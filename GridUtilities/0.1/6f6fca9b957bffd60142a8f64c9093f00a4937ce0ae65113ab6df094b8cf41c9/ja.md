```
History(datatype,[;htype::HistoryType = RegularHistory])
```

型 `datatype` のエントリを持つ空の履歴データベクターを作成します。代わりに、エントリの型の例となるインスタンスを渡すこともできます。オプションの引数 `htype` は履歴ベクターの型を指定します。デフォルトでは、これは `RegularHistory` ですが、代わりに `PeriodicHistory` に設定することもできます。後者の場合、履歴ベクターの長さが `n` の場合、`n+1` エントリは `1` エントリと同じであると見なされます。

`History` 型のルーチンを使用するためには、要素型 `datatype` が基本的な演算 `+`、`-`、スカラー乗算、および `fill!` を備えている必要があることに注意することが重要です。

別のコンストラクタは `History(h::History)` で、これは `h` と同じ型の履歴の空のインスタンスを作成します。 ```
