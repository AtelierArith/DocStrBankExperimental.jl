```
NodeReduction <: Operation
```

NodeReduction操作。2つ以上のノードを相互に削減することを表します。入力ノードのうち1つだけが保持され、他のすべては削除され、その親は保持されたノードの親に集約されます。

ノード削減が適用された後、グラフには`length(nr.input) - 1`個のノードが減少します。

# 成功した適用のための要件

ノードのベクトルは次の条件を満たす場合に削減できます：

  * すべてのノードがグラフに存在する。
  * すべてのノードが同じタスクタイプを持つ。
  * すべてのノードが同じ子ノードのセットを持つ。

[`is_valid_node_reduction_input`](@ref)を使用して、これらの要件を`@assert`できます。
