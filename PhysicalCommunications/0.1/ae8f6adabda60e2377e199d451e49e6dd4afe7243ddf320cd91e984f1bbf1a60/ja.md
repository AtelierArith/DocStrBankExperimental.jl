```
buildeye(x::Vector, y::Vector, tbit::Number, teye::Number; tstart::Number=0)
```

提供された `(x,y)` の `x` 値を `teye` の複数のウィンドウに折りたたむことによってアイダイアグラムを構築します。ウィンドウは各ビットの開始時点である "トリガー" に基づいて `tbit` ごとに開始されます。

トレースデータは、`(x,y)` サブベクトルのベクトルとして `DataEye` オブジェクトに格納されます。

入力:

  * tbit: ビット周期 (s)。各ビットの開始の "トリガーポイント" を定義します。
  * teye: アイ周期 (s)。 "トリガーポイント" の後にアルゴリズムが収集するアイデータの量 (x軸に沿って) を定義します。通常、これは `2.0*tbit` に設定されます。
  * tstart: 最初の "トリガーポイント" の時間 (s)。

Plots.jl を使用したプロットの例:

```
#仮定: ここで (x, y) データが生成されます。
tbit = 1e-9 #データビット周期が 1ns であると仮定します。

#アイを構築し、tstart を使用してデータを中央に配置します。
eye = buildeye(x, y, tbit, 2.0*tbit, tstart=0.2*tbit)

plot(eye.vx, eye.vy)
```
