```julia
Unknown(value = NaN; name = :u) 
```

`Unknown`はデフォルト値を持つ変数を作成するためのヘルパーです。デフォルト値は結果の型と形状を決定します。また、変数名が衝突しないようにメタデータを変数に追加します。表示可能な変数名は`gensym`に基づいています。`name`はメタデータとして保存され、方程式が`system`でフラット化されると、変数はサブシステム名と変数のベース名を含むように名前が変更されます。

例えば、`Unknown(name = :v)`は`var"##v#1057"(t)`として表示されるかもしれませんが、フラット化された後は`ss₊c1₊v(t)`のように表示されます（`ss`と`c1`はサブシステムです）。
