```
collapse_data(linked_data::AbstractDataFrame, translation_dict::Dict)
```

各タイムシリーズマイクロボットの軌跡を、各マイクロボットの要約データの単一行にまとめます。

[Translation Dictionary](@ref) からの translation_dict を使用して、実験データを含めます。

# 出力列の定義

  * `V` : 瞬時速度 `dp_um` の平均。常に正の値であり、これは大きさです。単位は µm/s。
  * `Vx` : x位置の数値微分の平均。正または負の値になる可能性があります。単位は µm/s。
  * `Vy` : y位置の数値微分の平均。正または負の値になる可能性があります。単位は µm/s。
  * `Area_um_mean` : マイクロボットの面積の平均。単位は µm^2。
  * `Ω_est` : リンクデータの `Major_um` 列のFFTから抽出された推定回転率。[`estimate_omega`](@ref) 関数を使用して実行されます。単位は Hz。
  * `R` : バウンディングサークルの半径/回転半径。主要軸 `Major_um` の95パーセンタイルを2で割った値として計算されます。単位は µm。
  * `Circularity` : フィットした楕円のアスペクト比に基づく定量化。ImageJ から計算されます。詳細は [their docs](https://imagej.nih.gov/ij/docs/menus/analyze.html) を参照してください。単位なし。
  * `total_displacement_um` : マイクロボットの *全軌跡* にわたる総移動量。これはすでに `linked_data` で一定であるため、値の1つを取るだけです。単位は µm。
