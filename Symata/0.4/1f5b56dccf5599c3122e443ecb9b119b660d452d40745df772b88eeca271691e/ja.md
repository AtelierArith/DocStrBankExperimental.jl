```
symatamath()
```

は、`Symbols` と `Numbers` の間、および `Symbols` 同士の間の算術演算子 `*`、`+`、`-`、`^` のための Base メソッドを定義します。`symatamath()` は、Symata を実行し、J(expr) を使用して Julia コードを評価する際には *必要ありません*。`J()` は、これらのメソッドがすでに定義されている Symata モジュール内で評価を行います。`symatamath()` は、式が `Main` で評価される Julia モードで便利です。

これらのメソッドは、デフォルトでは `Main` には定義されていません。なぜなら、コア Julia オブジェクト間の Base メソッドを定義すると、将来の Julia のバージョンと競合する可能性があるからです。
