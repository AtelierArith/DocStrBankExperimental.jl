```
gcop2arch(x::Matrix{Real}, inds::Vector{Pair{String,Vector{Int64}}}; naive::Bool = false, notnested::Bool = false, rng = Random.GLOBAL_RNG)
```

xはガウスのn変量分布からのt回のデータの行列を取ります。選択された周辺部分集合（inds[i][2]）がアーキメデスのサブコピュラ（inds[i][1]で示される）を持つ、xと同じサイズの行列を返します。すべての単変量周辺は変更されません。

例えば、inds = ["clayton" => [1,2]]は、インデックス1,2の周辺部分集合がクレイトンのサブコピュラに変更されることを意味します。indsにはより多くの要素が含まれる場合がありますが、周辺は重複してはいけません。

naiveがtrueの場合、ナイーブな再サンプリングが使用されます。notnestedがtrueの場合、ネストされたアーキメデスコピュラは使用されません。
