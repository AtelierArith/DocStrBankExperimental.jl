```
modeshape_extraction(freq, H, fn, ξn, dpi; type = :dis)
```

Sdof近似を使用してモードシェイプを抽出します

**入力**

  * `freq`: 周波数ベクトル
  * `H`: 周波数応答関数
  * `fn`: 自然周波数
  * `ξn`: 減衰比
  * `dpi`: 駆動点インデックス

      * `dpi[1]`: 測定メッシュ上の駆動点インデックス
      * `dpi[2]`: 励起メッシュ上の駆動点インデックス
  * `method`: モードシェイプを抽出するための方法

      * `PeakPicking`: ピークピッキング法（デフォルト）
      * `CircleFit`: 円フィッティング法
  * `type`: 自然周波数と減衰比を抽出するために使用されるFRFのタイプ

      * `:dis`: アドミタンス（デフォルト）
      * `:vel`: モビリティ
      * `:acc`: 加速度

**出力**

  * `ϕn`: モードシェイプ

**注意** 測定点の数が励起点の数より少ない場合、モードシェイプは励起点で推定されます（ロービングハンマーテスト）。そうでない場合、モードシェイプは測定点で推定されます（ロービング加速度計テスト）。
