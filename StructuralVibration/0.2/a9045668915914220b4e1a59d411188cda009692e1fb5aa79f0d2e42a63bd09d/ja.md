```
HarmonicModalTimeProblem(K, M, ξn, u0, t, F, ω = 0., n = size(K, 1); ismodal = false)
```

強制応答を計算するためのモーダル時間ソルバーのデータを含む構造体

**コンストラクタ**

  * `K::VecOrMat{Real}`:

      * `ismodal = false` の場合: 剛性行列
      * `ismodal = true` の場合: 自然角周波数のベクトル
  * `M::AbtractMatrix`:

      * `ismodal = false`: 質量行列
      * `ismodal = true`: 質量正規化されたモード形状
  * `ξn`: 減衰比
  * `u0::Tuple`: 初期条件

      * `u0[1]`: 初期変位（またはモーダル変位）
      * `u0[2]`: 初期速度（またはモーダル速度）
  * `t::AbstractRange`: 応答を評価する時間点
  * `F::Real`: 外部力行列（またはモーダル参加係数）
  * `freq::Real`: 励振周波数
  * `n::Int`: モーダル基底に保持するモードの数
  * `ismodal::Bool`: 問題がモーダルデータを含むかどうかを示すフラグ

**フィールド**

  * `K`: 剛性行列（またはモーダル剛性行列）
  * `M`: 質量行列（または質量正規化されたモード形状）
  * `ξn`: 減衰比
  * `u0`: 初期条件

      * `u0[1]`: 初期変位（またはモーダル変位）
      * `u0[2]`: 初期速度（またはモーダル速度）
  * `t`: 応答を評価する時間点
  * `F`: 振幅ベクトル（またはモーダル参加ベクトル）
  * `ω`: 励振角周波数
  * `n`: モーダル基底に保持するモードの数
  * `ismodal`: 問題がモーダルデータを含むかどうかを示すフラグ
