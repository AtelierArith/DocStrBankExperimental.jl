```
Plate(L, b, h, E, ρ, ν)
```

Structure containing the data of a homogeneous and isotropic bending plate

**Constructor parameters**

  * `L`: Length [m]
  * `b`: Width [m]
  * `E`: Young's modulus [Pa]
  * `ρ`: Density [kg/m³]
  * `ν`: Poisson's ratio

**Fields**

  * `L`: Length [m]
  * `b`: Width [m]
  * `m`: Surface mass [kg/m²]
  * `D`: Bending stiffness [N.m]
