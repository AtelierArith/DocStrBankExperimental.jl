```
Strings(L, S, D, ρ)
```

Structure containing the data of a homogeneous and isotropic string

**Constructor parameters**

  * `L`: Length [m]
  * `S`: Cross-section area [m²]
  * `D`: Tension [N]
  * `ρ`: Mass density [kg/m³]

**Fields**

  * `L`: Length [m]
  * `m`: Linear mass density [kg/m]
  * `D`: Tension [N]
