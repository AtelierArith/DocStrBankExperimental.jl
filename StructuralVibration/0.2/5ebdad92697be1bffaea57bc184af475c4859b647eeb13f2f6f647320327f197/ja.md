```
detrend(t, y, order = 1, bp = [])
```

信号 `y` を時間 `t` に対して、次数 `order` の多項式を使用してトレンドを除去します。

**入力**

  * `t`: 時間ベクトル
  * `y`: トレンドを除去する信号
  * `order`: 多項式の次数（デフォルトは1）
  * `bp`: 多項式のブレークポイント（デフォルトは空）

**出力**

  * `y_detrended`: トレンドを除去した信号

**注意事項**

`order` は、ブレークポイント `bp` によって定義されたセグメントごとにトレンドが異なる場合、ベクトルであることができます。

ブレークポイント `bp` は、多項式の次数が変わる点です。`bp` が空の場合、全体の信号に対して単一の多項式がフィッティングされます。 ```
