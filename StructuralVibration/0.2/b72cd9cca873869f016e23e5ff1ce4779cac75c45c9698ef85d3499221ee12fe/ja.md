```
freq_extraction(freq, H, method::SdofModalExtraction; type = :dis)
```

ボード図フィッティング法から固有周波数と減衰比を抽出します

**入力**

  * `freq`: 周波数ベクトル
  * `H`: 周波数応答関数
  * `method`: 固有周波数と減衰比を抽出するための方法

      * `PeakPicking`: ピークピッキング法（デフォルト）
      * `CircleFit`: 円フィッティング法
  * `type`: 固有周波数と減衰比を抽出するために使用されるFRFのタイプ

      * `:dis`: アドミタンス（デフォルト）
      * `:vel`: モビリティ
      * `:acc`: 加速度

**出力**

  * `fn`: 固有周波数
  * `ξn`: 減衰比

**注意** `H`はFRFの行または列であると想定されており、Hはベクトルであることを意味します。
