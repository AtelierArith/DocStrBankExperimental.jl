```
unitInputDemandGeneral(xT::Vector{Float64}, z::Real, b_g::Function, e_h::Vector{Function}) -> Vector{Float64}
```

タスク空間における H-1 の閾値の配列 `xT`、生産性値 `z`、タスク分布のための密度関数 `b_g`、およびタスクの複雑さの関数としての各労働タイプのコストを表す H 関数の配列 `e_h` に基づいて、単位労働需要を計算します。

この関数はまず、`b_g` が有効な密度関数であることを確認します。次に、`xT` に定義された閾値によって定義された区間で、比率 `b_g(x) / (z * e_h[h](x))` を数値的に積分することによって、各労働タイプの労働需要を計算します。

# 引数

  * `xT`: タスク空間における H-1 の閾値のベクトル。
  * `q`: 生産価値
  * `z`: 生産性値。
  * `b_g`: タスク分布のための密度関数。
  * `e_h`: タスクの複雑さの関数としての各労働タイプのコストを表す H 関数のベクトル。

# 戻り値

  * 各労働タイプの労働需要を表すベクトル。
