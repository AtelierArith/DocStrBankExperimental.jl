```
System(; latvecs::Vector{NTuple{3, Float}},
         pos::Vector{Vec3}
         c0::Float64 = 6.0
         c1::Float64 = 2.0)
```

三つのスーパーセル格子ベクトル `latvecs` において周期的なシステムを作成し、原子位置 `pos` を持ちます。オプションのパラメータ `c0` はエヴァルド和の精度を制御します。デフォルトは `c0 = 6` で、誤差は `1e-12` のオーダーになります（小さい `c0` はより速く実行されます）。オプションのパラメータ `c1` は実空間とフーリエ空間の和のバランスを制御します（大きい `c1` はより多くの実空間和を意味します）。
