```
β = logdensityderivatives( X, x, h; 様々なオプション引数 )

データベクター X の対数密度の導関数の推定値を、バンド幅 h を使用してベクター x の要素で評価します。評価点に対応する行と導関数に対応する列を持つ行列 β を返します。 S = 1 の場合でも、これは行列です。
```

# 必須引数

  * `X::Vector{T}`: データのベクター
  * `x::Vector{T}`: 評価点のベクター
  * `h::T`: バンド幅

T は浮動小数点型です。

# オプション引数

  * `g::Function=g`: 使用する関数 g (Pinkse と Schurter を参照)
  * `dg::Function=dg`: その導関数
  * `S::Int=1`: 計算される導関数の数
  * `minx::T=0.0`: サポートの左側
  * `maxx::T=Inf`: サポートの右側
  * `anal::Bool=true`: 多くのサニティチェックを実行するかどうか
