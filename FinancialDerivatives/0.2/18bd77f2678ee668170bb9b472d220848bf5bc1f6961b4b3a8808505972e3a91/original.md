```
AsianOption(s, k, r, σ, t, call)
```

[Asian option](https://en.wikipedia.org/wiki/Option_style#Asian_option).

# Arguments

  * `s`: underlying price
  * `k`: strike price
  * `r`: risk-free interest rate
  * `σ`: volatility
  * `t`: time to expiration
  * `call`: `true` if call, `false` if put
