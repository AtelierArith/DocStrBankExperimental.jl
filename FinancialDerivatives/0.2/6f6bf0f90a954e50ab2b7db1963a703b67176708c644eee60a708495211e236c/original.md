```
EuropeanOption(s, k, r, σ, t, call)
```

[European option](https://en.wikipedia.org/wiki/Option_style#American_and_European_options).

# Arguments

  * `s`: underlying price
  * `k`: strike price
  * `r`: risk-free interest rate
  * `q`: dividend yield
  * `σ`: volatility
  * `t`: time to expiration
  * `call`: `true` if call, `false` if put
