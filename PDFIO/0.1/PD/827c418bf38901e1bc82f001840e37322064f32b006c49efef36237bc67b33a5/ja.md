```
    pdDocGetPageLabel(doc::PDDoc, pageno::Int) -> String
```

ページに関連付けられたページラベルがある場合、そのページラベルを返します。

PDF仕様1.7のセクション12.4.2によれば、文書はオプションでページラベル（PDF 1.3）を定義して、各ページを画面上または印刷時に視覚的に識別することができます。ページラベルとページインデックスは一致する必要はありません：インデックスは固定され、最初のページから0から始まり文書全体を通じて連続して進行しますが、ラベルは特定の文書に適した方法で指定することができます。

# 例

```
julia> pdDocGetPageLabel(doc, 3)
"ii"
```
