```
    pdDocHasPageLabels(doc::PDDoc) -> Bool
```

ドキュメントにページラベルが定義されている場合は `true` を返します。

PDF仕様1.7のセクション12.4.2によれば、ドキュメントはオプションでページラベル（PDF 1.3）を定義して、画面上または印刷時に各ページを視覚的に識別することができます。ページラベルとページインデックスは一致する必要はありません：インデックスは固定され、最初のページから0から始まってドキュメント全体を通じて連続して進む必要がありますが、ラベルは特定のドキュメントに適した方法で指定することができます。

# 例

```
julia> PDFIO.PD.pdDocHasPageLabels(doc)
false
```
