```
    pdDocGetOutline(doc::PDDoc) -> PDOutline
```

PDFドキュメントは、`Document Catalog`辞書に利用可能なドキュメントのアウトライン（目次）を提供します。ドキュメントにアウトラインがない場合、このメソッドは`nothing`を返します。

PDFドキュメントには、準拠したリーダーが画面に表示できるドキュメントのアウトラインが含まれている場合があり、ユーザーはドキュメントの一部から別の部分へインタラクティブにナビゲートできます。アウトラインは、アウトライン項目（時にはブックマークと呼ばれる）のツリー構造の階層で構成されており、ユーザーにドキュメントの構造を表示する視覚的な目次として機能します。ユーザーは、マウスでクリックすることで個々の項目をインタラクティブに開いたり閉じたりできます。項目が開いていると、その階層内の直下の子項目が画面に表示されます；各子項目もまた開いたり閉じたりでき、階層のさらなる部分を選択的に表示または非表示にします。項目が閉じていると、その階層内のすべての子孫は隠されます。表示されている項目のテキストをクリックすると、その項目がアクティブになり、準拠したリーダーが目的地にジャンプしたり、その項目に関連付けられたアクションをトリガーしたりします。 - セクション 12.3.3 - ドキュメント管理 — ポータブルドキュメントフォーマット — パート 1: PDF 1.7
