MapTiler(variant; key)

[`Provider`](@ref) for MapTiler tiles.

(C) MapTiler (C) OpenStreetMap contributors

# 引数

  * `variant`: :Streets, :Basic, :Bright, :Pastel, :Positron, :Hybrid, :Toner, :Topo, :Voyager, :Ocean, :Backdrop および :Dataviz、デフォルトは Streets です。

## キーワード

  * `key`: MapTiler サービス用の API キー。
