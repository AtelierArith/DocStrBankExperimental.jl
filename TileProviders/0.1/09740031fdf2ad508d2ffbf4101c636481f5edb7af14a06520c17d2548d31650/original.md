MapTiler(variant; key)

[`Provider`](@ref) for MapTiler tiles.

(C) MapTiler (C) OpenStreetMap contributors

# Arguments

  * `variant`: :Streets, :Basic, :Bright, :Pastel, :Positron, :Hybrid, :Toner, :Topo, :Voyager, :Ocean, :Backdrop and :Dataviz, with a default of Streets.

## Keywords

  * `key`: Your API key for the MapTiler service.
