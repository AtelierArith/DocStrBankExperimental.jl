Jawg(variant; accesstoken)

[`Provider`](@ref) for Jawg tiles.

(C) **Jawg** Maps (C) OpenStreetMap contributors

# Arguments

  * `variant`: :Streets, :Terrain, :Lagoon, :Sunny, :Dark, :Light and :Matrix, with a default of Streets.

## Keywords

  * `accesstoken`: Your API key for the Jawg service.
