OpenWeatherMap(variant; apikey)

[`Provider`](@ref) for OpenWeatherMap tiles.

Map data (C) OpenWeatherMap

# Arguments

  * `variant`: :Clouds, :CloudsClassic, :Precipitation, :PrecipitationClassic, :Rain, :RainClassic, :Pressure, :PressureContour, :Wind, :Temperature and :Snow, with a default of Clouds.

## Keywords

  * `apikey`: Your API key for the OpenWeatherMap service.
