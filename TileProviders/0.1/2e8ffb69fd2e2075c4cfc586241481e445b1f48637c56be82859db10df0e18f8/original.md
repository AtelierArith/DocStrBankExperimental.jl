Thunderforest(variant; apikey)

[`Provider`](@ref) for Thunderforest tiles.

(C) Thunderforest, (C) OpenStreetMap contributors

# Arguments

  * `variant`: :OpenCycleMap, :Transport, :TransportDark, :SpinalMap, :Landscape, :Outdoors, :Pioneer, :MobileAtlas and :Neighbourhood, with a default of OpenCycleMap.

## Keywords

  * `apikey`: Your API key for the Thunderforest service.
