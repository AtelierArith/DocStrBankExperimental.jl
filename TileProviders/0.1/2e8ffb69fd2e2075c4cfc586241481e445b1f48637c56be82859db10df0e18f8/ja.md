Thunderforest(variant; apikey)

[`Provider`](@ref) for Thunderforest tiles.

(C) Thunderforest, (C) OpenStreetMap contributors

# 引数

  * `variant`: :OpenCycleMap, :Transport, :TransportDark, :SpinalMap, :Landscape, :Outdoors, :Pioneer, :MobileAtlas および :Neighbourhood、デフォルトは OpenCycleMap です。

## キーワード

  * `apikey`: Thunderforest サービス用の API キー。
