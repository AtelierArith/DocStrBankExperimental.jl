AzureMaps(variant; subscriptionkey)

[`Provider`](@ref) for AzureMaps tiles.

See https://docs.microsoft.com/en-us/rest/api/maps/render-v2/get-map-tile for details.

# Arguments

  * `variant`: :MicrosoftImagery, :MicrosoftBaseDarkGrey, :MicrosoftBaseRoad, :MicrosoftBaseHybridRoad, :MicrosoftTerraMain, :MicrosoftWeatherInfraredMain and :MicrosoftWeatherRadarMain, with a default of MicrosoftImagery.

## Keywords

  * `subscriptionkey`: Your API key for the AzureMaps service.
