# Arguments

  * `variant`: :Positron, :PositronNoLabels, :PositronOnlyLabels, :DarkMatter, :DarkMatterNoLabels, :DarkMatterOnlyLabels, :Voyager, :VoyagerNoLabels, :VoyagerOnlyLabels and :VoyagerLabelsUnder, with a default of Positron.
