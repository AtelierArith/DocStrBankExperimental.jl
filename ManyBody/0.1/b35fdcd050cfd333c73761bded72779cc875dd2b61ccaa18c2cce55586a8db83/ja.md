フェルミオン生成/消滅演算子の積を表現します。

フレーバーのフォック空間において、これは任意の数の生成および消滅演算子の積の抽象的な表現であり、必ずしも通常順序付けされているわけではありません。このような各積を標準形で書くことができます：

```
t(v,i,j,k) = v * prod(x -> c[x]', i) * prod(x -> c[x], reverse(j)) *
             prod(x -> c[x]*c[x]', k)
```

ここで `i`、`j` はフレーバーのタプルであり、それぞれ昇順に並べられています。また `k` は `i` および `j` の両方と互いに素なフレーバーのタプルです。上記の表現は通常順序付けされた積ではなく、項の効果に関する各フレーバーの「分類」を示しています。

このような積の要素に必要な `O(2^(2n))` の密なストレージまたは `O(2^n)` の疎なストレージの代わりに、このクラスは高速なオンザフライ計算を可能にする `O(1)` のビットフィールドのセットのみを格納します。
