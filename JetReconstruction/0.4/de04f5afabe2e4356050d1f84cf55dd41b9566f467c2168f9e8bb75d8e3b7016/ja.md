```
savejets(filename, jets; format="px py pz E")
```

ジェットデータをファイルに保存します。

# 引数

  * `filename`: ジェットデータを保存するファイルの名前。
  * `jets`: 保存するジェットオブジェクトの配列。
  * `format="px py pz E"`: （オプション）保存するジェットデータのフォーマットを指定する文字列。デフォルトのフォーマットは "px py pz E" です。

# 詳細

この関数は、特定のフォーマットでファイルにジェットデータを保存します。ファイル内の各行は1つのジェットを表し、指定されたフォーマットでジェットに関する情報を含みます。フォーマット文字列には以下のプレースホルダーを含めることができます：

  * "E" または "energy": ジェットエネルギー
  * "px": x軸に沿った運動量
  * "py": y軸に沿った運動量
  * "pz": z軸に沿った運動量
  * "pt2": 横運動量の二乗
  * "phi": 方位角
  * "rapidity": ラピディティ

'#' で始まる行はコメントとして扱われ、無視されます。

`format` 文字列には値や（おそらくカスタムの）区切り文字以外のものを入れることは強く推奨されません。
