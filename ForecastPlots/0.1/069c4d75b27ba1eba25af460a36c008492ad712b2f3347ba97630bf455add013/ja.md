パッケージ: ForecastPlots

```
splot(x, labels; plot = true)
```

パラメータ `labels` に基づいて x の季節プロットを描画します。

# 引数

  * `x`: 定期的な観測値
  * `labels`: このパラメータは整数、文字列、ベクトルの値を受け入れます。            - 整数の場合、ラベルは 1:labels です。           - ベクトルの場合、指定されたソートされたラベルが含まれます。           - 文字列の場合、"month"、"day"、"quarter" の値を受け入れ、デフォルトでは x の最初の値が "Jan"、"Mon"、または "Q1" に該当します。他の初期値が必要な場合は、ラベルをベクトルで指定する必要があります。
  * `plot`: `true` の場合、splot が表示され、それ以外の場合は季節ごとの列行列のみが返されます。

# 戻り値

オプションのプロット付きの列ごとの季節データの行列

# 例

splot(rand(120),"month");
splot(rand(120),"quarter");
splot(rand(120),"day"; plot=false)
