パッケージ: ForecastPlots

```
ccf(x1::{AbstractVector,DataFrame},
    x2::{AbstractVector,DataFrame};
    type = "cor",
    lag = Integer(ceil(10*log10(length(x1)))),
    alpha = (0.95,0.99);
    plot = true)
```

2つの単変量系列のクロス相関またはクロス共分散を計算し、オプションでプロットを表示します。

結果はホモスケダスティシティを保持するように正規化されています。データを正規化するために使用される分布は、正規分布を介したフィッシャー変換の近似です。返されるオブジェクトにはプロットレシピがあり、タイプが`cor`の場合、プロットは指定されたアルファ値の信頼区間も表示します。

与えられた整数`k`に対して、`x2`が`x1`の値を繰り返す場合、すなわちx1[t] = x2[t+k]がすべての`i`に対して成り立つ場合、高い相関値は結果の*中心の右側*に配置されます。つまり、この慣習はプロットで`x1_t = x2_{t+k} -> _____0__k__`として表され、x2の挙動はk単位でx1によって予測できることを意味します。

# 引数

  * `x1`: データのベクトルまたは1次元データフレーム。
  * `x2`: データのベクトルまたは1次元データフレーム。
  * `type`: 有効な値は相関のための"cor"（デフォルト）と共分散のための"cov"です。
  * `lag`: 最大ラグ数。
  * `alpha`: 信頼区間をプロットするための2つの閾値(t1,t2)を持つタプル。デフォルト値は0.95と0.99です。
  * `plot`: trueの場合、結果を含むプロットを表示します。

# 戻り値

オプションのプロットを伴うクロス相関ベクトル

# 例

```julia-repl
x1 = rand(100);
x2 = circshift(x1,6);
ccf(x1, x2; type="cor");
ccf(x1, x2; type="cor", plot = false)

  [...]
```
