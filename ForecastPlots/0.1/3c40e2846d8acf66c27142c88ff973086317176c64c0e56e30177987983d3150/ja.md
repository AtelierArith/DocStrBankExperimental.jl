パッケージ: ForecastPlots

```
pacf(x,
     type = "step-real",
     lag = Integer(ceil(10*log10(length(x)))),
     alpha = (0.95,0.99);
     plot = true)
```

オプションのプロットを伴う単変量系列の部分自己相関を計算します。

2つのバージョンがあります。「ステップ」バージョンは増加モデルの自己回帰パラメータを推定し、「リアル」バージョンはラグによって提供される線形情報を排除することによって実際の部分自己相関を推定します。デフォルトのタイプ「ステップワイズ-リアル」を使用すると、両方のバージョンが計算されます。

信頼区間を推定するために使用される分布は、正規分布を介したフィッシャー変換の近似です。返されるオブジェクトにはプロットレシピがあります。

# 引数

  * `x`: データのベクトル。
  * `type` = 有効な値は「ステップワイズ」、「リアル」、および「ステップワイズ-リアル」です。
  * `lag`: 最大ラグ数。
  * `alpha`: 信頼区間をプロットするための2つの閾値(t1,t2)を持つタプル。デフォルト値は0.95と0.99です。
  * `plot`: trueの場合、結果を含むプロットを表示します。

# 戻り値

CFオブジェクト

# 例

```julia-repl
x = rand(100);
pacf(x);
pacf(x; plot=false)
20×2 Matrix{Float64}:
 -0.0475437   -0.101204
  0.0407006    0.058935
 -0.0747269    0.0366026
```
