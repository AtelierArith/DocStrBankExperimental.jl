パッケージ: ForecastPlots

```
acf(x;
    type = "cor",
    lag = Integer(ceil(10*log10(length(x)))),
    alpha = (0.95,0.99);
    plot = true)
```

オプションのプロットを伴う単変量系列の自己相関または自己共分散を計算します。

結果は均一分散性を保持するように正規化されます。データを正規化するために使用される分布は、正規分布を介したフィッシャー変換の近似です。返されるオブジェクトにはプロットレシピがあり、タイプが `cor` の場合、プロットは指定されたアルファ値の信頼区間も表示します。

# 引数

  * `x`: データのベクトルまたは一次元データフレーム。
  * `type`: 有効な値は "cor"（デフォルト）で相関、"cov" で共分散です。
  * `lag`: 最大ラグ数。
  * `alpha`: 信頼区間をプロットするための2つの閾値 (t1,t2) を持つタプル。デフォルト値は 0.95 と 0.99 です。
  * `plot`: true の場合、結果を含むプロットを表示します。

# 戻り値

CFオブジェクトとオプションのプロット

# 例

```julia-repl
x = rand(100);
acf(x; type="cor");
acf(x; type="cor", plot=false)
```
