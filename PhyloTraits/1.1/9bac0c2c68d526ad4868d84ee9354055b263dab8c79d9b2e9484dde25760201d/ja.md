```
JC69(rate, relative)
```

ジュークス・カントール（1969）核酸置換モデルで、単一の速度パラメータを持ちます。`rate`は絶対対角要素に対応し、すなわち他の2つの状態への変化の速度です。個々の速度は`rate`/3です。`relative`がtrue（デフォルト）の場合、遷移行列[`Q`](@ref)は単位時間あたりの遷移の平均が1になるように正規化されます。この場合、`rate`は1.0に設定されます。

```jldoctest
julia> m1 = JC69([0.25], false)

絶対速度バージョン

速度行列Q:

       A       *  0.0833  0.0833  0.0833
       C  0.0833       *  0.0833  0.0833

       T  0.0833  0.0833  0.0833       *

       T  0.3333  0.3333  0.3333       *

julia> nparams(m2)
0
```
