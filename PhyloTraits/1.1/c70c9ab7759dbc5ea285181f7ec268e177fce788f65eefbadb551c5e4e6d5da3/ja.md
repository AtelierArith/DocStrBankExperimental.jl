```
ParamsMultiBM <: ParamsProcess
```

ネットワーク上の多変量ブラウン運動拡散（MBD）プロセスの型。フィールドは `mu`（期待値）、`sigma`（共分散行列）、`randomRoot`（ルートがランダムかどうか、デフォルトは `false`）、`varRoot`（ルートがランダムな場合、ルートの共分散行列、デフォルトは `[NaN]`）、`shift`（ShiftNet型、デフォルトは `missing`）、および `L`（`sigma` のコレスキー分解の下三角行列、自動的に計算される）

# 例とコンストラクタ

```jldoctest
julia> ParamsMultiBM([1.0, -0.5], [2.0 0.3; 0.3 1.0]) # シフトなし
ParamsMultiBM:
固定ルートのMBDのパラメータ:
mu: [1.0, -0.5]
Sigma: [2.0 0.3; 0.3 1.0]

julia> net = readnewick("((A:1,B:1):1,C:2);");

julia> shifts = ShiftNet(net.node[2], [-1.0, 2.0], net);

julia> ParamsMultiBM([1.0, -0.5], [2.0 0.3; 0.3 1.0], shifts) # シフトあり
ParamsMultiBM:
固定ルートのMBDのパラメータ:
mu: [1.0, -0.5]
Sigma: [2.0 0.3; 0.3 1.0]

ネットワーク上に2つのシフトがあります:
───────────────────────────────────────────
  エッジ番号  シフト値1  シフト値2
───────────────────────────────────────────
          2.0           -1.0            2.0
───────────────────────────────────────────

```
