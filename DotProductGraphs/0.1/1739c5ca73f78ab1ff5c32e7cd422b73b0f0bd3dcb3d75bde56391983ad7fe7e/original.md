```
`d_elbow(Σ)`
```

Estimates the optimal embedding dimension using the Elbow method.

Used by default in `truncated_svd` is `d` is left to `nothing`.

See M. Zhu, and A. Ghodsi (2006). Automatic dimensionality selection from the scree plot via the use of profile likelihood. *Computational Statistics and Data Analysis*, Vol. 51, 918–930.
