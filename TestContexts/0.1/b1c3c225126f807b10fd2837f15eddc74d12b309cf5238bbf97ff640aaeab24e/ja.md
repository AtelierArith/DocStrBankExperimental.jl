```
test_set(body::Function, name::String, data...)
```

`@testset`に似ていますが、指定された`name`の下でテストを実行するためにグローバルな`tc`テストコンテキストを使用します。オプションの`data`は、各エントリが`property => value`の形式であるゼロ個以上のエントリの系列でなければなりません。ここで、`property`はシンボルで、`value`は`PropertyValue`です。

テストセットのネストは許可されています。これは、実際のテストケースのためにテスト環境を段階的に設定する一般的なパターンです。BDD用語では、テストセットは「given」または「when」句に似ています。

プロパティ値にはまだアクセスできません。データに実際にアクセスするには`test_case`を参照してください。
