```
ipmt(rate::Real, per::Integer, nper::Integer, pv::Real, fv = 0.0, when = :end)
```

定期支払いの利息成分を計算します。これは、返済スケジュールのあるローンに役立ちます。

# 例

例えば、300000（現在価値）の住宅ローンを、月利4.25%で今後30年間返済する必要があるとします。私の月々の支払いの利息成分はどのくらいになるでしょうか？最初は、利息成分が支払いの大部分を占めますが、終わりに近づくにつれて、以下のように小さな部分になります。期間1、2、3、358、359、360の例を示します：

```
julia> pmt(0.0425/12, 30*12, 300000)
-1475.8196732384283

julia> ipmt.(0.0425/12, [1,2,3,358,359,360], 30*12, 300000)
6-element Array{Float64,1}:
 -1062.5
 -1061.04
 -1059.57
   -15.5702
   -10.3984
    -5.20841
```
