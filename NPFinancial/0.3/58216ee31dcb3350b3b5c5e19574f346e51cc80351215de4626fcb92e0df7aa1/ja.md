```
pmt(rate::Real, nper::Integer, pv::Real, fv = 0.0, when = :end)
```

現在価値 `pv`、期間ごとに1回複利計算される金利 `rate`、および `nper` 期間の数を考慮して、最後の期間の終わりに値が `fv` になるように支払いを計算します。支払いは、`when` 引数で指定されたように、各期間の開始 (`:begin`) または終了 (`:end`) に行われることが期待されます。

# 例

例えば、300000（現在価値）の住宅ローンを返済する必要があり、次の30年間で月利4.25%の金利がかかるとします。私の月々の支払いはいくらになりますか？

```
julia> pmt(0.0425/12, 30*12, 300000)
-1475.8196732384283
```
