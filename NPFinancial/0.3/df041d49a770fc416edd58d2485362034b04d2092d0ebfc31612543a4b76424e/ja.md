```
npv(rate::Real, values::AbstractVector{<:Real})
```

キャッシュフローシリーズ `values` の正味現在価値 (NPV) を、内部収益率 `rate` に基づいて計算します。

キャッシュフローの「イベント」間の（固定）時間間隔は、`rate` が与えられたものと同じでなければなりません（つまり、`rate` が年単位の場合、各キャッシュフローイベントの間には正確に1年が経過することが理解されます）。慣例として、投資または「預金」は負、収入または「引き出し」は正とされます。したがって、`values` は初期投資から始まる必要があり、通常 `values[1]` は負になります。

# 例

```
julia> npv(0.281,[-100, 39, 59, 55, 20])
-0.00847859163845488
```
