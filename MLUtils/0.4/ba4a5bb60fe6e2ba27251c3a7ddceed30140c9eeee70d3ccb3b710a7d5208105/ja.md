```
batch(xs)
```

`xs`の配列を追加の次元を持つ単一の配列にバッチ処理します。

`xs`の要素がタプル、名前付きタプル、または辞書である場合、出力は同じ型になります。

[`unbatch`](@ref)および[`batch_sequence`](@ref)も参照してください。

# 例

```jldoctest
julia> batch([[1,2,3], 
              [4,5,6]])
3×2 Matrix{Int64}:
 1  4
 2  5
 3  6

julia> batch([(a=[1,2], b=[3,4])
               (a=[5,6], b=[7,8])]) 
(a = [1 5; 2 6], b = [3 7; 4 8])
```
