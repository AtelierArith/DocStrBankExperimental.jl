```
leavepout(data, p = 1)
```

k-fold戦略を使用して`data`コンテナを再分割します。ここで、`k`は、結果のフォルダの各検証サブセットが大体`p`の観測値を含むように選ばれます。デフォルトは`p = 1`で、これは「leave-one-out」分割とも呼ばれます。

結果のフォルダのシーケンスは遅延イテレータとして返されます。実際のデータは[`getobs`](@ref)が呼び出されるまでコピーされません。

```julia
for (train, val) in leavepout(X, p=2)
    # numobs(X)が2で割り切れる場合、
    # 各イテレーションでnumobs(val)は2になります。
    # そうでない場合、最初の数回のイテレーションでは3になることがあります。
end
```

関連する関数については[`kfolds`](@ref)を参照してください。
