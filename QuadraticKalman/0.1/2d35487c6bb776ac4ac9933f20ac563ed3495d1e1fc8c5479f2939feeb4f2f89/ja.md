```
QKFOutput{T<:Real}
```

Quadratic Kalmanアルゴリズムからのフィルタリングおよびスムージング出力のための統合コンテナ。

このタイプは、過去および現在の観測に基づいて状態推定と関連する共分散を再帰的に計算する前方フィルタリングパスの結果と、将来の観測を取り入れることによってこれらの推定を洗練する後方スムージングパスの結果をカプセル化します。統一された構造は、診断分析、視覚化、およびさらなるモデルベースの推論タスクのための明確で便利なインターフェースを提供します。

# フィールド

  * `filter::FilterOutput{T}`: フィルタリングプロセスの出力を含み、各時間ステップでのフィルタリングされた対数尤度、状態推定、および共分散行列などが含まれます。これらの結果は、データが観測されたリアルタイムで得られたモデルの推定を表します。
  * `smoother::SmootherOutput{T}`: スムージングプロセスの出力を含み、全観測シーケンスからの情報を活用してフィルタ結果を洗練し改善します。これには通常、スムーズな状態推定と対応する共分散行列が含まれ、基礎となる状態動態のより正確な再構築を提供します。

# 詳細

QKFOutput構造を使用することで、ユーザーは瞬時の（フィルタリング）および遡及的に改善された（スムージング）推定の両方に便利にアクセスでき、事後分析、診断、または予測を行いやすくなります。
