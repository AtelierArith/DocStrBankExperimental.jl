```
KalmanSmootherPlot{M<:SmootherOutput{<:Real}}
```

カルマンスムーザー出力の視覚化を生成するためのコンテナ。このタイプは、スムーザーの結果をカプセル化しており、通常はスムーズな状態推定と対応する共分散行列を含み、これらの出力を信頼区間と共にプロットするための統一インターフェースを提供します。

# フィールド

  * `results::M`: 次の内容を含む `SmootherOutput` のインスタンス：

      * `Z_smooth`: スムーズな状態推定の行列。
      * `P_smooth`: スムーズな状態共分散の三次元配列。

    これらの出力は、将来の情報を使用して状態推定を洗練する逆再帰を実行することによって生成されます。

# 使用法

このタイプは、スムーザーのパフォーマンスを表示する診断プロットを生成するために、プロットレシピによって内部的に使用されます。標準的な視覚化により、ユーザーはスムーズな状態軌道を真の状態値（または他のベンチマーク）と視覚的に比較でき、不確実性は信頼区間として表されます。これにより、モデルの推定効率と収束特性の詳細な分析が可能になります。

# 詳細

スムーザーの結果をカプセル化することにより、`KalmanSmootherPlot` はスムージング診断をプロットフレームワークに統合することを簡素化します。関連するプロットは通常、次のことを示します：

  * 時間にわたる推定状態軌道。
  * これらの推定の不確実性、通常は95%信頼区間に対応する陰影のある領域を使用して描かれます。

この強化された視覚化は、スムージング手続きの精度と信頼性を評価するのに役立ちます。
