```
QKModel{T<:Real, T2<:Real}
```

二次カルマンフィルタに必要なすべてのパラメータとモーメントを含む主要な構造。

この複合モデルは、二次状態空間モデルを指定するために必要なすべてのコンポーネントをカプセル化します。標準的な状態進化パラメータ、二次項を含む測定パラメータ、より豊かなダイナミクスのための拡張状態パラメータ、時間にわたる状態の挙動を要約する無条件モーメントをまとめています。この設計により、統合されたフィルタリングおよびスムージング手法が統一されたフレームワーク内で可能になります。

# フィールド

  * `state::StateParams{T}`: 状態プロセスを支配するパラメータで、方程式 X*t = μ + Φ X*{t-1} + Ω ε*t によって定義されます。ここで、μ は初期状態の平均、Φ は状態遷移行列、Ω はプロセスノイズ ε*t をスケーリングします。
  * `meas::MeasParams{T}`: 測定モデルのパラメータで、Y*t = A + B X*t + α Y*{t-1} + ∑*{i=1}^M (X*t' C*i X*t) + D ε*t によって与えられます。ここには、切片 A、線形負荷 B、自己回帰項 α、行列 C_i を含む二次項、測定ノイズスケーリング D が含まれます。
  * `aug_state::AugStateParams{T,T2}`: 状態表現を拡張するための拡張状態空間のパラメータ。このコンポーネントには、変換された状態平均、遷移、および状態プロセスの非線形または補助的な特徴を捉える追加の行列が含まれます。二次の数値型 T2 を使用することで、自動微分との互換性が促進されます。
  * `moments::Moments{T}`: 状態と拡張状態の無条件（または定常）モーメント。これには、状態ダイナミクスおよび拡張状態の長期的な平均と共分散が含まれ、モデルの初期化および診断評価にとって重要です。

# 型パラメータ

  * `T`: ほとんどのパラメータに使用される主要な数値型（例：Float64）。これは Real のサブタイプでなければならず、数値的精度と標準的な算術演算との互換性を確保します。
  * `T2`: AugStateParams の Lambda のようなパラメータに特に使用される、潜在的に異なる数値型。自動微分（AD）技術を活用するためにしばしば使用されます。
