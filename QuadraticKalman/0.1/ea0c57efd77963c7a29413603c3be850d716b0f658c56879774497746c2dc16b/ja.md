```
params_to_model(params::Vector{T}, N::Int, M::Int) where T<:Real -> QKModel
```

パラメータベクトルを状態および測定パラメータを持つQKModelオブジェクトに変換します。

# 引数

  * `params::Vector{T}`: 制約のないパラメータのベクトル。
  * `N::Int`: 状態ベクトルの次元。
  * `M::Int`: 測定ベクトルの次元。

# 戻り値

次を含む`QKModel`オブジェクト:

  * **状態パラメータ:** (mu, Phi, Omega)   状態方程式は     Xₜ = μ + Φ Xₜ₋₁ + Omega εₜ です。ここで、OmegaはOmega = D*state * D*state′として構築され、D_stateは対角成分が正の下三角行列（サイズN×N）です。
  * **測定パラメータ:** (A, B, C, D, α)   測定方程式は     Yₜ = A + B Xₜ + α Yₜ₋₁ + ∑₍ᵢ₌₁₎ᴹ Xₜ′ Cᵢ Xₜ + D εₜ です。ここで、DはD = D*meas * D*meas′として構築され、D_measは対角成分が正の下三角行列（サイズM×M）です。
  * 拡張状態パラメータおよびモデルモーメント（ヘルパー関数を介して計算）。

# パラメータベクトルのレイアウト

パラメータベクトルは次を含むと仮定されます:

1. **状態パラメータ:**

      * 最初の`N`エントリ: 状態平均`mu`。
      * 次の`N^2`エントリ: `Phi`のエントリ（列ごとに格納）。
      * 次の`N(N+1)/2`エントリ: D_stateの制約のないパラメータ（Omegaを形成するために使用）。
2. **測定パラメータ:**

      * 次の`M`エントリ: `A`。
      * 次の`M×N`エントリ: `B`のエントリ（M×N行列として再形成）。
      * 次の`M×N^2`エントリ: `C`のエントリ。（サイズN×NのM行列として解釈）。
      * 次の`M(M+1)/2`エントリ: D_measの制約のないパラメータ（Dを形成するために使用）。
      * 最後の`M×M`エントリ: `α`のエントリ（M×M行列として再形成）。

# 期待される合計長:

```
N + N^2 + N(N+1)/2  +  M + M×N + M×N^2 + M(M+1)/2 + M^2
```
