```
QKData{T<:Real,N}
```

実数値の `M x T_bar` 配列 (`Y`) を保持するデータ構造で、次元から派生した2つの整数フィールド (`M`, `T_bar`) を持ちます。`M` は測定ベクトルの次元であり、'Y*t' を表し、`T*bar` は時間期間の数から1を引いたものです（モデルの自己回帰構造のため）。

# フィールド

  * `Y::AbstractArray{T, N}` : 基本となるデータ配列（N次元、要素型 `<: Real`）。
  * `M::Int` : `Y` が少なくとも2Dの場合の `Y` の最初の次元。`N == 1` の場合、`M = 1` と定義します。
  * `T_bar::Int` : 2Dの場合の「2番目の次元」より1少ない値、または1Dベクトルの場合の `length(Y) - 1`。
