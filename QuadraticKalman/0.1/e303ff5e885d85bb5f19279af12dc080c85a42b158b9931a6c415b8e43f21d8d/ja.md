```
qkf_smoother(filter_output::FilterOutput{T}, model::QKModel{T,T2}) where {T<:Real, T2<:Real}
```

フィルタリング出力を使用して、二次カルマンフィルタ（QKF）のためのアウトオブプレイスのバックワードスムージングを実行します。この関数は、フィルタリング中に生成された状態推定を洗練させ、将来の観測を組み込むことでバックワードスムージングパスを通じて行います。

# 引数

  * filter*output::FilterOutput{T}: フィルタリングフェーズからの出力を保持するコンテナで、以下を含みます:   • Z*tt: フィルタリングされた拡張状態推定の行列。   • P*tt: フィルタリングされた状態共分散の配列。   • Z*ttm1: 一歩先の予測状態を含む行列。   • P_ttm1: 一歩先の予測状態共分散を含む配列。
  * model::QKModel{T,T2}: スムージングプロセスに必要なパラメータを提供するモデル仕様。モデルには、以下を取得するためにアンパックされる`aug_state`フィールドが含まれます:   • H*aug: 拡張測定選択行列。   • G*aug: 二次形式を処理するために使用される拡張複製行列。   • Phi_aug: 拡張状態遷移行列。さらに、`state`フィールドは状態の次元（N）を抽出するために使用されます。

# 詳細

この関数は最初に、元のフィルタリング出力の変更を防ぐためにフィルタリングされた状態と共分散推定のコピーを作成します。その後、これらのコピーに対して、モデルのパラメータと一歩先の予測値を使用してインプレーススムーザー処理（`qkf_smoother!`）を呼び出します。最終的なスムーズ結果は`SmootherOutput`構造体にラップされ、新しいコピーとして返されます。これは自動微分（AD）ワークフローとの互換性にとって特に重要です。

# 戻り値

  * SmootherOutput: 以下を含む複合構造:   • Z*smooth: スムーズされた拡張状態推定の行列。   • P*smooth: スムーズされた状態共分散行列の配列。
