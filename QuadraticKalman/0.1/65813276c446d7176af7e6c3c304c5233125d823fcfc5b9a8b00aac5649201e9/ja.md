```
qkf_smoother!(filter_output::FilterOutput{T}, model::QKModel{T,T2}) where {T<:Real, T2<:Real}
```

フィルタリング中に得られた出力を使用して、二次カルマンフィルタ（QKF）のインプレース後方スムージングを実行します。

この関数は、将来の観測を取り入れることによってフィルタリングされた状態推定と共分散行列を洗練させ、スムーズな推定値のセットを生成します。最初に、与えられたFilterOutput構造体からフィルタリングされた状態（Z*tt）とそれに関連する共分散（P*tt）、および1ステップ先の予測（Z*ttm1およびP*ttm1）を抽出します。次に、QKModelから拡張された状態パラメータ（H*aug、G*aug、Phi_aug）と状態次元（N）をアンパックします。この関数は、元のフィルタリング結果の変更を避けるためにフィルタリングされた推定値のローカルコピーを作成し、次に1ステップ先の予測とモデルパラメータを使用してこれらのコピーを更新するために、下位レベルのインプレーススムージングルーチンを呼び出します。スムーズな状態と共分散は最終的にSmootherOutput構造体にカプセル化され、返されます。

パラメータ：

  * filter*output: 次を含むFilterOutputインスタンス:   • Z*tt    :: 時間ステップ0からT̄までのフィルタリングされた拡張状態推定の行列。   • P*tt    :: フィルタリングされた状態共分散行列の配列。   • Z*ttm1  :: 1ステップ先の予測された拡張状態の行列。   • P_ttm1  :: 1ステップ先の予測された共分散行列の配列。
  * model: スムージングに必要なモデルパラメータを提供するQKModelインスタンスで、次を含む:   • aug*state: 次を含む構造体:       - H*aug  :: 拡張された測定選択行列。       - G*aug  :: 二次形式を処理するための拡張された重複行列。       - Phi*aug:: 拡張された状態遷移行列。   • state: 状態ベクトルの次元（N）。

返り値：

  * 次を含むSmootherOutput構造体:   • Z*smooth :: スムーズな拡張状態推定の行列。   • P*smooth :: スムーズな状態共分散行列の配列。
