```
StateParams{T<:Real}
```

状態空間モデルにおける状態の進化を支配するパラメータを表す構造体。このタイプは、初期平均、遷移ダイナミクス、ノイズ特性、および結果として得られる共分散を含む、状態プロセスの挙動を記述するために必要な基本的なコンポーネントをカプセル化します。

# フィールド

  * N::Int   状態ベクトルの次元。 このパラメータは、状態変数の数を指定します。
  * mu::AbstractVector{T}   初期状態平均ベクトル。 長さはNと等しくなければなりません。
  * Phi::AbstractMatrix{T}   状態遷移行列で、状態が時間とともにどのように進化するかをモデル化します。 この行列はN×Nのサイズである必要があります。
  * Omega::AbstractMatrix{T}   状態ノイズ行列で、状態の進化に対する確率的ノイズの影響をスケーリングするために使用されます。 N×Nのサイズである必要があります。
  * Sigma::AbstractMatrix{T}   状態共分散行列で、通常はOmega * Omega'として計算されます。 この行列は、状態の不確実性を定量化します。

# 詳細

典型的な状態空間モデルの状態進化は、次の方程式で表されます：     Xₜ = mu + Phi * Xₜ₋₁ + Omega * εₜ、ここでεₜはホワイトノイズプロセスを表します。この構造体は、QuadraticKalmanフレームワーク内でのフィルタリングおよびスムージングプロセスを促進するための重要なコンポーネントであり、モデルのダイナミクスが正確にキャプチャされ、安定性条件が適切に検証されることを保証します。

この構造体は、自動微分ツールとスムーズに統合されるように設計されており、Juliaの型システムを活用して数値計算における精度とパフォーマンスの両方を提供します。
