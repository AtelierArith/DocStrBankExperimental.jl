```
preamble
```

A constant [`TypstString`](@ref) used at the beginning of Typst source files generated by [`render`](@ref) and some `show` methods.

May be configured using [`set_preamble`](@ref).

# Examples

```jldoctest
julia> preamble
TypstString(TypstText("#set page(margin: 1em, height: auto, width: auto, fill: white)\n#set text(16pt, font: \"JuliaMono\")\n"))
```
