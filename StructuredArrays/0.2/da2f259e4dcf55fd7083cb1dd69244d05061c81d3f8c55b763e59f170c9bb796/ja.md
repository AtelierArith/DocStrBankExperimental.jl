```
UniformArray(val, args...) -> A
UniformArray{T}(val, args...) -> A
UniformArray{T,N}(val, args...) -> A
```

不変の配列 `A` を構築し、その要素はすべて `val` に等しく、軸は `args...` で指定されます。ストレージ要件は、通常の配列 `A` の `O(length(A))` ではなく `O(1)` です。オプションのパラメータ `T` と `N` は、要素の型と配列の次元数を指定するためのものです。

不均一配列は従来の線形インデックスを実装しています：`A[i]` は、範囲 `1:length(A)` のすべての線形インデックス `i` に対して `val` を返します。

`A[i] = x` のような文は実装されていません。なぜなら、不均一配列は不変と見なされるからです。可変の不均一配列を作成するには [`MutableUniformArray(val, dims)`](@ref) を呼び出してください。
