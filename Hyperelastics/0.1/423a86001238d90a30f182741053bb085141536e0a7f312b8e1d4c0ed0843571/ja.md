# モデル: 論文を参照

# 引数:

  * `ℒinv=CohenRounded3_2()`: 使用される逆ランジュバン近似を設定します。
  * `n=21`: 球面積分のための数値積分の次数

# パラメータ:

  * μ: 小ひずみせん断弾性率
  * N: チェーンセグメントの数
  * p: 非アフィン伸びパラメータ
  * U: チューブ幾何学パラメータ
  * q: 非アフィンチューブパラメータ
