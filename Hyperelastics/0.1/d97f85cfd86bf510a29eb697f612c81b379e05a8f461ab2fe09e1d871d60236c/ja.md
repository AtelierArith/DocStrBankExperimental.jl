# モデル:

$$
W = \frac{\mu}{2 * b} (\exp(b(I_1 - 3)) - 1)
$$

# 引数:

  * `type=PrincipalValueForm()`: ひずみエネルギー密度関数の形式を設定します。`PrincipalValueForm()` または `InvariantForm()` のいずれかです。

# パラメータ:

  * `μ`
  * `b`
