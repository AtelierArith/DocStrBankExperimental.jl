# モデル:

$$
W = \sqrt{\frac{N_3+N_8}{2N_3}}W_{3Chain}+\sqrt{\frac{I_1}{3N_8}}W_{8Chain}
$$

# 引数:

  * `ℒinv=TreloarApproximation()`: 使用される逆ランジュバン近似を設定します

# パラメータ:

  * `μ`
  * `N₃`
  * `N₈`
