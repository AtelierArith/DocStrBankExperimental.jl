シュレーディンガー方程式を解くために、Magnus展開ではなく、DifferentialEquations.jlの微分方程式ソルバーを使用するシミュレーター。このシミュレーターは、高いアニーリング時間で問題が発生する可能性があります。APIはsimulate関数と同じです。

引数: ising*model - 辞書として表現されたイジングモデル。キュービットと結合はタプルとして表現され、重みは数値です。例えば: im = Dict((1,) => 1, (2,) => 0.5, (1,2) => 2) annealing*schedule - 構造体によって与えられた形式のアニーリングスケジュール reltol - ODEシミュレーションで使用される相対許容誤差

パラメータ: initial*state - 初期状態ベクトル。デフォルトはnキュービットの均一重ね合わせ状態 constant*field*x - 各キュービットのX基底における定数バイアスのベクトル。デフォルトはzeros(n) constant*field_z - 各キュービットのZ基底における定数バイアスのベクトル。デフォルトはzeros(n)
