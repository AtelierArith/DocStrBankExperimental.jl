量子アニーリングシミュレーションをマグナス展開（2次）を介して実行するためのメイン関数。ノイズは、ランダム化された定数場で複数回実行することでシミュレートできます。

引数: ising*model - 辞書として表現されたイジングモデル。キュービットと結合はタプルとして表現され、重みは数値です。例えば: im = Dict((1,) => 1, (2,) => 0.5, (1,2) => 2) annealing*schedule - 構造体によって与えられた形式のアニーリングスケジュール

パラメータ: initial*state - 初期状態ベクトル。デフォルトはnキュービットの均一重ね合わせ状態 constant*field*x - 各キュービットのX基底における定数バイアスのベクトル。デフォルトはzeros(n) constant*field*z - 各キュービットのZ基底における定数バイアスのベクトル。デフォルトはzeros(n) パラメータ`mean*tol`と`max_tol`は、望ましいシミュレーション精度を指定します。`silence`パラメータは、進行ログを抑制するために使用できます。
