Function to modify an existing annealing schedule to use a customized annealing schedule (asch).  These parameters are the same as those used in a dwisc call or a dwave schedule. Inputs: annealing*schedule - annealing*schedule

Parameters: asch - This is the annealing-schedule parameter.  This is a list of tuples of the form        [(s₀,s*effective₀), (s₀,s*effective₁), ..., (sₙ,s_effectiveₙ)].
