```julia
dskbrief(
    file;
    single,
    gaps,
    extended,
    timebounds,
    bysegment,
    full,
    sigdigs,
    version,
    help,
    usage,
    stdout,
    stderr,
    stdin,
    append,
    wait
)

```

DSKBRIEFは、1つまたは複数のバイナリデジタルシェイプカーネル（DSK）ファイルの空間カバレッジと追加属性の概要を表示するコマンドラインユーティリティプログラムです。

# 拡張ヘルプ

!!! warning
    以下のすべての説明は、コマンドラインプログラムのヘルプ/使用出力から手動で解析されたものです。


| 引数           | 同等のもの    | 説明                                          |
|:------------ |:-------- |:------------------------------------------- |
| `single`     | `-a`     | すべてのDSKファイルを単一のファイルとして扱う                    |
| `gaps`       | `-gaps`  | カバレッジのギャップを表示する（`-a`が使用されている場合のみ適用）         |
| `extended`   | `-ext`   | 拡張サマリーを表示する：これにはデータタイプ、データクラス、および時間範囲が含まれます |
| `timebounds` | `-tg`    | セグメントの時間範囲が一致することを要求してセグメントをグループ化する         |
| `full`       | `-full`  | 各セグメントの詳細なサマリーを表示する。データタイプ固有のパラメータを含む       |
| `sigdigs`    | `-d <n>` | 浮動小数点値の`n`桁の有効数字を表示する                       |
| `version`    | `-v`     | プログラムのバージョンを表示する                            |
| `help`       | `-h`     | ヘルプテキストを表示する                                |
| `usage`      | `-u`     | 使用テキストを表示する                                 |
