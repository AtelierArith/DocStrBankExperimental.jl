```
simplify
```

SymPyには、さまざまな種類の簡略化を行うための多数の関数があります。また、すべてのこれらの関数を知的に適用して式の最も単純な形に到達しようとする一般的な関数`simplify`もあります。（`simplify`およびその他の関連機能の詳細については、[Simplification](https://docs.sympy.org/latest/tutorial/simplification.html)を参照してください）。他の簡略化関数は`sympy`オブジェクトを通じて利用可能です。

非記号式に対しては、`simplify`はその最初の引数を返します。
