```
triangle(x, y, side_length)
```

Determine if points (x, y) are within an equilateral triangle aperture centered at the origin. The triangle has side length `side_length`.

# Arguments

  * `x`: x-coordinates of the points.
  * `y`: y-coordinates of the points.
  * `side_length`: Side length of the triangle.

# Returns

  * A boolean array indicating whether each point is within the triangular aperture.
