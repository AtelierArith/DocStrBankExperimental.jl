```
square(x, y, l)
```

Determine if points (x, y) are within a square aperture centered at the origin. The square has side length `l`.

# Arguments

  * `x`: x-coordinates of the points.
  * `y`: y-coordinates of the points.
  * `l`: Side length of the square.

# Returns

  * A boolean array indicating whether each point is within the square aperture.
