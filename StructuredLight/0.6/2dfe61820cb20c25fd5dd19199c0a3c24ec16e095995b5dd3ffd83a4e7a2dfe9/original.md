```
single_slit(x, y, a)
```

Determine if points (x, y) are within a single slit aperture centered at the origin. The slit has width `a` and extends infinitely in the y-direction.

# Arguments

  * `x`: x-coordinates of the points.
  * `y`: y-coordinates of the points.
  * `a`: Width of the slit.

# Returns

  * A boolean array indicating whether each point is within the single slit aperture.
