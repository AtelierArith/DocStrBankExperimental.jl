```
est_1st_order_markov_bg(vec_str::Vector{Sting}; laplace=1; F=FloatType)

マルコフ行列を推定します。例えば、
        A   C   G   T       
    A  0.1 0.3 0.4 0.2 
    C  0.5 0.1 0.1 0.3
    G  0.2 0.2 0.2 0.4
    T  0.7 0.1 0.1 0.1
シャッフルされた背景配列において。
行と列はA, C, G, Tの順に並んでいます。

また、初期分布を推定します。例えば、
    A   C   G   T       
   0.2 0.2 0.3 0.3 
列はA, C, G, Tの順に並んでいます。
```

入力:

  * `vec_str`: 文字列のベクトル
  * `laplace`: 加える擬似カウント（オプション、デフォルトは1）
  * `F`: 推定値のデータ型（オプション、デフォルトはFloat32）

出力:     推定された4x4マルコフ行列と4x1初期分布。
