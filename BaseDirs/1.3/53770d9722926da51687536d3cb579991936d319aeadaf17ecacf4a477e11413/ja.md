**DATA_DIRS** (`XDG_DATA_DIRS`)

データファイルを検索するための*優先順位付き*ベースディレクトリのセット。

**デフォルト値**

|              Linux |                         MacOS |       Windows |
| ------------------:| -----------------------------:| -------------:|
| `/usr/local/share` | `/Library/ApplicationSupport` | `ProgramData` |
|       `/usr/share` |                               |               |
