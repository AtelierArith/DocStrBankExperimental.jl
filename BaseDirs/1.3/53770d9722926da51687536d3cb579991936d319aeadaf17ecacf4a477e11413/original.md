**DATA_DIRS** (`XDG_DATA_DIRS`)

The set of *preference ordered* base directories relative to which data files should be searched.

**Default values**

|              Linux |                         MacOS |       Windows |
| ------------------:| -----------------------------:| -------------:|
| `/usr/local/share` | `/Library/ApplicationSupport` | `ProgramData` |
|       `/usr/share` |                               |               |
