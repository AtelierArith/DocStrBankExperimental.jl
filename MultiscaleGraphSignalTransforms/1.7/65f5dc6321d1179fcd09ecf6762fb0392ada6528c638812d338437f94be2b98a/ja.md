```
stem(f; x = 1:length(f[:]), c = :black, ms = 5, shape = :circle, lw = 1)
```

長さ N の離散シーケンスデータをステムプロットで表示します。

# 入力引数

  * `f`: 関数値はベクトルまたは N x 1 行列である可能性があります。
  * `x`: デフォルトは `1:N`。x 値。
  * `c::Symbol`: デフォルトは `:black`。マーカーの色。
  * `ms::Number`: デフォルトは `5`。マーカーのサイズ。
  * `shape::Symbol`: デフォルトは `:circle`。マーカーの形状。
  * `lw::Number`: デフォルトは `1`。ステムの線幅。
  * `subgplot::Int`: デフォルトは `1`。サブプロットインデックス。
