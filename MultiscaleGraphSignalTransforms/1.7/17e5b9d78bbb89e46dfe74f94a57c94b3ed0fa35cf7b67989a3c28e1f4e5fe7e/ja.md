```
ROT_Distance(A, B, Q; edge_length = 1, α = 1.0)
```

は、Aの列ベクトルからBの列ベクトルへのROT距離行列を計算します。A、Bがベクトル入力の場合、コスト値と最適輸送計画も返します。

# 入力引数

  * `A::Any`: 初期確率分布の列を持つベクトルまたは行列。
  * `B::Any`: 終端確率分布の列を持つベクトルまたは行列。
  * `Q::SparseMatrixCSC{Int64,Int64}`: グラフの非加重インシデンス行列。
  * `edge_length::Any`: 長さベクトル（デフォルト: `1`は非加重グラフを表す）
  * `α::Float64`: デフォルトは`1.0`。ROTパラメータ。
  * `retPlan::Bool`: 最適計画を返すかどうかのインジケーター（デフォルト: `false`）

# 出力引数

  * `dis::Matrix{Float64}`: 距離行列、dis[i,j] = d_{ROT}(aᵢ, bⱼ; α)。
