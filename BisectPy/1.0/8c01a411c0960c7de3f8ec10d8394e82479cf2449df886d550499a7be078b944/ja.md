```
bisect_right(a, x, lo = 1, hi = length(a) + 1)
```

配列 `a` にアイテム `x` を挿入するインデックスを返します。`a` は非減少順序であると仮定します。

返り値 `i` は、すべての `e` が `a[:(i - 1)]` において `e <= x` であり、すべての `e` が `a[i:]` において `e > x` であるようなものです。したがって、`x` がすでに配列に存在する場合、`insert!(a, i, x)` はすでにそこにある最も右の `x` の直後に挿入されます。

# 引数

オプション引数 `lo`（デフォルト `1`）と `hi`（デフォルト `length(a) + 1`）は、検索される `a` のスライスを制限します。

# 例

```jldoctest
julia> bisect_right([1, 2, 3, 4, 5], 3.5)
4

julia> bisect_right([1, 2, 3, 4, 5], 2)
3

julia> bisect_right([1, 2, 3, 3, 3, 5], 3)
6
```
