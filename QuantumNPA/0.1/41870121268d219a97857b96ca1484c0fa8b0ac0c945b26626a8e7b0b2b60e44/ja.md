```
npa2sdp(expr, moment; eq=[], ge=[])
```

与えられた量子最適化問題のNPA緩和を生成します。

この関数の主な目的は、入力問題から等式制約を排除することです。この関数は、入力問題の実部の表現も受け取ります。もしそれがすでに実数値でない場合です。

返り値は、1) 最適化する式と、2) 半正定値である必要がある行列のベクトルを含むタプルです。ベクトルの最初の要素は、入力モーメント行列から導出されます。追加のものは、提供された追加の不等式制約から導出されます。

# 引数

  * `expr`: 最適化したい期待値を持つ演算子。
  * `moment`: モーメント行列
  * `eq`=[] および `ge=[]`: それぞれ、期待値をゼロに等しく設定したい演算子と、下限をゼロに設定したい演算子のベクトル。
