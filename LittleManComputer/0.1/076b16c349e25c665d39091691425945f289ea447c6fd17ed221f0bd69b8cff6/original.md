The LittleManComputer module used to assemble, disassamble and run code written for the imaginary little man computer. This is a computer used for educational purposed, to teach beginner assembly code programming.

# Example

```
code = assemble("counter.lmc")
inputs = [4, 3]
outputs = execute(code, inputs)
```

Runs code stored in "counter.lmc" file with input `[4, 3]`.
