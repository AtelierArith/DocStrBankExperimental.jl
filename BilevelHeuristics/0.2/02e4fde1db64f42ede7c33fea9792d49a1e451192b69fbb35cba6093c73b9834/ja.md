```
SMS_MOBO(;
    ul = Metaheuristics.SMS_EMOA(;N = 100), # 上位レベルの最適化手法
    ll = Metaheuristics.NSGA2(;N = 50), # 下位レベルの最適化手法
    ul_offsprings = 10,
    options_ul = Options(iterations = 100, f_calls_limit=Inf),
    options_ll = Options(iterations = 50, f_calls_limit=Inf)
)
```

### パラメータ

  * `ul` は上位レベルの最適化手法です：`SMS_EMOA` は唯一の有効な最適化手法です。
  * `ll` は `NSGA2`、`SPEA2`、または `SMS_EMOA` であることができます。
  * `ul_offsprings` は各世代で生成される新しい解の数です。

最適解は `res = optimize(...)` のときに `res.population` から取得できます。または `method.parameters.archive` を通じて取得できます。
