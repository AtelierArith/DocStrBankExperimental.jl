```
isphysical(raysplitter(s))
```

与えられた `raysplitters` が物理的に妥当な特性を持っている場合は `true` を返します。

具体的には、(φは入射角、θは屈折角とします):

  * 臨界角は全反射を意味します: もし θ(φ) ≥ π/2 ならば Tr(φ) = 0
  * 透過確率は偶関数です: Tr(φ) ≈ Tr(-φ) at ω = 0
  * 屈折角は奇関数です: θ(φ) ≈ -θ(-φ) at ω = 0
  * 光線の反転は真です: θ(θ(φ, pflag, ω), !pflag, ω) ≈ φ
  * 磁気保存は真です: (ω*new(ω*new(ω, pflag), !pflag) ≈ ω
