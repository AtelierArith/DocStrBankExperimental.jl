```
evolve!([p::AbstractParticle,] bd::Billiard, t)
```

与えられた粒子 `p` をビリヤード `bd` の中で進化させます。もし `t` が `AbstractFloat` 型であれば、`t` の時間だけ進化させます。しかし、もし `t` が `Int` 型であれば、`t` 回の衝突のために進化させます。衝突の間の粒子の状態を返します。

この関数は粒子を変異させますので、そうでない場合は `evolve` を使用してください。粒子が指定されていない場合は、[`randominside`](@ref) を通じてランダムな粒子が選ばれます。

### 返り値

  * `ct::Vector{T}` : 衝突時間。
  * `poss::Vector{SVector{2,T}}` : 衝突時の位置。
  * `vels::Vector{SVector{2,T}})` : 衝突直後の速度。
  * `ω`、`T` または `Vector{T}` : 角速度（磁気粒子の場合のみ返されます）。

時間 `ct[i+1]` は、状態 `poss[i], vels[i]` から状態 `poss[i+1], vels[i+1]` に到達するのに必要な時間です。したがって、`ct[1]` は常に 0 です。なぜなら `poss[1], vels[1]` が初期条件だからです。角速度 `ω[i]` は、粒子が状態 `poss[i], vels[i]` から `i+1` に伝播している間のものです。

どの時点においても、速度ベクトル `vels[i]` は、`i-1` 番目の衝突の鏡面反射の *後* に得られるものです。

### レイ分割ビリヤード

```
evolve!(p, bd, t, raysplitters)
```

レイ分割を実装するために、`evolve!` 関数は第四の引数 `raysplitters` を追加され、これは [`RaySplitter`](@ref) インスタンスのタプルです。レイ分割が使用される場合、`evolve` は **常にビリヤードを変異させる** ことに注意してください！レイ分割の使用に関する詳細情報と指示については、公式ドキュメントを参照してください。
