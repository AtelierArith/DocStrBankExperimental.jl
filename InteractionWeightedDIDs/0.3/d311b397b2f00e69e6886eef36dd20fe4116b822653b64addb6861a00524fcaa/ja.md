```
ContrastResult{T,M,R,C} <: AbstractMatrix{T}
```

同じセルのセットとセルレベルの平均に対して計算された1つ以上の[`RegDIDResultOrAgg`](@ref)から得られた最小二乗重みを保持する行列タイプ。詳細は[`contrast`](@ref)を参照。

最小二乗重みは`Matrix`に格納されており、プロパティ名`:m`を使用して取得できます。各処置係数の重みは2列目から始まる列方向に格納されており、1列目には結果変数のセルレベルの平均が含まれています。セルのインデックスはプロパティ名`:r`を使用してアクセスでき、係数を識別するためのインデックスはプロパティ名`:c`を使用してアクセスできます。`ContrastResult`を生成するために使用された[`RegDIDResultOrAgg`](@ref)は、`parent`を呼び出すことでアクセスできます。
