```
Emitter3DFit{T} <: AbstractEmitter
```

不確実性と時間/追跡情報を持つフィッティングされた3Dローカリゼーション結果を表します。

# フィールド

  * `x::T`: フィッティングされたx座標（マイクロメートル単位）
  * `y::T`: フィッティングされたy座標（マイクロメートル単位）
  * `z::T`: フィッティングされたz座標（マイクロメートル単位）
  * `photons::T`: フィッティングされた光子の数
  * `bg::T`: フィッティングされた背景（光子/ピクセル単位）
  * `σ_x::T`: x位置の不確実性（マイクロメートル単位）
  * `σ_y::T`: y位置の不確実性（マイクロメートル単位）
  * `σ_z::T`: z位置の不確実性（マイクロメートル単位）
  * `σ_photons::T`: 光子数の不確実性
  * `σ_bg::T`: 背景の不確実性（光子/ピクセル単位）
  * `frame::Int`: 取得シーケンス内のフレーム番号
  * `dataset::Int`: 特定の取得/データセットの識別子
  * `track_id::Int`: フレーム間でのローカリゼーションをリンクするための識別子（0 = リンクされていない）
  * `id::Int`: データセット内のユニークな識別子
