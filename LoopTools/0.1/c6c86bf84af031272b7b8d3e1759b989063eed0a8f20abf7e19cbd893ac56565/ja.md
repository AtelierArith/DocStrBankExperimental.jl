```
A0i(id, m^2)
```

`id` のための1点テンソル係数

$$
\frac{μ^{4-D}}{iπ^{D/2} r_Γ} \int d^D q \frac{\{1, g_{μν} \} }{q^2-m^2}
$$

ここで $r_Γ = \frac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)}$, $D=4-2ε$。

特別なケース：

| `id`   | Int | 説明                          |
|:------ |:---:|:--------------------------- |
| `aa0`  | `1` | スカラー1点1ループ関数、すなわち `A0(m^2)` |
| `aa00` | `4` | $g_{μν}$ の係数                |
