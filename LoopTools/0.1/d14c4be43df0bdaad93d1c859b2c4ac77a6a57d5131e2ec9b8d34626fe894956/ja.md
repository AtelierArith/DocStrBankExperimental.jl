```
C0i(id, p1^2, p2^2, (p1+p2)^2, m1^2, m2^2, m3^2)
```

`id` の三点テンソル係数

$$
\frac{μ^{4-D}}{iπ^{D/2} r_Γ} \int
\frac{({\rm numerator})\, d^D q }{(q^2-m_1^2)\left[(q+p_1)^2-m_2^2\right]
\left[(q+p_1+p_2)^2-m_3^2\right]}
$$

ここで $r_Γ = \frac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)}$, $D=4-2ε$。

特別な場合:

| `id`     |  Int  | 説明                                |
|:-------- |:-----:|:--------------------------------- |
| `cc0`    |  `1`  | スカラー三点一ループ関数                      |
| `cc1`    |  `4`  | $k_{1μ}$ の係数                      |
| `cc2`    |  `7`  | $k_{2μ}$ の係数                      |
| `cc00`   | `10`  | $g_{μν}$ の係数                      |
| `cc11`   | `13`  | $k_{1μ} k_{1ν}$ の係数               |
| `cc12`   | `16`  | $k_{1μ} k_{2ν}$ の係数               |
| `cc22`   | `19`  | $k_{2μ} k_{2ν}$ の係数               |
| `...`    | `...` | `...`                             |
| `cc2222` | `64`  | $k_{2μ} k_{2ν} k_{2ρ} k_{2σ}$ の係数 |

ここで $k_{1,2}$ は $p_{1,2}$ に関連しており、$k_{j} = \sum_{i=1}^j p_i$ です。
