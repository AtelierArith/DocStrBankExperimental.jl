```
D0i(id, p1^2, p2^2, p3^2, p4^2, (p1+p2)^2, (p2+p3)^2, m1^2, m2^2, m3^2, m4^2)
```

`id` のための四点テンソル係数

$$
\frac{μ^{4-D}}{iπ^{D/2} r_Γ} \int
\frac{({\rm numerator})\, d^D q }{(q^2-m_1^2)\left[(q+p_1)^2-m_2^2\right]
\left[(q+p_1+p_2)^2-m_3^2\right] \left[(q+p_1+p_2+p_3)^2-m_4^2\right]}
$$

ここで $r_Γ = \frac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)}$, $D=4-2ε$。

特別なケース：

| `id`      |  Int  | 説明                                       |
|:--------- |:-----:|:---------------------------------------- |
| `dd0`     |  `1`  | スカラー四点一ループ関数                             |
| `dd1`     |  `4`  | $k_{1μ}$ の係数                             |
| `dd2`     |  `7`  | $k_{2μ}$ の係数                             |
| `dd3`     | `10`  | $k_{3μ}$ の係数                             |
| `dd00`    | `13`  | $g_{μν}$ の係数                             |
| `...`     | `...` | `...`                                    |
| `dd33333` | `238` | $k_{3μ} k_{3ν} k_{3ρ} k_{3σ} k_{3λ}$ の係数 |

ここで $k_{1,2,3}$ は外部モーメンタ $p_{1,2,3}$ に関連しており、$k_{j} = \sum_{i=1}^j p_i$ です。
