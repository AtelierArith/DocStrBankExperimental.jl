```
B0i(id, p^2, m1^2, m2^2)
```

`id` の二点テンソル係数

$$
\frac{μ^{4-D}}{iπ^{D/2} r_Γ} \int
\frac{({\rm numerator})\, d^D q }{(q^2-m_1^2)\left[(q+p)^2-m_2^2\right]}
$$

ここで $r_Γ = \frac{Γ^2(1-ε)Γ(1+ε)}{Γ(1-2ε)}$, $D=4-2ε$。

特別なケース：

| `id`               |     Int     | 説明                   |
|:------------------ |:-----------:|:-------------------- |
| `bb0`   (`dbb0`)   | `1` (`19`)  | (導関数の)スカラー二点一ループ関数   |
| `bb1`   (`dbb1`)   | `4` (`22`)  | (導関数の)$p_{μ}$の係数     |
| `bb00`  (`dbb00`)  | `7` (`25`)  | (導関数の)$g_{μν}$の係数    |
| `bb11`  (`dbb11`)  | `10` (`28`) | (導関数の)$p_μ p_ν$の係数   |
| `bb001` (`dbb001`) | `13` (`31`) | (導関数の)$g_{μν}p_ρ$の係数 |
| `bb111`            |    `16`     | $p_μ p_ν p_ρ$の係数     |

関数 `B0`, `B1`, `B00`, `B11`, `B001` および `B111` が定義されています。
