```
sim_epidemics(
    model::EpidemicModel{SIR,TG};
    patient_zero::Union{Vector{Int},Nothing}=nothing,
    γ::Union{Float64,Nothing}=nothing) where {TG<:Union{<:AbstractGraph,Vector{<:AbstractGraph}}}
```

SIR（感受性-感染-回復）モデルを使用して、疫病の発生をシミュレートします。

# 引数

  * `model`: 感染のダイナミクス、接触グラフ、およびその他のパラメータに関する情報をカプセル化したSIR疫病モデル。
  * `patient_zero`: （オプション）最初に感染した個体のインデックスを指定するベクトル。指定されていない場合（デフォルトは`nothing`）、患者ゼロは確率`γ`に基づいてランダムに選択されます。
  * `γ`: （オプション）患者ゼロである確率。`patient_zero`が指定されておらず、`γ`が提供されている場合、患者ゼロは確率`γ`でランダムに選ばれます。`patient_zero`と`γ`の両方が提供されていない場合（デフォルトは`nothing`）、患者ゼロは各個体に対して等しい確率でランダムに選択されます。
  * `rng`: （オプション）乱数生成器。デフォルトは`Xoshiro(1234)`です。

# 戻り値

  * 時間にわたる疫病の発生構成を表す行列。各行はノードに対応し、各列は時間ステップを表します。行列内の値は、各時間ステップにおける各ノードの状態を示します：0.0は感受性（S）、1.0は感染（I）、2.0は回復（R）を示します。
