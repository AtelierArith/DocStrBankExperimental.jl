```
bethe_lattice(z::Int, tmax::Int)
```

指定された次数と深さを持つベーテ格子（木）を生成します。

# 引数

  * `z::Int`: ベーテ格子の次数。
  * `tmax::Int`: ベーテ格子の最大深さ。

# 戻り値

  * `V::Vector{Int}`: ベーテ格子の頂点のリスト。
  * `E::Vector{Vector{Int}}`: ベーテ格子の辺のリスト。

# 説明

この関数は、指定された次数（`z`）と最大深さ（`tmax`）を持つベーテ格子（木）を生成します。ベーテ格子は、指定されたパラメータに従ってノードとエッジを反復的に追加することによって構築されます。ルートノードには常に頂点番号1が割り当てられ、頂点は深さ優先方式で番号付けされます。エッジは、各ペアが2つの頂点間のエッジを表す頂点のペアのリストとして保存されます。

この関数は、最初の要素（`V`）が頂点のリストであり、2番目の要素（`E`）がベーテ格子の辺のリストであるタプルを返します。
