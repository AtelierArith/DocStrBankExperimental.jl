```
struct SIRS <: InfectionModel
    εᵢᵗ::Array{Float64, 2} # 自己感染確率
    rᵢᵗ::Array{Float64, 2} # 回復確率
    σᵢᵗ::Array{Float64, 2} # 免疫喪失確率
end
```

`SIRS` 構造体は SIRS（感受性-感染-回復-感受性）感染モデルを表します。

# フィールド

  * `εᵢᵗ`: NVxT 配列で、時間にわたる自己感染確率を表します。ここで、NV はノードの数、T は時間ステップの数です。各要素 εᵢᵗ[i, t] は、ノード i が時間 t に自己感染する確率を示します。
  * `rᵢᵗ`: NVxT 配列で、時間にわたる回復確率を表します。ここで、NV はノードの数、T は時間ステップの数です。各要素 rᵢᵗ[i, t] は、ノード i が時間 t に感染から回復する確率を示します。
  * `σᵢᵗ`: NVxT 配列で、時間にわたる免疫喪失確率を表します。ここで、NV はノードの数、T は時間ステップの数です。各要素 σᵢᵗ[i, t] は、ノード i が時間 t に免疫を失い、再び感受性になる確率を示します。
