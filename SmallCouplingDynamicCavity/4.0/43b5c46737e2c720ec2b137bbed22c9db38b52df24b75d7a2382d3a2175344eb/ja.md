```
struct SIS <: InfectionModel
    εᵢᵗ::Array{Float64, 2} # 自己感染確率
    rᵢᵗ::Array{Float64, 2} # 回復確率
end
```

`SIS`構造体はSIS（感受性-感染-感受性）感染モデルを表します。

# フィールド

  * `εᵢᵗ`: NVxT配列で、時間にわたる自己感染確率を表します。ここで、NVはノードの数、Tは時間ステップの数です。各要素εᵢᵗ[i, t]は、ノードiが時間tに自己感染する確率を示します。
  * `rᵢᵗ`: NVxT配列で、時間にわたる回復確率を表します。ここで、NVはノードの数、Tは時間ステップの数です。各要素rᵢᵗ[i, t]は、ノードiが時間tに感染から回復する確率を示します。
