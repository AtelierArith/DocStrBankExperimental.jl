```
target!(score)
```

現在実行中のテストケースを更新して、指定されたスコアをそのターゲットとして追跡します。

`score` は `Float64` に `convert` 可能でなければなりません。

!!! danger "複数の更新"
    このスコアは一度だけ設定できます！繰り返しの呼び出しは無視されます。


!!! warning "呼び出し可能性"
    これは、テストケースが現在調査中であるか、`Possibility` の例が積極的に生成されている間にのみ呼び出すことができます。これは `@composed` や `@check` の内部、またはそれらの場所からのみ呼び出されることを意図した関数内で呼び出すことができます。
