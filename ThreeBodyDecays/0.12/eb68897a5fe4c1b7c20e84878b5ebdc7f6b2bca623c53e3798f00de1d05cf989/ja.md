```
DalitzPlot # ドキュメント用のみ、以下の例を参照
plot(intensity, ms)
plot(ms, intensity)
```

`DalitzPlot` のプロットレシピ。

このレシピは、指定された不変量の範囲にわたる関数の強度を視覚化するヒートマップとしてのダリッツプロットを生成します。このプロットは、三体崩壊や類似のプロセスの運動学的領域に関する洞察を提供します。

# パラメータ:

  * `intensity::Function`: 不変量 `(m23², m31², m12²)` の実関数で、特定の運動学的点での強度を返します。
  * `ms::MassTuple`: システムに関与する粒子の質量を表すタプル。

# キーワード引数:

  * `iσx`: x軸に使用する最初の不変量のインデックス、`1->m23²`、`2->m31²`、および `3->m12²`。デフォルトは1。
  * `iσy`: y軸に使用する2番目の不変量のインデックス。デフォルトは2。
  * `grid_size`: プロットグリッドの解像度。値が高いほど詳細が細かくなります。デフォルトは100。
  * `xlims`: 不変量範囲に関するx軸の制限。デフォルトは `lims(iσx, ms)`（自動的に計算されます）。
  * `ylims`: 不変量範囲に関するy軸の制限。デフォルトは `lims(iσy, ms)`（自動的に計算されます）。

# 出力:

このレシピは以下を生成します：

1. `σx` および `σy` 軸の不変量値のグリッド。
2. `values` の2D配列で、各要素は次のいずれかです：

      * Kibble条件によって対応する運動学的領域が禁止されている場合は `NaN`。
      * 提供された `intensity` 関数から計算された強度。

# 使用法:

プロットコマンドを呼び出すだけです、`julia plot(intensity, ms) plot(ms, intensity)``
