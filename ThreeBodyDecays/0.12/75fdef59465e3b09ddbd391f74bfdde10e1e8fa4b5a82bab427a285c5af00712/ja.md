```
amplitude(dc::DecayChain, σs::MandelstamTuple, two_λs; refζs = (1, 2, 3, 1))
```

与えられた崩壊系列と運動学的構成に対する総振幅を計算します。

# 引数

  * `dc::DecayChain`: 崩壊系列オブジェクト。
  * `σs::MandelstamTuple`: プロセスの運動学的不変量を記述するマンデルスタム変数を表すタプル。
  * `two_λs`: 崩壊に関与する粒子のヘリシティ値のコレクション。
  * `refζs`: アライメント回転のための参照ζインデックス（デフォルトは `(1, 2, 3, 1)`）。

# 戻り値

  * 振幅値の四次元配列。
