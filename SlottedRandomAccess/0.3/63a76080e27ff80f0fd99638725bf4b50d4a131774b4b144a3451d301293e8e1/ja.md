```
simulate!(sim::PLR_Simulation; kwargs...)
```

`PLR_Simulation`オブジェクト内の各負荷点のPLRを計算するためにシミュレーションを実行します。デフォルトでは、すべての利用可能なスレッドを使用します。juliaが単一スレッドで起動されている場合、関数は警告を送信します。

!!! note
    すでに有効なシミュレーション結果を含むポイントはスキップされ、新しいシミュレーションオブジェクトを明示的に作成して再計算する必要があります。


# キーワード引数

  * `logger`: シミュレーションの進行状況を表示するために使用するロガー。デフォルトのjuliaロガーを使用し、インタラクティブなjuliaセッション（つまりREPL）から実行されるときにTerminalLogger.jlを介してターミナルにも出力します。
  * `ntasks`: 各PLRポイントの並列計算に使用するタスクの数。指定されていない場合は、すべての利用可能なスレッドを使用します。
