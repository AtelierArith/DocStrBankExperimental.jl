```
ncread(filename, varname)
```

は、ファイル `filename` から変数 `varname` の値を読み取り、配列として値を返します。

### キーワード引数

  * `start` 各次元の開始インデックスを設定する長さ `ndim(v)` のベクトル
  * `count` 各次元に沿って読み取る値の数を設定する長さ `ndim(v)` のベクトル。値 -1 は、その次元に沿ってすべての値を読み取る特別なケースとして扱われます

### 例

3D NetCDF 変数の2番目のスライスを読み取るには、次のように書くことができます：

```
ncread("filename","varname", start=[1,1,2], count = [-1,-1,1])
```
