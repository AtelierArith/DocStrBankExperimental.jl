```
vl,vv = pcsaft_vsat(T,m,ϵ,σ)
```

入力温度 `T` における PCSAFT 状態方程式の飽和体積を返します。

入力:

  * `T`: 飽和温度 (ケルビン)
  * `m`: セグメント長 (単位なし)
  * `ϵ`: 縮小相互作用エネルギー `[K]`
  * `σ`: モノマー直径 `[m]`

出力:

  * `vl` : 飽和液体体積 `[m^3/mol]`.
  * `vv` : 飽和蒸気体積 `[m^3/mol]`.

補助範囲外の値の場合は `NaN,NaN` を返します (1 ≤ `m` ≤ 64 および `Tmin` < `T` < `Tc`).
