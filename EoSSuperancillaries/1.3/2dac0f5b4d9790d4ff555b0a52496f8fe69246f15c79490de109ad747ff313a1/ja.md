```
Θ,status = pcsaft_theta(T̃,m)
Θ,status = pcsaft_theta(T̃,m,T̃c)
```

PCSAFT飽和体積スーパーアンシラリーで使用される縮小スケール温度パラメータを計算します。

入力:

  * `T̃`: 縮小相互作用エネルギーで割った温度 (`T/ϵ`)（単位なし）
  * `m`: セグメント長（単位なし）
  * `T̃c`: （オプション）縮小臨界温度 (`Tc/ϵ`)（単位なし）

出力:

  * `Θ` : スケール温度パラメータ（単位なし）
  * `status` : `T̃`,`m` の組み合わせが有効かどうかを示すために使用される `Symbol`。次のいずれかを返すことができます：

      * `:inrange` : 組み合わせが有効な場合
      * `:nonfinite` : 入力のいずれかが有限でない場合
      * `:below_mmin` : `m` < 1.0 の場合
      * `:over_mmax` : `m` > 64.0 の場合
      * `:below_Tmin` : `T̃` < `T̃min` の場合、ここで `T̃min` = `T̃c*exp(-2.20078778)*m^0.37627892`
      * `:over_Tmax` : `T̃` > `T̃c` の場合
