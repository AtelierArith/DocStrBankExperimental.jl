```
create_completion(model::String, prompt::String; params::Dict{String,Any}=Dict{String,Any}())
```

Creates a chat completion.

Args:     model (String): The name of the model to use for the completion.     prompt (String): The prompt to use for the completion.     params (Dict{String,Any}, optional): Additional parameters to pass to the API. Defaults to Dict{String,Any}().

Returns:     Any: The response from the API.
