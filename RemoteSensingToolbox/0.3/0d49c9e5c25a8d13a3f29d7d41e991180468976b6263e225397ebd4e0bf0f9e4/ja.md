```
from_table(table, val_col, dim_col, dims; missingval=0)
```

テーブルを `Raster` または `RasterStack` に読み込みます。

# パラメータ

  * `table`: `Tables.jl` インターフェースを実装している任意のタイプ。
  * `val_col`: ラスタ値を含む列。シンボルまたはシンボルのタプルである必要があります。
  * `dim_col`: 各値の座標を含む列。座標が単一の列の下にタプルとして保存されている場合はシンボルであるべきです。座標が複数の列にスカラー値として保存されている場合は、シンボルのタプルでも構いません。
  * `dims`: 出力ラスタの次元。
  * `missingval`: 欠損データを示すために使用される値。

# 戻り値

`val_col` が `Symbol` の場合、`dims` と同じ次元を持つ単一の `Raster` を返します。それ以外の場合は、各レイヤーが単一の値列に対応する `RasterStack` を返します。
