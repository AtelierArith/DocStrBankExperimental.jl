```
xkb_log_level
```

ログレベルを指定します。

| 列挙子                    | 注釈                     |
|:---------------------- |:---------------------- |
| XKB*LOG*LEVEL_CRITICAL | 重大な内部エラーのみをログに記録します。   |
| XKB*LOG*LEVEL_ERROR    | すべてのエラーをログに記録します。      |
| XKB*LOG*LEVEL_WARNING  | 警告とエラーをログに記録します。       |
| XKB*LOG*LEVEL_INFO     | 情報、警告、およびエラーをログに記録します。 |
| XKB*LOG*LEVEL_DEBUG    | すべてをログに記録します。          |
