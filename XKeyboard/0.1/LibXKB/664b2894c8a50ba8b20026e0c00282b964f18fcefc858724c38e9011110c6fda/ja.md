```
xkb_rule_names
```

キーマップをコンパイルするための名前で、RMLVOとも呼ばれます。

これらの名前は、ユーザーがキーマップを選択するための一般的な設定値です。

構造体全体がNULLの場合、各フィールドはNULLと見なされます。独自のデフォルトを選択する代わりにNULLを渡すことを推奨します。

| フィールド   | 注釈                                                                                                                                                                                                                                                                                          |
|:------- |:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| rules   | 使用するルールファイル。ルールファイルは、モデル、レイアウト、バリアント、およびオプションフィールドの値を解釈する方法を説明します。NULLまたは空の文字列""の場合、デフォルト値が使用されます。XKB*DEFAULT*RULES環境変数が設定されている場合、それがデフォルトとして使用されます。そうでなければ、システムのデフォルトが使用されます。                                                                                                              |
| model   | キーコードとLEDを解釈するためのキーボードモデル。NULLまたは空の文字列""の場合、デフォルト値が使用されます。XKB*DEFAULT*MODEL環境変数が設定されている場合、それがデフォルトとして使用されます。そうでなければ、システムのデフォルトが使用されます。                                                                                                                                                      |
| layout  | キーマップに含めるレイアウト（言語）のカンマ区切りリスト。NULLまたは空の文字列""の場合、デフォルト値が使用されます。XKB*DEFAULT*LAYOUT環境変数が設定されている場合、それがデフォルトとして使用されます。そうでなければ、システムのデフォルトが使用されます。                                                                                                                                                  |
| variant | 各レイアウトごとに1つのバリアントのカンマ区切りリストで、さまざまな方法でそれぞれのレイアウトを修正または拡張することがあります。一般的には、空であるか、レイアウトの数と同じ数の値を持つべきです。 "intl,,neo"のように空の値を使用することができます。NULLまたは空の文字列""の場合、レイアウトにもデフォルト値が使用されると、デフォルト値が使用されます。そうでなければ、バリアントは使用されません。XKB*DEFAULT*VARIANT環境変数が設定されている場合、それがデフォルトとして使用されます。そうでなければ、システムのデフォルトが使用されます。 |
| options | ユーザーがレイアウトに関連しない好みを指定するためのオプションのカンマ区切りリストで、レイアウトを切り替えるために使用されるキーの組み合わせや、コンポーズキーとして使用されるキーなどがあります。NULLの場合、デフォルト値が使用されます。空の文字列""の場合、オプションは使用されません。XKB*DEFAULT*OPTIONS環境変数が設定されている場合、それがデフォルトとして使用されます。そうでなければ、システムのデフォルトが使用されます。                                                              |
