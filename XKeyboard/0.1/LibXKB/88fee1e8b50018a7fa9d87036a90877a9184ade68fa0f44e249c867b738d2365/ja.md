```
xkb_state_component
```

状態オブジェクトのための修飾子とレイアウトタイプ。この列挙型はビットマスク可能であり、例えば (XKB*STATE*MODS*DEPRESSED | XKB*STATE*MODS*LATCHED) はロックされた修飾子を除外するために有効です。

XKBでは、DEPRESSEDコンポーネントは「ベース」としても知られています。

| 列挙子                        | 注釈                                                                                         |
|:-------------------------- |:------------------------------------------------------------------------------------------ |
| XKB*STATE*MODS_DEPRESSED   | 押下された修飾子、すなわち物理的にキーがそれらを保持しています。                                                           |
| XKB*STATE*MODS_LATCHED     | ラッチされた修飾子、すなわち次の非修飾キーの押下後に解除されます。                                                          |
| XKB*STATE*MODS_LOCKED      | ロックされた修飾子、すなわちロックを引き起こすキーが再度押下された後に解除されます。                                                 |
| XKB*STATE*MODS_EFFECTIVE   | 効果的な修飾子、すなわち現在アクティブでキー処理に影響を与えます（他の状態コンポーネントから派生）。状態がどのように発生したかを明示的に気にしない限り、これを使用してください。   |
| XKB*STATE*LAYOUT_DEPRESSED | 押下されたレイアウト、すなわち物理的にキーがそれを保持しています。                                                          |
| XKB*STATE*LAYOUT_LATCHED   | ラッチされたレイアウト、すなわち次の非修飾キーの押下後に解除されます。                                                        |
| XKB*STATE*LAYOUT_LOCKED    | ロックされたレイアウト、すなわちロックを引き起こすキーが再度押下された後に解除されます。                                               |
| XKB*STATE*LAYOUT_EFFECTIVE | 効果的なレイアウト、すなわち現在アクティブでキー処理に影響を与えます（他の状態コンポーネントから派生）。状態がどのように発生したかを明示的に気にしない限り、これを使用してください。 |
| XKB*STATE*LEDS             | LED（他の状態コンポーネントから派生）。                                                                      |
