```
xkb_context_flags
```

Flags for context creation.

| Enumerator                       | Note                                                       |
|:-------------------------------- |:---------------------------------------------------------- |
| XKB_CONTEXT_NO_FLAGS             | Do not apply any context flags.                            |
| XKB_CONTEXT_NO_DEFAULT_INCLUDES  | Create this context with an empty include path.            |
| XKB_CONTEXT_NO_ENVIRONMENT_NAMES | Don't take RMLVO names from the environment.  \since 0.3.0 |
