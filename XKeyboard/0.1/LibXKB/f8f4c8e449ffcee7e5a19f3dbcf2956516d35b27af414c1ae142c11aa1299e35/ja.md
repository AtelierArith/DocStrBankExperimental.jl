```
xkb_context_flags
```

コンテキスト作成のためのフラグ。

| 列挙子                              | 注釈                              |
|:-------------------------------- |:------------------------------- |
| XKB*CONTEXT*NO_FLAGS             | いかなるコンテキストフラグも適用しない。            |
| XKB*CONTEXT*NO*DEFAULT*INCLUDES  | 空のインクルードパスでこのコンテキストを作成する。       |
| XKB*CONTEXT*NO*ENVIRONMENT*NAMES | 環境からRMLVO名を取得しない。  \since 0.3.0 |
