```
xkb_keymap_key_repeats(keymap, key)
```

キーが繰り返されるべきかどうかを判断します。

キー マップは、異なるキーに対して異なる繰り返し動作を指定することがあります。ほとんどのキーは一般的に繰り返し動作を示すべきです。たとえば、テキストエディタで 'a' キーを押し続けると、通常は数ミリ秒ごとに単一の 'a' 文字が挿入され、キーが放されるまで続きます。しかし、繰り返す必要がない、または繰り返すべきでないキーもあります。たとえば、左/右シフトやキャップスロックなどの修飾キーを繰り返すことは、一般的には有用でも望ましくもありません。

xkb_keymap

### 戻り値

キーが繰り返されるべきであれば 1、そうでなければ 0 を返します。
