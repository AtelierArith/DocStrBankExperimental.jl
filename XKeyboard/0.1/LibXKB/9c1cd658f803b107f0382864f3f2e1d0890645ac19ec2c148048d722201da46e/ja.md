```
xkb_context_set_log_verbosity(context, verbosity)
```

現在のログの冗長性を設定します。

ライブラリは、ライブラリの通常のユーザーには役に立たない多くの警告を生成する可能性があります。より多くの情報が必要な場合（例えば、新しいキーマップを開発しているとき）には、冗長性を増やすことができます。

デフォルトの冗長性は0です。コンテキストが作成された時に設定された環境変数XKB*LOG*VERBOSITYは、デフォルト値を上書きします。

最も冗長なメッセージは、XKB*LOG*LEVEL_WARNING以下のレベルです。

xkb_context

### パラメータ

  * `context`: 設定された冗長性を使用するコンテキスト。
  * `verbosity`: 使用する冗長性。現在使用されている値は1から10で、高い値ほど冗長です。0は冗長なメッセージがログに記録されないことを意味します。
