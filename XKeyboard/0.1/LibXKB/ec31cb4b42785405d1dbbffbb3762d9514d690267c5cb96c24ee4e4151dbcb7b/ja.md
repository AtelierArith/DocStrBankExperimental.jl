```
xkb_consumed_mode
```

消費された修飾子モード。

消費される修飾子とそうでない修飾子を決定するためのいくつかの方法があり、それぞれ異なるシステムや状況に適用されます。このモードは使用する方法を選択します。

すべての方法において、キーマップが修飾子を「保持」することを決定する場合があることに注意してください。つまり、修飾子は消費されたと報告されない場合があります。

| 列挙子                   | 注                                                                                                                                                                                                                                                                                                                |
|:--------------------- |:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| XKB*CONSUMED*MODE_XKB | これはXKB仕様で定義され、libX11によって使用されるモードです。修飾子は、キーの翻訳に*影響を与える*場合にのみ消費されます。たとえば、`Control+Alt+<Backspace>`がいくつかの割り当てられたキーシンボルを生成する場合、`<Backspace>`のみを押すと、`Control`と`Alt`は消費されます。これは、アクティブでなくても、アクティブであればキーの翻訳に影響を与えるからです。                                                                                                  |
| XKB*CONSUMED*MODE_GTK | これはGTK+ツールキットによって使用されるモードです。このモードは、次の2つの独立したヒューリスティックで構成されています：  - 現在アクティブな修飾子のセット（XKB*CONSUMED*MODE_XKBで説明されているようにキーに影響を与えない修飾子を除く）は、すべての修飾子がアクティブなときに生成されるキーシンボルが、修飾子がアクティブでないときに生成されるキーシンボルと異なる場合に消費されたと見なされます。  - 単一の修飾子は、その修飾子が唯一のアクティブ修飾子であるときに生成されるキーシンボルが、修飾子がアクティブでないときに生成されるキーシンボルと異なる場合に消費されたと見なされます。 |
