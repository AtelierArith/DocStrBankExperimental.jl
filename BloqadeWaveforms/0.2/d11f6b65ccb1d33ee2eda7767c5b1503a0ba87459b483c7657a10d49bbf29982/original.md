```
piecewise_linear(;clocks, values)
```

Create a piecewise linear waveform.

# Keyword Arguments

  * `clocks::Vector{<:Real}`: the list of clocks for the corresponding values.
  * `values::Vector{<:Real}`: the list of values at each clock.
