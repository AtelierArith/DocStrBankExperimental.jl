```
function (..)(first, last)
```

[`Intervals`](https://github.com/invenia/Intervals.jl) からエクスポートされ、`first..last` から閉区間を作成し、`Waveform` 構造体と共に使用して、Waveform の値のスライスを取得することができ、Waveform スライスの時間は 0 μs から始まり、期間は `last - first` になります。
