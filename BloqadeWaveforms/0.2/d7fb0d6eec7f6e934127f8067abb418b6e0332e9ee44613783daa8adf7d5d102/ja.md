```
linear_ramp(;duration, start_value, stop_value)
```

線形ランプ波形を作成します。

# キーワード引数

  * `duration::Real`: 波形全体の持続時間。
  * `start_value::Real`: 線形ランプの開始値。
  * `stop_value::Real`: 線形ランプの停止値。
