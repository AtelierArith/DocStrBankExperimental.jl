ローカル最適ブロック前処理共役勾配法 (LOBPCG)

`AX = λBX` を満たす `nev` の極値固有値とそれに対応する固有ベクトルを見つけます。

`A` と `B` は一般的な型である可能性がありますが、ベクトルおよびストライド行列 `X` と `C` に対して `Base.mul!(C, AorB, X)` が定義されている必要があります。また、`size(A, i::Int)` と `eltype(A)` も `A` に対して定義されている必要があります。

```
lobpcg(A, [B,] largest, nev; kwargs...) -> results
```

# 引数

  * `A`: 線形演算子;
  * `B`: 線形演算子;
  * `largest`: 最大の固有値が必要な場合は `true`、最小の固有値が必要な場合は `false`;
  * `nev`: 必要な固有値の数。

## キーワード

  * `log::Bool`: デフォルトは `false`；`true` の場合、`results.trace` に反復状態が保存されます；`false` の場合、`results.trace` は空になります;
  * `P`: 残差ベクトルの前処理器で、`ldiv!` をオーバーロードする必要があります;
  * `C`: 残差および解ベクトルをサブスペースに対して直交させるための制約で、`mul!` をオーバーロードする必要があります;
  * `maxiter`: 最大反復回数；デフォルトは 200;
  * `tol::Real`: 残差ベクトルのノルムが下回るべき許容誤差。

# 出力

  * `results`: `LOBPCGResults` 構造体。`r.λ` と `r.X` に固有値と固有ベクトルが格納されます。
