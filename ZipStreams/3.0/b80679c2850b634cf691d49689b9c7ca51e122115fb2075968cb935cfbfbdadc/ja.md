```
write_file(sink, fname, data; [keyword arguments])
```

`data`をアーカイブシンク内の新しいファイル`fname`に一度にアーカイブします。

これは便利なメソッドで、`fname`という名前の新しいファイルをアーカイブ内に作成し、そのファイルに`data`のすべてを書き込みます。`data`引数は、`write(io, data)`メソッドが定義されているものであれば何でも可能です。

アーカイブに書き込まれたバイト数を返します。

キーワード引数は、[`open(::ZipArchiveSink, ::AbstractString)`](@ref)が受け入れるものと同じです。

!!! note "メモリ要件"
    このメソッドは、アーカイブに書き込む前に`data`をバッファに読み込みます。`data`とバッファされた（圧縮される可能性のある）コピーは、同時にメモリに収まる必要があります。
