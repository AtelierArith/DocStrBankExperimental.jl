画像をログ-ガボールフィルタのバンクで畳み込む。

```
使用法: (EO, BP) = gaborconvolve(img,  nscale, norient, minWaveLength, mult,
                                 sigmaOnf, dThetaOnSigma, Lnorm)

引数:
畳み込みはFFTを介して行われます。多くのパラメータは、周波数平面におけるフィルタの仕様に関連しています。

  変数名       推奨値   説明
 ----------------------------------------------------------
   img                       畳み込む画像。
   nscale          = 4       ウェーブレットスケールの数。
   norient         = 6       フィルタの方向の数。
   minWaveLength   = 3       最小スケールフィルタの波長。
   mult            = 1.7     連続するフィルタ間のスケーリングファクタ。
   sigmaOnf        = 0.65    周波数領域におけるログガボールフィルタの
                             転送関数を記述するガウスの標準偏差と
                             フィルタ中心周波数の比。
   dThetaOnSigma   = 1.3     フィルタの方向間の角度間隔と
                             周波数平面でフィルタを構築するために使用される
                             角度ガウス関数の標準偏差の比。
   Lnorm            0        フィルタが正規化されるべきノルムを示す
                             オプションの整数。1の値はL1ノルムが同じフィルタを
                             生成し、2の値はL2ノルムが一致するフィルタを
                             生成します。デフォルト値の0は正規化を行わず
                             （フィルタは対数周波数スケール上で単位高さの
                             ガウス転送関数を持つ）
戻り値:

  EO - 複素値の畳み込み結果の2D配列
       EO[s,o] = スケールsおよび方向oの畳み込み結果。
       実部は偶対称フィルタとの畳み込みの結果であり、
       虚部は奇対称フィルタとの畳み込みの結果です。

       したがって：
       abs.(EO[s,o])はスケールsおよび方向oでの
                    画像上の畳み込みの大きさを返します。
       angle.(EO[s,o])は位相角を返します。

  BP - 各スケールsに対応するバンドパス画像の配列。
```

スペクトルエネルギーの均等なカバレッジを得るためのフィルタ設定に関するメモ

```
dThetaOnSigma 1.2 - 1.3
sigmaOnf  .90   mult 1.15
sigmaOnf  .85   mult 1.2
sigmaOnf  .75   mult 1.4       (帯域幅 ~1オクターブ)
sigmaOnf  .65   mult 1.7
sigmaOnf  .55   mult 2.2       (帯域幅 ~2オクターブ)
```

`sigmaOnf`が与えられたときの`mult`の決定は完全に経験的です。私が行うのは、周波数領域におけるフィルタ振幅の二乗の合計をプロットし、スペクトルのカバレッジがどれほど均等であるかを確認することです。スペクトルに同心円状の「ギャップ」がある場合は、`mult`を減少させるか、`sigmaOnf`を減少させる必要があります（これによりフィルタの帯域幅が増加します）。

外向きに放射状の「ギャップ」がある場合は、`dthetaOnSigma`を減少させる必要があります（フィルタの角度帯域幅が増加します）。
