画像内のエッジおよびコーナーの位相一致をログ-ガボールフィルターを使用して計算します。

引数は多く存在する可能性があります。以下が完全な使用法です：

```
  (M, m, or, ft, EO, T) = phasecong3(img; nscale, norient, minwavelength,
                          mult, sigmaonf, k, cutoff, g, noisemethod)
```

ただし、画像を除いてすべてのパラメータにはデフォルト値があり、使用法は次のようにシンプルにすることができます：

```
    (M,) = phasecong3(img)
```

キーワード引数：              デフォルト値      説明

nscale           4    - ウェーブレットスケールの数、3-6の値を試してください    norient          6    - フィルターの方向の数。    minwavelength    3    - 最小スケールフィルターの波長。    mult             2.1  - 連続するフィルター間のスケーリングファクター。    sigmaonf         0.55 - 周波数領域におけるログガボールフィルターの伝達関数を記述するガウスの標準偏差の比率。    k                2.0  - ノイズエネルギーの平均を超える標準偏差の数で、ノイズ閾値ポイントを設定します。                            ノイズの多い画像の場合、10または20まで変化させることをお勧めします。    cutoff           0.5  - 位相一致値がペナルティを受ける周波数分散の割合。    g                10   - 周波数分散の位相一致を重み付けするために使用されるシグモイド関数の遷移の鋭さを制御します。    noisemethod      -1   - ノイズ統計を決定するために使用されるメソッドを指定するパラメータ。                              -1 最小スケールフィルター応答の中央値を使用                              -2 最小スケールフィルター応答のモードを使用                               0+ noisemethod値を固定ノイズ閾値として使用

返される値：    M          - 位相一致共分散の最大モーメント。                 これはエッジ強度の指標として使用されます。    m          - 位相一致共分散の最小モーメント。                 これはコーナー強度の指標として使用されます。    or         - ラジアンでの方向画像 -pi/2からpi/2、 +ve反時計回り。                 0は垂直エッジに対応し、pi/2は水平です。    ft         - 画像内の各点での局所加重平均位相角。 pi/2の値は明るい線に対応し、0はステップに対応し、-pi/2は暗い線です。    EO         - 各スケールおよび方向の複素値畳み込み結果の2D配列    T          - 計算されたノイズ閾値（画像のノイズ特性を診断するのに役立つ場合があります）。 これを知ることで、固定閾値を指定し、計算時間を節約できます。

```

`EO[s,o]` = スケール`s`および方向`o`の畳み込み結果。 実部は偶対称フィルターとの畳み込みの結果であり、虚部は奇対称フィルターとの畳み込みの結果です。

したがって：       `abs.(EO[s,o])`はスケール`s`および方向`o`での画像上の畳み込みの大きさを返し、       `angle.(EO[s,o])`は位相角を返します。

畳み込みはFFTを介して行われます。 多くのパラメータは、周波数平面におけるフィルターの仕様に関連しています。 値は非常に重要ではないようで、デフォルトは通常問題ありません。 `nscales`およびノイズ補償ファクター`k`の値を試してみることをお勧めします。

スペクトルの均等なカバレッジを得るためのフィルターパラメータ

```

sigmaonf       .85   mult 1.3 sigmaonf       .75   mult 1.6     (フィルターバンド幅 ~1オクターブ) sigmaonf       .65   mult 2.1 sigmaonf       .55   mult 3       (フィルターバンド幅 ~2オクターブ) ```
