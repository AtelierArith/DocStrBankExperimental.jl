画像の位相一致性をモノジェニックフィルターを使用して計算します。

このコードは `phasecong3()` よりもかなり高速ですが、`phasecong3()` の方向性フィルターからの出力を好むかもしれません。

引数は多くの可能性があり、以下が完全な使用法です：

```
  (PC, or, ft, T) =
          phasecongmono(img; nscale, minwavelength, mult,
                        sigmaonf, k, cutoff, g, deviationgain, noisemethod)

ただし、画像を除いてすべてのパラメータにはデフォルトがあり、
使用法は次のようにシンプルにできます：

   (PC,) = phasecongmono(img)   # (PC,) を使用して、PCがすべての
                                # 戻り値のタプルではないようにします

一般的には、画像の後に設定したい一連のキーワード引数を渡し、
残りのパラメータはデフォルトに設定されます。例えば：

   (PC,) = phasecongmono(img, nscale = 5, minwavelength = 3, k = 2.5)

キーワード引数：
             デフォルト値      説明

   nscale           4    - ウェーブレットスケールの数、3-6の値を試してください
                           より低い値は、より細かいスケールの
                           特徴を明らかにします。より大きな値は「主要な」
                           特徴を強調します。
   minwavelength    3    - 最小スケールフィルターの波長。
   mult             2.1  - 連続するフィルター間のスケーリングファクター。
   sigmaonf         0.55 - ガウスの標準偏差の比率
                           ログガボールフィルターの伝達関数を
                           周波数領域で説明するためのもの。
   k                3.0  - ノイズエネルギーの標準偏差の数
                           平均を超えてノイズ閾値ポイントを設定します。
                           ノイズの多い画像の場合、10または
                           20まで変化させることをお勧めします。
   cutoff           0.5  - 位相一致性値がペナルティを受ける
                           周波数分散の分数測定。
   g                10   - 位相一致性を周波数分散に重み付けするために
                           使用されるシグモイド関数の遷移の鋭さを制御します。
   deviationgain    1.5  - 計算された位相偏差結果に適用する増幅。
                           これを増加させるとエッジ応答が鋭くなりますが、
                           増幅が大きすぎるとその大きさを減衰させることもあります。
                           使用するのに適切な値は1-2の範囲です。
   noisemethod      -1   - ノイズ統計を決定するために使用されるメソッドを指定するパラメータ。
                             -1 最小スケールフィルター応答の中央値を使用
                             -2 最小スケールフィルター応答のモードを使用
                              0+ noiseMethod値を固定ノイズ閾値として使用
                           値が0の場合、すべてのノイズ補償がオフになります。

戻り値：
   PC         - エッジの重要性を示す位相一致性
   or         - ラジアンでの方向画像 -pi/2 から pi/2、 +ve 反時計回り。
                0は垂直エッジに対応し、pi/2は水平です。
   ft         - 画像の各点での局所的な加重平均位相角。
                pi/2の値は明るい線に対応し、0はステップに、
                -pi/2は暗い線に対応します。
   T          - 計算されたノイズ閾値（画像のノイズ特性を診断するのに役立つことがあります）。
                これを知ることで、固定閾値を指定し、計算時間を節約できます。
```

畳み込みはFFTを介して行われます。多くのパラメータは周波数平面でのフィルターの仕様に関連しています。値はそれほど重要ではないようで、デフォルトは通常問題ありません。`nscales` と `k` の値、ノイズ補償係数を試してみることをお勧めします。

エッジ画像を取得するための典型的な操作のシーケンス：

```
 > (PC, or) = phasecongmono(img)
 > nm = nonmaxsup(PC, or, 1.5)   # 非最大抑制
 > bw = hysthresh(nm, 0.1, 0.3)  # ヒステリシスしきい値 0.1 - 0.3

スペクトルの均一なカバレッジを得るためのフィルター設定に関する注意
sigmaonf       .85   mult 1.3
sigmaonf       .75   mult 1.6     (フィルターバンド幅 ~1オクターブ)
sigmaonf       .65   mult 2.1
sigmaonf       .55   mult 3       (フィルターバンド幅 ~2オクターブ)
```

一般的に、大きなバンド幅フィルターを使用することでより良い結果が得られます。私は通常、`sigmaOnf` の値を0.55またはそれ以下に設定します。
