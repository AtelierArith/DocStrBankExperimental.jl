画像の位相値を量子化します。

```
使用法:  qimg = quantizephase(img, N)

引数: img::Array{T,2} where T <: Real - 処理する画像
                     N::Integer - 量子化された位相値の希望数

戻り値:  qimg::Array{Float64,2} - 位相量子化画像
```

画像の位相値は重要です。しかし、それにもかかわらず、知覚的損失がほとんどない状態で非常に強く量子化することができます。Nの値は4まで、あるいは3まで低くすることができます！N = 2を使用することも検討する価値があります。
