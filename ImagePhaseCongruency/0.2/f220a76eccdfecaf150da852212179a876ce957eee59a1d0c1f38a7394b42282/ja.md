画像の位相対称性をモノジェニックフィルタを使用して計算します。

この関数は、画像内の点の位相対称性を計算します。これは対比不変の対称性の測定値です。この関数は、ラインおよびブロブ検出器として使用できます。見つけたいラインのグレースケールの極性を指定できます。

このコードは`phasesym()`よりもかなり速いですが、`phasesym()`の方向性フィルタからの出力を好むかもしれません。

引数は多くありますが、完全な使用法は以下の通りです：

```
  (phSym, symmetryEnergy, T) =
               phasesymmono(img; nscale, minwaveLength, mult,
                            sigmaonf, k, polarity, noisemethod)
```

ただし、画像を除いてすべてのパラメータにはデフォルトがあり、使用法は次のようにシンプルにできます：

```
   (phSym,) = phasesymmono(img)
```

キーワード引数：              デフォルト値      説明

nscale           5    - ウェーブレットスケールの数、3-6の値を試してください    minwaveLength    3    - 最小スケールフィルタの波長。    mult             2.1  - 連続するフィルタ間のスケーリングファクター。    sigmaonf         0.55 - ガウスの標準偏差の比率                            ログガボールフィルタの伝達関数を記述する                            周波数領域でのフィルタ中心周波数に対して。    k                2.0  - ノイズエネルギーの標準偏差の数                            平均を超えてノイズ閾値ポイントを設定する。                            ノイズの多い画像の場合、10または                            20まで変化させることをお勧めします。    polarity         0    - 対称性特徴の「極性」を制御します。                             1 - 「明るい」点のみを返す                            -1 - 「暗い」点のみを返す                             0 - 明るい点と暗い点を返す。    noisemethod      -1   - パラメータは使用する方法を指定します                            ノイズ統計を決定するための。                              -1 最小スケールフィルタ応答の中央値を使用                              -2 最小スケールフィルタ応答のモードを使用                               0+ noiseMethod値を固定ノイズ閾値として使用                            0の値はすべてのノイズ補償をオフにします。

戻り値：    phSym                 - 位相対称性画像（値は0から1の間）。    symmetryEnergy        - 正規化されていない生の対称性エネルギーで、                            あなたの好みに合うかもしれません。    T                     - 計算されたノイズ閾値（画像のノイズ特性を診断するのに役立つことがあります）

```

畳み込みはFFTを介して行われます。多くのパラメータは、周波数平面でのフィルタの仕様に関連しています。値は非常に重要ではないようで、デフォルトは通常問題ありません。`nscales`と`k`、ノイズ補償ファクターの値を試してみることをお勧めします。

スペクトルの均等なカバレッジを得るためのフィルタ設定に関するメモ

```

sigmaonf       .85   mult 1.3 sigmaonf       .75   mult 1.6     (フィルタ帯域幅 ~1オクターブ) sigmaonf       .65   mult 2.1 sigmaonf       .55   mult 3       (フィルタ帯域幅 ~2オクターブ) ```
