画像の位相対称性をログ-ガボールフィルタを用いて計算します。

この関数は、画像内の点の位相対称性を計算します。これは対比不変の対称性の測定値です。この関数は、ラインおよびブロブ検出器として使用できます。見つけたいラインのグレースケールの極性を指定できます。

```
Usage:   (phSym, orientation, totalEnergy, T) =
                phasesym(img; nscale = 5, norient = 6, minwavelength = 3, mult = 2.1,
                         sigmaonf = 0.55, k = 2, polarity = 0, noisemethod = -1)

ただし、画像を除いて、すべてのパラメータにはデフォルト値があり、
使用法は次のようにシンプルにできます：

    (phSym,) = phasesym(img)

引数:
                    img  - 処理する画像。実数またはグレーの2D配列

キーワード引数:
              デフォルト値      説明
    nscale           5    - ウェーブレットスケールの数、3-6の値を試してください
    norient          6    - フィルタの方向の数。
    minwavelength    3    - 最小スケールフィルタの波長。
    mult             2.1  - 連続するフィルタ間のスケーリングファクタ。
    sigmaonf         0.55 - フィルタ中心周波数に対する
                            ログガボールフィルタの伝達関数を記述する
                            ガウスの標準偏差の比率。
    k                2.0  - ノイズエネルギーの標準偏差の数
                            平均を超えてノイズ閾値点を設定する。
                            ノイズの多い画像の場合、10または20まで
                            この値を変えてみることをお勧めします。
    polarity         0    - 対称性特徴の「極性」を制御します。
                             1 - 「明るい」点のみを返す
                            -1 - 「暗い」点のみを返す
                             0 - 明るい点と暗い点の両方を返す。
    noisemethod      -1   - ノイズ統計を決定するために使用される方法を指定するパラメータ。
                              -1 最小スケールフィルタ応答の中央値を使用
                              -2 最小スケールフィルタ応答のモードを使用
                               0+ noiseMethod値を固定ノイズ閾値として使用。

戻り値:
    phSym                 - 位相対称性画像（値は0から1の間）。
    orientation           - オリエンテーション画像。局所的な対称性エネルギーが最大となるオリエンテーション（ラジアン）
                            （-pi/2 - pi/2）、角度は反時計回りに正。注意
                            オリエンテーション情報は方向の数によって量子化されます。
    totalEnergy           - 正規化されていない生の対称性エネルギーで、あなたの好みに合うかもしれません。
    T                     - 計算されたノイズ閾値（画像のノイズ特性を診断するのに役立つことがあります）。これを知ることで
                            固定閾値を指定し、計算時間を節約できます。
```

畳み込みはFFTを介して行われます。多くのパラメータは、周波数平面におけるフィルタの仕様に関連しています。値は非常に重要ではないようで、デフォルトは通常問題ありません。`nscales`と`k`、ノイズ補償ファクタの値を試してみることをお勧めします。

スペクトルの均等なカバレッジを得るためのフィルタ設定に関するメモ

```
sigmaonf       .85   mult 1.3
sigmaonf       .75   mult 1.6     (フィルタ帯域幅 ~1オクターブ)
sigmaonf       .65   mult 2.1
sigmaonf       .55   mult 3       (フィルタ帯域幅 ~2オクターブ)
```
