画像に単一生成フィルタを適用して2D解析信号を取得します。

これはFelsbergの単一生成フィルタの実装です。

```
Usage: (f, h1f, h2f, A, theta, psi) =
            monofilt(img, nscale, minWaveLength, mult, sigmaOnf, orientWrap)
                             3         4           2     0.65    true/false
Arguments:
畳み込みはFFTを介して行われます。多くのパラメータは、周波数平面におけるフィルタの仕様に関連しています。

  Variable       Suggested   Description
  name           value
 ----------------------------------------------------------
   img                       畳み込む画像。実数またはグレースケールの配列。
   nscale          = 3       フィルタスケールの数。
   minWaveLength   = 4       最小スケールフィルタの波長。
   mult            = 2       連続するフィルタ間のスケーリングファクタ。
   sigmaonf        = 0.65    周波数領域におけるログガボールフィルタの
                             転送関数を記述するガウスの標準偏差の比率
                             とフィルタ中心周波数の比率。
   orientWrap       false    オプションのブールフラグで、-pi .. piの範囲から
                             0 .. piの範囲への方向データの「ラッピング」を
                             有効/無効にします。
                             これは位相角の解釈に影響します - 下記の
                             注釈を参照してください。デフォルトはfalseです。
Returns:
       f  - スケールに関するバンドパスフィルタ応答のベクトル。
     h1f  - スケールに関するバンドパスh1フィルタ応答のベクトル。
     h2f  - バンドパスh2フィルタ応答のベクトル。
       A  - 単一生成エネルギー応答のベクトル。
   theta  - 位相方向応答のベクトル。
     psi  - 位相角応答のベクトル。
```

`orientWrap`がtrueの場合、`theta`は`0 .. pi`の範囲で返されます。

`sigmaonf`の実験は、アプリケーションに応じて有用です。私は0.2（*非常に*大きな帯域幅を持つフィルタ）という値が有用であることをいくつかの場面で発見しました。
