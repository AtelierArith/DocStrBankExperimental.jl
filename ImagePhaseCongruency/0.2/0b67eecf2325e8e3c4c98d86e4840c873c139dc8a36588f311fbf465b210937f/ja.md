Moisanの周期的画像成分の2Dフーリエ変換。

```
Usage: (P, S, p, s) = perfft2(img)

Argument: img - 変換する画像
Returns:    P - 周期的画像成分の2D fft
            S - スムーズ成分の2D fft
            p - 周期的成分（空間領域）
            s - スムーズ成分（空間領域）
```

Moisanの「周期的かつスムーズな画像分解」は、画像を2つの成分に分解します。

```
    img = p + s
```

ここで、sは平均0の「スムーズ」成分であり、pは画像の境界を周期的に移動する際に鋭い不連続性を持たない「周期的」成分です。

この分解は、境界の不連続性から生じる最小限のアーティファクトで画像のFFTを取得したい場合に非常に便利です。画像pは、画像情報の大部分を集めますが、周期化アーティファクトを回避します。

この関数の典型的な使用法は、画像の「周期的のみ」のfftを取得することです。

```
  P = perfft2(img)
```

Pの振幅スペクトルを表示すると、通常見られる画像の境界から生じる典型的な垂直-水平の「交差」がないクリーンなスペクトルが得られます。

周波数領域でフィルタリングを行うためにこの関数を使用している場合は、s（空間領域のスムーズ成分）を保持し、最終的にフィルタリングされた結果に戻すことを検討してください。

「周期的のみ」のFFTを取得するための計算コストは、追加のFFTを取ることを含みます。
