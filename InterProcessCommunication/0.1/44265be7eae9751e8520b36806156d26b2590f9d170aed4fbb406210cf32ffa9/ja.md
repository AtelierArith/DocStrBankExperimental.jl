```
id = shmid(arg)
```

は、`arg`によって識別される既存のPOSIX共有メモリオブジェクトまたはSystem V共有メモリセグメントの識別子を返します。引数は次のいずれかです：

  * `SharedMemory`のインスタンス。
  * 共有メモリに格納されている内容を持つ`WrappedArray`のインスタンス。
  * POSIX共有メモリオブジェクトを識別するための`'/'`で始まる文字列（他の`'/'`は含まれない）。
  * 共有メモリセグメントに関連付けられたSystem V IPCキーを指定するための`IPC.Key`のインスタンス。この場合、オプションの第二引数`readonly`を`true`に設定すると、読み取り専用アクセスのみを要求します。それ以外の場合は、読み書きアクセスが要求されます。
  * System V共有メモリセグメントを指定するための`ShmId`のインスタンス。

参照：[`SharedMemory`](@ref)、[`shmrm`](@ref)。
