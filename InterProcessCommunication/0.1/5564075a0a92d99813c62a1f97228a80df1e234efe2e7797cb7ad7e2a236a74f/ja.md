```
ptr = shmat(id, readonly)
```

は、呼び出し元のアドレス空間にSystem V共有メモリセグメントをアタッチします。引数 `id` は共有メモリセグメントの識別子です。ブール引数 `readonly` は、セグメントを読み取り専用アクセス用にアタッチするかどうかを指定します。そうでない場合、セグメントは読み取りおよび書き込みアクセス用にアタッチされ、プロセスはセグメントに対する読み取りおよび書き込み権限を持っている必要があります。返される値は、呼び出し元のための共有メモリセグメントのアドレスです。

参照: [`shmdt`](@ref), [`shmrm`](@ref).
