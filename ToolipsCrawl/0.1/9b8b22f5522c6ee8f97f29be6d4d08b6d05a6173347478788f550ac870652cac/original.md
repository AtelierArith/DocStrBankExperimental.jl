### ToolipsCrawl

**ToolipsCrawl** provides a high-level web-scraping and web-crawling interface using the  `Toolips` web-development framework's requests and `Component` structures. The package also  featurs `ComponentFilters`, a set of premade filters for scraped components. See `?Crawler` for  usage

##### Module Composition

  * **ToolipsCrawl**
  * **ComponentFilters**
