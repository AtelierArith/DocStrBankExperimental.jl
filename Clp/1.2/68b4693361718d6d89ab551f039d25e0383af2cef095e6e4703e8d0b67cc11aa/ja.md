```
Clp_crash(model, gap, pivot)
```

クラッシュ - 現在は二重に焦点を当てており、二重が優先されクラッシュ基準が作成された場合は -2 を返します。二重が優先され、すべてのスラック基準が優先された場合は -1 を返します。基準がすべてスラックでなかった場合は 0 を返します。プライマルが優先され、すべてのスラック基準が優先された場合は 1 を返します。プライマルが優先され、クラッシュ基準が作成された場合は 2 を返します。

境界間のギャップが <="gap" の場合、変数を反転できます。

"pivot" が 0 の場合、ピボッティングは行われません（したがって、アルゴリズムの選択のみになります）。1 は単純なピボッティング（例：gub）、2 はミニイテレーションです。
