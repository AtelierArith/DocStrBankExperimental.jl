```
Frame <: AbstractDelay

Frame{K}(frame)
```

`Frame`は、ルール内から特定の前のタイムステップの[`Grid`](@ref)を使用することを可能にし、`get`を使用します。これはルールコード内でのみ使用されるべきであり、パラメータとしては使用されるべきではありません。

# 型パラメータ

  * `K::Symbol`: `init`内のグリッドの名前と一致します。

# 引数

  * `frame::Int`: 使用する正確なフレーム番号。

警告: この機能は実験的です。将来のバージョンで変更される可能性があり、すべてのケースで100%信頼できるわけではありません。問題が発生した場合は、GitHubの問題を報告してください。
