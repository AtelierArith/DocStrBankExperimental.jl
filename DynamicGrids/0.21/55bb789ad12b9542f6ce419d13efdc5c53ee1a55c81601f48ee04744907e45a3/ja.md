```
resume!(output::GraphicOutput, ruleset::Ruleset=ruleset(output); tstop, kw...)
```

最後に停止した場所からシミュレーションを再開します。引数については [`sim!`](@ref) を参照してください。キーワード引数 `tstop` を使用してシミュレーションの長さを延長できます。

# 引数

  * `output`: グリッドを保存するか、画面に表示するための [`Output`](@ref)。
  * `ruleset`: 1つ以上の [`Rule`](@ref) を含む [`Ruleset`](@ref)。これらは順番に実行されます。

# キーワード（オプション）

  * `tstop`: シミュレーションの新しい停止時間。デフォルトでは出力の長さから取得されます。
  * `fps`: 表示するフレーム毎秒。デフォルトでは出力から取得されます。
  * `simdata`: [`SimData`](@ref) オブジェクト。シミュレーション間で保持することで、重要な場合にパフォーマンスを向上させることができます。
