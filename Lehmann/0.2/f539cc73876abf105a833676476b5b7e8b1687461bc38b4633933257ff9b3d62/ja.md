```
struct DLRGrid
```

虚時間/マツバラ周波数相関関数のためのDLRグリッド

#メンバー:

  * `isFermi`: bool フェルミオンまたはボソン
  * `symmetry`: 粒子-ホール対称 :ph、または粒子-ホール非対称 :pha、または :none
  * `Euv` : スペクトル密度のUVエネルギースケール
  * `β` または `beta` : 逆温度
  * `Λ` または `lambda`: カットオフ = スペクトル密度のUVエネルギースケール * 逆温度
  * `rtol`: 許容絶対誤差
  * `size` : DLR基底の数
  * `ω` または `omega` : 選択された代表的な実周波数グリッド
  * `n` : 選択された代表的なマツバラ周波数グリッド（整数）
  * `ωn` または `omegaN` : (2n+1)π/β
  * `τ` または `tau` : 選択された代表的な虚時間グリッド
