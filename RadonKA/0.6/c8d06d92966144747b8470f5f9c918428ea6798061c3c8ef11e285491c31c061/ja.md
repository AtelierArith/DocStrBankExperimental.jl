```
RadonFlexibleCircle(N, in_height, out_height, weights)
```

  * `N` は `radon` の入力の最初と二番目の次元のサイズです。
  * `in_height` は、円の中心点に対する光線が円に入る垂直位置を示すベクトルまたは範囲で、中心点は `N ÷ 2 + 1` に位置しています。最大値と最小値はそれぞれ `(N+1) ÷ 2 - 1` と `-(N+1) ÷ 2 + 1` です。
  * `out_height` は、円の中心点に対する光線が円から出る垂直位置を示すベクトルまたは範囲で、中心点は `N ÷ 2 + 1` に位置しています。最大値と最小値はそれぞれ `(N+1) ÷ 2 - 1` と `-(N+1) ÷ 2 + 1` です。

一つの定義は次のようになります: `RadonFlexibleCircle(10, -4:4, zeros((9,)))` これは、光線が位置 `-4:4` で円に入りますが、すべての光線が円を出るときに位置 0 に集中することを示します。これはファンビームトモグラフィーの極端な形です。

  * `weights` は、各光線に異なる強さで重みを付けることができます。デフォルトでは `weights = 0 .* in_height .+ 1` です。
