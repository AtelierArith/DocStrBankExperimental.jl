```
contrast(anovaresult::AnovaResult, groupassignment::Vector{Int})
```

`groupassignment`で指定されたグループに対して単因子コントラストを計算します。有効なグループは「0」、「1」、および「2」であり、「0」は比較からグループを除外します。

注意: 非直交コントラストを使用する場合は、BonferroniまたはŠidák補正を使用してαレベル（結果が真である可能性が高いと考えるレベル）を調整してください:   Bonferroni: α′ = α/c（cはコントラストの数）   Šidák:      α′ = 1 - (1 - α)^(1 / c)（わずかに優れています）ここでc = コントラストの数

注意: 効果量は因子に関連する誤差項を使用して計算されます。他の選択肢も可能であり、各グループの誤差の平均やコントロールに関連する誤差を含むことができます。
