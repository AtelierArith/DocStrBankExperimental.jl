```
(xg, yg, ...) = ndgrid_array(v1, v2, ...)
```

ベクトルのセットから（密な）`Array`のタプルを構築するためのメソッドです。

このタプルは多くのメモリを使用する可能性があるため、一般的には避けるべきです！主にテストやタイミング比較のために提供されています。

各入力は、何らかの型の`AbstractVector`である必要があります。対応する出力Arrayは同じ要素型を持ちます。

このメソッドは、Matlabの`ndarray`関数と似た機能を提供しますが、ベクトルが任意の型であるため、より一般的です。

# 例

```jldoctest
julia> ndgrid_array(1:3, 1:2)
([1 1; 2 2; 3 3], [1 2; 1 2; 1 2])

julia> ndgrid(1:3, [:a,:b])
([1 1; 2 2; 3 3], [:a :b; :a :b; :a :b])
```
