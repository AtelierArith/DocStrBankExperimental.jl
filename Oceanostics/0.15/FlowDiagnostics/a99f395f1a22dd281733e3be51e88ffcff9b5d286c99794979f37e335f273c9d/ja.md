`Q`速度勾配テンソル不変量の値を計算します。これは通常`Q`と呼ばれ、流体の流れにおける渦を特定し視覚化するために一般的に使用されます。

定義と名称は、速度勾配テンソル`∂ⱼuᵢ`の固有値`λ`の方程式から来ています：

```
    λ³ + P λ² + Q λ + T = 0
```

ここで`Q`は次のように定義されます：

```
    Q = ½ (ΩᵢⱼΩᵢⱼ - SᵢⱼSᵢⱼ)
```

ここで`Sᵢⱼ= ½(∂ⱼuᵢ + ∂ᵢuⱼ)`および`Ωᵢⱼ= ½(∂ⱼuᵢ - ∂ᵢuⱼ)`です。詳細についてはdoi:10.1063/1.5124245を参照してください。
