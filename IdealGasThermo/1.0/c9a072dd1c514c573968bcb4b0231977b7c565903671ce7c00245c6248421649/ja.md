```
set_h!(gas::AbstractGas, hspec::Float64)
```

指定されたエンタルピーに対してガスの温度を非線形ニュートン-ラフソン法を用いて計算します。

# 例

```julia-repl
julia> gas = Gas();
julia> set_h!(gas, 0.0)
理想気体の状態
  T =  302.463 K
  P =  101.325 kPa
 cp =   29.108 J/K/mol
  h =    0.000 kJ/mol
  s =    0.199 kJ/K/mol

組成:
-----------------------------
 種類        Yᵢ  MW[g/mol]
-----------------------------
     空気     1.000     28.965
-----------------------------
     ΣYᵢ     1.000     28.965
```
