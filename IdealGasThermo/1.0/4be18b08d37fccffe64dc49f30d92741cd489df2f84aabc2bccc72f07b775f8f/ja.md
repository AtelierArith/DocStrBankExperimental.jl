```
Gas()
```

標準条件下の空気を表す `Gas` 型を返すコンストラクタ

関連情報は [`Gas`](@ref) を参照してください。

# 例

```julia-repl
julia> Gas()
理想気体で
  T =  298.150 K
  P =  101.325 kPa
 cp =   29.102 J/K/mol
  h =   -0.126 kJ/mol
  s =    0.199 kJ/K/mol

組成:
-----------------------------
 種類        Yᵢ  MW[g/mol]
-----------------------------
     空気     1.000     28.965
-----------------------------
     ΣYᵢ     1.000     28.965
```
