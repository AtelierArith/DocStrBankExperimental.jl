```
set_TP!(gas::AbstractGas, T::Float64, P::Float64)
```

温度と圧力 (T,P) をそれぞれ K と Pa で与えられたときの気体の状態を計算します。

# 例

```julia-repl
julia> gas = Gas(); # 標準条件下の空気からなる理想気体を作成
julia> set_TP!(gas, 298.15*2, 101325.0*2)
理想気体の状態
  T =  596.300 K
  P =  202.650 kPa
 cp =   30.418 J/K/mol
  h =    8.706 kJ/mol
  s =    0.214 kJ/K/mol

組成:
-----------------------------
 種類        Yᵢ  MW[g/mol]
-----------------------------
     空気     1.000     28.965
-----------------------------
     ΣYᵢ     1.000     28.965
```
