```
downscale_2D_climate!(glacier::Glacier2D)
```

与えられた氷河の2D気候構造を更新するために、気候データをダウンサンプリングします。

# 引数

  * `glacier::Glacier2D`: ダウンサンプリングされる気候データを含む氷河オブジェクト。

# 説明

この関数は、与えられた氷河の2D気候構造を以下のように更新します：

1. 2D気候ステップの温度、PDD（正の度日）、雪、雨のフィールドを気候ステップからの対応する値で更新します。
2. 2D気候ステップの勾配と平均勾配を更新します。
3. 現在の`S`を`RasterStack`構造で再投影することによって、温度勾配を適用し、選択された期間の雪/雨の割合を計算します。

# 注意

  * この関数は、`glacier`オブジェクトをその場で変更します。
