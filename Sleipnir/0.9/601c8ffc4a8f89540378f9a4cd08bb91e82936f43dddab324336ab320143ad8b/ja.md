```
initialize_glacier_climate!(glacier::AbstractGlacier, params::Parameters)
```

与えられた氷河の気候データを初期化します。

# 引数

  * `glacier::AbstractGlacier`: 気候データを初期化するための氷河オブジェクト。
  * `params::Parameters`: シミュレーション設定とパスを含むパラメータ。

# 説明

この関数は、氷河の気候データを次のように初期化します。

1. シミュレーションの時間範囲とステップに基づいてダミー期間を作成します。
2. NetCDFファイルから生の気候データを読み込みます。
3. ダミー期間の累積気候データを計算します。
4. 累積気候データを2Dグリッドにダウンサンプリングします。
5. 氷河の長期的な温度データを取得します。
6. 生の気候データ、累積気候データ、ダウンサンプリングされた2D気候データ、長期的な温度、平均温度、および平均勾配を含む気候データを氷河オブジェクトに格納します。
