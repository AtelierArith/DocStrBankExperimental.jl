```
get_cumulative_climate(climate; gradient_bounds=[-0.009, -0.003], default_grad=-0.0065)
```

与えられた気候データから累積気候統計を計算します。

# キーワード引数

  * `climate::Climate`: 温度、降水量、勾配データを含む気候オブジェクト。
  * `gradient_bounds::Vector{Float64}`: 勾配値の下限と上限を指定する2要素ベクトル。デフォルトは`[-0.009, -0.003]`。
  * `default_grad::Float64`: 使用するデフォルトの勾配値。デフォルトは`-0.0065`。

# 戻り値

  * `climate_sum::Dict{String, Any}`: 次のキーを含む辞書：

      * `"temp"`: 温度データからの正の度日（PDD）の合計。
      * `"prcp"`: 降水量データの合計。
      * `"gradient"`: 指定された範囲内にクリップされた勾配データの合計。
      * `"avg_temp"`: 平均温度。
      * `"avg_gradient"`: 平均勾配。
      * `"ref_hgt"`: 気候メタデータからの参照高さ。

# 注意事項

  * 温度データは正の度日（PDD）値のみを含むように修正されます。
  * 勾配データは、妥当な値を確保するために指定された範囲内にクリップされます。
