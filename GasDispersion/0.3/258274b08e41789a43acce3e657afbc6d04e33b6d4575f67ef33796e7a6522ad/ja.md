```
plume(scenario::Scenario, model::PlumeModel[, equationset::EquationSet])
```

与えられたシナリオでプルーム拡散モデルを実行し、濃度 c(x, y, z[, t]) を返す解を呼び出すことができます。

濃度は体積分率であり、`model` が指定されていない場合は、単純なガウスプルームモデルがデフォルトとなります。

`equationset` は、代替の相関関係のセットをモデルパラメータに使用することを指定するために使用されます。代替が存在する場合です。

すべてのモデルパラメータは、SI基本単位（すなわち、距離はm、速度はm/s、質量はkgなど）であると仮定されます。
