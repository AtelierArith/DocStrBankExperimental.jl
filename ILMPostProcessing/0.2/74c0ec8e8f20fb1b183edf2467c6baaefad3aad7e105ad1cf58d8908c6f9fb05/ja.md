```
displacement_field(v::VectorFieldSequence,x0,y0,Trange::Tuple[;dt=step(tr),alg=Euler()])
```

座標 `x0` と `y0` に初めている粒子の変位を、`v` における空間的に補間された速度場の列を使用して、時間の範囲 `Trange = (ti,tf)` にわたって計算します。（速度場のベクトルと時間配列を別々の引数として提供することもできます）。`Trange` の最終時間は、逆向きの軌道が望ましい場合、初期時間よりも早くすることができます。オプションのキーワード引数は `dt` で、時間ステップサイズ（デフォルトでは `tr` のステップサイズですが、1より大きい整数倍にすることもできます）。
