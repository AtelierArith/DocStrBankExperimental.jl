```
pod(X::Vector{T}[; tolerance=0.99])
```

配列のスナップショットデータ `X` からPODモードと関連する時間変化係数を計算します。この `X` はスナップショット行列の役割を果たし、その列はデータのスナップショットです。しかし、実際には `Vector{T}` 型として保存される必要があり、`T<:GridData` である必要があります。これは、`velocity(sol,sys,trange)` のような関数で生成できます。ここで、`sol` は `ODESolution`、`sys` は `ILMSystem`、`trange` は時間の配列で、例えば `trange=range(1,10,length=100)` のようになります。分解で保持されるPODモードの数は `tolerance` によって設定されます：これは保持する総エネルギーの割合を指定し、デフォルトは99パーセントです。

`PODModes` の出力は、以下のフィールドを持つ構造体です。

  * `Xmean`: データの時間平均。型 `T`
  * `Xnorm`: 平均が除去された元の `X` ベクトル。各要素は型 `T`
  * `phi`: PODモードのベクトル。各要素は型 `T`
  * `a`: POD係数の行列。列の数は `phi` のエントリの数と同じです。列 `k` は `phi` のモード `k` に対する時間変化係数を構成します。
