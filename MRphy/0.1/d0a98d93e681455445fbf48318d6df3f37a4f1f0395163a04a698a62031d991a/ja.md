```
UΦRot!(U, Φ, V, R)
```

`V`に対して軸-角、`U`-`Φ`に基づく回転を適用します。回転は`V`の3次元に沿ってブロードキャストされます。結果は`R`に上書きされます。

*入力*:

  * `U::TypeND(AbstractFloat,[2])` (nM, 3)、3Dの回転軸、単位ベクトルと仮定;
  * `Φ::TypeND(AbstractFloat,[1])` (nM、)、回転角;
  * `V::TypeND(AbstractFloat,[2,3])` (nM, 3, (3))、回転されるベクトル;
  * `R::TypeND(AbstractFloat,[2,3])` (nM, 3, (3))、回転されたベクトル、すなわち結果;

*出力*:

  * `R` 入力コンテナ`R`も便利のために返されます。

参照: [`UΦRot`](@ref)、[`B2UΦ!`](@ref)。
