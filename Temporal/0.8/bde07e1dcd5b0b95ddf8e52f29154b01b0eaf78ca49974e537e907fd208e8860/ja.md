```
merge(x::TS,y::TS;join::Char='o')::TS
```

インデックスによって2つの時系列オブジェクトをマージし、オプションで指定された結合タイプパラメータを使用します。

...

# 引数

  * `x::TS`: 結合の左側。
  * `y::TS`: 結合の右側。

オプション引数:

  * `join::Char='o'::TS`: マージを実行するために使用されるロジックを指定し、'o'（外部結合）、'i'（内部結合）、'l'（左結合）、または'r'（右結合）の値を取ることができます。デフォルトは外部結合で、その結果は`hcat(x, y)`または`[x y]`と同じです。
