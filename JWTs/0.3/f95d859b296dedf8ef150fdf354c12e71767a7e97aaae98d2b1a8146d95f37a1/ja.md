```
with_valid_jwt(f, jwt, keyset; kid=nothing)
```

有効なJWTで`f`を実行します。検証されたJWTは`f`への引数として渡されます。JWTが無効な場合、`ArgumentError`がスローされます。

引数:

  * `f`: 有効なJWTで実行する関数。検証されたJWTは`f`への引数として渡されます。
  * `jwt`: 使用するJWT文字列またはJWTオブジェクト。文字列が渡されると、JWTオブジェクトに変換されます。
  * `keyset`: 検証に使用するJWKSet。このキーセット内のキーのみが検証に使用されます。

キーワード引数:

  * `kid`: 検証に使用するキーID。指定されていない場合、JWTヘッダーからの`kid`が使用されます。
  * `algorithms`: リストされたアルゴリズムのいずれかで検証を確実にします。デフォルトでは強制されません。
