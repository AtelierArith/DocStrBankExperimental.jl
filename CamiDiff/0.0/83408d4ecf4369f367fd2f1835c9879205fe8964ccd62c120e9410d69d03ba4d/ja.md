```
Grid(ID, name, T, N, r, r′, r′′, h, r0, epn, epw, k, p, polynom)
```

フィールドを持つ型:

  * `.ID`:    グリッド識別子名 (`::Int`)
  * `.name`:  グリッド識別子名 (`::String`)
  * `.T`:     グリッドタイプ名 (`::Type`)
  * `.N`:     グリッドポイントの数 (`::Int`)
  * `.r`:   表にしたグリッド関数 (`::Vector{T}`)
  * `.r′`:   表にしたグリッド関数の一次導関数 (`::Vector{T}`)
  * `.r′′`:   表にしたグリッド関数の二次導関数 (`::Vector{T}`)
  * `.h` :    グリッドステップの乗数 (`::T`)
  * `.r0`:    グリッドスケールファクター (`::T`)
  * `.epn`:   台形法の端点補正に使用される端点の数（奇数でなければならない） (`::Int`)
  * `.epw`:   n=1:epn のための台形法の端点重み (`::Vector{Vector{T}}`)
  * `.k`:     有限差分の次数 (`::Int`)
  * `.p`:     切り捨て指数グリッドのみに使用; 切り捨ての指数 (`::Int`)
  * `.polynom`: 多項式グリッドのみに使用: 多項式 (`::Vector{T}`)

オブジェクト `Grid` は、関数 [`castGrid`](@ref) を使用して作成するのが最適です。
