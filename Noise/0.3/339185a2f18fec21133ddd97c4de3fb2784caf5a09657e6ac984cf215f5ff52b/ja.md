```
salt_pepper(X; salt_prob=0.5, salt=1.0, pepper=0.0[, prob=0.1])
```

配列 `X` に塩と胡椒のノイズが影響を与えます。 `X` は配列またはRGBまたはグレイ画像であることができます。 `prob` は、ピクセルがノイズの影響を受ける確率のためのオプションの引数です。 `salt_prob` は塩ノイズの確率を表すキーワード引数です。 したがって、胡椒ノイズの確率は 1-`salt_prob` です。 `salt` は塩ノイズの値を指定するためのキーワード引数です。 `pepper` は胡椒ノイズの値を指定するためのキーワード引数です。
