```
poisson(X; scaling=nothing, clip=false)
```

ポアソンノイズの影響を受けた配列 `X` を返します。各位置でポアソンノイズは強度に個別に影響を与え、位置の値はポアソン分布の期待値を表します。ポアソンノイズは離散的なイベントによるものであるため、オプションの引数 `scaling` を提供する必要があります。この `scaling` は配列の最大値を離散的なイベントの数に接続します。最大値はスケーリングされ、その後ポアソンノイズが適用されます。その後、元の強度が保持されるように全体の配列を再スケーリングしますが、ポアソンノイズが適用されています。`clip` はキーワード引数です。指定された場合、値を [0, 1] にクリップします。
