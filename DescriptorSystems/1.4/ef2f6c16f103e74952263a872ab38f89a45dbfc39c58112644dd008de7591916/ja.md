```
sys = dss(A, E, B, C, D; Ts = 0, check_reg = false, 
          atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ )
```

`Ts = 0` の場合、連続時間状態空間システムの形式 `sys = (A-λE,B,C,D)` の記述子システムモデルを作成します。

```
Edx(t)/dt = Ax(t) + Bu(t) ,
y(t)      = Cx(t) + Du(t) ,
```

ここで `x(t)`、`u(t)`、`y(t)` はそれぞれシステム状態ベクトル、システム入力ベクトル、システム出力ベクトルであり、連続時間変数 `t` に対して定義されます。

非ゼロの正のサンプリング時間 `Ts = ΔT` の場合、記述子システムモデルは次の形式の離散時間状態空間システムを指定します。

```
Ex(t+ΔT) = Ax(t) + Bu(t)
y(t)     = Cx(t) + Du(t)
```

ここで、時間変数 `t` の離散値は `t = 0, ΔT, 2ΔT, ...` です。サンプリング時間が指定されていない場合は、`Ts = -1` を使用します。この場合、慣例として `ΔT = 1` となります。

ゼロフィードスルーマトリックス `D` を持つシステムの場合、`D = 0`（スカラーゼロ）に設定することが可能です。

標準状態空間システムの場合、`E` は単位行列です。この場合、`E = I`（ブール均一スケーリング）に設定することが可能です。あるいは、

```
sys = dss(A, B, C, D; Ts = 0)
```

を使用して標準システムを作成します。

静的ゲイン `D` に対応するシステムの場合は、次のようにします。

```
sys = dss(D; Ts = 0)
```

すべてまたは一部の行列を使用して記述子システムを指定することが可能です。

```
sys = dss(A = mat1, E = mat2, B = mat3, C = mat4, D = mat5; Ts = 0, check_reg = false, 
          atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ)
```

ここで `A`、`E`、`B`、`C`、`D` はそれぞれ適切な行列値 `mat1`、`mat2`、`mat3`、`mat4`、`mat5` に設定されたキーワードパラメータです。システム行列の一部が省略された場合、適切なサイズのゼロ行列が代わりに使用されます。

ペンシル `A-λE` が正則であると仮定します（すなわち、`det(A-λE) ̸≡ 0`）。したがって、効率の観点から、`A-λE` の正則性はデフォルトで*テストされません*。`check_reg = true` の場合、`A-λE` の正則性が追加でチェックされます。この場合、キーワード引数 `atol1`、`atol2`、`rtol` はそれぞれ、`A` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A` と `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は正方行列 `A` と `E` の次数、`ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` を使用して、同時に `atol1 = atol` および `atol2 = atol` を設定することができます。
