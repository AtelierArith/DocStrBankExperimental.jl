```
 sysf = grsfg(sys, γ; fast = true, stabilize = true, offset = β, 
              atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ)
```

記述子システム `sys = (A-λE,B,C,D)` に対して、伝達関数行列 `G(λ)` と ${\small γ > \|G(λ)\|_∞}$ を用いて、最小位相右スペクトル因子 `sysf = (Af-λEf,Bf,Cf,Df)` を求めます。このとき、伝達関数行列 `F(λ)` は `F(λ)'*F(λ) = γ^2*I-G(λ)'*G(λ)` を満たします。`stabilize = true`（デフォルト）の場合、`sys` の事前安定化が行われます。この場合、`sys` は連続時間の場合は虚軸上に極を持たず、離散時間の場合は単位円上に極を持たない必要があります。`stabilize = false` の場合、事前安定化は行われず、この場合 `sys` は安定でなければなりません。

安定性領域 `Cs` の境界に極が存在するかどうかを評価するために、キーワードパラメータ `offset = β` を介して境界オフセット `β` を指定できます。したがって、連続時間システムの場合、`Cs` の境界には実部が区間 `[-β,β]` にある複素数が含まれ、離散時間システムの場合、`Cs` の境界には絶対値が区間 `[1-β,1+β]` にある複素数が含まれます。`β` のデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業用マシン精度です。

`stabilize = true` の場合、`A-λE` の有限および無限の固有値の事前分離が、`fast = true` の場合は列ピボットを用いたランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、`A`、`E`、および `C` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` を使用すると、`atol1 = atol`、`atol2 = atol`、`atol3 = atol` を同時に設定できます。

*方法:* [1] の因子分解アプローチの拡張が使用されます。
