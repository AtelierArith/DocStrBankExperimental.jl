```
gnrcf(sys; fast = true, ss = false, 
     atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ) -> (sysn, sysm)
```

記述子システム `sys = (A-λE,B,C,D)` に対して、正規化された右互素因子分解の因子 `sysn = (An-λEn,Bn,Cn,Dn)` と `sysm = (An-λEn,Bn,Cm,Dm)` を計算します。もし `sys`、`sysn` および `sysm` がそれぞれ伝達関数行列 `G(λ)`、`N(λ)` および `M(λ)` を持つなら、`G(λ) = N(λ)*inv(M(λ))` となり、`N(λ)` と `M(λ)` は適切で安定した伝達関数行列であり、`[N(λ);M(λ)]` は内部です。結果として `En = I` となるのは `ss = true` の場合です。

ペンシル削減アルゴリズムが使用され、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づいてランクの決定が行われ、`fast = false` の場合はより信頼性の高いSVD分解が行われます。

キーワード引数 `atol1`、`atol2` および `rtol` はそれぞれ、`A`、`B`、`C` および `D` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A`、`E`、`B`、`C` および `D` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は `A` の要素型のマシンイプシロンで、`n` はシステム `sys` の次数です。キーワード引数 `atol` を使用すると、`atol1 = atol`、`atol2 = atol` を同時に設定できます。

*方法:* ペンシル削減アルゴリズムが使用され、伝達関数行列 `[G(λ); I]` の内部範囲空間 `R(λ)` を [1] に記載された方法に基づいて決定します。これは [2] の削減アルゴリズムに基づいています。次に、因子 `N(λ)` と `M(λ)` は `R(λ)` の分割から得られ、`R(λ) = [N(λ);M(λ)]` となります。

[2] C. Oara.     スペクトルおよび内部–外部因子分解に対する構成的解法      ディスクに関して. Automatica,  41, pp. 1855–1866, 2005.  ```
