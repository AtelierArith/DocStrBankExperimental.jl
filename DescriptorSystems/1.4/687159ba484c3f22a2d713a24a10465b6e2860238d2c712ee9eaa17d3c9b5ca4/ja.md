```
r = RationalTransferFunction(num::AbstractPolynomial, den::AbstractPolynomial, Ts:Real)
```

分子と分母の多項式 `num` と `den` から、ラショナル伝達関数モデル `r` を構築し、サンプリング時間 `Ts` を指定します。

`r::RationalTransferFunction{T,λ,P <: Polynomial(T,λ),Ts}` は、`r(λ) = num(λ)/den(λ)` と定義されるラショナル伝達関数システムモデルオブジェクトであり、ここで `num(λ)` と `den(λ)` は係数が `T` の多項式で、未定義変数 `λ` を持ち、`Ts` はサンプリング時間です。次のようになります：

`r.num` は分子多項式 `num(λ)` です； 

`r.den` は分母多項式 `den(λ)` です。 

サンプリング時間 `Ts` は次の値を持つことができます：

  * `Ts = 0` は連続時間システムを示し、`λ = s` はラプラス変換における複素変数です；
  * `Ts > 0` または `Ts = -1` は離散時間システムを示し、`λ = z` は `Z` 変換における複素変数です； `Ts = -1` はサンプリング時間が未指定の離散時間システムを示します。

サンプリング時間は `r.Ts` として取得できます。未定義変数 `λ` に使用される記号（または *変数*）は、分子 `num(λ)` と分母 `den(λ)` の多項式の未定義変数に一般的に使用される記号であり、`r.var` として取得できます。分子多項式 `num(λ)` の根（`r(λ)` の *ゼロ* とも呼ばれる）は `r.zeros` として取得でき、分母多項式 `den(λ)` の根（`r(λ)` の *極* とも呼ばれる）は `r.poles` として取得できます。分子多項式 `num(λ)` と分母多項式 `den(λ)` の主係数の比（`r(λ)` の *ゲイン* とも呼ばれる）は `r.gain` として取得できます。
