```
 H = freqresp(sys, ω)
```

記述システム `sys = (A-λE,B,C,D)` の周波数応答 `H` を実数周波数 `ω` で計算します。

出力が `p` 個、入力が `m` 個のシステムに対して、`ω` の `N` 個の周波数値に対して、`H` は `p × m × N` の配列であり、`H[:,:,i]` には周波数応答の `i` 番目の値が含まれます。この値は `C*((w[i]*E - A)^-1)*B + D` として計算され、ここで `w[i] = im*ω[i]` は連続時間システムの場合、`w[i] = exp(im*ω[i]*|Ts|)` はサンプリング時間 `Ts` を持つ離散時間システムの場合です。

*方法:* 多くの `w[i]` の値に対して `C*((w[i]*E - A)^-1)*B + D` を効率的に評価するために、入力-出力同等の変換されたシステム `sys = (At-λEt,Bt,Ct,Dt)` に対して、行列 `w[i]*Et - At` が上ヘッセンベルグになるように事前に直交座標変換が行われます。これにより、[1] で提案されたヘッセンベルグ形式に基づくアプローチを使用して周波数応答を効率的に計算できます。

*参考文献:*

[1] Laub, A.J., "Efficient Multivariable Frequency Response Computations",     IEEE Transactions on Automatic Control, AC-26 (1981), pp. 407-408.
