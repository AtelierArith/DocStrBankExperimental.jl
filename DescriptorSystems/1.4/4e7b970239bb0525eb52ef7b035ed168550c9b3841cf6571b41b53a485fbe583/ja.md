```
gss2ss(sys; Eshape = "ident", atol = 0, atol1 = atol, atol2 = atol, rtol = nϵ) -> (sysr, r)
```

記述系 `sys = (A-λE,B,C,D)` を、非動的モードを持たず、同じ伝達関数行列を持つ入力-出力等価の記述系実現 `sysr = (Ar-λEr,Br,Cr,Dr)` に変換します。結果として得られる `Er` は、SVDのような形 `Er = blockdiag(E1,0)` であり、ここで `E1` は `r × r` の非特異行列で、`r` は `E` のランクです。

キーワード引数 `Eshape` は、`E1` の形状を次のように指定します：

  * `Eshape = "ident"`（デフォルトオプション）の場合、`E1` はランク `r` の単位行列であり、`E` が非特異であれば、結果として得られる系 `sysr` は標準状態空間系です。
  * `Eshape = "diag"` の場合、`E1` はランク `r` の対角行列であり、対角要素は `E` の非ゼロ特異値を減少順に並べたものです。
  * `Eshape = "triu"` の場合、`E1` はランク `r` の上三角非特異行列です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、および `A` と `E` の非ゼロ要素に対する相対許容誤差を指定します。キーワード引数 `atol` は、同時に `atol1 = atol` および `atol2 = atol` を設定するために使用できます。

`Eshape = "triu"` の場合、`E` と `A` の削減は、列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われます。`Eshape = "ident"` または `Eshape = "diag"` の場合、削減はより信頼性の高いSVD分解を使用して行われます。
