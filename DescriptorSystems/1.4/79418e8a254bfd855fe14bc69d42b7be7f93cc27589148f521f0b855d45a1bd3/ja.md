```
val = gzero(sys; fast = false, prescale, atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ)
```

記述システム `sys = (A-λE,B,C,D)` に対して、システム行列ペンシルの有限および無限スミスゼロを含む複素ベクトル `val` を返します。

```
           | A-λE | B | 
   S(λ) := |------|---| .
           |  C   | D |
```

`val` の値は、ペンシル `S(λ)` の *不変ゼロ* と呼ばれ、`A-λE` が *正則* であり、記述システムの実現 `sys = (A-λE,B,C,D)` が *不可約* である場合、システムの伝達関数行列の *伝達ゼロ* となります。

`prescale = true` の場合、記述システム行列の事前バランシングが行われます。デフォルト設定は `prescale = gbalqual(sys) > 10000` であり、ここで `gbalqual(sys)` は記述システムモデル `sys` のスケーリング品質です（[`gbalqual`](@ref) を参照）。

ゼロの計算は、直交類似変換を使用してペンシル `S(λ)` を適切なクロンカー様式に還元することによって行われ、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定が含まれ、`fast = false` の場合は、より信頼性の高いSVD分解が使用されます。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A`、`B`、`C` および `D` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`A`、`E`、`B`、`C` および `D` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズ、`ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。
