```
gzeroinfo(sys; smarg, fast = false, prescale, atol = 0, atol1 = atol, atol2 = atol, 
          rtol = n*ϵ, offset = sqrt(ϵ)) -> (val, info)
```

記述システム `sys = (A-λE,B,C,D)` に対して、システム行列ペンシル `S(λ)` の有限および無限スミスゼロを含む複素ベクトル `val` を返します。

```
          | A-λE | B | 
   S(λ) = |------|---| 
          |  C   | D |
```

および、ペンシル `S(λ)` のクロンカー構造に関する情報を含む名前付きタプル `info` を返します。 `val` の値はペンシル `S(λ)` の *不変ゼロ* と呼ばれ、`A-λE` が *正則* であり、記述システムの実現 `sys = (A-λE,B,C,D)` が *不可約* である場合、システムの伝達関数行列の *伝達ゼロ* となります。

`prescale = true` の場合、記述システム行列の事前バランスが実行されます。 デフォルト設定は `prescale = gbalqual(sys) > 10000` であり、ここで `gbalqual(sys)` は記述システムモデル `sys` のスケーリング品質です（[`gbalqual`](@ref) を参照）。

安定性分析の目的のために、有限ゼロに対して安定性マージン `smarg` を指定でき、安定性領域境界オフセット `β` と組み合わせて、次のように安定性領域の境界に属する有限ゼロを数値的に評価します：連続時間の場合、これらは実部が区間 `[smarg-β, smarg+β]` にある有限ゼロであり、離散時間の場合、これらはモジュールが区間 `[smarg-β, smarg+β]` にある有限ゼロです。 安定性マージン `smarg` のデフォルト値は、連続時間システムの場合は `0`、離散時間システムの場合は `1` です。 `β` に使用されるデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業機械精度です。

名前付きタプル `info` には次の情報が含まれます：

`info.nfz` はペンシル `S(λ)` の有限固有値の数（`sys` の有限ゼロの数でもあります）です；

`info.niev` はペンシル `S(λ)` の無限固有値の数です；

`info.nisev` はペンシル `S(λ)` の *単純* 無限固有値の数です；

`info.niz` はシステム `sys` の無限ゼロの数です；

`info.nfsz` は有限安定ゼロの数、すなわち、連続時間または離散時間システムに対してそれぞれ `smarg-β` より小さい実部またはモジュールを持つ有限ゼロの数です；

`info.nfsbz` は安定性領域の境界上の有限ゼロの数、すなわち、連続時間または離散時間システムに対してそれぞれ `[smarg-β, smarg+β]` の区間に実部またはモジュールを持つ有限ゼロの数です；

`info.nfuz` は有限不安定ゼロの数、すなわち、連続時間または離散時間システムに対してそれぞれ `smarg+β` より大きい実部またはモジュールを持つ有限ゼロの数です；

`info.nrank` はペンシル `S(λ)` の通常のランクです；

`info.miev` は整数ベクトルで、ペンシル `S(λ)` の無限固有値の重複度を含みます（`S(λ)` のクロンカー形式における無限ブロックの次元でもあります）；

`info.miz` は整数ベクトルで、次のように `S(λ)` の無限ゼロの重複度に関する情報を含みます：`S(λ)` は重複度 `i` の無限ゼロ `info.mip[i]` を持ち、`S(λ)` に無限ゼロがない場合は空です；

`info.rki` は整数ベクトルで、ペンシル `S(λ)` の *右クロンカーインデックス* を含みます（正則ペンシルの場合は空です）；

`info.lki` は整数ベクトルで、ペンシル `S(λ)` の *左クロンカーインデックス* を含みます（正則ペンシルの場合は空です）；

`info.regular` はペンシル `S(λ)` が正則である場合は `true` に設定され、ペンシル `S(λ)` が特異である場合は `false` に設定されます；

`info.stable` はペンシル `S(λ)` が安定な有限ゼロのみを持ち、すべての無限ゼロが単純である場合は `true` に設定され、それ以外の場合は `false` に設定されます。

*注意:* ペンシル `S(λ)` の有限ゼロと有限固有値は同じですが、無限固有値の重複度は無限ゼロの重複度よりも1つ多いです。

ゼロの計算は、ペンシル `S(λ)` を適切なクロンカー様式に減少させることによって行われ、直交類似変換を使用し、ランクを明らかにするQR分解（列ピボットを使用）に基づくランク決定を含みます。 `fast = true` の場合は、より信頼性の高いSVD分解を使用します。 `fast = false` の場合。

キーワード引数 `atol1`、`atol2` および `rtol` は、`A`、`B`、`C` および `D` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、および `A`、`E`、`B`、`C` および `D` の非ゼロ要素の相対許容誤差をそれぞれ指定します。 デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズで、`ϵ` は作業機械エプシロンです。 キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。 ```
