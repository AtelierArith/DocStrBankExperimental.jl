```
c2d(sysc, Ts, meth = "zoh"; x0, u0, standard = true, fast = true, prewarp_freq = freq, 
           state_mapping = false, simple_infeigs = true, 
           atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ) -> (sysd, xd0, Mx, Mu)
```

連続時間記述子システム `sysc = (A-sE,B,C,D)` に対して、適切な伝達関数行列 `Gc(λ)` を用いて、サンプリング時間 `Ts` に対する対応する離散化記述子システム `sysd = (Ad-zEd,Bd,Cd,Dd)` を、選択された離散化手法 `meth` に従って計算します。キーワード引数 `standard` は、`standard = true`（デフォルト）の場合に `Ed = I` の標準状態空間実現を計算するオプションを指定し、`standard = false` の場合には記述子システムの実現を計算します。キーワード引数 `simple_infeigs = true` は、ペア `(A,E)` の単純無限固有値のみが期待されることを示します（デフォルト）。`simple_infeigs = false` の設定は、ペア `(A,E)` の制御不能または観測不能な高次無限一般化固有値が存在し、それを除去する必要があることを示します。`xd0` は、連続時間システム `sysc` の初期条件 `x0` と入力 `u0` から決定された離散時間システム `sysd` の状態のマッピングされた初期条件です。キーワード引数 `state_mapping = true` は、連続時間システム `sysc` と離散時間システム `sysd` のシステム状態ベクトルの値 `xc(t)` と `xd(t)`、および入力ベクトル `u(t)` が `xc(t) = Mx*xd(t)+Mu*u(t)` のように関連付けられるように、状態マッピング行列 `Mx` と `Mu` を計算するオプションを指定します。`state_mapping = false`（デフォルトオプション）の場合、`Mx = nothing` および `Mu = nothing` となります。

以下の離散化手法は、`meth` を適切に選択することで実行できます：

```
"zoh"     - 入力のゼロ次保持（デフォルト）； 

"foh"     - 入力の線形補間（一次保持とも呼ばれる）；

"impulse" - インパルス不変離散化； 

"Tustin"  - タスティン変換（台形積分とも呼ばれる）：非ゼロの前方変換周波数
            `freq` をキーワードパラメータ `prewarp_freq = freq` を使用して指定することで、
            `Gd(exp(im*freq*Ts)) = Gc(im*freq)` を保証します。
```

基礎となるペンシル操作アルゴリズムは、`fast = true`（デフォルト）の場合は列ピボッティングを伴うランクを明らかにするQR分解を使用し、`fast = false` の場合はSVD分解に基づくランク決定を行います。SVD分解に基づくランク決定は一般的により信頼性がありますが、関与する計算コストは高くなります。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A`、`B`、`C`、`D` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、および `A`、`B`、`C`、`D` および `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は正方行列 `A` と `E` の次数、`ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` を使用して、`atol1 = atol` および `atol2 = atol` を同時に設定できます。
