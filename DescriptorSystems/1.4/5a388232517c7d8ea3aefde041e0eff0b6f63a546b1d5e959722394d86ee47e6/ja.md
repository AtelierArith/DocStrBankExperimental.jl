```
gpoleinfo(sys; smarg, fast = false, atol = 0, atol1 = atol, atol2 = atol, 
          rtol = n*ϵ, offset = sqrt(ϵ)) -> (val, info)
```

記述システム `sys = (A-λE,B,C,D)` に対して、複素ベクトル `val` を返します。このベクトルには、システムポールペンシル `P(λ) := A-λE` の有限および無限のゼロが含まれ、名前付きタプル `info` にはポールペンシル `P(λ)` の固有値構造に関する情報が含まれます。`val` の値は、`A-λE` が *正則* であり、記述システムの実現 `sys = (A-λE,B,C,D)` が *不可約* である場合、`sys` の伝達関数行列の *ポール* です。ペンシル `A-λE` が特異な場合、`val` には `NaN` 要素も含まれ、その数はペンシル `A-λE` のランク欠損を示します。

安定性分析の目的で、有限固有値に対して安定性マージン `smarg` を指定でき、安定性領域境界オフセット `β` と組み合わせて、次のように安定性領域の境界に属する有限固有値を数値的に評価します：連続時間の場合、これらは実部が区間 `[smarg-β, smarg+β]` にある有限固有値であり、離散時間の場合、これらは絶対値が区間 `[smarg-β, smarg+β]` にある有限固有値です。安定性マージン `smarg` のデフォルト値は、連続時間システムの場合は `0`、離散時間システムの場合は `1` です。`β` に使用されるデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業機械精度です。

名前付きタプル `info` には次の情報が含まれます：

`info.nfev` はペンシル `A-λE` の有限固有値の数（`sys` の有限ポールの数でもあります）です；

`info.niev` はペンシル `A-λE` の無限固有値の数です；

`info.nisev` はペンシル `A-λE` の *単純* 無限固有値の数（非動的モードとも呼ばれます）です；

`info.nip` はシステム `sys` の無限ポールの数です；

`info.nfsev` は有限安定固有値の数、すなわち、連続時間または離散時間システムに対してそれぞれ `smarg-β` より小さい実部または絶対値を持つ有限固有値の数です；

`info.nfsbev` は安定性領域の境界にある有限固有値の数、すなわち、連続時間または離散時間システムに対してそれぞれ `[smarg-β, smarg+β]` の区間に実部または絶対値がある有限固有値の数です；

`info.nfuev` は有限不安定固有値の数、すなわち、連続時間または離散時間システムに対してそれぞれ `smarg+β` より大きい実部または絶対値を持つ有限固有値の数です；

`info.nhev` は `NaN` に設定された *隠れた* 固有値の数（ペンシル `A-λE` が特異な場合のみ非ゼロになる可能性があります）です；

`info.nrank` はペンシル `A-λE` の通常のランクです；

`info.miev` は整数ベクトルで、ペンシル `A-λE` の無限固有値の重複度を次のように含みます：`i` 番目の要素 `info.miev[i]` は無限の基本除数の次数（すなわち、無限固有値の重複度）であり、無限ポールの数は `info.miev` の成分の合計です；

`info.mip` は整数ベクトルで、`A-λE` の無限ゼロの重複度に関する情報を次のように含みます：`i` 番目の要素 `info.mip[i]` は `k-1` に等しく、ここで `k` は無限の基本除数の次数で `k > 0` であり、無限ポールの数は `info.mip` の成分の合計です；

`info.rki` はペンシル `A-λE` の *右クロンネッカー指数* を含む整数ベクトル（正則ペンシルの場合は空）です；

`info.lki` はペンシル `A-λE` の *左クロンネッカー指数* を含む整数ベクトル（正則ペンシルの場合は空）です；

`info.regular` はペンシル `A-λE` が正則である場合は `true` に設定され、ペンシル `A-λE` が特異である場合は `false` に設定されます；

`info.proper` はペンシル `A-λE` が正則であり、すべての無限固有値が単純（非動的モードのみを持つ）である場合は `true` に設定され、ペンシル `A-λE` が特異であるか、より高次の無限固有値を持つ場合は `false` に設定されます；

`info.stable` はペンシル `A-λE` が正則であり、安定な有限固有値のみを持ち、すべての無限固有値が単純（非動的モードのみを持つ）である場合は `true` に設定され、それ以外の場合は `false` に設定されます。

*注意:* 有限ポールとペンシル `P(λ)` の有限固有値は同じですが、無限固有値の重複度は無限ポールの重複度に対して1つ多くなります。

ペンシル `P(λ)` を適切なクロンネッカー様式に削減するために、列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を含む直交類似変換が行われます。`fast = true` の場合は、より信頼性の高いSVD分解が行われます。

キーワード引数 `atol1`、`atol2` および `rtol` は、`A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`A` および `E` の非ゼロ要素の相対許容誤差をそれぞれ指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `P(λ)` のサイズで、`ϵ` は作業機械エプシロンです。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。 ```
