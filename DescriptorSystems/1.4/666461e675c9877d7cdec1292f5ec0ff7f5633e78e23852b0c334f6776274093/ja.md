```
r = rtf(z, p, k; Ts = rts, var = rvar)
```

ゼロ `z`、極 `p`、ゲイン `k`、サンプリング時間 `rts` および変数名 `rvar` から、合理的な伝達関数 `r(λ) = k*num(λ)/den(λ)` を作成します。ここで、`num(λ)` と `den(λ)` はそれぞれ `z` と `p` に等しい根を持つ単項式多項式であり、`r.Ts = rts`（デフォルトは `rts = 0`）および `r.var = rvar`（デフォルトは `Ts = 0` の場合は `rvar = :s`、`Ts ≠ 0` の場合は `rvar = :z`）となります。
