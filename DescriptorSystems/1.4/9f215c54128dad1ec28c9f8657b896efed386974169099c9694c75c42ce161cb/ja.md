```
isproper(sys; atol = 0, atol1 = atol, atol2 = atol, rtol = = n*ϵ, fast = true)
```

伝達関数行列 `G(λ)` が適切である場合は `true` を返し、そうでない場合は `false` を返します。  

制御不能および観測不能な無限固有値を持たない記述系の実現 `sys = (A-λE,B,C,D)` に対して、ペンシル `A-λE` に無限固有値がないこと、または無限固有値が存在する場合はすべての無限固有値が単純であることが確認されます。元の記述実現に制御不能または観測不能な無限固有値がある場合、これらは直交ペンシル削減アルゴリズムを使用して排除されます。 

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `A`、`B`、`C`、`D` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`A`、`B`、`C`、`D` および `E` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` の次数、`ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` を使用すると、`atol1 = atol` および `atol2 = atol` を同時に設定できます。
