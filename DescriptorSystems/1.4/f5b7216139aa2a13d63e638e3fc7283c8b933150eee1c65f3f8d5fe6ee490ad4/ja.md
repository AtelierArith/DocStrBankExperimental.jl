```
glinfnorm(sys, hinfnorm = false, rtolinf = 0.001, fast = true, offset = sqrt(ϵ), atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ) -> (linfnorm, fpeak)
```

記述子システム `sys = (A-λE,B,C,D)` に対して、伝達関数行列 `G(λ)` の `L∞` ノルム `linfnorm`（すなわち、`G(λ)` のピークゲイン）と、ピークゲインが達成される対応するピーク周波数 `fpeak` を計算します。`L∞` ノルムは、`G(λ)` が安定性領域の境界上に極を持つ場合、すなわち、連続時間の場合は拡張虚軸上に、離散時間の場合は単位円上にある場合は無限大になります。ペンシル `A-λE` が安定性領域の境界上に制御不能および/または観測不能な固有値を持つ場合、これらの固有値を排除するために、まず次元削減実現が決定されます（以下を参照）。

安定性領域の境界上に極がないことを確認するために、ペンシル `A-λE` の固有値は、連続時間システムの場合は区間 `[-β,β]` に実部を持たない必要があり、離散時間システムの場合は、モジュリが区間 `[1-β,1+β]` にない必要があります。ここで、`β` は安定性領域の境界オフセットです。使用するオフセット `β` は、キーワードパラメータ `offset = β` を介して指定できます。`β` に使用されるデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業機械精度です。

キーワード引数 `rtolinf` は、計算された無限ノルムの相対精度を指定します。`rtolinf` に使用されるデフォルト値は `0.001` です。

`hinfnorm = true` の場合、`H∞` ノルムが計算されます。この場合、`A-λE` のゼロの安定性が追加でチェックされ、安定していないシステムの場合、`H∞` ノルムは無限大になります。安定性を確認するために、ペンシル `A-λE` の固有値は、連続時間システムの場合は実部が `-β` より小さい必要があり、離散時間システムの場合はモジュリが `1-β` より小さい必要があります。

`E` が特異な連続時間システム `sys` の場合、まず制御不能および観測不能な有限および無限の固有値を持たない次元削減実現が決定されます。離散時間システムまたは可逆な `E` を持つシステムの場合、まず制御不能および観測不能な有限の固有値を持たない次元削減実現が決定されます。実施された次元削減におけるランクの決定は、`fast = true` の場合は列ピボットを用いたランクを明らかにするQR分解に基づき、`fast = false` の場合はより信頼性の高いSVD分解に基づきます。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、行列 `A`、`B`、`C`、`D` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、および `A`、`B`、`C`、`D` および `E` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は作業機械エプシロンで、`n` はシステム `sys` の次数です。キーワード引数 `atol` を使用して、`atol1 = atol` および `atol2 = atol` を同時に設定できます。
