```
 res = dssubsel(sys, S; minimal = true, fast = true, atol = 0, atol1 = atol, atol2 = atol, rtol = nϵ)
```

与えられた記述子システム `sys = (A-λE,B,C,D)` に対して、バイナリ構造行列 `S` によって指定されたゼロ-ワンパターンに対応するサブシステム `res` を選択します。もし `G(λ)` が `sys` の伝達関数行列であれば、`res` の伝達関数行列は要素ごとの積 `S .* G(λ)` です。

`minimal = true`（デフォルト）の場合、結果のシステム `res` の不可約実現が計算されます。そうでない場合、`minimal = false` の場合は、非最小の実現が返されます。

基礎となるペンシル操作に基づく最小実現アルゴリズムは、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解を使用し、そうでない場合はSVD分解に基づくランク決定を行います。SVD分解に基づくランク決定は一般的により信頼性が高いですが、関与する計算コストは高くなります。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ行列 `A`、`B`、`C`、`D` の非ゼロ要素に対する絶対許容誤差、行列 `E` の非ゼロ要素に対する絶対許容誤差、そして行列 `A`、`B`、`C`、`D` および `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `nϵ` であり、ここで `ϵ` は作業用 *マシンイプシロン* で、`n` はシステム `sys` の次数です。キーワード引数 `atol` は、`atol1 = atol` と `atol2 = atol` を同時に設定するために使用できます。
