```
isstable(sys[, smarg]; fast = true, atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ, offset = sqrt(ϵ))
```

安定な極のみを持つ記述系 `sys = (A-λE,B,C,D)` の場合は `true` を返し、そうでない場合は `false` を返します。

極ペンシル `P(λ) := A-λE` に無限の固有値が存在しないこと、または無限の固有値が存在する場合はすべての無限の固有値が単純であること、さらにすべての有限の固有値の実部が連続時間系の場合は `smarg-β` より小さく、離散時間系の場合は `smarg-β` より小さい絶対値を持つことが確認されます。ここで `smarg` は安定性マージン、`β` は安定性領域境界のオフセットです。連続時間系の安定性マージン `smarg` のデフォルト値は `0`、離散時間系のデフォルト値は `1` です。固有値の安定性を数値的に評価するために使用されるオフセット `β` は、キーワードパラメータ `offset = β` を介して指定できます。`β` のデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業用の機械精度です。

`E` が特異な場合、極の計算は、直交類似変換を使用してペンシル `P(λ)` を適切なクロンカー様式に削減することによって行われ、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定が含まれ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `A`、`B`、`C`、`D` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`A`、`B`、`C`、`D` および `E` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` の次数、`ϵ` は作業用の機械エプシロンです。キーワード引数 `atol` を使用すると、`atol1 = atol` と `atol2 = atol` を同時に設定できます。
