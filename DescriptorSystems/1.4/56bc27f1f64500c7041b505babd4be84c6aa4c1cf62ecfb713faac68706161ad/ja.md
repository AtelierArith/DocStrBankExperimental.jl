```
sys = rdss(n, p, m; id = [ ], disc = false, Ts, T = Float64, stable = false, nfuc = 0, iduc = [ ], 
           nfuo = 0, iduo = [ ], randlt = true, randrt = true)
```

ランダム化された記述子状態空間システム `sys = (A-λE,B,C,D)` を生成します。ここで、`p` は出力の数、`m` は入力の数であり、すべての行列は型 `T` のランダムに生成されたものです。結果として得られる `sys` は、`disc = false` の場合は連続時間システムであり、`disc = true` の場合は離散時間システムです。離散時間システムの場合、サンプル時間 `Δ` はキーワード引数 `Ts = Δ` を使用して指定できます（デフォルト: `Δ = -1`、すなわち指定なし）。

ベクトル `id` が空でない場合、`id[i]` は結果のペンシル `A-λE` の `i` 番目の無限基本除数の次数を指定します。したがって、`n` 個の有限固有値と `ni = sum(id)` 個の制御可能かつ観測可能な無限固有値を持ちます。`nfuc+nfuo > 0` の場合、システム `sys` は非最小であり、`A` は `nfuc` 個の制御不能な有限固有値と `nfuo` 個の観測不能な有限固有値を持ちます。ベクトル `iduc` が空でない場合、`iduc[i]` は結果のペンシル `A-λE` の制御不能な無限固有値を持つ `i` 番目の無限基本除数の次数を指定します。したがって、`niuc = sum(iduc)` 個の制御不能な無限固有値を持ちます。ベクトル `iduo` が空でない場合、`iduo[i]` は結果のペンシル `A-λE` の観測不能な無限固有値を持つ `i` 番目の無限基本除数の次数を指定します。したがって、`niuo = sum(iduo)` 個の観測不能な無限固有値を持ちます。`niuc+niuo > 0` の場合、システム `sys` は非最小であり、`A` は `niuc` 個の制御不能な無限固有値と `niuo` 個の観測不能な無限固有値を持ちます。

したがって、結果のペンシル `A-λE` は `n+nfuc+nfuo` 個の有限固有値と `ni+niuc+niuo` 個の無限固有値を持ちます。`stable = true` の場合、システム `sys` の適切な部分は安定しており、連続時間システムの場合は `A` がすべての有限固有値を持ち、実部が負であるか、離散時間システムの場合はモジュールが1未満である必要があります。

`randlt = true` の場合、ランダムに生成された直交またはユニタリ変換が `A`、`E`、および `B` に左から追加で適用されます。`randrt = true` の場合、ランダムに生成された直交またはユニタリ変換が `A`、`E`、および `C` に右から追加で適用されます。`randlt = false` および `randrt = false` の場合、システム行列 `A`、`E`、`B`、および `C` は、`A-λE` の制御不能および観測不能な有限および無限固有値を示すブロック構造形式になります：

```
A-λE = diag(A1-λE1, A2-λE2, A3-λE3, A4-λE4, A5-λE5, A6-λE6),  
B = [B1; B2; 0; 0; B5; B6 ], 
C = [C1 C2 C3 C4 0 0]
```

対角ブロック `A1`、`A2`、`A3`、`A4`、`A5`、`A6` の次数はそれぞれ `n`、`ni`、`nfuc`、`niuc`、`nfuo` および `niuo` です。
