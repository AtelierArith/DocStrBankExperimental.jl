```
isregular(sys; atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ)
```

`sys = (A-λE,B,C,D)` という記述子システムが正則ポールペンシル `A-λE` を持つ場合は `true` を返し、そうでない場合は `false` を返します。

ペンシル `A-λE` が正則であるかどうかをテストするために（すなわち、`det(A-λE) ̸≡ 0`）、基礎となる計算手順はペンシル `A-λE` を適切なクロンカー様形式に還元し、`A-λE` のランクに関する情報を提供します。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A` および `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズ、`ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` を使用すると、同時に `atol1 = atol` および `atol2 = atol` を設定できます。
