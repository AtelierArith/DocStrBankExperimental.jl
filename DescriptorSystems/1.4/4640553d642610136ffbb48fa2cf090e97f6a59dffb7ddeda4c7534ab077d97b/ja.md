```
 rtfbilin(type = "c2d"; Ts = T, Tis = Ti, a = val1, b = val2, c = val3, d = val4) -> (g, ginv)
```

一般的に使用されるいくつかの双線形変換とその逆の有理伝達関数を構築します。結果として得られる `g` は、双線形変換 `λ = g(δ)` における有理伝達関数 `g(δ)` を表し、`ginv` は双線形変換 `δ = ginv(λ)` におけるその逆変換 `ginv(λ)` を表します。`type` の値とキーワード引数の値 `Ts`、`Tis`、`a`、`b`、`c`、`d` に応じて、結果として得られる `g` と `ginv` は、次の形の一次有理伝達関数を含みます：`g(δ) = (a*δ+b)/(c*δ+d)` および `ginv(λ) = (d*λ-b)/(-c*λ+a)`。これらはそれぞれ `g(ginv(λ)) = λ` および `ginv(g(δ)) = δ` を満たします。

`type` の値に応じて、次の種類の変換が `Ts`、`Tis`、`a`、`b`、`c`、`d` で指定されたパラメータとともに生成されます：

```
"Cayley" - Cayley 変換: `s = g(z) = (z-1)/(z+1)` および `z = ginv(s) = (s+1)/(-s+1)`; 
           サンプリング時間 `g.Ts` は値 `T ≠ 0` に設定され（デフォルト `T = -1`）、`ginv.Ts = 0` に設定されます;
           g(z) と ginv(s) はそれぞれ連続から離散への変換および離散から連続への変換としても知られています; 

"c2d"    - "Cayley" のエイリアス

"Tustin" - Tustin 変換（台形積分とも呼ばれる）: `s = g(z) = (2*z-2)/(T*z+T)` および `z = ginv(s) = (T*s+2)/(-T*s+2)`; 
           サンプリング時間 `g.Ts` は値 `T ≠ 0` に設定され（デフォルト `T = -1`）、`ginv.Ts = 0` に設定されます;
           非ゼロの前方周波数 `freq` はキーワードパラメータ `prewarp_freq = freq` を使用して指定できます;

"Euler"  - オイラー積分（または前方オイラー積分）: `s = g(z) = s = (z-1)/T` および `z = ginv(s) = T*s+1`; 
           サンプリング時間 `g.Ts` は値 `T ≠ 0` に設定され（デフォルト `T = -1`）、`ginv.Ts = 0` に設定されます;

"BEuler" - 後方オイラー積分（または後方オイラー積分）: `s = g(z) = (z-1)/(T*z)` および `z = ginv(s) = 1/(-T*s+1)`; 
           サンプリング時間 `g.Ts` は値 `T ≠ 0` に設定され（デフォルト `T = -1`）、`ginv.Ts = 0` に設定されます;

"Moebius" - 一般的な（モビウス）双線形変換: `λ = g(δ) = (a*δ+b)/(c*δ+d)` および `δ = ginv(λ) = (d*λ-b)/(-c*λ+a)`;  
            サンプリング時間 `g.Ts` と `ginv.Ts` はそれぞれ値 `T` と `Ti` に設定されます;
            パラメータ `a`、`b`、`c`、`d` のデフォルト値は `a = 1`、`b = 0`、`c = 0`、`d = 1` です。
            一部の有用な特定のモビウス変換は、次のパラメータの選択に対応します: 
            - _平行移動_: `a = 1`、`b ≠ 0`、`c = 0`、`d = 1` (`λ = δ+b`)
            - _スケーリング_: `a ≠ 0`、`b = 0`、`c = 0`、`d = 1` (`λ = a*δ`)
            - _回転_: `|a| = 1`、`b = 0`、`c = 0`、`d = 1` (`λ = a*δ`)
            - _逆転_: `a = 0`、`b = 1`、`c = 1`、`d = 0`  (`λ = 1/δ`)

"lft"     - "Moebius"（線形分数変換）のエイリアス。
```
