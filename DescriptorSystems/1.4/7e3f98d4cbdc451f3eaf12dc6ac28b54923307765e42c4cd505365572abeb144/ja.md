```
gbilin(sys, g; compact = true, minimal = false, standard = true, atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ) -> (syst, ginv)
```

記述子システム `sys = (A-λE,B,C,D)` に対して、伝達関数行列 `G(λ)` と一次の実数有理伝達関数 `g = g(δ)` を用いて、ビリニア変換 `λ = g(δ) = (aδ+b)/(cδ+d)` に対応する `G(g(δ))` の記述子システム実現 `syst = (At-δEt,Bt,Ct,Dt)` を計算します。連続時間伝達関数 `g(δ)` の場合、`δ = s` はラプラス変換における複素変数であり、離散時間伝達関数の場合、`δ = z` は `Z`-変換における複素変数です。`syst` は `sys1` のサンプリング時間を継承します。`sysi1` はビリニア変換 `g(δ)` の逆を表す伝達関数 `ginv(λ) = (d*λ-b)/(-c*λ+a)` です（すなわち、`g(ginv(λ)) = 1`）。

キーワード引数 `compact` は、非動的モードを持たないコンパクトな記述子実現を計算するオプションを指定するために使用できます。`compact = true`（デフォルトオプション）であれば、非動的モードの排除を行い、`compact = false` であれば、非動的モードの排除を無効にします。

キーワード引数 `minimal` は、最小記述子実現を計算するオプションを指定します。`minimal = true` の場合は最小実現、`minimal = false` の場合は非最小実現（デフォルトオプション）となります。

キーワード引数 `standard` は、`syst` の標準状態空間実現（可能な場合）を計算するオプションを指定します。`standard = true`（デフォルト）であれば標準状態空間実現、`standard = false` であれば記述子システム実現となります。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A`、`B`、`C`、`D` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A`、`B`、`C`、`D` および `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は正方行列 `A` と `E` の次数、`ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。
