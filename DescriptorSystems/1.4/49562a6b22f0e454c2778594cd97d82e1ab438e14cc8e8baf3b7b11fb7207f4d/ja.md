```
gl2norm(sys, h2norm = false, fast = true, offset = sqrt(ϵ), atol = 0, atol1 = atol, atol2 = atol, atol3 = atol, atolinf = atol, rtol = n*ϵ)
```

記述子システム `sys = (A-λE,B,C,D)` の伝達関数行列 `G(λ)` の `L2` ノルムを計算します。`L2` ノルムは、`G(λ)` が安定性領域の境界上に極を持つ場合、すなわち連続時間の場合は拡張虚軸上に、離散時間の場合は単位円上にある場合は無限大になります。また、連続時間システムが無限大でのゲインが `atolinf` より大きい場合も `L2` ノルムは無限大になります。ペンシル `A-λE` が安定性領域の境界上に制御不能および/または観測不能な固有値を持つ場合、これらの固有値を排除するためにまず次元削減実現が決定されます（以下を参照）。

安定性領域の境界上に極がないことを確認するために、ペンシル `A-λE` の固有値は、連続時間システムの場合は実部が区間 `[-β,β]` にない必要があります。また、離散時間システムの場合は、モジュールが区間 `[1-β,1+β]` にない必要があります。ここで、`β` は安定性領域の境界オフセットです。使用するオフセット `β` は、キーワードパラメータ `offset = β` を介して指定できます。`β` に使用されるデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業機械精度です。

`h2norm = true` の場合、`H2` ノルムが計算されます。*極ペンシル* `A-λE` が不安定なゼロ（すなわち不安定な極）を持つ場合、または連続時間システムが無限大でのゲインが `atolinf` より大きい場合、`H2` ノルムは無限大になります。安定性を確認するために、ペンシル `A-λE` の固有値は、連続時間システムの場合は実部が `-β` より小さい必要があります。また、離散時間システムの場合はモジュールが `1-β` より小さい必要があります。

`E` が特異な連続時間システム `sys` の場合、まず制御不能および観測不能な有限および無限の固有値を持たない次元削減実現が決定されます。離散時間システムまたは可逆な `E` を持つシステムの場合、まず制御不能および観測不能な有限の固有値を持たない次元削減実現が決定されます。実施された次元削減におけるランクの決定は、`fast = true` の場合は列ピボットを用いたランクを明らかにするQR分解に基づいており、`fast = false` の場合はより信頼性の高いSVD分解に基づいています。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `A`、`B`、`C`、`D` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、および `A`、`B`、`C`、`D` および `E` の非ゼロ要素の相対許容誤差を指定します。キーワード引数 `atol3` は `B` の非ゼロ要素の絶対許容誤差を指定し、`h2norm = false` の場合に不安定な固有値の制御可能性テストにのみ使用されます。キーワード引数 `atolinf` は `λ = ∞` における `G(λ)` のゲインの絶対許容誤差です。使用されるデフォルト値は `atolinf = 0` です。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は作業機械エプシロンで、`n` はシステム `sys` の次数です。キーワード引数 `atol` は、同時に `atol1 = atol`、`atol2 = atol` および `atol3 = atol` を設定するために使用できます。
