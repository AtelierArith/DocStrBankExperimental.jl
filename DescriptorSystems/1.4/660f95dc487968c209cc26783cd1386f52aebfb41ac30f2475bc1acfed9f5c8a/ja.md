```
sysinv = inv(sys; atol = 0, atol1 = atol, atol2 = atol, rtol, checkinv = true)
```

記述子システム `sys = (A-λE,B,C,D)` の転送関数行列 `G(λ)` に対して、その逆システムの記述子実現 `sysinv = (Ai-λEi,Bi,Ci,Di)` を計算します。このとき、`sysinv` の転送関数行列 `Ginv(λ)` は `G(λ)` の逆である必要があります（すなわち、`G(λ)*Ginv(λ) = I`）。`sysinv` の実現は、逆計算を行わない式を用いて決定され、逆行列の条件がチェックされます。ただし、`checkinv = false` の場合は除きます。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A`、`B`、`C`、`D` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A`、`B`、`C`、`D` および `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は正方行列 `A` と `E` の次数、`ϵ` は作業用マシンイプシロンです。キーワード引数 `atol` を使用すると、`atol1 = atol` および `atol2 = atol` を同時に設定できます。
