```
sysrdiv = rdiv(sys1, sys2; atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ, checkinv = true)  
sysrdiv = sys1 / sys2
```

記述子システム `sys1 = (A1-λE1,B1,C1,D1)` の伝達関数行列 `G1(λ)` と `sys2 = (A2-λE2,B2,C2,D2)` の伝達関数行列 `G2(λ)` に対して、記述子実現 `sysrdiv = (Ai-λEi,Bi,Ci,Di)` を計算します。これは `sysrdiv = sys1*inv(sys2)` であり、その伝達関数行列 `Gri(λ)` は右除算の結果を表します。すなわち、`Gri(λ) = G1(λ)*inv(G2(λ))` です。`sysrdiv` の実現は、逆行列を使わない公式を用いて決定され、`checkinv = false` でない限り、`sys2` の可逆性条件がチェックされます。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A1`、`B1`、`C1`、`D1`、`A2`、`B2`、`C2`、`D2` の非ゼロ要素に対する絶対許容誤差、`E1` および `E2` の非ゼロ要素に対する絶対許容誤差、`A1`、`B1`、`C1`、`D1`、`A2`、`B2`、`C2`、`D2`、`E1` および `E2` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は正方行列 `A1` と `A2` の次数の最大値、`ϵ` は作業用マシンイプシロンです。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。
