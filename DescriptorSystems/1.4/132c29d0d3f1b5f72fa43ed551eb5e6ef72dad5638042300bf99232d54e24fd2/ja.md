```
 sys = dss(A, E, B, F, C, G, D, H; compacted = false, 
           atol = 0, atol1 = atol, atol2 = atol, atol3 = atol, rtol = min(atol1,atol2,atol3)>0 ? 0 : n*ϵ)
```

入力-出力同等の記述子システム表現 `sys = (Ad-λdE,Bd,Cd,Dd)` を、ペンシルベースの線形化 `(A-λE,B-λF,C-λG,D-λH)` に対して構築します。これは次の条件を満たします。

```
            -1                        -1
 Cd*(λEd-Ad)  *Bd + Dd = (C-λG)*(λE-A)  *(B-λF) + D-λH .
```

`compacted = true` の場合、行列 `F`、`G`、および `H` の可能なランク不足を利用して圧縮された記述子システムの実現が決定されます。行列 `F`、`G`、および `H` のいずれかを `missing` に設定できます。

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `F` の非ゼロ要素の絶対許容誤差、`G` の非ゼロ要素の絶対許容誤差、`H` の非ゼロ要素の絶対許容誤差、および `F`、`G`、`H` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズ、`ϵ` は `A` の要素型のマシンイプシロンです。キーワード引数 `atol` は、同時に `atol1 = atol`、`atol2 = atol`、`atol3 = atol` を設定するために使用できます。
