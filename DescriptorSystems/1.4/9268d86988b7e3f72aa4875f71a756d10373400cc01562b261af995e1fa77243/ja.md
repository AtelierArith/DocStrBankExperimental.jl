```
val = gpole(sys; fast = false, atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ)
```

記述システム `sys = (A-λE,B,C,D)` に対して、複素ベクトル `val` はシステムポールペンシル `P(λ) := A-λE` の有限および無限のゼロを含みます。 `val` の値は、`A-λE` が *正則* であり、記述システムの実現 `sys = (A-λE,B,C,D)` が *不可約* である場合、`sys` の伝達関数行列の極を示します。ペンシル `A-λE` が特異な場合、`val` には `NaN` 要素も含まれ、その数はペンシル `A-λE` のランク欠損を示します。

`E` が非特異の場合、`val` にはペア `(A,E)` の一般化固有値が含まれます。`E` が特異な場合、`val` には `P(λ)` のゼロが含まれ、これはペンシル `P(λ)` を適切なクロンカー様形式に削減することによって計算され、直交類似変換を使用し、列ピボッティングを伴うランクを明らかにするQR分解に基づくランク決定が含まれます（`fast = true` の場合）または、より信頼性の高いSVD分解（`fast = false` の場合）を使用します。

`A-λE` の正則性は暗黙的にチェックされます。`check_reg = true` の場合、ペンシル `A-λE` が特異であるとエラーメッセージが発行されます。`check_reg = false` でペンシル `A-λE` が特異な場合、`n-r` 個の極は `NaN` に設定されます。ここで、`n` はシステムの次数、`r` は `A-λE` の通常のランクです。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`A` および `E` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は作業用マシンエプシロンです。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。
