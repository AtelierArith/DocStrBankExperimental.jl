```
glinfldp(sys1, sys2, [, γ]; nehari = false, reltol = 0.0001, fast = true, offset = β, 
         atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ) -> (sysx, mindist)
```

記述子システム `sys1 = (A1-λE1,B1,C1,D1)` と `sys2 = (A2-λE2,B2,C2,D2)` に対して、伝達関数行列 $G_1(λ)$ と $G_2(λ)$ をそれぞれ持つ記述子システム `sysx` を求めます。このとき、$X(λ)$ は2ブロック `L∞` *最小距離問題* (`LDP`) の解である伝達関数行列です。

$$
      \text{mindist} := \min \|G_1(λ)-X(λ) \mid   G_2(λ) \|_\infty 
$$

`mindist` は最適解に対応する達成された最小距離です。もし `sys2 = []` の場合、1ブロック `LDP` が解かれます。`sys1` と `sys2` は安定性領域 `Cs` の境界上に極を持ってはいけません。`sys2` は安定性領域の境界上に極を持たないと仮定します（以下を参照）。得られた解は、`sys1` が安定性領域の境界上に極を持たない限り、安定です。

もし ${\small γ > \|G_2(λ)\|_\infty}$ が望ましいサブ最適度である場合、`γ`-サブ最適 `LDP` 

$$
     \text{mindist} := \|G_1(λ)-X(λ) \mid G_2(λ) \|_\infty < γ
$$

が解かれ、`mindist` は達成されたサブ最適距離です。

次の呼び出しは

```
glinfldp(sys[, m2[, γ]]; nehari = false, fast = true, offset = β, 
         atol = 0, atol1 = atol, atol2 = atol, rtol = n*ϵ) -> (sysx, mindist)
```

複合記述子システム `sys = (A-λE,[B1 B2],C,[D1 D2])` を使用し、ここで `B2` は `m2` 列を持ち、記述子システム `sys1 = (A-λE,B1,C,D1)` と `sys2 = (A-λE,B2,C,D2)` を定義します（すなわち、`A1-λE1 = A2-λE2 = A-λE` および `C1 = C2 = C`）。もし `m2 = 0` の場合、1ブロック `LDP` が解かれます。`sys2` は安定性領域 `Cs` の境界上に極を持ってはいけません。

もし `nehari = true` の場合、最適またはサブ最適なネハリ近似が `LDP` を解くために使用されます。もし `nehari = false`（デフォルト）の場合、最適解は `γ`-反復 [1] を使用して計算されます。

キーワード引数 `reltol` は、`γ`-反復の望ましい精度のための相対許容誤差を指定します。反復は、最適距離 $γ_o$ の最大 $γ_u$ と最小 $γ_l$ の現在の推定値が 

$$
     γ_u-γ_l \leq \text{reltol} * \text{gap} ,
$$

を満たすまで行われます。ここで `gap` は元のギャップ（内部で決定される）です。

安定性領域 `Cs` の境界上に極が存在するかどうかを評価するために、キーワードパラメータ `offset = β` を介して境界オフセット `β` を指定できます。したがって、連続時間設定の場合、`Cs` の境界は実部が区間 `[-β,β]` にある複素数を含み、離散時間設定の場合、`Cs` の境界は絶対値が区間 `[1-β,1+β]` にある複素数を含みます。`β` に使用されるデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業機械精度です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`A` および `E` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は作業機械イプシロンで、`n` はシステム `sys` の次数です。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。

基礎となるペンシル操作アルゴリズムにおけるランク決定は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づき、`fast = false` の場合はより信頼性の高いSVD分解に基づきます。

*メソッド:* [1] のアプローチが2ブロック最小距離問題の解決に使用されます。安定した解を持つ `LDP` の解決に必要な条件は、`sys1` と `sys2` が安定性領域 `Cs` の境界上に極を持たないことです。

*参考文献:* [1] C.-C. Chu, J. C. Doyle, and E. B. Lee     H∞最適制御理論における一般的な距離問題、     Int. J. Control, vol 44, pp. 565-596, 1986. ```
