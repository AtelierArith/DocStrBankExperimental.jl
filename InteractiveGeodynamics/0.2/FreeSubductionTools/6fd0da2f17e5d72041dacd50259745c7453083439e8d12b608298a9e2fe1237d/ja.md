subduction(;  host = HTTP.Sockets.localhost, port=8050, wait=false, width="80vw", height="45vh", cores=1)

これは無料のsubduction GUIを開始します

# オプションのパラメータ

  * `host`   : IPアドレス
  * `port`   : ポート番号
  * `wait`   : trueの場合、LaMEMの出力が表示され、シミュレーションが終了した後にのみ図のウィンドウが表示されます
  * `width`  : メイン図の相対的な幅
  * `height` : メイン図の相対的な高さ
