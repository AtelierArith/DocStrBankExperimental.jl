```
visibility(mimg, p)
```

モデル `m` の複素視認性を座標 `p` で計算します。`p` はモデルの座標に対応します。これらは `U`、`V`、および時には時間のための `Ti` と周波数のための `Fr` のプロパティを持っている必要があります。

# 注意

多数の位置で視認性を計算したい場合は、[`visibilitymap`](@ref visibilitymap) の使用を検討してください。

# 警告

これは解析モデルにのみ定義されています。非解析モデルの単一点の視認性を計算したい場合は、`visibilitymap` 関数を使用し、単一点を持つ `UnstructuredDomain` を作成してください。
