```
rf = RF_fun(f::Function, T::Real, N::Int64)
```

関数波形からサンプリングされた振幅を持つRFシーケンスを生成します。

!!! note
    この関数はこのKomaMRIバージョンでは使用されていません。


# 引数

  * `f`: (`::Function`, [`T`]) RF振幅波形のための関数
  * `T`: (`::Real`, [`s`]) RFパルスの持続時間
  * `N`: (`::Int64`) RFパルスのサンプル数

# 戻り値

  * `rf`:(`::RF`) 関数`f`によって定義された振幅を持つRF構造体
