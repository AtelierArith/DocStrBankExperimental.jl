```
y = trapz(Δt, x)
```

ファントムのすべてのスピンに対する台形積分。

!!! note
    実際には、この関数はすべてのスピンに対して (Gx * x + Gy * y + Gz * z) * Δt を積分するために使用されます。 `NΔt` は `Δt` の長さです。 `Ns` はファントムのスピンの数を表します。 `x` は行が異なるスピンを表し、列が異なる時間を表し、要素がフィールド Gx * x + Gy * y + Gz * z の値です。


# 引数

  * `Δt`: (`1 x NΔt ::Matrix{Float64}`, `[s]`) デルタ時間の1行配列
  * `x`: (`Ns x (NΔt+1) ::Matrix{Float64}`, `[T]`) フィールド Gx * x + Gy * y + Gz * z の大きさ

# 戻り値

  * `y`: (`Ns x 1 ::Matrix{Float64}`, `[T*s]`) ファントムのすべてのスピンに対する (Gx * x + Gy * y + Gz * z) * Δt の積分であるベクトル
