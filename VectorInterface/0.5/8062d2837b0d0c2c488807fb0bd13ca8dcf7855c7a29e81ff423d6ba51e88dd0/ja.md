```
add!(y, x, [α::Number = One(), β::Number = One()])
```

`y` と `x` を加算するか、より一般的には線形結合 `y * β + x * α` を構築し、その結果を `y` に格納します。互換性のないスカラー型や不適合なサイズの場合はエラーになります。

!!! note
    新しい型は四引数バージョンのみを実装するべきです。必要に応じて、二引数および三引数バージョンは `α` と `β` が型 `One` であることに特化することで区別できます。
