```
add!!(y, x, [α::Number = 1, β::Number = 1])
```

`y` と `x` を加算するか、より一般的には線形結合 `y * β + x * α` を構築し、その結果を `y` に格納しようとします。これは、不変性、互換性のないスカラー型、または不適合なサイズのために失敗した場合、新しいオブジェクトが作成されます。

!!! note
    新しい型は、4引数バージョンのみを実装するべきです。必要に応じて、2引数および3引数バージョンは、`α` と `β` が型 `One` であることに特化することで区別できます。
