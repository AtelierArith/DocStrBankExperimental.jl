```
StopAfter <: StoppingCriterion
```

完全な実行時間を見ているときに停止するしきい値を保存します。`time_ns()`を使用して時間を測定し、例えば`Minute(15)`のように時間制限として`Period`を提供します。

# フィールド

  * `threshold`は停止する`Period`を保存します
  * `start`はアルゴリズムが開始されたときの開始時間を保存し、これは`i=0`での呼び出しです。
  * `time`は経過時間を保存します
  * `at_iteration`は停止基準が満たされたイテレーション（`i=0`を含む）を示し、満たされていない間は`-1`です。

# コンストラクタ

```
StopAfter(t)
```

停止するための`Period t`に停止基準を初期化します。
