```
Polyak(; kwargs...)
Polyak(M::AbstractManifold; kwargs...)
```

ポリャックによって提案された方法に従ってステップサイズを計算します。これは、[Bertsekas:2015](@cite)のセクション3.2で議論された動的ステップサイズに関連しています。これは、リーマンの場合と最小コスト値を近似するために一般化されています。

$$
f_{\text{best}
$$

は、いくつかの反復最適化アルゴリズムの間にこれまでに見た最良のコスト値であり、$γ_k$は平方和可能だが和不可能な数の列です。

ここで計算されるステップサイズは次のようになります。

$$
s_k = \frac{f(p^{(k)}) - f_{\text{best} + γ_k}{\lVert ∂f(p^{(k)})} \rVert_{}},
$$

ここで、$∂f$は現在の反復点$p^{(k)}$における$f$の非ゼロサブグラディエントを示します。

# コンストラクタ

```
Polyak(; γ = k -> 1/k, initial_cost_estimate=0.0)
```

ポリャックのステップサイズを特定の列と$f_{	ext{best}}$の初期推定値に初期化します。
