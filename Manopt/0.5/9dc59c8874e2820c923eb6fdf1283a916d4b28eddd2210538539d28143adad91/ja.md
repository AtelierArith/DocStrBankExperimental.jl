```
DebugWhenActive <: DebugAction
```

アクティブなブール値が設定されている場合にのみデバッグを評価して印刷します。これは外部から設定でき、例えばサブソルバーのデバッグで[`DebugEvery`](@ref)によってトリガーされます。

このメソッドは自体で印刷を行わず、その子の印刷に依存します。

現在のところ、主な相互作用は[`DebugEvery`](@ref)とのもので、これがこのデバッグを有効または無効にする可能性があります。

# フィールド

  * `active`:        デバッグをオン/オフするために外部から（非）アクティブ化できるブール値
  * `always_update`: イテレーション `<=0` の非アクティブ状態でオーダーデバッグを呼び出すかどうか

# コンストラクタ

```
DebugWhenActive(d::DebugAction, active=true, always_update=true)
```
