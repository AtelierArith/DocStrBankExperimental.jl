```
AbstractManifoldObjective{E<:AbstractEvaluationType}
```

最適化関数 $f: \mathcal M → ℝ$（またはベクトル範囲）のコレクションと、その対応する要素を説明します。これらの要素は、例えば勾配や（1つ以上の）近接写像である可能性があります。

これらの要素は通常、次のような関数として実装されるべきです `(M, p) -> ...` または `(M, X, p) -> ...` すなわち

  * これらの関数の最初の引数は、それらが定義されている多様体 `M` であるべきです
  * 引数 `X` は、計算が `X` のインプレースで行われる場合に存在します（[`InplaceEvaluation`](@ref)を参照）
  * 引数 `p` は、関数 ($f$ またはその要素の1つ) が評価される場所です。

型 `T` は、グローバルな [`AbstractEvaluationType`](@ref) を示します。
