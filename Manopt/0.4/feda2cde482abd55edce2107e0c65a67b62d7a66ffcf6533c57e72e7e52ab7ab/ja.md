```
DebugWarnIfCostIncreases <: DebugAction
```

コストが増加した場合に警告を表示します。

これは、デフォルトの定数ステップサイズを持つ勾配降下法に対して追加の警告を提供します。

# コンストラクタ

```
DebugWarnIfCostIncreases(warn=:Once; tol=1e-13)
```

警告レベル（`:Once`）で警告を初期化し、`1e-13`のテストのための許容誤差を導入します。

`warn`レベルは、コストが増加したときに最初の1回だけ警告するために`:Once`に設定でき、毎回増加を報告するために`:Always`に設定でき、警告を無効にするために`:No`に設定できるため、この[`DebugAction`](@ref)は非アクティブになります。その他のシンボルはすべて`:Always`として処理されます。
