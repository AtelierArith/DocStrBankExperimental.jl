```
PolyakStepsize <: Stepsize
```

ステップサイズを計算します

$$
α_i = \frac{f(p^{(i)}) - f_{\text{best}}+γ_i}{\lVert{ ∂f(p^{(i)})} \rVert^2},
$$

ここで、$f_{\text{best}}$は現在の反復までに見られた最良のコスト値であり、$γ_i$は平方和可能だが和は収束しない数の列です（和は無限大に発散する必要があります）。さらに、$∂f$は現在の反復点$p^{(i)}$における$f$のサブグラディエントを示します。

ステップサイズは、[Bertsekas:2015](@cite)のセクション3.2で議論された動的ステップサイズからの適応であり、リーマン的な場合と最小コスト値の近似に対応しています。

# フィールド

  * `γ`               : 列を表す関数 `i -> ...`。
  * `best_cost_value` : 値 `f_{\text{best}}` を格納します。

# コンストラクタ

```
PolyakStepsize(;
    γ = i -> 1/i,
    initial_cost_estimate=0.0
)
```

Polyakステップサイズを特定の列と$f_{\text{best}}$の初期推定値に初期化します。
