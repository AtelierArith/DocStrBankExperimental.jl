```
const ELEMENTS
```

元素に関連する定数です。これは5つのキーを持つ辞書です。

  * `:MW`: 原子量。
  * `:ABUNDANCE`: 自然存在比。
  * `:ISOTOPES`: 可能な同位体。
  * `:PARENTS`: 親元素。
  * `:DECODES`: `parse_compound`のためにエンコードされた文字列をデコードします。

# 最も一般的な元素（親元素）

C, H, O, N, P, S, Li, Na, K, F, Cl, Ag 

# マイナー同位体

[13C], [2H](D), [17O], [18O], [15N], [33S], [34S], [35S], [6Li], [40K], [41K], [37Cl], [109Ag]

デフォルトでは、親元素はマイナー同位体によって置き換えられる可能性があると見なされますが、`Isotopomers`の`parent`化学においては、置き換えの数はフィールド`isotopes`によって制限されます。一方、マイナー元素は固定されています。
