```
struct JacobiSmoother{L,F,V} <: AbstractSmoother
    loops::L
    omega::F
    x_temp::V
end
```

加重ヤコビスムーザーを使用するための情報を保持する構造体。

# フィールド

  * `loops` はスムージング反復の回数を表します。
  * `omega` は緩和重みを表し、1は重み付けなしに対応します。通常、残差場の高周波数をフィルタリングするために2/3の重みが使用されます。
  * `x_temp` は内部で中間解を保存するために使用されるベクトルです。

# コンストラクタ

```
JacobiSmoother(mesh::AbstractMesh)
```

このコンストラクタは、スムーザーを作成するためにメッシュオブジェクトを入力として受け取ります。メッシュ内のセルの数は `x_temp` を割り当てるために使用されます。ループの数とスムーザー係数はそれぞれ5と1に設定されます。

```
JacobiSmoother(; domain, loops, omega=2/3)
```

`JacobiSmoother` の代替コンストラクタ。

# キーワード引数

  * `domain` は `AbstractMesh` 型のメッシュオブジェクトを表します。
  * `loops` は使用される反復の回数です。
  * `omega` は重み付け係数を表し、1はシステムを緩和せず、2/3は残差場の高周波数をスムージングするのにうまく機能することがわかっています。
