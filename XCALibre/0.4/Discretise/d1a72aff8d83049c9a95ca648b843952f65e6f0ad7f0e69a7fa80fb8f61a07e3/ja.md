```
DirichletFunction(ID, value) <: AbstractDirichlet
```

ユーザー提供の関数で定義されたディリクレ境界条件。

# 入力

  * `ID` シンボルとして提供される境界名、例: :inlet
  * `value` ディリクレ境界条件のためのカスタム関数。

# 関数の要件

この境界条件に渡される関数は、以下のシグネチャを持つ必要があります：

```
f(coords, time, index) = SVector{3}(ux, uy, uz)
```

ここで、`coords` は面の座標を含むベクトル、`time` は過渡シミュレーションにおける現在の時間（定常シミュレーションにおける反復番号）、`index` はローカル面インデックス（1から `N` まで、ここで `N` は与えられた境界の面の数）です。この関数は、速度ベクトルを表すSVector（StaticArrays.jlから）を返さなければなりません。
