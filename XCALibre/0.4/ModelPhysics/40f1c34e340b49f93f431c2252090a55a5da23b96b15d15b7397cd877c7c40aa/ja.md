```
Physics(; time, fluid, turbulence, energy, domain)::Physics{T,F,M,Tu,E,D,BI}
```

`Physics` コンストラクタはトップレベル API の一部です。これは、シミュレーションに関連する物理学とモデルを定義するために使用できます。このコンストラクタはキーワード引数を使用して、ユーザーがシミュレーションを微調整できるようにし、一部のフィールドは便利さのために背後で自動生成されます（`Momentum` と `boundary_info`）。ここで：

  * `time` - 時間モデルを指定します（定常または過渡）
  * `fluid` - 流体の種類を指定します（非圧縮性など）
  * `turbulence` - turbulence モデルを指定します
  * `energy` - エネルギー処理を指定します
  * `domain` - 使用するメッシュを提供します（ターゲットバックエンドデバイスに適応する必要があります）
