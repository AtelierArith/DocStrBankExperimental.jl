```
Session(args...; kwargs...)
```

ADCMEセッションを作成します。デフォルトでは、ADCMEは開始時にすべてのGPUリソースを使用します。必要に応じてGPUの使用量を増やしたい場合は、ADCMEを開始する前に環境変数を設定する必要があります。

```julia
ENV["TF_FORCE_GPU_ALLOW_GROWTH"] = "true"
```

# 設定

Sessionは、いくつかのランタイム最適化設定を受け入れます。

  * `intra`: 個々のオペレーション内で並列処理に使用されるスレッドの数
  * `inter`: 独立したオペレーション間で並列処理に使用されるスレッドの数
  * `CPU`: 使用する最大CPU数
  * `GPU`: 使用する最大GPUデバイス数
  * `soft`: True/有効に設定すると、オペレーションがGPUではなくCPUに配置されることを促進します

!!! note
    `CPU`は使用されるCPUの数を制限し、コアやスレッドの数を制限するものではありません。
