```
pad(o::PyObject, paddings::Array{Int64, 2}, args...; kwargs...)
```

境界の値で `o` をパディングします。

# 例

```julia
o = rand(3,3)
o = pad(o, [1 4      # 最初の次元
             2 3])   # 2番目の次元
run(sess, o)
```
