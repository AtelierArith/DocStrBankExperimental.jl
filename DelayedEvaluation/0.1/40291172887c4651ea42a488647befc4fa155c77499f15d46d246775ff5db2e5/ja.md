このモジュールは、`Base.Fix1` と `Base.Fix2` のアイデアを任意の関数に拡張し、配列インデックスのようなシンプルなインデックス構文を使用して遅延評価関数オブジェクトを作成します。このパッケージは型の略奪に依存しており、`Base.Callable` のために `Base.getindex` を定義しています。

例:

```
`sin[1.0]() == sin(1.0)`

`map[!,[1,2,3]](x->x+1) == map(x->x+1,[1,2,3])`

`map[!][[1,2,3]]() == map(x->x+1,[1,2,3])`

`sort[by=x->x[1]]([(2,:a),(1,:b)]) == sort([(2,:a),(1,:b)];by=x->x[1])`
```
