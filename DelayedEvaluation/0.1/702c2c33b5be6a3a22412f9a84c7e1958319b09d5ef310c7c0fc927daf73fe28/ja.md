`DelayEval{F,T}(f,x,kw)` は、変数 `x` とキーワード引数 `kw` を使った `f` の遅延評価のためのオブジェクトを返します。このようなオブジェクトを構築するための構文は `f[x1,x2,...,xn,k1=v1,k2=v2,...,km=vm]` であり、固定引数とキーワード引数で関数をインデックス付けすることを意味します。配列のように、`!` を使用すると、呼び出し時に追加の引数を供給するための空白が残ります。

例:

```
`sin[1.0]() == sin(1.0)`

`map[!,[1,2,3]](x->x+1) == map(x->x+1,[1,2,3])`

`map[!][[1,2,3]]() == map(x->x+1,[1,2,3])`

`sort[by=x->x[1]]([(2,:a),(1,:b)]) == sort([(2,:a),(1,:b)];by=x->x[1])`
```
