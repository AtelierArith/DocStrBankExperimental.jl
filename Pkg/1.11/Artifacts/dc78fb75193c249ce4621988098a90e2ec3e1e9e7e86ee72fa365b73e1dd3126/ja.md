```
remove_artifact(hash::SHA1; honor_overrides::Bool=false)
```

指定されたアーティファクト（SHA1 git ツリーハッシュで識別される）をディスクから削除します。アーティファクトが複数のデポにインストールされている場合、それはすべてのデポから削除されます。オーバーライドされたアーティファクトの削除が要求された場合、それは静かに無視されます。このメソッドはオーバーライドされたアーティファクトを削除しようとは決してしません。

一般的に、アーティファクトのインストールを管理するために `Pkg.gc()` を使用することをお勧めします。`remove_artifact()` を直接使用することは避けた方が良いでしょう。なぜなら、アーティファクトが他のパッケージによって使用されているかどうかを知るのが難しいからです。
