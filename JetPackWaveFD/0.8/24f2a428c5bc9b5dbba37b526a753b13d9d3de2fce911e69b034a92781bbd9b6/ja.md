```
srcillum(J)
srcillum(J, m)
srcillum!(y, J, m)
```

`Jets` 演算子 `J` のソースイルミネーションを計算して返します。ソースイルミネーションは、有限差分進化の時間にわたってモデル内のどこでも波場振幅の二乗の合計として定義されます。

`J::Jop` が `Jets` ブロック演算子または分散ブロック演算子である場合、すべてのブロックからのソースイルミネーションは単純な合計で蓄積されます。

`srcillum(J)` は、`Jet` で定義された現在の位置を使用して、`J::JopLn` からのソースイルミネーションを持つ配列を作成して返します。

`srcillum(J, m)` は、位置 `m` に対して `J::Jop` からのソースイルミネーションを持つ配列を作成して返します。

`srcillum!(y, J, m)` は、渡された配列 `y` をゼロにし、次に位置 `m` における `J::Jop` からのソースイルミネーションを `y` に蓄積します。
