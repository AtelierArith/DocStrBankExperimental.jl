```
VARIMASettings(...)
```

VARIMA仕様を管理する不変構造体を定義します。

# 引数

  * `Y_levels`: 観測された測定値 (`nxT`) - レベルで
  * `Y`: 観測された測定値 (`nxT`) - 差分を取り、平均を引いたもの
  * `μ`: サンプル平均（シリーズごと）
  * `n`: シリーズの数
  * `d`: 差分の次数
  * `p`: 自己回帰モデルの次数
  * `q`: 移動平均モデルの次数
  * `nr`: n*max(p, q+1)
  * `np`: n*p
  * `nq`: n*q
  * `nnp`: (n^2)*p
  * `nnq`: (n^2)*q
