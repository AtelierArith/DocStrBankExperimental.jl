```
bregman(A, x, b, options)
```

線形化されたブレグマン反復法は、次のシステムに対して行われます。

$$
\frac{1}{2} ||TD \ x||_2^2 + λ ||TD \ x||_{1,w}  \ \ \ s.t Ax = b
$$

例えば、スパース性を促進するデノイジング（すなわちLSRTM）の場合

# 必要な引数

  * `A`: 前方演算子（例：JまたはLSRTMのための前処理されたJ）
  * `x`: 初期推定
  * `b`: 観測データ

# オプションの引数

  * `callback` : コールバック関数。入力として`result` callback(x::result)を受け取る必要があります。

# 非必須の引数

  * `options`: ブレグマンオプション、デフォルトはbregman_options(); options.TDはスパース化変換（例：カーブレット）を提供し、options.wは加重l1のための重みベクトルを提供します。
