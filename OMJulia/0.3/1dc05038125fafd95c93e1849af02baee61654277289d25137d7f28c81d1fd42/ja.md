```
setLinearizationOptions(omc, name)
```

線形化オプションを設定します。

## 引数

  * `omc::OMCSession`:        OpenModelicaコンパイラセッション。
  * `name::Union{<:AbstractString, Array{<:AbstractString,1}}`:  文字列 "Name=value" または                                                              文字列のベクター ["Name1=value1","Name2=value2","Name3=value3"]
