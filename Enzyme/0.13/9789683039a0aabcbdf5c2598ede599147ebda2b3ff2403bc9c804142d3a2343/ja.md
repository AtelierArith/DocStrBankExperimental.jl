```
hvp(f::F, x::X, v::X) where {F, X}
```

スカラー出力関数 `f` のヘッセ行列-ベクトル積を、`x` で評価されたときのベクトル `v` との積として計算します。

言い換えれば、hessian(f)(x) * v を計算します。

結果を既存のバッファに格納するバージョンについては [`hvp!`](@ref) を、hvp と勾配の両方を単一の呼び出しで計算する関数については [`hvp_and_gradient!`](@ref) を参照してください。

例:

```jldoctest hvp; filter = r"([0-9]+\.[0-9]{8})[0-9]+" => s"\1***"
f(x) = sin(x[1] * x[2])

hvp(f, [2.0, 3.0], [5.0, 2.7])

# output
2-element Vector{Float64}:
 19.6926882637302
 16.201003759768003
```
