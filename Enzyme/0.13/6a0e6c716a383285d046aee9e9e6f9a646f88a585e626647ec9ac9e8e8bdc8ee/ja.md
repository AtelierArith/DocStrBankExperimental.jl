```
hvp!(res::X, f::F, x::X, v::X) where {F, X}
```

配列入力スカラー出力関数 `f` のインプレースヘッセ行列-ベクトル積を計算します。これは `x` で評価された関数 `f` にベクトル `v` を掛けたものです。結果は `res` に格納されます。この関数は中間勾配を格納するためのバッファを割り当ててゼロに初期化しますが、ユーザーには返されません。

言い換えれば、`res .= hessian(f)(x) * v` を計算します。

両方のhvpと勾配を1回の呼び出しで計算する関数については [`hvp_and_gradient!`](@ref) を参照してください。

例:

```jldoctest hvpip; filter = r"([0-9]+\.[0-9]{8})[0-9]+" => s"\1***"
f(x) = sin(x[1] * x[2])

res = Vector{Float64}(undef, 2)
hvp!(res, f, [2.0, 3.0], [5.0, 2.7])

res
# output
2-element Vector{Float64}:
 19.6926882637302
 16.201003759768003
```
