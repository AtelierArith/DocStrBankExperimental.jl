```
EngTicks(kind=:number; suffix="", digits=0, space=true)
```

エンジニアリング表記を使用するティックフォーマッタ：科学的表記のように、しかし指数は常に3の倍数です。例えば、`10000`は`kind=:number`のとき`10×10³`になり、`kind=:symbol`のとき`10k`になります。

## 引数

  * `kind`: 使用するエンジニアリング表記の種類。`:number`または`:symbol`にできます。
  * `suffix`: ティックラベルに追加する文字列。例えば、`suffix="m"`は「10 km」のようなラベルを生成する可能性があります。
  * `digits`: 表示する小数点以下の桁数（実験的であり、使用例がない場合は削除される可能性があります）。
  * `space`: 数字とサフィックスの間にスペースを含めるかどうか。
