```
peektimers()
```

すべてのタイマーによってカウントされた経過秒数を、停止せずに配列として返します。

## 例

```
julia> laptimer()
3         [ Info:         16.380142899s: 16秒, 380ミリ秒
2         [ Info:         21.480662472s: 21秒, 480ミリ秒
1         [ Info:         23.888862411s: 23秒, 888ミリ秒

julia> peektimers()
3-element Vector{Float64}:
 18.56985403
 23.670373603
 26.078573542
```
