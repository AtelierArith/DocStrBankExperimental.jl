`pca(A, k; [q=10], [exact_svd=false], [s=5])`

行列 `A` の高速ランダム化PCAを実行し、最初の `k` コンポーネントを返します。

スパース行列で動作するように設計されています。

# キーワード引数

  * `exact_svd` - `true` の場合、ランダム投影に対して正確なSVDを使用します。それ以外の場合は `eig_svd` の近似を使用します。これを `true` に設定すると、小さな固有値に対するアルゴリズムの精度が向上しますが、大きな行列（>1M行）ではパフォーマンスが大幅に低下します。
  * `q` は `A` に対するパスの数で、`q` は 1 より大きく、`q` 回のパスは `(q-2)/2` 回のパワーイテレーションに等しい必要があります。
  * `s` はランダム行列の余分な次元です。このパラメータは通常変更することを想定していません。

# 例

```julia-repl
using SparseArrays
pca(sprand(100, 100, 0.2), 10; q=3)
```
