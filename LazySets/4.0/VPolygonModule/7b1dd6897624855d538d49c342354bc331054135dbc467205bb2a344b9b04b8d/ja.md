# 拡張ヘルプ

```
σ(d::AbstractVector, P::VPolygon)
```

### 出力

方向のノルムがゼロの場合、最初の頂点が返されます。

### アルゴリズム

この実装は、ポリゴンの頂点が10個を超える場合にバイナリサーチアルゴリズムを使用し、10個以下の場合にはブルートフォースサーチを使用します。ブルートフォースサーチは、与えられた方向に沿った各ベクトルの投影を比較し、$O(n)$で実行されます。ここで、$n$は頂点の数です。バイナリサーチは$O(log n)$で実行され、[この実装](http://geomalgorithms.com/a14-_extreme_pts.html#polyMax_2D())に従い、[ORourke98](@citet)で説明されたアルゴリズムに基づいています。
