# 拡張ヘルプ

```
remove_redundant_generators(S::SparsePolynomialZonotope)
```

## 注意事項

結果は、`S`の配列タイプに関係なく、密な配列を使用します。

### アルゴリズム

`G`を従属生成行列、`E`を指数行列、`GI`を`S`の独立生成行列とします。次の簡略化を行います：

  * `G`のゼロ列と`E`の対応する列を削除します。
  * `GI`のゼロ列を削除します。
  * `E`のゼロ列に対して、対応する`G`の列を中心に追加します。
  * `E`の重複した列を、対応する`G`の列を合計することでまとめます。
