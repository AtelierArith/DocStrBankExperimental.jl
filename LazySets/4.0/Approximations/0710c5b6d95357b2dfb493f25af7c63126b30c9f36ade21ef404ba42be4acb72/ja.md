```
box_approximation(Z::AbstractZonotope)
```

軸に整列したボックスでゾノトープの厳密な上限近似を返します。

### 入力

  * `Z` – ゾノトープ

### 出力

ハイパーレクタングル。

### アルゴリズム

この関数は[AlthoffSB10; Section 5.1.2](@citet)の方法を実装しています。ゾノトープ $Z = ⟨c, G⟩$ は、中心が $c$ であり、次元 $i$ に沿った半径が $G$ の $i$ 行の絶対値の列和であるような軸に整列したハイパーレクタングルによって厳密に上限近似されます。ここで、$i = 1,…, p$ であり、$p$ は $Z$ の生成器の数です。 ```
