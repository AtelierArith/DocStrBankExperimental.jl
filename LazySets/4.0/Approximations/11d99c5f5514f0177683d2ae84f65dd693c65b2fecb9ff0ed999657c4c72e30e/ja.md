```
overapproximate(cap::Intersection{N, <:LazySet, <:AbstractPolyhedron},
                dirs::AbstractDirections;
                kwargs...
               ) where {N}
```

集合と多面体の交差点を、テンプレート方向のセットを考慮して過大評価します。

### 入力

  * `cap`    – 集合と多面体の交差
  * `dirs`   – テンプレート方向
  * `kwargs` – サポート関数アルゴリズムに渡される追加の引数

### 出力

各半空間の法線方向が `dirs` の要素によって与えられるH表現の多面体または多面体。

### アルゴリズム

`di` をテンプレート方向のセット `dirs` から引き出された方向とします。`X` を集合、`P` を多面体とします。与えられたテンプレート方向 `dirs` を使用して、集合 `X ∩ H` を制約表現の多面体または多面体で過大評価します。

アイデアは、集合 `P` の各半空間に対して一変数最適化問題 `ρ(di, X ∩ Hi)` を解き、最小値を取ることです。これにより、正確なサポート関数の過大評価が得られます。

このアルゴリズムは [Frehse012](@citet) からインスパイアを受けています。

### 注意事項

この方法は、多面体 `P` の `constraints_list` が利用可能であることに依存します。

この方法は、元の集合が空であっても非空の集合を返す場合があります。
