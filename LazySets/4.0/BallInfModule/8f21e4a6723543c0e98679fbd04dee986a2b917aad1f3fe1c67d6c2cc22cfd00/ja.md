# 拡張ヘルプ

```
ρ(d::AbstractVector, B::BallInf)
```

### アルゴリズム

$$
B
$$

を中心 $c$ と半径 $r$ を持つ無限ノルムのボールとし、$d$ を関心のある方向とします。次元が $30$ 未満のボールに対しては、`AbstractHyperrectangle` 用に調整された `BallInf` の実装を使用し、次のように計算します。

$$
    ∑_{i=1}^n d_i (c_i + \textrm{sgn}(d_i) · r)
$$

ここで、$\textrm{sgn}(α) = 1$ もし $α ≥ 0$ であり、$\textrm{sgn}(α) = -1$ もし $α < 0$ です。

次元が高いボールに対しては、サポートベクトル $v = σ(d, B) = c + \textrm{sgn}(d) · (r, …, r)ᵀ$ を利用します。これにより、

$$
    ρ(d, B) = ⟨d, v⟩ = ⟨d, c⟩ + ⟨d, \textrm{sgn}(d) · (r, …, r)ᵀ⟩ = ⟨d, c⟩ + r · ∑_{i=1}^n |d_i|
$$

となります。ここで、$⟨·, ·⟩$ はドット積を示します。
