# 拡張ヘルプ

```
reflect(P::LazySet)
```

### 出力

集合 `-P`。これは、`P` が多面体（すなわち、有界）である場合は `HPolytope` 型であり、そうでない場合は `HPolyhedron` 型です。

### アルゴリズム

この関数は、集合 `P` の制約のリストが利用可能であることを要求します。すなわち、$P = \{z ∈ ℝⁿ: ⋂ sᵢᵀz ≤ rᵢ, i = 1, ..., N\}$ と書くことができる必要があります。

この関数は、ミンコフスキー差の代替定義を実装するために使用できます。

$$
    A ⊖ B = \{a − b \mid a ∈ A, b ∈ B\} = A ⊕ (-B)
$$

`minkowski_sum(A, reflect(B))` を呼び出すことで実行できます。
