# 拡張ヘルプ

```
vertices_list(Z::AbstractZonotope; [apply_convex_hull]::Bool=true)
```

### 入力

  * `apply_convex_hull` – （オプション、デフォルト: `true`）`true`の場合、点の凸包で計算を後処理します。

### アルゴリズム

#### 二次元の場合

すべての生成子が第一象限または第三象限（同符号）にある2次元ゾノトープ集合の頂点を列挙する速度を上げるためのトリックを使用します。具体的には、生成子を角度でソートし、増加順に時計回りで追加し、減少順に反時計回りで追加します。より詳細な説明は[こちら](https://math.stackexchange.com/q/3356460)で見つけることができます。

両方向からの累積和を別々に避けるために、生成子を1つの行列-ベクトル積で両方向に合計するための2Dインデックス行列を構築します。

#### 一般的な場合

ゾノトープ集合に$p$個の生成子がある場合、各頂点は中心に生成子の線形結合を加算した結果です。このとき、結合係数は$ξ_i ∈ \{-1, 1\}$です。

最大で$2^p$の異なる頂点があります。フラグ`apply_convex_hull`を使用して、この方法で計算された頂点に凸包アルゴリズムが適用されるかどうかを制御します。そうしないと、冗長な頂点が存在する可能性があります。
