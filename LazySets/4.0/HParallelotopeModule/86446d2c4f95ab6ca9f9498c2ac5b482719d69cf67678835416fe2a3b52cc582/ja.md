# 拡張ヘルプ

```
rand(::Type{HParallelotope}; [N]::Type{<:Real}=Float64, [dim]::Int=2,
     [rng]::AbstractRNG=GLOBAL_RNG, [seed]::Union{Int, Nothing}=nothing)
```

### アルゴリズム

すべての数値は平均0、標準偏差1の正規分布に従います。

方向行列とオフセットベクトルはランダムに生成されます。平均して、この結果得られるセットが空である可能性が高いです。その後、オフセットを修正して非空であることを保証します。

結果として得られるセットが無限大のセットを表す可能性があります。この実装はその場合をチェックし、新しいセットをサンプリングします。
