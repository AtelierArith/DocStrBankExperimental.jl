```
constraints_list(rm::ResetMap)
```

ポリヘドロンリセットマップの制約のリストを返します。

### 入力

  * `rm` – ポリヘドロンのリセットマップ

### 出力

リセットマップの制約のリスト。

### 注意事項

基になる集合 `rm.X` がポリヘドロンであると仮定します。すなわち、`constraints_list(X)` メソッドを提供します。

### アルゴリズム

集合 `rm.X` がハイパーレクタングラーである場合、すべての次元を通じて反復します。各リセットに対して対応する（平坦な）制約を構築し、他の次元では基になる集合の対応する制約を構築します。

より一般的な集合の場合、リセットマップのアフィンマップビューにおける `A`-行列の `LinearMap` の `constraints_list` にフォールバックします。各リセット次元 $i$ はゼロに投影され、各リセット次元に対して2つの制約で表現されます。次に、これらの制約を新しい値にシフトする必要があります。

例えば、次元 $5$ が $4$ にリセットされた場合、制約 $x₅ ≤ 0$ と $-x₅ ≤ 0$ が存在します。次に、これらの制約の右辺をそれぞれ $x₅ ≤ 4$ と $-x₅ ≤ -4$ に修正します。
