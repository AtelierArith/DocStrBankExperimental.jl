# 拡張ヘルプ

```
isbounded(am::AbstractAffineMap; cond_tol::Number=DEFAULT_COND_TOL)
```

### 入力

  * `cond_tol` – （オプション）行列の条件の許容誤差（行列が可逆であるかどうかを確認するために使用）

### アルゴリズム

まず、行列がゼロであるか、ラップされた集合が有界であるかを確認します。そうでない場合、行列が可逆であるかどうかを十分に確認します。行列が可逆であれば、マップが有界であることはラップされた集合が有界であることと同等であり、したがってマップは無限大です。そうでなければ、[`_isbounded_unit_dimensions`](@ref)を介して有界性を確認します。
