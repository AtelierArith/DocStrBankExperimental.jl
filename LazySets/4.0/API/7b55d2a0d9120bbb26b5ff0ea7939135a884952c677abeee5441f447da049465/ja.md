```
distance(x::AbstractVector, X::LazySet; [p]::Real=2)
distance(X::LazySet, x::AbstractVector; [p]::Real=2)
```

点と集合の間の標準距離（$p$-ノルムによって誘導される）を計算します。

### 入力

  * `x` – 点/ベクトル
  * `X` – 集合
  * `p` – （オプション; デフォルト: `2`）$p$-ノルムの値

### 出力

`x` と `X` の間の距離を表す実数。

### 注意事項

標準距離は、点が集合の内部にある場合はゼロになり、集合が空である場合は無限大になります。それ以外の場合、点と集合内の他の任意の点との間の最短線分の$p$-ノルムです。形式的には、

$$
    \inf_{y ∈ X} \{ d(x, y) \}.
$$
