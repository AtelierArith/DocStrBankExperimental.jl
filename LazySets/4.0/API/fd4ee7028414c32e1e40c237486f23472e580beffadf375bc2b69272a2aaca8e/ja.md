```
distance(X::LazySet, Y::LazySet; [p]::Real=2)
```

2ノルムによって誘導される2つの集合間の標準距離を計算します。

### 入力

  * `X` – 集合
  * `Y` – 集合
  * `p` – （オプション; デフォルト: `2`）$p$-ノルムの値

### 出力

`X`と`Y`の間の距離を表す実数。

### 注意事項

標準距離は、集合が交差する場合はゼロになり、いずれかの集合が空である場合は無限大になります。それ以外の場合、任意の点のペア間の最短線分の$p$-ノルムです。形式的には、

$$
    \inf_{x ∈ X, y ∈ Y} \{ d(x, y) \}.
$$
