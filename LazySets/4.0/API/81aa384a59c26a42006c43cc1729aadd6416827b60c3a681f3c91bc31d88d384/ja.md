```
volume(X::LazySet)
```

集合の体積、またはルベーグ測度を計算します。

### 入力

  * `X` – 集合

### 出力

`X` のルベーグ測度を表す実数。

### 注意事項

[ルベーグ測度](https://en.wikipedia.org/wiki/Lebesgue_measure)には以下の一般的な特別なケースがあります：

  * 1次元では、*長さ*と一致します。
  * 2次元では、*面積*と一致します（[`area`](@ref)も参照）。
  * 3次元では、*体積*と一致します。

高次元では、*ハイパーボリューム*または単に*体積*とも呼ばれます。
