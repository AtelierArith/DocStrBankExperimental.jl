```
is_interior_point(v::AbstractVector{<:Real}, X::LazySet; kwargs...) end
```

点が集合の内部に含まれているかどうかを確認します。

### 入力

  * `v`  – 点/ベクトル
  * `X`  – 集合
  * `p`  – （オプション; デフォルト: `Inf`）誤差許容範囲を適用するために使用される球のノルム
  * `ε`  – （オプション; デフォルト: `_rtol(eltype(X))`）チェックの誤差許容範囲

### 出力

点 `v` が誤差許容範囲 `ε` で `X` に厳密に含まれている場合は `true`。
