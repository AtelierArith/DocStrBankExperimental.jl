```
linear_combination(X::LazySet, Y::LazySet)
```

2つの集合の線形結合を計算します。

### 入力

  * `X` – 集合
  * `Y` – 集合

### 出力

`X` と `Y` の線形結合を表す集合。

### 注意事項

2つの集合 $X$ と $Y$ の線形結合は次のように定義されます。

$$
    \left\{\frac{1}{2}(1+λ)x + \frac{1}{2}(1-λ)y \mid x ∈ X, y ∈ Y, λ ∈ [-1, 1]\right\}.
$$

$$
X
$$

と $Y$ が凸である場合、その線形結合はそれらの和集合 $X ∪ Y$ の凸包と同一です。
