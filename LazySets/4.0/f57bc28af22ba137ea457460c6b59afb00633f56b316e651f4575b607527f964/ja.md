# 拡張ヘルプ

```
area(X::LazySet)
```

### 注意事項

このアルゴリズムは、`vertices_list`を介して頂点のリストを計算できる任意の多角形集合`X`に適用可能です。

### アルゴリズム

`m`を`X`の頂点の数とします。以下のインスタンスを考えます：

  * `m = 0, 1, 2`：出力はゼロです。
  * `m = 3`：三角形の場合は、3点を用いてシューレースの公式で解決されます。
  * `m = 4`：四角形の場合は、4点を用いてシューレースの公式の因子版で解決されます。

それ以外の場合は、一般的なシューレースの公式が使用されます。詳細については、[Wikipediaのページ](https://en.wikipedia.org/wiki/Shoelace_formula)を参照してください。
