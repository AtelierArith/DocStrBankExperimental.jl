# 拡張ヘルプ

```
constraints_list(Z::AbstractZonotope{<:AbstractFloat})
```

### 注意事項

主なアルゴリズムは、生成行列がフルランクであることを前提としています。この前提の下では、結果は $2 \binom{p}{n-1}$ （ここで $p$ は生成器の数、$n$ は周囲の次元）制約を持ち、これは最適です。この前提が与えられない場合、実装は回避策を試みます。

### アルゴリズム

私たちは [AlthoffSB10](@citet) に示されたアルゴリズムに従います。そのアルゴリズムではカバーされていない3つのケースがあるため、それらを別々に処理します。最初のケースは1次元のゾノトープです。2番目のケースは、生成器の数が次元よりも少ない場合、$p < n$、または生成行列がフルランクでない場合であり、この場合は（遅い）頂点表現に基づく計算に戻ります。3番目のケースは、ゾノトープがいくつかの次元で平坦である場合であり、この場合はゾノトープを非平坦な次元に投影し、後で結果を拡張します。
