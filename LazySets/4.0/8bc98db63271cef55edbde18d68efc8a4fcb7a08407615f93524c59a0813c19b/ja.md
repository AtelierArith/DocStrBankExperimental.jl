# 拡張ヘルプ

```
vertices_list(am::AbstractAffineMap; [apply_convex_hull]::Bool=true)
```

### 入力

  * `apply_convex_hull` – （オプション、デフォルト: `true`）`true`の場合、アフィンマップによって変換された頂点のリストに対して凸包操作を適用します。

### アルゴリズム

この実装は、まず `X` のすべての頂点を計算し、それらをアフィンマップを通じて変換します。すなわち、各頂点 `x` に対して `x ↦ M*x + v` です。デフォルトでは、このリストを返す前に凸包操作が行われます。次元が3以上の場合、この操作は具体的な多面体ライブラリ `Polyhedra.jl` の機能に依存します。

アフィンマップの下で得られた頂点の凸包を取ることに興味がない場合は、キーワード引数として `apply_convex_hull=false` を渡してください。

基礎となる集合 `X` が具体的または遅延的に多面体であると仮定しています。すなわち、関数 `vertices_list` は適用可能であるべきです。
