# 拡張ヘルプ

```
σ(d::AbstractVector, P::HPolygon;
  [linear_search]::Bool=(length(P.constraints) < 10))
```

### 入力

  * `linear_search` – （オプション、デフォルト：以下を参照）線形探索を行うか二分探索を行うかを制御するフラグ

### 出力

結果は常に頂点の1つであり、特に方向のノルムがゼロの場合は、任意の頂点が返されます。

### アルゴリズム

方向の比較は極角を使用して行われます。2次元ベクトルのための `⪯` の定義を参照してください。

制約が10以上の多角形の場合、デフォルトで二分探索を使用します。
