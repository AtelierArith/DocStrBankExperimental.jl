```
tosimplehrep(constraints::AbstractVector{<:HalfSpace}; [n]::Int=0)
```

線形制約のリストから単純なH-表現 $Ax ≤ b$ を返します。

### 入力

  * `constraints` – 線形制約のリスト
  * `n`           – （オプション; デフォルト: `0`）制約の次元

### 出力

正規方向の行列 $A$ とオフセットのベクトル $b$ からなるタプル `(A, b)`。

### 注意

パラメータ `n` は、制約がないが非ゼロ次元の行列を作成するために使用できます。もし `n` ≤ 0 の場合、このメソッドは最初の制約の次元を取ります。
