```
VarξClustering( clusteron :: Symbol )
```

VarξClustering型の変数を作成します。これは、標準誤差がクラスタリングの仮定の下で計算されるべきであることを示すために使用されます。この選択は効率に影響を与えません。また、解オブジェクトの一部として行列V(ξ)の推定値を生成し、これを可能な第二段階への入力として使用できます。引数はクラスタリングすべき変数であり、例えば*VarξClustering( :market )*は、Grumpsが製品スプレッドシートの列見出し*market*を含む列の変数に基づいてクラスタリングすべきであることを示唆しています。
