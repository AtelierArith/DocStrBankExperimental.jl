```
Variables( ; 
  market              :: Symbol = :market,
  product             :: Symbol = :product,
  choice              :: Symbol = :choice,
  interactions        :: Mat{Symbol} = [],
  randomcoefficients  :: Vec{Symbol} = [],
  outsidegood         :: String = "outsidegood",
  share               :: Symbol = :share,
  marketsize          :: Symbol = :N,
  regressors          :: Vec{Symbol} = [],
  instruments         :: Vec{Symbol} = [],
  dummies             :: Vec{Symbol} = [],
  nuisancedummy       :: Symbol = :none,
  microinstruments    :: Mat{Symbol} = [],
  user                :: Mat{Symbol} = []
    )
```

このメソッドは、[`Sources()`](@ref)で指定したソースから回帰変数、インストゥルメント、ランダム係数、相互作用、変数ラベルなどを指定するために使用されます。これは*GrumpsVariables*型のオブジェクトを作成します。

例えば、*market*オプションは市場記述子（名前）を含む列の列見出しを指定します。他のすべての引数についても同様ですが、*outsidegood*は製品が外部財であることを示すスプレッドシートのエントリを説明します。外部財の同じラベルは、すべてのスプレッドシートとすべての市場で使用する必要があります。外部財のエントリは、消費者のマイクロデータでのみ使用されるべきであり、実際に外部財を選択する消費者がマイクロデータに存在する場合に限ります。すべての記述子は大文字とスペースに敏感です。

個々の消費者の効用に入る変数と「平均効用」にのみ入る変数の間には区別があります。例えば、*interactions*はGrumpsにどの相互作用項を使用するかを指示し、*randomcoefficients*はどの製品レベルの回帰変数がランダム係数を持つかを指示します。それに対して、*regressors*は平均効用成分に入るものであり、「第二段階」（βが回収される場所）の回帰変数です。定数を使用することを示すために特別なシンボル*:constant*を使用することができます; スプレッドシートにはその見出しの列を含める必要はありません。

ダミー変数を第二段階の回帰変数として入力する方法は3つあり、ブランド、企業、または市場効果をδに組み込むのに役立ちます。最初の方法は*regressors*を介してであり、この場合、正しい数値を持っていることをユーザーが確認する責任があります。第二の可能性は*dummies*引数を介してです。*dummies*を介して渡された各シンボルに対して、*nuisancedummy*引数を介して1つを渡すことができます。これにより、計算時間とメモリの両方が節約されます。外部ダミーに変換できるカテゴリ変数は最大1つのみですが、カテゴリは任意の数を持つことができます。これらのダミーと外部ダミーは自動的に外生的であると見なされ、インストゥルメントにも含まれます。

*market*は、すべての入力データセットにおける市場指標を含む変数を指します。文字列（例："Amarillo, Texas"）は市場指標自体に最適ですが、必須ではありません。

*product*は、製品データセットにおける製品指標を含む変数を指します。文字列（例："Camry"）は製品指標自体に最適ですが、必須ではありません。

*choice*は、消費者レベルのデータセットにおける選択指標を示す変数を指します。文字列は選択指標自体に最適ですが、必須ではありません。これらは製品データセットの*product*列または*outsidegood*の値を取るべきです。

*interactions*は、消費者と製品の変数の相互作用を示す変数を指します（各行には消費者変数、製品変数が含まれます）。

*randomcoefficients*は、ランダム係数を持つ製品レベルの変数を指します。

*outsidegood*は、外部財に使用されるラベルを指します。

*share*は、製品レベルのシェアに使用されるラベルを指します。これらは外部財を含む分母を持つシェアです。

*marketsize*は、市場のサイズ（人数）を指します。

*regressors*は、第二段階の回帰変数に使用されるラベルを指します。

*instruments*は、第二段階のインストゥルメントに使用されるラベルを指します。

*dummies*は、第二段階のダミー回帰変数およびインストゥルメントに変換される離散変数を指します。

*nuisancedummy*は、係数値が関心のない第二段階のダミー回帰変数およびインストゥルメントに変換される最大1つの変数を指します。

*microinstruments*は、gmmスタイルの手続きにのみ関連するマイクロインストゥルメントを指します。

*user*は、ユーザー指定の手順を使用して消費者-製品相互作用に追加される変数のリストを指します。これは、Grumpsの仕様自体が不十分な場合にのみ必要です: 詳細については[Extending Grumps](@ref)を参照してください。
