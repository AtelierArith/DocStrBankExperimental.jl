```
DefaultMacroIntegrator( T )
```

基本的なモンテカルロ積分器を10,000回のサンプリングを使用して作成します。これは推奨される数よりも少ないため、他の方法を使用して任意の数を設定してください。型Tは省略可能で、その場合はFloat64になります。オプションの*options*引数は、デフォルトからの2つの可能な変更を示すために使用できます。すなわち、*:randomize*はランダム化を要求するために使用でき、*:replacement*は置換を伴うランダム化を示します。*options*は何もないか、シンボルのベクターであることに注意してください。両方のデフォルトはfalseです。*options*のさらに別の使い方は、重みを含む列見出しを指定することです。このシンボルは、サンプリングスプレッドシート内の希望する列見出しに対応する必要があります。
