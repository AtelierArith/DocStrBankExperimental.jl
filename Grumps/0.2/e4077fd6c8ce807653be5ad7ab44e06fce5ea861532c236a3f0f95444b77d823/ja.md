```
Save( fn, sol :: GrumpsSolution; keywords... )
```

`Save`の形式は、事前に指定されたmimeタイプと同じですが、mimeタイプはファイル拡張子から推測されます。キーワードも同様の意味を持ちます。すなわち...

以下に説明されているいくつかのキーワードがあり、その中にはいくつかのmimeタイプでは無視されるものもあります。許可されるmimeタイプは`text/plain`、`text/csv`、および`text/tex`です。

| キーワード              | 説明         | デフォルト   |
|:------------------ |:---------- |:------- |
| `colsep`           | 列の区切り文字    | `","`   |
| `adorned`          | 出力を美しくする？  | `true`  |
| `printθ`           | θの結果を印刷する？ | `true`  |
| `printβ`           | βの結果を印刷する？ | `true`  |
| `printδ`           | δの結果を印刷する？ | `false` |
| `printconvergence` | 収束統計を表示する？ | `true`  |
