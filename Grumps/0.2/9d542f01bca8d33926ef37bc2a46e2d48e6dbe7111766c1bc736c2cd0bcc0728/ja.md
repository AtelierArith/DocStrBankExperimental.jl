```
Estimator( s :: String )
```

GrumpsEstimator型を作成して返します。Grumpsはあなたが何を望んでいるのかをうまく理解するので、例えば*Estimator( "maximum likelihood" )*は、ペナルティなしのGrumps最大尤度推定量を返します。

現在プログラムされている推定量には以下が含まれます：

  * 完全なCLER推定量
  * 同じ限界分布を持つCLERの安価な代替
  * MDLE、すなわち積のレベルモーメントなしのCLER
  * シェア制約を持つ混合ロジット
  * マイクロデータのみを使用した混合ロジット推定量
  * 同じモデルのGMM推定量（進行中：推奨されません）
