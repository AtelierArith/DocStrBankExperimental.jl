```
kaiser_normalize(Λ)
```

ロード行列 `Λ` のカイザー正規化を実行します。

正規化されたロード行列と関連する重みのタプルを返します。

## 例

```jldoctest
julia> L = [
           0.830 -0.396
           0.818 -0.469
           0.777 -0.470
           0.798 -0.401
           0.786  0.500
           0.672  0.458
           0.594  0.444
           0.647  0.333
       ];

julia> L_norm, weights = kaiser_normalize(L);
```
