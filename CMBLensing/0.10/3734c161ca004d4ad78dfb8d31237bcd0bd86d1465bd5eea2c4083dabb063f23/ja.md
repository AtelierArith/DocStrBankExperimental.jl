```
ud_grade(f::Field, θnew, mode=:map, deconv_pixwin=true, anti_aliasing=true)
```

フィールド `f` を新しい解像度 `θnew` にアップグレードまたはダウングレードします（整数ステップのみ）。`mode` 引数で指定された2つのモードが利用可能です：

  * `:map`     — マップ空間でピクセルを複製/平均化することによるアップ/ダウングレード
  * `:fourier` — フーリエグリッドを拡張/切り詰めることによるアップ/ダウングレード

`:map` モードでは、2つの追加オプションが可能です。`deconv_pixwin` が true の場合、ダウングレードされたマップからピクセルウィンドウ関数を逆畳み込みし、新旧マップのスペクトルが同じになるようにします。`anti_aliasing` が true の場合、ダウンサンプリング前にナイキストを超える周波数をフィルタリングします。
