δᵥの計算

この関数は、与えられたパラメータに基づいてδᵥの値を計算します。

ケース

3. ラウンド ≡ 計算 ∩ キュービット ∈ 入力セット  → δᵥ = ϕᵥ + (θᵥ + xᵥπ) + rᵥπ

引数:

  * `::ComputationRound`: 計算ラウンド。
  * `::InputQubits`: 入力キュービット。
  * `ϕ`: ϕの値。
  * `Sx`: Sxの値。
  * `Sz`: Szの値。
  * `θᵥ`: θᵥの値。
  * `rᵥ`: rᵥの値。
  * `xᵥ`: xᵥの値。

戻り値:

  * `δᵥ`: 計算されたδᵥの値。

# 例

```julia
julia> compute_angle_δᵥ(ComputationRound(),InputQubits(),0,0,[0,0,0],0,0,0)
0
```
