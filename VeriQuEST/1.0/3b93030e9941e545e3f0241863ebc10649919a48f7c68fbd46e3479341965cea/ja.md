```
assert_flow(flow_type, resource, vertex)
```

リソースグラフにおけるフローの有効性を確認します。

引数:

  * `flow_type`: チェックするフローのタイプ。
  * `resource`: グラフを表すリソース状態。
  * `vertex`: チェックする頂点。

この関数はフローを検証するためにいくつかのアサーションを実行します：

  * 指定された頂点のフローがグラフの頂点集合に含まれているかをチェックします。
  * 指定された頂点が頂点のフローの近傍に含まれているかをチェックします。
  * 指定された頂点が頂点のフロー以下であるかをチェックします。
  * 頂点のフローの近傍にあるすべての頂点が指定された頂点以上であるかをチェックします。

# 例

```julia
assert_flow(ForwardFlow(), resource, vertex) # フォワードフローの有効性をチェックします
```
