```
MBQCFlow(forward_flow, backward_flow)
```

MBQCにおけるフローを表す構造体。

# フローの定義

  * `forward_flow`, `f`: Oᶜ → Iᶜ はマッピング `v ↦ f(v)` であり、その逆は `f⁻¹(v) ↦ v` であり、部分順序 "≤" を持つ。部分順序は現在を未来に、または現在を過去にマッピングすると言われる。
  * (a) `v ∼ f(v)`、ここで "∼" は近傍 `N(f(v))` を定義し、`v` は集合のメンバーシップを持つ。
  * (b) `v ≤ f(v)`
  * (c) `w ∼ f(v)` の場合、∀ `v`、`v ≤ w`

## 例

  * 一次元格子、"パスグラフ"、ここで頂点は "()" で表され、辺は "–-" で表される。
  * `i` を各頂点のインデックスとし、`i = {1, 2, 3, 4}` とする。
  * `p := (1)---(2)---(3)---(4)`
  * `f(i) := i + 1`
  * `f⁻¹(i) := i - 1`
  * `f([1, 2, 3]) = [2, 3, 4]`、4には未来がないため、4 + 1の答えはない。
  * `f⁻¹([2, 3, 4]) = [1, 2, 3]`、1には過去がないため、1 - 1の答えはない。

# パラメータ

  * `forward_flow`: 入力頂点を受け取り、フローの定義が成り立つように出力頂点を返すマッピング。前方フローファンクションは、頂点インデックスを入力として受け取り、新しい頂点インデックスを出力する任意のコンテナである。
  * `backward_flow`: 出力頂点を受け取り、入力頂点を返すマッピング。後方フローファンクションは前方フローの逆をマッピングする。

# 例

```julia
# 前方および後方フローファンクションを定義
forward_flow(io) = io[2]
backward_flow(io) = io[1]

# MBQCFlowを作成
mbqc_flow = MBQCFlow(forward_flow, backward_flow)
```
