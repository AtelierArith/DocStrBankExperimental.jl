```
admissible_subspace([Partition{UInt16},] C, A, b[; verbose, atol])
```

半正定問題の最適な許容分割部分空間を返します。

> ```
>   minimize: ⟨Cᵀ, x⟩
> subject to: A·x = b
>             Mat(x) ⪰ 0
> ```


ここで `C` と `A` は対称です。

問題は、その最適値を変えずに部分空間に制限できます。

分割は、**ランダム**な平方で飽和するジョルダン還元アルゴリズムによって見つかります。確率1で返される部分空間はジョルダン代数です（すなわち、線形結合と平方に対して閉じています）。Permenter論文のセクション5.2を参照してください。

# 出力

  * `Partition`部分空間 `P`。
