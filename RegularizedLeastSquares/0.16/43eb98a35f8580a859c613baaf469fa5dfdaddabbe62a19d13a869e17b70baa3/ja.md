```
power_iterations(AᴴA; rtol=1e-3, maxiter=30, verbose=false)
```

正規演算子または正方行列の最大固有値を決定するためのパワーイテレーション。抽象配列や`AbstractLinearOperator`でないカスタムのAᴴAの場合、計算中に使用される`size(AᴴA, 2)`のベクトル`b`を渡すことができます。

# 引数

  * `AᴴA`                 - 演算子または行列; 正方行列である必要があります
  * b                     - （オプション）、計算中に使用されるベクトル

# キーワード引数

  * `rtol=1e-3`           - 相対許容誤差; 最大固有値の変化がこの値より小さい場合、関数は終了します
  * `maxiter=30`          - 最大パワーイテレーション数
  * `verbose=false`       - `true`の場合、最大固有値を出力します

# 出力

演算子の最大固有値 ```
