```
itsample([rng], iter, [wv]; kwargs...)
```

イテレータのランダムな要素を返します。オプションで `rng`（デフォルトは `Random.default_rng()`）と `wv` 関数を指定できます。イテレータが空の場合、`nothing` を返します。

---

```
itsample([rng], iter, [wv], n::Int; replace = false, ordered = false, kwargs...)
```

イテレータのランダムな要素のベクトルを `n` 個返します。オプションで `rng`（デフォルトは `Random.default_rng()`）を指定できます。

`replace` は、置換ありでサンプリングを行うかどうかを決定します。`ordered` は、順序付きサンプル（イテレータ内のアイテムが同じ順序で表示されるサンプルとも呼ばれる）を取得するかどうかを決定します。

イテレータの要素が `n` 個未満の場合、置換なしでサンプリングする場合は、それらの要素のベクトルを返します。
