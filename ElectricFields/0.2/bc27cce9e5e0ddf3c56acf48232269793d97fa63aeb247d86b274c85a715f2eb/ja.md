```
@field(name) do ... end
```

電場を作成し、それを変数 `name` に格納するためのフロントエンドマクロ。

# 例

```julia
julia> @field(F) do
           I₀ = 2.0
           T = 2.0
           σ = 3.0
           Tmax = 3.0
       end
線形偏光場で
  - I₀ = 2.0000e+00 au = 7.0188904e16 W cm⁻² =>
    - E₀ = 1.4142e+00 au = 727.2178 GV m⁻¹
    - A₀ = 0.4502 au
  – 固定キャリア @ λ = 14.5033 nm (T = 48.3777 as, ω = 3.1416 Ha = 85.4871 eV)
  – および170.8811 asの持続時間のガウス包絡線 (強度FWHM; ±2.00σ)
```
