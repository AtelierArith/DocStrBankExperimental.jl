```
Crystal(material, d, R[, ∂k∂ω₀ = 0])
```

は、異なる屈折率を持つ一般的な結晶の厚さ `d` と回転 `R` によって与えられる方向を通じての分散を記述します。これらの屈折率は、結晶の異なる軸に沿った `material` の成分によって与えられます。

`material` が2つの成分を持つ特別な場合、我々は*単軸*結晶を持ち、これは二重屈折性を持ち、`material[1]` は*通常*の屈折率と呼ばれ、`material[2]` は*異常*の屈折率と呼ばれます。

オプションの `∂k∂ω₀` は、分散関係から線形傾きを引くために使用でき、すなわち時間シフトを取り除くことができ、中央角周波数 `ω₀` を持つパルスが基準系の中で中心に留まるようにします。
