```
srcwa_reftra(ψin,m,grd,λ)
srcwa_reftra(ψin,εsup,εsub,S,grd,λ)
```

S行列法に従って反射と透過を計算します

# 引数

  * `ψin` :  入射（ソース）振幅ベクトル
  * `m` :  RCWAモデルオブジェクト
  * `grd` : RCWAグリッドオブジェクト
  * `λ` : 自由空間波長
  * `S` :  デバイスの全散乱行列（指定がない場合はmから計算されます）
  * `εsup` :  スーパー ストレート材料オブジェクト（指定がない場合はmから取得されます）
  * `εsub` :  基板材料オブジェクト（指定がない場合はmから取得されます）

# 出力

  * `R` : デバイスによる反射
  * `T` : デバイスによる透過
