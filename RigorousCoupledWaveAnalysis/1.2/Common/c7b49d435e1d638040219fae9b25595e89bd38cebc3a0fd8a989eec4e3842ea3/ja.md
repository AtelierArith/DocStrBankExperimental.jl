```
getfields(ain,bout,em::Eigenmodes,grd::RCWAGrid,xypoints,zpoints,λ,window,padding)
```

層内の電場と磁場を計算します

# 引数

  * `ain` : 前の層から層に入る振幅ベクトル
  * `bout` : 前の層に向かって層を出る振幅ベクトル
  * `em` : 層の固有モード
  * `grd` : 逆空間グリッドオブジェクト
  * `xypoints` : フィールドを計算するためのx、yのポイント数を指定する2要素ベクトル、より良い偶数、そうでない場合は偶数に切り捨てられます
  * `zpoints` : 層の上部インターフェースに対する希望するz軸ポイントの配列
  * `λ` : 波長
  * `window` : ギブス現象に対して使用するウィンドウのタイプ、利用可能: Hann（デフォルト）、None
  * `padding` : xおよびyのウィンドウのパディング、デフォルトは[0,0]、まだ実装されていません

# 出力

  * `efield` : 電場のための4Dテンソル（次元はx、y、z、および成分（E*xまたはE*yまたはE_z）
  * `hfield` : 磁場のための4Dテンソル（次元はx、y、z、および成分（E*xまたはE*yまたはE_z）
