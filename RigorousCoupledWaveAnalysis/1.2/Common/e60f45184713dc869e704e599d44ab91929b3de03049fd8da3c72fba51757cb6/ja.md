```
rcwagrid(Nx,Ny,px,py,θ,α,λ,sup,use_cuda=false)
```

RCWAシミュレーションのための逆空間グリッドを作成します

# 引数

  * `nx` : x方向の最大次数
  * `ny` : y方向の最大次数
  * `px` : x方向の構造ピッチ
  * `py` : y方向の構造ピッチ
  * `θ` : 入射波の傾斜角
  * `α` : 入射波の方位角
  * `λ` : 波長
  * `sup` : 上層材料
  * `use_cuda` : オプション、CUDA GPUソルバーに切り替え

# 出力

  * `grd`: RCWAグリッド構造体
