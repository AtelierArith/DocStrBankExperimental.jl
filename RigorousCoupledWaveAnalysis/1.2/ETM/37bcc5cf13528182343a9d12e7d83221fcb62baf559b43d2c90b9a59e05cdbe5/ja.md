```
etm_reftra(ψin,m,grd,λ)
etm_reftra(ψin,m,grd,λ,em,sup,sub)
```

ETM法に従って反射と透過を計算します

# 引数

  * `ψin` :  入射（ソース）振幅ベクトル
  * `m` :  RCWAモデルオブジェクト
  * `grd` : RCWAグリッドオブジェクト
  * `λ` : 自由空間の波長
  * `em` :  すべての層の固有モードを含む配列（指定されていない場合はmから計算されます）
  * `sup` :  スーパーストレート半空間オブジェクト（指定されていない場合はmから計算されます）
  * `sub` :  基板半空間オブジェクト（指定されていない場合はmから計算されます）

# 出力

  * `R` : デバイスによる反射
  * `T` : デバイスによる透過
