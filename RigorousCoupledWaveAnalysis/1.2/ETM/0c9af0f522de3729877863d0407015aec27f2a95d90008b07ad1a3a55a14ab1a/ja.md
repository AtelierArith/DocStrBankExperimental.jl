```
etm_amplitudes(ψin,m,grd,λ)
etm_amplitudes(ψin,m,grd,λ,em,sup,sub)
```

ETM法を通じて前方および後方に伝播する波の振幅ベクトルを計算します。

# 引数

  * `ψin` :  入射（ソース）振幅ベクトル
  * `m` :  RCWAモデルオブジェクト
  * `grd` : RCWAグリッドオブジェクト
  * `λ` : 自由空間の波長
  * `em` :  すべての層の固有モードを含む配列（指定されていない場合はmから計算されます）
  * `sup` :  スーパーストレート半空間オブジェクト（指定されていない場合はmから計算されます）
  * `sub` :  基板半空間オブジェクト（指定されていない場合はmから計算されます）

# 出力

  * `a` : 前方波の振幅ベクトル
  * `b` : 後方波の振幅ベクトル
