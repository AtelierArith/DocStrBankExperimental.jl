```
etm_propagate(sup,sub,em,ψin,grd)
etm_propagate(sup,sub,em,ψin,grd,get_r)
```

ETM法に従った波の伝播

# 引数

  * `sup` :  上層半空間オブジェクト
  * `sub` :  下層半空間オブジェクト
  * `em` :  すべての層の固有モードを含む配列
  * `ψin` :  入射（ソース）振幅ベクトル
  * `grd` : RCWAグリッドオブジェクト
  * `get_r` : 内部の後方伝播波が必要ない場合はfalseに設定（吸収計算にのみ必要）

# 出力

  * `ψref` :  反射振幅ベクトル
  * `ψtra` :  透過振幅ベクトル
  * `ψp` :  各層の内部後方伝播波ベクトルの配列
  * `ψm` :  各層の内部前方伝播波ベクトルの配列
