```
etm_reftra_flows(ψin,m,grd,λ)
etm_reftra_flows(ψin,m,grd,λ,em,sup,sub)
```

ETM法に従って、層間の反射と透過、ならびにネットパワーフローを計算します。

# 引数

  * `ψin` :  入射（ソース）振幅ベクトル
  * `m` :  RCWAモデルオブジェクト
  * `grd` : RCWAグリッドオブジェクト
  * `λ` : 自由空間波長
  * `em` :  すべての層の固有モードを含む配列（指定がない場合はmから計算されます）
  * `sup` :  スーパーストレート半空間オブジェクト（指定がない場合はmから計算されます）
  * `sub` :  基板半空間オブジェクト（指定がない場合はmから計算されます）

# 出力

  * `R` : デバイスによる反射
  * `T` : デバイスによる透過
  * `flw` : 層間のパワーフローを含む配列
