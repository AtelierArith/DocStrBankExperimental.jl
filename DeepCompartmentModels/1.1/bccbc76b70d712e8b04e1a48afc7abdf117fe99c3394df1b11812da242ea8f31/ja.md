```
two_comp_abs!(dA, A, p, t)
```

吸収コンパートメントを持つ二コンパートメントモデルのインプレース関数：

```
dA₁/dt = I - kₐ ⋅ A₁
dA₂/dt = kₐ ⋅ A₁ + A₃ ⋅ k₃₂ - A₂ ⋅ (k₂₀ + k₂₃)
dA₃/dt = A₂ ⋅ k₂₃ - A₃ ⋅ k₃₂
```

# モデルパラメータ

  * `kₐ`: 薬物吸収の速度。
  * `CL`: 第一コンパートメント（A₂）からの薬物クリアランス。
  * `V₁`: 中央分布容積（A₂内）。
  * `Q`: コンパートメント間クリアランス（A₃内）。
  * `V₂`: 周辺分布容積（A₃内）。
  * `I`: 薬物注入の速度；コールバック関数によって処理される。
