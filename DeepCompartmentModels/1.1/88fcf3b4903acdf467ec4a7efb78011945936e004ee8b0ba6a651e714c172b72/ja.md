```
Combine([out_dim], pairs...)
```

特定のブランチをLuxモデル内の特定の出力に接続します。このモデルにはBranchLayerが含まれています。同じ出力を指すブランチからの出力は、積を使用して結合されます。

# 引数:

  * `pairs...`: どのブランチがどの出力に接続されるかを示すペア。ペアは次の形式を取ります: i => [j] または [j, k, ...]

# 例

```jldoctest
julia> layer = Combine(1 => [1], 2 => [1], 3 => [2]) # ブランチ1を出力1に接続し、ブランチ2を出力1に接続します。
Combine()
```
