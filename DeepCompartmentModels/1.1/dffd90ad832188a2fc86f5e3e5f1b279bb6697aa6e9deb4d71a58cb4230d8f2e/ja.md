two_comp!(dA, A, p, t)

2コンパートメントモデルのインプレース関数：

```
dA₁/dt = (I / V₁) + A₂ ⋅ k₂₁ - A₁ ⋅ (k₁₀ + k₁₂)
dA₂/dt = A₁ ⋅ k₁₂ - A₂ ⋅ k₂₁
```

# モデルパラメータ

  * `CL`: 最初のコンパートメントからの薬物クリアランス。
  * `V₁`: 中心分布容積（A₁内）。
  * `Q`: コンパートメント間クリアランス。
  * `V₂`: 周辺分布容積（A₂内）。
  * `I`: 薬物注入の速度；コールバック関数によって処理される。

# 推定パラメータ

  * `k₁₀`: CL / V₁
  * `k₁₂`: Q / V₁
  * `k₂₁`: Q / V₂
