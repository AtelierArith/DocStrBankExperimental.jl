```
Magnus4Evolution(reg, start_clock, durations, hamiltonian, options)
```

`Magnus4Evolution`オブジェクトを作成します。

# 引数

  * `reg`: レジスタオブジェクト。
  * `start_clock`: 発展の開始クロック。
  * `durations`: 各時間ステップでの持続時間のリスト。
  * `hamiltonian`: [`Hamiltonian`](@ref)型の低レベルハミルトニアンオブジェクト。
  * `options`: [`KrylovOptions`](@ref)型の発展のオプション。
