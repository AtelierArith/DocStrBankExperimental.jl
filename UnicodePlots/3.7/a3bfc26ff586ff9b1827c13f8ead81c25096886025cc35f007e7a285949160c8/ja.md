```
annotate!(plot, x, y, text; kw...)
```

# 説明

位置 `(x, y)` にテキストをプロットに追加します。

# 引数

  * **`text`** : テキストの文字列。
  * **`x`** : 各点の水平方向の位置。
  * **`y`** : 各点の垂直方向の位置。
  * **`color::Symbol = :auto`** : （`:green`, `:blue`, `:red`, `:yellow`, `:cyan`, `:magenta`, `:white`, `:normal`, `:auto`）から選択するか、`[0-255]` の整数を使用するか、`RGB` コンポーネントとして `3` つの整数を提供します。
