# 説明

「モデル - ビュー - プロジェクション」変換行列を構築します（codinglabs.net/article*world*view*projection*matrix.aspxを参照）。

これは通常、3Dプロットの表示方法を調整するために使用されます。`surfaceplot`[@ref]、`isosurface`[@ref]の`projection`キーワードも参照してください。
