```
MVP(x, y, z; projection = :orthographic, elevation = 35.264389682754654, azimuth = 45.0, zoom = 1.0, up = :z)
```

# Description

Build up the "Model - View - Projection" transformation matrix (see codinglabs.net/article*world*view*projection*matrix.aspx).

This is typically used to adjust how 3D plot is viewed, see also the `projection` keyword in [`surfaceplot`](@ref), [`isosurface`](@ref).
