TernaryLines

Draw a line using barycentric coordindates, `x`, `y`, `z`, i.e. `x + y + z = 1`. Attributes are passed to `lines`. 

## Attributes

Available attributes and their defaults for `MakieCore.Plot{TernaryDiagrams.ternarylines}` are: 

```
  color      :red
  linestyle  :solid
  linewidth  4
```
