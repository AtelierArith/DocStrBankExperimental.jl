TernaryAxis

Draw the base triangle without any data to form a barycentric axis. Use the "adjustment" kwargs to adjust various formatting options.

## Attributes

Available attributes and their defaults for `MakieCore.Plot{TernaryDiagrams.ternaryaxis}` are: 

```
  arrow_label_fontsize                  16
  arrow_label_rotation_adjustment       0.85
  arrow_scale                           0.4
  grid_line_color                       :grey
  grid_line_width                       0.5
  hide_triangle_labels                  false
  hide_vertex_labels                    false
  label_edge_vertical_adjustment        0.1
  label_edge_vertical_arrow_adjustment  0.08
  label_fontsize                        18
  label_vertex_vertical_adjustment      0.05
  labelx                                "labelx"
  labelx_arrow                          "nothing"
  labely                                "labely"
  labely_arrow                          "nothing"
  labelz                                "labelz"
  labelz_arrow                          "nothing"
  tick_fontsize                         8
```
