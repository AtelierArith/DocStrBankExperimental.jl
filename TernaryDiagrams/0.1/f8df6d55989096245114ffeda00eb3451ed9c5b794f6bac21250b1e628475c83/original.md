TernaryScatter

Draw scattered data points using barycentric coordindates, `x`, `y`, `z`, i.e. `x + y + z = 1`. Attributes are passed to `scatter`. 

## Attributes

Available attributes and their defaults for `MakieCore.Plot{TernaryDiagrams.ternaryscatter}` are: 

```
  color       :red
  marker      :circle
  markersize  8
```
