```
@chain(initial_value, block::Expr)
```

ブロック式を再構成して、各行の結果を次の行に渡します。初期値は最初の引数によって与えられます。

すべての行で、アンダースコアは前の行の結果に置き換えられます。アンダースコアがなく、式がシンボルである場合、そのシンボルは前の結果を唯一の引数とする関数呼び出しに書き換えられます。アンダースコアがなく、式が関数呼び出しまたはマクロ呼び出しである場合、その呼び出しは前の結果を最初の引数として追加します。

例:

```
x = @chain [1, 2, 3] begin
    filter(!=(2), _)
    sqrt.(_)
    sum
end
x == sum(sqrt.(filter(!=(2), [1, 2, 3])))
```
