```
@chain(initial_value, args...)
```

一連の引数、式またはシンボルを再構成し、各行の結果を次の行に渡します。初期値は最初の引数によって与えられます。

すべての引数では、アンダースコアが引数の結果に置き換えられます。アンダースコアがなく、引数がシンボルの場合、そのシンボルは前の結果を唯一の引数とする関数呼び出しに書き換えられます。アンダースコアがなく、引数が関数呼び出しまたはマクロ呼び出しの場合、その呼び出しは前の結果を最初の引数として追加します。

例:

```
x = @chain [1, 2, 3] filter(!=(2), _) sqrt.(_) sum

x == sum(sqrt.(filter(!=(2), [1, 2, 3])))
```
