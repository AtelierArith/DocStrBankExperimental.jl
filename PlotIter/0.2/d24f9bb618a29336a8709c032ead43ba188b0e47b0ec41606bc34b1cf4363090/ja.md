```
clims_convex_hull!(plots)
clims_convex_hull!(plots...)
```

すべての `plots` の c 軸の制限を、既存の c 軸の制限をすべて含む最小の区間に設定します。

引数が `Subplot` の場合、特定の引数の制限に関して曖昧さはありません。しかし、引数が `Plot` の場合、次のようにします：     * `n` をすべての引数にわたる最小のサブプロットの数とします。     * `i` を `1:n` の範囲で取り、すべての `i` 番目のサブプロットに clims*convex*hull! を適用します。

これは、2つのプロットが視覚的に比較可能であることを保証するために便利です。
