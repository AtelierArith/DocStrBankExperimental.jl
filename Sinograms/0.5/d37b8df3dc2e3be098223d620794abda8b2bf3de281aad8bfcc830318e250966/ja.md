```
h, n = fbp_ramp(rg::SinoGeom, N::Int)
```

'ランプ状'フィルタは、平行ビームおよびファンビームのFBP再構成に使用されます。このサンプリングされたバンド制限アプローチは、周波数領域でランプを直接サンプリングすることによって引き起こされるエイリアシングを回避します。

# 入力

  * `rg::SinoGeom`
  * `N::Int` : サンプル数 (偶数でなければならない)

# 出力

  * `h::Vector{<:RealU}` : バンド制限ランプフィルタのサンプル
  * `n::UnitRange{Int64}` : -(N÷2):(N÷2-1)
