```
Hk = fbp_filter(rg::RayGeom ;
    npad=0, ds::RealU = rg.d, decon1::Bool=true, window=Window())
```

FBP画像再構成に使用されるランプ状フィルタの周波数応答を計算します。2Dおよび3Dの平行ビームおよびファンビームトモグラフィー幾何学をサポートします。このコードは、周波数領域でランプを直接サンプリングすることによって引き起こされるエイリアシングを回避するために、バンド制限されたランプをサンプリングします。

# 入力

  * `rg::RayGeom`

# オプション

  * `npad::Int` パディングされたサンプルの数。（デフォルト：次の2の累乗）
  * `ds::Td` 検出器サンプル間隔（デフォルトは`st`から）
  * `decon1::Bool` 線形補間器の効果を逆変換しますか？（デフォルト：true）
  * `window::Window` アポダイザー；デフォルト：`Window()`

# 出力

  * `Hk::Vector` アポダイズされたランプフィルタの周波数応答
