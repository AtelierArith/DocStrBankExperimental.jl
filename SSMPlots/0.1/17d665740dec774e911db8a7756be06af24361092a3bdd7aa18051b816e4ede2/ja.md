```
plot!([cur_plot], d::SSM2D; t_range=default_range(d), kwargs...)
```

既存のプロットに多選択逐次サンプリングモデルの確率密度を追加します

# 引数

  * `cur_plot`: オプションの現在のプロット
  * `d::SSM2D`: 多選択逐次サンプリングモデルのモデルオブジェクト

# キーワード

  * `t_range`: 確率密度がプロットされる時間点の範囲
  * `m_args=()`: 該当する場合、`rand`に渡されるオプションの位置引数
  * `kwargs...`: プロットオプションを設定するためのオプションのキーワード引数
