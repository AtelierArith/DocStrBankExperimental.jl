```
histogram(d::SSM2D;  kwargs...)
```

マルチオルタナティブ逐次サンプリングモデルのヒストグラムをプロットします。

# 引数

  * `d::SSM2D`: マルチオルタナティブ逐次サンプリングモデルのモデルオブジェクト

# キーワード

  * `norm=true`: trueの場合、密度に応じてスケーリングされたヒストグラム
  * `m_args=()`: `rand`に渡されるオプショナルな位置引数
  * `n_sim=2000`: ヒストグラムに使用されるデータポイントの数
  * `kwargs...`: プロットオプションを設定するためのオプショナルなキーワード引数
