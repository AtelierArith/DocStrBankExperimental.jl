```
histogram!([cur_plot], d::ContinuousMultivariateSSM;  kwargs...)
```

既存のプロットに多選択逐次サンプリングモデルのヒストグラムを追加します

# 引数

  * `cur_plot`: オプションの現在のプロット
  * `d::SSM2D`: 多選択逐次サンプリングモデルのモデルオブジェクト

# キーワード

  * `norm=true`: trueの場合、密度に応じてスケーリングされたヒストグラム
  * `m_args=()`: `rand`に渡されるオプションの位置引数
  * `n_sim=2000`: ヒストグラムに使用されるデータポイントの数
  * `kwargs...`: プロットオプションを設定するためのオプションのキーワード引数
