```
gen_fn = Switch(gen_fns::GenerativeFunction...)
```

引数タプルの型 `Tuple{Int, ...}` を受け取る新しい生成関数を返します。最初のインデックスは呼び出すブランチを示します。

```
gen_fn = Switch(d::Dict{T, Int}, gen_fns::GenerativeFunction...) where T
```

引数タプルの型 `Tuple{Int, ...}` または引数タプルの型 `Tuple{T, ...}` を受け取る新しい生成関数を返します。最初のインデックスは、呼び出すブランチを示すか、選択されたブランチにマッピングされる `d` へのインデックスを示します。この形式は便利さのために設計されています - プログラマーが `d` を if-else または case ステートメントのように使用できるようにします。

`Switch` は if-else 制御フローのパターンを表現するために設計されています。`gen_fns` はいくつかの要件を満たす必要があります：

1. `gen_fns` の各 `gen_fn` は同じ引数型を受け入れなければなりません。
2. `gen_fns` の各 `gen_fn` は同じ戻り値型を返さなければなりません。

そうでなければ、各 `gen_fn` は異なるモデリング言語から来ることができ、異なるトレースを持つことができます。
