```
HomogeneousMixture(distribution::Distribution, dims::Vector{Int})
```

いくつかの単一基底分布のインスタンスの混合物である新しい分布を定義します。

最初の引数は、混合物の各成分の基底分布を定義します。

2番目の引数は、基底分布が取る引数の数と同じ長さでなければなりません。ベクトルの位置に0がある場合は、対応する基底分布への引数がスカラーであることを示し、i >= 1の整数値iは、対応する引数がi次元配列であることを示します。

例:

```julia
mixture_of_normals = HomogeneousMixture(normal, [0, 0])
```

結果として得られる分布（上記の`mixture_of_normals`など）は、`normal`のような組み込み分布の値のように使用できます。この分布は、基底分布が取る引数の数をnとした場合、`n+1`の引数を取ります。分布への最初の引数は、非負の混合重みのベクトルであり、合計は1.0でなければなりません。分布への残りの引数は基底分布の引数に対応しますが、異なる型を持ちます: 基底分布への引数が型`T`のスカラーである場合、混合分布への対応する引数は`Vector{T}`であり、このベクトルの各要素は対応する混合成分への引数です。基底分布への引数が`Array{T,N}`である場合、混合分布への対応する引数は`arr::Array{T,N+1}`の形式であり、`arr[:,:,...,i]`の形の配列の各スライスが`i`番目の混合成分への引数です。

例:

```julia
mixture_of_normals = HomogeneousMixture(normal, [0, 0])
mixture_of_mvnormals = HomogeneousMixture(mvnormal, [1, 2])

@gen function foo()
    # 2つの正規分布の混合
    # 平均 -1.0 と 1.0
    # 標準偏差 0.1 と 10.0
    # 最初の正規分布の重みは0.4; 2番目は0.6
    x ~ mixture_of_normals([0.4, 0.6], [-1.0, 1.0], [0.1, 10.0])

    # 2つの多変量正規分布の混合
    # 平均: [0.0, 0.0] と [1.0, 1.0]
    # 共分散行列: [1.0 0.0; 0.0 1.0] と [10.0 0.0; 0.0 10.0]
    # 最初の多変量正規分布の重みは0.4;
    # 2番目は0.6
    means = [0.0 1.0; 0.0 1.0] # または、cat([0.0, 0.0], [1.0, 1.0], dims=2)
    covs = cat([1.0 0.0; 0.0 1.0], [10.0 0.0; 0.0 10.0], dims=3)
    y ~ mixture_of_mvnormals([0.4, 0.6], means, covs)
end
```
