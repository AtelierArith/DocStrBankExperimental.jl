```
sc_io_error_t
```

ioのエラー値。

| 列挙子               | 注釈                                      |
|:----------------- |:--------------------------------------- |
| SC*IO*ERROR_NONE  | ゼロの値はエラーがないことを意味します。                    |
| SC*IO*ERROR_FATAL | ioオブジェクトは現在機能不全です。                      |
| SC*IO*ERROR_AGAIN | 別のio操作がそれを解決するかもしれません。返された関数は何もしませんでした。 |
