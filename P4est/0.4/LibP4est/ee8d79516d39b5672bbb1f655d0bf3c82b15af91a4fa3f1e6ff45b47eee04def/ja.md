```
p4est_iter_face_side
```

森林の顔の一側に関する情報。

*quad* がローカルである場合（*is_ghost* が false）、その *quadid* は木の象限配列をインデックスします。そうでない場合は、ゴースト配列をインデックスします。顔がハンギングしている場合、象限は z-順序でリストされます。象限が存在する必要があるが、ゴースト層に含まれていない場合、quad = NULL、is_ghost は true、quadid = -1 になります。

| フィールド      | 注釈                                      |
|:---------- |:--------------------------------------- |
| treeid     | この側の木                                   |
| face       | 顔が接触する象限の側                              |
| is_hanging | ブール値: 完全な quad (0) または 2 つの小さな quad (1) |
