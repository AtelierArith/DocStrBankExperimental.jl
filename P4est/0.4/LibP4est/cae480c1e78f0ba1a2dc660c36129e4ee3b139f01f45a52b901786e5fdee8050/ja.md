sc_hash

[`sc_hash`](@ref)はハッシュテーブルを実装しています。要素としてリンクリストを持つ配列を使用します。

| フィールド      | 注釈                               |
|:---------- |:-------------------------------- |
| elem_count | 含まれるオブジェクトの総数                    |
| slots      | スロット数はslots->elem_count          |
| user_data  | ハッシュ関数に渡されるユーザーデータ               |
| allocator  | [`sc_link_t`](@ref)を割り当てる必要があります |
