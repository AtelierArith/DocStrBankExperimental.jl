`p8est`ツリーのデータ型

| フィールド            | 注釈                             |
|:---------------- |:------------------------------ |
| quadrants        | ローカルに保存された四分木                  |
| first_desc       | 最初のローカル子孫                      |
| last_desc        | 最後のローカル子孫                      |
| quadrants_offset | このプロセッサ上の以前のツリーに対する累積和（ローカルのみ） |
| maxlevel         | 最高のローカル四分木レベル                  |
