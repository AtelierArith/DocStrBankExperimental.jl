```
sc_mempool
```

[`sc_mempool`](@ref) オブジェクトは、同じサイズの要素の大きなプールを提供します。プールは要素の割り当てに応じて動的に成長します。要素はそのアドレスによって参照され、アドレスは決して変わりません。要素は解放（つまり、プールに戻すこと）され、透過的に再利用されます。zero*and*persist オプションが選択されている場合、新しい要素は作成時にすべてゼロに初期化され、要素の内容は解放と再返却の間に変更されることはありません。

| フィールド            | 注記                    |
|:---------------- |:--------------------- |
| elem_size        | 単一要素のサイズ              |
| elem_count       | 有効な要素の数               |
| zero*and*persist | ブール値; コンストラクタで設定されます。 |
| mstamp           | 固定サイズのチャンクアロケータ       |
| freed            | 解放された要素をバッファリング       |
