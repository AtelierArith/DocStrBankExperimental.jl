```
p8est_revision(p8est_)
```

フォレストのリビジョンカウンタを返します。全てのランクでリビジョン値は同じですが、集団的ではありません。新しく作成されたフォレストはリビジョンカウンタがゼロから始まります。メッシュを実際に変更するすべてのリファイン、コースン、パーティション、およびバランスは、カウンタを1つ増やします。効果のない操作は古い値を保持します。

### パラメータ

  * `p8est`:[in] フォレストは有効でなければなりません。

### 戻り値

非負の数。

### プロトタイプ

```c
long p8est_revision (p8est_t * p8est);
```
