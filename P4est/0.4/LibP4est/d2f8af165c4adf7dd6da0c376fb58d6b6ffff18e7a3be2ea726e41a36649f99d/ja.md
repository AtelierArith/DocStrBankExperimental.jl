```
p4est_connectivity_new_icosahedron()
```

球面をマッピングするための接続を作成します。 

正20面体は、各面が正三角形である20の面を持つ多面体です。`p4est`接続を構築するために、面を2つずつグループ化して10の四角形を形成し、したがって10の木を作成します。

この接続は、球面をマッピングするためにp4est*geometry*new_icosahedronと一緒に使用されることを意図しています。

平面接続は次のようになります：頂点番号：

A00 A01 A02 A03 A04 / \ / \ / \ / \ / \ A05–-A06–-A07–-A08–-A09–-A10 \ / \ / \ / \ / \ / \ A11–-A12–-A13–-A14–-A15–-A16 \ / \ / \ / \ / \ / A17 A18 A19 A20 A21

原点はA05にあります。

木の番号付け：

0 2 4 6 8 1 3 5 7 9

### プロトタイプ

```c
p4est_connectivity_t *p4est_connectivity_new_icosahedron ();
```
