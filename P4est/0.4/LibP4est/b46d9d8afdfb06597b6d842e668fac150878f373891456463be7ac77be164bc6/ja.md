```
sc_mstamp_init(mst, stamp_unit, elem_size)
```

メモリスタンプコンテナを初期化します。新しいメモリを毎回要求することなく、固定サイズのメモリアイテムの割り当てを提供します。代わりに、複数のアイテムを持つスタンプと呼ばれるブロック内で割り当てを行います。割り当てが行われなくても、コンテナの内部メモリは最終的にsc*mstamp*resetによって解放される必要があります。

### パラメータ

  * `mst`:[in,out] スタンプ構造体への合法的なポインタ。
  * `stamp_unit`:[in] 割り当てる各メモリブロックのサイズ。要素サイズより大きい場合、1つのブロックに複数の要素を配置することができます。0を渡すことは合法で、各アイテムを1つだけ保持するスタンプを強制します。
  * `elem_size`:[in] 各アイテムのサイズ。0を渡すことは合法です。その場合、sc*mstamp*allocはNULLを返します。

### プロトタイプ

```c
void sc_mstamp_init (sc_mstamp_t * mst, size_t stamp_unit, size_t elem_size);
```
