```
p8est_iterate(p4est_, ghost_layer, user_data, iter_volume, iter_face, iter_edge, iter_corner)
```

ローカルフォレスト内のすべてのボリューム、フェイス、エッジ、コーナーでユーザーが提供したコールバック関数を実行します。

ghost*layerはNULLである可能性があります。*user*data*ポインタは[`p8est_iterate`](@ref)によって変更されませんが、各コールバックに渡されます。コールバック関数のいずれかはNULLである可能性があります。コールバック関数は互いに交互に実行されます。つまり、いくつかのフェイスコールバックはボリュームコールバックの間に発生し、いくつかのエッジコールバックはフェイスコールバックの間に発生します。

1. ボリュームコールバックはソートされたモートンインデックス順に発生します。 2) フェイスコールバックは、それを共有するクアドラントのボリュームコールバックが実行された後にのみ実行されます。 3) エッジコールバックは、エッジに接触するすべてのフェイスのフェイスコールバックが実行された後にのみ実行されます。 4) コーナーコールバックは、コーナーに接触するすべてのエッジのエッジコールバックが実行された後にのみ実行されます。 5) 特定のクアドラントに対するすべてのフェイスコールバックがエッジまたはコーナーコールバックの前に実行されるとは限らず、特定のクアドラントに対するすべてのエッジコールバックがコーナーコールバックの前に実行されるとは限りません。 6) コールバックは、ローカルフォレストのセクションで隣接していないゴーストクアドラントのみを含むフェイス、エッジ、またはコーナーでは実行されません。

### パラメータ

  * `p4est`:[in] フォレスト
  * `ghost_layer`:[in] オプション: 指定されていない場合、ローカルパーティションの境界でのコールバックはゴーストクアドラントに関するデータを提供できません: 欠落している（[`p8est_quadrant_t`](@ref) *）ポインタはNULLに設定され、欠落しているインデックスは-1に設定されます。
  * `user_data`:[in,out] 各コールバックに供給するオプションのコンテキスト
  * `iter_volume`:[in] 各クアドラントの内部に対するコールバック関数
  * `iter_face`:[in] クアドラント間の各フェイスに対するコールバック関数
  * `iter_edge`:[in] クアドラント間の各エッジに対するコールバック関数
  * `iter_corner`:[in] クアドラント間の各コーナーに対するコールバック関数

### プロトタイプ

```c
void p8est_iterate (p8est_t * p4est, p8est_ghost_t * ghost_layer, void *user_data, p8est_iter_volume_t iter_volume, p8est_iter_face_t iter_face, p8est_iter_edge_t iter_edge, p8est_iter_corner_t iter_corner);
```
