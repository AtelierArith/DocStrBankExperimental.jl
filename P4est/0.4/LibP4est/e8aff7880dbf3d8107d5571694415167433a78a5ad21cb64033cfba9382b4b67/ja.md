```
p6est_connectivity
```

この構造体は、2D+1Dのツリー間接続情報を保持します。基本的には、3次元がどのように埋め込まれているかに関する追加情報を持つ2D p4est*connecitivity*tデータ型のラッパーです。

| フィールド        | 注釈                                                                                  |
|:------------ |:----------------------------------------------------------------------------------- |
| conn4        | 2D接続性; 所有; 頂点はシートの底の頂点として解釈されます                                                     |
| top_vertices | NULLの場合、均一な垂直プロファイル、それ以外の場合はシートの上部の頂点: *conn4*->tree*to*vertexと同じサイズである必要があります; 所有。 |
| height       | *top_vertices* == NULLの場合、シートの底から上部までのオフセットを示します                                    |
