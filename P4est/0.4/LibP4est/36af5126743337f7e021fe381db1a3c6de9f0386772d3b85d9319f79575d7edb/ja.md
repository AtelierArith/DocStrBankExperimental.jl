```
p4est_partition(p4est_, allow_for_coarsening, weight_fn)
```

森林を均等に分割します。分割は要素数またはユーザー定義の重みで行うことができます。

森林はプロセッサ間で分割され、ほぼ同じ数の四分木（または重みの合計）を持つようになります。

1つのプロセスでは、関数は何もしない（noops）で、重みコールバックは呼び出されません。それ以外の場合、重みコールバックは順番に各四分木ごとに1回呼び出されます。

### パラメータ

  * `p4est`:[in,out] 分割される森林。
  * `allow_for_coarsening`:[in] 四分木ファミリーがランク間で分割されないように、分割をわずかに修正します。
  * `weight_fn`:[in] 重み付け関数または均一分割のためのNULL。mpisize == 1で実行している場合は、決して呼び出されません。それ以外の場合、NULLでない限り、すべての四分木に対して順番に呼び出されます。各四分木に対して重みが1の定数重み付け関数は、weight_fn == NULLと同等ですが、他の定数重み付けは異なる均一分割をもたらす可能性があります。

### プロトタイプ

```c
void p4est_partition (p4est_t * p4est, int allow_for_coarsening, p4est_weight_t weight_fn);
```
