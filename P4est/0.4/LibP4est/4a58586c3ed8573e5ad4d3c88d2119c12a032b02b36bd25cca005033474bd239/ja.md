```
p8est_ghost_new(p8est_, btype)
```

ゴーストレイヤーを構築します。

これは、所有しているものの周りに顔、エッジ、コーナーのゴースト要素の1層を構築するために、各隣接プロセスから四分木を集めます。

### パラメータ

  * `p8est`:[in] ゴーストレイヤーが生成されるフォレスト。
  * `btype`:[in] どのゴーストを含めるか（顔、エッジ、またはコーナー/フル）。

### 戻り値

完全に初期化されたゴーストレイヤー。

### プロトタイプ

```c
p8est_ghost_t *p8est_ghost_new (p8est_t * p8est, p8est_connect_type_t btype);
```
