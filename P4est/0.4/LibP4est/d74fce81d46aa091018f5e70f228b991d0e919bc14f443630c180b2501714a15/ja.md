```
p8est_iter_face_side
```

森林内の面の一側に関する情報。*quad*がローカルである場合（*is_ghost*がfalseの場合）、その*quadid*は木の象限配列をインデックスします。それ以外の場合は、ゴースト配列をインデックスします。面がハンギングしている場合、象限はz順でリストされます。象限が存在する必要があるが、ゴースト層に含まれていない場合、quad = NULL、is_ghostはtrue、quadid = -1となります。

| フィールド      | 注釈                                 |
|:---------- |:---------------------------------- |
| treeid     | この側の木                              |
| face       | 面が接触する象限の側                         |
| is_hanging | ブール値：1つの完全なquad（0）または4つの小さなquad（1） |
