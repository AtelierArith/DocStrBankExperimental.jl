```
sc_mstamp
```

同じサイズのメモリアイテムを作成するためのデータコンテナです。割り当てはバンドルされているため、小さなメモリサイズの場合は高速です。作成されたアイテムは、コンテナが破棄されるまで有効のままです。アイテムをコンテナに戻すオプションはありません。その目的にはsc*mempool*tを参照してください。

| フィールド      | 注釈                 |
|:---------- |:------------------ |
| elem_size  | 入力パラメータ：アイテムごとのサイズ |
| per_stamp  | スタンプごとのアイテム数       |
| stamp_size | スタンプに割り当てられたバイト数   |
| cur_snext  | スタンプ内の次の番号         |
| current    | 現在のスタンプのメモリ        |
| remember   | すべてのスタンプを収集        |
