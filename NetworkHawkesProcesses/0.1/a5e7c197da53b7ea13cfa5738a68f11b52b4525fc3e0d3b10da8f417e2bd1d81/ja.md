LogGaussianCoxProcess

固定されたグリッドポイントでのガウス過程の実現から構築されたログガウスコックス過程。

モデルは次のようになります。

```
y ~ GP(0, K)
λ(t) = exp(m + y(t))
s ~ PP(λ(t))
```

任意のグリッドポイントのセット `x[1], ..., x[N]` に対して、ガウス過程の対応するサンプル `y[1], ..., y[N]` は `N(0, Σ)` 分布を持ち、ここで

```
Σ[i, j] = K(x[i], x[j])
```

プロセスは強度値 `λ[1], ..., λ[N]` の間を補間することによってサンプリングされます。

# 引数

  * `x::Vector{Float64}`: サンプリンググリッドポイントの厳密に増加するベクトル。
  * `λ::Vector{Vector{Float64}}`: `λ[k][i] = λ[k]([x[i])` となる非負の強度ベクトルのリスト。
  * `Σ::Matrix{Float64}`: 正定値分散行列。
  * `m::Vector{Float64}`: 均質プロセスの `log(λ0)` に等しい強度オフセット。
