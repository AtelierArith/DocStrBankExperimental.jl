```
離散対数ガウスコックス過程
```

固定グリッドポイントでのガウス過程の実現から構築された離散対数ガウスコックス過程。

確率モデルは次の通りです：

```
y ~ GP(0, K)
λ[t] = exp(m + y[t])
s ~ PP(λ[t])
```

任意のグリッドポイントのセットに対して、`x[1] = 1, ..., x[N] = T`、対応するガウス過程のサンプル`y[1], ..., y[N]`は`N(0, Σ)`分布を持ち、ここで

```
Σ[i, j] = K(x[i], x[j])
```

プロセスは強度値`λ[1], ..., λ[N]`の間を補間することによってサンプリングされます。

# 引数

  * `x::Vector{Int64}`: サンプリンググリッドポイントの厳密に増加するベクトル。
  * `λ::Matrix{Float64}`: 非負の`T x N`強度行列、すなわち`λ[i, n] = λ_n([x[i])`。
  * `Σ::Matrix{Float64}`: 正定値分散行列。
  * `m::Float64`: 均質プロセスの`log(λ0)`に等しい強度オフセット。
