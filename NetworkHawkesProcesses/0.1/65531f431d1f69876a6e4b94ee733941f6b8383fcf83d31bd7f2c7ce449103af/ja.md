```
DiscreteHomogeneousProcess
```

離散的同質ポアソン過程。

このモデルは、確率モデルのベイズ推論をサポートします：

```
λ[i] ~ Gamma(λ[i] | α0, β0) (i = 1, ..., N)
x[i, t] ~ Poisson(x[i, t] | λ[i] * dt) (t = 1, ..., T)
```

# 引数

  * `λ::Vector{Float64}`: 強度のベクトル。
  * `α0::Float64`: ガンマ事前分布の形状パラメータ。
  * `β0::Float64`: ガンマ事前分布の逆スケール（すなわち、レート）パラメータ。
  * `dt::Float64`: 各時間ステップ `t` に対応する物理時間。
