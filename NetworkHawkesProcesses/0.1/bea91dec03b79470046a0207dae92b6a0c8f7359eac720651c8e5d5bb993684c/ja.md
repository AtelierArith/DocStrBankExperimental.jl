ExponentialImpulseResponse

ガンマ事前分布を持つ指数インパルス応答関数。

このモデルはホークス過程の構成要素です。ノード `i` に起因するノード `j` の子イベントの数が与えられたとき、レートパラメータ `θ[i, j]` を持つ指数分布に従ってイベント時間を生成します。

ベイズ推論のために、`θ` に対して一様ガンマ事前分布を仮定します：

```
`θ[i, j] ~ Gamma(κ, ν) for all i, j`
```

ただし、得られるモデルはサンプリング期間が無限大に近づく限り共役であるだけです。したがって、指数過程に対するベイズ推論は近似的です。

# 引数

  * `θ::Float64`: 指数分布のレート（すなわち、`1 / scale`）。
  * `α`: `θ` のガンマ事前分布の形状パラメータ（デフォルト: 1.0）。
  * `β`: `θ` のガンマ事前分布のレートパラメータ（デフォルト: 1.0）。
