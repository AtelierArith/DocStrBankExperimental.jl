```
checkneighbour!(mdbm::MDBM_Problem{fcT,N,Nf,Nc,t01T,t11T,IT,FT,aT}; interpolationorder::Int=0, maxiteration::Int=0)
```

隣接するn-キューブの面をチェックして、失われた（欠落した）解の部分を発見します。

解の部分多様体（例：曲線）が完全に検出されていない可能性があります（中断されています）。このようにして、欠落した部分は継続法に似た方法で探索できます（最小限の関数評価で）。

解オブジェクトの次元がゼロより大きい場合にのみ機能します（多様体は点の集合ではありません）。

```
# 引数
- `interpolationorder::Int=0`: チェックされる隣接の補間順序法
- `maxiteration::Int=0~`: '継続的な' 探索における最大ステップ数。ゼロの場合、無限のステップが許可されます
```
