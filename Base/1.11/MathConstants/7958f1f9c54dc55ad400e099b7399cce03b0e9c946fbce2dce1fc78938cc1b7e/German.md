Die Konstante ℯ.

Unicode `ℯ` kann eingegeben werden, indem man `\euler` schreibt und die Tabulatortaste im Julia REPL und in vielen Editoren drückt.

Siehe auch: [`exp`](@ref), [`cis`](@ref), [`cispi`](@ref).

# Beispiele

```jldoctest
julia> ℯ
ℯ = 2.7182818284590...

julia> log(ℯ)
1

julia> ℯ^(im)π ≈ -1
true
```
