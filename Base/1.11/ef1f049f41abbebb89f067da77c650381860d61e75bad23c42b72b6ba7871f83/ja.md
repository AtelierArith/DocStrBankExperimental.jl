```
@raw_str -> String
```

補間やアンエスケープなしで生文字列を作成します。例外として、引用符はまだエスケープする必要があります。バックスラッシュは、引用文字の前にバックスラッシュのシーケンスがある場合にのみ、引用符と他のバックスラッシュの両方をエスケープします。したがって、2nのバックスラッシュの後に引用符が続くと、nのバックスラッシュとリテラルの終わりをエンコードし、2n+1のバックスラッシュの後に引用符が続くと、nのバックスラッシュと引用文字が続きます。

# 例

```jldoctest
julia> println(raw"\ $x")
\ $x

julia> println(raw"\"")
"

julia> println(raw"\\\"")
\"

julia> println(raw"\\x \\\"")
\\x \"
```
