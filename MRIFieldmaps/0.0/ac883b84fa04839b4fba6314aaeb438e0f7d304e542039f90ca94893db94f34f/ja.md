```
finit = b0init(ydata, echotime; kwargs...)
```

異なるエコー時間での複素画像の位相差に基づくクラシックなB0フィールドマップ推定。感度マップ（`smap`）が提供されている場合、最初に複素コイルの組み合わせが行われます。このコードは任意の次元（2D、3Dなど）の画像と複数のコイルで動作します。

通常、`echotime`の単位が秒である場合、返されるB0フィールドマップの単位はHzになります。

`df`が空でない場合（これは水-脂肪のケースでは常に成り立ちます）、`fdict`を使用して離散最大尤度推定を行います。

# 入力

  * `ydata (dims..., nc, ne)` `ne`セットの複素画像、コイル数`nc ≥ 1`
  * `echotime (ne)` `ne ≥ 2`のエコー時間のベクトル（最初の2つのみ使用）

# オプション

  * `smap (dims..., nc)` 複素コイルマップ; デフォルトは`nothing`
  * `threshold` `|y1| < threshold * max(|y1|)`の場所で`finit`値を設定し、`|y1| ≥ threshold * max(|y1|)`の「良い」値の平均にします。デフォルト: `0.1`

# 水-脂肪ケースのオプション:

  * `df` 水-脂肪イメージングにおけるΔf値（デフォルト: `[]`）単位Hz、例: `[440]` at 3T
  * `relamp` 多成分水-脂肪における相対振幅（デフォルト: `[]`）
  * `fband` `fdict`の周波数帯域幅; デフォルトは`floor(1 / minimum(echo time spacing))`
  * `nf` 試す離散周波数の数; デフォルトは`1+floor(fband)`で、約1Hzの間隔
  * `fdict` 試す離散周波数値の「辞書」; デフォルトは`LinRange(-1/2,1/2,nf) * fband`

# 出力

  * `finit` Hz単位の初期B0フィールドマップ推定
