```
Graph(;axiom, rules = nothing, data = nothing)
```

公理、1つ以上のルール、およびオプションでグラフレベルの変数から動的グラフを作成します。

## 引数

  * `axiom`: `Node`から継承された単一のオブジェクトまたはグラフ構築DSLで生成されたサブグラフ。動的グラフの初期状態を表す必要があります。

## キーワード

  * `rules`: 単一の`Rule`オブジェクトまたは`Rule`オブジェクトのタプル（オプション）。グラフのすべてのグラフ書き換えルールを含む必要があります。
  * `data`: 任意のユーザー定義型の単一のオブジェクト（オプション）。これは、グラフに適用される任意のルールまたはクエリからアクセス可能なグラフレベルの変数になります。
  * `FT`: グラフに関連付けられた3Dジオメトリを生成する際に使用される浮動小数点精度。

## 詳細

すべての引数はキーワードで指定されます。グラフを作成する際に公理とルールは深くコピーされますが、グラフレベルの変数（可変性のためにコピーが必要な場合、ユーザーがそれに注意する必要があります）はコピーされません。

## 戻り値

動的グラフを表す`Graph`型のオブジェクト。 このオブジェクトを印刷すると、グラフに格納されているデータの型の人間可読な説明が得られます。

## 例

```jldoctest
julia> struct A0 <: Node end;

julia> struct B0 <: Node end;

julia> axiom = A0() + B0();

julia> no_rules_graph = Graph(axiom = axiom);

julia> rule = Rule(A0, rhs = x -> A0() + B0());

julia> rules_graph = Graph(axiom = axiom, rules = rule);
```
