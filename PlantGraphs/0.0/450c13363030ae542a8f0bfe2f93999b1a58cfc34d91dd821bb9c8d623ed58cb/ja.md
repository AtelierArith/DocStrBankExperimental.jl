```
rewrite!(g::Graph)
```

グラフに保存されたグラフ書き換えルールを適用します。

## 引数

  * `g::Graph`: 書き換えられるグラフ。インプレースで変更されます。

## 詳細

この関数は、グラフ内のすべてのルールの左辺を一致させます。もし任意のノードが複数のルールによって一致した場合、エラーが発生します。ルールは、マッチしたノードをルールの右辺を実行した結果で置き換えるために順番に適用されます。ルールはグラフに保存されている順序で適用されますが、ノードが処理される順序は定義されていません。グラフの書き換えは意味的には並列プロセスであるため、ルールはその機能のために特定の順序に依存すべきではありません。

## 戻り値

この関数は `nothing` を返しますが、入力として渡されたグラフはルールの実行によって変更されます。

# 例

```jldoctest
julia> struct A <: Node end;

julia> struct B <: Node end;

julia> axiom = A() + B();

julia> rule = Rule(A, rhs = x -> A() + B());

julia> g = Graph(axiom = axiom, rules = rule);

julia> rewrite!(g);
```
