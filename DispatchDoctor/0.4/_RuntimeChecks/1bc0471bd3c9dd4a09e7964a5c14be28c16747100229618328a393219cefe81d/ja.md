```
allow_unstable(f::F) where {F<:Function}
```

提供された関数 `f` 内でタイプディスパッチドクターの不安定性チェックをグローバルに無効にします。

この関数は、タイプ不安定性チェックが無効になっているコードブロックを実行することを可能にします。ブロックが実行された後、エラーが発生してもチェックが再度有効になることを保証します。

この関数は `ReentrantLock` を使用しており、同時に2つのタスクから使用するとエラーが発生します。

# 使用法

```
allow_unstable() do
    # 不安定な処理を行う
end
```

# 引数

  * `f::F`: タイプ不安定性チェックが無効な状態で実行される関数。

# 戻り値

  * 関数 `f` の結果。

# 注意事項

同時に2つのタスクから `allow_unstable` を呼び出すことはできません。そうしようとするとエラーが発生します。
