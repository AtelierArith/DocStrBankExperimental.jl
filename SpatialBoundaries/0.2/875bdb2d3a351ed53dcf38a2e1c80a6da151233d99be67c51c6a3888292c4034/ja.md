```
LatticeWomble{T <: Number} <: Womble
```

空間に定期的に配置されたデータ（および座標）のセットは、`wombling`を通過した後に`LatticeWomble`型になります。この型のフィールドは次のとおりです。

  * `m`、各 (x,y) 座標での変化率の *行列*
  * `θ`、各 (x,y) 座標での変化の方向の *行列*
  * `x` および `y`、各セルの中心の座標

`x` と `y` の型は `m` と `θ` の要素型と同じである必要があります。なぜなら、これらの値はすべて変化率を計算する際に使用され、`x` と `y` はそれぞれ緯度と経度に対応するからです。
