```
getHyperParameters()
```

デフォルトのハイパーパラメータの値を返します

  * `nU = 1`: すべてのインスタンスに影響を与えると仮定される潜在的な混乱変数の数。これらの値に対して推論が行われます。
  * `nOuter = 20`: 引く後方サンプルの数。
  * `nMHInner = 5`: 各後方サンプルごとに行う内部メトロポリス-ヘイスティング更新の数。
  * `nESInner = 5`: 潜在的な混乱因子と二項治療のための各後方サンプルごとに行う楕円スライスサンプリング更新の数。
  * `nBurnIn = 5`: 予測と推定を行う際に破棄する後方サンプルの数。
  * `stepSize = 1`: 後方サンプルを使用する頻度（1はburnIn後のすべて、より高い値は`stepSize`*番目*ごと）。
  * `predictionCovarianceNoise=1e-10`: ガウス過程での予測には対称正定値の共分散行列の使用が必要であり、この対角上の共分散ノイズは、これらの操作が安定かつ一貫した方法で実行できることを保証します。
