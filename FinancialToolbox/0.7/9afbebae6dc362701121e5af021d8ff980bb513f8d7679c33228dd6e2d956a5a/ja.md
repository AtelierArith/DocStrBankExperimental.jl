ヨーロピアンオプションのブラックインプライドボラティリティ

```
	σ=blkimpv(F0,K,r,T,Price,FlagIsCall=true,xtol=1e-14,ytol=1e-15)
```

ここで：

```
	F0         = フォワードの価値。
	K          = オプションの行使価格。
	r          = ゼロレート。
	T          = オプションの満期までの時間。
	Price      = オプションの価格。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	σ          = ヨーロピアンオプションのインプライドボラティリティ。
```

# 例

```julia-repl
julia> blkimpv(10.0,10.0,0.01,2.0,2.0)
0.36568658096623635
```
