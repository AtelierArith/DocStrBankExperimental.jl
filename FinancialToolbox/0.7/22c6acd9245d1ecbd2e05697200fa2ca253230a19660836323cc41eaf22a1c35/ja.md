ブラック・ショールズの欧州オプションのインプライド・ボラティリティ

```
	σ=blsimpv(S0,K,r,T,Price,d=0.0,FlagIsCall=true,xtol=1e-14,ytol=1e-15)
```

ここで：

```
	S0         = 基礎資産の価値。
	K          = オプションの行使価格。
	r          = ゼロ金利。
	T          = オプションの満期までの時間。
	Price      = オプションの価格。
	d          = 基礎資産のインプライド配当。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	σ          = 欧州オプションのインプライド・ボラティリティ。
```

# 例

```julia-repl
julia> blsimpv(10.0,10.0,0.01,2.0,2.0)
0.3433730534290586
```
