ブラック・ショールズ・バンナ（Vanna） for 欧州オプション

```
	Vanna=blsvanna(S0,K,r,T,σ,d=0.0,FlagIsCall=true)
```

どこで：

```
	S0         = 基礎資産の価値。
	K          = オプションの行使価格。
	r          = ゼロ金利。
	T          = オプションの満期までの時間。
	σ          = インプライド・ボラティリティ
	d          = 基礎資産のインプライド・配当。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	Vanna        = 欧州オプションのバンナ。
```

# 例

```julia-repl
julia> blsvanna(10.0,10.0,0.01,2.0,0.2,0.01)
0.2737576307142566
```
