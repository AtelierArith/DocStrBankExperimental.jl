ブラック・ショールズのヨーロピアンオプションにおけるロー

```
	ρ=blsrho(S0,K,r,T,σ,d=0.0,FlagIsCall=true)
```

ここで：

```
	S0         = 基礎資産の価値。
	K          = オプションの行使価格。
	r          = ゼロ金利。
	T          = オプションの満期までの時間。
	σ          = インプライド・ボラティリティ。
	d          = 基礎資産のインプライド・配当。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	ρ          = ヨーロピアンオプションのロー。
```

# 例

```julia-repl
julia> blsrho(10.0,10.0,0.01,2.0,0.2,0.01)
8.699626722294234
```
