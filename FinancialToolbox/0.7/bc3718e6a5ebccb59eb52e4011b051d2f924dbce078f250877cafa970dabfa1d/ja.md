ブラック・ショールズ価格（ヨーロピアンオプション用）

```
	Price=blsprice(S0,K,r,T,σ,d=0.0,FlagIsCall=true)
```

ここで：

```
	S0         = 基礎資産の価値。
	K          = オプションの行使価格。
	r          = ゼロ金利。
	T          = オプションの満期までの時間。
	σ          = インプライド・ボラティリティ。
	d          = 基礎資産のインプライド・配当。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	Price      = ヨーロピアンオプションの価格。
```

# 例

```julia-repl
julia> blsprice(10.0,10.0,0.01,2.0,0.2,0.01)
1.1023600107733191
```
