ブラック・ショールズのプサイ（Psi） for 欧州オプション

```
	Ψ=blspsi(S0,K,r,T,σ,d=0.0,FlagIsCall=true)
```

ここで：

```
	S0         = 基礎資産の価値。
	K          = オプションの行使価格。
	r          = ゼロ金利。
	T          = オプションの満期までの時間。
	σ          = インプライド・ボラティリティ。
	d          = 基礎資産のインプライド・配当。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	Ψ          = 欧州オプションのプサイ。
```

# 例

```julia-repl
julia> blspsi(10.0,10.0,0.01,2.0,0.2,0.01)
-10.904346743840872
```
