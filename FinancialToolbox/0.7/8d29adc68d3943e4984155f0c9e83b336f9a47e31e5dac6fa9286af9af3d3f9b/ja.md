ブラック・ショールズ価格（欧州オプション用）

```
	Price=blkprice(F0,K,r,T,σ,FlagIsCall=true)
```

ここで：

```
	F0         = フォワードの価値。
	K          = オプションの行使価格。
	r          = ゼロ金利。
	T          = オプションの満期までの時間。
	σ          = インプライド・ボラティリティ。
	FlagIsCall = コールオプションの場合はtrue、プットオプションの場合はfalse。

	Price      = 欧州オプションの価格。
```

# 例

```julia-repl
julia> blkprice(10.0,10.0,0.01,2.0,0.2)
1.1023600107733191
```
