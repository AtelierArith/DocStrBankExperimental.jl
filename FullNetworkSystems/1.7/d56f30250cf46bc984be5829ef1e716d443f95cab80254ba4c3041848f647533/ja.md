`Branch`のコンストラクタ。ユーザーは`Branch`を線として定義するオプションがあります。例えば：

```
line1 = Branch("1", "A", "B", 10.0, 10.0, true, (100.0, 102.0), (5.0, 6.0), 1.0, 1.0)
```

ここで、最後の2つの値（`resistance`と`reactance`）は指定しなくても構いません。また、ユーザーは`Branch`を変圧器として定義することもできます：

```
trnasformer1 = Branch(
    "4", "A", "C", 10.0, 10.0, true, (100.0, 102.0), (5.0, 6.0), 1.0, 1.0, 0.5, 30.0
)
```

ここでは、`tap`と`angle`を表す2つの追加パラメータが最後に提供されています。
