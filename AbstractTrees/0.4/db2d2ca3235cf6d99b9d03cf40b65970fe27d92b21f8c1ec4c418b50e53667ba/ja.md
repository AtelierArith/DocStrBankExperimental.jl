```
PreOrderDFS{T,F} <: TreeIterator{T}
```

ツリーのノードを訪れるためのイテレータで、親が子よりも先に訪問されることを保証します。

オプションでフィルタ関数を受け取り、イテレータがノードの子を継続して反復するべきか（子がある場合）またはそのノードを葉と見なすべきかを決定します。

例えば、次のツリーの場合

```
Any[Any[1,2],Any[3,4]]
├─ Any[1,2]
|  ├─ 1
|  └─ 2
└─ Any[3,4]
   ├─ 3
   └─ 4
```

`[[[1, 2], [3, 4]], [1, 2], 1, 2, [3, 4], 3, 4]`が得られます。
