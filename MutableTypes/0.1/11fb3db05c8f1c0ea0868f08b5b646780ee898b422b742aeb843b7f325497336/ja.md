```
toString(x::Complex;
         format::Char='E',
         precsion::Int=5,
         aligned::Bool=false)::String
```

ここで `format` ∈ {'e', 'E', 'f', 'F'}（'e' または 'E' は指数または科学的表記を返し、'f' または 'F' は固定小数点表記を返します）、`precsion` ∈ {3, …, 7} は有効数字の数を示します。キーワード `aligned` は、数値文字列が他の数値文字列と列揃えされる場合、例えば行列を印刷する際に、`y` が非負の実部を持つときにスペースを印刷するために true に設定されるべきです。返される文字列は、実際の浮動小数点数の完全な精度を表す必要はありません。toString メソッドは、視覚的に消費するための数値を印刷することを目的としています。
