```
aws_socket_close(socket)
```

ソケットに対して `close()` を呼び出し、すべての I/O 操作をイベントループから登録解除します。この関数は、リスニングソケットでない限り、イベントループのスレッドから呼び出す必要があります。リスニングソケットの場合は、非イベントループスレッドまたはソケットが現在割り当てられているイベントループから呼び出すことができます。イベントループの外部から呼び出すと、この関数はソケットが閉じるのを待つためにブロックします。割り当てられているイベントループスレッド以外のイベントループスレッドから呼び出すとデッドロックの可能性があるため、そうしないでください。

Apple Network Framework を使用している場合は、サーバーソケットであろうとクライアントソケットであろうと、常にイベントループスレッドからこの関数を呼び出すべきです。

### プロトタイプ

```c
int aws_socket_close(struct aws_socket *socket);
```
