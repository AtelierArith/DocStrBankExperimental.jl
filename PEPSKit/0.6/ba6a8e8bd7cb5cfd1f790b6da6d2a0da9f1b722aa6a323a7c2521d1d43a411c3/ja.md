コーナー転送行列環境は、コーナーおよびエッジテンソルのユニットセル配列を含みます。配列の最後の2つのインデックスはユニットセルの行および列のインデックスに対応し、最初のインデックスはコーナーまたはエッジテンソルの方向に対応します。方向は、北西のコーナーと北のエッジから時計回りにラベル付けされています。

コーナーの配列 `c` とエッジの配列 `t` は、ユニットセル内のサイト `(r, c)` でネットワークテンソル `P` に接続されます：

```
   c[1,r-1,c-1]---t[1,r-1,c]----c[2,r-1,c+1]
   |              |             |
   t[4,r,c-1]-----P[r,c]--------t[2,r,c+1]
   |              |             |
   c[4,r+1,c-1]---t[3,r+1,c]----c[3,r+1,c+1]
```

ここで `P` は効果的なローカル構成テンソルを表します。これは、単一のランク4テンソル、PEPSテンソルのペア、またはネットワークが収束されるに応じてPEPS-PEPO-PEPSテンソルのスタックのいずれかです。

## フィールド

  * `corners::Array{C, 3} where C`: 空間方向を指定する最初の次元を持つコーナーテンソルの4 x 行 x 列の配列
  * `edges::Array{T, 3} where T`: 空間方向を指定する最初の次元を持つエッジテンソルの4 x 行 x 列の配列
