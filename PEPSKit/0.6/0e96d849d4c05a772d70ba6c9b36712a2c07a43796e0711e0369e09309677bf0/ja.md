```
InfinitePartitionFunction(A; unitcell=(1, 1))
```

テンソルと単位セルを指定することで `InfinitePartitionFunction` を作成します。

単位セルは行列としてラベル付けされているため、パーティション関数テンソルまたは環境テンソルに関係なく、単位セル内の任意のテンソルは行と列のインデックス `[r, c]` をそれぞれ1ずつシフトすることで得られます：

```
   |            |          |
---C[r-1,c-1]---T[r-1,c]---T[r-1,c+1]---
   |            |          |
---T[r,c-1]-----AA[r,c]----AA[r,c+1]----
   |            |          |
---T[r+1,c-1]---AA[r+1,c]--AA[r+1,c+1]--
   |            |          |
```

単位セルは周期境界条件を持つため、`[r, c]` は単位セルのサイズでモジュロインデックス付けされます。
