```
mat_fun(f, A; sep, max_deg, scale, color, ε, checknative)
```

行列変数関数 `f(A)` を計算します。行列を返します。

`f` は一般的なスカラー関数で、`f(x)` として呼び出されます。

`A` は任意の正方行列です。

`sep` は初期の分離距離で、`A` の固有値をクラスタに分割するために使用されます。これにより、(i) min{|λ - μ|: λ ∈ cl*p, μ ∈ cl*q, p ≠ q} > sep. (ii) |cl*p| > 1 の場合、∀ λ ∈ cl*p に対して、∃ μ ∈ cl*p かつ μ ≠ λ で、|λ - μ| ≤ sep となります。ここで cl_p はインデックス p のクラスタです。`sep=Inf` の場合、固有値は `color` によってのみ分割されます。

`max_deg` は対角ブロック計算のための最大テイラー級数の次数です。

`tol_taylor` は対角ブロックのテイラー級数を評価するための終了許容誤差です。

`scale` はテイラー級数の誤差のスケーリングで、各分割クラスタの広がりを制御するために使用されます。`scale=Inf` の場合、固有値は `sep` で一度だけ分割されます。

`color(x::Number)::Integer` はインデックスマッピング関数です。これは、クラスタ内のすべての固有値が同じ色のインデックスを持つことを保証するためのものです。デフォルトでは、`color` 関数は `x` を `1` にマッピングします。非連続関数の場合、ユーザーは各連続区間に異なる色を割り当てることができます。例えば、`sign` 関数の場合、ユーザーは `color = x -> Int(real(sign(x)))` を使用して色のマッピングをカスタマイズできます。

`ε` はデフォルトの誤差です。

`ε_im` は f(A) の虚部のデフォルトの誤差（複素シュール分解に使用されます）です。

`checknative` は `f` が Julia のネイティブ関数であることを確認します。
