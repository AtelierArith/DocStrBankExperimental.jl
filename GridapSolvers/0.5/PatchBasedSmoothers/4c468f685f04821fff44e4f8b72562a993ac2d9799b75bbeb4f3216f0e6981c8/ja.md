パッチベースの手法のための部分組み立て線形ソルバー。双線形形式 `a` とパッチ空間によって与えられる空間分解 `V = Σ_i V_i` が与えられたとき、集約された局所補正によって与えられるグローバル補正を返します。すなわち、

```
dx = Σ_i w_i I_i inv(A_i) (I_i)^* x 
```

ここで `A_i` は次のように定義されるパッチローカルシステム行列です。

```
(A_i u_i, v_i) = a(u_i,v_i) ∀ v_i ∈ V_i
```

また、`I_i` はパッチ空間からグローバル空間への自然な注入です。集約は無重み（すなわち `w_i = 1`）または重み付きで行うことができ、ここで `w_i = 1/#(i)` です。
