```
Init(data ; fast, label)
```

関数 `Problem` で使用する初期データを生成します。

`data` には以下のいずれかを含める必要があります。

  * 関数 `η` と関数 `v` （この順序で）;
  * `mesh` （型 `Mesh`）と `η(mesh.x)` および `v(mesh.x)` を表す2つのベクトル（この順序で）;
  * コレクションポイントの配列と `η(x)` および `v(x)` を表す2つのベクトル（この順序で）。

最後の2つのケースでは、オプションのキーワード引数 `fast` を `true` に設定できます（デフォルトは `false`）。この場合、アルゴリズムはより高速で、メモリの割り当てが少なくなりますが、精度は低下します。

最後のケースでは、コレクションポイントは等間隔でなければならず、そうでない場合は `ErrorException` が発生します。

キーワード `label::String` （出力ストリームに情報を表示するために使用される）が提供されていない場合、`"user-defined"` に設定されます。
