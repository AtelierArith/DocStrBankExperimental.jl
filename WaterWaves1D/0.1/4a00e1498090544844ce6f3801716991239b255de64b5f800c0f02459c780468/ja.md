```
modifiedMatsuno(param;kwargs)
```

初期値問題を解決するために、[Duchêne and Melinand](https://arxiv.org/abs/2203.03277)によって提案された修正マツノモデルの`AbstractModel`型のオブジェクトを定義します。

# 引数

`param`は`NamedTuple`型で、以下を含む必要があります。

  * 無次元パラメータ`ϵ`（非線形性）と`μ`（分散）；
  * オプションで、浅水/深水スケーリングファクター`ν`。デフォルトでは、`μ≦1`の場合は`ν=1`、それ以外の場合は`ν=1/√μ`です。`ν=0`または`μ=Inf`の場合は無限層ケースを設定します。
  * `mesh`がキーワード引数として提供されていない場合、コレクションポイントのメッシュを構築するための数値パラメータ。

## オプションのキーワード引数

  * `IL`：`IL=true`（または`μ=Inf`、または`ν=0`）の場合は無限層ケースを設定し、この場合`ϵ`は傾斜パラメータです。デフォルトは`false`です。
  * `mesh`：コレクションポイントのメッシュ。デフォルトでは`mesh = Mesh(param)`です；
  * `ktol`：ローパスKrasnyフィルターの許容誤差（デフォルトは`0`、すなわちフィルタリングなし）；
  * `dealias`：Orliczルール`1-dealias/(dealias+2)`によるディーリアシング（デフォルトは`0`、すなわちディーリアシングなし）；
  * `label`：将来の参照用のラベル（デフォルトは`"modified Matsuno"`）；

# 戻り値

`solve!`を介して初期値問題を解決するために必要な要素を生成します：

1. 明示的時間積分ソルバーで呼び出される関数`modifiedMatsuno.f!`；
2. `(η,v)`の型`InitialData`から計算が実行される生データ行列を提供する関数`modifiedMatsuno.mapto`；
3. そのようなデータ行列から実ベクトルのタプル`(η,v,x)`を返す関数`modifiedMatsuno.mapfro`、ここで

      * `η`はコレクションポイント`x`での表面変形の値です；
      * `v`は`x`での速度ポテンシャルのトレースの導関数です。
