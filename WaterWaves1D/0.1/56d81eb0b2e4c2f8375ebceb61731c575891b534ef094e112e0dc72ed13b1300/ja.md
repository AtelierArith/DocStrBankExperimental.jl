```
Mesh(args)
```

`N` 個のコレクションポイントを `xmin`（含む）と `xmax`（除外）との間に均等に配置し、関連するフーリエモードを持つ周期メッシュを構築します。

# 引数

次のいずれかであることができます。

  * `N`、`xmin`、`xmax` を含む `NamedTuple`；または
  * `N` と `L` を含む `NamedTuple`（この場合、`xmin=-L` および `xmax=L`）；または
  * 均等に配置されたコレクションポイントのベクトル。

# 戻り値

`m=Mesh(args)` はパラメトリック型であり、次の情報を提供します。

  * `m.N`: コレクションポイントおよびフーリエモードの数；
  * `m.xmin`: メッシュの最小値（コレクションポイントのベクトルに含まれる）；
  * `m.xmax`: メッシュの最大値（コレクションポイントのベクトルに除外される）；
  * `m.dx`: 2つのコレクションポイント間の距離；
  * `m.x`: コレクションポイントのベクトル；
  * `m.k`: 波数のベクトル；
  * `m.kmin`: 波数の最小値（波数のベクトルに含まれる）；
  * `m.kmax`: 波数の最大値（波数のベクトルに含まれる）；
  * `m.dk`: 2つのフーリエモード間の距離。
