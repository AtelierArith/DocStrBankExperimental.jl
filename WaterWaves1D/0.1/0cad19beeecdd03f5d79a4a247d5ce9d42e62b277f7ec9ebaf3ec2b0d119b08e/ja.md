```
Times(param; ns, Ns)
```

初期値問題で使用するための時間のメッシュを構築します（`Problem`を参照）。

# 引数

`param` は次のいずれかです。

  * `dt`（タイムステップ）と `T`（計算の最終時間）を含む `NamedTuple`；または
  * 計算された時間のベクトル。

## オプションのキーワード引数

  * `ns`  : データは `ns` 回の計算ごとに保存されます（オプション、デフォルトは 1）。
  * `Ns`  : `Ns` 個のデータ（初期データに加えて）が保存されます（オプション、デフォルトは `floor( tfin/dt)`）。

`Ns` と `ns` の両方が指定されている場合、`Ns` が `ns` を上書きします。

# 戻り値

`t=Times(args)` はパラメトリック型であり、次のことを提供します。

  * `t.Nc`: 計算された時間の数（初期データを含む）；
  * `t.Ns`: 保存された時間の数（初期データを含む）；
  * `t.ns`: 2つの保存された時間の間の計算された時間の数；
  * `t.tfin`: 最終時間；
  * `t.dt`: タイムステップ；
  * `t.tc` : 計算された時間のベクトル；
  * `t.ts`: 保存された時間のベクトル。
