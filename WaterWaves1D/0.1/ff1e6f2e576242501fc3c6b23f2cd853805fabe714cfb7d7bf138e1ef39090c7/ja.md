```
Random(param;args)
```

提供された（オプションの引数に基づいて）ランダムに生成された初期データ：

  * `L` は典型的な波長（デフォルトは `L=1`）、
  * `s` は（実数の）ソボレフ指数の正則性（デフォルトは `s=∞`）、
  * `λ` は空間的局所化の長さ（デフォルトは `λ=∞`、局所化なし）、
  * `a` は表面変形と速度の振幅の組（デフォルトは `a=(1,1)`）。

初期値問題 `Problem(model, init, param)` で使用する初期データ `init::InitialData` を返します。ここで、`init.η` と `init.v` は `random(x;L,s,λ)` の2つのサンプルであり、`x` は `Mesh(param)` によって生成されたコレクションポイントのセットです。
