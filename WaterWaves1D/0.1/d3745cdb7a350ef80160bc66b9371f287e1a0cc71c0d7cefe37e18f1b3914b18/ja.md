```
WaterWaves(param; kwargs)
```

初期値問題を解くための `AbstractModel` 型のオブジェクトを定義します（水波システムのために、共形写像を介して、[Zakharov, Dyachenko and Vasilyev](https://doi.org/10.1016/S0997-7546(02)01189-5)を参照）。

# 引数

`param` は `NamedTuple` 型であり、以下を含む必要があります。

  * 無次元パラメータ `ϵ`（非線形性）および `μ`（分散）;
  * オプションで、浅水/深水スケーリングファクター `ν`。デフォルトでは、`μ≦1` の場合は `ν=1`、それ以外の場合は `ν=1/√μ` です。`ν=0` または `μ=Inf` の場合は無限層ケースを設定します。
  * キーワード引数として `mesh` が提供されていない場合、コレクションポイントのメッシュを構築するための数値パラメータ。

## オプションのキーワード引数

  * `IL`: `IL=true` の場合は無限層ケースを設定し、この場合 `ϵ` は傾斜パラメータです。デフォルトは `false` です。
  * `mesh`: コレクションポイントのメッシュ。デフォルトでは `mesh = Mesh(param)` です;
  * `method ∈ {1,2,3}`: 共形写像を初期化するために使用されるメソッド、固定点問題 `F(u)=u` として

      * `method == 1` の場合、標準的な収束固定点反復を使用します;
      * `method == 2` の場合、ヤコビアンを反転するために GMRES 反復ソルバーを使用したニュートンアルゴリズムを使用します;
      * `method == 3` の場合、ヤコビアンを反転するために直接ソルバーを使用したニュートンアルゴリズムを使用します;
  * `tol`: 固定点アルゴリズムの（相対的）許容誤差（デフォルトは `1e-16`）;
  * `maxiter`: 固定点アルゴリズムの最大反復回数（デフォルトは `100`）;
  * `ktol`: ローパスKrasnyフィルターの許容誤差（デフォルトは `0`、すなわちフィルタリングなし）;
  * `dealias`: Orlicz ルールによるディーリアス処理 `1-dealias/(dealias+2)`（デフォルトは `0`、すなわちディーリアス処理なし）;
  * `label`: 将来の参照用のラベル（デフォルトは `"water waves"`）;
  * `verbose`: `true` の場合に情報を印刷します（デフォルトは `true`）。

# 戻り値

`solve!` を介して初期値問題を解くために必要な要素を生成します：

1. 明示的な時間積分ソルバーで呼び出される関数 `WaterWaves.f!`（また `WaterWaves.f1!` および `WaterWaves.f2!` はシンプレクティックオイラーソルバー用）;
2. `(η,v)` の型 `InitialData` から計算が実行される生データ行列を提供する関数 `WaterWaves.mapto`;
3. そのようなデータ行列から実ベクトルのタプル `(η,v,x)` を返す関数 `WaterWaves.mapfro`、ここで

      * `x` はコレクションポイントのベクトル（不規則に間隔を空けた）;

```
- `η` はコレクションポイント `x` での表面変形の値です;
- `v` は `x` での速度ポテンシャルのトレースの導関数です。
```
