```
substitute(b, a)
```

B系列 `b` を B系列 `a` に代入します。B系列 `b` は空の木の係数がゼロであると仮定されています。

Chartier、Hairer、Vilmart (2010) の表記法では、`substitute(b, a) = b ★ a` となります。
