```
Smoothing(transition::Float64=0.0, gain::Float64=1.0)
```

不連続性（ステップ関数）を含む計算を平滑化するためのパラメータ。

平滑化を無効にするデフォルト値は、`NoSmoothing` 定数として利用可能です。

## 背景と目的

稼働時間、燃料消費などのいくつかの計算は、ゼロで不連続なステップ関数を含みます：

step(x) = 0.0 if x==0.0 and 1.0 if 0 < x ≤ 1.0

このステップは、0 ≤ x ≤ `transition` の範囲で線形に平滑化できます。`transition` パラメータは以下の範囲であるべきです：

  * 0.0:平滑化なし（デフォルト値）
  * 1.0:最も強い平滑化（すなわち、凸緩和）

勾配ベースの最適化を使用する際には、平滑化を推奨し、その場合は0.05から0.30の間の「十分に小さい」`transition` 値が推奨されます。

また、この平滑化は真のステップを過小評価するため、オプションの `gain`≥1 を適用することができ、これにより step(x) = gain for x ≥ `transition` となります。
