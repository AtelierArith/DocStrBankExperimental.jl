```
go_game_graph(board_graph::Graphs.SimpleGraph)
```

無向グラフ `board_graph` 上で囲碁をプレイすることに対応する有向グラフを返します。囲碁は伝統的に正方形のグリッドでプレイされますが、ルールは任意の無向グラフに適用できます。すべての可能なボードの位置はゲームグラフの頂点として機能し、手は辺として機能します。空のボードは頂点1に対応します。結果は、サイクルが多数存在する疎な有向グラフであり、強連結です。これは `Graphs.SimpleDiGraph` として返されます。

ゲームグラフのサイズは、対応するボードグラフの頂点数に対して指数関数的に増加します。この関数は最大12頂点のボードグラフをサポートします。サイズ制限はオーバーライドできますが、警告されています。

```
go_game_graph(board_graph::SimpleGraph; allow_self_capture = false)
```

キーワード引数 `allow_self_capture` を `false` に設定すると、プレイヤーが自分の石を失う原因となる手を禁止できます。これにより、自己捕獲が許可されている場合よりもエッジの少ないゲームグラフが生成されます。

（補足：囲碁のルールのすべてのバリエーションは手を打つメカニクスに同意していますが、どの手が有効かについては異なる意見を持つことがあります。複数の石の自己捕獲を許可するかどうかはそのような違いの一つです。しかし、すべての囲碁ルールは、単一の石の自己捕獲は無効であると同意しています。なぜなら、それは位置を変えないからです。グラフの観点から言えば、`allow_self_capture` の設定に関係なく自己辺は存在しません。）

```
go_game_graph(n::Integer; kwargs...)
```

`go_game_graph(go_board_graph(n); kwargs...)` のショートカットです。
