```
Board(id::Integer)
```

与えられた `id` を持つ囲碁ボードグラフの表現を生成します。ノードの数は、`2^(n*(n-1)/2)>id` を満たす最小の `n` です。エッジは次の方法で見つけられます：

  * 潜在的なエッジを `(1,2), (1,3), ..., (1,n), (2,3), (2,4), ..., (2,n), ..., (n,n)` の順に列挙します。
  * `id` を `n*(n-1)/2` 桁の2進数に変換します。
  * 2進数の列に一致するリスト内のエッジがグラフに含まれます。

注意：これらのグラフの多くは同型です。ノードが2から12のボードがサポートされています（`1 <= id < 2^66` に対応）。

```
Board(edges::Vector{Vector{Int}})
```

各ノードからの指定された出力エッジのリストからボードグラフを構築します。これは無向グラフを記述していない場合、ゲームグラフを構築する際に問題が発生します。
