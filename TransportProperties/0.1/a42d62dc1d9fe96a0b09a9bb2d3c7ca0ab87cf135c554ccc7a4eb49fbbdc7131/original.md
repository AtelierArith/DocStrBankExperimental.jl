viscosity(sp*trd::Array{TransportData}, T::Float64, molwt::Array{Float64}, mole*fracs::Array{Float64}) Calculation of mixture viscosity in Kg/m-s

# Usage

```
viscosity(sp_trd, T, molwt, mole_fracs)
```

  * sp_trd: Species transport data
  * T: Temperature
  * molwt : molecular weight vector
  * mole_fracs : Mole fractions vector
