```
 dmuEfficiency(I₀,O₀,I,O)
```

バニラデータエンベロープ分析を使用してDMUの効率スコアを計算します。

## パラメータ:

  * `I₀`: このDMUの入力 (nI)
  * `O₀`: このDMUの出力 (n0)
  * `I`: すべてのDMUの入力 (nDMU x nI)
  * `O`: すべてのDMUの出力 (nDMU x n0)

## 戻り値:

  * 次の内容を持つ名前付きタプル:

      * `computed`: 基本的な最適化が成功したかどうかを示すブール値
      * `eff`: DMUが効率的かどうかを示すブール値
      * `obj`: DMUの効率スコアを表すスカラー
      * `wI`: 最適な入力重みのベクトル
      * `wO`: 最適な出力重みのベクトル
      * `refSet`: 参照DMUの数値をキーとし、

    相対制約双対を値とする辞書
