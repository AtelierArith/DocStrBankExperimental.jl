```
 dmuEfficiencyDual(I₀,O₀,I,O)
```

バニラデータエンベロープ分析を使用して、二重表現でDMUの効率スコアを計算します。二重パス法を使用します。

## パラメータ:

  * `I₀`: このDMUの入力 (nI)
  * `O₀`: このDMUの出力 (n0)
  * `I`: すべてのDMUの入力 (nDMU x nI)
  * `O`: すべてのDMUの出力 (nDMU x n0)

## 戻り値:

  * 次の内容を持つ名前付きタプル:

      * `computed`: 基本的な最適化が成功したかどうかを示すブール値
      * `eff`: DMUが効率的かどうかを示すブール値
      * `obj`: DMUの効率スコアを表すスカラー
      * `λ`
      * `θ`
      * `dualsI`
      * `dualsO`
      * `s⁻`
      * `s⁺`
