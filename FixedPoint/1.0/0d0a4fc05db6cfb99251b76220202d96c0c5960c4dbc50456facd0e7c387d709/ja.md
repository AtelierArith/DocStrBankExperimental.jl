```
afps!(f!, x; iters::Int = 5000, vel::Float64 = 0.9, ep::Float64 = 0.01, tol::Float64 = 1e-12, grad_norm=x->maximum(abs,x))
```

方程 `f(x) = x` を次のように解きます：

```
`f!` : 固定点を見つけるための関数のインプレースバージョンで、`f!(out,x)` を呼び出すことは `out = f(x)` と同等です。

`x` : 初期条件で、理想的には最終解に近い必要があります。

`vel` : Nesterov加速の量で、範囲は [0,1] です。

`ep` : 学習率で、通常は ]0,1[ の範囲です。

`tol` : |f(x)-x| に対する絶対許容誤差です。

`grad_norm` : |f(x)-x| のノルムを評価するための関数です。
```

返されるのは名前付きタプル (x, error, iters) で、次のようになります：

```
`x` : f(x)=x のために見つかった解です。

`error` : 解の点での f(x)-x のノルムです。

`iters` : 実行された総反復回数です。
```
