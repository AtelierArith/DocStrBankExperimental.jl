```
RepeatingDecimal
```

中間構造体は、循環小数を表すためのものです。

# 例

```julia-repl
julia> RepeatingDecimal(false, 12743, 857142, 2, 6)
       2|--|------|6
    -127.43(857142)
----------- --------------
有限部分 循環部分

julia> RepeatingDecimal(1//17)
         0||----------------|16
        +0.(0588235294117647)
----------- -------------------
有限部分 循環部分
```
