```
@rd_str
```

有理数を作成するための文字列マクロ。

# 例

```jldoctest
julia> r = rd"123.4(56)"  # 123.4565656...
61111//495

julia> rd"1.234r56e2"  # 他の表記
61111//495

julia> rd"123.45656..."  # もサポートされています。
61111//495

julia> float(r)  # 浮動小数点数の近似を確認します。
123.45656565656566

julia> rd"0.(9)"  # 0.999... は 1 に等しいです。
1//1

julia> rd"0.99(9)", rd"1", rd"1.000_000"  # 繰り返し小数の表記は一意ではありません。
(1//1, 1//1, 1//1)
```
