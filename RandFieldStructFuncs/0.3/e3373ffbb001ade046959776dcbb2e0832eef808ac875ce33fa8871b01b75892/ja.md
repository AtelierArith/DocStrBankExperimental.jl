```
A = EmpiricalStructFunc{T}(S[, plan])
```

は、浮動小数点型 `T` の値を持つ（空の）経験的構造関数をサポート `S` に対して生成します。パラメータ `T` は、引数から浮動小数点型を決定するために省略可能です。

オプション引数 `plan` が指定されていない場合、経験的構造関数は高速フーリエ変換（FFT）によって更新され、FFTプランを構築するためにキーワード `flags` と `timelimit` を指定することができます（デフォルトは `flags=FFTW.ESTIMATE` と `timelimit=Inf` です）。そうでない場合、`plan` は事前に計算された適切なFFTプランまたは **遅い** 更新方法を使用するための `nothing` である可能性があります。

基本メソッド `push!` を使用して、経験的構造関数オブジェクトにデータを *更新* することができます：

```
push!(A, φ)
```

ここで `φ` は、`S` と同じサイズの配列であるランダムサンプルです。`plan` が `nothing` の場合、`φ` はサポート `S` の非ゼロの数と同じ長さのベクトルでもかまいません。

経験的構造関数オブジェクト `A` には、以下のプロパティがあります：

```
A.support # サポート
A.num     # 累積分子
A.den     # 分母
A.nobs    # 観測数
```

以下のメソッドが適用可能です：

```
nobs(A)         # 観測数
Dᵩ = values(A)  # 経験的構造関数を計算
valtype(A)      # `Dᵩ` の要素型
```

`values(A)` によって計算された経験的構造関数 `Dᵩ` は `OffsetArray` インスタンスであり、次のようになります：

```
Dᵩ[Δr]
```

は、カルテシアンインデックスとして指定できる変位 `Δr` に対する経験的構造関数の値を返します。
