```
cov(Dᵩ::AbstractStructFunc, S, σ=0; pack=false) -> Cᵩ
```

は、構造関数が `Dᵩ` であり、サポートが `S` で定義され、ピストンモードの標準偏差が `σ` であるランダムフィールドの共分散を返します。

ランダムフィールドに対して考慮するインデックスの範囲は、キーワード `pack` が true でない限り `S` と同じです。この場合、サポート `S` 内のインデックスのみが考慮されます。

結果は、`pack` が false の場合は `n = length(S)` のフラットな `n×n` 共分散行列であり、`pack` が true の場合はサポート `S` 内の非ゼロの数に基づく `n` の共分散行列です。

実装されたメソッドは、このパッケージに付随するノートで説明されています。
