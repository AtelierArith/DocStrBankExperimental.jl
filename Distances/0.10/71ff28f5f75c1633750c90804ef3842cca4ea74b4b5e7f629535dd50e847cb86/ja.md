```
Euclidean([thresh])
```

ユークリッド距離を作成します。

多数の点間の距離を計算する際、次の式を利用することで、はるかに効率的になることがあります。

```
(x-y)^2 = x^2 - 2xy + y^2
```

しかし、これにより丸め誤差が生じる可能性があります。`thresh`（デフォルトは0）は、距離を再計算するために、`2xy`と`x^2 + y^2`を比較した相対的な平方距離の許容範囲を指定します。これにより、より正確な直接（要素ごとの引き算）式を使用して距離を強制的に再計算します。

# 例:

```julia
julia> x = reshape([0.1, 0.3, -0.1], 3, 1);

julia> pairwise(Euclidean(), x, x)
1×1 Array{Float64,2}:
 7.45058e-9

julia> pairwise(Euclidean(1e-12), x, x)
1×1 Array{Float64,2}:
 0.0
```
