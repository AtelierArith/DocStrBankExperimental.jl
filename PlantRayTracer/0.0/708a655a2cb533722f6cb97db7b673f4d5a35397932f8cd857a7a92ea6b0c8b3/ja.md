```
Phong(;τ = 0.0, ρd = 0.0, ρsmax = 0.0, n = 2)
```

透過率（`τ`）、拡散反射率（`ρd`）、最大Phong鏡面反射率（`ρsmax`）の値から`Phong`材料オブジェクトを作成します。`n`は「Phong反射ローブ」を制御する鏡面指数です。複数の波長がシミュレーションされる場合は、各光学特性に対して値のタプルを渡す必要があります（例：`τ = (0.1, 0.3)`）。

## 例

```jldoctest
julia> p = Phong(τ = 0.1, ρd = 0.2, ρsmax = 0.5);

julia> p = Phong(τ = (0.1, 0.45), ρd = (0.2, 0.45), ρsmax = (0.5, 0.5));
```
