```
RTSettings(;verbose = true, parallel = false, pkill = 0.2, maxiter = 2,
            sampler = Random.Xoshiro(123456789), nx = 3, ny = 3, nz = 0, dx = 0.0,
            dy = 0.0, dz = 0.0)
```

レイトレーサーの設定:

  * `verbose` はレイトレーサーが警告やその他の情報を出力するかどうかを示します。
  * `parallel` はレイトレーサーが単一コアで実行されるか、Juliaのマルチスレッドサポートに基づいてマシンの複数コアを利用するかを示します。`pkill` は、レイが散乱された後にロシアンルーレットによって終了される確率であり、`maxiter` 回数の間に散乱された後に適用されます（これは `Sensor` タイプの材料との相互作用を除外します）。
  * `sampler` はレイトレーサーによって使用される擬似乱数生成器です。
  * `nx` と `ny` はメッシュがグリッドクローンによって各方向（x軸とy軸）にクローンされる回数です（例: `nx = 1` と `ny = 1` を設定すると、元のメッシュの3 x 3のクローンのグリッドが生成されます）。一方、`dx` と `dy` は各新しいクローンが生成される距離（軸に沿って）です。
  * `nz` はメッシュが垂直方向にクローンされる回数です。水平クローンとは異なり、垂直クローンは常に `z` 軸の正の方向に行われ、クローンの数は正確に `nz` になります。

レイトレーサーに関する詳細はVPLのドキュメントを参照してください。

## 例

```jldoctest
julia> RTSettings(parallel = true, maxiter = 3);
```
