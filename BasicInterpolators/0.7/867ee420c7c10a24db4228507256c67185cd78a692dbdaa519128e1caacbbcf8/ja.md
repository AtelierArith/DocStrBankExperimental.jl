```
vandermonde(x, y)
```

座標 `x` で定義された点と値 `y` を通る任意の次数の多項式の係数を生成します。n 個の点に対して、n 個の係数 $[c_0, c_1, ..., c_{n-1}]$ が返され、これが多項式 $c_0 + c_1x + ... + c_{n-1}x^{n-1}$ を形成します。

!!! warning
    高次の多項式の係数を解くことは、悪条件の問題として知られています。5 または 6 より大きい次数の場合は推奨されませんが、アプリケーションによって異なります。高次の多項式で補間する必要がある場合は、係数を計算する代わりに [`neville`](@ref) 関数を使用する方が良いです。
