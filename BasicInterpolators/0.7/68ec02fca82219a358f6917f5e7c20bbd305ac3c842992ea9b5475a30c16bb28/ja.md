```
BichebyshevInterpolator(x, y, Z)
```

座標（`x`、`y`）と値 `Z` で定義された点のグリッドのための `BichebyshevInterpolator` を構築します。与えられた点は、各方向でチェビシェフグリッド上に存在する必要があります。これらは [`chebygrid`](@ref) 関数を使用して生成することができるか、以下のメソッドを使用して関数から直接補間器を構築することができます。

!!! warning
    Bichebyshev 補間器は *スレッドセーフではありません*。コサイン展開を計算し、オブジェクトに保存された配列を使用してインプレースでいくつかの線形代数を行います。単一の `BichebyshevInterpolator` は、同時に複数のスレッドによって呼び出されるべきではありません。
