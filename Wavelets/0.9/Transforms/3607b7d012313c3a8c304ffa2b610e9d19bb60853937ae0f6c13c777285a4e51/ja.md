```
dwt(x, wt[, L=maxtransformlevels(x)])
```

配列 `x` の離散ウェーブレット変換を実行します。ウェーブレットタイプ `wt` は変換タイプ（フィルターまたはリフティング）とウェーブレットクラスを決定します。詳細は `wavelet` を参照してください。

変換レベルの数 `L` は、`x` のサイズが `L` で割り切れる任意の非負整数にすることができます。`L==0` の場合は恒等変換を実行します。

# 例

```julia
dwt(x, wavelet(WT.coif6))
```

**関連情報:** `idwt`, `dwt!`, `wavelet`
