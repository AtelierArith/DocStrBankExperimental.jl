```
eigenbox(A[, method=Rohn()])
```

`A`のすべての固有値の囲いを返します。`A`が対称である場合、出力は実数の区間になりますが、そうでない場合は複素数の区間になります。

### 入力

  * `A` – 正方形の区間行列
  * `method` – 対称区間固有値問題を解くために使用される方法（一般行列の固有値の境界を求めることも対称の場合に還元されます）。可能な値は次のとおりです。

    ```
    - `Rohn` -- (デフォルト) 対称区間行列の固有値の囲いを計算するための高速な方法
    - `Hertz` -- 対称区間行列の固有値の正確なハルを見つけますが、指数的な複雑さがあります。
    ```

### アルゴリズム

関数によって使用されるアルゴリズムは[[HLA13]](@ref)に記載されています。

### 注意事項

囲いは厳密ではなく、内部で解決される実数固有値問題は通常の浮動小数点計算を利用します。

### 例

```jldoctest
julia> A = [0 -1 -1; 2 -1.399.. -0.001 0; 1 0.5 -1]
3×3 Matrix{Interval{Float64}}:
 [0, 0]  [-1, -1]                       [-1, -1]
 [2, 2]       [-1.39901, -0.000999999]    [0, 0]
 [1, 1]        [0.5, 0.5]               [-1, -1]

julia> eigenbox(A)
[-1.90679, 0.970154] + [-2.51903, 2.51903]im

julia> eigenbox(A, Hertz())
[-1.64732, 0.520456] + [-2.1112, 2.1112]im
```
