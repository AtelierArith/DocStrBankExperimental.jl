```
Problem(ℓ::Real, d::Real, ℐ::Integer, t::AbstractRange{Real}; O = 4, M_s = 0, M_b = 0, static_bottom = true)
```

流体領域の長さ `ℓ` と深さ `d`、`ℐ` の調和数および `N` の時間ステップに対応する IBV 問題オブジェクトを構築します。

出力は次のフィールドを持つ Problem オブジェクトです：

  * `ℓ` は流体領域の長さ (m)、
  * `d` は水深 (m)、
  * `ℐ` は調和数、
  * `t` は時間範囲、
  * `Δt` は時間ステップ (s)、
  * `N` は時間ステップの数、
  * `O` は時間積分スキームの次数、
  * `κ` は波数 (rad/m)、
  * `η̂` は自由表面の高さ振幅 (m)、
  * `η̇` は自由表面の垂直速度振幅 (m/s)、
  * `β̂` は底面の高さ振幅 (m)、
  * `β̇` は底面の垂直速度振幅 (m/s)、
  * `ϕ̂` は平底の速度ポテンシャル振幅 (m²/s)、
  * `ϕ̇` は平底の加速度ポテンシャル振幅 (m²/s²)、
  * `ψ̂` は不均一底の速度ポテンシャル振幅 (m²/s)、
  * `ψ̇` は不均一底の加速度ポテンシャル振幅 (m²/s²)、
  * `p̂` は表面圧力頭振幅 (m)、
  * `χ` は波動発生器のパドル変位 (m)、
  * `ξ` は波動発生器のパドル速度 (m/s)、
  * `ζ` は波動発生器のパドル加速度 (m/s²)、
  * `𝒯` は双曲線正接のルックアップ値、
  * `𝒮` は双曲線余接のルックアップ値、
  * `static_bottom` は底が静的かどうかを示すブールフラグです。
