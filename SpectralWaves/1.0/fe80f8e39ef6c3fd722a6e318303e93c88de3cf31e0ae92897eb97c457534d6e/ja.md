```
solve_problem!(p::Problem; msg_flag = true)
```

波動問題の解決係数 `η̂`, `η̇`, `ϕ̂`, `ϕ̇`, `ψ̂`, `ψ̇` を計算します。

インプレースで変更される変数:

  * `η̂` は自由表面の標高ポテンシャル振幅 (m) です。
  * `η̇` は自由表面の垂直速度振幅 (m/s) です。
  * `ϕ̂` は平底の速度ポテンシャル振幅 (m²/s) です。
  * `ϕ̇` は平底の加速度ポテンシャル振幅 (m²/s²) です。
  * `ψ̂` は不均一底の速度ポテンシャル振幅 (m²/s) です。
  * `ψ̇` は不均一底の加速度ポテンシャル振幅 (m²/s²) です。

入力変数:

  * `β̂` は底面の標高振幅 (m) です。
  * `β̇` は底面の垂直速度振幅 (m/s) です。
  * `p̂` は表面圧力頭振幅 (m) です。
  * `κ` は波数 (rad/m) です。
  * `𝒯` は双曲線正接のルックアップ値です。
  * `𝒮` は双曲線セカントのルックアップ値です。
  * `ℐ` は調和数です。
  * `M_s` は非線形自由表面境界条件の次数です。
  * `M_b` は非線形底境界条件の次数です。
  * `Δt` は時間ステップ (s) です。
  * `O` は時間ステッピングスキームの次数です。
  * `N` は時間ステップの数です。
  * `χ` は波作りパドルの変位 (m) です。
  * `ξ` は波作りパドルの速度 (m/s) です。
  * `ζ` は波作りパドルの加速度 (m/s²) です。
  * `ℓ` は流体領域の長さ (m) です。
  * `d` は水深 (m) です。

キーワード引数:

  * `static_bottom` は底が静的であるかどうかを示すブールフラグです。
  * `msg_flag` は進行状況メッセージを印刷するかどうかを示すブールフラグです。
