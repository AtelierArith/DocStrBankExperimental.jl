```
support_enumeration(g)
```

2人の通常形ゲームに対して、サポート列挙によって等しいサポートサイズの混合行動ナッシュ均衡を計算します。非退化ゲームの入力に対して、これらはすべてのナッシュ均衡です。

アルゴリズムはすべての等サイズのサポートペアをチェックします。プレイヤーが同じ数 n のアクションを持つ場合、2n choose n マイナス 1 のペアがあります。したがって、これは小さなゲームにのみ使用するべきです。

# 引数

  * `g::NormalFormGame{2,T}`: 2人用のNormalFormGameインスタンス。

# 戻り値

  * `::Vector{NTuple{2,Vector{S}}}`: 見つかった混合行動ナッシュ均衡で、`S` は `T` が Int または Float の場合は Float、`T` が Rational の場合は Rational です。
