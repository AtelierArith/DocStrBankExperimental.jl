```
MPBOptions(; kwargs...)
```

アルゴリズムを構成するためのオプション。`MPBOptions`型のインスタンスは、キーワード引数`options`を使って最適化メソッドに渡すことができます。

`MPBOptions`の可能な`kwargs`は次のとおりです：

  * `max_iter`: 最大反復回数（デフォルト：1000）。デフォルト：1000
  * `max_fun_calls`: 最大関数評価回数（デフォルト：1000）。デフォルト：1000
  * `mL`: $tL ∈ [0,1]$の決定のためのラインサーチパラメータ（デフォルト：1e-2）。デフォルト：0.01
  * `mR`: $tR$の決定のためのラインサーチパラメータ（デフォルト：0.5）。デフォルト：0.5
  * `t_bar`: 長いステップまたは短いステップの分類のためのラインサーチパラメータ（デフォルト：1e-2）。デフォルト：0.01
  * `t_min`: ラインサーチでテストされる最小ステップサイズ（デフォルト：`eps(Float64)*1e2`）。デフォルト：eps(Float64) * 100.0
  * `max_fun_calls_ls`: ラインサーチでの最大関数呼び出し回数（デフォルト：100）。デフォルト：100
  * `gamma_default`: 提供されていない場合のフォールバック非凸性定数（デフォルト：0.5）。デフォルト：0.5
  * `tol_acc`: 最終精度許容誤差（デフォルト：1e-5）。デフォルト：1.0e-5
  * `initial_weight`: 初期重み $u^1$（デフォルト：NaN）。デフォルト：NaN
  * `max_subgrads`: 保存されるサブ勾配の最大数（デフォルト：10）。デフォルト：10
