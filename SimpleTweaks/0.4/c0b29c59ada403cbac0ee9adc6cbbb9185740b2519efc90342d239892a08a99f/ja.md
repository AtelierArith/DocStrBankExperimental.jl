```
curve_length(f::Function, a::Real, b::Real) -> Float64
```

与えられた `f` によって定義される曲線の長さを計算します。ここで `f` は1つのパラメータの関数であり、指定されたパラメータの区間 <`a`, `b`> で定義されています。

# 引数

  * `f::Function`: 測定される曲線の関数、
  * `a::Real`: 積分する区間の左端、
  * `b::Real`: 積分する区間の右端、

# キーワード

# 戻り値

  * `Float64`: 区間 <`a`, `b`> での `f` によって定義される曲線の長さ。
