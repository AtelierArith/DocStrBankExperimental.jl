```
WhiteTest(X, e; type = :White)
```

ホワイトの（またはブリュッシュ-パガンの）ヘテロスケダスティシティのテストを計算します。

`X` は回帰変数の行列で、`e` は元のモデルからの残差のベクトルです。キーワード引数 `type` は、ブリュッシュ-パガン/コエンカーのテスト用の `:linear`、線形および二次項のみ（交差積なし）のホワイトのテスト用の `:linear_and_squares`、または完全なホワイトのテスト（線形、二次および交差積項）のデフォルトである `:White` のいずれかです。`X` には定数と少なくとも1つの回帰変数が含まれている必要があり、観測値は行に、回帰変数は列に配置されます。一部のアプリケーションでは、`X` は元のモデルの回帰変数のサブセットであるか、単にフィッティングされた値です。これにより自由度が節約され、より強力なテストが得られる場合があります。`lm`（ラグランジュ乗数）テスト統計量は T*R2 であり、ここで R2 は上記の項に対する `e^2` の回帰から得られます。帰無仮説の下では、これは `Chisq(dof)` として分布し、ここで `dof` は独立した項の数（定数をカウントしない）であるため、テスト統計量が十分に大きいときに帰無仮説は棄却されます。
