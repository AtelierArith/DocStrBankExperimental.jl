```
sane_bonds = bond_sanity_check(crystal)
```

`crystal.bonds`に対してサニティチェックを実行します。

  * ボンドグラフは完全に接続されていますか？すなわち、ボンドグラフ内のすべての頂点（=原子）は少なくとも1つのエッジを持っていますか？
  * 各水素は1つのボンドしか持てません
  * 各炭素は最大4つのボンドを持つことができます

サニティチェックに失敗した場合は、[`write_bond_information`](@ref)を参照してボンドを視覚化するための.vtkを書き出します。

サニティチェックに失敗した場合は警告を表示します。サニティチェックが成功した場合は`true`を返し、そうでない場合は`false`を返します。
