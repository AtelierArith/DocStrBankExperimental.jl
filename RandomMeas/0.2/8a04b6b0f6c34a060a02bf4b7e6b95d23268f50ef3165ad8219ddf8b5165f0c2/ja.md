```
get_h_tensor(s::Index, s_prime::Index) -> ITensor
```

与えられたインデックスのためのハミングテンソルを構築します。

# 引数

  * `s::Index`: 非プライムサイトインデックス。
  * `s_prime::Index`: プライムサイトインデックス。

# 戻り値

  * `Hamming_tensor::ITensor`: `s` と `s_prime` を接続するハミングテンソル。

# メソッド

  * インデックス `s` と `s_prime` で `ITensor` を初期化します。
  * ハミング距離操作を表す値を割り当てます：

      * 対角要素は `1.0` に設定されます。
      * 非対角要素は `-0.5` に設定されます。
