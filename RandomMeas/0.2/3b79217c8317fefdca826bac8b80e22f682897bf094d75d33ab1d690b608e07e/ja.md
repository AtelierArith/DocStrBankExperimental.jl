```
get_Born_MPS(ρ::MPO)
```

密度行列 ρ の MPO 表現から Born 確率ベクトルを MPS として構築します。

この関数は Born 確率ベクトル P(s) = ⟨s|ρ|s⟩ を計算します。ここで |s⟩ は基底状態です。これは、MPO ρ の各テンソルを適切なデルタテンソルと収束させ、未プライムインデックスとプライムインデックスの間の等式を強制します。結果は計算基底に対する Born 確率を表す MPS として返されます。

# 引数

  * `ρ::MPO`: 密度行列を表す行列積演算子。

# 戻り値

Born 確率ベクトルを表す MPS。

# 例

```julia
P = get_Born_MPS(ρ)
```
