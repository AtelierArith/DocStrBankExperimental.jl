```
get_average_mps(ψ_list::Vector{MPS}, χ::Int64, nsweeps::Int64)
```

MPSのコレクションから平均状態σを近似するためのDMRGのようなアルゴリズムを使用します。

このアルゴリズムは、平均状態σ = Average(ψ_list)を近似する最大結合次元χのMPS ψを見つけます。収束を監視するために、次のように定義されたコスト関数を追跡します：

```
cost_function = ⟨ψ|ψ⟩ - ⟨ψ|σ⟩ - ⟨σ|ψ⟩,
```

これは(||σ - ψ||² - ⟨σ|σ⟩)に等しいです。

# 引数

  * `ψ_list::Vector{MPS}`: 個々の量子状態を表すMPSオブジェクトのベクター。
  * `χ::Int64`: 平均化されたMPSのための望ましい最大結合次元。
  * `nsweeps::Int64`: DMRGのようなアルゴリズムで実行するスイープ（反復）の数。

# 戻り値

結合次元χを持つ近似平均状態を表すMPS。

# 例

```julia
avg_state = get_average_mps(ψ_list, 20, 10)
```
