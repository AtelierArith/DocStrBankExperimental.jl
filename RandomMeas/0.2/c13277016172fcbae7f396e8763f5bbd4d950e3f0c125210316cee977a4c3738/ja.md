```
get_selfXEB(ψ::MPS)
```

純粋状態をMPSとして表現した場合の自己XEB（交差エントロピーベンチマーキング）メトリックを計算します。

自己XEBは次のように定義されます：

```
selfXEB = 2^N * ∑ₛ |ψ(s)|⁴ - 1
```

ここで、合計はすべての計算基底状態sにわたるもので、Nはサイト数（キュービット）です。この関数はまずψからボルン確率MPSを計算し、次に確率MPSの自己内積を計算し、結果を2^Nでスケーリングし、最後に1を引きます。

# 引数

  * `ψ::MPS`: 純粋量子状態を表す行列積状態。

# 戻り値

自己XEB値を表すスカラー（Float64）。

# 例

```julia
x = get_selfXEB(ψ)
```
