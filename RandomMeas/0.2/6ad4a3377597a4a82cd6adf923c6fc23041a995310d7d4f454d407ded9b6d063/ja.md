```
get_Born_MPS(ψ::MPS)
```

MPS表現ψからボルン確率ベクトルP(s) = |ψ(s)|²をMPSとして構築します。

この関数は、MPS ψの各テンソルをその複素共役と収束させ、インデックスの等しさを強制するために適切なデルタテンソルを使用することによって、各計算基底状態の確率を計算します。結果として得られるMPSは、状態のボルン確率分布を表します。

# 引数

  * `ψ::MPS`: 純粋量子状態を表す行列積状態。

# 戻り値

各テンソルP[i]がサイトiでの確率寄与に対応するボルン確率ベクトルを表すMPS。

# 例

```julia
P = get_Born_MPS(ψ)
```
