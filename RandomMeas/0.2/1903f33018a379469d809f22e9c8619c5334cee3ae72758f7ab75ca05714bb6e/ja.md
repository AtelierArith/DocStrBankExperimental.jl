```
random_Pauli_layer(ξ::Vector{Index{Int64}}, p::Vector{Float64})
```

ランダムな単一量子ビットパウリ演算の層を構築して、局所的な脱極化をシミュレートします。平均化すると、これは強度 p の局所脱極化チャネルに対応します。

各量子ビット（インデックス i を持つ）に対して、次の確率でランダムなパウリ演算が適用されます：

  * 確率 1 - 3p_i/4 で：操作は適用されず（量子ビットは変更されません）。
  * 確率 p_i/4 でそれぞれ：X、Y、または Z ゲートを適用します。

ここで、p_i は量子ビット i の脱極化確率です。

# 引数

  * `ξ::Vector{Index{Int64}}`: 量子ビットサイトを表す ITensor インデックスのベクトル。
  * `p::Vector{Float64}`: 脱極化確率のベクトル（量子ビットごとに1つ）。

# 戻り値

適用されたパウリゲートを表す ITensor のベクトル。サイトにゲートが適用されない場合（確率 1 - 3p_i/4）、そのサイトは返される回路から省略されます。

# 例

```julia
circuit = random_Pauli_layer(siteinds("Qubit", 5), 0.05 * ones(5))
```
