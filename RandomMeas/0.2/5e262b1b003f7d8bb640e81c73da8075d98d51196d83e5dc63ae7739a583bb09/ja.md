```
get_trace_moment(spec::ITensor, k::Int)
```

ITensor `spec` によって表されるエンタングルメントスペクトルの k 番目のモーメントを計算します。

この関数は、`spec` が平方 ITensor であり、その対角要素が縮小密度行列の特異値（シュミット係数）に対応していると仮定します。k 番目のモーメントは次のように計算されます：

```
pk = ∑ₗ (spec[l, l]^(2*k))
```

これは、特異値の二乗の k 番目の累乗の合計を効果的に計算します。

# 引数

  * `spec::ITensor`: エンタングルメントスペクトルを表す平方 ITensor。
  * `k::Int`: 計算するモーメントの次数（整数 ≥ 1 でなければなりません）。

# 戻り値

k 番目のモーメントに対応するスカラー (Float64)。

# 例

```julia
moment = get_trace_moment(spec, 2)
```
