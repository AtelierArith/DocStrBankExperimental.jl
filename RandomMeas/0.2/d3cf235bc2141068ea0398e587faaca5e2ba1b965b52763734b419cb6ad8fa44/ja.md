```
apply_depo_channel(ρ::MPO, p::Vector{Float64})
```

MPOにローカルデポライズチャネルを適用し、各サイトテンソルをデポライズ確率に従って修正します。

各サイトに対して、チャネルは次のように作用します：

ρ[i] → (1 - p[i]) * ρ[i] + (p[i] / 2) * (ρ[i] * δ(s, s') * δ(s, s'))

ここで、δ(s, s')はサイトインデックスをそのプライム対応物と収束させるデルタテンソルです。

# 引数

  * `ρ::MPO`: 密度行列を表す入力マトリックスプロダクトオペレーター。
  * `p::Vector{Float64}`: 各サイトごとのデポライズ確率のベクトル。

# 戻り値

各サイトにデポライズチャネルが適用されたMPO。
