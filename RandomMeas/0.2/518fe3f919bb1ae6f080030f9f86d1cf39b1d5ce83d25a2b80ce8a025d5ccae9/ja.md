```
random_circuit(ξ::Vector{Index{Int64}}, depth::Int64)
```

与えられた深さのランダム回路を作成します。この関数は、回路内のゲートを表すITensorのベクターを返します。

  * `depth == 0` の場合、各サイトに対して単一量子ビットのランダムユニタリが適用されます。
  * `depth > 0` の場合、回路は層ごとに構築されます：

      * 奇数層では、サイト1-2、3-4などにランダムな二量子ビットゲートが適用されます。
      * 偶数層では、サイト2-3、4-5などにランダムな二量子ビットゲートが適用されます。

# 引数

  * `ξ::Vector{Index{Int64}}`: 量子ビットのためのサイトインデックスのベクター。
  * `depth::Int64`: 回路の深さ（非負整数）。

# 戻り値

ランダム回路ゲートを表すITensorのベクター。

# 例

```julia
circuit = random_circuit(siteinds("Qubit", 10), 3)
```
