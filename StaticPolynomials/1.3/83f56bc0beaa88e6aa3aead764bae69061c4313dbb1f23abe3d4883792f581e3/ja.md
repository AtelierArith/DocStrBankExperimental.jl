```
@evalpoly_derivative(z, c...)
```

多項式 $\sum_k c[k] z^{k-1}$ を評価し、その導関数を求めます。すなわち、係数は `z` の冪に従って昇順で与えられます。このマクロは、ホーナー法または、複素数 `z` の場合はより効率的なゴートゼル類似のアルゴリズムを使用する効率的なインラインコードに展開されます。

```jldoctest
julia> @evalpoly(3, 1, 0, 1)
(10, 6)

julia> @evalpoly(2, 1, 0, 1)
(5, 4)

julia> @evalpoly(2, 1, 1, 1)
(7, 5)
```
