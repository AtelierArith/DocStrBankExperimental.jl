```
MinesightAngles(θ₁, θ₂, θ₃)
```

右手系に従ったMineSight z-x'-y''の内因的回転規則。すべての角度は度単位で、符号規則はCW、CCW、CWが正です。

最初の回転 `θ₁` はZ軸周りの水平回転で、正の方向は時計回りです。2番目の回転 `θ₂` は新しいX軸周りの回転で、Y軸を所望の位置に移動させ、回転の正の方向は上向きです。3番目の回転 `θ₃` は新しいY軸周りの回転で、X軸を所望の位置に移動させ、回転の正の方向は上向きです。
