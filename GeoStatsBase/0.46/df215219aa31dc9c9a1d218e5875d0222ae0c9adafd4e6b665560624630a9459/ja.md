```
GslibAngles(θ₁, θ₂, θ₃)
```

GSLIB z-x'-y'' 内因回転規則は左手の法則に従います。すべての角度は度単位で、符号規則はCW、CCW、CWが正です。Yは主軸です。

最初の回転 `θ₁` はZ軸周りの回転で、これは方位角とも呼ばれます。2番目の回転 `θ₂` は新しいX軸周りの回転で、これは傾斜とも呼ばれます。3番目の回転 `θ₃` は新しいY軸周りの回転で、これは傾きとも呼ばれます。
