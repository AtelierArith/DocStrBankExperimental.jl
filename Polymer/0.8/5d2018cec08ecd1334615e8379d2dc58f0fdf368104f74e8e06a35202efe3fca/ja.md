```
PolymerParameter{T, R} <: AbstractParameter
```

型はポリマーシステムの物理パラメータを表します。特定のT（通常はシンボル）を持つ各型は、パラメータを定義します。例えば、

````
```julia
PolymerParameter{:ϕ} # ϕパラメータを定義
```
````

事前定義されたパラメータシンボル：

  * χ: フローリー・ハギンズ相互作用パラメータ。
  * N: モノマーまたはクーンセグメントの数における鎖の長さ。
  * f: ブロックコポリマーにおけるブロックの体積分率。
  * ϕ: ポリマーシステムにおける鎖の種類の体積分率。
  * Rg: ポリマー鎖の回転半径。
  * C: 鎖密度 C = ρ₀Rg^3/N。
  * b: モノマーまたはクーンセグメントの長さ。
  * α: 参照鎖に対する鎖の長さ。
  * τ: 2つ以上のブロックの長さの比。

フィールド

  * `description`: パラメータの説明。
  * `variable_name`: Juliaの変数名として使用できるパラメータの文字列表現。
  * `ascii_label`: ファイル名、ディレクトリ名などに使用される純粋なascii文字のラベル。
  * `plot_label`: プロット用のラベルとして使用されるLaTeXStrings。
  * `value_type`: パラメータの数値型（例：整数、浮動小数点、複素数など）。
  * `allowed_min`: 許可される最小値。負の無限大の場合は、`typemin(Type)`を使用。
  * `allowed_max`: 許可される最大値。正の無限大の場合は、`typemax(Type)`を使用。
  * `allowed_values`: 離散的な有限集合にのみ使用。許可される値が無限の場合は、代わりに`allowed_min`と`allowed_max`を使用。
  * `disallowed_values`: 無効な値の配列。

ユーザーは、次の2つのフィールド値を定義する限り、自分自身のパラメータシンボルを定義できます。他のフィールドは、パラメトリック型 `T` から自動的に推測されます。

  * `description`
  * `plot_label`
