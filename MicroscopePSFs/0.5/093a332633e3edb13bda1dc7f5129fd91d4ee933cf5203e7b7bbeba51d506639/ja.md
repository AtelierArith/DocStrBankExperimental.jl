```
(psf::AbstractPSF)(x, y)
```

位置 (x,y) における正規化された PSF 強度を PSF 中心に対して評価します。

# 引数

  * `x, y`: PSF 中心に対するマイクロメートル単位の位置

# 戻り値

  * 1 に統合されるように正規化された強度

# 座標

入力座標 (x,y) は PSF 中心に対する物理単位（マイクロメートル）です。

# 例

```julia
psf = AiryPSF(1.4, 0.532)  # NA=1.4, λ=532nm
intensity = psf(0.1, 0.2)  # (0.1μm, 0.2μm) で評価
```
