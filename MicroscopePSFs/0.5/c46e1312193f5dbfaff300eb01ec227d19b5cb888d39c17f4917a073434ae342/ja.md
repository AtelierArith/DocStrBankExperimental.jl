```
amplitude(psf::VectorPSF, x::Real, y::Real, z::Real)
```

指定された位置での複素ベクトル振幅を計算します。

# 引数

  * `psf`: ベクトルPSFインスタンス
  * `x, y`: PSF中心に対するマイクロメートル単位の横方向位置
  * `z`: カバーガラスの上の深さを表すマイクロメートル単位の軸方向位置

# 戻り値

  * 複素場振幅のベクトル [Ex, Ey]

# 注意事項

  * z座標はカバーガラスの上の深さを表します
  * PSFのz_stageは、ステージが名目焦点面からどれだけ移動したかを示します
  * UAFおよびSAFの寄与を自動的に含みます
  * 回転する双極子には意味がなく（PSFに複数の開口がある場合はエラーをスローします）
