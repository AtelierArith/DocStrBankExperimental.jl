```
ZernikeCoefficients{T<:Real}
```

可変構造体で、瞳関数の大きさと位相のZernike係数を保持します。インデックス1から始まるNollインデックス規則を使用します。

# フィールド

  * `mag::Vector{T}`: 大きさの係数（Noll規則に従って1インデックス）
  * `phase::Vector{T}`: 位相の係数（Noll規則に従って1インデックス）

# 注意事項

  * 最初の係数（インデックス1）は通常、ピストンを表します
  * 大きさの係数は通常、mag[1] = 1で正規化されています
  * 位相の係数はラジアンでの位相を表します
  * RMS正規化（Noll規則）が使用されているため、係数はRMS波面誤差を直接表します
