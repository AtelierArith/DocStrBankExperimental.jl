```
cheb_lobatto_quadrature_weights([TF=Float64], n::Integer) where {TF<:AbstractFloat}
```

Compute quadrature weights for Chebyshev points of the 2nd kind.

# Arguments

  * `TF`: Type parameter for the weights (e.g., Float64)
  * `n`: Number of points
