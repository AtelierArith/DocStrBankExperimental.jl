```
dihedral(v1, v2, v3, v4; degrees=true)
dihedral(v::AbstractVector; degrees=true)
```

ベクトル `v1-v2` と `v2-v3`、および `v2-v3` と `v3-v4` によって形成される平面間の二面角を計算します。入力ベクトルは3つの要素を持っている必要があります。この関数は、ラジアンまたは度で二面角を返します。入力が4つのベクトルを持つベクトルの場合、関数はベクトル `v[1]-v[2]` と `v[2]-v[3]`、および `v[2]-v[3]` と `v[3]-v[4]` によって形成される平面間の二面角を計算します。

オプションの引数 `degrees` は、出力が度（デフォルト）またはラジアンであるかを指定します。
