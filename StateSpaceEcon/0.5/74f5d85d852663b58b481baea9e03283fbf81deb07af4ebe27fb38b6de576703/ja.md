```
simulate(model, plan, data; <options>)
```

与えられたモデル、シミュレーションプラン、および外生データのシミュレーションを実行します。

### 引数

```
* `model` - シミュレーションする[`Model`](@ref ModelBaseEcon.Model)インスタンス。
* `plan` - シミュレーションのための[`Plan`](@ref)。
* `data` - 外生データを含む2D `Array`。これには初期条件と最終条件が含まれます。
```

### キーワード引数としてのオプション

```
* `fctype::`[`FinalCondition`](@ref) - シミュレーションのための希望する最終条件タイプを設定します。デフォルト値は[`fcgiven`](@ref)です。他の可能な値には[`fclevel`](@ref)、[`fcslope`](@ref)、および[`fcnatural`](@ref)が含まれます。
* `initial_guess::AbstractMatrix{Float64}` - 解の初期推測を含む2D `Array`。これはニュートン-ラフソンアルゴリズムを開始するために使用されます。デフォルト値は空の配列（`zeros(0,0)`）であり、その場合は初期条件に外生データを使用します。定常状態の解を使用するには[`steadystatearray`](@ref)を使用できます。
* `deviation::Bool` - `data`が定常状態からの偏差で与えられている場合は`true`に設定します。この場合、シミュレーション結果も定常状態からの偏差として返されます。デフォルト値は`false`です。
* `anticipate::Bool` - すべてのショックがエージェントによって予測されていないことをソルバーに指示するために`false`に設定します。デフォルト値は`true`です。
* `solver::Symbol` - シミュレーションソルバーを指定します。利用可能なオプションは:stackedtimeと:firstorderです。指定しない場合、デフォルトは:stackedtimeです。
* `verbose::Bool` - 進行状況情報を印刷するかどうかを制御します。デフォルト値は`model.options`から取得されます。
* `tol::Float64` - 希望する精度を設定します。デフォルト値は`model.options`から取得されます。
* `maxiter::Int` - 希望する精度がこの最大反復回数内に達しない場合、アルゴリズムは失敗します。デフォルト値は`model.options`から取得されます。
```

次のオプションは`:stackedtime`ソルバーに特有です。  

  * `sim_solver` - 使用する非線形ソルバーを指定します。利用可能なオプションは次のとおりです。  

      * `:sim_nr` : （デフォルト）ニュートン-ラフソン、可能な減衰を含む、以下を参照。
      * `:sim_lm` : レヴェンバーグ–マルクワルト
      * `:sim_gn` : ガウス-ニュートン
  * `linesearch::Bool` - `true`の場合、ニュートン-ラフソンはfの十分な減少のための降下方向に沿った探索を含むように修正されます。これは各反復で行われます。デフォルトは`false`です。（以下に説明する`damping`オプションによって上書きされます）
  * `damping` - ニュートン非線形ソルバーに適用できる減衰のスタイルを指定します。利用可能なオプションは次のとおりです。  

      * 指定しない場合、デフォルトの動作は減衰なし、すなわち各反復で減衰係数は1.0に設定されます。
      * 数値：減衰係数は指定された数値（1ではなく）に設定されます。
      * 数値のベクトル：各反復での減衰係数は、指定されたベクトルの対応するエントリの数値に設定されます。ニュートンの反復回数がベクトルの長さを超える場合、残りの反復では最後のエントリが使用されます。
      * `:linesearch`または`:armijo` : `linesearch=true`を設定するのと同じです。Armijoルールは「C.T.Kelly, Iterative Methods for Linear and Nonlinear Equations, ch.8.1, p.137」から取られています。
      * `(:armijo, :sigma => 0.5, :alpha => 1e-4)` - Armijoルールのデフォルトパラメータを上書きします。
      * `:br81` : （実験的）「Bank, R.E., Rose, D.J. Global approximate Newton methods. Numer. Math. 37, 279–295 (1981)」の減衰アルゴリズムを実装します。
      * `(:br81, :rateK => 10, :delta => 0.1)` : Bank & Rose (1981)アルゴリズムのデフォルトパラメータを上書きします。
