```
SoftDot
```

SoftDotノードは、ドット積演算子デルタノードの代替として使用できます（出力変数は他の2つのドット積です）。これは、次のようにガウスノイズを追加することでデルタ制約を緩和します：

```
y ~ N(dot(θ, x), γ^(-1))
```

インターフェース：

1. y - "ソフト"ドット積の結果、
2. θ - 乗算される最初の変数、
3. x - 乗算される2番目の変数、
4. γ - ガウスノイズの精度。

SoftDotを使用する利点は、信念伝播と変分メッセージ伝播の両方に対して、扱いやすく最適化された閉形式の変分メッセージを提供することです。
