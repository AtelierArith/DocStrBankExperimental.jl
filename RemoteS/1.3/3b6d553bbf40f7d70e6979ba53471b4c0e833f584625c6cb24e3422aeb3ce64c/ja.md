```
findscenes(lon::Real, lat::Real; kwargs...)
```

位置点 `lon`, `lat` をカバーするシーンの名前を、kwargsで指定された日付と衛星によって決定された期間内で見つけます。

  * `day`: 軌道の昼間部分のみを検索します。
  * `night`: 軌道の夜間部分のみを検索します。
  * `oc`: AQUAまたはTERRA衛星の場合、クロロフィル含量のシーンのみを選択します。
  * `sst`: AQUAまたはTERRA衛星の場合、海面温度含量のシーンのみを選択します。
  * `sat`, `SAT` または `satellite`: 使用する衛星の名前; (文字列またはシンボル) から選択: :TERRA, :AQUA
  * `start`: 検索期間の開始を指定するDateTimeオブジェクトまたは`DateTime(start)`で変換可能な文字列。省略した場合、UTCの現在時刻が使用されます。
  * `duration`: シーンが検索される期間の長さ。期間は日数で期待され、負の数であれば`start`からその日数を遡って検索します。
  * `stop`: `duration`の代わりに検索の終了日を提供します。`start`と同じ条件です。
  * `tle` または `TLE`: 特定の衛星と期間のTLEデータを含むファイル名。TLEファイルの1行目と2行目を含む2要素の文字列ベクターでも構いません。

### 戻り値

シーン名の文字列ベクター

## 例:

「2021-09-07T17:00:00」の2日前にポイント(-8, 36)をカバーするクロロフィル-a (oceancolor)のAQUAシーンを見つけます。これは2021年9月の正確なものであることに注意してください。他の日付には更新されたTLEが必要です。

```
tle1 = "1 27424U 02022A   21245.83760660  .00000135  00000-0  39999-4 0  9997";
tle2 = "2 27424  98.2123 186.0654 0002229  67.6025 313.3829 14.57107527 28342";
findscenes(-8,36, start="2021-09-07T17:00:00", sat=:aqua, day=true, duration=-2, oc=1, tle=[tle1, tle2])

2-element Vector{String}:
"A2021251125500.L2_LAC_OC.nc"
"A2021252134000.L2_LAC_OC.nc"
```
