```
cal_Rnl(Tmax, Tmin, ea, cld)
cal_Rnl(Tmax, Tmin, ea, Rsi, Rsi_o)
```

ネット放射長波放射。

# 引数

  * `Tmax`: 2mの高さでの1日の最大気温（摂氏）。
  * `Tmin`: 2mの高さでの1日の最小気温（摂氏）。
  * `ea`: 実際の水蒸気圧（kPa）。最大または最小気温と平均相対湿度から推定できます。
  * `Rsi`: 作物表面での入射短波放射（MJ m-2 day-1）。デフォルトは`nothing`。
  * `Rsi_o`: 晴天時の入射短波放射、すなわち、地球外放射を晴天時の透過率（すなわち、a + b、aとbはアングストローム式の係数。通常は0.75）で掛けたもの（MJ m-2 day-1）。デフォルトは`nothing`。
  * `cld`: 雲量の割合。提供された場合、Rsoよりもネット放射長波放射を計算するために直接使用されます。デフォルトは`nothing`。

# 戻り値

  * `W m-2`でのネット放射長波放射。
