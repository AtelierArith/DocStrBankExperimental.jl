```
ET0_eq(Rn::T, Tair::T, Pa::T=atm, args...)
```

平衡蒸発、`ET0_eq = Δ / (Δ + γ) * Rn`

# 引数

  * `Rn`   : 正味放射 (W m-2)
  * `Tair` : 2mの空気温度 (degC)
  * `VPD`  : 水蒸気圧不足 (kPa)
  * `Pa`   : 表面気圧 (kPa)

# 戻り値

  * `λ` : 蒸発潜熱 (MJ kg-1)
  * `Δ`  : 飽和水蒸気圧曲線の傾き (kPa degC-1)
  * `γ`  : サイクロメトリック定数 (kPa degC-1)
  * `Eeq`    : 平衡蒸発率 (mm day-1)

# オプションのキーワード引数

  * `args...` : 追加の引数 (この関数では使用されません)

# 例

```julia
julia> ET0_eq(200.0, 20.0)
(2.4536, 0.14474018811241365, 0.06725596686893338, 4.808405926729265)
```
