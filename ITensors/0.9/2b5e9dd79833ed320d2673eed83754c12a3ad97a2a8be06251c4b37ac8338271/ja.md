```
nullspace(T::ITensor, left_inds...; tags="n", atol=1E-12, kwargs...)
```

ITensor `T`を行空間として提供された`left_inds`、残りのインデックスを右インデックスまたは列空間として見た場合、`nullspace`関数は右のヌル空間を計算します。つまり、`T*N`がゼロになるように、`T`の右インデックスに作用するテンソル`N`を返します。返されるテンソル`N`は、ヌル空間の「ベクトル」をインデックスするラベル「n」を持つ新しいインデックスも持ちます。

例えば、`T`がインデックス`i,j,k`を持つ場合、`N = nullspace(T,i,k)`を呼び出すと、インデックス`j`を持つ`N`が返され、次のようになります。

```
       ___       ___
  i --|   |     |   |
      | T |--j--| N |--n  ≈ 0
  k --|   |     |   |
       ---       ---
```

インデックス`n`は、`n = uniqueindex(N,T)`を呼び出すことで取得できます。

この関数の実装は将来的に変更される可能性があるため、特定のヌル空間のサイズを与える正確な`atol`値は、ITensorの将来のバージョンで変更される可能性があります。

キーワード引数:

  * `atol::Float64=1E-12` - この値未満のT†*Tの特異値がヌル空間を定義します
  * `tags::String="n"` - ヌル空間の要素を選択するインデックスのタグを選択します
