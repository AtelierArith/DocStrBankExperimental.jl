ITensorは、正確で効率的なテンソルネットワークアルゴリズムを迅速に作成するためのライブラリです。

ITensorは、そのインターフェースがメモリレイアウトに依存しないテンソルです。ITensorのインデックスは「インテリジェント」であり、追加情報を持ち、ITensorを収束または加算する際に自動的にお互いを「認識」します。

ITensorライブラリには、行列積状態や行列積演算子など、テンソルネットワークを最適化および変換するための合成可能で拡張可能なアルゴリズムが含まれています。

# 使用例

テンソルインデックスiとjを定義します

```
i = Index(2, "i")
j = Index(3, "j")
```

これらのインデックスを持つITensorを作成します

```
A = ITensor(i,j)
```

i==2,j==1の要素を-2.6に設定します

```
A[j=>1,i=>2] = -2.6
A[i=>2,j=>1] = -2.6 #これには同じ効果があります
```

ランダムな要素を持つITensorを作成します

```
B = random_itensor(j,i)
```

ITensor AとBを一緒に加算します（異なる順序のインデックスでも大丈夫です）

```
C = A + B
```

# ITensorのその他の機能

  * 行列積状態（MPS）/ テンソルトレイン（TT）などの**テンソルネットワーク**のためのツール
  * MPS形式の線形方程式を解くための**アルゴリズム**（DMRGなど）や、微分方程式を統合するための「時間発展MPS」
  * ITensorは、ブロックスパース性や対角スパース性などの**スパースデータ**を内部に持つことができ、密なITensorと同じインターフェースを持ちます
  * ITensorは、インデックスの群変換に対して**対称性特性**（不変性または同変性）を持つことができます。物理学用語では、そのようなITensorは量子数を保存します。
