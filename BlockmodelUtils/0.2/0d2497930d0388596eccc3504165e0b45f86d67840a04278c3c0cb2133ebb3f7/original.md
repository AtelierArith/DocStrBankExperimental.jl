```
permuteplot!(ax, bm)
permuteplot(bm)
```

Plot the permuted adjacency matrix with a grid outlining the blockmodel blocks.

# Attributes

  * `cellcolor = :black`
  * `linecolor = :black`
  * `framecolor = linecolor`
  * `rotate_xlabels = fale`
  * `xticklabels = bm.labels`
  * `yticklabels = bm.labels`
