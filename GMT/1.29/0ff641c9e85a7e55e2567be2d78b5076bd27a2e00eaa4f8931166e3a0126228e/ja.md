```
ds = gmt2gd(GI)
```

GIの内容から、グリッドまたは画像としてGDALデータセットを作成します。

```
ds = gmt2gd(D, save="", geometry="")
```

Dの内容からGDALデータセットを作成します。DはGMTdataset、GMTdatasetのベクトル、またはMxN配列であることができます。`save`キーワードはGDALに内容をOGRファイルとして保存するよう指示します。フォーマットはファイル拡張子によって決まります。`geometry`は「polygon」という文字列で、ファイルがDが単一またはマルチセグメントオブジェクトであるかに応じてポリゴン/マルチポリゴンに変換されるか、または「point」でマルチポイントジオメトリに変換されます。
