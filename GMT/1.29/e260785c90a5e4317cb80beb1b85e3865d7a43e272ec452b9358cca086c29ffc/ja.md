```
pcolor(X, Y, C::Matrix{<:Real}; kwargs...)
```

行列 `C` の値を使用して色付きセルプロットを作成します。各セルの色は、色テーブル (cpt) を参照した後の `C` の各値に依存します。オプション `cmap=xxx` を介して色テーブルが提供されていない場合、デフォルトのものを計算します。

### 引数

  * `X`, `Y`: 頂点の x および y 座標を持つベクトルまたは 1 行の行列。`X` の要素数は `C` の列数と一致する必要があります (グリッド登録モデルを使用している場合) または 1 つ多い必要があります (ピクセル登録)。`Y` と `C` の行数についても同様です。`X` と `Y` は等間隔である必要はありません。
  * `X`, `Y`: 頂点の x および y 座標を持つ行列。この場合、`X` と `Y` が m-by-n グリッドを定義する場合、`C` は (m-1)-by-(n-1) 行列である必要がありますが、m-by-n であることも許可されており、その場合は `C` から最後の行と列を削除します。
  * `C`: セルの色付けに使用される値を持つ行列。

### キーワード引数

この形式の `pcolor` は実際には $plot$ のラップであるため、そのモジュールの任意のオプションをここで使用できます。

  * `labels`: この $keyword$ が使用される場合、対応するセル内の各ノードの値をプロットします。`label=n` を使用し、$n$ は整数で印刷される小数点以下の桁数を表します。$true$、$"y"$ または $:y$ のような他の値は、プログラムに小数点以下の桁数を推測させます。
  * `font`: `label` が使用されると、テキストフォント設定も制御できます。オプションは $text$ の `attrib` オプションのサブセットです。具体的には、角度と $font$ です。例: $font=(angle=45, font=(5,:red))$。指定されていない場合、デフォルトは $font=(font=(6,:black),)$ です。

---

```
D = pcolor(X, Y; kwargs...)
```

この形式は、色行列なしで、以前と同様に `X` と `Y` を受け入れますが、GMTdatasets のベクトル内のタイルを返します。例えば、投影設定を渡すために `kwargs` オプションを使用します (例えば $proj=:geo$)。

---

```
pcolor(G::GMTgrid; kwargs...)
```

この形式は、グリッド (またはそのファイル名) を入力として受け取り、一定の色でセルを塗ります。

  * `outline`: タイルのアウトラインを描画し、デフォルトのペンが気に入らない場合はカスタムペンを指定します。
  * `kwargs`: この形式の `pcolor` は $grdview$ のラップであるため、そのモジュールの任意のオプションをここで使用できます。例えば、$grdview$ の $tiles$ オプションを介してタイルオプションを制御できます。

---

```
pcolor(GorD; kwargs...)
```

`GorD` が GMTgrid または $GMT.cor()$ で得られたピアソン相関行列を含む GMTdataset のいずれかである場合、処理は特別な扱いを受けます。この場合、`labels` キーワードの他に、ユーザーは自動的に選択された x 注釈の角度が正しいかどうかを確認することにも興味があります。そうでない場合は、`rotx` (または `slanted`) キーワードを設定することで強制できます。

### 例

```julia
    # 例のグリッドを作成
	G = GMT.peaks(N=21);

	pcolor(G, outline=(0.5,:dot), show=true)

	# 非正規形式の G の x,y 座標を使用
	pcolor(G.x, G.y, G.z, show=true)

	# デフォルト設定 (フォントサイズ = 6p) を使用してセルにラベルを追加
	pcolor(G.x, G.y, G.z, labels=:y, show=true)

	# 上記と似ていますが、ラベルの小数点以下の桁数とフォント設定も設定
	pcolor(G.x, G.y, G.z, labels=2, font=(angle=45, font=(5,:red)), show=1)

	# 不規則なグリッド
	X,Y = meshgrid(-3:6/17:3);
	XX = 2*X .* Y;	YY = X.^2 .- Y.^2;
	pcolor(XX,YY, reshape(repeat([1:18; 18:-1:1], 9,1), size(XX)), lc=:black, show=true)
```

ピアソンの相関行列を表示

```julia
pcolor(GMT.cor(rand(4,4)), labels=:y, colorbar=1, show=true)
```
