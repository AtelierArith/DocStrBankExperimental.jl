```
grdtrend(cmd0::String="", arg1=nothing, arg2=nothing; kwargs...)
```

2次元グリッドファイルを読み込み、これらのデータに低次の多項式トレンドを[オプションで重み付けされた]最小二乗法でフィットします。

## パラメータ

  * **N** | **model** :: [Type => Str | Number]

    フィットするモデルパラメータの数を設定します。
  * **D** | **diff** :: [Type => Str | []]

    差分（入力データ - トレンド）を計算します。結果をディスクに保存するためのファイル名をオプションで提供できます。
  * **T** | **trend** :: [Type => Str | []]

    トレンドサーフェスを計算します。結果をディスクに保存するためのファイル名をオプションで提供できます。
  * **W** | **weights** :: [Type => Str]

    weight.ncが存在する場合、それが読み込まれ、重み付き最小二乗問題を解くために使用されます。
  * **R** | **region** | **limits** :: [Type => Str or list or GMTgrid|image]	$Arg = (xmin,xmax,ymin,ymax)$

    興味のある領域を指定します。提供されていない場合はデータの最小バウンディングボックスに設定されます。
  * **V** | **verbose** :: [Type => Bool or Str]		$Arg = [level]$

    進行状況レポートをstderrに送信する冗長性レベルを選択します。

完全なドキュメントを見るには、次のように入力してください: $@? grdtrend$
