```
FV = cylinder(r, h; base=0.0, center=(0.0, 0.0, 0.0), geog=false, unit="m", np=36) -> GMTfv
```

半径 `r` と高さ `h` の円柱を作成します。

### 引数

  * `r`: 円柱の半径。地理的円柱の場合、デフォルトはメートルです。ただし、以下の `unit` を参照してください。
  * `h`: 円柱の高さ。`r` と同じ単位である必要があります。

### キーワード引数

  * `base`: 円柱の基底の高さ。デフォルトは 0 です。
  * `center`: 体の原点を定義する3要素の配列またはタプル。デフォルトは `(0.0, 0.0, 0.0)` です。
  * `closed`: true の場合（デフォルト）、円柱の上部と下部を閉じます。
  * `geog`: true の場合、地理座標で円柱を作成します。
  * `unit`: 地理的円柱のみ。半径がメートルでない場合は、`unit=:km`、`unit=:Nautical`、または `unit=:Miles` のいずれかを使用します。
  * `np`: 円の頂点の数。デフォルトは 36 です。

Faces-Vertices データセットを返します。

### 例

```julia
	FV = cylinder(50, 100)
	viz(FV)
```
