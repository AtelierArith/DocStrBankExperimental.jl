```
era5time(; year="", month="", day="", hour="") -> String
```

CDS ERA5データセットから1つ以上の日付時刻を選択します。

この関数は、$ecmwf()$関数の`params`オプションに入力として使用できるJSON形式の文字列を返します。

### Kwargs

  * `year`: 選択する年。ユニークな年を選択するための文字列、または複数の年を選択するための文字列またはIntのベクターで指定できます。また、年の範囲（例："2010:2020"）を指定することもできます。
  * `month`: 選択する月。ユニークな月を選択するための文字列、または複数の月を選択するための文字列またはIntのベクターで指定できます。また、月の範囲（例："01:06"）を指定することもできます。
  * `day`: 選択する日。ユニークな日を選択するための文字列、または複数の日を選択するための文字列またはIntのベクターで指定できます。また、日の範囲（例："01:20"）を指定することもできます。
  * `hour`: 選択する時間。ユニークな時間を選択するための文字列、または複数の時間を選択するための文字列またはIntのベクターで指定できます。また、時間の範囲（例："01:10"）を指定することもできます。

### Returns

JSON形式の日付時刻を含む文字列。

### Example

```julia
# 2023年のすべての時刻
var = era5time(year="2023")
```
