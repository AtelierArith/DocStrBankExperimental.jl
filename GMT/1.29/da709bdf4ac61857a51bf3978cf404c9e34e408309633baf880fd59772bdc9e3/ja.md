```
function loxodrome_inverse(lon1, lat1, lon2, lat2, a=6378137.0, f=0.0033528106647474805)
```

楕円体上のロクソドロームの逆問題を計算します。

楕円体上のP1とP2の緯度と経度が与えられたとき、ロクソドロームP1P2の方位a12と、ロクソドローム曲線に沿った弧の長さsを計算します。

Args:

  * `lon1, lat1, lon2, lat2`: - 開始点と終了点の経度と緯度（度）。
  * `a` - 楕円体の長軸（メートル）。WGS84のデフォルト値
  * `f` - 楕円体の扁平率（デフォルト = 1 / 298.257223563）

### Returns

  * P1からP2までの距離（メートル）と方位

## 例: 点(0,0)と(5,5)の間の距離と方位を計算する

```
dist, azim = loxodrome_inverse(0,0,5,5)
```
