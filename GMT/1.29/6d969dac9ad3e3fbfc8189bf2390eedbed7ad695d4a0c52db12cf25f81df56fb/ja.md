```
era5vars(varID; single::Bool=true, pressure::Bool=false) -> String
```

CDS ERA5データセットから1つ以上の変数を選択します。

この関数は、$ecmwf()$関数の`params`オプションに入力として使用できるJSON形式の文字列を返します。利用可能な変数のリストについては、$listecmwfvars()$関数を参照してください。

### 引数

  * `varID`: 変数名。ユニークな変数を選択するための文字列またはシンボル、または複数の変数を選択するための文字列/シンボルのベクターです。

### キーワード引数

  * `single`: trueの場合、単一レベルの変数のみがリストされます。falseの場合、圧力レベルの変数がリストされます [デフォルトはtrue]。
  * `pressure`: trueの場合、圧力レベルの変数のみがリストされます。falseの場合、単一レベルの変数がリストされます [デフォルトはfalse]。

### 戻り値

JSON形式の変数名を含む文字列。

### 例

```julia
# "t2m"は2mの温度で、"skt"は地表温度です
var = era5vars(["t2m", "skt"])
```
