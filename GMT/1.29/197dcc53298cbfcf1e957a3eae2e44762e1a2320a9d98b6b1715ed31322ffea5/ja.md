```
D = mat2ds(mat::Vector{<:AbstractMatrix}; hdr=String[], kwargs...)::Vector{GMTdataset}
```

ベクトル `mat` に渡された行列からマルチセグメント GMTdataset（GMTdataset のベクトル）を作成します。`mat` の行列要素は同じ行数である必要はありません。これは、同じ設定を共有する線/点のグループを指定することを考えてください。この形式の KWarg オプションは一般的なケースよりも数が限られていますが、Vector{Vector}、Vector またはスカラーの形式を取ることができます。前者の場合（Vector{Vector}）、各 Vector[i] の長さは各 mat[i] の行数と等しくなければなりません。

  * `hdr`: 1つまたは `length(mat)` のマルチセグメントヘッダーを持つオプションの String ベクトル。
  * `pen`: フルペン設定。`length = length(mat)` のペン設定を持つ文字列または文字列の配列。
  * `lc` または `linecolor` または `color`: オプションの色または色名/値の文字列/シンボルの配列。
  * `linethick` または `lt`: 異なる線の太さを選択するためのもの。`color` のように機能しますが、数値のベクトルである必要があります。または、すべての線に適用される単一の数値である必要があります。
  * `ls` または `linestyle`: 線のスタイル。`length = length(mat)` の線スタイルを持つ文字列または文字列の配列。
  * `front`: フロントラインスタイル。`length = length(mat)` のフロントラインスタイルを持つ文字列または文字列の配列。
  * `fill`: 色名または "パターン" の配列を持つオプションの文字列配列（またはカンマ区切りの色名の文字列、または色名のタプル）。
  * `fillalpha`: `fill` オプションが使用されている場合、このオプションを使用して、[0-1] または ]1-100] の間の数値を持つ配列（ベクトルまたは1行の行列）で、塗りつぶされたポリゴンやシンボルの透明度を設定できます。ここで、100（または1）は完全な透明度を意味します。

### 例:

D = mat2ds([rand(6,3), rand(4,3), rand(3,3)], fill=[[:red], [:green], [:blue]], fillalpha=[0.5,0.7,0.8]) ```
