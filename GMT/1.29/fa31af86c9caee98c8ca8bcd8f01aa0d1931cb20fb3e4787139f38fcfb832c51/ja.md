```
blockmean(cmd0::String="", arg1=nothing; kwargs...)
```

L2ノルムによるブロック平均 (x,y,z) データテーブル。

## パラメータ

  * **R** | **region** | **limits** :: [Type => Str or list or GMTgrid|image]	$Arg = (xmin,xmax,ymin,ymax)$

    興味のある領域を指定します。提供されていない場合はデータの最小バウンディングボックスに設定されます。
  * **I** | **inc** | **increment** | **spacing** :: [Type => Str]	$Arg = xinc[unit][+e|n][/yinc[unit][+e|n]]]$

    *x_inc* [およびオプションで *y_inc*] はグリッド間隔です。オプションで、増分単位を追加できます。
  * **A** | **field** | **fields** :: [Type => Str]

    個別のグリッドに書き込むフィールドを選択します。利用可能なフィールドのカンマ区切りコードを追加します: **z** (平均データz、ただし**statistic**を参照)、**s** (標準偏差)、**l** (最小値)、**h** (最大値) および **w** (出力重み; **weights**が必要)。 [デフォルトは**z**のみ]。
  * **C** | **center** :: [Type => Bool]

    ブロックの中心を出力位置として使用します [デフォルトは平均位置を使用]。**-A**を使用している場合は使用されません。
  * **E** | **extend** | **extended** :: [Type => Str | []]

    各ブロックの平均に関する標準偏差s、最小値l、および最大値hを含む拡張レポートを提供します。出力順序はx,y,z,s,l,h[,w]になります。[デフォルトはx,y,z[,w]を出力します。-Wのw出力を参照してください。-Epが使用される場合、重みは1/(sigmaの二乗)と仮定し、sは平均の伝播誤差になります。
  * **G** | **save** | **outgrid** | **outfile** :: [Type => Str]

    1つ以上のフィールドをディスク上のグリッドに直接書き込みます; テーブルデータは返されません。**A**を介して複数のフィールドが指定されている場合、grdfileはフィールドコードをファイル名に埋め込むためにフォーマットフラグ%sを含む必要があります。提供されていないが**A**が使用されている場合、1つ以上のGMTgridタイプを返します。
  * **S** | **statistic** :: [Type => Str | Symb] 

    `statistic=:n`を使用して各ブロック内のポイント数を報告し、`statistic=:s`を使用してブロック内のすべてのz値の合計を報告し、`statistic=:w`を使用して重みの合計を報告します。[デフォルト（または`statistic=:m`）は平均値を報告します]。
  * **mean** :: [Type => Any] 

    各ブロック内のポイントの平均値を報告します。
  * **npts** | **counts** :: [Type => Any] 

    各ブロック内のポイント数を報告します。
  * **sum** :: [Type => Any] 

    各ブロック内のすべてのz値の合計を報告します。
  * **sum_weights** :: [Type => Any] 

    重みの合計を報告します。
  * **grid** :: [Type => Bool] 

    上記のいずれかのオプション（`statistic`、`npts`、`sum`）を使用すると、このオプションはGMTdatasetの代わりにグリッドを返します。
  * **W** | **weights** :: [Type => Str | []]

    重みなしの入力と出力は3列x,y,zを持ちます; 重み付きi/oは4列x,y,z,wを持ちます。重みは入力に使用して各ブロックの重み付き平均値を構築できます。
  * **yx** :: [Type => Str or Bool or []]     $Arg = [i|o]$

    入力および/または出力で1列目と2列目を入れ替えます。

完全なドキュメントを見るには、次のように入力してください: $@? blockmean$ ```
