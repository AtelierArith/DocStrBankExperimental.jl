```
xcov(x, y, [lags]; demean=true)
```

実数値ベクトルまたは行列 `x` と `y` の間のクロス共分散関数 (CCF) を計算し、オプションで `lags` を指定します。`demean` は、CCF を計算する前に `x` と `y` のそれぞれの平均を引くべきかどうかを指定します。

もし `x` と `y` がベクトルであれば、`lags` と同じ長さのベクトルを返します。それ以外の場合は、`x` と `y` の各列のペア間のクロス共分散を計算します。

### Kwargs

  * `demean`: `x` と `y` のそれぞれの平均を引くべきかどうかを指定します。
  * `lags`: 指定されていない場合、`maxlags=0` のとき、使用されるラグは `-min(size(x,1)-1, 10*log10(size(x,1)))` から `min(size(x,1), 10*log10(size(x,1)))` までの整数です。
  * `maxlags`: ラグ範囲を `-maxlag` から `maxlag` に制限します。
