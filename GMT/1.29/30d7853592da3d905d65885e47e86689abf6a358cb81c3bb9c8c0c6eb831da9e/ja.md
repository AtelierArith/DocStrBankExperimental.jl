```
loxodrome_direct(lon, lat, azimuth, distance, a=6378137.0, f=0.0033528106647474805)
```

楕円体上のロクソドロームの直接問題を計算します。

P1の緯度と経度、ロクソドロームP1P2の方位a12、およびロクソドローム曲線に沿った弧の長さsが与えられたとき、P2の緯度と経度を計算します。

引数:

  * `lon, lat`: - 開始点の経度、緯度（度）。
  * `azimuth`:  - 方位（度）
  * `distance`: - （lat,lon）から移動する距離（メートル）
  * `a` - 楕円体の長軸（メートル）。WGS84のデフォルト値
  * `f` - 楕円体の扁平率（デフォルト = 1 / 298.257223563）

### 戻り値

  * [距離]メートルを[方位]方向に移動した後の目的地の[lons lat]。

## 例: 点0,0から方位45度で10000メートルの終点を計算する

```
loxo = loxodrome_direct(0,0,45, 10000)
```
