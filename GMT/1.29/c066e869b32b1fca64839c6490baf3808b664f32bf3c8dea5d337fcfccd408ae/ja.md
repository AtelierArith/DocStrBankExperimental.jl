```
zvals = polygonlevels(D::GDtype, ids::VecOrMat{String}, vals::Vector{<:Real}; kw...) -> Vector{Float64}
```

`plot`で使用するための`zvals`を持つベクトルを作成し、`length(zvals) == length(D)`であることを確認します。

`zvals`の要素は`vals`から構成されます。

### 引数

  * `D`:      GMTdatasetのベクトル内のデータ。
  * `ids`:    GMTdataset `D`のid（属性名）の文字列ベクトルまたは行列。行列の場合（2列のみ）、以下の`att`も2つの要素を持つ文字列ベクトルでなければならず、`ids`の各行の2つの要素に対して一致させる必要があります。ここでのアイデアは、2つの条件を一致させることです：`att[1] == ids[n,1] && att[2] == ids[n,2]`
  * `vals`:   プロット`level`でポリゴンの色を付けるために使用される数値のベクトル。

### キーワード引数

  * `attrib`または`att`: `ids`文字列の内容と一致させる際に使用する属性を選択します。
  * `nocase`または`insensitive`: `ids`の内容と`attrib`で指定された属性との間で大文字小文字を区別しない比較を行います。デフォルトでは大文字小文字を区別して比較します。
  * `repeat`: ポリゴンに属性がない場合（国の島々に発生する可能性があります）に新しいセグメントIDが見つかるまで、以前に知られている値を複製します。

Dのセグメント数と同じ長さのVector{Float64}を返します。その内容は`vals`の内容から構成されますが、同じ`ids`を持つ同じファミリーの各ポリゴンが同じ値を持つように繰り返される場合があります。
