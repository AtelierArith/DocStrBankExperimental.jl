```
arrows(cmd0::String="", arg1=nothing; arrow=(...), kwargs...)
```

矢印フィールドをプロットします。

キーワード `arrow=(...)` または `vector=(...)` が使用される場合、方向（水平から反時計回りの度数）と長さは列3と4に見つける必要があり、サイズがコマンドラインで指定されていない場合は列5に存在する必要があります。サイズはベクトルヘッドの長さです。ベクトルの幹の幅はオプション `pen` または `line_attrib` によって設定されます。

`vecmap=(...)` のバリエーションは上記と似ていますが、方位（北から東の度数）が方向の代わりに与えられる必要があります。方位は選択した地図投影に基づいて角度にマッピングされます。長さがプロット単位ではなく任意のユーザー単位（例：mm/yrのレート）である場合、対応する列を +sscale 修飾子を介してスケーリングするために *input_col* オプションを使用できます。

`geovec=(...)` または `geovector=(...)` キーワードは地理ベクトルをプロットします。地理ベクトルでは、方位（北から東の度数）と地理的長さは列3と4に見つける必要があります。サイズはベクトルヘッドの長さです。ベクトルの幅は `pen` または `line_attrib` によって設定されます。注意：地理ベクトルの幹は薄い塗りつぶされたポリゴンとして描画されるため、破線や点線のようなペン属性は利用できません。許可される地理的単位については、`units=()` オプションを参照してください。

完全な `arrow` オプションリストは [Vector Attributes](@ref) で確認できます。

  * **B** | **frame** | **axis** | **xaxis yaxis**:: [Type => Str] 

    地図の境界フレームと軸の属性を設定します。
  * **J** | **proj** | **projection** :: [Type => String]

    地図の投影を選択します。デフォルトは15x10 cmの線形（非投影）地図です。
  * **R** | **region** | **limits** :: [Type => Str or list or GMTgrid|image]	$Arg = (xmin,xmax,ymin,ymax)$

    興味のある地域を指定します。提供されていない場合はデータの最小バウンディングボックスに設定されます。
  * **W** | **pen** | **line_attrib** :: [Type => Str]

    線またはシンボルのアウトラインのためのペン属性を設定します。
  * **savefig** | **figname** | **name** :: [Type => Str]

    `figname=name.ext` で図を保存します。ここで `ext` は図のフォーマットを選択します（例：figname="name.png"）。

例:

```
arrows([0 8.2 0 6], limits=(-2,4,0,9), arrow=(len=2,stop=1,shape=0.5,fill=:red), axis=:a, pen="6p", show=true)
```
