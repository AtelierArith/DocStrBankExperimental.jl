```
morse_interval(lc::LefschetzComplex, mvf::CellSubsets,
               ms::CellSubsets)
```

モース集合区間のための孤立不変集合を見つけます。

入力引数 `lc` はレフシェッツ複体を含み、`mvf` は多重ベクトル場を説明します。モース集合のコレクションは `ms` に含まれています。これらの集合はすべて、流れグラフの強連結成分としての意味でモース集合である必要があります。（それにもかかわらず、これは関数内で強制されます！）言い換えれば、`ms` の集合は `morse_sets` 関数を使用して決定されるべきです！

この関数は、モース集合とその接続を含む最小の孤立不変集合を `Vector{Int}` として返します。
