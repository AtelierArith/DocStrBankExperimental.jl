```
create_cubical_complex(cubes::Vector{String}; p::Int=2)
```

キュービカル複体からLefschetz複体を初期化します。`p=0`の場合は有理数上、`p>0`の場合は`GF(p)`上の複体です。

ベクトル`cubes`には、キュービカル複体を定義するために必要なすべての最高次元のキューブのリストが含まれています。各キューブは次のように文字列で表されます：

  * d個の整数、これはd次元ユークリッド空間における点の座標に対応します
  * 点`.`
  * d個の整数0または1、これはそれぞれの次元における区間の長さを示します

最初のd個の整数はすべて同じ文字数を占める必要があります。さらに、各座標に占められるスペースがL文字である場合、座標は0から10^L - 2までの値しか取ることができません。これは、境界演算子が特定の座標に1を加えるため、同じL桁内で表現可能である必要があるからです。

例えば、文字列`030600.101`は三次元における点`(3,6,0)`に対応します。次元は1, 0, 1であり、したがってこの文字列はキューブ`[3,4] x [6] x [0,1]`に対応します。同じキューブは`360.101`や`003006000.101`でも表現できます。

!!! 警告     すべてのラベルは同じ形式である必要があります！

# 例

```jldoctest
julia> cubes = ["00.11", "01.01", "02.10", "11.10", "11.01", "22.00"];

julia> lc = create_cubical_complex(cubes);

julia> lc.ncells
17

julia> homology(lc)
3-element Vector{Int64}:
 2
 1
 0
```
