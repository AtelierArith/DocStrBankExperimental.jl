```
filter_shallow_pairs(lc::LefschetzComplex, phi)
```

フィルターのためのすべての浅いペアを見つけます。

この関数は、フィルター `phi` のためのすべての浅いペアを見つけます。これらは、次元が1異なる面-共面ペア `(x,y)` であり、`y` が `x` のコバウンダリー上で最小のフィルタ値を持ち、`x` が `y` のバウンダリー上で最大のフィルタ値を持つようなペアです。

フィルターが単射である場合、これらのペアは基礎となるレフシェッツ複体上のフォーマンベクトル場を生じます。非単射フィルターの場合、一般にはこれは成り立ちません。
