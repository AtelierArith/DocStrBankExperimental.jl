```
create_cubical_box(nx::Int, ny::Int, nz::Int;
                   p::Int=2, randomize::Real=0.0)
```

空間内のボックスを覆う立方体複体を作成します。複体は `p=0` の場合は有理数上にあり、`p>0` の場合は `GF(p)` 上にあります。

ボックスは空間の部分集合 `[0,nx] x [0,ny] x [0,nz]` によって与えられ、各単位立方体は結果として得られる立方体複体内の三次元立方体を提供します。この関数は以下のオブジェクトを返します：

  * 立方体複体 `cc::LefschetzComplex`
  * 頂点座標のベクトル `coords::Vector{Vector{Float64}}`

オプションのパラメータ `randomize` に正の実数分数 `r` が 0.5 未満で割り当てられた場合、実際の座標はランダム化されます。それらは各頂点を中心とした半径 `r` の球から一様に選ばれます。
