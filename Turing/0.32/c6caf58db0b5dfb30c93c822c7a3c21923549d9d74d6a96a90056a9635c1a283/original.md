The *Poisson distribution* with logarithmic parameterization of the rate parameter describes the number of independent events occurring within a unit time interval, given the average rate of occurrence $exp(log位)$.

The distribution has the probability mass function

$$
P(X = k) = \frac{e^{k \cdot log位}{k!} e^{-e^{log位}}, \quad \text{ for } k = 0,1,2,\ldots.
