```
Gibbs(algs...)
```

構成的MCMCインターフェース。ギブスサンプリングは、モデル内の異なる変数のセットからサンプリングする1つ以上のサンプリングアルゴリズムを組み合わせます。

例：

```julia
@model function gibbs_example(x)
    v1 ~ Normal(0,1)
    v2 ~ Categorical(5)
end

# 'v2'変数にはPGを使用し、'v1'変数にはHMCを使用します。
# v2は離散的であるため、PGサンプラーの方がHMCよりも適切です。
alg = Gibbs(HMC(0.2, 3, :v1), PG(20, :v2))
```

各ギブスコンポーネントの反復回数を次の構文を使用して渡すこともできます：

  * `alg = Gibbs((HMC(0.2, 3, :v1), n_hmc), (PG(20, :v2), n_pg))`

ここで、`n_hmc`と`n_pg`は各ギブス反復のHMCおよびPGの反復回数です。

ヒント：

  * `HMC`と`NUTS`は高速サンプラーであり、Particle Gibbsのような粒子ベースの手法を妨げる可能性があります。粒子サンプリングの効果を高めるために、粒子サンプラーにより多くの粒子を含めることができます。
