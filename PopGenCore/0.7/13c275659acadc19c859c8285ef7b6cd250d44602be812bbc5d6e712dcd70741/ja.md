```
featurematrix(data::PopData, matrixtype::Union{String, Symbol} = "genotype")
```

### 位置引数

```
- `data`: PopDataオブジェクト
- `matrixtype`: `genotype`または`allele`の`String`または`Symbol`（デフォルト: `genotype`）
```

**遺伝子型フィーチャーマトリックス**

サンプルに対応する行と、ロキに対応する列を持つダミーエンコードされた遺伝子型の行列を返します（0,1,2...）。欠損している遺伝子型は`-1`としてエンコードされます。二対立遺伝子のロキにおいて、`0`はアレル1のホモ接合体をエンコードし、`1`はヘテロ接合体をエンコードし、`2`はアレル2のホモ接合体をエンコードします。

**アレルフィーチャーマトリックス**

サンプルに対応する行と、ロキ内のアレルに対応する列を持つダミーエンコードされたアレルの行列を返します。各ロキに対してアレルの数だけ列があります。欠損しているアレル（欠損遺伝子型から）は`-1`としてエンコードされます。
