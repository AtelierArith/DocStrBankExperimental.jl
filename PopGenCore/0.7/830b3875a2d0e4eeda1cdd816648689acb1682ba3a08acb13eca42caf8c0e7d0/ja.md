```
locimatrix(data::PopData)
```

遺伝子型の行列を返します。次元は `samples × loci` です。行はサンプル、列はロキです。サンプル間で倍数が異なる場合はエラーを返します。

**例**

```julia
julia> locimatrix(@nancycats)
237×9 Matrix{Union{Missing, Tuple{Int16, Int16}}}:
 missing     …  (208, 208)
 missing        (208, 208)
 (135, 143)     (210, 210)
 (133, 135)     (208, 208)
 (133, 135)     (208, 208)
 (135, 143)  …  (208, 208)
 ⋮           ⋱
 (133, 141)     (208, 220)
 (133, 143)     (208, 208)
 (135, 141)     (208, 208)
 (137, 143)  …  (208, 208)
 (135, 141)     (208, 208)
```
