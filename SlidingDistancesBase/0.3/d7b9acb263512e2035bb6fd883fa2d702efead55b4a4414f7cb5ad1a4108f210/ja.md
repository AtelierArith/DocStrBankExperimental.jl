```
ZNormalizer{T} <: AbstractNormalizer{T, 1}
```

入力配列をその場で正規化するユーティリティオブジェクト。ベクトルのように動作し、ウィンドウにインデックスを付けるには `z[i]` を使用し、正規化されたウィンドウにインデックスを付けるには `z[!,i]` を使用します。

# 引数:

  * `x::Vector{T}`: 操作対象のベクトル
  * `n::Int`: ウィンドウの長さ
  * `μ::T`: ウィンドウの平均
  * `σ::T`: ウィンドウの標準偏差
  * `s::T`: ウィンドウの合計
  * `ss::T`: ウィンドウの合計^2
  * `i::Int`: 現在のインデックス
  * `buffer::Vector{T}`: 計算された正規化値を格納します。これにはアクセスしないでください。完全でない可能性があります。
  * `bufi::Int`: 最後の正規化値のインデックス
