```
degree(i)
degree(p)
```

Return the "relative scale degree" (0-6) to which the interval points (unison=`0`, 2nd=`1`, octave=`0`, 2nd down=`6`, etc.). For pitches, return the integer that corresponds to the letter (C=`0`, D=`1`, ...).
