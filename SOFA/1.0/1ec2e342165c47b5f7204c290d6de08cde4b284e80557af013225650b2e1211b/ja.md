地上の観測者のために、CIRSと観測座標間の変換のための星に依存しない天文学パラメータを準備します。呼び出し元は、地球の姿勢情報と屈折定数、ならびにサイトの座標を提供します。

この関数は、国際天文学連合のSOFA（基礎天文学の標準）ソフトウェアコレクションの一部です。

ステータス：サポート関数。

与えられたもの： sp     double      TIOロケータs'（ラジアン、注1）    theta  double      地球回転角（ラジアン）    elong  double      経度（ラジアン、東+ve、注2）    phi    double      測地緯度（ラジアン、注2）    hm     double      楕円体上の高さ（m、測地的注2）    xp,yp  double      極運動座標（ラジアン、注3）    refa   double      屈折定数A（ラジアン、注4）    refb   double      屈折定数B（ラジアン、注4）

返されるもの：    astrom iauASTROM*  星に依存しない天文学パラメータ：       pmt    double       変更なし       eb     double[3]    変更なし       eh     double[3]    変更なし       em     double       変更なし       v      double[3]    変更なし       bm1    double       変更なし       bpn    double[3][3] 変更なし       along  double       経度 + s'（ラジアン）       xpl    double       地元子午線に対する極運動xp（ラジアン）       ypl    double       地元子午線に対する極運動yp（ラジアン）       sphi   double       測地緯度の正弦       cphi   double       測地緯度の余弦       diurab double       日周異常ベクトルの大きさ       eral   double       "ローカル" 地球回転角（ラジアン）       refa   double       屈折定数A（ラジアン）       refb   double       屈折定数B（ラジアン）

注：

1. sp、TIOロケータs'は、最も正確なアプリケーションにのみ必要な微小量です。ゼロに設定するか、SOFA関数iauSp00を使用して予測できます。
2. 地理的座標はWGS84基準楕円体に関してです。経度の符号に注意してください：現在の関数で必要な経度は東正（すなわち右手系）であり、地理的慣習に従っています。
3. 極運動xp,ypはIERSの公報から取得できます。値は、国際地球基準系に対する天体中間極の座標（ラジアン）であり、それぞれ子午線0度および90度西に沿って測定されます。多くのアプリケーションでは、xpとypはゼロに設定できます。

内部的に、極運動は地元子午線に回転した形で保存されます。

4. 屈折定数refaとrefbは、dZ = A*tan(Z)+B*tan^3(Z)モデルで使用されます。ここで、Zは観測された（すなわち屈折された）天頂距離であり、dZは屈折量です。
5. 入力パラメータの単位に注意を払うことが推奨されます。なぜなら、ありそうもない値の入力パラメータも受け入れられ、使用されるモデルに従って処理されるからです。
6. 呼び出し元が地球回転情報と屈折定数を提供したくない場合、関数iauApio13を使用することができます。これはUTCおよび気象データなどから始まり、他のSOFA関数を使用して適切な値を計算します。
7. これは、星に依存しないパラメータをastrom構造に挿入するいくつかの関数の1つであり、天文学的変換のチェーンICRS <-> GCRS <-> CIRS <-> 観測に必要です。

さまざまな関数は、異なる観測者のクラスと変換チェーンの部分をサポートします：

```
     関数             観測者        変換

  iauApcg iauApcg13    地心的      ICRS <-> GCRS
  iauApci iauApci13    地上        ICRS <-> CIRS
  iauApco iauApco13    地上        ICRS <-> 観測
  iauApcs iauApcs13    宇宙        ICRS <-> GCRS
  iauAper iauAper13    地上        地球回転の更新
  iauApio iauApio13    地上        CIRS <-> 観測
```

"13"で終わる名前のものは、現代のSOFAモデルを使用してさまざまな天体暦を計算します。他のものは、呼び出し元が提供した天体暦を受け入れます。

ICRSからGCRSへの変換は、空間運動、視差、光の屈折、異常をカバーします。GCRSからCIRSへの変換は、フレームバイアスと歳差- nutationを含みます。CIRSから観測への変換は、地球回転、極運動、日周異常、視差（ICRS <-> GCRS変換に組み込まれていない限り）、および大気屈折を考慮します。

8. この関数によって生成されたコンテキスト構造astromは、iauAtioqおよびiauAtoiqによって使用されます。

呼び出し：    iauPvtob     地上ステーションの位置/速度    iauAper      天文学パラメータ：ERAの更新
