ラジアンを時間、分、秒、分数に分解します。

この関数は国際天文学連合のSOFA（基礎天文学の標準）ソフトウェアコレクションの一部です。

ステータス：ベクトル/行列サポート関数。

与えられた：    ndp     int     解像度（注1）    angle   double  ラジアンでの角度

返される：    sign    char    '+' または '-'    ihmsf   int[4]  時間、分、秒、分数

呼び出し：    iauD2tf      日をhmsに分解

注意：

```
1. 引数ndpは次のように解釈されます：
```

| ndp |      解像度      |
|:---:|:-------------:|
|  :  | ...00,000,000 |
| -7  |  10,000,000   |
| -6  |   1,000,000   |
| -5  |    100,000    |
| -4  |    10,000     |
| -3  |     1,000     |
| -2  |      100      |
| -1  |      10       |
|  0  |       1       |
|  1  |      0.1      |
|  2  |     0.01      |
|  3  |     0.001     |
|  :  |   0.000...    |

```
2. ndpの最大の有用な正の値は、角度の大きさ、ターゲットプラットフォームの倍精度の形式、およびihmsf[4]のオーバーフローのリスクによって決まります。典型的なプラットフォームでは、角度が2πまでの場合、利用可能な浮動小数点精度はndp=12に対応するかもしれません。しかし、実際の制限は通常ndp=9で、これは32ビットintの容量によって設定され、intが16ビットのみの場合はndp=4です。

3. 角度の絶対値は2πを超える場合があります。そうでない場合は、呼び出し元が角度が非常に近く2πであり、24時間に丸められる場合をテストして処理する責任があります。ihmsf[1]=24をテストし、ihmsf[1:4]をゼロに設定します。
```
