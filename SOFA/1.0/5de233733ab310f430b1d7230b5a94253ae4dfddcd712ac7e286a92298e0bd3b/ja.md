ラジアンを度、分、秒、分数に分解します。この関数は国際天文学連合のSOFA（基礎天文学の標準）ソフトウェアコレクションの一部です。

ステータス：ベクトル/行列サポート関数。

与えられた：

  * `ndp::Integer` 解像度（注1）
  * `angle::Real` ラジアンでの角度

返される：

  * `sign::Char`    '+' または '-'
  * `idmsf::Array{Int32, 1}`  度、分、秒、分数

呼び出し： iauD2tf      日をhmsに分解

注：

1. 引数ndpは次のように解釈されます：

| ndp |      解像度      |
|:---:|:-------------:|
|  :  | ...00,000,000 |
| -7  |  10,000,000   |
| -6  |   1,000,000   |
| -5  |    100,000    |
| -4  |    10,000     |
| -3  |     1,000     |
| -2  |      100      |
| -1  |      10       |
|  0  |       1       |
|  1  |      0.1      |
|  2  |     0.01      |
|  3  |     0.001     |
|  :  |   0.000...    |

2. ndpの最大の正の有用値は、角度の大きさ、ターゲットプラットフォームの倍精度の形式、およびidmsf[3]のオーバーフローのリスクによって決まります。典型的なプラットフォームでは、角度が2πまでの場合、利用可能な浮動小数点精度はndp=12に対応するかもしれません。しかし、実際の制限は通常ndp=9で、32ビット整数の容量によって設定され、整数が16ビットのみの場合はndp=4です。
3. 角度の絶対値は2πを超える場合があります。超えない場合は、呼び出し元が角度が非常に近く2πで360度に丸められるケースをテストして処理する責任があります。これはidmsf[0]=360をテストし、idmsf[0-3]をゼロに設定することによって行います。
