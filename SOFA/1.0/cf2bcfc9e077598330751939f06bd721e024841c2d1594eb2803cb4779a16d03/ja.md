地上の観測者のために、ICRSと観測座標間の変換のための星に依存しない天文学パラメータを準備します。呼び出し元は、地球のエフェメリス、地球の回転情報、屈折定数、およびサイトの座標を提供します。

この関数は、国際天文学連合のSOFA（基礎天文学の標準）ソフトウェアコレクションの一部です。

ステータス：サポート関数。

与えられたもの： date1  double       TDBとしての2部...    date2  double       ...ユリウス日（注1）    ebpv   double[2][3] 地球重心のPV（au、au/日、注2）    ehp    double[3]    地球の太陽中心のP（au、注2）    x,y    double       CIP X,Y（単位ベクトルの成分）    s      double       CIOロケーターs（ラジアン）    theta  double       地球の回転角（ラジアン）    elong  double       経度（ラジアン、東+ve、注3）    phi    double       緯度（測地、ラジアン、注3）    hm     double       楕円体上の高さ（m、測地、注3）    xp,yp  double       極運動座標（ラジアン、注4）    sp     double       TIOロケーターs'（ラジアン、注4）    refa   double       屈折定数A（ラジアン、注5）    refb   double       屈折定数B（ラジアン、注5）

返されるもの： astrom iauASTROM*   星に依存しない天文学パラメータ：       pmt    double       PM時間間隔（SSB、ユリウス年）       eb     double[3]    SSBから観測者へのベクトル（au）       eh     double[3]    太陽から観測者への単位ベクトル       em     double       太陽から観測者までの距離（au）       v      double[3]    重心観測者の速度（ベクトル、c）       bm1    double       sqrt(1-|v|^2)：ローレンツ因子の逆数       bpn    double[3][3] バイアス-歳差- nutation行列       along  double       経度 + s'（ラジアン）       xpl    double       地元の子午線に対する極運動xp（ラジアン）       ypl    double       地元の子午線に対する極運動yp（ラジアン）       sphi   double       測地緯度の正弦       cphi   double       測地緯度の余弦       diurab double       日周のアベレーションベクトルの大きさ       eral   double       "ローカル" 地球の回転角（ラジアン）       refa   double       屈折定数A（ラジアン）       refb   double       屈折定数B（ラジアン）

注：

1. TDB日付date1+date2はユリウス日であり、2つの引数の間で任意の便利な方法で分配されます。例えば、    JD(TDB)=2450123.7は、以下のように表現できます：

```
     date1          date2

  2450123.7           0.0       （JD法）
  2451545.0       -1421.3       （J2000法）
  2400000.5       50123.2       （MJD法）
  2450123.5           0.2       （日付と時間法）
```

JD法は、いくつかの小数点以下の桁数の解像度の損失が許容される場合に最も自然で便利です。J2000法は、引数が内部で処理される方法に最も適しており、最適な解像度を提供します。MJD法と日付と時間法は、解像度と便利さの良い妥協点です。この関数のほとんどのアプリケーションでは、選択は全く重要ではありません。

TTは、精度に重大な影響を与えることなくTDBの代わりに使用できます。

2. ベクトルeb、eh、およびすべてのastromベクトルは、BCRS軸に関してです。
3. 地理的座標はWGS84基準楕円体に関してです。経度の符号規則に注意してください：現在の関数で必要な経度は右手系、すなわち東が正であり、地理的規則に従っています。
4. xpとypは、国際地球基準系に対する天体中間極の座標（ラジアン）であり、それぞれ子午線0度および90度西に沿って測定されます。spは、赤道上の地球中間原点を位置付けるTIOロケーターs'（ラジアン）です。多くのアプリケーションでは、xp、yp、および（特に）spはゼロに設定できます。

内部的に、極運動は地元の子午線に回転した形で保存されます。

5. 屈折定数refaとrefbは、dZ = A*tan(Z)+B*tan^3(Z)モデルで使用され、ここでZは観測された（すなわち屈折された）天頂距離であり、dZは屈折の量です。
6. 入力パラメータの単位に注意を払うことが推奨されます。なぜなら、ありそうもない値の入力パラメータも受け入れられ、使用されるモデルに従って処理されるからです。
7. 呼び出し元が地球のエフェメリス、地球の回転情報、および屈折定数を提供したくない場合、iauApco13関数を現在の関数の代わりに使用できます。これはUTCや気象データなどから始まり、他のSOFA関数を使用して適切な値を計算します。
8. これは、ICRS <-> GCRS <-> CIRS <-> 観測の天文学的変換の連鎖に必要な星に依存しないパラメータをastrom構造に挿入するいくつかの関数の1つです。

さまざまな関数は、異なる観測者のクラスと変換チェーンの部分をサポートします：

```
     関数             観測者        変換

  iauApcg iauApcg13    地心      ICRS <-> GCRS
  iauApci iauApci13    地上      ICRS <-> CIRS
  iauApco iauApco13    地上      ICRS <-> 観測
  iauApcs iauApcs13    宇宙      ICRS <-> GCRS
  iauAper iauAper13    地上      地球の回転を更新
  iauApio iauApio13    地上      CIRS <-> 観測
```

"13"で終わる名前のものは、さまざまなエフェメリスを計算するために現代のSOFAモデルを使用します。他のものは、呼び出し元が提供したエフェメリスを受け入れます。

ICRSからGCRSへの変換は、空間運動、視差、光の屈折、およびアベレーションをカバーします。GCRSからCIRSへの変換は、フレームバイアスと歳差-ヌテーションを含みます。CIRSから観測への変換は、地球の回転、極運動、日周のアベレーションおよび視差（ICRS <-> GCRS変換に組み込まれていない限り）、および大気の屈折を考慮します。

9. この関数によって生成されたコンテキスト構造astromは、iauAtioq、iauAtoiq、iauAtciq*およびiauAticq*によって使用されます。

呼び出し：    iauAper      天文学パラメータ：ERAを更新    iauC2ixys    天体から中間行列への変換、X,Yおよびsが与えられた場合    iauPvtob     地上ステーションの位置/速度    iauTrxpv     r行列の転置とpvベクトルの積    iauApcs      天文学パラメータ、ICRS-GCRS、宇宙観測者    iauCr        r行列のコピー
