地上の観測者のために、ICRSと観測座標間の変換のための星に依存しない天文学パラメータを準備します。呼び出し元はUTC、サイトの座標、周囲の空気条件、および観測波長を提供し、SOFAモデルを使用して地球のエフェメリス、CIP/CIO、および屈折定数を取得します。

この関数によって生成されるパラメータは、ICRS/CIRS変換の視差、光の屈折、アベレーション、およびバイアス-歳差- nutationの部分で必要です。

この関数は、国際天文学連合のSOFA（基礎天文学の基準）ソフトウェアコレクションの一部です。

ステータス：サポート関数。

与えられた：    utc1   double     UTCを2部...    utc2   double     ...準ジュリア日（注1、2）    dut1   double     UT1-UTC（秒、注3）    elong  double     経度（ラジアン、東+ve、注4）    phi    double     緯度（測地、ラジアン、注4）    hm     double     楕円体上の高さ（m、測地、注4、6）    xp,yp  double     極運動座標（ラジアン、注5）    phpa   double     観測者の圧力（hPa = mB、注6）    tc     double     観測者の周囲温度（℃）    rh     double     観測者の相対湿度（範囲0-1）    wl     double     波長（マイクロメートル、注7）

返される：    astrom iauASTROM* 星に依存しない天文学パラメータ：       pmt    double       PM時間間隔（SSB、ジュリア年）       eb     double[3]    SSBから観測者へのベクトル（au）       eh     double[3]    太陽から観測者への単位ベクトル       em     double       太陽から観測者までの距離（au）       v      double[3]    重心観測者の速度（ベクトル、c）       bm1    double       sqrt(1-|v|^2)：ローレンツ因子の逆数       bpn    double[3][3] バイアス-歳差- nutation行列       along  double       経度 + s'（ラジアン）       xpl    double       地元の子午線に対する極運動xp（ラジアン）       ypl    double       地元の子午線に対する極運動yp（ラジアン）       sphi   double       測地緯度の正弦       cphi   double       測地緯度の余弦       diurab double       日周アベレーションベクトルの大きさ       eral   double       "ローカル"地球回転角（ラジアン）       refa   double       屈折定数A（ラジアン）       refb   double       屈折定数B（ラジアン）    eo     double*    原点の方程式（ERA-GST）

注：

1. utc1+utc2は準ジュリア日（注2）であり、2つの引数の間で任意の便利な方法で配分されます。たとえば、utc1がジュリア日数で、utc2が1日の分数である場合です。

ただし、JDは、特別な措置が講じられない限り、うるう秒中のUTCを明確に表すことはできません。現在の関数の慣例では、JDの日は、長さが86399、86400、または86401 SI秒であっても、UTCの日を表します。

アプリケーションは、カレンダーの日付と時間を2部の準ジュリア日に変換するために、iauDtf2d関数を使用する必要があります。これは、前述のうるう秒の曖昧さの慣例を実装しています。

2. 警告ステータス「疑わしい年」は、時間スケールの導入前のUTCや、信頼できないほど未来のUTCをフラグします。詳細については、iauDatを参照してください。
3. UT1-UTCはIERSの公報に表形式で示されています。これは、UT1-UTCを+/- 0.9秒以内に保つために導入された各正のUTCうるう秒の終わりに正確に1秒増加します。注意：この慣行は見直し中であり、将来的にはUT1-UTCが本質的に無制限に増加する可能性があります。
4. 地理座標はWGS84基準楕円体に関しています。経度の符号に注意してください：現在の関数で必要な経度は東が正（すなわち右手系）であり、地理的慣例に従っています。
5. 極運動xp,ypはIERSの公報から取得できます。値は、国際地球基準系に対する天体中間極の座標（ラジアン）であり、それぞれ0度および90度西の子午線に沿って測定されます。多くのアプリケーションでは、xpとypをゼロに設定できます。

内部的に、極運動は地元の子午線に回転した形で保存されます。

6. 観測ステーションの楕円体上の高さhm（メートル）が不明な場合でも、圧力phpa（hPa = mB）が利用可能であれば、次の式からhmの適切な推定値を得ることができます。

```
     hm = -29.3 * tsl * log ( phpa / 1013.25 );
```

ここで、tslはKでの近似海面気温です（Astrophysical Quantities, C.W.Allen, 第3版、セクション52を参照）。同様に、圧力phpaが不明な場合は、観測ステーションの高さhmから次のように推定できます。

```
     phpa = 1013.25 * exp ( -hm / ( 29.3 * tsl ) );
```

ただし、屈折は圧力にほぼ比例し、正確なphpa値は精密作業にとって重要です。

7. 引数wlは、マイクロメートル単位での観測波長を指定します。光学からラジオへの遷移は、100マイクロメートル（約3000 GHz）で発生すると仮定されています。
8. 単位に注意を払うことが推奨されます。入力パラメータのありそうもない値であっても、使用されるモデルに従って受け入れられ、処理されます。
9. 呼び出し元が独自の地球エフェメリス、地球回転情報、および屈折定数を提供したい場合は、現在の関数の代わりにiauApco関数を使用できます。
10. これは、ICRS <-> GCRS <-> CIRS <-> 観測の天文学的変換の連鎖に必要な星に依存しないパラメータをastrom構造に挿入するいくつかの関数の1つです。

さまざまな関数は、異なる観測者のクラスと変換チェーンの部分をサポートします：

```
     関数             観測者        変換

  iauApcg iauApcg13    地心的      ICRS <-> GCRS
  iauApci iauApci13    地上        ICRS <-> CIRS
  iauApco iauApco13    地上        ICRS <-> 観測
  iauApcs iauApcs13    宇宙        ICRS <-> GCRS
  iauAper iauAper13    地上        地球回転の更新
  iauApio iauApio13    地上        CIRS <-> 観測
```

「13」で終わる名前のものは、さまざまなエフェメリスを計算するために現代のSOFAモデルを使用します。他のものは、呼び出し元が提供したエフェメリスを受け入れます。

ICRSからGCRSへの変換は、空間運動、視差、光の屈折、およびアベレーションをカバーします。GCRSからCIRSへの変換は、フレームバイアスと歳差- nutationを含みます。CIRSから観測への変換は、地球回転、極運動、日周アベレーション、および視差（ICRS <-> GCRS変換に組み込まれていない限り）、および大気の屈折を考慮します。

11. この関数によって生成されたコンテキスト構造astromは、iauAtioq、iauAtoiq、iauAtciq*、およびiauAticq*によって使用されます。
