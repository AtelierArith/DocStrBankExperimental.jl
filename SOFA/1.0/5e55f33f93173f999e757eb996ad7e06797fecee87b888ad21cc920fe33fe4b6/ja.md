この関数は国際天文学連合のSOFA（基礎天文学の標準）ソフトウェアコレクションの一部です。

ステータス：サポート関数。

与えられたもの：     s         char    sign:  '-' = 負、その他は正     ideg      int     度     iamin     int     分     asec      double  秒

返されるもの：     rad       double  ラジアンでの角度

返される（関数値）：     int status:  0 = OK                  1 = ideg が範囲 0-359 の外                  2 = iamin が範囲 0-59 の外                  3 = asec が範囲 0-59.999... の外 注意：     1. 結果は、範囲チェックが失敗しても計算されます。
