```
McCullagh_test(mat::Matrix{Int})
```

平方のコンティンジェンシーテーブル `mat` に対してマカラのテストを実行します。

kxk コンティンジェンシーテーブルに対するマカラのテスト;

Biometrika, 1977, 64(3), 449-453 を参照してください。

テストケース（p. 452 の表 1）

例

```jldoctest
julia> mat = [43 8 3 0; 2 2 5 3; 1 0 7 2; 0 0 1 5]
julia> McCullagh_test(mat)
(0.005469174895116946, 1.4504988072997458, 1.502600073417028, 0.5221345956920705, 2.778017046308073)
```

期待される N 行列

```jldoctest
3×3 Matrix{Int64}:
 14   4  0
  4  12  3
  0   3  6
```

期待される R ベクトル

```jldoctest
[11 11 5]'
```

期待される Δ1 = 1.45, Δ2 = 1.50, 標準誤差 = 0.53 ```
