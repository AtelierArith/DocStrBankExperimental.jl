```
DistMeshSetup
```

キーワード引数は以下の通りです：

```
iso (デフォルト: 0): 等値面を抽出するための値、内部の面は負
deltat (デフォルト: 0.1): 各イテレーションで適用するエッジ変位の割合
sort (デフォルト: false): trueで固定点がない場合、ヒルベルトソートを使用して点をソートします。
sort_interval (デフォルト: 20): 指定された再三角形化の後にヒルベルトソートを使用します
distribution (デフォルト: :regular): 初期点分布、:regularまたは:packedのいずれか。
```
