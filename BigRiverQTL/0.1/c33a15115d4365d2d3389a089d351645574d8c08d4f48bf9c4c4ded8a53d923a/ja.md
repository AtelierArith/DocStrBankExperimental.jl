```
get_geno_completecases(geno_missing::Geno) -> Geno
```

欠損値を含む遺伝子型データ構造 `Geno` から行または列を特定して削除し、欠損データの割合が最も高いものを優先します。

# 引数

  * `geno_missing::Geno`: 欠損値を含む可能性のある遺伝子型データ構造。

# 戻り値

  * `Geno`: 入力と同じフィールドを持つ新しい遺伝子型データ構造。ただし、欠損データを含む行と列は体系的に削除され、欠損データが残らないようにします。

# 説明

この関数の核心は、欠損データの割合が最も高い行または列を反復的に特定して削除することです。この決定は、行全体の欠損データの最大割合が列全体のそれを超えるかどうかに基づいています。行または列は、行列に欠損値が残らなくなるまで削除されます。

クリーンアップ後、関数は `Geno` データ構造を再構築します。データの次元を調整し、染色体ごとに再グループ化し、マーカーおよびサンプル識別子が正しく整列されることを確認することで、データが期待される形式に整合するようにします。
