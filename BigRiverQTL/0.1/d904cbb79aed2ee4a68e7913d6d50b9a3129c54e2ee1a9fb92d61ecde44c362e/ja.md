```
get_geno_completecases(data_missing::GeneticStudyData) -> GeneticStudyData
```

欠損値を含むデータ構造 `GeneticStudyData` から行または列を特定して削除し、欠損データの割合が最も高いものを優先します。

# 引数

  * `data_missing`: 欠損値を含む可能性のある遺伝学的研究データ構造。

# 戻り値

  * `GeneticStudyData`: 入力と同じフィールドを持つ新しい遺伝学的研究データ構造。ただし、欠損データを含む行と列は体系的に削除され、欠損データが残らないようにします。

# 説明

この関数の核心は、genoオブジェクト内の欠損データの割合が最も高い行または列を反復的に特定して削除することです。この決定は、行全体の欠損データの最大割合が列全体のそれを超えるかどうかに基づいています。行または列は、行列内に欠損値が残らないまで削除されます。

クリーンアップ後、関数は `GeneticStudyData` データ構造を再構築します。データの次元を調整し、染色体ごとに再グループ化し、マーカーとサンプルの識別子が正しく整列されるようにして、期待される形式にデータが整合することを保証します。
