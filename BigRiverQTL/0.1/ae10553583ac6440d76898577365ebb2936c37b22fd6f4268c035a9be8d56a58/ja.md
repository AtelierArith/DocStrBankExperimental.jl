```
pmap2df(pmap::Pmap) -> DataFrame
```

表現型マップオブジェクト（`Pmap`）をDataFrameに変換します。

# 引数

  * `pmap`: マッピング情報を含む遺伝子マップオブジェクト。このオブジェクトは、以下の期待される構造に準拠している必要があります。

      * `chr`: 染色体識別子の配列。
      * `marker_name`: 染色体ごとに整理されたマーカー名のネストされた配列。
      * `pos`: 染色体ごとに整理されたマーカー位置のネストされた配列。

# 戻り値

  * `DataFrame`: 表現型マップデータを表すDataFrameで、マーカー名（`Locus`）、染色体識別子（`Chr`）、およびマーカー位置（`Pos`）の列を持ちます。
