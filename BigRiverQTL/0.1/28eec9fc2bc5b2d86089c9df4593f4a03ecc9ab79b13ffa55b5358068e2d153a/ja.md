```
 kinship_std(genmat::Array{Float64,2})
```

標準化（または正規化）された遺伝子型マトリックス（線形カーネル）によって親族関係を計算します。すなわち、遺伝子型からマーカーの平均を引き、マーカーの標準偏差で割ります。また、気候情報データでも実行できます。詳細は[`kinship_gs`](@ref)を参照してください。

# 引数

  * `genmat` : 遺伝子型データの行列（0,1,2）。size(genmat)=(n,p) は `n` 人の個体 x `p` マーカーを表します。

# 出力

n x n の対称行列を返します。詳細は[`kinship_ctr`](@ref)を参照してください。
