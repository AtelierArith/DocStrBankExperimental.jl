```
get_control_file(filename::String) -> (String, String)
```

制御ファイルを見つけ、そのファイルが存在するディレクトリを特定します。

# 引数

  * `filename::String`: ファイルまたはディレクトリへのパスを表す文字列。

# 戻り値

  * `(String, String)`: 制御ファイルのディレクトリと制御ファイルへのパスを含むタプル。

提供された `filename` がディレクトリである場合、関数はその中のJSONファイルを検索し、最初に見つかったJSONファイルへのパスを返します。

# 説明

関数は最初に提供された `filename` の絶対パスを解決します。`filename` が既存のファイルを指している場合、関数はこのファイルを含むディレクトリを特定します。`filename` がディレクトリを指定している場合、関数はこのディレクトリ内のすべてのエントリをリストし、最初のJSONファイルを検索し、`filename` をこのJSONファイルへのパスに更新します。
