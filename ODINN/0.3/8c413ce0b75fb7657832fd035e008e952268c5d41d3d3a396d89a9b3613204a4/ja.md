```
run!(simulation::FunctionalInversion)
```

与えられた `FunctionalInversion` シミュレーションのトレーニングプロセスを実行します。

# 引数

  * `simulation::FunctionalInversion`: 機能的反転プロセスのためのパラメータと設定を含むシミュレーションオブジェクト。

# 説明

この関数は、提供されたシミュレーションのための普遍的微分方程式 (UDE) のトレーニングを開始します。トレーニングプロセスの開始を示すメッセージを印刷し、`train_UDE!` 関数を呼び出してトレーニングを実行し、結果を `results_list` に収集します。結果は `Sleipnir.save_results_file!` を使用して保存されることを意図していますが、このステップは現在コメントアウトされており、最適化が機能するようになったら有効になります。最後に、メモリを解放するためにガーベジコレクタがトリガーされます。

# 注意事項

  * `Sleipnir.save_results_file!` 関数の呼び出しは現在コメントアウトされており、最適化プロセスが機能していることが確認されたら有効にする必要があります。
  * メモリ使用量を管理するために、ガーベジコレクタが明示的に `GC.gc()` を使用して実行されます。
