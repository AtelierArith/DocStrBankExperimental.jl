```
@Select(...)
```

`@Select`マクロは、`NamedTuple`のようなオブジェクトのプロパティの任意の変換を行う関数を返します。

入力式は、カンマ区切りの`lhs = rhs`ペアのリストです。`lhs`は計算する新しいプロパティの名前です。`rhs`は、入力プロパティ名の前に`:(`が付加された標準的なJuliaコードです。例えば、入力プロパティ`)a`を`b`と呼ばれるように名前を変更したい場合は、`@Select(b = :a)`を使用します。

特別なケースとして、プロパティを単に複製する場合は`= rhs`部分を省略できます。例えば、`@Select(a)`は`@Select(a = $a)`と同義です。

# 例

```julia
julia> nt = (a = 1, b = 2.0, c = false)
(a = 1, b = 2.0, c = false)

julia> @Select(a, sum_a_b = $a + $b)(nt)
(a = 1, sum_a_b = 3.0)
```
