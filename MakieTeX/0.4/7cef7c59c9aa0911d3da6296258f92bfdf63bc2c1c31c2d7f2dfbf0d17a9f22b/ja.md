```
CachedTypst(doc::TypstDocument)
```

`TypstDocument`をコンパイルし、キャッシュされたTypstオブジェクトを返します。

`CachedTypst`構造体は、ドキュメントとそのコンパイルされた形式、およびそれに対するプログラム内のバージョンへのポインタをいくつか保存します。また、ページの寸法も保存します。

コンストラクタは以下のフィールドを保存します：

  * `doc`
  * `pdf`
  * `ptr`
  * `surf`
  * `dims`

!!! note
    これは`mutable struct`です。なぜなら、Popplerハンドルへのポインタが変更される可能性があるからです。TODO: これをハンドルへのRefを持つ不変構造体にする?? あるいは、サーフェス自体を持つこともできる...


!!! note
    もし事前にレンダリングされたPDFを図に挿入したいだけであれば、`doc`フィールドに`nothing`を指定して`CachedTypst`を手動で構築することも可能です。
