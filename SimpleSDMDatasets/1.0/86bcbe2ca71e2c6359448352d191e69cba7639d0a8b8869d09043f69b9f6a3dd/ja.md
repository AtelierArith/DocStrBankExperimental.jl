```
layers(::R) where {R <: RasterData}
```

このメソッドは、データセットに名前付きレイヤーがあるかどうかを制御します。これが `nothing`（デフォルト）である場合、データセットには単一のレイヤーがあることを意味します。

複数のレイヤーが利用可能な場合、このメソッドのオーバーロードが*必要*であり、通常は `String` の `Vector` を返さなければなりません。デフォルトでは、レイヤーには `Integer` を使用してもアクセスでき、その場合 `layer=i` は `layers(data)` の *i*-番目のエントリになります。

戻り値が `nothing` でないデータセットは、*必ず* `layer` キーワードを受け入れなければなりません。
