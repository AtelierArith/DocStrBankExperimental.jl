```
serve_sync(port=3000, shared=false, print_stack=true, async=false)
```

デーモンを実行し、クライアント関数によって送信されたすべてのファイルと式を実行します。

# オプション

  * port: リッスンするポート（デフォルトは3000）。
  * shared: 呼び出し間で環境を共有します。false（デフォルト）の場合、各実行は独自の環境を持つため、変数/関数は共有されません。
  * print_stack: エラーが発生したときに完全なスタックを印刷します。デフォルトではtrueです。
  * async: クライアントを同時に異なるクライアントで実行します。
  * threaded: 各クライアントを新しいスレッドで実行します（asyncがtrueの場合はデフォルトでtrue）。
