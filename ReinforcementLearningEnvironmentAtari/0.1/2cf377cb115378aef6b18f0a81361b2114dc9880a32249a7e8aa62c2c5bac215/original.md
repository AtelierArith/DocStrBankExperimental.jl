```
AtariEnv(name; colorspace = "Grayscale", frame_skip = 4, noopmax = 20,
               color_averaging = true, repeat_action_probability = 0.)
```

Returns an AtariEnv that can be used in an RLSetup of the [ReinforcementLearning](https://github.com/jbrea/ReinforcementLearning.jl) package. Check the deps/roms folder of the ArcadeLearningEnvironment package to see all available `name`s.
