```
const F = GaloisField(p)
const F,α = GaloisField(p, :β => [1, 0, 1])
const F,α = GaloisField(p, n, :β)
```

有限体を表す型を返します。

単一引数のシグネチャは有限体 $ℤ/pℤ$ を返します。

二引数のシグネチャは、その体の代数拡張を返し、最小多項式は第二引数によって与えられます：単項式の密な表現で、次数が昇順です。

三引数のシグネチャは、その体の代数拡張を返し、最小多項式は $(p,n)$ のための [Conway 多項式](https://en.wikipedia.org/wiki/Conway_polynomial_(finite_fields)) に等しいです。`GaloisFields` パッケージには Conway 多項式のデータベースが付属しており、 $(p,n)$ のエントリが含まれていない場合はエラーが発生します。

後者の二つのケースでは、変数名（上記の β など）は型の一部であることに注意してください。これにより、同型（部分）体間の同定を定義できます。例えば、次の定義を用いて

```
const F = @GaloisField! 𝔽₂ β^2 + β + 1
const G = @GaloisField! 𝔽₂ γ^2 + γ + 1
```

体 $F$ と $G$ は同型ですが、標準的ではありません。次のように定義することができます。

```
@GaloisFields.identify β => γ + 1
@GaloisFields.identify γ => β + 1
```

これにより、次のような変換が可能になります。

```
G(β)
convert(F, γ + 1)
```

Conway の場合、独自の同定を定義する必要はありません。なぜなら、Conway 多項式は互換性関係を満たしており、それらの間の特定の区別された包含を使用することができます。 ```
