このパッケージは、元々以下で定義された `Squares test statistic` の累積分布関数の評価を実装しています。

Frederik Beaujean と Allen Caldwell. *A Test Statistic for Weighted Runs.*

https://arxiv.org/abs/1005.3233

著者たちはさらに、大量の観測値に対しても累積を計算できるように近似を導出しました。

Frederik Beaujean と Allen Caldwell. *Is the bump significant? An axion-search example* 

https://arxiv.org/abs/1710.06642 

ここで、彼らは加重ラン統計量の名前を `*Squares statistic*` に変更しました。このコードは、Frederik Beaujean による c++ と mathematica の元の実装に基づいています。

https://github.com/fredRos/runs
