```
struct Circle{I <: Integer} <: AbstractCircle
    position::Point{I}
    diameter::I
end
```

`position` と `diameter` を使用する利点は、中心と半径の代わりに、同じ型が直径の奇数値と偶数値の両方を処理できることです。この型のインスタンスは、次の条件が満たされている場合にのみ有効と見なされます：

1. `diameter > 0`

[`FilledCircle`](@ref) や [`ThickCircle`](@ref) も参照してください。
