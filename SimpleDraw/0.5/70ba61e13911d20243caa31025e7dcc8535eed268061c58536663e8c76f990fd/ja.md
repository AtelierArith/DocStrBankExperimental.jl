```
struct Line{I <: Integer} <: AbstractLine
    point1::Point{I}
    point2::Point{I}
end
```

現在の線描画アルゴリズムは一般的に対称ではありません。点の順序を逆にすると異なる結果が得られる場合があります。

[`ThickLine`](@ref)も参照してください。
