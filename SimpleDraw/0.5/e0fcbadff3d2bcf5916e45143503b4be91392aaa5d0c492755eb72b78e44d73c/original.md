```
struct Rectangle{I <: Integer} <: AbstractRectangle
    position::Point{I}
    height::I
    width::I
end
```

An instance of this type is considered valid only if the following conditions hold true:

1. `height > 0`
2. `width > 0`

See also [`FilledRectangle`](@ref), [`ThickRectangle`](@ref).
