```
struct Image{I <: Integer, A} <: AbstractShape
    position::Point{I}
    image::A
end

既存の画像を特定の位置にある画像の上に描画するために使用できます。色を明示的に渡す必要はなく、サブ画像はすでに色付けされている必要があります。一方、`Bitmap`では色を明示的に渡す必要があり、単一の色のみである必要があります。
```

See also [`Bitmap`](@ref).
