```
struct TextLine{I <: Integer, S, F <: AbstractFont} <: AbstractShape
    position::Point{I}
    text::S
    font::F
end
```

この時点で利用可能なモノスペースASCIIフォントは6つです：

1. `TERMINUS_16_8`（高さ16ピクセル、幅8ピクセル）
2. `TERMINUS_BOLD_16_8`（高さ16ピクセル、幅8ピクセル）
3. `TERMINUS_24_12`（高さ24ピクセル、幅12ピクセル）
4. `TERMINUS_BOLD_24_12`（高さ24ピクセル、幅12ピクセル）
5. `TERMINUS_32_16`（高さ32ピクセル、幅16ピクセル）
6. `TERMINUS_BOLD_32_16`（高さ32ピクセル、幅16ピクセル）

現時点ではASCII文字のグリフのみが利用可能です。

詳細は[`Character`](@ref)を参照してください。
