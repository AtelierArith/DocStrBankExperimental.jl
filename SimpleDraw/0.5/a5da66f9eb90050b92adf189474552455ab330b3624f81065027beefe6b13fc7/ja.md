```
struct ThickCircle{I <: Integer} <: AbstractCircle
    position::Point{I}
    diameter::I
    thickness::I
end
```

`center` と `radius` の代わりに `position` と `diameter` を使用する利点は、同じ型が奇数および偶数の `diameter` の両方を処理できることです。この型のインスタンスは、次の条件がすべて満たされている場合にのみ有効と見なされます：

1. `diameter > 0`
2. `thickness > 0`
3. `diameter` が奇数の場合、`2 * thickness <= diameter + 1`
4. `diameter` が偶数の場合、`2 * thickness <= diameter`

[`Circle`](@ref) や [`FilledCircle`](@ref) も参照してください。
