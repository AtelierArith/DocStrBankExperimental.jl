```
files(root, dirs=true)
```

絶対パス `root` で指定されたディレクトリ内のファイルを提供するためのミドルウェアです。

`req[:path]` は `root` と組み合わされてファイルパスを生成します。ファイルパスが `root` 内に含まれている場合（正規化後）で、既存のファイル（または `dirs=true` の場合はディレクトリ）を指している場合、ファイル（またはディレクトリリスト）で応答します。それ以外の場合は、次のミドルウェアを呼び出します。

現在の作業ディレクトリまたはサーバーが定義されているファイルを含むディレクトリに対して `root` を指定したい場合は、`pwd()` または `@__DIR__` を使用し、（必要に応じて）`joinpath` または `normpath` を使用できます。

# 例

```
files(pwd()) # 現在の作業ディレクトリからファイルを提供
files(@__DIR__) # スクリプトがあるディレクトリからファイルを提供

# スクリプトがあるディレクトリ内の assets ディレクトリからファイルを提供:
files(joinpath(@__DIR__, "assets"))

# スクリプトがあるディレクトリの上のディレクトリ内の assets ディレクトリからファイルを提供:
files(normpath(@__DIR__, "../assets))
```
