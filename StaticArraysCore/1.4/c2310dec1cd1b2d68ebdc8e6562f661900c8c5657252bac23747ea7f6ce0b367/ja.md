```
MMatrix{S1, S2, T, L}(undef)
MMatrix{S1, S2, T, L}(x::NTuple{L, T})
MMatrix{S1, S2, T, L}(x1, x2, x3, ...)
```

静的サイズの可変行列 `MMatrix` を構築します。データは構築時にオプションで提供でき、後で変更可能です。`L` パラメータは配列の `length` であり、常に `S1 * S2` に等しいです。コンストラクタは、入力から推測可能な場合、`L`、`T`、さらには `S2` パラメータを省略することができます（例：`L` は常に `S1` と `S2` から推測可能です）。

```
MMatrix{S1, S2}(mat::Matrix)
```

`mat` のデータを使用して、次元 `S1 × S2` の静的サイズの可変行列を構築します。パラメータ `S1` と `S2` は必須です。なぜなら、`mat` のサイズはコンパイラには不明だからです（要素の型もオプションで指定できます）。
