```
SVector{S, T}(x::NTuple{S, T})
SVector{S, T}(x1, x2, x3, ...)
```

静的サイズのベクトル `SVector` を構築します。この型は不変であるため、データは構築時に提供する必要があり、後で変更することはできません。コンストラクタは、入力から推論可能な場合、`T` および `S` パラメータを省略することができます（例：`SVector(1,2,3)` は `SVector{3, Int}` を構築します）。

```
SVector{S}(vec::Vector)
```

`vec` からのデータを使用して、長さ `S` の静的サイズのベクトルを構築します。パラメータ `S` は必須であり、`vec` の長さはコンパイラには不明であるためです（要素型はオプションで指定することもできます）。
