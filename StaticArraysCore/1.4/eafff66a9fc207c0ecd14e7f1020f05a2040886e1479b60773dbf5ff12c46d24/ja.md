```
MArray{S, T, N, L}(undef)
MArray{S, T, N, L}(x::NTuple{L})
MArray{S, T, N, L}(x1, x2, x3, ...)
```

静的サイズの可変配列 `MArray` を構築します。データは構築時にオプションで提供でき、後で変更可能です。`S` パラメータは配列の次元またはサイズを指定するタプル型であり、例えば 3×4×5 サイズの配列の場合は `Tuple{3,4,5}` となります。`N` パラメータは配列の次元であり、`L` パラメータは配列の `length` で、常に `prod(S)` に等しくなります。コンストラクタは、入力から推測可能な場合、`L`、`N`、および `T` パラメータを省略できます（例えば、`L` は常に `S` から推測可能です）。

```
MArray{S}(a::Array)
```

データ `a` を使用して、次元 `S`（`Tuple{...}` として表現）を持つ静的サイズの可変配列を構築します。`S` パラメータは必須であり、`a` のサイズはコンパイラには不明なためです（要素型もオプションで指定できます）。
