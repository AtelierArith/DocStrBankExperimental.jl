```
SMatrix{S1, S2, T, L}(x::NTuple{L, T})
SMatrix{S1, S2, T, L}(x1, x2, x3, ...)
```

静的サイズの行列 `SMatrix` を構築します。この型は不変であるため、データは構築時に提供する必要があり、後で変更することはできません。`L` パラメータは配列の `length` であり、常に `S1 * S2` に等しくなります。コンストラクタは、入力から推測可能な場合、`L`、`T`、さらには `S2` パラメータを省略することができます（例：`L` は常に `S1` と `S2` から推測可能です）。

```
SMatrix{S1, S2}(mat::Matrix)
```

`mat` のデータを使用して、次元 `S1 × S2` の静的サイズの行列を構築します。パラメータ `S1` と `S2` は必須であり、`mat` のサイズはコンパイラには不明であるためです（要素型はオプションで指定することもできます）。
