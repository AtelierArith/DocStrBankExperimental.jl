```
 SphericalShell
```

球殻の基底関数情報を保持するオブジェクト。

# フィールド

| 名前     | 説明               |
|:------ |:---------------- |
| `l`    | 角運動量数            |
| `coef` | 原始関数の展開係数を保持する配列 |
| `exp`  | 原始関数の指数を保持する配列   |

# 例

```julia
julia> χ = SphericalShell(1, [1/√2, 1/√2], [5.0, 1.2])
P shell with 3 basis built from 2 primitive gaussians

χ₁₋₁ =    0.7071067812⋅Y₁₋₁⋅r¹⋅exp(-5.0⋅r²)
     +    0.7071067812⋅Y₁₋₁⋅r¹⋅exp(-1.2⋅r²)

χ₁₀  =    0.7071067812⋅Y₁₀⋅r¹⋅exp(-5.0⋅r²)
     +    0.7071067812⋅Y₁₀⋅r¹⋅exp(-1.2⋅r²)

χ₁₁  =    0.7071067812⋅Y₁₁⋅r¹⋅exp(-5.0⋅r²)
     +    0.7071067812⋅Y₁₁⋅r¹⋅exp(-1.2⋅r²)
```
