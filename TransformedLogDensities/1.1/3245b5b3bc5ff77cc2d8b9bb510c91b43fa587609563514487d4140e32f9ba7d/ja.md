# TransformedLogDensities.jl

![ライフサイクル](https://img.shields.io/badge/lifecycle-maturing-blue.svg) [![ビルド](https://github.com/tpapp/TransformedLogDensities.jl/workflows/CI/badge.svg)](https://github.com/tpapp/TransformedLogDensities.jl/actions?query=workflow%3ACI) [![コードカバレッジ](https://codecov.io/github/tpapp/TransformedLogDensities.jl/graph/badge.svg?token=wKzH6Q1Aot)](https://codecov.io/github/tpapp/TransformedLogDensities.jl)

[TransformVariables.jl](https://github.com/tpapp/TransformVariables.jl/) からの変換を使用して、[LogDensityProblems.jl](https://github.com/tpapp/LogDensityProblems.jl) API に準拠した変換された対数密度を作成します。

公開されているシンボルは [`TransformedLogDensity`](@ref) のみです。詳細な例は [LogDensityProblems.jl](https://github.com/tpapp/LogDensityProblems.jl) のドキュメントにあります。
