```
split_string(str::AbstractString, size::Int) -> String, String
```

文字列 `str` を特定の印刷可能な `size` の文字数の後で分割します。この関数は、分割点の前後の2つの文字列を返します。

アルゴリズムは、最初に返される文字列の印刷可能な幅が常に `size` に等しくなることを保証します。ただし、`size` が負の値であるか、`str` の印刷可能なサイズより大きい場合は除きます。最初のケースでは、最初の文字列は空になりますが、2番目のケースでは最初の文字列は `str` に等しくなります。

!!! note
    分割点の文字が印刷するのに1文字以上必要な場合（例えば、一部のUTF-8文字のように）、すべてがスペースで埋められます。
