```
fmi2GetVersion(fmu::FMU2)
```

FMUの関数をコンパイルするために使用された「fmi2Functions.h」ヘッダーファイルのバージョンを返します。

# 引数

  * `fmu::FMU2`: FMI 2.0.2標準におけるFMUとそのすべてのインスタンスを表す可変構造体。

# 戻り値

  * Cスタイルの（NUL終端）文字列のアドレスからの文字列を返します。この文字列は、FMUの関数をコンパイルするために使用された「fmi2Functions.h」ヘッダーファイルのバージョンを表します。この関数は、このヘッダーファイルで定義されている「fmiVersion」を返します。この仕様書に文書化されている標準のヘッダーファイルのバージョンは「2.0」です。
