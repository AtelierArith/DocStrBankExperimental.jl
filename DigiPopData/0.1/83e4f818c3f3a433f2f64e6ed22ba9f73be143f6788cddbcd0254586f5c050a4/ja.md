`MetricBinding` は、**シナリオ**、**エンドポイント**、および具体的な `AbstractMetric` の説明を結びつけて、ログ記録、表示、または最適化/検証ルーチンに渡すことができる単一のユニットにまとめるコンテナです。

# フィールド

|         名前 |                型 |                                        説明 |
| ----------:| ----------------:| -----------------------------------------:|
|       `id` |         `String` |                            バインディングの一意の識別子 |
| `scenario` |         `String` |            メトリックが評価されるシナリオ（例：シミュレーションアーム） |
|   `metric` | `AbstractMetric` | メトリックの実装（`MeanMetric`、`CategoryMetric`、…） |
| `endpoint` |         `String` |                     メトリックが計算される観測可能/モデル変数 |
|   `active` |           `Bool` |             バインディングが有効かどうか（デフォルトは `true`） |
