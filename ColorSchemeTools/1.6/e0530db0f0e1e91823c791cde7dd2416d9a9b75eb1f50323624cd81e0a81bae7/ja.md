```
sineramp(rows, cols;
        amplitude = 12.5,
        wavelength = 8,
        p = 2)
```

`rows × cols` の値の配列を生成し、上から下に向かって振幅が減少する正弦波を表示します。

使用法:

```julia
using Images
scheme = ColorSchemes.dracula
img = Gray.(sineramp(256, 512, amp = 12.5, wavelen = 8, p = 2))
cimg = zeros(RGB, 256, 512)
for e in eachindex(img)
    cimg[e] = get(mscheme, img[e])
end
cimg
```

デフォルトの波長は8ピクセルです。ピクセルピッチが0.25mmのコンピュータモニターでは、これは2mmの波長に相当します。600mmのモニター視距離では、これは視角の0.19度、または約5.2サイクル/度に相当します。これは、ほとんどの人が正弦波の格子に対して最大のコントラスト感度を持つ空間周波数の範囲（3-7サイクル/度）に該当します（これは平均輝度によって変わります）。8ピクセルの波長は、正弦波の合理的な離散表現を提供するのにも十分です。目的は、刺激が人間の視覚システムの性能に適合するように提示されることであり、主に評価しているのはカラースキームの知覚コントラストであり、視聴者の視覚性能ではありません。

デフォルトの振幅は12.5に設定されており、ピークから谷までの局所的な特徴の大きさは25です。これは、典型的なカラースキームの256レベルの約10%に相当します。この大きさの特徴を隠す知覚的なフラットスポットを持つカラースキームは珍しくありません。

画像の幅は、正弦波の整数サイクル数を持つように調整されています。これは、テスト画像を使用して周期的なカラースキームを評価する場合に役立ちます。ただし、画像の上部にはわずかな周期的な不連続性が見られますが、これは下部では消えます。
