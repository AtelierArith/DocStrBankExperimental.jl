```
make_colorscheme(f1::Function, f2::Function, f3::Function;
    model    = :RGB,
    length   = 100,
    category = "",
    notes    = "functional ColorScheme")
```

関数を使用してカラースキームを作成します。各引数は、0から1の間の値に対して赤、緑、青の成分の値を返す関数です。

`model`はカラーモデルで、`:RGB`、`:HSV`、または`:LCHab`のいずれかを指定できます。

`length`キーワードを使用して、カラースキームの色の数を設定します。

デフォルトのカラーモデルは`:RGB`で、関数は適切な範囲の値を返す必要があります：

  * f1 - [0.0 - 1.0]   - 赤
  * f2 - [0.0 - 1.0]   - 緑
  * f3 - [0.0 - 1.0]   - 青

`:HSV`カラーモデルの場合：

  * f1 - [0.0 - 360.0] - 色相
  * f2 - [0.0 - 1.0]   - 彩度
  * f3 - [0.0 - 1.0]   - 値（明るさ）

`:LCHab`カラーモデルの場合：

  * f1 - [0.0 - 100.0] - 輝度
  * f2 - [0.0 - 100.0] - クロマ
  * f3 - [0.0 - 360.0] - 色相

### 例

赤成分を0からπまで上昇し、再び0に戻る正弦曲線として定義し、緑成分は常に0、青成分はπから始まり0.5で0に到達するカラースキームを作成します（それ以降は0に制限されます）。

```julia
make_colorscheme(n -> sin(n * π), n -> 0, n -> cos(n * π))
```
