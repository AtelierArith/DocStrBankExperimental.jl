```
mass_sorbed(isotherm::IsothermData, polymer_mass_g::Number, component=:)
```

ポリマーの質量（**g**単位）を考慮して、ポリマー相に吸着された浸透物質の質量を取得します。

!!! note
    複数成分の等温線で成分が指定されていない場合、**合計吸着質量**ではなく、質量の`vector`が返されます。


!!! warning
    *この場合のポリマーの質量*は、ポリマーの*のみ*の質量を意味します。ポリマー相の質量ではありません。ほとんどの場合、この区別は重要ではありませんが、非常に吸着性の高いポリマーでは、この違いを考慮しないと重大な誤差を引き起こす可能性があります。
