```
Ctr
```

抽象型は、いくつかの他の型がサブタイプとなる参照位置を指定します。

# 可能なサブタイプ

  * `CtrCorner`: 参照ピクセルをコーナーに設定
  * `CtrFFT`: 参照ピクセルをFFTの中心に設定。
  * `CtrFT`: 参照ピクセルをFTの中心に設定。FTは、ゼロ周波数がFFTの慣例的中心（`size ÷ 2 + 1`）にあることを意味します。
  * `CtrRFFT`: 参照ピクセルをRFFTの中心に設定。`CtrFFT`と同じですが、最初の次元は1で中心にあります。
  * `CtrRFT`: 参照ピクセルをRFTの中心に設定。FTは、ゼロ周波数がFFTの慣例的中心（`size ÷ 2 + 1`）にあることを意味します。`CtrFT`と同じですが、最初の次元は1で中心にあります。
  * `CtrMid`: 参照ピクセルを実際の中央に設定します。奇数配列の場合は中央のピクセル、偶数配列の場合は半ピクセルを中心にします。
  * `CtrEnd`: 参照を端のコーナー（最後のピクセル）に設定
