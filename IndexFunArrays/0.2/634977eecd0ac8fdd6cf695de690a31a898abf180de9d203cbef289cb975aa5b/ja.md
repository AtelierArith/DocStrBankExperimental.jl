```
box1([T=Float64], size::NTuple{N, Int};
    offset=CtrFT,
    dims=ntuple(+, N),
    scale=ScaUnit)
```

すべてのオプションの説明については `rr2` を参照してください。

中心からのピクセルの任意の横方向の距離が1.0以下である場合、1であるIndexFunArrayを返します。この関数は、指定されたボックスサイズに一致するボックスの長さを保証する関数 `box` の補助関数として主に使用されます。この関数はピクセルごとに計算を行うため、均等なピクセル幅のボックスを希望する場合は、サブピクセルオフセットを適用する必要があります。ボックスのサイズは `scale` パラメータによって調整できます。より便利なバージョンについては `box` を参照してください。
