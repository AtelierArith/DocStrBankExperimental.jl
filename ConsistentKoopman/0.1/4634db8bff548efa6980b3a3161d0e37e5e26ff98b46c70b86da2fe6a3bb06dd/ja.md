```
sparseW_sepband(X::Matrix{Float64}, eps::Float64, m̂::Float64, 
D::Matrix{Float64}, N::Matrix{Integer}; NN::Integer = 0, sym::Bool = true)

与えられたϵからガウス多重バンド幅カーネルを用いてスパースカーネル行列Wを計算します。詳細は10.1038/s41467-021-26357-xの補足および関連文献を参照してください。

引数
=================
- X: 元のデータ（埋め込み後）、次元2はサイズNの埋め込み空間
- eps: ガウスカーネルのイプシロン値、厳密に正のFloat64
- m̂: 次元x2の推定値
- D: 距離行列、距離情報は行列Nによって与えられます

キーワード引数
=================
- NN: 使用される最近傍
- sym: オプションの演算子の対称化のためのブール値
```
