```
randomgraph!(net;<keyword arguments>)
```

ネットワーク *net* にERランダムグラフを作成します。

ネットワークは単純グラフであることが保証されているわけではありませんが、大規模なスパースネットワークでは高い確率で単純です。

デフォルトでは、ネットワークが収容できるすべてのノードとリンクが使用され、すべてのノードは状態1に設定されます。この動作は、以下のキーワード引数によって制御できます。

  * N : ランダムグラフの作成に使用されるノードの数。その他のノードはネットワークから削除されます。
  * K : ランダムグラフの作成に使用されるリンクの数。その他のリンクはネットワークから削除されます。
  * S : ノードの状態。すべてのノードはこの状態に設定されます。

# 例

```jldoctest
julia> using Fastnet

julia> net=FastNet(1000,2000,1,[])
ノード0およびリンク0のネットワーク

julia> randomgraph!(net)
ノード1000およびリンク2000のネットワーク

julia> nullgraph!(net)
ノード0およびリンク0のネットワーク

julia> randomgraph!(net,N=100,K=10)
ノード100およびリンク10のネットワーク
```
