A spherical particle.

  * `r`: Radius of the sphere.
  * `x`: x coordinate of the sphere.
  * `y`: y coordinate of the sphere.
  * `z`: z coordinate of the sphere.
  * `m`: Complex refractive index of the sphere. Default is `1.0`.
  * `Î²`: Complex chiral factor of the sphere. Default is `0.0`.
  * `t_matrix`: The T matrix file specifying the scattering properties of the sphere. Default is `""` (empty).
