```
latitudes(genealogy[, vs])
```

系譜の内部頂点の（サブセットの）緯度を返します。

単一の頂点の緯度を取得するには、[`latitude`](@ref)も参照してください。

# 実装

`latitudes(::AbstractGenealogy, ::Any)`のデフォルト実装が利用可能です。`latitudes(::T)`のみが必要です。
