```
H = okid(d::AbstractIdData, nx, l = 5nx; p = 1, λ=0, estimator = /)
```

オブザーバー・カルマンフィルタの同定。マルコフパラメータ（インパルス応答）`H` をサイズ `ny × nu × (l+1)` で返します。

パラメータ `l` は調整が必要な場合が多く、インパルス応答がほぼ消散するのに十分な大きさの `l` を選ぶのが合理的な出発点です。

# 引数:

  * `nx`: モデル次数
  * `l`: 推定するマルコフパラメータの数（インパルス応答の長さ）。
  * `λ`: 正則化パラメータ
  * `smooth`: true の場合、`λ` による正則化は推定されたインパルス応答の曲率にペナルティを与えます。`okid` の後に [`era`](@ref) を使用する場合は、`smooth=true` で小さな `λ` を好みますが、インパルス応答を目視で検査する場合は、より大きなスムージングがインパルス応答の視覚的により正確な推定をもたらすことがあります。
  * `p`: オプションで、初期条件が 0 でないことを考慮して内部ハンケル行列の最初の `p` 列を削除します。`x0 != 0` の場合、`p` を `l` と同じ値に設定してみてください。
  * `estimator`: マルコフパラメータを推定するために使用する関数。デフォルトは `/`（最小二乗）ですが、`TotalLeastSquares.irls / flts` や `TotalLeastSquares.tls` のようなロバストなオプションも使用できます（変数の誤差に対する全最小二乗解）。
