```
era(d::AbstractIdData, nx; m = 2nx, n = 2nx, l = 5nx, p = l, λ=0, smooth=false)
era(ds::Vector{IdData}, nx; m = 2nx, n = 2nx, l = 5nx, p = l, λ=0, smooth=false)
```

固有値実現アルゴリズム。初期ステップとして[`okid`](@ref)を使用してマルコフパラメータを見つけます。

パラメータ`l`は調整が必要な場合があり、インパルス応答がほぼ消失するのに十分大きい`l`を選ぶのが合理的な出発点です。

データセットのベクトルが提供される場合、各実験から推定されたマルコフパラメータは`era`を呼び出す前に平均化されます。これにより、複数の実験からのデータを使用してモデルの推定を改善できます。

# 引数:

  * `nx`: モデル次数
  * `l`: 推定するマルコフパラメータの数。
  * `λ`: 正則化パラメータ（これを過剰に使用しないでください。代わりに、より多くの実験を行うことをお勧めします）
  * `smooth`: trueの場合、`λ`によって与えられる正則化は推定されたインパルス応答の曲率にペナルティを与えます。
  * `p`: オプションで、初期条件が0でないことを考慮して内部ハンケル行列の最初の`p`列を削除します。`x0 != 0`の場合、`era`では`p`は`l`にデフォルト設定され、[`okid`](@ref)を直接呼び出す場合は`p`は0にデフォルト設定されます。
