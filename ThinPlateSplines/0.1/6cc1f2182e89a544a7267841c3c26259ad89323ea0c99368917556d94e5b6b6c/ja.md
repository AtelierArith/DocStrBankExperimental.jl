```
tps_deform(x1, x2, y, λ; compute_affine=true)
```

は、`x1`、`y`、および `λ` に基づいて tps 変形を計算し、それを `x2` に適用します。最初に `tps_solve` が適用され、その後に `tps_deform` が適用されます。

# 引数

```
`x1`: tps 生成のための制御点
`x2`: tps によって変形される点の座標
`y`: tpd 生成のための変形された制御点
`λ`: 剛性係数
`compute_affine`: アフィン変換が計算されるかどうかを定義するオプション引数。
```

返されるのは、行列としての変形された座標です。
