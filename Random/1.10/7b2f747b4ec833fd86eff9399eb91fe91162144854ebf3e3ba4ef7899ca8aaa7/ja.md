```
randn([rng=default_rng()], [T=Float64], [dims...])
```

平均0、標準偏差1のタイプ`T`の正規分布に従う乱数を生成します。オプションで、正規分布に従う乱数の配列を生成します。`Base`モジュールは現在、[`Float16`](@ref)、[`Float32`](@ref)、および[`Float64`](@ref)（デフォルト）のタイプに対する実装を提供しており、それらの[`Complex`](@ref)対応物も含まれています。タイプ引数が複素数の場合、値は分散1の円対称複素正規分布から引き出されます（実部と虚部が平均0、分散`1/2`の独立した正規分布を持つことに対応します）。

インプレースで動作するには、[`randn!`](@ref)も参照してください。

# 例

```jldoctest
julia> using Random

julia> rng = MersenneTwister(1234);

julia> randn(rng, ComplexF64)
0.6133070881429037 - 0.6376291670853887im

julia> randn(rng, ComplexF32, (2, 3))
2×3 Matrix{ComplexF32}:
 -0.349649-0.638457im  0.376756-0.192146im  -0.396334-0.0136413im
  0.611224+1.56403im   0.355204-0.365563im  0.0905552+1.31012im
```
