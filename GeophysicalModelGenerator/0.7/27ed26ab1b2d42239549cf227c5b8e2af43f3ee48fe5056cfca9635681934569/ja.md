```
LithosphericPhases(Layers=[10 20 15], Phases=[1 2 3 4], Tlab=nothing )
```

これにより、層状のリソスフェアを定義できます。層は上から下に向かって定義されます。

# パラメータ

  * Layers : 各層の厚さで、上から下に向かって順番に並べられています。最後の層の厚さは指定する必要はありません。
  * Phases : 層の相で、上から下に向かって順番に並べられています。
  * Tlab   : リソスフェアアセノスフェア境界の温度。指定された場合、T>Tlabの位置にある相はPhases[end]に設定されます。
