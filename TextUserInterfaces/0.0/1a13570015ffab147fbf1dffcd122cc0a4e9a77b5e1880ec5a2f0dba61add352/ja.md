```
function create_field(height::Int, width::Int, y::Int, x::Int, buffer::String = "", id::String = "", offscreen::Int = 0, nbuffers::Int = 0; ...)
```

`id`のフィールドを作成し、高さ`height`と幅`width`を持ち、`y`および`x`座標に配置します。初期バッファ文字列は変数`buffer`で設定できます。オフスクリーン行の数は`offscreen`で設定され、バッファの数は`nbuffers`で設定されます。

# キーワード

  * `color_foreground`: フィールドの前景で使用される色マスク。関数`ncurses_color`を参照してください。負の値の場合、色は変更されません。(**デフォルト** = -1)
  * `color_background`: フィールドの背景で使用される色マスク。関数`ncurses_color`を参照してください。負の値の場合、色は変更されません。(**デフォルト** = -1)
  * `justification`: フォームの整列。左は`:l`、中央は`:c`、右は`:r`です。他のシンボルの場合、左揃えが使用されます。(**デフォルト** = `:l`)
  * `visible`: `true`の場合、コントロールは画面に表示されます。(**デフォルト** = `true`)
  * `active`: `true`の場合、コントロールはアクティブです。(**デフォルト** = `true`)
  * `public`: `true`の場合、フィールドのデータは入力中に表示されます。たとえば、パスワードフィールドの場合は`false`に設定します。(**デフォルト** = `true`)
  * `edit`: `true`の場合、フィールドのデータを変更できます。(**デフォルト** = `true`)
  * `wrap`: `true`の場合、単語は複数行のフィールドで折り返されます。(**デフォルト** = `true`)
  * `blank`: `true`の場合、最初のフィールド位置に文字を入力すると、フィールド全体が消去されます。(**デフォルト** = `false`)
  * `autoskip`: `true`の場合、フィールドが埋まると自動的にスキップされます。(**デフォルト** = `false`)
  * `nullok`: `true`の場合、空のフィールドに対して検証は**適用されません**。(**デフォルト** = `true`)
  * `passok`: `true`の場合、すべての終了時に検証が行われます。そうでない場合、フィールドが変更されたときのみ検証が行われます。(**デフォルト** = `false`)
  * `static`: `true`の場合、フィールドは初期の寸法に固定されます。そうでない場合、入力されたデータに合わせて伸縮します。(**デフォルト** = `true`)
