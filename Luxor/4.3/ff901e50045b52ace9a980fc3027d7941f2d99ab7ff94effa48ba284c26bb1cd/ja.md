```
textextents(str)
```

現在のフォント設定（Toy API）を使用して、文字列 `str` の測定値を含む6つのFloat64の配列を返します：

1. x_bearing
2. y_bearing
3. width
4. height
5. x_advance
6. y_advance

xおよびyのベアリングは、基準点からバウンディングボックスの左上隅までの変位です。xの変位は通常ゼロまたは小さな正の値ですが、「j」のような文字の場合は負のxになることがあります。yの変位はほぼ常に負の値です。

幅と高さはバウンディングボックスのサイズを説明します。アドバンスは次の文字のための推奨基準点に移動します。ベアリングが負である場合や、アドバンスが幅が示唆するよりも小さい場合、後続のテキストブロックのバウンディングボックスが重なる可能性があることに注意してください。

例：

```
textextents("R")
```

は次のように返します

```
[1.18652; -9.68335; 8.04199; 9.68335; 9.74927; 0.0]
```
