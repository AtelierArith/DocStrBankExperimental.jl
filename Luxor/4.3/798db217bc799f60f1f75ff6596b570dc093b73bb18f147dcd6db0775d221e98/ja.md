```
textfit(str, bbox::BoundingBox, maxfontsize = 800;
     horizontalmargin=12,
     leading=100)
```

文字列 `str` をバウンディングボックス `bbox` にフィットさせるために、フォントサイズと行の改行を調整します。

現在のフォントサイズを使用する代わりに、可能な限り大きな値が計算されます。`maxfontsize` でサイズ制限を指定でき、テキストはこの値より大きくなることはありませんが、より小さくなる必要があるかもしれません。

`horizontalmargin` は両側に適用されます。

オプションで、`leading` を指定することができ、これは最終的に計算されたフォントサイズのパーセンテージとして解釈されます。デフォルト値は110（%）です。

この関数は、計算された値に関する情報を持つ名前付きタプルを返します：

```julia
(fontsize = 37.6, linecount = 5, finalpos = Point(-117.43, 92.60)
```

!!! note
    この関数は改善が必要です。特定のフォントでのテキストの行の高さを見つけるのは非常に難しいです。（FreeTypeをインポートしない限り。）改善の提案を歓迎します！
