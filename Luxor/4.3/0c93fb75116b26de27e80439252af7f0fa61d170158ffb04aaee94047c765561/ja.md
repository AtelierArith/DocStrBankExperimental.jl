```
polysuper(center::Point = Point(0, 0);
    n1 = 1,
    n2 = 1,
    n3 = 1,
    m = 2,
    a = 1,
    b = 1,
    radius = 100,
    action = :none,
    vertices = false,
    reversepath = false,
    stepby = π / 120)
```

スーパシェイプを構築し、スーパエリプスの一般化を行い、`action`を適用します。

ヨハン・ギリスによる方程式に基づいています。

参照: `squircle()`...

## 例

```julia
polysuper(m=4,
    n1=13, n2=11, n3=3,
    a=3, b=4,
    action=:fill,
    radius=200)
```

これは奇妙な目/レモンの形を生成し、次のようになります:
