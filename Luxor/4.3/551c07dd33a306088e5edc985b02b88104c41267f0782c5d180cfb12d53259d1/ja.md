```
setantialias(n)
```

現在のアンチエイリアスを0から6の間の値に設定します：

```
antialias_default  = 0, サブシステムおよびターゲットデバイスのデフォルトのアンチエイリアス
antialias_none     = 1, 二値アルファマスクを使用
antialias_gray     = 2, 単色アンチエイリアスを使用（例えば、白い背景に黒いテキストのためのグレーの色合いを使用）
antialias_subpixel = 3, LCDパネルなどのデバイス上のサブピクセル要素の順序を利用
antialias_fast     = 4, いくつかのアンチエイリアスを実行するが、品質よりも速度を優先
antialias_good     = 5, 品質とパフォーマンスのバランスを取る
antialias_best     = 6, 最高の品質でレンダリングし、必要に応じて速度を犠牲にする
```

これはその後のグラフィックスに影響しますが、テキストには影響せず、すべてのタイプの出力ファイルに適用されるわけではありません。
