```
offsetpoly(plist, shape::Function)
```

ポリラインからオフセットされ、囲まれた閉じたポリゴンを返します。

入力された点のセット `plist` はポリラインとして扱われ、ソースポリからオフセットされた閉じたポリゴンを形成する別の点のセットが作成されます。

ポリラインには少なくとも4つの点が必要です。

この `offsetpoly()` メソッドは、点のリストを `n` 頂点として扱い、`n - 1` 本の線で接続します。（別のメソッド `offsetpoly(plist, d)` は、点のリストを `n` 頂点として扱い、`n` 本の線で接続します。）

提供された関数は線の幅を決定します。`f(0, θ)` は開始時の幅を与え（その点での曲線の傾きは θ で提供されます）、`f(1, θ)` は終了時の幅を提供し、`f(n, θ)` は点 `n/l` の幅です。

# 例

この例では、両端が20（10 + 10）単位の幅で始まり、中央で50（10 + 10 + 15 + 15）に膨らむチルダを描きます。ここで、f(0.5) = 25 です。

```julia
f(x, θ) = 10 + 15sin(x * π)
sinecurve = [Point(50x, 50sin(x)) for x in (-π):(π / 24):π]
pgon = offsetpoly(sinecurve, f)
poly(pgon, :fill)
```

この例では、曲線の垂直部分を強調し、水平方向の部分を細くします。

```julia
g(x, θ) = rescale(abs(sin(θ)), 0, 1, 0.1, 30)
sinecurve = [Point(50x, 50sin(x)) for x in (-π):(π / 24):π]
pgon = offsetpoly(sinecurve, g)
poly(pgon, :fill)
```
