```
offsetpoly(plist;
    startoffset = 10,
    endoffset   = 10,
    easingfunction = lineartween)
```

オープンポリゴンからオフセットされた閉じたポリゴンを返します。

入力された点のセット `plist` はオープンポリゴンとして扱われ、ソースポリゴンから `...offset` 単位離れた位置にあるポリゴンを形成する別の点のセットが作成されます。

`offsetpoly()` メソッドは、点のリストを `n` 頂点として扱い、`n - 1` 本の線で接続します。これにより、線の始点から終点までのオフセットを変えることができます。

もう一つのメソッド `offsetpoly(plist, d)` は、点のリストを `n` 頂点として扱い、`n` 本の線で接続します。

# 拡張ヘルプ

この関数は、Luxor に組み込まれたイージング機能を使用してオフセットを制御するためのキーワード引数を受け入れます。デフォルトでは関数は `lineartween()` であり、オフセットは `startoffset` と `endoffset` の間で線形に変化します。関数：

```julia
f(a, b, c, d) = 2sin((a * π))
```

は、`a` が 0 から 1 に変化する間、0 から 2 まで行き、再び戻ります。この量でオフセットがスケーリングされます。
