```
intersectioncirclecircle(cp1, r1, cp2, r2)
```

2つの円が交差する点を見つけます。最初の円は `cp1` を中心とし、半径 `r1` を持ち、2番目の円は `cp1` を中心とし、半径 `r1` を持ちます。

返す値

```
(flag, ip1, ip2)
```

ここで、`flag` は円が点 `ip1` と `ip2` で交差する場合は真のブール値 `true` です。円が全く交差しない場合、または一方が完全に他方の内部にある場合、`flag` は `false` となり、両方の点は Point(0, 0) になります。

2つの重なり合う円の面積を求めるには `intersection2circles()` を使用してください。

純粋な数学の世界では、2つの円が「キス」している場合、単一の交差点しか持たないことが可能であるはずです。現在、このロマンチックでない関数は、2つのキスしている円が交差点を持たないと報告します。
