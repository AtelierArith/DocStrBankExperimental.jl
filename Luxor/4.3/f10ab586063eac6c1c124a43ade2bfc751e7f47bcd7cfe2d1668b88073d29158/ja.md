```
hypotrochoid(R, r, d;
    action=:none,
    stepby=0.01,
    period=0.0,
    vertices=false)
hypotrochoid(R, r, d, action;
    stepby=0.01,
    period=0.0,
    vertices=false)
```

短い線分でハイポトロコイドを作成し、現在のパスに追加します。（スピログラフのように。）この曲線は、半径 `r` の円が半径 `R` の固定円の内側を転がるときに、その中心から距離 `d` の点によって描かれます。非整数値を指定すると、面白いことが起こります。

特別なケースには、`d` = `r` の場合のハイポサイクロイドや、`R` = `2r` の場合の楕円があります。

`stepby` は角度のステップ値で、ポリゴンの詳細度、すなわち滑らかさを制御します。

`period` が指定されていない場合、または 0 の場合、最小の周期が自動的に計算されます。

この関数はポリゴン（点のリスト）を返すか、指定された `action` を使用して点を直接描画します。点が描画される場合、関数は描画された点の数と周期が `π` の倍数であったことを示すタプルを返します。
