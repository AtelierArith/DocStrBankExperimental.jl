```
hexcube_round(x, y, origin, width = 10.0, height = 10.0)
```

Return the hexagon containing the point `x`, `y`, on the hexagonal grid centered at `origin`, and with tiles of `width`/`height` point in Cartesian space can be mapped to the index of the hexagon that contains it.

For more information about making hexagons and hexagonal grids, see [Luxor.Hexagon](@ref).
