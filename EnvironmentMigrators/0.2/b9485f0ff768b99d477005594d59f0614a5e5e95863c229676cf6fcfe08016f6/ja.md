```
backup(m::AbstractEnvironmentMigrator)
```

環境をバックアップします。通常、これは Project.toml と Manifest.toml をタイムスタンプ付きのバックアップサブフォルダーにコピーすることを含みます。

# 拡張ヘルプ

標準のバックアップ手順は次のとおりです。

1. 対象のプロジェクトとマニフェスト TOML が同じディレクトリにあることを確認します。
2. プロジェクトまたはマニフェスト TOML のいずれかが存在することを確認します。
3. "yyyy-mm-dd*HH*MM_SS" 形式のタイムスタンプを作成します。
4. プロジェクト TOML と同じディレクトリに `backups/[timestamp]` というサブパスを作成します。
5. プロジェクトとマニフェスト TOML が存在する場合、バックアップパスにコピー（または移動）します。
