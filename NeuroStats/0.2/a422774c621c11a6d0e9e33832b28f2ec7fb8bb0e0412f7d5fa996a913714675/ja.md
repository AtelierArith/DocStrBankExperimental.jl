```
f1(; <キーワード引数>)
```

F1スコアを使用して分類モデルのパフォーマンスを評価します。F1スコアの値は0から1の範囲です。

# 引数

  * `tp::Int64`: 真陽性の数
  * `tn::Int64`: 真陰性の数
  * `fp::Int64`: 偽陽性の数
  * `fn::Int64`: 偽陰性の数

# 戻り値

名前付きタプルには以下が含まれます：

  * `f1::Float64`: F1スコア
  * `p::Float64`: 精度
  * `r::Float64`: 再現率
