```
friedman(m)
```

フリードマンの非パラメトリック二元分散分析（およびケンドールの一致係数、正規化されたフリードマンのQ統計量）を推定します。

# 引数

  * `m::AbstractArray`: 値 × グループ

# 戻り値

名前付きタプルを含む:

  * `q::Float64`: フリードマンのQ統計量
  * `w::Float64`: ケンドールの一致係数
  * `p::Float64`: p値

# 注意事項

  * H0 (フリードマン) は、処置が等しいことです
  * H0 (ケンドール) は、ランキングまたはテスト結果の間に合意があることです
  * ケンドールの一致係数は0から1の範囲で、0は評価者（審査員）間に合意がないことを意味します
