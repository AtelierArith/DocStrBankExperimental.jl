```
res_norm(x, g)
```

残差の正規分布をテストします。

# 引数

  * `x::AbstractVector`: データ値
  * `g::Vector{Int64}`: 各データ値が属するグループ

# 戻り値

名前付きタプルを含みます：

  * `adt_p::Vector{Float64}`: 正規分布に対するkサンプルアンダーソン–ダーリングテストのp値
  * `ks_p::Vector{Float64}`: 正規分布に対する1サンプルの正確なコルモゴロフ–スミルノフテストのp値

# 注意事項

p値は各グループおよび全体サンプルに対して報告されます。グループが1つだけの場合、p値は全体サンプルのp値のみが報告されます。
