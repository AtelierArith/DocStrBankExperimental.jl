```
power_c2g(; <キーワード引数>)
```

連続変数の研究力を計算します（グループ1対グループ2）。

# 引数

  * `m1::Real`: 研究グループ1の平均
  * `s1::Real`: 研究グループ1の標準偏差
  * `n1::Int64`: 研究グループ1のサンプルサイズ
  * `m2::Real`: 研究グループ2の平均
  * `s2::Real`: 研究グループ2の標準偏差
  * `n2::Int64`: 研究グループ2のサンプルサイズ
  * `alpha::Float64=0.05`: 第I種過誤の確率

# 戻り値

  * `p::Float64`: 研究力
