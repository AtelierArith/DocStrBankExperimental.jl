```
deltacomplex(s :: Vector{<:Integer})
```

単一の点を持つ向きのある表面の三角形分割を作成します。対応する辺を接合することによって行います。

`s` は、反時計回りの順序で多角形の辺を表す非ゼロ整数の配列である必要があります。 i 番目の辺は、`s[i]>0` の場合は反時計回りに向いており、`s[i]<0` の場合は時計回りに向いています。 `s[i]` と `s[j]` が同じ絶対値を持つ場合、それらは向きを尊重しながら接合されます。

# 例

次の結果は、1つの点を持つ *トーラス* の三角形分割になります：

```julia-rep
julia> deltacomplex([1,2,-1,-2])
```

次の結果は、1つの点を持つ *クラインボトル* の三角形分割になります：

```julia-rep
julia> deltacomplex([1,2,-1,2])
```
