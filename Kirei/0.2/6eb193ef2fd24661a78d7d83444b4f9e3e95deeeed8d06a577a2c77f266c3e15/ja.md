```
@head_default f([io::IO=stdout], a::Int, b=0)
```

引数リストの先頭にデフォルト値を持つ関数 `f` を生成します。曖昧さを防ぐために、いくつかの引数の型を指定する必要があります。

通常、ユニークな型を持つ引数のデフォルトは最大で1つだけにすることが推奨されます。

上記は次のように等価です。

```julia
f(io::IO, a::Int) = f(io, a, 0)
f(a::Int, b) = f(stdout, a, b)
f(a::Int) = f(stdout, a, 0)
```

関数定義の前に `@head_default` を使用することもできます。 ```
