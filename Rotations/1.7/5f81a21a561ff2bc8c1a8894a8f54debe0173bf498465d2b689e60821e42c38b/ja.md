```
QuatRotation{T} <: Rotation
```

4パラメータの姿勢表現で、特異点がありません。単位ノルムのクォータニオンはSO(3)のダブルカバーを表します。`QuatRotation`は単位ノルム制約を厳密には強制しませんが、特定のメソッドは単位クォータニオンを持っていると仮定します。クォータニオンに対してハミルトンの規約に従います。

# コンストラクタ

```julia
QuatRotation(w,x,y,z)
QuatRotation(q::AbstractVector)
```

ここで、`w`はスカラー（実数）部分、`x`、`y`、および`z`はベクトル（虚数）部分であり、`q = [w,x,y,z]`です。
