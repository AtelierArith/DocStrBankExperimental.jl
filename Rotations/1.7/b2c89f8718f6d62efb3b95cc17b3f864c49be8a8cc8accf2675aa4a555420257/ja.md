```
struct AngleAxis{T} <: Rotation{3,T}
AngleAxis(Θ, x, y, z)
```

任意の軸 `[x, y, z]` の周りの角度 θ による 3D 回転によってパラメータ化された 3×3 回転行列。

θ = 0 の場合、軸は一意ではなく、このパラメータ化は恒等回転の近傍を連続的にマッピングしないため（したがって、自動微分や最適化の目的には適さない可能性があります）。

注意: デフォルトでは、コンストラクタは入力を再正規化して、軸の長さが 1 になるようにします（x² + y² + z² = 1）。

再正規化は、追加のコンストラクタ引数として `false` を渡すことでスキップできます。この場合、ユーザーは入力引数が正規化された回転軸を表すことを保証します。この方法で再正規化をスキップして作成された、単位ノルムを持たない回転軸を持つ `AngleAxis` に対する操作は、意味のある結果を保証するものではありません。
