```
value(q)
value(q, pos)
```

与えられた位置 `pos` での変動量 `q` の値を取得します。`pos` は `(x, y, z)` タプル、`(x, z)` タプル、または `z` 値として指定できます。

# 例

```julia
value(q)             # 定数量の値を取得
value(q, -10)        # z=-10 での深さ依存量の値を取得
value(q, (1000,-10)) # x=1000, z=-10 での位置依存量の値を取得
value(q, (0,0,-10))  # (0,0,-10) での位置依存量の値を取得
```
