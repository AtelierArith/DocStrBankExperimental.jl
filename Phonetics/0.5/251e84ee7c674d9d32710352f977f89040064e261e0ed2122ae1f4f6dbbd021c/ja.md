```
vowelplot(f1, f2, cats; meansOnly=false, addLabels=true, ell=false, ellPercent=0.67, nEllPts=500, kw...)
```

F1-by-F2の母音プロットを作成します。`f1`の値はx軸に表示され、`f2`の値はy軸に表示され、`cats`内の各ユニークな母音クラスは新しい色で表されます。凡例の系列ラベルは`cats`に含まれるユニークな値を取ります。F2がx軸に、F1がy軸に逆転して表示される代替表示は、`f1`引数にF2の値を渡し、`f2`引数にF1の値を渡し、その後`Plots`パッケージによって提供される`:flip`マジック引数を使用することで作成できます。

`meansOnly`がtrueに設定されている場合、各母音カテゴリの平均値のみがプロットされます。`ell=true`を使用すると、`ellPercent`で指定されたデータの割合を大まかに包含するデータエリプスがプロットされます。エリプスは、`nEllPts`で指定された数の点によって表されます。`plot`への他の引数は、スプラットされた`kw`引数を通じて渡されます。`addLabels`引数をtrueに設定すると、平均の上部と右側に母音カテゴリのテキストラベルが追加されます。

引数の構造はプロットレシピの使用から推測されます。`xlim`、`ylim`、`color`、および`size`などのパラメータは、標準的な`plot`呼び出しと同様にキーワード引数として渡す必要があります。プロットパラメータ`markersize`のデフォルトは`3`、`linewidth`のデフォルトは`3`です。

# Args

  * `f1` x軸にプロットする値、またはF1の値
  * `f2` y軸にプロットする値、またはF2の値
  * `cats` 各F1、F2ペアに関連付けられた母音カテゴリ
  * `meansOnly` 各カテゴリの平均値のみをプロット
  * `addLabels` 平均の近くに各カテゴリのラベルをプロットに追加
  * `ell` プロットにデータエリプスを追加するかどうか
  * `ellPercent` エリプスがカバーすべきデータ分布の割合（おおよそ）
  * `nEllPts` エリプスをプロットする際に使用する点の数
