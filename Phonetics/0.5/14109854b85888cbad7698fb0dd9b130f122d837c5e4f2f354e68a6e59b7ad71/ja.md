```
acdist(s1, s2; [method=:dtw, dist=SqEuclidean(), radius=10])
```

`s1` と `s2` の間の音響距離を計算します。`method` は動的時間伸縮のバージョンで、`dist` は内部距離関数です。`method=:dtw` を使用すると、通常の動的時間伸縮が使用され、`method=:fastdtw` を使用すると、ファスト dtw の近似が使用されます。動的時間伸縮は三角不等式を必ずしも満たさず、不可分の同一性を保証しないため、これは真の数学的距離メトリックではないことに注意してください。

# 引数

  * `s1` 比較する最初の音の特徴-時間配列
  * `s2` 比較する2番目の音の特徴-時間配列
  * `method` (キーワード) 使用する動的時間伸縮の方法
  * `dist` (キーワード) `Distances` パッケージの `SemiMetric` インターフェースを実装する任意の距離関数
  * `dtwradius` (キーワード) 通常の動的時間伸縮の最大ワーピング半径; 値が渡されない場合、ワーピング制約は使用されません。method=:fastdtw の場合は引数は未使用
  * `fastradius` (キーワード) ファスト dtw メソッドに使用する半径; method=:dtw の場合は引数は未使用
