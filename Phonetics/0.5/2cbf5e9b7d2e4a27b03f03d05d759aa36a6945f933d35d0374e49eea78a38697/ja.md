```
lev(s, t)
```

Levenshtein距離は、文字列だけでなく任意の反復可能なオブジェクトに対して計算されます。ウィキペディアの[Levenshtein距離のページ](https://en.wikipedia.org/wiki/Levenshtein_distance)にある擬似コードから実装されています。

# パラメータ

  * **s** 比較のための最初の反復可能なオブジェクト。文字列、配列、タプルなど、インデックス指定が可能であれば何でも構いません。
  * **t** 比較のための2番目の反復可能なオブジェクト。文字列、配列、タプルなど、インデックス指定が可能であれば何でも構いません。

# 戻り値

  * `s` と `t` の間の計算されたLevenshtein距離。
