```
pnd(corpus::Array, queries::Array; [progress=true])
```

`corpus`内のアイテムに基づいて、`queries`内の各アイテムの音韻的隣接密度（pnd）を計算します。この関数は、隣接アイテムの検索を迅速化するために、視点ツリーのデータ構造を使用して検索空間を剪定します。この関数は、`queries`内のアイテムが`corpus`に存在するかどうかに関係なく機能するはずです。

# パラメータ

  * **corpus** 音韻的隣接を照会するためのコーパス
  * **queries** `corpus`内の音韻的隣接を照会するアイテム
  * **progress** 進行メーターを表示するかどうか

  * 最初の列にクエリ、2番目の列に音韻的隣接密度を持つ`DataFrame`
