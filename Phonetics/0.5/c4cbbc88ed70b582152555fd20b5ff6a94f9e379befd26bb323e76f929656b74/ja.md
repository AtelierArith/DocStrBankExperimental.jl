```
neareyI(f1, f2, vowel, speaker)
```

Nearey形式のフォルマント内在的正規化ルーチンを実行し、フォルマントを記録し、対数F1値から対数F1値の平均値を引き、対数F2値からF2値の平均値を引きます。詳細については、Nearey (1978, 母音の音声特徴システム, インディアナ大学言語学クラブ) を参照してください。

# 引数

  * `f1`  F1値
  * `f2` F2値
  * `vowel` 母音カテゴリ; 計算には使用されませんが、適切なフォルマントとスピーカー情報にリンクさせるために渡されます
  * `speaker` スピーカーIDの`Array`; 整数、シンボル、文字列、及び`DataFrames`パッケージの`groupby`関数でサポートされている他のデータ型を使用できます
