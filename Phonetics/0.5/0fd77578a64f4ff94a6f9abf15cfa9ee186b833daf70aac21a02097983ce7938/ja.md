```
upt(corpus, queries; [inCorpus=true])
```

`corpus`内のアイテムに基づいて、`queries`内のアイテムの音韻的ユニークネスポイント（upt）を計算します。アイテムがコーパスに存在することが期待される場合、この関数は、分岐が1つの単語のみを表すと見なされるユニークネスポイントを計算します。アイテムがコーパスに存在しないことが期待される場合、ユニークネスポイントは、ツリーをこれ以上トラバースできなくなる深さとして取られます。

# パラメータ

  * **corpus** ユニークネスポイントを計算する際に比較対象となるコーパスを構成するアイテム
  * **queries** ユニークネスポイントを計算するアイテム
  * **inLexicon** クエリアイテムがコーパスに存在することが期待されるかどうか

# 戻り値

  * 最初の列にクエリ、2番目の列にユニークネスポイントを持つ`DataFrame`
