```
PointFluxes([fluxes, points; offset=(0, 0), gauge=:axial])
```

与えられたフラックスとポイントを使用して `PointFluxes` オブジェクトを構築します。

オプションの `gauge` 引数は、フィールドのゲージを指定するために使用できます。

## 引数

  * `fluxes`: フラックス値のベクトル。単一の値でも構いません。その場合、すべてのポイントにブロードキャストされます。
  * `points`: ポイントのベクトルまたは `AbstractLattice` オブジェクト。後者の場合、サイトはポイントとして解釈されます。

両方の引数が省略された場合、空のフィールドが作成されます。後で `push!` または `append!` を使用してフラックスを追加できます。

## キーワード引数

  * `offset`: ポイントに追加するオフセット、デフォルトは `(0, 0)` です。`points` が `AbstractLattice` の場合のみ有効で、それ以外の場合はエラーがスローされます。
  * `gauge`: フィールドのゲージ。可能な値は `:axial` と `:singular` です。デフォルトは `:axial` です。
