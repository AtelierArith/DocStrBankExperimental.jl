```
UnitCell(translations[, basis; offset, rotate])
```

与えられた変換ベクトルと基底サイトの位置を持つブラヴァイ格子単位セルを構築します。

## 引数

  * `translations`: 格子の変換ベクトルを表すサイズ `N×N` の `AbstractMatrix`。
  * `basis`: 基底サイトの位置を表すサイズ `N×NB` の `AbstractMatrix`。 提供されない場合、格子基底は単位セルの左下隅に位置する1つのサイトで構成されます。

## キーワード引数

  * `offset`: 格子基底をシフトする方法を指定するキーワード引数。 可能な値：

      * `:origin`: シフトなし（デフォルト）。
      * `:center`: 基底の中心が単位セルの原点に来るように格子をシフトします。
      * `:centeralign`: 基底の中心が単位セルの中心に来るように格子をシフトします。
      * カスタムベクトルで格子をシフトするためにサイズ `N` の `AbstractVector` も受け入れます。
  * `rotate`: 格子基底を回転する方法を指定するキーワード引数。 可能な値：

      * `nothing`: 回転なし（デフォルト）。
      * 格子を回転させるためのサイズ `N×N` の `AbstractMatrix`。
      * この角度をラジアンで回転させるための `Real` 数。
      * 格子基底を回転させるためのサイズ `N×N` の `AbstractMatrix` も受け入れます。
