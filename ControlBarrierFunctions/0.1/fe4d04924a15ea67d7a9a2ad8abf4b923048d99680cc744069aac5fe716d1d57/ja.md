```
制御アフィンシステム
```

ダイナミクス $ẋ = f(x) + g(x)u$ で記述される制御アフィンシステム。ここで、$x ∈ ℝⁿ$ はシステム状態、$u ∈ ℝᵐ$ は制御入力、$f : ℝⁿ → ℝⁿ$ および $g : ℝⁿ → ℝⁿˣᵐ$ はダイナミクスを表します。

# フィールド

  * `name::String` : システムの名前、例："ダブルインテグレーター"、"ユニサイクル"、"クアッドローター" など。
  * `n::Int` : 状態次元
  * `m::Int` : 制御次元
  * `f::Function` : ドリフトダイナミクス
  * `g::Function` : 制御方向
