```julia
struct ParMM{S<:AbstractFloat}
```

# 説明

マッチングモーメント推定入力を格納するための構造体。

# フィールド

  * `lb_global`: グローバルステージにおけるパラメータの下限
  * `ub_global`: グローバルステージにおけるパラメータの上限
  * `lb_hard`: パラメータのハード下限 - 外れるとローカルステージでペナルティが発生
  * `ub_hard`: パラメータのハード上限 - 外れるとローカルステージでペナルティが発生
  * `labels`: 推定されたパラメータのラベル
  * `momdat`: 一致させるデータからのモーメント
  * `mmomdat`: 一致させるデータからの正規化因子モーメント
  * `W`: 重み行列
  * `mdifrec`: モーメント条件を再中心化するための事前に定められた量
