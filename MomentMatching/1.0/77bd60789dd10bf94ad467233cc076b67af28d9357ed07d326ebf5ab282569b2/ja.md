```julia
struct ComputationSettings{T<:Integer}
```

# 説明

計算設定

# フィールド

  * `location`: 計算が行われる場所。現在は 'local' と 'slurm' がサポートされています。デフォルト: local
  * `num_procs`: プロセスの数。1を指定するとマルチプロセッシングを回避します（ワーカーを1つ追加するだけではパフォーマンスに悪影響を及ぼすため、マスターはループで使用されません）。クラスターではノードの数を指定してください（これを再確認する必要があります）。デフォルト: 1
  * `num_tasks`: プロセスごとのタスクの数。実際の（仮想または物理??）スレッドの数よりもやや多く指定するのが良いでしょう。デフォルト: Threads.nthreads() * 2
  * `num_threads`: 各プロセスが開始されるスレッドの数。デフォルト: num_tasks
  * `maxmem`: このメモリ使用量で集中的なガーベジコレクションをトリガーします。デフォルト: -1
  * `clustermanager_settings`: その他の設定。デフォルト: Dict(:x => "")
