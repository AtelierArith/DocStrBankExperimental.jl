```
SLD(classifier; kwargs...)
```

Saerens-Latinne-Decaestecker法、別名EMQまたはSaerensらによる期待値最大化に基づく定量化（2002年）：*新しい事前確率に対する分類器の出力を調整する：シンプルな手続き*。

多項式の次数 `o > 0` と正則化の影響 `λ > 0` は、Bunseらによって提案された順序定量化のためのo-SLD法を生み出します（2022年）：*宇宙粒子物理学における知識獲得のための機械学習*。

**キーワード引数**

  * `o = 0` はo-SLDの多項式の次数です。
  * `λ = 0.0` はo-SLDの多項式の影響です。
  * `a = Float64[]` は展開分析のための受容因子です。
  * `fit_classifier = true` は与えられた`classifier`をフィットさせるかどうかです。
