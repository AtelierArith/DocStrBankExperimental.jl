```
SVD(transformer; kwargs...)
```

ホエッカーとカートヴェリシヴィリによる特異値分解に基づく展開法、1996年: *データ展開へのSVDアプローチ*。

**キーワード引数**

  * `strategy = :softmax` は解法戦略です（下記参照）。
  * `τ = 1e-6` は順序量化のための正則化強度です。
  * `n_df = -1` （`strategy==:original` の場合のみ使用）は有効ランクで、`0 < n_df < C` である必要があります。ここで `C` はクラスの数です。
  * `a = Float64[]` は展開分析のための受容因子です。

**戦略**

ホエッカーとカートヴェリシヴィリの損失関数、特徴変換、および正則化は、複数の戦略で最適化できます。

  * `:softmax` （デフォルト; 我々の方法）は、技術的な正則化項を導入する代わりに、1つの潜在パラメータをゼロに設定することで `:softmax_full_reg` を改善します。
  * `:original` はホエッカーとカートヴェリシヴィリ（1996年）によって提案された元の解析的解法です。
  * `:constrained` は、ホプキンズとキングによって提案された適切な確率密度に最適化を制約します。2010年: *社会科学のための自動非パラメトリックコンテンツ分析の方法*。
  * `:softmax_full_reg` （我々の方法）は、制約を無効にするソフトマックス層を導入します。この戦略は、バンセによって提案された技術的な正則化項を使用します。2022年: *調整された分類とカウントの多クラス拡張について*。
  * `:softmax_reg` （我々の方法）は、技術的な正則化項を導入するのに加えて、1つの潜在パラメータをゼロに設定する `:softmax` の変種です。
  * `:unconstrained` （我々の方法）は `:original` に似ていますが、より一般的なソルバーを使用します。
