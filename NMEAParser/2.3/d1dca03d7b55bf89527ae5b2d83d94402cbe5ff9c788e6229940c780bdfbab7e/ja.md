```
parse_msg!(s::NMEAData, line::AbstractString)
```

NMEA 0183データの行を解析し、NMEADataオブジェクトの状態を更新します。

# 引数

s : NMEAData 	NMEAメッセージから解析されたデータを格納するオブジェクト。 line : AbstractString 	有効なNMEA 0183メッセージを含む文字列。

# 戻り値

DataType 	解析されたメッセージの型、またはメッセージがサポートされていない場合はNothing。

# エラー

ArgumentError 	行が有効なNMEA 0183メッセージでない場合。
