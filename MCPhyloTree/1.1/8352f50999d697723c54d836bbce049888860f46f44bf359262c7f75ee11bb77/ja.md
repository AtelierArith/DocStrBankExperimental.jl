```
function cov2tree(covmat::Array{<:T, 2}, names::Vector{<:AbstractString}, numbers::Vector{Int64}; tol::Real=1e-7)::GeneralNode{T, Int64} where T<:Real
```

この関数は、共分散行列からツリーを再構築します。共分散行列、葉の名前のベクトル、およびノード番号のベクトルを必須引数として受け取ります。2つのベクトルの順序は、共分散行列の行と列の順序に対応している必要があります。オプションで、`tol` パラメータは、すべての値がゼロとして扱われる境界を示します。

供給された共分散行列に対応するツリーのルートノードを返します。

  * `covmat` : 共分散行列
  * `names` : 行列の列/行インデックスに一致する名前のリスト
  * `numbers` : 行列の列/行インデックスに一致する整数のリスト
  * `tol` : すべての値がゼロとして扱われるカットオフ値
