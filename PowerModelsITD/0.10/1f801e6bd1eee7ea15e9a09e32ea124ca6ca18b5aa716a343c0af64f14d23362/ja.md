```
function replicate(
    sn_data::Dict{String,<:Any},
    count::Int;
    global_keys::Set{String}=Set{String}()
)
```

与えられた単一ネットワーク pmitd データを、指定されたネットワークの `count` 複製を持つマルチネットワークデータに変換します。この関数はネットワークデータのディープコピーを実行することに注意してください。データの複製を最小限に抑えたアプリケーション特有のマルチネットワーク構築方法を構築することで、しばしば重要なマルチネットワークスペースの節約が達成できます。`sn_data` は複製されるデータであり、`count` は複製されるネットワークの数です。
