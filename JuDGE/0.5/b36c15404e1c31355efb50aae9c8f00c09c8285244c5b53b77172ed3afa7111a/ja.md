```
tree_from_file(filename::String)
```

ファイルからツリーを構築します。ファイル内の各行は、ノード "A" からノード "B" への弧を表す形式 B,A,... です。列の総数は任意です。ファイルの最初の行は n,p,... であり、これらの列ヘッダーは `data` をインデックスするためにシンボルに変換されます。各列自体は辞書に変換され、ノードによってインデックスされます。

### 必要な引数

`string` はツリーを含むファイルのフルパスです。

### 例

```
tree, data = tree_from_file(joinpath(@__DIR__,"tree.csv"))
```
