```
find_wrapped(nav::Array{Float64, 4}, nav_time::Array{Float64, 2}, trace::Array{Float64, 2}, slices::Int64)
```

ナビゲーター位相推定におけるラップされたポイントの位置を特定します。呼吸ベルトの記録が必要です。ラップされたポイントの位置、各ナビゲータースライスとトレースデータとの相関、整列および補間されたトレースデータを返します。

# 引数

  * `nav::Array{Float64, 4}`      - ナビゲーター位相推定
  * `nav_time::Array{Float64, 2}` - 各スライスの1日の開始からのミリ秒単位のナビゲーターデータのタイムスタンプ
  * `trace::Array{Float64, 2}`    - 生理学的トレース記録。2列のベクトル（1:時間 [ms], 2:トレース）。最初の列には1日の開始からのミリ秒単位のタイムスタンプが含まれています。                                   画像取得の前後の時間点を含めてください（少なくとも2秒）。
  * `slices::Int64`               - スライスの数
