```
cubic_spline_τ_to_ωn!(
    Cn::AbstractVector{Complex{E}},
    Cτ::AbstractVector{T},
    β::E, Δτ::E;
    spline_type::String = "C2",
    M1::T = NaN,
    M2::T = NaN
) where {E<:AbstractFloat, T<:Number}
```

マツバラ周波数表現を計算します

$$
\begin{align*}
C(\mathrm{i}\omega_n) = \int_0^\beta d\tau \ e^{\mathrm{i}\omega_n\tau} C(\tau)
\end{align*}
$$

は、$L + 1$ の虚時間点 $\tau \in \{ 0, \Delta\tau, 2\Delta\tau, \ldots, (\beta-\Delta\tau), \beta \}$ の正規グリッド上で定義された虚時間相関関数 $C(\tau)$ のものです。これは、$C(\tau)$ 点を通る三次スプラインをフィッティングし、その後スプラインを変換することで行われます。ここで、マツバラ相関 $C(\mathrm{i}\omega_n)$ はベクトル `Cn` に書き込まれ、虚時間相関 $C(\tau)$ はベクトル `Cτ` に格納されます。`C[1]` と `C[end]` がそれぞれ $C(\tau = 0)$ と $C(\tau = \beta)$ に対応することが仮定されています。また、$L = \beta/\Delta\tau$ であることも仮定されています。

ベクトル `Cn` の長さは、相関関数がフェルミオン的であるかボソニックであるかを決定します。

フェルミオン相関関数の場合、`mod(M, 2L) == 0` であり、ここで `M = length(Cn)` および `L = length(Cτ) - 1` です。言い換えれば、`length(Cn) == 2*n*(length(Cτ)-1)` である必要があり、ここで `n` は正の整数で $n ≥ 1$ です。この場合、ベクトル `Cn` には $n \in [-N, (N-1)]$ の範囲のマツバラ周波数相関 $C(\mathrm{i}\omega_n)$ が含まれ、ここで `N = M÷2` および $\omega_n = (2n+1)\pi/\beta$ です。

ボソニック相関関数の場合、`mod(M+1, 2L) == 0` であり、ここで `M = length(Cn)` および `L = length(Cτ) - 1` です。言い換えれば、`length(Cn) == 2*n*(length(Cτ)-1) - 1` である必要があり、ここで `n` は正の整数で $n ≥ 1$ です。この場合、ベクトル `Cn` には $n \in [-N, N]$ の範囲のマツバラ周波数相関 $C(\mathrm{i}\omega_n)$ が含まれ、ここで `N = (M-1)÷2` および $\omega_n = 2n\pi/\beta$ です。

`spline_type` 引数は、使用するスプラインのタイプを指定するために `"C2"`、`"akima"`、または `"makima"` に設定できます。デフォルトの引数（ほとんどのケースで最適）は `"C2"` で、これは連続した一次および二次導関数を持つ C2 三次スプラインを指します。この場合、境界条件が課せられ、`τ=0` および `τ=β` でのスプラインの二次導関数が、それぞれ二次前進および後退有限差分を使用して計算された二次導関数に等しく設定されます。

`M1` および `M2` 引数は、既知であればスペクトル関数の第一および第二モーメントを指定するために使用できます。デフォルトでは `NaN` に設定されており、これにより虚時間相関関数データポイントと対応するスプラインフィットに基づいて内部的に計算されます。
