ベースマップの上にネットワークをマップする

ネットワークプロットのための受け入れられたキーワード引数:

  * `lines::Bool` : 線を表示
  * `linealpha::Union{Float, Vector{Float}}` : 線の透明度
  * `nodesize::Union{Float, Vector{Float}}` : ノードのサイズ
  * `nodehover::Union{String, Vector{String}}` : ノードのホバーテキスト
  * `linewidth::Float` : 線の幅
  * `linecolor::String` : 線の色
  * `buffer::Float`
  * `size::Tuple` : 図のサイズ
  * `xlim::Tuple` : x軸を切り取る
  * `ylim::Tuple` : y軸を切り取る
  * `showleged::Bool` : 凡例を表示
  * `nodecolor::String` : ノードの色
  * `nodealpha::Float` : ノードの透明度
  * `legend_font_color::String` : 凡例のフォントの色

マップのための受け入れられたキーワード引数:

  * `map_lines::Bool` : 線を表示
  * `map_linealpha::Union{Float, Vector{Float}}` : 線の透明度
  * `map_nodesize::Union{Float, Vector{Float}}` : ノードのサイズ
  * `map_nodehover::Union{String, Vector{String}}` : ノードのホバーテキスト
  * `map_linewidth::Float` : 線の幅
  * `map_linecolor::String` : 線の色
  * `map_buffer::Float`
  * `map_size::Tuple` : 図のサイズ
  * `map_xlim::Tuple` : x軸を切り取る
  * `map_ylim::Tuple` : y軸を切り取る
  * `map_showleged::Bool` : 凡例を表示
  * `map_nodecolor::String` : ノードの色
  * `map_nodealpha::Float` : ノードの透明度
  * `map_legend_font_color::String` : 凡例のフォントの色
