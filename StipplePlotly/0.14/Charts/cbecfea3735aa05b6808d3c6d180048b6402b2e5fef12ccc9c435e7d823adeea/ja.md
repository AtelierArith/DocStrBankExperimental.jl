```
function watchplot(selector::AbstractString)
```

DOM要素のプロットイベントをそれぞれのモデルフィールド（例：`plot_selected`、`plot_hover`など）に転送するjsスクリプトを生成します。プレフィックスが指定されていない場合は、クラスリストから取得され、少なくとも1つのエントリ `'sync_[prefix]'`（例：`sync_plot`）を含む必要があります。この関数は、すでにUIに存在するプロットに対して動作します。これは、バックエンドリスナーでのプロット特有のイベントバインディングの稀なケースを目的としています。

プロットイベントを転送する通常の方法は、`js_mounted()`内で`watchplots()`を呼び出すことです。
