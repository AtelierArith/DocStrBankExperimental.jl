# プロパティ

---

  * `columns::String` - グリッドの列数。2D `subplots` 配列を提供した場合、その最長の行の長さがデフォルトとして使用されます。`xaxes` 配列を与えた場合、その長さがデフォルトとして使用されます。しかし、非直交サブプロットのために最後に行を残したい場合は、異なる長さにすることも可能です。タイプ: `1`以上の整数
  * `domain_x::Vector{Float64}` - このグリッドサブプロットの水平ドメインを設定します（プロットの割合で）。最初と最後のセルは、エッジの周りにグラウトがない状態で、ドメインのエッジで正確に終了します。デフォルト: `[0, 1]`
  * `domain_y::Vector{Float64}` - このグリッドサブプロットの垂直ドメインを設定します（プロットの割合で）。最初と最後のセルは、エッジの周りにグラウトがない状態で、ドメインのエッジで正確に終了します。デフォルト: `[0, 1]`
  * `pattern::String` - `subplots`、`xaxes`、または `yaxes` が与えられていないが、`rows` と `columns` がある場合、連続した軸IDを使用してデフォルトを生成できます。 "coupled" は各列に1つのx軸、各行に1つのy軸を提供します。 "independent" は各セルに新しいxyペアを使用し、各行を左から右に、次に `roworder` に従って行を反復します。タイプ: `"coupled"` または `"independent"` のいずれか | デフォルト: `"coupled"`
  * `roworder::String` - 最初の行は上ですか、それとも下ですか？列は常に左から右に列挙されることに注意してください。列挙されたものの中から ( `"top to bottom"` | `"bottom to top"` ) | デフォルト: `"top to bottom"`
  * `rows::Int` - グリッドの行数。2D `subplots` 配列または `yaxes` 配列を提供した場合、その長さがデフォルトとして使用されます。しかし、非直交サブプロットのために最後に行を残したい場合は、異なる長さにすることも可能です。タイプ: `1`以上の整数
  * `subplots::Matrix{String}` - 一部の軸がサブプロット間で共有されるが、他の軸は共有されない自由形式のグリッドに使用されます。各エントリは、"xy" や "x3y2" のような直交サブプロットID、またはそのセルを空にするための "" であるべきです。同じ列内でx軸を再利用し、同じ行内でy軸を再利用できます。非直交サブプロットと `domain` をサポートするトレースは、`gridcell` 属性を使用してこのグリッドに別々に配置できます。
  * `xaxes::Vector{String}` - x軸とy軸が列と行で共有されるときに `yaxes` と共に使用されます。各エントリは "x"、"x2" などのx軸ID、またはその列にx軸を置かないための "" であるべきです。"" 以外のエントリは一意でなければなりません。`subplots` が存在する場合は無視されます。`yaxes` が存在するが `xaxes` が欠けている場合、連続したIDが生成されます。
  * `xgap::Float64` - グリッドセル間の水平スペースで、1つのセルに利用可能な総幅の割合として表現されます。カップル軸グリッドの場合はデフォルトで0.1、独立グリッドの場合は0.2です。タイプ: `0`と`1`の間の数
  * `xside::String` - x軸のラベルとタイトルの位置を設定します。"bottom" はグリッドの最下部を意味します。"bottom plot" は各x軸が使用される最も低いプロットです。"top" と "top plot" も同様です。タイプ: 列挙されたものの中から ( `"bottom"` | `"bottom plot"` | `"top plot"` | `"top"` ) | デフォルト: `"bottom plot"`
  * `yaxes::Vector{String}` - x軸とy軸が列と行で共有されるときに `yaxes` と共に使用されます。各エントリは "y"、"y2" などのy軸ID、またはその行にy軸を置かないための "" であるべきです。"" 以外のエントリは一意でなければなりません。`subplots` が存在する場合は無視されます。`xaxes` が存在するが `yaxes` が欠けている場合、連続したIDが生成されます。
  * `ygap::Float64` - グリッドセル間の垂直スペースで、1つのセルに利用可能な総高さの割合として表現されます。カップル軸グリッドの場合はデフォルトで0.1、独立グリッドの場合は0.3です。タイプ: `0`と`1`の間の数
  * `yside::String` - y軸のラベルとタイトルの位置を設定します。"left" はグリッドの最左端を意味します。"left plot" は各y軸が使用される最も左のプロットです。"right" と "right plot" も同様です。タイプ: 列挙されたものの中から ( `"left"` | `"left plot"` | `"right plot"` | `"right"` ) | デフォルト: `"left plot"`
