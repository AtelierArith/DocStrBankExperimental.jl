ErrorBar()

# 例

---

```
julia> error_x = ErrorBar(
    array = [1.2, 2.6, 3.1],
    type = "data"
)
```

# プロパティ

---

  * `array::Vector{Float64}` - 各エラーバーの長さに対応するデータを設定します。値は基になるデータに対して相対的にプロットされます。
  * `arrayminus::Vector{Float64}` - 垂直（水平）バーの下（左）方向の各エラーバーの長さに対応するデータを設定します。値は基になるデータに対して相対的にプロットされます。
  * `color::String` - エラーバーのストロークカラーを設定します。
  * `symmetric::Bool` - エラーバーが両方向（垂直バーの場合は上下、水平バーの場合は左右）で同じ長さを持つかどうかを決定します。
  * `thickness::Int` - エラーバーの太さ（px単位）を設定します。タイプ：0以上。デフォルト：2
  * `traceref::Int` - タイプ：0以上の整数。デフォルト：0
  * `tracerefminus::Int` - タイプ：0以上の整数。デフォルト：0
  * `type::String` - エラーバーを生成するために使用されるルールを決定します。 "constant"の場合、バーの長さは一定の値になります。この定数は`value`で設定します。 "percent"の場合、バーの長さは基になるデータのパーセンテージに対応します。このパーセンテージは`value`で設定します。 "sqrt"の場合、バーの長さは基になるデータの平方に対応します。 "data"の場合、バーの長さはデータセット`array`で設定されます。タイプ：列挙型、( "percent" | "constant" | "sqrt" | "data" )のいずれか
  * `value::Float64` - "percent"に設定されている場合のパーセンテージ、または"constant"に設定されている場合の定数の値を設定します。タイプ：0以上。デフォルト：10
  * `valueminus::Float64` - 垂直（水平）バーの下（左）方向のエラーバーの長さに対応するパーセンテージ（`type`が"percent"に設定されている場合）または定数（`type`が"constant"に設定されている場合）の値を設定します。タイプ：0以上の数 | デフォルト：10
  * `visible::Bool` - このエラーバーのセットが表示されるかどうかを決定します。
  * `width::Int` - エラーバーの両端のクロスバーの幅（px単位）を設定します。タイプ：0以上
