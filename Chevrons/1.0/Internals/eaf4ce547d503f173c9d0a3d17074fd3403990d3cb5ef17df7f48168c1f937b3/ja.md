再帰的に `>>` チェーン関数呼び出しを置き換えます。

  * `x >> f(y, z)` は `f(x, y, z)` に変わります。
  * `x >> f(y, _, z)` は `f(y, x, z)` に変わります。
  * `x >> f(y) >> g(z)` は `g(f(x, y), z)` に変わります。

また、逆方向の `<<` も置き換えます。

  * `f(y, z) << x` は `f(x, y, z)` に変わります。
  * `f(y, __, z) << x` は `f(y, x, z)` に変わります（ダブルアンダースコアに注意）。
  * `x >> f(y) << z` は `f(z, x, y)` に変わります。

また、`>>>` を使用して前の値を保持することもできます。

  * `x >>> f() >> g()` は `tmp = x; f(tmp); g(tmp)` に変わります。
