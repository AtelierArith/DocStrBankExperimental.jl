```
get_status(wc::WinchController)
```

指定された `WinchController` インスタンスの現在のステータスを取得し、ログ記録およびデバッグ目的で使用します。

# 引数

  * `wc::WinchController`: ステータスを取得するウインチコントローラーオブジェクト。

# 戻り値

  * ウインチコントローラーの現在のステータス。以下を含む配列：

      * `reset`: コントローラーがリセット状態にあるかどうかを示すブール値。
      * `active`: コントローラーがアクティブであるかどうかを示すブール値。
      * `force`: 現在設定されている力の値、または設定されていない場合はゼロ。
      * `f_set`: 設定された力の値。
      * `v_set_out`: スピードコントローラーによって設定された出力速度。
      * `v_set_ufc`: 上部力コントローラーによって設定された出力速度。
      * `v_set_lfc`: 下部力コントローラーによって設定された出力速度。
