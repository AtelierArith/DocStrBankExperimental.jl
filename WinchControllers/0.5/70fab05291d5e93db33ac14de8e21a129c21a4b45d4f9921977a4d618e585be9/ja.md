```
mutable struct WCSettings
```

ウインチコントローラの設定

# フィールド

  * `dt::Float64`: ウインチコントローラのタイムステップ
  * `test::Bool`: ユニットテストを実行するためにtrueに設定 デフォルト: false
  * `fac::Float64`: 下部力コントローラのIおよびPの係数 デフォルト: 0.25
  * `max_iter::Int64`: PIDソルバの最大反復回数の制限 デフォルト: 100
  * `iter::Int64`: PIDソルバの実際の最大反復回数 デフォルト: 0
  * `t_startup::Float64`: ソフトスタートのためのスタートアップ時間 デフォルト: 0.25
  * `t_blend::Float64`: ミキサーのブレンド時間（秒） デフォルト: 0.25
  * `v_sat_error::Float64`: スピードコントローラの入力飽和ブロックで使用される巻き出し速度誤差の制限 デフォルト: 1.0
  * `v_sat::Float64`: スピードコントローラの出力飽和ブロックで使用される巻き出し速度の制限 デフォルト: 8.0
  * `v_ri_max::Float64`: 最大巻き取り速度 [m/s] デフォルト: 8.0
  * `p_speed::Float64`: スピードコントローラのP値 デフォルト: 0.125
  * `i_speed::Float64`: スピードコントローラのI値 デフォルト: 4.0
  * `kb_speed::Float64`: スピードコントローラのアンチワインドアップループのためのバック計算定数 デフォルト: 4.0
  * `kt_speed::Float64`: スピードコントローラのトラッキング定数 デフォルト: 5.0
  * `vf_max::Float64`: 設定された力が最大に達する巻き出し速度 デフォルト: 2.75
  * `pf_low::Float64`: 下部力コントローラのP定数 デフォルト: 0.000144
  * `if_low::Float64`: 下部力コントローラのI定数 デフォルト: 0.0075 * 1.5
  * `df_low::Float64`: 下部力コントローラのD定数 デフォルト: 2.0e-5 * 1.7
  * `nf_low::Float64`: 上部力コントローラのフィルター定数n デフォルト: 7.0
  * `kbf_low::Float64`: 下部力コントローラのアンチワインドアップループのためのバック計算定数 デフォルト: 1.0
  * `ktf_low::Float64`: 下部力コントローラのトラッキング定数 デフォルト: 8.0
  * `f_low::Float64`: 下部力制限 [N] デフォルト: 350
  * `f_reelin::Float64`: 巻き取りフェーズの設定力 [N] デフォルト: 700
  * `f_high::Float64`: 上部力制限 [N] デフォルト: 3800
  * `pf_high::Float64`: 上部力コントローラのP定数 デフォルト: 0.000144 * 1.6
  * `if_high::Float64`: 上部力コントローラのI定数 デフォルト: 0.0075 * 1.6
  * `df_high::Float64`: 上部力コントローラのD定数 デフォルト: 2.0e-5 * 1.7
  * `nf_high::Float64`: 上部力コントローラのフィルター定数n デフォルト: 15.0
  * `kbf_high::Float64`: 上部力コントローラのアンチワインドアップループのためのバック計算定数 デフォルト: 1.0
  * `ktf_high::Float64`: 上部力コントローラのトラッキング定数 デフォルト: 10.0
  * `winch_iter::Float64`: ウインチモデルの反復回数 デフォルト: 10
  * `max_acc::Float64`: ウインチの最大加速度（巻き出し速度の設定値の導関数） デフォルト: 8.0
  * `kv::Float64`: 平方根法則の比例係数、関数calc_vroを参照 デフォルト: 0.06
