```
Pie(args...)
```

与えられた引数で円グラフを表現します。

# キーワード引数

## 必須

  * `category`: 各スライスを示すカテゴリ列。デフォルト: `0`

## 円グラフオプション

  * `endangle`: 終了角度（度）。デフォルト: `360`
  * `missingmode`: カテゴリまたはグループの欠損値をどのように処理するかを示します。 `0` = 何もしない、`1` = カテゴリに欠損なし、`2` = グループに欠損なし、`3` = カテゴリまたはグループに欠損なし、`4` = カテゴリとグループに欠損なし。デフォルト: `0`
  * `response`: `stat`キーワード引数に基づいて集約された値を使用して各スライスの半径を決定する数値列。
  * `sort`: `true`の場合、スライスは合計角度に基づいてソートされます。デフォルト: `false`
  * `startangle`: 開始角度（度）。デフォルト: `0`
  * `stat`: `response`キーワード引数を集約するための関数。`response`が渡されると、キーワードのデフォルト値は`IMD.sum`に変更されますが、ユーザーはこの引数に任意の関数を渡すことができます。関数は2つの引数`f`（フォーマット）と`x`（入力値）を受け取り、集約された値を返す必要があります。

## グルーピング

  * `group`: この引数に単一の列を渡すことで、ネストされた円グラフが作成されます。
  * `groupspace`: ネストされた円グラフの間のスペース。デフォルト: `0.01`

## 円グラフの外観

  * `colormodel`: マークに使用するカラースキームを指定します。デフォルト: `:category`
  * `innerradius`: ドーナツの半径。0と1の間の数値でなければなりません。デフォルト: `0`
  * `opacity`: マークの不透明度。0（透明）から1（不透明）まで。デフォルト: `1`
  * `outerradius`: 外半径。0と1の間の数値でなければなりません。デフォルト: `1`
  * `outlinecolor`: マークのアウトラインカラー。デフォルト: `:white`
  * `outlinethickness`: マークのアウトラインの厚さ。デフォルト: `1`
  * `piecorner`: スライスのコーナーのピクセル数。デフォルト: `0`
  * `space`: スライス間のスペース。0と1の間の任意の数値を指定できます。デフォルト: `0`

## 円グラフラベル

  * `decimal`: パーセンテージが使用されるときの小数点以下の桁数。デフォルト: `1`
  * `label`: スライスラベルに使用する情報。`:category`、`:percent`、または`:both`のいずれかです。
  * `labelalign`: テキストの配置。デフォルト: `:center`
  * `labelangle`: テキストの角度。
  * `labelbaseline`: テキストのベースライン。デフォルト: `:middle`
  * `labelcolor`: マークのデフォルトカラー。ユーザーは色の名前をシンボル（例: `:red`）、文字列（例: `"red"`）、HTMLカラー値（例: `"#4682b4"`）として渡すことができます。デフォルト: `:black`
  * `labeldir`: テキストの方向、すなわち`:ltr`または`:rtl`。デフォルト: `:ltr`
  * `labelfont`: テキストを表示するためのフォント名。
  * `labelfontweight`: テキストを表示するためのフォントの太さ。細いフォントには100、太字には900を使用します。
  * `labelitalic`: `true`の場合、テキストを表示する際にイタリックスタイルが使用されます。
  * `labellimit`: テキストマークの最大長（ピクセル）。レンダリングサイズが制限を超えると、テキスト値は自動的に切り捨てられます。
  * `labelopacity`: テキストの不透明度。デフォルト: `1`
  * `labelpos`: ラベルの垂直位置。デフォルト: `0.5`
  * `labelrotate`: テキストを90度回転させます。デフォルト: `false`
  * `labelsize`: テキストを表示するためのフォントサイズ。
  * `labelthreshold`: `:labelthreshold`よりも小さい角度のスライスのラベルは削除されます。デフォルト: `0.0`
  * `tooltip`: マウスオーバー時にツールチップが表示されます。デフォルト: `false`

## 凡例

  * `legend`: ユーザーはこのキーワード引数にシンボルを渡して、対応するマークの凡例のためにさらにカスタマイズが渡されることを示すことができます。ユーザーは`Legend`グローバルキーワードを介して追加のカスタマイズを提供する必要があります。

## その他

  * `clip`: マークをクリップするかどうかを示します。
