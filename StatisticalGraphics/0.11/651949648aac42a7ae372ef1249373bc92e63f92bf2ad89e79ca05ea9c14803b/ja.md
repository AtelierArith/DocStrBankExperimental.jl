```
gradient([colors...]; direction = [0,1,0,0])
```

線形グラデーションは、開始点から終了点までの線に沿って色を補間します。デフォルトでは、線形グラデーションは水平方向に、左から右に実行されます。方向を使用してグラデーションの方向を設定します。たとえば、[0,0,0,1]はグラデーションを垂直に実行します。すべての座標は、色付けされるアイテムのバウンディングボックスに対して正規化された[0, 1]座標空間で定義されます。

## 例

```julia
gradient()
gradient(:red, :white, :blue)
```
