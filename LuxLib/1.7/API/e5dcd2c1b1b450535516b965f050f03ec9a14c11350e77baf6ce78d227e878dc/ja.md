```
fast_activation!!(σ::F, x::AbstractArray) where {F}
```

`σ.(x)`を可能な限り最適な実装で計算します。`x`をインプレースで書き換えることが可能な場合は、そうします。`x`が不変配列である場合は、一般的な実装にフォールバックします。

!!! note
    この関数は`σ`を`NNlib.fast_act(σ, ...)`で置き換えることはありません。必要に応じてユーザーが行う必要があります。


!!! tip "より高速な活性化のために`SLEEFPirates.jl`をロードする"
    特定の活性化関数は、FP32用の[SLEEFPirates.jl](https://github.com/JuliaSIMD/SLEEFPirates.jl)からの特化した実装に置き換えられます。これによりパフォーマンスが向上する可能性がありますが、精度がわずかに低下する可能性があります（浮動小数点の限界において）。


## 引数

  * `σ`: 活性化関数
  * `x`: 入力配列

## 戻り値

  * `x`と同じサイズの出力配列
