```
fast_activation(σ::F, x::AbstractArray) where {F}
```

`σ.(x)`を可能な限り最適な実装で計算します。CPUではループを展開し、LoopVectorization.jlを使用して計算をベクトル化します。GPUでは単にブロードキャスティングを使用します。

!!! note
    この関数は`σ`を`NNlib.fast_act(σ, ...)`で置き換えるものではなく、必要に応じてユーザーが行う必要があります。


## 引数

  * `σ`: 活性化関数
  * `x`: 入力配列

## 戻り値

  * `x`と同じサイズの出力配列
