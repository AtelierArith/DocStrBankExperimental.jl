```
function multi_product(g1::Graph{F,W}, g2::Graph{F,W}, c1=F(1), c2=F(1)) where {F,W,C}
```

`c1*g1 * c2*g2` の多重積を表すグラフを返します。もし `g1 == g2` の場合、`Power(2)` 演算子を用いて `c1*c2 * (g1)^2` を表すグラフを返します。

# 引数:

  * `g1`:  最初の計算グラフ
  * `g2`:  2番目の計算グラフ
  * `c1`:  最初のスカラー倍 (デフォルトは1)。
  * `c2`:  2番目のスカラー倍 (デフォルトは1)。
