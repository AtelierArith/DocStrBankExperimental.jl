```
function linear_combination(g1::FeynmanGraph{F,W}, g2::FeynmanGraph{F,W}, c1, c2) where {F,W}
```

`c1*g1 + c2*g2` の線形結合を表すグラフを返します。もし `g1 == g2` であれば、`(c1+c2)*g1` を表すグラフを返します。フェインマングラフ `g1` と `g2` は同じ図のタイプ、次数、および外部頂点を持っている必要があります。

# 引数:

  * `g1`  最初のフェインマングラフ
  * `g2`  2番目のフェインマングラフ
  * `c1`:  最初のスカラー倍 (デフォルトは1)。
  * `c2`:  2番目のスカラー倍 (デフォルトは1)。
