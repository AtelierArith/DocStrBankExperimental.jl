```
function merge_multi_product(g::Graph{F,W}) where {F,W}

複数の積を持つ計算グラフ `g` のコピーを返します。これらは同じ演算子（`Prod`）を共有している場合にマージされます。
もし `g.operator == Prod` の場合、この関数は `N` 個の同一のサブグラフを `Power(N)` 演算子を持つ単一のサブグラフにマージします。
この関数は、各ユニークなサブグラフが適切にカウントされ、マージされることを保証し、それに関連する任意の異なる subgraph_factors を保持します。
```

# 引数:

  * `g::Graph`: 修正されるグラフ

# 戻り値

  * `Prod` 演算子を持つ繰り返しサブグラフがあった場合、潜在的に少ないサブグラフを持つマージされた計算グラフを返します。入力グラフの演算子が `Prod` でない場合、関数は入力グラフを変更せずに返します。
