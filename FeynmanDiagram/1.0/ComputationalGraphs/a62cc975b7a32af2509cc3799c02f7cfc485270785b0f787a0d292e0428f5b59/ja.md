```
multi_product(graphs::Vector{Graph{F,W}}, constants::AbstractVector=ones(F, length(graphs))) where {F,W,C}
```

複数の入力グラフから積グラフを構築します。各グラフは定数によって重み付けされることができます。複数回繰り返されるグラフに対しては、繰り返しを表すために部分グラフに累乗演算子が追加されます。さらに、結果の積グラフ内の任意の自明な単項演算子を最適化します。

# 引数:

  * `graphs::Vector{Graph{F,W}}`: 乗算される入力グラフのベクトル。
  * `constants::AbstractVector`: スカラー倍のベクトル。提供されない場合、`graphs`と同じ長さの1のベクトルがデフォルトとなります。

返り値:

  * ユニークな部分グラフ（またはその累乗版）と関連する定数を部分グラフ因子として持つ新しい積グラフ。

# 例:

```
グラフ `g1`、`g2`、`g1` と定数 `c1`、`c2`、`c3` が与えられた場合、関数は `(c1*c3)*(g1)^2 * c2*g2` を計算します。
```
