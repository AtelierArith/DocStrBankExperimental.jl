```
function leafstates(leaf_maps::Vector{Dict{Int,G}}, maxloopNum::Int)

葉の値のインデックスから葉ノードへの葉マッピングから葉情報を抽出します。 
情報には、初期値、タイプ、順序、入出力時間、およびループ運動量インデックスが含まれます。
すべてのグラフのループ基底も取得されます。
```

# 引数:

  * `leaf_maps`: 葉の値のインデックスをこの葉のグラフにマッピングする辞書のベクトル。               各辞書は、(order, Gorder, Vorder)のようなグラフパーティションに対応します。
  * `maxloopNum`: 最大ループ運動量番号。

# 戻り値

  * グラフの葉に関する情報を含むベクトルのタプルで、初期値、タイプ、順序、入力および出力時間インデックス、ループ運動量インデックスが含まれます。
  * すべてのグラフのループ運動量基底 (`::Vector{Vector{Float64}}`)。
