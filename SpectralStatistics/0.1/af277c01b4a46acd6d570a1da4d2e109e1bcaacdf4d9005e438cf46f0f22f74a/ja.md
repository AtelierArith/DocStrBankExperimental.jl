```
BerryRobnik <: Model
```

`BerryRobnik` は、Berry-Robnik モデル（2 つのコンポーネントを持つ）を表すために使用される具体的な型です。

## 説明

このモデルは、古典的な位相空間が規則的な運動と混沌とした運動の両方を特徴とするシステムのスペクトル統計（半古典的制限において）を記述します。これらのシステムは、位相空間が分割されたシステムまたは混合型システムと一般に呼ばれます。このモデルは、半古典的制限において、異なるタイプの運動（規則的および混沌とした）を記述する状態に属するスペクトルのサブスペクトルが別々のコンポーネントに分解されるという議論に基づいています。スペクトルの規則的な部分は [`Poisson`](@ref) 統計によってモデル化され、混沌とした部分は [`GOE`](@ref) 統計によってモデル化されます。

## 属性

  * `rho`: 結合された規則的コンポーネントのリウヴィル測度。
