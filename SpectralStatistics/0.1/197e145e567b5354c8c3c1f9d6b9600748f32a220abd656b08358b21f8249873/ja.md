```
level_spacing(spect::UnfoldedSpectrum; n::Int = 1) → s::Vector
```

レベル間隔の順序 `n` を返します。最近接隣接レベル間隔がデフォルトで、n=1 で与えられます。

## 説明

順序 n のレベル間隔は、スペクトルの n 番目の連続レベルの差によって与えられます。最近接隣接レベル間隔が最も一般的に考慮されます。

## 引数

  * `spect`: [`UnfoldedSpectrum`](@ref) 型のインスタンスとして与えられる展開されたエネルギースペクトル。

## キーワード引数

  * `n=1` : レベル間隔の順序。

## 戻り値

  * `s` : レベル間隔のベクトル。
