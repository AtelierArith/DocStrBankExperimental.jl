```
level_spacing_pdf(model::Model, pts::Vector; n::Int = 1) → p::Vector
```

選択したモデルに対応するレベル間隔確率密度関数の解析的表現を返し、位置 `pts` で評価します。

## 説明

最近接隣接レベル間隔分布は、最も一般的に研究されるスペクトル統計です。

## 引数

  * `model`: [`Model`](@ref) の具体的なサブタイプのインスタンスとして与えられるモデル。
  * `pts`: 確率密度関数を評価する位置。

## キーワード引数

  * `n=1` : レベル間隔の順序。

## 返り値

  * `p` : 確率のベクトル。
