```
rand_pgdevroye(b::Integer, z::Real, [rng::AbstractRNG = Random.default_rng()]])
```

Devroye法を使用してポリア-ガンマ分布からサンプルを取得します。

# 引数

  * `b::Integer`: 形状パラメータ
  * `z::Real`: 指数傾斜パラメータ
  * `rng::AbstractRNG`: `rand`用のオプションの乱数生成器オブジェクト

# 戻り値

  * 形状パラメータ `b` と指数傾斜パラメータ `z` を持つポリア-ガンマ分布からのサンプル。

# 注意事項

  * このメソッドは正確ですが、`b` が増加するにつれて遅くなります。
  * このサンプラーは整数 `b` を必要とします。
  * `b < 13` で整数の場合、自動的にメソッドを選択します。
