```
rand_pgnormalapprox(b::Real, z::Real, [rng::AbstractRNG = Random.default_rng()])
```

ポリア-ガンマ分布から正規近似法を使用してサンプルを取得します。

# 引数

  * `b::Real`: 形状パラメータ
  * `z::Real`: 指数傾斜パラメータ
  * `rng::AbstractRNG`: `rand`用の乱数生成器オブジェクト

# 戻り値

  * 形状パラメータ `b` と指数傾斜パラメータ `z` を持つポリア-ガンマ分布からのサンプル。

# 注意事項

  * このメソッドは近似であり、非整数の `b` をサポートし、大きな `b` に対して非常に効率的です。
  * `b >= 170` の場合、このメソッドが自動的に選択されます。
