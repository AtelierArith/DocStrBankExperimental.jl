```
rand_pggammasum(b::Real, z::Real, [rng::AbstractRNG = Random.default_rng()])
```

ポリア-ガンマ分布から、ガンマのトランケート和表現を使用してサンプリングします。

# 引数

  * `b::Real`: 形状パラメータ
  * `z::Real`: 指数傾斜パラメータ
  * `rng::AbstractRNG`: `rand`用の乱数生成器オブジェクト

# 戻り値

  * 形状パラメータ `b` と指数傾斜パラメータ `z` を持つポリア-ガンマ分布からのサンプル。

# 注意事項

  * このメソッドは非整数の `b` をサポートしますが、これは近似に過ぎず、`b < 1` の場合のみを意図しています。
  * `b` が大きすぎる場合は警告は表示されません。
  * 和は論文の推奨に従って200項目にトランケートされます。
  * `b < 1` の場合、または非整数の `b` に対してデブロイ法と組み合わせて使用される場合は、このメソッドが自動的に選択されます。
