```
rand_pgsaddlepoint(b::Real, z::Real, [rng::AbstractRNG = Random.default_rng()])
```

サドルポイント近似法を使用してポリア-ガンマ分布からサンプルを取得します。

# 引数

  * `b::Real`: 形状パラメータ
  * `z::Real`: 指数傾斜パラメータ
  * `rng::AbstractRNG`: `rand`用の乱数生成器オブジェクト

# 戻り値

  * 形状パラメータ `b` と指数傾斜パラメータ `z` を持つポリア-ガンマ分布からのサンプル。

# 注意事項

  * このメソッドは近似であり、非整数の `b` をサポートし、大きな `b` に対して非常に効率的です。
  * `b >= 13` の場合は自動的にこのメソッドが選択されます。
