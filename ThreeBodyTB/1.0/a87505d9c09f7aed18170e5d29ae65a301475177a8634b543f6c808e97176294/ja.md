```
scf_energy(c::crystal)
```

結晶のエネルギー、力、応力を計算します。自己無矛盾場（SCF）計算を行います。自己無矛盾静電気を使用する場合。

エネルギー、タイトバインディング結晶オブジェクト、エラーフラグを返します。

# 引数

  * `c::crystal`: 計算対象の構造。唯一の必須引数。
  * `database=missing`: 係数のソース。欠落している場合は事前フィット係数からロードされます。
  * `smearing=0.01`: ガウススミアリング温度（Ryd）。通常はデフォルトのままで大丈夫です。
  * `grid=missing`: k点グリッド、例：[10,10,10]、デフォルトは自動的に選択されます。
  * `conv_thr = 1e-5`: SCF収束閾値（Ryd）。
  * `sparse = :auto`: デフォルトは `nat < 100` の場合に密行列を使用します。強制的に選択するために `true` または `false` にすることができます。
  * `iter = 75`: より保守的な設定に切り替える前の反復回数。
  * `mix = -1.0`: 初期ミキシング。-1.0はデフォルトのミキシングを使用することを意味します。SCFが収束しない場合は自動的にミキシングを調整します。大きな単位セルの場合、開始デフォルトは小さくなります。
  * `mixing_mode = :simple`: デフォルトはシンプルです。他のオプションは `:simple` と `:DIIS` / `:pulay`（反復部分空間の直接反転）です。Pulayが失敗した場合は自動的にシンプルに切り替わります。
