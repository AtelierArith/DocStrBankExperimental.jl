```
function band_summary(tbc, kgrid, fermi=missing)
```

バンド構造の要約を生成します。k点を自動的に生成する関数の詳細バージョンについては、以下の関数を参照してください。

注意: ギャップは、電子数が奇数の非磁性系では明確に定義されていません。これは金属である必要があります。

`direct_gap, indirect_gap, gaptype, bandwidth`を返します。

  * `direct_gap`: 名目上満たされたバンドと空のバンドの間の最小ギャップ。金属ではゼロでない場合があります。
  * `indirect_gap`: LUMO - HOMO。材料がどこでも直接ギャップを持つ場合、しかしあるk点での伝導バンドが異なるk点での価電子バンドの下にある場合、負になることがあります。物理的にはこれらは間接ギャップ半金属です。
  * `gaptype`: すべての金属に対しては `:metal`、絶縁体に対しては `:direct` または `:indirect` です。
  * `bandwidth`: HOMO - 最小*バンド*エネルギー。TB計算に含まれる場合は半コア状態も含まれます。
