```
relax_structure(c::crystal; mode="vc-relax")
```

結晶 `c` の最低エネルギー原子配置を見つけます。

# 引数

  * `c::crystal`: リラックスさせる構造、必須の引数
  * `mode="vc-relax"`: デフォルト（可変セルリラックス）は構造とセルをリラックスさせます。それ以外は構造のみをリラックスさせます。
  * `database=missing`: 係数データベース、デフォルトは事前フィットされたpbesolデータベースを使用します
  * `smearing=0.01`: スミア温度（リュード）、デフォルト = 0.01
  * `grid=missing`: k点グリッド、例: [10,10,10]、デフォルトは自動的に選択されます
  * `nsteps=100`: 最大反復回数
  * `update_grid=true`: リラクゼーション中に自動k点グリッドを更新
  * `conv_thr = 2e-3`: 勾配の収束閾値
  * `energy_conv_thr = 2e-4`: リュードにおけるエネルギーの収束閾値
  * `sparse = :auto`: デフォルトは `nat < 100` の場合に密行列を使用します。強制的に選択するために `true` または `false` にすることができます。
