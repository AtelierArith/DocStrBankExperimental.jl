```
ll(y, model, θ; initiate = "first")
```

カウントデータモデルの尤度関数。

  * `y`: 時系列
  * `model`: モデル仕様
  * `θ`: パラメータ（ベクトルまたはパラメータ）
  * `initiate`: 時系列はどのように初期化されますか？

時系列は「first」、最初に観測された値、「intercept」、切片パラメータ（対数リンクの場合は指数化される可能性があります）、または「marginal」、プロセスの周辺平均によって初期化できます。

# 例

```julia-repl
ll(y, model, pars)
```
