```
predict(results, h, nChain, Xnew)
```

カウントデータモデルの予測のための関数。

  * `results`: 推定結果
  * `h`: 予測するステップ数
  * `nChain`: シミュレーションに基づく予測のためのチェーン数（オプション）
  * `Xnew`: 回帰変数の新しい値（回帰変数がある場合のみ）

# 例

```julia-repl
# 10ステップ先の予測
predict(results, 10, 10000)
```

この関数は、`nChain`が指定されていない場合はポイント予測を返すか、推定結果に従って複数の時系列を生成します。後者は予測区間を計算するために使用され、INARMAモデルのデフォルトです。
