```
apply(d, by::Pair)
```

`Tables.jl`互換のテーブル`d`の指定された列に対して関数を要素ごとに適用し、結果を返します。

関数`f`が受け入れる引数に応じて、引数`by`は`column_index => f`または`column_indices => f`のいずれかとして指定されます。ここで、`column_index`は`d`の列に対する`Symbol`または`Int`であり、`column_indices`は複数の列に対するそのようなインデックスの反復可能なコレクションです。`f`は指定された各列に要素ごとに適用され、返された値の配列が得られます。
