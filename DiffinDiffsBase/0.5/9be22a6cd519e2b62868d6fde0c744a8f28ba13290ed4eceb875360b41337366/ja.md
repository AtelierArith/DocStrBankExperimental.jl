```
TableIndexedMatrix{T,M,R,C} <: AbstractMatrix{T}
```

行の値に基づいて選択できる行列と列のインデックスを持つ行列で、`Tables.jl`互換のテーブルにそれぞれ対応しています。これは、要素が行列に格納される方法がテーブルの行によって決定される場合に便利です。

# パラメータ

  * `T`: 行列の要素の型。
  * `M`: 行列の型。
  * `R`: 行インデックスに対応するテーブルの型。
  * `C`: 列インデックスに対応するテーブルの型。

# フィールド

  * `m::M`: 要素を格納する行列。
  * `r::R`: `m`と同じ数の行を持つテーブル。
  * `c::C`: 行数が`m`の列数と等しいテーブル。
