```
print_tree([io::IO,] tree::Node, depth=-1, indent=0; sigdigits=4, feature_names=nothing)
```

指定された `tree` のテキスト視覚化を印刷します。たとえば、ある入力パターンに対して「Feature 3」の値が「-30」で、「Feature 2」の値が「100」の場合、以下のサンプル出力に従って、主要クラスの予測は7です。さらに、この入力データと同じ葉で終わる10555のトレーニングサンプルのうち、2493が主要クラスを予測していることがわかり、クラス7の確率的予測は `2493/10555` になります。非主要クラスの比率は表示されません。

# 例の出力:

```
Feature 3 < -28.15 ?
├─ Feature 2 < -161.0 ?
   ├─ 5 : 842/3650
   └─ 7 : 2493/10555
└─ Feature 7 < 108.1 ?
   ├─ 2 : 2434/15287
   └─ 8 : 1227/3508
```

サードパーティパッケージを使用してツリーの視覚化を容易にするために、`DecisionTree.Leaf` オブジェクト、`DecisionTree.Node` オブジェクト、または `DecisionTree.Root` オブジェクトをラップして、AbstractTrees.jl インターフェースを実装するツリー構造を取得できます。詳細については [`wrap`](@ref) を参照してください。
