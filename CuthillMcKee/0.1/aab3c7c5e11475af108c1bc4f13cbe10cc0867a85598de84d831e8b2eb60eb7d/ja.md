```
 symrcm(adjac, sdegr,[ rev, inv, warnunconnected])
```

与えられた隣接リスト `adjac` のグラフのカシル・マッキー置換を計算します。隣接リストは `Vector{Vector{Int}}` 型である必要があり、`adjac[i]` は頂点 `i` に隣接する頂点番号のベクトルで、頂点の次数が昇順に並んでいる必要があります。追加の必須引数 `sdegr` は、頂点の次数が昇順に並んだ頂点番号の配列です。デフォルトでは逆置換を計算します。すなわち、`rev` は `true` にデフォルト設定されています。オプションとして、`inv` が `true` に設定されている場合、置換は反転されます。デフォルトでは、グラフ内の未接続領域の数を表示する警告は無効になっています。これを有効にするには、`warnunconnected` を `true` に設定します。
