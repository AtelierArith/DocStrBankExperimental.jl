```
voxel_id = xf_to_id(n_pts, xf)
```

ユニットキューブが `n_pts[1]` x `n_pts[2]` x `n_pts[3]` ボクセルの規則的なグリッドに分割されるときに、その中に落ちるボクセルのインデックスを返します。周期境界条件が適用されます。

# 引数

  * `n_pts::Tuple{Int, Int, Int}`: `Grid` の各軸のポイント数
  * `xf::Array{Float64, 1}`: id に変換される分数座標

# 戻り値

  * `id::Array{Int, 1}`: この空間内のポイントを格納するための配列インデックス
