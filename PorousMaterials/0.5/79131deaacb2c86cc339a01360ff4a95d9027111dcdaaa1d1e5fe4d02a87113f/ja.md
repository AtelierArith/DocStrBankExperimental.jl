```
repfactors = replication_factors(unitcell, r_cutoff)
```

指定されたカットオフ半径に収まる球をフィットさせるのに十分な大きさのスーパーセルを作るために必要な複製因子を見つけます。PorousMaterials.jlでは、シミュレーションボックスとして機能するスーパーセルを構築するために、ホームユニットセル内の原子を複製するのではなく、ホームユニットセルを複製してスーパーセル（シミュレーションボックス）を形成します。この関数は、最近接画像の規約を適用できるように、十分な複製因子を確保します。

複製されていないスーパーセルは、各次元で複製因子が1（`repfactors = (1, 1, 1)`）です。

# 引数

  * `unitcell::Box`: 結晶のユニットセル
  * `r_cutoff::Float64`: ポテンシャルエネルギーをゼロと定義するカットオフ半径（単位：オングストローム）

# 戻り値

  * `repfactors::Tuple{Int, Int, Int}`: a, b, c方向の複製因子
