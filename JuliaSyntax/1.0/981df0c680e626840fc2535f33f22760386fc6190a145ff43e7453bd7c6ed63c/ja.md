```
K"name"
Kind(namestr)
```

`Kind` は、構文木のトークンと内部ノードの型を指定するための型タグです。抽象的には、このタグは構文木ノードのための独自の *和型* を定義するために使用されます。これは、(a) Julia に和型がないため、(b) 効率のために Julia コンパイラの観点からユニタイプの具体的なデータ構造を望むために、Julia の型システムの外で明示的に行います。

命名規則：

  * 正確に1つのテクスチャ形式に対応する Kind は、そのテキストで表されます。これには K"for" のようなキーワードや K"*" のような演算子が含まれます。
  * 多くのテクスチャ形式を表す Kind は UpperCamelCase の名前を持ちます。これには K"Identifier" や K"Comment" のような Kind が含まれます。
  * 単に区切りとして存在する Kind はすべて大文字です。
