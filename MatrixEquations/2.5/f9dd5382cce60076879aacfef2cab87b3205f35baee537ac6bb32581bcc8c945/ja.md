```
γ = opnorm1(op)
```

線形演算子 `op` の誘導 `1`-ノルム `γ` を、関連する `m x n` 行列 $M$ `= Matrix(op)` の列の `1`-ノルムの最大値として計算します：

$$
\gamma = \|op\|_1 := \max_{1 ≤ j ≤ n} \|M_j\|_1
$$

ここで、$M_j$ は $M$ の `j`-列です。この関数は、大きな次数の演算子には使用しないことをお勧めします。

# 例

```jldoctest
julia> A = [-6. -2. 1.; 5. 1. -1; -4. -2. -1.]
3×3 Array{Float64,2}:
 -6.0  -2.0   1.0
  5.0   1.0  -1.0
 -4.0  -2.0  -1.0

julia> opnorm1(lyapop(A))
30.0

julia> opnorm1(invlyapop(A))
3.7666666666666706
```
