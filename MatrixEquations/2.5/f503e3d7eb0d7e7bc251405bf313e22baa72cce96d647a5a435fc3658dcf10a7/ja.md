```
U = plyapd(A, B)
```

離散リャプノフ方程式の解 `X = UU'` の上三角因子 `U` を計算します。

```
  AXA' - X + BB' = 0,
```

ここで、`A` は正方形の実数または複素行列であり、`B` は `A` と同じ行数を持つ行列です。`A` は絶対値が1未満の固有値のみを持つ必要があります。

```
U = plyapd(A', B')
```

離散リャプノフ方程式の解 `X = U'U` の上三角因子 `U` を計算します。

```
  A'XA - X + B'B = 0,
```

ここで、`A` は正方形の実数または複素行列であり、`B` は `A` と同じ列数を持つ行列です。`A` は絶対値が1未満の固有値のみを持つ必要があります。
