```
X = lyapd(A, E, C)
```

一般化離散リャプノフ方程式の解 `X` を計算します。

```
     AXA' - EXE' + C = 0,
```

ここで `A` と `E` は実数または複素数の正方行列であり、`C` は正方行列です。ペンシル `A-λE` は、`α` と `β` の2つの固有値が `αβ = 1` でないことを満たさなければなりません。解 `X` は、`C` が対称またはエルミートであれば対称またはエルミートです。

以下の特定のケースも扱います。

```
X = lyapd(A,β*I,C)  または  X = lyapd(A,β,C)
```

行列方程式 `AXA' - βXβ' + C = 0` を解きます。

```
X = lyapd(α*I,E,C)  または  X = lyapd(α,E,C)
```

行列方程式 `αXα' - EXE' + C = 0` を解きます。

```
X = lyapd(α*I,β*I,C)  または  X = lyapd(α,β,C)
```

行列方程式 `(αα'-ββ')X + C = 0` を解きます。

```
x = lyapd(α,β,γ)
```

方程式 `(αα'-ββ')x + γ = 0` を解きます。

# 例

```jldoctest
julia> A = [3. 4.; 5. 6.]
2×2 Array{Float64,2}:
 3.0  4.0
 5.0  6.0

julia> E = [ 1. 2.; 0. -1.]

 1.0   2.0
 0.0  -1.0

julia> C = [1. 1.; 1. 2.]
2×2 Array{Float64,2}:
 1.0  1.0
 1.0  2.0

julia> X = lyapd(A, E, C)
2×2 Array{Float64,2}:
  1.775  -1.225
 -1.225   0.775

julia> A*X*A' - E*X*E' + C
2×2 Array{Float64,2}:
 -2.22045e-16  -4.44089e-16
 -1.33227e-15   1.11022e-15
```
