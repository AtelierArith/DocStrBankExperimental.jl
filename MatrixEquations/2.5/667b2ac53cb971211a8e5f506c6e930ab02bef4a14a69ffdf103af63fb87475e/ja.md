```
(X,Y) = sylvsyss!(A,B,C,D,E,F)
```

シルベスター行列方程式系を解きます

```
            AX + YB = C
            DX + YE = F,
```

ここで、`(A,D)`、`(B,E)` は同じサイズの正方行列のペアで、一般化シュール形式にあります。ペンシル `A-λD` と `-B+λE` は正則であり、共通の固有値を持ってはいけません。計算された解 `(X,Y)` は `(C,F)` に含まれます。

*注:* これは、`A`、`B`、`D` および `E` が実行列であり、`C` と `F` が複素行列である場合もカバーするために、`LAPACK.tgsyl!` 関数への拡張インターフェースです。
