```
γ = opnorm1est(op)
```

逆通信ベースの計算を使用して、線形演算子 `op` の `1`-ノルムの下限である `γ` を計算します。`op * x` および `op' * x` を評価することが期待されます。ほとんどの場合、$\gamma > \|op\|_1/10$ であることが期待され、これは線形演算子の条件数を推定するために通常受け入れられます。

# 例

```jldoctest
julia> A = [-6. -2. 1.; 5. 1. -1; -4. -2. -1.]
3×3 Array{Float64,2}:
 -6.0  -2.0   1.0
  5.0   1.0  -1.0
 -4.0  -2.0  -1.0

julia> opnorm1est(lyapop(A))
18.0

julia> opnorm1est(invlyapop(A))
3.76666666666667
```
