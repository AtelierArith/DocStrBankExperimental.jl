```
U = plyapc(A, E, B)
```

一般化連続リャプノフ方程式の解 `X = UU'` の上三角因子 `U` を計算します。

```
  AXE' + EXA' + BB' = 0,
```

ここで `A` と `E` は正方行列の実数または複素数であり、`B` は `A` と同じ行数を持つ行列です。ペンシル `A - λE` は負の実部を持つ固有値のみを持たなければなりません。

```
U = plyapc(A', E', B')
```

一般化連続リャプノフ方程式の解 `X = U'U` の上三角因子 `U` を計算します。

```
  A'XE + E'XA + B'B = 0,
```

ここで `A` と `E` は正方行列の実数または複素数であり、`B` は `A` と同じ列数を持つ行列です。ペンシル `A - λE` は負の実部を持つ固有値のみを持たなければなりません。
