```
X = hulyapc!(U, Q; adj = false)
```

上三角行列 `U` とエルミート行列 `Q` に対して、連続 H-リャプノフ行列方程式の上三角解 `X` を計算します。

```
            U'*X + X'*U = Q   ただし adj = false の場合、
```

または

```
            U*X' + X*U' = Q   ただし adj = true の場合。
```

解 `X` は行列 `Q` を上書きし、`U` は変更されません。

`U` が非特異である場合、`adj = false` の場合は後退消去法が使用され、`adj = true` の場合は前進消去法が使用されます。このアプローチは、[`DiffOpt.jl`](https://github.com/jump-dev/DiffOpt.jl) パッケージの関数 `dU_from_dQ!` で採用されている方法を適応させたものです。`n×n` の特異な `U` の場合、共役勾配法に基づく反復法を使用して、上三角行列 `X` の最小二乗解が決定されます。これは、上三角行列 `X` を上三角行列 `Y` にマッピングする適切に定義された T-リャプノフ線形演算子 `L:X -> Y` に適用され、関連する行列 `M = Matrix(L)` は $n(n+1)/2 \times n(n+1)/2$ です。この場合、キーワード引数 `tol`（デフォルト: `tol = sqrt(eps())`）を使用して計算された解の精度に対する所望の許容誤差を提供でき、キーワード引数 `maxiter` を使用して最大反復回数を設定できます（デフォルト: maxiter = 1000）。
