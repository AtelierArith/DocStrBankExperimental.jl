```
plyapds!(A, R; adj = false)
```

正の離散リャプノフ行列方程式を解きます

```
            op(A)Xop(A)' - X + op(R)*op(R)' = 0
```

ここで、`X = op(U)*op(U)'`、`op(K) = K` もし `adj = false` の場合、`op(K) = K'` もし `adj = true` の場合です。`A` は実シュール形式の正方行列または複素シュール形式の正方行列であり、`R` は上三角行列です。`A` は絶対値が1未満の固有値のみを持つ必要があります。`R` には出力として上三角解 `U` が含まれます。
