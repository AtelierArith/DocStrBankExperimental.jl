```
lyapcs!(A,C;adj = false)
```

連続リャプノフ行列方程式を解きます。

```
            op(A)X + Xop(A)' + C = 0,
```

ここで、`op(A) = A` もし `adj = false` の場合、`op(A) = A'` もし `adj = true` の場合です。`A` は実シュール形式の正方行列、または複素シュール形式の正方複素行列であり、`C` は対称行列またはエルミート行列です。`A` は、2つの固有値 `α` と `β` が `α+β = 0` でないことが必要です。`C` には出力として解 `X` が含まれます。
