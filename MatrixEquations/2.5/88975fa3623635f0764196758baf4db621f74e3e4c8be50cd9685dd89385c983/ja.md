```
X = tsylvckr(A,B,C; atol::Real=0, rtol::Real=atol>0 ? 0 : N*ϵ)
```

連続T-シルベスター行列方程式の解を計算します。

```
            A*X + transpose(X)*B = C
```

方程式のクロネッカー積展開を使用します。`A`、`B`、および `C` はそれぞれ `m×n`、`n×m`、および `m×m` 行列であり、`X` は `n×m` 行列です。`atol` と `rtol` はそれぞれランク計算に使用される絶対および相対許容誤差です。デフォルトの相対許容誤差は `N*ϵ` であり、ここで `N = 4*min(m,n)^2` で、ϵ は `A` の要素型の機械精度です。この関数は大きな次数の行列には推奨されません。
