```
lyapds!(A, C; adj = false)
```

離散リャプノフ行列方程式を解きます

```
            op(A)Xop(A)' - X + C = 0,
```

ここで `op(A) = A` もし `adj = false` の場合、`op(A) = A'` もし `adj = true` の場合です。`A` は実数または複素数のシュール形式であり、`C` は対称行列またはエルミート行列です。`A` は二つの固有値 `α` と `β` が `αβ = 1` でない必要があります。計算された対称またはエルミートの解 `X` は `C` に含まれています。
