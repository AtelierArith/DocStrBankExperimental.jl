```
U = plyapd(A, E, B)
```

一般化離散リャプノフ方程式の解 `X = UU'` の上三角因子 `U` を計算します。

```
  AXA' - EXE' + BB' = 0,
```

ここで `A` と `E` は正方形の実数または複素行列であり、`B` は `A` と同じ行数を持つ行列です。ペンシル `A - λE` は、絶対値が1未満の固有値のみを持たなければなりません。

```
U = plyapd(A', E', B')
```

一般化離散リャプノフ方程式の解 `X = U'U` の上三角因子 `U` を計算します。

```
  A'XA - E'XE + B'B = 0,
```

ここで `A` と `E` は正方形の実数または複素行列であり、`B` は `A` と同じ列数を持つ行列です。ペンシル `A - λE` は、絶対値が1未満の固有値のみを持たなければなりません。
