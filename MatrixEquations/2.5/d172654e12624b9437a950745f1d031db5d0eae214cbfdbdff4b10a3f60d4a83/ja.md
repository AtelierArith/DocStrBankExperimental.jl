```
M = gsylvop(A, B, C, D; mx, nx, htype = false)
```

一般化T-シルベスター演算子 `M: X -> ∑ A_i*X*B_i + ∑ C_j'*transpose(X)*D_j` を定義します。`htype = false` の場合、または一般化H-シルベスター演算子 `M: X -> ∑ A_i*X*B_i + ∑ C_j'*X'*D_j` を定義します。`htype = true` の場合。`A_i` と `C_j` は同じ行次元を持つ行列であり、`B_i` と `D_j` は同じ列次元を持つ行列です。`A_i` と `B_i` はそれぞれ行列 `A` と `B` の `k`-ベクトルに含まれ、`C_j` と `D_j` はそれぞれ行列 `C` と `D` の `l`-ベクトルに含まれます。任意の成分行列は `UniformScaling` として与えることができます。キーワードパラメータ `mx` と `nx` は、データから推測できない場合に `X` の行次元と列次元を指定するために使用できます。
