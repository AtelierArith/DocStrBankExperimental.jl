```
X = lyapd(A, C)
```

離散リャプノフ方程式の解 `X` を計算します。

```
   AXA' - X + C = 0,
```

ここで `A` は実数または複素数の正方行列であり、`C` は正方行列です。`A` は、2つの固有値 `α` と `β` が `αβ = 1` であることがない必要があります。解 `X` は、`C` が対称またはエルミートであれば対称またはエルミートです。

以下の特定のケースも扱います。

```
X = lyapd(α*I,C)  または  X = lyapd(α,C)
```

行列方程式 `(αα'-1)X + C = 0` を解きます。

```
x = lyapd(α,γ)
```

方程式 `(αα'-1)x + γ = 0` を解きます。

# 例

```jldoctest
julia> A = [3. 4.; 5. 6.]
2×2 Array{Float64,2}:
 3.0  4.0
 5.0  6.0

julia> C = [1. 1.; 1. 2.]
2×2 Array{Float64,2}:
 1.0  1.0
 1.0  2.0

julia> X = lyapd(A, C)
2×2 Array{Float64,2}:
  0.2375  -0.2125
 -0.2125   0.1375

julia> A*X*A' - X + C
2×2 Array{Float64,2}:
 5.55112e-16  6.66134e-16
 2.22045e-16  4.44089e-16
```
