```
X = hsylvckr(A,B,C; atol::Real=0, rtol::Real=atol>0 ? 0 : N*ϵ)
```

連続H-シルベスター行列方程式の解を計算します。

```
            A*X + adjoint(X)*B = C
```

方程式のクロネッカー積展開を使用します。`A`、`B`、および`C`はそれぞれ`m×n`、`n×m`、および`m×m`の行列であり、`X`は`n×m`の行列です。`atol`と`rtol`は、それぞれランク計算に使用される絶対および相対許容誤差です。デフォルトの相対許容誤差は`N*ϵ`であり、ここで`N = 4*min(m,n)^2`、ϵは`A`の要素型の機械精度です。この関数は大きな次数の行列には推奨されません。
