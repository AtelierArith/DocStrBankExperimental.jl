```
rcond = oprcondest(op; exact = false)
```

線形演算子 `op` の `1`-ノルム逆条件数の推定値 `rcond` を計算します。ここで、`op` は定義されたリャプノフまたはシルベスター演算子のいずれかです。この推定値は、`exact = false` の場合は `1`-ノルムの推定値を使用して $\text{rcond} = 1 / (\|op\|_1\|inv(op)\|_1)$ として計算され、`exact = true` の場合は `1`-ノルムの正確な値が計算されます。`exact = true` オプションは、大きな次数の演算子には推奨されません。
