```
plyapcs!(A,R;adj = false)
```

正の連続リャプノフ行列方程式を解きます

```
            op(A)X + Xop(A)' + op(R)*op(R)' = 0
```

ここで、`X = op(U)*op(U)'`、`op(K) = K` ならば `adj = false` の場合、`op(K) = K'` ならば `adj = true` です。`A` は実シュール形式の正方行列または複素シュール形式の正方行列であり、`R` は上三角行列です。`A` は負の実部を持つ固有値のみを持たなければなりません。出力の `R` には解 `U` が含まれます。
