```
X = tlyapckr(A,C, isig = 1; atol::Real=0, rtol::Real=atol>0 ? 0 : N*ϵ)
```

`isig = ±1` の場合に連続 T-Lyapunov 行列方程式の解を計算します。

```
            A*X + isig*transpose(X)*transpose(A) + C = 0
```

方程式のクロンカー積展開を使用します。`A` と `C` はそれぞれ `m×n` および `m×m` 行列であり、`X` は `n×m` 行列です。行列 `C` は `isig = 1` の場合は対称でなければならず、`isig = -1` の場合は反対称でなければなりません。`atol` と `rtol` はそれぞれランク計算に使用される絶対および相対許容誤差です。デフォルトの相対許容誤差は `N*ϵ` であり、ここで `N = 4*min(m,n)^2` で、ϵ は `A` の要素型の機械精度です。この関数は大きな次数の行列には推奨されません。
