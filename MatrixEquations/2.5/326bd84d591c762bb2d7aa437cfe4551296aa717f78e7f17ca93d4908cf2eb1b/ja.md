```
X = sylvd(A,B,C)
```

離散シルベスター行列方程式を解きます。

```
            AXB + X = C
```

バーテルス-スチュワートのシュール形式に基づく拡張を使用します。`A` と `B` は正方行列であり、`A` と `-B` は共通の逆固有値を持ってはいけません。

以下の特定のケースも扱います。

```
X = sylvd(α*I,B,C)  または  X = sylvd(α,B,C)
```

行列方程式 `X(αB+I)  = C` を解きます。

```
X = sylvd(A,β*I,C)   または  X = sylvd(A,β,C)
```

行列方程式 `(βA+I)X = C` を解きます。

```
X = sylvd(α*I,β*I,C)  または  X = sylvd(α,β,C)
```

行列方程式 `(αβ+1)X = C` を解きます。

```
x = sylvd(α,β,γ)
```

方程式 `(αβ+1)x = γ` を解きます。

# 例

```jldoctest
julia> A = [3. 4.; 5. 6.]
2×2 Array{Float64,2}:
 3.0  4.0
 5.0  6.0

julia> B = [1. 1.; 1. 2.]
2×2 Array{Float64,2}:
 1.0  1.0
 1.0  2.0

julia> C = [-1. -2.; 2. -1.]
2×2 Array{Float64,2}:
 -1.0  -2.0
  2.0  -1.0

julia> X = sylvd(A, B, C)
2×2 Array{Float64,2}:
 -2.46667  -2.73333
  2.4       1.86667

julia> A*X*B + X - C
2×2 Array{Float64,2}:
  8.88178e-16   8.88178e-16
 -3.9968e-15   -5.55112e-15
```
