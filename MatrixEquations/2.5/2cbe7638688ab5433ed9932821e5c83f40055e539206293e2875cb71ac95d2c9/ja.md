```
X = lyapc(A, C)
```

連続リャプノフ方程式の解 `X` を計算します。

```
  AX + XA' + C = 0,
```

ここで `A` は実数または複素数の正方行列であり、`C` は正方行列です。`A` は、`α` と `β` の2つの固有値が `α+β = 0` であってはなりません。解 `X` は、`C` が対称またはエルミートであれば対称またはエルミートです。

以下の特定のケースも扱います。

```
X = lyapc(α*I,C)  または  X = lyapc(α,C)
```

行列方程式 `(α+α')X + C = 0` を解きます。

```
x = lyapc(α,γ)
```

方程式 `(α+α')x + γ = 0` を解きます。

# 例

```jldoctest
julia> A = [3. 4.; 5. 6.]
2×2 Array{Float64,2}:
 3.0  4.0
 5.0  6.0

julia> C = [1. 1.; 1. 2.]
2×2 Array{Float64,2}:
 1.0  1.0
 1.0  2.0

julia> X = lyapc(A, C)
2×2 Array{Float64,2}:
  0.5  -0.5
 -0.5   0.25

julia> A*X + X*A' + C
2×2 Array{Float64,2}:
 -8.88178e-16   2.22045e-16
  2.22045e-16  -4.44089e-16
```
