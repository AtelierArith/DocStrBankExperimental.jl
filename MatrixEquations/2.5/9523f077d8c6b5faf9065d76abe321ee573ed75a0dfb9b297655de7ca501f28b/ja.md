```
lyapcs!(A, E, C; adj = false)
```

一般化連続リャプノフ行列方程式を解きます

```
            op(A)Xop(E)' + op(E)Xop(A)' + C = 0,
```

ここで `op(A) = A` および `op(E) = E` で `adj = false` の場合、`op(A) = A'` および `op(E) = E'` で `adj = true` の場合です。ペア `(A,E)` は一般化された実数または複素数のシュール形式にあり、`C` は対称行列またはエルミート行列です。ペンシル `A-λE` は、`α` と `β` の2つの固有値が `α+β = 0` であってはなりません。計算された対称またはエルミートの解 `X` は `C` に含まれます。
