```
X = gsylv(A,B,C,D,E)
```

一般化されたシルベスター行列方程式を解きます

```
          AXB + CXD = E
```

一般化されたシュール形式に基づいて。`A`、`B`、`C`、および `D` は正方行列です。ペンシル `A-λC` と `D+λB` は正則であり、共通の固有値を持ってはいけません。

以下の特定のケースも扱います：

```
X = gsylv(A,B,E)
```

一般化されたシルベスター行列方程式 `AXB  = E` を解きます。

```
X = gsylv(A,B,γ*I,E)  or  X = gsylv(A,B,γ,E)
```

一般化されたシルベスター行列方程式 `AXB +γX = E` を解きます。

```
X = gsylv(A,B,γ*I,D,E)  or  X = gsylv(A,B,γ,D,E)
```

一般化されたシルベスター行列方程式 `AXB +γXD = E` を解きます。

```
X = gsylv(A,B,C,δ*I,E)  or  X = gsylv(A,B,C,δ,E)
```

一般化されたシルベスター行列方程式 `AXB +CXδ = E` を解きます。

```jldoctest
julia> A = [3. 4.; 5. 6.]

 5.0  6.0

julia> B = [1. 1.; 1. 2.]

 1.0  1.0
 1.0  2.0

julia> C = [-1. -2.; 2. -1.]

  2.0  -1.0

julia> D = [1. -2.; -2. -1.]

  1.0  -2.0
 -2.0  -1.0

julia> E = [1. -1.; -2. 2.]

  1.0  -1.0
 -2.0   2.0

 -0.52094   -0.0275792
```
