```
plyapds!(A,E,R;adj = false)
```

一般化された正の離散リャプノフ行列方程式を解きます。

```
            op(A)Xop(A)' - op(E)Xop(E)' + op(R)*op(R)' = 0
```

ここで、`X = op(U)*op(U)'`、`op(K) = K` もし `adj = false` の場合、`op(K) = K'` もし `adj = true` の場合です。正方行列のペア `(A,E)` は一般化シュール形式にあり、`R` は上三角行列です。`A-λE` は絶対値が1未満の固有値のみを持たなければなりません。出力の `R` には上三角解 `U` が含まれます。
