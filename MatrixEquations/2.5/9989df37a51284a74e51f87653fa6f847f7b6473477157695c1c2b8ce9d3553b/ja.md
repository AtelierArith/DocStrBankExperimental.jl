```
plyapcs!(A,E,R;adj = false)
```

一般化された正の連続リャプノフ行列方程式を解きます。

```
            op(A)Xop(E)' + op(E)*Xop(A)' + op(R)*op(R)' = 0
```

ここで、`X = op(U)*op(U)'`、`op(K) = K` もし `adj = false` の場合、`op(K) = K'` もし `adj = true` の場合です。ペア `(A,E)` は一般化された実数/複素数シュール形式にあり、`R` は上三角行列です。ペンシル `A-λE` は負の実部を持つ固有値のみを持たなければなりません。出力の `R` には解 `U` が含まれます。
