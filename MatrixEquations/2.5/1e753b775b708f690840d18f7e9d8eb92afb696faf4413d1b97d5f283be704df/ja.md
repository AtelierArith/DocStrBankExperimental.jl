```
lyapci(A, E, C; abstol, reltol, maxiter) -> (X,info)
```

一般化連続リャプノフ方程式の解 `X` を計算します

```
AXE' + EXA' + C = 0,
```

ここで `A` と `E` は正方形の実数または複素行列であり、`C` は正方行列です。最小二乗解 `X` は、適切に定義されたリャプノフ線形演算子 `L:X -> Y` に適用される共役勾配法に基づく反復法を使用して決定されます。ここで `L(X) = C` または `norm(L(X) - C)` が最小化されます。キーワード引数 `abstol`（デフォルト: `abstol = 0`）と `reltol`（デフォルト: `reltol = sqrt(eps())`）は、計算された解の精度に対する望ましい許容誤差を提供するために使用できます。また、キーワード引数 `maxiter` は最大反復回数を設定するために使用できます（デフォルト: `maxiter = 1000`）。
