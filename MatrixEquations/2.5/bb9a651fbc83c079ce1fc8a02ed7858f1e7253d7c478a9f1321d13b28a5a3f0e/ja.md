```
U = plyapc(A, B)
```

連続リャプノフ方程式の解 `X = UU'` の上三角因子 `U` を計算します。

```
  AX + XA' + BB' = 0,
```

ここで `A` は実数または複素数の正方行列であり、`B` は `A` と同じ行数を持つ行列です。`A` は負の実部を持つ固有値のみを持たなければなりません。

```
U = plyapc(A', B')
```

連続リャプノフ方程式の解 `X = U'U` の上三角因子 `U` を計算します。

```
  A'X + XA + B'B = 0,
```

ここで `A` は実数または複素数の正方行列であり、`B` は `A` と同じ列数を持つ行列です。`A` は負の実部を持つ固有値のみを持たなければなりません。
