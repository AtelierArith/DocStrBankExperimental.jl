```
lacn2!(V, X, EST, KASE, ISAVE ) -> (EST, KASE )
```

複素線形演算子 `A` の 1-ノルムを推定するために、ベクトルに演算子またはその随伴を適用する逆通信を使用します。`KASE` はノルム評価プロセスを制御するパラメータで、次のように動作します。最初の呼び出しでは、`KASE` は `0` である必要があります。中間の戻り値では、`KASE` は `1` または `2` になり、次の呼び出しで複素ベクトル `X` を `A * X` または `A' * X` で上書きするかどうかを示します。最終的な戻り値では、`KASE` は再び `0` になり、`EST` は `A` の 1-ノルムの推定値（下限）です。`V` は複素作業ベクトルで、`ISAVE` は呼び出し間で情報を保存するために使用される 3 次元整数ベクトルです。LAPACK サブルーチン ZLACN2/CLACN2 へのインターフェース。
