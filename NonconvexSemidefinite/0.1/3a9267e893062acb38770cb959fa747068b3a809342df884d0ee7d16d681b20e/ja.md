```
SDPBarrierOptions(; kwargs...)
```

指定できるキーワード引数は次のとおりです：

  * `c_init`: (デフォルト 1.0) バリア項に掛けられる係数 `c` の初期値で、複数の半正定値制約がある場合は実数またはベクトルである可能性があります。
  * `c_decr`: (デフォルト 0.1) 各イテレーションでバリア項に掛けられる減少率 (< 1) で、複数の半正定値制約がある場合は実数またはベクトルである可能性があります。
  * `n_iter`: (デフォルト 20) バリア法で解決するサブ問題の数。
  * `sub_options`: サブ問題のソルバーのオプション
  * `keep_all`: (デフォルト `falue`) `true` に設定されている場合、`SDPBarrierResult` はすべてのイテレーションからの結果を保存します。
