```
merge!(sc, sc1...)
```

これは、SortedDictsまたはSortedMultiDicts `sc1` などを `sc` にマージすることで `sc` を更新します。これらはすべて同じキー-バリュータイプを持っている必要があります。引数の間でキーが重複している場合、キーを所有する右側の引数の値がSortedDictに保存されます。SortedMultiDictの場合、すべてのキー-バリューペアが保存され、重複するキーについては左から右への順序が維持されます。この関数はSortedSetには利用できませんが、`union!` 関数（下記参照）が同等の機能を提供します。時間: O(*cN* log *N*), ここで *N* はすべての引数の合計サイズです。
