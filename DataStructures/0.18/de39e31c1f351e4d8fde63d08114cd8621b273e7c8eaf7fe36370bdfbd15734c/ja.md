```
isequal(sc1,sc2)
```

2つのコンテナが同じアイテムを含んでいるという意味で等しいかどうかをチェックします。キーは `eq` メソッドを使用して比較され、値は `isequal` 関数で比較されます。SortedMultiDictの場合、等価性は特定のキーに関連付けられた値が同じ順序（つまり、同じ挿入順序）であることを要求します。この意味での `isequal` は、`sc1` のアイテムに対するセミトークンと `sc2` のそれとの間に対応関係があることを示唆するものではありません。キーと値に関連付けられた等価性テストメソッドが SortedDict の場合にハッシュ等価性を示唆する場合、コンテナ全体の `isequal` はコンテナのハッシュ等価性を示唆します。時間: O(*cn* + *n* log *n*)
