```
Stress(; kwargs...)(adj_matrix)
stress(adj_matrix; kwargs...)
```

ストレス主要化を使用してグラフレイアウトを計算します。ネットワークの隣接行列表現を取り、ノードの座標を返します。

解決すべき主な方程式は、Gansner, Koren, and North (2005, [doi 10.1007/978-3-540-31843-9_25](https://doi.org/10.1007/978-3-540-31843-9_25)) の式 (8) です。

## 入力:

  * `adj_matrix`: ペアワイズ距離の行列。

## キーワード引数

  * `dim=2`, `Ptype=Float64`: 次元と出力タイプ `Point{dim,Ptype}` を決定します。
  * `iterations=:auto`: 最大反復回数（`:auto` は `400*N^2` を意味し、`N` は頂点の数です）
  * `abstols=0`

    ストレスの収束に対する絶対許容誤差。2つの連続するストレスの差がabstol未満になると、反復が終了します。
  * `reltols=10e-6`

    ストレスの収束に対する相対許容誤差。2つの連続するストレスの差が現在のストレスに対してreltol未満になると、反復が終了します。
  * `abstolx=10e-6`

    レイアウトの収束に対する絶対許容誤差。2つの連続するレイアウトのフロベニウスノルムがabstolx未満になると、反復が終了します。
  * `weights=Array{Float64}(undef, 0, 0)`

    重みの行列。空である場合（つまり、指定されていない場合）、`δ[i,j]` が非ゼロの場合は `weights[i,j] = δ[i,j]^-2` にデフォルト設定され、そうでない場合は `0` になります。
  * `initialpos=Point{dim,Ptype}[]`

    初期位置の `Vector` または `Dict` を提供します。すべての位置は正規分布からのランダム座標を使用して初期化されます。ランダム位置は、この引数で提供されたキー-バリューペアを使用して上書きされます。
  * `pin=[]`: ノード位置をピン留めします（更新されません）。ノードインデックス -> 値のペアリングの `Vector` または `Dict` として指定できます。値は次のいずれかです。

      * `(12, 4.0)` : 初期位置を上書きし、ピン留めします
      * `true/false` : この位置をピン留めします
      * `(true, false, false)` : 特定の座標のみをピン留めします
  * `seed=1`: ランダム初期位置のシード。
  * `rng=DEFAULT_RNG[](seed)`

    シードに基づいてrngを作成します。デフォルトは `MersenneTwister` で、`DEFAULT_RNG[]` を上書きすることで指定できます。
  * `uncon_dist=(maxdist, Ncomps)->maxdist*Ncomps^(1/3)`

    デフォルトでは、グラフ内の接続されていない頂点には、接続された成分の数と成分内の最大距離にスケーリングされるペアワイズ「理想」距離が与えられます。
