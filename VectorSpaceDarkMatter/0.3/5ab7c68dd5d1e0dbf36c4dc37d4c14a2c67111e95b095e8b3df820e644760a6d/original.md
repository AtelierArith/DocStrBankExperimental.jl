```
kinematic_I(lnvnq_max, model::ModelDMSM, 
            v_basis::RadialBasis, q_basis::RadialBasis)
```

Computes the kinematic scattering matrix $\mathcal{I}^\ell_{n,n'}$ for a given `model`, `v_basis`, and `q_basis`. `lnvnq_max` $= (\ell_{\textrm{max}}, n_{\textrm{max}}^v, n_{\textrm{max}}^q)$.
