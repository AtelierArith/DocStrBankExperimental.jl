現在のサブプロットインデックスを設定します。

デフォルトでは、現在のプロットはウィンドウ全体をカバーします。複数のプロットを表示するには、ウィンドウを行と列の数に分割し、現在のプロットが結果のグリッド内の1つまたは複数のセルをカバーするようにします。

サブプロットインデックスは1から始まり、左上隅から始まります。新しい行は、**num_columns**サブプロットごとに始まります。

:param num*rows: サブプロット行の数 :param num*columns: サブプロット列の数 :param subplot_indices: 	- 現在のプロットで使用されるサブプロットインデックス 	- 現在のプロットがカバーすべきサブプロットのペア

**使用例:**

.. code-block:: julia

```
julia> # 2x3グリッドの2番目のサブプロットに現在のプロットを設定
julia> subplot(2, 3, 2)
julia> # 4x2グリッドの最初の2行をカバーするように現在のプロットを設定
julia> subplot(4, 2, (1, 4))
julia> # 現在のプロットにウィンドウ全体を使用
julia> subplot(1, 1, 1)
```
