現在の図をファイルに保存します。

この関数は、GRのワークステーショントタイプの1つを使用して、指定された名前のファイルを作成するために現在の図を描画します。サポートされているファイルタイプはインストールされているワークステーショントタイプによって異なりますが、GRは通常、.png、.jpg、.pdf、.ps、.gifおよびさまざまな他のファイル形式のサポートを備えています。

:param filename: 図を保存するファイル名

**使用例:**

.. code-block:: julia

```
julia> # 簡単なプロットを作成
julia> x = 1:100
julia> plot(x, 1 ./ (x .+ 1))
julia> # 図をファイルに保存
julia> savefig("example.png")
```
