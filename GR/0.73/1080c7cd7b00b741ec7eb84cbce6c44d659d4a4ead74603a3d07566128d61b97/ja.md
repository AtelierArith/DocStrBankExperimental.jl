```
setspace(zmin::Real, zmax::Real, rotation::Int, tilt::Int)
```

現在のワールド座標空間に三次元出力プリミティブをマッピングするために使用される抽象Z空間を設定します。

**パラメータ:**

`zmin` :     Z軸の最小値。 `zmax` :     Z軸の最大値。 `rotation` :     X軸の回転角度（度）。 `tilt` :     Z軸の視点角度（度）。

`setspace`は抽象Z軸の限界を設定し、シミュレートされた三次元グラフの回転角度と視点角度（ティルト）を定義します。これにより、対応する出力プリミティブを現在のウィンドウにマッピングします。これらの設定は、他の値が指定されるまで、すべての後続の三次元出力プリミティブに使用されます。回転角度と視点角度は0°から90°の間で指定する必要があります。
