```
drawpath(points, codes, fill::Int)
```

線分とベジェ曲線からなる単純および複合のアウトラインを描画します。

**パラメータ:**

`points` :     (N, 2) 配列の (x, y) 頂点 `codes` :     N 長さの配列のパスコード `fill` :     結果のパスを塗りつぶすかどうかを示すフラグ

以下のパスコードが認識されます：

```
+----------+-----------------------------------------------------------+
|      STOP|全体のパスを終了する                                     |
+----------+-----------------------------------------------------------+
|    MOVETO|指定された頂点に移動する                                 |
+----------+-----------------------------------------------------------+
|    LINETO|現在の位置から指定された頂点まで線を描画する           |
+----------+-----------------------------------------------------------+
|    CURVE3|2次ベジェ曲線を描画する                                  |
+----------+-----------------------------------------------------------+
|    CURVE4|3次ベジェ曲線を描画する                                  |
+----------+-----------------------------------------------------------+
| CLOSEPOLY|現在のパスの開始点まで線分を描画する                   |
+----------+-----------------------------------------------------------+
```
