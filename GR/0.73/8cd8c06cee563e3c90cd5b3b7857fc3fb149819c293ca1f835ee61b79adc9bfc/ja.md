```
setwindow(xmin::Real, xmax::Real, ymin::Real, ymax::Real)
```

`setwindow` は、プロットされるワールド座標のウィンドウまたは長方形のサブスペースを確立します。対数スケーリングや軸のミラーイメージを希望する場合は、SETSCALE関数を使用してください。

**パラメータ:**

`xmin` :     ウィンドウの左水平座標 (`xmin` < `xmax`). `xmax` :     ウィンドウの右水平座標. `ymin` :     ウィンドウの下垂直座標 (`ymin` < `ymax`). `ymax` :     ウィンドウの上垂直座標.

`setwindow` は、指定された正規化変換に関連付けられるワールド座標空間 (WC) の長方形部分を定義します。WCウィンドウと正規化デバイス座標 (NDC) ビューポートは、すべての出力プリミティブがマッピングされる正規化変換を定義します。WCウィンドウは、長方形のNDCビューポートにマッピングされ、さらにそれは、デバイス座標で開いているアクティブなワークステーションの表示面にマッピングされます。デフォルトでは、GRはワールド座標での範囲 [0,1] x [0,1] を正規化変換ウィンドウとして使用します。
