```
setwsviewport(xmin::Real, xmax::Real, ymin::Real, ymax::Real)
```

ワークステーションのグラフィックスウィンドウのサイズをメートル単位で定義します。

**パラメータ:**

`xmin` :     ワークステーションビューポートの左水平座標。 `xmax` :     ワークステーションビューポートの右水平座標。 `ymin` :     ワークステーションビューポートの下垂直座標。 `ymax` :     ワークステーションビューポートの上垂直座標。

`setwsviewport` は、指定されたサイズのワークステーションウィンドウをディスプレイ上に配置します。このコマンドは、ディスプレイやハードコピーデバイスに対してワークステーションウィンドウを正確にサイズ調整できるようにし、デスクトップパブリッシングアプリケーションのグラフサイズ調整に役立ちます。
