```
setwswindow(xmin::Real, xmax::Real, ymin::Real, ymax::Real)
```

ワークステーションウィンドウに描画されるNDCビューポートの領域を設定します。

**パラメータ:**

`xmin` :     ワークステーションウィンドウの左の水平座標。 `xmax` :     ワークステーションウィンドウの右の水平座標 (0 <= `xmin` < `xmax` <= 1)。 `ymin` :     ワークステーションウィンドウの下の垂直座標。 `ymax` :     ワークステーションウィンドウの上の垂直座標 (0 <= `ymin` < `ymax` <= 1)。

`setwswindow` は、デバイスに出力される正規化デバイス座標空間の長方形の領域を定義します。デフォルトでは、ワークステーショントランスフォーメーションは、NDCの範囲 [0,1] x [0,1] をワークステーションの表示面上の最大の正方形にマッピングします。ワークステーションウィンドウのアスペクト比は1対1に維持されます。
