```
setfillstyle(index::Int)
```

次の塗りつぶし領域に使用される塗りつぶしスタイルを設定します。

**パラメータ:**

`index` :     使用する塗りつぶしスタイルのインデックス

`setfillstyle` は、`setfillintstyle` 関数によって PATTERN 塗りつぶしまたは HATCH 塗りつぶしが要求されたときにインデックスを指定します。内部スタイルが PATTERN に設定されている場合、塗りつぶしスタイルのインデックスはデバイスに依存しないパターンテーブルを指します。内部スタイルが HATCH に設定されている場合、塗りつぶしスタイルのインデックスは異なるハッチスタイルを示します。内部スタイルに HOLLOW または SOLID が指定されている場合、塗りつぶしスタイルのインデックスは使用されません。
