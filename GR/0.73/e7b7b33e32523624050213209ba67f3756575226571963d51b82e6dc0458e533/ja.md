```
grid(x_tick::Real, y_tick::Real, x_org::Real, y_org::Real, major_x::Int, major_y::Int)
```

線形および/または対数グリッドを描画します。

**パラメータ:**

`x_tick`, `y_tick` :     マイナーグリッド線の間隔のワールド座標における長さ。 `x_org`, `y_org` :     グリッドの原点（交差点）のワールド座標。 `major_x`, `major_y` :     メジャーグリッド線の間のマイナーグリッド線の数を指定する単位のない整数値。 0または1の値は、グリッド線がないことを意味します。

メジャーグリッド線は、軸の原点およびメジャー目盛りに対応し、表示されているかどうかにかかわらず存在します。マイナーグリッド線は、マイナー目盛りと等しい点に描画されます。メジャーグリッド線は黒い線で描画され、マイナーグリッド線は灰色の線で描画されます。
