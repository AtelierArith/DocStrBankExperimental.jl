```
spline(x, y, m, method)
```

最初のデータポイントから最後のデータポイントまでの間で、三次スプラインフィットを生成します。

**パラメータ:**

`x` :     X座標を含むリスト `y` :     Y座標を含むリスト `m` :     描画されるポリゴンの点の数（`m` > len(`x`）） `method` :     スムージングメソッド

`x` と `y` の値はワールド座標で表されます。スプラインフィットの外観を制御する属性は、線の種類、線の幅、色インデックスです。

`method` が > 0 の場合、一般化交差検証スムージングスプラインが計算されます。`method` が 0 の場合、補間自然三次スプラインが計算されます。`method` が < -1 の場合、三次Bスプラインが計算されます。
