```
textext(x::Real, y::Real, string)
```

現在のテキスト属性を使用して、位置 `x`、`y` にテキストを描画します。文字列は基本的な数学的表現やギリシャ文字を作成するために定義できます。

**パラメータ:**

`x` :     テキスト文字列の開始位置のX座標 `y` :     テキスト文字列の開始位置のY座標 `string` :     描画するテキスト

XおよびYの値は正規化デバイス座標で表されます。テキストの外観を制御する属性は、テキストフォントと精度、文字拡張係数、文字間隔、テキストカラーインデックス、文字の高さ、文字の上向きベクトル、テキストパス、およびテキストの整列です。

文字列は単純な数学的式として解釈されます。以下の表記が適用されます：

下付き文字と上付き文字：これらはキャレット（'^'）とアンダースコア（'_'）で示されます。下付き文字または上付き文字が複数の文字を含む場合は、中括弧（'{}'）で囲む必要があります。

分数は A '/' B で組版され、A は分子、B は分母を表します。

ギリシャ文字を含めるには、バックスラッシュ（'\'）文字の後に対応するキーワードを指定する必要があります。テキスト翻訳者は、キーワードのケースに応じて大文字または小文字のギリシャ文字を生成します。
