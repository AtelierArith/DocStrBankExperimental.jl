```
setclip(indicator::Int)
```

クリッピングインジケーターを設定します。

**パラメータ:**

`indicator` :     クリッピングがオンかオフかを指定するインジケーター。

```
+----+---------------------------------------------------------------+
|   0|クリッピングはオフです。ウィンドウの外側のデータが描画されます。     |
+----+---------------------------------------------------------------+
|   1|クリッピングはオンです。ウィンドウの外側のデータは描画されません。  |
+----+---------------------------------------------------------------+
```

`setclip` は、現在のウィンドウに描画される画像のクリッピングを有効または無効にします。クリッピングは、定義されたビューポートの外にあるグラフの部分を削除することとして定義されます。クリッピングがオンの場合、GRはビューポートの境界を超えて生成された出力プリミティブを描画しません。クリッピングがオフの場合、プリミティブはビューポートの境界を超えることができ、ワークステーションウィンドウの端まで描画されます。デフォルトでは、クリッピングはオンです。
