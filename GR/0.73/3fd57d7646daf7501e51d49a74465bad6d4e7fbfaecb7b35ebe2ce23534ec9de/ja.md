```
fillarc(xmin::Real, xmax::Real, ymin::Real, ymax::Real, a1::Real, a2::Real)
```

指定された長方形を覆う円弧または楕円弧を塗りつぶします。

**パラメータ:**

`xmin` :     長方形の左下の辺 `xmax` :     長方形の右下の辺 `ymin` :     長方形の左上の辺 `ymax` :     長方形の右上の辺 `a1` :     開始角度 `a2` :     終了角度

結果として得られる弧は `a1` で始まり `a2` 度で終わります。角度は0度が3時の位置にあると解釈されます。弧の中心は与えられた長方形の中心です。
