```
selntran(transform::Int)
```

`selntran` は、ワールド座標から正規化デバイス座標への事前定義された変換を選択します。

**パラメータ:**

`transform` :     正規化変換番号。

```
+------+----------------------------------------------------------------------------------------------------+
|     0|ウィンドウとビューポートの範囲が 0 から 1 である恒等変換を選択します。                          |
+------+----------------------------------------------------------------------------------------------------+
|  >= 1|`setwindow` と `setviewport` によって定義された正規化変換を選択します。                       |
+------+----------------------------------------------------------------------------------------------------+
```
