```
openws(workstation_id::Int, connection, workstation_type::Int)
```

グラフィカルワークステーションを開きます。

**パラメータ:**

`workstation_id` :     ワークステーション識別子。 `connection` :     接続識別子。 `workstation_type` :     希望するワークステーションの種類。
