```
contour(px, py, h, pz, major_h::Int)
```

三次元データセットの等高線を描画します。データの値は長方形のメッシュ上で指定されます。等高線はオプションでラベル付けできます。

**パラメータ:**

`x` :     X座標を含むリスト `y` :     Y座標を含むリスト `h` :     高さ値のZ座標を含むリスト `z` :     長さが `len(x)` * `len(y)` のリストまたは適切な次元の配列で、Z座標を含みます `major_h` :     GRに等高線をラベル付けするよう指示します。例えば、値が3の場合は毎第3の線にラベルが付けられます。値が1の場合はすべての線にラベルが付けられます。値が0の場合はラベルは付きません。色付きの等高線を生成するには、`major_h` に1000のオフセットを追加します。
