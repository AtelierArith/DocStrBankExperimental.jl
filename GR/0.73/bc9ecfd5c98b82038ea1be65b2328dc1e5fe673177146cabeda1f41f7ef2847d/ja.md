```
setviewport(xmin::Real, xmax::Real, ymin::Real, ymax::Real)
```

`setviewport` は正規化デバイス座標の矩形サブスペースを確立します。

**パラメータ:**

`xmin` :     ビューポートの左水平座標。 `xmax` :     ビューポートの右水平座標 (0 <= `xmin` < `xmax` <= 1)。 `ymin` :     ビューポートの下垂直座標。 `ymax` :     ビューポートの上垂直座標 (0 <= `ymin` < `ymax` <= 1)。

`setviewport` は、指定された正規化変換に関連付けられる正規化デバイス座標 (NDC) 空間の矩形部分を定義します。NDCビューポートとワールド座標 (WC) ウィンドウは、すべての出力プリミティブが通過する正規化変換を定義します。WCウィンドウは、矩形のNDCビューポートにマッピングされ、さらにオープンでアクティブなワークステーションの表示面にデバイス座標でマッピングされます。
