```
setshadow(offsetx::Real, offsety::Real, blur::Real)
```

`setshadow` は、グラフィックスオブジェクトの下に描かれた画像によって実現される影を描画することを可能にし、影がグラフィックスオブジェクトに投影される光源の効果を模倣します。

**パラメータ:**

`offsetx` :     オブジェクトから影が水平方向にどれだけオフセットされるかを指定するxオフセット `offsety` :     オブジェクトから影が垂直方向にどれだけオフセットされるかを指定するyオフセット `blur` :     オブジェクトがハードエッジかディフューズエッジかを指定するぼかし値
