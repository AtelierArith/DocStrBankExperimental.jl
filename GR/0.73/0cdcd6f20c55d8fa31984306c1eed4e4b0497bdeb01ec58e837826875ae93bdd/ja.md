```
setfillintstyle(style::Int)
```

塗りつぶし領域の内部スタイルを設定します。

**パラメータ:**

`style` :     使用する塗りつぶしのスタイル

`setfillintstyle` は、後続の塗りつぶし領域出力プリミティブの内部スタイルを定義します。デフォルトの内部スタイルは HOLLOW です。

```
+---------+---+--------------------------------------------------------------------------------+
|HOLLOW   |  0|塗りつぶしなし。境界ポリラインを描画するだけです。                             |
+---------+---+--------------------------------------------------------------------------------+
|SOLID    |  1|塗りつぶし色インデックスを使用してポリゴンの内部を塗りつぶします。               |
+---------+---+--------------------------------------------------------------------------------+
|PATTERN  |  2|スタイルインデックスをパターンインデックスとして使用してポリゴンの内部を塗りつぶします。 |
+---------+---+--------------------------------------------------------------------------------+
|HATCH    |  3|スタイルインデックスを交差ハッチスタイルとして使用してポリゴンの内部を塗りつぶします。 |
+---------+---+--------------------------------------------------------------------------------+
```
