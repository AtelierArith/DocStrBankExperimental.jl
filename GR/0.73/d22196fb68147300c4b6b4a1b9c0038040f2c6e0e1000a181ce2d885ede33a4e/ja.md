```
drawarc(xmin::Real, xmax::Real, ymin::Real, ymax::Real, a1::Real, a2::Real)
```

指定された矩形をカバーする円弧または楕円弧を描画します。

**パラメータ:**

`xmin` :     矩形の左下の辺 `xmax` :     矩形の右下の辺 `ymin` :     矩形の左上の辺 `ymax` :     矩形の右上の辺 `a1` :     開始角度 `a2` :     終了角度

結果として得られる弧は `a1` から始まり `a2` 度で終わります。角度は0度が3時の位置にあると解釈されます。弧の中心は指定された矩形の中心です。
