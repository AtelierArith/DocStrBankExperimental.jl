```
subgrad_distance(M, q, p[, c = 2; atol = 0])
subgrad_distance!(M, X, q, p[, c = 2; atol = 0])
```

距離のサブグラディエントを計算します（`X`の代わりに）

$$
f(p) = \frac{1}{c} d^c_{\mathcal M}(p, q)
$$

多様体`M`上の固定点`q`に対して、`c`は整数です。サブグラディエントは次のように表されます。

$$
\partial f(p) = -d_{\mathcal M}^{c-2}(p, q)\log_pq
$$

ただし、$c\neq 1$または$p\neq  q$の場合です。残りのケース$c=1$、$p=q$では、関数は微分可能ではありません。この場合、サブグラディエントは、ノルムが1以下の点`p`での接ベクトルによって与えられます。

# オプション

  * `c` – （`2`）距離の指数、すなわちデフォルトは距離です
  * `atol` – （`0`）`p`と`q`の間の距離を評価する際に使用する許容誤差です。
