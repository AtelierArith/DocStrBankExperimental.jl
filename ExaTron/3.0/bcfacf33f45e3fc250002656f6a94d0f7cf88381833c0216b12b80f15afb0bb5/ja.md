サブルーチン dspcg

このサブルーチンは、サブ問題の近似最小化子のシーケンスを生成します。

```
min { q(x) : xl <= x <= xu }.
```

二次関数は次のように定義されます。

```
q(x[0]+s) = 0.5*s'*A*s + g'*s,
```

ここで、x[0]はユーザーによって提供された基準点、Aは圧縮列ストレージの対称行列、gはベクトルです。

各ステージで、近似最小化子x[k]を持ち、サブ問題に対して前処理共役勾配法を使用して方向p[k]を生成します。

```
min { q(x[k]+p) : || L'*p || <= delta, s(fixed) = 0 },
```

ここで、fixedはx[k]で固定された変数の集合、deltaは信頼領域の境界、Lはサブ行列の不完全なコレスキー分解です。

```
B = A(free:free),
```

ここで、freeはx[k]で自由な変数の集合です。p[k]が与えられると、次の最小化子x[k+1]は投影探索によって生成されます。

このサブルーチンの出発点はx[1] = x[0] + sであり、ここでx[0]は基準点、sはコーシーステップです。

サブルーチンは、ステップsが次の条件を満たすと収束します。

```
|| (g + A*s)[free] || <= rtol*|| g[free] ||
```

この場合、最終的なxは自由変数によって定義された面での近似最小化子です。

サブルーチンは、信頼領域の境界がさらなる進展を許さないときに終了します。すなわち、|| L'*p[k] || = deltaです。この場合、最終的なxはq(x) < q(x[k])を満たします。
