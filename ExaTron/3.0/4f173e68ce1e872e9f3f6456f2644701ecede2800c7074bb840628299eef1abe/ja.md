サブルーチン dprsrch

このサブルーチンは、二次関数に対して十分な減少条件を満たすステップを計算するために、投影探索を使用します。

```
q(s) = 0.5*s'*A*s + g'*s,
```

ここで、Aは圧縮列ストレージの対称行列であり、gはベクトルです。パラメータ alpha が与えられたとき、ステップは次のようになります。

```
s[alpha] = P[x + alpha*w] - x,
```

ここで、wは探索方向であり、Pはn次元区間[xl,xu]への投影です。最終的なステップ s = s[alpha] は、十分な減少条件を満たします。

```
q(s) <= mu_0*(g'*s),
```

ここで、mu_0は(0,1)の範囲の定数です。

探索方向 w は、x における二次関数 q の下降方向でなければならず、二次関数は 0 <= alpha <= 1 の範囲で x + alpha*w の光線に沿って減少している必要があります。

MINPACK-2 プロジェクト。1999年3月。アーゴン国立研究所。チー・ジェン・リンとホルヘ・J・モレ。
