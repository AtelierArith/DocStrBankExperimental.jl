サブルーチン dicf

圧縮行ストレージにあるスパース対称行列 A が与えられたとき、このサブルーチンは不完全なコレスキー分解を計算します。

dicf の実装は、Jones-Plassmann コードに基づいています。配列 indf と list はデータ構造を定義します。j 番目の列の計算の開始時に、

```
k < j の場合、indf[k] は k 番目の列の最初の非ゼロ l[i,k] の A のインデックスであり、i >= j です。

k < j の場合、list[i] は i = L.rowval[indf[k]] の列インデックス k のリンクリストへのポインタです。
```

j 番目の列の計算のために、配列 indr は行インデックスを記録します。したがって、nlj が j 番目の列の非ゼロの数である場合、indr[1],...,indr[nlj] は行インデックスです。また、i > j の場合、indf[i] は j 番目の列の行インデックスを示し、l[i,j] がゼロでない場合は indf[i] = 1 となります。

MINPACK-2 プロジェクト。1998年5月。アーゴン国立研究所。リン・チーゼンとホルヘ・J・モレ。
