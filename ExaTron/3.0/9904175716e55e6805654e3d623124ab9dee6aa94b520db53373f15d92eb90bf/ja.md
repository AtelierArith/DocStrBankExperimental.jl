サブルーチン dcauchy

このサブルーチンは、信頼領域制約と十分な減少条件を満たすコーシーステップを計算します。

コーシーステップは、二次形式に対して計算されます。

```
q(s) = 0.5*s'*A*s + g'*s
```

ここで、Aは圧縮行ストレージの対称行列であり、gはベクトルです。パラメータ alpha が与えられたとき、コーシーステップは次のようになります。

```
s[alpha] = P[x - alpha*g] - x,
```

ここで、Pはn次元の区間[xl,xu]への射影です。コーシーステップは信頼領域制約と十分な減少条件を満たします。

```
|| s || <= delta,    q(s) <= mu_0*(g'*s),
```

ここで、mu_0は(0,1)の間の定数です。

MINPACK-2プロジェクト。1999年3月。アーゴンヌ国立研究所。チー・ジェン・リンとホルヘ・J・モレ。
