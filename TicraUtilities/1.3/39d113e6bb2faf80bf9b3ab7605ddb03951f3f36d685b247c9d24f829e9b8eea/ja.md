```
tepp2s(tep::TEPperiodic, d; title="")
```

周期的ユニットセルTEPオブジェクト（`TEPperiodic`型）を散乱面TEPオブジェクト（`TEPscatter`型）に変換します。また、`tep`が複数の周波数を含む場合は、そのようなオブジェクトのベクトルに変換します。

`d`は前面と背面の参照平面間の距離であり、`Unitful`長さ量です。`title`キーワード引数は出力オブジェクトの`title`フィールドに使用されます。デフォルトの空の値のままにすると、`"TEPscatter object created by tepp2s"`に置き換えられます。

## 使用例

```
using Unitful: @u_str
d = 2.3u"mm"
tep_scatter = tepp2s(tep_periodic, d)
```

# 拡張ヘルプ

入力引数`d`は必須です。なぜなら、`TEPperiodic`はユニットセルの実際の前面と背面の表面に位置する位相参照平面を使用するのに対し、`TEPscatter`は前面のみを前面および背面の入射の位相参照平面として使用するからです。したがって、`d`は背面入射反射および前面と背面の入射透過のために必要な位相補正を計算するために必要です。この位相補正は、一部の係数に必要な符号の変更に加えて行われます。
