```
sph2cut(sphfile::AbstractString; theta, phi, ipol) -> cut::Cut

sph2cut(sph:SPHQPartition; theta, phi, ipol) -> cut::Cut

sph2cut(sphs:Vector{SPHQPartition}; theta, phi, ipol) -> cuts::Vector{Cut}
```

Q型球面波モード係数のセットを遠方場の電場値に変換し、`Cut`オブジェクトとして返します。

単一の位置引数は、Ticra互換のQ型球面波ファイルの名前を含む文字列、または`read_sphfile`を使用してそのようなファイルを読み取ったときに返される値のいずれかです。

## オプションのキーワード引数:

  * `theta`: 電場が評価されるべき望ましい極角（コラティチュード）を度単位で示す抽象範囲です。空の範囲が提供された場合（デフォルト）、値は`sph`内のモーダルコンテンツを調べることによって自動的に決定されます。
  * `phi`: 電場が評価されるべき望ましい方位角を度単位で示す抽象範囲です。空の範囲が提供された場合（デフォルト）、値は`sph`内のモーダルコンテンツを調べることによって自動的に決定されます。
  * `ipol`: 計算された電場値のための望ましい偏光分解を示す0:3の範囲の整数です。これらの値の意味は次のとおりです:

      * 0: (デフォルト値) 最大のピークコポルマグニチュードを生成する選択肢2または3の中から基底を選択します。
      * 1: (θ̂, ϕ̂)基底を使用します。
      * 2: (R̂, L̂)（すなわち、円偏光）基底を使用します。
      * 3: (ĥ, v̂)（ルートヴィヒ3）基底を使用します。

## 使用例

```
cut = sph2cut("testfile.sph"; phi=0:5:355, theta=0:1:180, ipol=2)
```
