```
(x,y,z0,z90) = phscen(cut, fghz=11.80285268; min_dropoff=-10)
```

カットファイルまたは `Cut` オブジェクトの位相中心をNSI最小二乗法を使用して推定します。

4つの出力は、データを記録するために使用された原点に対する位相中心の位置の推定値です。 `fghz` が引数として渡されると、値はインチ単位で表されます。それ以外の場合、長さは波長に正規化されます。 `z0` と `z90` は、位相中心位置のphi = 0ᵒおよびphi = 90ᵒ平面の推定値であることに注意してください。位相中心の位置を決定する際には、ピークフィールドに対して `min_dropoff` より大きいdBの大きさを持つフィールド値のみが考慮されます。
