```
valgrad!(g, c::SimpleChain, arg, params)
```

`g` は `params` と同じサイズの `AbstractVector` か、`Tuple{A,G}` であることができます。もし `g` がタプルであれば、最初の要素は `arg` に関する勾配であり、`nothing`（この勾配を取らない場合）であるか、`arg` と同じ `size` である必要があります。2 番目の要素は `params` に関する勾配であり、同様に `nothing` であるか、`params` と同じサイズである必要があります。

許可される破壊:

```
valgrad_layer!
```

前の層の戻り値 (`B`) を受け取り、出力 `C` を返します。内部層の場合、`B` を破壊することが許可されています（例: ドロップアウト層）。

```
pullback!
```

その戻り値の随伴 (`C̄`) を受け取ります。これを破壊することが許可されています。また、前の層の戻り値 `B` を破壊して `B̄` を生成することも許可されています（受け取った `C̄`）。したがって、プルバックは `C` に依存することは許可されていません。なぜなら、`C̄` を生成する際に破壊されている可能性があるからです。
