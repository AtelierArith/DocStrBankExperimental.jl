```
partial_fitted(model::MixedModel,
               fe::AbstractVector{<:AbstractString},
               re::Dict{Symbol}=Dict(fn => fe for fn in fnames(model));
               mode=:include,
               type=:linpred)
```

「部分的な」適合値を計算します。

部分的な適合値は、部分残差を計算するのに役立ちます。これは、選択したモデル項をゼロに設定し、他の値を元の推定値のまま保持することによって得られる適合値です。

使用する固定効果の係数（`fe`）は、文字列のベクトルとして指定されます。係数を指定しない場合は、空の文字列ベクトル `String[]` を使用します。

ランダム効果 `re` は、辞書として指定され、グループ変数がキーとして、グループレベルの係数がベクトルとして指定されます。例えば、`Dict(:subj => ["(Intercept)"])` は `(1|subj)` を保持することを指定します。デフォルトは、すべてのグループ変数に対して指定された固定効果と一致させることですが、固定効果の指定が任意のグループ変数と互換性がない場合、これは失敗します。

キーワード引数 `mode` は、固定効果とランダム効果の指定が `:include` または `:exclude` の係数として扱われるかどうかを指定します。

`GeneralizedLinearMixedModel` の場合、キーワード引数 `type` は、予測が線形予測子のスケール（`:linpred`）で返されるべきか、応答スケール（`:response`）で返されるべきかを指定します。

!!! warning
    部分的な適合値は、リンク関数の非線形性のため、応答スケールでの一般化線形混合モデルに対して誤解を招く可能性があります。例えば、ロジスティック回帰では、部分的な適合値は線形予測子スケール、すなわち対数オッズスケールで計算され、その後応答スケール、すなわち確率スケールに変換されます。しかし、対数オッズスケールでの単純な加算的寄与は、確率スケールでは加算的ではありません。より直接的には、個々の予測因子の効果を元のスケールで単純な加算的寄与に分解することは不可能です。


!!! note
    固定効果とランダム効果の両方において、関連するエンティティは係数名であり、元の項名ではありません。


!!! warning
    切片は**自動的に** / 暗黙的に含まれず、常に明示的に指定する必要があります。


!!! warning
    この機能は、ランク欠陥のある固定効果を持つモデルでテストされておらず、したがって動作が確認されていません。


この機能は、Rの [remef](https://github.com/hohenstein/remef) パッケージに似ています。
