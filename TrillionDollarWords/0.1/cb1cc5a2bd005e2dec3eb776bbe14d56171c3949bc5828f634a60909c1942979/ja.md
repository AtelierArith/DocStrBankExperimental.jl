```
load_training_sentences()
```

アーティファクトからトレーニング文のデータセットをロードします。これは、記者会見、会議の議事録、スピーチからの文を含む結合データセットです。

  * `sentence` 列は文そのものです。
  * `year` 列はイベントの年です。
  * `label` のラベルは論文からの手動で注釈されたラベルです。
  * `seed` 列は、論文でデータをトレーニングセットとテストセットに分割するために使用されたシードです。
  * `sentence_splitting` 列は、文が分割されたかどうかを示します（詳細は論文を参照してください）。
  * `event_type` 列はイベントの種類（会議の議事録、スピーチ、または記者会見）です。
  * `split` 列は、文がトレーニングセットまたはテストセットに含まれているかどうかを示します。
