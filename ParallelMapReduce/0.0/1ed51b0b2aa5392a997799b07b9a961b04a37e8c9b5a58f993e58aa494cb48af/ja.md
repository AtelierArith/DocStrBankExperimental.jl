```
pmapreduce(f, op, [::AbstractWorkerPool], c...; algorithm = :even)
```

利用可能なワーカーを使用した関数 `mapreduce` の並列バージョンで、`f` が `c` の各要素に適用され、結果が `op` を使用して縮約されます。

`algorithm` には3つの選択肢があります：

  * `:even`、`c` の要素をワーカー間で均等に分割します。詳細は `@distributed` を参照してください。
  * `:reduction_local`、`c` の要素が非同期にワーカーに配布されます。各要素は次の利用可能なワーカーに送信されます。`f` の結果は縮約され、ローカルに保存されます。`c` が尽きると、ローカルの結果がマスターに送信され、最終的な縮約が行われます。
  * `:reduction_master`、`:reduction_local` に似ていますが、`f` の各計算の結果がマスターに送信され、すべての縮約が行われます。
