```
Choco(p1, μ0, ϕ0, μ1, ϕ1; p_mid=0, ϕ_mid=100, k0=0, k1=1)
```

選択-信頼 (Choco) モデル分布を構築します。これは、スケールの両側にそれぞれ1つずつの順序付きベータ分布（[`OrderedBeta`](@ref)を参照）を混合したものであり、スケールの中心にクラスタリングされたスコアを許可するための3つ目の（オプションの）非スケールベータ分布を含みます。ベータ分布は、[`BetaPhi2`](@ref) パラメータ化を使用して定義されます。

# 引数

  * `p1`: 左半分（すなわち、0から0.5の間の回答）に対して右半分（すなわち、0.5から1の間の回答）が存在する全体的な確率。デフォルトは0.5で、これは両側（すなわち、「選択」）が同じ確率であることを意味します。
  * `μ0`, `μ1`: 左半分および右半分のベータ分布の平均。
  * `ϕ0`, `ϕ1`: 左半分および右半分のベータ分布の精度。
  * `p_mid`: スケールの中央（すなわち、0.5の周りの回答）に存在する回答の確率。デフォルトは0で、これはモデルが他の2つのベータ分布の単純な混合であることを意味します。
  * `ϕ_mid`: スケールの中央のベータ分布の精度（`p_mid` > 0の場合に関連）。デフォルトは100です。このパラメータは、1のように低くなるべきではなく、フラットな分布を生成し、分布が同定不可能になるためです（同じパターンが他のパラメータの組み合わせでも観察される可能性があります）。
  * `k0`, `k1`: 極端な値（ゼロと1）のカットポイントに対応します。デフォルト値は0と1で、これによりその影響が取り除かれ（分布は通常のベータ分布と同等になります）。

パラメータの詳細については、[`BetaPhi2`](@ref) および [`OrderedBeta`](@ref) を参照してください。

# 詳細

*Beta Phi2* は、従来の *Mu-Phi* 位置-精度パラメータ化の変種です。この修正 - ϕを1/2の係数でスケーリングすること - により、μが中心（すなわち、0.5）にあるとき、ϕが1であるベータ分布がフラットな事前分布（すなわち、$Beta(1, 1)$）を生成します。回帰モデルにおいてϕの事前分布を対数スケールで設定するのに便利であり、$Normal(0, 1)$の事前分布はフラットな分布（ϕ=1）に最も確率を割り当てます。

![](https://github.com/DominiqueMakowski/SubjectiveScalesModels.jl/blob/main/docs/img/animation_Choco1.gif?raw=true)

スケールの中央（0.5）にクラスタリングされた回答の場合、0.5の周りに中心を持つ3つ目の（非スケール）ベータ分布を追加することが可能です。

![](https://github.com/DominiqueMakowski/SubjectiveScalesModels.jl/blob/main/docs/img/animation_Choco2.gif?raw=true)
