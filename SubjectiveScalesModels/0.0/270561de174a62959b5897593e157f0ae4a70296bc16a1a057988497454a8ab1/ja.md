```
beta_bins(n::Int=10)
```

ヒストグラムに便利なビンの境界のベクトルを生成するシンプルな関数です。極端な値（0と1）のために左端と右端にビンを追加し、[`OrderedBeta`](@ref) モデルに適したデータに便利です。

```jldoctest
julia> beta_bins(3)
6-element Vector{Float64}:
 -0.3333333333333333
  2.220446049250313e-16
  0.3333333333333335
  0.6666666666666667
  1.0
  1.3333333333333333

julia> beta_bins([0, 0.5, 0.2, 0.6, 0.8, 1, 0.5, 0.4])
6-element Vector{Float64}:
 -0.3333333333333333
  2.220446049250313e-16
  0.3333333333333335
  0.6666666666666667
  1.0
  1.3333333333333333
```
