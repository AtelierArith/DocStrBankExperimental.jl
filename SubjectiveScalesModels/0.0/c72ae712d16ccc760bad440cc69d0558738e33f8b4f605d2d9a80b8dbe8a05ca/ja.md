```
BetaPhi2(μ, ϕ)
```

平均 `μ` と精度 `ϕ` のパラメータを持つベータ分布を構築します。これは `Beta(μ * 2ϕ, (1 - μ) * 2ϕ)` と定義されます。

# 引数

  * `μ`: ロケーションパラメータ（範囲: $]0, 1[$）。
  * `ϕ`: 精度パラメータ（$> 0$ でなければなりません）。

# 詳細

*Beta Phi2* は従来の *Mu-Phi* ロケーション-精度パラメータ化の変種です。修正 - ϕを1/2の係数でスケーリングすることにより、μが中心（すなわち0.5）のとき、ϕが1であるとフラットな事前分布（すなわち $Beta(1, 1)$）を生成するベータ分布が作成されます。回帰モデルにおいて、ϕの事前を対数スケールで設定することが有用であり、$Normal(0, 1)$ の事前がフラットな分布（ϕ=1）に最も確率を割り当てます。

![](https://github.com/DominiqueMakowski/SubjectiveScalesModels.jl/blob/main/docs/img/animation_BetaPhi2.gif?raw=true)

赤い領域は、分布が極端な値（0および/または1に向かう）に最も高い確率を割り当てる領域を示しています。青い領域は、分布が「凸」であり、$]0, 1[$ の区間内でピークを持つ領域を示しています。

# 例

```jldoctest
julia> BetaPhi2(0.5, 1)
BetaPhi2{Float64}(μ=0.5, ϕ=1.0)
```
