```
relaxation_step!(grid::VoronoiGrid, dt::Float64; rusanov::Bool = true)
```

変数を修復速度 `dv` を考慮して更新します。修復速度は `relaxation_step!` が呼び出される前に知られている必要があります。`find_dv!` を参照してください。各セルの質量、運動量、エネルギーは、輸送問題の1ステップを解くことによって更新されます。キーワードのブール引数 `rusanov` は、Rusanov近似リーマンソルバーの使用を制御します。Rusanovフラックスは、新しい極値（負の密度など）の生成を防ぎ、エントロピーの減少を抑えるために推奨されます。これはメッシュ緩和手順の第二ステップです（または、シミュレーションに多相プロジェクターが含まれている場合は第三ステップかもしれません）。
