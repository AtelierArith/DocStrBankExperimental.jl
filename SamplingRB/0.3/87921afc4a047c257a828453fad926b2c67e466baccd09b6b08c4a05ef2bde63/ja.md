```
sgd_Lagrangian(B::Vector{Float64}, α::Float64, loss_sampler::Function;
               maxiters::Int=10000, ϵ::Float64=1e-12, debug::Int=0)
```

資産に対する投資エクスポージャーを計算し、最適化問題のラグランジアン形式に基づいてCV@R-αリスクバジェットポートフォリオを構築するために、確率的勾配法を使用します。ここでは（恣意的に）乗数を1に設定します： `math    min._{v} CV@R_alpha[L(v)] - sum B_i log(v_i)`

これは、CVaRを期待値で置き換える補助変数tの導入によって可能になり、確率的勾配を簡単に計算できるようになります。

アルゴリズムはmaxiter回の反復後に停止します。

対数のため、vの正しい（かつ凸）実行可能領域に留まるためには、すべてのエントリが厳密に正であることを確認する必要があります。これは、非常に小さな正の数であるべきパラメータϵを使用して行われます。

debug >= 1は、反復回数、エクスポージャー、および現在のV@Rの推定値を出力します。

ペア(v, V@R-α)を返します。
