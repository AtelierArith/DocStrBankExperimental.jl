```
cutting_planes_evar(B::Vector{Float64}, α::Float64, losses::Array{Float64, 2};
                    tol::Float64=1e-6, maxiters::Int=1000, ub_w::Float64=2000., debug::Int=0, normalize::Bool=true)
```

資産に対する投資ウェイトを計算し、カッティングプレーン法を使用してエントロピーV@R-alphaリスクバジェットポートフォリオを構築します。

ポートフォリオは、各資産 j が総リスクに B[j] という形で寄与するように構成されており、これはリスクアペタイトとも呼ばれます。

アルゴリズムは、証明可能な最適性ギャップが (絶対または相対) 許容範囲 tol に達するか、maxiters 回の反復後に収束しなかった場合に停止します。

最初に、アルゴリズムはウェイトのバウンディングボックスが必要です。対数が w > 0 を暗示するため、すべてのウェイトに対して十分に大きな上限 ub_w を使用します。アルゴリズムがそのような大きなウェイトを持つ解に収束した場合は、健全性を確認する必要があります。最も一般的なケースは、問題が下に無制限である場合で、たとえば alpha が十分に大きくない場合です。

debug >= 1 は反復番号、ギャップ、ウェイトの変化を出力します。debug >= 2 は現在の上限と下限、および試行ポイント (ウェイト、t) も出力します。debug >= 3 はリスクバジェッティング制約と第一段階のステータスも出力します。

normalize == true の場合、ウェイトの合計が1になる三つ組 (status, weights, t) を返します。
