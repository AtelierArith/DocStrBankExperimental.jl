```
load(loader::DataLoader{driver}, source::Any, as::Type)
```

特定の `loader` を使用して、`source` から与えられたデータの形式で `as` の情報を取得します。

これは全体のデータフローのこのコンポーネントを満たします：

```
  ╭────loader─────╮
  ╵               ▼
Data          Information
```

ローダーが `nothing` を生成した場合、これは何らかの理由でデータをロードできなかったことを示し、可能であれば別のローダーを試すべきであると見なされます。これはソフトフェイルと考えることができます。他の値は有効な情報と見なされます。
