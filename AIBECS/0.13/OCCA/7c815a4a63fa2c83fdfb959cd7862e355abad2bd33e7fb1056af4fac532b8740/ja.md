この `OCCA` モジュールは、AIBECSで使用するためにOCCA行列とグリッドをロードするために使用されます。

!!! tip
    OCCA行列とグリッドをロードするには、次のようにします。

    ```
    julia> grd, T = OCCA.load()
    ```

    詳細については *Forget* (2010) を参照してください。
