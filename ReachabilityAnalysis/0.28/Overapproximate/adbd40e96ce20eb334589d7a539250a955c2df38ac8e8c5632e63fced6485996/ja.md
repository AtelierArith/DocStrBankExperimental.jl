```
relative_error(x, x_ref)
```

区間 `x` と参照区間 `xref` の相対誤差を計算します。

### 入力

  * `x`    – 区間
  * `xref` – 参照区間

### 出力

`x` と参照区間 `xref` に対する相対誤差（パーセンテージ）の区間を表します。

### アルゴリズム

もし $x = [x_L, x_H]$`および``xref = [xref_L, xref_H]``であれば、出力は区間``y = 100 * [y_L, y_H]``で計算され、``y_L = -(x_L - xref_L) / den``および``y_H = (x_H - xref_H) / den``となります。ここで``den = xref_H - xref_L``です。

この関数は、区間 `x` と参照区間 `x_ref` の相対誤差を、下限と上限の範囲を別々に考慮して測定します（詳細は [AlthoffGK18; Eq. (20)](@citet) を参照）。
