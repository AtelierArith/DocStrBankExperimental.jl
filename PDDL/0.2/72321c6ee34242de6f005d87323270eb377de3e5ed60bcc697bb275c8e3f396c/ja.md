```
compiled(domain, state)
compiled(domain, problem)
```

`domain` と `state` をコンパイルし、結果として得られたコンパイル済みのドメインとコンパイル済みの状態を返します。`problem` は状態の代わりに提供される場合があります。

!!! warning "トップレベルのみ"
    `compiled` は新しい型とメソッドを定義するため、ワールドエイジエラーを避けるためにトップレベルでのみ呼び出すべきです。


!!! warning "プリコンパイルはサポートされていません"
    `compiled` は `PDDL` モジュール内でコードを評価するため、他のモジュールやパッケージで使用するとプリコンパイルエラーが発生します。したがって、`compiled` を呼び出すモジュールはプリコンパイルを無効にする必要があります。
