```
get_datasets(; datasets=["S2_10m.json", "spectral.json"], data_loc=joinpath(dirname(@__FILE__), "..", "data"))
```

指定されたリモートロケーションから事前定義されたデータセットをダウンロードし、ローカルディレクトリに保存します。

# キーワード引数

  * `datasets::Array{String,1}`: ダウンロードするデータセットファイル名のリスト。デフォルトは `["S2_10m.json", "spectral.json"]` です。
  * `data_loc::String`: ダウンロードしたデータセットが保存されるローカルディレクトリのパス。デフォルトはスクリプトのディレクトリから1つ上の `data` ディレクトリです。

# 説明

この関数はデータセットファイル名のリストを反復処理し、各データセットを事前定義されたリモートURLからダウンロードし、指定されたローカルディレクトリに保存します。リモートURLは現在、リスト内の任意のデータセットに対して特に "S2_10m.json" ファイルをダウンロードするようにハードコーディングされています。異なるデータセットに異なるURLが必要な場合は、関数またはその使用法を適宜調整してください。

# 例

```julia
get_datasets()  # デフォルトのデータセットをデフォルトの場所にダウンロードします

get_datasets(; datasets=["custom_dataset.json"], data_loc="path/to/custom/directory")
```

これは、さらなる処理や分析のために必要なデータファイルを持つローカル環境を設定するのに特に便利です。
