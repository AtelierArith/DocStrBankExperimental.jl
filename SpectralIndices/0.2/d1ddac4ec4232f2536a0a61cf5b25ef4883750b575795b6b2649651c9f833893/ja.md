```
compute(si::SpectralIndex, params::Dict=Dict(); kwargs...) -> Any
```

提供された `SpectralIndex` インスタンス、パラメータ、およびオプションのキーワード引数に基づいてスペクトルインデックスを計算します。

# パラメータ

  * `si`: 計算されるスペクトルインデックスの名前と詳細を含む `SpectralIndex` のインスタンス。
  * `params`: （オプション）計算の入力として使用されるパラメータの辞書。提供されない場合、パラメータはキーワード引数を使用して渡すことができます。
  * `kwargs`: 計算の入力として使用される追加のパラメータで、キーワードペアとして提供されます。`params` が空の場合に使用されます。

# 戻り値

  * 計算されたスペクトルインデックス。戻り値の型は、入力パラメータおよび計算される特定のスペクトルインデックスに依存します。

# 例

```julia-repl
julia> compute(NDVI; N=0.643, R=0.175)

```

```julia-repl
julia> compute(NDVI; N=fill(0.643, (5, 5)), R=fill(0.175, (5, 5)))

```
