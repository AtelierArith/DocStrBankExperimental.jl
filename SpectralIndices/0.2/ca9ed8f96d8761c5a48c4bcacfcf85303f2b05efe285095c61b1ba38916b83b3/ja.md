```
poly(a::T, b::T, c::T, p::T) where T <: Number
poly(a::T, b::T, c::T, p::T) where T <: AbstractArray
poly(params::Dict{String, T})
poly(params::DataFrame)
poly(params::YAXArray)
```

多項式カーネル `(a * b + c) ^ p` を計算します。この関数は、数値、配列、辞書、データフレーム、YAXArray など、さまざまな入力タイプをサポートしています。

# 引数

  * `a`: 多項式カーネルの最初のパラメータ。数値または配列である可能性があります。
  * `b`: 多項式カーネルの第二のパラメータ。数値または配列である可能性があります。
  * `c`: `a` と `b` の積に加えられる係数。数値または配列である可能性があります。
  * `p`: 積と係数の合計が累乗される指数。数値または配列である可能性があります。
  * `params`: パラメータ "a"、"b"、"c"、および "p" を含む辞書、データフレーム、または YAXArray。

# 戻り値

  * `(a * b + c) ^ p` の結果。出力タイプは入力タイプに依存します：

      * `a`、`b`、`c`、および `p` が数値の場合、結果は数値です。
      * `a`、`b`、`c`、および `p` が配列の場合、結果は要素ごとに操作が適用された配列です。
      * `params` が使用される場合、結果は `params` の形式（辞書、DataFrame、または YAXArray）に一致します。

# 例

```julia
# 数値を使用
result = poly(2, 3, 1, 2)

# 配列を使用
result = poly([1, 2, 3], [4, 5, 6], [1, 1, 1], [2, 2, 2])

# 辞書を使用
result = poly(Dict("a" => 2, "b" => 3, "c" => 1, "p" => 2))

# データフレームを使用
df = DataFrame(; a=[1, 2, 3], b=[4, 5, 6], c=[1, 1, 1], p=[2, 2, 2])
result = poly(df)
```
