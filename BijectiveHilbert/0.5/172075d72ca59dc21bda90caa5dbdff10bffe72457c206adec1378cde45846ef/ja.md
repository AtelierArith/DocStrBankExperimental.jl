```
GlobalGray(b, n)
GlobalGray(T, b, n)
```

`T` はヒルベルトインデックスのデータ型です。符号付きまたは符号なしで、少なくとも `n * b` ビットを持っていれば構いません。`n` は次元の数で、`b` は次元ごとのビット数です。したがって、各軸の値はゼロベースのインターフェースの場合、0 から $2^b - 1$ の範囲である必要があります。1 ベースのインターフェースの場合は、1 から $2^b$ の範囲である必要があります。

GlobalGray アルゴリズムは、簡略化された実装の n 次元ヒルベルト曲線です。これは、John Skilling による「Programming the Hilbert Curve」という記事に従っています。707 (2004), http://dx.doi.org/10.1063/1.1751381。私はこれを「Global Gray」と呼んでいます。なぜなら、この記事の洞察は、単一のグローバルグレイコードをすべての np ビットのヒルベルト長に適用できるということだからです。
