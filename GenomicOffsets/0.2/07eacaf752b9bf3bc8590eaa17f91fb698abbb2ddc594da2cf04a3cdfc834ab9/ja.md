```
fit(::Type{GeometricGO}, Y::AbstractMatrix{T1}, X::AbstractMatrix{T2}) where {T1<:Real,T2<:Real}
```

ジオメトリックゲノムオフセットモデル（すなわち、LFMMをフィットする）をデータ `Y` と `X` を使用してフィットします。

# 引数

  * `Y::AbstractMatrix{T1}`: 個体の遺伝子型またはアレル頻度を持つ NxL 行列。
  * `X::AbstractMatrix{T}`: 環境データを持つ NxP 行列。
  * `K`: LFMMモデルをフィットする際に使用する潜在因子の数。エルボールールを使用して決定できます。何も提供されない場合、潜在因子の数を決定するためのトレーシー・ウィドム統計が使用されます（これは過剰に保守的かもしれません）。
  * オプションの引数:

      * `center::Bool=true`: 遺伝子型と環境データの両方をセンタリングします。予測された環境データは、Xと同じ平均を使用してセンタリングされます。falseの場合、データはセンタリングされていると見なされます。
      * `scale::Bool=true`: 環境データをスケーリングします。予測された環境データは、Xと同じ標準偏差を使用してスケーリングされます。falseの場合、データはスケーリングされていると見なされます。
      * tw_threshold::Real=1e-3: トレーシー・ウィドム統計を使用する場合に潜在因子の数を決定するための閾値。デフォルトでは、1e-3が使用されます。
      * λ::Real=1e-5: LFMMの正則化パラメータ。

# 戻り値

  * GeometricGOモデル。
