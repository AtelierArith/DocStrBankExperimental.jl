```
SimpleDFSane(;
    σ_min::Real = 1e-10, σ_max::Real = 1e10, σ_1::Real = 1.0,
    M::Union{Int, Val} = Val(10), γ::Real = 1e-4, τ_min::Real = 0.1, τ_max::Real = 0.5,
    nexp::Int = 2, η_strategy::Function = (f_1, k, x, F) -> f_1 ./ k^2
)
```

大規模非線形方程式系を解くための低オーバーヘッドのdf-saneメソッドの実装。すべてのパラメータとアルゴリズムに関する詳細情報については、[la2006spectral](@citet)を参照してください。

### キーワード引数

  * `σ_min`: アルゴリズムのステップサイズに関連するスペクトル係数 `σ_k` の最小値。デフォルトは `1e-10`。
  * `σ_max`: アルゴリズムのステップサイズに関連するスペクトル係数 `σ_k` の最大値。デフォルトは `1e10`。
  * `σ_1`: アルゴリズムのステップサイズに関連するスペクトル係数 `σ_k` の初期値。デフォルトは `1.0`。
  * `M`: アルゴリズムの単調性はこの正の整数によって決まります。`M` の値が1の場合、関数 `f` のL2ノルムの減少において厳密な単調性が得られます。しかし、より高い値はこの減少においてより柔軟性を許可します。それにもかかわらず、アルゴリズムはGrippo-Lampariello-Lucidi条件に従った非単調線形探索アルゴリズムを使用することで、グローバル収束を保証します。通常、5から20の範囲の値が十分ですが、場合によってはより高い `M` の値が必要になることがあります。デフォルト設定は10です。
  * `γ`: 提案されたステップが受け入れられるかどうかに影響を与えるパラメータ。`γ` の値が高いほど、アルゴリズムはステップの受け入れに対してより制限的になります。デフォルトは `1e-4`。
  * `τ_min`: ステップが拒否された場合、新しいステップサイズは因子で乗算され、このパラメータはその因子の最小値です。デフォルトは `0.1`。
  * `τ_max`: ステップが拒否された場合、新しいステップサイズは因子で乗算され、このパラメータはその因子の最大値です。デフォルトは `0.5`。
  * `nexp`: 損失の指数、すなわち $f_k=||F(x_k)||^{nexp}$。論文では `nexp ∈ {1,2}` を使用しています。デフォルトは `2`。
  * `η_strategy`: パラメータ `η_k` を決定する関数で、$||F||^2$ の成長を可能にします。`η_k = η_strategy(f_1, k, x, F)` として呼び出され、`f_1` は $f_1=||F(x_1)||^{nexp}$ で初期化され、`k` は反復回数、`x` は現在の `x` 値、`F` は現在の残差です。$η_k > 0$ および $∑ₖ ηₖ < ∞$ を満たす必要があります。デフォルトは $||F||^2 / k^2$ です。
