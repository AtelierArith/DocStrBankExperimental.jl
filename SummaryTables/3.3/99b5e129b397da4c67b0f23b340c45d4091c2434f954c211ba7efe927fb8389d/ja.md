```
Annotated(value, annotation; label = AutoNumbering())
```

`Table`で使用されるフットノート注釈が与えられる`Annotated`オブジェクトを作成します。`label`キーワードが`AutoNumbering()`の場合、注釈には出現順に1からNまでの番号ラベルが付けられます。`nothing`の場合、ラベルは表示されません。他の任意の`label`はフットノートラベルとして直接使用されます。

各ユニークなラベルはユニークな注釈とペアでなければなりませんが、同じ組み合わせが単一のテーブル内に複数回存在することは可能です。
