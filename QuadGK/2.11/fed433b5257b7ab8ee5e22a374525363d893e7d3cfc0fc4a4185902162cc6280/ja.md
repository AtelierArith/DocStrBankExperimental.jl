```
gauss(J::AbstractMatrix, unitintegral::Real=1, [ (a₀,b₀) => (a,b) ])
```

点ガウス求積法を$I[f] = \int_a^b w(x) f(x) dx$のために構築します。ここで、$n \times n$対称三重対角ヤコビ行列`J`は、その重み付き積分に対する直交多項式に対応します。`unitintegral`の値は$I[1]$、すなわち重み関数の積分であるべきです。

オプションの引数`(a₀,b₀) => (a,b)`を使用すると、`J`が元々異なる区間$(a_0, b_0)$のために定義されていたことを指定でき、これを与えられた$(a, b)$に再スケールしたい場合に使用します。(`gauss`はポイントと重みを自動的に再スケールします。)

関数を積分するための$n$個の求積点$x[i]$と重み$w[i]$のペア`(x, w)`を返します。すなわち、`sum(w .* f.(x))`は積分$I[f]$を近似します。
