```
kronrod([T,] n)
kronrod([T,] n, a, b)
```

$$
2n+1
$$

の Kronrod 点 `x[i]` と重み `w[i]` を、Laurie (1997) の付録 A に基づいて計算します。これは区間 $(a,b)$ での積分に使用され、デフォルトでは $[-1,1]$ になります。

`a` と `b` が渡されない場合、ルールは対称であるため、`x <= 0` の `n+1` 点のみが返されます。この関数は、埋め込まれた `n` 点のガウス数値積分の重み `wg` も計算します（再び `a` と `b` が渡されない場合は `x <= 0` に対して）、これは点 `x[2:2:end]` に対応します。O(`n`²) の操作で `(x,w,wg)` を返します。

`T` はオプションのパラメータで、浮動小数点型を指定し、デフォルトは `Float64` です。任意の精度（`BigFloat`）もサポートされています。

これらの点と重みを用いて、被積分関数 `f(x)` に対する推定積分 `I` と誤差 `E` を次のように計算できます：

```julia
x, w, wg = kronrod(n)
fx⁰ = f(x[end])                # f(0)
x⁻ = x[1:end-1]                # x < 0 の Kronrod 点
fx = f.(x⁻) .+ f.((-).(x⁻))    # f(x < 0) + f(x > 0)
I = sum(fx .* w[1:end-1]) + fx⁰ * w[end]
if isodd(n)
    E = abs(sum(fx[2:2:end] .* wg[1:end-1]) + fx⁰*wg[end] - I)
else
    E = abs(sum(fx[2:2:end] .* wg[1:end])- I)
end
```
