```
quadgk_print([io], f, args...; kws...)
```

[`quadgk`](@ref) と同じですが、**各被積分関数の評価をストリーム `io`（デフォルトは `stdout`）に次の形式で出力します**:

```
f(x1) = y1
f(x2) = y2
...
```

これは教育やデバッグに役立ちます。

また、[`quadgk_count`](@ref) のように、推定された積分 `I`、推定された誤差境界 `E`、および被積分関数 `f` が評価された回数の `count` からなる三つ組 `(I, E, count)` を返します。
