```
lwc_crosshair_highlight_bar(; kw...) -> LWCPlugin
```

Adds additional highlighting to the cursor crosshairs.

## Keyword arguments

| Name::Type      | Default (Possible values) | Description                |
|:--------------- |:------------------------- |:-------------------------- |
| `color::String` | `"rgba(0, 0, 0, 0.2)"`    | Crosshair highlight color. |
