```
lwc_vert_line(; kw...) -> LWCPlugin
```

カーソルに従って値（およびマウスボタンを押し続けている間の変化）を表示する動的ツールチップをチャートに追加します。

## キーワード引数

| 名前::型                | デフォルト (可能な値)           | 説明              |
|:-------------------- |:---------------------- |:--------------- |
| `line_color::String` | `"rgba(0, 0, 0, 0.2)"` | 線の色。            |
| `show_time::Bool`    | `false`                | 詳細な時間表示。        |
| `top_offset::Int64`  | `20`                   | ツールチップの垂直オフセット。 |
