```
lwc_layout(panels::LWCPanel...; kw...) -> LWCLayout
```

複数の `panels` を共通のレイアウトに結合します。

## キーワード引数

| 名前::型                 | デフォルト (可能な) 値                  | 説明                          |
|:--------------------- |:------------------------------ |:--------------------------- |
| `name::String`        | `"LightweightCharts ❤️ Julia"` | レイアウト名（ブラウザタブのタイトルに表示されます）。 |
| `sync::Bool`          | `true`                         | チャートのスクロールの同期。              |
| `min_height::Integer` | `300`                          | レイアウトの最小高さ（px単位）。           |
| `resizable::Bool`     | true                           | パネルのリサイズモードを有効にします。         |
