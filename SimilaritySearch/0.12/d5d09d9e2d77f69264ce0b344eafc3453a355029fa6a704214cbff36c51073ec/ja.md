```
getminbatch(minbatch, n)
```

`Polyester.jl` に基づく並列処理を使用する関数によって使用され、ミニバッチごとにスレッドが使用されるときに、スレッドごとに解決されるクエリ（または他の何か）の数を指定します。

# 引数

  * `minbatch`

      * 整数 $1 ≤ minbatch ≤ n$ は有効な値です（ここで n は処理するオブジェクトの数、すなわちクエリです）
      * デフォルトは 0 で、利用可能なコアの数と `n` に基づいてデフォルトの数を計算します。
      * 並列処理を回避するには `minbatch=-1` を設定します。
