```
fubini(f, [zs], [ys], xs; rtol=missing, kws...)
```

1, 2, または 3 入力変数の `f` を積分します。

`zs` は (x,y) に依存する場合があり、`ys` は x に依存する場合があります。

## 例

```
# 単位正方形の上で積分
fubini((x,y) -> sin(x-y), (0,1), (0,1))

# 三角形の上で積分
fubini((x,y) -> 1, (0,identity), (0,1 ))

#
f(x,y,z) = x*y^2*z^3
fubini(f, (0,(x,y) ->  x+ y), (0, x -> x), (0,1))
```

!!! 注意     これは `quadgk` へのネストされた呼び出しを使用します。矩形領域を作るために変数変換を行った後、通常は `hcubature` の使用が推奨されます。相対許容誤差は各ネストされたレベルで増加します。
