```
bulk_register!(s, who, what, Δ, randomize)
```

時刻 `s.now+Δ` にイベントを `s.event_queue` に追加し、`who` のすべての `w` に対して `what(scheduler, w)` を実行します。`randomize` が `false` の場合、`who` は自然な順序で走査され、それ以外の場合はランダムな順序で走査されます。`what` は `Scheduler` と `eltype(who)` の2つの引数を正確に受け入れる必要があります。この関数は `Δ` が有効（有限）な数であるかどうかをチェックしません。挿入されたバルク `Action` を返します。

この関数は、同じアクションを多くのエージェントによって同じシミュレーション時間に実行する必要がある場合に効率的に処理するように設計されています。
