```
Sp,Sm,Sz,Sy,Sx,O,Id = spinOps(s)
```

Generates operators for a heisenberg model (spin-`s`)

#Outputs:

  * `Sx`: spin-x operator
  * `Sy`: spin-y operator
  * `Sz`: spin-z operator
  * `Sp`: raising operator
  * `Sm`: lowering operator
  * `H`: Hadamard gate
  * `O`: zero matrix
  * `Id`: identity matrix
