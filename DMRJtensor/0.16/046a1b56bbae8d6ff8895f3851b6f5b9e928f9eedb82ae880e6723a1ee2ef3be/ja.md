```
transfermatrix(dualpsi,psi,i,j[,transfermat=])
```

物理インデックスに沿ってサイト `i` と `j`（両方を含む）間の転送行列（MPS `psi` とその双対 `dualpsi` を収束させたもの）を形成します。指定されていない場合、`transfermat` フィールドは `i-1` サイトからの転送行列で初期化されます。

転送行列の形式は以下のようになります（双対波動関数テンソルが上、共役）：

1 –––––– 3         |         |         |         | 2 –––––– 4

この関数のインプレースバージョンはありません。
