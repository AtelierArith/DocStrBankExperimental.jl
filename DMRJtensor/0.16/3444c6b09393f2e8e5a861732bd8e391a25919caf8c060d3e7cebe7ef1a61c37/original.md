```
Sp,Sm,Sz,Sy,Sx,O,Id = spinOps([,s=0.5])
```

Generates operators for a heisenberg model (spin-`s`, default 1/2)

#Outputs:

  * `Sx`: spin-x operator
  * `Sy`: spin-y operator
  * `Sz`: spin-z operator
  * `Sp`: raising operator
  * `Sm`: lowering operator
  * `H`: Hadamard gate
  * `O`: zero matrix
  * `Id`: identity matrix
