```
chbv(A, vec)
```

行列の指数関数をいくつかのベクトルに作用させるために、チェビシェフ法を使用して計算します。

# 入力

  * `A`   – 密または疎の行列
  * `vec` – 行列 `A` の指数関数が適用されるベクトル

# 注意事項

このJulia実装は、Roger B. SidjeによるExpokitのCHBV Matlabコードに基づいています。以下を参照してください。

---

```
y = chbv( H, x )
CHBVは行列の指数関数の直接作用を計算します
ベクトルに対して: y = exp(H) * x。これは、タイプ(14,14)の一様有理チェビシェフ近似の部分分数展開を使用します。
行列Hが対称負定である場合、約14桁の精度が期待されます。それ以外の場合、アルゴリズムの動作が悪くなる可能性があります。
PADM、EXPOKITも参照してください。

Roger B. Sidje (rbs@maths.uq.edu.au)
EXPOKIT: 行列指数関数を計算するためのソフトウェアパッケージ。
ACM - Transactions On Mathematical Software, 24(1):130-156, 1998
```
