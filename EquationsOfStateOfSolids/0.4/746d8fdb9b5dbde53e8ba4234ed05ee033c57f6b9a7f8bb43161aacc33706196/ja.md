```
BirchMurnaghan4th(v0, b0, b′0, b″0, e0=zero(v0 * b0))
```

バーチ–マーナハン4次状態方程式を作成します。

この状態方程式は単位を持つことができます。単位は[`Unitful.jl`](https://github.com/PainterQubits/Unitful.jl)の`@u_str`スタイルで指定されます。

# 引数

  * `v0`: ゼロ圧力での固体の体積。
  * `b0`: ゼロ圧力での固体の体積弾性率。
  * `b′0`: ゼロ圧力での固体の一次圧力微分体積弾性率。
  * `b″0`: ゼロ圧力での固体の二次圧力微分体積弾性率。
  * `e0`: ゼロ圧力での固体のエネルギー。

!!! 注     4次方程式は、次の条件を満たすと3次方程式と同一になります。

````
```math
b″0 = -\frac{ 1 }{ 9b0 } (9b′0^2 - 63b′0 + 143).
```
````
