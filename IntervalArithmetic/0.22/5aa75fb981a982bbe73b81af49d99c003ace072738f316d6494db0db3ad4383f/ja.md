```
IntervalArithmetic
```

区間算術を使用した検証された数値のためのライブラリで、値を区間として表現することにより数値計算の誤差を厳密に制約し、計算結果が真の値を含むことを保証するツールを提供します。科学計算、コンピュータ支援証明、および認証された数値結果を必要とする任意の分野に理想的な、正確で効率的な方法を提供します。

主な機能：

  * **境界タイプ**: 区間の境界を表すために使用されるデフォルトの数値タイプ。デフォルトは `Float64` ですが、精度を調整するために他の [`IntervalArithmetic.NumTypes`](@ref) のサブタイプを使用できます。
  * **フレーバー**: IEEE標準1788-2015に準拠した区間表現。デフォルトでは、無限大を区間の一部としない集合ベースのフレーバーを使用します。詳細はこちら: [`IntervalArithmetic.Flavor`](@ref)。
  * **区間丸め**: 区間算術演算の丸め動作を制御します。デフォルトでは、ライブラリは正しい丸めを使用して、境界ができるだけ厳密に計算されることを保証します。詳細はこちら: [`IntervalArithmetic.IntervalRounding`](@ref)。
  * **パワーモード**: べき演算のためのパフォーマンス設定。デフォルトモードは、迅速な計算を優先する効率的なアルゴリズムを使用しますが、必要に応じてより正確で遅い計算に調整できます。詳細はこちら: [`IntervalArithmetic.PowerMode`](@ref)。
  * **行列乗算モード**: 行列乗算演算のためのパフォーマンス設定。デフォルトモードは、迅速な計算を優先する効率的なアルゴリズムを使用しますが、行列乗算の標準定義を使用するように変更できます。詳細はこちら: [`IntervalArithmetic.MatMulMode`](@ref)。

上記のデフォルトの動作は、[`IntervalArithmetic.configure`](@ref)を介して構成できます。

**表示設定**: 区間の表示方法を制御します。デフォルトでは、区間は標準的な数学的表記 $[a, b]$ を使用して表示され、装飾と最大6桁の有効数字が含まれます。詳細はこちら: [`setdisplay`](@ref)。

# 使用法

```julia
using IntervalArithmetic

# 区間を作成
x = interval(1.0, 2.0)

# 厳密な計算を実行
x + exp(interval(π))
```

詳細はこちら: https://github.com/JuliaIntervals/IntervalArithmetic.jl
