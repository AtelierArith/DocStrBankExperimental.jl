```
Piecewise(pairs... ; continuity = fill(-1, length(pairs) - 1))
```

区間ごとに定義された関数（各区間が関数に関連付けられています）。区間と標準数の両方をサポートします。

```jl
julia> myabs = Piecewise(
          Domain{:open, :closed}(-Inf, 0) => x -> -x,
          Domain{:open, :open}(0, Inf) => identity
       )
Piecewise関数は2つの区間を持っています：
  (-Inf, 0] -> var"#119#120"()
  (0, Inf) -> identity

julia> myabs(-22.3)
22.3

julia> myabs(interval(-5, 5))
Interval{Float64}(0.0, 5.0, def)
```

定数の区間については、`Constant`を使用することをお勧めします。これにより、区間との完全な互換性が得られます。

ドメインは増加順に指定する必要があり、重複してはいけません。

`continuity`のオプション引数は、区間の数を`N`とした場合に`N - 1`の整数のベクトルを受け取り、部分ドメイン間の端点での区分関数の挙動を決定します。可能性は次の通りです：

  * `-1` : 関数はドメイン間で不連続です。
  * `0` : 関数は連続ですが、ドメイン間で微分不可能です。
  * `n > 0` : 関数はドメイン間で`n`回連続的に微分可能です。これは、関数の導関数を計算するために`ForwardDiff`を使用する場合にのみ重要です。

この情報は、複数のドメインの端点をカバーする区間の装飾を決定するために使用されます。

入力区間が区分関数の定義域の外に出ると、出力は常に自明な（`trv`）装飾を持ちます。標準数の場合は、`DomainError`がスローされます。

区分関数は、2つの区間の間にギャップを持つことがあります。この場合、`continuity`のオプション引数は無視され、ギャップを跨ぐ区間は常に`trv`の装飾を持ちます。
