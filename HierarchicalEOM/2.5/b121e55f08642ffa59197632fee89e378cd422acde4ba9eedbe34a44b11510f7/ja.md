```
struct Exponent <: BathTerm
```

バス相関関数の分解内の単一指数拡張項（ナイーブには、励起モード）を記述するオブジェクトです。

バス相関関数の拡張は次のように表現できます： $C(t) = \sum_i \eta_i \exp(-\gamma_i t)$。

# フィールド

  * `op::QuantumObject` : システム-バス相互作用に従ったシステム結合演算子。
  * `η::Number` : バス相関関数における係数 $\eta_i$。
  * `γ::Number` : バス相関関数における係数 $\gamma_i$。
  * `types::String` : 指数のタイプタグ。

Exponentの異なるタイプ：

  * `"bR"` : ボソニック相関関数の実部から $C^{u=\textrm{R}}(t)$
  * `"bI"` : ボソニック相関関数の虚部から $C^{u=\textrm{I}}(t)$
  * `"bRI"` : 結合された（実部と虚部の）ボソニックバス相関関数から $C(t)$
  * `"bA"` : 吸収ボソニック相関関数から $C^{\nu=+}(t)$
  * `"bE"` : 放出ボソニック相関関数から $C^{\nu=-}(t)$
  * `"fA"` : 吸収フェルミオン相関関数から $C^{\nu=+}(t)$
  * `"fE"` : 放出フェルミオン相関関数から $C^{\nu=-}(t)$
