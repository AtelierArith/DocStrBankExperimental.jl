```
struct FermionBath <: AbstractBath
```

システムとフェルミオンバスとの相互作用を記述するオブジェクト

# フィールド

  * `bath` : 相互作用を記述する異なるフェルミオンバスタイプのオブジェクト
  * `op` : システム-フェルミオンバス相互作用に従ったシステムの「放出」演算子。
  * `Nterm` : 異なるバス項の総数。
  * `δ` : HEOM行列に終端子を追加するために使用される近似の不一致 (関数: addTerminator を参照)

# メソッド

`bath::FermionBath` から $k$-番目の項を取得するには、次のように呼び出します: `bath[k]`。`HierarchicalEOM.jl` は次の呼び出し (メソッド) もサポートしています :

```julia
bath[1:k];   # `1`-番目から `k`-番目の項を含むベクトルを返します。
bath[1:end]; # すべての項を含むベクトルを返します
bath[:];     # すべての項を含むベクトルを返します
from b in bath
    # 何かをする
end
```
