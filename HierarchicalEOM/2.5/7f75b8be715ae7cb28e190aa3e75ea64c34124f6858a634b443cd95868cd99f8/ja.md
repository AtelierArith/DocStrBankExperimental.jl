```
struct BosonBath <: AbstractBath
```

システムとボソンバスとの相互作用を記述するオブジェクト

# フィールド

  * `bath` : システムとボソンバスとの相互作用を記述する異なるボソンバス型オブジェクト
  * `op` : システム結合演算子で、エルミートでなければならず、フェルミオン系の場合は電荷保存と互換性を持つために偶数パリティである必要があります。
  * `Nterm` : 異なるバス項の総数。
  * `δ` : HEOM行列に終端子を追加するために使用される近似誤差（関数: addTerminatorを参照）

# メソッド

`bath::BosonBath`から$k$-番目の項を取得するには、`bath[k]`を呼び出します。`HierarchicalEOM.jl`は以下の呼び出し（メソッド）もサポートしています：

```julia
bath[1:k];   # `1`-番目から`k`-番目の項を含むベクトルを返します。
bath[1:end]; # すべての項を含むベクトルを返します
bath[:];     # すべての項を含むベクトルを返します
from b in bath
    # 何かをする
end
```
