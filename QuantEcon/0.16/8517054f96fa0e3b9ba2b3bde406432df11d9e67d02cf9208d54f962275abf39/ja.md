```
SimplexGrid
```

`simplex_grid`のイテレーターバージョン、すなわち、(m-1)次元単体 $\{x \mid x_1 + \cdots + x_m = n, x_i \geq 0\}$ の整数点を反復するイテレーター、または同等に、nのm部分構成を辞書式順序で反復します。

# フィールド

  * `m::Int` : 各点の次元。正の整数でなければなりません。
  * `n::Int` : 各点の座標の合計となる数。非負整数でなければなりません。

# 例

```julia
julia> sg = SimplexGrid(3, 4);

julia> for x in sg
           @show x
       end
x = [0, 0, 4]
x = [0, 1, 3]
x = [0, 2, 2]
x = [0, 3, 1]
x = [0, 4, 0]
x = [1, 0, 3]
x = [1, 1, 2]
x = [1, 2, 1]
x = [1, 3, 0]
x = [2, 0, 2]
x = [2, 1, 1]
x = [2, 2, 0]
x = [3, 0, 1]
x = [3, 1, 0]
x = [4, 0, 0]
```
