```
@dms_str
```

`"deg:arcmin:arcsec"` 形式の文字列を直接角度に解析します。デフォルトではラジアンとして解析されますが、文字列の末尾にフラグを追加することで角度を選択できます。

  * `dms"..."rad` -> ラジアン（デフォルト）
  * `dms"..."deg` -> 度
  * `dms"..."ha` -> 時間角

# 例

```jldoctest
julia> dms"12:17:25.3"
0.21450726764795752

julia> dms"12:17:25.3"rad # デフォルト
0.21450726764795752

julia> dms"12:17:25.3"deg
12.29036111111111

julia> dms"12:17:25.3"ha
0.8193574074074074
```
