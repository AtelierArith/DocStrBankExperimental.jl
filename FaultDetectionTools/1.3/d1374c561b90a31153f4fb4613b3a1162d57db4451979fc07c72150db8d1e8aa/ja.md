```
fdimodset(sys; controls, c, disturbances, d, faults, f, fa, faults_sen, fs, noise, n, aux) -> sysf::FDIModel
```

与えられた線形時不変システムモデル `sys = (A-λE,B,C,D)` に対して、FDIフィルタの分析および合成関数と併用するための合成モデルオブジェクト `sysf::FDIModel` を構築します。 `sys` がシステムモデルのベクトルである場合、`sysf` は合成モデルオブジェクトのベクトルとして結果が得られます。

制御、外乱、故障、ノイズ、および補助入力の入力コンポーネントの分割に関する情報は、次のキーワード引数を使用して指定できます。

`controls = inpu` または `c = inpu` は制御入力のインデックス `inpu` を指定します（デフォルト：無効）

`disturbances = inpd` または `d = inpd` は外乱入力のインデックス `inpd` を指定します（デフォルト：無効）

`faults = inpf` または `f  = inpf` は故障入力のインデックス `inpf` を指定します（デフォルト：無効）

`fa = inpfa` はアクチュエータ故障の影響を受ける制御入力のインデックス `inpfa` を指定します（デフォルト：無効）

`faults_sen = inpfs` または `fs = inpfs` はセンサ故障入力の影響を受けるシステム出力のインデックス `inpfs` を指定します（デフォルト：無効）

`noise = inpn` または  `noise = inpn` はノイズ入力のインデックス `inpn` を指定します（デフォルト：無効）

`aux = inpa` は補助入力のインデックス `inpa` を指定します（デフォルト：無効）

入力または出力のインデックスは、整数ベクトル、整数スカラー、または整数 `UnitRange` として指定できます。

結果として得られる `sysf` には、分割されたシステム `sysf.sys = (A-λE,[Bu Bd Bf Bw Bv],C,[Du Dd Df Dw Dv])` が含まれ、ここで `Bu`、`Bd`、`Bf`、`Bw` および `Bv` はそれぞれ制御入力 `u`、外乱入力 `d`、故障入力 `f`、ノイズ入力 `w` および補助入力 `v` からの入力行列であり、`Du`、`Dd`、`Df`、`Dw` および `Dv` はそれらの入力からのフィードスルーメトリックスです。制御、外乱、故障、ノイズおよび補助入力ベクトルの次元は、それぞれ `sysm.mu`、`sysm.md`、`sysm.mf`、`sysm.mw` および `sysm.ma` に含まれています。

*メソッド:* `G(λ)` が `sys` の `p x m` 転送関数行列である場合、結果として得られるシステム `sysf` は、次の関係がコンポーネント行列を定義する同等の入力出力形式 `[Gu(λ) Gd(λ) Gf(λ) Gw(λ) Gv(λ)]` を持ちます：`Gu(λ) = G(λ)*Su`、`Gd(λ) = G(λ)*Sd`、`Gf(λ) = [G(λ)*Sf Ss]`、`Gw(λ) = G(λ)*Sw`、`Gv(λ) = G(λ)*Sv`、ここで選択行列 `Su`、`Sd`、`Sf`、`Sw` および `Sv` は `m` 次の単位行列の列から形成され、`Ss` は `p` 次の単位行列の列から形成されます。
