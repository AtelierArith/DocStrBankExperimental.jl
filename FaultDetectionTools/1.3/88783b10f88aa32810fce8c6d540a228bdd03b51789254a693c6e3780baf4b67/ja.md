```
 fdiscond(sysr::FDFilterIF, SFDI, freq) -> (scond, β, γ)
```

検出および隔離感度条件ベクトル `scond`（および関連する量 `β` と `γ`）を計算します。これは、故障入力から故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` の残差への転送チャネルの安定転送関数行列 `Rf(λ)` に関連する `q × mf` バイナリ構造行列 `SFDI` に関連しています。フィルタ `sysr` は、`N` 個の個別の FDI フィルタ `sysr.sys[i]` で構成されており、`i = 1, ..., N` の範囲で、`sysr.sys[i][:,sysr.faults]` は `i` 番目のフィルタの故障から残差へのチャネルであり、`Rfi(λ)` は対応する転送関数行列です。グローバル転送関数行列 `Rf(λ)` は、`N` 個の個別フィルタの転送関数行列の行の連結によって形成されます。すなわち、`Rf(λ) := [ Rf1(λ); Rf2(λ); ...; RfN(λ)]` です。ベクトル `scond`、`β` および `γ` の i 番目の要素には次の量が含まれます： `β[i]` - `Rffi(λ)` の非ゼロ列の H∞-インデックス、`γ` - `Rfi(λ)` の非ゼロ列のノルムの最大値、対応する故障検出感度条件 `scond[i]` は `scond[i] := β[i]/γ[i]` として評価されます。ここで、`Rffi(λ)` は `Rf(λ)` の i 番目の行の `j` 番目の列で、`S[i,j] = 1` であるものから形成され、`Rfi(λ)` は `Rf(λ)` の i 番目の行です。`SFDI[i,j] = 1` である各 `j` に対して、`Rfi(λ)` の `j` 番目の列は非ゼロであり、`SFDI[i,j] = 0` である各 `j` に対して、`Rfi(λ)` の `j` 番目の列はゼロであると仮定します。`freq` が実数の周波数値のベクトルである場合、`β[i]` と `γ[i]` は `freq` に含まれる周波数に対して評価されます。 ```
