 γ = fdimmperf(sysr::FDIFilterIF[, nrmflag])

安定した故障検出および隔離フィルタ内部形式オブジェクト `sysr::FDIFilterIF` のモデルマッチング性能 `γ` を計算します。フィルタ `sysr` は `N` 個の個別の FDI フィルタ `sysr.sys[i]` で構成されており、`i = 1, ..., N` の範囲です。ここで、`sysr.sys[i][:,sysr.noise]` は `i` 番目のフィルタのノイズから残差へのチャネルで、対応する伝達関数行列は `Rwi(λ)` です。したがって、`γ` は `N` 次元のベクトルであり、`i` 番目の成分は `nrmflag = Inf`（デフォルト）の場合は `Rwi(λ)` の H∞-ノルムであり、`nrmflag = 2` の場合は `Rwi(λ)` の H2-ノルムです。`γ` の `i` 番目の成分は、フィルタが不安定な場合、または `nrmflag = 2` で連続時間システムの伝達関数行列 `Rwi(λ)` が厳密に適切でない場合は無限大になります。 ```
