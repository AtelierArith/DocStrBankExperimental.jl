```
 γ = fdimmperf(sysr::FDFilterIF, SFDI[, nrmflag])
```

故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` のモデルマッチング性能 `γ` を、与えられたバイナリ構造行列 `SFDI` に対して計算します。もし `Rf(λ)` が故障入力から残差 `sysr.sys[:,sysr.faults]` への伝達チャネルの伝達関数行列であり、`Rw(λ)` がノイズ入力から残差 `sysr.sys[:,sysr.noise]` への伝達チャネルの伝達関数行列であるならば、`γ` は `[Rdf(λ) Rw(λ)]` の H∞-ノルムであり、`nrmflag = Inf`（デフォルト）で、`γ` は `[Rdf(λ) Rw(λ)]` の H2-ノルムであり、`nrmflag = 2` の場合です。ここで、`Rdf(λ) = .!SFDI .* Rf(λ)`（すなわち、`.!SFDI` と `Rf(λ)` の要素ごとの積）です。`γ` の値は、不安定なフィルタの場合や、`nrmflag = 2` で連続時間システムの伝達関数行列 `[Rdf(λ) Rw(λ)]` が厳密に適切でない場合には無限大になります。 ```
