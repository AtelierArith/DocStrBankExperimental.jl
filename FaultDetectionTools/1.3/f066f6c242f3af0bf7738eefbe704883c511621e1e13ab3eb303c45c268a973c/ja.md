```
 fdiscond(sysr::FDFilterIF,freq) -> (scond, β, γ)
```

故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` から残差への伝達チャネルの安定伝達関数行列 `Rf(λ)` のために、次の量を計算します: `β` - `Rf(λ)` の H∞-インデックス、`γ` - `Rf(λ)` の列ノルムの最大値、そして故障検出感度条件 `scond` は `scond := β/γ` として評価されます。もし `freq` が実周波数値のベクトルであれば、`β` と `γ` は `freq` に含まれる周波数に対して評価されます。
