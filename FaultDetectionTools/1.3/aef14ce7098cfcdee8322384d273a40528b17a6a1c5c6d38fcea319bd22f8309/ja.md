```
MDModel(sys; mu, md, mw, ma) -> sysc::MDModel
```

線形不変記述子システム `sys = (A-λE,B,C,D)` のために、モデル検出フィルタの分析および合成関数と併用するためのコンポーネント合成モデルオブジェクト `sysm::MDModel` を構築します。

結果として得られる合成モデルオブジェクト `sysc` には、入力が分割されたコンポーネントモデルが含まれ、 `sysc.sys = (A-λE,[Bu Bd Bw Bv],C,[Du Dd Dw Dv])` となります。ここで、 `Bu`、 `Bd`、 `Bw`、および `Bv` は `B` の連続する列から形成され、制御入力 `u`、外乱入力 `d`、ノイズ入力 `w`、および補助入力 `v` からの入力行列です。また、 `Du`、 `Dd`、 `Dw`、および `Dv` は `D` の連続する列から形成され、これらの入力からのフィードスルーマトリックスです。制御、外乱、ノイズ、および補助入力ベクトルの次元はそれぞれ `sysm.mu`、 `sysm.md`、 `sysm.mw`、および `sysm.ma` に含まれています。

制御、外乱、ノイズ、および補助入力のコンポーネントの分割に関する情報は、次のキーワード引数を使用して指定できます。

`mu = nu` は制御入力ベクトル `u` の次元 `nu` を指定します（デフォルト: `nu = 0`）

`md = nd` は外乱入力ベクトル `d` の次元 `nd` を指定します（デフォルト: `nd = 0`）

`mw = nw` はノイズ入力ベクトル `w` の次元 `nw` を指定します（デフォルト: `nw = 0`）

`ma = na` は補助入力ベクトル `v` の次元 `na` を指定します（デフォルト: `na = 0`）
