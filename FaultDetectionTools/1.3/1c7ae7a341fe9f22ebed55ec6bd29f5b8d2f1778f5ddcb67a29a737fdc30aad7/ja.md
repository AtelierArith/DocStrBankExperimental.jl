```
 γ = fdimmperf(sysr::FDIFilterIF, SFDI[, nrmflag])
```

安定した故障検出および隔離フィルタ内部形式オブジェクト `sysr::FDIFilterIF` と関連するバイナリ構造行列 `SFDI` のモデルマッチング性能 `γ` を計算します。フィルタ `sysr` は `N` 個の個別の FDI フィルタ `sysr.sys[i]` から構成され、`i = 1, ..., N` の範囲で、`sysr.sys[i][:,sysr.faults]` は `i` 番目のフィルタの故障から残差へのチャネルで、対応する伝達関数行列は `Rfi(λ)` です。また、`sysr.sys[i][:,sysr.noise]` は `i` 番目のフィルタのノイズから残差へのチャネルで、対応する伝達関数行列は `Rwi(λ)` です。次に、`γ` は `N` 次元ベクトルで、その `i` 番目の成分は `[Rfdi(λ) Rwi(λ)]` の H∞-ノルムであり、`nrmflag = Inf`（デフォルト）の場合、または `[Rfdi(λ) Rwi(λ)]` の H2-ノルムであり、`nrmflag = 2` の場合です。ここで、`Rfdi(λ)` は、`SFDI[i,j] = 1` の場合はゼロであり、`SFDI[i,j] = 0` の場合は `Rfi(λ)` の `j` 番目の列に等しい伝達関数行列です。`γ` の `i` 番目の成分は、フィルタが不安定な場合、または `nrmflag = 2` で連続時間システムの伝達関数行列 `[Rfdi(λ) Rwi(λ)]` が厳密に適切でない場合は無限大になります。
