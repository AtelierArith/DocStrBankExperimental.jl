```
FDFilter(sys, ny, mu) -> Q::FDFilter
```

与えられた線形時間不変記述子システムモデル `sys = (A-λE,B,C,D)` に対して、FDIフィルタの合成関数を用いて決定される故障検出フィルタオブジェクト `Q` を構築します。 `ny` と `mu` はそれぞれ測定出力の数と制御入力の数です。 `B = [By Bu Bv]` および `D = [Dy Du Dv]` は、`By` と `Dy` が `ny` 列を持ち、`Bu` と `Du` が `mu` 列を持つように分割された行列であると仮定します。ここで、`By` と `Bu` は測定出力 `y` と制御入力 `u` からの入力行列であり、`Dy` と `Du` は測定出力 `y` と制御入力 `u` からのフィードスルー行列です。

結果として得られる `Q` には、分割されたシステム `Q.sys = (A-λE,[By Bd],C,[Dy Du])` が含まれ、分割されたフィルタ入力ベクトルの次元は `測定出力` と `制御入力` として、`Q.ny` および `Q.mu` に含まれる整数としてアクセスできます。
