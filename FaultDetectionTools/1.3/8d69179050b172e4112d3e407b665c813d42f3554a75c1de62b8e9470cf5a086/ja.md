```
  fdif2ngap(sysr::FDFilterIF, SFDI, freq; atol = √ϵ) -> (gap, β, γ)
```

故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` に関連する安定伝達関数行列 `Rf(λ)` と残差へのノイズ入力の伝達チャネルの `Rw(λ)` に対して、`q × mf` バイナリ構造行列 `SFDI` の故障からノイズへのギャップ `gap`（および関連する量 `β` と `γ`）を計算します。ベクトル `gap`、`β` および `γ` の `i` 番目の要素には次の量が含まれます： `β[i]` - `Rffi(λ)` の H∞-インデックス、`γ[i]` - `[Rdfi(λ) Rwi(λ)]` の H∞-ノルム、そして故障からノイズへのギャップ `gap` は `gap[i] := β[i]/γ[i]` として評価されます。ここで、`Rffi(λ)` は `S[i,j] = 1` である `Rf(λ)` の `i` 番目の行の `j` 番目の列から構成され、`Rdfi(λ)` は `S[i,j] = 0` である `Rf(λ)` の `i` 番目の行の `j` 番目の列から構成されます。`[Rdfi(λ) Rwi(λ)] = 0` の場合、`gap[i] = ∞` となり、`Rffi(λ) = 0` の場合、`gap[i] = 0` となります。`freq` が実数の周波数値のベクトルである場合、`β[i]` と `γ[i]` は `freq` に含まれる周波数に対して評価されます。`atol` はノルム `Rwi(λ)` の絶対許容誤差であり、ノルム値が `atol` 以下の場合はゼロと見なされます（デフォルト： `√ϵ`、ここで `ϵ` は作業機械精度です。）
