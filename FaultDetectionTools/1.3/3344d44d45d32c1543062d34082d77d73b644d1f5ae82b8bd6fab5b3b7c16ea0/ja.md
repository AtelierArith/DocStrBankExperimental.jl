```
mdmatch(sysQ::MDFilter, sysc::MDModel; MDfreq, minimal = false, rtolinf, offset, atol, atol1 = atol, atol2 = atol, rtol, fast = true) -> (mdgain,fpeak,mind)
```

モデル検出フィルタオブジェクト `sysQ::MDFilter` をコンポーネントモデル `sysc::MDModel` に適用して得られる距離マッピング性能ベクトル `mdgain`、対応するピーク周波数ベクトル `fpeak`、および最小ゲイン値が達成される `mdgain` のコンポーネントのインデックス `mind` を計算します。

`i` 番目のフィルタ `sysQ.sys[i]` が伝達関数行列 `Qi(λ)` を持ち、コンポーネントモデル `sysc::MDModel` が分割された伝達関数行列 `G(λ) = [Gu(λ)  Gd(λ) Gw(λ) Gv(λ)]` を持つ場合、これはそれぞれ制御、外乱、ノイズ、および補助入力に対応しています。したがって、与えられたコンポーネントモデルに適用された `i` 番目のフィルタの距離マッピング性能は `mdgain[i] = || Ri(λ) ||∞` として計算されます。ここで、`Ri(λ)` は対応する内部形式です。

```
 Ri(λ) = Qi(λ) * | Gu(λ)  Gd(λ) Gw(λ) Gv(λ) | .
                 |  I     0     0     0     |
```

`minimal = true` の場合、`Ri(λ)` の最小記述子実現が計算され、`minimal = false`（デフォルト）の場合は（おそらく）非最小の実現が決定されます。

個々の入力-出力チャネルの最小実現の計算は、ペンシル操作アルゴリズムに依存しており、これは `fast = true`（デフォルト）の場合は列ピボットを用いたランク明示QR分解、または `fast = false` の場合はより信頼性の高いSVD分解に基づくランク決定を使用します。

`(Ari-λEri,Bri,Cri,Dri)` が `Ri(λ)` のフルオーダー記述子実現である場合、キーワード引数 `atol1`、`atol2`、および `rtol` はそれぞれ、行列 `Ari`、`Bri`、`Cri`、`Dri` の非ゼロ要素に対する絶対許容誤差、`Eri` の非ゼロ要素に対する絶対許容誤差、`Ari`、`Bri`、`Cri`、`Dri` および `Eri` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `ni*ϵ` であり、ここで `ϵ` は作業用 *マシンイプシロン* で、`ni` は `Ri(λ)` の実現の次数です。キーワード引数 `atol` を使用して `atol1 = atol` および `atol2 = atol` を同時に設定できます。
