```
 fdiscond(sysr::FDFilterIF,SFDI,freq) -> (scond, β, γ)
```

検出および隔離感度条件 `scond`（および関連する量 `β` と `γ`）を、故障入力から故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` の残差に関連する安定伝達関数行列 `Rf(λ)` に関連付けられたバイナリ構造ベクトル `SFDI` に対して計算します。もし `Rff(λ)` が `SFDI[j] = 1` の場合の `Rf(λ)` の `j` 番目の列から形成される伝達関数行列であるならば、`β` - `Rff(λ)` の H∞- 指数、`γ` - `Rff(λ)` の列ノルムの最大値、そして故障検出感度条件 `scond` は `scond := β/γ` と評価されます。もし `freq` が実数周波数値のベクトルであるならば、`β` と `γ` は `freq` に含まれる周波数に対して評価されます。
