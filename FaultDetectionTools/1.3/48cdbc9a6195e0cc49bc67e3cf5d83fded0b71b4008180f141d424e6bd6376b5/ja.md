```
 γ = fdimmperf(sysr::FDFilterIF[, nrmflag])
```

故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` のモデルマッチング性能 `γ` を計算します。 `Rw(λ)` がノイズ入力から残差 `sysr.sys[:,sysr.noise]` への伝達チャネルの伝達関数行列である場合、`nrmflag = Inf`（デフォルト）のときは `γ` は `Rw(λ)` の H∞-ノルムであり、`nrmflag = 2` のときは `Rw(λ)` の H2-ノルムです。 `γ` の値は、不安定なフィルタの場合や、`nrmflag = 2` で連続時間システムの伝達関数行列 `Rw(λ)` が厳密に適切でない場合は無限大になります。
