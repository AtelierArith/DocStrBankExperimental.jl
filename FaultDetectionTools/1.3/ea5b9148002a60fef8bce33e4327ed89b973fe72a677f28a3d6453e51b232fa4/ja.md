```
 fdif2ngap(sysr::FDFilterIF, SFDI, freq) -> (gap, β, γ)
```

安定した故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` と関連するバイナリ構造ベクトル `SFDI` に対して、故障からノイズへのギャップ `gap`（および関連する量 `β` と `γ`）を計算します。 `sysr.sys[:,sysr.faults]` はフィルタの故障から残差へのチャネルであり、対応する伝達関数行列は `Rf(λ)` です。また、`sysr.sys[:,sysr.noise]` はフィルタのノイズから残差へのチャネルであり、対応する伝達関数行列は `Rw(λ)` です。 `Rff(λ)` は、`SFDI[j] = 1` の場合の `Rf(λ)` の `j` 番目の列から形成される伝達関数行列であり、`Rdf(λ)` は、`SFDI[j] = false` の場合の `Rf(λ)` の `j` 番目の列から形成される伝達関数行列です。したがって、`β` は `Rff(λ)` の H∞-インデックスであり、`γ` は `[Rdf(λ) Rw(λ)]` の H∞-ノルムであり、`gap` は `gap := β/γ` として評価される故障からノイズへのギャップです。 `freq` が実数の周波数値のベクトルである場合、`β` と `γ` は `freq` に含まれる周波数に対して評価されます。 `[Rdf(λ) Rw(λ)] = 0` の場合、`gap = ∞` となり、`Rff(λ) = 0` の場合、`gap = 0` となります。
