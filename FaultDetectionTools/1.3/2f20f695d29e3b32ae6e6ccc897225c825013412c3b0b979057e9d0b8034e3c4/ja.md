```
 fdif2ngap(sysr::FDFilterIF, freq) -> (gap, β, γ)
```

安定した故障検出フィルタ内部形式オブジェクト `sysr::FDFilterIF` に対して、故障からノイズへのギャップ `gap`（および関連する量 `β` と `γ`）を計算します。フィルタの故障から残差チャネル `sysr.sys[:,sysr.faults]` に対して、対応する伝達関数行列 `Rf(λ)` があり、フィルタのノイズから残差チャネル `sysr.sys[:,sysr.noise]` に対して、対応する伝達関数行列 `Rw(λ)` があります。`β` は `Rf(λ)` の H∞-インデックスであり、`γ` は `Rw(λ)` の H∞-ノルムであり、`gap` は `gap := β/γ` として評価される故障からノイズへのギャップです。`freq` が実数の周波数値のベクトルである場合、`β` と `γ` は `freq` に含まれる周波数で評価されます。ノイズ入力がない場合、`gap = ∞` となり、故障入力がない場合、`gap = 0` となります。
