```
FDIModel(sys; mu, md, mf, mw, ma) -> sysf::FDIModel
```

線形不変記述子システム `sys = (A-λE,B,C,D)` のために、故障検出フィルタの分析および合成関数と併用するための合成モデルオブジェクト `sysf::FDIModel` を構築します。

結果として得られる合成モデルオブジェクト `sysf` には、入力が分割されたコンポーネントモデル `sysc.sys = (A-λE,[Bu Bd Bf Bw Bv],C,[Du Dd Df Dw Dv])` が含まれます。ここで、`Bu`、`Bd`、`Bf`、`Bw`、および `Bv` は `B` の連続した列から形成され、制御入力 `u`、外乱入力 `d`、故障入力 `f`、ノイズ入力 `w`、および補助入力 `v` からの入力行列です。また、`Du`、`Dd`、`Df`、`Dw`、および `Dv` は `D` の連続した列から形成され、これらの入力からのフィードスルーマトリックスです。制御、外乱、故障、ノイズ、および補助入力ベクトルの次元はそれぞれ `sysf.mu`、`sysf.md`、`sysf.mf`、`sysf.mw`、および `sysf.ma` に含まれています。

制御、外乱、故障、ノイズ、および補助入力の入力コンポーネントの分割に関する情報は、以下のキーワード引数を使用して指定できます。

`mu = nu` は制御入力ベクトル `u` の次元 `nu` を指定します（デフォルト: `nu = 0`）

`md = nd` は外乱入力ベクトル `d` の次元 `nd` を指定します（デフォルト: `nd = 0`）

`mf = nf` は故障入力ベクトル `f` の次元 `nf` を指定します（デフォルト: `nf = 0`）

`mw = nw` はノイズ入力ベクトル `w` の次元 `nw` を指定します（デフォルト: `nw = 0`）

`ma = na` は補助入力ベクトル `v` の次元 `na` を指定します（デフォルト: `na = 0`）
