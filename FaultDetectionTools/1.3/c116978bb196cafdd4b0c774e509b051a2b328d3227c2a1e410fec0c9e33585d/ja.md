```
 fdhinfminus(sys,freq) -> (β, ind, fr)
```

安定した記述子システム `sys = (A-λE,B,C,D)` の伝達関数行列 `G(λ)` の `H∞-` 指数 `β` を計算します。`freq = missing`（デフォルト）の場合、`β` は `G` の列の最小 `H∞-ノルム` であり、`ind` は最小ノルム列のインデックス、`fr` は列の最小 `H∞-ノルム` が達成される周波数です。`freq` が実数値または実数ベクトルの周波数値の場合、`β` は `freq` に含まれるすべての値に対して評価された `G` の周波数応答の列の 2-ノルムの最小値であり、`ind` は最小が達成される列のインデックス、`fr` は対応する周波数です。
