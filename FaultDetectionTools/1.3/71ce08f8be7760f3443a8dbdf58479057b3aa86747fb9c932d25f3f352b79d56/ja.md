```
 S = mdsspec(sysR::MDFilterIF, freq; cdinp = false, MDGainTol = 0.01, 
             atol, atol1 = atol, atol2 = atol, rtol = 0, fast = true)
```

与えられた実周波数のセット `freq` に対して、モデル検出フィルタのコレクションの強いバイナリ構造行列 `S` を、モデル検出フィルタ内部形式オブジェクト `sysR::MDFilterIF` を使用して計算します。

`freq` には、持続的な制御および干渉信号のクラスを特徴づける実周波数値または `nf` 個の実周波数のベクトルを含める必要があります（デフォルト: `freq = 0`、すなわち定常信号を特徴づける）および、持続的信号のクラスを特徴づける複素周波数の集合 `Ω` を定義します。次のように、`f` が `freq` の実周波数である場合、対応する複素周波数 `λ` は、連続時間システムの場合は `λ := im*f`、離散時間システムの場合は `λ := exp(im*f*abs(Ts))` です。

`M × N` のフィルタの配列 `sysR` に対して、`S` は次のように決定される `M × N` のバイナリ配列です。`(i,j)`-th コンポーネントフィルタ `sysR.sys[i,j]` の入力-出力形式は次のようになります。

```
 rij = Ruij(λ)*u + Rdij(λ)*dj + Rwij(λ)*wj + Rvij(λ)*vj ,
```

ラプラス変換または Z 変換された残差出力 `rij`、制御入力 `u`、干渉入力 `dj`、ノイズ入力 `wj`、および補助入力 `vj` を持ち、対応する伝達関数行列は `Ruij(λ)`, `Rdij(λ)`, `Rwij(λ)` および `Rvij(λ)` です。したがって、`S[i,j] = 1` となるのは、`Ruij(λs)` が任意の `λs ∈ Ω` に対して非ゼロであり、かつ `cdinp = false`（デフォルト）である場合、または `[Ruij(λs) Rdij(λs)]` が任意の `λs ∈ Ω` に対して非ゼロであり、かつ `cdinp = true` である場合です。それ以外の場合、`S[i,j] = 0` となります。

`MDGainTol = tol` は、周波数応答ゲインの非ゼロの大きさに対する絶対閾値 `tol` を指定します（デフォルト: `tol = 0.01`）。

個々の入力-出力チャネルの最小実現の計算は、ペンシル操作アルゴリズムに依存しており、`fast = true`（デフォルト）の場合は列ピボッティングを用いたランク明示化 QR 分解、または `fast = false` の場合はより信頼性の高い SVD 分解に基づくランク決定を使用します。

`(Arij-λErij,Brij,Crij,Drij)` が `sysR.sys[i,j]` の記述子実現である場合、キーワード引数 `atol1`, `atol2`, および `rtol` はそれぞれ、行列 `Arij`, `Brij`, `Crij`, `Drij` の非ゼロ要素に対する絶対許容誤差、`Erij` の非ゼロ要素に対する絶対許容誤差、および `Arij`, `Brij`, `Crij`, `Drij` および `Eirj` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `nij*ϵ` であり、ここで `ϵ` は作業用 *マシンイプシロン* で、`nij` はシステム `sysR.sys[i,j]` の次数です。キーワード引数 `atol` は、`atol1 = atol` および `atol2 = atol` を同時に設定するために使用できます。 ```
