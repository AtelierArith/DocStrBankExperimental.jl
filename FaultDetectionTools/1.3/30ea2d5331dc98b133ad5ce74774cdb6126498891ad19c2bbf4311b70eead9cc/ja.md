```
MDMModel(sys; mu, md, mw, ma) -> sysm::MDMModel
```

線形不変システムモデルのベクトル `sys` に対して構築され、`i` 番目のコンポーネントモデル `sys[i] = (Ai-λEi,Bi,Ci,Di)` を持つ複数合成モデルオブジェクト `sysm::MDMModel` は、モデル検出フィルタの分析および合成関数と共に使用されます。

結果として得られる複数合成モデルオブジェクト `sysm` には、入力が分割されたコンポーネントモデルのベクトル `sysm.sys` が含まれ、`i` 番目のモデル `sysm.sys[i] = (Ai-λEi,[Bui Bdi Bwi Bvi],Ci,[Dui Ddi Dwi Dvi])` となります。ここで、`Bui`、`Bdi`、`Bwi`、および `Bvi` は `Bi` の連続する列から形成され、制御入力 `u`、外乱入力 `di`、ノイズ入力 `wi`、および補助入力 `vi` からの入力行列です。また、`Dui`、`Ddi`、`Dwi`、および `Dvi` は `Di` の連続する列から形成され、これらの入力からのフィードスルー行列です。制御、外乱、ノイズ、および補助入力ベクトルの次元はそれぞれ `sysm.mu`、`sysm.md[i]`、`sysm.mw[i]`、および `sysm.ma[i]` に含まれています。

`N` がコンポーネントモデルの数である場合、制御、外乱、ノイズ、および補助入力の入力コンポーネントの分割に関する情報は、次のキーワード引数を使用して指定できます。

`mu = nu` は制御入力ベクトル `u` の次元 `nu` を指定します（デフォルト: `nu = 0`）

`md = nd` は外乱入力ベクトルの次元を含むベクトル `nd` を指定し、`i` 番目の外乱ベクトル `di` は次元 `nd[i]` を持ちます（デフォルト: `nd = zeros(Int,N)`）

`mw = nw` はノイズ入力ベクトルの次元を含むベクトル `nw` を指定し、`i` 番目のノイズベクトル `wi` は次元 `nw[i]` を持ちます（デフォルト: `nw = zeros(Int,N)`）

`ma = na` は補助入力ベクトルの次元を含むベクトル `na` を指定し、`i` 番目の補助ベクトル `vi` は次元 `na[i]` を持ちます（デフォルト: `na = zeros(Int,N)`）
