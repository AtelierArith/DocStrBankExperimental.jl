```
 mdperf(sysR::MDFilterIF; MDfreq, cdinp = false, rtolinf = 0.00001, 
        offset, atol, atol1 = atol, atol2 = atol, rtol = 0, fast = true) -> (mdgain, fpeak)
```

モデル検出フィルタのコレクションの距離マッピング性能を、モデル検出フィルタ内部形式オブジェクト `sysR::MDFilterIF` を使用して計算します。 `M × N` のフィルタ配列 `sysR` に対して、モデル検出性能ゲインの `M × N` 配列 `mdgain` と対応するピーク周波数 `fpeak` は次のように決定されます。 `(i,j)`-th コンポーネントフィルタ `sysR.sys[i,j]` は、次の入力出力形式を持ちます。

```
 rij = Ruij(λ)*u + Rdij(λ)*dj + Rwij(λ)*wj + Rvij(λ)*vj ,
```

ここで、ラプラス変換または Z 変換された残差出力 `rij`、制御入力 `u`、外乱入力 `dj`、ノイズ入力 `wj`、および補助入力 `vj` があり、`Ruij(λ)`, `Rdij(λ)`, `Rwij(λ)` および `Rvij(λ)` は対応する伝達関数行列です。次に、`(i,j)`-th 性能ゲインは、`cdinp = false`（デフォルト）の場合は `mdgain[i,j] = ||Ruij(λ)||∞` と評価され、`cdinp = true` の場合は `mdgain[i,j] = ||[Ruij(λ) Rdij(λ)]||∞` となり、`fpeak[i,j]` には対応するピーク周波数が含まれます。

`MDfreq = ω` の場合、ここで `ω` は与えられた実周波数値のベクトルであり、各ゲイン `mdgain[i,j]` は `ω` のすべての周波数に対して評価された 2-ノルムの点ごとの最大値を表し、`fpeak[i,j]` は対応するピーク周波数です。

安定性境界オフセット `β` は、安定性領域の境界に属する有限極を評価するために使用され、キーワードパラメータ `offset = β` を介して指定できます。したがって、連続時間システムの場合、これらは実部が区間 `[-β,β]` 内にある有限極であり、離散時間システムの場合、これらはモジュールが区間 `[1-β,1+β]` 内にある有限極です。 `β` に使用されるデフォルト値は `sqrt(ϵ)` であり、ここで `ϵ` は作業機械精度です。

ペンシル削減アルゴリズムは H∞-ノルムを計算するために使用されます。これらのアルゴリズムは、`fast = true`（デフォルト）の場合は列ピボットを伴うランクを明らかにする QR 分解に基づいてランク決定を行い、`fast = false` の場合はより信頼性の高い SVD 分解を行います。

`(Arij-λErij,Brij,Crij,Drij)` が `sysR.sys[i,j]` の記述子実現である場合、キーワード引数 `atol1`, `atol2`, および `rtol` はそれぞれ、行列 `Arij`, `Brij`, `Crij`, `Drij` の非ゼロ要素の絶対許容誤差、`Erij` の非ゼロ要素の絶対許容誤差、および `Arij`, `Brij`, `Crij`, `Drij` および `Eirj` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `nij*ϵ` であり、ここで `ϵ` は作業 *機械イプシロン* で、`nij` はシステム `sysR.sys[i,j]` の次数です。キーワード引数 `atol` を使用して、`atol1 = atol` および `atol2 = atol` を同時に設定できます。

キーワード引数 `rtolinf = tol` は、無限ノルムを計算するために使用される相対精度 `tol` を指定します。使用されるデフォルト値は `tol = 0.00001` です。
