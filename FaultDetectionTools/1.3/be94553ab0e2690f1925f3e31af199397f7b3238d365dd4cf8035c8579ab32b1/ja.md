```
FDIFilter(sys, ny, mu) -> Q::FDIFilter
```

同じ数の入力を持つ線形時間不変記述子システムモデルのベクトル `sys[i] = (Ai-λEi,Bi,Ci,Di)` に対して、FDIフィルタの合成関数で決定される故障検出および隔離フィルタオブジェクト `Q` を構築します。 `ny` と `mu` はそれぞれ測定出力の数と制御入力の数です。各 `Bi = [Byi Bui Bvi]` および `Di = [Dyi Dui Dvi]` は、`Byi` と `Dyi` が `ny` 列を持ち、`Bui` と `Dui` が `mu` 列を持つように分割された行列であると仮定します。ここで、`Byi` と `Bui` は測定出力 `y` と制御入力 `u` からの入力行列であり、`Dyi` と `Dui` は測定出力 `y` と制御入力 `u` からのフィードスルー行列です。

結果として得られる `Q` には、分割されたシステムのベクトル `Q.sys[i] = (Ai-λEi,[Byi Bdi],Ci,[Dyi Dui])` が含まれ、分割されたフィルタ入力ベクトルの次元は `測定出力` と `制御入力` として、`Q.ny` および `Q.mu` に含まれる整数としてアクセスできます。
