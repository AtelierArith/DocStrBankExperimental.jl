```
mdmodset(sys; controls, c, disturbances, d, noise, n, aux) -> sysm::MDMModel
```

線形時不変システムモデルのベクトル `sys` に対して構築され、`i` 番目のコンポーネントモデル `sys[i] = (Ai-λEi,Bi,Ci,Di)` を持つ複数合成モデルオブジェクト `sysm::MDMModel` が、モデル検出フィルタの分析および合成関数と共に使用されます。

制御、外乱、ノイズ、および補助入力の入力コンポーネントの分割に関する情報は、次のキーワード引数を使用して指定できます。

`controls = inpu` または `c = inpu` は、制御入力のインデックス `inpu` を指定します（デフォルト: void）

`disturbances = inpd` または `d = inpd` は、外乱入力のインデックスまたはインデックスのベクトル `inpd` を指定します（デフォルト: void）

`noise = inpn` または  `noise = inpn` は、ノイズ入力のインデックスまたはインデックスのベクトル `inpn` を指定します（デフォルト: void）

`aux = inpa` は、補助入力のインデックスまたはインデックスのベクトル `inpa` を指定します（デフォルト: void）

入力のインデックスは、整数ベクトル、整数スカラー、または整数 `UnitRange` として指定できます。外乱、ノイズ、および補助入力については、各コンポーネントモデルに対して異なるインデックスのセットを指定するために、整数ベクトルのベクトルまたは整数 `UnitRange` のベクトルを使用できます。

結果として得られる `sysm` には、分割されたシステムのベクトル `sysm.sys` が含まれ、`i` 番目のモデルは `sysm.sys[i] = (Ai-λEi,[Bui Bdi Bwi Bvi],Ci,[Dui Ddi Dwi Dvi])` となります。ここで、`Bui`、`Bdi`、`Bwi`、および `Bvi` はそれぞれ制御入力 `u`、外乱入力 `di`、ノイズ入力 `wi`、および補助入力 `vi` からの入力行列であり、`Dui`、`Ddi`、`Dwi`、および `Dvi` はそれらの入力からのフィードスルー行列です。制御、外乱、ノイズ、および補助入力ベクトルの次元は、それぞれ `sysm.mu`、`sysm.md[i]`、`sysm.mw[i]`、および `sysm.ma[i]` に含まれています。

*メソッド:* `Gi(λ)` が `sys[i]` の `p x mi` 転送関数行列である場合、結果として得られるコンポーネントシステム `sysm.sys[i]` は、次の関係がコンポーネント行列を定義する同等の入力出力形式 `[Gui(λ) Gdi(λ) Gwi(λ) Gvi(λ)]` を持ちます: `Gui(λ) = Gi(λ)*Su`、`Gdi(λ) = Gi(λ)*Sdi`、`Gwi(λ) = G(λ)*Swi`、`Gvi(λ) = Gi(λ)*Svi`。選択行列 `Su`、`Sdi`、`Swi`、および `Svi` は、`mi` 次の単位行列の列から形成されます（`Su` はすべてのコンポーネントモデルで同じであることに注意してください）。
