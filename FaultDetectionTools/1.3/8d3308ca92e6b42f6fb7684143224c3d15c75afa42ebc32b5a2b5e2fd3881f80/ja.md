```
 fdiscond(sysr::FDIFilterIF, SFDI, freq) -> (scond, β, γ)
```

検出および隔離感度条件 `scond`（および関連する量 `β` と `γ`）を計算します。これは、故障入力から故障検出および隔離フィルタ内部形式オブジェクト `sysr::FDIFilterIF` の残差に関連する安定なグローバルトランスファ関数行列 `Rf(λ)` に関連付けられた `N × mf` バイナリ構造行列 `SFDI` に対して行われます。フィルタ `sysr` は、`N` 個の個別の FDI フィルタ `sysr.sys[i]` で構成されており、`i = 1, ..., N` の範囲で、`sysr.sys[i][:,sysr.faults]` は `i` 番目のフィルタの故障から残差へのチャネルであり、`Rfi(λ)` は対応するトランスファ関数行列です。グローバルトランスファ関数行列 `Rf(λ)` は、`N` 個の個別フィルタのトランスファ関数行列の行の連結によって形成されます。すなわち、`Rf(λ) := [ Rf1(λ); Rf2(λ); ...; RfN(λ)]` です。`SFDI[i,j] = 1` である各 `j` に対して、`Rfi(λ)` の `j` 番目の列は非ゼロであり、`SFDI[i,j] = 0` である各 `j` に対して、`Rfi(λ)` の `j` 番目の列はゼロであると仮定します。ベクトル `scond`、`β` および `γ` の `i` 番目の要素には、次の量が含まれます： `β[i]` - `Rfi(λ)` の非ゼロ列の H∞-インデックス、`γ` - `Rf(λ)` の非ゼロ列ノルムの最大値、そして故障検出感度条件 `scond` は `scond[i] := β[i]/γ[i]` として評価されます。`freq` が実数の周波数値のベクトルである場合、`β[i]` と `γ[i]` は `freq` に含まれる周波数に対して評価されます。 ```
