```
TotalMassFlux(scale = si_unit(:day), max_abs = nothing, max_rel = nothing)
```

通常、主変数として使用される変数。面を通過する総質量フラックスを表します。典型的な使用法は、[`MultiSegmentWell`](@ref)のセグメントを通る質量フローです。

フローの方向は、複雑なシミュレーション中にセグメントを通じて符号が切り替わることがよくあります。したがって、`max_rel`を`nothing`以外の値に設定すると、フローの反転の場合に深刻な収束問題を引き起こす可能性があります。

# フィールド（キーワード引数として）

  * `scale`: 変数のスケーリング
  * `max_abs`: ニュートン反復間の最大絶対変化
  * `max_rel`: ニュートン反復間の最大相対変化
