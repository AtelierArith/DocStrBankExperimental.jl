```
compute_peaceman_index(Δ, K, radius; kwargs...)
```

与えられたグリッドブロックのためのPeaceman井戸インデックスを計算します。

# 引数

  * `Δ`: グリッドブロックのサイズを表すタプル `(dx, dy, dz)`
  * `K`: 媒体の透過性（全テンソルの場合は行列、またはスカラー）。
  * `radius`: 井戸の半径。

# キーワード引数

  * `dir::Symbol = :z`: 井戸の方向、`:x`、`:y`、または `:z` を指定できます。
  * `net_to_gross = 1.0`: ネット対粗比率、垂直方向の値をスケールするために使用されます。
  * `constant = 0.14`: 等価半径の計算に使用される定数。TPFA特有。
  * `Kh = nothing`: 水平透過性、提供されていない場合は計算されます。
  * `drainage_radius = nothing`: 排水半径、提供されていない場合は計算されます。
  * `skin = 0`: 皮膚係数、近井戸効果を考慮するために使用されます。
  * `check = true`: 負の井戸インデックス値をチェックするためのフラグ。

# 戻り値

  * `Float64`: 計算されたPeaceman井戸インデックス。
