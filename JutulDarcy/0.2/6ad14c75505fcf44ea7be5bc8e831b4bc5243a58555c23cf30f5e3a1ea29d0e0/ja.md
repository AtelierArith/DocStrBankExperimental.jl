```
reservoir_domain(g; permeability = convert_to_si(0.1, :darcy), porosity = 0.1, kwarg...)
```

与えられたメッシュまたは他の表現 `g` のために `DataDomain` インスタンスを設定します。`permeability` と `porosity` はその後ドメインに追加されます。スカラーが渡されると、それらはすべてのセルをカバーするように展開されます。配列はすべてのセルと一致することが確認されます。透過率は、セルごとに1つの値（対角スカラー）、各行に与えられた次元ごとの1つの値（対角テンソルの場合）、またはコンパクトな全テンソル表現を表すベクトル（3Dでは6要素、2Dでは3要素）である必要があります。

# デフォルトデータとその値

|                           名前 |                    説明 |         単位 |  デフォルト |
| ----------------------------:| ---------------------:| ----------:| ------:|
|               `permeability` |          岩石の流体流動を導く能力 |      $m^2$ | 100 mD |
|                   `porosity` | 流動に開放された岩石の空隙率 (0から1) |          - |    0.3 |
|               `rock_density` |               岩石の質量密度 | $kg^3/m^3$ | 2000.0 |
|         `rock_heat_capacity` |               岩石の比熱容量 | $J/(kg K)$ |  900.0 |
|  `rock_thermal_conductivity` |               岩石の熱伝導率 |    $W/m K$ |    3.0 |
| `fluid_thermal_conductivity` |              流体相の熱伝導率 |    $W/m K$ |    0.6 |
|    `component_heat_capacity` |             流体成分の比熱容量 | $J/(kg K)$ | 4184.0 |

デフォルト値は、流体相の水と岩石の砂岩の値を大まかに取ることに注意してください。値の選択はシミュレーション結果に大きな影響を与える可能性があるため、物理システムが使用する値を確認することに注意してください！

# オプション値

|                            名前 |              説明 |      単位 | デフォルト |
| -----------------------------:| ---------------:| -------:| -----:|
|                `net_to_gross` | 流動に利用可能な空隙率の大きさ |       - |   1.0 |
|                   `diffusion` |        各成分の拡散係数 | $m^2/s$ |   0.0 |
|   `transmissibility_override` |  各面の透過率のオーバーライド |   $m^2$ |   NaN |
| `transmissibility_multiplier` |       各面の透過率の乗数 |       - |   1.0 |

これらの値はオプションであり、指定された場合にのみ追加されます。例えば、ネットからグロスおよび透過率の乗数については、コード内で存在しない場合はデフォルト値1.0が仮定されます。

透過率のオーバーライドは、幾何学や他の特性から計算された透過率をオーバーライドするために使用できます。これは、外部で計算されたモデルがある場合に便利です。値はメッシュのすべての面に対して与えられる必要があります。NaNまたはInfの値は欠損として扱われ、その面については標準の透過率計算機が代わりに使用されます。
