```
InjectorControl(target, mix, density = 1.0, phases = ((1, 1.0)), temperature = 293.15)
```

貯留層への注入を指定する制御です。`target`はターゲットのタイプを指定し、`mix`は注入中のモデル内のすべての種の注入質量分率を定義します。

例えば、CO2、H2O、H2からなる三成分系の場合、[0.1, 0.6, 0.3]を設定すると、注入流は質量で1部のCO2、6部のH2O、3部のH2を含むことを意味します。不混和系（例：`LiquidPhase(), VaporPhase()`）の場合、種は相に対応し、[0.3, 0.7]は質量で液体と蒸気の3対7の混合を意味します。

注入された流体の表面条件下での密度は`density`によって与えられ、指定がない場合はデフォルトで1.0になります。

また、[`ProducerControl`](@ref)、[`DisabledControl`](@ref)も参照してください。
