```
replacer_channel(
    ρ :: State,
    σ :: State,
    μ :: Real
) :: State
```

リプレイサーチャネルは、量子状態 `ρ` を確率 `μ` で量子状態 `σ` に置き換えます。リプレイサーチャネルは次のように表現されます：

$$
    \mathcal{R}_{\mu}(\rho) = \mu \rho + (1-\mu)\sigma
$$

次の場合に `DomainError` がスローされます：

  * `μ` が `1 ≥ μ ≥ 0` を満たさない場合
  * `ρ` と `σ` のサイズが異なる場合
