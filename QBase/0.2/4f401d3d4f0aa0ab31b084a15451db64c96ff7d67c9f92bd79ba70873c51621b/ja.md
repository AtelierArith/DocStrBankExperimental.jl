```
choi_matrix( 𝒩 :: Function, dims :: Vector{Int} ) :: Matrix{ComplexF64}
```

チャネルのチョイ演算子を返します。チョイ行列は次のように構築されます。

$$
        \Lambda^{AB} = \sum_{i,j \in [d_A]} E^A_{i,j} \otimes \mathcal{N}^{A' \to B}(E_{i,j}^{A'}) ,
$$

ここで、$[d_A]$は入力空間をインデックス付けする有限アルファベットであり、$E_{i,j}$は次元$d_A$の正方行列で、$(i,j)$のエントリに$1$があり、他のすべての場所には$0$があります。入力$\Lambda$は出力次元です。

入力関数`𝒩`は任意の入力`X`に対して`𝒩(X)`として呼び出されます。複数のパラメータを持つチャネル関数は、固定された`(x,y)`に対して`𝒩_xy(ρ) = 𝒩(ρ,x,y)`という関数を宣言することで考慮でき、その後、`choi(𝒩_xy, dim_in, dim_out)`と呼び出します。
