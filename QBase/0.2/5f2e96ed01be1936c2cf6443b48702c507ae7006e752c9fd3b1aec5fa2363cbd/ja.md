```
erasure_channel( ρ :: State, μ :: Real ) :: State
```

消失チャネルは、量子状態 `ρ` をエラーフラグ $|F\rangle$ と混合します。このエラーフラグは `ρ` のヒルベルト空間に直交しています。引数 `μ` は、`ρ` がエラーフラグに置き換えられる確率を表します。量子状態 $\rho$ に対して、消失チャネルは次のように表されます：

$$
    \mathcal{E}_{\mu}(\rho) = \mu \rho + (1 - \mu) |F \rangle \langle F|
$$

消失チャネルはヒルベルト空間の次元を1増加させることに注意してください。

`μ` が `1 ≥ μ ≥ 0` を満たさない場合、`DomainError` がスローされます。
