コンテキストによって定義された暗黙の方程式。

この関数は、一般化代数理論（GAT）を本質的に代数的理論として表現することを可能にします。つまり、方程式によって定義されたドメインを持つ部分関数としてです。

参考文献：

  * (Cartmell, 1986, Sec 6: "本質的に代数的理論と有限限界を持つカテゴリ")
  * (Freyd, 1972, "トポスの側面")

この関数は、`args`の要素にアクセサ関数を適用することから推測できる`c.context`の要素を計算するための式を提供します。

例：

> equations({f::Hom(a,b), g::Hom(b,c)}, {a::Ob, b::Ob, c::Ob}, ThCategory)


ways*of*computing = Dict(a => [dom(f)], b => [codom(f), dom(g)], c => [codom(g)],                          f => [f], g => [g])
