`restyle!` メソッドは、plotly.js の `Plotly.restyle` 関数のセマンティクスに従います。具体的には、トレース `ind` の属性 `k` を値 `v` に設定しようとする際に、以下のルールが適用されます。ここで `ind` は `ind` のベクターにリストされている `i` 番目のトレースです（`ind` がスカラーの場合、`i` は常に 1 に等しいです）。

  * `v` が配列またはタプルである場合（どちらも `json(v)` が呼ばれると JavaScript 配列に変換されます）、`p.data[ind][k]` は `v[i]` に設定されます。以下に例を示します。
  * `v` が他の任意の型（任意のスカラー型）の場合、`k` は直接 `v` に設定されます。

**例**

```julia
# 最初の 2 つのトレースのマーカーの色を赤に設定
restyle!(p, [1, 2], marker_color="red")

# トレース 1 のマーカーの色を緑に、トレース 2 のマーカーの色を赤に設定
restyle!(p, [2, 1], marker_color=["red", "green"])

# トレース 1 のマーカーの色を赤に設定。緑は使用されません
restyle!(p, 1, marker_color=["red", "green"])

# トレース 1 の最初のマーカーを赤に、2 番目のマーカーを緑に設定
restyle!(p, 1, marker_color=(["red", "green"],))

# p に 3 つのトレースがあると仮定します。
# トレース 1 のマーカーの色を ["red", "green"] に設定
# トレース 2 のマーカーの色を "blue" に設定
# トレース 3 のマーカーの色を ["red", "green"] に設定
restyle!(p, 1:3, marker_color=(["red", "green"], "blue"))
```
