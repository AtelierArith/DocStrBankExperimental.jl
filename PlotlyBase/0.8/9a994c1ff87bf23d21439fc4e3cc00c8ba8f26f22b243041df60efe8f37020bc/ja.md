**パラメータ**

  * `rows`: サブプロットグリッドの行数。ゼロより大きくなければなりません。
  * `cols`: サブプロットグリッドの列数。ゼロより大きくなければなりません。
  * `shared_xaxes`: 2Dカーテシアンサブプロットのための共有（リンクされた）x軸を割り当てます。

      * true または "columns": 同じ列のサブプロット間で軸を共有します
      * "rows": 同じ行のサブプロット間で軸を共有します
      * "all": グリッド内のすべてのサブプロット間で軸を共有します
  * `shared_yaxes`: 2Dカーテシアンサブプロットのための共有（リンクされた）y軸を割り当てます。

      * "columns": 同じ列のサブプロット間で軸を共有します
      * true または "rows": 同じ行のサブプロット間で軸を共有します
      * "all": グリッド内のすべてのサブプロット間で軸を共有します。
  * `start_cell`: サブプロットのドメイングリッドを設定するために使用されるサブプロットグリッドの開始セルを選択します。

      * "top-left": サブプロットは左上隅の (1, 1) で番号付けされます
      * "bottom-left": サブプロットは左下隅の (1, 1) で番号付けされます
  * `horizontal_spacing`: 正規化されたプロット座標におけるサブプロット列間のスペース。0と1の間の浮動小数点数でなければなりません。すべての列に適用されます（"specs" サブプロット依存のスペーシングを使用）
  * `vertical_spacing`: 正規化されたプロット座標におけるサブプロット行間のスペース。0と1の間の浮動小数点数でなければなりません。すべての行に適用されます（"specs" サブプロット依存のスペーシングを使用）
  * `subplot_titles`: 行優先順序のリストとしての各サブプロットのタイトル。サブプロットタイトルがそのスペースに必要ない場合は、空の文字列 ("") をリストに含めることができ、タイトルが適切にインデックスされるようにします。
  * `specs`: サブプロットタイプ、行/列のスパン、およびスペーシングの各サブプロット仕様。

      * "specs" の行数は "rows" と等しくなければなりません。
      * "specs" の列数
      * "specs" リストの各アイテムは、サブプロットグリッド内の1つのサブプロットに対応します。（注：サブプロットグリッドには正確に "rows" 倍の "cols" セルがあります。）
      * 空白のサブプロットセル（または列/行スパンを超えるために）には missing を使用します。
      * "specs" の各アイテムは `Spec` のインスタンスです。詳細については `Spec` のドキュメントを参照してください。
      * 注：サブプロット間のスペーシングを調整するには `horizontal_spacing` と `vertical_spacing` を使用します。
  * `insets`: インセット仕様。インセットはグリッドサブプロットに重ねられたサブプロットです。

      * "insets" の各アイテムは `Inset` のインスタンスです。詳細については `Inset` のドキュメントを参照してください。
  * `column_widths`: サブプロットの各列の相対幅の長さ `cols` の配列。値は内部で正規化され、全体の幅（パディングを除く）を列間で分配するために使用されます。
  * `row_heights`: サブプロットの各行の相対高さの長さ `rows` の配列。値は内部で正規化され、全体の高さ（パディングを除く）を行間で分配するために使用されます。
  * `column_titles`: 各列の最上部のサブプロットの上に配置するタイトルの長さ `cols` のリスト。
  * `row_titles`: 各サブプロットの行の右側に配置するタイトルの長さ `rows` のリスト。
  * `x_title`: 最下行のサブプロットの下に配置するタイトルで、水平に中央揃え。
  * `y_title`: 左列のサブプロットの左側に配置するタイトルで、垂直に中央揃え。
