```
NormalInverseChisq(μ, σ2, κ, ν)
```

正規-χ^-2分布は、平均と分散が不明な正規分布の共役事前分布です。パラメータは次の通りです：

  * μ: 期待される平均
  * σ2 > 0: 期待される分散
  * κ ≥ 0: 平均の信頼度
  * ν ≥ 0: 分散の信頼度

パラメータは、平均と分散が不明な正規分布の事前分布として使用されるときに自然な解釈を持ちます：μとσ2は期待される平均と分散であり、κとνはそれぞれの信頼度の度合い（「擬似カウント」で表現）です。解釈可能なパラメータが重要な場合、これは共役事前分布の少し便利なパラメータ化となります。

次のパラメータを持つ`NormalInverseGamma`分布と同等です：

  * m0 = μ
  * v0 = 1/κ
  * shape = ν/2
  * scale = νσ2/2

Murphy "Gaussian distributionの共役ベイズ分析"に基づいています。
