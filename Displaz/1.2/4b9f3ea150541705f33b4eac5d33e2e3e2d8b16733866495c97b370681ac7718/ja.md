現在のプロットに3Dポイントまたはラインを追加します。

```
  plot3d([plotobj,] position; attr1=value1, ...)
```

`position`配列は、3xN配列または`FixedVector{3}`の`Vector`として指定されたN個の頂点位置のセットである必要があります。`plotobj`引数はオプションで、データを送信するプロットウィンドウを決定します。使用しない場合、データは`current()`によって返されるプロットウィンドウに送信されます。

### データセット属性

次の属性は、各`plot3d`呼び出しでデータセットに添付できます：

  * `label` - データセットをラベル付けする文字列

### 頂点属性

各ポイントには、視覚的表現を制御し、検査のためにポイントにタグを付けるための一連の頂点属性を添付できます。任意のカスタム情報に対して`Float32`値の`Vector`を渡すことができますが、以下はデフォルトシェーダーによってサポートされています：

  * `color`       - 各ポイントの色または色のベクトル; 以下に指定方法を示します。
  * `intensity`   - 各ポイントの強度のベクトル（0と1の間）
  * `markersize`  - 頂点マーカーサイズ
  * `markershape` - 頂点マーカー形状のベクトル。形状は、Charまたは0から4の間の数値識別子で表すことができます：

```
                    sphere - '.' 0    square - 's' 1
                    circle - 'o' 2    times  - 'x' 3
                    cross  - '+' 4
```

#### 色の指定

色は次の3つの方法のいずれかで提供できます：

  * ColorTypesパッケージの型のインスタンスとして、例えば、HSV(180,1,1)。これらはRGBコンストラクタを使用してRGBに変換されます。
  * 0.0から1.0の間の赤、緑、青の3要素の`Vector`として。
  * Matlabのような単一の色の文字名文字列または`Char`を使用して。サポートされているのは赤、緑、青、シアン、マゼンタ、黄、黒、白で、すべて色名の最初の文字で略されますが、黒の場合は「k」が使用されます。

各ポイントごとに色を`Color`サブタイプの`Vector`または赤、緑、青を行に持つ3xN行列として提供できます。

### ポイントのプロット

原点からの距離によって色を決定し、ランダムなマーカー形状で10000のランダムポイントをプロットするには：

```
  P = randn(3,10000)
  c = 0.5./sumabs2(P,1) .* [1,0,0]
  plot3d(P, color=c, markershape=rand(1:4,10000))
```

### ラインのプロット

10000のランダム頂点間の区分線形曲線をプロットするには：

```
  plot3d(randn(3,10000), markershape="-")
```

ラインをプロットする際には、`linebreak`キーワード引数を使用して位置配列を複数のラインセグメントに分割できます。ラインブレーク配列の各インデックスは、ラインセグメントの初期インデックスです。
