# LatexSVG

これは、すべての主要なLaTeXディストリビューションにバンドルされた`dvisvgm`実行可能ファイルを利用して、LaTeXをSVGとしてレンダリングする小さなパッケージです。カスタムプリアンブルを設定でき、`xelatex`、`pdflatex`、または`lualatex`エンジンによって有効と見なされる任意のLaTeXコードをレンダリングすることができるため、JavaScriptベースの代替手段よりもはるかに柔軟で強力です。

詳細については[ドキュメント](https://ehbaozi.github.io/LatexSVG.jl/dev/)をお読みください。
