```
init_sim!(s::AKM; t_end=1.0, stiffness_factor=0.5, delta=0.001, upwind_dir=-pi/2, prn=false)
```

モデルの積分器を初期化します。

パラメータ:

  * s:     抽象凧モデルのインスタンス
  * t_end: シミュレーションの終了時間; 通常は必要ありません
  * stiffness_factor: 初期化中にテザーの剛性に適用される係数
  * delta: 定常状態計算中のテザーの初期伸び
  * upwind_dir: ラジアンでの上流方向、風が吹いてくる方向。ゼロは北; 時計回りが正。デフォルト: -pi/2、西からの風。
  * prn: trueに設定すると、詳細なソルバー結果を印刷します

返り値: DAE積分器のインスタンス。
