```
init_sim!(s::RamAirKite, measure::Measurement; prn=true, precompile=false) -> Nothing
```

カイトパワーシステムモデルを初期化します。

現在の構成に対するシリアライズされたモデルが存在する場合、それをロードし、状態変数のみを更新します。そうでない場合は、ゼロから新しいモデルを作成します。

# 高速パス（シリアライズされたモデルが存在する場合）:

1. ディスクから既存のODEProblemをロードします
2. `reinit!`を呼び出して状態変数を更新します
3. 現在の測定値で積分器を設定します

# 遅いパス（シリアライズされたモデルが存在しない場合）:

1. すべての方程式を持つ記号的MTKシステムを作成します
2. システム方程式を簡略化します
3. ODEProblemを作成し、ディスクにシリアライズします
4. 高速パスに進みます

# 引数

  * `s::RamAirKite`: カイトシステムの状態オブジェクト
  * `measure::Measurement`: 初期状態の測定値
  * `prn::Bool=true`: 進行状況情報を印刷するかどうか
  * `precompile::Bool=false`: 事前コンパイルのために問題を構築するかどうか

# 戻り値

`Nothing`
