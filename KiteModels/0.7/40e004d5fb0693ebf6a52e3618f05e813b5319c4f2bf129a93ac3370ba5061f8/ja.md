```
reinit!(s::RamAirKite; prn=true, precompile=false) -> Nothing
```

既存のカイトパワーシステムモデルを新しい状態値で再初期化します。

この関数は以下の操作を行います：

1. まだ積分器が存在しない場合：

      * ディスクからシリアライズされたODEProblemをロードします
      * 新しいODE積分器を初期化します
      * システムのゲッター/セッター関数を生成します
2. 測定データをクォータニオンの向きに変換します
3. 新しい位置で点質量システムを初期化します
4. すべての状態変数の初期値を設定します
5. ODE積分器を再初期化します
6. 線形化された空力モデルを更新します

これは`init!`よりも効率的で、既存のモデル構造を再利用し、現在の`measure`に一致するように状態変数のみを更新します。

# 引数

  * `s::RamAirKite`: カイトパワーシステムの状態オブジェクト
  * `prn::Bool=true`: 進行状況情報を印刷するかどうか

# 戻り値

  * `Nothing`

# 例外

  * `ArgumentError`: シリアライズされた問題が存在しない場合（最初に`init_sim!`を実行してください）
