```
log_variance_est(x::Array; m::Int=200)
```

ログ分散を使用して長期記憶の推定を計算します。

# 引数

  * `x::Array`: 時系列データ。

# 出力

  * `d_var::Real`: ログ分散プロットの線形回帰の傾きであるベータを用いて計算された推定長期記憶パラメータ (beta+1)/2。

# オプション引数

  * `m::Int`: 時系列の分割数。デフォルトは20。m ≈ length(x)/2の値が推奨されます。

# 注意事項

この関数は、長期記憶パラメータを推定するためにログ分散に対して線形回帰法を使用します。

# 例

```julia
julia> log_variance_est(randn(100,1); m = 40)
```
