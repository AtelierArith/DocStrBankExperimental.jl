```
rescaled_range_est(x::Array; k::Int = 20)
```

時系列のリスケールドレンジ (R/S) 統計を使用して、長期記憶パラメータを推定します。

# 引数

  * `x::Array`: 時系列。

# 出力

  * `d::Real`: 推定された長期記憶パラメータ。

# オプション引数

  * `k::Int`: 時系列の分割数。デフォルトは20。

# 注意事項

この関数は、リスケールドレンジの対数に対して線形回帰法を使用して長期記憶パラメータを推定します。ハースト係数は、式 H = d + 1/2 によって長期記憶パラメータ d に関連しています。

# 例

```julia
julia> rescaled_range_est(randn(100))
```
