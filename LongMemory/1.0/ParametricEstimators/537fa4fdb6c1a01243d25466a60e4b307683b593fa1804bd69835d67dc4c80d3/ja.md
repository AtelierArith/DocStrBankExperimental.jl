```
csa_mle_est(x::Array)
```

データ `x` に基づいて、CSAプロセスのパラメータ `p` と `q` の最大尤度推定値とCSAプロセスの標準偏差を計算します。

# 引数

  * `x::Array`: データ。

# 出力

  * `p::Real`: CSAプロセスの最初のパラメータの最大尤度推定値。
  * `q::Real`: CSAプロセスの2番目のパラメータの最大尤度推定値。
  * `σ::Real`: CSAプロセスの標準偏差の最大尤度推定値。

# 注意事項

この関数は、対数尤度関数を最小化するために `Optim` パッケージを使用します。

# 例

```julia
julia> csa_mle_est(randn(100,1))
```
