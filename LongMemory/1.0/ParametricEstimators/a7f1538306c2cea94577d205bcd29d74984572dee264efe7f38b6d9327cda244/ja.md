```
csa_var_matrix(T::Int, d::Real)
```

CSAプロセスの自己共分散行列を、遅れ0, 1, ..., `T-1`でパラメータ`p`と`q`を用いて構築します。

# 引数

  * `T::Int`: 計算する遅れの数。
  * `p::Real`: CSAプロセスの最初のパラメータ。
  * `q::Real`: CSAプロセスの2番目のパラメータ。

# 出力

  * `V::Array`: 遅れ0, 1, ..., `T-1`でパラメータ`p`と`q`を持つCSAプロセスの自己共分散行列。

# 例

```julia
julia> csa_var_matrix(10, 1.4, 1.8)
```
