```
fi_llk(d::Real, x::Array)
```

パラメータ `d` に基づく分数差分過程の対数尤度をデータ `x` に対して計算します。

# 引数

  * `d::Real`: 分数差分パラメータ。
  * `x::Array`: データ。

# 出力

  * `llk::Real`: データ `x` に対してパラメータ `d` の分数差分過程の対数尤度。

# 注意事項

この関数は、データ `x` に対してパラメータ `d` の分数差分過程の集中対数尤度関数を計算します。

# 例

```julia
julia> fi_llk(0.4, randn(100,1))
```
