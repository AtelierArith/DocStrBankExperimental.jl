```
fi_var_vals(T::Int,d::Real)
```

パラメータ `d` を持つ分数差分過程の自己共分散関数をラグ 0, 1, ..., `T-1` で計算します。

# 引数

  * `T::Int`: 計算するラグの数。
  * `d::Real`: 分数差分パラメータ。

# 出力

  * `vars::Array`: パラメータ `d` を持つ分数差分過程の自己共分散関数をラグ 0, 1, ..., `T-1` で。

# 注意事項

この関数は分数差分過程の自己共分散関数の再帰的な公式を使用します。

# 例

```julia
julia> fi_var_vals(10, 0.4)
```
