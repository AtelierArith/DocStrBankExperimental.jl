```
csa_llk(p::Real, q::Real, x::Array)
```

CSAプロセスのパラメータ`p`と`q`に基づいて、データ`x`に対する対数尤度を計算します。

# 引数

  * `p::Real`: CSAプロセスの最初のパラメータ。
  * `q::Real`: CSAプロセスの2番目のパラメータ。
  * `x::Array`: データ。

# 出力

  * `llk::Real`: データ`x`に基づくパラメータ`p`と`q`を持つCSAプロセスの対数尤度。

# 注意事項

この関数は、データ`x`に基づくパラメータ`p`と`q`を持つCSAプロセスの集中対数尤度関数を計算します。

# 例

```julia
julia> csa_llk(1.4, 1.8, randn(100,1))
```
