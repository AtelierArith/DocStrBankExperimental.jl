```
fi_forecast_plot(x::Array, h::Int, d::Real, σ::Real)
```

長期記憶時系列の予測を分数差分法を用いてプロットします。

# 引数

  * `x::Array`: 時系列。
  * `h::Int`: 予測する期間の数。
  * `d::Real`: 分数差分パラメータ。
  * `σ::Real`: 予測誤差の標準偏差。
  * `p1::Plot`: 元の時系列と信頼区間を持つ予測のプロット。

# 注意事項

複数のディスパッチを使用して、信頼区間の有無にかかわらず時系列の予測をプロットします。プロットは `fi_forecast` 関数を呼び出すことで行われます。

# 例

```julia
julia> fi_forecast_plot(fi_gen(100,0.2), 10, 0.2, 1.0)
```
