```
csa_forecast(x::Array, h::Int, p::Real, q::Real, σ::Real)
```

CSA法を使用して長期記憶時系列の予測を計算します。

# 引数

  * `x::Array`: 時系列。
  * `h::Int`: 予測する期間の数。
  * `p::Real`: CSAプロセスのパラメータp。
  * `q::Real`: CSAプロセスのパラメータq。
  * `σ::Real`: 予測誤差の標準偏差。

# 出力

  * `xfor::Array`: 時系列の予測を行列として返します。最初の列が予測、2列目が下側信頼帯、3列目が上側信頼帯です。最初のT要素は元の時系列です。

# 注意事項

複数ディスパッチを使用して、信頼帯の有無にかかわらず時系列の予測を計算します。

# 例

```julia
julia> csa_forecast(csa_gen(100,1.4,1.4), 10, 1.4, 1.4, 1.0)
```
