```
fi_forecast(x::Array, h::Int, d::Real, σ::Real)
```

長期記憶時系列の予測を分数差分法を用いて計算します。

# 引数

  * `x::Array`: 時系列。
  * `h::Int`: 予測する期間の数。
  * `d::Real`: 分数差分パラメータ。
  * `σ::Real`: 予測誤差の標準偏差。

# 出力

  * `xfor::Array`: 時系列の予測を行列として返します。最初の列が予測、2列目が下側信頼帯、3列目が上側信頼帯です。最初の T 要素は元の時系列です。

# 注意事項

複数ディスパッチを使用して、信頼帯の有無にかかわらず時系列の予測を計算します。

# 例

```julia
julia> fi_forecast(fi_gen(100,0.2), 10, 0.2)
```
