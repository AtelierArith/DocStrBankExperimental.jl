```
fi_forecast(x::Array, h::Int, d::Real)
```

長期記憶時系列の予測を分数差分法を使用して計算します。

# 引数

  * `x::Array`: 時系列。
  * `h::Int`: 予測する期間の数。
  * `d::Real`: 分数差分パラメータ。

# 出力

  * `xfor::Array`: 時系列の予測を列ベクトルとして返します。最初の T 要素は元の時系列です。

# 注意事項

複数のディスパッチを使用して、信頼区間の有無にかかわらず時系列の予測を計算します。

# 例

```julia
julia> fi_forecast(fi_gen(100,0.2), 10, 0.2)
```
