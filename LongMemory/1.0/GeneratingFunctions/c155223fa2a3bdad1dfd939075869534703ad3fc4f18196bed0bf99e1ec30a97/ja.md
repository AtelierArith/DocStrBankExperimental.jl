```
     fi*survival*probs(N::Int,d)

エラーデュレーションモデルのサバイバル確率を生成します（Parke (1999) に従う）。

# 引数

  * `N::Int`: 時系列の長さ
  * `d::Float64`: 分数差分パラメータ

# 出力

  * `p::Vector`: サバイバル確率

# 注意事項

サバイバル確率は、数値オーバーフローを避けるために再帰的な式 `p_{t+1} = p_t * (t + d - 1) / (t + 1 - d)` を使用して計算されます。

# 例

```

julia-repl julia> fi*survival*probs(100,0.4) ```
