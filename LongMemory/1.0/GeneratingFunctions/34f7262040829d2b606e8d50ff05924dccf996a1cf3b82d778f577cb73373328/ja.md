```
csadiff(x,p,q)
```

移動平均表現を使用して横断的集約プロセスの長期記憶を生成します。詳細については[Vera-Valdes(2021)](https://www.mdpi.com/2225-1146/9/4/39)を参照してください。

# 引数

  * `x::Vector`: 時系列
  * `p::Float64`: 横断的集約プロセスの最初のパラメータ
  * `q::Float64`: 横断的集約プロセスの2番目のパラメータで、分数差分パラメータ `d` に関連し、`q = 2(1-d)` となります。

# 出力

  * `dx::Vector`: 長期記憶を持つ時系列

# 注意事項

`q` は横断的集約プロセスの長期記憶パラメータを決定します。関係式 `q = 2(1-d)` が成り立ち、ここで `d` は分数差分パラメータです。自己回帰式を使用して、横断的集約プロセスの移動平均表現の係数を効率的に計算します。

# 例

```julia-repl
julia> csadiff(randn(100,1),1.2,1.4)
```
