```
csa_gen(T::Int,N::Int,p,q;t=0.01;μ=0,σ=1)
```

長期記憶パラメータ `q` と長さ `T` を持つ時系列を生成します。これは、Granger (1980) による 'N' の AR(1) プロセスの横断的集約を使用しています。

# 引数

  * `T::Int`: 時系列の長さ
  * `N::Int`: AR(1) プロセスの数
  * `p::Float64`: 横断的集約プロセスの最初のパラメータ
  * `q::Float64`: 横断的集約プロセスの2番目のパラメータで、分数差分パラメータ `d` に関連しており、`q = 2(1-d)` です。

# オプション引数

  * `t::Float64`: テーパーの長さ
  * `μ::Float64`: 時系列の平均
  * `σ::Float64`: 時系列の標準偏差

# 注意事項

引数に 'N' が含まれている場合、有限サンプルプロセスを生成するために多重ディスパッチが使用されます。

# 出力

  * `x::Vector`: 長期記憶を持つ時系列

# 例

```julia-repl
julia> csa_gen(100,100,1.2,1.4)
```
