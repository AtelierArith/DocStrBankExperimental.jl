```
csa_gen(T::Int,p,q;μ=0,σ=1)
```

長期記憶パラメータ `q` と長さ `T` を持つ時系列を、横断的集約プロセスを使用して生成します。詳細は [Vera-Valdes(2021)](https://www.mdpi.com/2225-1146/9/4/39) を参照してください。

# 引数

  * `T::Int`: 時系列の長さ
  * `p::Float64`: 横断的集約プロセスの最初のパラメータ
  * `q::Float64`: 横断的集約プロセスの2番目のパラメータで、分数差分パラメータ `d` に関連しており、`q = 2(1-d)` です。

# オプション引数

  * `μ::Float64`: 時系列の平均
  * `σ::Float64`: 時系列の標準偏差

# 出力

  * `x::Vector`: 長期記憶を持つ時系列

# 例

```julia-repl
julia> csa_gen(100,1.2,1.4)
```
