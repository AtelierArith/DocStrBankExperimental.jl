# LongMemory.jl

このパッケージは、長期記憶時系列モデルを生成、推定、予測するための関数を提供します。さらに、パッケージは、古典的な診断プロットを含むさまざまな推定器と予測器のためのプロット機能を提供します。

## 著者

[J. Eduardo Vera-Valdés](https://everval.github.io/)

## ドキュメント

パッケージのドキュメントは[こちら](https://everval.github.io/LongMemory.jl/)で見つけることができます。

## クイックスタート

このパッケージには、古典的なナイル川と北半球の温度データセットが含まれています。これらには、次のように入力することでアクセスできます。

```julia
NileData()
```

または

```julia
NHTempData()
```

それぞれ。パッケージには、データと診断プロットをプロットするための関数*NileDataPlot()*と*NHTempDataPlot()*も含まれています。同様のことは、*LMPlot()*関数を使用して任意のデータに対しても行うことができます。
