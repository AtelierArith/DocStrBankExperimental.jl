```
NHTempDataPlot()
```

1850年1月から2023年9月までの北半球の温度データの月次プロットを返します。

# 出力

  * `pp::Plot`: ナイル川データのプロット。
  * 4つのプロットが2x2のグリッドで返されます：

      * 最初のプロットは北半球の温度データの時系列です。
      * 2番目のプロットは、`autocorrelation_plot`関数を使用した北半球の温度データの自己相関関数です。
      * 3番目のプロットは、`periodogram_plot`関数を使用した北半球の温度データの周期グラムです。
      * 4番目のプロットは、`variance_plot`関数を使用した北半球の温度データの分散プロットです。

# 注意事項

この関数は、データをロードするために`NHTempData()`関数を使用します。

# 例

```julia
julia> NHTempDataPlot()
```
