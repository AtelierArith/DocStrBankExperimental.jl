```
NileDataPlot()
```

622年から1284年までの年間ナイル川の最小データのプロットを返します。

# 出力

  * `pp::Plot`: ナイル川データのプロット。
  * 2x2のグリッドに4つのプロットが返されます：

      * 最初のプロットはナイル川データの時系列です。
      * 2番目のプロットは、`autocorrelation_plot`関数を使用したナイル川データの自己相関関数です。
      * 3番目のプロットは、`periodogram_plot`関数を使用したナイル川データの周期グラムです。
      * 4番目のプロットは、`variance_plot`関数を使用したナイル川データの分散プロットです。

# 注意事項

この関数は、データをロードするために`NileData()`関数を使用します。

# 例

```julia
julia> NileDataPlot()
```
