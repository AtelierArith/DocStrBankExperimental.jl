```
MDN(; mixtures=5, layers=[128], η=1e-3, epochs=1, batchsize=32)
```

与えられたハイパーパラメータを使用してMDNモデルを定義します。

# パラメータ

  * `mixtures`: 条件付き分布を推定するために使用するガウス混合の数（デフォルト=5）。
  * `layers`: 各隠れ層のノード数を示すベクトル（デフォルト=[128,]）。
  * `η`: モデルをトレーニングする際に使用する学習率（デフォルト=1e-3）。
  * `epochs`: モデルをトレーニングするエポック数（デフォルト=1）。
  * `batchsize`: トレーニング中に使用するバッチサイズ（デフォルト=32）。
