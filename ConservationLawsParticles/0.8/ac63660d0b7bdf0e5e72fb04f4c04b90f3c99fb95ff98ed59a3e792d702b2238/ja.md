```
pwc_densities(xs::AbstractVector{<:Real}...)
pwc_densities(xs::Tuple{Vararg{AbstractVector{<:Real}}})
```

長さ `l₁, …, lₙ` の `n` 個の粒子のファミリーのリストを与えると、`s` 番目のエントリが `n×2×lₛ` 配列 `densₛ` である `n`-タプルを返します。

この配列には、`s` 番目の種の粒子の周りのすべての種の密度に関する情報が含まれています。この配列は `[o,side,i]` でインデックス付けされており、ここで `o∈{1, …, n}` は知りたい他の種のインデックス、`side` は左の密度を知りたい場合は `1`、右の密度を知りたい場合は `2`、`i∈{1, …, lₛ}` は粒子のインデックスです。

言い換えれば、`densₛ[o,side,i]` は `s` 番目の種の `i` 番目の粒子の左側（`side=1`）または右側（`side=2`）の `o` 番目の種の密度です。

インプレースバージョンについては [`pwc_densities!`](@ref) を参照してください。

# 例

```jldoctest
julia> pwc_densities([0,1,2], [0,2,3])
([0.0 0.5; 0.0 0.25;;; 0.5 0.5; 0.25 0.25;;; 0.5 0.0; 0.25 0.5], [0.0 0.5; 0.0 0.25;;; 0.5 0.0; 0.25 0.5;;; 0.0 0.0; 0.5 0.0])
```
