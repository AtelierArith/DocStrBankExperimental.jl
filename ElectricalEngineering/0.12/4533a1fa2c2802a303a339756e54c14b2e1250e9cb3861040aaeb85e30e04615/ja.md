# 関数呼び出し

`∥(z...)`

`∥`記号を自動補完するには、`tab`キーを押します。

# 説明

これはインピーダンスの並列接続を計算します。オプションとして、計算はUnitfulモジュールの単位を使用して行うことができます。

**重要な注意:** 演算子の優先順位が（低い）関係演算子の優先順位と等しいため、`∥`演算子は括弧内で使用してください。または、誤計算を避けるために引数を持つ関数呼び出しを使用してください。

関数`par`と`∥`は冗長な実装です。

# 変数

`z` インピーダンスのベクトル

# 例

```julia
julia> using Unitful,Unitful.DefaultSymbols,ElectricalEngineering
julia> ∥(4,6)
2.4
julia> ∥(4Ω,6Ω)
2.4 Ω
julia> 2Ω+(4Ω∥6Ω) # 括弧は常に必要です
4.4 Ω
julia> 2Ω+4Ω∥6Ω # は (2Ω+4Ω)∥6Ω と等しく、通常は意図されていません
3.0 Ω
julia> (2.4Ω∥(4Ω∥6Ω))
1.2 Ω
julia> ∥(4Ω,6Ω,2.4Ω)
1.2 Ω

```

`Ω`記号を自動補完するには、`tab`キーを押します。
