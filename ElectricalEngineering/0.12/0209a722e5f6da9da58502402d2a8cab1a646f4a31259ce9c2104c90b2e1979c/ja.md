# 関数呼び出し

`mmfwinding(w, i)`

# 説明

巻線レイアウトと多相巻線の実際の電流に基づいて、磁気動圧力分布を計算します。

# 入力変数

`w[:,:]` スロットごとに等しい数の巻線を持つ巻線配列で、ここで

  * `size(w,1)` = 層の数、例：1または2
  * `size(w,2)` = スロットの数

`i[:]` 実際の位相電流、例：3つの位相のために、1にスケーリングされています。

# 出力変数

`mmf[:,:]` 磁気動圧力で、ここで

  * `size(mmf,1)` = 位相の数
  * `size(mmm,1)` = スロットの数
