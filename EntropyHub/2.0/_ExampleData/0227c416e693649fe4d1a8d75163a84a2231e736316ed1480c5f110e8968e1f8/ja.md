Data = ExampleData(SigName::String)

特定の特性を持つサンプルデータの時系列をインポートし、さまざまなエントロピー手法のパフォーマンスを評価するためのベンチマークとして一般的に使用されます。ExampleData()によって返されるデータセットは、www.EntropyHub.xyzやその他の場所で提供されるドキュメントの例で使用されます。***注意*** ExampleData()は、必要なデータセットをダウンロードしてインポートするためにインターネット接続を必要とします！

`Data`は、文字列入力`SigName`に対応してインポートされたサンプルデータセットであり、次のいずれかの文字列である可能性があります：

# 引数：

`SigName`   -  

```
        `uniform`          - 範囲 [0 1] の一様分布のランダム数列、N = 5000
        `randintegers`     - 範囲 [1 8] のランダムに分布した整数列、N = 4096
        `gaussian`         - 正規分布の数列 [平均: 0, 標準偏差: 1]、N = 5000
        `henon`            - ヘノンアトラクタのXおよびY成分 [α: 1.4, β: .3, Xo = 0, Yo = 0]、N = 4500
        `lorenz`           - ローレンツアトラクタのX、Y、およびZ成分 [σ: 10, β: 8/3, ρ: 28, Xo = 10, Yo = 20, Zo = 10]、N = 5917
        `chirp`            - チャープ信号 (f0 = .01, t1 = 4000, f1 = .025)、N = 5000
        `uniform2`         - 範囲 [0,1] の二つの一様分布のランダム数列、N = 4096
        `gaussian2`        - 二つの正規分布の数列 [平均: 0, 標準偏差: 1]、N = 3000
        `randintegers2`    - 範囲 [1 8] の二つの一様分布の擬似ランダム整数列、N = 3000
        `uniform_Mat`      - 範囲 [0 1] の一様分布のランダム数の行列、N = 50 x 50
        `gaussian_Mat`     - 正規分布の数の行列 [平均: 0, 標準偏差: 1]、N = 60 x 120
        `randintegers_Mat` - 範囲 [1 8] のランダムに分布した整数の行列、N = 88 x 88
        `mandelbrot_Mat`   - 範囲 [0 255] の値を持つマンデルブロ集合のフラクタル画像を表す行列、N = 92 x 115
        `entropyhub_Mat`   - 範囲 [0 255] の値を持つEntropyHubロゴを表す行列、N = 127 x 95
                     
これらのグレイン処理に関する詳細情報は、`EntropyHubガイド <https://github.com/MattWillFlood/EntropyHub/blob/main/EntropyHub%20Guide.pdf>`_を参照してください。
```
