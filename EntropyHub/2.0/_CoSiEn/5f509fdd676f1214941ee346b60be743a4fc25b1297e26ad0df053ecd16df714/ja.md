```
   CoSi, Bm = CoSiEn(Sig)
```

デフォルトのパラメータを使用してデータシーケンス（`Sig`）から推定されたコサイン類似度エントロピー（`CoSi`）と対応するグローバル確率を返します：埋め込み次元 = 2、時間遅延 = 1、角度しきい値 = .1、対数 = 基数 2、

```
   CoSi, Bm = CoSiEn(Sig::AbstractArray{T,1} where T<:Real; m::Int=2, tau::Int=1, r::Real=.1, Logx::Real=2, Norm::Int=0)
```

指定された「キーワード」引数を使用してデータシーケンス（`Sig`）から推定されたコサイン類似度エントロピー（`CoSi`）を返します：

# 引数：

`m`     - 埋め込み次元、1より大きい整数   

`tau`   - 時間遅延、正の整数    

`r`     - 角度しきい値、範囲 [0 < r < 1] の値   

`Logx`  - 対数の基数、正のスカラー（自然対数の場合は0を入力） 

`Norm`  - `Sig` の正規化、次の整数のいずれか：    

```
       [0]  正規化なし - デフォルト
       [1]  中央値（`Sig`）を取り除いて `Sig` を正規化
       [2]  平均（`Sig`）を取り除いて `Sig` を正規化
       [3]  SD（`Sig`）に対して `Sig` を正規化
       [4]  `Sig` の値を範囲 [-1 1] に正規化
```
