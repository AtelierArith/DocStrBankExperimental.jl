```
  MSx, Sn, CI = hMSEn(Sig, Mobj)
```

階層ツリーの各ノードで計算されたエントロピー値のベクトル（`MSx`）、各スケールでのすべてのノードの平均エントロピー値（Sn）、および階層ツリーの複雑性指数（`CI`）（すなわち、データシーケンス（`Sig`）に対する合計（Sn））を、マルチスケールオブジェクト（`Mobj`）によって指定されたパラメータを使用して3つの時間スケール（デフォルト）で返します。 MSxのエントロピー値は、ルートノード（S.00）からスケールTのNthサブノード（S.TN）まで順序付けられています：すなわち、S.00、S.10、S.11、S.20、S.21、S.22、S.23、S.30、S.31、S.32、S.33、S.34、S.35、S.36、S.37、S.40、...、S.TN。 Snの平均エントロピー値も同様に順序付けられ、ルートノードの値が最初に与えられます：すなわち、S0、S1、S2、...、ST

```
  MSx, Sn, CI = hMSEn(Sig::AbstractArray{T,1} where T<:Real, Mobj::NamedTuple; 
                          Scales::Int=3, RadNew::Int=0, Plotx::Bool=false)
```

階層ツリーの各ノードで計算されたエントロピー値のベクトル（`MSx`）、各スケールでのすべてのノードの平均エントロピー値（Sn）、およびデータシーケンス（`Sig`）に対する全体の階層ツリーの複雑性指数（CI）を、以下の「キーワード」引数を使用して返します：

# 引数：

`Scales`   - 時間スケールの数、1より大きい整数（デフォルト：3）                各スケール（T）で、2^(T-1)ノードのエントロピーが推定されます。

`RadNew`   - 半径再スケーリング方法、範囲[1 4]の整数。                `Mobj`によって指定されたエントロピーが`SampEn`または`ApEn`の場合、                 `RadNew`はツリーの各ノードで半径閾値を更新できるようにします。                 `Mobj`によって指定された半径値（`r`）がある場合、                これは再スケーリング係数となり、そうでない場合は                0.2（デフォルト）に設定されます。 `RadNew`の値は、次のいずれかの方法を指定します：

```
           [1]    標準偏差          - r*std(Xt)

           [2]    分散              - r*var(Xt)

           [3]    平均絶対偏差     - r*mean_ad(Xt)

           [4]    中央絶対偏差     - r*med_ad(Xt,1)
```

`Plotx`    - `Plotx` == true の場合、各時間スケールでの平均エントロピー                 値（すなわち、マルチスケールエントロピー曲線）のプロットと、                 階層ツリー分解における各ノードのエントロピー値を示す階層グラフを                 返します。（デフォルト：false）
