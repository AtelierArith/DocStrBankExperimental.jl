# 拡張ヘルプ

間隔が1の均一なノットを仮定すると、ここで実装されている`i`番目の二次スプラインは次のように定義されます：

```
y_i(x) = cm p(x-i) + c q(x) + cp p(1-(x-i))
```

ここで

```
p(δx) = (δx - 1)^2 / 2
q(δx) = 3/4 - δx^2
```

および`X ∈ {m,_,p}`の`cX`の値は事前フィルタリングされた係数です。

将来の参照のために、これは次の多項式に展開されます：

```
y_i(x) = cm * 1/2 * (x-i-1)^2 + c * (3/4 - x + i)^2 + cp * 1/2 * (x-i)^2
```

境界条件を導出する際には、導関数`y_1'(x-1)`および`y_1''(x-1)`を使用します。
