# 拡張ヘルプ

間隔が1の均一なノットを仮定すると、ここで実装されている`i`番目の線形bスプラインは次のように定義されます。

```
y_i(x) = c p(x) + cp p(1-x)
```

ここで

```
p(δx) = x
```

および値`cX`は`X ∈ {_, p}`の係数です。

線形bスプラインは自然に補間を行い、事前フィルタリングを必要としません。したがって、境界条件を提供する必要はありません。

また、実装は一般的なbスプラインのために構築されたフレームワークを再利用するために若干異なりますが、結果として得られる補間関数は、隣接するデータポイントのペアを結ぶ片側線形関数に過ぎません。
