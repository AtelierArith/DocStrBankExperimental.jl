# 拡張ヘルプ

間隔が1の均一なノットを仮定すると、ここで実装されている`i`番目の三次スプラインの部分は次のように定義されます。

```
y_i(x) = cm p(x-i) + c q(x-i) + cp q(1- (x-i)) + cpp p(1 - (x-i))
```

ここで

```
p(δx) = 1/6 * (1-δx)^3
q(δx) = 2/3 - δx^2 + 1/2 δx^3
```

および値`cX`は`X ∈ {m, _, p, pp}`の事前フィルタリングされた係数です。

将来の参照のために、これは次の多項式に展開されます。

```
y_i(x) = 1/6 cm (1+i-x)^3 + c (2/3 - (x-i)^2 + 1/2 (x-i)^3) +
         cp (2/3 - (1+i-x)^2 + 1/2 (1+i-x)^3) + 1/6 cpp (x-i)^3
```

境界条件を導出する際には、導関数`y_0'(x)`および`y_0''(x)`を使用します。
