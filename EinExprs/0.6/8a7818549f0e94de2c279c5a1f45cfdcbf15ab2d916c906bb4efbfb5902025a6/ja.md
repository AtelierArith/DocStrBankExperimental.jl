```
Exhaustive(; outer = false)
```

徹底的な収縮経路最適化器。最適な収縮経路を見つけることを保証しますが、大きなコストがかかります。

# キーワード

  * `outer` は最適な収縮経路を探す際に外積（別名テンソル積）を考慮するよう指示します。内積のみを考慮する場合に比べてほとんど利点がないため、デフォルトでは `false` です。

!!! warning
    `outer = true` の機能はまだ実装されていません。


# 実装

アルゴリズムは、`outer = true` の場合は $\mathcal{O}(n!)$ の時間計算量を持ち、`outer = false` の場合は $\mathcal{O}(\exp(n))$ です。
