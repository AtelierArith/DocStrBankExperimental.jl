`Ket`は*量子波動関数*を表し、数学的には複素数の列ベクトルに相当します。Ketのノルムは常に1であるべきです。$n=2$のキュービット数を持つシステムを表す`Ket`は$2^n$の状態を持ちます。慣例として、キュービット1は最も左のビットであり、その後に続くすべてのキュービットがあります。したがって、2キュービットの`Ket`は4つの複素数値の係数$a_{ij}$を持ち、それぞれが状態$\left|ij\right\rangle$に対応し、次の順序で表されます：

$$
\psi = \begin{bmatrix}
    a_{00}  \\
    a_{10}  \\
    a_{01}  \\
    a_{11}  \\
    \end{bmatrix}.
$$

# 例

Ketは、`fock`基底のような事前構築された基底を使用して初期化できます。この関数の詳細については[`fock`](@ref)を参照してください。

```jldoctest
julia> ψ = fock(2, 4)
4-element Ket{ComplexF64}:
0.0 + 0.0im
0.0 + 0.0im
1.0 + 0.0im
0.0 + 0.0im


```

好ましい方法ではありませんが、列ベクトルを初期化子として渡すことによって、Ketオブジェクトを直接構築することもできます。

```jldoctest
julia> using Snowflurry

julia> ψ = Ket([1.0; 0.0; 0.0; 0.0])
4-element Ket{ComplexF64}:
1.0 + 0.0im
0.0 + 0.0im
0.0 + 0.0im
0.0 + 0.0im


```
