Read `prefix_r.dat`.

# Return

  * `Rvectors`: $\mathbf{R}$-vectors on which operators are defined
  * `r_x`: $x$-component of position operator
  * `r_y`: $y$-component of position operator
  * `r_z`: $z$-component of position operator
  * `header`: the first line of the file
