```
create_tensor_h1_lagrange_basis(ceed, dim, ncomp, p, q, qmode)
```

テンソル積ラグランジュ基底を作成します。

# 引数:

  * `ceed`:  [`Basis`](@ref) が作成される [`Ceed`](@ref) オブジェクト。
  * `dim`:   要素のトポロジカル次元。
  * `ncomp`: フィールドコンポーネントの数（スカラー場の場合は1）。
  * `p`:     一次元のガウス・ロバットノードの数。結果の $Q_k$ 要素の多項式次数は $k=p-1$ です。
  * `q`:     一次元の積分点の数。
  * `qmode`: $q$ 積分点の分布（積分の精度に影響します）。
