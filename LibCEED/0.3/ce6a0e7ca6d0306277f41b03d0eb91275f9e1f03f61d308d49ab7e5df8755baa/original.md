```
EvalMode
```

Evaluation mode used in the specification of input and output fields for Q-functions, e.g. in [`@interior_qf`](@ref).

One of:

  * `EVAL_NONE`
  * `EVAL_INTERP`
  * `EVAL_GRAD`
  * `EVAL_DIV`
  * `EVAL_CURL`
  * `EVAL_WEIGHT`

See the [libCEED documentation](https://libceed.org/en/latest/api/CeedBasis/?highlight=EVAL_MODE#c.CeedEvalMode) for further information.
