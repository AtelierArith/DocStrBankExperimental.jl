```
EvalMode
```

Q関数の入力および出力フィールドの仕様で使用される評価モード、例えば[`@interior_qf`](@ref)で。

次のいずれか：

  * `EVAL_NONE`
  * `EVAL_INTERP`
  * `EVAL_GRAD`
  * `EVAL_DIV`
  * `EVAL_CURL`
  * `EVAL_WEIGHT`

詳細については、[libCEEDのドキュメント](https://libceed.org/en/latest/api/CeedBasis/?highlight=EVAL_MODE#c.CeedEvalMode)を参照してください。
