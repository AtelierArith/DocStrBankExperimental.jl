```
@note!(df, args...)
```

データフレームの列にノートを割り当てるには、`:col = note` 構文を使用します。これは、TablesMetadataTools.jlの `note!(df, col, note)` の短縮形です。

長い説明には `@note!` を使用します。主に見栄えを良くするための短い説明には `@label!` を使用します。

`df` を返し、`df` のノートが修正されます。

```julia-repl

julia> df = DataFrame(wage = 12);

julia> @note! df :wage = "
       2014年のUSDでの時間あたりの賃金は、IPUMSが提供するACSデータから取得されています。

       時間あたりの賃金は、時間給労働者に対して直接測定されます。サラリー労働者の場合、給与 / 労働時間に等しいです。
       ";

julia> printnotes(df)
Column: wage
────────────

2014年のUSDでの時間あたりの賃金は、IPUMSが提供するACSデータから取得されています。

時間あたりの賃金は、時間給労働者に対して直接測定されます。サラリー労働者の場合、給与 / 労働時間に等しいです。



julia> @note! df :wage = "賃金は99パーセンタイルで制限されています";

julia> printnotes(df)
Column: wage
────────────

2014年のUSDでの時間あたりの賃金は、IPUMSが提供するACSデータから取得されています。

時間あたりの賃金は、時間給労働者に対して直接測定されます。サラリー労働者の場合、給与 / 労働時間に等しいです。

賃金は99パーセンタイルで制限されています
```
