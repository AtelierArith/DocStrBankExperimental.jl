```
set_outgroup!(node::Node)
```

ノード `node` をそれが属する木の外群として設定します。これにより、`node` とその親の間に根ノードが導入されます。木が三分岐の根を持つ場合、これにより三分岐が解消されます（つまり、木を根付けます）。木がすでに二分岐している場合、これにより木が再根付けされます。（これら二つの機能は異なることをするため、分けるのが賢明かもしれません...すなわち、根付け/再根付け）。

```julia
julia> tr = nw"(A,(B,(C,D)),E);"
(A,(B,(C,D)),E);
julia> NewickTree.set_outgroup!(tr[2][1])
(((C,D),(A,E)),B);
```
