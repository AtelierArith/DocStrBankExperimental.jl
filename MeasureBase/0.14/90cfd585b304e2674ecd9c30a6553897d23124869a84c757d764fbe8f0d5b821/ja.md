もし

  * μ が `AbstractMeasure` であるか、または Measure インターフェースを満たす場合、かつ
  * k が μ のサポートから値を取り、測度を返す関数である場合

そのとき `μ ↣ k` は測度であり、*モナディックバインド*と呼ばれます。Soss.jl のような確率的プログラミング言語では、これを次のように表現できます。

バインドは通常 `>>=` と書かれますが、この記号は Julia では使用できません。

```
bind = @model μ,k begin 
    x ~ μ
    y ~ k(x)
    return y
end
```

`bind` と `Bind` も参照してください。
