```
smf(μ, x::Real) ::Real
```

測度 `μ` の *Stieltjes 測度関数 (SMF)* を点 `x` で計算します。

SMF は確率論からの *累積分布関数 (CDF)* の測度理論的一般化です。SMF `F(x) = smf(μ, x)` は以下の特性を持たなければなりません：

1. F は *単調非減少*
2. F は *右連続*：`F(x)` は `lim_{δ→0} F(x + |δ|)` と同じであるべきです。
3. μ((a,b]) = F(b) - F(a)

CDF とは異なり、SMF は定数の加算によってのみ決定されることに注意してください。多くのアプリケーションでは、-∞ で SMF を評価する必要があります。したがって、`smf(μ, -Inf)` が高速であることが重要です。実際には、通常 `smf(μ, static(-Inf))` として呼び出されます。これにより、速度を確保し、複雑な制御フローを避けるために、メソッド `smf(μ::M, ::StaticFloat64{-Inf})` を追加することが簡単になります。

`sinh` を「sinch」と発音するユーザーは、`smf` を「smurf」と発音することをお勧めします。
