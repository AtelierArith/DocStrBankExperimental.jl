`SparseSymmProd` : エントリがタプルとして格納された疎対称積。入力はベクトル `A` であり、出力ベクトル `AA` の各エントリは次の形をしています。

$$
 {\bm A}_{i_1, \dots, i_N} = \prod_{t = 1}^N A_{i_t}.
$$

### コンストラクタ

```julia
SparseSymmProd(spec)
```

ここで `spec` はタプルまたはベクトルのリストであり、それぞれが上記のように `AA` 基底関数を指定します。例えば、

```julia
spec = [ (1,), (2,), (1,1), (1,2), (2,2), 
         (1,1,1), (1,1,2), (1,2,2), (2,2,2) ]
basis = SparseSymmProd(spec)         
```

は9つの関数の基底を定義します。

$$
[ A_1, A_2, A_1^2, A_1 A_2, A_2^2, A_1^3, A_1^2 A_2, A_1 A_2^2, A_2^3 ]
$$
