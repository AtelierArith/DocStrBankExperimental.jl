```
F2p(b::Vector{Fb}, α::Fe)::Fe
```

F2要素のベクトルを与えられた基底要素を使用してフィールド要素に変換します。

# 引数

  * `b::Vector{Fb}`: F2要素のベクトル
  * `α::Fe`: 対象フィールドの基底要素

# 戻り値

  * `Fe`: ベクトルによって表されるフィールド要素

# 例外

  * `AssertionError`: 入力ベクトルの長さがフィールドの拡張次数と一致しない場合

# 例

```julia
julia> F4, α = GaloisField(2, 2, :α)
julia> F2p([F2(1), F2(0)], α)
1
```
