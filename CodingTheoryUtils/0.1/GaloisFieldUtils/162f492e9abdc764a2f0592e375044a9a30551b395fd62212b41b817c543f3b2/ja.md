```
get_tr_one_elem(a::F)::F where F <: GaloisFields.AbstractGaloisField
```

トレースが1のガロア体の要素を見つけます。

# 引数

  * `a::F`: ゼロでないガロア体の要素（原始要素であると仮定）

# 戻り値

  * `F`: トレースが1の要素

# 例外

  * `AssertionError`: 入力要素がゼロの場合

# 注意事項

  * この関数は、trace(b) = 1となる要素bを見つけます
  * このような要素は、特性2の有限体に常に存在します
  * 戻り値の要素は、トレース双対基底の基底要素として使用できます
