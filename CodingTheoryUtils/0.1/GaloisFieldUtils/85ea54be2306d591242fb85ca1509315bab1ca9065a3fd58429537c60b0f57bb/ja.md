```
trace(a::F)::F2 where F <: GaloisFields.AbstractGaloisField
```

ガロワ体の要素のトレースを計算します。

# 引数

  * `a::F`: ガロワ体の要素

# 戻り値

  * `F2`: 要素のトレース (0 または 1)

# 注意事項

  * GF(2^m) における要素 a のトレースは Tr(a) = a + a^2 + a^4 + ... + a^(2^(m-1)) と定義されます。
  * トレースは常に基底体 GF(2) の要素です。
  * トレースは GF(2) 上の線形関数です。
  * ゼロのトレースはゼロです。
