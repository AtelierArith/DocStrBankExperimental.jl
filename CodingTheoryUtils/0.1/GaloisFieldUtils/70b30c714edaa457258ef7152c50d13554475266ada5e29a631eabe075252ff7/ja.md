```
gf_pretty(v::Vector{F2})::String
```

バイナリベクトル (F2) を多項式文字列 (例: "1 + x^2") としてきれいに表示します。これは、`v` の要素が係数である多項式に対応します。

# 引数

  * `v::Vector{F2}`: バイナリベクトル。

# 戻り値

  * `String`: 多項式の文字列表現、またはゼロ/空のベクトルの場合は "0"。

# 例

```julia
gf_pretty([F2(1), F2(0), F2(1)]) == "1 + x^2"
gf_pretty([F2(0), F2(1)]) == "x"
gf_pretty(F2[]) == "0"
```
