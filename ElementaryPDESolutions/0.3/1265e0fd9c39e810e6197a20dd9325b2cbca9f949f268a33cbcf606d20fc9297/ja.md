```
struct Polynomial{N,T}
```

`N`変数の多項式で、係数の型は`T`です。

ファンクタインターフェースが実装されているため、`p(x)`は多項式を評価します。パフォーマンスの理由から、`x`は`Tuple`であることが期待されます。

# 例

単一の項を持つ多項式は、次数と係数をマッピングするペアを渡すことで作成できます：

```jldoctest; output = false
julia> Polynomial((1,1)=>2)
2xy
```

複数の項が存在する場合、それらはペアのベクトル（またはタプル）として渡す必要があります：

```jldoctest
julia> Polynomial([(1,1)=>2, (0,1)=>-1])
-y + 2xy
```

空間次元は、次数タプルの長さから自動的に推測されます：

```jldoctest
julia> Polynomial((1,1,1)=>2)
2xyz
```
