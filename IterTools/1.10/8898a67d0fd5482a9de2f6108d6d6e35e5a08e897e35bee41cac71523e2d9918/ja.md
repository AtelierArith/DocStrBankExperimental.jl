```
subsets(xs)
subsets(xs, k)
subsets(xs, Val{k}())
```

インデックス可能なコレクション `xs` のすべての部分集合を反復処理します。部分集合を特定のサイズ `k` に制限することができます。

`Val{k}()` の形式で部分集合のサイズを指定することで、コンパイラは要求された特定のサイズに最適化されたコードを生成できます。これにより、`k` が小さく、コンパイル時に知られている場合は手書きのループに匹敵するパフォーマンスが得られますが、それ以外の場合はパフォーマンスが向上するかどうかは不明です。

```jldoctest
julia> for i in subsets([1, 2, 3])
          @show i
       end
i = Int64[]
i = [1]
i = [2]
i = [1, 2]
i = [3]
i = [1, 3]
i = [2, 3]
i = [1, 2, 3]

julia> for i in subsets(1:4, 2)
          @show i
       end
i = [1, 2]
i = [1, 3]
i = [1, 4]
i = [2, 3]
i = [2, 4]
i = [3, 4]

julia> for i in subsets(1:4, Val{2}())
           @show i
       end
i = (1, 2)
i = (1, 3)
i = (1, 4)
i = (2, 3)
i = (2, 4)
i = (3, 4)
```
