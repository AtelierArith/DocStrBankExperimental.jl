```
create_params(
    defn_path::AbstractString,
    param_paths...;
    error_on_unused::Bool = false,
    out_path::AbstractString = "",
)
```

エントリ関数 - パラメータ定義とファイルへのパスを与えると、パラメータ辞書を解析して作成します。

defn*path と param*paths は、単一の yaml ファイルまたは yaml ファイルを含むディレクトリのいずれかであることができます。

# 引数

  * `defn_path`: パラメータ定義へのパス
  * `param_paths...`: パラメータファイルまたはディレクトリへのパス。ファイルは渡された順序でマージされるため、最初のパラメータのアイテム 'a' は、2 番目のパラメータのアイテム 'a' によって上書きされます。
  * `out_path`: 計算されたパラメータが保存されるディレクトリへのパス（渡された場合）
  * `error_on_unused`: 未使用のパラメータがある場合にハードエラーをスローするか、そうでない場合は警告のみが表示されます

# 戻り値

  * 必要に応じてデフォルトが埋め込まれた計算済み/検証済みのモデルパラメータを含む辞書
