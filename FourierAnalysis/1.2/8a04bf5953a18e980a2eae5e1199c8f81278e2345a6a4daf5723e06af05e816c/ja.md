```julia
function f2b(f :: IntOrReal,
            sr :: Int,
            wl :: Int;
        DC :: Bool = false)
```

**f**requency **to b**in。サンプリングレート `sr` とウィンドウ長 `wl` が与えられたとき、周波数 `f`（Hz単位）に最も適合する実数FFTベクトル内のビン（位置）を返します。周波数は整数または実数として与えることができます。

要求された `f` が2つのフーリエ離散周波数のちょうど間にある場合、2つの等距離の周波数のうち小さい方が返されます。

FFTベクトルは1ベースであると仮定されます（Juliaでは常にそうです）。`DC` が false の場合、最初の離散周波数はビン1にあると仮定されます。そうでない場合、DCはビン1にあり、最初の離散周波数はビン2にあると仮定されます。

`DC` が false の場合、周波数が周波数分解能の半分未満の場合は0を返します。

**関連情報**: [`fres`](@ref), [`b2f`](@ref), [`fdf`](@ref), [`brange`](@ref)。

**例**:

```julia
using FourierAnalysis
f2b(10, 512, 1024) # return 20
```
