```
nearest_cor!(A, alg=nothing; kwargs...)
```

行列 $A$ に最も近い正定値相関行列を返します。このメソッドは $A$ を上書きします。

これは「バッテリー同梱」メソッドであり、できるだけ考えずに機能するように設計されています。`solve!` とは異なり、このメソッドは `NCMSolution` オブジェクトではなく、直接最も近い相関行列を返します。さらに、解が正定値であることが確認され、そうでない場合は修正されます。

# 例

```julia-repl
julia> import LinearAlgebra: isposdef

julia> r = [
    1.00 0.82 0.56 0.44
    0.82 1.00 0.28 0.85
    0.56 0.28 1.00 0.22
    0.44 0.85 0.22 1.00
];

julia> isposdef(r)
false

julia> nearest_cor!(r)
4×4 Matrix{Float64}:
 1.0       0.817095  0.559306  0.440514
 0.817095  1.0       0.280196  0.847352
 0.559306  0.280196  1.0       0.219582
 0.440514  0.847352  0.219582  1.0

julia> isposdef(r)
true
```
