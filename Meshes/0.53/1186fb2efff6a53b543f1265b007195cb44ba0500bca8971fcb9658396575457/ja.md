```
Cylinder(bottom, top, radius)
```

与えられた `radius` を持ち、`bottom` と `top` 平面で区切られた R³ に埋め込まれた固体円筒。

```
Cylinder(start, finish, radius)
```

または、`start` と `finish` の端点を持つ線分に沿って、与えられた `radius` の直立円筒を構築します。

```
Cylinder(start, finish)
```

または、`start` と `finish` の端点を持つ線分に沿って、単位半径の直立円筒を構築します。

```
Cylinder(radius)
```

最後に、与えられた `radius` を持つ直立円形円筒を構築します。
