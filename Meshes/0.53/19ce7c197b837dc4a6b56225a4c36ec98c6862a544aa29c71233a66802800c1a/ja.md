```
CylinderSurface(bottom, top, radius)
```

与えられた `radius` で R³ に埋め込まれた円筒面で、`bottom` と `top` 平面によって区切られています。

```
CylinderSurface(start, finish, radius)
```

または、`start` と `finish` の端点を持つ線分に沿った与えられた `radius` の直円筒面を構築します。

```
CylinderSurface(start, finish)
```

または、`start` と `finish` の端点を持つ線分に沿った単位半径の直円筒面を構築します。

```
CylinderSurface(radius)
```

最後に、与えられた `radius` の直立円筒面を構築します。
