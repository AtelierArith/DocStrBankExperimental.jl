```
connect(indices, [PL])
```

グローバルベクトルの[`Point`](@ref)からの`indices`のリストを[`Polytope`](@ref)タイプ`PL`に接続します。

タイプ`PL`は[`Ngon`](@ref)である場合があり、その場合、インデックスの長さが実際の多面体タイプを特定するために使用されます。

最後に、タイプ`PL`は省略可能です。この場合、インデックスは[`Ngon`](@ref)または[`Segment`](@ref)として接続されると見なされます。

## 例

インデックスを三角形に接続します：

```julia
connect((1,2,3), Triangle)
```

インデックスをN-gon、三角形、四角形に接続します：

```julia
connect.([(1,2,3), (2,3,4,5)], Ngon)
```

インデックスをN-gonまたはセグメントに接続します：

```julia
connect((1,2)) # セグメント
connect((1,2,3)) # 三角形
connect((1,2,3,4)) # 四角形
```
