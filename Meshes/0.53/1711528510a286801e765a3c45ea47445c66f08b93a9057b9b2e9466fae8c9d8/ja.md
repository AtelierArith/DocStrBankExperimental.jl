```
FibonacciSampling(n, ϕ = (1 + √5)/2)
```

パラメータ `ϕ` を用いて `n` 個のフィボナッチ点を生成します。

黄金比が `ϕ` のデフォルト値として使用されますが、他の無理数も使用できます。

詳細は [https://observablehq.com/@meetamit/fibonacci-lattices](https://observablehq.com/@meetamit/fibonacci-lattices) と [https://www.johndcook.com/blog/2023/08/12/fibonacci-lattice](https://www.johndcook.com/blog/2023/08/12/fibonacci-lattice) を参照してください。
