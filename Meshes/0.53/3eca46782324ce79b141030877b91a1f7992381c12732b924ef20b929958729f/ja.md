```
Point(x₁, x₂, ..., xₙ)
Point((x₁, x₂, ..., xₙ))
```

次元 `Dim` の空間における、長さ単位（デフォルトはメートル）での座標を持つ点。

点の座標は標準的なユークリッド基底に対して与えられ、整数座標は浮動小数点に変換されます。

## 例

```julia
# 2D 点
Point(1.0, 2.0) # デフォルト単位を追加
Point(1.0m, 2.0m) # 期待通りの倍精度
Point(1f0km, 2f0km) # 期待通りの単精度
Point(1m, 2m) # 整数は設計上浮動小数点に変換される

# 3D 点
Point(1.0, 2.0, 3.0) # デフォルト単位を追加
Point(1.0m, 2.0m, 3.0m) # 期待通りの倍精度
Point(1f0km, 2f0km, 3f0km) # 期待通りの単精度
Point(1m, 2m, 3m) # 整数は設計上浮動小数点に変換される
```

### 注意

整数座標はサポートされていません。なぜなら、ほとんどの幾何処理アルゴリズムは連続空間を前提としているからです。浮動小数点への変換は `InexactError` やその他の予期しない結果を回避します。
