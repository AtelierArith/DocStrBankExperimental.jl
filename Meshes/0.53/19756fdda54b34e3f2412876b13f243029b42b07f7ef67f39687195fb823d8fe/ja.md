```
Vec(x₁, x₂, ..., xₙ)
Vec((x₁, x₂, ..., xₙ))
```

線形代数のための長さ単位（デフォルトはメートル）での座標を持つ `Dim` 次元空間の幾何学的ベクトル。

デフォルトでは、整数座標は浮動小数点に変換されます。

ベクトルは、2つの [`Point`](@ref) オブジェクトを引き算することで得られます。

## 例

```julia
A = Point(0.0, 0.0)
B = Point(1.0, 0.0)
v = B - A

# 2D ベクトル
Vec(1.0, 2.0) # デフォルトの単位を追加
Vec(1.0m, 2.0m) # 期待通りの倍精度
Vec(1f0km, 2f0km) # 期待通りの単精度
Vec(1m, 2m) # 整数は設計上浮動小数点に変換される

# 3D ベクトル
Vec(1.0, 2.0, 3.0) # デフォルトの単位を追加
Vec(1.0m, 2.0m, 3.0m) # 期待通りの倍精度
Vec(1f0km, 2f0km, 3f0km) # 期待通りの単精度
Vec(1m, 2m, 3m) # 整数は設計上浮動小数点に変換される
```

### 注意

`Vec` は StaticArrays.jl の `StaticVector` のサブタイプです。
