```
GrahamScan()
```

Compute the convex hull of a set of points or geometries using the Graham's scan algorithm. See [https://en.wikipedia.org/wiki/Graham_scan](https://en.wikipedia.org/wiki/Graham_scan).

The algorithm has complexity `O(n*log(n))` where `n` is the number of points.

## References

  * Cormen et al. 2009. [Introduction to Algorithms](https://mitpress.mit.edu/books/introduction-algorithms-third-edition)
