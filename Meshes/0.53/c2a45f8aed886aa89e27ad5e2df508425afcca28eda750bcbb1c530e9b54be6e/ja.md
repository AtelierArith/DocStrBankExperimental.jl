```
Rotate(R)
```

回転 `R` でジオメトリまたはドメインを回転させます。これは Rotations.jl からのものです。

```
Rotate(u, v)
```

軸 `u` に向かう方向を軸 `v` に向かう方向にマッピングする回転です。より正確には、法線ベクトル `u` を持つ原点を通る平面を法線ベクトル `v` を持つ原点を通る平面にマッピングします。

```
Rotate(θ)
```

角度 `θ`（ラジアン単位）で2Dジオメトリまたはドメインを回転させます。`Angle2d` 回転を使用します。

## 例

```julia
Rotate(one(RotXYZ{Float64})) # 単位回転
Rotate(AngleAxis(0.2, 1.0, 0.0, 0.0)) # X軸周りに0.2ラジアン回転
Rotate(rand(QuatRotation{Float64})) # ランダム回転
Rotate(Vec(1, 0, 0), Vec(1, 1, 1)) # (1, 0, 0) から (1, 1, 1) への回転
Rotate(π / 2) # ラジアン単位の角度での2D回転
```
