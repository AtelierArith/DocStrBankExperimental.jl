```
構造格子(X, Y, Z, ...)
構造格子{M,C}(X, Y, Z, ...)
```

ソートされた座標 `X`, `Y`, `Z`, ... に頂点を持つ構造格子で、マンフォールド `M`（デフォルトは `𝔼`）およびCRSタイプ `C`（デフォルトは `Cartesian`）を持ちます。

## 例

`x` 次元で均等な間隔、`y` 次元で不均等な間隔の2D構造格子を作成します：

```julia
julia> X = repeat(0.0:0.2:1.0, 1, 6)
julia> Y = repeat([0.0, 0.1, 0.3, 0.7, 0.9, 1.0]', 6, 1)
julia> 構造格子(X, Y)
```
