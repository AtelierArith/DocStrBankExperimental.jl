```
≻(A::Point, B::Point)
```

点 `A` と `B` の積順序 (`\succ`)。

`A ≻ B` は、すべての座標 `aᵢ` と `bᵢ` に対して `aᵢ > bᵢ` の場合です。

See [https://en.wikipedia.org/wiki/Partially*ordered*set#Orders*on*the*Cartesian*product*of*partially*ordered*sets](https://en.wikipedia.org/wiki/Partially_ordered_set#Orders_on_the_Cartesian_product_of_partially_ordered_sets)
