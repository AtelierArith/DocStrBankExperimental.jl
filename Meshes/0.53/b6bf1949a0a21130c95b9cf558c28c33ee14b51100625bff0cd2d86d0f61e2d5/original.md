```
Multi(geoms)
```

A collection of geometries `geoms` seen as a single [`Geometry`](@ref).

In geographic information systems (GIS) it is common to represent multiple polygons as a single entity (e.g. country with islands).

### Notes

Type aliases are [`MultiPoint`](@ref), [`MultiSegment`](@ref), [`MultiRope`](@ref), [`MultiRing`](@ref), [`MultiPolygon`](@ref).
