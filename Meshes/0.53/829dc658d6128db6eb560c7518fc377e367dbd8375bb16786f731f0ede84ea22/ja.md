```
Repair(K)
```

コード `K` を使用して修復操作を実行します。

## 利用可能な操作

  * K =  0: 重複した頂点と面が削除されます
  * K =  1: 未使用の頂点が削除されます
  * K =  2: 非多様体の面が削除されます
  * K =  3: 劣化した面が削除されます
  * K =  4: 非多様体の頂点が削除されます
  * K =  5: 非多様体の頂点が閾値によって分割されます
  * K =  6: 近接する頂点がマージされます（半径が指定されます）
  * K =  7: 面が一貫して向けられます
  * K =  8: ゼロ面積の耳が削除されます
  * K =  9: 多角形のリングがソートされます
  * K = 10: 多角形の外側のリングが拡張されます
  * K = 11: 多角形のリングが一貫して向けられます
  * K = 12: 劣化した多角形のリングが削除されます

## 例

```
# 重複と劣化を削除
mesh |> Repair(0) |> Repair(3)
```
