bulkscan_null(Y, G, K, nb; reml = true)

すべての特性とマーカーのペアに対するLODスコアを計算します。これは、特性のブロックに対する（マルチスレッド）ループとLiteQTL型アプローチによるものです。

# 引数

  * Y = 2d 配列の Float; 1つの特性または複数の特性の行列
  * G = 2d 配列の Float; ジェノタイプ確率の行列
  * K = 2d 配列の Float; 親族関係行列
  * nb = Int; 必要な特性のブロック数; 理想的には並列化に使用されるスレッドの数と同じであるべきです

# 値

  * LOD = 2d 配列の Float; すべての特性とマーカーのペアに対するLODスコア
