```
scan(y, g, K, nperm, nprocs, rndseed, reml)
```

遺伝子マーカーごとに単変量特性を置換することによるいくつかの再サンプリングに対してゲノムスキャンを実行します（1自由度のテスト）。分散成分はヌルモデルから推定され、マーカー間で同じであると仮定されます。

# 引数

  * y = 特定の定量的特性に対するNの観測値からなる1d配列（次元：N*1）
  * g = すべてのp遺伝子マーカーからなる2d配列（次元：N*p）
  * K = Nの観測値の遺伝的関連性からなる2d配列（次元：N*N）

# キーワード引数

  * nperms = 要求された置換の数
  * nprocs = 同時プロセスの数
  * rndseed = 置換を実行するための乱数シード
  * reml = 分散成分がREML（true）またはML（false）によって推定されるかどうかを示すブールフラグ

# 値

  * lod = 各置換された特性（行）とマーカー（列）に対するLODスコアからなる2d配列
