```
get_growingseason(GPPd, tGPP; ws=15, min_int=5, warngap=true)
```

平滑された日次GPPデータに基づいて成長期の年間時系列をフィルタリングします。

# 引数

  * `GPPd`:    日次GPP（任意の単位）
  * `tGPP`:    GPP閾値（GPP時系列の95パーセンタイルの割合）。              0から1の間の値を取ります。

オプション               

  * `ws`:      GPP時系列平滑化に使用されるウィンドウサイズ
  * `min_int`: 成長期の特定の状態のための最小時間間隔（日数）
  * `warngap`: 非欠損データが少なすぎる場合の警告を抑制するにはfalseに設定

# 詳細

成長期フィルタの基本的な考え方は、植生がその炭素吸収（GPP）が指定された閾値を超えているときに活発であると見なされることです。この閾値は、年に観測されたピークGPP（95パーセンタイル）に対して定義されます。GPP閾値は次のように計算されます：

$$
GPP_{threshold} = quantile(GPPd,0.95)*tGPP
$$

GPPd時系列は、成長期の境界の変動を避けるために移動平均で平滑化されます。ウィンドウサイズはデフォルトで15日ですが、エコシステムによっては他の値が適切な場合があります。

引数`min_int`は、成長期と非成長期の状態の短い変動を避けるために、状態の最小長を定義します。`min_int`より短い時間間隔が成長期または非成長期としてラベル付けされると、それは隣接する値の状態、すなわちその反対に変更されます。

# 値

エントリを持つNamedTuple

  * `is_growingseason`: 入力GPPdと同じ長さの`BitVector`で、`false`は成長期なし（休眠期）を示し、`true`は成長期を示します。
  * `GPPd_smoothed`: 平滑化されたGPPd
