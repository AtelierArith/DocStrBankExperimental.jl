```
CartPoleEnv(;kwargs...)
```

# Keyword arguments

  * `T = Float64`
  * `continuous = false`
  * `rng = Random.default_rng()`
  * `gravity = T(9.8)`
  * `masscart = T(1.0)`
  * `masspole = T(0.1)`
  * `halflength = T(0.5)`
  * `forcemag = T(10.0)`
  * `max_steps = 200`
  * `dt = 0.02`
  * `thetathreshold = 12.0 # degrees`
  * `xthreshold` = 2.4`
