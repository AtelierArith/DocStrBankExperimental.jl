```
assemble_dirichletcondition!(
    A::SparseMatrixCSC{Float64, Int64},
    DI::Set{Int64}; 
    rhs = [],
    bc = [],
    qdim::Int64 = 1,
    insert = 1.0
)
```

与えられたディリクレ条件に従って剛性行列と右辺を修正します。

DIは条件が有効であるべきノードインデックスのセットでなければなりません。ベクトル値の状態の場合、DIはディリクレ条件を持つべき各コンポーネントに設定することができるか、すべてのコンポーネントが条件を持つべき場合はqdimが設定されます。rhsを更新する必要がある場合、bcも明示的に指定する必要があります。値insertは対角要素として配置されます。通常、ここでは1.0を望みます。
