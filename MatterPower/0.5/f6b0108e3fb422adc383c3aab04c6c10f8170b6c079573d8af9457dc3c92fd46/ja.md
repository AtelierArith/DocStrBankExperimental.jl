```
setup_growth(Ωm, ΩΛ[, a1=1e-2])
```

`growth!` で与えられた連立微分方程式の解を返します。初期条件は `a = a1` で $δ(a) = a$ です。デフォルト値は `a1 = 1e-2` です。

# 引数

  * `Ωm::Real`: 現在の総物質密度パラメータ。
  * `ΩΛ::Real`: 現在のダークエネルギー密度パラメータ（宇宙定数用）。

# オプション引数

  * `a1::Real=1e-2`: 初期スケールファクターで、初期条件が δ(a1) = a1 と設定されます。
