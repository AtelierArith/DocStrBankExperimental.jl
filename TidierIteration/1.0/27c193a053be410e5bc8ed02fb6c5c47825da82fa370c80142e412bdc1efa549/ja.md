```
map_dfc(x, f)
```

関数 `f` を `x` の各要素に適用し、すべての列をデータフレームに結合します。

# 引数

  * `x`: イテラブルなコレクション。
  * `f`: データフレームを返す関数。

# 例

```julia-repl
julia> map_dfc(1:3, x -> DataFrame(Symbol("a" * string(x)) => rand(3)))
3×3 DataFrame
 Row │ a1        a2        a3       
     │ Float64   Float64   Float64  
─────┼──────────────────────────────
   1 │ 0.864102  0.342032  0.99839
   2 │ 0.160245  0.912323  0.465663
   3 │ 0.235012  0.516522  0.368654
```
