```
map_dfr(x, f)
```

関数 `f` を `x` の各要素に適用し、すべての行をデータフレームに結合します。

# 引数

  * `x`: イテラブルなコレクション。
  * `f`: データフレームを返す関数。

# 例

```julia-repl
julia> map_dfr(1:3, x -> DataFrame(a = rand(3)))
9×1 DataFrame
 Row │ a         
     │ Float64   
─────┼───────────
   1 │ 0.719786
   2 │ 0.208629
   3 │ 0.689885
   4 │ 0.334989
   5 │ 0.0575872
   6 │ 0.0717808
   7 │ 0.825819
   8 │ 0.289853
   9 │ 0.495595
```
