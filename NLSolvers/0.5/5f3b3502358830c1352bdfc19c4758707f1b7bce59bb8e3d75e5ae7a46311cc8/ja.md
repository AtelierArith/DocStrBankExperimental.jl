```
DFSANE(memory[ = 4], sigma_limits[ = (1e-5, 1e5)])
```

非線形方程系を解くためのDFSANEアルゴリズムのメソッドインスタンスを構築します。memoryキーワードは、非単調線形探索のために残差ノルム値をいくつ保存するかを制御するために使用されます（論文中の`M`）。`sigma_limit`キーワードは、スペクトル係数を制約するために使用される値のタプルを提供します（論文中の`σ_min`と`σ_max`）。

```
- decrease（論文中のγ）は、関数値の減少に対して十分な量を制御するために使用されます。0 < decrease < 1となるように選択する必要があります。
```
