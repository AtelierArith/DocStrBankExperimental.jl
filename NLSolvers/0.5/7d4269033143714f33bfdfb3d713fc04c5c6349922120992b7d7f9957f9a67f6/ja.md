# SimulatedAnnealing

## コンストラクタ

```julia
SimulatedAnnealing(; neighbor = default_neighbor, temperature = log_temperature)
```

コンストラクタは二つのキーワードを取ります：

  * `neighbor = a(x_now, [x_candidate])`、現在の状態に基づいて新しい反復を生成する関数
  * `temperature = b(iteration)`、現在の反復の関数で、温度を返します

## 説明

シミュレーテッドアニーリングは、最適化のための導関数フリーの手法です。これは、元々熱力学システムからサンプルを生成するために使用されたメトロポリス-ヘイスティングスアルゴリズムに基づいており、ベイズ推論を行う際に事後分布からのサンプルを生成するためにしばしば使用されます。そのため、これは関数の最小値を見つけるための確率的手法であり、しばしばかなり大きな領域にわたって行われます。上記の歴史的理由から、このアルゴリズムは冷却、温度、受容確率といった用語を使用します。
