```
バックトラッキング
```

バックトラッキング線形探索アルゴリズムを制御するオブジェクトです。このアルゴリズムは初期点を試し、その後、ゼロのステップ長の周りの一次テイラー近似と比較して改善されることによって十分なステップ長を見つけることを反復的に試みます。

```
Backtracking(; kwargs...)
```

`Backtracking` コンストラクタは以下のキーワード引数を受け取ります：

  * `ratio`: 現在の試行ステップ長と次のステップ長の比率。これは大きなステップサイズを探すのにどれだけの時間を費やすかを制御します。一般的な選択肢は1/2で、ここではデフォルトです。
  * `decrease`: `ratio` より小さい正の閾値で、降下が十分であるかどうかを決定します。デフォルト値は0.0001です。
  * `maxiter`: より良いステップ長を探す最大回数。デフォルト値は26です。補間がなく、比率が1/2の場合、ステップサイズが `sqrt(eps(Float64))` に達したときに探索が終了します。
  * `interp`: 使用する補間の種類（ある場合）。デフォルトは `FixedInterp()`、すなわち補間なしです。
  * `steprange`: ステップ長の許可範囲をタプルまたは名前付きタプルで指定します。デフォルトは `(; lower=0, upper=Inf)` です。
  * `verbose`: `true` の場合、アルゴリズムが進行するにつれて探索に関する情報が印刷されます。デフォルトは `false` です。

!!! note
    パラメータを選択する際には、`0 < decrease < ratio` が成り立つ必要があることに注意することが重要です。
