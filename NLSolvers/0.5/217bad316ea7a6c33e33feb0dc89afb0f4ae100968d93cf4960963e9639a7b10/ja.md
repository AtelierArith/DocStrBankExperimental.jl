```
HZAW
```

Hager-Zhang近似Wolfe線探索アルゴリズムを制御するオブジェクトです。[^HZ2005]

```
HZAW(; kwargs...)
```

`HZAW`コンストラクタは、以下のキーワード引数を受け取ります。デフォルト値は、[^HZ2005]のセクション5で使用されているものに対応しています。

  * `decrease`: 0と1の間のパラメータで、`curvature`以下であり、Armijoルールに従った目的関数の十分な減少を指定します。デフォルトは0.1です。
  * `curvature`: 0と1の間のパラメータで、`decrease`以上であり、曲率条件に従った勾配の十分な減少を指定します。デフォルトは0.9です。
  * `theta`: 0と1の間のパラメータで、区間更新のブレッキング間隔を制御します。デフォルトは1/2で、二分法を示します。（[^HZ2005]のセクション4の区間更新手順のステップU3を参照してください。）
  * `gamma`: アルゴリズムの各反復でブレッキング間隔の長さが減少する割合です。デフォルトは2/3です。

元のアルゴリズムをわずかに調整し、元のステップ長が有限でない関数値をもたらす場合には、実行可能な領域にバックトラックします。

[^HZ2005]: Hager, W. W., & Zhang, H. (2005). A New Conjugate Gradient Method        with Guaranteed Descent and an Efficient Line Search. SIAM Journal        on Optimization, 16(1), 170–192. doi:10.1137/030601880
