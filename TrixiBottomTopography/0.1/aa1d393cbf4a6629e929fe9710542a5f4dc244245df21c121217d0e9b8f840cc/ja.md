```
BilinearBSpline(path::String)
```

`BilinearBSpline`(@ref)のための`x`、`y`、`z`の値を.txtファイルから読み込む関数です。入力値は次のとおりです：

  * `path`: 特定の.txtファイルのパスの文字列

.txtファイルは、この関数によって解釈されるために次の構造を持っている必要があります：

  * 最初の行：コメント `# x値の数`
  * 二行目：`x`値の数を示す整数
  * 三行目：コメント `# y値の数`
  * 四行目：`y`値の数を示す整数
  * 五行目：コメント `# x値`
  * 続く行：各値が独自の行を持つ`x`値
  * x値の後の行：コメント `# y値`
  * 続く行：各値が独自の行を持つ`y`値
  * y値の後の行：コメント `# z値`
  * 残りの行：各値が独自の行を持つ`z`の値で、次の順序で配置されています：                  z*11, z*12, ... z*1n, z*21, ... z*2n, ..., z*m1, ..., z_mn

例は[こちら](https://gist.githubusercontent.com/maxbertrand1996/7b1b943eac142d5bc836bb818fe83a5a/raw/74228e349e91fbfe1563479f99943b469f26ac62/Rhine_data_2D_10.txt)で見つけることができます。
