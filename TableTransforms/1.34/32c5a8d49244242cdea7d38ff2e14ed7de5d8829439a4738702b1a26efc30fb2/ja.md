```
Map(cols₁ => fun₁ => target₁, cols₂ => fun₂, ..., colsₙ => funₙ => targetₙ)
```

`funᵢ` 関数を `colsᵢ` で選択された列に適用し、結果を `targetᵢ` という新しい列に保存します。

列の選択は、単一の列識別子（インデックスまたは名前）、識別子のコレクション、または正規表現（regex）で行うことができます。

ターゲット列名を渡すことはオプションであり、省略した場合は関数名と選択された列名を結合して新しい名前が生成されます。ターゲット列がすでにテーブルに存在する場合、元の列は置き換えられます。

# 例

```julia
Map(1 => sin)
Map(:a => sin, "b" => cos => :cos_b)
Map([2, 3] => ((b, c) -> 2b + c))
Map([:a, :c] => ((a, c) -> 2a * 3c) => :col1)
Map(["c", "a"] => ((c, a) -> 3c / a) => :col1, "c" => tan)
Map(r"[abc]" => ((a, b, c) -> a^2 - 2b + c) => "col1")
```

## 注意

  * 無名関数は、上記の例のように括弧付きで渡す必要があります；
  * 一部の関数名は特別な方法で扱われます。それらは：

      * 無名関数: `#1` -> `f1`;
      * 合成関数: `outer ∘ inner` -> `outer_inner`;
      * `Base.Fix1` 関数: `Base.Fix1(f, x)` -> `fix1_f`;
      * `Base.Fix2` 関数: `Base.Fix2(f, x)` -> `fix2_f`;
