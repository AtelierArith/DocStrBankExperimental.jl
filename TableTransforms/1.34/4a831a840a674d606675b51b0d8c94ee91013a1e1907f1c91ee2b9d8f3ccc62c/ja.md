```
Replace(cols₁ => pred₁ => new₁, pred₂ => new₂, ..., colsₙ => predₙ => newₙ)
```

`predᵢ` の述語が `true` を返すすべての値を、選択された列 `colsᵢ` の `newᵢ` 値で置き換えます。

列の選択を渡すことはオプションであり、省略した場合はテーブル内のすべての列が選択されます。列の選択は、単一の列識別子（インデックスまたは名前）、識別子のコレクション、または正規表現（regex）であることができます。

述語は、単一の引数を受け取りブール値を返す関数、または値であることができます。述語が値である場合、次の関数に変換されます：`x -> x === value`。

# 例

```julia
Replace(1 => -1, 5 => -5)
Replace(2 => 0.0 => 1.5, 5.0 => 5.5)
Replace(:b => 0.0 => 1.5, 5.0 => 5.5)
Replace("b" => 0.0 => 1.5, 5.0 => 5.5)
Replace([1, 3] => >(5) => 5)
Replace([:a, :c] => isequal(2) => -2)
Replace(["a", "c"] => (x -> 4 < x < 6) => 0)
Replace(r"[abc]" => (x -> isodd(x) && x > 10) => 2)
```

## 注意

  * 無名関数は、上記の例のように括弧を付けて渡す必要があります。
  * 置き換えは定義された順序で適用されるため、同じ列に対して複数の置き換えがある場合、最初の有効なものが適用されます。
