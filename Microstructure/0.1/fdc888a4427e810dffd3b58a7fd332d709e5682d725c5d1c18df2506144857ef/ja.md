```
Iso(diff::Float64, t2=Float64)
```

拡散率 `diff` と T2 緩和時間 `t2` を持つ等方性テンソルを返します。このコンパートメントは、CSF（`diff` = 自由水）やドットコンパートメント（`diff` = 0）を表すために使用できます。後者は、通常 ex vivo 組織で見られる不動水のためのものです。このコンパートメントは、自由水よりも拡散率 `diff` が遅い等方性細胞外環境を表すこともできます。

# 例

```julia-repl
julia> Iso(diff = 3.0e-9,t2 = 2000.0e-3)
Iso(3.0e-9, 2.0)
```

```julia-repl
julia> Iso(diff = 0.0)
Iso(0.0, 0.0)
```
