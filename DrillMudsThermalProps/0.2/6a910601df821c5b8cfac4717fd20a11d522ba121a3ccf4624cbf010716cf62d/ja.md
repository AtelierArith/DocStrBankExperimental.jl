```
TestSolid <: Solid
```

パッケージ内で新しい固体を定義するために必要なすべての最小パラメータを持つテンプレート固体として定義された型。この固体を使用した関数呼び出しは、未実装のプロパティに対してエラーをスローする必要があります。

# パラメータ

  * `name::Symbol`: 固体の名前。デフォルト値: `:TestSolid`。
  * `ρ::Float64`: 標準条件下での固体の密度 [kg/m³]。デフォルト値: `1000.0`。
  * `cₚ::Float64`: 標準条件下での固体の比熱 [J/kg K]。デフォルト値: `4000.0`。
  * `k::Float64`: 標準条件下での固体の熱伝導率 [W/m K]。デフォルト値: `0.58`。
