```
lattice_vectors(a, b, c, α, β, γ)
```

グローバルな直交座標系における結晶格子セルの形状を定義する$3×3$出力行列の列として、格子ベクトルを返します。逆に、出力行列は格子系に対するグローバルな直交座標系を定義していると見ることもできます。

格子定数$(a, b, c)$は長さの単位を持ち、角度$(α, β, γ)$は度単位です。逆のマッピングは[`lattice_params`](@ref)です。

# 例

```julia
latvecs = lattice_vectors(1, 1, 2, 90, 90, 120)
a1, a2, a3 = eachcol(latvecs)
@assert a1 ≈ [1, 0, 0]       # a1は常にグローバルxに整列
@assert a2 ≈ [-1/2, √3/2, 0] # a2は常にグローバル(x,y)平面にある
@assert a3 ≈ [0, 0, 2]       # a3は一般に(x,y,z)の組み合わせである可能性がある
```
