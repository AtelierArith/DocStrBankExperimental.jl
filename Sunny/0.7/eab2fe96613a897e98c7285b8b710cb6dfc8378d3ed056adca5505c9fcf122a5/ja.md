```
(cell1, cell2, cell3, i) :: Site
```

単一のサイトを識別する四つのインデックスです [`System`](@ref) 内で。最初の三つのインデックスは単位セルを選択し、最後のインデックスはサブラティスを選択します。つまり、単位セル内の $i$ 番目の原子です。

このオブジェクトは、`System` の `dipoles` および `coherents` フィールドをインデックスするために使用できます。また、[`set_field_at!`](@ref) や [`set_exchange_at!`](@ref) のような関数を介して不均一な相互作用を指定するためにも `Site` が必要です。

システムが再形成されると、セルの定義が変更される可能性があることに注意してください。この場合、元の格子ベクトルの分数座標で位置を常に受け取る [`position_to_site`](@ref) を使用して `Site` を構築することが便利です。
