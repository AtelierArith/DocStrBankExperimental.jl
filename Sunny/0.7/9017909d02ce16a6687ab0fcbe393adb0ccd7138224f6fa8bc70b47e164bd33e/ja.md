```
LocalSampler(; kT, nsweeps=1.0, propose=propose_uniform)
```

個々のスピンに対するメトロポリス更新を含むモンテカルロシミュレーション。[`step!`](@ref) 関数への1回の呼び出しは、提供された [`System`](@ref) に対して `nsweeps` のMCMCサンプリングを実行します。デフォルト値の `1.0` は、`step!` が平均してスピンごとに1回の試行更新を行うことを意味します。

エルゴード性を仮定すると、`LocalSampler` はターゲット温度 `kT` の熱平衡からサンプリングします。

試行スピン更新は `propose` 関数を使用してサンプリングされます。オプションには [`propose_uniform`](@ref)、[`propose_flip`](@ref)、および [`propose_delta`](@ref) が含まれます。複数の提案はマクロ [`@mix_proposals`](@ref) で混合できます。

返されるオブジェクトは、ネットエネルギーと双極子の累積変化をそれぞれ表すフィールド `ΔE` と `ΔS` を格納します。

!!! warning "効率に関する考慮事項"
    ほとんどの場合、[`Langevin`](@ref) サンプリングを優先してください。ランジュバン力学は、通常、連続的に変化するハイゼンベルグ様スピンのサンプリングに対してはるかに効率的です。`LocalSampler` は離散スピン状態からのサンプリングに最も有用です。特に、強いイージー軸異方性によって生じるイジング様スピンのサンプリングには、[`propose_flip`](@ref) が必要になる場合があります。強いが有限の単一イオン異方性の場合、`Langevin` と `LocalSampler` の更新ステップを交互に行うことを検討してください。
