```
print_wrapped_intensities(sys::System; nmax=10)
```

ブラヴァイス格子の場合: 瞬時（静的）構造因子強度に従って、最大で `nmax` の波ベクトルを降順で表示します。非ブラヴァイス格子の場合: 各スピンサブ格子について独立して同じ分析を行います; 出力の重みはサブ格子間で単純に平均され、位相シフト情報は考慮されません。この手順により、すべての波ベクトルが最初のブリルアンゾーンにラップされます。各波ベクトルの座標は、逆格子単位（RLU）で $-1/2$ と $1/2$ の間で与えられます。この関数の出力は通常、[`suggest_magnetic_supercell`](@ref) の入力として使用されます。

この関数はサブ格子間の平均に位相情報を組み込んでいないため、印刷された重みは実験と直接比較することはできません。その目的のためには、代わりに [`SampledCorrelationsStatic`](@ref) を使用してください。
