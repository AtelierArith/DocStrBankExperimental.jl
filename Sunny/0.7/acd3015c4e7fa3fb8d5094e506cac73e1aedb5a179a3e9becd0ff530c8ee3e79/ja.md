```
set_field!(sys::System, B_μB)
```

外部磁場 $𝐁$ をボーア磁子 $μ_B$ でスケーリングします。このスケーリングされた場はエネルギーの単位を持ち、無次元の [`magnetic_moment`](@ref) に直接結合します。すべてのサイトで、ゼーマン結合はエネルギー $+ (𝐁 μ_B) ⋅ (g 𝐒)$ に寄与し、局所の $g$-テンソルとスピン角運動量 $𝐒$ が関与します。一般に、$g ≈ +2$ であるため、$𝐒$ は適用された場 $𝐁$ と反対に整列することが好まれます。特定の [`Units`](@ref) のシステムは、場とエネルギーの次元を変換するためにボーア磁子を暗黙的に使用します。

# 例

```julia
# meV 単位で、z方向に2テスラの場を適用
units = Units(:meV, :angstrom)
set_field!(sys, [0, 0, 2] * units.T)
```
