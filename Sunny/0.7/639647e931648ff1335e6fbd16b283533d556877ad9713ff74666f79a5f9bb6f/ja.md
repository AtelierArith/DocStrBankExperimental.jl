```
set_exchange!(sys::System, J, bond::Bond; biquad=0)
```

指定された `bond` に沿った交換相互作用 $𝐒_i⋅J 𝐒_j$ を設定します。この相互作用は、結晶対称性に従って同等の結合に伝播されます。これらの結合に対する以前の相互作用は上書きされます。パラメータ `bond` は `Bond(i, j, offset)` の形式を持ち、ここで `i` と `j` は単位格子内の原子インデックスであり、`offset` は単位格子内の変位です。

便利なことに、スカラー `J` を使用してハイゼンベルグ相互作用を指定できます。また、関数 [`dmvec(D)`](@ref dmvec) を使用して、交換の反対称部分を構築することができ、ここで `D` はジャリョーシンキー-モリヤ擬似ベクトルです。結果として得られる相互作用は $𝐃⋅(𝐒_i×𝐒_j)$ になります。

オプションの数値パラメータ `biquad` は、適切な場合にスカラーの二次相互作用 $(𝐒_i⋅𝐒_j)^2$ を [相互作用の再正規化](@ref) で乗算します。より一般的な相互作用には、代わりに [`set_pair_coupling!`](@ref) を使用してください。

# 例

```julia
using LinearAlgebra

# ハイゼンベルグおよびDM相互作用を設定: 2Si⋅Sj + D⋅(Si×Sj)
D = [0, 0, 3]
set_exchange!(sys, 2I + dmvec(D), bond)

# 明示的な交換行列として同じ相互作用
J = [2 3 0;
    -3 2 0;
     0 0 2]
set_exchange!(sys, J, bond)
```
