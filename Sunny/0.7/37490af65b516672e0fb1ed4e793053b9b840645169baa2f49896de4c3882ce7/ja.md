```
System(crystal::Crystal, moments, mode; dims=(1, 1, 1), seed=nothing)
```

スピンシステムは、[`Crystal`](@ref) 単位セル、スピン `moments` の対称的に異なるサイトの仕様、および計算 `mode` から構築されます。相互作用は、例えば [`set_exchange!`](@ref) を使用してシステムに追加できます。デフォルトのスーパーセルの寸法は 1×1×1 化学セルですが、`dims` を使用して変更できます。

スピン `moments` は、対称的に異なる原子の完全なセットである `[i1 => Moment(...), i2 => ...]` のペアのリストで構成されています。各 [`Moment`](@ref) にはスピンと $g$-因子の情報が含まれています。

`mode` の主な2つのオプションは `:SUN` と `:dipole` です。前者では、各スピン-$s$ 自由度は SU(*N*) コヒーレント状態として記述され、すなわち $N = 2s + 1$ レベルの量子重ね合わせです。この形式は、多極子スピンの揺らぎや局所的なエンタングルメント効果を捉えるのに役立ちます。

モード `:dipole` は、SU(*N*) 動力学を純粋な双極子の制限された空間に投影します。実際には、Sunny はランダウ-リフシッツ動力学をシミュレートしますが、単一イオン異方性と二次交換相互作用は精度を向上させるために再正規化されます。この再正規化を無効にするには、形式的な $s → ∞$ 限界に対応するモード `:dipole_uncorrected` を使用します。詳細については、ドキュメントページを参照してください: [Interaction Renormalization](@ref)。

このシステムに対する確率的操作は、このシステムの擬似乱数生成器の `seed` を選択することで再現可能にできます。デフォルトのシステムシードは、Julia のタスクローカル RNG から生成され、これ自体は `Random.seed!` を使用してシードを設定できます。

すべてのスピンは最初にグローバル $z$-方向に偏極されています。
