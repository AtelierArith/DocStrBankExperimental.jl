```
stevens_matrices(s)
```

スピン-`s` 表現におけるスティーブンス演算子を返します。返り値 `O` は `O[k,q]` のようにインデックス指定でき、ここで $0 ≤ k ≤ 6$ は SO(3) の不変表現をラベル付けし、$-k ≤ q ≤ k$ です。これにより、$N = 2s + 1$ の $N×N$ 行列が生成されます。スティーブンス演算子の線形結合は、局所的な可観測量を分解するための「物理基底」として使用できます。この分解を見るには、[`print_stevens_expansion`](@ref) を使用してください。

`s == Inf` の場合、記号演算子が返されます。この無限次元表現では、スティーブンス演算子は可換スピン演算子の同次多項式になります。

# 例

```julia
O = stevens_matrices(2)
S = spin_matrices(2)

A = (1/20)O[4,0] + (1/4)O[4,4] + (102/5)I
B = S[1]^4 + S[2]^4 + S[3]^4
@assert A ≈ B
```

他に [`spin_matrices`](@ref) と [Interaction Renormalization](@ref) も参照してください。
