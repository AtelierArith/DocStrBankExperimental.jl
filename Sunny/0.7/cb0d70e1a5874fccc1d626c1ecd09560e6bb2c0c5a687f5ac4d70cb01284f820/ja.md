```
propose_delta(magnitude)
```

ガウス摂動を既存のスピン状態に加える提案関数を生成します。`:dipole`モードでは、手順はまずランダムな三次元ベクトル摂動 $𝐒' = 𝐒 + |𝐒| ξ$ を導入し、その後適切に正規化されたスピン $|𝐒| (𝐒'/|𝐒'|)$ を返します。ランダムベクトル $ξ$ の各成分は標準偏差 `magnitude` のガウス分布に従います；後者は次元のないもので、通常は1より小さいです。

`:SUN`モードでは、手順は類似していますが、今度はSU(*N*)コヒーレント状態の$N$個の複素成分それぞれに対するガウス摂動を含みます。

非常に大きな `magnitude` の限界では、この関数は [`propose_uniform`](@ref) と一致します。

また、拒否なしのサンプリングである [`Langevin`](@ref) サンプリングも考慮してください。
