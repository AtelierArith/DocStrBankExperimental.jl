```
homology(curricula; <キーワード引数>)
```

`Curriculum`データオブジェクトのコレクションを入力として、各カリキュラムがコレクション内の他のカリキュラムとどれだけ類似しているか（共有コースの観点から）を示す視覚化を提供します。デフォルトのカラースキームでは、明るい色が高い類似性を示し、主対角線上の色（カリキュラム自身との類似性）は最大の類似性を示します。

# 引数

必須:

  * `curricula::Array{Curriculum,1}` : 類似性を分析するカリキュラムのコレクション。

キーワード:

  * `strict::Bool` : trueの場合、コースを厳密に一致させます（コースIDを含む）；false（デフォルト）の場合、コース名またはコースの接頭辞と番号のみを一致させます。
  * `title::AbstractString` : 視覚化の上に表示されるタイトル、デフォルトは空文字列です。
  * `xlabel::AbstractString` : x軸の下に表示されるラベル、デフォルトは空文字列です。
  * `ylabel::AbstractString` : y軸の左に表示されるラベル、デフォルトは空文字列です。
  * `color::Symbol` : 類似性値をプロットするために使用されるカラ―グラデーション、デフォルトはシンボル`:thermal`です。他のカラースキームのシンボル
  * `legend::Bool` : プロットの凡例を表示するかどうか、デフォルトは`false`です。

利用可能なカラ―ライブラリを確認するには、次のように入力します:

```julia-repl
julia> clibraries()
```

次に、これらのカラ―ライブラリのいずれかを選択し、次の関数を使用してそのライブラリで利用可能なカラースキームのシンボルを確認します。例えば、

```julia-repl
julia> showlibrary(:cmocean)
```
