```
visualize(degree_plan; <keyword arguments>))
```

学位プランを視覚化します。

# 引数

必須:

  * `degree_plan::DegreePlan` : 視覚化する学位プラン。

キーワード:

  * `changed` : コールバック関数の引数で、インターフェースを通じて学位プランが変更されるたびに呼び出されます。デフォルトは `nothing`。
  * `notebook` : ブール引数で、`true` に設定すると、Blinkは視覚化のために新しいウィンドウを作成せず、Jupyterノートブックの出力セルに表示されます。
  * `serve_local`: ブール引数で、`true` に設定すると、視覚化はlocalhostを通じて提供されます。それ以外の場合は、プリセットのリモートサーバー（インターネットが必要）を使用します。デフォルトは `false`。
  * `edit` : ブール引数で、`true` に設定すると、ユーザーは視覚化インターフェースを通じて学位プランを編集できます。デフォルトは `false`。
  * `output_file` : 編集された学位プランを保存するCSVファイルへの相対パスまたは絶対パス。デフォルトは `edited_degree_plan.csv`。
  * `show_delay` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップに遅延要因メトリックが表示されます。デフォルトは `true`。
  * `show_blocking` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップにブロッキング要因メトリックが表示されます。デフォルトは `true`。
  * `show_centrality` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップに中心性メトリックが表示されます。デフォルトは `true`。
  * `show_complexity` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップに複雑さメトリックが表示されます。デフォルトは `true`。
  * `scale` : 出力ウィンドウのサイズをスケーリングするために使用される実数値。
