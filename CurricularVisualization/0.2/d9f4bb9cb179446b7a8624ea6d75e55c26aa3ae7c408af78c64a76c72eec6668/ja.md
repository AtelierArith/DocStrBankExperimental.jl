```
 visualize(curriculum; <keyword arguments>))
```

カリキュラムを視覚化します。

# 引数

必須:

  * `curriculum::Curriculum` : 視覚化するカリキュラム。

キーワード:

  * `changed` : コールバック関数の引数で、インターフェースを通じてカリキュラムが変更されるたびに呼び出されます。デフォルトは `nothing`。
  * `notebook` : ブール引数で、`true` に設定すると、Blink は視覚化のために新しいウィンドウを作成せず、Jupyter ノートブックの出力セルに表示されます。
  * `serve_local`: ブール引数で、`true` に設定すると、視覚化はローカルホストを通じて提供されます。それ以外の場合は、プリセットのリモートサーバー（インターネットが必要）を使用します。デフォルトは `false`。
  * `edit` : ブール引数で、`true` に設定すると、ユーザーは視覚化インターフェースを通じてカリキュラムを編集できます。デフォルトは `false`。
  * `output_file` : 編集されたカリキュラムを保存するCSVファイルへの相対パスまたは絶対パス。デフォルトは `edited_curriculum.csv`。
  * `show_delay` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップに遅延要因メトリックが表示されます。デフォルトは `false`。
  * `show_blocking` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップにブロッキング要因メトリックが表示されます。デフォルトは `false`。
  * `show_centrality` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップに中心性メトリックが表示されます。デフォルトは `false`。
  * `show_complexity` : ブール引数で、`true` に設定すると、コースにマウスをホバーしたときにツールチップに複雑さメトリックが表示されます。デフォルトは `false`。
  * `scale` : 出力ウィンドウのサイズをスケーリングするために使用される実数値。
