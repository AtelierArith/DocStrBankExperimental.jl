```
tphaseplot(
    f :: "Complex -> Complex",
    limits = (-1, 1, -1, 1);
    pixels = (720, 720),
    real = false,
    imag = false,
    rect = false,
    angle = false,
    abs = false,
    polar = false,
    box = nothing,
    kwargs...
)
```

複素値関数を受け取り、[ColorCET](https://colorcet.com)のCBTC1サイクリックカラーマップを使用して、色覚異常のある視聴者向けの位相プロットを生成します。

赤は位相$0$、白は$\frac{\pi}{2}$、シアンは$\pi$、黒は$\frac{3\pi}{2}$に対応します。

# 引数

  * **`f`** はプロットする複素関数です。
  * **`limits`** はプロットする矩形の限界で、形式は`(minRe, maxRe, minIm, maxIm)`です。1つまたは2つの数値が提供された場合、実数軸と虚数軸に沿って対称的に取られます。

# キーワード引数

  * **`pixels`** は、実数軸と虚数軸で計算するピクセル数で、1つの数値が提供された場合は両方に同じ数が取られます。
  * **`real`** は、実数軸に対して直交する黒から白へのグラデーションを、単位増加ごとに1つのグラデーションの割合でプロットします。数値が設定されている場合は、代わりに幅として使用されます。
  * **`imag`** は、虚数軸に対して直交する黒から白へのグラデーションを、単位増加ごとに1つのグラデーションの割合でプロットします。数値が設定されている場合は、代わりに幅として使用されます。
  * **`rect`** は`real = true`および`imag = true`のショートカットです。
  * **`angle`** は、位相角に対して直交する黒から白へのグラデーションを、1回転あたり8つのグラデーションの割合でプロットします。異なる割合を指定するために整数に設定できます。
  * **`abs`** は、絶対値の自然対数の単位増加ごとに1つのグラデーションの割合で黒から白へのグラデーションをプロットします。数値が設定されている場合は、対数の底として使用されます。関数に設定された場合は、その出力の単位増加が代わりに使用されます。
  * **`polar`** は`angle = true`および`abs = true`のショートカットです。`abs`に使用する基準を設定することもでき、その場合は`angle`に適切な割合が選択されます。
  * **`box`** は`(a, b, s)`に設定されている場合、出力がボックス`a`と`b`の間にある領域を色`s`でシェーディングします。`(f, s)`に設定されている場合、色付けされた領域は`f(w) == true`によって定義されます。複数のボックスのリストでも構いません。

残りのキーワード引数はプロットバックエンドに渡されます。
