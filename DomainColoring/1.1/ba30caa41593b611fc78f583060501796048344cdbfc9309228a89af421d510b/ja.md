```
pdphaseplot(
    f :: "Complex -> Complex",
    limits = (-1, 1, -1, 1);
    pixels = (720, 720),
    real = false,
    imag = false,
    rect = false,
    angle = false,
    abs = false,
    polar = false,
    box = nothing,
    kwargs...
)
```

複素数値関数を受け取り、[ColorCET](https://colorcet.com)のCBC1サイクリックカラーマップを使用して、プロタノピックおよびデュタノピックの視聴者向けに位相プロットを生成します。

黄色は位相 $0$ に対応し、白は $\frac{\pi}{2}$、青は $\pi$、黒は $\frac{3\pi}{2}$ に対応します。

# 引数

  * **`f`** はプロットする複素関数です。
  * **`limits`** はプロットする長方形の限界で、形式は `(minRe, maxRe, minIm, maxIm)` です。1つまたは2つの数値が提供された場合、それらは実軸と虚軸に沿って対称的に取られます。

# キーワード引数

  * **`pixels`** は、実軸と虚軸で計算するピクセル数で、1つの数値が提供された場合は両方に同じ数が取られます。
  * **`real`** は、実軸に対して直交する黒から白へのランプを、単位増加ごとに1つのランプの割合でプロットします。数値が設定されている場合は、代わりに幅として使用されます。
  * **`imag`** は、虚軸に対して直交する黒から白へのランプを、単位増加ごとに1つのランプの割合でプロットします。数値が設定されている場合は、代わりに幅として使用されます。
  * **`rect`** は `real = true` および `imag = true` のショートカットです。
  * **`angle`** は、位相角に対して直交する黒から白へのランプを、1回転あたり8つのランプの割合でプロットします。異なる割合を指定するために整数に設定できます。
  * **`abs`** は、絶対値の自然対数の単位増加ごとに1つのランプの割合で黒から白へのランプをプロットします。数値が設定されている場合は、対数の底として使用されます。関数に設定された場合は、その出力の単位増加が代わりに使用されます。
  * **`polar`** は `angle = true` および `abs = true` のショートカットです。また、`abs` に使用する基準を設定することもでき、その場合は `angle` に適切な割合が選択されます。
  * **`box`** は `(a, b, s)` に設定されている場合、出力がボックス `a` と `b` の範囲内にある領域を色 `s` でシェードします。`(f, s)` に設定されている場合、色付けされた領域は `f(w) == true` によって定義されます。複数のボックスのリストでも構いません。

残りのキーワード引数はプロットバックエンドに渡されます。
