```
checkerplot(
    f :: "Complex -> Complex",
    limits = (-1, 1, -1, 1);
    pixels = (720, 720),
    real = false,
    imag = false,
    rect = false,
    angle = false,
    abs = false,
    polar = false,
    box = nothing,
    hicontrast = false,
    kwargs...
)
```

複素関数を取り、チェッカープロットを生成します。

# 引数

  * **`f`** はプロットする複素関数です。
  * **`limits`** はプロットする矩形の限界で、形式は `(minRe, maxRe, minIm, maxIm)` です。1つまたは2つの数値が提供された場合、それらは実軸と虚軸に沿って対称的に取られます。

# キーワード引数

  * **`pixels`** は、実軸と虚軸で計算するピクセル数で、1つの数値が提供された場合は両方に同じ数が取られます。

以下のオプションが設定されていない場合、プロットは `rect = true` にデフォルト設定されます。

  * **`real`** は、実軸に対して直交する黒と白のストライプを、単位の増加ごとに1本のストライプの割合でプロットします。数値が設定された場合は、幅として使用されます。
  * **`imag`** は、虚軸に対して直交する黒と白のストライプを、単位の増加ごとに1本のストライプの割合でプロットします。数値が設定された場合は、幅として使用されます。
  * **`rect`** は `real = true` と `imag = true` のショートカットです。
  * **`angle`** は、位相角に対して直交する黒と白のストライプを、1回転ごとに8本のストライプの割合でプロットします。異なる割合を指定するために整数に設定できます。
  * **`abs`** は、絶対値の自然対数の単位増加ごとに1本のストライプの割合で黒と白のストライプをプロットします。数値が設定された場合は、対数の底として使用されます。関数に設定された場合は、その出力の単位増加が代わりに使用されます。
  * **`polar`** は `angle = true` と `abs = true` のショートカットです。`abs` に使用する基準を設定することもでき、その場合は `angle` に適切な割合が選択されます。
  * **`box`** が `(a, b, s)` に設定された場合、出力がボックス `a` と `b` の範囲内にある領域を色 `s` でシェードします。`(f, s)` に設定された場合、色付けされた領域は `f(w) == true` によって定義されます。複数のボックスのリストでも構いません。
  * **`hicontrast`** は、柔らかいデフォルトの代わりに黒と白を使用します。

残りのキーワード引数はプロットバックエンドに渡されます。
