```
update!(m::MixedModel, re...)
```

混合モデルを更新して、ランダム効果の共分散行列を使用します。

`re`は[`create_re`](@ref)を使用して作成できます。

それらは`VarCorr(m)`で指定された順序で指定する必要があります。

!!! warning
    このメソッドを直接呼び出すことは推奨しません。代わりに、キーワード引数を使用して異なる`re`を名前で指定するメソッドを使用してください。


!!! note
    これは特定のパラメータベクトルと結果のモデルフィットをインストールするための便利な関数です。実際には最適化のタイプを実行しません。


# 詳細

`re`はモデルの`ReTerm`のλフィールドとして使用され、共分散行列の下Cholesky因子として指定する必要があります。
