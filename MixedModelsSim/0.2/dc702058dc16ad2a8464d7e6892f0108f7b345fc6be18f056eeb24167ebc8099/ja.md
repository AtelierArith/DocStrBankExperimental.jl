```
update!(m::MixedModel; namedre...)
update!(m::MixedModel; θ)
```

混合モデルを更新して、ランダム効果の共分散行列を使用します。

`namedre`は[`create_re`](@ref)を使用して作成できます。`namedre`はブロッキング変数の名前によって指定されます。例えば、`subj=create_re(...)`のように指定します。

!!! warning
    モデルの応答は、この関数を呼び出す前に非定数値に初期化されている必要があります。そうしないと、モデルの更新は`PosDefException`で失敗します。


!!! warning
    θをキーワード引数として直接設定することは非推奨です。


!!! note
    これは特定のパラメータベクトルとその結果としてのモデルフィットをインストールするための便利な関数です。実際には最適化のタイプを実行しません。


# 詳細

`re`はモデルの`ReTerm`のλフィールドとして使用され、共分散行列の下Cholesky因子として指定されるべきです。
