```
create_re(sigmas...; corrmat=Matrix{Float64}(I, length(sigmas), length(sigmas))
```

標準偏差と相関行列からランダム効果の共分散因子を作成します。

`sigmas`は、`VarCorr(m)`の出力におけるランダムスロープの順序と同じ順序で指定する必要があります。

!!! note
    `sigmas`は残差標準偏差に対して**相対的に**指定されます。絶対値は、仮定された残差標準偏差で割ることによってスケーリングする必要があります。


相関行列はデフォルトで単位行列、すなわちランダム効果間に相関がないことを意味します。

!!! note
    戻り値は共分散行列の下三角のコレスキー因子であり、これは[`update!`](@ref)が要求するものです。
