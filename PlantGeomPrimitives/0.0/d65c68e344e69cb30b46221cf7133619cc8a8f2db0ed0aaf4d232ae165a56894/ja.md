```
Mesh(nt, type)
```

`nt` 三角形を格納するのに十分なメモリを割り当てて三角形の密なメッシュを生成します。この動作は空のメッシュを生成するのと同等ですが、大量のプリミティブを追加する際に計算効率が向上する可能性があります。より低い浮動小数点精度が必要な場合は、`Mesh(10, Float32)` のようにオプションの第3引数として指定できます。

# 引数

  * `nt`: メモリを割り当てる三角形の数。
  * `type`: メッシュデータの浮動小数点精度タイプ（デフォルトは `Float64`）。

# 戻り値

頂点や法線を持たない `Mesh` オブジェクト。

# 例

```jldoctest
julia> m = Mesh(1_000);

julia> nvertices(m);

julia> ntriangles(m);

julia> Mesh(1_000, Float32);
```
