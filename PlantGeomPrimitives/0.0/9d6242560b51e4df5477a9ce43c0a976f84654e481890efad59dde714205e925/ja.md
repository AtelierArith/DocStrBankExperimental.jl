```
add_property!(m::Mesh, prop::Symbol, data, nt = ntriangles(m))
```

メッシュにプロパティを追加します。プロパティは名前（`prop`）で識別され、三角形ごとに値の配列（`data`）として保存されます。プロパティがすでに存在する場合、新しいデータは既存のプロパティに追加されます。そうでない場合は、新しいプロパティが作成されます。`data`に単一のオブジェクトを渡すことも可能で、その場合、プロパティはすべての三角形に対して同じ値に設定されます。

# 引数

  * `mesh`: プロパティを追加するメッシュ。
  * `prop`: 追加されるプロパティの名前（`Symbol`）。
  * `data`: プロパティに追加されるデータ（配列または単一の値）。
  * `nt`: `data`が配列でない場合に仮定される三角形の数。デフォルトでは、これはメッシュ内の三角形の数です。

# 戻り値

更新されたプロパティを持つメッシュ。

# 例

```jldoctest
julia> r = Rectangle();

julia> add_property!(r, :absorbed_PAR, [0.0, 0.0]);
```
