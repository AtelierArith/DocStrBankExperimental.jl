```
Trapezoid(;length = 1.0, width = 1.0, ratio = 1.0)
```

`length` と大きい `width` および小さい幅と大きい幅の間の `ratio` によって与えられた寸法を持つ台形を作成します。台形は標準の位置と向きで生成されます。

# 引数

  * `length = 1.0`: 台形の長さ。
  * `width = 1.0`: 台形の大きい幅（台形の下底）。
  * `ratio = 1.0`: 小さい幅と大きい幅の間の比率。

# 例

```jldoctest
julia> Trapezoid(;length = 1.0, width = 1.0, ratio = 1.0);
```
