```
Ellipse(;length = 1.0, width = 1.0, n = 20)
```

与えられた `length` と `width` によって指定された楕円を近似する三角形メッシュを作成し、`n` 三角形（偶数でなければならない）に離散化し、標準の位置と向きを持ちます。

# 引数

  * `length = 1.0`: 楕円の長さ。
  * `width = 1.0`: 楕円の幅。
  * `n = 20`: メッシュに使用される三角形の数。

# 例

```jldoctest
julia> Ellipse(;length = 1.0, width = 1.0, n = 20);
```
