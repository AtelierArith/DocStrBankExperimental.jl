```
HollowCube(;length = 1.0, width = 1.0, height = 1.0)
```

標準の位置と向きを持つ、`length`、`width`、`height` で指定された寸法の中空の立方体（プリズム）を作成します。

# 引数

  * `length = 1.0`: 立方体の長さ。
  * `width = 1.0`: 立方体の基底の幅。
  * `height = 1.0`: 立方体の基底の高さ。

## 例

```jldoctest
julia> HollowCube(;length = 1.0, width = 1.0, height = 1.0);
```
