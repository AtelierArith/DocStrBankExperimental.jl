```
get_triangle(m::Mesh, i)
```

メッシュ内の i 番目の三角形の頂点を取得します。

# 引数

  * `mesh`: 三角形を取得するメッシュ。
  * `i`: 取得する三角形のインデックス。

# 戻り値

i 番目の三角形を定義する三つの頂点を含むベクトル。

# 例

```jldoctest
julia> v = [Vec(0.0, 0.0, 0.0), Vec(0.0, 1.0, 0.0), Vec(1.0, 0.0, 0.0),
            Vec(0.0, 0.0, 0.0), Vec(0.0, 1.0, 0.0), Vec(0.0, 0.0, 1.0)];

julia> m = Mesh(v);

julia> get_triangle(m, 2);
```
