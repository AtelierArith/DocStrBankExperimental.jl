```
time_heuristic(g::OSMGraph)
```

最短経路計算に使用されるヒューリスティック関数を返します。これは、`weight_type=:time` または `weight_type=:lane_efficiency` のグラフで使用する必要があります。ヒューリスティック関数は2つの引数を受け取り、`u` は現在のノード、`v` は隣接ノードを表し、両者の間の推定移動時間を返します。これは、ハーヴァーサイン距離を固定の最大速度 `100` で割ることによって計算されます。最適な経路を達成するためには、`goal` までの残りのコストを最もよく推定する*過小評価*のヒューリスティックを選ぶことが重要であるため、すべての道の中で最大の最大速度を選択します。
