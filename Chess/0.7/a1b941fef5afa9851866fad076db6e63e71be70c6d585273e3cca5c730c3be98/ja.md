```
bishopattacks(b::Board, s::Square)
```

このボード上で、`s`のマスにいるビショップが攻撃するマスの集合。

空のマスと敵または味方の駒が占めるマスの両方が集合に含まれます。

# 例

```julia-repl
julia> b = fromfen("5k2/8/4q3/8/2B5/8/4P3/3K4 w - -");

julia> pprint(b, highlight=bishopattacks(b, SQ_C4))
```
