```
rookattacks(b::Board, s::Square)
```

このボード上で、`s`のマスにいるルークが攻撃するマスの集合。

空のマスと敵または味方の駒が占有しているマスの両方が集合に含まれます。

# 例

```julia-repl
julia> b = fromfen("2r2k2/8/8/8/2R3P1/8/4P3/3K4 w - -");
```
