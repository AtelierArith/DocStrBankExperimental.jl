```
pprint(b::Board, color = false, highlight = SS_EMPTY, unicode = false)
```

`Board`を標準出力にきれいに表示します。

24ビットカラーサポートのある端末では、`color = true`を使用して色付きのボードを表示します。`highlight`パラメータを使用して、ハイライトしたい`SquareSet`を含めます。

フォントと端末がサポートしている場合は、Unicodeピース出力のために`unicode = true`を使用します。
