```
queenattacks(b::Board, s::Square)
```

このボード上で、`s`の位置にいるクイーンが攻撃することになるマスの集合。

空のマスと敵または味方の駒が占有しているマスの両方が集合に含まれます。

julia> b = fromfen("2r2k2/8/8/8/2Q3P1/8/4P3/3K4 w - -");

julia> pprint(b, highlight=queenattacks(b, SQ_C4))
