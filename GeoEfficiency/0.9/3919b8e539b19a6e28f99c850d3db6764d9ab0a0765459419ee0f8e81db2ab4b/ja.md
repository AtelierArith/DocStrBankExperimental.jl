# GeoEfficiencyパッケージ

放射線検出器の一般的な形状（円筒、ボアホール、井戸型）に対して、ソースから見た`幾何学的効率`をバッチまたは個別に計算するための迅速かつ柔軟なツールを紹介します。ソースは点、円盤、またはシリンダーである可能性があります。

# クイック使用法

  * geoEff()	: 1つの幾何学的セットアップの幾何学的効率を計算し、幾何学的効率の値のみを返します。
  * calc() 	: 1つの幾何学的セットアップの幾何学的効率を計算し、コンソールに完全な情報を表示します。
  * calcN()	: 幾何学的セットアップの幾何学的効率を計算し、ユーザーが終了するまでコンソールに完全な情報を表示します。
  * batch()	: **`/home/terasaki/GeoEfficiency`**フォルダー内のデータを使用して、バッチモードで幾何学的効率を計算します。

**詳細情報や更新については、[`GitHub.com`](https://github.com/DrKrar/GeoEfficiency.jl/)のリポジトリを参照してください。**
