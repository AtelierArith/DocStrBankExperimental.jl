```
ppr_diffusion(g::GNNGraph{<:COO_T}, alpha =0.85f0) -> GNNGraph
```

GNNGraphのエッジ重み行列に基づいてパーソナライズドページランク（PPR）拡散を計算し、PPR行列から導出された新しいエッジ重みでグラフを更新します。参考文献: [The pagerank citation ranking: Bringing order to the web](http://ilpubs.stanford.edu:8090/422)

この関数は以下のステップを実行します：

1. グラフのエッジ重みを使用して修正された隣接行列 `A` を構築します。ここで、`A` は `(α - 1) * A + I` によって調整され、`α` は減衰係数（`alpha_f32`）で、`I` は単位行列です。
2. 各列の合計が1になるように `A` を正規化し、遷移確率を表します。
3. PPRの式 `α * (I + (α - 1) * A)^-1` を適用して拡散行列を計算します。
4. PPR拡散行列に基づいてグラフの元のエッジ重みを更新し、PPR行列から各エッジの新しい重みを割り当てます。

# 引数

  * `g::GNNGraph`: PPR拡散を計算するための入力グラフ。エッジ重みが利用可能である必要があります。
  * `alpha_f32::Float32`: PPR計算に使用される減衰係数で、ランダムウォークにおけるテレポート確率を制御します。デフォルトは `0.85f0` です。

# 戻り値

  * PPR拡散計算に従って更新されたエッジ重みを持つ、`g` と同じ構造の新しい `GNNGraph` インスタンス。
