```
parameter_observed(indp, sym)
```

`indp`における`sym`の観測関数を返します。この関数は[`observed`](@ref)と同様に機能しますが、返される関数の引数として`u`は含まれません。時間依存システムの場合、返される関数は`(p, t) -> [values...]`というシグネチャを持つ必要があります。時間非依存システムの場合、返される関数は`(p) -> [values...]`というシグネチャを持つ必要があります。

デフォルトでは、この関数は`nothing`を返し、インデックスプロバイダーがパラメータ観測関数の生成をサポートしていないことを示します。
