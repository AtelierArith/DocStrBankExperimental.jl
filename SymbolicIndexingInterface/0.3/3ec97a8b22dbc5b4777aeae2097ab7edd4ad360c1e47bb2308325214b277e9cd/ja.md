```
get_all_timeseries_indexes(indp, sym)
```

シンボリック変数 `sym` の変数のすべてのユニークなタイムシリーズインデックスの `Set` を返します。 `sym` はシンボリック変数または式、シンボリックの配列、インデックス、またはインデックスの配列である可能性があります。連続変数は [`ContinuousTimeseries`](@ref) タイムシリーズインデックスに対応します。非タイムシリーズパラメータにはタイムシリーズインデックスがありません。タイムシリーズパラメータは [`timeseries_parameter_index`](@ref) によって返されるのと同じタイムシリーズインデックスを持ちます。独立変数は `ContinuousTimeseries` タイムシリーズインデックスに対応することに注意してください。

あいまいさは変数の有利に解決されるべきです。たとえば、`1` がインデックス `1` の変数を指す場合とインデックス `1` のパラメータを指す場合がある場合、それは変数として解釈されるべきです。

デフォルトでは、この関数は `Set([ContinuousTimeseries()])` を返します。
