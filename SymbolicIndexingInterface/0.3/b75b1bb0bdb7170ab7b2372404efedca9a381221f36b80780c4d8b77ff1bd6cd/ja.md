```
is_markovian(indp)
```

インデックスプロバイダーがマルコフシステムを表すかどうかを確認します。マルコフシステムは、シミュレーションを行うために過去の状態の知識を必要としません。この関数は、`is_time_dependent(indp)` が `true` を返すインデックスプロバイダーにのみ適用されます。

非マルコフインデックスプロバイダーは、異なるシグネチャの [`observed`](@ref) 関数を返します。非マルコフインデックスプロバイダーに関連付けられたすべての値プロバイダーは、[`get_history_function`](@ref) を実装する必要があります。

デフォルトでは `true` を返します。
