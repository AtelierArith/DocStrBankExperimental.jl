```
 DoubleBendingPowerLaw_Bis(α₀, f₁, Δα₁, Δf, Δα₂)

パワースペクトル密度のための二重ベンディングパワーローモデル
```

  * `α₀`: 最初のパワーロー指数
  * `f₁`: 最初のベンド周波数
  * `Δα₁`: 最初のパワーロー指数の差
  * `Δf`: 第二のベンド周波数のスケール、`f₂ = f₁ * Δf`
  * `Δα₂`: 第二のパワーロー指数の差

$$
\mathcal{P}(f) =  \frac{(f/f₁)^{-α_0}}{1 + (f / f₁)^{α_0+\Delta α₁}}\frac{1}{1 + (f / f₁ \Delta f)^{\Delta α₁ + \Delta α₂}}
$$
