```
bopik(xae, xes)
```

オーピックの衝突パラメータベクトル $\mathbf{B}$ の座標を計算します。ハイパーボリックな惑星の接近時における衝突パラメータベクトル。以下のフィールドを持つ名前付きタプルを返します：

  * `ξ` = $\mathbf{B}\cdot\hat{\mathbf{\xi}}/R_E$ および `ζ` = $\mathbf{B}\cdot\hat{\mathbf{\zeta}}/R_E$、ここで $\mathbf{B}$ は衝突パラメータベクトル、$(\hat{\mathbf{\xi}}, \hat{\mathbf{\zeta}})$ はオーピックのフレーム、$R_E$ は地球の半径（au単位）です。
  * `U` は以下のフィールドを持つ別の名前付きタプルです：

      * `y` = $U_y$。
      * `norm` = $||\mathbf{U}||$、ここで $\mathbf{U}$ は惑星中心の速度ベクトル（km/s単位）です。
  * `b` = $b_E$、ここで $b_E$ は地球の衝突断面積（「クリティカルB」）です。[`crosssection`](@ref)を参照してください。

# 引数

  * `xae`: 小惑星の地心位置/速度ベクトル（au単位、au/日単位）で、最接近時のもの。
  * `xes`: 小惑星の最接近時における惑星の太陽中心位置/速度ベクトル（au単位、au/日単位）。

!!! reference
    https://doi.org/10.1007/s10569-019-9914-4 の14ページの式（37）-（38）を参照してください。
