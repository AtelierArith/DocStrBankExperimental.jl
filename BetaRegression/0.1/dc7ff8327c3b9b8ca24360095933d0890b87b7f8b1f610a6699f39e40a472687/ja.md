```
BetaRegressionModel{T,L1,L2,V,M} <: RegressionModel
```

ベータ分布に従う応答値の回帰モデルを表す型で、開区間 (0, 1) にあります。これは Ferrari と Cribari-Neto (2004) によって説明されています。

平均応答は、リンク関数の型 `L1 <: Link01` によって線形予測子に関連付けられます。すなわち、リンクは $(0, 1) \mapsto \mathbb{R}$ にマッピングされ、GLM パッケージのリンク関数のインターフェースを使用する必要があります。GLM に対してのようにベータ回帰モデルに対する標準的なリンク関数は存在しませんが、ロジットが最も一般的な選択肢です。

精度は、型 `L2 <: Link` のリンク関数によって変換され、$\mathbb{R} \mapsto \mathbb{R}$ または理想的には $(0, \infty) \mapsto \mathbb{R}$ にマッピングされる必要があります。なぜなら、精度は正でなければならないからです。最も一般的な選択肢は、恒等、対数、および平方根のリンクです。
