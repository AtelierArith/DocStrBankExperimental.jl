```
random_sequence(motifs::Vector{M}, q, limits, translate, δq = 0; kwargs...)
```

モチーフのタイプ `M` のランダムなシーケンスを作成します。このシーケンスは、"length" `ℓ` が `q - δq ≤ ℓ ≤ q + δq` の範囲内で**正確に**なければなりません。シーケンス自体と、それを作成するために使用された `motifs` のインデックスのシーケンスを返します。確率のベクトル `weights` をキーワード引数として指定することができ、これにより作成される初期シーケンスの各 `motifs` のエントリのサンプリング確率が決まります。

ここでの "length" は、`limits` および `translate` 関数に基づいて `M` 構造体によって定義された抽象的な長さを意味します。これは要素の数を指すものではありません！

`M` は、次の2つの関数があれば何でも構いません。

  * `limits(motif)` : モチーフを与えると、そのモチーフの `(start, fine)` を `q` と同じ単位で返す関数です。この関数は長さの尺度を確立し、単に `fine - start` となります。
  * `translate(motif, t)` : モチーフを与えると、`t`（負または正のいずれか）によって変換された*新しい*モチーフを返す関数で、`q` と同じ単位に関してです。

## その他のキーワード

アルゴリズムの完全な説明については、ソースコードを参照してください（`@which` を使用）。

  * `tries = 5` : 受け入れられる初期ランダムシーケンスの最大数。
  * `taulcut = 2` : 初期推測から要素が削除される最大回数。
  * `summands = 3` : シーケンスを完成させるために組み合わされる可能性のあるモチーフの最大数。
