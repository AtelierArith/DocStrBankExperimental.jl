```
dnPl(x, l::Integer, n::Integer, [cache::AbstractVector])
```

$$
P_\ell(x)
$$

の$n$-階導関数$d^n P_\ell(x)/dx^n$を計算します。オプションで、事前に割り当てられたベクトル`cache`を提供することができ、これは最小長さ`l - n + 1`を持ち、計算中に上書きされる可能性があります。

導関数の次数`n`は非負でなければなりません。`n == 0`の場合、この関数は単にレジェンドル多項式を返します。

# 例

```jldoctest
julia> dnPl(0.5, 3, 2) # x = 0.5におけるP3(x)の2階導関数
7.5

julia> dnPl(0.5, 4, 0) == Pl(0.5, 4) # ゼロ階導関数 == Pl(x)
true
```
