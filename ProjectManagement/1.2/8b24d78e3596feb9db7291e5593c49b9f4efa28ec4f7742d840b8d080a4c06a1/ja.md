```
Project(task_durations, links)
```

  * `task_durations::NamedTuple`: タスクの名前がキーで、値がタイミングを示す名前付きタプル。

      * タイミングの値は、固定の期間を表す `Real` 値（例えば 5.0）であるか、確率的にのみ知られている期間を表すサンプル可能なものである必要があります。すなわち、`rand(rng, dur)::Real` が定義されている任意のオブジェクト `dur` です。例えば、任意のコレクションや分布オブジェクトです。特に関連するのは、PERT推定の標準として使用される `PertBeta(min, mode, max)` です。
  * `links::Vector{Pair{Symbol, Symbol}}`: タスクのすべての依存関係のペアのコレクション。例えば `[:a => :b, :a=>:c, :c=>:d]`。これを書く便利な方法は、しばしばブロードキャスティングと行列表記を使用して vcat を使うことです: `[:a .=> [:b, :c]; :c => :d]`。

タスク名 `start` と `finish` は特別で、プロジェクトの開始と終了のマイルストーンを表します。これらは `task_durations` に期間 `0` で含まれている必要があり、それぞれ `links` で記述されたグラフのルートであり唯一の葉でなければなりません。
