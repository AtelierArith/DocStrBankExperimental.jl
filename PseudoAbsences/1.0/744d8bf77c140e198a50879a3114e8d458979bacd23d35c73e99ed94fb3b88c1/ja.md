```
pseudoabsencemask( ::Type{DistanceToEvent}, presences::SDMLayer{Bool}; f = minimum, )
```

イベントまでの距離法を使用して擬似欠損のマスクを生成します。候補セルは、既知の観察からの距離に応じて重み付けされ、遠くの場所ほど可能性が高くなります。距離の分布によっては、このレイヤーを `log` や指数を使用して平坦化することが非常に良いアイデアかもしれません。`f` 関数は、報告される距離を決定するために使用されます（デフォルトは `minimum` ですが、他の関連する関数を使用することもできます）。
