```
^(g::Perm, n::Integer)
```

順列 `g` の $n$ 乗を返します。

デフォルトでは、`n > 3` の場合、`g^n` は `g` のサイクル分解によって計算されます。`Generic.power_by_squaring` は、特定のケースに応じて速い場合もあればそうでない場合もある、異なる累乗方法を提供します。サイクル構造のキャッシュにより、`g` の繰り返し累乗はデフォルトの方法でより速くなります。

# 例

```jldoctest
julia> g = Perm([2,3,4,5,1])
(1,2,3,4,5)

julia> g^3
(1,4,2,5,3)

julia> g^5
()
```
