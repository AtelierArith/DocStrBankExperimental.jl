```
spinloop()
```

`spinloop()` がスピンループの中にあることをコンパイラ、ランタイム、およびハードウェアに示すヒントです。現在のスレッドが前進するためには、他のワーカースレッドが前進する必要があるスピンループ内でこれを呼び出してください。

!!! warning
    上記の文は、特にワーカー *スレッド* に言及しており、*`Task`* には言及していないことに注意してください。Julia プログラマーは、API がスレッドについて話すときは常に警戒すべきです。チャネルや条件変数などの高レベルの API を好んで使用してください。

    `spinloop` の適切な使用には、Julia スケジューラで待機するフォールバックや、他のスレッドが存在し、スピンループを中断できるタスクが実行中または最終的にスケジュールされることを考慮したメカニズムなど、追加の注意が必要です。
