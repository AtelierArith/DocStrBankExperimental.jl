```
bravaistype(sgnum::Integer, D::Integer=3; normalize::Bool=false)  -->  String
```

`sgnum`のブレヴァイタイプを次元`D`で文字列として返します（単一文字の結晶略語と中心タイプの連結として）。

## キーワード引数

  * **`normalize`:** `sgnum`に関連付けられた中心タイプが`'A'`の場合、キーワード引数`normalize`（デフォルトは`false`）に応じて、中心タイプ`'C'`に「正規化」することを選択できます。`'A'`と`'C'`の中心の違いは基底の変更に過ぎないためです。`normalize=true`の場合、私たちは14の標準的なブレヴァイタイプのみを持つことになります。すなわち、`unique(bravaistype.(1:230, 3), normalize=true)`は15ではなく、14の異なるタイプのみを返します。

    これは空間群38-41にのみ影響を与え（それらの従来のブレヴァイタイプを`"oA"`から`"oC"`に正規化します）。
