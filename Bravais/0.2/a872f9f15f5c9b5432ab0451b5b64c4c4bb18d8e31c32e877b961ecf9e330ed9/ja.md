```
directbasis(sgnum, D=3;    abclims, αβγlims)
directbasis(sgnum, Val(D); abclims, αβγlims) --> DirectBasis{D}
```

空間群番号 `sgnum` と次元 `D` に対応する結晶のランダムな（従来の）`DirectBasis` を返します。格子ベクトルの自由パラメータはランダムに選ばれ、`abclims`（長さ）および `αβγlims`（角度）でオプションの制限が供給されます。慣例として、最初の格子ベクトル（`a`）の長さは単位に設定され、2 番目と 3 番目の格子ベクトル（`b` と `c`）の長さは最初のものに対して相対的です。

長さ `b` と `c` の相対的な一様分布に関する制限は 2 タプルのキーワード引数 `abclims` として指定できます。同様に、角度 `α`、`β`、`γ` に関する制限は αβγlims を介して設定できます（斜方、単斜、三斜格子にのみ影響します）。
