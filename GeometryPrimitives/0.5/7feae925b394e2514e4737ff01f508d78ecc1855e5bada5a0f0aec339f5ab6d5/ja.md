```
volfrac(voxel, nout, r₀)
```

ボリュームフラクション `rvol = vol(voxel ⋂ half-space) / vol(voxel)` を返し、ボクセルの体積のどれだけの部分が平面によって定義された半空間に含まれているかを示します。結果は0.0から1.0の間で、両端を含みます。半空間の内側と外側は `nout` によって決定され、これは外向きの法線です。（点が平面の `nout` 側にある場合、それは外側にあります。）

ボクセルは直交座標軸に整列している必要があり、ボクセルのx、y、z境界を指定する `v = ([xn,yn,zn], [xp,yp,zp])` で記述されます。半空間は境界平面によって記述されます。境界平面は外向きの法線ベクトル `nout = [nx, ny, nz]` と平面上の点 `r₀ = [rx, ry, rz]` によって記述されます。`nout` は正規化されている必要はありません。
