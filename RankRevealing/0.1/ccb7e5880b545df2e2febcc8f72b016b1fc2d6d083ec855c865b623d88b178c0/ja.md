```
pluq(A) -> PLUQ
```

行列 `A` のランク感度のあるLU因子分解を実行します。`n` 行 `m` 列の行列の因子分解は、`O(n m r^(ω-2))` ステップで計算され、ここで `r = rank(A)` であり、`ω` は行列の乗算の複雑さの指数です。

これは、基本的な算術演算 `+`, `-`, `*`, `/`, `one`, `zero` を実装する任意の型で動作する正確な関数です。

得られた因子分解 `F` は次の式を満たします。

```
F.P * [F.L ; F.M] * [F.U F.V] * F.Q == A
```

アルゴリズムの元の説明については[こちらのリンク](https://arxiv.org/pdf/1301.4438.pdf)を参照してください。
