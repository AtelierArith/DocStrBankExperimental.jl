```
PLUQ <: Factorizarion
```

任意の矩形行列 `A` の完全な `LU` 因数分解のための行列因数分解型です。これは [`pluq`](@ref) の戻り値の型です。

因数分解は次の式を満たします。

```
P * [L ; M] * [U V] * Q == A
```

| コンポーネント | 説明              |
|:------- |:--------------- |
| `F.L`   | フルランクの単位下三角部分   |
| `F.U`   | フルランクの上三角部分     |
| `F.M`   | 行次元の残り          |
| `F.V`   | 列次元の残り          |
| `F.P`   | 行ごとの置換 `Matrix` |
| `F.Q`   | 列ごとの置換 `Matrix` |
| `F.p`   | 行ごとの置換 `Vector` |
| `F.q`   | 列ごとの置換 `Matrix` |

反復と分解は、コンポーネントを `p`, `L`, `U`, `V`, `M`, q の順序で生成します。

`L` と `U` は、`A` と同じランクを持つ正方形のフルランク行列です。
