```
GeneralizedRankRevealing <: Factorization
```

行列 `A` と `B` の一般化ランクレベリング分解のための行列因子分解型で、同じ列数を持ちます。これは関数 `grr(A,B)` を介して得られます。

`A` と `B` の行空間の和の次元を `d_+` と呼ぶと、この因子分解は行列を次のように分解します。

```
                | I 0 0 |
| A | = | X 0 | | 0 0 I |
| B |   | 0 Y | | 0 I 0 | H
                | 0 0 I |
```

| コンポーネント | 説明                     |
|:------- |:---------------------- |
| `F.X`   | `m_A x r_A` フルコラムランク行列 |
| `F.Y`   | `m_B x r_B` フルコラムランク行列 |
| `F.H`   | `d_+ x n` フルローランク行列    |

他の便利なフィールドは次のとおりです：

  * `F.r1`: `A` のランクから交差部分を引いたもの
  * `F.r2`: `B` のランクから交差部分を引いたもの
  * `F.r3`: 交差部分のランク
