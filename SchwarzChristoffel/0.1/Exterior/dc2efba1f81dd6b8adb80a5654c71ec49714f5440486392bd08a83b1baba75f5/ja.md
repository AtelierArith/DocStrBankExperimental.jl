```
summary(m::ConformalMap)
```

共形写像のデータの要約を返します

# 例

```jldoctest
julia> p = Polygon([-1.0,0.2,1.0,-1.0],[-1.0,-1.0,0.5,1.0]);

julia> m = ExteriorMap(p);

julia> summary(m)
シュワルツ・クリストッフェル写像：単位円から4つの頂点を持つ多角形の外部への写像
   頂点: (-1.0,-1.0), (0.2,-1.0), (1.0,0.5), (-1.0,1.0),
   内角/π: 0.5, 0.656, 0.422, 0.422,
   円上の前頂点: (1.0,0.0), (0.3764,-0.9265), (-0.9024,-0.4309), (-0.1868,0.9824),
   前頂点の角度/π: -0.7291, -0.3519, 0.1291, 0.7111,
   定数 = 0.6722 + 0.7669im, 精度 = 1.0e-8,
   多重極係数の数 = 100
```
