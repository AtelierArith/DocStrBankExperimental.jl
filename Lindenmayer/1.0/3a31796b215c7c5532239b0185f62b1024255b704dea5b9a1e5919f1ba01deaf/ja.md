Lindenmayerシステムは、再帰的なパターンを構築するためのルールのセットです。Lindenmayer.jlでは、LSystem構造は以下で構成されています：

  * ルール：検索と置換の文字列のセット
  * シード：システムの初期状態を定義する文字列
  * 状態：現在の状態（初期は空、システムが評価されると追加される）

次のようにL-Systemを定義できます：

```
using Lindenmayer
koch = LSystem(["F" => "F+F--F+F"], "F")
```

これは、1つのルールがあることを示しています。「F」を検索し、各反復で「F+F–F+F」に置き換えます。最初の「シード」は、単一の「F」からなる初期状態です。

# 拡張ヘルプ

LSystemを描画するために、Luxor.jlのTurtleを使用します。これは、ルール内の文字を命令やコマンドとして解釈します。例えば、「F」は`Luxor.Forward()`に変換され、`+`は時計回りに回転し、`-`は反時計回りに回転します。

```
drawLSystem(LSystem(["F" => "5F+F--F+Ftt"], "F"),
    startingx = -400,
    forward = 4,
    turn = 80,
    iterations = 6)
```

`drawLSystem`のキーワードオプションには以下が含まれます：

```
forward=15,
turn=45,
iterations=6,
filename="/tmp/lsystem.pdf",
width=1000,
height=1000,
startingpen=(0.3, 0.6, 0.8), # 開始色 RGB
startingx=0,
startingy=0,
startingorientation=0,
showpreview=true
```

LSystemルールで認識される文字は以下の通りです。

F - 前に進む

G - Fと同じ

B - 後ろに進む

V - Bと同じ

f - 半歩前に進む

b - 180°回転して半歩前に進む

U - ペンを上げる（描画を停止）

D - ペンを下ろす（描画を開始）

`+` - 角度で回転

`-` - 角度で反時計回りに回転

r - 10°、15°、30°、45°、または60°でランダムに回転

T - 色相をランダムに変更

t - 色相を5°シフト

c - 彩度をランダム化

O - ランダムな不透明度の値を選択

l - ステップサイズを1増加

s - ステップサイズを1減少

5 - 線の幅を5に設定

4 - 線の幅を4に設定

3 - 線の幅を3に設定

2 - 線の幅を2に設定

1 - 線の幅を1に設定

n - 線の幅を0.5に設定

o - 半径をステップ/4とする円を描く

q - 辺の長さをステップ/4とする正方形を描く

`@` - 5°回転

`&` - -5°回転

[ - 現在の状態をスタックにプッシュ

] - 現在の状態をスタックからポップ

`*` - `asteriskfunction()`として渡された任意の関数を実行
