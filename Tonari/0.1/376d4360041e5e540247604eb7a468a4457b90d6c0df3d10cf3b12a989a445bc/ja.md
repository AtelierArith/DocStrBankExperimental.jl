```
timmer_koenig(psd, rng::random.AbstractRNG, alternative=false)
```

与えられたパワースペクトル密度（PSD）を使用して時間系列を生成する[1995A&A...300..707T](@cite)メソッド

1. パワースペクトル密度（PSD）𝓟のN値を与える
2. 標準正規分布から2N値を引く。
3. ランダム化された周期グラムの振幅はA = N + i Mで与えられる
4. ランダム化された周期グラムは𝓟_rand = √(𝓟 / 2) * Aで与えられる
5. ランダム化された周期グラムの最初の値を挿入し、0に設定する
6. 時間系列は、ランダム化された周期グラムの逆フーリエ変換を取ることによって得られ、関数FFTW.irfftを使用する。時間系列の長さは2*(N+1)-1を使用する。

代替パラメータ化は次のように与えられる

1. パワースペクトル密度（PSD）𝓟のN値を与える
2. χ²(2)分布からN-1値を引き、χ²₁(1)から1値を引く。これがランダム化された周期グラムの振幅Aである
3. 0と1の間の一様分布からN値を引き、最後の値を0に設定する。これがランダム化された周期グラムの位相θである
4. ランダム化された周期グラムは𝓟_rand = √(𝓟 / 2 * A) * exp(2πiθ)で与えられる
5. ランダム化された周期グラムの最初の値を挿入し、0に設定する
6. 時間系列は、ランダム化された周期グラムの逆フーリエ変換を取ることによって得られ、関数FFTW.irfftを使用する。時間系列の長さは2*(N+1)-1を使用する。

# 引数

  * `𝓟::Array{Float64, 1}`: プロセスに関連するPSD
  * `rng::Random.AbstractRNG`: ランダム数生成器。
  * `α::Real`: ランダム化された周期グラムの乗数因子。デフォルトは1.0。これは時間系列に位相を追加するための複素ベクトルである可能性がある。

# 戻り値

  * `x::Array{Float64, 1}`: 与えられたPSDを持つ時間系列。
