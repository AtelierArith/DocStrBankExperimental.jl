```
adapt!(
    chain, 
    options; 
    t_rate = .20, 
    kwargs...
)
```

ターゲット受け入れ率を達成するために半径を反復的に適応させます。半径は次の因子 `c` に従って調整されます：

```julia
c = exp(λ * d_rate)
```

ここで `λ` は適応率、`d_rate` は受け入れ率とターゲット受け入れ率の差です。

# 引数

  * `chain`: パラメータ空間を探索するためのチェーン
  * `options`: アルゴリズムを構成するためのオプションのセット

# キーワード引数

  * `t_rate = .20`: ターゲット受け入れ率
  * `kwargs...`: 処理されないキーワード引数
