```
Interpolator{T, S, F}
Interpolator(schedule, rate, ceil_fn = x -> ceil(Int, x))
```

出力が `schedule(t / rate)` であるスケジュール（すなわち、`schedule(t)` を補間します）。

これは、コードが固定レートで実数を反復処理する場合（例えば、固定時間ステップの微分方程式ソルバーで）に便利ですが、整数を離散的に反復処理するスケジュールを使用したい場合に役立ちます。

エポックの単位で `schedule` を指定し、ミニバッチの単位でそれを反復処理するためにも使用できます。

`ceil_fn` を指定して、`t / rate` に天井（または床）関数を適用します。
