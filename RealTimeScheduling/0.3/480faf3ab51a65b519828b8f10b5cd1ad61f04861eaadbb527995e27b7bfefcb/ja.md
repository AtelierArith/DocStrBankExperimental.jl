```
schedule_gfl(T, m, time; kill=false)
```

タスクシステム `T` の先行的グローバル公正遅延 (GFL) スケジュールを `m` プロセッサで指定された `time` の間シミュレートします。これは、準拠ベクトル分析の下での任意のGEDFのようなスケジューラの中で最も低い遅延境界を提供します。詳細については、Ericksonの「ソフトリアルタイムシステムにおける遅延境界とオーバーロードの管理」を参照してください。 DOI: [10.17615/fvp3-q039](https://doi.org/10.17615/fvp3-q039)。

`kill` が `true` の場合、ジョブは期限に達した時点でまだ完了していない場合に終了されます。

より一般的なグローバルスケジューリングについては、[`schedule_global`](@ref) を参照してください。
