```
normalize!(x::AbstractMatrix; nthreads=false) -> AbstractMatrix
```

対行列`x`を`インプレース`で正規化し、修正された`x`を返します。

# 引数

  * `x`: 行列
  * `x`: 行列
  * `nthreads`: 正規化にマルチスレッドを使用するかどうかを決定します。デフォルトはいいえです。

# 注意

あまり大きくない行列（10,000 × 10,000未満）に対しては、マルチスレッドを使用しないことをお勧めします。これは、マルチスレッドには追加のメモリと時間のオーバーヘッドがあるためです。
