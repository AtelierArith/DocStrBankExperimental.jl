```
±(C::MT, S::MT) where {T, MT<:AbstractMatrix{T}}
```

中心と半径がそれぞれ行列 `C` と `S` で与えられる区間行列を返します。

### 入力

  * `C` – 中心行列
  * `S` – 半径行列

### 出力

区間行列 `M` を返します。ここで `M[i, j]` は中心が `C[i, j]` で半径が `S[i, j]` の区間に対応します。すなわち、$M = C + [-S, S]$ です。

### 注意事項

半径行列は非負である必要があります。すなわち、各 `i` と `j` に対して `S[i, j] ≥ 0` です。

### 例

```jldoctest
julia> [1 2; 3 4] ± [1 2; 4 5]
2×2 IntervalMatrix{Float64, Interval{Float64}, Matrix{Interval{Float64}}}:
  [0.0, 2.0]   [0.0, 4.0]
 [-1.0, 7.0]  [-1.0, 9.0]
```
