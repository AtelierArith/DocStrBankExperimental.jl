```
NKFtool
```

[Julia](https://julialang.org) package to guess and convert encodings of Japanese characters. This is a wrapper to Network kanji filter, aka [nkf](https://osdn.net/projects/nkf/), and provides the interface to use nkf command installed in your system from Julia.

NKFtool requires nkf to be installed in your system. Major Unix-like operating systems offer its precompiled version in their package management systems.
