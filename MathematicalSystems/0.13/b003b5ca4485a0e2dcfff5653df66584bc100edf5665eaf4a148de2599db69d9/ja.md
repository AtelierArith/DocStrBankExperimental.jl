```
map(ex, args)
```

与えられた式に対応するマップ型のインスタンスを返します。

### 入力

  * `ex`   – 無名関数の形でマップを定義する式
  * `args` – 追加のオプション引数

### 出力

与えられた式に最も適したマップ。

### 例

まず、このマクロを使って線形マップを作成しましょう：

```jldoctest
julia> @map x -> [1 0; 0 0]*x
LinearMap{Int64, Matrix{Int64}}([1 0; 0 0])
```

アフィンシステムも作成できます：

```jldoctest
julia> @map x -> [1 0; 0 0]*x + [2, 0]
AffineMap{Int64, Matrix{Int64}, Vector{Int64}}([1 0; 0 0], [2, 0])
```

追加の引数は、関数呼び出し形式を使用して `@map` に渡すことができ、すなわち引数をカンマで区切り、マクロ呼び出しの周りに括弧を使用します。たとえば、次元5の恒等マップは次のように定義できます：

```jldoctest
julia> @map(x -> x, dim=5)
IdentityMap(5)
```

そのようなマップに対する状態制約は、追加の引数 `x ∈ X` を渡すことで指定できます。

恒等マップは、恒等行列のサイズを `Id(n)` として与えることでも作成できます。たとえば：

```jldoctest
julia> @map x -> Id(5)*x
IdentityMap(5)
```
