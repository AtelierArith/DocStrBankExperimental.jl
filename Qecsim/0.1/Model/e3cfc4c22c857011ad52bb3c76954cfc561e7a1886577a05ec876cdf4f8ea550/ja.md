```
probability_distribution(error_model, p::Real) -> NTuple{4,Real}
```

単一量子ビットの確率分布をパウリ I, X, Y, Z の間で返します。ここで `p` は単一量子ビットにおけるエラーの全体的な確率です。

!!! note "抽象メソッド [オプション]"
    このメソッドは**コアモジュール**によって呼び出されることはありません。デコーダにとってしばしば有用であるため、テンプレートとして提供されており、具体的なサブタイプやダックタイピングされた [`ErrorModel`](@ref) の実装が適切な場合に実装することが推奨されます。特に IID エラーモデルに対して。
