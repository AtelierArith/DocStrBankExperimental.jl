```
decode(decoder, code, syndrome::AbstractVector{Bool}; kwargs...) -> DecodeResult
```

与えられた `code` と `syndrome` に対する回復操作を解決するか、デコードの成功を評価します。これはデコード結果にカプセル化されています。

シンドロームの長さはコードスタビライザーの数と等しく、要素の値が 0 または 1（偽または真）は、対応するスタビライザーがエラーと可換であるかどうかを示します。

キーワード引数 `kwargs` はクライアントによって提供される場合があり、例えば [`App`](@ref App) のように、`error_model`、`error_probability`、`error` などのコンテキスト値を含むことができます。ほとんどの実装はそのようなパラメータを無視しますが、使用される場合は、実装はそれらを明示的に宣言し、オプションとして扱うべきです。

[`DecodeResult`](@ref) も参照してください。

!!! note "抽象メソッド"
    このメソッドは、具体的なサブタイプまたは [`Decoder`](@ref) のダックタイピングされた実装のために実装されるべきです。
