```
is_zero(a::MPolyQuoIdeal)
```

`a` がゼロイデアルであれば `true` を返し、そうでなければ `false` を返します。

# 例

```jldoctest
julia> R, (x, y) = polynomial_ring(QQ, [:x, :y]);

julia> A, _ = quo(R, [x^2-y, y^2-x+y]);

julia> a = ideal(A, [x^2+y^2, x+y])
Ideal generated by
  x^2 + y^2
  x + y

julia> is_zero(a)
false

julia> b = ideal(A, [x^2-y])
Ideal generated by
  x^2 - y

julia> is_zero(b)
true
```
