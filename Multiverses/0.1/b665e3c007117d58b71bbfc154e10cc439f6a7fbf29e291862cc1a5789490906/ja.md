```
multiverse = @enter begin
    @choose x = 1:5
    y = 2 * x + 3
    @measure z = sqrt(y)
end
```

提供されたコードブロックに基づいてマルチバース分析を準備します。

選択肢は `@choose` マクロコールによって示され、変数にコレクションを割り当てる必要があります。測定は `@measure` マクロコールによって示され、変数に値を割り当てる必要があります。

マルチバースは、`explore!` メソッドを使用して、各選択セットのユニバース内の測定値を一度にまたは反復的にポピュレートすることができます。
