```
get_genotype_transition_matrix(r, θ, α, q, table)
```

非相同性遺伝子型における隠れマルコフ連鎖の遷移行列を計算します。これはSesia et al.による「隠れマルコフモデルのノックオフによる遺伝子ハンティング」の式9です。

# 入力

`r`: 長さ `p` のベクトル、「再結合率」 `θ`: サイズ `p × K` の行列、`θ[j, k]` は SNP `p` の `k` 番目のハプロタイプモチーフに対してアレルが1である確率 `α`: サイズ `p × K` の行列、ハプロタイプモチーフが互いに成功する確率。行は1に合計されるべきです。 `q`: 初期確率の長さ `K` のベクトル `table`: マルコフ連鎖の状態 k = 1, ..., K+(K+1)/2 をハプロタイプペア (ka, kb) にマッピングする `MarkovChainTable`。
