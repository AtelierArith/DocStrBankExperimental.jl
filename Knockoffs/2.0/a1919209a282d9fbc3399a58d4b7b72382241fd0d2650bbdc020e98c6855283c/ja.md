```
simulate_ER(p::Int; [invert])
```

クラスタ化されたエルデシュ＝レーニーグラフから共分散行列をシミュレートします。これは、各ブロックがエルデシュ＝レーニーグラフであるブロック対角行列です。結果は相関行列にスケーリングされます。

詳細については、LiとMaathiusの第5.1節の4番目のシミュレーションルーチンを参照してください https://academic.oup.com/jrsssb/article/83/3/534/7056103?login=false

# 入力

  * `p`: 共分散行列の次元
  * `ϕ`: 任意の2つのノード間でエッジを形成する確率
  * `lb`: エッジの値の下限（一様分布から抽出）
  * `ub`: エッジの値の上限（一様分布から抽出）
  * `invert`: 共分散行列を反転するかどうか（精度を得るため）
  * `λmin`: 結果の共分散行列の最小固有値
  * `blocksize`: 各ERグラフ内の変数の数。
