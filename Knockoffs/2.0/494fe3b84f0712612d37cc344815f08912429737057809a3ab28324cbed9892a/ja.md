```
solve_s_graphical_group(Σ::Symmetric, groups::Vector{Int}, group_reps::Vector{Int},
method; [m], [verbose])
```

グループノックオフ問題を解決しますが、凸最適化問題は代表者のみに対して実行されます。非代表変数は、代表者に条件付けるときにグループごとに独立であると仮定されます。

# 入力

  * `Σ`: 対称 `p × p` 共分散行列
  * `groups`: グループメンバーシップの `p` 次元ベクトル
  * `group_reps`: 代表者のインデックス。
  * `method`: グループノックオフ問題を解決するための方法
  * `m`: 特徴ごとに生成するノックオフの数
  * `verbose`: 情報的な中間結果を印刷するかどうか
  * `kwargs...`: [`solve_s_group`](@ref) のための追加引数

# 出力

  * `S`: 代表者に対する最適化問題を解決することで得られた行列。
  * `D`: 代表者および非代表変数の S 行列に対応する `p × p`（密）行列。ノックオフサンプリングはこの行列を使用する必要があります。グラフィカル条件付き独立性の仮定が正確に満たされている場合、この行列はスパースであるべきですが、`cond_indep_corr` を使用して共分散行列がそれを満たすように強制しない限り、常にスパースではありません。
  * `obj`: 代表者に対する最適化問題を解決するための目的値。
