```
choose_group_reps(Σ::Symmetric, groups::AbstractVector; [threshold=0.5], [prioritize_idx], [Σinv])
```

グループ代表を選択します。代表者のインデックスを `Σ` から返します。R がグループ内の選択された変数の集合であり、O がグループ外の変数の集合である場合、R に変数を追加し続け、R によって説明される分散の割合を R と O によって説明される分散の割合で割った値が `threshold` を超えるまで続けます。

# 入力

  * `Σ`: `Symmetric` 引数でラップされた相関行列。
  * `groups`: グループメンバーシップのベクトル。

# オプションの入力

  * `threshold`: グループごとの代表者の数を制御する0と1の間の値。大きいほど代表者が多くなります（デフォルトは0.5）
  * `prioritize_idx`: 代表者として選ばれる優先権を持つべき変数のインデックス、デフォルトは `nothing`
  * `Σinv`: 事前に計算された `inv(Σ)` （提供されていない場合は計算されます）
