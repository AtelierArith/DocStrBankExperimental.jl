```
solve_max_entropy(Σ::AbstractMatrix)
```

固定XおよびモデルXノックオフに対する最大エントロピーのノックオフ問題を相関行列Σに対して解決します。ユーザーはこの関数の代わりに`solve_s`を呼び出すべきです。

# 参考文献

「再構成可能性の最小化による強力なノックオフ」からのアルゴリズム2.2: https://arxiv.org/pdf/2011.14625.pdf

# 注意

Spector、Asher、およびLucas Janson（2020）による「再構成可能性の最小化による強力なノックオフ」のアルゴリズムには、MEノックオフを計算するための誤植があります。補足セクションの式59では、`c_m = D^t_{-j,j}D^{-1}_{-j,-j}D_{-j,j}`を評価する必要がありました。彼らはFANOK論文（Askari et al.（2020）による「FANOK: KNOCKOFFS IN LINEAR TIME」）が`c_m = ||v_m||^2`を示唆していると主張しました。しかし、FANOK論文のセクションA.1.2によると、実際の更新は`D^t_{-j,j}D^{-1}_{-j,-j}D_{-j,j} = ζ*||c_m||^2 / (ζ + ||c_m||^2)`であるべきのようです。ここで、`ζ = 2Σ_{jj} - s_j`です。
