# NewtonTrustRegion

## コンストラクタ

```julia
NewtonTrustRegion(; initial_delta = 1.0,
                    delta_hat = 100.0,
                    delta_min = 0.0,
                    eta = 0.1,
                    rho_lower = 0.25,
                    rho_upper = 0.75,
                    use_fg = true)
```

コンストラクタには5つのキーワードがあります：

  * `initial_delta`、開始時の信頼領域の半径。デフォルトは`1.0`です。
  * `delta_hat`、許可される最大の信頼領域の半径。デフォルトは`100.0`です。
  * `delta_min`、許可される最小の信頼領域の半径。更新された半径がこの値より小さい場合、最適化は停止します。デフォルトは`sqrt(eps(Float64))`です。
  * `eta`、`rho`が少なくとも`eta`である場合、ステップを受け入れます。デフォルトは`0.1`です。
  * `rho_lower`、`rho`が`rho_lower`より小さい場合、信頼領域を縮小します。デフォルトは`0.25`です。
  * `rho_upper`、`rho`が`rho_upper`より大きい場合、信頼領域を拡大します。デフォルトは`0.75`です。
  * `use_fg`、真の場合、常にサブプロブレムを解いた後の値で勾配を評価します。fとgが高価な計算を共有する場合、これがより効率的です。デフォルトは`true`です。

## 説明

`NewtonTrustRegion`メソッドは、関数を最適化するための信頼領域を持つニュートン法を実装しています。このメソッドは、関数のヘッセ行列における二次の情報を活用するように設計されていますが、関数が二次でグローバルに良好に近似されていない場合でも、ニュートン法よりも安定性があります。これは、関数が局所的に二次であると仮定される動的サイズの信頼領域内で二次近似を繰り返し最小化することによって達成されます。信頼領域法の実践については、WrightとNocedalおよびWright（第4章、2006）を参照してください。
