# AdaMax

## 定数 `alpha` ケース（デフォルト）コンストラクタ:

```julia
    AdaMax(; alpha=0.002, beta_mean=0.9, beta_var=0.999, epsilon=1e-8)
```

## スケジュールされた `alpha` ケースコンストラクタ:

上記の（デフォルトの）使用法の代替として、`alpha` がすべての反復に対して固定された定数である場合、次のコンストラクタは、現在の反復カウントを現在の最適化反復の更新ステップに使用される `alpha` の値にマッピングする呼び出し可能なオブジェクト（スケジューラ）として `alpha` に柔軟性を提供します。これにより、次の使用法を使用して、反復にわたって `alpha` をスケジュールすることができます。

```julia
    # alpha_scheduler を反復 -> alpha 値マッピングの呼び出し可能オブジェクトとする
    AdaMax(; alpha=alpha_scheduler, other_kwargs...)
```

## 説明

AdaMax は、勾配ベクトルの最初の2つのモーメントの推定を構築することによって探索方向を選択する勾配ベースの最適化手法です。これにより、確率的な目的とそれに伴う勾配を持つ問題に適しています。この手法は [1] で紹介されており、関連する Adam 手法も紹介されています。この手法についての詳細は `?Adam` を参照してください。
