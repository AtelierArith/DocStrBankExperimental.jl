```
eigenvalues(res::Result, branch; class=["physical"])
```

`branch`の調和方程式のヤコビ行列の固有値を[Result](@ref)構造体の1次元スイープに対して計算します。

# 引数

  * `res::Result`: 解とヤコビ情報を含む結果オブジェクト
  * `branch`: 分析する解のブランチのインデックス
  * `class=["physical"]`: 含める解のクラスのフィルタ、デフォルトは物理的解

# 戻り値

  * 解のブランチに沿ったフィルタされた固有値のベクトル

# 注意事項

  * 現在、1次元のパラメータスイープ（D=1）のみをサポートしています
  * branchにNaN値が含まれている場合はエラーをスローします
  * 固有値は指定された解のクラスに基づいてフィルタリングされます
