```
eigenvectors(res::Result, branch; class=["physical"])
```

get_Jacobiannchを分析するために

  * `class=["physical"]`: 含める解のクラスをフィルタリングします。デフォルトは物理的解です。

# 戻り値

  * 解のブランチに沿ったフィルタリングされた固有ベクトルのベクトル

# 注意事項

  * 現在、1次元のパラメータスイープ（D=1）のみをサポートしています。
  * ブランチにNaN値が含まれている場合はエラーが発生します。
  * 固有ベクトルは指定された解のクラスに基づいてフィルタリングされます。
