```
DistributedLoads(assembly, ielem; kwargs...)
```

ビーム要素に対して分布荷重を事前に統合し、解析に使用します。

# 引数

  * `assembly`: ビーム要素のアセンブリ（[`Assembly`](@ref)型）
  * `ielem`: ビーム要素のインデックス

# キーワード引数

  * `s1 = 0.0`: ビーム要素の開始点（分布荷重の統合のみに使用）
  * `s2 = 1.0`: ビーム要素の終了点（分布荷重の統合のみに使用）
  * `method = (f, s1, s2) -> gauss_quadrature(f, s1, s2)`: 関数を統合するメソッド

`f`を`s1`から`s2`まで統合します。デフォルトは各要素に4点のガウス・レジェンドルの数値積分です。

  * `fx = (s) -> 0.0`: 分布x方向力
  * `fy = (s) -> 0.0`: 分布y方向力
  * `fz = (s) -> 0.0`: 分布z方向力
  * `mx = (s) -> 0.0`: 分布x方向モーメント
  * `my = (s) -> 0.0`: 分布y方向モーメント
  * `mz = (s) -> 0.0`: 分布z方向モーメント
  * `fx_follower = (s) -> 0.0`: 分布x方向フォロワー力
  * `fy_follower = (s) -> 0.0`: 分布y方向フォロワー力
  * `fz_follower = (s) -> 0.0`: 分布z方向フォロワー力
  * `mx_follower = (s) -> 0.0`: 分布x方向フォロワーモーメント
  * `my_follower = (s) -> 0.0`: 分布y方向フォロワーモーメント
  * `mz_follower = (s) -> 0.0`: 分布z方向フォロワーモーメント
