```
compliance_matrix(nodes, elements; cache=initialize_cache(nodes, elements),
    gxbeam_order=true, shear_center=true)
```

ノードと要素によって記述された有限要素メッシュに基づいて、コンプライアンス行列を計算します。

# 引数

  * `nodes`: メッシュ内のすべてのノードを含むベクター
  * `elements`: メッシュ内のすべての要素を含むベクター
  * `cache::SectionCache`: ノードの数、要素の数、接続性が同じである場合に、この関数への複数回の呼び出しでアロケーションを減らすために渡すことができる事前に割り当てられたキャッシュ。
  * `gxbeam_order::Bool`: コンプライアンス行列がセクション解析で使用される内部順序ではなく、GXBeamによって期待される順序で提供されるべきかどうかを示します。
  * `shear_center::Bool`: コンプライアンス行列がせん断中心に関して提供されるべきかどうかを示します。

# 戻り値

  * `S::Matrix`: コンプライアンス行列
  * `sc::Vector{float}`: せん断中心のx, y位置（横方向/せん断力がトルクを生じさせない位置、すなわちビームがねじれない位置）
  * `tc::Vector{float}`: 張力中心、別名弾性中心、別名重心のx, y位置（軸方向の力が曲げを生じさせない位置、すなわちビームがまっすぐのままである位置）
