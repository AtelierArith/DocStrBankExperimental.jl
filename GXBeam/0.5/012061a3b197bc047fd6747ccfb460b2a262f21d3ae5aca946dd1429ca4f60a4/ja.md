```
Assembly(points, start, stop; kwargs...)
```

接続された非線形ビーム要素のアセンブリを構築します。ビームの長さと中点は、ビーム要素が曲がっている場合に手動で指定できます。

# 引数

  * `points`: すべてのビーム要素の端点の配列
  * `start`: 各ビーム要素が開始する点のインデックスを含む配列
  * `stop`: 各ビーム要素が停止する点のインデックスを含む配列

# キーワード引数

  * `frames`: 各ビーム要素のための (3 x 3) 変換行列の配列。局所的な変形していないビームフレームからグローバルフレームへの変換を行います。デフォルトは単位行列です。
  * `compliance`: 各ビーム要素のための (6 x 6) ばね定数行列の配列。各ビーム要素のデフォルトは `zeros(6,6)` です。
  * `mass`: 各ビーム要素のための (6 x 6) 質量行列の配列。各ビーム要素のデフォルトは `zeros(6,6)` です。
  * `damping`: 各ビーム要素のための (6) 構造的減衰係数の配列。各ビーム要素のデフォルトは `fill(0.01, 6)` です。
  * `lengths`: 各ビーム要素の長さを含む配列。デフォルトはビームの端点間の距離です。
  * `midpoints`: 各ビーム要素の中点を含む配列。デフォルトはビーム要素の端点の平均です。
