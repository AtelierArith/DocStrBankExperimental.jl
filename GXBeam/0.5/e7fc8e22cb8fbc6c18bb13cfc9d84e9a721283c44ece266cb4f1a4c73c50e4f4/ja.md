```
StaticSystem([TF=eltype(assembly),] assembly; kwargs...)
```

[`StaticSystem`](@ref)のオブジェクトを初期化します。

# 引数:

  * `TF:`(オプション) 浮動小数点型、デフォルトは`assembly`の浮動小数点型
  * `assembly`: 剛接続された非線形ビーム要素のアセンブリ

# キーワード引数

  * `force_scaling`: システムの力/モーメントを内部的にスケーリングするために使用される係数。指定されていない場合、ビーム要素のコンプライアンス行列のエントリに基づいて適切なデフォルトが選択されます。
