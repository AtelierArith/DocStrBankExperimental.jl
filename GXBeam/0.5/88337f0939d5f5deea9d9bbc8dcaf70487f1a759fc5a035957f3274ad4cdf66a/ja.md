```
correlate_eigenmodes(C)
```

現在の反復の固有モードを前の反復の固有モードに関連付ける順列と関連する破損インデックスベクトルを返します。これは相関行列 `C` に基づいています。

相関行列は次のいずれかの形式を取ることができます（優先順位順）：

  * `C = U_p*M*V`
  * `C = U*M_p*V_p`
  * `C = V_p'*V`
  * `C = V'*V_p`

ここで、`U` は共役左固有ベクトルの行列、`M` はシステム質量行列、`V` は右固有ベクトルの行列であり、`()_p` は前の反復からの変数を示します。

相関行列の次の2つの形式は、上記の対応物に比べて著しく劣っているようです：`C = U*M*V_p` および `C = U_p*M_p*V`。これは、左固有ベクトル行列の計算方法によるものと思われます。

破損インデックスは、選択されなかった `C` の特定の行の中での最大の大きさを選択された固有モードの大きさで割ったものです。これは、左固有ベクトルを使用する相関行列の形式を使用する場合に最も意味があります。なぜなら、正しい固有モードは大きさが1に近く、誤った固有モードは大きさが0に近くなるからです。

新しいモード番号がすでに割り当てられている場合は、次に高い未割り当てのモード番号が使用されます。この場合、1より大きい破損インデックスが返されます。そうでない場合、破損インデックスの値は常に0と1の間に制約されます。

Eldred、Vankayya、Andersonによる「Aeroelastic Analysisにおける新しいモード追跡方法」を参照してください。
