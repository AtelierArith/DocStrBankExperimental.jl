```
JointRecurrenceMatrix(x, y, rthres; kwargs...)
```

軌道 `x` と `y` から共同再発行列を作成します。`rthres` と `kwargs` の可能な値については [`RecurrenceMatrix`](@ref) を参照してください。

共同再発行列は、`x` と `y` の軌道の再発を別々に考慮し、両方が同時に再発する点を探します。これは、`x` と `y` の再発行列の要素ごとの乗算によって計算されます。`x` と `y` の長さが異なる場合、再発は最短のものの長さまでのみ計算されます。

詳細、参考文献、キーワードについては [`RecurrenceMatrix`](@ref) を参照してください。
