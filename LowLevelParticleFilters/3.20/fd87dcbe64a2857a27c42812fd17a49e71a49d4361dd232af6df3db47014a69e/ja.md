```
MerweParams(; α = 1.0, β = 2.0, κ = 0.0)
MerweParams(; ακ = 1.0, β = 2.0) # ακのためのパラメータが1つだけの簡略化されたインターフェース
```

van der Merweらによって提案された無香料変換パラメータ。

  * `α`: シグマ点の広がりのためのスケーリングパラメータ (0,1]。`α`を減少させると広がりが減少します。
  * `β`: 状態の分布に関する事前知識を組み込みます。
  * `κ`: 通常0に設定される二次スケーリングパラメータ。`κ`を増加させるとシグマ点の広がりが増加します。

$$
α^2 (L + κ) < L
$$

の場合、ここで$L$はシグマ点の次元であり、中心平均重みは負になります。これは許可されていますが、場合によっては不定の共分散行列につながる可能性があります。

点の広がりは$α^2 (L + κ)$であり、ここで$L$は各点の次元です。広がりを視覚化するには

```julia
using Plots
μ = [0.0, 0.0]
Σ = [1.0 0.0; 0.0 1.0]
pars = LowLevelParticleFilters.MerweParams(α = 1e-3, β = 2.0, κ = 0.0)
xs = LowLevelParticleFilters.sigmapoints(μ, Σ, pars)
unscentedplot(xs, pars)
```

簡略化された調整ルール 

  * シグマ点の広がりを減少させたい場合は、$κ = 0$および$α < 1$を使用します。
  * シグマ点の広がりを増加させたい場合は、$κ > 0$および$α = 1$を使用します。

このルールは、単一の関数引数$ακ$のみを持つインターフェースを使用する際に使用できます。詳細については、Nielsen, K. et al., 2021, "UKF Parameter Tuning for Local Variation Smoothing"を参照してください。
