```
debugplot(pf, u, y, p=parameters(pf); runall=false, kwargs...)
```

役立つプロットを生成します。カスタマイズオプション（`kwargs...`）については、`?pplot`を参照してください。

  * `runall=false:` trueの場合、表示する前にすべての時間ステップを実行します（速い）。falseの場合、各時間ステップの後にプロットを表示します。

生成されたプロットはかなり重くなります。最初は、クラッシュしないことを確認するために、入力を100時間ステップに制限してみてください。

!!! note
    この関数を使用する前に`using Plots`を呼び出す必要があります。
