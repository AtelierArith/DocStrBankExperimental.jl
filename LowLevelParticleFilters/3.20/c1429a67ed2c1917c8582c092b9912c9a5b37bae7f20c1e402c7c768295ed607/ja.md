```
unscentedplot(ukf;          n_std = 2, N = 100, dims=1:2)
unscentedplot(sigmapoints;  n_std = 2, N = 100, dims=1:2)
```

シグマポイントとそれに対応する共分散楕円をプロットします。`dims`はプロットする2つの次元を示し、デフォルトでは最初の2つの次元になります。

UKFが渡された場合、最後の動力学更新後のシグマポイントがフィルタから抽出されます。出力のシグマポイントをプロットするには、それらを手動で渡します。これらは`ukf.measurement_model.cache.x0`および`ukf.measurement_model.cache.x1`として利用可能で、測定モデルの入力ポイントと出力ポイントを示します。

注意: シグマポイントの共分散は一般に状態の予測共分散と等しくありません。なぜなら、状態共分散は`cov(sigmapoints) + R1`として更新されるからです。`AUGD = true`（拡張動力学）の場合のみ、状態の共分散は最初の`nx`シグマポイントによって与えられます。

`covplot`も参照してください。

!!! note
    この関数を使用する前に`using Plots`を呼び出す必要があります。
