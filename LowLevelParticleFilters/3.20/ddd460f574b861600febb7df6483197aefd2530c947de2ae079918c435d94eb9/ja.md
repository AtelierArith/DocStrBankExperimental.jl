```
update!(imm::IMM, u, y, p, t; correct_kwargs = (;), predict_kwargs = (;), interact = true)
```

[`IMM`](@ref)フィルタの結合更新は、以下のステップを実行します：

1. 測定値 `y` と制御入力 `u` で各モデルを補正します。
2. モデルを単一の状態と共分散に結合します。
3. モデルを相互作用させて、それぞれの状態と共分散を更新します。
4. 各モデルを次の時間ステップに予測します。

これは他のフィルタの更新ステップとは少し異なり、更新の最後にフィルタの状態が1ステップ先の*予測*値であるのに対し、ここでは各個別フィルタが予測された状態を持っていますが、IMMフィルタの[`combine!`](@ref)ステップはまだ予測に対して実行されていません。したがって、IMMフィルタの状態は $x(t|t)$ であり、他のフィルタのように $x(t+1|t)$ ではなく、IMM内部の各フィルタも同様です。

# 引数：

  * `correct_kwargs`: [`correct!`](@ref)に送信されるオプションの名前付きタプルのキーワード引数。
  * `predict_kwargs`: [`predict!`](@ref)に送信されるオプションの名前付きタプルのキーワード引数。
  * `interact`: 相互作用ステップを実行するかどうか。
