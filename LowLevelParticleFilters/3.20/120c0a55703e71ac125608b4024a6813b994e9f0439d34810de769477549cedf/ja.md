```
ParticleFilteringSolution{F, Tu, Ty, Tx, Tw, Twe, Tll} <: AbstractFilteringSolution
```

# フィールド:

  * `f`: ソリューションを生成するために使用されるフィルター。
  * `u`: 入力
  * `y`: 出力 / 測定値
  * `x`: パーティクル。この配列のサイズは `(N,T)` で、`N` はパーティクルの数、`T` は時間ステップの数です。各要素は、重み `we` によって与えられる重み付き状態仮説を表します。
  * `w`: 重み（対数空間）。これらは内部計算に使用されます。
  * `we`: 重み（指数化された / 元の空間）。これらは重み付き平均などを計算するために使用され、各時間ステップで合計は1になります。
  * `ll`: 対数尤度

# プロット

ソリューションオブジェクトはプロットできます。

```
plot(sol; nbinsy=30, xreal=nothing, dim=nothing, ploty=true, q=nothing)
```

デフォルトでは、各状態変数に対して重み付きの2Dヒストグラムがプロットされます。`q` に分位数のベクトルが提供されると、ヒストグラムの代わりに分位数がプロットされます。`xreal` が提供されると、真の状態がヒストグラムの上に散布図としてプロットされます。`dim` が提供されると、指定された次元のみがプロットされます。`ploty` が真の場合、測定値もプロットされます。
