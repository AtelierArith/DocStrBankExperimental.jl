# 引数

  * `tr::Abstract2DTrajectory` - 二次元軌道
  * `image::Matrix`            - 変換される画像
  * `correctionMap=[]`         - 補正マップはオフレゾナンスとリラクゼーションのフィールドの合計
  * `alpha::Float64 = 1.75`    - オーバーサンプリング係数
  * `m::Float64=4.0`           - カーネルサイズ
  * `K::Int64=28`              - 補正項を近似するために使用される係数のランク
  * `method="nfft"`            - 補正項の近似係数を取得するために使用されるメソッド
  * (`sensmaps=[]`)            - コイル感度の配列
  * `kargs...`                 - 追加のキーワード引数
