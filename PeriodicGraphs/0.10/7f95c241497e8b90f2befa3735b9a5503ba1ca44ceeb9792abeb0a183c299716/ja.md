```
hash_position(x::PeriodicVertex{N}, n::Integer) where N
```

与えられた `x` は `PeriodicVertex{N}` であり、グラフ内の頂点識別子の数 `n` に対して、与えられた頂点のための一意の正の整数ハッシュを計算します。

このハッシュ関数は、周期的グラフのすべての頂点の集合と正の整数の集合との間の全単射です。その値は `1+n*(2d-1)^N`（または `d == 0` の場合は `1`）と `n*(2d+1)^N` の間の整数であり、ここで `d = maximum(abs.(x.ofs))` です。

特に、これは、単位セル B が原点からのマンハッタン距離において別の A よりも遠い場合、B のすべての頂点が A のすべての頂点よりも大きなハッシュを持つことを意味します。
