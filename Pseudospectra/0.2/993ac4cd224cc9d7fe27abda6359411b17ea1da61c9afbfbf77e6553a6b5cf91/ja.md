```
psa_abscissa(A,ϵ [,d]) -> α,z
```

密行列のϵ擬似スペクトルのアブシッサを計算します。

密行列 `A` のϵ擬似スペクトルのアブシッサ `α` を計算するための二方向の二次収束法です。また、擬似スペクトルがアブシッサに達する点のベクトル `z` も返します。Burkeらのクロスクロスアルゴリズムを使用しています。

ϵ擬似スペクトルのアブシッサは次のように定義されます。

```
   maximum(real(z)) for z s.t. minimum(σ(A-zI)) == ϵ
```

オプション引数:

  * `d`: 事前に知られている場合のAの固有値

キーワード引数:

  * `verbosity: 0 for quiet, 1 for noise`
