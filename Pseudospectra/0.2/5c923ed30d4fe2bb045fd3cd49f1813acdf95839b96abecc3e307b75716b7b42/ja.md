```
psa_radius(A,ϵ [,d]) -> r,z
```

密行列のϵ擬似スペクトル半径を計算します。

密行列`A`のϵ擬似スペクトル半径`r`を計算するための二方向の二次収束法です。また、擬似スペクトルが半径`r`の円と交差する点のベクトル`z`も返します。OvertonとMengiの「クロスクロス」アルゴリズムを使用しています。

ϵ擬似スペクトル半径は次のように定義されます。

```
   maximum(abs(z)) for z s.t. minimum(σ(A-zI)) == ϵ
```

オプション引数：

  * `d`: 事前に知られている場合、Aの固有値

キーワード引数：

  * `verbosity: 0は静か、1はノイズ
