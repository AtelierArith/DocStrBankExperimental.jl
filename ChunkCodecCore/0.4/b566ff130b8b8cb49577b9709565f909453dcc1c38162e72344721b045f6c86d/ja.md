```
struct ShuffleCodec <: Codec
ShuffleCodec(element_size::Integer)
```

バイトシャッフル。要素サイズはシャッフルをデコードするために必要です。

例えば：

```julia
[
0x11, 0x12, 0x13,
0x21, 0x22, 0x23,
0x31, 0x32, 0x33,
0x41, 0x42, 0x43,
]
```

要素サイズ3でエンコードされると、

```julia
[
0x11, 0x21, 0x31, 0x41,
0x12, 0x22, 0x32, 0x42,
0x13, 0x23, 0x33, 0x43,
]
```

データの長さが要素サイズで割り切れない場合、余りのデータが追加されます。

例えば、要素サイズ3の「12312312312345」は「11112222333345」としてエンコードされます。

`ShuffleCodec`はエンコーダーまたはデコーダーとして使用できます。
