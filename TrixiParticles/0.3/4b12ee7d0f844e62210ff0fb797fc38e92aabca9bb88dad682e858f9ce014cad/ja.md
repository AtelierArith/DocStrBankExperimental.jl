```
初期条件(; 座標, 密度, 速度=zeros(size(座標, 1)),
                 質量=nothing, 圧力=0.0, 粒子間隔=-1.0)
```

粒子の初期構成を保持する構造体です。

以下のセットアップは、一般的に使用されるセットアップのための`初期条件`を返します：

  * [`RectangularShape`](@ref)
  * [`SphereShape`](@ref)
  * [`RectangularTank`](@ref)
  * [`ComplexShape`](@ref)
  * [`extrude_geometry`](@ref)

`初期条件`は、より複雑な幾何学を構築するために、集合演算`union`、`setdiff`、`intersect`をサポートしています。`初期条件`はまた、[`load_geometry`](@ref)によって返される`TrixiParticles.TriangleMesh`および`TrixiParticles.Polygon`と共に、集合演算`setdiff`および`intersect`をサポートしています。

# 引数

  * `座標`: $i$-番目の列に粒子$i$の座標を保持する配列。
  * `密度`:     各粒子の密度を保持するベクトル、                または各粒子の座標をその密度にマッピングする関数、                またはすべての粒子に対して一定の密度のスカラー。

# キーワード

  * `速度`:   $i$-番目の列に粒子$i$の速度を保持する配列、               または各粒子の座標をその速度にマッピングする関数、               または一定の流体速度の場合、この速度を保持するベクトル。               速度はデフォルトでゼロです。
  * `質量`:       粒子の密度と間隔から自動的に粒子質量を計算するための`nothing`（デフォルト）、               または各粒子の質量を保持するベクトル、               または各粒子の座標をその質量にマッピングする関数、               またはすべての粒子に対して一定の質量のスカラー。
  * `圧力`:   各粒子の圧力を保持するベクトル、               または各粒子の座標をその圧力にマッピングする関数、               またはすべての粒子に対して一定の圧力のスカラー。これはオプションであり、               [`EntropicallyDampedSPHSystem`](@ref)を使用する場合にのみ必要です。
  * `粒子間隔`: 粒子間の間隔。このスカラーは、間隔が均一であると仮定されるため、                     必要です。これは、`初期条件`に対して集合演算を使用する場合や、                     自動質量計算のためにのみ必要です。

# 例

```jldoctest; output = false
# 粒子で満たされた長方形
初期条件 = RectangularShape(0.1, (3, 4), (-1.0, 1.0), 密度=1.0)

# 一つの初期条件に二つの球
初期条件 = union(SphereShape(0.15, 0.5, (-1.0, 1.0), 1.0),
                          SphereShape(0.15, 0.2, (0.0, 1.0), 1.0))

# 球状の穴のある長方形
形状1 = RectangularShape(0.1, (16, 13), (-0.8, 0.0), 密度=1.0)
形状2 = SphereShape(0.1, 0.35, (0.0, 0.6), 1.0, sphere_type=RoundSphere())
初期条件 = setdiff(形状1, 形状2)

# 長方形と球の交差。これは、交差にある長方形の粒子を保持しますが、`intersect(形状2, 形状1)`は
# 交差にある球の粒子で構成されます。
形状1 = RectangularShape(0.1, (16, 13), (-0.8, 0.0), 密度=1.0)
形状2 = SphereShape(0.1, 0.35, (0.0, 0.6), 1.0, sphere_type=RoundSphere())
初期条件 = intersect(形状1, 形状2)

# ファイルから読み込まれた幾何学を使用した集合演算
形状 = RectangularShape(0.05, (20, 20), (0.0, 0.0), 密度=1.0)
ファイル =  pkgdir(TrixiParticles, "examples", "preprocessing", "data", "circle.asc")
幾何学 = load_geometry(ファイル)

初期条件_1 = intersect(形状, 幾何学)
初期条件_2 = setdiff(形状, 幾何学)

# `初期条件`を手動で構築
座標 = [0.0 1.0 1.0
               0.0 0.0 1.0]
速度 = zero(座標)
質量 = ones(3)
密度 = 1000 * ones(3)
初期条件 = 初期条件(; 座標, 速度, 質量, 密度)

# 関数を使用して
初期条件 = 初期条件(; 座標, 速度=x -> 2x, 質量=1.0, 密度=1000.0)

# 出力

│ 初期条件{Float64}                                                                        │
│ ═════════════════════════                                                                        │
│ #次元: ……………………………………………… 2                                                                │
│ #粒子: ………………………………………………… 3                                                                │
│ 粒子間隔: ………………………………… -1.0                                                             │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘
```
