```
ArtificialViscosityMonaghan(; alpha, beta=0.0, epsilon=0.01)
```

モナハンによる人工粘性 ([Monaghan1992](@cite), [Monaghan1989](@cite))。

実装された粘性モデルの概要と比較については、[`Viscosity`](@ref viscosity_sph)を参照してください。

# キーワード

  * `alpha`: ほとんどのシミュレーションには通常 `0.02` の値が使用されます。運動粘性との関係については、上記の説明を参照してください。
  * `beta=0.0`: `0.0` の値は、ほとんどの流体シミュレーションや中程度の強さの衝撃を伴うシミュレーションでうまく機能します。マッハ数が非常に高くなる可能性のあるシミュレーション、例えば天体物理学的計算では、`beta=2.0` と `alpha=1.0` を選択することで良好な結果が得られます。
  * `epsilon=0.01`: 特異点を防ぐためのパラメータ。
