スパイキーカーネルは、特に流体シミュレーションにおける境界近くの特徴を保持する望ましい特性のために、SPHで頻繁に使用されるカーネルの一つです [Mueller2003](@cite)。これは次のように定義されます：

$$
 W(r, h) = \frac{1}{h^d} w(r/h)
$$

ここで：

$$
w(q) = \sigma \begin{cases}
    (1 - q)^3    & \text{if } 0 \leq q < 1, \\
    0            & \text{if } q \geq 1,
\end{cases}
$$

ここで、$d$は次元の数で、正規化係数$\sigma$は2次元で$10 / \pi$、3次元で$15 / \pi$です。

このカーネル関数は、$[0, h]$のコンパクトサポートを持っています。

スパイキーカーネルは、特に固体境界近くの流体シミュレーションにおいて鋭い特徴を保持するのに役立つ鋭い勾配で知られています。境界でのこれらの鋭い勾配は、安定性の低下を引き起こす可能性があるため、最大の欠点でもあります。

スムージング長さは通常、$[1.5\delta, 3.0\delta]$の範囲にあり、ここで$\delta$は典型的な粒子間隔です。

一般的な情報と使用法については、[Smoothing Kernels](@ref smoothing_kernel)を参照してください。
