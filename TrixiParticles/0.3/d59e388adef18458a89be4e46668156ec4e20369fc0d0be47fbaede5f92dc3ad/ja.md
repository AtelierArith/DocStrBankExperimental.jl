```
BernoulliPressureExtrapolation(; pressure_offset=0, factor=1)
```

`density_calculator` for `BoundaryModelDummyParticles`.

# キーワード

  * `pressure_offset=0`:   境界圧力を人工的に増加させる必要がある場合があり、この値を増加させることでそれが可能です。
  * `factor=1`         :   `factor`を設定することで、動的圧力部分の強度を増加させることができます。
  * `allow_loop_flipping=true`: 圧力外挿のためにループ順序を反転させることを許可します。異なるスレッド数のシミュレーション間でのエラーの変動を防ぐために無効にします。通常、最初の（マルチスレッド）ループは境界粒子に対して、2番目のループは流体の隣接粒子に対して行われます。境界粒子の数が `ceil(0.5 * nthreads())` 倍の流体粒子の数よりも大きい場合、ループ順序を反転させて流体粒子を最初にループする方が通常は効率的です。この因子はスレッド数に依存します。反転したループはスレッド並列化できないため、異なるスレッド数のシミュレーション間でエラーの変動を引き起こす可能性があります。
