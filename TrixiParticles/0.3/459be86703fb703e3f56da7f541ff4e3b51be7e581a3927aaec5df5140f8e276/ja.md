```
RoundSphere(; start_angle=0.0, end_angle=2π)
```

完全に丸い同心球を重ねることで球体（または球体セグメント）を構築します。結果として得られるボールは完全に丸いですが、規則的な内部構造は持ちません。

# キーワード

  * `start_angle`: 球体セグメントの開始角度（ラジアン）。セグメントの開始点を決定します。デフォルトは `0.0` で、正の x 軸を表します。
  * `end_angle`: 球体セグメントの終了角度（ラジアン）。セグメントの終了点を定義します。デフォルトは `2pi` で、完全な球を完成させます。

!!! note "使用法"
    [`SphereShape`](@ref) を参照して、使用方法を確認してください。


!!! warning
    球体セグメントは2D幾何学および中空の球体を対象としています。充填された球体や3Dコンテキストで使用した場合、結果が正確でない可能性があります。
