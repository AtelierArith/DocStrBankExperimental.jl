```
SphereShape(particle_spacing, radius, center_position, density;
            sphere_type=VoxelSphere(), n_layers=-1, layer_outwards=false,
            cutout_min=(0.0, 0.0), cutout_max=(0.0, 0.0), tlsph=false,
            velocity=zeros(length(center_position)), mass=nothing, pressure=0.0)
```

完全に満たされた球体（デフォルト）または中空の球体（`n_layers`を渡すことで）を生成します。

球体のタイプ [`VoxelSphere`](@ref) では、ボクセルの球体（粒子がボクセルの中心に配置される）を作成し、内部構造は規則的ですが、表面には角があります。基本的に、粒子のグリッドが生成され、球体の外にあるすべての粒子が削除されます。球体のタイプ [`RoundSphere`](@ref) では、内部構造が不完全な完全に丸い球体が作成されます。

球体からは、`cutout_min` と `cutout_max` として負の座標方向と正の座標方向の2つのコーナーを指定することで、直方体を切り取ることができます。

# 引数

  * `particle_spacing`:   粒子間の間隔。
  * `radius`:             球体の半径。
  * `center_position`:    球体の中心の座標。
  * `density`:            各粒子の座標をその密度にマッピングする関数、またはすべての粒子に対して一定の密度のスカラー。

# キーワード

  * `sphere_type`:    [`VoxelSphere`](@ref) または [`RoundSphere`](@ref) のいずれか（上記の説明を参照）。
  * `n_layers`:       中空の球体を生成するために0より大きい整数に設定し、シェルは `n_layers` 層で構成されます。
  * `layer_outwards`: `false`（デフォルト）の場合、`radius` は球体の外半径です。`true` に設定すると、`radius` は球体の内半径になります。これは `n_layers > 0` の場合にのみ使用されます。
  * `cutout_min`:     球体から切り取る直方体の負の座標方向のコーナー。
  * `cutout_max`:     球体から切り取る直方体の正の座標方向のコーナー。
  * `tlsph`:          [`TotalLagrangianSPHSystem`](@ref) では、粒子は流体のように1粒子の半径離れた位置ではなく、形状の境界に配置する必要があります。`tlsph=true` の場合、粒子は形状の境界に配置されます。
  * `velocity`:   各粒子の座標をその速度にマッピングする関数、または一定の流体速度の場合はこの速度を保持するベクトル。デフォルトでは速度はゼロです。
  * `mass`:       `nothing`（デフォルト）で粒子の密度と間隔から自動的に粒子の質量を計算するか、各粒子の座標をその質量にマッピングする関数、またはすべての粒子に対して一定の質量のスカラー。
  * `pressure`:   各粒子の座標をその圧力にマッピングする関数、またはすべての粒子に対して一定の圧力のスカラー。これはオプションで、[`EntropicallyDampedSPHSystem`](@ref) を使用する場合にのみ必要です。

# 例

```jldoctest; output = false
# 半径0.5、中心が(0.2, 0.4)、粒子間隔が0.1の充填された円
SphereShape(0.1, 0.5, (0.2, 0.4), 1000.0)

# 前と同じですが、完全に丸い
SphereShape(0.1, 0.5, (0.2, 0.4), 1000.0, sphere_type=RoundSphere())

# 約3層の中空の円、外半径0.5、中心が(0.2, 0.4)、粒子間隔が0.1。
SphereShape(0.1, 0.5, (0.2, 0.4), 1000.0, n_layers=3)

# 前と同じですが、完全に丸い
SphereShape(0.1, 0.5, (0.2, 0.4), 1000.0, n_layers=3, sphere_type=RoundSphere())

# 3層の中空の円、内半径0.5、中心が(0.2, 0.4)、粒子間隔が0.1。
SphereShape(0.1, 0.5, (0.2, 0.4), 1000.0, n_layers=3, layer_outwards=true)

# 半径0.1、中心が(0.0, 0.0)、粒子間隔0.1の充填された円ですが、
# 長方形 [0, 1] x [-0.2, 0.2] が切り取られます。
SphereShape(0.1, 1.0, (0.0, 0.0), 1000.0, cutout_min=(0.0, -0.2), cutout_max=(1.0, 0.2))

# 半径0.5、中心が(0.2, 0.4, 0.3)、粒子間隔が0.1の充填された3D球体
SphereShape(0.1, 0.5, (0.2, 0.4, 0.3), 1000.0)

# 前と同じですが、完全に丸い
SphereShape(0.1, 0.5, (0.2, 0.4, 0.3), 1000.0, sphere_type=RoundSphere())
```
