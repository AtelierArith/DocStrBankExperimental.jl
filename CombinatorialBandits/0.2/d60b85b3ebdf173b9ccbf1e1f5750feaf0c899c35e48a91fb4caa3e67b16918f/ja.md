```
Trace{T}
```

環境におけるバンディットの進化に関するすべての詳細を記憶します。このオブジェクトは、長いエピソードの場合、非常に重くなる可能性があります。

情報は複数のベクトルに格納されており、すべてのベクトルはバンディットが使用されたラウンドの数と同じ長さです：

  * `states`: エピソード中にバンディットが入った状態のリスト
  * `arms`: エピソード中にプレイされたアームのセットのリスト
  * `reward`: エピソード中にポリシーが受け取った報酬のリスト
  * `policy_details`: エピソード中のポリシーによって定義された詳細のリスト（通常、計算時間）
  * `time_choose_action`: エピソード中のラウンド決定にかかる時間（ミリ秒単位）のリスト
