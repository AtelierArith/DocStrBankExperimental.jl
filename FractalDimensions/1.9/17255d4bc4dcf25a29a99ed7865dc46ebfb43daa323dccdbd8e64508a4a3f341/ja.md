```
Exceedances(p::Real, estimator::Symbol)
```

[`extremevaltheory_dims_persistences`](@ref)および関連する関数のための指示タイプ。このメソッドはしきい値を設定し、超過値を一般化パレート分布（GPD）にフィットさせます。パラメータ `p` は0と1の間の数で、しきい値と極値インデックスの計算のためのp-分位数を決定します。引数 `estimator` は、GPDがデータにどのようにフィットされるかを決定するシンボルです。これは、[`estimate_gpd_parameters`](@ref)のように `:exp, :pwm, :mm` の値を取ることができます。

## 説明

`X` の各状態空間点 $\mathbf{x}_i$ に対して、$g_i = -\log(||\mathbf{x}_i - \mathbf{x}_j|| ) \; \forall j = 1, \ldots, N$ を計算します。ここで、$||\cdot||$ はユークリッド距離です。次に、$g_i$ の分布に対して極端な分位数確率 $p$（例：0.99）を選択します。$g_p$ を $g_i$ の $p$-分位数として計算します。次に、$E_i = \{ g_i - g_p: g_i \ge g_p \}$ と定義される $g_i$ の超過値を収集します。すなわち、$g_p$ 以上のすべての $g_i$ の値を、$g_p$ でシフトしたものです。$E_i$ には合計で $n = N(1-q)$ の値があります。極値理論によれば、$N \to \infty$ の限界において、値 $E_i$ はパラメータ $\sigma,\xi$ を持つ二項パラメータ一般化パレート分布（GPD）に従います（GPDの第三のパラメータ $\mu$ は、$E$ の正定値構造によりゼロです）。この極値理論アプローチの中で、状態空間点 $\textbf{x}_i$ に割り当てられる局所次元 $\Delta^{(E)}_i$ は、データにフィットしたGPDの $\sigma$ パラメータの逆数として与えられます[^Lucarini2012]。$\Delta^{(E)}_i = 1/\sigma$ です。$\sigma$ は `estimator` キーワードに従って推定されます。

このプロセスのより正確な説明は、レビュー論文 [Datseris2023](@cite) に記載されています。
