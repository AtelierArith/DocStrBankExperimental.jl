```
identify_cocos(B0, Ip, q, psi, clockwise_phi, a)
```

COCOS座標系を特定するためのユーティリティ関数。複数のCOCOSが可能な場合は、すべてが返されます。

引数:

  * B0 - トロイダル磁場（符号付き）
  * Ip - プラズマ電流（符号付き）
  * q  -  安全係数プロファイル（符号付き）としてのpsiの関数
  * psi::AbstractVector - 磁気軸からプラズマ境界までのポロイダルフラックスのベクトル
  * clockwise_phi::Bool - （オプション）[true, false] phi角が時計回りに定義されているかどうか。これは奇数と偶数のCOCOSを特定するために必要です。この情報はコードの出力からは決定できません。
  * a::AbstractVector - （オプション）psiの関数としてのフラックスサーフェスのマイナー半径。これはpsi定義における2*pi項を特定するために必要です。
