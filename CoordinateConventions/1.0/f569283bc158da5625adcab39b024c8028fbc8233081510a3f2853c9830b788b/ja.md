COCOS 構造

```
cocos::Int           = COCOS ID 番号
exp_Bp::Int          = 0 または 1、psi が 2pi で割られているかどうかに応じて、それぞれ
sigma_Bp::Int        = +1 または -1、psi が Ip と B0 が正のときに増加しているか減少しているかに応じて
sigma_RpZ::Int       = +1 または -1、(R,phi,Z) が右手系か (R,Z,phi) かに応じて、それぞれ
sigma_rhotp::Int     = +1 または -1、(rho, theta, phi) が右手系か (rho,phi,theta) かに応じて、それぞれ
sign_q_pos::Int      = +1 または -1、q が Ip と B0 が正のときに正か負かに応じて
sign_pprime_pos::Int = +1 または -1、dp/dpsi が Ip と B0 が正のときに正か負かに応じて
```
