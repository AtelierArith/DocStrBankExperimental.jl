```
atan(y)
atan(y, x)
```

`y` 또는 `y/x`의 역탄젠트를 계산합니다.

하나의 실수 인수에 대해, 이는 양의 *x*-축과 점 (1, *y*) 사이의 라디안 각도로, $[-\pi/2, \pi/2]$ 구간의 값을 반환합니다.

두 개의 인수에 대해, 이는 양의 *x*-축과 점 (*x*, *y*) 사이의 라디안 각도로, $[-\pi, \pi]$ 구간의 값을 반환합니다. 이는 표준 [`atan2`](https://en.wikipedia.org/wiki/Atan2) 함수에 해당합니다. 관례에 따라 `atan(0.0,x)`는 $\pi$로 정의되고, `atan(-0.0,x)`는 `x < 0`일 때 $-\pi$로 정의됩니다.

도에 대한 [`atand`](@ref)도 참조하십시오.

# 예제

```jldoctest
julia> rad2deg(atan(-1/√3))
-30.000000000000004

julia> rad2deg(atan(-1, √3))
-30.000000000000004

julia> rad2deg(atan(1, -√3))
150.0
```
