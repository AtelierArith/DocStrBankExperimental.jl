```
dot(x, A, y)
```

두 벡터 `x`와 `y` 사이의 일반화된 내적 `dot(x, A*y)`를 계산하되, `A*y`의 중간 결과를 저장하지 않습니다. 두 인수 [`dot(_,_)`](@ref)와 마찬가지로, 이 함수는 재귀적으로 작동합니다. 또한, 복소 벡터의 경우 첫 번째 벡터는 켤레화됩니다.

!!! compat "Julia 1.4"
    세 인수 `dot`는 최소한 Julia 1.4가 필요합니다.


# 예제

```jldoctest
julia> dot([1; 1], [1 2; 3 4], [2; 3])
26

julia> dot(1:5, reshape(1:25, 5, 5), 2:6)
4850

julia> ⋅(1:5, reshape(1:25, 5, 5), 2:6) == dot(1:5, reshape(1:25, 5, 5), 2:6)
true
```
