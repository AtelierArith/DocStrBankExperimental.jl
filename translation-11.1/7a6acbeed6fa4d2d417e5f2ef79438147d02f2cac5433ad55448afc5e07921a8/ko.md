```
<<(x, n)
```

왼쪽 비트 시프트 연산자, `x << n`. `n >= 0`인 경우, 결과는 `x`를 `n` 비트만큼 왼쪽으로 시프트하며, `0`으로 채워집니다. 이는 `x * 2^n`과 같습니다. `n < 0`인 경우, 이는 `x >> -n`과 같습니다.

# 예제

```jldoctest
julia> Int8(3) << 2
12

julia> bitstring(Int8(3))
"00000011"

julia> bitstring(Int8(12))
"00001100"
```

자세한 내용은 [`>>`](@ref), [`>>>`](@ref), [`exp2`](@ref), [`ldexp`](@ref)를 참조하세요.
