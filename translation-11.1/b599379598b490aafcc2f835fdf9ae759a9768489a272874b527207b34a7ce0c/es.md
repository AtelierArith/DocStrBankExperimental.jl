```
sincos(A::AbstractMatrix)
```

Calcula el seno y el coseno de matriz de una matriz cuadrada `A`.

# Ejemplos

```jldoctest
julia> S, C = sincos(fill(1.0, (2,2)));

julia> S
2×2 Matrix{Float64}:
 0.454649  0.454649
 0.454649  0.454649

julia> C
2×2 Matrix{Float64}:
  0.291927  -0.708073
 -0.708073   0.291927
```
