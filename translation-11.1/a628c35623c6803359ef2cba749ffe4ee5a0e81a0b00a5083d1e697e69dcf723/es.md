```
Cstring
```

Una cadena de estilo C compuesta por el tipo de carácter nativo [`Cchar`](@ref)s. Los `Cstring` están terminados en NUL. Para cadenas de estilo C compuestas por el tipo de carácter ancho nativo, consulte [`Cwstring`](@ref). Para más información sobre la interoperabilidad de cadenas con C, consulte el [manual](@ref man-bits-types).
