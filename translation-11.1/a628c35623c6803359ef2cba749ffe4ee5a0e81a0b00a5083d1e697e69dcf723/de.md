```
Cstring
```

Ein C-ähnlicher String, der aus dem nativen Zeichentyp [`Cchar`](@ref)s besteht. `Cstring`s sind NUL-terminiert. Für C-ähnliche Strings, die aus dem nativen breiten Zeichentyp bestehen, siehe [`Cwstring`](@ref). Für weitere Informationen zur String-Interoperabilität mit C siehe das [Handbuch](@ref man-bits-types).
