```
tanh(x)
```

`x`의 쌍곡선 탄젠트를 계산합니다.

자세한 내용은 [`tan`](@ref), [`atanh`](@ref)를 참조하세요.

# 예제

```jldoctest
julia> tanh.(-3:3f0)  # 여기서 3f0은 Float32입니다.
7-element Vector{Float32}:
 -0.9950548
 -0.9640276
 -0.7615942
  0.0
  0.7615942
  0.9640276
  0.9950548

julia> tan.(im .* (1:3))
3-element Vector{ComplexF64}:
 0.0 + 0.7615941559557649im
 0.0 + 0.9640275800758169im
 0.0 + 0.9950547536867306im
```
