```
mul!(Y, A, B) -> Y
```

행렬-행렬 또는 행렬-벡터 곱 $A B$를 계산하고 결과를 `Y`에 저장하여 기존의 `Y` 값을 덮어씁니다. `Y`는 `A` 또는 `B`와 별칭이 되어서는 안 됩니다.

# 예제

```jldoctest
julia> A = [1.0 2.0; 3.0 4.0]; B = [1.0 1.0; 1.0 1.0]; Y = similar(B);

julia> mul!(Y, A, B) === Y
true

julia> Y
2×2 Matrix{Float64}:
 3.0  3.0
 7.0  7.0

julia> Y == A * B
true
```

# 구현

사용자 정의 행렬 및 벡터 유형의 경우, 가능하다면 3-인수 `mul!`를 직접 구현하기보다는 5-인수 `mul!`를 구현하는 것이 좋습니다.
