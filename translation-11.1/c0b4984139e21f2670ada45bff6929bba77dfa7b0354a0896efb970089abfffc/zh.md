```
muladd(A, y, z)
```

组合乘加，`A*y .+ z`，用于矩阵-矩阵或矩阵-向量乘法。结果的大小始终与 `A*y` 相同，但 `z` 可能更小，或是一个标量。

!!! compat "Julia 1.6"
    这些方法需要 Julia 1.6 或更高版本。


# 示例

```jldoctest
julia> A=[1.0 2.0; 3.0 4.0]; B=[1.0 1.0; 1.0 1.0]; z=[0, 100];

julia> muladd(A, B, z)
2×2 Matrix{Float64}:
   3.0    3.0
 107.0  107.0
```
