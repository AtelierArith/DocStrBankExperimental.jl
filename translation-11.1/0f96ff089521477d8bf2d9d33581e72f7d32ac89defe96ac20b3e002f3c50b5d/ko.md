| 오프셋 | 설명                              |
|:--- |:------------------------------- |
| 0   | 태그 바이트 (0x37)                   |
| 1-2 | 서명 바이트 "JL"                     |
| 3   | 프로토콜 버전                         |
| 4   | 비트 0-1: 엔디안: 0 = 리틀, 1 = 빅      |
| 4   | 비트 2-3: 플랫폼: 0 = 32비트, 1 = 64비트 |
| 5-7 | 예약됨                             |
