```
Serialization.writeheader(s::AbstractSerializer)
```

向指定的序列化器写入一个标识头。头由以下8个字节组成：

| 偏移量 | 描述                          |
|:--- |:--------------------------- |
| 0   | 标签字节 (0x37)                 |
| 1-2 | 签名字节 "JL"                   |
| 3   | 协议版本                        |
| 4   | 位 0-1: 字节序: 0 = 小端, 1 = 大端  |
| 4   | 位 2-3: 平台: 0 = 32位, 1 = 64位 |
| 5-7 | 保留                          |
