```
/(x, y)
```

Rechter Divisionsoperator: Multiplikation von `x` mit dem Inversen von `y` auf der rechten Seite.

Gibt Fließkommaergebnisse für ganzzahlige Argumente zurück. Siehe [`÷`](@ref div) für ganzzahlige Division oder [`//`](@ref) für [`Rational`](@ref) Ergebnisse.

# Beispiele

```jldoctest
julia> 1/2
0.5

julia> 4/2
2.0

julia> 4.5/2
2.25
```
