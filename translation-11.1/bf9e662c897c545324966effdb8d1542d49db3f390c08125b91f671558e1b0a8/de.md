```
@which
```

Angewendet auf einen Funktions- oder Makroaufruf bewertet es die Argumente des angegebenen Aufrufs und gibt das `Method`-Objekt für die Methode zurück, die für diese Argumente aufgerufen werden würde. Wird es auf eine Variable angewendet, gibt es das Modul zurück, in dem die Variable gebunden wurde. Es ruft die [`which`](@ref)-Funktion auf.
