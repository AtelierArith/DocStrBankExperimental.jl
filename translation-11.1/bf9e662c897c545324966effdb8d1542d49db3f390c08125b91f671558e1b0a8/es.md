```
@which
```

Aplicado a una llamada de función o macro, evalúa los argumentos de la llamada especificada y devuelve el objeto `Method` para el método que se llamaría para esos argumentos. Aplicado a una variable, devuelve el módulo en el que se vinculó la variable. Llama a la función [`which`](@ref).

Véase también: [`@less`](@ref), [`@edit`](@ref).
