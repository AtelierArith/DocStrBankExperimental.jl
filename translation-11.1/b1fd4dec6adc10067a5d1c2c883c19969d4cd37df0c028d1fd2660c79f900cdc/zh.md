```
+(x, y...)
```

加法运算符。

中缀 `x+y+z+...` 调用此函数并传入所有参数，即 `+(x, y, z, ...)`，默认情况下从左侧开始调用 `(x+y) + z + ...`。

请注意，对于大多数整数类型，包括默认的 `Int`，在添加大数字时可能会发生溢出。

# 示例

```jldoctest
julia> 1 + 20 + 4
25

julia> +(1, 20, 4)
25

julia> [1,2] + [3,4]
2-element Vector{Int64}:
 4
 6

julia> typemax(Int) + 1 < 0
true
```
