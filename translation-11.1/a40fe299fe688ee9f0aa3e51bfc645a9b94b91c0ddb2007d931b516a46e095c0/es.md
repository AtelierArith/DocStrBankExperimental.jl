```
acosh(A::AbstractMatrix)
```

Calcula el coseno hiperbólico inverso de una matriz cuadrada `A`. Para la teoría y las fórmulas logarítmicas utilizadas para calcular esta función, consulta [^AH16_4].

[^AH16_4]: Mary Aprahamian y Nicholas J. Higham, "Funciones Trigonométricas Inversas de Matrices y Funciones Hiperbólicas Inversas: Teoría y Algoritmos", MIMS EPrint: 2016.4. [https://doi.org/10.1137/16M1057577](https://doi.org/10.1137/16M1057577)
