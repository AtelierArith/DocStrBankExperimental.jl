```
abstract type Group end
```

巡回群インターフェース型です。群のコンストラクタはインスタンスの検証を行い、構築できない場合は`ArgumentError`をスローすることが期待されています。単位元は明示的な引数`allow_one=true`で構築でき、検証はコンストラクタの引数`skip_validation=true`を使用することで省略でき、これはローカルの安全な場所からのデシリアライズに役立ちます。

群の要素はバイトのベクター（オクテット）からの構築をサポートし、`octet`メソッド自体もサポートします。さらに、`value`メソッドはデバッグ目的に役立つ最も単純な表現を提供することが期待されています。与えられた`g::Group`に対して、次のコンストラクタメソッドがサポートされています：

```julia
G = typeof(g)
g == G(octet(g)) == G(value(g))
one(G) == G(octet(one(G)), allow_one=true)
```

群は`order`、`*`、`^`、`rem`および単位元を`one`を介してサポートしなければなりません。具体的な実装については`PGroup`および`ECGroup`を参照してください。
