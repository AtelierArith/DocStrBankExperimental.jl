```
signal(x,[samplerate])
```

`x`を信号に強制変換し、オプションでサンプルレート（通常はHz）を指定します。すべての信号演算子は、各引数に対して最初に`signal(x)`を呼び出します。これは、追加の引数を渡したいときにのみ`signal`を呼び出す必要があることを意味します。

!!! note
    `signal`をパイプする場合（例：`myobject |> signal(2kHz)`）、渡す場合はサンプルレートの単位を指定する必要があります。単位のない数値は常に定数の無限長信号として解釈されます（下記参照）。
