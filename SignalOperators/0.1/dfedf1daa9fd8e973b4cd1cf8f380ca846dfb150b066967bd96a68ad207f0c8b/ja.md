```
fadeto(x,y,[len=10ms],[fn=x->sinpi(0.5x)])
```

xをyに追加し、`len`秒間のスムーズな遷移でxからyにフェードします（したがって、合計の長さは`duration(x) + duration(y) - len`です）。

このフェードは、`x`の[`rampoff`](@ref)と`y`の[`rampon`](@ref)によって実現されます。`fn`は、定義域[0,1]に対して範囲[0,1]で単調非減少である必要があります。すなわち、`fn(0) == 0`および`fn(1) == 1`である必要があります。

`len`と`fn`はオプションの引数です：どちらか一方または両方を指定できますが、`len`が存在する場合は`fn`の前に来る必要があります。
