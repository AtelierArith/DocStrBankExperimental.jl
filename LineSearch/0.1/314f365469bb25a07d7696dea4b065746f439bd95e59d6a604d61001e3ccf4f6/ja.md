```
RobustNonMonotoneLineSearch(; gamma = 1 // 10000, sigma_0 = 1, M::Int = 10,
    tau_min = 1 // 10, tau_max = 1 // 2, n_exp::Int = 2, maxiters::Int = 100,
    η_strategy = (fn₁, n, uₙ, fₙ) -> fn₁ / n^2)
```

ロバスト非単調線形探索は、DF Saneによる導関数フリーの線形探索法です [la2006spectral](@cite)。

### キーワード引数

  * `M`: アルゴリズムの単調性は、この正の整数によって決まります。`M`の値が1の場合、関数`f`のL2ノルムの減少において厳密な単調性が得られます。しかし、より高い値はこの減少においてより柔軟性を許します。それにもかかわらず、アルゴリズムはGrippo-Lampariello-Lucidi条件に従った非単調線形探索アルゴリズムを使用することで、グローバル収束を保証します。通常、5から20の範囲の値で十分ですが、場合によっては`M`のより高い値が必要になることがあります。デフォルト設定は10です。
  * `gamma`: 提案されたステップが受け入れられるかどうかに影響を与えるパラメータです。`gamma`の値が高いほど、アルゴリズムはステップを受け入れる際により制限的になります。デフォルトは`1e-4`です。
  * `tau_min`: ステップが拒否された場合、新しいステップサイズは因子で乗算され、このパラメータはその因子の最小値です。デフォルトは`0.1`です。
  * `tau_max`: ステップが拒否された場合、新しいステップサイズは因子で乗算され、このパラメータはその因子の最大値です。デフォルトは`0.5`です。
  * `n_exp`: 損失の指数、すなわち $f_n=||F(x_n)||^{n\_exp}$ です。論文では `n_exp ∈ {1, 2}` を使用しています。デフォルトは`2`です。
  * `η_strategy`: パラメータ `η` を決定する関数で、$||f_n||^2$ の成長を可能にします。`η = η_strategy(fn_1, n, x_n, f_n)` として呼び出され、`fn_1` は $fn_1=||f(x_1)||^{n\_exp}$ として初期化され、`n` は反復回数、`x_n` は現在の `x` 値、`f_n` は現在の残差です。$η > 0$ および $∑ₖ ηₖ < ∞$ を満たす必要があります。デフォルトは $fn_1 / n^2$ です。
  * `maxiters`: アルゴリズムの内部ループで許可される最大反復回数です。デフォルトは`100`です。
