```
 pdlqry(psys, Q, R, S; kwargs...) -> (F, EVALS)
```

最適な周期的安定化ゲイン行列 `F(t)` を計算します。これは、次の形式の周期的離散時間状態空間モデル `psys` に対して

```
x(t+1) = A(t)x(t) + B(t)u(t) + Bw(t)w(t)
  y(t) = C(t)x(t) + D(t)u(t) + Dw(t)w(t),
```

状態フィードバック制御則

```
 u(t) = F(t)x(t)
```

が二次インデックス

```
 J = Sum {y'(t)Q(t)y(t) + u'(t)R(t)u(t) + 2*y'(t)S(t)u(t)}
```

を最小化するようにします。

制御入力 `u(t)` が `m` 個、出力が `p` 個のシステムにおいて、`Q(t)` と `R(t)` はそれぞれ `p×p` および `m×m` の対称行列であり、`S(t)` は `p×m` の行列です。`u(t)` の次元は `R(t)` の次元から推測されます。`R`、`Q`、および `S` は、定数実行列として代わりに提供することもできます。行列 `S` は省略された場合、ゼロに設定されます。

また、`A(t)+B(t)F(t)` の閉ループ特性指数 `EVALS` も返されます。

`kwargs` に含まれるキーワード引数は、関数 [`PeriodicMatrixEquations.prdric`](@extref) で使用されるものです。

注意: ペア `(A(t),B(t))` は安定化可能でなければならず、`[Q S;S' R]` は非負定義でなければなりません。 ```
