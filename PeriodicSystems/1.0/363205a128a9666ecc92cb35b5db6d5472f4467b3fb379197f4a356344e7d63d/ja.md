```
 pdkeg(psys, Qw, Rv, Sn; kwargs...) -> (L, EVALS)
```

離散時間周期状態空間モデル `psys` のカルマン推定器周期ゲイン行列 `L(t)` を計算します。このモデルは次の形式です。

```
x(t+1) = A(t)x(t) + B(t)u(t) + w(t)
  y(t) = C(t)x(t) + D(t)u(t) + v(t)
```

およびノイズ共分散データ `E{w(t)w(t)'} = Qw(t)`, `E{v(t)v(t)'} = Rv(t)`, `E{w(t)v(t)'} = Sn(t)` に対して、カルマン推定器は次のようになります。

```
  xe(t+1) = A(t)xe(t) + B(t)u(t) + L(t)(y(t)-C(t)xe(t)-D(t)u(t))
```

出力が `p` の順序 `n` のシステムに対して、`Qw` と `Rv` はそれぞれ `n×n` および `p×p` の対称行列であり、`Sn` は `n×p` の行列です。`Qw`、`Rv` および `Sn` は定数実行列として提供することもできます。行列 `Sn` は省略された場合はゼロに設定されます。

また、閉ループ特性指数 `EVALS` が `A(t)-L(t)C(t)` として返されます。

`kwargs` に含まれるキーワード引数は、関数 [`PeriodicMatrixEquations.pfdric`](@extref) で使用されるものです。

注意: ペア `(A(t),C(t))` は検出可能でなければならず、`[Qw Sn;Sn' Rv]` は非負定義でなければなりません。
