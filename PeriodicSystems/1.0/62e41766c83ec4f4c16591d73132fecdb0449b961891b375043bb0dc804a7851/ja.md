```
 pclqry(psys, Q, R, S; intpol = true, kwargs...) -> (F, EVALS)
```

最適な周期的安定化ゲイン行列 F(t) を計算します。これは、次の形式の周期的連続時間状態空間モデル `psys` に対して、        .       x(t) = A(t)x(t) + B(t)u(t) + Bw(t)w(t)        y(t) = C(t)x(t) + D(t)u(t) + Dw(t)w(t) ,

状態フィードバック制御則

```
 u(t) = F(t)x(t)
```

が二次インデックス

```
 J = Integral {y'(t)Q(t)y(t) + u'(t)R(t)u(t) + 2*y'(t)S(t)u(t)} dt
```

を最小化するようにします。

`m` 個の制御入力 `u(t)` と `p` 個の出力を持つシステムにおいて、`Q(t)` と `R(t)` はそれぞれ `p×p` および `m×m` の対称行列であり、`S(t)` は `p×m` の行列です。 `u(t)` の次元は `R(t)` の次元から推測されます。 `R`、`Q` および `S` は、定数実行列として代わりに提供することもできます。 行列 `S` は省略された場合、ゼロに設定されます。

また、`A(t)+B(t)F(t)` の閉ループ特性指数 `EVALS` も返されます。

`intpol = true`（デフォルト）の場合、結果として得られる周期的ゲイン `F(t)` は、補間に基づく式を使用して、連続時間周期的行列微分リカッティ方程式の安定化解から計算されます。 `intpol = false` の場合、ゲイン `F(t)` は、対応する常微分方程式の積分を使用して、最近接点の値を初期条件としてリカッティ微分方程式の多点解から計算されます。このオプションは、デフォルトで使用されるシンプレクティック積分器と共同で使用することは推奨されません。

`kwargs` に含まれるキーワード引数は、関数 [`PeriodicMatrixEquations.prcric`](@extref) に使用されるものです（intpolを除く）。

注意: ペア `(A(t),B(t))` は安定化可能でなければならず、`[Q S;S' R]` は非負定義でなければなりません。 ```
