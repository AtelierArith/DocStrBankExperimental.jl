```
 psmrc2d(sys, Ts; ki, ko) -> psys::PeriodicStateSpace{PeriodicMatrix}
```

線形時間不変システムのマルチレート離散化を行います。

連続時間状態空間システム `sys = (A,B,C,D)`、基本サンプリング時間 `Ts`、およびそれぞれ入力および出力サンプリングサブ期間の数を含む整数ベクトル `ki` と `ko` に対して、周期 `T = n*Ts` の対応する離散化された周期システム `psys = (Ap,Bp,Cp,Dp)` が決定されます。ここで、`n` は `ki` と `ko` に含まれる整数の最小公倍数です。連続時間システム `sys` に対しては、ゼロオーダーホールドに基づく離散化手法が使用され、`i` 番目の入力は長さ `ki[i]*Ts` の間隔中は一定です。出力ホールド装置が使用され、`i` 番目の出力は長さ `ko[i]*Ts` の間隔中は一定の間隔出力を提供します。定義されたサンプル時間 `Δ` を持つ離散時間システムの場合、`Δ` を基本サンプル時間として入力および出力の再サンプリングが行われ、`Ts` の値は無視されます。システムのサンプル時間が定義されていない場合、`Ts` の値が基本サンプル時間として使用されます。
