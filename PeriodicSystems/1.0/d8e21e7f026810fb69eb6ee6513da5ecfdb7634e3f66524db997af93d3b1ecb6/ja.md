```
 psyscl = pssofeedback(psys, F, K, (inp, out); negative = false)
```

与えられた周期的システム `psys` に対して、入力ベクトル `u` と出力ベクトル `y` のために、周期的状態フィードバックゲイン `F(t)` と周期的カルマンゲイン `K(t)` を用いて、メモリーレス状態フィードバック `u[inp] = F(t)*xe + v` に対応する閉ループ周期的システム `psyscl` を構築します。また、状態 `xe` と入力 `[u[inp]; y[out]]` を持つ完全状態推定器を使用します。ここで、`inp` と `out` はインデックス、インデックスのベクトル、インデックス範囲、`:` またはそれらの任意の組み合わせです。異なるインデックスのみを指定できます。`negative = true` の場合、負の状態フィードバック `u[inp] = -F(t)*xe + v` が使用されます。
