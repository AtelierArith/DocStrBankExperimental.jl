```
 psyscl = psfeedback(sys, K, (inp, out); negative = false)
```

標準状態空間システム `sys` に対して、入力ベクトル `u` と出力ベクトル `y` および周期的出力フィードバックゲイン `K(t)` を用いて、メモリーレス出力フィードバック `u[inp] = K(t)*y[out] + v` に対応する閉ループ周期システム `psyscl` を構築します。ここで、`inp` と `out` はインデックス、インデックスのベクトル、インデックスの範囲、`:` またはそれらの任意の組み合わせです。指定できるのは異なるインデックスのみです。`negative = true` の場合、負のフィードバック `u[inp] = -K(t)*y[out] + v` が使用されます。連続時間システム `sys` の場合、`K` は周期的スイッチング行列または離散時間周期行列でなければなりません。一方、離散時間システム `sys` の場合、`K` は同じサンプル時間を持つ離散時間周期行列でなければなりません。
