```
pszero(psys::PeriodicStateSpace{HarmonicArray}[, N]; P, atol, rtol, fast) -> val
pszero(psys::PeriodicStateSpace{PeriodicFunctionMatrix}[, N]; P, atol, rtol, fast) -> val
pszero(psys::PeriodicStateSpace{PeriodicSymbolicMatrix}[, N]; P, atol, rtol, fast) -> val
```

連続時間周期システム `psys = (A(t), B(t), C(t), D(t))` の有限および無限のゼロを `val` に計算します。ここで、周期システム行列 `A(t)`、`B(t)`、`C(t)`、および `D(t)` は *Harmonic Array*、*Periodic Function Matrix*、または *Periodic Symbolic Matrix* 表現であり（最後の2つの表現は自動的に *Harmonic Array* 表現に変換されます）。`N` はシステム行列のフーリエ級数における選択された調和成分の数です（デフォルト: `N = max(20,nh-1)`、ここで `nh` は調和項の最大数です）およびキーワードパラメータ `P` は考慮される完全な周期の数です（デフォルト: `P = 1`）で、切り捨てられたブロックトピレッツ行列に基づく周波数リフトされたLTI表現を構築します。

*複素* リフトされたシステムのゼロの計算は、対応するシステムペンシルを有限および無限の固有値を示す適切なクロンカー様式に削減することによって行われます。削減は直交類似変換を使用して行われ、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解に基づくランクの決定が含まれ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。周期 `T` のシステム `psys` に対して、有限ゼロは虚部が区間 `[-ω/2, ω/2]` にある固有値として決定されます。ここで、`ω = 2π/(P*T)` です。可能な偽の有限固有値を排除するために、`N` および `N+2` 調和成分に対して得られた2つのリフトされたシステムの2つの有限固有値集合の交差が計算されます。無限ゼロは、ランダムな時間値 `ti` に対して得られるLTIシステム `(A(ti), B(ti), C(ti), D(ti))` の無限ゼロとして決定されます。*警告:* この無限ゼロの数の評価はほとんど正しい結果を提供しますが、このアプローチの理論的評価はありません（反例は歓迎します！）。

キーワード引数 `atol` および `rtol` は、それぞれ基礎となるリフトされたシステムペンシルの非ゼロ要素の絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` はペンシルの最小次元のサイズ、`ϵ` は作業用マシンエプシロンです。 ```
