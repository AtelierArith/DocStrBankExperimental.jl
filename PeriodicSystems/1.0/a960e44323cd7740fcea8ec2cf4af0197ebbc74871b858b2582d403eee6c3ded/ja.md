```
 psyscl = pssfeedback(psys, F, inp; negative = false)
```

与えられた周期的システム `psys` と入力ベクトル `u` および出力ベクトル `y`、および周期的状態フィードバックゲイン `F(t)` に対して、メモリーレス状態フィードバック `u[inp] = F(t)*x + v` に対応する閉ループ周期的システム `psyscl` を構築します。ここで、`inp` はインデックスのベクトル、インデックス範囲、`:` またはそれらの任意の組み合わせです。異なるインデックスのみを指定できます。`negative = true` の場合、負の状態フィードバック `u[inp] = -F(t)*x + v` が使用されます。
