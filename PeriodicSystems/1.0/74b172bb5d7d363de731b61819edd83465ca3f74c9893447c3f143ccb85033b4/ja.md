```
 pckeg(psys, Qw, Rv, Sn; intpol = true, kwargs...) -> (L, EVALS)
```

連続時間周期状態空間モデル `psys` のカルマン推定器周期ゲイン行列 `L(t)` を計算します。このモデルは次の形式です。

```
  .
  x(t) = A(t)x(t) + B(t)u(t) + w(t)
  y(t) = C(t)x(t) + D(t)u(t) + v(t)
```

およびノイズ共分散データ `E{w(t)w(t)'} = Qw(t)`, `E{v(t)v(t)'} = Rv(t)`, `E{w(t)v(t)'} = Sn(t)` に対して、カルマン推定器は次のように表されます。

```
   .
  xe(t) = A(t)xe(t) + B(t)u(t) + L(t)(y(t)-C(t)xe(t)-D(t)u(t))
```

順序 `n` のシステムで `p` 出力を持つ場合、`Qw` と `Rv` はそれぞれ `n×n` および `p×p` の対称行列であり、`Sn` は `n×p` 行列です。`Qw`、`Rv` および `Sn` は、定数実行列として代わりに提供することもできます。行列 `Sn` は省略された場合、ゼロに設定されます。

また、閉ループ特性指数 `EVALS` が `A(t)-L(t)C(t)` として返されます。

`intpol = true`（デフォルト）の場合、結果の周期ゲイン `L(t)` は、補間に基づく式を使用して連続時間周期行列微分リカッチ方程式の安定化解から計算されます。`intpol = false` の場合、ゲイン `L(t)` は、対応する常微分方程式の積分を使用して最近接点の値を初期条件としてリカッチ微分方程式の多点解から計算されます。このオプションは、デフォルトで使用されるシンプレクティック積分器と一緒に使用することは推奨されません。

`kwargs` に含まれるキーワード引数は、関数 [`PeriodicMatrixEquations.pfcric`](@extref) に使用されるものです（`intpol` を除く）。

注意: ペア `(A(t),C(t))` は検出可能でなければならず、`[Qw Sn;Sn' Rv]` は非負定でなければなりません。 ```
