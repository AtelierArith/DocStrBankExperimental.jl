```
 pdlqr(psys, Q, R, S; kwargs...) -> (F, EVALS)
```

最適な周期的安定化ゲイン行列 `F(t)` を計算します。これは、次の形式の離散時間周期状態空間モデル `psys` に対して

```
x(t+1) = A(t)x(t) + B(t)u(t) + Bw(t)w(t) 
  y(t) = C(t)x(t) + D(t)u(t) + Dw(t)w(t)
```

状態フィードバック制御則

```
 u(t) = F(t)x(t)
```

が二次インデックス

```
 J = Sum {x'(t)Q(t)x(t) + u'(t)R(t)u(t) + 2*x'(t)S(t)u(t)}
```

を最小化するようにします。

制御入力 `u(t)` が `m` の順序 `n(t)` のシステムに対して、`Q(t)` と `R(t)` はそれぞれ `n(t)×n(t)` および `m×m` の対称行列であり、`S(t)` は `n(t)×m` の行列です。行列 `S` は省略された場合はゼロに設定されます。`u(t)` の次元は `R(t)` の次元から推測されます。`R`、`Q`、および `S` は、定数実行列として代わりに提供することもできます。

また、`A(t)+B(t)F(t)` の閉ループ特性指数 `EVALS` も返されます。

`kwargs` に含まれるキーワード引数は、関数 [`PeriodicMatrixEquations.prdric`](@extref) に使用されるものです。

注意: ペア `(A(t),B(t))` は安定化可能でなければならず、`[Q S;S' R]` は非負定義でなければなりません。 ```
