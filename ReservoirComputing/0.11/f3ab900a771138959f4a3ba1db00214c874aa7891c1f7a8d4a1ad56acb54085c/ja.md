```
ESN(train_data; kwargs...) -> ESN
```

エコー状態ネットワーク（ESN）を作成します。

# 引数

  * `train_data`: トレーニングデータの行列 `num_features x time_steps`。
  * `variation`: ESNのバリエーション（デフォルト: `Default()`）。
  * `input_layer`: ESNの入力層。
  * `reservoir`: ESNのリザーバー。
  * `bias`: 各タイムステップのバイアスベクトル。
  * `rng`: 重みの初期化に使用される乱数生成器。デフォルトは `Utils.default_rng()`。
  * `reservoir_driver`: リザーバー状態を進化させるメカニズム（デフォルト: `RNN()`）。
  * `nla_type`: 非線形活性化タイプ（デフォルト: `NLADefault()`）。
  * `states_type`: 状態を保存するためのフォーマット（デフォルト: `StandardStates()`）。
  * `washout`: 廃棄する初期タイムステップ（デフォルト: `0`）。
  * `matrix_type`: 内部で使用される行列のタイプ（デフォルト: `train_data`のタイプ）。
