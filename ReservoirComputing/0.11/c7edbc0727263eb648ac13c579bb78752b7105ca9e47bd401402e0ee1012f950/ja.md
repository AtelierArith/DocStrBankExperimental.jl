```
Generative(prediction_len)
```

モデルが自らの出力を次の予測ステップの入力として再帰的にフィードバックすることによって、自律的なマルチステップ予測を生成することを可能にする予測戦略です。

# パラメータ

  * `prediction_len`: 予測する未来のステップ数。

# 説明

`Generative`予測メソッドは、モデルが以前の予測を未来の予測の入力として使用することによって、マルチステップ予測を行うことを可能にします。

各ステップで、モデルは現在の入力を取り、予測を生成し、その予測を次のステップの入力に組み込みます。この再帰的プロセスは、指定された予測ステップ数（`prediction_len`）に達するまで続きます。
