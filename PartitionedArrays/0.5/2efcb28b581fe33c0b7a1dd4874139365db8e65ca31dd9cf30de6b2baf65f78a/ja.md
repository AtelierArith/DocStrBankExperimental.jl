```
MPIArray{T,N}
```

要素型 `T` と次元数 `N` の配列を表し、配列内の各アイテムは別々のMPIプロセスに格納されます。つまり、各MPIランクは1つのアイテムのみを格納します。ランクごとに複数のアイテムを格納できる配列については、[`PVector`](@ref) または [`PSparseMatrix`](@ref) を参照してください。この構造体はJuliaの配列インターフェースを実装しています。ただし、パフォーマンス上の理由（通信コスト）から `setindex!` と `getindex!` の使用は無効になっています。

# プロパティ

この構造体のフィールド（および内部コンストラクタ）はプライベートです。`MPIArray` のインスタンスを生成するには、関数 [`distribute_with_mpi`](@ref) を使用してください。

# スーパークラス階層

```
MPIArray{T,N} <: AbstractArray{T,N}
```
