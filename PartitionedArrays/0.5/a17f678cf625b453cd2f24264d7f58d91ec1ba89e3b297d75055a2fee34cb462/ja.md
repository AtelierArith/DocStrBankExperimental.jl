```
struct PVector{V,A,B,...}
```

`PVector`（パーティションベクトル）は、エントリが異なる部分に分散（すなわち、パーティション化）されているベクトルを表す型で、分散メモリ並列計算のために使用されます。

この型は、対応する並列実装を持つ多数の配列のような操作をオーバーロードします。

# プロパティ

  * `vector_partition::A`
  * `index_partition::B`

`vector_partition[i]` には、データ分配における `i` 番目の部分のローカル値のベクトルが含まれています。最初の型パラメータ `V` は `typeof(values[i])` に対応し、ローカル値を格納するために使用されるベクトル型です。アイテム `index_partition[i]` は [`AbstractLocalIndices`](@ref) インターフェースを実装し、`i` 番目の部分におけるローカル、所有、およびゴーストインデックスに関する情報を提供します。

この構造体の残りのフィールドと型パラメータはプライベートです。

# スーパタイプ階層

```
PVector{V,A,B,...} <: AbstractVector{T}
```
