```
vtk_surface([f::Function], filename, xs, ys, zs; kwargs...)
```

2D関数のサーフェスプロットを表す非構造化グリッドファイル（".vtu"）を作成します。座標（`xs`、`ys`）に対して値`zs`を持ちます。

座標`xs`と`ys`は次のように指定できます：

  * 正規グリッドの場合は、それぞれの次元`Nx`と`Ny`の1D配列；
  * 不規則グリッドの場合は、次元が`(Nx, Ny)`の2D配列。

値`zs`は次元が`(Nx, Ny)`の2D配列で指定する必要があります。

# 追加データの含め方

オプションとして、関数`f`を介して生成されたファイルに追加データを書き込むことができます。この関数は[`vtk_grid`](@ref)と同様に動作します。

例えば、サーフェスプロットを高さ`z`で色付けすることが一般的です。これを行うには、`zs`行列をポイントデータとして書き込む必要があります：

```julia
julia> xs = 0:0.5:10; ys = 0:1.0:20;

julia> zs = @. cos(xs) + sin(ys');

julia> vtk_surface("surf", xs, ys, zs) do vtk
           vtk["z_values"] = zs
       end
```

含まれるデータは、次元が`(Nx, Ny)`（ポイントデータの場合）または`(Nx - 1, Ny - 1)`（セルデータの場合）である必要があることに注意してください。
