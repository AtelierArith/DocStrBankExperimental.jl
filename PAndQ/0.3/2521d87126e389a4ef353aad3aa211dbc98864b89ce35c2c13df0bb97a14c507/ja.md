```
TruthTable(ps)
```

与えられた命題の[真理値表](https://en.wikipedia.org/wiki/Truth_table)を構築します。

ヘッダーは命題のシーケンスです。ボディは、行がヘッダー内の各命題の[`解釈`](@ref)である行列です。

真理値[@ref nullary_operators]に論理的に同等な命題は左にグループ化され、次に原子命題に同等なものが続き、最後に他のすべての命題が続きます。論理的に同等な命題は一緒にグループ化されます。同じテキスト表現を持つ命題は一度だけ表示されます。

# 例

```jldoctest
julia> TruthTable([⊤])
┌───┐
│ ⊤ │
├───┤
│ ⊤ │
└───┘

julia> @atomize TruthTable([¬p])
┌───┬────┐
│ p │ ¬p │
├───┼────┤
│ ⊤ │ ⊥  │
│ ⊥ │ ⊤  │
└───┴────┘

julia> @atomize TruthTable([p ∧ ¬p, p → q, ¬p ∨ q])
┌────────┬───┬───┬───────────────┐
│ p ∧ ¬p │ p │ q │ p → q, ¬p ∨ q │
├────────┼───┼───┼───────────────┤
│ ⊥      │ ⊤ │ ⊤ │ ⊤             │
│ ⊥      │ ⊥ │ ⊤ │ ⊤             │
├────────┼───┼───┼───────────────┤
│ ⊥      │ ⊤ │ ⊥ │ ⊥             │
│ ⊥      │ ⊥ │ ⊥ │ ⊤             │
└────────┴───┴───┴───────────────┘
```
