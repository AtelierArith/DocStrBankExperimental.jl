```
@atomize(x)
```

定数と変数をインラインで初期化します。

定数は `$` 補間構文を使用して初期化されます。変数は以前に未定義のシンボルで初期化されます。

!!! warning
    このマクロは、値が割り当てられているシンボルを無視しようとします。たとえば、`@atomize f(; x = p) = x ∧ q` は `f(; x = @atomize p) = x ∧ @atomize q` と同等であるべきです。しかし、この機能は進行中であり、一部のケースでのみ機能します。実装は、まだ処理できない式の部分をスキップするように注意しています。


# 例

```jldoctest
julia> @atomize x = p ∧ q
p ∧ q

julia> @atomize x → r
(p ∧ q) → r

julia> @atomize $1 ∧ $(1 + 1)
$(1) ∧ $(2)
```
