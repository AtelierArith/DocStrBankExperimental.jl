plotcamera - カメラのポーズを示すグラフィカルな表現をプロットします。

```
Usage: plotcamera(C, l; col=[0,0,1], plotCamPath=false, fig=nothing)

Arguments:
           C - カメラ構造体（またはカメラ構造体の配列）
           l - カメラの視野を示す長方形の円錐の辺の長さ。
Keyword Arguments:
         col - 使用するRGB色を指定するオプションの三要素ベクトル。デフォルトは青。
 plotCamPath - カメラの中心位置を結ぶ線をプロットするためのオプションのフラグ true/false。省略または空の場合はデフォルトでfalse。
         fig - 使用するオプションの図番号。指定されていない場合は新しい図が作成されます。
```

この関数は、現在の図に1つまたは複数のカメラのポーズを示すグラフィカルな表現をプロットします。これは、カメラの中心に頂点がある長方形の円錐で構成され、カメラの視野を示します。カメラの座標X軸とY軸もカメラの中心でプロットされます。
