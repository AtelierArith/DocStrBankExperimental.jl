ransacfithomography - RANSACを使用して2Dホモグラフィをフィットします

```
Usage:   (H, inliers) = ransacfithomography(x1, x2, t)

Arguments:
         x1  - 2xNまたは3xNの同次点のセット。データが
               2xNの場合、同次スケールファクターは1であると仮定されます。
         x2  - x1<->x2となる同次点の2xNまたは3xNのセット。
         t   - データポイントとモデル間の距離の閾値
               これは、ポイントがインライヤーであるかどうかを決定するために使用されます。
               ポイント座標は、原点からの平均距離がsqrt(2)になるように正規化されます。
               tの値はこれに対して設定する必要があります。範囲は
               0.001 - 0.01のようにします。
```

x1とx2のマッチングは仮定的であり、マッチの一部が間違っていることが期待されることに注意してください。

```
Returns:
         H       - x2 = H*x1となる3x3のホモグラフィ。
         inliers - 最良モデルのインライヤーであったx1、x2の要素のインデックスの配列。
```

See Also: ransac(), homography2d(), homography1d()
