fitline2d - 2Dポイントのセットに対する最小二乗法による直線フィッティング

```
Usage:   (C, dist) = fitline2d(XY)

Where:   XY  - 2xNpts配列のxy座標で、データに直線をフィットさせるための
               形式は
               [x1 x2 x3 ... xN
                y1 y2 y3 ... yN]

               XYは同次座標の3xNpts配列でも構いません。

Returns: C    - 直線係数の3x1配列で、形式は
                c[1]*X + c[2]*Y + c[3] = 0
         dist - フィットした直線から供給されたデータポイントまでの距離の配列。
                distは、関数が2つの出力引数で呼び出された場合にのみ計算されます。
```

Cの大きさはスケーリングされ、直線方程式は次のようになります。

```
   sin(theta)*X + (-cos(theta))*Y + rho = 0
```

ここで、thetaは直線とx軸の間の角度で、rhoは原点から直線までの垂直距離です。このように係数を再スケーリングすることで、任意の点(x,y)から直線までの垂直距離を単純に計算できます。

```
   r = abs(c[1]*X + c[2]*Y + c[3])
```

この直線表現を古典的な形式に変換したい場合は、

```
     Y = a*X + b
を使用します。
  a = -c[1]/c[2]
  b = -c[3]/c[2]
```

ただし、これはc[2]がゼロでないことを前提としています。
