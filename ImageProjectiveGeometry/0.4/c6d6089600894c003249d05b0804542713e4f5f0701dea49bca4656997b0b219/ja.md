ransacfitfundmatrix - 基本行列をRANSACを使用してフィットします

```
使用法:   (F, inliers) = ransacfitfundmatrix(x1, x2, t)

引数:
         x1  - 2xNまたは3xNの同次点のセット。データが
               2xNの場合、同次スケールファクターは1であると仮定されます。
         x2  - x1<->x2となる同次点のセットで、2xNまたは3xNです。
         t   - データポイントとモデル間の距離の閾値
               点がインライヤーであるかどうかを決定するために使用されます。
               点の座標は、原点からの平均距離がsqrt(2)になるように正規化されます。
               tの値はこれに対して設定する必要があります。範囲は
               0.001 - 0.01の間であるべきです。
```

x1とx2のマッチングは仮定的であり、マッチの一部が間違っていることが期待されることに注意してください。

```
戻り値:
         F       - x2'Fx1 = 0となる3x3の基本行列。
         inliers - 最良モデルのインライヤーであったx1、x2の要素のインデックスの配列。
```

関連情報: ransac(), fundmatrix()
