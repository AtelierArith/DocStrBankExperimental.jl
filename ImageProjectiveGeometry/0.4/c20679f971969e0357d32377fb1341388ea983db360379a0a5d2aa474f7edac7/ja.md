affinefundmatrix - 4つ以上の点からアフィン基本行列を計算します

関数は、ステレオペアの画像における4つ以上の対応する点からアフィン基本行列を計算します。

```
Usage:   (F, e1, e2) = affinefundmatrix(x1, x2)
         (F, e1, e2) = affinefundmatrix(x)

Arguments:
         x1, x2 - 対応する2つの点のセットで、非同次画像座標の
                  2xN配列として定義されます。

         x      - 単一の引数が供給される場合、それは
                  x = [x1; x2] の形式であると仮定されます。
Returns:
         F      - x2'*F*x1 = 0 となる3x3の基本行列。
         e1     - F*e1 = 0 となる画像1のエピポール
         e2     - F'*e2 = 0 となる画像2のエピポール
```

単一の引数を使用する場合は、この関数をransac()と共に使用することを意図しています。

HartleyとZissermanによって示されたゴールドスタンダードアルゴリズム（第2版、p351）が使用されます。

関連情報: fundmatrix()
