homotrans - 同次変換の点/線

2Dまたは3Dの同次または非同次座標に対して変換を実行する関数。結果の座標は、同次スケールが1になるように正規化されます。

```
Usage:     t = homotrans(P, v; checkinf = false)

Arguments:
          P  - 3 x 3 または 4 x 4 同次変換行列
          v  - 3 x n または 4 x n の同次座標の行列または ...
               2 x n または 3 x n の非同次座標の行列
Keyword Argument:
    checkinf - trueの場合、コードは変換された点の同次座標が無限大（スケールパラメータがゼロ）であるかどうかをチェックします。
               そうであれば、点の正規化は試みられません。

Returns   t  - 変換された座標。入力データvに合わせて同次または非同次。
```

入力vは、行数がsize(P,2) - 1に等しい場合、非同次であると仮定されます。
