ransacfitline - RANSACを使用して3Dポイントの配列に線をフィットさせる

```
Usage  (V, L, inliers) = ransacfitline(XYZ, t, feedback)

この関数はRANSACアルゴリズムを使用して、3Dデータポイントのセットまたはスケール値が1の2D同次点のセットに対してロバストに線をフィットさせます。

Arguments:
         XYZ - 線をフィットさせるためのxyz座標の3xNpts配列。
         t   - データポイントと線の間の距離しきい値
               ポイントがインライヤーかどうかを決定するために使用されます。
    feedback - RANSACフィードバック情報をオンにするためのオプションのブールフラグ。

Returns:.
          V - インライヤーと見なされるポイントに対して単純なフィッティングによって得られた線。線はインライヤーポイントの計算された平均を通り、主固有ベクトルに平行です。線は最大固有値の平方根でスケーリングされます。
              この線はnx2行列として返されます。最初の列は始点、2番目の列は線の終点です。
          L - 最も多くのインライヤーを持つ線を定義するために見つかったデータセット内の2つのポイント。
              Lの2つの列が2つのポイントを定義します。
    inliers - フィットした線に対してインライヤーと見なされたポイントのインデックス。
```

See also:  ransac(), fitplane(), ransacfitplane()
