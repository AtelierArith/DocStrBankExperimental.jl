grey2lbp - 画像のグレースケール値をローカルバイナリパターンに変換します。

```
Usage:  lbpim = grey2lbp(img, rc, window)

Arguments:
          img - 処理するグレースケール画像
           rc - [2 x 2*nPairs] の整数 (行;列) 座標の配列。
                各連続する列のペアは、画像のグレースケール値を互いに比較するための
                点のペアを提供し、ある中心特徴位置の周りのパッチの
                バイナリ記述子を形成します。配列 rc は、64 ビットのエンコーディングに
                対応する 128 列を超えてはなりません。
       window - ローカル標準偏差を計算する際に考慮されるウィンドウのサイズを指定する
                2ベクトル。ウィンドウサイズの値は rc の値の範囲と一致し、
                奇数である必要があります。

Returns:
        lbpim - ローカルバイナリパターンでエンコードされた UInt64 画像。
```

各ピクセルは、ピクセルのペアを比較することによって形成されたビットパターンでエンコードされます。ピクセル間の差が負であれば 1 でエンコードされ、そうでなければ 0 でエンコードされます。これにより、照明の変動に対して不変な方法で周囲のピクセルに基づいてピクセルを記述するエンコーディングが提供されます。ただし、エンコーディングは画像の向きに依存します。また、エンコーディングはほぼ一定の領域に対してノイズに敏感です。

マッチング時には、エンコードされたピクセル値を比較するためにハミング距離を使用してください。

関連情報: grey2lbp!(), grey2census(), briefcoords()
