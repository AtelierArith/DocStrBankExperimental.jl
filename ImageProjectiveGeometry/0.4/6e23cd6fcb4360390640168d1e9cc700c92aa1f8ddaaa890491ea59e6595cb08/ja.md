stereorectifytransforms

画像ペアをステレオ整列ペアに変換するホモグラフィーを計算します。

```
Usage:  (P1r, H1, P2r, H2, dmin, dmax, dmed) = stereorectifytransforms(P1, img1, P2, img2,
                                   xy1=zeros(0), xy2=zeros(0); scale=1.0, disparitytruncation=0)

Arguments:
          P1, P2 - 整列される画像の投影行列。
      img1, img2 - 2つの画像、マルチチャネルである可能性があり（同じサイズであると仮定）。
         xy1,xy2 - オプション: 画像内の対応する同次画像座標の2セット
                   [3 x N]のサイズ。 提供された場合、これらは
                   整列された画像点間の相対的な変位を最小化することを目指す
                   整列を構築するために使用されます。
Keyword argument:
                scale - 画像の希望するスケーリング、デフォルトは1.0
  disparitytruncation - 画像座標の不均一性のヒストグラムの端からクリップされる
                        パーセンテージで、不均一性範囲の決定に使用されます。

Returns:
        P1r, P2r - 整列された画像の投影行列。
          H1, H2 - img1およびimg2に適用されるホモグラフィーで、
                   ステレオ整列ペアを取得します。
      dmin, dmax - 整列された画像座標から導出された画像間の不均一性の範囲（提供された場合）。
            dmed - 不均一性の中央値。
```

xy1とxy2を提供することの価値は、画像間の任意の一致点でのピクセルの局所的な空間配置がより類似している可能性が高く、したがって画像記述子がより良く機能する可能性があることです。また、画像間の不均一性の範囲はできるだけ均一に保たれます。

不均一性の範囲は、指定されたトランケーションが適用された提供された画像座標間の不均一性のヒストグラムから導出され、外れ値を除外します。

See also: stereorectify()
