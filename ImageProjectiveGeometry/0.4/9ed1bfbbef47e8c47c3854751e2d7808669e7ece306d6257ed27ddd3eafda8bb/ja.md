fundmatrix - 8点以上の点から基本行列を計算します

関数は、ステレオペアの画像における8点以上の対応する点から基本行列を計算します。HartleyとZissermanによって示された正規化8点アルゴリズム（p265）が使用されます。正確な結果を得るためには、12点以上の使用が推奨されます。

```
Usage:   (F, e1, e2) = fundmatrix(x1, x2)
         (F, e1, e2) = fundmatrix(x)

Arguments:
         x1, x2 - 対応する3xNの同次点の2つのセット。

         x      - 単一の引数が供給される場合、それはx = [x1; x2]の形式であると仮定されます。
Returns:
         F      - x2'*F*x1 = 0となる3x3の基本行列。
         e1     - F*e1 = 0となる画像1のエピポール
         e2     - F'*e2 = 0となる画像2のエピポール
```

単一の引数を使用した場合の使い方は、この関数をransac()と共に使用することを意図しています。

参照: affinefundmatrix()
