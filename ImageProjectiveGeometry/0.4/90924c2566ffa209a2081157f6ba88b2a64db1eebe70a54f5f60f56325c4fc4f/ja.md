ransacfitaffinefundmatrix - アフィン基本行列をRANSACを使用してフィッティングします

```
Usage:   (F, inliers) = ransacfitaffinefundmatrix(x1, x2, t)

Arguments:
         x1  - 2xNまたは3xNの同次点のセット。データが
               2xNの場合、同次スケール因子は1であると仮定されます。
         x2  - x1<->x2となる同次点の2xNまたは3xNのセット。
         t   - データポイントとモデル間の距離の閾値
               これは、ポイントがインライヤーであるかどうかを決定するために使用されます。
               ポイント座標は、原点からの平均距離がsqrt(2)になるように正規化されます。
               tの値はこれに対して設定する必要があります。例えば、範囲は
               0.001 - 0.01の間です。
```

x1とx2のマッチングは仮定的であり、マッチの一部が間違っていることが期待されることに注意してください。

```
Returns:
         F       - x2'Fx1 = 0となる3x3の基本行列。
         inliers - 最良モデルのインライヤーであったx1、x2の要素のインデックスの配列。
```

関連情報: ransac(), fundmatrix(), affinefundmatrix()
