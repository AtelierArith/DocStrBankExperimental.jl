fastradial - LoyとZelinskiの高速放射状特徴検出器

```
Usage: (S, So) = fastradial(img, radii, alpha=2, beta=0)

Arguments:
           img   - 分析する画像
                   ::Array{T,2} または ::ImageMeta{T,2} 
           radii - 処理する整数半径値のベクトル
                   推奨される半径は [1, 3, 5] です
           alpha - 放射状の厳密さパラメータ。
                   1 - ゆるい、両側対称性を持つ特徴を受け入れます。
                   2 - 妥当な妥協。
                   3 - 厳密、放射状対称性のみを受け入れます。
                       ... さらに高くすることもできます
           beta  - 勾配の閾値。この閾値未満の勾配は
                   対称性測定に寄与せず、デフォルトは0です。

Returns    S     - 対称性マップ。高い対称性を持つ明るい点は
                   大きな正の値でマークされます。高い対称性を持つ
                   暗い点は大きな負の値でマークされます。
           So    - 向きのみに基づく対称性マップ。
```

ポイントをローカライズするには、S、-S、またはabs(S)に対してnonmaxsuppts()を使用します。何を見つけようとしているかによります。

顔画像内の目を検出するための良いアルゴリズムです。

入力がImageMetaの場合、cimgはプロパティspatialorderが["y","x"]に設定されたImageMetaになります。

参考文献:   Loy, G.  Zelinsky, A.  高速放射状対称性による興味点の検出。  IEEE PAMI, Vol. 25, No. 8, 2003年8月。 pp 959-973。
