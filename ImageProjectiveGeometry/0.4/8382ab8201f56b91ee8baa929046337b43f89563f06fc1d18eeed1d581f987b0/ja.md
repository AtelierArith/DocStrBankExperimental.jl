harris - ハリスコーナー検出器

```
Usage:                cimg = harris(img, sigma=1; k=0.04)
              (cimg, r, c) = harris(img, sigma=1; k=0.04, args...)

Arguments:   
           img    - 処理する画像。
                    ::Array{T,2} または ::ImageMeta{T,2} 
           sigma  - ガウス和ウィンドウの標準偏差。 
                    使用する典型的な値は1-3です。デフォルトは1です。

Keyword arguments:
                k - 構造テンソルのトレースに適用するスケーリングファクター。デフォルトは伝統的な値 
                    の0.04です。

          args... - 非最大抑制を実行するためのキーワード引数のシーケンス。
                    これらは次のとおりです（詳細はnonmaxsuppts()を参照）。
           radius - 非最大抑制で考慮される領域の半径。使用する典型的な値は
                    1-3ピクセルです。デフォルト（および最小値）は1です。
           thresh - 閾値、閾値より大きい値を持つ特徴のみが返されます。デフォルトは0です。
                N - 返す最大特徴数。この場合、'thresh'を超える
                    N個の最も強い特徴が返されます。デフォルトはInfです。
         subpixel - trueに設定すると、特徴がサブピクセル
                    精度にローカライズされます。デフォルトはfalseです。
             img  - オプションの画像データ。画像が提供されると、
                    検出されたコーナーがこの画像にオーバーレイされます。これは
                    パラメータ調整に役立ちます。デフォルトは[]です。

Returns:
           cimg   - コーナー強度画像。  
           r      - コーナーポイントの行座標。
           c      - コーナーポイントの列座標。
```

入力がImageMetaの場合、cimgはプロパティspatialorderが["y","x"]に設定されたImageMetaとなり、r、cで返される座標と一貫性が保たれます。

オプションの非最大キーワード引数が指定されていない場合、`cimg`のみが生のコーナー強度画像として返されます。その後、'cimg'内の値を確認して使用する適切な閾値を決定することができます。ハリスコーナー強度は、強度勾配の4乗に比例して変化することに注意してください。入力画像のコントラストの小さな変化は、適切な閾値に大きな変化をもたらします。

ハリス測定はdet(M) - k*trace^2(M)であり、ここでkは設定する必要があるパラメータ（伝統的にk = 0.04）で、Mは構造テンソルです。パラメータkを設定する必要がないようにしたい場合は、ノーブルの測定を使用してください。ただし、実際に望むのはおそらくシー・トマシの測定です。

参照: noble(), shi_tomasi(), hessianfeatures(), nonmaxsuppts(), derivative5()
