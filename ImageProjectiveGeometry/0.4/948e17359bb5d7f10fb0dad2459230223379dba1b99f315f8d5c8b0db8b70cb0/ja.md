grey2census - 画像のグレースケール値をセンサス値に変換する

```
使用法:  cimg = grey2census(img, window)

引数:
          img - 処理するグレースケール画像
       window - センサス変換を計算する際に考慮されるウィンドウのサイズを指定するオプションの2ベクトル [行, 列]
                デフォルトは [7, 9] です。値は奇数でなければならず、
                その積は64を超えてはいけません。
戻り値:
         cimg - センサスエンコードされた UInt64 画像。
```

各ピクセルは、周囲のウィンドウ内のピクセルと比較することによって形成されたビットパターンでエンコードされます。ウィンドウ内のピクセルが中心ピクセルよりも小さい場合、その対応するビットがセンサスエンコーディングで設定されます。これにより、照明の変動に対して不変な方法で周囲のピクセルに基づいてピクセルを記述するエンコーディングが提供されます。ただし、エンコーディングは画像の向きに依存することに注意してください。

マッチング時にエンコードされたピクセル値を比較するためにハミング距離を使用します。

```
  hammingDist = count_ones(img1[r,c] $ img2[r,c])
```
