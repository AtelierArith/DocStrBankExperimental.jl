カメラ - ブラウン歪みモデルに従ったカメラのパラメータを定義する構造体

```
      fx::Float64      # 焦点距離。
      fy::Float64
     ppx::Float64      # 主点。
     ppy::Float64
      k1::Float64      # 放射レンズ歪みパラメータ。
      k2::Float64
      k3::Float64
      p1::Float64      # 接線レンズ歪みパラメータ。
      p2::Float64
    skew::Float64
    rows::Integer   # オプション、行と列を提供することで
    cols::Integer   # 画像の境界外に投影される点の検出を可能にします。
       P::Array     # 世界座標におけるカメラの位置。
    Rc_w::Array     # カメラフレームに対する世界の向きを定義する回転行列。
```

接線歪みパラメータp1とp2の順序は、文献によって常に一貫しているわけではありません。cameraproject()内では、xとyはそれぞれ列と行の座標を指し、以下の順序で使用されます。

```
   dx = 2*p1*x*y          +  p2*(r^2 + 2*x^2)
   dy = p1*(r^2 + 2*y^2)  +  2*p2*x*y

```

コンストラクタ:

```
      Camera(keyword = value, ....)

      Camera(P)     # Pは3x4の投影行列です。
                    # この場合、fx、fy、skew、P、およびRc_wのみが定義でき、
                    # その他の（歪み）パラメータは0に設定されます。
```
