ransacfitplane - 3Dポイントの配列に平面をRANSACを使用してフィットさせます

```
Usage  (B, P, inliers) = ransacfitplane(XYZ, t, feedback)

この関数はRANSACアルゴリズムを使用して、3Dデータポイントのセットに対して堅牢に平面をフィットさせます。

Arguments:
         XYZ - 平面にフィットさせるためのxyz座標の3xNpts配列。
         t   - データポイントと平面の間の距離の閾値
               ポイントがインライヤーかどうかを決定するために使用されます。
         feedback - RANSACフィードバック情報をオンにするためのオプションのフラグ0または1。

Returns:
          B - 平面係数の4x1配列
              b[1]*X + b[2]*Y +b[3]*Z + b[4] = 0の形
              Bの大きさは1です。
              この平面は、インライヤーと見なされたすべてのポイントに対する最小二乗フィットによって得られます。したがって、この平面は以下のPによって定義されるものとはわずかに異なります。
          P - インライヤーの数が最も多い平面を定義するために見つかったデータセット内の3つのポイント。
              Pの3つの列が3つのポイントを定義します。
    inliers - フィットした平面に対してインライヤーと見なされたポイントのインデックス。
```

See also:  ransac(), fitplane()
