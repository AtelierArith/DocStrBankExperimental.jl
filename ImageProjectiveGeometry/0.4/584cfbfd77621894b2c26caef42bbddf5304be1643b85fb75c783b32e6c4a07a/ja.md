hline - 同次座標で定義された2Dラインをプロットします。

2つの点で定義された2D同次ラインまたは単一の同次ベクトルで定義されたラインをプロットするための関数

```
Usage 1:   hline(p1,p2, linestyle="b-")
Arguments: p1, p2 - 同次座標で定義された2つの3ベクトル

Usage 2:    hline(l, linestyle="b-")
Argument:   l - 同次座標で定義された3ベクトルによるライン

```

引数として同次ラインが提供される場合、描画されるラインの範囲は現在の軸の制限に依存することに注意してください。これには、この関数を呼び出す前にPyPlot.axis()を呼び出して希望の制限を設定する必要があります。
