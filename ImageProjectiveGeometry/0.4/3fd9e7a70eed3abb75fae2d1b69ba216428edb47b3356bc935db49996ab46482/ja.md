ransac - RANSACアルゴリズムを使用してデータにモデルをロバストにフィットさせる

```
Usage:

(M, inliers) = ransac(x, fittingfn, distfn, degenfn s, t, feedback,
                      maxDataTrials, maxTrials)

Arguments:
    x         - モデルMにフィットさせようとしているデータセット
                xはサイズ[d x Npts]であると仮定されます
                ここでdはデータの次元数、Nptsは
                データポイントの数です。

    fittingfn - xからのsデータにモデルをフィットさせる関数への参照。
                関数は次の形式であると仮定されます:
                   M = fittingfn(x)
                フィッティング関数が複数のモデルを返すことがあることに注意してください
                （例えば、7対のマッチしたポイントに対して最大3つの基本行列がフィットできます）。
                この場合、フィッティング関数はモデルの配列を返すと仮定されます。
                この関数がモデルをフィットできない場合、Mは
                空の行列として返す必要があります。

    distfn    - モデルからデータxへの距離を評価する関数への参照。
                関数は次の形式であると仮定されます:
                   (inliers, M) = distfn(M, x, t)
                この関数は、ポイントとモデル間の距離を評価し、
                モデルからの距離't'以内にあるポイント、すなわち
                インライヤーであるxの要素のインデックスを返す必要があります。
                さらに、Mが可能なモデルの配列である場合、distfn()は
                最も多くのインライヤーを持つモデルを返します。
                モデルが1つだけの場合でも、この関数は
                モデルを出力にコピーする必要があります。この呼び出しの後、Mは
                単一のモデルを表す単一のオブジェクトになります。

    degenfn   - データポイントのセットが退化モデルを生成するかどうかを判断する関数への参照。
                これは、役に立たないモデルを生成しないランダムサンプルを破棄するために使用されます。
                degenfnは次の形式のブール関数であると仮定されます:
                   r = degenfn(x)
                退化性のテストを考案できない場合は、常に
                trueの値を返すダミー関数を書き、データセットが退化している場合は
                fittingfn()が空のモデルを返すことに依存する必要があります。

    s         - モデルをフィットさせるためにfittingfnが必要とするxからのサンプルの最小数。

    t         - データポイントとモデル間の距離の閾値
                ポイントがインライヤーであるかどうかを決定するために使用されます。

    feedback  - オプションのブールフラグ。trueに設定すると、試行回数と
                必要な試行の推定総数が各ステップで出力されます。
                デフォルトはfalseです。

maxDataTrials - 非退化データセットを選択するための最大試行回数。
                このパラメータはオプションで、デフォルトは100です。

    maxTrials - 最大反復回数。このパラメータはオプションで
                デフォルトは1000です。

    p         - アウトライヤーのないサンプルを少なくとも1つ選択する
                望ましい確率、デフォルトは0.99です。

Returns:
    M         - 最も多くのインライヤーを持つモデル。
    inliers   - 最良のモデルのインライヤーであったxの要素のインデックスの配列。
```

この関数の使用例については、ransacfithomography()またはransacfitplane()を参照してください。
