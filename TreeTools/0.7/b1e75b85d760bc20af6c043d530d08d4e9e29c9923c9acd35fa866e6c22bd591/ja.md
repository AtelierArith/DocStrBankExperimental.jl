```
root!(tree; method=:midpoint, topological = false)
```

ルートツリーを `method` を使用して。実装されているメソッドは `:midpoint` のみです。

# メソッド

## `:midpoint`

ノード間の距離は、トポロジカル（枝の数）または枝の長さに基づくことができます。ルートでポリトミーを作成しません：ミッドポイントが既存の内部ノード（ルートではない）である場合、それのすぐ下に新しいルートノードを作成します。

ミッドポイントルーティングは、以下の場合に何もせずに終了します。

  * 最も遠い2つの葉の間の距離が `0` の場合。これは、すべての枝の長さが 0 の場合に発生します。
  * 現在のルートがすでにミッドポイントである場合。

## `:model`

キーワード引数 `model::Tree` を提供します。`model` のように `tree` をルートしようとします。2つのツリーがルーティングによってのみ異なる場合、最終的には同じトポロジーになります。それ以外の場合、ツリーは再ルーティングされますが、警告が表示されます。
