非線形かつ非凸の軌道最適化問題を混合補完問題（MCP）としてキャストし、PATHを介して解決するソルババックエンド。

MCPは問題のKKT条件から導出され、次の形式を取ります。

find   z s.t.   lᵢ == zᵢ       Fᵢ(z) >= 0        lᵢ <  zᵢ <  u, Fᵢ(z) == 0              zᵢ == u, Fᵢ(z) <= 0

# 注意

PATHソルバはオープンソースではありませんが、無料ライセンスを提供しています。ライセンスキーを設定しない場合、このバックエンドは小さな問題にのみ対応します。ライセンスキーの読み込みについては、[PATHSolver.jl](https://github.com/chkwon/PATHSolver.jl)のドキュメントを参照してください。
