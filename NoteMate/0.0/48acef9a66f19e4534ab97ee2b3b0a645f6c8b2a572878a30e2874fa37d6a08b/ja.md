```
get_sections
```

全体の解析されたMarkdownドキュメントをヘッダーの間のテキストのブロックに分けます。

# 引数

  * `contents`: Julia標準ライブラリ`Markdown.parse`からのMD構造体のコンテンツ構造体の`Vector`
  * `headers`: 提出されたドキュメントで見つかった`Header`構造体のベクター

# キーワード引数

  * `name_sections = true`: 辞書を取得するためのブール設定、ベクターの代わりに。

# 戻り値

  * デフォルトでは、ヘッダーの最初の単語をキーとし、ヘッダーの間のセクションを持つ辞書
  * ドキュメントのヘッダーの間のすべてのセクションテキストのベクター
