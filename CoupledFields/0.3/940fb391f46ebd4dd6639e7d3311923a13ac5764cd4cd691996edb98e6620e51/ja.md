```
gradient(Z::Array; axs::Tuple=axes(Z), smoothness=1.0)
```

フィールド `Z` の勾配を計算します。 `axs` を使用して各次元のポイント範囲のタプルを指定します。

このメソッドはガウスカーネルの勾配関数に基づいています： `smoothness` はガウスのσの自動推定をスケーリングし、大きな値は勾配関数を線形にします。

`Z` は任意の次元を持つことができますが、 `length(Z)>10³` の場合、メソッドは遅くなる可能性があります。

```
gradient(X::Matrix, Y::Matrix; smoothness=1.0)
```

$$
∇g
$$

を計算します。ここで、 `Y = g(X)` であり、 `X` は位置値（行 = ポイント）、 `Y` はフィールド値（例：1列の行列）です。ポイントは不規則な位置を持つことがあります。 `size(X, 1)>10³` の場合、メソッドは遅くなる可能性があります。
