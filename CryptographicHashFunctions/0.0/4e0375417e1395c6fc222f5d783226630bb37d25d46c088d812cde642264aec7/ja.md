```
digest!(obj)
```

HMACオブジェクトまたはハッシュアルゴリズムのハッシュコンテキスト`obj`のダイジェストを返します。

`reset!`が呼び出される前でない限り、`update!`または`digest!`へのさらなる呼び出しは許可されません。

```
digest!(ctx, len)
```

XOFアルゴリズムのハッシュコンテキスト`ctx`のダイジェストの次の`len`バイトを返します。

`reset!`が呼び出される前でない限り、`update!`へのさらなる呼び出しは許可されません。プロバイダーがストリーミングをサポートしていない場合、`digest!`へのさらなる呼び出しも禁止されます。
