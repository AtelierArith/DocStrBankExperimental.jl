```
nodes_in_area(a::Area, ℒ::Vector{<:Link}; n_nodes=1000)
```

エリア `a` に接続されたすべてのノードのベクターを返します。接続はリンク `ℒ` を通じて行われます。このアプローチは幅優先探索に基づいており、関数 [getnodesinarea](@ref) とは異なり、すべての接続されたノードを提供します。

# 引数

  * **`a::Area`** は評価されるべきエリアです。
  * **`ℒ::Vector{<:Link}`** は評価されるべきリンクのベクターです。

# キーワード引数

  * **`n_nodes=1000`** ループが終了するまでに追加されるノードの数です。これはエリア `a` のノードの数以上でなければなりません。
