```
reduce_to_wignerseitz(v::StaticVector, Vs::BasisLike)  -->  v′
```

基底 `Vs` における点 `v` の周期的な画像 `v′` を返します。

`v` は格子基底（すなわち `Vs` に対して相対的）で提供されると仮定され、`v′` は同様の方法で返されます。

返される点 `v′` は `Vs` によって定義されたウィグナー-ザイツセル（またはその境界）内にあり、`v` のすべての同等な画像の中で最も小さいノルムを持ち、`mod(v, 1) ≈ mod(v′, 1)` となるように整数格子の平行移動によって `v` から異なることが最大である。
