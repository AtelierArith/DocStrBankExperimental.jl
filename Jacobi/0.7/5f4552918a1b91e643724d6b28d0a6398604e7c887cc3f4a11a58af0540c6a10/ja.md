```
jacobi_zeros(m, alpha, beta, x)
```

ジャコビ多項式のゼロを計算します

この関数はジャコビ多項式のゼロを計算します：

  * :P_m

    ^{a,b}(x) = 0 $

`jacobi_zeros!`は修正バージョンで、ゼロが保存されるメモリが事前に割り当てられています。非修正バージョンの`jacobi_zeros`はメモリを割り当てて修正バージョンを呼び出します。関数`legendre_zeros`はレジェンドル多項式のゼロを計算します（`a = b = 0`）

パラメータ：

  * `m` 多項式の次数
  * `alpha` ジャコビ多項式の重みα
  * `beta` ジャコビ多項式の重みβ
  * `x` ゼロを保存する場所
  * `T` ゼロを計算するための浮動小数点型
  * 戻り値：ゼロを含む配列。

# 例

```julia-repl
z = jacobi_zeros(7, 0.3, 0.2)
```
