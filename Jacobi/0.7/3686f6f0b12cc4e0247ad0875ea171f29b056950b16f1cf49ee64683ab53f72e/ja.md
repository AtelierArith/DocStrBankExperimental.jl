```
jacobi(x, n [, a [, b]])
djacobi(x, n [, a [, b]])
```

ジャコビ多項式とジャコビ多項式の導関数を計算します。

この関数は、点 x での重み a と b を持つ次数 n のジャコビ多項式を計算します ($P_n^{a,b}(x)$)。デフォルト値を持つ関数のいくつかのバリアントがあります：

  * `jacobi(x, n)` （`a` と `b` は両方ともゼロ）
  * `jacobi(x, n, a)` （`b` はゼロ）

ジャコビ多項式の導関数 ($\frac{dP_n^{a,b}(x)}{dx}$) は、その名前の前に `d` を追加する関数で計算されます：

  * `djacobi`

パラメータ：

  * `x` 多項式を計算する場所
  * `n` 多項式の次数
  * `a` ジャコビ多項式の重み α
  * `b` ジャコビ多項式の重み β
  * 戻り値：多項式の値

### 例

```julia
julia> x = 0.3; a = 0.2; b = 0.1; m = 5;

julia> y = jacobi(x, m, a, b)
0.3780304524166406

julia> dy = djacobi(x, m, a, b)
-0.4343764194492186
```
