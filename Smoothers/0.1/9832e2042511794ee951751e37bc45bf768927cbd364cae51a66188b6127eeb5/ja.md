パッケージ: Forecast

```
hma(x, n)
```

データセット `x` に対して `n` 項のヘンダーソン移動平均フィルターを適用します。

「ヘンダーソン移動平均は、1916年にロバート・ヘンダーソンによって保険数理の応用のために導出されたフィルターです。これらはトレンドフィルターであり、季節調整された推定値を平滑化してトレンド推定を生成するために、時系列分析で一般的に使用されます。

これらは、最大3次の多項式を再現できるため、単純な移動平均よりも好まれて使用され、トレンドの転換点を捉えることができます。

ABS（オーストラリア統計局）は、季節調整された系列からトレンド推定を生成するためにヘンダーソン移動平均を使用しています。ABSが公表するトレンド推定は、通常、月次系列には13項のヘンダーソンフィルターを、四半期系列には7項のヘンダーソンフィルターを使用して導出されます。

ヘンダーソンフィルターは、対称または非対称のいずれかです。対称移動平均は、時系列の端から十分に離れた点に適用できます。この場合、時系列の特定の点の平滑化された値は、データポイントの両側にある同数の値から計算されます。」 - オーストラリア統計局 (www.abs.gov.au)

# 引数

  * `x`: 観測値のサポート。
  * `n`: 観測値、通常は四半期データの場合は13または7。

# 戻り値

`s` に提供されるヘンダーソンフィルターで平滑化された値の配列。

# 例

```julia-repl
julia> hma(rand(1000), 303)
1000-element Vector{Float64}}:
[...]
```
