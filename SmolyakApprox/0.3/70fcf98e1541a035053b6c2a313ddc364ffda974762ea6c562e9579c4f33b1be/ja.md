`N` 個の均等に間隔を空けた点を [1.0,-1.0] 上で計算し、`domain` で指定された区間にスケーリングします（デフォルトは [1.0,-1.0]）。指定されたドメイン上に位置する `N` 点を含むベクトルを返します。

クレンショー・カーティスの等間隔点は、スモリャク近似が区分線形基底関数に基づいている場合に使用されます。

# シグネチャ

points = clenshaw*curtis*equidistant(N,domain)

# 例

```
julia> points = clenshaw_curtis_equidistant(4)
[-1.0
 -0.33333333333333337
  0.3333333333333335
  1.0]

julia> points = clenshaw_curtis_equidistant(4,[3.0,-1.0])
[-1.0
  0.33333333333333326
  1.666666666666667
  3.0]
```
