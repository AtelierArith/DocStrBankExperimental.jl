```
spgbox(f, g!, x::AbstractVecOrMat; lower=..., upper=..., options...)`
```

追加のヘルプについては `spgbox!` を参照してください。

関数 `f` を初期点 `x` から最小化し、勾配を計算する関数 `g!` を与えます。 `f` は `f(x)` の形式で、`g!` は `g!(g,x)` の形式でなければなりません。ここで `g` は修正される勾配ベクトルです。

オプションの下限および上限のボックス境界は、オプションのキーワード引数 `lower` と `upper` を使用して提供できます。

```
spgbox(fg!, x::AbstractVecOrMat; lower=..., upper=..., options...)`
```

勾配ベクトル `g` を更新し、関数値を返す単一の関数 `fg!(g,x)` を与えることで、関数を最小化します。

これらの関数は、`x` に見つかった最良の解と最終目的関数 `f` を含む `SPGBoxResult` 型の構造体を返します。

これらの関数は `x` ベクトルを変更せず、代わりにそれの（深い）コピーを作成します（インプレースの代替手段については `spgbox!` を参照してください）。
