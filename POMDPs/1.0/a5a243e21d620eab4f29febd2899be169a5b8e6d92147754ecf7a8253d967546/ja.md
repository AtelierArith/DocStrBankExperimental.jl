```
history(b)
```

信念 `b` に関連する行動-観察履歴を返します。

履歴は `AbstractVector`、`Tuple`（または `end` でインデックス付けをサポートする類似のオブジェクト）で、`:a` と `:o` のキーを持つ `NamedTuples` で満たされている必要があります。つまり、`history(b)[end][:a]` は `b` に至るまでに行われた最後の行動であり、`history(b)[end][:o]` は受け取った最後の観察であるべきです。

利用可能な履歴の一部だけを返すことは許容されますが、常に現在の観察で終わるべきです。例えば、長さ 3 の `Vector{NamedTuple{(:o,),Tuple{O}}` に含まれる最後の 3 つの観察のみを含む構造を返すことは許容されます。
