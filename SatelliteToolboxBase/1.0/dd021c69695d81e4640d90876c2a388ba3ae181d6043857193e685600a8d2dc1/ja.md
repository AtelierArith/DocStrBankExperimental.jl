```
jd_to_date([T,] JD::Number)
```

ジュリア日 `JD` をグレゴリオ暦に変換します。オプションのパラメータ `T` は返される型を定義します。

# 戻り値

`T` が省略されるか `Int` の場合、次のデータを含むタプルが返されます：

  * 年。
  * 月（`1` => **1月**, `2` => **2月**, ...）。
  * 日。
  * 時間（0 - 24）。
  * 分（0 - 59）。
  * 秒（0 - 59）。

`T` が `Int` の場合、秒のフィールドは `Int` に丸められます。それ以外の場合は浮動小数点になります。

`T` が `Date` の場合、ジュリア構造体 `Date` が返されます。構造体 `Date` は時間、分、秒をサポートしていないため、これらは無視されます。

`T` が `DateTime` の場合、ジュリア構造体 `DateTime` が返されます。

# 備考

!!! note
    このメソッドは、1582年以前の年にグレゴリオ暦のうるう年規則を使用してグレゴリオ暦を遡って拡張することによって得られるグレゴリオ暦の前向きカレンダーで正確な日付を提供しません。特に、Y < 400 の場合、このメソッドは失敗します。
