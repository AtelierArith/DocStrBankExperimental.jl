```
ConstantForcing(rtol)
```

常に値 `rtol` を返す `ForcingTerm` であり、`rtol` は区間 `[0, 1)` にある必要があります。`x` と `f!` が特定の仮定を満たす場合、この強制項はニュートン-クリロフ法が線形収束し、漸近的な収束率が最大で `rtol` であることを保証します。`rtol` が 0（または `eps(FT)`）の場合、これは `Δx[n]` の近似が正確（または機械精度内で正確）であることを強制し、ニュートン-クリロフ法が二次収束することを保証します。より小さい値の `rtol` はより速い漸近的収束を保証しますが、過剰解決の確率も高くなります。
