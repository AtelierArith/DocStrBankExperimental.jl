```
apply(f::Function, [T], p::PointCloud, attrs...; kws...)
```

関数 `f` を点群 `p` の各点の属性 `attrs` に適用します。属性名は `Symbol` として渡され、関数は各属性名に対して1つの引数を取る必要があります。

出力の要素型は関数と引数の型から自動的に決定されます。また、自動的に決定された型が狭すぎるまたは広すぎる場合は、`T` 引数を使用して設定することもできます。

# キーワード

  * `neighbors`: `true` または整数 `k` に設定されている場合、点の隣接点の属性に関数を適用します。この場合、関数 `f` は隣接点の属性値を持つ `AbstractVector` を受け取ります。`k` 最近接隣接点のインデックスは、利用可能な場合は `p.neighbors` から読み取られます（[`neighbors`](@ref)/[`neighbors!`](@ref) を参照）。`p.neighbors` が利用できない場合や `k` インデックス未満の要素を含む場合、隣接点の検索が（`p.neighbors` を更新せずに）再実行されます。デフォルト: `false`
