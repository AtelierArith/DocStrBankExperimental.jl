```
gettiles([src::DataSource]; lat, lon, crs = "EPSG:4326", limit = 100)
```

ポイントクラウドデータのタイルを取得するために、利用可能なデータソースまたは特定のソース `src` にクエリを実行します。これにより、データベースのエントリのみを含む `PointCloudTile` の配列が返されます。これらの値は、ポイントデータ自体をダウンロードして読み込むために `LAS` コンストラクタに渡すことができます。また、各タイルがカバーする座標範囲を取得するために `minimum` / `maximum` / `extent` に渡すこともできます。

`lon`（または同等の `x`）および `lat`（または `y`）キーワード引数は、[WGS 84](https://en.wikipedia.org/wiki/World_Geodetic_System) 座標参照系（CRS）で指定された空間クエリのための緯度と経度の値を指定します。これはGPSや多くのオンラインマップでも使用されています。クエリは、1つ、2つ、または複数の座標値が提供されるかに応じて、ポイント、ボックス、またはポリゴン内のタイルを検索します。座標は、`crs` 引数を指定することで異なるCRSでも指定できます。

`limit` 引数は、各データソースから要求されるタイルの最大数を指定します。クエリされるAPIには、超えることができない独自の制限がある場合があります（ScienceBaseの場合は1000）。それを超える場合はページングが必要であり（つまり、オフセットを使用してより多くの結果を要求すること）、現在 `gettiles` ではサポートされていません。
