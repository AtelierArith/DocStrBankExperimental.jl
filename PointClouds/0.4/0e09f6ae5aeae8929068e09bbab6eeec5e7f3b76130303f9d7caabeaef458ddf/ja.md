```
rasterize(p::PointCloud, dims; kws...)
```

`PointCloud`内のポイントをターゲットエリア内のグリッド化された位置に割り当てることによってラスタライズします。ラスタは`dims`（整数のタプルとして与えられる）という次元を持ち、x-/y座標の軸に整列し、ポイントの完全なバウンディングボックスをカバーします。ただし、キーワード引数で範囲が狭められた場合は除きます。デフォルトでは、各ラスタ位置はその「ピクセルエリア」内にあるポイントにマッピングされますが、`radius`または`neighbors`のキーワード引数が指定されている場合は別です。現在、2Dラスタライズのみがサポートされています。

デフォルトの「ピクセルエリア」メソッドでは、各ポイントはラスタの境界内にある場合、正確に1つのラスタ位置に割り当てられます。`radius`および`neighbors`メソッドでは、ポイントが複数またはゼロのラスタ位置に割り当てられる可能性があります。さらに、`neighbors`メソッドのみが各ラスタ位置にポイントが割り当てられることを保証します。他のメソッドでは空のピクセルが生成される可能性があります。

# キーワード

  * `extent`: 座標の下限と上限を持つタプルとしてラスタエリアの範囲を指定します。各エントリはx値とy値のタプルです。例えば、`extent = ((0, 0), (1, 2))`は$0 ≤ x ≤ 1$および$0 ≤ y ≤ 2$を表します。
  * `radius`: ピクセル中心から`radius`の距離内にあるすべてのポイントをそのラスタ位置に割り当てます。`radius`の単位はx-/y座標と同じです。ポイントが複数またはゼロのラスタ位置に割り当てられる可能性があることに注意してください。
  * `neighbors`: ピクセル中心に最も近いポイントに基づいて、各ラスタ位置に特定の数`neighbors`のポイントを割り当てます。
