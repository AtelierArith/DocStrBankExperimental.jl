```
resistance_distance_matrix(g)
```

グラフの抵抗距離行列を計算します。

# 引数

  * `g`: 抵抗距離行列を計算するグラフ。

# 戻り値

  * i 行 j 列の要素がノード `i` と `j` の間の抵抗距離を表す行列。

# 注意事項

  * 抵抗距離行列は、グラフのラプラシアン行列に関する抵抗距離の公式を使用して計算されます。
  * グラフのラプラシアン行列は、`laplacian_matrix` 関数を使用して計算されます。
  * ラプラシアン行列の擬似逆行列は、LinearAlgebra モジュールの `pinv` 関数を使用して計算されます。
