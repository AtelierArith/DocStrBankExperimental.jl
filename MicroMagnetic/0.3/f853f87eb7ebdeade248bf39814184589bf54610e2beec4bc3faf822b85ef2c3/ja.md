```
add_exch(sim::AtomisticSim, J1::NumberOrArray; name="exch", J2=0, J3=0, J4=0)
```

原子シミュレーション `sim` に交換相互作用を追加します。

$$
\mathcal{H}_\mathrm{ex} = -J \sum_{\langle i, j\rangle} \mathbf{m}_{i} \cdot \mathbf{m}_{j}
$$

### パラメータ

  * `J1::NumberOrArray`: 最初の最近接隣接交換結合定数。単一の数値（均一に適用）または最初の最近接隣接の数に等しい長さの配列であることができます。
  * `J2, J3, J4`（オプション）: それぞれ第二、第三、第四の最近接隣接のための交換結合定数。各々は単一の数値または対応する隣接数に一致する長さの配列であることができます。
  * `name::String`: 交換相互作用のオプションの識別子（デフォルトは `"exch"`）。

### 使用例

最初の最近接隣接のために均一な交換結合を設定するには：

```julia
add_exch(sim, 0.1*meV)
```

複数の隣接シェルに対して異なる交換結合を設定するには：

```julia
Js4 = [0.1*meV, 0.2*meV, 0.3*meV, 0.1*meV, 0.2*meV, 0.3*meV]
add_exch(sim, 0.1*meV, J2=0.2meV, J3=0.0, J4=Js4)
```

#### 注意

J1、J2、J3、またはJ4に配列が提供される場合、その長さはシステムのメッシュで定義された対応する隣接の数と一致する必要があります。そうでない場合、エラーが発生します。
