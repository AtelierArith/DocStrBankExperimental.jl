```
FDMesh(; dx=1e-9, dy=1e-9, dz=1e-9, nx=1, ny=1, nz=1, pbc="open")
```

シミュレーションのための有限差分メッシュ（`FDMesh`）を作成します。このメソッドは、[マイクロ磁気モデル](@ref)用に設計されています。

このコンストラクタは、指定されたパラメータで有限差分メッシュを初期化します。`pbc`（周期境界条件）パラメータは、メッシュが対応する軸をラップするかどうかを決定する文字 `"x"`、`"y"`、および `"z"` の任意の組み合わせにすることができます。

**パラメータ:**

  * `dx::Float64`: x方向のグリッド間隔（デフォルト: `1e-9`）。
  * `dy::Float64`: y方向のグリッド間隔（デフォルト: `1e-9`）。
  * `dz::Float64`: z方向のグリッド間隔（デフォルト: `1e-9`）。
  * `nx::Int`: x方向のグリッドポイント数（デフォルト: `1`）。
  * `ny::Int`: y方向のグリッドポイント数（デフォルト: `1`）。
  * `nz::Int`: z方向のグリッドポイント数（デフォルト: `1`）。
  * `pbc::String`: 周期境界条件で、`"x"`、`"y"`、および `"z"` の任意の組み合わせ（デフォルト: `"open"`）。

**例:**

```julia
mesh = FDMesh(dx=1e-8, dy=1e-8, dz=1e-8, nx=10, ny=10, nz=10, pbc="xyz")
```

これは、各方向に10のグリッドポイントを持ち、すべての3つの軸に周期境界条件を持つFDMeshを作成します。
