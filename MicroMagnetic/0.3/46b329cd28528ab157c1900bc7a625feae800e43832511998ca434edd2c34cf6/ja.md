```
read_table(filename::String)
```

指定されたファイル `filename` からデータテーブルを読み込みます。ファイルには `#` で始まるコメント行が含まれている必要があり、最初のコメント行には物理量の名前が含まれ、2番目のコメント行には単位が含まれます。データは `start_idx` 行から始まり、これは実際のデータの最初の行です。

### パラメータ

  * `filename::String` : データファイルへのパス。

### 戻り値

  * 2つの辞書:

      * `data` : キーが物理量の名前で、値が対応する浮動小数点データのベクトルである辞書。
      * `units` : キーが物理量の名前で、値が対応する単位文字列である辞書。

### 例

```julia data, units = read_table("data.txt")

```
