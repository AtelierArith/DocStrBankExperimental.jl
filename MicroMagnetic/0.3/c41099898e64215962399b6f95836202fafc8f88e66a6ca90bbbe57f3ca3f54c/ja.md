```
CylindricalTubeMesh(;dz=1e-9, R=20e-9, nz=1, nr=10, pbc="open")
```

+z方向に沿った円筒チューブメッシュを作成します。

スピンは円筒チューブ上に均等に配置され、次のようにインデックス付けされています：

```julia
  id = index(i, 1, k, nr, 1, nz)
```

これは、スピンが最初にリングでラベル付けされ、その後`nz`がリングの数であることを意味します。各リングのスピンの座標は次のように与えられます：(R cos(2*pi*(i-1)/nr), R sin(2*pi*(nr-1)/nr))。

最近接の隣接点は次のようにインデックス付けされています：

|  1      2        3       4            |左   右    下   上 |
