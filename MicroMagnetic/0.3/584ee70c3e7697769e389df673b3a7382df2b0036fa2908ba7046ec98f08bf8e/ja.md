```
TriangularMesh(; dx=1e-9, nx=1, ny=1, nz=1, pbc="open")
```

三角メッシュを作成します。最近接隣接点と次最近接隣接点のインデックスは以下の通りです：

| 最近接インデックス | 位置  | 次最近接インデックス | 位置  |
|:---------:|:---:|:----------:|:---:|
|     1     |  右  |     1      | 右上  |
|     2     | 右上  |     2      |  上  |
|     3     | 左上  |     3      | 左上  |
|     4     |  左  |     4      | 左下  |
|     5     | 左下  |     5      |  下  |
|     6     | 右下  |     6      | 右下  |
|     7     |  上  |            |     |
|     8     |  下  |            |     |
