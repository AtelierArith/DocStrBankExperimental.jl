```
instance(md, i)
```

マルチモーダルデータセットの `i` 番目のインスタンスを返します。

```
instance(md, i_modality, i_instance)
```

`i_modality` 番目のモダリティからの変数のみを持つマルチモーダルデータセットの `i_instance` 番目のインスタンスを返します。

```
instance(md, i_instances)
```

マルチモーダルデータセットの `i_instances` にあるインスタンスを返します。

```
instance(md, i_modality, i_instances)
```

`i_modality` 番目のモダリティからの変数のみを持つマルチモーダルデータセットの `i*instances` にあるインスタンスを返します。
