```
RBiQuaternion{T<:Real} <: Number
```

Reduced BiQuaternion number type with real and imaginary parts of type `T`.

`RBiQuaternionF16`, `RBiQuaternionF32`, and `RBiQuaternionF64` are aliases for `RBiQuaternion{Float16}`, `RBiQuaternion{Float32}`, and `RBiQuaternion{Float64}`, respectively.

See also: [`rbiquat`](@ref), [`real`](@ref), [`imag`](@ref).
