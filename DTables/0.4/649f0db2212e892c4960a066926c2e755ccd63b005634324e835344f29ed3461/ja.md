```
leftjoin(d1::DTable, d2; on=nothing, l_sorted=false, r_sorted=false, r_unique=false, lookup=nothing)
```

`d1`と任意の`Tables.jl`互換テーブルタイプとの左結合を実行します。結果を持つ`DTable`を返します。

基になるテーブルタイプが`leftjoin`の実装を持ち、以下の`DTable`関連のキーワード引数が提供されない場合、特化した関数が使用されます。その良い例は、基になるタイプが`DataFrame`の`DTable`で`DataFrame`タイプの`d2`を呼び出すことです。

# キーワード引数

  * `on`: 結合する列のシンボル。列名が異なる場合はシンボルのペアとして提供できます。複数の列での結合の場合は、前述のシンボルのベクターを提供できます。
  * `l_sorted`: 左のテーブルがソートされていることを示します - `r_sorted`も`true`に設定されている場合にのみ有用です。
  * `r_sorted`: 右のテーブルがソートされていることを示します。
  * `r_unique`: 右のテーブルが一意のキーのみを含むことを示します。
  * `lookup`: 内部行の直接マッチングを可能にする辞書のような構造を提供します。この構造は、`Tuple`の形式のキーと、関連する行インデックスを含む`Vector{UInt}`の形式の値を含む必要があります。
