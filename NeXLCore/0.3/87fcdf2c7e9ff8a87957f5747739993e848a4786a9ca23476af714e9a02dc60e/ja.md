```
coated_particle(mat::Material, radius::Float64, coating::Material, thickness::Float64, substrate::Union{Nothing,Material} = nothing)
```

オプションの基板上にコーティングされた粒子を構築します。

このモデルは、より複雑なモデルがどのように構築されるかの良い例です。 `dump(..)`を使用して構造を表示します。 チャンバーがルート`Region`として機能することに気付くでしょう。 `coating`と`substrate`はチャンバー内にあります（チャンバー`Region`の子）。 `particle`は`coating` `Region`の子であり、粒子はコーティングによって完全に囲まれています。 コーティング内の電子は、粒子を表す体積に入るとすぐに粒子に入り、コーティングを離れます。 電子は、空間の任意の点で子最下層の領域内にあるように見えます。 したがって、典型的な軌道は、チャンバーから始まり、コーティングに入り、コーティングの厚さを横断し、次に粒子に入るかもしれません。 最終的には粒子を離れ、コーティングに再入し、コーティングを出てチャンバーに再入し、最終的に基板に静止することになります。
