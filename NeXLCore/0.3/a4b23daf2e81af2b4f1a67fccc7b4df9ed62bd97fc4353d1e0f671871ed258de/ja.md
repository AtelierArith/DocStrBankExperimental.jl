平均イオン化ポテンシャルを実装するアルゴリズム。平均イオン化ポテンシャルは、物質中の電子エネルギー損失の連続減速モデルにおける主要なパラメータです。電子は主に2つのメカニズムを通じてエネルギーを失います - 1) Jでパラメータ化された衝突停止力、すなわち平均イオン化ポテンシャルによるもの; そして2) ブレムストラールング生成。衝突損失に対しては2桁または3桁多くのエネルギーが失われるため、衝突損失がプロセスを支配し、ブレムストラールング生成による損失は衝突損失の不確実性に対して無視できるほど小さいです。

これを実装します：

```
J(::Type{<:NeXLMeanIonizationPotential}, z)  # eV単位
```

また、次のものも提供されています：

```
J(::Type{<:NeXLMeanIonizationPotential}, elm::Element)  # eV単位
```
