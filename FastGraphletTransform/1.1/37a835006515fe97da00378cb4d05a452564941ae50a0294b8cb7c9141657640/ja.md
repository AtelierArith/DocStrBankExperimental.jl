```
fglt(A::SparseMatrixCSC{F, I})
```

与えられた隣接行列 `A` で記述されたグラフに対して高速グラフレット変換 (FGLT) を実行し、生の頻度行列 (`f`) とネット頻度行列 (`fnet`) のタプルを返します。これらの行列はサイズ `n` x 16 で、`n` はグラフのノード数です。

計算された頻度は、以下のグラフレットに対応します：

| sigma | 説明                |
|:-----:|:----------------- |
|   0   | シングルトン            |
|   1   | 1-パス、端に           |
|   2   | 2-パス、端に           |
|   3   | バイフォーク、ルートに       |
|   4   | 3-クリーク、任意のノードに    |
|   5   | 3-パス、端に           |
|   6   | 3-パス、内部ノードに       |
|   7   | クロー、葉に            |
|   8   | クロー、ルートに          |
|   9   | ディッパー、ハンドル先端に     |
|  10   | ディッパー、基底ノードに      |
|  11   | ディッパー、中心に         |
|  12   | 4-サイクル、任意のノードに    |
|  13   | ダイヤモンド、オフコードノードに  |
|  14   | ダイヤモンド、オンコードノード14 |
|  15   | 4-クリーク、任意のノードに    |
