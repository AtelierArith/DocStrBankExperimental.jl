```
distancetobase(::Type{SPM}, N::SpeciesInteractionNetwork{<:Unipartite{T}, <:Interactions}, sp::T, f) where {T, SPM <: ShortestPathMethod}
```

食物網における基底種からの種 `sp` の距離を測定します。基底種は、[`generaliry`](@ref) が 0 であると定義されます。

*例えば* [Thompson2012Food](@citet) に従い、基底種には基底までの距離を 1 と割り当てます。一次消費者は距離 2 を持ち、などとなります。

[Post2002long](@citet) は、栄養レベルは生産者への最大、平均、または最小距離から得られることを指摘しています。消費者が複数の生産者に接続されている可能性があるため、これらの接続のモードまたは中央値が関連するかもしれないと主張することができます。この理由から、関数 `f` は距離の配列を受け取り、スカラーを返します。

最小を支持する立場からは、ほとんどのエネルギー移動は短いチェーンに沿って行われるべきだと主張できます。しかし、長さ 5 のチェーンの頂点にいる消費者が生産者に直接接続されている場合、最小値はその消費者に栄養レベル 2 を与え、食物網の頂点にいる位置を隠してしまいます。

最大を支持する立場からは、より高いチェーンが食物網の底から来るエネルギーがどれだけ遠くまで行けるかの良い指標を提供すると主張できます。これは、どれだけ*垂直的に*多様であるかの強い指標です [Duffy2007functional](@cite)。
