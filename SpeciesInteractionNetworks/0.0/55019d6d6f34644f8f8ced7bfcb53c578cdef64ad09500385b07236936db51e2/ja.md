```
Bipartite{T <: Any} <: Partiteness{T}
```

二部グラフの種の集合は、`top` と `bottom` と呼ばれる二つの種の集合で表されます。両方の種の集合は `Vector{T}` として表され、いくつかの特定の制約があります：

1. `T` は `Number` であってはならない（*すなわち* ノードは名前を持っているか、他のオブジェクトでなければなりません）
2. `top` のすべての種は一意でなければなりません
3. `bottom` のすべての種は一意でなければなりません
4. どの種も `bottom` と `top` の両方に存在してはなりません
