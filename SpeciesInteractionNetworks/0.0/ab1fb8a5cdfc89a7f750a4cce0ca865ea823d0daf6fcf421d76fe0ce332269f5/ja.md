```
modularity(N::SpeciesInteractionNetwork{<:Partiteness[T], <:Binary}, L::Dict{T,Int}) where {T}
```

ネットワークとその分割のモジュラリティ。第二引数は、`N`のすべての種がモジュールのアイデンティティを表す整数値に関連付けられた辞書です。この関数は、二部ネットワークとその単一部投影に対して同じ値を返します。この測定は通常$Q$と呼ばれます。

[Newman2006Modularity](@citet)では、ヌルモデルは$2m$で修正されます。ここで$m$はネットワーク内のリンクの数です。これは*無向*相互作用に適しており、したがってコードでは$m$でのみ割ります。このことは、モジュラリティ行列が正しい特性を持つことを保証します。*すなわち*、その行と列の合計は（おおよそ）0になります。
