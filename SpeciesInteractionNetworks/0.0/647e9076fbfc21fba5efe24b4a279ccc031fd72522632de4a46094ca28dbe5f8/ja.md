```
centrality(::Type{KatzCentrality}, N::SpeciesInteractionNetwork{<:Unipartite, <:Union{Binary, Probabilistic}}; α::AbstractFloat=0.1)
```

この指標は、各接続に異なる重みを与えます（`α`）。`α`は重みであり、特にノードからの各移動の減衰を示すため、正でなければなりません。

この関数は内部的に線形代数のショートカットを使用して（パスの長さの合計ではなく）中心性を計算するため、より高速です。その結果、各ネットワークには、隣接行列の主固有値の絶対値の逆数である最大の`α`の値があります。この`α`は、隣接行列に複素固有値がある場合、解析的な解を持ちません。大きな値の`α`を使用する場合は、この中心性アルゴリズムを`try`/`catch`ブロックの一部として使用することをお勧めします。
