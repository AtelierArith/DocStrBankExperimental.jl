```
PermutationConstraint
```

`PermutationConstraint`は、どの構造的制約が強制されるかを指定します。これは抽象型として定義されており、サブタイプは`swap!`の第二引数として渡すことができます。

現在サポートされている制約は、`Degree`（次数分布が維持される）、`Generality`（外向きリンクの数が維持される）、`Vulnerability`（内向きリンクの数が維持される）、および`Connectance`（接続性のみが維持される）です。*さらに*、種は次数/次数分布に対して制約が作用していなくても、切断されることはできないことに注意してください。
