```
interactions(N::SpeciesInteractionNetwork)
```

ネットワーク内のすべての相互作用のベクターを返します。

各相互作用は、ソース、デスティネーション、および値の3つの要素からなる名前のないタプルとして返されます。例えば、バイナリネットワークの場合、`:a` から `:b` への相互作用は `(:a,:b,true)` として表されます。返されるタプルの型は `eltype(N)` によって示され、ネットワークの反復を使用したときの出力と同じです。

このメソッドは、ネットワークの基礎となるデータ構造における非ゼロ相互作用のインデックス付けに関連する理由から、反復 (`int for int in N`) を使用するよりも大幅に高速でメモリ効率が良いことに注意してください。
