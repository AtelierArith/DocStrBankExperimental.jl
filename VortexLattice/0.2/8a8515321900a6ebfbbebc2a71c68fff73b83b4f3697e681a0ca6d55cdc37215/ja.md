```
lifting_line_coefficients(system, r, c; frame=Body())
```

幾何の揚力線表現の各スパン方向セグメントに対する力とモーメント係数（単位スパンあたり）を返します。

この関数は、パネル力を取得するために `system` に対して近接場解析が実行されている必要があります。

# 引数

  * `system`: 事前に計算されたシステムプロパティを保持する [`System`](@ref) 型のオブジェクト。
  * `r`: 表面の数と等しい長さのベクトルで、各要素は結果の揚力線座標の x, y, z 座標を含むサイズ (3, ns+1) の行列です。
  * `c`: 表面の数と等しい長さのベクトルで、各要素は各揚力線座標での弦長を含む長さ `ns+1` のベクトルです。

# キーワード引数

  * `frame`: `cf` と `cm` を返すフレーム、可能なオプションは [`Body()`](@ref)（デフォルト）、[`Stability()`](@ref)、および [`Wind()`](@ref) です。

# 返り値の引数:

  * `cf`: 表面の数と等しい長さのベクトルで、各要素は各スパン方向セグメントの x, y, z 方向の力係数（単位スパンあたり）を含むサイズ (3, ns) の行列です。
  * `cm`: 表面の数と等しい長さのベクトルで、各要素は各スパン方向セグメントの x, y, z 方向のモーメント係数（単位スパンあたり）を含むサイズ (3, ns) の行列です。
