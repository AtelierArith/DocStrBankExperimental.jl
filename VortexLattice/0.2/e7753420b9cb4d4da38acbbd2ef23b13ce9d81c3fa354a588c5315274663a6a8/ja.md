```
wing_to_grid(xle, yle, zle, chord, theta, phi, ns, nc;
    fc = fill(x -> 0, length(xle)),
    mirror = false,
    fcore = (c, Δs) -> 1e-3,
    spacing_s = Cosine(),
    spacing_c = Uniform(),
    interp_s = (x, y, xpt) -> FLOWMath.linear(x, y, xpt))
```

翼を `ns` スパン方向および `nc` 進行方向のパネルに離散化し、スパン方向の離散化スキーム `spacing_s` および進行方向の離散化スキーム `spacing_c` に従って関連する渦環を生成します。

パネルのコーナーを含む (3, i, j) の次元を持つグリッドを返し、表面パネルに変換する際に制御点を配置するための比率のベクトルを返します。

# 引数

  * `xle`: 各翼型セクションの前縁 x 座標
  * `yle`: 各翼型セクションの前縁 y 座標
  * `zle`: 各翼型セクションの前縁 z 座標
  * `chord`: 各翼型セクションの弦長
  * `theta`: 各翼型セクションのねじれ
  * `phi`: 各翼型セクションの二重角、x 軸を中心とした右手回転によって定義される
  * `ns`: スパン方向のパネル数
  * `nc`: 進行方向のパネル数
  * `fc`: (オプション) 各翼型セクションのキャンバー線関数 y=f(x)
  * 'reference_line': 2D 配列、各行は翼型の基準点の x, y 座標です。これにより、xle、yle、および zle を前縁ではない点に基づいて定義できます
  * `mirror`: X-Z 平面に対して幾何学をミラーリングしますか？デフォルトは `false`
  * `fcore`: 弦長 (x 方向) および/またはパネル幅 (y/z 方向) に基づいて有限コアサイズを設定するための関数。デフォルトは `(c, Δs) -> 1e-3`
  * `spacing_s`: スパン方向の離散化スキーム、デフォルトは `Cosine()`
  * `spacing_c`: 進行方向の離散化スキーム、デフォルトは `Uniform()`
  * `interp_s`: スパン方向のステーション間の補間関数、デフォルトは線形補間
  * 'invert_cambers': グリッドのキャンバーを反転しますか？デフォルトは `false`。ローター/タービンの生成に使用されます
