```
lifting_line_geometry(grids, xc=0.25)
```

`grids`内の表面の揚力線表現を、正規化された弦方向位置`xc`で構築します。揚力線の座標と弦長を返します。

# 引数

  * `grids`: 表面の数と等しい長さのベクトル。ベクトルの各要素は、パネルへの表面の離散化を定義する形状(3, nc, ns)のグリッドです。`nc`は弦方向のパネルの数、`ns`は翼幅方向のパネルの数です。
  * `xc`: 前縁からの揚力線の正規化された弦方向位置。デフォルトは四分の一弦です。

# 戻り値の引数:

  * `r`: 表面の数と等しい長さのベクトルで、各要素は(3, ns+1)のサイズを持つ行列で、結果の揚力線座標のx、y、z座標を含みます。
  * `c`: 表面の数と等しい長さのベクトルで、各要素は揚力線座標ごとの弦長を含む長さ`ns+1`のベクトルです。
