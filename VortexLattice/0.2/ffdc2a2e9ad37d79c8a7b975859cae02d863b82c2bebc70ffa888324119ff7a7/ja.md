```
steady_analysis(surfaces, reference, freestream; kwargs...)
```

定常渦格子法分析を実行します。システム状態を含む[`System`](@ref)型のオブジェクトを返します。

# 引数

  * `surfaces`:

      * 揚力面を表す形状 (3, nc+1, ns+1) のグリッドのベクター

    または

      * サーフェスパネルを含む形状 (nc, ns) の行列のベクター（[`SurfacePanel`](@ref）を参照）で、`nc`は弦方向パネルの数、`ns`は翼幅方向パネルの数です。
  * `reference`: 参照パラメータ（[`Reference`](@ref）を参照）
  * `freestream`: 自由流パラメータ（[`Freestream`](@ref）を参照）

# キーワード引数

  * `symmetric`: 各サーフェスに対して、誘導速度を計算する際にX-Z平面を横断する鏡像を使用するかどうかを示すフラグ。各サーフェスのデフォルトは`false`です。
  * `wakes`: 各サーフェスのためのウェイクパネルの行列（[`WakePanel`](@ref）を参照）。各行列の形状は(nw, ns)で、`nw`は弦方向ウェイクパネルの数、`ns`は各サーフェスの翼幅方向パネルの数で、各サーフェスのデフォルトはウェイクパネルなしです。
  * `nwake`: `wakes`の各ウェイクから使用する弦方向ウェイクパネルの数。各サーフェスのデフォルトはすべてのウェイクパネルです。
  * `surface_id`: 各サーフェスのサーフェスID。共有する同じIDのサーフェス/ウェイクの影響を計算する際に有限コアモデルが無効になります。デフォルトでは、すべてのサーフェスに独自のIDが割り当てられます。
  * `wake_finite_core`: 各ウェイクに対して、同じサーフェスIDを持つサーフェス/ウェイクに対するウェイクの影響を計算する際に有限コアモデルを有効にするかどうかを示すフラグ。各サーフェスのデフォルトは`true`です。
  * `trailing_vortices`: 各サーフェスのための後流渦を有効/無効にするフラグ。各サーフェスのデフォルトは`true`です。
  * `xhat`: 後流渦を発生させる方向。デフォルトは`[1, 0, 0]`です。
  * `additional_velocity`: 位置の関数として追加の速度を定義する関数。
  * `fcore`: 弦長および/またはパネル幅に基づいて各サーフェスの有限コアサイズを設定する関数。デフォルトは`(c, Δs) -> 1e-3`です。グリッド入力のみに使用されます。
  * `calculate_influence_matrix`: 空力的影響係数行列がすでに計算されているかどうかを示すフラグ。基礎となるジオメトリが変更されていない場合、同じAIC行列を再利用することで計算時間が短縮されます。デフォルトは`true`です。この引数は、この関数の事前割り当てバージョンにのみ有効です。
  * `near_field_analysis`: パネル速度、循環、および力を取得するために近接場分析を実行するかどうかを示すフラグ。デフォルトは`true`です。
  * `derivatives`: 自由流変数に関する導関数を計算するかどうかを示すフラグ。デフォルトは`true`です。
