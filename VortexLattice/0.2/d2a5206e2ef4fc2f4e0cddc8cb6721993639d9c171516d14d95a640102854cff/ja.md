```
grid_to_surface_panels(xyz, ns, nc;
    mirror = false,
    fcore = (c, Δs) -> 1e-3,
    spacing_s = Cosine(),
    spacing_c = Uniform(),
    interp_s = (x, y, xpt) -> FLOWMath.linear(x, y, xpt),
    interp_c = (x, y, xpt) -> FLOWMath.linear(x, y, xpt))
```

グリッドで定義された潜在的に曲がった揚力面を離散化します。グリッドの次元は (3, i, j) で、`i` は弦方向（前縁から後縁へ順に）を、`j` はスパン方向（左から右へ順に）を表します。これを `ns` のスパン方向パネルと `nc` の弦方向パネルに分割し、スパン方向の離散化スキーム `spacing_s` と弦方向の離散化スキーム `spacing_c` に従って関連する渦環を配置します。境界渦は各パネルの1/4弦に配置され、制御点は3/4弦に配置されます。

次元が (3, i, j) のグリッドを返し、パネルコーナーを含み、サーフェスパネルに変換する際の制御点を配置するための比率のベクトルと、生成されたパネルを含む次元 (i, j) の行列を返します。

# 引数

  * `xyz`: 次元が (3, i, j) のグリッドで、`i` は弦方向、`j` はスパン方向に対応します。
  * `ns`: スパン方向パネルの数
  * `nc`: 弦方向パネルの数

# キーワード引数

  * `mirror`: X-Z平面に沿ってジオメトリをミラーリングしますか？デフォルトは `false` です。
  * `fcore`: 弦長（x方向）および/またはパネル幅（y/z方向）に基づいて有限コアサイズを設定するための関数。デフォルトは `(c, Δs) -> 1e-3`
  * `spacing_s`: スパン方向の離散化スキーム、デフォルトは `Cosine()`
  * `spacing_c`: 弦方向の離散化スキーム、デフォルトは `Uniform()`
  * `interp_s`: スパン方向の補間関数、デフォルトは線形補間
  * `interp_c`: 弦方向の補間関数、デフォルトは線形補間
