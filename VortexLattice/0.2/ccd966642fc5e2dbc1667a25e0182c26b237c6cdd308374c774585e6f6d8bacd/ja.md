```
grid_to_surface_panels(xyz;
    ratios = zeros(2, size(xyz, 2)-1, size(xyz, 3)-1) .+ [0.5;0.75],
    mirror = false,
    fcore = (c, Δs) -> 1e-3)
```

潜在的に曲がった揚力面によって定義されたグリッド（3, i, j）の次元を持つパネルのセットを構築します。ここで、`i`は弦方向（先端から後端へ順に並んでいます）を表し、`j`はスパン方向（左から右へ順に並んでいます）を表します。各リング渦の先端は1/4弦に配置され、制御点は各パネルの3/4弦に配置されます。

次元が(3, i, j)のグリッドを返し、パネルのコーナーを含み、サーフェスパネルに変換する際に制御点を配置するための比率のベクトルと、生成されたパネルを含む次元が(i, j)の行列を返します。

# キーワード引数

  * `mirror`: X-Z平面に沿って幾何学をミラーリングしますか？デフォルトは`false`です。
  * `fcore`: 弦長（x方向）および/またはパネル幅（y/z方向）に基づいて有限コアサイズを設定するための関数です。デフォルトは`(c, Δs) -> 1e-3`です。
