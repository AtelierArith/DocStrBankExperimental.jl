```
struct ProjectedConstrainedNonSmoothProblem
```

制約が投影演算子を通じて課せられる制約付き非滑らか最適化問題を表します。この構造体は、`NonSmoothProblem` インターフェースにもう1つのメンバーを提供します：

  * `project`: 投影演算子。これは `dimension` の Float64 のベクトルを受け取り、`dimension` の Float64 のベクトルを生成します。返される点は、最適化問題の定義に従って実行可能でなければなりません。
