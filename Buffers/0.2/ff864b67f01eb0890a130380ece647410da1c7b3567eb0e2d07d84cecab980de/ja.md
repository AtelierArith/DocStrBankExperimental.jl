```
MAllocBuffer{T}
```

型 `T` のデータをオフセット付きで格納するバッファオブジェクトです。バッファメモリは手動で割り当ておよび解放され、拡張はできません。

バッファは、バッファが拡張可能かどうかを確認し、割り当てられた配列へのポインタがバッファと同じメモリを指さないようにするために、最初に余分な要素を割り当てます。

バッファが [`reshape_buf!`](@ref) と共に使用される場合、オフセットはゼロに設定されます。

!!! warning
    メモリは手動で割り当てられるため、バッファも手動で解放する必要があります。


!!! tip
    バッファは、使用後にバッファを解放する [`@buffer`](@ref) マクロと共に使用することを意図しています。


# 例

```julia
julia> @buffer buf(100) begin
         A = alloc!(buf, 10, 10)
         B = alloc!(buf, 10, 10)
         C = alloc!(buf, 10, 10)
         A .= 1.0
         B .= 2.0
         @tensor C[i,j] = A[i,l] * B[l,j]
       end
```
