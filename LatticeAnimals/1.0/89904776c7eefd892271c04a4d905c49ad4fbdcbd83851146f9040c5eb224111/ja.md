```
Poly(n, p, basis, neighbours; doPlot)
```

メトロポリスアルゴリズムを使用して、pの浸透分布からサイズnのランダムd次元ポリフォームを生成します。neighboursは、格子によって定義されたそれぞれの隣接格子点への座標の差のリストであり、生成中の接続性チェックのパフォーマンスを向上させるために時計回りにソートされている必要があります。デフォルトの混合時間は2*n^2です。

# 引数

```
* `n`: ポリフォームのサイズ
* `p`: [0, 1)の範囲の浸透因子
* `basis`: ポリフォームの格子基底
* `neighbours`: 現在のポリフォームのそれぞれの隣接タイルへの差のリスト
* `doPlot`: ポリフォームを視覚化する散布図を作成するかどうか（2dポリフォームのみ利用可能）
```
