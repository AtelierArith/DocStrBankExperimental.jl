### ToolipsSVG

This module brings an array of different Components, as well as making things work more thoroughly with paths.

###### exports

Path interface

  * `M!`
  * `L!`
  * `Z!`
  * `Q!`
  * `C!`
  * `A!`

Position, shape, and size.

  * `size(::Component{<:Any})`
  * `set_size!`
  * `get_position`
  * `set_position!`
  * `get_shape`
  * `set_shape`

Regular Components

  * `text`
  * `path`
  * `image`
  * `circle`
  * `rect`
  * `line`
  * `ellipse`
  * `polygon`
  * `polyline`
  * `use`
  * `g`
  * `svg`
  * `div`
  * `tmd`

SVG Shapes

  * `star`
  * `polyshape`
