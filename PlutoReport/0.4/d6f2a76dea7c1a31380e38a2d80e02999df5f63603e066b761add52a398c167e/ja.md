```
presentation_controls(aside::Bool = true) :: PlutoUI.CombineNotebook.CombinedBonds
```

PlutoUIウィジェットを返し、[`PresentationControls`](@ref)を返し、[`presentation_ui`](@ref)の制御パネルとして機能します。`PlutoUI.bind`を使用して変数にバインドする必要があります。

# 引数

  * `aside::Bool`: 制御パネルが浮いているかどうか

他に[`presentation_ui`](@ref)、[`PresentationControls`](@ref)を参照してください。
