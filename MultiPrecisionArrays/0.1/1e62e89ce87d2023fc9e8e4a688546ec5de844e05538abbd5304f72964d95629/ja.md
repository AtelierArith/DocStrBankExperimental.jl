mpglu!(MPG::MPGEFact, A::AbstractArray{TW,2}) where TW <: Real は、ストレージを再利用して新しい行列 A の多倍長因子分解 MPF を上書きします。

もちろん、これにより元の因子分解は破棄されます。

これを使用するには、次のようにします。

```
MPG=mpglu!(MPG,A)
```

単に次のように使用することは、

```
mpglu!(MPG,A) # これはしないでください。
```

（つまり、明示的に MPG を返さずに）

あなたが望むことをしない可能性があります。なぜなら、多倍長因子分解構造は不変であり、MPF.AF.info は変更できないからです。

MPG を再割り当てすることは機能し、元の配列のほとんどすべてのストレージを再利用します。
