mplu!(MPA::MPArray; residterm=residtermdefault)

プレーンバニラ MPArray 因数分解: 低精度コピーを因数分解し、高精度行列はそのままにします。出力として因数分解オブジェクトが得られ、線形システムを解くために `\` を使用できます。

中間精度転送に関する話は、MPArrayを構築する際にブール値 `onthefly` を設定できるということです。`mplu` を使用すると、デフォルトが得られます。

  * `onthefly == false` の場合、ソルバーは解く前に残差をダウンキャストし、N^2の中間精度転送を回避します。
  * `onthefly == true` の場合、ソルバーは中間精度転送をその場で行い、そのためにN^2の中間精度転送コストが発生します。

    `onthefly == true` は、低精度因数分解をIR-GMRESの前処理として使用する予定がある場合や、Float16で作業していて行列が非常に条件が悪い場合に使用する必要があります。

    `onthefly == nothing` は、デフォルトを取ることを意味します。

キーワード引数 `residterm` は終了基準を設定します。 `residterm == true`（デフォルト）は、小さな残差で反復を終了します。 `residterm == false` は、小さなノルム逆誤差で反復を終了します。詳細についてはドキュメントを参照してください。

この内容で静的配列を使用したい場合は、可変の @MArray コンストラクタを使用してください。
