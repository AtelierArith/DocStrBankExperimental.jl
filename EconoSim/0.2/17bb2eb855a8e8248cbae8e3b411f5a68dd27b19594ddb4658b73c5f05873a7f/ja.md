```
struct Balance
```

バランスシート、現在のバランスシートの状態に至るまでの取引の履歴を含む。

# プロパティ

  * assets: バランスシートの資産側。
  * def*min*asset: 資産残高エントリのデフォルトの下限。
  * min*assets: 最小資産値。取引の検証に使用される。エントリは def*min_asset を上書きする。
  * liabilities: バランスシートの負債側。
  * def*min*liability: 負債残高エントリのデフォルトの下限。
  * min*liabilities: 最小負債値。取引の検証に使用される。エントリは def*min_liability を上書きする。
  * log_transactions: 取引が記録されるかどうかを示すフラグ。取引を記録しないことでパフォーマンスが向上する。
  * transactions: 取引タプルの時系列リスト。各タプルは次のように構成される: タイムスタンプ、エントリタイプ（資産または負債）、残高エントリ、金額、新しい残高値、コメント。
  * properties: ユーザー定義のプロパティを持つ辞書。辞書のキーがシンボルの場合、値は balance.symbol で取得/設定できる。
