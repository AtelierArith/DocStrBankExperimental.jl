```
process_sumsy!(balance::Balance, sumsy::SuMSy, step::Int)
```

タイムスタンプがSuMSyのインターバルの倍数である場合、デマレージと保証された収入を処理します。それ以外の場合、この関数は何もしません。預けられた保証された収入の金額と差し引かれたデマレージを返します。この関数がタイムスタンプ == 0 で呼び出されると、バランスは「シード」されます。シード金額は返される収入に追加されます。

  * sumsy: 計算に使用するSuMSyの実装。
  * balance: SuMSyを適用するバランス。
  * timestamp: 現在のタイムスタンプ。アクションを取る必要があるかどうかを判断するために使用されます。
