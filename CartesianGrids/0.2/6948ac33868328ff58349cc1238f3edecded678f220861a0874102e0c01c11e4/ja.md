```
grid_interpolate!(out:ScalarGridData,in::ScalarGridData) -> ScalarGridData
```

スカラーグリッドデータ `in` をスカラーグリッドデータ `out` に1次元対称補間します。`in` または `out` のいずれかはエッジコンポーネントデータであり、もう一方はノードデータでなければなりません。補間の方向は `in` と `out` の関係によって決まります。たとえば、`in` がデュアルノード（セルごと）で、`out` がプライマル x-エッジコンポーネント（セルごとエッジ）である場合、補間は y 方向で行われます。なぜなら、この方向では異なるタイプだからです。（x 方向では同じタイプ（セル）であるため、その方向では補間は行われません。）
