```
coordinates(w::GridData;[dx=1.0],[I0=(1,1)])
```

グリッドデータ `w` の各方向における物理座標の範囲のタプルを返します。`w` が `Nodes` 型の場合、`xg,yg` の形式のタプルを返します。`w` が `Edges` または `NodePair` 型の場合、`xgu,ygu,xgv,ygv` の形式のタプルを返します。

オプションのキーワード引数 `dx` はグリッド間隔を設定し、そのデフォルトは `1.0` です。オプションのキーワード `I0` は整数のタプルを受け入れ、原点と一致する原始ノードのインデックスペアを設定します。デフォルトは `(1,1)` です。

# 例

```jldoctest
julia> w = Nodes(Dual,(12,22));

julia> xg, yg = coordinates(w,dx=0.1)
(-0.05:0.1:1.05, -0.05:0.1:2.0500000000000003)
```
