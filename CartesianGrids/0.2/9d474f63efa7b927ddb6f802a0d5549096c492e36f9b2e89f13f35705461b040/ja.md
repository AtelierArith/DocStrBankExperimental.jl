```
grad(w::Nodes{Primal}) --> Edges{Primal}
```

プライマルノードデータ `w` の離散勾配を評価します。この操作は、Grad型のオブジェクトを作成し、`*` で適用することでも実行できます。

# 例

julia> w = Nodes(Primal,(8,6));

julia> w[3,4] = 1.0;

julia> G = Grad();
