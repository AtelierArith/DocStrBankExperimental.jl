```
Regularize(x,y,dx,[ddftype=Yang3],[graddir=0],[I0=(1,1)], [weights=1.0], [filter=false],
                   [issymmetric=false])
```

データをグリッドに埋め込まれた点からグリッド自体のフィールドに対して、またその逆に補正および補間するためのオペレーターを設定するためのコンストラクタです。提供された `x` と `y` は埋め込まれた点の物理座標を表し、`dx` はグリッドの均一な物理セルサイズを示します。別々の引数 `x` と `y` は、座標を保持する `VectorData` 型の単一の引数 `X` に置き換えることができます。

補正と補間の操作は、離散デルタ関数 (ddf) を使用して行われ、デフォルトでは `Yang3` 型です。他にも `Roma`、`Goza`、または `M3` などの型も可能です。オプションの引数 `graddir` が 1 または 2 に設定されている場合、埋め込まれた点でグリッドフィールドの勾配のそれぞれの成分の負の値を評価する補間オペレーターが生成されます。この引数のデフォルト値は 0 で、単に補間を行います。この勾配タイプの補正形式も可能であることに注意してください。

オプションのタプル `I0` は、`(x,y) = (0,0)` に一致する主要ノードのインデックスを表します。これはデフォルトで `(1,1)` で、1 層のゴースト (双対) セルを残し、内部双対セルのグリッドの左下隅に物理的な原点を設定します。

もう一つのオプションのパラメータ `weights` は、補正における各点の重みを設定します。これは一般的に、曲線上のデータの補正のための微分弧長などで設定されます。これは、`x` と `y` と同じ長さのベクトルであるか、均一であればスカラーでもかまいません。デフォルトは 1.0 です。

オプションのブールパラメータ `filter` は、補間の前にグリッドデータにフィルタリングを適用することが望ましい場合に `true` に設定できます (Goza et al, J Comput Phys 2016 を参照)。これは一般的に、埋め込まれた点に対する力の解を前処理する文脈でのみ使用されます。

オプションのブールパラメータ `issymmetric` が `true` に設定されている場合、補正と補間は互いの転置として構築されます。このオプションは、提供された重みを上書きすることに注意してください。このパラメータのデフォルトは `false` です。

結果として得られるオペレーターは、補正と補間のいずれの方向でも使用でき、最初の引数は *ターゲット* (補正/補間する対象) を表し、2 番目の引数は *ソース* (補正/補間する元) を表します。補正はフィルタリングオプションを使用しません。
