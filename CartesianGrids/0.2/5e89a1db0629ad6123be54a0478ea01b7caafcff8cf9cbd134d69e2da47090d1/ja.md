```
diff!(out:ScalarGridData,in::ScalarGridData) -> ScalarGridData
```

スカラーグリッドデータ `in` の1次元中央有限差分をスカラーグリッドデータ `out` に返します。`in` または `out` のいずれかはエッジコンポーネントデータであり、もう一方はノードデータでなければなりません。差分の方向は `in` と `out` の関係によって決まります。たとえば、`in` がデュアルノード（セルごと）で、`out` がプライマルxエッジコンポーネント（セルごとにエッジ）である場合、差分はy方向で行われます。なぜなら、この方向では異なるタイプだからです。
