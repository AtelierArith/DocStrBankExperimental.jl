```
grid_interpolate!(q::Edges{Dual},w::Nodes{Dual})
```

双対ノードデータ `w` を双対セルのエッジに補間し、結果を `q` に返します。

# 例

```jldoctest
julia> w = Nodes(Dual,(8,6));

julia> w[3,4] = 1.0;

julia> q = Edges(Dual,w);
```
