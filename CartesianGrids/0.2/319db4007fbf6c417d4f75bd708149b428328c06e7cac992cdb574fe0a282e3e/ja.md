```
DDF([ddftype=Roma],[dx=1.0])
```

離散デルタ関数演算子を構築します。これは一般的に`Regularize`演算子によって内部的にのみ必要とされるため、ユーザーが直接アクセスする必要はあまりありません。デフォルトのDDFは`Roma`関数で、サポートは3つのグリッドセルです。他の選択肢としては、28セルのサポートを持つ切断ガウス関数である`Goza`演算子や、2セルのサポートを持つ`Witchhat`があります。結果として得られる演算子は、1つ、2つ、または3つの座標引数で評価され、それぞれ1次元、2次元、または3次元のスミアデルタ関数を生成します。また、引数の配列に対して通常のJuliaのベクトル化されたドット表記を使用して呼び出すこともできます。オプションのセル間隔引数`dx`は、座標をこの間隔でスケーリングし、結果もこの間隔（次元数の累乗）でスケーリングされます。この間隔引数のデフォルトは1.0です。

```jldoctest
julia> ddf = DDF(ddftype=Roma)

julia> ddf(1)
0.16666666666666666

julia> ddf(-1)
0.16666666666666666

julia> ddf.([-1,0,1])
3-element Array{Float64,1}:
 0.16666666666666666
 0.6666666666666666
 0.16666666666666666
```
