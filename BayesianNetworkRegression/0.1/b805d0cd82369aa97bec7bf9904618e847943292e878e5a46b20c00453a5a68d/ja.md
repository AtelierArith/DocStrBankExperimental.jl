```
Fit!(X::AbstractArray{T}, y::AbstractVector{U}, R; η=1.01,V=30,ζ=1.0,ι=1.0,aΔ=1.0,bΔ=1.0, 
     ν=10, nburn=30000, nsamples=20000, maxburn=50000, mingen=0, maxgen=0, psrf_cutoff=1.01, x_transform=true, suppress_timer=false, 
     num_chains=2, seed=nothing, purge_burn=nothing, filename="parameters.log") where {T,U}
```

ベイジアンネットワーク回帰モデルをフィットさせます。世代数を制御する2つの代替スキームがあります。

  * 「従来の」スキームでは、アルゴリズムは`nburn`バーニング後に`nsamp`ギブスサンプルを生成し、破棄します。
  * 「倍増世代」スキームでは、アルゴリズムは`mingen`サンプルを生成し、最初の半分を破棄します。Gelman-Rubin PSRFカットオフが達成されない場合、`mingen`の追加サンプルが生成されます（すべての世代の最初の半分をバーニングとして考慮）し、収束が達成されるか、`maxgen`サンプルが生成されるまで続きます。

デフォルトでは、従来のスキームが使用されます。`mingen`と`maxgen`に有効な値が提供されると、「倍増世代」スキームが使用されます。

fit!のロードマップ：

  * 「従来の」スキームの場合は[`generate_samples!`](@ref)を、 「倍増世代」スキームの場合は[`generate_samples_dbl!`](@ref)を直接呼び出します
  * `generate_samples!`は、各チェーンで[`initialize_and_run!`](@ref)を呼び出します
  * `initialize_and_run!`は、[`initialize_variables!`](@ref)と[`gibbs_sample!`](@ref)を呼び出します

# 引数

  * `X`: 行列、必須、予測子として使用される重みなし対称隣接行列の行列。2つのオプション：        各行が1つのサンプルに関連する隣接行列の上三角である2D行列       各行がノード同士を関連付ける隣接行列である1D行列
  * `y`: ベクトル、必須、応答変数のベクトル
  * `R`: 整数、必須、潜在変数uの次元、ハイパーパラメータ
  * `η`: 浮動小数点、デフォルト=1.01、πᵥパラメータの0値をサンプリングするために使用されるハイパーパラメータ、1より大きくなければならない
  * `ζ`: 浮動小数点、デフォルト=1.0、θをサンプリングするために使用されるハイパーパラメータ
  * `ι`: 浮動小数点、デフォルト=1.0、θをサンプリングするために使用されるハイパーパラメータ
  * `aΔ`: 浮動小数点、デフォルト=1.0、Δをサンプリングするために使用されるハイパーパラメータ
  * `bΔ`: 浮動小数点、デフォルト=1.0、Δをサンプリングするために使用されるハイパーパラメータ
  * `ν`: 整数、デフォルト=10、Mをサンプリングするために使用されるハイパーパラメータ、Rより大きくなければならない
  * `nburn`: 整数、デフォルト=30000、生成して破棄するバーニングサンプルの数
  * `nsamples`: 整数、デフォルト=20000、バーニング後に生成するギブスサンプルの数
  * `mingen`: 整数、デフォルト0、「倍増世代」のために生成する最小サンプル数（バーニング + 残存）
  * `maxgen`: 整数、デフォルト0、「倍増世代」のために生成する最大サンプル数（バーニング + 残存）
  * `psrf_cutoff`: 浮動小数点、デフォルト=1.2、収束が達成されたと見なされる値
  * `x_transform`: ブール、デフォルト=true、Xがサンプルごとに1行に事前変換されている場合はfalseに設定します。そうでない場合、Xは自動的に変換されます。
  * `suppress_timer`: ブール、デフォルト=false、「進行メーター」出力を抑制するためにtrueに設定します
  * `num_chains`: 整数、デフォルト=2、収束を確認するために実行する別々のサンプリングチェーンの数
  * `seed`: 整数、デフォルト=nothing、再現性のために使用されるランダムシード
  * `purge_burn`: 整数、デフォルト=nothing、設定されている場合はバーニングサンプルの数より少なくなければならない（理想的にはバーニングはこの値の倍数であるべきです）。以前のバーニングサンプルを削除するためのバーニングサンプルの数。
  * `filename`: フィットに使用されたパラメータのログファイル、デフォルト="parameters.log"。新しい名前が指定されない場合、ファイルは上書きされます。
