```julia
SetScrollHereX()
SetScrollHereX(center_x_ratio)

```

Adjust scrolling amount to make current cursor position visible. center*x*ratio=0.0: left, 0.5: center, 1.0: right. When using to make a "default/current item" visible, consider using SetItemDefaultFocus() instead.
