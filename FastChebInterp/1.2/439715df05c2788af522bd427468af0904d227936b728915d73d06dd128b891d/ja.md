```
chebregression(x, y, [lb, ub,] order)
```

与えられた `order` のチェビシェフ多項式の最小二乗フィットを行うことによって構築されたチェビシェフ多項式（`ChebPoly`）を返します。ここで、`x` はデータポイント `y` の座標です。 `lb` と `ub` はそれぞれチェビシェフ領域の下限と上限であり、通常は `x` のすべての点を含む必要があります。省略した場合は、`x` の最小および最大座標がデフォルトとなります。

1次元の場合、`x` はスカラーの配列であり、`lb < ub` はスカラーで、`order` は整数です。N次元の場合、`order` は整数の `N`-タプル（各次元の次数）であり、`lb` と `ub` は `N`-成分ベクトルで、`x` は `N`-成分ベクトルの配列（または `N` 列を持つ行列で、ベクトル成分として解釈されます）です。

`y` はベクトルまたは数値、またはベクトル値のチェビシェフフィット用のベクトルのベクトルであることができます。後者のケースは、列がベクトル成分である行列としても入力できます。 `size(x,1)` と `size(y,1)` は一致し、`prod(order .+ 1)` を超えなければなりません。
