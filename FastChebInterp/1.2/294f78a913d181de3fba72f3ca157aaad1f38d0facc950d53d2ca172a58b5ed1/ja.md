ハイパーキューブ（直交積）ドメイン上での高速多次元チェビシェフ補間。分離可能（テンソル積）グリッドのチェビシェフ補間点を使用します。

ドメインの上限と下限 `lb` と `ub`、および与えられた `order` タプルに対して、関数 `f` の補間器を次のように作成します：

```
using FastChebInterp
x = chebpoints(order, lb, ub) # StaticVector の配列
c = chebinterp(f.(x), lb, ub)
```

その後、点 `y`（ベクトル）に対して補間器を評価するには `c(y)` を使用します。

また、点 `y` における補間器とその勾配のタプル `(c(y), ∇c(y))` を返す関数 `chebgradient(c,y)` も提供しています。

FastChebInterp パッケージは、複素数およびベクトル値関数 `f` もサポートしています。この場合、`c(y)` は補間器のベクトルを返し、`chebjacobian(c,y)` を使用して、補間器とそのヤコビ行列のタプル `(c(y), J(y))` を計算できます。
