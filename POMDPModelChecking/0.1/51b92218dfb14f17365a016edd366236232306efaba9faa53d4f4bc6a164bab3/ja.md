```
ReachabilitySolver{S} <: Solver
```

MDPおよびPOMDPにおける到達可能性および制約付き到達可能性問題を解決します。指定された状態のセットに到達する確率を最大化するポリシーを返します。到達する状態のセットと回避する状態のセット、さらに基盤となるソルバーを入力として受け取ります。POMDPs.jlの任意のソルバーがサポートされています。

# フィールド

フィールドはソルバーへのキーワード引数として指定されます。

  * `reach::Set{S}` 到達する状態のセット
  * `avoid::Set{S}` 回避する状態のセット
  * `solver::Solver` 使用する基盤となるソルバー（デフォルトは`ValueIterationSolver`）
