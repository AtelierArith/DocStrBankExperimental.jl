Modify the color and thickness of the line when Tracing is enabled

Tracing can be enabled by pressing T in the Editor || setting `EnableTrace` to `True` in the Vehicle Settings

Args:     color*rgba (list) desired RGBA values from 0.0 to 1.0     thickness (float, optional) Thickness of the line     vehicle*name (string, optional) Name of the vehicle to set Trace line values for
