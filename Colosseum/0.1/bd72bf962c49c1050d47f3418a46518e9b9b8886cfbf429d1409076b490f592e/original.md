Get a single image Returns bytes of png format image which can be dumped into abinary file to create .png image `string_to_uint8_array()` can be used to convert into Numpy unit8 array See https://microsoft.github.io/AirSim/image*apis/ for details Args:     camera*name (str) Name of the camera, for backwards compatibility, ID numbers such as 0,1,etc. can also be used     image*type (ImageType) Type of image required     vehicle*name (str, optional) Name of the vehicle with the camera     external (bool, optional) Whether the camera is an External Camera Returns:     Binary string literal of compressed png image
