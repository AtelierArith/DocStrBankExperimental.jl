```
    (1) ispos(λ::Vector{T};
	<
	tol::Real=0,
	rev=true,
	🔔=true,
	msg="">)

    (2) ispos(Λ::𝔻{T};
	< (1) と同じオプションのキーワード引数 > )

	すべての上記: ここで T<:Real

(1) の実ベクトル $λ$ または (2) の実 `Diagonal` 行列 $Λ$ のすべての数が $tol$ より劣っていない場合は $true$ を返し、そうでない場合は $false$ を返します。これは、すべての固有値が正であることを確認するために、例えばスペクトル関数で使用されます。

!!! note "Nota Bene"
    $tol$ は $λ$ (1) または $Λ$ (2) の型の `Base.eps` の平方根にデフォルト設定されています。これは、約半分の有効桁数を超える正を要求することに対応します。

以下は *<オプションのキーワード引数>* です：

  * $rev=true$ の場合、(1) の $λ$ の要素または (2) の $Λ$ の対角要素は逆順でチェックされます。これは、要素がソートされている場合に非常に高速なチェックを可能にし、どこからチェックを開始するのが最適かがわかっている場合に行われます。

結果が $false$ の場合：

  * $🔔=true$ の場合、ベル文字が印刷されます。ほとんどのシステムでは、これによりコンピュータでベルが鳴ります。
  * 文字列 $msg$ が提供されている場合、警告が $msg$ を印刷し、次に：

"位置 *pos* で"、ここで *pos* は最初の非正の要素が見つかった位置です。

**例**

```

julia using PosDefManifold a=[1, 0, 2, 8] ispos(a, msg="非正の要素が見つかりました")

# 次のように印刷されます：

# ┌ Warning: 非正の要素が見つかりました 位置 2

# └ @ [ここでジュリアは警告を発行したコードの行を指します]

```
