```julia
    (1) geodesic(metric::Metric, P::ℍ{T}, Q::ℍ{T}, a::Real) where T<:RealOrComplex
    (2) geodesic(metric::Metric, D::𝔻{S}, E::𝔻{S}, a::Real) where S<:Real
```

(1) 点 $P$ から点 $Q$ までの [geodesic](@ref) に沿って、指定されたメトリック（[Metric::Enumerated type](@ref) のタイプ）を使用して、*arclegth* $0<=a<=1$ で移動します。

すべてのメトリックに対して、

  * $$
    a=0
    $$

    の場合、$P$ に留まります。
  * $$
    a=1
    $$

    の場合、$Q$ まで移動します。
  * $$
    a=1/2
    $$

    の場合、$P$ と $Q$ の中点（平均）に移動します。

フィッシャーメトリックを使用すると、引数 $a$ は *任意の* 実数であり、例えば：

  * $$
    0<a<1
    $$

    の場合、$Q$ に向かって移動します（*引力*）。
  * $$
    a>1
    $$

    の場合、$Q$ を越えて移動します（*外挿*）。
  * $$
    a<0
    $$

    の場合、$Q$ から離れて移動します（*反発*）。

$$
P
$$

と $Q$ は、julia によって `Hermitian` としてフラグ付けされる必要があります。 [typecasting matrices](@ref) を参照してください。

フィッシャーの測地線移動は、Iannazzo(2016)[🎓](@ref) の Eq. 4.2 に示されている Cholesky-Schur アルゴリズムによって計算されます。 $Q=I$ の場合、フィッシャーの測地線移動は単に $P^a$ です（この関数を呼び出す必要はありません）。

!!! note "Nota Bene"
    [logdet zero](@ref) および [Jeffrey](@ref) メトリックについては、著者の知識の限りでは測地線の閉じた形の表現は利用できないため、この場合、測地線は [`mean`](@ref) 関数を使用して加重平均として求められます。[Von Neumann](@ref) については、平均の表現すら利用できないため、この場合、測地線は提供されず、*警告* が表示されます。


(2) (1) と同様ですが、2 つの実正定値対角行列 $D$ と $E$ に対してです。

**数学**

点 $P$、$Q$ および弧長 $a$ に対して、$b=1-a$ とすると、サポートされているメトリックの測地線方程式は次のとおりです：

|   メトリック    | 測地線方程式                                                                                          |
|:----------:|:----------------------------------------------------------------------------------------------- |
|   ユークリッド   | $bP + aQ$                                                                                       |
|  逆ユークリッド   | $\big(bP^{-1} + aQ^{-1}\big)^{-1}$                                                              |
| チョー・ユークリッド | $TT^*$、ここで $T=bL_P + aL_Q$                                                                      |
|  対数ユークリッド  | $\text{exp}\big(b\hspace{2pt}\text{log}(P) + a\hspace{2pt}\text{log}(Q)\big)$                   |
|  対数コレスキー   | $TT^*$、ここで $T=S_P+a(S_Q-S_P)+D_P\hspace{2pt}\text{exp}\big(a(\text{log}D_Q-\text{log}D_P)\big)$ |
|   フィッシャー   | $P^{1/2} \big(P^{-1/2} Q P^{-1/2}\big)^a P^{1/2}$                                               |
|  logdet0   | 加重平均アルゴリズム [`logdet0Mean`](@ref) を使用                                                            |
|   ジェフリー    | 加重平均 [`mean`](@ref) を使用                                                                         |
|  ボン・ノイマン   | N.A.                                                                                            |
|  ワッサースタイン  | $b^2P+a^2Q +ab\big[(PQ)^{1/2} +(QP)^{1/2}\big]$                                                 |

**凡例:** $L_X$、$S_X$ および $D_X$ はそれぞれ $X$ のコレスキー下三角行列、その厳密に下三角の部分および対角部分です（したがって、$S_X+D_X=L_X$、$L_XL_X^*=X$）。

**関連情報**: [`mean`](@ref)。

**例**

```julia
using PosDefManifold
P=randP(10)
Q=randP(10)
# ワッサースタイン平均
M=geodesic(Wasserstein, P, Q, 0.5)
# フィッシャーメトリックを使用して外挿
E=geodesic(Fisher, P, Q, 2)
```
