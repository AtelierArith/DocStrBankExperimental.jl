```
SpatioTemporalEmbedding{Φ,BC,X} → embedding
```

空間時間遅延座標構造をファンクタとして使用します。`Φ` 空間次元のデータに適用され、次元 `X` の埋め込みを提供します。

```
embedding(rvec, s, t, α)
```

`rvec`（長さ `X`）に対してインプレースで動作し、時刻 `t` での空間時系列 `s` からベクトルを再構築します。カートesianインデックス `α` に注意してください。`t` に対する境界チェックはありません。

`s` は `Vector{<:AbstractArray{T,Φ}}` であると仮定されています。

## コンストラクタ

ここには直感的な埋め込みを返す便利なコンストラクタがいくつかあります：

  * [`cubic_shell_embedding`](@ref)
  * [`light_cone_embedding`](@ref)

「メイン」コンストラクタは

```
SpatioTemporalEmbedding{X}(τ, β, bc, fsize)
```

であり、空間時間埋め込みを完全に制御できます。

  * `Χ == length(τ) == length(β)` : 再構築された空間の次元。
  * `τ::Vector{Int}` = 再構築された空間の**各エントリ**のための時間遅延のベクトル（昇順にソート）。
  * `β::Vector{CartesianIndex{Φ}}` = 再構築された空間の**各エントリ**のための空間遅延の*相対*インデックスのベクトル。
  * `bc::BC` : 境界条件。
  * `fsize::NTuple{Φ, Int}` : 時系列における各状態のサイズ。

例えば、時間遅延 τ = 2, 4 のために空間隣接点 [0, ±1] を持ちたい場合は、

```
τ = [2,2,2,4,4,4]
β = CartesianIndex.([1, 0, -1, 1, 0, -1])
```
