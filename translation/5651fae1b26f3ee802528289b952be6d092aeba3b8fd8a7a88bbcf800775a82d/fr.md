```
pi
```

La constante pi.

Unicode `π` peut être tapé en écrivant `\pi` puis en appuyant sur tab dans le REPL Julia, et dans de nombreux éditeurs.

Voir aussi : [`sinpi`](@ref), [`sincospi`](@ref), [`deg2rad`](@ref).

# Exemples

```jldoctest
julia> pi
π = 3.1415926535897...

julia> 1/2pi
0.15915494309189535
```
