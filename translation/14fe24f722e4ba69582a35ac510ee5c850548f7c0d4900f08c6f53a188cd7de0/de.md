```
diskstat(path=pwd())
```

Gibt Statistiken in Bytes über das Laufwerk zurück, das die Datei oder das Verzeichnis enthält, auf das `path` verweist. Wenn kein Argument übergeben wird, werden Statistiken über das Laufwerk zurückgegeben, das das aktuelle Arbeitsverzeichnis enthält.

!!! compat "Julia 1.8"
    Diese Methode wurde in Julia 1.8 hinzugefügt.
