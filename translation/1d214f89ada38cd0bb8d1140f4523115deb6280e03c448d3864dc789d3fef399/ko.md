```
public
```

`public`는 모듈 내에서 Julia에게 어떤 이름이 모듈의 공개 API의 일부인지 알려주는 데 사용됩니다. 예를 들어: `public foo`는 이름 `foo`가 공개적임을 나타내지만, 모듈을 [`using`](@ref)할 때 사용할 수 있도록 하지는 않습니다. 자세한 내용은 [모듈에 대한 매뉴얼 섹션](@ref modules)을 참조하십시오.

!!! compat "Julia 1.11"
    public 키워드는 Julia 1.11에 추가되었습니다. 이전에는 공개성의 개념이 덜 명확했습니다.
