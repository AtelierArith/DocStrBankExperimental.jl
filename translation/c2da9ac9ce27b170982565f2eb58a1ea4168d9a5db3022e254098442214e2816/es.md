```
Cwstring
```

Una cadena de estilo C compuesta por el tipo de carácter ancho nativo [`Cwchar_t`](@ref)s. Los `Cwstring` están terminados en NUL. Para cadenas de estilo C compuestas por el tipo de carácter nativo, consulte [`Cstring`](@ref). Para más información sobre la interoperabilidad de cadenas con C, consulte el [manual](@ref man-bits-types).
