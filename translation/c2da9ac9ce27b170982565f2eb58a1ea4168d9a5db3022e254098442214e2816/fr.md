```
Cwstring
```

Une chaîne de caractères de style C composée du type de caractère large natif [`Cwchar_t`](@ref). Les `Cwstring` sont terminées par un NUL. Pour les chaînes de caractères de style C composées du type de caractère natif, voir [`Cstring`](@ref). Pour plus d'informations sur l'interopérabilité des chaînes avec C, voir le [manuel](@ref man-bits-types).
