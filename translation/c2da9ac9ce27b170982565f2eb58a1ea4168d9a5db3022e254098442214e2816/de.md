```
Cwstring
```

Ein C-ähnlicher String, der aus dem nativen Wide-Character-Typ [`Cwchar_t`](@ref)s besteht. `Cwstring`s sind NUL-terminiert. Für C-ähnliche Strings, die aus dem nativen Zeichentyp bestehen, siehe [`Cstring`](@ref). Für weitere Informationen zur String-Interoperabilität mit C siehe das [Handbuch](@ref man-bits-types).
