```
@which
```

Appliqué à un appel de fonction ou de macro, il évalue les arguments de l'appel spécifié et renvoie l'objet `Method` pour la méthode qui serait appelée pour ces arguments. Appliqué à une variable, il renvoie le module dans lequel la variable a été liée. Il appelle la fonction [`which`](@ref).

Voir aussi : [`@less`](@ref), [`@edit`](@ref).
