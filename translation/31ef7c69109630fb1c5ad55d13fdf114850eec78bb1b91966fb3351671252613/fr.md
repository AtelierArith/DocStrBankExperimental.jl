```
new, ou new{A,B,...}
```

Fonction spéciale disponible pour les constructeurs internes qui crée un nouvel objet du type. La forme new{A,B,...} spécifie explicitement les valeurs des paramètres pour les types paramétriques. Voir la section du manuel sur [Inner Constructor Methods](@ref man-inner-constructor-methods) pour plus d'informations.
