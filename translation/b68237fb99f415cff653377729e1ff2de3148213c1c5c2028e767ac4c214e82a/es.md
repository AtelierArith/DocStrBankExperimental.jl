```
mean!(r, v)
```

Calcula la media de `v` sobre las dimensiones singleton de `r`, y escribe los resultados en `r`.

# Ejemplos

```jldoctest
julia> using Statistics

julia> v = [1 2; 3 4]
2×2 Matrix{Int64}:
 1  2
 3  4

julia> mean!([1., 1.], v)
2-element Vector{Float64}:
 1.5
 3.5

julia> mean!([1. 1.], v)
1×2 Matrix{Float64}:
 2.0  3.0
```
