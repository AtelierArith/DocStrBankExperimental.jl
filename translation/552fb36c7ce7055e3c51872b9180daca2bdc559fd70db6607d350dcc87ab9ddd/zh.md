```
gcdx(a, b)
```

计算 `a` 和 `b` 的最大公约数（正）及其贝祖系数，即满足 $ua+vb = d = gcd(a, b)$ 的整数系数 `u` 和 `v`。`gcdx(a, b)` 返回 $(d, u, v)$。

参数可以是整数和有理数。

!!! compat "Julia 1.4"
    有理数参数需要 Julia 1.4 或更高版本。


# 示例

```jldoctest
julia> gcdx(12, 42)
(6, -3, 1)

julia> gcdx(240, 46)
(2, -9, 47)
```

!!! note
    贝祖系数 *不是* 唯一确定的。`gcdx` 返回通过扩展欧几里得算法计算的最小贝祖系数。（参考：D. Knuth, TAoCP, 2/e, p. 325, 算法 X。）对于有符号整数，这些系数 `u` 和 `v` 在 $|u| < |b/d|$ 和 $|v| < |a/d|$ 的意义下是最小的。此外，`u` 和 `v` 的符号被选择为使得 `d` 为正。对于无符号整数，系数 `u` 和 `v` 可能接近它们的 `typemax`，此时等式仅通过无符号整数的模运算成立。
