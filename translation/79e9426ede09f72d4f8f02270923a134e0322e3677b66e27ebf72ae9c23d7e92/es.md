```
uperm(file)
```

Obtiene los permisos del propietario del archivo como un campo de bits de

| Valor | Descripción          |
|:----- |:-------------------- |
| 01    | Permiso de Ejecución |
| 02    | Permiso de Escritura |
| 04    | Permiso de Lectura   |

Para los argumentos permitidos, consulta [`stat`](@ref).
