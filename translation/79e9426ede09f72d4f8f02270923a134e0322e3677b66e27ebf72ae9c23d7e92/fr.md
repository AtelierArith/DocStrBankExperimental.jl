```
uperm(fichier)
```

Obtenez les permissions du propriétaire du fichier sous forme de champ de bits

| Valeur | Description            |
|:------ |:---------------------- |
| 01     | Permission d'exécution |
| 02     | Permission d'écriture  |
| 04     | Permission de lecture  |

Pour les arguments autorisés, voir [`stat`](@ref).
