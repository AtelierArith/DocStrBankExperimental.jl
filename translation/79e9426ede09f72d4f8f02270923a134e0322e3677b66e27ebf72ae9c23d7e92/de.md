```
uperm(datei)
```

Holen Sie sich die Berechtigungen des Eigentümers der Datei als Bitfeld von

| Wert | Beschreibung            |
|:---- |:----------------------- |
| 01   | Ausführungsberechtigung |
| 02   | Schreibberechtigung     |
| 04   | Leseberechtigung        |

Für erlaubte Argumente siehe [`stat`](@ref).
