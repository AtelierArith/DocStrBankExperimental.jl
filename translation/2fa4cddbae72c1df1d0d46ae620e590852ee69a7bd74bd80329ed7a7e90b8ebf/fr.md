```
/(x, y)
```

Opérateur de division à droite : multiplication de `x` par l'inverse de `y` à droite.

Donne des résultats en virgule flottante pour les arguments entiers. Voir [`÷`](@ref div) pour la division entière, ou [`//`](@ref) pour les résultats [`Rational`](@ref).

# Exemples

```jldoctest
julia> 1/2
0.5

julia> 4/2
2.0

julia> 4.5/2
2.25
```
