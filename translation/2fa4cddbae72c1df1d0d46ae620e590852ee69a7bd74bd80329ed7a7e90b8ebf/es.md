```
/(x, y)
```

Operador de división a la derecha: multiplicación de `x` por el inverso de `y` a la derecha.

Devuelve resultados de punto flotante para argumentos enteros. Consulta [`÷`](@ref div) para división entera, o [`//`](@ref) para resultados de [`Rational`](@ref).

# Ejemplos

```jldoctest
julia> 1/2
0.5

julia> 4/2
2.0

julia> 4.5/2
2.25
```
