```
muladd(A, y, z)
```

결합된 곱셈-덧셈, `A*y .+ z`, 행렬-행렬 또는 행렬-벡터 곱셈을 위한 것입니다. 결과는 항상 `A*y`와 같은 크기를 가지지만, `z`는 더 작거나 스칼라일 수 있습니다.

!!! compat "Julia 1.6"
    이러한 메서드는 Julia 1.6 이상이 필요합니다.


# 예제

```jldoctest
julia> A=[1.0 2.0; 3.0 4.0]; B=[1.0 1.0; 1.0 1.0]; z=[0, 100];

julia> muladd(A, B, z)
2×2 Matrix{Float64}:
   3.0    3.0
 107.0  107.0
```
