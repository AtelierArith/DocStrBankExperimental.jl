```
추상형
```

`추상형`은 인스턴스화할 수 없는 형을 선언하며, 형 그래프의 노드로만 사용되어 관련된 구체적 형의 집합을 설명합니다: 그 구체적 형들은 이들의 자손입니다. 추상형은 줄리아의 형 시스템을 단순한 객체 구현의 집합 이상으로 만드는 개념적 계층을 형성합니다. 예를 들어:

```julia
abstract type Number end
abstract type Real <: Number end
```

[`Number`](@ref)에는 슈퍼타입이 없고, 반면에 [`Real`](@ref)는 `Number`의 추상 하위형입니다.
