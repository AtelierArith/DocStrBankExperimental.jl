```
rounding(T)
```

Obtiene el modo de redondeo de punto flotante actual para el tipo `T`, controlando el redondeo de las funciones aritméticas básicas ([`+`](@ref), [`-`](@ref), [`*`](@ref), [`/`](@ref) y [`sqrt`](@ref)) y la conversión de tipos.

Consulta [`RoundingMode`](@ref) para los modos disponibles.
