```
Cstring
```

Une chaîne de caractères de style C composée du type de caractère natif [`Cchar`](@ref). Les `Cstring` sont terminées par un NUL. Pour les chaînes de caractères de style C composées du type de caractère large natif, voir [`Cwstring`](@ref). Pour plus d'informations sur l'interopérabilité des chaînes avec C, voir le [manuel](@ref man-bits-types).
