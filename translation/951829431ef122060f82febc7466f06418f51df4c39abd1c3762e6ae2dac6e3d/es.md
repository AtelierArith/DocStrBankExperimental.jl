```
AbstractArray{T,N}
```

Supertipo para arreglos de `N` dimensiones (o tipos similares a arreglos) con elementos del tipo `T`. [`Array`](@ref) y otros tipos son subtipos de esto. Consulta la sección del manual sobre la interfaz [`AbstractArray`](@ref man-interface-array).

Véase también: [`AbstractVector`](@ref), [`AbstractMatrix`](@ref), [`eltype`](@ref), [`ndims`](@ref).
