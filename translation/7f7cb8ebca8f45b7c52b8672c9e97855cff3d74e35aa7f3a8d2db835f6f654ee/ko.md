```
@. expr
```

모든 함수 호출 또는 연산자를 `expr`에서 "점 호출"로 변환합니다(예: `f(x)`를 `f.(x)`로 변환) 그리고 `expr`에서 모든 할당을 "점 할당"으로 변환합니다(예: `+=`를 `.+=`로 변환).

`expr`에서 선택된 함수 호출에 대해 점을 추가하는 것을 *피하고 싶다면*, `$`로 해당 함수 호출을 분리하십시오. 예를 들어, `@. sqrt(abs($sort(x)))`는 `sqrt.(abs.(sort(x)))`와 동등합니다(여기서 `sort`에는 점이 없습니다).

(`@.`는 `@__dot__`에 대한 호출과 동등합니다.)

# 예시

```jldoctest
julia> x = 1.0:3.0; y = similar(x);

julia> @. y = x + 3 * sin(x)
3-element Vector{Float64}:
 3.5244129544236893
 4.727892280477045
 3.4233600241796016
```
