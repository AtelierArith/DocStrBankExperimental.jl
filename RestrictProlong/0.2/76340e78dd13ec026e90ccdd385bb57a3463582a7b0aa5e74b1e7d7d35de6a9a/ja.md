```
Ap = prolong(A, [sz::Tuple])
```

サイズを二重に拡張します。`sz`が省略された場合、すべての次元が拡張されます。`sz`が指定された場合、次のいずれかになります：

  * `sz[d] = size(A, d)`: 拡張は行われません
  * `sz[d] = 2*size(A, d) - 1`: 奇数サイズへの拡張で、次元`d`に沿ったすべての点の間の半グリッドに点を追加（補間による）
  * `sz[d] = 2*size(A, d)`: 偶数サイズへの拡張で、現在の範囲の両端を超えた1点を含む1/4および3/4グリッド上の点を推定

他の選択肢はエラーになります。デフォルトの選択は奇数拡張を行うことです。`prolong`は、`A`の合計をおおよそ保持するように正規化されています。

演算子として考えると、`prolong`は`restrict`の転置と等しいです。
