```
nid(F::System; options...)
```

[`numerical_irreducible_decomposition`](@ref)を呼び出します。

### 例

```julia-repl
julia> @var x, y
julia> f = [x^2 + y^2 - 1]
julia> N = nid(f)

1つの成分を持つ数値的不可約分解
• 次元1の成分が1つあります。

成分の次数表:
╭───────────┬───────────────────────╮
│ 次元      │ 成分の次数           │
├───────────┼───────────────────────┤
│     1     │           2           │
╰───────────┴───────────────────────╯
```
