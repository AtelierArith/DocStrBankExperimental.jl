```
rand_subspace(n::Integer; dim | codim, affine = true, real = false)
```

Generate a random [`LinearSubspace`](@ref) with given dimension `dim` or codimension `codim` (one of them has to be provided) in ambient space of dimension `n`. If `real` is `true`, then the extrinsic description is real. If `affine` then an affine linear subspace is generated. The subspace is generated by drawing each entry of the extrinsic description indepdently from a normal distribuation using `randn`.

```
rand_subspace(x::AbstractVector; dim | codim, affine = true)
```

Generate a random [`LinearSubspace`](@ref) with given dimension `dim` or codimension `codim` (one of them has to be provided) in ambient space of dimension `length(x)` going through the given point `x`.
