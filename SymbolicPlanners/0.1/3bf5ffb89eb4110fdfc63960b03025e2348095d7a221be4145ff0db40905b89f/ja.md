```
ProbForwardPlanner(;
    search_noise::Float64 = 1.0,
    kwargs...
)
```

確率的な前方最良優先探索のバリアントです。常に探索フロンティア内の最小の $f$ 値を持つノードを展開するのではなく、ボルツマン分布に従って展開するノードをサンプリングします。ここで、フロンティアノードの $f$ 値は展開の非正規化対数確率として扱われます。

ボルツマンサンプリングの温度は `search_noise` によって定義されます。値が高いほどランダムな探索が行われ、値が低いほど決定論的な探索が行われます。

特に限られた `max_nodes` 予算と組み合わせることで、潜在的に最適でない計画の多様性をシミュレートするのに役立ちます。

`ForwardPlanner{Float64}` のエイリアスです。他の引数については [`ForwardPlanner`](@ref) を参照してください。
