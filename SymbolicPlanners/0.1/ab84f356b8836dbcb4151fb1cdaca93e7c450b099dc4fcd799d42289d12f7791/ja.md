```
MetricHeuristic(metric, fluents[, coeffs])
```

現在の状態と指定された数値 `fluents` の目標との間の `metric` 距離を計算するヒューリスティックであり、メトリック計算の前にスカラー `coeffs` で (オプションで) 乗算されます。

このヒューリスティックは、提供された `fluents` に対する等式制約のリストを含む目標式とのみ使用できます。

# 引数

  * `metric`

    現在の状態と目標のフルーエント値の差のベクトルを与えられたときにスカラー値を返す関数。
  * `fluents`

    状態内の数値フルーエントを参照する `Term` のリスト。
  * `coeffs`

    各フルーエント値がメトリック計算の前に乗算されるスカラー係数のリスト。すべてのフルーエントに対してデフォルトは `1` です。
