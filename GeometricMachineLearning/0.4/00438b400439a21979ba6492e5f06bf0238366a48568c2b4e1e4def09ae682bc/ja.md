```
バッチ
```

`Batch`は、`DataLoader`のインスタンスに作用して、1エポックのトレーニングのためのトレーニングサンプルのシーケンスを生成するファンクタを持つ構造体です。

異なるコンストラクタについては、[`Batch(::Int)`](@ref)および[`Batch(::Int, ::Int, ::Int)`](@ref)を参照してください。

# ファンクタ

`Batch`のインスタンスは、`DataLoader`のインスタンスに呼び出されて、すべての入力データを含むサンプルのシーケンスを生成します。つまり、1エポックのトレーニングのためです。

`batch:Batch`を`dl::DataLoader`に適用した結果は、整数のベクトルのタプルです。これらのベクトルのそれぞれには2つの整数が含まれています。最初の整数は*時間インデックス*で、2番目の整数は*パラメータインデックス*です。

# 例

以下は、ベクトルで構築された`DataLoader`のインスタンスに対してサイズ2のバッチを描画する例です。

```jldoctest
using GeometricMachineLearning
import Random

rng = Random.TaskLocalRNG()
Random.seed!(rng, 123)

dl = DataLoader(rand(rng, 5))
batch = Batch(2)

batch(dl)

# 出力

[ Info: 入力として行列を提供しました。軸は(i) システム次元および(ii) パラメータの数として解釈されます。
([(1, 5), (1, 3)], [(1, 4), (1, 1)], [(1, 2)])
```

ここで、最初のインデックスは常に1（時間次元）です。合計で3つのバッチが得られます。最後のバッチはサイズ1のみです。なぜなら、*置換なしでサンプリング*を行うからです。また、[`DataLoader(::AbstractVector)`](@ref)のドキュメントも参照してください。
