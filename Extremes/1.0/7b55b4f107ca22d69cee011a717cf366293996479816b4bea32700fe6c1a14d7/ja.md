```
cint(..., confidencelevel::Real=.95)
```

信頼区間またはベイズ推定の場合の信用区間を計算します。

この関数は、モデルパラメータの信頼区間を取得するために、任意の [`AbstractFittedExtremeValueModel`](@ref) サブタイプに適用できます。また、リターンレベルの信頼区間を取得するために [`ReturnLevel`](@ref) タイプにも適用できます。

# 実装

区間を計算するために使用されるメソッドは、推定方法によって異なります。最尤推定の場合、信頼区間は近似パラメータ推定の共分散行列に基づくワルド近似を使用して計算されます。ベイズ推定の場合、リターン区間はMCMCサンプルに基づく最高の事後密度推定です。確率加重モーメント推定の場合、区間はブートストラップ手法を使用して計算されます。
