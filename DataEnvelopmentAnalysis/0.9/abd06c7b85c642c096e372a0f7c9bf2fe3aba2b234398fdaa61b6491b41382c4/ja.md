```
deaddf(X, Y; Gx, Gy)
```

入力 `X` と出力 `Y` のためのデータ包絡分析方向距離関数モデルを、方向 `Gx` と `Gy` を使用して計算します。

# 方向の指定:

方向 `Gx` と `Gy` は以下のシンボルのいずれかである必要があります。

  * `:Zeros`: ゼロを使用します。
  * `:Ones`: 一を使用します。
  * `:Observed`: 観測値を使用します。
  * `:Mean`: 列の平均を使用します。

また、希望する方向を持つベクトルまたは行列を提供することもできます。

# オプション引数

  * `rts=:CRS`: 定常規模の収益を選択します。変動規模の収益を選択するには `:VRS` を選択します。
  * `slack=true`: 入力と出力のスラックを計算します。
  * `Xref=X`: ユニットが評価される基準となる入力の参照セットを特定します。
  * `Yref=Y`: ユニットが評価される基準となる出力の参照セットを特定します。
  * `names`: 意思決定ユニットの名前を含む文字列のベクトルです。
