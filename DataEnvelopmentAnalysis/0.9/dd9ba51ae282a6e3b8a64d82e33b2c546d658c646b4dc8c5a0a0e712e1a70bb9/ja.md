```
deaprofit(X, Y, W, P; Gx, Gy)
```

データ包絡分析モデルを使用して、入力 `X`、出力 `Y`、入力の価格 `W`、および出力の価格 `P` に基づいて利益効率を計算します。

# 方向の指定:

方向 `Gx` と `Gy` は次のシンボルのいずれかである必要があります。

  * `:Zeros`: ゼロを使用します。
  * `:Ones`: 一を使用します。
  * `:Observed`: 観測値を使用します。
  * `:Mean`: 列の平均を使用します。
  * `:Monetary`: 利益の非効率性が貨幣的価値で表現されるように方向を使用します。

または、希望する方向を持つベクトルまたは行列を提供することもできます。

# オプション引数

  * `monetary=false`: 正規化された用語での分解。`true` の場合は貨幣的用語。
  * `names`: 意思決定単位の名前を含む文字列のベクトル。
