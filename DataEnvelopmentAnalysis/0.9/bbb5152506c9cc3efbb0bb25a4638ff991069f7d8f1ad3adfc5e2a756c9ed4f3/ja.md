```
malmluen(X, Y, B; Gx, Gy, Gb)
```

マルムキスト・ルエンバーガー生産性指数を、入力 `X`、良好な出力 `Y`、および悪い出力 `B` に対してデータ包絡分析を使用して計算します。方向は `Gx`、`Gy`、および `Gb` を使用します。

# 方向の指定:

方向 `Gx`、`Gy`、および `Gb` は次のシンボルのいずれかである必要があります。

  * `:Zeros`: ゼロを使用します。
  * `:Ones`: 一を使用します。
  * `:Observed`: 観測値を使用します。
  * `:Mean`: 列の平均を使用します。

# オプションの引数

  * `refperiod=:Geomean`: 技術変化の基準期間を選択します: `:Base`、`:Comparison` または `:Geomean`。
  * `names`: 意思決定単位の名前を含む文字列のベクター。
