```
Significance
```

Supertype used to test for significance in [`significant_transitions`](@ref). Changes that are statistically significant are "transitions".

Valid subtypes are:

  * [`SurrogatesSignificance`](@ref).
  * [`SigmaSignificance`](@ref).
  * [`QuantileSignificance`](@ref).
  * [`ThresholdSignificance`](@ref).
