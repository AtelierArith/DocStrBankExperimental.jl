```
inverse_iteration(A, x₀, μ₀; tol = 1e-4, maxiter_eig = 10, maxiter_solve = 2, τ = 0.1)
```

# 説明

逆反復法を使用して、初期固有値推定（μ₀, x₀）に対応する行列 A の固有対 (λ, x) を計算します。

# 引数

  * `A::Matrix` - 固有対を計算する行列。
  * `x₀::Vector` - 固有ベクトルの初期推定。
  * `μ₀::Real` - 固有値の初期推定。

# キーワード引数

  * `tol::Real` - 反復固有値ソルバーのためのコーシー基準の許容誤差。
  * `maxiter_eig::Int` - 反復固有値ソルバーの最大反復回数。
  * `maxiter_solve::Int` - 反復線形ソルバーの最大反復回数。
  * `τ::Real` - 不完全LU因子分解のドロップ許容誤差。

# 戻り値

  * `x::Vector` - 固有ベクトル。
  * `λ::Real` - 固有値。
