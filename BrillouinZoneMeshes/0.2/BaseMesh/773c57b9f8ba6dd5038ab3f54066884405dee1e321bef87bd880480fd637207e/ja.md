```
abstract type AbstractUniformMesh{T,DIM} <: AbstractMesh{T,DIM}
```

すべての均一メッシュの親タイプです。

均一メッシュのメッシュポイントは、平行四辺形の領域内に均等に分布していると仮定されます。均一メッシュは分数座標をサポートします。

この抽象タイプのすべての具体的な実装は、以下のフィールドを持つと仮定されます：

# 必須フィールド：

  * `origin`: 領域の原点（左下の点）
  * `shift`: メッシュの分数座標のシフト。これは、DFTで一般的に使用されるM-Pメッシュを再現するのに便利です。

および以下のメソッドが実装されていると仮定されます：

# 必須メソッド：

  * `lattice_vector`: 表現された領域の格子ベクトルを返す
  * `inv_lattice_vector`: 格子ベクトルの逆を返す
  * `cell_volume`: セルの体積を返す
