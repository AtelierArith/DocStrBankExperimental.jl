```
hc_label(matrix::AbstractMatrix{<:Real}, n::Int64; linkage::Symbol=:ward)
```

系統的クラスタリングから系統樹のグループに対する予測ラベルを取得します。

# 引数

  * `matrix`: N * N のペアワイズ距離行列。
  * `n`: クラスターの数。
  * `linkage`（デフォルトは `:ward`）: 使用するクラスタリンク関数。各イテレーションでどのクラスターがマージされるかに影響します：

      * `:single`: クラスターのメンバー間の最小距離を使用
      * `:average`: クラスターのメンバー間の平均距離を使用
      * `:complete`: メンバー間の最大距離を使用
      * `:ward`: 2つのクラスターをマージした後の、点とそのクラスター重心との平均二乗距離の増加
      * `:ward_presquared`: `:ward` と同じですが、`d` の距離がすでに二乗されていると仮定します。

# 出力

各木に対する予測ラベル（その木が属するクラスター）の長さ N の `Vector` オブジェクト。
