```julia
∂(I)

```

与えられた座標に沿った偏微分。

!!! NOTE     偏微分は現在**実験的**であり、十分にテストされていません。APIは事前の通知や非推奨なしにいつでも変更される可能性があります。

以下は同等であり、$\partial_1 \partial^2_2$を表します。すなわち、最初の軸に沿った最初の導関数と、2番目の軸に沿った2番目の偏微分です。

```@jldoctest
julia> ∂(1, 2)
∂(1, 2)

julia> ∂((1, 2))
∂(1, 2)
```

vararg形式のみが末尾のゼロを許可し、それは削除されます：

```@jldoctest
julia> ∂(1, 0)
∂(1)

julia> ∂((1, 0))
ERROR: ArgumentError: I ≡ () || last(I) ≠ 0 must hold.
```

導関数がない場合は空の形式を使用します：

```@jldoctest
julia> ∂()
∂()
```

`union`または`∪`を使用して導関数を組み合わせます：

```jldoctest
julia> ∂(1, 2) ∪ ∂(2, 1)
union(∂(2, 1), ∂(1, 2))
```
