## hexahedron

3D type finite element

### Constructor

```julia
Hexahedron(nod, nodof)
```

### Arguments

```julia
* nod::Int       : Number of nodes for finite element (8, 14, 20)
* nodof::Int     : Number of degrees of freedom per node
```
