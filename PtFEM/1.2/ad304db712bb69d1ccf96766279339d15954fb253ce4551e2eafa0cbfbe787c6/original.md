## Tetrahedron

3D type finite element

### Constructor

```julia
Tetrahedron(nod, nodof)
Tetrahedron(nodof)
```

### Arguments

```julia
* nod::Int       : Number of nodes for finite element (defaults to 4)
* nodof::Int     : Number of degrees of freedom per node
```
