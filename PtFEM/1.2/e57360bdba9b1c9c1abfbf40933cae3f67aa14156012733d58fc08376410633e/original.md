## Line (Interval)

1D type finite element

### Constructor

```julia
Line(nod, nodof)
Line(nodof)
```

### Arguments

```julia
* nod::Int       : Number of nodes for finite element, defaults to 2
* nodof::Int     : Number of degrees of freedom per node
```
