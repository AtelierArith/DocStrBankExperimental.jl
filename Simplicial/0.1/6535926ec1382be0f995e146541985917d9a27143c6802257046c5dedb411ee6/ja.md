```
simplexmap(C; check_complete=true, check_max=true)
```

整数 `m` と部分集合の配列 `T` を返します。これは `C == trunksmorphism(delta(m),T)` です。`C` が交差完備であるかどうかのチェックや、ユニークな最大要素があるかどうかのチェックは時間がかかる可能性があるため、キーワード引数を使用してこれらのチェックをスキップできます。

`check_complete = false` は `C` の交差完備性の計算をスキップし、`C` がすでに交差完備であると仮定します。この場合、返されるものに関しては保証はありません。

`check_max = false` は `C` にユニークな最大コードワードがあるかどうかのチェックをスキップします。再度、`C` にすでにユニークな最大コードワードがない場合、この場合に返されるものに関しては保証はありません。
