`cartan(type, rank [,bond])`

有限コクセター群をタイプとランクで表すカルタン行列。認識されるタイプは `:A, :B, :Bsym, :C, :D, :E, :F, :Fsym, :G, :Gsym, :I, :H` です。タイプ `:I` の場合、2つの生成子の間の結合を説明する第3の引数を指定する必要があります。`sym` タイプは、すべての根が同じ長さを持つ（非結晶的な）根系に対応し、結晶的な根系が持たない自己同型を提供し、「非常にねじれた」シェヴァレ群を定義することを可能にします。

```julia-repl
julia> cartan(:F,4)
4×4 Matrix{Int64}:
  2  -1   0   0
 -1   2  -1   0
  0  -2   2  -1
  0   0  -1   2

julia> cartan(:I,2,5)
2×2 Matrix{Cyc{Int64}}:
       2  ζ₅²+ζ₅³
 ζ₅²+ζ₅³        2

julia> cartan(:Bsym,2)
2×2 Matrix{Cyc{Int64}}:
   2  -√2
 -√2    2
```
