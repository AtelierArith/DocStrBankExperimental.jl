`coxeter_group(C)` または `coxgroup(C)`

`C` は整数、有理数、または実数の巡回数からなる正方行列である必要があります。この関数は、カルタン行列が `C` であるコクセター群を、次の生成子を持つ行列群として返します。`V` を次元 `size(C,1)` の実ベクトル空間とし、`eᵢ` を `V` の標準基底とします。生成子は反射 `sᵢ(eⱼ)=eⱼ-Cᵢⱼ eᵢ` です。

```julia-repl
julia> W=coxgroup([2 -2;-2 2])
coxeter_group([2 -2; -2 2])

julia> gens(W) # 行列生成子
2-element Vector{Matrix{Int64}}:
 [-1 0; 2 1]
 [1 2; 0 -1]
```

上記はアフィン・ワイル群 `Ã₁` を構成する方法の一つです。
