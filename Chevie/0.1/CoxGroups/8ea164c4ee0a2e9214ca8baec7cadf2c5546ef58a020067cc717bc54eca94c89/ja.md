`braid_relations(W)`

この関数は、反射群 `W` のブレイド群を表す関係を返します。これらは、生成子間の同じ長さのホモジニアス（両側が同じ長さ）な関係であり、`W` の生成反射と一対一対応しています。`W` の提示は、生成子の順序を指定する関係を追加することによって得られます。

```julia-repl
julia> W=complex_reflection_group(29)
G₂₉

julia> braid_relations(W)
7-element Vector{Tuple{Vector{Int64}, Vector{Int64}}}:
 ([1, 2, 1], [2, 1, 2])
 ([2, 4, 2], [4, 2, 4])
 ([3, 4, 3], [4, 3, 4])
 ([2, 3, 2, 3], [3, 2, 3, 2])
 ([1, 3], [3, 1])
 ([1, 4], [4, 1])
 ([4, 3, 2, 4, 3, 2], [3, 2, 4, 3, 2, 4])
```

各関係は、左側のインデックスに従った生成子の積が右側のインデックスに従った積に等しいことを指定するリストのペアとして表されます。`diagram` も参照してください。
