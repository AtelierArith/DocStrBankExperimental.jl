複素反射群のキャラクターと共役類。

有限複素反射群 `W` の `CharTable` は、`W` の不可約群への分解を使用して計算されます（`refltype` を参照）。各不可約群について、キャラクターテーブルは無限級数の再帰的な公式を使用して計算されるか、例外的なタイプのためにライブラリファイルからシステムに読み込まれます。したがって、非常に大きな群（例：E₈）に対してもキャラクターテーブルを迅速に取得できます。同様のことが共役類にも当てはまります。

不可約有限複素反射群の共役類と不可約キャラクターは、特定の組合せオブジェクトによって標準的にラベル付けされています。これらのラベル付けは、私たちが提供するテーブルで使用されます。クラスについては、無限級数の場合は分割または分割タプルであり、例外的なコクセター群の場合はカーターの許容図 [Carter1972](biblio.htm#Car72) です。他の原始的な複素反射群については、クラスを指定するために生成子の単語を使用します。キャラクターについては、無限級数の場合は再び分割または分割タプルであり、他のものについては、キャラクターの次数 `d` と、与えられたキャラクターを構成要素として含む反射表現の最小対称冪 `e` のペア `(d,e)` です（キャラクターの `b`-不変量）。この情報は、関数 `classinfo` と `charinfo` によって提供されます。キャラクターテーブルを表示すると、クラスとキャラクターの標準的なラベル付けが表示されます。

典型的な例は `coxgroup(:A,n)` で、対称群 `𝔖ₙ₊₁` であり、クラスとキャラクターは `n+1` の分割によってパラメータ化されます（これは `coxsym(n+1)` にも当てはまります）。

反射群にはフェイク次数があります（[`fakedegrees`](@ref) を参照）。その評価と次数は、`W` の各不可約キャラクターに対して2つの整数 `b,B` を与えます。特別群（有限コクセター群を含む）については、ヘッケ代数の一般的な次数の評価と次数が2つの整数 `a,A` を与えます（コクセター群については、詳細は [Carter1985, Ch.11](biblio.htm#Car85) を参照）。これらの整数は、切り捨てられた帰納法の操作にも使用されます。[`j_induction_table`](@ref) および [`J_induction_table`](@ref) を参照してください。

岩堀-ヘッケ代数および巡回ヘッケ代数にもキャラクターテーブルがあります。対応する章を参照してください。

各タイプについて、クラスとキャラクターのラベル付けの慣習を説明します。

タイプ `Aₙ` (`n≥0`)。この場合、`W ≅ 𝔖ₙ₊₁` です。クラスとキャラクターは `n+1` の分割によってラベル付けされています。クラスをラベル付けする分割は、そのクラスの要素のサイクル型です。`.classtext` の代表は、各部分に対応する単語の連結であり、部分 `i` の単語は `i-1` 個の連続する生成子の積です（前の部分で使用された最後の生成子より1つ高いところから始まります）。キャラクターをラベル付けする分割は、トリビアルキャラクターがその部分群から誘導され、そのキャラクターが重複度 `1` で含まれ、他のすべてのキャラクターがこの誘導されたキャラクターにおいてより高い `a`-値を持つようなヤング部分群の型を記述します。したがって、符号キャラクターは分割 `(1ⁿ⁺¹)` でラベル付けされ、トリビアルキャラクターは分割 `(n+1)` でラベル付けされます。`W` の反射表現のキャラクターは `(n,1)` でラベル付けされます。

タイプ `Bₙ` (`n≥2`)。この場合、`W=W(Bₙ)` は、次数 `2` の循環群と対称群 `𝔖ₙ` の輪積に同型です。したがって、クラスとキャラクターは、部分の合計が `n` に等しい分割のペアによってパラメータ化されます。クラスに対応するペアは、そのクラスの要素の符号付きサイクル型を記述します。私たちは、`(λ,μ)` がそのペアである場合、`λ` が正のサイクルに、`μ` が負のサイクルに対応するという慣習を使用します。したがって、`(1ⁿ,-)` と `(-,1ⁿ)` はそれぞれトリビアルクラスと最長要素のクラスをラベル付けします。

不可約キャラクターに対応するペアは、クリフォード理論を介して決定されます。次のようにします。`W(Bₙ)=N ⋊ 𝔖ₙ` の半直積分解があり、`N` は標準の `n` 次元 `𝔽₂ⁿ` ベクトル空間です。`n=a+b` となる `a,b ≥ 0` に対して、$η_{a,b}$ を `N` の不可約キャラクターとし、最初の `a` 個の標準基底ベクトルで値 `1` を取り、最後の `b` 個の標準基底ベクトルで値 `-1` を取ります。次に、$η_{a,b}$ の慣性部分群は $T_{a,b}=N.(𝔖_a × 𝔖_b)$ の形を持ち、$η_{a,b}$ を $T_{a,b}$ の不可約キャラクター $η̃_{a,b}$ に自明に拡張できます。`α` と `β` をそれぞれ `a` と `b` の分割とします。`𝔖_a` と `𝔖_b` の対応する不可約キャラクターのテンソル積を取り、これを $T_{a,b}$ の不可約キャラクターと見なします。このキャラクターを $η̃_{a,b}$ と掛け合わせて `W(Bₙ)` に誘導すると、`W(Bₙ)` の不可約キャラクター $χ= χ_{(α,β)}$ が得られます。これにより、不可約キャラクターとペアの分割との対応が定義されます。

例えば、ペア `((n),-)` はトリビアルキャラクターをラベル付けし、`(-,(1ⁿ))` は符号キャラクターをラベル付けします。自然な反射表現のキャラクターは `((n-1),(1))` でラベル付けされます。

タイプ `Dₙ` (`n≥4`)。この場合、`W=W(Dₙ)` はコクセター群 `W(Bₙ)` の指数 `2` の部分群として埋め込むことができます。`W(Bₙ)` のクラスと `W(Dₙ)` の交差は、空であるか、`W(Dₙ)` の単一のクラスであるか、または `W(Dₙ)` の2つのクラスに分かれます。これにより、前述のようにペアの分割 `(λ,μ)` によって `W(Dₙ)` のクラスをパラメータ化することができますが、`μ` の部分の数は偶数であり、`μ` が空であり、`λ` のすべての部分が偶数である場合にはこのタイプの2つのクラスがあります。この場合、`W(Dₙ)` の2つのクラスを `(λ,+)` と `(λ,-)` で表し、`(λ,+)` でラベル付けされたクラスには、`{s₁,s₃,…,sₙ}` の単語として書くことができる代表が含まれ、`(λ,-)` には `{s₂,s₃, …,sₙ}` の単語として書くことができる代表が含まれます。

クリフォード理論によれば、`W(Bₙ)` の不可約キャラクターの `W(Dₙ)` への制限は、不可約であるか、2つの不可約成分に分かれます。`(α,β)` を部分の合計が `n` に等しいペアの分割とします。`α!=β` の場合、`(α,β)` と `(β,α)` でラベル付けされた `W(Bₙ)` の不可約キャラクターの制限は不可約で等しいです。`α=β` の場合、`(α,α)` でラベル付けされたキャラクターの制限は、`(α,+)` と `(α,-)` という2つの不可約成分に分かれます。これは `n` が偶数である場合にのみ発生することに注意してください。記法を固定するために、[Stembridge1989](biblio.htm#Ste89) の結果を使用し、これらの2つのキャラクターの差の値をクラス `(λ,+)` の形式で、対称群 `𝔖_{n/2}` のキャラクター値に関して記述します。`λ'` を各部分を `2` で割った `n/2` の分割とします。次に、`χ_{(α,-)}-χ_{(α,+)}` の値は、クラス `(λ,+)` の要素に対して `2^{k(λ)}` 倍の `α` でラベル付けされた `𝔖_{n/2}` の不可約キャラクターの値によって与えられます。（ここで、`k(λ)` は `λ` の非ゼロ部分の数を示します。）

トリビアル、符号、自然反射キャラクターのラベルは、`W(Bₙ)` のものと同じです。これらのキャラクターは、`W(Bₙ)` の対応するキャラクターの制限です。

群 `G(d,1,n)`。これらは、次数 `d` の循環群と対称群 `𝔖ₙ` の輪積に同型です。したがって、クラスとキャラクターは、部分の合計が `n` に等しい `d`-タプルの分割によってパラメータ化されます。`d>2` の場合、クラスの代表として選ばれた単語は、ヘッケ代数のキャラクターを計算するためにラムとハルバソンが使用する単語と一致するように、わずかに異なる方法で計算されます。まず、`d` 個の分割の部分を1つの大きな分割にマージし、昇順にソートします。次に、`j` 番目の分割から来る部分 `i` に対して、単語 `(l+1…1… l+1)ʲ⁻¹l+2…l+i` が関連付けられます。ここで、`l` は前の部分を表現するために使用された最高の生成子です。

不可約キャラクターに対応する `d`-タプルは、`Bₙ` の場合と同様にクリフォード理論を介して決定されます。単位キャラクターは、1つの部分が `n` で、他の部分が空の最初の分割を持ちます。反射表現のキャラクターは、最初の2つの分割がそれぞれ `n-1` と `1` で、他の分割が空の部分を持ちます。

群 `G(de,e,n)`。これらは、`G(de,1,n)` の指数 `e` の正規部分群です。商は循環的で、`G(de,1,n)` の最初の生成子の像 `g` によって生成されます。クラスは、分割するクラスの成分に対して追加情報を持つ `G(de,e,n)` のクラスとしてパラメータ化されます。

[Hugues1985](biblio.htm#Hu85) によれば、`G(de,1,n)` のクラス `C` は、`de`-分割 $(S₀,…,S_{de-1})$ によってパラメータ化され、`G(de,e,n)` に含まれるのは、`e` が $∑ᵢ i ∑_{p∈ Sᵢ}p$ を割り切る場合です。`d` が `e` を割り切る最大の `d` のクラスに分割され、すべての `Sᵢ` の部分が空であり、`d` が `i` を割り切らない場合には `Sᵢ` が空です。`w` が `C` にある場合、`gⁱ w g⁻ⁱ` は `G(de,e,n)` のクラスの代表であり、`C` に出会います。これらは、クラス `C` のラベルに整数 `i` を追加することによって記述されます。

キャラクターはクリフォード理論によって記述されます。`G(de,1,n)` のキャラクターのラベルに `g` が作用します。`g` の作用は、`de`-タプルの分割を `d` で円形に置換します。キャラクターは、`G(de,e,n)` への制限に対して同じ制限を持ちます。その制限の不可約成分の数は、`g` の冪の下での安定化子の順序 `k` に等しいです。`G(de,e,n)` のキャラクターは、まず、制限がそれを含むキャラクターの最小の辞書順ラベルを選択することによってエンコードされます。次に、このラベルは、動機が `k` 回繰り返される周期的です。これらの動機の1つを表現し、制限のどの成分を選択するかを説明するために `E(k)ⁱ` を追加します。

タイプ `G₂` と `F₄`。キャラクター値の行列と不可約キャラクターの順序およびラベル付けは、[Carter1985, p.412/413](biblio.htm#Car85) とまったく同じです。タイプ `G₂` では、キャラクター `φ₁,₃'` は長い単純根に関連する反射で値 -1 を取ります。タイプ `F₄` では、キャラクター `φ₁,₁₂'`、`φ₂,₄'`、`φ₄,₇'`、`φ₈,₉'` および `φ₉,₆'` は、短い単純根に対応する `A₂` からの単位の誘導に現れます。ペア (`φ₂,₁₆'`、`φ₂,₄″`) および (`φ₈,₃'`、`φ₈,₉″`) は符号によってテンソル化されて関連しています。最後に、`φ₆,₆″` は反射表現の外積です。ただし、私たちは、長い根をダインキン図の左側に置いて、[Lusztig1985, (4.8) および (4.10)](biblio.htm#Lus85) の慣習に従います。

クラスはカーターの許容図 [Carter1972](biblio.htm#Car72) によってラベル付けされています。キャラクターはペア `(d,b)` でラベル付けされ、`d` は次数を、`b` は対応する `b`-不変量を示します。同じペア `(d,b)` を持つ複数のキャラクターがある場合、[Carter1985](biblio.htm#Car85) のように、プライムを付けます。

タイプ `E₆,E₇,E₈`。キャラクターテーブルは、ヘッケ代数のそれを特殊化することによって得られます。クラスはカーターの許容図 [Carter1972](biblio.htm#Car72) によってラベル付けされています。キャラクターはペア `(d,b)` でラベル付けされ、`d` は次数を、`b` は対応する `b`-不変量を示します。これらのタイプでは、キャラクターのユニークなラベル付けが得られます。

非結晶型タイプ `I₂(m)`、`H₃`、`H₄`。これらの場合、クラスに対する標準的なラベル付けはありません。私たちはそれらを簡約表現によってラベル付けします。

タイプ `H₃` の各キャラクターは、ペア `(d,b)` によって一意に決定され、`d` は次数、`b` は対応する `b`-不変量です。タイプ `H₄` では、対応するペアが同じであるキャラクターは2つだけです（次数 `30` のもの）。これらの2つのキャラクターは、フェイク次数によって区別されます：キャラクター `φ₃₀,₁₀'` はフェイク次数 `q¹⁰+q¹²+` の高次項を持ち、`φ₃₀,₁₀″` はフェイク次数 `q¹⁰+q¹⁴+` の高次項を持ちます。タイプ `H₄` のテーブルにあるキャラクターは、[Alvis and Lusztig1982](biblio.htm#AL82) と同じ順序で並べられています。

最後に、タイプ `I₂(m)` の次数 `2` のキャラクターは次のように順序付けられます。次数 `2` のキャラクターを提供する行列表現は次のように与えられます：$ρ_j : s₁s₂ ↦ \begin{pmatrix}\zeta_m^j&0\\0&\zeta_m^{-j}\end{pmatrix},  s₁↦\begin{pmatrix}0&1\\1&0\end{pmatrix},$ ここで `1 ≤ j ≤  ⌊(m-1)/2⌋` です。反射表現は `ρ₁` です。テーブルのキャラクターは、まず次数1のキャラクターをリストし、その後 `ρ₁,ρ₂,…` の順に並べられます。

原始的な複素反射群 `G₄` から `G₃₄`。群 `G₂₃=H₃`、`G₂₈=F₄`、`G₃₀=H₄` は例外的なコクセター群であり、上記で説明されました。他の群のキャラクターのラベルは、主にペア `(d,b)` で構成され、`d` は次数を、`b` は対応する `b`-不変量を示します。これは `G₄`、`G₁₂`、`G₂₂` および `G₂₄` に対して十分です。他の群については、同じ `(d,b)` 値を持つキャラクターのペアまたはトリプルがあります。これらは、`G₂₇`、`G₂₉`、`G₃₁`、`G₃₃` および `G₃₄` の [Malle2000](biblio.htm#Mal00) の慣習に従って区別されます：

  * `G₂₇` の場合：

キャラクター `φ₃,₅'` （および `φ₃,₂₀'`、`φ₈,₉″`）のフェイク次数は、`φ₃,₅″` （および `φ₃,₂₀″`、`φ₈,₉'`）のものよりも小さいです。キャラクター `φ₅,₁₅'` と `φ₅,₆'` は、最初と3番目の生成子によって生成されるタイプ `A₂` の部分群のトリビアルキャラクターから誘導されるときに重複度1で現れます（[Malle2000](biblio.htm#Mal00) では、`φ₅,₆″` がこの誘導に現れないと誤って主張されています；実際には重複度2で現れます）。

  * `G₂₉` の場合：

キャラクター `φ₆,₁₀‴` は `φ₄,₁` の外積であり、その複素共役は `φ₆,₁₀⁗` です。キャラクター `φ₁₅,₄″` は `φ₄,₁⊗φ₄,₃` に現れ、キャラクター `φ₁₅,₁₂″` は `φ₁₅,₄″` から符号キャラクターによってテンソル化されます。最後に、`φ₆,₁₀'` は、最初、2番目、4番目の生成子によって生成される標準的な部分群のトリビアルキャラクターから誘導されます。

  * `G₃₁` の場合：

キャラクター `φ₁₅,₈'`、`φ₁₅,₂₀'` および `φ₄₅,₈″` は `φ₄,₁⊗φ₂₀,₇` に現れ、キャラクター `φ₂₀,₁₃'` は `φ₂₀,₇` の複素共役です。キャラクター `φ₄₅,₁₂'` は `φ₄₅,₈'` の符号によってテンソル化されます。`φ₃₀,₁₀'` のフェイク次数の最大次数の2つの項は `q⁵⁰+q⁴⁶` であり、`φ₃₀,₁₀″` のそれは `q⁵⁰+2q⁴⁶` です。

  * `G₃₃` の場合：

キャラクター `φ₁₀,₈'` のフェイク次数の最大次数の2つの項は `q²⁸+q²⁶` であり、`φ₁₀,₈″` のそれは `q²⁸+q²⁴` です。キャラクター `φ₄₀,₅'` のフェイク次数の最大次数の項は `q³¹+q²⁹` であり、`φ₄₀,₅″` のそれは `q³¹+2q²⁹` です。キャラクター `φ₁₀,₁₇'` は `φ₁₀,₈'` の符号によってテンソル化され、キャラクター `φ₄₀,₁₄'` は `φ₄₀,₅'` の符号によってテンソル化されます。

  * `G₃₄` の場合：

キャラクター `φ₂₀,₃₃'` は `φ₆,₁⊗φ₁₅,₁₄` に現れます。キャラクター `φ₇₀,₉'` は有理数です。キャラクター `φ₇₀,₉″` は `φ₆,₁⊗φ₁₅,₁₄` に現れます。キャラクター `φ₇₀,₄₅'` は有理数です。キャラクター `φ₇₀,₄₅″` は `φ₇₀,₉″` の行列式キャラクターによってテンソル化されます。キャラクター `φ₅₆₀,₁₈'` は有理数です。キャラクター `φ₅₆₀,₁₈‴` は `φ₆,₁⊗φ₃₃₆,₁₇` に現れます。キャラクター `φ₂₈₀,₁₂'` は `φ₆,₁⊗φ₃₃₆,₁₇` に現れます。キャラクター `φ₂₈₀,₃₀″` は `φ₆,₁⊗φ₃₃₆,₁₇` に現れます。キャラクター `φ₅₄₀,₂₁'` は `φ₆,₁⊗φ₁₀₅,₂₀` に現れます。キャラクター `φ₁₀₅,₈'` は `φ₁₀₅,₄` の複素共役であり、キャラクター `φ₈₄₀,₁₃'` は `φ₈₄₀,₁₁` の複素共役です。キャラクター `φ₈₄₀,₂₃'` は `φ₈₄₀,₁₉` の複素共役です。最後に、キャラクター `φ₁₂₀,₂₁'` は、`G₃₄` の生成子のうち3番目を省略した標準的な部分群のトリビアルキャラクターから誘導されます。

群 `G₅` と `G₇` については、次の慣習を採用します。`G₅` の場合、[MalleRouquier2003](biblio.htm#MR03) および [BroueMalleMichel2014](biblio.htm#BMM14) のものと互換性があります。

  * `G₅` の場合：

`W=complex_reflection_group(5)` とし、生成子は `W(1)` と `W(2)` です。

キャラクター `φ₁,₄'`（および `φ₁,₁₂'`、`φ₂,₃'`）は `W(1)` で値 `1` を取ります（それぞれ `ζ₃`、`-ζ₃`）。キャラクター `φ₁,₈″` は `φ₁,₁₆` の複素共役であり、キャラクター `φ₁,₈'` は `φ₁,₄'` の複素共役です。キャラクター `φ₂,₅″` は `φ₂,₁` の複素共役であり、`φ₂,₅'` は `W(1)` で値 `-1` を取ります。キャラクター `φ₂,₇'` は `φ₂,₅'` の複素共役です。

  * `G₇` の場合：

`W=complex_reflection_group(7)` とし、生成子は `W(1)`、`W(2)`、`W(3)` です。

キャラクター `φ₁,₄'` と `φ₁,₁₀'` は `W(2)` で値 `1` を取ります。キャラクター `φ₁,₈″` は `φ₁,₁₆` の複素共役であり、キャラクター `φ₁,₈'` は `φ₁,₄'` の複素共役です。キャラクター `φ₁,₁₂'` と `φ₁,₁₈'` は `W(2)` で値 `ζ₃` を取ります。キャラクター `φ₁,₁₄″` は `φ₁,₂₂` の複素共役であり、キャラクター `φ₁,₁₄'` は `φ₁,₁₀'` の複素共役です。キャラクター `φ₂,₃'` は `W(2)` で値 `-ζ₃` を取り、`φ₂,₁₃'` は `W(2)` で値 `-1` を取ります。キャラクター `φ₂,₁₁″`、`φ₂,₅″`、`φ₂,₇‴` および `φ₂,₁` はガロア共役であり、キャラクター `φ₂,₇'`、`φ₂,₁₃'`、`φ₂,₁₁'` および `φ₂,₅'` も同様です。キャラクター `φ₂,₉'` は `φ₂,₁₅` の複素共役であり、キャラクター `φ₂,₉‴` は `φ₂,₃'` の複素共役です。

最後に、残りの群 `G₆`、`G₈` から `G₁₁`、`G₁₃` から `G₂₁`、`G₂₅`、`G₂₆`、`G₃₂` および `G₃₃` には、同じ値 `(d,b)` を持つキャラクターのペアしかありません。これらのキャラクターに均一にラベルを付けるために、次のルールを順に適用します：

  * 2つのキャラクターが異なるフェイク次数を持つ場合、フェイク次数が最小の方に `φ_{d,b}'` のラベルを付けます。
  * まだラベル付けされていないペアについて、2つのキャラクターのガロア軌道の中で、少なくとも1つのキャラクターがその `(d,b)`-不変量によって区別される場合、`φ_{d,b}'` のラベルを付けます。
  * まだラベル付けされていないペアについて、2つのキャラクターのガロア軌道の中で `(d,b)`-値の最小値が異なる場合、最小の最小値を持つキャラクターに `φ_{d,b}'` のラベルを付けます。
  * キャラクターに対して新しい不変量を定義します：すべてのペアの不可約キャラクター `χ` と `ψ` を考慮し、`φ` が `χ⊗ψ` において非ゼロの重複度 `m` で現れるように、`(d(χ),b(χ),d(ψ),b(ψ),m)` の最小（辞書順）リストを `t(φ)` と定義します。

まだラベル付けされていないペアについて、`t`-不変量が異なる場合、最小の `t`-不変量を持つキャラクターに `φ_{d,b}'` のラベルを付けます。

最後のルールを適用した後、考慮された群のすべてのペアにラベルが付けられます。得られたラベル付けは、`G₂₅`、`G₂₆`、`G₃₂` および `G₃₃` に対して [Malle2000](biblio.htm#Mal00) のものと互換性があり、`G₈` に対しては [MalleRouquier2003](biblio.htm#MR03) で説明されたものと互換性があります。

すべての群に対して、いくつかの例外を除いて、キャラクターのパラメータは非標準的な選択に依存しないことを強調すべきです。例外は、`e>1` の `G(de,e,n)` と、`G₅`、`G₇`、`G₂₇`、`G₂₈`、`G₂₉` および `G₃₄` であり、これらの群は反射の集合を保持する外部自己同型を持つため、これらの自己同型によって不変でないキャラクターの特定の生成子に対して特定の値を選択する必要があります。

クラスのラベル。例外的な複素反射群の場合、クラスのラベルは、生成子の積としてクラスの代表の分解を表し、追加の慣習として「z」は中心の生成子を表し、十分生成された群の場合「c」はコクセター要素を表します（最高反射次数の正則要素である生成子の積）。
