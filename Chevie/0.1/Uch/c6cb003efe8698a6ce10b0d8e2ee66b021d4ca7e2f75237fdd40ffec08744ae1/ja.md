`deligne_lusztig_lefschetz(h,m=0)` または `dllefschetz(h,m=0)`

ここで `h` はコクセター群 `W` またはコクセターコセット `Wϕ` に関連するヘッケ代数の要素であり、これは代数群 `𝐆` に関連しています。 [DigneMichel1985](biblio.htm#DM85) によれば、$g∈  𝐆^F$ の場合、要素 `wϕ∈Wϕ` に関連するデリーニュ＝ルスティグ多様体上の `Fᵐ` の不変点の数は、`m` が十分大きな整数 `d` で割り切れるとき、$∑_φ φ_{(qᵐ)}(T_wϕ)R_φ(g)$ の形を持ち、ここで `φ` は $Wϕ$ の不可約キャラクタを走り、$R_φ$ は対応するほぼキャラクタであり、$φ_{(qᵐ)}$ はパラメータ `qᵐ` を持つヘッケ代数 $H(Wϕ,qᵐ)$ のキャラクタ値です。この表現はデリーニュ＝ルスティグ多様体の*レフシェッツキャラクタ*と呼ばれます。`qᵐ` を不定元 `x` と見なすと、これは一般的なヘッケ代数 $H(Wϕ,x)$ のキャラクタ値の係数を持つユニポテントキャラクタの和として見ることができます。`m` が `d` に素でない場合、ユニポテントキャラクタに付随するフロベニウスの固有値を含むより複雑な式が適用されます。この関数は、第二のパラメータ `m≠0` が与えられたときにこの式を返します。

関数 'dllefschetz' はヘッケ要素を引数として取り、対応するレフシェッツキャラクタを返します。これは線形性によってヘッケ代数全体で定義されています。デリーニュ＝ルスティグ多様体に関連するさまざまな多様体のレフシェッツキャラクタは、ヘッケ代数のさまざまな要素でレフシェッツキャラクタを取ることによって得られます。

```julia-repl
julia> W=coxgroup(:A,2)
A₂

julia> H=hecke(W,Pol(:q))
hecke(A₂,q)

julia> T=Tbasis(H);

julia> dllefschetz(T(1,2))
[A₂]:<111>-q<21>+q²<3>

julia> dllefschetz((T(1)+T())*(T(2)+T()))
[A₂]:q<21>+(q²+2q+1)<3>
```

最後の行はコクセター要素のデリーニュ＝ルスティグ多様体のサメルソン＝ボットの非特異化のレフシェッツキャラクタを示しています。

次に、コセット（単位群に対応）を用いた例を示します。

```julia-repl
julia> H=hecke(spets(W,Perm(1,2)),Pol(:q)^2)
hecke(²A₂,q²)

julia> T=Tbasis(H);dllefschetz(T(1))
[²A₂]:-<11>-q<²A₂>+q²<2>
```

最後に、`dllefschetz(H::HeckeAlgebra,w,i=0)` という第二の形式があります。ここでの引数はヘッケ代数と要素 `w` です。これは、`w` の `H` の `CharTable` の列を知っているが、特別なヘッケ代数のキャラクタテーブルの他の列を知らないスぺツェスに使用できます。
