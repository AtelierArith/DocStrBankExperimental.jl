`Family(f [, charNumbers [, opt]])`

この関数は、2つの可能な方法で新しいファミリーを作成します。

最初のケースでは、`f`はChevieによって知られているファミリーを示す文字列です。例としては、3、4、5要素の対称群のDrinfeldダブルとして得られるファミリーを示す"S3"、"S4"、"S5"や、順序2の巡回群のDrinfeldダブルを示す"C2"があります。

2番目のケースでは、`f`はすでに`struct Family`です。

他の（オプションの）引数は、最初の引数で定義されたファミリーに情報を追加します。指定された場合、2番目の引数は`f.charNumbers`になります。指定された場合、3番目の引数`opt`は、結果のファミリーに追加されるキーを持つ`Dict`です。

`opt`にキー`signs`がある場合、これは'1'と'-1'のリストである必要があり、その後、フーリエ行列はそれらの符号の対角行列によって共役されます。これは、Spetsesで非可換次数の符号の選択に合わせて行列を調整するために使用されます。

```julia-repl
julia> Family("C2")
Family(C₂)
Drinfeld double D(ℤ/2)
┌──────┬────────────────────────────┐
│label │eigen                       │
├──────┼────────────────────────────┤
│(1,1) │    1 1//2  1//2  1//2  1//2│
│(g₂,1)│    1 1//2  1//2 -1//2 -1//2│
│(1,ε) │    1 1//2 -1//2  1//2 -1//2│
│(g₂,ε)│   -1 1//2 -1//2 -1//2  1//2│
└──────┴────────────────────────────┘

julia> Family("C2",4:7;signs=[1,-1,1,-1])
Family(C₂,4:7,signs=[1, -1, 1, -1])
Drinfeld double D(ℤ/2)
┌──────┬──────────────────────────────────┐
│label │eigen signs                       │
├──────┼──────────────────────────────────┤
│(1,1) │    1     1  1//2 -1//2 1//2 -1//2│
│(g₂,1)│    1    -1 -1//2  1//2 1//2 -1//2│
│(1,ε) │    1     1  1//2  1//2 1//2  1//2│
│(g₂,ε)│   -1    -1 -1//2 -1//2 1//2  1//2│
└──────┴──────────────────────────────────┘
```
