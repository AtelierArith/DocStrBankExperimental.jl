`Aut(F::NumberField,k::Union{Integer,Mod})`

*ガロア自己同型* `σₖ` は、サイクリック体 `CF(n)` の `n` 次根を `k` の冪に上げます。これは `k` が `n` と互いに素である場合に存在します。もし `F` が `CF(n)` の部分体であれば、ガロア群 `galois(CF(n))` における `F` の安定化子に対する `σₖ` の軌道の要素は、`F` への同じ制限を持ちます。`F` の自己同型は、この軌道の標準代表 `σₗ` によって表されます。これが `Aut(F,k)` の結果です。数 `k` は整数または `Mod(k,n)` として与えることができます。

```julia-repl
julia> F=NF(root(5))
NF(5,-1₅)

julia> s=Aut(F,3)
Aut(NF(5,-1₅),2₅)

julia> root(5)^s # s のサイクリック体への作用
Cyc{Int64}: -√5
```
