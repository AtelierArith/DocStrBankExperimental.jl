数体は `ℚ` の有限拡張です。現在扱えるのは、サイクリック体の部分体、つまりその要素が `Cyc` であるものです。これらはまた、`Gal(K/ℚ)` がアーベルである数体 `K` として特徴付けられます。例えば、`ℚ (√5)` はサイクリック体ではありませんが、サイクリック体 `ℚ (ζ₅)` に含まれる数体です。

数体のデフォルトコンストラクタは、いくつかの数を引数として受け取り、その引数を含む最小の数体を構築します。

```julia-repl
julia> F=NF(E(5)) # 完全なサイクリック体は CF として表示されます
CF(5)

julia> K=NF(root(5)) # 部分体
NF(5,-1₅)

julia> conductor(K) # K が CF(n) の部分体であるための最小の n
5

julia> E(5)+E(5,-1) in NF(root(5)) # 要素が部分体に含まれているかテスト
true
```

数体 `K` は、これを含む最小のサイクリック体 `F` の導体と、`F` のガロワ群における `K` の安定化子の生成元を与えることによって表示されます。上記の `NF(5,-1₅)` は、複素共役によって安定な `CF(5)` の部分体を表します。

```julia-repl
julia> elements(galois(F))
4-element Vector{Chevie.Nf.NFAut}:
 Aut(CF(5),1₅)
 Aut(CF(5),2₅)
 Aut(CF(5),-1₅)
 Aut(CF(5),-2₅)
```

`CF(5)` のガロワ群の要素 `-2₅` は、単位根の五乗根を -2 乗することによって作用します。したがって、`-1₅` は複素共役を表します。

```julia-repl
julia> NF(root(3),root(5)) # ここでは安定化子に 2 つの生成元が必要です
NF(60,-11₆₀,-1₆₀)
```
