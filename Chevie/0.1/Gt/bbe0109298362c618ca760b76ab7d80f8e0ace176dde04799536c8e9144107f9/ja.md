`ClassTypes(G[,p])`

`G` は有限還元群 $𝐆 ^F$ を表す根データまたはねじれ根データであり、`p` は素数である必要があります。この関数は、特性 `p` における `G` のクラス型を返します（`p` が省略された場合は良い特性で）。$𝐆 ^F$ の2つの要素は、その中心化群が共役であれば同じクラス型を持ちます。要素 `x` のジョルダン分解を `su` とすると、`x` のクラス型はその半単純部分 `s` のクラス型と、$C_𝐆 (s)$ における `u` の非可換クラスによって決まります。

関数 `ClassTypes` は現在、単純連結群に対してのみ実装されており、$C_𝐆 (s)$ は連結です。このセクションはやや実験的であり、将来的に変更される可能性があります。

`ClassTypes` は、半単純要素のクラス型のリストを含む `struct` を返します。これらは `subspets` によって表され、$C_𝐆 (s)$ の非可換クラスに関する追加情報を含みます。

いくつかの例を示しましょう：

```julia-repl
julia> t=ClassTypes(rootdatum(:sl,3))
ClassTypes(A₂,good characteristic)
┌──────────┬─────────┐
│C_G(s)    │ |C_G(s)|│
├──────────┼─────────┤
│A₂₍₎=Φ₁²  │      Φ₁²│
│A₂₍₎=Φ₁Φ₂ │     Φ₁Φ₂│
│A₂₍₎=Φ₃   │       Φ₃│
│A₂₍₁₎=A₁Φ₁│   qΦ₁²Φ₂│
│A₂        │q³Φ₁²Φ₂Φ₃│
└──────────┴─────────┘
```

デフォルトでは、半単純中心化群の型とその一般的な順序に関する情報のみが返されます。

```julia-rep1
julia> xdisplay(t;unip=true)
ClassTypes(A₂,good characteristic)
┌──────────┬───────────────┐
│C_G(s)    │    u |C_G(su)|│
├──────────┼───────────────┤
│A₂₍₎=Φ₁²  │    1       Φ₁²│
│A₂₍₎=Φ₁Φ₂ │    1      Φ₁Φ₂│
│A₂₍₎=Φ₃   │    1        Φ₃│
│A₂₍₁₎=A₁Φ₁│   11    qΦ₁²Φ₂│
│          │    2       qΦ₁│
│A₂        │  111 q³Φ₁²Φ₂Φ₃│
│          │   21      q³Φ₁│
│          │    3       3q²│
│          │ 3_ζ₃       3q²│
│          │3_ζ₃²       3q²│
└──────────┴───────────────┘
```

ここでは、中心化群に関する非可換クラスの情報を表示しました。

```julia-rep1
julia> xdisplay(t;nClasses=true)
ClassTypes(A₂,good characteristic)
┌──────────┬─────────────────────────┐
│C_G(s)    │       nClasses  |C_G(s)|│
├──────────┼─────────────────────────┤
│A₂₍₎=Φ₁²  │(q²-5q+2q₃+4)/6       Φ₁²│
│A₂₍₎=Φ₁Φ₂ │       (q²-q)/2      Φ₁Φ₂│
│A₂₍₎=Φ₃   │  (q²+q-q₃+1)/3        Φ₃│
│A₂₍₁₎=A₁Φ₁│       (q-q₃-1)    qΦ₁²Φ₂│
│A₂        │             q₃ q³Φ₁²Φ₂Φ₃│
└──────────┴─────────────────────────┘
```

ここでは、与えられた型を持つ半単純共役類がいくつあるかに関する情報を追加しました。一般的に、答えは `gcd(q-1,a)` を表す形の変数 `qₐ` を含みます。

最後に、悪い特性の例を示します：

```julia-rep1
julia> t=ClassTypes(coxgroup(:G,2),2);xdisplay(t;nClasses=true)
ClassTypes(G₂,char. 2)
ClassTypes(G₂,char. 2)
┌──────────┬──────────────────────────────┐
│C_G(s)    │         nClasses     |C_G(s)|│
├──────────┼──────────────────────────────┤
│G₂₍₎=Φ₁²  │(q²-8q+2q₃+10)/12          Φ₁²│
│G₂₍₎=Φ₁Φ₂ │        (q²-2q)/4         Φ₁Φ₂│
│G₂₍₎=Φ₁Φ₂ │        (q²-2q)/4         Φ₁Φ₂│
│G₂₍₎=Φ₆   │    (q²-q-q₃+1)/6           Φ₆│
│G₂₍₎=Φ₃   │    (q²+q-q₃+1)/6           Φ₃│
│G₂₍₎=Φ₂²  │ (q²-4q+2q₃-2)/12          Φ₂²│
│G₂₍₁₎=A₁Φ₁│       (q-q₃-1)/2       qΦ₁²Φ₂│
│G₂₍₁₎=A₁Φ₂│       (q-q₃+1)/2       qΦ₁Φ₂²│
│G₂₍₂₎=Ã₁Φ₁│          (q-2)/2       qΦ₁²Φ₂│
```

`q` が `2` の冪であり、`q≡2 (mod 3)` である場合、すなわち `q₃=1` のとき、いくつかのクラス型が存在しないことに注意します。特定の値を `q₃` に与えることで何が起こるかを見てみましょう：
