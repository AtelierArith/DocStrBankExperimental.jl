`reflectionMatrix(root,  coroot)` は、与えられたルートとコルートに対する反射の行列です。

（複素）反射は、複素数の部分体上のベクトル空間の線形群 `GL(V)` の有限順序要素 `s` であり、その不変点はハイパープレーン `H` を形成します（`s` の *反射ハイパープレーン*）。非自明な固有値 `ζ`（単位根の一つ）に対する固有ベクトル `r` は、`s` の *ルート* と呼ばれます。ハイパープレーン `H` を定義する線形形式 `rᵛ` を選び、`rᵛ(r)=1-ζ` となるようにすると（`s` の *コルート*）、`s` は `x↦ x-rᵛ(x)r` で与えられます。

`s` を指定する方法の一つは、ルートとコルートを与えることであり、これらは `s` によってスカラーの乗算を除いて一意に決定されます。関数 `reflectionMatrix` は、`V` の標準基底における対応する反射の行列を与えます。ここで、`root` と `coroot` は `V` と `Vᵛ` の標準基底で与えられたベクトルであり、ペアリング `rᵛ(r)` は `transpose(root)*coroot` として得られます。

```
julia> r=reflectionMatrix([1,0,0],[2,-1,0])
3×3 Matrix{Int64}:
 -1  0  0
  1  1  0
  0  0  1

julia> r==reflrep(coxgroup(:A,3),1)
true

julia> r*[2,-1,0]
3-element Vector{Int64}:
 -2
  1
  0

julia> [1 0 0]*r
1×3 Matrix{Int64}:
 -1  0  0
```

最後の行からわかるように、Julia では行列は行ベクトルのベクトル空間 `V` に対して右から作用し、列ベクトルの双対空間 `Vᵛ` に対して左から作用します。
