`reflection_representation(W::ComplexReflectionGroup,w)` または `reflrep(W,w)`

`V` を `W` が反射群として作用する空間とし、`w∈ W` を根の置換として表現します。関数 `reflrep` は、`V` の要素に対して (*右から*) 行ベクトルとして見たときの `w` の作用を表す行列を返します。これは、コルートの直交に対して自明に作用し、単純な根に対して `w` と同じ効果を持つ `V` の線形変換です。この関数は、根を安定化する `GL(V)` の要素によって誘導される根の置換に対しても一般的に意味を持ちます（したがって特に `W` を正規化します）；したがって、反射コセットに対しても機能します。コセット `Wσ` に対応する `rootdatum` に対して、`Wσ` が `X(𝐓)` に対して作用することが得られます。

```julia-repl
julia> W=reflection_subgroup(rootdatum("E7sc"),1:6)
E₇₍₁₂₃₄₅₆₎=E₆Φ₁

julia> reflrep(W,longest(W))
7×7 Matrix{Int64}:
  0   0   0   0   0  -1  2
  0  -1   0   0   0   0  2
  0   0   0   0  -1   0  3
  0   0   0  -1   0   0  4
  0   0  -1   0   0   0  3
 -1   0   0   0   0   0  2
  0   0   0   0   0   0  1
```
