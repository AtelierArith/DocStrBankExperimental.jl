`cartan(W::PermRootGroup)`    `W` のカルタン行列。

`s₁,…,sₙ` を反射とし、それに関連する根 `rᵢ` とコルート `cᵢ` を考えます。エントリ `Cᵢ,ⱼ=cᵢ(rⱼ)` を持つ行列 `C` は、`s₁,…,sₙ` の *カルタン行列* と呼ばれます。反射はスカラーに対して根とコルートを決定するため、`C` は対角行列による共役を除いて `s₁,…,sₙ` によって一意に決まります。

`s₁,…,sₙ` が反射群 `W` を生成する場合、対角行列による共役を除いて `C` は `W` の反射表現の不変量です。`C` が可逆であれば、この行列はこの表現を決定します。なぜなら、その場合 `rᵢ` は基底を形成し、行列 `sᵢ` は単位行列と異なるのは `i` 行目だけであり、そこでは `C` の対応する行が引かれています。

一般に `cartan(W)==simplecoroots(W)*permutedims(simpleroots(W))` です。

```julia-repl
julia> W=coxgroup(:A,3)
A₃

julia> cartan(W)
3×3 Matrix{Int64}:
  2  -1   0
 -1   2  -1
  0  -1   2
```
