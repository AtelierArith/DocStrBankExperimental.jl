```
plot_bolt_pattern(points; A = 1, udf_pivot = false)
```

ボルトパターンと重心をプロットします。

`points` は x および y 座標のタプル ([x], [y]) です。 `points` は関数 [`circle`](@ref) および [`rectangle`](@ref) によって生成される場合があります。 `A` はボルトの応力面積で、すべてのボルトに対して単一の値として入力するか、各ボルトに対して異なる値（すなわちベクトル）を入力します。 `udf_pivot` は計算される場合は false で、特定のピボットポイントを手動で入力する場合は [x,y] として入力されます。単位は `Unitful` パッケージに準拠する必要があります。
