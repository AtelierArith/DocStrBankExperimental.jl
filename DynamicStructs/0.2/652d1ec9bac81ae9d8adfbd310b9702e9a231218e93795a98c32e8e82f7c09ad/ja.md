```
@dynamic [mutable] struct ... end
```

動的構造体を定義します：

  * 動的構造体のインスタンスは、実行時に追加/削除できる動的プロパティを持っています。これらのプロパティは、パフォーマンスの観点から構造体の `Any` 型フィールドに似ています。
  * フィールドは静的に型付けされており、それにアクセスすることは構造体と同様にコンパイルされるため、パフォーマンスに大きな影響はないはずです。
  * マクロは、遅延初期化を伴う動的プロパティを格納するための隠れたフィールドを追加します。つまり、基盤となるストレージは最初の動的プロパティが追加されるまで割り当てられません。
  * 動的構造体のデフォルトコンストラクタは、動的プロパティのためのキーワード引数を受け入れ、これを反映する `Base.show` メソッドがあります。これらは、カスタムコンストラクタなどの非フィールド式がブロックに存在しない場合にのみ存在します。
