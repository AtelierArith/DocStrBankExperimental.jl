```
(s::Spectrum)(ω::Real,usteady,ps=[];wtol=0)
```

[`Spectrum`](@ref) のインスタンス `s` から、周波数 `ω` におけるスペクトルパワー密度を実際に計算します。方程式 `x=inv(A)*b` を数値的に解きます。ここで `x` はフーリエ変換された相関関数を含むベクトルであり、すなわちスペクトルは `real(x[1])` によって与えられます。`A` と `b` はそれぞれ、スペクトルを得るために解く必要がある線形方程式系を記述する行列とベクトルです。許容誤差 `wtol=0` は、周波数がゼロとして扱われる範囲を指定します。すなわち、`abs(ω) <= wtol` の場合、発散を避けるために `1/(im*ω)` に比例する項は無視されます。
