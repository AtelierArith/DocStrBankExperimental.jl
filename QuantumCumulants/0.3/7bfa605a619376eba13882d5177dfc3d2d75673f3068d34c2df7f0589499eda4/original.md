```
scaleME(me::IndexedMeanfieldNoiseEquations)
```

Function, that evaluates a given [`IndexedMeanfieldNoiseEquations`](@ref) entity and returns again equations, where indices have been inserted and sums evaluated, regarding the same relations, as done when calculating with oparators using a [`ClusterSpace`](@ref).

# Arguments

*`me::IndexedMeanfieldNoiseEquations`: A [`IndexedMeanfieldNoiseEquations`](@ref) entity, which shall be evaluated.
