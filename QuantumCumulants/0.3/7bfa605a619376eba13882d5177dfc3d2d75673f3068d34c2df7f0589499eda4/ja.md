```
scaleME(me::IndexedMeanfieldNoiseEquations)
```

関数で、与えられた [`IndexedMeanfieldNoiseEquations`](@ref) エンティティを評価し、インデックスが挿入され、合計が評価された方程式を再び返します。これは、[`ClusterSpace`](@ref) を使用して演算を行う際に行われるのと同じ関係に基づいています。

# 引数

*`me::IndexedMeanfieldNoiseEquations`: 評価されるべき [`IndexedMeanfieldNoiseEquations`](@ref) エンティティ。
