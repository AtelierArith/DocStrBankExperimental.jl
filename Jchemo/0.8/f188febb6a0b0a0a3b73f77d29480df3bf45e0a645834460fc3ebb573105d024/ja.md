```
segmkf(n::Int, K::Int; rep = 1)
segmkf(group::Vector, K::Int; rep = 1)
```

K分割交差検証のための観測値のセグメントを構築します。

  * `n` : データセット内の観測値の総数。サンプリングは1:nで実装されています。
  * `group` : 観測値のブロックを定義するベクトル(n)。
  * `K` : `n`の観測値を分割するフォールド(セグメント)の数。

キーワード引数:

  * `rep` : サンプリングの繰り返し回数。

各繰り返しに対して、関数は`n`の観測値を`K`のセグメントに分割し、K分割交差検証に使用できるようにします。

`group`が使用される場合（長さ`n`のベクトルである必要があります）、関数は観測値の代わりに全体のグループ（= ブロック）をサンプリングします。このようなブロックサンプリングは、データがブロックによって構造化されている場合や、予測する応答がブロック内で相関している場合に必要です。これにより、一般化誤差の過小評価を防ぎます。

関数は`rep`要素のリスト（ベクトル）を返します。リストの各要素には`K`のセグメント（= `K`のベクトル）が含まれています。各セグメントには、サンプリングされた観測値のインデックス（1:`n`内の位置）が含まれています。

## 例

```julia
using Jchemo 

n = 10 ; K = 3
rep = 4 
segm = segmkf(n, K; rep)
i = 1 
segm[i]
segm[i][1]

n = 10 
group = ["A", "B", "C", "D", "E", "A", "B", "C", "D", "E"]    # 観測値のブロック
tab(group) 
K = 3 ; rep = 4 
segm = segmkf(group, K; rep)
i = 1 
segm[i]
segm[i][1]
group[segm[i][1]]
group[segm[i][2]]
group[segm[i][3]]
```
