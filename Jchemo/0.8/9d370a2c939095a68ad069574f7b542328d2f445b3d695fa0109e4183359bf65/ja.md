```
samprand(n::Int, k::Int; replace = false)
```

トレーニングセットとテストセットをランダムサンプリングによって構築します。

  * `n` : 観測の総数。
  * `k` : サンプリングするテスト観測の数。

キーワード引数:

  * `replace` : ブール値。`false` の場合、サンプリングは重複なしで行われます。

2つの出力が返されます（データの行インデックス）:

  * `train` (`n` - `k`),
  * `test` (`k`)。

出力 `test` は `1:n` の範囲内でランダムサンプリングによって構築されます。

## 例

```julia
using Jchemo

n = 10
samprand(n, 4)
```
