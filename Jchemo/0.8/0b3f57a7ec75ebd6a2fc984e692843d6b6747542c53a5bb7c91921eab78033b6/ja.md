```
sampsys(y, k::Int)
```

定量変数に対する系統的サンプリングによってトレーニングセットとテストセットを構築します。

  * `y` : サンプリングする定量変数 (n)。
  * `k` : サンプリングするテスト観測の数。 2以上でなければなりません。

2つの出力が返されます (= データの行インデックス):

  * `train` (n - `k`),
  * `test` (`k`)。

出力 `test` は `y` の観測値のランクに対する系統的サンプリングによって構築されます。例えば、`k` / n ~ .3 の場合、ソートされた `y` の3つの観測値のうち1つが選択されます。

出力 `test` には常に `y` の最小値と最大値のインデックスが含まれます。

## 例

```julia
using Jchemo 

y = rand(7)
[y sort(y)]
res = sampsys(y, 3)
sort(y[res.test])
```
