```
matW(X, y, weights::Weight)
```

クラス内共分散行列。

  * `X` : Xデータ (n, p)。
  * `y` : クラスメンバーシップを定義するベクトル (n)。
  * `weights` : 観測値の重み (n)。タイプ `Weight` でなければなりません (例えば、関数 `mweight` を参照)。

`X` の (非修正) クラス内およびプール共分散行列 (それぞれ出力 `Wi` と `W`) を計算します。

クラス i に観測値が1つしか含まれていない場合、Wi は次のように計算されます：

  * `covm(`X`,`weights`)`。

例については、関数 `matB` を参照してください。
