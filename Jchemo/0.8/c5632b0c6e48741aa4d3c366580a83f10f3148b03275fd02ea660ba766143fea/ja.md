```
xresid(object, X; nlv = nothing)
xresid!(object, X::Matrix; nlv = nothing)
```

双線形モデル（例：PCA）からの残差行列。

  * `object` : フィッティングされたモデル。
  * `X` : モデルから近似される新しいXデータ。モデル`object`をフィットさせるために使用されたXデータと同じスケールでなければなりません。すなわち、センタリングおよび最終的なスケーリングの前です。

キーワード引数：

  * `nlv` : 考慮すべき成分の数（PCまたはLV）。`nothing`の場合、最大成分数です。

残差行列を計算します：

  * E = `X` - X_fit

ここで、X_fitは関数`xfit`によって返されるフィッティングされたXです。例については`xfit`を参照してください。  ```
