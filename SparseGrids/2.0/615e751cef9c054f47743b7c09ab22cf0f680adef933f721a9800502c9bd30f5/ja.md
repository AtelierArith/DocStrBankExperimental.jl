```
sparsegrid( D::Int, k::Int, f::Function=gausshermite; sym::Bool=true )
```

スパースグリッドノードと関連する重みの計算

  * `D` : 被積分関数の次元
  * `k` : 数値積分法の次数
  * `f` : 整数入力に対して1次元のノードと重みを生成する関数 – その順序で
  * `sym` : ノードを対称化するかどうかを決定するブール変数

ノードが対称であるべき場合（ガウス・エルミート法のように）、スパースセットの和において重複して出現するノードを正しく特定するために、そのようにする必要があります。
