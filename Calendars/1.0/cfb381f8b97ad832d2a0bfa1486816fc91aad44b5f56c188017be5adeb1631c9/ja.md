```julia
ConvertDate(date::CDate, calendar::DPart, string::Bool, show::Bool)
```

与えられた日付をカレンダー `calendar` の日付に変換します。カレンダーの許可される名前は CDate のドキュメントに記載されています。

オプションのパラメータ `string` が `true` の場合、関数は日付を文字列として返します。そうでない場合は整数のタプルとして返します。`string` はデフォルトで `false` です。

オプションのパラメータ `show` が 'true' に設定されている場合、両方の日付が印刷されます。`show` はデフォルトで 'false' です。

例えば：

```julia
julia> ConvertDate(CE, 1756, 1, 27, AM) 
```

また、次のように書くこともできます。

```julia
julia> ConvertDate("Gregorian", 1756, 1, 27, "Hebrew") 
```

これはグレゴリオ日（1756, 1, 27）をヘブライ日（5516, 11, 25）に変換します。デフォルトではカレンダー表現 (4, 5516, 11, 25) が返されます。

`string` が true の場合、文字列 "AM-5516-11-25" が返されます。`show` が 'true' の場合、次の行が印刷されます：

```julia
    "CE-1756-01-27 -> AM-5516-11-25"
```

エラーが発生した場合は (0, 0, 0, 0) （無効な日付を表す）が返されます。
