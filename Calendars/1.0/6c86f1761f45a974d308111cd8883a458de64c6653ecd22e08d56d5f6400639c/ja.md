```julia
Duration(a::CDate, b::CDate, show=false)
```

このdurationは、最初の日付`a`と2番目の日付`b`の間の日数を示します。最初の日付は含まれますが、2番目の日付は含まれません。したがって、これは右半開区間を表します。開始日と終了日は異なるカレンダーで指定できます。

オプションのパラメータ`show`が'true'に設定されている場合、両方の日付が印刷されます。`show`はデフォルトで'false'です。

```julia
julia> Duration((CE, 2022, 1, 1), (ID, 2022, 1, 1), true)
```

驚くべきことに、これらの日付は2日間隔です。

```julia
    CE-2022-01-01 <-> ID-2022-01-01 -> Duration 2
```
