```
PNTerm{T,PNOrder,c⁻¹Exponent}
```

このオブジェクトは、PNExpansionの単一の項を表します。 それには単一のフィールド `coeff` があり、これは項の係数です。 型パラメータ `T` は係数の型です。 型パラメータ `PNOrder` は、PN展開のPN次数を表す半整数（[`PNSystem`](@ref)と同様）です。 そして、型パラメータ `c⁻¹Exponent` は、PN展開パラメータ $1/c$ の指数を表す整数です。

`PNTerm` はスカラーで乗算および除算でき、整数で累乗して別の `PNTerm` を生成できます。 また、他の `PNTerm` に加算して `PNExpansion` を生成することもできます。

`PNTerm` または `PNExpansion` を定義する簡単な方法は、PN展開パラメータを定義することです。

```julia
c = PNExpansionParameter(pnsystem)
```

そして、それを自然に数式で使用します。

```julia
e = 1 + (v/c)^2 * (-ν/12 - 3//4) + (v/c)^4 * (-ν^2/24 + 19ν/8 - 27//8)
```

望ましい `PNOrder` よりも高い指数は自動的にゼロに設定されます。

便利な事実：

  * `v` の次数は `1/c`
  * `x` の次数は `1/c^2`
  * `γ` の次数は `1/c^2`
  * `1/r` の次数は `1/c^2`
