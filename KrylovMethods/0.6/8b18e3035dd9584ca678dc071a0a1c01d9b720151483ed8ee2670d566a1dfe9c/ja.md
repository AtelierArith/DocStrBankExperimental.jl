x,flag,err,iter,resvec = blockFGMRES(A,b,restrt,tol=1e-2,maxIter=100,M=1,x=[],out=0)

ブロック（フレキシブル）一般化最小残差（（F）GMRES(m)）法をA*x = bに適用したものです。

これはブロックGMRESの「右前処理」バージョンです：A*M^{-1}Mx = b。

入力：

A       - A*xを計算する関数 b       - 右辺行列（ベクトルのセット） restrt  - 再起動間の反復回数 tol     - 誤差許容値 maxIter - 最大反復回数 M       - 前処理器、M\xを計算する関数 x       - 初期推定値 out     - 出力フラグ（0 : エラーのみ、1 : 最終ステータス、2: 各反復のエラー）

出力：

x       - 近似解 flag    - 終了フラグ（  0 : 望ましい許容値達成、 	-1 : 収束せずにmaxIterに達した 	-9 : 右辺がゼロだった） 	err     - 相対残差のノルム、すなわち、ノルム(A*x-b)/ノルム(b) 	iter    - 反復回数 	resvec  - 各反復における相対残差のノルム

```
前処理器M(r)は行列のコピーを返さなければなりません。rのメモリを再利用することはできません。
```
