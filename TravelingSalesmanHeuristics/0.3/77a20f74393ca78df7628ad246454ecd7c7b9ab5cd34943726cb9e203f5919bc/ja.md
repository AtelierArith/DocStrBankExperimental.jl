```
nearest_neighbor(distmat)
```

近似的にTSPを最近傍法を用いて解決します。`distmat`は、`distmat[i,j]`が都市`i`から都市`j`までの距離を表す正方形の実行列です。この行列は対称である必要はなく、負の値を含むことができます。タプル`(path, pathcost)`を返します。

# オプションのキーワード引数:

  * `firstcity::Union{Int, Nothing}`: パスを開始する都市を指定します。`nothing`を渡すか、値を指定しない場合はランダム選択に対応します。
  * `closepath::Bool = true`: 最後に訪れた都市から最初の都市への弧をコスト計算に含めるかどうか。trueの場合、最初の都市はパスの最初と最後に現れます。
  * `do2opt::Bool = true`: 2-optスイッチによって見つかったパスを洗練するかどうか（平面ユークリッドの場合のパスの交差を取り除くことに対応します）。
