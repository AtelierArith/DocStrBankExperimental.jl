```
cheapest_insertion(distmat::Matrix, initpath::AbstractArray{Int})
```

距離行列と初期パスが与えられたとき、最も安い挿入を繰り返すことでツアーを完成させます。タプル `(path, cost)` を返します。初期パスの長さは少なくとも2でなければなりませんが、単に `[i, i]` という形で、ある都市インデックス `i` から始めるループとして使用することができます。

!!! note
    挿入は常に現在のパスの内部で行われるため、このヒューリスティックは非閉じたTSPパスにも使用できます。


現在の実装は単純な $n^3$ アルゴリズムです。
