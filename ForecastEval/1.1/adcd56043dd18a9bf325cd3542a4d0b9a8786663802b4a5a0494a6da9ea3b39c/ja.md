```
MCSBoot(data ; alpha, basecaseindex, kwargs...)
```

依存ブートストラップを使用してMCSテストを行うためのメソッドタイプ。関連するキーワード引数は次のとおりです：

```
alpha=0.05 <- テストで使用する信頼水準

basecaseindex=1 <- MCSには自然なベースケースがありません。ただし、入力データは損失の行列であり、損失の差分ではなく、メソッド自体はすべての可能な損失差分の組み合わせを考慮します。これは、ブートストラップの最適なブロック長を決定するためにチェックするにはあまりにも多すぎるため、適切なブロック長を自動的に推定したいユーザーは、ベースケースに対するすべての損失差分の最適なブロック長を計算する目的で、仮想的なベースケース変数インデックスを指定できます。
blocklength=0.0 <- ブートストラップで使用するブロック長。0.0のデフォルトは、データから最適にブロック長が推定されることを意味し、依存ブートストラップパッケージによって最も適切と見なされる方法（通常はPatton、Politis、White（2009）の選択手順）を使用します。

numresample=1000 <- ブートストラップ時に使用するリサンプルの数。

bootmethod=:stationary <- 使用するブートストラップの方法論で、デフォルトはPolitisとRomano（1993）の定常ブートストラップです。
```

注意：BootInputコンストラクタの有効なキーワード引数に関する詳細は、DependentBootstrapのドキュメントを参照してください。
