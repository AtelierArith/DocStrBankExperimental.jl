```
heisenbergtrottercircuit(nqubits::Integer, nlayers::Integer; topology=nothing)
```

ハイゼンベルクハミルトニアンのトロッター化に対応する回路を作成します。トポロジーは、キュービットインデックスのペアのリストとして指定できます。トポロジーが指定されていない場合、ブリックレイヤートポロジーが使用されます。ゲートは、すべてのXX、次にすべてのYY、次にすべてのZZのレイヤーとしてではなく、XX-YY-ZZゲートのレイヤーとして適用されることに注意してください。ブリックレイヤートポロジーでは、これらは同等です。
