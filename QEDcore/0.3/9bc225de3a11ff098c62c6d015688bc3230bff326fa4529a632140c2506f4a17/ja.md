```
AbstractCoordinateSet{N}
```

`N` 要素を持つ座標のセットを表す抽象型です。

`AbstractCoordinateSet` のサブタイプは、散乱イベントのための位相空間パラメータ化など、物理プロセスを記述するために使用される特定の座標系やレイアウトを定義するために使用されます。パラメータ `N` は、セット内の座標の数を指定し、`N` は単変量座標（例：エネルギー、急速）には1、より複雑なシステムにはそれ以上の値を取ることができます。

# 型パラメータ

  * `N`: セット内の座標の数。

# 意図された使用法

`AbstractCoordinateSet{N}` のサブタイプは、特定の座標系の構造と動作を実装します。`coordinate_names` や `coordinate_name` のような関数は、セット内の座標の人間が読み取れる名前を取得するために使用されます。
