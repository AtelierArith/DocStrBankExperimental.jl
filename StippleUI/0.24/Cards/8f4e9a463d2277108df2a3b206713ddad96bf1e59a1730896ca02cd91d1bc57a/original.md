```
card_1
```

Premade card component with many options. This is a good starting point for creating your own card component. Allows for:

  * showing and hidding of avatar, media, separator.
  * customizing avatar, media, title, subtitle, content, buttons, menu.
  * controlling the content of the components with buttons*component, menu*component, media_component, content.
  * customizing the props of the components with card*props, media*props, title*props, subtitle*props, content*props, separator*props.

# Example

card*1(; show*avatar = false, title = "Welcome to Genie", subtitle = "The best Julia Framework", content = "",             buttons*component = nothing, menu*component = nothing, show_separator = false)
