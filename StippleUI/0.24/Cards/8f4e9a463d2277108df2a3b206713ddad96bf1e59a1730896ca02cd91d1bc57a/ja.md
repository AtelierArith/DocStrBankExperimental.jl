```
card_1
```

多くのオプションを持つプレメイドカードコンポーネント。これは、自分自身のカードコンポーネントを作成するための良い出発点です。以下を可能にします：

  * アバター、メディア、セパレーターの表示と非表示。
  * アバター、メディア、タイトル、サブタイトル、コンテンツ、ボタン、メニューのカスタマイズ。
  * ボタン*コンポーネント、メニュー*コンポーネント、メディアコンポーネント、コンテンツでコンポーネントの内容を制御。
  * card*props、media*props、title*props、subtitle*props、content*props、separator*propsでコンポーネントのプロパティをカスタマイズ。

# 例

card*1(; show*avatar = false, title = "Welcome to Genie", subtitle = "The best Julia Framework", content = "",             buttons*component = nothing, menu*component = nothing, show_separator = false)
