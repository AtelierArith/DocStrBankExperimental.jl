```
snoop_bot(config::BotConfig, path_to_example_script::String, test_modul=Main; snoop_mode=:auto)
```

Generate precompile statements using a precompile script. `config` can be generated by [`BotConfig`](@ref). `path_to_example_script` is preferred to be an absolute path. The example script will be run in the module specified by `test_modul`. `snoop_mode` can be `:auto`, `:snoopi` (to run [`SnoopCompileCore.@snoopi`](@ref)), or `:snoopc` (to run [`SnoopCompileCore.@snoopc`](@ref)), where `:auto` chooses `:snoopi` on supported versions of Julia.

See the [online documentation](https://timholy.github.io/SnoopCompile.jl/stable/bot/) for a more complete overview.

# Extended help

## Example

In this case, the bot-running script is placed in the same directory as the precompile script, so we can use `@__DIR__` to find it:

```julia
using SnoopCompile

snoop_bot(BotConfig("MatLang"), "$(@__DIR__)/example_script.jl")
```
