| フィールド                     | ノート                                                                                                                                                                                        |
|:------------------------- |:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| rc                        | 参照カウンタ。                                                                                                                                                                                    |
| num*ghosts*elements       | 非ローカルゴースト要素のカウント                                                                                                                                                                           |
| num*remote*elements       | 他のプロセスに対してゴーストであるローカル要素のカウント。                                                                                                                                                              |
| ghost_type                | どの隣接要素がゴーストと見なされるかを説明します。                                                                                                                                                                  |
| ghost_trees               | ゴーストツリーデータ：global_id。eclass。要素。線形ID順。                                                                                                                                                      |
| global*tree*to*ghost*tree | ghost*treesへのインデックス。グローバルツリーID I が与えられたとき、ツリーが ghost*trees[i] にあるようなインデックス i を返します。                                                                                                        |
| process_offsets           | プロセスが与えられたとき、最初のゴーストツリーとその中のそのプロセスの最初の要素を返します。                                                                                                                                             |
| remote_ghosts             | 他のプロセスのためにゴースト要素を持つローカルツリーの配列。各ツリーに対して、ローカルゴースト要素の配列 [`t8_element_t`](@ref) *。また、ツリー内のこれらの要素のローカルインデックスの配列 [`t8_locidx_t`](@ref)。リモートプロセスのランクでハッシュ化されたハッシュテーブルです。各プロセス内で線形IDによってソートされています。 |
| remote_processes          | ローカル要素がゴーストであるプロセスのランク。整数の配列。                                                                                                                                                              |
