```
t8_stash_joinface
```

コミットされる前のcmeshに保存される面接続情報。

| フィールド       | 注釈                                    |
|:----------- |:------------------------------------- |
| id1         | 接続の最初の木のグローバルツリーID。                   |
| id2         | 2番目の木のグローバルツリーID。id1<=id2を保証します。      |
| face1       | 接続された面の最初の面番号。                        |
| face2       | 2番目の面の面番号。                            |
| orientation | 面接続の向き。  # t8*cmesh*types.hも参照してください。 |
