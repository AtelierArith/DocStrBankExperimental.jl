```
t8_stash_attribute
```

コミットされる前のcmeshに保存される属性情報。ペア (package_id, key) はデータを特定するためのルックアップキーとして機能します。

| フィールド      | 注釈                                |
|:---------- |:--------------------------------- |
| id         | グローバルツリーID                        |
| attr_size  | この属性のサイズ（バイト単位）                   |
| attr_data  | 属性データを格納する*サイズ*バイトの配列。            |
| is_owned   | データがコピーされた場合は真、ユーザーがまだ所有している場合は偽。 |
| package_id | この属性を設定したパッケージのID。                |
| key        | この属性を特定するためにパッケージが使用するキー。         |
