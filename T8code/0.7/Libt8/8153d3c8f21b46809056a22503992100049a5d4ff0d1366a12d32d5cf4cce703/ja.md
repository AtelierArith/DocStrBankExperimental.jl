t8ツリーのデータ型

| フィールド           | 注釈                             |
|:--------------- |:------------------------------ |
| elements        | ローカルに保存された要素                   |
| eclass          | このツリーの要素クラス                    |
| first_desc      | 最初のローカル子孫                      |
| last_desc       | 最後のローカル子孫                      |
| elements_offset | このプロセッサ上の以前のツリーに対する累積和（ローカルのみ） |
