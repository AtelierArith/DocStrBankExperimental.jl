```
t8_eclass
```

この列挙型には、すべての可能な要素クラスが含まれています。

| 列挙子                | 注釈                                                           |
|:------------------ |:------------------------------------------------------------ |
| T8*ECLASS*VERTEX   | 頂点は唯一のゼロ次元要素クラスです。                                           |
| T8*ECLASS*LINE     | 線は唯一の一次元要素クラスです。                                             |
| T8*ECLASS*QUAD     | 四角形は二次元の要素クラスのうちの一つです。                                       |
| T8*ECLASS*TRIANGLE | 三角形のための要素クラスです。                                              |
| T8*ECLASS*HEX      | 六面体は三次元の要素クラスの一つです。                                          |
| T8*ECLASS*TET      | 四面体は別の三次元の要素クラスです。                                           |
| T8*ECLASS*PRISM    | プリズムは五つの面を持ちます：三つの四角形で結ばれた二つの対向する三角形です。                      |
| T8*ECLASS*PYRAMID  | ピラミッドは四角形の底面と四つの三角形の側面を持ちます。                                 |
| T8*ECLASS*COUNT    | これは要素クラスではありませんが、要素クラスの数として使用できます。                           |
| T8*ECLASS*INVALID  | これは要素クラスではありませんが、t8codeがサポートしていないサードパーティライブラリのクラスの場合に使用できます。 |
