```
t8_ghost_type_t
```

このタイプは、どの隣接要素がゴースト要素としてカウントされるかを制御します。現在、面隣接要素をサポートしています。頂点および辺の隣接要素は最終的に追加される予定です。

| 列挙子               | 注釈                               |
|:----------------- |:-------------------------------- |
| T8*GHOST*NONE     | ゴースト層を作成しない。                     |
| T8*GHOST*FACES    | すべての面（コドメイン1）隣接要素を考慮する。          |
| T8*GHOST*EDGES    | すべての辺（コドメイン2）および面隣接要素を考慮する。      |
| T8*GHOST*VERTICES | すべての頂点（コドメイン3）および辺および面隣接要素を考慮する。 |
