```
t8_scheme_cxx_unref(pscheme)
```

スキームの参照カウンタを減少させます。カウンタがゼロに達すると、このスキームは破棄されます。

# 引数

  * `pscheme`:[in,out] 入力時に、指し示すスキームは正の参照カウントで存在している必要があります。参照カウントがゼロに達すると、スキームは破棄され、このポインタはNULLに設定されます。それ以外の場合、ポインタは変更されず、スキームは他の方法で変更されません。

### プロトタイプ

```c
void t8_scheme_cxx_unref (t8_scheme_cxx_t **pscheme);
```
