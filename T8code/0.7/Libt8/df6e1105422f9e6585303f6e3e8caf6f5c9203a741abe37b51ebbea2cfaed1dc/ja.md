```
t8_forest_ghost_unref(pghost)
```

ゴースト構造体の参照カウントを減少させます。カウントがゼロに達すると、ゴースト構造体は破棄されます。cmeshへの最後の参照が削除されることがわかっている場合は、t8*forest*ghost_destroyを使用することが推奨されます。

# 引数

  * `pghost`:[in,out] 入力時に、ポインタが指すゴースト構造体は正の参照カウントで存在している必要があります。参照カウントがゼロに達すると、ゴースト構造体は破棄され、このポインタはNULLに設定されます。それ以外の場合、ポインタは変更されません。

### プロトタイプ

```c
void t8_forest_ghost_unref (t8_forest_ghost_t *pghost);
```
