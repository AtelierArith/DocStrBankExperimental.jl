```
t8_forest_unref(pforest)
```

フォレストの参照カウンタを減少させます。カウンタがゼロに達すると、このフォレストは破棄されます。この場合、フォレストはそのcmeshおよびschemeメンバーの参照を解除します。

# 引数

  * `pforest`:[in,out] 入力時に、ポインタが指すフォレストは正の参照カウントで存在している必要があります。任意の状態である可能性があります。参照カウントがゼロに達すると、フォレストは破棄され、このポインタはNULLに設定されます。それ以外の場合、ポインタは変更されず、フォレストは他の方法で修正されません。

### プロトタイプ

```c
void t8_forest_unref (t8_forest_t *pforest);
```
