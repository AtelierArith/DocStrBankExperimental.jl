```
t8_cmesh_set_derive(cmesh, set_from)
```

この関数は、cmeshを派生元として設定します。デフォルトでは、すべてのデータを手動で指定してスタンドアロンのcmeshを作成します。また、既存のcmeshから派生させることで粗いメッシュを構築することもできます。他のcmeshからの派生は、各ツリーの再分配や均一な細分化と組み合わせることもオプションで可能です。この関数は、以前に設定された派生元のcmeshを上書きします。

# 引数

  * `cmesh`:[in,out] 初期化されている必要がありますが、コミットされていない必要があります。スタンドアロンに戻すためにNULLにすることもできます。
  * `set_from`:[in,out] このcmeshの参照カウンタが増加します。t8*cmesh*commitによって減少され、その後*from*はもはや記憶されません。それ以外は、fromオブジェクトは変更されません。

### プロトタイプ

```c
void t8_cmesh_set_derive (t8_cmesh_t cmesh, t8_cmesh_t set_from);
```
