```
t8_load_mode
```

この列挙型には、保存されたcmeshを開くことができるすべてのモードが含まれています。cmeshは、保存されたプロセスよりも多くのプロセスでロードでき、モードはどのプロセスがファイルを開き、データを分配するかを制御します。

| 列挙子            | 注釈                                                                                                                                              |
|:-------------- |:----------------------------------------------------------------------------------------------------------------------------------------------- |
| T8*LOAD*SIMPLE | シンプルモードでは、最初のnプロセスがファイルをロードします。                                                                                                                 |
| T8*LOAD*BGQ    | BGQモードでは、ファイルはnノードで、各ノードの1つのプロセスからロードされます。これにはMPIバージョン3.1以上が必要です。                                                                               |
| T8*LOAD*STRIDE | n番目のプロセスがファイルをロードします。注意して扱ってください。これは、JuqueenでMPI-3が利用できなかったために導入されました。パラメータnは追加のパラメータとして渡す必要があります。  # 参照 [`t8_cmesh_load_and_distribute`](@ref) |
| T8*LOAD*COUNT  |                                                                                                                                                 |
