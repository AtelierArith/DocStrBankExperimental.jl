```
t8_forest_get_maxlevel(forest)
```

フォレスト内の任意の要素に対して許可される最大の細分化レベルを返します。

# 引数

  * `forest`:[in] フォレスト。

# 戻り値

このフォレスト内の要素に対して許可される最大の細分化レベル。*forest*内の任意の木はこの回数だけ細分化できることが保証されており、それ以上の細分化は許可されていません。*forest*はこの関数を呼び出す前にコミットされている必要があります。単一要素クラス（非ハイブリッド）のフォレストの場合、maxlevelはこの要素クラスの最大細分化レベルですが、ハイブリッドフォレストの場合、maxlevelはこのフォレスト内の要素クラスのすべてのmaxlevelの最小値です。

### プロトタイプ

```c
int t8_forest_get_maxlevel (const t8_forest_t forest);
```
