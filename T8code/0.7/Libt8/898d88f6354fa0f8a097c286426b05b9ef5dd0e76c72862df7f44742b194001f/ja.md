```
t8_stash
```

スタッシュデータ構造は、cmeshがコミットされる前の情報を保存するために使用されます。特に、ツリーのeクラス、面の接続、およびツリーの属性を保存します。スタッシュ構造を使用することで、非常に柔軟なインターフェースを持つことができます。新しいメッシュを構築する際に、ユーザーはこれらのメッシュエンティティを任意の順序で指定できます。cmeshがコミットされると、情報はスタッシュからcmeshに順序正しくコピーされます。

| フィールド      | 注釈                                             |
|:---------- |:---------------------------------------------- |
| classes    | ツリーのeクラスを保存します。  # 参照 [`t8_stash_class`](@ref) |
| joinfaces  | 面の接続を保存します。  # 参照 [`t8_stash_joinface`](@ref)  |
| attributes | 属性を保存します。  # 参照 [`t8_stash_attribute`](@ref)   |
