```
t8_netcdf_variable_type
```

この列挙型には、すべての可能なnetCDF変数データ型（int、int64、double）が含まれています。

| 列挙子              | 注釈                                |
|:---------------- |:--------------------------------- |
| T8*NETCDF*INT    | 32ビット整数データを保持するnetCDF変数データ型を象徴します |
| T8*NETCDF*INT64  | 64ビット整数データを保持するnetCDF変数データ型を象徴します |
| T8*NETCDF*DOUBLE | 倍精度データを保持するnetCDF変数データ型を象徴します     |
