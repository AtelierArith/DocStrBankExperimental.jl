```
t8_netcdf_variable_type
```

This enumeration contains all possible netCDF variable datatypes (int, int64, double).

| Enumerator       | Note                                                                |
|:---------------- |:------------------------------------------------------------------- |
| T8_NETCDF_INT    | Symbolizes netCDF variable datatype which holds 32-bit integer data |
| T8_NETCDF_INT64  | Symbolizes netCDF variable datatype which holds 64-bit integer data |
| T8_NETCDF_DOUBLE | Symbolizes netCDF variable datatype which holds double data         |
