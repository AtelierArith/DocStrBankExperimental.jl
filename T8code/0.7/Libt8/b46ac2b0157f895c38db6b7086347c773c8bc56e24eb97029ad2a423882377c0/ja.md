```
t8_cmesh_unref(pcmesh)
```

cmeshの参照カウンタを減少させます。カウンタがゼロに達すると、このcmeshは破棄されます。最後の参照が削除されることがわかっている場合は、t8*cmesh*destroyを使用することが推奨されます。

# 引数

  * `pcmesh`:[in,out] 入力時に、指し示されているcmeshは正の参照カウントで存在している必要があります。任意の状態である可能性があります。参照カウントがゼロに達すると、cmeshは破棄され、このポインタはNULLに設定されます。それ以外の場合、ポインタは変更されず、cmeshは他の方法で変更されません。

### プロトタイプ

```c
void t8_cmesh_unref (t8_cmesh_t *pcmesh);
```
