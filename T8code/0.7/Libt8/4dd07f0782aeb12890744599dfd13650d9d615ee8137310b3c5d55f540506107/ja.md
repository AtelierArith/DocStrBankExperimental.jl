```
t8_geometry_type
```

この列挙型には、すべての可能なジオメトリが含まれています。

| 列挙子                                  | 注釈                                                     |
|:------------------------------------ |:------------------------------------------------------ |
| T8*GEOMETRY*TYPE_ZERO                | ゼロジオメトリはすべての点をゼロにマッピングします。                             |
| T8*GEOMETRY*TYPE_LINEAR              | 線形ジオメトリは、木の頂点間を補間するために線形補間を使用します。                      |
| T8*GEOMETRY*TYPE*LINEAR*AXIS_ALIGNED | 線形の軸に整列したジオメトリは、軸に整列しているため、2つの頂点のみを使用します。              |
| T8*GEOMETRY*TYPE_LAGRANGE            | ラグランジュジオメトリは、ラグランジュ多項式を用いたマッピングを使用して曲線要素を近似します。        |
| T8*GEOMETRY*TYPE_ANALYTIC            | 解析ジオメトリは、ユーザー定義の解析関数を使用して物理領域にマッピングします。                |
| T8*GEOMETRY*TYPE_CAD                 | opencascadeジオメトリは、CAD形状を使用して木を基盤となるCADモデルに正確にマッピングします。 |
| T8*GEOMETRY*TYPE_COUNT               | これはジオメトリタイプではありませんが、ジオメトリタイプの数として使用できます。               |
| T8*GEOMETRY*TYPE_UNDEFINED           | これはジオメトリタイプではありませんが、タイプが定義されていないすべてのジオメトリに使用されます。      |
