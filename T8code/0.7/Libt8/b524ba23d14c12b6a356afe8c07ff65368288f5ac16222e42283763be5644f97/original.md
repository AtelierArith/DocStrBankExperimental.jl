```
t8_vec_orthogonal_tripod(v1, v2, v3)
```

Compute an orthogonal coordinate system from a given vector.

# Arguments

  * `v1`:[in] 3D vector.
  * `v2`:[out] 3D vector.
  * `v3`:[out] 3D vector.

### Prototype

```c
static inline void t8_vec_orthogonal_tripod (const double v1[3], double v2[3], double v3[3]);
```
