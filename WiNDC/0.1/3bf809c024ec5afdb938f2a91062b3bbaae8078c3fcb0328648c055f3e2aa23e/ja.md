```
calibrate(data::AbstractNationalTable)
```

これは現在、国のデータセットをキャリブレーションすることを目的としています。これを一般的なキャリブレーション関数にするために作業を進めています。

キャリブレーションされた値とモデルを持つ新しいAbstractNationalTableを返します。

主なバランス操作は3つあります：

1. ゼロ利益 - 列の合計が等しい
2. 市場クリアランス - 行の合計が等しい
3. マージンバランス - マージンがバランスする

3つの税率は固定されています。税率は次のとおりです：

1. 生産税率
2. 吸収税率
3. 輸入関税率

以下は固定されています：

1. 労働補償
2. 輸入
3. 輸出
4. 家庭供給

ゼロの値はゼロのまま残ります。
