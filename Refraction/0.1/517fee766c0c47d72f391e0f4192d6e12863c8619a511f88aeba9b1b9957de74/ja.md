```
(m::Material)(wavelength::Real; [default::Real=1.0]) -> Real
```

与えられた波長に対する材料の屈折率を計算するファンクタメソッドです。波長が材料の波長範囲外の場合、メソッドは波長を範囲内に制限します。材料が屈折率に関するデータを含まない場合、メソッドはデフォルト値を返します。ファンクタメソッドは `dispersion` メソッドを呼び出します。

# 引数

  * `m::Material` : 屈折率を計算する材料オブジェクト。
  * `wavelength::Real` : 屈折率を計算する波長（単位は $\mu m$）。
  * `default::Real=1.0` : 材料が屈折率に関するデータを含まない場合に返すデフォルト値。
