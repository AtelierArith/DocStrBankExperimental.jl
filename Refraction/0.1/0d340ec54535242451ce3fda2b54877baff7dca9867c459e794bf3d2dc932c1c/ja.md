```
extinction(m::Material, wavelength::Real; [default::Float64=0.0]) -> Real
```

与えられた波長に対する材料の消失係数を計算します。波長が材料の波長範囲外の場合、メソッドは波長を範囲内に制限します。材料が消失係数に関するデータを含まない場合、メソッドはデフォルト値を返します。

# 引数

  * `m::Material` : 消失係数を計算する材料オブジェクト。
  * `wavelength::Real` : 消失係数を計算する波長（単位は $\mu m$）。
  * `default::Float64=0.0` : 材料が消失係数に関するデータを含まない場合に返すデフォルト値。
