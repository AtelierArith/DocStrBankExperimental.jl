```
drop(r)
drop(r; kind=:lowest, n=1)
```

ロールの最低部分を削除します。`DiceRoll`の場合、これはロールされたサイコロの中で最低のサイコロを削除することを意味しますが、複合サイコロの場合は、全体のセグメントを削除することを意味することがあります。例えば、

```
drop(4d6)
```

は、振られた4つの6面サイコロの中で最低の値を持つサイコロを削除します。これは次のように等価です。

```
v = roll.([d6, d6, d6, d6])
sum(v) - minimum(v)
```

一方、

```
drop((2d4 - 1) * (2d4 - 1))
```

は、積の中で最低の値を削除します。つまり、これは次のように等価です。

```
v = roll.([2d4 - 1, 2d4 - 1])
sum(v) - minimum(v)
```

**キーワード引数**

  * `kind`: 最低または最高のロールまたはロールを削除するために、`:lowest`または`:highest`のいずれか。
  * `n`: 削除するサイコロの数。
