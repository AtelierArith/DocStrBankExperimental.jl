```
 ffm2hr(Afun::FourierFunctionMatrix; atol = 0, rtol = √ϵ, squeeze = true) -> Ahr::HarmonicArray
```

フーリエ周期行列オブジェクトの調和（フーリエ）表現を計算します。

周期 `T` のフーリエ関数行列オブジェクト `Afun` は、部分周期 `T′ = T/k` の周期行列 `Afun(t)` のフーリエ級数表現を使用して構築されます。ここで、`Afun(t)` の各エントリは次の形式を持ちます。

```
         p
  a_0 +  ∑ ( ac_i*cos(i*t*2*π/T′)+as_i*sin(i*2*π*t/T′) ) ,
        i=1
```

ここで、`k ≥ 1` は部分周期の数です（デフォルト: `k = 1`）。   

周期 `T` の調和配列オブジェクト `Ahr` は、部分周期 `T′′ = T/k′` の周期行列 `Ahr(t)` の調和表現を使用して構築されます。形式は次の通りです。

```
                 p′
 Ahr(t) = A_0 +  ∑ ( Ac_i*cos(i*t*2*π/T′′)+As_i*sin(i*2*π*t/T′′) ) ,
                i=1
```

ここで、`p′` は `Ac_j` および/または `As_j` が非ゼロである最大インデックス `j` です。非ゼロ要素を評価するために使用される許容誤差は `tol = max(atol, rtol*maxnorm)` であり、ここで `maxnorm` は `Afun(t)` の係数 `ac_i` および `as_i` の最大絶対値です。許容誤差のデフォルト値は `atol = 0` および `rtol = √ϵ` であり、ここで `ϵ` は作業用の機械精度です。結果として得られる調和近似 `Ahr(t)` は、調和配列オブジェクト `Ahr` に返されます（詳細は [`HarmonicArray`](@ref) を参照）。
