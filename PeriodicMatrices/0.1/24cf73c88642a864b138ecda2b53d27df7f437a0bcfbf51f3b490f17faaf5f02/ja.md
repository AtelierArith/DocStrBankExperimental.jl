```
 HarmonicArray(Ahr, T; nperiod = k) -> A::HarmonicArray
```

連続時間ハーモニック配列表現。

周期 `T` のハーモニック配列オブジェクト `A` は、部分周期 `T′ = T/k` の周期行列 `Ahr(t)` のハーモニック表現を使用して構築されます。形式は次の通りです。

```
                 p
 Ahr(t) = A_0 +  ∑ ( Ac_i*cos(i*t*2*π/T′)+As_i*sin(i*2*π*t/T′) ) ,
                i=1
```

ここで `k ≥ 1` は部分周期の数（デフォルト: `k = 1`）です。`m×n×(p+1)` の複素配列 `Ahr` には、ハーモニック成分が次のように含まれています: `Ahr[:,:,1]` には定数項 `A_0`（平均値）が含まれ、`Ahr[:,:,i+1]` の実部と虚部はそれぞれ係数行列 `Ac_i` と `As_i` を含みます。ハーモニック成分を含む複素行列 `Ahr`、周期 `T`、および部分周期の数 `k` は、それぞれ `A.values`、`A.period`、および `A.nperiod` を介してアクセスできます。
