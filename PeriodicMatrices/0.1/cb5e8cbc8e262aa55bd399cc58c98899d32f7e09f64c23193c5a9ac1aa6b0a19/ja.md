```
SwitchingPeriodicArray(M, ns, T; nperiod = k) -> A::SwitchingPeriodicArray
```

離散時間スイッチング周期配列の表現。

離散時間スイッチング周期配列オブジェクト `A` は、`m×n×p` の実配列 `M`、増加する正の整数の `p` ベクトル `ns`（離散スイッチングモーメントを表す）、関連する時間周期 `T`、およびキーワード引数 `nperiod = k` を介して指定されたサブ周期の数から構築されます。

`M` には、サイクル成分行列 `M[:,:,i]`（`i = 1,..., p`）が含まれており、これは行列のシーケンス `N := ns[p]` を定義します。すなわち、`S[1], ..., S[N]` であり、`S[j] = M[:,:,i]` は `j ∈ [ns[i-1]+1, ..., ns[i]]` の場合に成り立ち、`ns[0] := 0` です。`S[j]` は、サブ周期 `T′ := T/k` の時間周期行列 `A(t)` の `j` 番目の値 `A(Δ(j-1))` であり、`Δ := T′/N` は関連するサンプル時間です。成分行列 `M`、整数ベクトル `ns`、周期 `T`、サブ周期の数 `k`、離散周期 `N`、およびサンプル時間 `Δ` は、それぞれ `A.M`、`A.ns`、`A.period`、`A.nperiod`、`A.dperiod`、および `A.Ts` を介してアクセスできます。

`j` 番目の時間値 `A(Δ(j-1))` は `A[j]` として決定できます。任意の `j` に対して `A[j] := A[mod(j-1,N)+1]` であると仮定されます。
