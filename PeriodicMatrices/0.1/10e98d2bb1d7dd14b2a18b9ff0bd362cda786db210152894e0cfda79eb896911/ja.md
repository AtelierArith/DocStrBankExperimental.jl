```
PeriodicArray(M, T; nperiod = k) -> A::PeriodicArray
```

離散時間周期配列の表現。

離散時間周期配列オブジェクト `A` は、`m×n×p` の実配列 `M`、関連する時間周期 `T`、およびキーワード引数 `nperiod = k` を介して指定されたサブ周期の数から構築されます。 `M` には、周期的な行列 `M(t)` の値 `M(Δ(i-1))` を表すサイクリックコンポーネント行列 `M[:,:,i]` が含まれ、ここで `i = 1,..., p` です。周期 `T/k` の `M[:,:,k] := M[:,:,mod(k-1,p)+1]` が任意の `k` に対して成り立つと仮定します。コンポーネント行列 `M`、周期 `T`、サブ周期の数 `k`、離散周期 `p`、およびサンプル時間 `Δ` は、それぞれ `A.M`、`A.period`、`A.nperiod`、`A.dperiod`、および `A.Ts` を介してアクセスできます。
