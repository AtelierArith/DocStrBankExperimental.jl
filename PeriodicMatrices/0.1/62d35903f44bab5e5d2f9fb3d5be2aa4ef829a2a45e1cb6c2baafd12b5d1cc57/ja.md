```
 hr2btupd(Ahr::HarmonicArray, N; P, nperiod, shift]) -> Ab::Matrix
```

周期行列の調和（フーリエ）配列表現の更新されたブロック・トプリッツ行列を構築します。

`BT` が、[`hr2bt`](@ref) を使用して構築された、サブ周期 `T′` の `n × n` 周期行列 `Ahr` の調和配列表現のブロック・トプリッツ行列である場合（[`HarmonicArray`](@ref) を参照）、更新された行列 Ab = BT-NT が構築されます。ここで、`NT` は `n × n` の対角ブロックを持つブロック対角行列です。`NT` の `k` 番目の対角ブロックは、対角行列 `im*(μ + k*ωh)*I` であり、ここで `μ` はキーワードパラメータ `shift = μ`（デフォルト: `μ = 0`）を介して指定されたシフトであり、`ωh` は `ωh = 2π*nperiod/(P*T′)` として計算される基準周波数です。シフトの値は `0 ≤ μ ≤ ωh/2` を満たす必要があります。
