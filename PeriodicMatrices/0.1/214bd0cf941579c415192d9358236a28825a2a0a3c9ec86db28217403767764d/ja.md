```
PeriodicTimeSeriesMatrix(At, T; nperiod = k) -> A::PeriodicTimeSeriesMatrix
```

連続時間周期的時系列行列の表現。

周期 `T` の連続時間周期的時系列行列オブジェクト `A` は、実行列の `p`-ベクトル `At` と関連するサブ周期 `T′ = T/k` から構築されます。ここで、`k ≥ 1` はサブ周期の数（デフォルト: `k = 1`）です。`At` には、サイクル成分行列 `At[i]` が含まれており、`i = 1,..., p` の範囲で、`At[i]` は周期 `T′` の時間周期行列 `A(t)` の値 `A(Δ*(i-1))` を表します。ここで、`Δ := T′/p` は関連するサンプリング時間です。任意の `i` に対して `At[i] := At[mod(i-1,p)+1]` であると仮定します。すべての成分行列は同じ次元を持つ必要があります。成分行列 `At`、周期 `T`、およびサブ周期の数 `k` は、それぞれ `A.values`、`A.period`、および `A.nperiod` を介してアクセスできます。
