```
 psordschur!(S::Array{Float64,3}, Z::Array{Float64,3}, select::Union{Vector{Bool},BitVector}; rev, schurindex)
```

固有値の順序を変更します。積 `Π = S[:,:,p]*...*S[:,:,2]*S[:,:,1]` の場合、`rev = true`（デフォルト）または `Π = S[:,:,1]*S[:,:,2]*...*S[:,:,p]` の場合、`rev = false` で、`Π` は実シュール形式であり、論理配列 `select` で選択された固有値が先頭の位置に移動されるようにします。3次元配列 `S` と `Z` には、周期的シュール形式の行列 `S[:,:,1]`, `...`, `S[:,:,p]` と、対応する直交変換行列 `Z[:,:,1]`, `...`, `Z[:,:,p]` が含まれています。`S` と `Z` は更新された行列で上書きされます。共役複素数の固有値のペアは、`select` を介して両方とも含めるか、両方とも除外する必要があります。
