```
psceig(A[, k]; fast = false) -> ce
```

`p` 個の行列の循環行列積の特性指数を計算します。

離散時間周期行列 `A(t)` の特性指数を計算します。特性指数は、関数 [`pseig(::PeriodicMatrix)`](@ref) を呼び出すことによって決定される特性乗数から計算されます。 `fast = false`（デフォルト）の場合、特性乗数は周期的シュール分解に基づくアプローチを使用して計算されます[1]。一方、`fast = true` の場合は、適切なリフトペンシルの構造を利用した削減[2]が使用されます。この後者のオプションは、大量の行列に対して不正確な結果をもたらすことがあります。

引数 `k` は、コンポーネント行列の開始インデックスを指定します（デフォルト: `k = 1`）。

*注:* `ce` の最初の `nmin` コンポーネントには *コア特性指数* が含まれており、ここで `nmin` はコンポーネント行列の最小行次元です。一方、`ce` の最後のコンポーネントはゼロです。
