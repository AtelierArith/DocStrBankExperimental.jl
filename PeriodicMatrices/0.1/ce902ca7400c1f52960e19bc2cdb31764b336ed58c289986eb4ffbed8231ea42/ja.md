```
 tvmeval(Ahr::HarmonicArray, t; ntrunc, exact = true) -> A::Vector{Matrix}
```

ハーモニック配列の時間応答を評価します。

ハーモニック配列 `Ahr` は、連続時間周期行列 `A(t)` を表し、時間値のベクトル `t` に対して、各時間値 `t[i]` に対して `A[i] = A(t[i])` が計算されます。 `exact = true (デフォルト)` の場合、正確な評価が計算され、`exact = false` の場合、線形補間に基づく近似が計算されます（サンプル間の点でより正確な可能性があります）。 キーワード引数 `ntrunc` は、評価に使用されるハーモニクスの数を指定します（デフォルト：可能な最大数のハーモニクス）。
