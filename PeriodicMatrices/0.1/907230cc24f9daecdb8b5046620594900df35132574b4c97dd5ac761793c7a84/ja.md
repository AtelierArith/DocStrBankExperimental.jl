```
 HarmonicArray(A0, Ac, As, T) -> A::HarmonicArray
```

調和成分から調和配列表現を構築します。

調和配列オブジェクト `A` は、周期 `T` の周期行列の調和表現 `Ahr(t)` のために構築され、次の形式になります。

```
                 p
 Ahr(t) = A_0 +  ∑ ( Ac_i*cos(i*t*2*π/T)+As_i*sin(i*2*π*t/T) ) ,
                i=1
```

ここで、定数項 `A_0` は実行列 `A0` に含まれ、`Ac` と `As` は実行列のベクトルであり、`i` 番目の (コサイン) 係数行列 `Ac_i` は `Ac[i]` に含まれ、`i` 番目の (サイン) 係数行列 `As_i` は `As[i]` に含まれます。 `p` は行列 `Ac` と `As` のベクトルの最大長です。 `Ac` または `As` の長さが `p` よりも短い場合、対応する行列にはゼロのトレーリング行列があると見なされます。すべての成分行列は同じ次元を持たなければなりません。調和成分を含む複素行列と周期 `T` は、それぞれ `A.values` と `A.period` を介してアクセスできます。 ```
