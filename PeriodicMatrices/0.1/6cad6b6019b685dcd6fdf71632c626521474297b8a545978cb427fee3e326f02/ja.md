```
psceighr(Ahr::HarmonicArray[, N]; P, nperiod, shift, atol) -> ce
```

連続時間周期行列の特性指数を*ハーモニックアレイ*表現で計算します。

周期 `T` の与えられた正方形の連続時間周期関数行列 `Ahr(t)` に対して、特性指数 `ce` は、基本ストリップ `-ω/2 <  Im(λ) ≤ ω/2` にある切り捨てられたハーモニック状態演算子 `A(N)-E(N)` の固有値として計算されます。ここで `ω = 2π/T` です。`Ahr(t)` にハーモニック成分 `A_0`, `A_1`, ..., `A_p` がある場合、`N ≥ p`、`P = 1` および `nperiod = 1` のとき、行列 `A(N)` と `E(N)` は次のように構築されます。

```
       ( A_0  A_{-1} …  A_{-p}        0    )           ( -im*ϕ_{-N}I                                 0        )
       ( A_1   A_0             ⋱           )           (     ⋮       ⋱                                        )
       (  ⋮         ⋱            ⋱         )           (               -im*ϕ_{-1}*I                           )
A(N) = ( A_p             ⋱          A_{-p} ) , E(N) =  (                           -im*ϕ_{0}*I                )
       (        ⋱           ⋱         ⋮    )           (     ⋮                                  ⋱              )
       (  0        A_p      …         A_0  )           (     0                                   -im*ϕ_{N}I   )
```

ここで `ϕ_{i} := shift+i*ω` です。`N < p` の場合、最初の `N` ハーモニック成分を使用して切り捨てられた*フル*ブロックToeplitz行列 A(N) が構築されます。`N` のデフォルト値は `N = max(10,p-1)` です。

一般に、与えられた `P ≥ 1` および `nperiod ≥ 1` に対して、ブロックToeplitz行列 `A(N)`（および `E(N)`）は `(2N*np+1)×(2N*np+1)` ブロックで構築され、`np = P*nperiod` であり、各 `A_i` はその列の前に `np-1` のゼロブロックがあり、各 `A_{-i}` はその行の前に `np-1` のゼロブロックがあり、すべての対角ブロックは `A_0` に等しいです。

キーワード引数 `atol`（デフォルト: `atol = 1.e-10`）は、関連する固有ベクトルの漸近的（指数的）減衰を検証するために使用される末尾成分の大きさに対する許容誤差です。このチェックを満たす固有値のみが `ce` に返されます。
