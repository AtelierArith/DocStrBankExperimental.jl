```
 hreval(Ahr::HarmonicArray, t; ntrunc, exact = true) -> A::Matrix
```

ハーモニック配列 `Ahr` を評価し、連続時間の時間周期行列 `A(t)` を時間値 `t` に対して求めます。実数値 `t` に対して、`exact = true (デフォルト)` の場合は正確な評価が計算され、`exact = false` の場合は線形補間に基づく近似が計算されます（サンプル間の点でより正確な可能性があります）。キーワード引数 `ntrunc` は評価に使用されるハーモニクスの数を指定します（デフォルト：可能な最大数）。
