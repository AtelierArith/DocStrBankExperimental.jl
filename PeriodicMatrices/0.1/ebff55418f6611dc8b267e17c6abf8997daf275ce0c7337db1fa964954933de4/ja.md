```
 pfm2hr(A::PeriodicFunctionMatrix; nsample, NyquistFreq) -> Ahr::HarmonicArray
```

周期関数行列を調和配列表現に変換します。`A(t)`が周期`T`の周期関数行列である場合、調和配列表現`Ahr`は、サンプリングされた配列`A(iΔ)`に高速フーリエ変換を適用することによって決定されます。ここで、`Δ = T/k`はサンプリング周期であり、`k`はキーワード引数`nsample = k`によって指定されたサンプル数です（デフォルト: `k = 128`）。ナイキスト周波数`f`がキーワード引数`NyquistFreq = f`を介して指定されている場合、信号のエイリアシングを避けるために`k`は`k = 2*f*T`として選択されます。
