```
 psordschur!(S::Vector{Matrix{Float64}}, Z::Vector{Matrix{Float64}}, select; rev, schurindex)
```

行列の積 `Π = S[p]*...*S[2]*S[1]` のコア固有値を順序を変更します。`rev = true`（デフォルト）である場合、または `Π = S[1]*S[2]*...*S[p]` である場合、`rev = false` の場合、`Π` は実シュール形式であり、論理配列 `select` で選択された固有値が先頭の位置に移動されるようにします。`p` ベクトル `S` と `Z` には、拡張周期シュール形式の行列 `S[1]`, `...`, `S[p]` が含まれており、`S[schurindex]` の先頭の正方行列ブロックは実シュール形式であり、対応する直交変換行列 `Z[1]`, `...`, `Z[p]` がそれぞれ含まれています。`S` と `Z` は更新された行列で上書きされます。共役複素数の固有値のペアは、`select` を介して両方とも含まれるか、両方とも除外されなければなりません。`select` の次元はコア固有値の数（すなわち、ベクトル `S` 内の行列の最小次元）と等しくなければなりません。
