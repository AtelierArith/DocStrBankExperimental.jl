```
 pm2pa(At::PeriodicMatrix) -> A::PeriodicArray
```

離散時間周期行列オブジェクトを離散時間周期配列オブジェクトに変換します。

離散時間周期行列オブジェクト `At` は、実行列 `At[i]` の `p`-ベクトル `At`、関連する時間周期 `T`、およびサブ周期の数 `k` を含みます。結果として得られる周期 `T` とサブ周期の数 `k` の離散時間周期配列オブジェクト `A` は、`m×n×p` の実配列 `A` から構築され、`A[:,:,i]` には `At[i]` が含まれ、`i = 1,..., p` の範囲で適切にゼロエントリでパディングされています。非定数次元の場合、結果として得られる `m` と `n` はそれぞれ最大の行と列の次元であり、結果の成分行列 `A[:,:,i]` には `At[i]` が含まれ、適切にゼロエントリでパディングされています。
