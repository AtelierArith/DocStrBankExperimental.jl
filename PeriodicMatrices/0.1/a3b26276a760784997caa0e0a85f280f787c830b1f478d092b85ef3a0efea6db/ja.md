```
psceigfr(A::FourierFunctionMatrix[, N]; P, atol) -> ce
```

*Fourier Function Matrix* 表現における連続時間周期行列の特性指数を計算します。

周期 `T` の与えられた正方形の連続時間周期関数行列 `A(t)` に対して、特性指数 `ce` は状態演算子 `A(t)-D*I` の固有値として計算され、基本ストリップ `-ω/2 <  Im(λ) ≤ ω/2` に位置します。ここで、`ω = 2π/T` です。`A(t)-D*I` を近似するために、次数 `n*(2*N*P+1)` の有限次元切り捨て行列が構築されます。ここで、`n` は `A(t)` の次数、`N` はフーリエ表現における選択された調和成分の数、`P` は周期乗数（デフォルト: `P = 1`）です。`N` に対して使用されるデフォルト値は `N = max(10,p-1)` であり、ここで `p` は `A(t)` の調和項の数です（[`FourierFunctionMatrix`](@ref) を参照）。

キーワード引数 `atol`（デフォルト: `atol = 1.e-10`）は、関連する固有ベクトルの漸近的（指数的）減衰を検証するために使用される末尾成分の大きさに対する許容誤差です。このチェックを満たす固有値のみが `ce` に返されます。
