```
 ts2hr(A::PeriodicTimeSeriesMatrix; atol = 0, rtol, n, squeeze = true) -> Ahr::HarmonicArray
```

周期的な行列を時間系列行列によって指定された周期的行列 `A(t)` の調和（フーリエ）近似を計算します。周期的行列 `A(t)` は、`A.values` の行列ベクトルに含まれる `m` 個の行列として指定された連続時間周期的時間系列行列 `A` として指定され、`A.values[k]` は時間の瞬間 `(k-1)T/m` における `A(t)` の値であり、ここで `T = A.period` は周期です。得られた調和近似 `Ahr(t)` は次の形を持ちます。

```
                 p
 Ahr(t) = A_0 +  ∑ ( Ac_i*cos(i*t*2*π/T)+As_i*sin(i*2*π*t/T) ) 
                i=1
```

ここで `A_0` は定数項（平均値）、`Ac_i` と `As_i` はそれぞれ `i` 番目のコサインおよびサイン項の係数行列です。近似の次数 `p` は、最大次数 `n`（デフォルト: `n = (m-1)/2`）および絶対および相対許容誤差 `atol` と `rtol` に基づいて決定されます。`p` は `n`、`(m-1)/2` および `Ac_k` および/または `As_k` が非ゼロである最大インデックス `k` の最小値です。非ゼロ要素を評価するために使用される許容誤差は `tol = max(atol, rtol*maxnorm)` であり、ここで `maxnorm` は `A.values` に含まれる行列の最大ノルムです。許容誤差のデフォルト値は `atol = 0` および `rtol = 10*p*ϵ` であり、ここで `ϵ` は作業機械精度です。

得られた調和近似 `Ahr(t)` は調和配列オブジェクト `Ahr` に返されます（詳細は [`HarmonicArray`](@ref) を参照）。  ```
