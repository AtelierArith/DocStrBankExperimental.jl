```
PeriodicMatrix(M, T; nperiod = k) -> A::PeriodicMatrix
```

離散時間周期行列の表現。

離散時間周期行列オブジェクト `A` は、実行列の `p`-ベクトル `M`、関連する時間周期 `T`、およびキーワード引数 `nperiod = k` で指定されたサブ周期の数から構築されます。

`M` には、周期行列 `M(t)` の値 `M(Δ(i-1))` を表すサイクリックコンポーネント行列 `M[i]`、`i = 1,..., p` が含まれています。ここで、周期は `T/k` であり、`Δ := T/(k*p)` は関連するサンプル時間です。任意の `i` に対して `M[i] := M[mod(i-1,p)+1]` であると仮定されています。すべてのコンポーネント行列は、任意の（時間変動）次元を持つことが許可されています。コンポーネント行列 `M`、周期 `T`、サブ周期の数 `k`、離散周期 `p`、およびサンプル時間 `Δ` は、それぞれ `A.M`、`A.period`、`A.nperiod`、`A.dperiod`、および `A.Ts` を介してアクセスできます。
