```
 pgordschur!(S::Array{Float64,3}, s::Union{Vector{Bool},BitVector}, Z::Array{Float64,3}, select::Union{Vector{Bool},BitVector}; rev, schurindex)
```

固有値の順序を変更します。積 `Π = S[:,:,p]^s[p]*...*S[:,:,2]^s[2]*S[:,:,1]^s[1]` の固有値を、`rev = true`（デフォルト）または `Π = S[:,:,1]^s[1]*S[:,:,2]^s[2]*...*S[:,:,p]^s[p]` の場合、`rev = false` の場合に、`s[j] = ±1` であるとき、`Π` は実シュール形式であり、論理配列 `select` で選択された固有値が先頭の位置に移動されるようにします。3次元配列 `S` と `Z` は、一般化された周期シュール形式の行列 `S[:,:,1]`, `...`, `S[:,:,p]` を含み、`S[:,:,schurindex]` は準上三角（実シュール）形式であり、対応する直交変換行列 `Z[:,:,1]`, `...`, `Z[:,:,p]` をそれぞれ含みます。`S` と `Z` は更新された行列で上書きされます。共役対の固有値は、`select` を介して両方とも含まれるか、両方とも除外されなければなりません。
