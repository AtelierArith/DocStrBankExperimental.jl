```
PeriodicSwitchingMatrix(At, ts, T; nperiod = k) -> A::PeriodicSwitchingMatrix
```

連続時間周期スイッチング行列の表現。

周期 `T` の連続時間周期スイッチング行列オブジェクト `A` は、実行列の `p`-ベクトル `At`、`ts[1] = 0` を持つ増加順のスイッチング時間値の `p`-ベクトル `ts`、および関連するサブ周期 `T′ = T/k` から構築されます。ここで、`k ≥ 1` はサブ周期の数（デフォルト: `k = 1`）です。`At` には、周期 `T′` の時間周期行列 `A(t)` の定数値である循環成分行列 `At[i]`（`i = 1,..., p`）が含まれ、`i < p` の場合は `t ∈ [ts[i],ts[i+1})`、`i = p` の場合は `t ∈ [ts[i],T′)` です。任意の `i` に対して `At[i] := At[mod(i-1,p)+1]` および `ts[i] := ts[mod(i-1,p)+1]` であると仮定します。すべての成分行列は同じ次元を持たなければなりません。成分行列 `At`、スイッチング時間 `ts`、周期 `T`、およびサブ周期の数 `k` は、それぞれ `A.values`、`A.ts`、`A.period`、および `A.nperiod` を介してアクセスできます。
