```
SwitchingPeriodicMatrix(M, ns, T; nperiod = k) -> A::SwitchingPeriodicMatrix
```

離散時間スイッチング周期行列の表現。

離散時間スイッチング周期行列オブジェクト `A` は、実行列の `p`-ベクトル `M`、離散スイッチングモーメントを表す増加する正の整数の `p`-ベクトル `ns`、関連する時間周期 `T`、およびキーワード引数 `nperiod = k` を介して指定されたサブ周期の数から構築されます。

`M` には、成分行列 `M[i]` が含まれ、`i = 1,..., p` の範囲で、行列のシーケンス `N := ns[p]` を定義します。`S[1], ..., S[N]` は、`S[j] = M[i]` であり、`j ∈ [ns[i-1]+1, ..., ns[i]]` となります。ここで `ns[0] := 0` です。`S[j]` は、サブ周期 `T′ := T/k` の時間周期行列 `A(t)` の `j`-番目の値 `A(Δ(j-1))` であり、`Δ := T′/N` は関連するサンプル時間です。すべての成分行列は任意の（時間変動）次元を持つことが許可されています。成分行列 `M`、整数ベクトル `ns`、周期 `T`、サブ周期の数 `k`、離散周期 `N`、およびサンプル時間 `Δ` は、それぞれ `A.M`、`A.ns`、`A.period`、`A.nperiod`、`A.dperiod`、および `A.Ts` を介してアクセスできます。

j-番目の時間値 `A(Δ(j-1))` は `A[j]` として決定できます。任意の `j` に対して `A[j] := A[mod(j-1,N)+1]` であると仮定されます。
