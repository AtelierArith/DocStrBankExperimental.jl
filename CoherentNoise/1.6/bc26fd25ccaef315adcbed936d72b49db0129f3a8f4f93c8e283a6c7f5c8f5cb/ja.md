```
select(x::AbstractSampler, y::AbstractSampler; kwargs...)
```

サンプラー `z` の出力に応じて、サンプラー `x` または `y` のいずれかを出力する修飾子サンプラーを構築します。

サンプラー `z` の出力が `min` と `max` で示される範囲内にある場合、サンプラー `y` の出力が選択されます。この範囲の外に出力がある場合、サンプラー `x` の出力が選択されます。

# 引数

  * `min`: 選択範囲の下限を定義する -1.0 と 1.0 の間の実数。
  * `max`: 選択範囲の上限を定義する -1.0 と 1.0 の間の実数。
  * `falloff`: 遷移の滑らかさを指定する 0.0 と 1.0 の間の実数。
