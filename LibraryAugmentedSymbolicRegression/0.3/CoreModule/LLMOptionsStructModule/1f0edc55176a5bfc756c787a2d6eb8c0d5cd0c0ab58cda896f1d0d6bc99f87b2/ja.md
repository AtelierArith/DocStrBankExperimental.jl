```
LLMOperationWeights(;kws...)
```

異なるLLMベースの突然変異操作の確率を定義します。注意：LLM操作はその記号的対応物よりも大幅に遅くなる可能性があるため、高い確率は遅い操作をもたらします。デフォルトでは、すべての確率を0.0に設定します。これらのパラメータの最大値は1.0（100%の時間）です。

# 引数

  * `llm_crossover::Float64`: LLMバージョンの交差を呼び出す確率。
  * `llm_mutate::Float64`: LLMバージョンの突然変異を呼び出す確率。
  * `llm_gen_random::Float64`: LLMバージョンのgen_randomを呼び出す確率。
