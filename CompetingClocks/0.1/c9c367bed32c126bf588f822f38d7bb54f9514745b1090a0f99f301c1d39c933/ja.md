```
TrackWatcher{K,T}()
```

このウォッチャーはサンプリングを行いません。すべての有効なものを記録します。有効なクロックをforループで反復処理できます。モデルが有効または無効にされる遷移の変化を提供すると考えると、このウォッチャーはそれらの変化を蓄積して、すべての有効な遷移の一貫したリストを提供します。モデルとこのウォッチャーを組み合わせることで、セミマルコフコア行列、または現在知られているその行を提供します。

```julia
for entry in tracker
    entry.clock
    entry.distribution
    entry.te
    entry.when
end
```
