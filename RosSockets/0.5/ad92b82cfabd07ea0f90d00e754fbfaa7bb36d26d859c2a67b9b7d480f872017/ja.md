```
send_receive(connection::Connection, msg::String, timeout::Real = 10.0)
```

メッセージを送信し、応答を待ちます。応答は返されます。この関数は、待機中に実行をブロックし、指定されたタイムアウト期間まで待機します。タイムアウト期間が経過してもデータが到着しない場合、TimeoutError例外がスローされます。注意: 返されるペイロードは生の形式になります。文字列に変換するには、`String(payload)`を使用します。この文字列は、JSONなどの他の形式にさらに変換することができます。

# 引数

  * `connection`: `open_connection`から取得した`Connection`。
  * `msg`: 送信するメッセージ。注意: 一部のTCPサーバーでは、メッセージが特定の形式（例えばJSON）でフォーマットされる必要があり、メッセージを終了するために改行文字（例えば`\n`）が必要な場合があります。
  * `timeout`: データを待機する最大時間（秒）。
