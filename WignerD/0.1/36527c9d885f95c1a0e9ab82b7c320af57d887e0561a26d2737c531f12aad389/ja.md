```
wignerD!(D, j, α::Real, β::Real, γ::Real, [Jy = zeros(ComplexF64, 2j+1, 2j+1)])
```

角運動量 $j$ とオイラー角 $α$, $β$, $γ$ に対して、要素 $D^j_{m,n}(α,β,γ)$ を持つウィグナー D-行列を評価し、その結果を `D` に格納します。運動量 $j$ は整数または半整数であり、非負でなければなりません。オプションで、事前に割り当てられた行列 `Jy` を提供することができ、これはサイズが `(2j+1, 2j+1)` の `ComplexF64` 行列であり、計算中に上書きされる可能性があります。
