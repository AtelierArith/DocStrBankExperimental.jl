```
GCPair(smarts,name;group_order = 1)
```

グループの説明を保持するために使用される構造体。SMILESクエリ内でグループを一致させるために必要なSMARTS文字列と、割り当てられた名前を含みます。`group_order`パラメータは、コンスタンティノウ-ガニアプローチに従うグループに使用されます：`group_order = 1`の`GCPair`のリストは厳密なカバレッジで一致し（カバーするために欠けている原子がある場合は失敗）、2次およびそれ以上のグループは完全なカバレッジのために厳密にチェックされません。各オーダーのグループは独立して一致します。
