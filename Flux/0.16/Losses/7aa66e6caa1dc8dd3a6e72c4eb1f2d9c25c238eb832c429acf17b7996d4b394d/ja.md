```
focal_loss(ŷ, y; dims=1, agg=mean, gamma=2, eps=eps(eltype(ŷ)))
```

分類タスクで使用できる[焦点損失](https://arxiv.org/pdf/1708.02002.pdf)を返します。これは、クラスの不均衡が大きい場合に役立ちます。正しく分類された例の重みを下げ、難しい例に焦点を当てます。入力の'ŷ'は正規化されていることが期待されます（すなわち、[ソフトマックス](@ref Softmax)の出力）。

調整因子`γ == gamma`は、重み付けの強さを制御します。`γ == 0`の場合、損失は数学的に[`Losses.crossentropy`](@ref)と同等です。

# 例

```jldoctest
julia> y = [1  0  0  0  1
            0  1  0  1  0
            0  0  1  0  0]
3×5 Matrix{Int64}:
 1  0  0  0  1
 0  1  0  1  0
 0  0  1  0  0

julia> ŷ = softmax(reshape(-7:7, 3, 5) .* 1f0)
3×5 Matrix{Float32}:
 0.0900306  0.0900306  0.0900306  0.0900306  0.0900306
 0.244728   0.244728   0.244728   0.244728   0.244728
 0.665241   0.665241   0.665241   0.665241   0.665241

julia> Flux.focal_loss(ŷ, y) ≈ 1.1277571935622628
true
```

バイナリ（ワンホットではない）ラベルについては、[`Losses.binary_focal_loss`](@ref)も参照してください。
