```
Mill.string_start_code()
```

Get the current value of the `string_start_code` parameter used as a code point of the abstract string-start character. The default value of the parameter is `0x02`, which corresponds to the `STX` character in ASCII encoding.

See also: [`Mill.string_start_code!`](@ref), [`Mill.string_end_code`](@ref), [`Mill.string_end_code!`](@ref).
