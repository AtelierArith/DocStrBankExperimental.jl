```
SimpleGMRES(; restart::Bool = true, blocksize::Int = 0, warm_start::Bool = false,
    memory::Int = 20)
```

平方非エルミート線形システムのためのシンプルなGMRES実装です。

この実装は、特化したディスパッチを使用して、均一なサイズの正方行列ブロックを持つブロック対角行列を処理します。

## 引数

  * `restart::Bool`: `true`の場合、ソルバーは`memory`回の反復後に再起動します。
  * `memory::Int = 20`: 再起動前の反復回数。この値は、再起動が無効な場合にメモリを割り当てるために使用され、後でより多くのメモリが必要な場合に拡張されます。
  * `blocksize::Int = 0`: `blocksize`が`> 0`の場合、ソルバーは行列がサイズ`blocksize`の均一なサイズのブロック対角構造を持つと仮定します。このオプションを誤用すると不正確な結果が得られます。

      * これが`≤ 0`に設定され、実行時にブロック対角行列が得られた場合、特化したディスパッチが使用できるかどうかを確認します。

!!! warning
    ほとんどのユーザーは、この実装の代わりに`KrylovJL_GMRES`ソルバーを使用するべきです。


!!! tip
    行列が均一なサイズの正方行列ブロックを持つブロック対角行列であるかどうかを自動的に検出できます。これが当てはまる場合、特化したディスパッチを使用できます。ただし、ほとんどの現代のシステムでは、単一の行列-ベクトル乗算を行う方が、複数の小さな行列-ベクトル乗算を行うよりも速いです（ブロック対角行列の場合）。行列を密にすることをお勧めします（サイズが許す場合）し、`blocksize`引数を指定してください。
