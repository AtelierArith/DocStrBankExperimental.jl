```
radial_cut(r, Θ, h; narc::Int=197)
```

放射状カットを表す多角形をレンダリングします（金属のない放射状スタブのように）。この多角形は、基準面から引かれる必要があります。

パラメータ `h` は、弧の焦点（多角形の上部）を簡単に導波管の中心に合わせることができるため、メソッドシグネチャで `a` の代わりに利用可能です。もし `a` を制御したい場合は、三角関数を使用してください: `a/2 = h*tan(Θ/2)`。

パラメータは以下の通りで、Xが原点を示し、（*原点の上にあるものは結果の多角形の一部ではありません*）:

```
                       Λ
                      /│\
                     / │ \
                    /  |  \
              .    /   │Θ/2\
             .    /    │----\
            /    /   h │     \
           /    /      │      \
          /    /       │       \
         r    /        │        \
        /    /         │         \
       /    /----------X----------\
      /    /{--------- a ---------}\
     .    /                         \
    .    /                           \
        /                             \
       /                               \
      /                                 \
      --┐                             ┌--
        └--┐                       ┌--┘
           └--┐                 ┌--┘
              └--┐           ┌--┘
                 └-----------┘
                 (円弧)
```
