```
calcDynamics!(N::JTSystem)
```

JitterTimeシステム`N`の全体的なシステムダイナミクスを計算します。

全体システム（すべてのサブシステムを含む）の状態をxと仮定します。

連続時間の進化は次の方程式に従います。

```
dx(t) = N.Ac x(t)dt + dvc(t)
```

ここで`vc`は強度`N.Rc`の連続ホワイトノイズです。

離散イベントの進化（システム`S`を実行する際）は、次の方程式に従います。`x+(t_k) = S.Ad x(t_k) + vd(t_k)``ここでvdは分散`S.Rd`のホワイトガウスノイズです。
