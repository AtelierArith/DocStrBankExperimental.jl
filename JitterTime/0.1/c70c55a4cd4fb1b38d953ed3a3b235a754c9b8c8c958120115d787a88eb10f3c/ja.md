```
ContinuousSystem(sysid, A, B, C, D, inputid [, Rc, Qc])
```

連続時間線形システムを作成します

```
u(t) --> sys --> y(t)
         x(t)
```

状態は初めに平均値 `E(x) = 0` と共分散 `V(x) = 0` を持ちます。

## 引数:

  * `sysid`:          このシステムのための一意の正の `Integer` 識別子（任意のものを選択）。他のシステムから参照されるときに使用されます。
  * `[A, B, C, D]`:   厳密に適切で遅延のない連続時間LTIシステムの状態空間形式。
  * `inputid`:        入力システム識別子のベクトル。対応するシステムの出力がこのシステムへの入力として使用されます。このシステムの入力の数は、入力システムの出力の総数と等しくなければなりません。空のベクトル（またはゼロ）は、システムの入力が接続されていないことを示します。
  * `Rc`:             （オプション）連続状態または入力ノイズ強度行列。ノイズベクトルは、x（状態空間システム）のサイズと同じであると仮定されます。デフォルトはゼロと仮定されます。
  * `Qc`:             （オプション）連続コスト関数は `E(Integral [x(t); u(t)]' * Qc * [x(t); u(t)] dt)` （状態空間システムの場合）です。デフォルトはゼロと仮定されます。
