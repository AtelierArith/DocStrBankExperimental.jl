```
DiscreteSystem(sysid, Phi, Gam, C, D, inputid [, Rd, Qc])
```

離散時間線形システムを作成します

```
u(k) --> sys --> y(k)
         x(k)
```

状態/出力は初めに平均値 `E([x; y]) = 0` と共分散 `V([x; y]) = 0` を持ちます。

## 引数:

  * `sysid`:              このシステムのための一意の正の `Integer` 識別子（任意のものを選択）。他のシステムから参照されるときに使用されます。
  * `[Phi, Gam, C, D]`:   状態空間形式の離散時間LTIシステム（`Phi`、`Gam`、および `C` が欠けている場合、静的ゲインとして解釈されます）。
  * `inputid`:            入力システム識別子のベクトル。対応するシステムの出力がこのシステムへの入力として使用されます。このシステムの入力の数は、入力システムの出力の総数と等しくなければなりません。空のベクトル（またはゼロ）は、システムの入力が接続されていないことを示します。
  * `Rd`:                 （オプション）離散状態/出力/入力ノイズ共分散行列。ノイズベクトルは `[x; y]`（状態空間システムの場合）と同じサイズであると仮定されます。システムが実行されるたびにノイズが追加されます。デフォルトはゼロと仮定されます。
  * `Qc`:                 （オプション）連続コスト関数は `E(Integral [x(t); u(t)]' * Qc * [x(t); u(t)] dt)`（状態空間システムの場合）です。システムの実行間で `x(t)` と `y(t)` の両方が一定であることに注意してください。デフォルトはゼロと仮定されます。
