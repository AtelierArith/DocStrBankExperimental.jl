```
infile(record,annotator)
```

This function reads the data from a wbdb file.

Arguments:

  * record: is the name of the record
  * annotator: is the annotator of the record

!!! note
    In order to use the infile function for wfdb files, the WFDB Software Package from Pysionet is required. See [Installation](installation) for more information.
