```
show_agent(sim, Type{T}, [id=0; max=5, neighborstate = []]) 
show_agent(sim, id=0; [max=5, neighborstate = []])
```

エージェントのID `id` に関する詳細情報を表示します。もし `id` が 2^36 未満の値であれば、タイプ T の n 番目のエージェントの情報が表示されます。`id` が 0（デフォルト値）の場合、タイプ T のランダムなエージェントが選択されます。

エージェントのIDを返します（特にランダムなエージェントが選択された場合に便利です）。

キーワード引数:

`max` は、ネットワークごと（方向ごと）に表示される最大エッジ数を制御します。

エッジのソース側にあるエージェントのフィールドが `neighborstate` ベクターにリストされている場合、このフィールドの値も表示されます。
