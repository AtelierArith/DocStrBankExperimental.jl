```
all_agentids(sim, ::Type{T}, [all_ranks=true])
```

この関数は、シミュレーション `sim` のタイプ T のすべてのエージェントの現在の ID のベクターを取得します。

これらの ID は安定しておらず、並列シミュレーション中に変更される可能性があります。たとえば、[`finish_init!`](@ref) を呼び出すことによって（将来的には動的負荷分散を通じて変更されることを期待しています）。

`all_ranks` 引数は、並列シミュレーションで現在のランクだけでなくすべてのランクのエージェントを含めるかどうかを決定します。`all_ranks` が `true` の場合、関数はすべてのランクにわたるすべてのエージェント識別子のベクターを返します。

[`all_agents`](@ref) と `all_agentids` によって返されるエージェントの状態と ID は同じ順序です。

!!! info
    `all_agentids` は、引数 `all_ranks` が true に設定されている場合、遷移関数内で呼び出すことはできません。


他にも [`all_agents`](@ref)、[`add_agents!`](@ref) および [`num_agents`](@ref) を参照してください。
