```
DataFrame(sim::Simulation, T::DataType; types = false, localnr = false)
```

エージェントまたはタイプ `T` のエッジの現在の状態を持つ DataFrame を作成します。デフォルトでは、ID 列は完全な [`AgentID`](@ref) を表示します。可読性のために（および [`show_agent`](@ref) の使用のために）、`localnr` 引数を true に設定することで、最下位の 36 ビットのみを表示することが有用であり、これによりはるかに可読性の高い値が得られます。この場合、エッジのソースおよびターゲットエージェントのタイプ情報はもはや利用できないため、`types` 引数を使用してこれらのエージェントのタイプを含む追加の列を作成できます。

!!! info
    `DataFrame` は、モデル実装によって DataFrame.jl パッケージがインポートされている場合にのみ利用可能です。


!!! warning
    並列シミュレーションでは、`DataFrame` メソッドは現在の MPI ランクに存在するローカルグラフパーティション（エージェントまたはエッジ）のみを変換します。全体の分散グラフのグローバルビューは提供されません。すべてのランクにわたるすべてのエージェントの状態の完全なリストを取得するには、[`all_agents`](@ref) 関数を使用してください。同様に、エッジについては、分散グラフ内のすべてのエッジのグローバルビューを取得するために [`all_edges`](@ref) 関数を使用してください。
