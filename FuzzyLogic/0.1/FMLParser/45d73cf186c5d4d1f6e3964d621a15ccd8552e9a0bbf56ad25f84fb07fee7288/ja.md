```
parse_fml(s::String)::AbstractFuzzySystem
```

Fuzzy Markup Language (FML) の文字列表現からファジィ推論システムを解析します。

### 入力

  * `s::String` – IEEE 1855-2016 標準に準拠した FML でファジィシステムを記述する文字列。

### 注意事項

パーサーは IEEE 1855-2016 に準拠した FML を読み取ることができますが、以下の点に留意してください：

  * Mamdani と Sugeno のみがサポートされています。
  * 演算子 `and` と `or` の定義はルールベースブロックで設定する必要があります。個々のルールでの定義は無視されます。
  * 修飾子はサポートされていません。
