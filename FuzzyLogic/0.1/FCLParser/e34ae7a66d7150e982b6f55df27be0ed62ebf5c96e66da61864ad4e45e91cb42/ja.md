```
parse_fcl(s::String)::AbstractFuzzySystem
```

Fuzzy Control Language (FCL) の文字列表現からファジィ推論システムを解析します。

### 入力

  * `s::String` – IEC 1131-7 標準に準拠した FCL でファジィシステムを記述する文字列。

### 注意事項

パーサーは IEC 1131-7 に準拠した FCL を読み取ることができ、以下の点に留意してください：

  * Sugeno（単一出力を持つシステム）は、デフuzzifier として COGS を使用する必要があります。
  * ファジフィケーションおよびデフuzzification ブロックの両方に `RANGE` キーワードが必要です。
  * 必要な `MAX` アキュムレータのみがサポートされています。
  * デフuzzification のデフォルト値はサポートされていません。
  * オプションのローカル変数はサポートされていません。

上記の例外を除いて、パーサーは標準のすべての必須およびオプション機能（表 6.1-1 および 6.1-2）をサポートしています。さらに、以下の機能もサポートしています：

  * 区分線形関数は任意の数の点を持つことができます。
  * 区分線形関数の点におけるメンバーシップ度は $0$ と $1$ の間の任意の数であることができます。
