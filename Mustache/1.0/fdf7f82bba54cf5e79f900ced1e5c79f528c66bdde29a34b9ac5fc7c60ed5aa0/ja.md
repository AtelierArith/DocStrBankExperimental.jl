```
render([io], tokens, view)
render([io], tokens; kwargs...)
(tokens::MustacheTokens)([io]; kwargs...)
```

トークンのセットをビューでレンダリングし、オプションの `io` オブジェクトを使用して印刷または保存します。

## 引数

  * `io::IO`: オプションの `IO` オブジェクト。
  * `tokens`: Mustache トークン、またはトークンに解析する文字列
  * `view`: ビューは、マスタッシュの波括弧で区切られた未解決のシンボルを検索するためのコンテキストを提供します。ビューは、辞書、モジュール、合成型、ベクター、名前付きタプル、データフレーム、`Tables` オブジェクト、またはキーワード引数によって指定できます。

!!! note
    `render` メソッドは現在エクスポートされていますが、このエクスポートは将来的に非推奨になる可能性があります。
