```
detrend(y, x = [eachindex(y);]; λ = 0, mean_only::Bool = false)
```

信号のトレンドを除去します（平均を除去し、オプションで傾きを除去）。

**引数:**

  * `y`:         長さ`N`の観測データベクトル
  * `x`:         （オプション）`N` x `Nf`の入力データ行列（`Nf`は特徴の数）
  * `λ`:         （オプション）リッジパラメータ
  * `mean_only`: （オプション）trueの場合、平均のみを除去（傾きは除去しない）

**戻り値:**

  * `y`: 長さ`N`の観測データベクトル、トレンドを除去されたもの
