```
map_expand(map_map::Matrix, pad::Int = 1)
```

各エッジにパディングを追加してマップを拡張し、離散フーリエ変換の不連続性を排除します。マップは周期的にするために「ラップアラウンド」されます。パディングはマップを7スムーズ次元に拡張し、上方/下方継続中により高速な高速フーリエ変換アルゴリズムを使用できるようにします。

**引数:**

  * `map_map`: `ny` x `nx` 2D グリッドマップデータ
  * `pad`:     マップエッジに沿った最小パディング（グリッドセル）

**戻り値:**

  * `map_map`: `ny` x `nx` 2D グリッドマップデータ、拡張（パディング）
  * `padx`:    最初のエッジに適用されたx方向のパディング（グリッドセル）
  * `pady`:    最初のエッジに適用されたy方向のパディング（グリッドセル）
