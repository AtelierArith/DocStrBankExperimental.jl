```
get_XYZ0(xyz_file::String,
         traj_field::Symbol = :traj,
         ins_field::Symbol  = :ins_data;
         info::String       = splitpath(xyz_file)[end],
         flight             = 1,
         line               = 1,
         year               = 2023,
         doy                = 154,
         dt                 = 0.1,
         tt_sort::Bool      = true,
         silent::Bool       = false)
```

保存されたCSV、HDF5、またはMATファイルからMagNavに必要な最小データセットを取得します。`XYZ0`フライトデータ構造内のすべてのフィールドは必要ではありません。CSV、HDF5、またはMATファイルに必要な最小データは以下を含みます：

  * `lat`, `lon`, `alt`（位置）
  * `mag_1_c` または `mag_1_uc`（スカラー磁気計測）

他のすべてのフィールドは計算またはシミュレーションできます。

CSVまたはHDF5ファイルが提供される場合、ファイル内の可能な列/フィールドは次のとおりです：

| **フィールド**   | **タイプ** | **説明**                                                                             |
|:----------- |:------- |:---------------------------------------------------------------------------------- |
| `dt`        | スカラー    | 測定時間ステップ [s]                                                                       |
| `tt`        | ベクトル    | 時間 [s]                                                                             |
| `lat`       | ベクトル    | 緯度  [deg]                                                                          |
| `lon`       | ベクトル    | 経度 [deg]                                                                           |
| `alt`       | ベクトル    | 高度  [m]                                                                            |
| `vn`        | ベクトル    | 北方向の速度 [m/s]                                                                       |
| `ve`        | ベクトル    | 東方向の速度 [m/s]                                                                       |
| `vd`        | ベクトル    | 下方向の速度 [m/s]                                                                       |
| `fn`        | ベクトル    | 北方向の特定の力 [m/s]                                                                     |
| `fe`        | ベクトル    | 東方向の特定の力 [m/s]                                                                     |
| `fd`        | ベクトル    | 下方向の特定の力 [m/s]                                                                     |
| `Cnb`       | 3x3xN   | 方向余弦行列（ボディからナビゲーション）[-]、CSVファイルには無効（代わりに`roll`、`pitch`、および`yaw`を使用）                |
| `roll`      | ベクトル    | ロール [deg]                                                                          |
| `pitch`     | ベクトル    | ピッチ [deg]                                                                          |
| `yaw`       | ベクトル    | ヨー [deg]                                                                           |
| `ins_dt`    | スカラー    | INS測定時間ステップ [s]                                                                    |
| `ins_tt`    | ベクトル    | INS時間 [s]                                                                          |
| `ins_lat`   | ベクトル    | INS緯度  [deg]                                                                       |
| `ins_lon`   | ベクトル    | INS経度 [deg]                                                                        |
| `ins_alt`   | ベクトル    | INS高度  [m]                                                                         |
| `ins_vn`    | ベクトル    | INS北方向の速度 [m/s]                                                                    |
| `ins_ve`    | ベクトル    | INS東方向の速度 [m/s]                                                                    |
| `ins_vd`    | ベクトル    | INS下方向の速度 [m/s]                                                                    |
| `ins_fn`    | ベクトル    | INS北方向の特定の力 [m/s]                                                                  |
| `ins_fe`    | ベクトル    | INS東方向の特定の力 [m/s]                                                                  |
| `ins_fd`    | ベクトル    | INS下方向の特定の力 [m/s]                                                                  |
| `ins_Cnb`   | 3x3xN   | INS方向余弦行列（ボディからナビゲーション）[-]、CSVファイルには無効（代わりに`ins_roll`、`ins_pitch`、および`ins_yaw`を使用） |
| `ins_roll`  | ベクトル    | INSロール [deg]                                                                       |
| `ins_pitch` | ベクトル    | INSピッチ [deg]                                                                       |
| `ins_yaw`   | ベクトル    | INSヨー [deg]                                                                        |
| `ins_P`     | 17x17xN | INS共分散行列、シミュレーションデータにのみ関連し、そうでなければゼロ[-]、CSVファイルには無効                                |
| `flux_a_x`  | ベクトル    | フラックスA x方向の磁場 [nT]                                                                 |
| `flux_a_y`  | ベクトル    | フラックスA y方向の磁場 [nT]                                                                 |
| `flux_a_z`  | ベクトル    | フラックスA z方向の磁場 [nT]                                                                 |
| `flux_a_t`  | ベクトル    | フラックスA総磁場 [nT]                                                                     |
| `flight`    | ベクトル    | フライト番号                                                                             |
| `line`      | ベクトル    | 行番号、すなわち`flight`内のセグメント                                                            |
| `year`      | ベクトル    | 年                                                                                  |
| `doy`       | ベクトル    | 年の日                                                                                |
| `diurnal`   | ベクトル    | 測定された日変化、すなわち時間的変動または宇宙天候の影響 [nT]                                                  |
| `igrf`      | ベクトル    | 国際地磁気基準場（IGRF）、すなわちコアフィールド [nT]                                                    |
| `mag_1_c`   | ベクトル    | Mag 1補正済み（クリーン）スカラー磁気計測 [nT]                                                       |
| `mag_1_uc`  | ベクトル    | Mag 1非補正（破損）スカラー磁気計測 [nT]                                                          |

MATファイルが提供される場合、上記のフィールドも提供される可能性がありますが、非INSフィールドは指定された`traj_field` MAT構造内に、INSフィールドは指定された`ins_field` MAT構造内にあり、`ins_`プレフィックスなしである必要があります。これはMATLABコンパニオンがデータを出力する標準的な方法です。

**引数:**

  * `xyz_file`:   フライトデータCSV、HDF5、またはMATファイルのパス/名前（`.csv`、`.h5`、または`.mat`拡張子が必要）
  * `traj_field`: （オプション）使用するMATファイル内の軌道構造フィールド、CSVまたはHDF5ファイルには関連しない
  * `ins_field`:  （オプション）使用するMATファイル内のINS構造フィールド、利用できない場合は`:none`、CSVまたはHDF5ファイルには関連しない
  * `tt_sort`:    （オプション）trueの場合、時間でデータをソート（行ではなく）
  * `silent`:     （オプション）trueの場合、出力なし

`xyz_file`に提供されていない場合：

  * `info`:   （オプション）フライトデータ情報
  * `flight`: （オプション）フライト番号
  * `line`:   （オプション）行番号、すなわち`flight`内のセグメント
  * `year`:   （オプション）年
  * `doy`:    （オプション）年の日
  * `dt`:     （オプション）測定時間ステップ [s]

**戻り値:**

  * `xyz`: `XYZ0`フライトデータ構造
