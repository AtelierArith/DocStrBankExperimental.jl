```
INS{T1 <: Signed, T2 <: AbstractFloat} <: Path{T1, T2}
```

慣性航法システム (INS) 構造体。`Path` のサブタイプ。

| **フィールド** | **型**         | **説明**                                                |
|:--------- |:------------- |:----------------------------------------------------- |
| `N`       | `T1`          | サンプル数 (インスタンス)                                        |
| `dt`      | `T2`          | 測定時間ステップ [s]                                          |
| `tt`      | Vector{`T2`}  | 時間 [s]                                                |
| `lat`     | Vector{`T2`}  | 緯度 [rad]                                              |
| `lon`     | Vector{`T2`}  | 経度 [rad]                                              |
| `alt`     | Vector{`T2`}  | 高度 [m]                                                |
| `vn`      | Vector{`T2`}  | 北方向の速度 [m/s]                                          |
| `ve`      | Vector{`T2`}  | 東方向の速度 [m/s]                                          |
| `vd`      | Vector{`T2`}  | 下方向の速度 [m/s]                                          |
| `fn`      | Vector{`T2`}  | 北方向の特定の力 [m/s]                                        |
| `fe`      | Vector{`T2`}  | 東方向の特定の力 [m/s]                                        |
| `fd`      | Vector{`T2`}  | 下方向の特定の力 [m/s]                                        |
| `Cnb`     | Array{`T2,3`} | `3` x `3` x `N` 方向余弦行列 (ボディからナビゲーション) [-]             |
| `P`       | Array{`T2,3`} | `17` x `17` x `N` 共分散行列、シミュレーションデータにのみ関連し、それ以外はゼロ [-] |
