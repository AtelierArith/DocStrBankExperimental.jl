```
create_TL_coef(Bx, By, Bz, B;
               Bt         = sqrt.(Bx.^2+By.^2+Bz.^2),
               λ          = 0,
               terms      = [:permanent,:induced,:eddy],
               pass1      = 0.1,
               pass2      = 0.9,
               fs         = 10.0,
               pole::Int  = 4,
               trim::Int  = 20,
               Bt_scale   = 50000,
               return_var = false)
```

ベクトルおよびスカラー磁力計測定を使用して、バンドパス、ローパス、またはハイパスフィルタを使用してトレス・ローソン係数を作成します。

**引数:**

  * `Bx`,`By`,`Bz`: ベクトル磁力計測定 [nT]
  * `B`:            スカラー磁力計測定 [nT]
  * `Bt`:           （オプション）ベクトル磁力計測定または修正トレス・ローソンのスカラー磁力計測定の大きさ [nT]
  * `λ`:            （オプション）リッジパラメータ
  * `terms`:        （オプション）使用するトレス・ローソン項 {`:permanent`,`:induced`,`:eddy`,`:bias`}
  * `pass1`:        （オプション）最初のパスバンド周波数 [Hz]
  * `pass2`:        （オプション）2番目のパスバンド周波数 [Hz]
  * `fs`:           （オプション）サンプリング周波数 [Hz]
  * `pole`:         （オプション）バターワースフィルタの極の数
  * `trim`:         （オプション）フィルタリング後にトリムする要素の数
  * `Bt_scale`:     （オプション）誘導および渦電流項のスケーリングファクタ [nT]
  * `return_var`:   （オプション）trueの場合、`B_var`も返す

**戻り値:**

  * `coef`:  トレス・ローソン係数
  * `B_var`: `return_var = true`の場合、フィット誤差分散
