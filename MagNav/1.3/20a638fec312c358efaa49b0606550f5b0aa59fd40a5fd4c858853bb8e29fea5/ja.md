```
get_bpf(; pass1 = 0.1, pass2 = 0.9, fs = 10.0, pole::Int = 4)
```

バターワースバンドパス（またはローパスまたはハイパス）フィルターオブジェクトを作成します。ローパスフィルターの場合は `pass1 = -1`、ハイパスフィルターの場合は `pass2 = -1` に設定します。

**引数:**

  * `pass1`: （オプション）最初のパスバンド周波数 [Hz]
  * `pass2`: （オプション）2番目のパスバンド周波数 [Hz]
  * `fs`:    （オプション）サンプリング周波数 [Hz]
  * `pole`:  （オプション）バターワースフィルターの極の数

**戻り値:**

  * `bpf`: フィルターオブジェクト
