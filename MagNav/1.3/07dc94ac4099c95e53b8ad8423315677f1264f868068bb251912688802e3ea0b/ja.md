```
MapS3D{T2 <: AbstractFloat} <: Map{T2}
```

3D（多層）スカラー磁気異常マップ構造体。`Map`のサブタイプ。

| **フィールド** | **型**         | **説明**                                    |
|:--------- |:------------- |:----------------------------------------- |
| `info`    | String        | マップ情報                                     |
| `map`     | Array{`T2,3`} | `ny` x `nx` x `nz` 3D（多層）スカラー磁気異常マップ [nT] |
| `xx`      | Vector{`T2`}  | `nx` マップ x方向（経度）座標 [rad]                  |
| `yy`      | Vector{`T2`}  | `ny` マップ y方向（緯度）座標 [rad]                  |
| `alt`     | Vector{`T2`}  | `nz` マップ高度レベル [m]                         |
| `mask`    | BitArray{3}   | `ny` x `nx` x `nz` 有効（未記入）マップデータのマスク      |
