```
FILTres{T2 <: AbstractFloat}
```

フィルタ結果構造体。

| **フィールド** | **型**         | **説明**                             |
|:--------- |:------------- |:---------------------------------- |
| `x`       | Matrix{`T2`}  | フィルタされた状態、すなわち E(x*t   y*1,..,y_t) |
| `P`       | Array{`T2,3`} | 非線形共分散行列                           |
| `r`       | Matrix{`T2`}  | 測定残差 [nT]                          |
| `c`       | `Bool`        | true の場合、フィルタが収束したことを示す            |
