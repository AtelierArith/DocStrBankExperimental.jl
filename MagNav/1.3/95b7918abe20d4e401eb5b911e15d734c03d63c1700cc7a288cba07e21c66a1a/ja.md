```
get_ins(ins_file::String, field::Symbol = :ins_data;
        dt            = 0.1,
        tt_sort::Bool = true,
        silent::Bool  = false)
```

保存されたCSV、HDF5、またはMATファイルから慣性航法システムデータを取得します。必要なフィールドは`ins_lat`、`ins_lon`、および`ins_alt`（位置）のみです。

CSVまたはHDF5ファイルが提供される場合、ファイル内の可能な列/フィールドは次のとおりです：

| **フィールド**   | **タイプ** | **説明**                                                                             |
|:----------- |:------- |:---------------------------------------------------------------------------------- |
| `ins_dt`    | スカラー    | INS測定時間ステップ [s]                                                                    |
| `ins_tt`    | ベクトル    | INS時間 [s]                                                                          |
| `ins_lat`   | ベクトル    | INS緯度  [度]                                                                         |
| `ins_lon`   | ベクトル    | INS経度 [度]                                                                          |
| `ins_alt`   | ベクトル    | INS高度  [m]                                                                         |
| `ins_vn`    | ベクトル    | INS北方向の速度 [m/s]                                                                    |
| `ins_ve`    | ベクトル    | INS東方向の速度 [m/s]                                                                    |
| `ins_vd`    | ベクトル    | INS下方向の速度 [m/s]                                                                    |
| `ins_fn`    | ベクトル    | INS北方向の特定の力 [m/s]                                                                  |
| `ins_fe`    | ベクトル    | INS東方向の特定の力 [m/s]                                                                  |
| `ins_fd`    | ベクトル    | INS下方向の特定の力 [m/s]                                                                  |
| `ins_Cnb`   | 3x3xN   | INS方向余弦行列（ボディからナビゲーション）[-]、CSVファイルには無効（代わりに`ins_roll`、`ins_pitch`、および`ins_yaw`を使用） |
| `ins_roll`  | ベクトル    | INSロール [度]                                                                         |
| `ins_pitch` | ベクトル    | INSピッチ [度]                                                                         |
| `ins_yaw`   | ベクトル    | INSヨー [度]                                                                          |
| `ins_P`     | 17x17xN | INS共分散行列、シミュレーションデータにのみ関連し、それ以外はゼロ[-]、CSVファイルには無効                                  |

MATファイルが提供される場合、上記のフィールドも提供される可能性がありますが、指定された`field` MAT構造体内にあり、`ins_`プレフィックスは不要です。これはMATLABコンパニオンがデータを出力する標準的な方法です。

**引数:**

  * `ins_file`: INSデータCSV、HDF5、またはMATファイルのパス/名前（`.csv`、`.h5`、または`.mat`拡張子が必要）
  * `field`:    （オプション）使用するMATファイル内の構造体フィールド、CSVまたはHDF5ファイルには関連しない
  * `dt`:       （オプション）測定時間ステップ [s]、`ins_file`にない場合のみ使用
  * `silent`:   （オプション）trueの場合、出力なし

**戻り値:**

  * `ins`: `INS`慣性航法システム構造体
