```
psd(map_map::Matrix, dx, dy)
```

ポテンシャルフィールド（すなわち、磁気異常フィールド）マップのパワースペクトル密度。フーリエ変換における放射状波数（空間周波数）に沿ったスペクトルエネルギー分布を決定するために、高速フーリエ変換を使用します。

**引数:**

  * `map_map`: `ny` x `nx` 2D グリッドマップデータ
  * `dx`:      x方向マップステップサイズ [m]
  * `dy`:      y方向マップステップサイズ [m]

**戻り値:**

  * `map_psd`: `ny` x `nx` 2D グリッドマップデータのパワースペクトル密度
  * `kx`:      `ny` x `nx` x方向放射状波数
  * `ky`:      `ny` x `nx` y方向放射状波数
