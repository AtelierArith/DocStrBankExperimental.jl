```
MapSd{T2 <: AbstractFloat} <: Map{T2}
```

スカラー磁気異常マップ構造体は、ドレープマップ用の追加の高度マップを格納するために使用されます。`Map`のサブタイプです。

| **フィールド** | **型**        | **説明**                        |
|:--------- |:------------ |:----------------------------- |
| `info`    | String       | マップ情報                         |
| `map`     | Matrix{`T2`} | `ny` x `nx` スカラー磁気異常マップ [nT]  |
| `xx`      | Vector{`T2`} | `nx` マップのx方向（経度）座標 [rad]      |
| `yy`      | Vector{`T2`} | `ny` マップのy方向（緯度）座標 [rad]      |
| `alt`     | Matrix{`T2`} | `ny` x `nx` 高度マップ [m]         |
| `mask`    | BitMatrix    | `ny` x `nx` 有効（未記入）マップデータのマスク |
