```
Traj{T1 <: Signed, T2 <: AbstractFloat}
```

軌道構造体、すなわちGPSまたはその他の真実の飛行データ。`Path`のサブタイプ。

| **フィールド** | **型**         | **説明**                                  |
|:--------- |:------------- |:--------------------------------------- |
| `N`       | `T1`          | サンプル数（インスタンス数）                          |
| `dt`      | `T2`          | 測定時間ステップ [s]                            |
| `tt`      | Vector{`T2`}  | 時間 [s]                                  |
| `lat`     | Vector{`T2`}  | 緯度 [rad]                                |
| `lon`     | Vector{`T2`}  | 経度 [rad]                                |
| `alt`     | Vector{`T2`}  | 高度 [m]                                  |
| `vn`      | Vector{`T2`}  | 北向き速度 [m/s]                             |
| `ve`      | Vector{`T2`}  | 東向き速度 [m/s]                             |
| `vd`      | Vector{`T2`}  | 下向き速度 [m/s]                             |
| `fn`      | Vector{`T2`}  | 北向き特定力 [m/s]                            |
| `fe`      | Vector{`T2`}  | 東向き特定力 [m/s]                            |
| `fd`      | Vector{`T2`}  | 下向き特定力 [m/s]                            |
| `Cnb`     | Array{`T2,3`} | `3` x `3` x `N` 方向余弦行列（ボディからナビゲーション）[-] |
