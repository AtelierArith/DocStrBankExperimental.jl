```
corrupt_mag(mag_c, flux;
            dt           = 0.1,
            cor_sigma    = 1.0,
            cor_tau      = 600.0,
            cor_var      = 1.0^2,
            cor_drift    = 0.001,
            cor_perm_mag = 5.0,
            cor_ind_mag  = 5.0,
            cor_eddy_mag = 0.5)
```

ランダム、FOGM、ドリフト、およびトレス-ローソンノイズを使用して補償された（クリーンな）磁力計測定を汚染し、非補償（汚染された）スカラー磁力計測定を作成します。FOGMは一次ガウス-マルコフ確率過程です。

**引数:**

  * `mag_c`:        補償された（クリーンな）スカラー磁力計測定 [nT]
  * `flux`:         `MagV`ベクトル磁力計測定構造体
  * `dt`:           （オプション）測定時間ステップ [s]
  * `cor_sigma`:    （オプション）汚染FOGMキャッチオールバイアス [nT]
  * `cor_tau`:      （オプション）汚染FOGMキャッチオール時間定数 [s]
  * `cor_var`:      （オプション）汚染測定（ホワイト）ノイズ分散 [nT^2]
  * `cor_drift`:    （オプション）汚染測定線形ドリフト [nT/s]
  * `cor_perm_mag`: （オプション）汚染永久場TL係数標準偏差
  * `cor_ind_mag`:  （オプション）汚染誘導場TL係数標準偏差
  * `cor_eddy_mag`: （オプション）汚染渦電流TL係数標準偏差

**戻り値:**

  * `mag_uc`:   非補償（汚染された）スカラー磁力計測定 [nT]
  * `TL_coef`:  トレス-ローソン係数（部分的に）`mag_uc`を作成するために使用される
  * `cor_fogm`: 汚染FOGM部分 [nT]
