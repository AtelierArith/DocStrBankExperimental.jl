```
MapS{T2 <: AbstractFloat} <: Map{T2}
```

スカラー磁気異常マップ構造体。`Map`のサブタイプ。

| **フィールド** | **型**        | **説明**                        |
|:--------- |:------------ |:----------------------------- |
| `info`    | String       | マップ情報                         |
| `map`     | Matrix{`T2`} | `ny` x `nx` スカラー磁気異常マップ [nT]  |
| `xx`      | Vector{`T2`} | `nx` マップのx方向（経度）座標 [rad]      |
| `yy`      | Vector{`T2`} | `ny` マップのy方向（緯度）座標 [rad]      |
| `alt`     | `T2`         | マップの高度 [m]                    |
| `mask`    | BitMatrix    | `ny` x `nx` 有効（未記入）マップデータのマスク |
