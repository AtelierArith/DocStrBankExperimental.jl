```
Map_Cache
```

マップキャッシュ構造体、可変。

| **フィールド**        | **型**                     | **説明**                                           |
|:---------------- |:------------------------- |:------------------------------------------------ |
| `maps`           | Vector{`MapS`{`Float64`}} | `MapS`スカラー磁気異常マップ構造体のベクター                        |
| `map_sort_ind`   | Vector{`Int64`}           | 高度によってソートされた`maps`のインデックス                        |
| `fallback`       | `MapS`{`Float64`}         | フォールバック`MapS`スカラー磁気異常マップ構造体                      |
| `map_cache`      | Dict                      | 複数の高度でのスカラーマップ補間関数（`f(lat,lon)`）の`maps`キャッシュ     |
| `fallback_cache` | Dict                      | 複数の高度でのスカラーマップ補間関数（`f(lat,lon)`）の`fallback`キャッシュ |
| `dz`             | Real                      | マップ高度レベル間のステップサイズ [m]                            |
