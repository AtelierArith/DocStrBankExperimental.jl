```
downward_L(map_map::Matrix, dx, dy, dz, α::Vector;
           map_mask::BitMatrix = map_params(map_map)[2],
           expand::Bool        = true)
```

正則化パラメータのシーケンスを使用して特性L曲線を作成するための下方継続。最適な正則化パラメータはL曲線の局所的な最小値にあり、これは曲率の局所的な最大値です。曲率のグローバルな最大値が最適な正則化パラメータであるかどうかは不明です。

**引数:**

  * `map_map`:   `ny` x `nx` 2D グリッドマップデータ
  * `dx`:        x方向のマップステップサイズ [m]
  * `dy`:        y方向のマップステップサイズ [m]
  * `dz`:        z方向の上方/下方継続距離 [m]
  * `α`:        (幾何学的) 正則化パラメータのシーケンス
  * `map_mask`: (オプション) 有効な（未入力の）マップデータのための `ny` x `nx` マスク
  * `expand`:   (オプション) true の場合、一時的にマップを拡張してエッジ効果を減少させる

**戻り値:**

  * `norms`: 連続したD.C.解の間の差のL-無限大ノルム
