```
synth(kwargs...)
```

この関数は合成スペクトルを作成します。 [`synthesize`](@ref) よりも使いやすいですが、制御が少なくなります。 [`synthesize`](@ref) とは異なり、**`(wavelengths, rectified_flux, cntm)`** のタプルを返します（波長はÅ、正規化されたフラックスは0から1の間の無次元数、連続体はerg/s/cm^5）。 `Korg.synth` は、スペクトルを後処理するためのいくつかのショートカットも提供します（LSFの適用、回転など）。

# キーワード引数

  * `Teff`: 効力温度（K単位）（デフォルト: 5000）
  * `logg`: 表面重力（cgs単位）（デフォルト: 4.5）
  * `m_H`: 金属量、[metals/H]（デフォルト: 0.0）（この解釈の詳細については、[`format_A_X`](@ref) を参照してください。）
  * `alpha_H`: アルファ強化、[α/H]（デフォルト: `m_H`）（この解釈の詳細については、[`format_A_X`](@ref) を参照してください。）
  * *任意の原子記号*（例: `Fe` または `C`）を使用して、（太陽相対的な、[*X*/H]）の豊富さを指定できます。これらは `m_H` と `alpha_H` を上書きします。個別の豊富さを指定すると、真の金属量とアルファが `m_H` と `alpha_H` の値と正確に一致しないことになります。詳細については、[`format_A_X`](@ref) を参照してください。
  * `linelist`: リスト（デフォルト: [`get_VALD_solar_linelist()`](@ref)）。また、[`read_linelist`](@ref) も参照してください。
  * `wavelengths`: 開始波長と終了波長のタプル（デフォルト: (5000, 6000)）、または `(λstart, λstop)` のペアのベクトル。波長を指定する方法の詳細については、[`Wavelengths`](@ref) を参照してください。
  * `rectify`: スペクトルを正規化するかどうか（デフォルト: true）
  * `R`: 解像度（デフォルト: `Inf`、LSFは適用されません）。 `R` はスカラーまたは波長（Å単位）から解像度に関する関数であることができます。手動でこれを行う方法の詳細については、[`apply_LSF`](@ref) を参照してください。
  * `vsini`: 投影回転速度（km/s単位）（デフォルト: 0）。手動でこれを行う方法の詳細については、[`apply_rotation`](@ref) を参照してください。
  * `vmic`: マイクロ乱流速度（km/s単位）（デフォルト: 1.0）。
  * `synthesize_kwargs`: [`synthesize`](@ref) に渡す追加のキーワード引数。
  * `format_A_X_kwargs`: [`format_A_X`](@ref) に渡す追加のキーワード引数。
