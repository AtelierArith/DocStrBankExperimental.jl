```
eccentricity(g[, v][, distmx])
eccentricity(g[, vs][, distmx])
```

頂点 / 頂点リスト `v` または頂点の集合 `vs` の偏心率を返します。デフォルトではグラフ全体が対象です。オプションでエッジの距離の行列を指定できます。指定しない場合、エッジの距離は `1` にデフォルト設定されます。

頂点の偏心率は、その頂点とグラフ内の他のすべての頂点との間の最長の最短経路の距離です。

出力は、単一の頂点が提供された場合は単一の浮動小数点数、頂点ベクトルが提供された場合は浮動小数点数のベクトルになります。頂点ベクトルが提供されない場合、返されるベクトルはグラフ内の各頂点に対応します。

### パフォーマンス

この関数は、引数リストに提供されたすべての頂点の最短経路を計算する必要があるため、時間がかかる場合があります。

### 実装ノート

`eccentricity()` によって返される偏心率ベクトルは、以下の距離測定の残りの部分の入力として使用できます。偏心率ベクトルが提供されている場合、それが使用されます。そうでない場合、関数の各呼び出しに対して偏心率ベクトルが計算されます。したがって、複数の距離測定が必要な場合は、偏心率を計算して保存し、渡す方が効率的です。

無限の経路長は、距離行列の `typemax` で表されます。

# 例

```jldoctest
julia> using Graphs

julia> g = SimpleGraph([0 1 0; 1 0 1; 0 1 0]);

julia> eccentricity(g, 1)
2

julia> eccentricity(g, [1; 2])
2-element Vector{Int64}:
 2
 1

julia> eccentricity(g, [1; 2], [0 2 0; 0.5 0 0.5; 0 2 0])
2-element Vector{Float64}:
 2.5
 0.5
```
