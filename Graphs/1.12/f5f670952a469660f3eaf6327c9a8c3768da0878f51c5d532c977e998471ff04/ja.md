```
karger_min_cut(g)
```

グラフ `g` の最小カットを見つけるために[Karger Minimum Cut](https://en.wikipedia.org/wiki/Karger%27s_algorithm)を実行します。カットは `vertices(g)` を2つの非空の集合に分割することです。カットのサイズは、2つの非空の集合を横切るエッジの数です。

### 実装ノート

カットは整数配列で表されます。もし `cut[v] == 1` ならば `v` は最初の非空集合に属します。もし `cut[v] == 2` ならば `v` は2番目の非空集合に属します。`cut[1] = 1` です。

もし |V| < 2 ならば、すべての `v` に対して `cut[v] = 0` です。

### パフォーマンス

実行時間: O(|E|) メモリ: O(|E|)
