| :desargues                | [デサルググラフ](https://en.wikipedia.org/wiki/Desargues_graph)。                               |
| :diamond                  | [ダイヤモンドグラフ](http://en.wikipedia.org/wiki/Diamond_graph)。                                |
| :dodecahedral             | [プラトン十二面体グラフ](https://en.wikipedia.org/wiki/Platonic_graph)。                            |
| :frucht                   | [フルヒトグラフ](https://en.wikipedia.org/wiki/Frucht_graph)。                                  |
| :heawood                  | [ヒーウッドグラフ](https://en.wikipedia.org/wiki/Heawood_graph)。                                |
| :house                    | 家のクラシックな輪郭を模したグラフ。                                                                      |

| :karate                   | [ザカリーの空手クラブ](https://en.wikipedia.org/wiki/Zachary%27s_karate_club)と呼ばれるソーシャルネットワークグラフ。 |

| :pappus                   | [パッパスグラフ](http://en.wikipedia.org/wiki/Pappus_graph)。                                   |
| :petersen                 | [ピーターセングラフ](http://en.wikipedia.org/wiki/Petersen_graph)。                               |
