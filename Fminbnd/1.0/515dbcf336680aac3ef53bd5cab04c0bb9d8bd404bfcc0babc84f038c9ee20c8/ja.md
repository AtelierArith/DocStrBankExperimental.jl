```
使用される方法は、黄金比セクション探索と
```

連続的な放物線補間の組み合わせです。 収束は、フィボナッチ探索のそれよりも遅くなることはありません。 f が最小値で正の連続二次導関数を持つ場合（ax または bx ではない）、収束は超線形であり、通常は約 1.324 のオーダーです。関数 f は、eps*abs(fmin) + (tol/3) よりも近い2点で評価されることはありません。ここで、eps は相対的な機械精度の平方根にほぼ相当します。 f が単調関数であり、計算された f の値が常に eps*abs(x) + (tol/3) 以上離れている場合、fmin は区間 ax,bx における f のグローバル最小値の abcissa を 3*eps*abs(fmin) + tol よりも小さい誤差で近似します。 f が単調でない場合、fmin は同じ精度で局所的な最小値（おそらく非グローバル）を近似することがあります。この関数サブプログラムは、リチャード・ブレントの「導関数なしの最小化アルゴリズム」に記載されている algol 60 手続き localmin のわずかに修正されたバージョンです。プレンティス・ホール社（1973年）。
