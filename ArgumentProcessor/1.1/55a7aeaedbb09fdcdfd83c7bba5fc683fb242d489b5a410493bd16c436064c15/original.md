```
Varformat
```

must be one of:

  * `"%s"`        string
  * `"%f"/"%g"`   decimal float
  * `"%c"`        complex float
  * `"%h"`        hexadecimal **integer**
  * `"%o"`        octal **integer**
  * `"%b"`        binary **integer**
  * `"%d"`        integer
  * `"%l"`        logical (true, false, 0 or 1)
