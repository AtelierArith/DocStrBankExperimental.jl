```
ClosedCurve(f; tol=<default>)
ClosedCurve(f,a,b; tol=<default)
```

複素値関数 `point` から `ClosedCurve` オブジェクトを構築します。この関数は引数を区間 [0,1] で受け取ります。コンストラクタは `f(0)≈f(1)` が許容誤差 `tol` で成り立つかどうかをチェックします。もし `a` と `b` が指定されている場合、それらは供給された `f` への呼び出しにおけるパラメータの限界となります。しかし、結果として得られるオブジェクトは [0,1] で定義され、内部的には [a,b] にスケーリングされます。

```
ClosedCurve(f,df[,a,b]; tol=<default>)
```

複素値関数 `f` と `df` によって与えられる点の位置と接線を持つ閉曲線を構築します。オプションで、パラメータの限界を指定することができます。
