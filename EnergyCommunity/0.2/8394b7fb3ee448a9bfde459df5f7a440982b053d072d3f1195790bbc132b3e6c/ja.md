```
create_example_data(parent_folder, config_name::String = "default")
```

エネルギーコミュニティモデルの例データを作成します。この関数は、指定されたフォルダーに、指定された構成名に基づいてエネルギーコミュニティモデルを実行するために必要なデータを作成します。

## パラメータ

  * parent_folder : AbstractString   例データが作成される親フォルダー
  * config_name : String   例データを作成するために使用される構成名   サポートされている値: "default"
