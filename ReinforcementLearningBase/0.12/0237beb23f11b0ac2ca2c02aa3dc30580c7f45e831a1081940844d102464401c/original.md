[ReinforcementLearningBase.jl](https://juliareinforcementlearning.org/docs/rlbase/) (**RLBase**) provides some common constants, traits, abstractions and interfaces in developing reinforcement learning algorithms in Julia. 

Basically, we defined the following two main concepts in reinforcement learning:
