p-adic環上の局所属。

奇素数 `= p` の成分 `p^m A` の属記号は `(m,n,d)` の形をしており、ここで

  * `m` = 成分の評価
  * `n` = Aの階数
  * `d = det(A) \in \{1,u\}` は正規化された二次非残余 `u` のためのもの。

成分 `2^m A` の属記号は `(m, n, s, d, o)` の形をしており、ここで

  * `m` = 成分の評価
  * `n` = `A` の階数
  * `d` = `det(A)` は `{1,3,5,7}` の中の値
  * `s` = 偶数の場合は 0 (または奇数の場合は 1)
  * `o` = `A` の奇数性 (= s = 0 の場合は 0) は `Z/8Z` における `A` の対角化のトレース

属記号は、Jordanブロック `A_1,...,A_t` の各々に対して `m` によって順序付けられたそのような記号のリストです。

参考文献: [CS99](@cite) 第15章、第7節。

# 引数

  * `prime`: 素数
  * `symbol`: 整数のリストのリストとして与えられるJordanブロック `A_t,...,A_t` の不変量のリスト
