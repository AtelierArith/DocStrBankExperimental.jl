```
class_group(O::AbsSimpleNumFieldOrder; bound = -1,
                      redo = false,
                      GRH = true)   -> FinGenAbGroup, Map
```

グループ $A$ と、$A$ から $O$ のイデアルの集合への写像 $f$ を返します。写像の逆は、主イデアルのグループを除いたイデアルのグループへの射影です。

デフォルトでは、正しさは一般化リーマン予想 (GRH) を仮定した場合にのみ保証されます。

キーワード引数:

  * `redo`: 再計算をトリガーし、キャッシュを回避します。
  * `bound`: 指定された場合、これは因子基準のための上限として使用されます。
  * `GRH`: `false` の場合、結果の正しさは GRH に依存しません。
