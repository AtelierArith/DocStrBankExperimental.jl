```
genus(S::Vector{HermLocalGenus}, signatures) -> HermGenus
```

ローカル属記号のセット `S` と署名のセット `signatures` が与えられたとき、対応するグローバル属記号 `G` を返します。`S` のローカル属記号は同じ階数でなければならず、それがしたがって `G` の階数でもあります。

このコンストラクタは、`S` が空でないこと、署名が非負であること、ローカル不変量がヒルベルト記号の積公式を尊重することを要求します。

`signatures` は、無限の場所をキーとし、対応する署名を値とする辞書であるか、またはタプルのリスト $(::InfPlc, ::Int)$ であることができます。
