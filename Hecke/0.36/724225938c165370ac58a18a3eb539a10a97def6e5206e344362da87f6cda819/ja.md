```
division_polynomial_univariate(E::EllipticCurve, n::Int, [x]) -> Poly
```

楕円曲線 E の n 次単変数除法多項式を、Mazur と Tate に従って計算します。デフォルトでは、結果は `E` の基底環上の単変数多項式です。`x` が与えられた場合、出力は与えられた値 `x` を使用して評価されます。

オブジェクトの三重項が返されます：

  * 非ゼロの二重トーション点での重複度 2 を持つ単変数多項式としての n 次除法多項式。
  * n が偶数のとき、単変数 2 トーション多項式で割った n 次除法多項式。
  * 補完因子、すなわち最初の出力を第二の出力で割ったもの。
