```
order(A::AbstractAssociativeAlgebra{QQFieldElem}, B::Vector{<: AbstractAssociativeAlgebraElem{QQFieldElem}}; check::Bool = true,
      isbasis::Bool = false, cached::Bool = true)
  -> AlgAssAbsOrd
```

Returns the order of $A$ generated by $B$. If `check` is set, it is checked whether $B$ defines an order. If `isbasis` is set, then the elements are assumed to form a $\mathbb Z$-basis.
