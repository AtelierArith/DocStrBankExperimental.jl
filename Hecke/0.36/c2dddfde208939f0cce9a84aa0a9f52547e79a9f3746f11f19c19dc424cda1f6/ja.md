```
is_primary_with_prime(L::ZZLat) -> Bool, ZZRingElem
```

与えられた $\mathbb Z$-ラティス `L` に対して、`L` がプライマリであるかどうかを返します。つまり、`L` が整数であり、その判別群（[`discriminant_group`](@ref）を参照）がある素数 `p` の `p`-群であるかどうかを確認します。もしそうであれば、`p` も第二の出力として返されます。

単位モジュラーラティスの場合、この関数は `(true, 1)` を返します。ラティスがプライマリでない場合、第二の返り値はデフォルトで `-1` です。
