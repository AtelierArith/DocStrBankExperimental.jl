```
quadratic_lattice(K::Field, B::PMat ; gram = nothing,
                                      check:::Bool = true) -> QuadLat
```

擬似行列 `B` のエントリが体 `K` にある場合、擬似行列 `B` によって張られる二次格子を、グラム行列 `gram` を持つ体 `K` 上の二次空間内で返します。

`gram` が指定されていない場合、周囲空間のグラム行列は、`B` の列数のサイズを持つ体 `K` 上の単位行列になります。

デフォルトでは、`B` がフルランクであることが確認されます。このテストは、`check` を false に設定することで無効にできます。
