```
division_polynomial_univariate(E::EllipticCurve, n::Int, [x]) -> Poly
```

楕円曲線 E の n 番目の一変数除法多項式を、Mazur と Tate に従って、体 k 上で定義されたものとして計算します。デフォルトでは、結果は `E` の基底環上の一変数多項式です。`x` が与えられた場合、出力は与えられた値の `x` を使用して評価されます。

オブジェクトの三重項が返されます：

  * 非ゼロの二次トーション点での重複度 2 を持つ一変数多項式としての n 番目の除法多項式。
  * n が偶数のとき、一変数 2-トーション多項式で割った一変数多項式としての n 番目の除法多項式。
  * 補完因子、すなわち最初の出力を第二の出力で割ったもの。
