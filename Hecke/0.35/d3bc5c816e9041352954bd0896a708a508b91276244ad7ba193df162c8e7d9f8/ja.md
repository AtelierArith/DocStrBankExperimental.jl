```
hermitian_lattice(E::NumField, B::PMat; gram = nothing,
			             check::Bool = true) -> HermLat
```

次数2の数体`E`のエントリを持つ擬似行列`B`が与えられたとき、グラム行列`gram`を持つ`E`上のエルミート空間内で擬似行列`B`によって張られるエルミート格子を返します。

`gram`が指定されていない場合、周囲空間のグラム行列は`B`の列数のサイズの`E`上の単位行列になります。

デフォルトでは、`B`がフルランクであることがチェックされます。このテストは、`check`をfalseに設定することで無効にできます。
