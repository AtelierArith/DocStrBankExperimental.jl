```
hyperkaehler_lattice(S::Symbol; n::Int = 2)
```

`S` と `n` によって変形型が決まるハイパーカラーハイポマンフォールド上のボーヴィル-ボゴモロフ-藤木形式に対応する整数格子を返します。

  * `S == :K3` または `S == :Kum` の場合、`n` は 2 より大きい整数でなければなりません。
  * `S == :OG6` または `S == :OG10` の場合、`n` の値は影響を与えません。
