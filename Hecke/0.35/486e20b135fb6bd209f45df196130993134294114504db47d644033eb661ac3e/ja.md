ベクトル空間 $V$ と $W$ の間のスカラーの変化のマップのためのコンテナ型であり、それぞれ非退化のセスキリニア形式を備えています。ここで、$V$ はある数体 $K$ の $K$-ベクトル空間であり、$W$ は有限単純拡張 `$E/K$` の $E$-ベクトル空間です。

注: 現在、$K = \mathbb{Q}$ であり、$E$ が数体である場合のみが利用可能です。

基になるマップ `f` は、実際には $V$ から $W$ へのマップとして考慮されます。したがって、特に、$V$ のある $v \in V$ に対して $f(v)$ はスカラーを $K$ から $E$ に拡張するために使用され、$W$ の $w \in W$ に対する前像 $f\(w)$ はスカラーを $E$ から $K$ に制限するために使用されます。

$$
(a_1, \ldots, a_n)\in E^n
$$

を $K$ の基底とし、$B_V = (v_1, \ldots, v_l)$ を $V$ の $K$-基底、$B_W = (w_1, \ldots, w_m)$ を $W$ の $E$-基底とし、$l = m\times n$ とします。すると、マップ `f` は $V$ の $K$-基底 $(v_1, \ldots, v_l)$ を $W$ の $K$-基底 $(a_1w_1, a_2w_1, \ldots, a_nw_1, a_1w_2, \ldots, a_nw_m)$ に送ることによって、$V$ から $W$ への $K$-線形全単射を定義します。

異なる基底 $B_V$ と $B_W$ を選択することができます。ただし、現時点では、$K = \mathbb{Q}$ に対する $E$ の基底は [`absolute_basis`](@ref) によって固定されています。

デフォルトでは、$B_V$ は $V$ の標準 $K$-基底であり、$B_W$ は $W$ の標準 $E$-基底です。
