```
is_reduced(f::QuadBin{ZZRingElem}) -> Bool
```

`f`が次の意味で還元されているかどうかを返します。`f = [a, b, c]`が判別式`D`を持つとします。

`f`が正定値の場合（`D < 0`かつ`a > 0`）、`f`は`|b| <= a <= c`であり、`a = b`または`a = c`のときに`b >= 0`である場合に限り還元されていると言えます。

`f`が負定値の場合（`D < 0`かつ`a < 0`）、`f`は`[-a, b, -c]`が還元されている場合に限り還元されていると言えます。

`f`が不定の場合（`D > 0`）、`f`は`|sqrt{D} - 2|a|| < b < \sqrt{D}`または`a = 0`かつ`-b < 2c <= b`または`c = 0`かつ`-b < 2a <= b`である場合に限り還元されていると言えます。
