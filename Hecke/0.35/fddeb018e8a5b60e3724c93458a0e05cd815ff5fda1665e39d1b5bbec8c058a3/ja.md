```
isogeny_from_kernel_factored(E::EllipticCurve, psi::RingElem) -> Isogeny
```

入力多項式 $\psi$ の根である $x$-座標を持つ点を含む $\phi$: $E$ -> $E'$ のカーネルを持つイソジェニーを計算します。与えられたカーネルを持つイソジェニーの合成に等しいイソジェニーの配列を返します。因数分解は次のように決定されます。最初の写像はカーネル内の2-トーション点のみを含みます。2-トーションが残っていない場合、残りの写像はイソジェニーの次数の素因数であるすべての素数 $p$ に対して $p$-トーションに含まれるカーネルを持つイソジェニーで構成されます。 ```
