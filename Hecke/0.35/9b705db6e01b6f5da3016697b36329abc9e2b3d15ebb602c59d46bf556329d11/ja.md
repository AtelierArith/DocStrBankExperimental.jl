p-adic環上の局所属。

奇素数 `= p` の成分 `p^m A` の属記号は、次の形 `(m,n,d)` です。

  * `m` = 成分の評価
  * `n` = Aの階数
  * `d = det(A) \in \{1,u\}` 正規化された二次非残余 `u` のため。

成分 `2^m A` の属記号は、次の形 `(m, n, s, d, o)` です。

  * `m` = 成分の評価
  * `n` = `A` の階数
  * `d` = `det(A)` in `{1,3,5,7}`
  * `s` = 偶数の場合は 0 (または 1) 奇数の場合
  * `o` = `A` の奇数性 (= s = 0 の場合は 0) in `Z/8Z`     = `A` の対角化のトレース

属記号は、Jordanブロック `A_1,...,A_t` の各々に対して、`m` によって順序付けられたそのような記号のリストです。

参考文献: [CS99](@cite) 第15章、第7節。

# 引数

  * `prime`: 素数
  * `symbol`: 整数のリストのリストとして与えられたJordanブロック `A_t,...,A_t` の不変量のリスト
