```
hybridclades_support(boot_net::Vector{HybridNetwork}, ref_net::HybridNetwork; rooted=false)
```

参照ネットワーク内のハイブリッドノードを、ブートストラップネットワークやネットワークの事後サンプルの配列内のノードと一致させます。すべてのネットワークは完全に解決されており、同じ分類群セットに属している必要があります。`rooted=true`の場合、すべてのネットワークは事前に適切に根付けられていると見なされます。それ以外の場合、各ハイブリッドエッジの起源は根なしの二分割として考慮されます（デフォルト）。

2つのネットワーク内の2つのハイブリッドエッジは、同じ「ハイブリッド」クレード（または受容体）と同じ「ドナークレード」を共有する場合に一致すると言います。ドナークレードはネットワーク内のハイブリッドクレードの姉妹です。ハイブリッドクレードには2つの親エッジがあるため、同時に2つのクレードの姉妹となります：1つはその主要な姉妹（γ>0.5の主要なハイブリッドエッジに従う）で、もう1つはその小さい姉妹（γ<0.5の主要なハイブリッドエッジに従う）です。

特定のハイブリッドノードでこれらのハイブリッドおよび姉妹クレードを計算するには、最初にネットワークから他のすべてのハイブリッドエッジを削除します。次に、ハイブリッドクレードは、いずれかのハイブリッドエッジのハードワイヤードクラスター（子孫）であり、主要/小クレードは主要/小ハイブリッド親の兄弟エッジのハードワイヤードクラスターです。`rooted=false`の場合、姉妹クレードは二分割として考慮されます。

出力：

1. 「ノード」データフレーム（以下を参照）
2. 「エッジ」データフレーム（以下を参照）
3. ハイブリッドまたは姉妹として見つかったすべてのクレードの構成を説明する「クレード」データフレーム。すべての分類群をリストする`taxa`列から始まります。他のすべての列は特定のクレードに対応し、真/偽の値を含みます。`true`は、特定の分類群が特定のクレードに属することを意味します。たとえば、`H1`という名前のクレードがあり、データフレームが`cla`と名付けられている場合、このクレードの分類群のリストは`cla[:taxa][cla[:H1]]`で取得できます。
4. サンプル内の各ネットワークに1行、参照ネットワーク内の各ハイブリッドエッジに対して2列（主要/小）を持つガンマ値の配列。もしこのハイブリッドエッジがサンプルネットワーク内で見つかった場合（すなわち、すべての他のハイブリッドノードを削除した後、同じハイブリッドおよび姉妹クレード）、その継承ガンマ値がここに記録されます。そうでない場合、ガンマエントリは0.0です。
5. 参照ネットワーク内の各ハイブリッドエッジの番号を、上記のガンマ値の配列の列の順序と同じ順序で持つベクター。

「ノード」データフレームは、クレードごとに1行、9列を持ちます：

  * `:clade`: クレードの名前、単一の分類群である場合は分類群名、または参照ネットワーク内のハイブリッドタグ（例：'H1'）
  * `:node`: 参照ネットワーク内のノード番号。このクレードがこのネットワークに存在しない場合は欠落します。
  * `:hybridnode`: 通常は上記と同じノード番号ですが、参照ネットワーク内のハイブリッドクレードの場合は異なります。その場合、ハイブリッドノード番号がここに記載されます。
  * `:edge`: 参照ネットワーク内で見つかった場合、列2のノードの親エッジの番号。そうでない場合は欠落します。
  * `:BS_hybrid`: クレードがハイブリッドクレードであると見なされるサンプルネットワークの割合。
  * `:BS_sister`: クレードがいくつかのハイブリッドクレードの姉妹であると見なされるサンプルネットワークの割合（次の2列の合計）
  * `:BS_major_sister`: クレードがいくつかのハイブリッドクレードの主要な姉妹であると見なされるサンプルネットワークの割合
  * `:BS_minor_sister`: 前述と同じですが、小さい方
  * `:BS_hybrid_samesisters`: クレードがハイブリッドであり、参照ネットワークと同じ姉妹クレードのセットを持つサンプルネットワークの割合。参照ネットワーク内で見つかったハイブリッドクレードにのみ適用され、他のすべてのクレードには欠落します。

「エッジ」データフレームは、各クレードのペアごとに1行、8列を持ちます：

  * `:edge`: 参照ネットワーク内にエッジが存在する場合、ハイブリッドエッジ番号。そうでない場合は欠落します。
  * `:hybrid_clade`: ハイブリッドであると見なされるクレードの名前、'edge'の子孫
  * `:hybrid`: 参照ネットワーク内に存在する場合、そのクレードのノード番号。そうでない場合は欠落します。
  * `:sister_clade`: 'edge'の姉妹であるクレードの名前、すなわちハイブリッドの姉妹である
  * `:sister`: 参照ネットワーク内にある場合、そのクレードのノード番号。
  * `:BS_hybrid_edge`: 'edge'がハイブリッドエッジであると見なされるサンプルネットワークの割合、すなわち、'hybrid'列のクレードがハイブリッドであり、'sister'列のクレードがその姉妹の1つである場合。
  * `:BS_major`: 'edge'が主要なハイブリッドエッジであると見なされるサンプルネットワークの割合、すなわち、'hybrid'がハイブリッドクレードであり、'sister'がその主要な姉妹である場合。
  * `:BS_minor`: 前述と同じですが、小さい方
