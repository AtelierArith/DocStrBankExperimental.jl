```
setgamma!(Edge, new γ, change_other=true)
```

エッジの継承確率 γ を設定します。これはハイブリッドエッジでなければなりません。新しい γ は [0,1] の範囲である必要があります。「パートナー」ハイブリッドエッジの γ はそれに応じて 1-γ に変更されます。フィールド `ismajor` もそれに応じて変更されます。新しい γ が約 0.5 の場合、`Edge` はメジャー親に設定され、そのパートナーはマイナー親に設定されます。

`net` が HybridNetwork オブジェクトである場合、`printedges(net)` はエッジのリストとその γ を表示します。3 番目のハイブリッドエッジ（例えば）の γ を 0.2 に変更するには、`setgamma!(net.edge[3],0.2)` を使用します。これにより、パートナーのハイブリッドエッジの γ は自動的に 0.8 に設定されます。

最後の引数はデフォルトで true です。false の場合：パートナーエッジは更新されません。これは、新しい γ が 0.5 で、パートナーの γ がすでに 0.5 の場合に便利で、その場合 `ismajor` 属性は変更されずに残ることができます。

詳細は [`PhyloNetworks.setmultiplegammas!`](@ref) を参照してください。
