```
shrink2cycles!(net::HybridNetwork, unroot::Bool=false)
```

もし `net` に2サイクルが含まれている場合、サイクルを長さ tA + γt1+(1-γ)t2 + tB の1つのエッジに圧縮し、trueを返します。そうでない場合はfalseを返します。2サイクルとは、同じ親ノードから同じハイブリッド子ノードへの2つの平行なハイブリッドエッジのセットです。

```
       A                A
       | tA             |
     parent             |
       | \              |
t2,1-γ |  | t1,γ        | tA + γ*t1 + (1-γ)*t2 + tB
       | /              |
     hybrid             |
       | tB             |
       B                B
```

2サイクルに関連する長さやガンマのいずれかが欠けている場合、結合された長さも欠けています。もしγが欠けている場合、枝の長さはγ=0.5を使用して計算されます。

もし `unroot` がfalseで、ルートが削除の対象である場合、次数が2以上である限り保持されます。もし `unroot` がtrueで、ルートが削除の対象である場合、次数が3以上である限り保持されます。次数が1のルートノードは、どちらの場合でも削除されます。
