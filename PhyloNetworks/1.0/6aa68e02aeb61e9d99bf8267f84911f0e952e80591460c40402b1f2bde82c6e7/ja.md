```
directedges!(net::HybridNetwork; checkMajor::Bool=true)
```

エッジの属性 `ischild1` をルートの配置に従って更新します。また、既存のハイブリッドエッジの方向と互換性のある他の可能なルート配置のために、エッジの属性 `containroot` も更新します。ハイブリッドノードが正確に1つの主要なハイブリッド親エッジを持つことに依存しますが、`checkMajor` が true の場合はそれを確認します。

警告:

1. ハイブリッドエッジ上の `ischild1` が正しいと仮定しています（どのノードがハイブリッドであるかのアイデンティティを変更しないため）。
2. サイクルをチェックしません（ネットワークのDAGステータスを維持するため）。

ネットワークを返します。ルートが任意のハイブリッドエッジの方向と矛盾している場合は、'RootMismatch' 例外をスローします。
