```
insupport(d::UnivariateDistribution, x::Any)
```

`x` がスカラーの場合、`x` が `d` のサポート内にあるかどうかを返します（例：`insupport(d, x) = minimum(d) <= x <= maximum(d)`）。`x` が配列の場合、`x` のすべての要素が `d` のサポート内にあるかどうかを返します。

一般的なフォールバックメソッドが提供されていますが、`insupport` はより効率的に実行できることが多く、したがって専門的な `insupport` が望まれます。また、サポートが複数の離散区間で構成されている場合は、この関数をオーバーライドする必要があります。
