a = gauss*newton*fit(x, y, fun, ∇fun!, a0[[, eps,] maxiter])

ガウス・ニュートン非線形最小二乗法。ベクトル `x` と `y` が与えられたとき、最小二乗近似を使用してパラメータ `a` を関数 `f` にフィットさせようとします：

$$
y = f(x, a₁, ..., aₙ)
$$

より一般的な近似については、[`gauss_newton_fit`](@ref)を参照してください。

### 引数：

  * `x` x 値のベクトル
  * `y` y 値のベクトル
  * `fun` `fun(x, a)` として呼び出される関数で、`a` はパラメータのベクトルです。
  * `∇fun!` パラメータ `a` に関する導関数を計算する関数
  * `a0` パラメータの初期推定値のベクトル
  * `eps` 収束のための最大残差
  * `maxiter` 収束のための最大反復回数

## 戻り値

収束した配列を持つベクトル。収束が達成されない場合、関数はエラーをスローします。

## フィッティング関数の仕様

フィッティングされるべき関数は、次のシグネチャを持つJulia関数として指定されるべきです：

```julia
fun(x::T, a::AbstractVector{T}) where {T<:Number}
```

各フィッティングパラメータ `a[i]` に関する導関数は、次のシグネチャを持つべきです：

```julia
∇fun!(x::T, a::AbstractVector{T}, df::AbstractVector{T}) where {T<:Number}
```

戻り値は期待されず、導関数は引数 `df` に返されます。

## 初期近似（推定）

初期近似が十分でない場合、発散する可能性があります。

**注意** 0 に近いパラメータには注意してください。初期推定値は決して 0.0 であってはならず、パラメータの初期値は残差を計算するための基準値として使用されます。

## 収束基準

引数 `maxiter` は、実行すべき最大反復回数を指定します。各反復で、

$$
aₖⁿ⁺¹ = aₖⁿ + δₖ
$$

収束は次の条件を満たすときに達成されます：

$$
|δᵏ / aₖ⁰| < ε
$$

## 例

```julia
x = 1.0:10.0
a = [3.0, 2.0, 1.0]
y = a[1] + a[2]*x + a[3]*x^2
fun(x, a) = a[1] + a[2]*x + a[3]*x^2

function ∇fun!(x, a, df) 
    df[1] = 1.0
    df[2] = x
    df[3] = x^2
end

a = gauss_newton_fit(x, y, fun, ∇fun!, [0.5, 0.5, 0.5], 1e-8, 30)
```
