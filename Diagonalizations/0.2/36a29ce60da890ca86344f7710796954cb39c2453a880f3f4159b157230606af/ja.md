```julia
function spForm(P::Union{Mat, Real, Complex})
```

スケーリングされた置換形式からの逸脱の測定値である $n⋅n$ 正方行列 `P` の定義は次のとおりです。

$$
\frac{1}{2(n-1)}\bigg(\sum_{row}1-\beta(row)+\sum_{col}1-\beta(col)\bigg)
$$

,

ここで、`P` の各 *行* および *列* に対して、β は絶対値の最大値を絶対値の合計で割ったものです。

この指標は、各行および列において $P$ が非ゼロ要素を1つだけ持つ場合、すなわち $P$ がスケーリングされた置換行列である場合に $0$ に等しくなります。指標が大きいほど、$P$ はこの形式から遠くなります。

この測定値といくつかの既存の変種は、盲目的なソース分離 / 独立成分分析コミュニティでよく知られており、シミュレーションデータ上で近似的な共同対角化アルゴリズムを比較するために使用されます。実際、$A$ がデータを生成するために使用される近似的な共同対角化器の逆行列であり、$B$ がアルゴリズムによって推定された近似的な共同対角化器である場合、$P=BA$ はスケーリングされた置換行列にできるだけ近くなければなりません（[scale and permutation](@ref)を参照）。

`P` が実数または複素数の場合は 0.0（ゼロ）を返します。

**例:**

```julia
using Diagonalizations, PosDefManifold
# 20 のランダムな可換行列を作成
# それらはすべて同じ固有ベクトルを持つ
Cset=randP(3, 20; eigvalsSNR=Inf, commuting=true)
# 近似的な共同対角化器 (ajd) を推定
a=ajd(Cset)
# ajd は Cset のいずれかの行列の固有ベクトル行列と同等でなければならない
spForm(a.F'*eigvecs(Cset[1]))+1.0≈1.0 ? println(" ⭐ ") : println(" ⛔ ")
```
