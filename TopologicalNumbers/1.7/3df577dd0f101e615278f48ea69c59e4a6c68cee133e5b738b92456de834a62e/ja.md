スー・シュリーファー・ヒーガー模型のハミルトニアン。

```
 SSH(k::T1, p::T2) where {T1<:Real,T2<:Real}
```

# 引数

  * `k::T1`: 一次元波数 `k`。
  * `p::T2`: 以下に定義されたパラメータ。

# 定義

スー・シュリーファー・ヒーガー模型のハミルトニアンは次のように定義されます。

$$
H(k)=\begin{pmatrix}
    0                 & t_{1}+t_{2}e^{-ik} \\
    t_{1}+t_{2}e^{ik} & 0
\end{pmatrix}
$$

ここで、$t_{1}$ と $t_{2}$ はタイトバインディングモデルにおける最近接隣接ホッピングの振幅です。$t_{1}=1$ および $t_{2}=$`p` で無次元化します。

# 例

```julia
julia> SSH(π/3, 0.5)
2×2 Matrix{ComplexF64}:
  0.0+0.0im       1.25-0.433013im
 1.25+0.433013im   0.0+0.0im
```
