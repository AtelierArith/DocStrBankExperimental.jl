```
molarfrac(Tk, HumidityType, x, P)
```

湿度パラメータに基づいて水蒸気のモル分率を計算します。湿度パラメータは以下のいずれかになります：

  * `MolarFrac` - 基本的に恒等関数...
  * `RelHum` - 小数表記の相対湿度（パーセントではありません！）
  * `DewPoint` - ケルビンでの露点温度
  * `HumRat` - 湿度比、kgの蒸気 / kgの乾燥空気
  * `SpecHum` - 水蒸気の比湿 = mv / (mv + ma)
  * `WetBulb` - ケルビンでの湿球温度 - 断熱飽和温度によって推定されます。

通常通り、すべての計算は通常のSI単位を使用して行われます。この関数で使用されるパラメータは次のとおりです：

  * `Tk` - ケルビンでの温度
  * 湿度タイプ - 湿度を特徴付ける上記のリストにあるタイプ
  * 湿度パラメータの値。
  * P - パスカルでの圧力
