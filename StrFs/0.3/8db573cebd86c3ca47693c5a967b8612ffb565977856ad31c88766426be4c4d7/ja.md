```
StrF{S}(::String)
```

`S`バイト未満の文字列で、0で終わるUTF-8を使用して表現されます。

# 内部表現

`bytes`は`SVector{S, UInt8}`として格納され、`S`は最大バイト数です。これらはUTF-8に準拠しているかどうかは検証されません。終端の`0x00`は、**Sバイト未満**の文字列の場合にのみ続きます。

これは、[Stata](https://www.stata.com/help.cgi?dta)の`str#`（"sturfs"）表現に従っています。

# 操作

比較、変換、およびいくつかの操作がサポートされていますが、これは主にストレージタイプとして意図されています。複雑な操作を行う場合は、`String`に変換することをお勧めします。
