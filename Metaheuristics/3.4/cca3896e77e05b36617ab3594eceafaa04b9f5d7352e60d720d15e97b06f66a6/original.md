```
sample(method, [bounds])
```

Return a matrix with data by rows generated by using `method` (real representation) in inclusive interval [0, 1]. Here, `method` can be [`LatinHypercubeSampling`](@ref), [`Grid`](@ref) or [`RandomInBounds`](@ref).

### Example
