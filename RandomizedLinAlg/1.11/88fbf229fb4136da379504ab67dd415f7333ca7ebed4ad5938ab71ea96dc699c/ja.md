```
rcond(A, iters=1)
```

行列の条件数をランダムに推定します。

# 引数

  * `A`: 条件数を推定する行列。正方行列であり、前乗算（`A*⋅`）および解法（`A\⋅`）をサポートする必要があります。
  * `iters::Int = 1`: 実行するパワー反復の回数。

## キーワード

  * `p::Real = 0.05`: 推定が上限として成立しない確率。

# 出力

区間 `(x, y)` は、確率 `1 - p` で `κ(A)` を含みます。

# 実装ノート

[^Dixon1983] は、p と望ましい精度パラメータ `θ=y/x` を考慮して必要なパワー反復の回数を計算することで行うことができる計算としてこれを最初に説明しました。しかし、これらの境界は正確な算術の仮定の下でのみ導出されました。経験的に、`iters≥4` は計算された区間が真の条件数を含まないという誤った結果をもたらすことがあることが確認されています。したがって、この実装では `iters` を明示的にユーザーが制御可能なパラメータとし、そこから精度パラメータを推測し、したがって `κ(A)` を含む区間を得るようにしています。 ```
