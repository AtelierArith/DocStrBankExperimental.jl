```
rnorms(A, iters=1)
```

行列 `A` のノルムをランダムに推定します。

行列 `A` のノルムに対する確率的上限を計算します。

```
ρ = √(‖(A'A)ʲω‖/‖(A'A)ʲ⁻¹ω‖)
```

これは、正規化された `ω` から始めて `iters` ステップの冪法によって生成された `A` のスペクトルノルムの推定値であり、真のノルムに対する下限を次の因子で示します。

```
ρ ≤ α ‖A‖
```

確率 `1 - p` より大きい場合、ここで `p = 4\sqrt(n/(iters-1)) α^(-2iters)` です。

# 引数

  * `A`: ノルムを推定する行列。
  * `iters::Int = 1`: 実行する冪反復の数。

## キーワード

  * `p::Real = 0.05`: 上限が失敗する確率。
  * `At = A'`: `A` の転置。

# 出力

推定値 ‖A‖。

異なる推定器である [`rnorm`](@ref) も参照してください。これは `A'` での前乗算を必要としません。
