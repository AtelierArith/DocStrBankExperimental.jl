```
coordinates = haar_coordinates(height, width, feat)
```

指定されたタイプのHaar-like特徴の座標を含む配列を返します。

haar_coordinates関数の呼び出し元は、画像内の長方形の領域をその高さと幅、および特定の特徴タイプ（例：横に並んだ2つの長方形(:x2)）で指定します。この関数は、すべての可能な位置に対して特定の長方形の構成の座標を計算し、すべての可能な長方形の幅と高さを考慮します。

パラメータ：

  * height        = 座標が計算される近傍/ウィンドウの高さ
  * width         = 座標が計算される近傍/ウィンドウの幅
  * feat          = 検出されるHaar-like特徴のタイプを指定するシンボル
