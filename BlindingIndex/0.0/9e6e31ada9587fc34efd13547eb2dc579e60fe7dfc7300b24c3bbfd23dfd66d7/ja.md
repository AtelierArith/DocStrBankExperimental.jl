```
bijames(x; <キーワード引数>)
```

ジェームズのブラインディングインデックスを計算します。

# 引数

  * `x::Matrix{Int64}`: 2×2 または 3×2 の整数行列のクロス集計カウント
  * `weights::Matrix{Float64}=[0 0.5; 0.5 0; 1 1]`: 代替の重みが指定されない限り、デフォルトの1996年のジェームズの重みを使用します（正しい推測には重み0が割り当てられ、不正確な推測には重み0.5が割り当てられ、わからない推測には重み1が割り当てられます）
  * `conf::Float64=0.95`: 返される信頼区間の信頼レベル
  * `alternative::Symbol=:two`: 返される信頼区間のタイプ（両側、`:two`）または片側（`:less` または `:greater`）

# 戻り値

次の内容を含む名前付きタプル：

  * `bi_est::Float64`: 推定値
  * `bi_se::Float64`: 標準誤差
  * `bi_lcl::Float64`: 下側信頼区間の境界
  * `bi_ucl::Float64`: 上側信頼区間の境界

# 注意事項

クロス集計カウントと重みの形式は次の通りです：

```
         治療  プラセボ
```

治療    xxx        xxx  プラセボ      xxx        xxx  わからない   xxx        xxx

2×2のクロス集計カウント行列が提供される場合、最後の行はゼロで埋められます。
