```
bi(x; <キーワード引数>)
```

ジェームズとバングのブラインディングインデックスを計算します。

# 引数

  * `x::Matrix{Int64}`: クロス集計されたカウントの2×2または3×2の整数行列
  * `weights::Matrix{Float64}=[0 0.5; 0.5 0; 1 1]`: 代替の重みが指定されない限り、デフォルトの1996年ジェームズの重みを使用します（正しい推測には重み0が割り当てられ、不正確な推測には重み0.5が割り当てられ、わからない推測には重み1が割り当てられます）
  * `conf::Float64=0.95`: 返される信頼区間の信頼レベル
  * `alternative::Symbol=:two`: 返される信頼区間のタイプ（両側、`:two`）または片側（`:less`または`:greater`）
  * `groups::Vector{String}=["Treatment", "Placebo"]`: 治療群の名前
  * `output::Bool=true`: trueの場合、出力を表示します

# 戻り値

以下を含む名前付きタプル:

  * `bi_james`: ジェームズのインデックス: 全体の推定値、標準誤差、下信頼区間限界、上信頼区間限界
  * `bi_bang`: バングのインデックス: グループ1および2の推定値、標準誤差、下信頼区間限界、上信頼区間限界

# 注意事項

クロス集計されたカウントと重みの形式は次のとおりです:

```
         Treatment  Placebo
```

Treatment    xxx        xxx  Placebo      xxx        xxx  Don't Know   xxx        xxx

2×2のクロス集計カウント行列が提供されると（治療とプラセボの行）、最後の行（わからない）はゼロで埋められます。
