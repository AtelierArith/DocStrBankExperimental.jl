`PonchonSavarit` provides a set of functions to compute  the number of theoretical stages of a distillation column  using the Ponchon-Savarit method.

Author: Alexandre Umpierre `aumpierre@gmail.com`

Maintainer's repository: `https://github.com/aumpierre-unb/Ponchon-Savarit.jl`
