`qR2S(data::Matrix{Float64}, z::Vector{Float64}, q::Number, R::Number)`

`qR2S`は、平衡状態における液体と蒸気の分率およびそのエンタルピーのx-h-y-H行列、生成物の分率ベクトル、供給品質、塔の上部での再フラックス比、塔の下部での再フラックス比の2つのパラメータを用いて、蒸留塔の底部での再フラックス比をPonchon-Savarit法を用いて計算します。

供給が飽和液体の場合、供給品質q = 1、供給品質はq = 1 - 1e-10にリセットされます。

`qR2S`は、Juliaの`PonchonSavarit`ツールボックスの主要な関数です。

関連情報: `stages`, `refmin`, `qS2R`, `RS2q`.

# 例

アセトンとメタノールの蒸留塔の底部での再フラックス比を計算します。液体と蒸気の分率およびそのエンタルピーの平衡状態に関連する行列が与えられ、蒸留液の組成は93%、供給の組成は41%、底部の組成は7%、供給は飽和液体で、塔の上部での再フラックス比は2です：

```
data=[2.5e-4 3.235 1.675e-3 20.720; # エンタルピー (kcal/mol)
      0.05   2.666 0.267    20.520;
      0.1    2.527 0.418    20.340;
      0.15   2.459 0.517    20.160;
      0.2    2.422 0.579    20.000;
      0.3    2.384 0.665    19.640;
      0.4    2.358 0.729    19.310;
      0.5    2.338 0.779    18.970;
      0.6    2.320 0.825    18.650;
      0.7    2.302 0.87     18.310;
      0.8    2.284 0.915    17.980;
      0.9    2.266 0.958    17.680;
      1.     2.250 1.       17.390];
x=[0.93;0.41;0.07];
S=qR2S(data,x,1,2)
```

酸素と窒素の蒸留塔の底部での再フラックス比を計算します。液体と蒸気の分率およびそのエンタルピーの平衡状態に関連する行列が与えられ、蒸留液の組成は88%、供給の組成は44%、底部の組成は8%、供給品質は55%、塔の上部での再フラックス比は2です：

```
data=[0.    0.420 0.    1.840; # エンタルピー (kcal/mmol)
      0.075 0.418 0.193 1.755;
      0.17  0.415 0.359 1.685;
      0.275 0.410 0.50  1.625;
      0.39  0.398 0.63  1.570;
      0.525 0.378 0.75  1.515;
      0.685 0.349 0.86  1.465;
      0.88  0.300 0.955 1.425;
      1.    0.263 1.    1.405];
x=[0.88;0.44;0.08];
S=qR2S(data,x,0.55,2)
```
