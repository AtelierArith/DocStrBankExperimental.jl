```
リラクゼーション
```

リラクゼーションカスケード（蛍光およびオージェ電子生成）のパラメータを定義するために使用される構造体

# 必須フィールド

  * 該当なし

# オプションフィールド - デフォルト値付き

  * `ηmin::Float64 = 0.001` : 電子カスケードに続く特定のリラクゼーション粒子生成の最小確率。
  * `interaction_types::Dict{Tuple{DataType,DataType},Vector{String}} = Dict((Photon,Electron) => ["P"],(Electron,Electron) => ["P"],(Positron,Electron) => ["P"],(Photon,Photon) => ["P"],(Electron,Photon) => ["P"],(Positron,Photon) => ["P"])` : 相互作用プロセスタイプの辞書で、形式は（入射粒子、出力粒子）=> 関連する相互作用タイプのリストであり、値は次のように対応します：

      * `(Photon,Electron) => ["P"]` : 入射光子によるサブシェルのイオン化に続くオージェ電子の生成（光電効果による）。
      * `(Electron,Electron) => ["P"]` : 入射電子によるサブシェルのイオン化に続くオージェ電子の生成（モラー相互作用による）。
      * `(Positron,Electron) => ["P"]` : 入射陽電子によるサブシェルのイオン化に続くオージェ電子の生成（ババハ相互作用による）。
      * `(Photon,Photon) => ["P"]` : 入射光子によるサブシェルのイオン化に続く蛍光の生成（光電効果による）。
      * `(Electron,Photon) => ["P"]` : 入射電子によるサブシェルのイオン化に続く蛍光の生成（モラー相互作用による）。
      * `(Positron,Photon) => ["P"]` : 入射陽電子によるサブシェルのイオン化に続く蛍光の生成（ババハ相互作用による）。
