```
pixel_shuffle(x, r::Integer)
```

ピクセルシャッフル操作、スケーリングファクター `r` によるアップスケーリング。

`N` 画像を表す4次元配列の場合、この操作は入力 `size(x) == (W, H, r^2*C, N)` を出力サイズ `(r*W, r*H, C, N)` に変換します。`D` 次元データの場合、チャネルとバッチ次元を持つ `ndims(x) == D+2` を期待し、チャネル数を `r^D` で割ります。

高解像度の特徴に向けてアップサンプリングするために超解像ネットワークで使用されます。参考文献: Shi et. al., "Real-Time Single Image and Video Super-Resolution ...", CVPR 2016, https://arxiv.org/abs/1609.05158

```jldoctest
julia> x = [10i + j + channel/10 for i in 1:2, j in 1:3, channel in 1:4, batch in 1:1]

 21.1  22.1  23.1

[:, :, 2, 1] =

 21.2  22.2  23.2

[:, :, 3, 1] =

 21.3  22.3  23.3

[:, :, 4, 1] =

 21.4  22.4  23.4
```
