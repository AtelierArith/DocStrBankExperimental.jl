```
leakyrelu(x, a=0.01) = max(a*x, x)
```

Leaky [Rectified Linear Unit](https://en.wikipedia.org/wiki/Rectifier_(neural_networks)) 活性化関数。係数を明示的に指定することもできます。例えば、`leakyrelu(x, 0.01)`。
