```
elu(x, α=1) = x > 0 ? x : α * (exp(x) - 1)
```

指数線形ユニット活性化関数。["Fast and Accurate Deep Network Learning by Exponential Linear Units"](https://arxiv.org/abs/1511.07289)を参照してください。係数を明示的に指定することもできます。例えば、`elu(x, 1)`のように。
