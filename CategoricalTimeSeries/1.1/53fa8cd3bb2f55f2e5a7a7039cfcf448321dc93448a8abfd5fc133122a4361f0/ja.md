```
IB_optimize(model::IB; abs_thres = 10^-3, rel_thres = 0.01, ct_thres = 5, n_conv = 1000)
```

データのIBモデルを最適化します。キーワード引数は次のとおりです：     - abs*thres : 最適化関数（ラグランジアン）Lの絶対変化の収束のための閾値。     - rel*thres : Lの相対変化の収束のための閾値。     - ct*thres : Lがabs*thresまたはrel*thresより小さい必要があるステップの数、収束を宣言するため。     - n*conv : 最適化アルゴリズムの反復回数の制限。     - scan (bool): 'DIB'アルゴリズムを使用する際、コスト関数の削減につながる場合にクラスタをチェックしてマージします。     - report : 収束がいつどのように達成されたかを報告するかどうか、およびどのクラスタがマージされたかを示します。
