```
detect_motifs(ts, w, d, t = w - d; iters = 1000, tolerance = 0.7)
```

長さ 'w' のモチーフを検出し、偶然よりも頻繁に発生し、'd' の違いまで同一であるものを検出します。入力:

```
w : 探索するモチーフの長さ
d : モチーフ間の許容誤差（違い）
t : 検出に使用するランダム投影のマスクのサイズ
iters : ランダム投影プロセスの反復回数（デフォルトは1000）
tolerance : モチーフ識別の閾値。
```

返り値 :     motifs : 検出されたモチーフに関するすべての有用な情報（位置、頻度、形状など）を含むモチーフインスタンスのリスト。
