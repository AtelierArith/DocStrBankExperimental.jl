```
cramer_coefficient(Serie, Lags::Array{Int64,1})
```

指定された「Lags」値に対するクレーマーのV係数を返します。

クレーマーのvは、与えられたカテゴリ変数のセットと別の変数のセットとの関連の度合いをテストするために、カテゴリデータ分析で使用されます（例：目の色と髪の色）。ここでは、時系列のカテゴリ値と、後の遅延時間での値との関連を測定します。

Vは[0,1]の範囲にあります。0は関連なし、1は完全な関連を示します。これは対称的であり、v(A,B) = v(B,A)となるため、変数間の非対称性に含まれる情報は失われます。

```
input :
- Serie : データを含む時系列
- Lags : Vが計算される遅延を含む配列。整数が与えられた場合、
            単一の点値が返されます。

output :
- V : 指定された遅延に対するvの値。
```
