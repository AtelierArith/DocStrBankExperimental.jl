```
function MTKParameters(sys::AbstractSystem, p, u0 = Dict(); t0 = nothing)
```

システム `sys` のための `MTKParameters` オブジェクトを作成します。`p` (`u0`) はパラメータ（未知数）からその値への記号的マップです。値は他のパラメータ/未知数の値に基づいて評価される記号的表現であることもできます。`u0` はパラメータの値が未知数に依存する場合にのみ必要です。`t0` は初期時間であり、時間依存システムの場合に必要です。記号的表現がシステムの独立変数を使用する場合にのみ必要です。

これには、システムに対して `complete` が呼び出されている必要があります（通常は `structural_simplify` または `@mtkbuild` を介して）およびキーワード `split = true` が渡されている必要があります（これはデフォルトの動作です）。
