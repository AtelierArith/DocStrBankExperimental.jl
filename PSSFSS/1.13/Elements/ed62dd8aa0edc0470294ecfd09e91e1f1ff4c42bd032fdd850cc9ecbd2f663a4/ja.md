```
function manji(; L1, L2, L3, w, s1, s2, ntri, units, a=0, w2=0, CCW=false, orient=0, kwargs...)
```

# 説明:

「まんじ」（スワスティカ形状の日本語）タイプのジオメトリの三角形分割を含む`RWGSheet`型の変数を作成します。返される値は、`s₁`、`s₂`、`β₁`、`β₂`、`ρ`、`e1`、`e2`、`fv`、`fe`、および`fr`のフィールドが適切に初期化されています。

# 引数:

すべての引数はキーワード引数であり、任意の順序で入力できます。

## 必須引数:

  * `L1`、`L2`、`L3`、`w`: ![https://simonp0420.github.io/PSSFSS.jl/stable/assets/manjidef.png](https://simonp0420.github.io/PSSFSS.jl/stable/assets/manjidef.png) の図で定義された幾何学的パラメータ。`a` ≤ `w` の場合、図に示された中心の正方形は存在しません。同様に、`L3` ≤ `2*w` の場合、腕の曲がった部分は幅 `L3` の単一のストリップで構成されます（中央に隙間はありません）。
  * `s1` と `s2`: 単位セル格子ベクトルを含む2ベクトル。
  * `units`: 長さの単位（`mm`、`cm`、`inch`、または `mil`）
  * `ntri`: 希望する三角形の総数。この数はガイド/リクエストであり、実際の数は異なる可能性があります。

## オプション引数:

  * `a::Real=0`: 上記の図で定義された幾何学的パラメータ。`a` ≤ `w` の場合、図に示された中心の正方形は存在せず、腕は構造の中心まで途切れることなく続きます。
  * `w2::Real=0`: 上記の図に示された正方形リングの境界の幅。`w2` ≤ 0 の場合、正方形リングは三角形分割に含まれません。`w2 > 0` は正方形単位セルにのみ許可されます。
  * `L4`: 正方形リングの外側の辺の長さ。`0 < L4 ≤ norm(s1)`。指定されていない場合、`w2 > 0` のとき、`L4` のデフォルト値は単位セルの正方形の寸法です。`L4` が他の部分と干渉しないように十分大きいことを確認するのはユーザーの責任です。
  * `CCW::Bool=false`: デフォルトでは、キラル構造は時計回りの感覚を持っています。`CCW` が `true` の場合、構造は反時計回りになります。
  * `orient::Real=0.0`: 回転していない単位セル内で構造に適用される反時計回りの回転角度（度）。`w2 > 0` の場合に存在する外側の正方形リングは、非ゼロの `orient` 値を使用して回転されません。
  * `class::Char='J'`  クラスを指定します。`'J'` または `'M'` のいずれかです。`'J'` の場合、未知数は電気表面電流であり、ワイヤまたは金属パッチ型FSSをモデル化するために使用されます。`'M'` の場合、未知数は磁気表面電流であり、完全導体平面のスロットまたは開口部をモデル化するために使用されます。
  * `dx::Real=0.0`、`dy::Real=0.0`: これらは、単位セル全体とその内容に適用されるxおよびy方向のオフセットを指定します。長さの単位は `units` キーワードで指定されたものです。
  * `rot::Real=0.0`: 単位セル全体とその内容に適用される反時計回りの回転角度（度）。この回転は、`dx` および `dy` で指定されたオフセットの前に適用されます。
  * `Zsheet::Complex=0.0`: FSS導体の周波数に依存しない表面インピーダンス（[Ω]単位）。クラス `'J'` のシートにのみ指定できます。`Zsheet` が指定されている場合、`sigma`（または `σ`）は指定できません。
  * `sigma` または `σ`: DC、バルク導電率 [S/m]。クラス `'J'` のシートにのみ許可されます。`Zsheet` と同時に指定することはできません。これは、PSSFSSによって各周波数での有効なシート表面インピーダンスを計算するために使用され、グラデーションモデル（Grujić 2022）を使用します。
  * `Rq=0.0`: RMS表面粗さ [m]。クラス `'J'` の場合のみ合法です。`sigma`（または `σ`）が指定されている場合にのみ使用されます。その場合、`sigma` とともに使用され、グラデーションモデルを使用して周波数依存のシートインピーダンスを計算します。デフォルト値の0は滑らかな表面を示します。
  * `disttype::Symbol=:normal`: 表面粗さの確率分布タイプ。デフォルトは `:normal` です。他の合法的な値は `:rayleigh` です。
  * `fufp::Bool`: このキーワードは通常必要ありません。`fufp` は「ユニークな面ペアを見つける」の略です。`true` の場合、コードは三角形分割内でトイプレッツの意味で同等の三角形ペアのクラスを検索します。すなわち、三角形ペア (A,B) と (C,D) が同じ同値クラスに属する場合、ペア (A,B) の6つの頂点は単純な平行移動によってペア (C,D) の頂点と一致させることができます。このような同等のペアが多数存在する場合、同値を利用することで行列のフィル時間が大幅に短縮されます。トレードオフは、それらを特定するのに必要な時間です。デフォルト値は、`strip`、`diagstrip`、`meander`、`manji`、`loadedcross`、`jerusalemcross`、`pixels`、4辺の `polyring` スタイル（構造化メッシュを使用するもの）および `sympixels` では `true` であり、残りのスタイル（非構造化メッシュを使用するもの）では `false` です。
