```
meander(;a::Real, b::Real, h::Real, w1::Real, w2::Real, ntri::Int,
              units::PSSFSSLength, orient=0, kwarg...) --> sheet::RWGSheet
```

# 説明:

メアンダーラインストリップの三角形分割を含む`RWGSheet`型の変数を返します。返される`sheet`は、`s₁`、`s₂`、`β₁`、`β₂`、`ρ`、`e1`、`e2`、`fv`、`fe`、および`fr`のコンポーネントが適切に初期化されています。幾何学的パラメータは以下の図に示されています。

```
  - - - - - - - - - - - - - - - - - - - - - - - - -             ^
 |                                                |             |
 |                                                |             |
 |                                                |             |
 |                                                |             |
 |                                                |             |
 |            <-------- a/2 ------->              |             |
 |               (center-to-center)               |             |
 |                                                |             |
 |          ----------------------------          |  ^    ^     b
 |          |                          |          |  w2   |     |
 |          |                          |          |  |    |     |
 |          | -----------------------  |          |  v    |     |
 |          | |                     |  |          |             |
 |       -->| |<--w1           w1-->|  |<--       |       h     |
 ------------ |                     |  ------------  ^          |
 |            |                     |             |  w2   |     |
 |            |                     |             |  |    |     |
 ------------ - - - - - - - - - - - ---------------  v    v     v

 <-------------------- a ------------------------->
```

`a`と`b`はユニットセルの寸法です。`w1`と`w2`はそれぞれ垂直および水平ストリップの幅です。`h`はメアンダーの全高です。

# 引数:

すべての引数はキーワード引数であり、任意の順序で入力できます。

## 必須引数:

  * `a`,`b`,`h`,`w1`, `w2`: 上記で定義された幾何学的パラメータ。
  * `units`: 長さの単位（`mm`、`cm`、`inch`、または`mil`）
  * `ntri`: 希望する三角形の総数。この値はガイドであり、実際の数は異なる可能性があります。

## オプション引数:

  * `orient::Real=0.0`: 回転角度（度）で、メアンダーラインの向きを回転させるために使用されます。ユニットセルが正方形（すなわち`a` == `b`）の場合にのみ非ゼロ値が許可されます。許可される値は90の正または負の倍数のみです。
  * `class::Char='J'`  クラスを指定します。`'J'`または`'M'`のいずれかです。`'J'`の場合、未知数は電気表面電流であり、ワイヤまたは金属パッチ型FSSをモデル化するために使用されます。`'M'`の場合、未知数は磁気表面電流であり、完全導体平面のスロットまたは開口部をモデル化するために使用されます。
  * `dx::Real=0.0`, `dy::Real=0.0`: これらは、ユニットセル全体とその内容に適用されるxおよびy方向のオフセットを指定します。長さの単位は`units`キーワードで指定されたものです。
  * `rot::Real=0.0`: ユニットセル全体とその内容に適用される反時計回りの回転角度（度）です。この回転は、`dx`および`dy`で指定されたオフセットの前に適用されます。
  * `Zsheet::Complex=0.0`: FSS導体の周波数に依存しない表面インピーダンスで、単位は[Ω]です。クラス`'J'`のシートに対してのみ指定できます。`Zsheet`が指定されている場合、`sigma`（または`σ`）は指定できません。
  * `sigma`または`σ`: DC、バルク導電率 [S/m]。クラス`'J'`のシートに対してのみ許可されます。`Zsheet`と同時に指定することはできません。PSSFSSによって、各周波数での有効なシート表面インピーダンスを計算するために`Rq`と共に使用されます（Gradient Model (Grujić 2022)を使用）。
  * `Rq=0.0`: RMS表面粗さ [m]。クラス`'J'`に対してのみ合法です。`sigma`（または`σ`）が指定されている場合にのみ使用されます。その場合、`sigma`と共に使用して、Gradient Modelを使用して周波数依存のシートインピーダンスを計算します。デフォルト値0は滑らかな表面を示します。
  * `disttype::Symbol=:normal`: 表面粗さの確率分布タイプ。デフォルトは`:normal`です。他の合法的な値は`:rayleigh`です。
  * `fufp::Bool`: このキーワードは通常必要ありません。`fufp`は「ユニークな面ペアを見つける」の略です。真の場合、コードは三角形のペアの同等クラスを探します。すなわち、三角形ペア（A,B）と（C,D）が同じ同値クラスに属する場合、ペア（A,B）の6つの頂点は単純な平行移動によってペア（C,D）の頂点と一致させることができます。そのような同等ペアが多数存在する場合、同値を利用することで行列のフィル時間が大幅に短縮されます。トレードオフは、それらを特定するのに必要な時間です。デフォルト値は`strip`、`diagstrip`、`meander`、`manji`、`loadedcross`、`jerusalemcross`、`pixels`、4辺の`polyring`スタイル（構造化メッシュを使用するもの）、および`sympixels`に対して`true`であり、残りのスタイル（非構造化メッシュを使用するもの）に対しては`false`です。
