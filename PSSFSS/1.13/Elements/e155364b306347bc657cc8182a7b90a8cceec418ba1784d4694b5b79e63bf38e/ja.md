```
rectstrip(;Lx::Real, Ly::Real, Nx::Int, Ny::Int, Px::Real, Py::Real, units::PSSFSSLength, kwargs...)
```

長方形ストリップの三角形分割を含む `RWGSheet` 型の変数を返します。

# 引数:

すべての引数はキーワード引数であり、任意の順序で入力できます。

## 必須引数:

  * `units`:  長さの単位（`mm`、`cm`、`inch`、または `mil`）
  * `Lx` と `Ly`:  ストリップの x および y 方向の長さ。
  * `Px` と `Py`:  長方形ユニットセルの x および y 方向の長さ（周期）。
  * `Nx` と `Ny`:  ストリップを長方形に分割するための x および y 方向の線分の数。各長方形には対角線が追加されて三角形化されます。

## オプション引数:

  * `orient::Real=0.0`:  回転しないユニットセル内でストリップに適用される反時計回りの回転角度（度）。この回転は、`dx` および `dy` で指定されたオフセットや、`rot` で指定されたユニットセルの回転の前に適用されます。
  * `class::Char='J'`  クラスを指定します。`'J'` または `'M'` のいずれかです。`'J'` の場合、未知数は電気表面電流であり、ワイヤまたは金属パッチ型 FSS をモデル化するために使用されます。`'M'` の場合、未知数は磁気表面電流であり、完全導体平面のスロットまたは開口をモデル化するために使用されます。
  * `dx::Real=0.0`, `dy::Real=0.0`:  これらは、ユニットセル全体とその内容に適用される x および y 方向のオフセットを指定します。長さの単位は `units` キーワードで指定されたものです。
  * `rot::Real=0.0`:  ユニットセル全体とその内容に適用される反時計回りの回転角度（度）。この回転は、`dx` および `dy` で指定されたオフセットの前に適用されます。
  * `Zsheet::Complex=0.0`:  FSS 導体の周波数に依存しない表面インピーダンス（単位は [Ω]）。クラス `'J'` のシートにのみ指定できます。`Zsheet` が指定されている場合、`sigma`（または `σ`）は指定できません。
  * `sigma` または `σ`:  DC、バルク導電率 [S/m]。クラス `'J'` のシートにのみ許可されます。`Zsheet` と同時に指定することはできません。これは、PSSFSS によって各周波数での有効なシート表面インピーダンスを計算するために使用され、グラデーションモデル（Grujić 2022）を使用します。
  * `Rq=0.0`:  RMS 表面粗さ [m]。クラス `'J'` の場合のみ合法です。`sigma`（または `σ`）が指定されている場合にのみ使用されます。その場合、`sigma` とともに使用して、グラデーションモデルを使用して周波数依存のシートインピーダンスを計算します。デフォルト値の 0 は滑らかな表面を示します。
  * `disttype::Symbol=:normal`:  表面粗さの確率分布タイプ。デフォルトは `:normal` です。他の合法的な値は `:rayleigh` です。
  * `fufp::Bool`:  このキーワードは通常必要ありません。`fufp` は「ユニークな面ペアを見つける」の略です。真の場合、コードは三角形ペアの同等クラスを探します。すなわち、三角形ペア (A,B) と (C,D) が同じ同値クラスに属する場合、ペア (A,B) の 6 つの頂点は、単純な平行移動によってペア (C,D) の頂点と一致させることができます。このような同等ペアが多数存在する場合、同値を利用することで行列のフィル時間が大幅に短縮されます。トレードオフは、それらを特定するのに必要な時間です。デフォルト値は、`strip`、`diagstrip`、`meander`、`manji`、`loadedcross`、`jerusalemcross`、`pixels`、4 辺の `polyring` スタイル（構造化メッシュを使用するもの）、および `sympixels` では `true` であり、残りのスタイル（非構造化メッシュを使用するもの）では `false` です。
