```
loadedcross(;s1::Vector{<:Real}, s2::Vector{<:Real}, L1::Real, L2::Real, w::Real, 
             ntri::Int, units::PSSFSSLength, kwargs...)
```

# 説明:

"loaded cross" 型のジオメトリの三角形分割を含む `RWGSheet` 型の変数を作成します。返される値は、フィールド `s₁`、`s₂`、`β₁`、`β₂`、`ρ`、`e1`、`e2`、`fv`、`fe`、および `fr` が適切に初期化されています。

以下の（非常に貧弱な）"ascii art" は、ジオメトリパラメータ `L1`、`L2` および `w` の定義を示そうとしています。構造は、水平および垂直の中心線に関する反射に対して対称であり、x軸に対して45度の角度で整列した線に関する反射に対しても対称であることに注意してください。

```
 ^                 ----------------
 |                 |  _________   |
 |                 |  |       |   |
 |                 |  |       |   |
 |                 |  |    -->|   |<--- W
 |                 |  |       |   |
 |                 |  |       |   |
 |     ------------   |       |   -------------
 |     |  |-----------|       |------------|  |
 |     |  |                                |  |
 L1    |  |                                |  |
 |     |  |                                |  |
 |     |  |                                |  |
 |     |  ------------          ------------  |
 |     |-----------   |        |  ------------|
 |                 |  |        |  |
 |                 |  |        |  |
 |                 |  |        |  |
 |                 |  |        |  |
 |                 |  |________|  |
 |                 |              |
 V                 ----------------

                   <---- L2 ------>
```

# 引数:

すべての引数はキーワード引数であり、任意の順序で入力できます。

## 必須引数:

  * `s1` と `s2`:  ユニットセル格子ベクトルを含む2ベクトル。
  * `L1`、`L2`、`w`: 上記で定義されたジオメトリパラメータ。`w ≥ L2/2` を指定することが許可されており、その場合、固体（すなわち、単一接続）クロスが生成されます。その場合、`L2` 次元はクロス部品の幅として使用されます。
  * `units`: 長さの単位（`mm`、`cm`、`inch`、または `mil`）
  * `ntri`: 希望する三角形の総数。この値はガイド/リクエストであり、実際の数は異なる可能性があります。

## オプション引数:

  * `orient::Real=0.0`: ロードクロスの初期頂点を配置するために使用される反時計回りの回転角度（度）。デフォルトは正のx軸上に頂点を配置することです。
  * `class::Char='J'`  クラスを指定します。`'J'` または `'M'` のいずれかです。`'J'` の場合、未知数は電気表面電流であり、ワイヤまたは金属パッチ型FSSをモデル化するために使用されます。`'M'` の場合、未知数は磁気表面電流であり、完全導体平面のスロットまたは開口をモデル化するために使用されます。
  * `dx::Real=0.0`、`dy::Real=0.0`: これらは、ユニットセル全体とその内容に適用されるxおよびy方向のオフセットを指定します。長さの単位は `units` キーワードで指定されたものです。
  * `rot::Real=0.0`: ユニットセル全体とその内容に適用される反時計回りの回転角度（度）。この回転は、`dx` および `dy` で指定されたオフセットの前に適用されます。
  * `Zsheet::Complex=0.0`: FSS導体の周波数に依存しない表面インピーダンス [Ω] 単位。クラス `'J'` のシートにのみ指定できます。`Zsheet` が指定されている場合、`sigma`（または `σ`）は指定できません。
  * `sigma` または `σ`: DC、バルク導電率 [S/m]。クラス `'J'` のシートにのみ許可されます。`Zsheet` と同時に指定することはできません。これは、PSSFSSによって各周波数での効果的なシート表面インピーダンスを計算するために、`Rq` と共に使用されます（Grujić 2022）。
  * `Rq=0.0`: RMS表面粗さ [m]。クラス `'J'` のみ合法です。`sigma`（または `σ`）が指定されている場合にのみ使用されます。その場合、`sigma` と共に使用して、グラデーションモデルを使用して周波数依存のシートインピーダンスを計算します。デフォルト値の0は滑らかな表面を示します。
  * `disttype::Symbol=:normal`: 表面粗さの確率分布タイプ。デフォルトは `:normal` です。他の合法的な値は `:rayleigh` です。
  * `fufp::Bool`: このキーワードは通常必要ありません。`fufp` は "Find Unique Face Pairs" の略です。真の場合、コードは三角形分割内でトイプレッツの意味で同等の三角形ペアのクラスを検索します。すなわち、三角形ペア (A,B) と (C,D) が同じ同値クラスに属する場合、ペア (A,B) の6つの頂点は、単純な平行移動によってペア (C,D) の頂点と一致させることができます。そのような同等のペアが多数存在する場合、同値性を利用することで行列のフィル時間が大幅に短縮されます。トレードオフは、それらを特定するのに必要な時間です。デフォルト値は、`strip`、`diagstrip`、`meander`、`manji`、`loadedcross`、`jerusalemcross`、`pixels`、4辺の `polyring` スタイル（構造化メッシュを使用するもの）および `sympixels` では `true` であり、残りのスタイル（非構造化メッシュを使用するもの）では `false` です。
  * `structuredtri::Bool=true`: 真の場合、三角形分割に構造化メッシュを使用します。偽の場合、PSSFSSバージョン1.2まで標準であった非構造化メッシュ生成器が使用されます。構造化メッシュはより効率的に分析できますが、非構造化メッシュ生成器によって作成される三角形の数は通常 `ntri` に近いです。
