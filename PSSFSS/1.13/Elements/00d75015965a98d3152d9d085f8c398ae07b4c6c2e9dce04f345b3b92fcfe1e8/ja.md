```
sinuous(; arms, b, w, g, sides, ntri, units, s1, s2, kwargs...) --> RWGSheet
```

`RWGSheet`型の変数を返し、図に示されるような曲線要素を表します: ![https://simonp0420.github.io/PSSFSS.jl/stable/assets/sinuousdef.png](https://simonp0420.github.io/PSSFSS.jl/stable/assets/sinuousdef.png)

# 引数:

すべての引数はキーワード引数であり、任意の順序で入力できます。

## 必須引数:

  * `arms::Int`: 構造内のアームの数。
  * `rc::Real > 0`: 中央円の半径。 `rc` は `w` 以上でなければなりません。
  * `b`: 外側のポリゴンリングの外半径を提供する n-ベクトル (n ≥ 1)。エントリは正であり、厳密に増加している必要があり、隣接するリング間の差は `w` を超えなければなりません。
  * `w`: アーム内のトレースの幅。
  * `g`: 隣接するアームを分ける長方形の隙間の幅を含むスカラー。
  * `sides::Int`: リングセクションが作成される背景の正規環状ポリゴンのポリゴンの辺の数 (>= 4)。
  * `ntri::Int`: 希望する三角形の総数。この数はガイド/リクエストであり、実際の数は異なる可能性があります。
  * `units`: 長さの単位 (`mm`, `cm`, `inch`, または `mil`)
  * `s1` と `s2`: 単位セル格子ベクトルを含む 2-ベクトル。

## オプション引数:

  * `orient::Real=0.0`: 最初のアームの中心を位置決めするために使用される反時計回りの回転角度（度）。
  * `w2::Real=0.0`: 包囲する正方形ループ「リム」のトレース幅。 `w2 > 0` は正方形の単位セルに対してのみ許可されます。
  * `L2`: `w2 > 0` の場合に存在する正方形「リム」の外側の寸法（すなわち、完全な辺の長さ）。ユーザーは、リムが構造の曲線アームに干渉しないように十分大きな `L2` を選択する責任があります。 `L2` は正方形の単位セルの寸法以下でなければなりません。指定されていない場合は、単位セルの寸法がデフォルトになります。
  * `c2::Real=0.0`: 包囲する正方形ループ「リム」の隅に表示される小さな正方形の外側の寸法。 `c2==0` の場合、正方形は含まれず、外側のループは単純な正方形ループになります。
  * `class::Char='J'`  クラスを指定します。 `'J'` または `'M'` のいずれかです。 `'J'` の場合、未知数は電気表面電流であり、ワイヤまたは金属パッチ型 FSS をモデル化するために使用されます。 `'M'` の場合、未知数は磁気表面電流であり、完全導体平面のスロットまたは開口部をモデル化するために使用されます。
  * `dx::Real=0.0`, `dy::Real=0.0`: これらは、単位セル全体とその内容に適用される x および y 方向のオフセットを指定します。長さの単位は `units` キーワードで指定されたものです。
  * `rot::Real=0.0`: 単位セル全体とその内容に適用される反時計回りの回転角度（度）。この回転は、`dx` および `dy` で指定されたオフセットの前に適用されます。
  * `Zsheet::Complex=0.0`: FSS 導体の周波数に依存しない表面インピーダンス [Ω] の単位。クラス `'J'` のシートに対してのみ指定できます。 `Zsheet` が指定されている場合、`sigma` (または `σ`) は指定できません。
  * `sigma` または `σ`: DC、バルク導電率 [S/m]。クラス `'J'` のシートに対してのみ許可されます。 `Zsheet` と同時に指定することはできません。これは、PSSFSS によって各周波数での効果的なシート表面インピーダンスを計算するために使用され、勾配モデル（Grujić 2022）を使用します。
  * `Rq=0.0`: RMS 表面粗さ [m]。クラス `'J'` の場合のみ合法です。 `sigma` (または `σ`) が指定されている場合にのみ使用されます。その場合、`sigma` とともに使用され、勾配モデルを使用して周波数依存のシートインピーダンスを計算します。デフォルト値の 0 は滑らかな表面を示します。
  * `disttype::Symbol=:normal`: 表面粗さの確率分布タイプ。デフォルトは `:normal` です。他の合法的な値は `:rayleigh` です。
  * `fufp::Bool`: このキーワードは通常必要ありません。 `fufp` は「ユニークな面ペアを見つける」の略です。真の場合、コードは三角形の対の同等クラスを探すために三角形分割を検索します。すなわち、三角形ペア (A,B) と (C,D) が同じ同値クラスに属する場合、ペア (A,B) の 6 つの頂点は、単純な平行移動によってペア (C,D) の頂点と一致させることができます。このような同等ペアが多数存在する場合、同値を利用することで行列のフィル時間が大幅に短縮されます。トレードオフは、それらを特定するのに必要な時間です。デフォルト値は、`strip`、`diagstrip`、`meander`、`manji`、`loadedcross`、`jerusalemcross`、`pixels`、4 辺の `polyring` スタイル（構造化メッシュを使用するもの）および `sympixels` では `true` であり、残りのスタイル（非構造化メッシュを使用するもの）では `false` です。
