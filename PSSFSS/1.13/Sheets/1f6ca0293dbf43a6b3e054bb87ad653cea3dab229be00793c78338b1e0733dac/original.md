```
RWGSheet
```

A type that represents a zero-thickness sheet of periodically patterned metalization.  Particular instances  are created by calling a constructor function for a specific type of sheet geometry.  These include: `diagstrip`, `jerusalemcross`, `loadedcross`, `manji`, `meander`, `pecsheet`, `pmcsheet`, `polyring`, `rectstrip`, `sinuous`, and `splitring`.
