```
Layer <: Any
```

`Layer`型のインスタンスは、物理構造の単一の誘電層を表します。これは、誘電層の電気的特性を含んでいます。Gblockに含まれていない層の場合、Layerのインスタンスは周期性（逆格子ベクトルを介して）を指定し、層内に存在するフロケモードのモード定数を保存します。

```
Layer(;width::0mm, ϵᵣ=1.0, tanδ=0.0, μᵣ=1.0, mtanδ=0.0)
Layer(;width::0mm, epsr=1.0, tandel=0.0, mur=1.0, mtandel=0.0)
```

指定された電気的特性を持つ`Layer`インスタンスを作成します。すべての引数はオプションのキーワード引数であり、上記のようにデフォルト値が設定されています。これらは任意の順序で指定できます。通常、複合FSSまたはPSSFSS構造の最初と最後の`Layer`は、`Layer()`の呼び出しによって生成され、構造を囲む半無限の真空領域を表します。デフォルトの`width`が`0mm`であることは、位相基準面がこれらの半無限の端`Layer`に隣接する表面に位置することを意味します。

# 引数

  * `width`: 層の幅（すなわち厚さ）を[`Unitful`](https://github.com/timholy/Unitful.jl)の長さ量として表現します。便利のために、このモジュールによって以下の単位接尾辞がエクスポートされています: `m`, `cm`, `mil`, `inch`。例えば、`width=20mil`のように指定できます。`width`は、複合構造の最初および/または最後の層に対して負の値を取ることができ、これは位相基準面を複合構造の内部に向かってシフトさせる効果があります。これは、Ticraの`GRASP`プログラムのTEPファイル生成など、他のプログラムとインターフェースする際に必要になることがあります。
  * `ϵᵣ`または`epsr`: 誘電体の相対誘電率。
  * `tanδ`または`tandel`: 誘電体の損失タンジェント（電気的）。
  * `μᵣ`または`mur`: 誘電体の相対透磁率。
  * `mtanδ`または`mtandel`: 誘電体の損失タンジェント（磁気的）。
