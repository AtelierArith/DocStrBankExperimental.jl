```
@ftuple (x...; y...)
@ftuple (a, x=t, b, y=u)
```

与えられたタプル式から[`FrankenTuple`](@ref)を構築します。このタプル式は、位置引数と名前付き引数の両方を含むことができます。タプルは、位置引数と名前付き引数をセミコロンで区切ることによって、関数シグネチャと同様に「セクション化」することができ、位置引数と名前付き引数は混在させることもでき、任意の順序で発生します。

# 例

```jldoctest
julia> @ftuple (1, 2; a=3, b=4)
FrankenTuple((1, 2), (a = 3, b = 4))

julia> @ftuple (1, a=3, 2, b=4)
FrankenTuple((1, 2), (a = 3, b = 4))
```
