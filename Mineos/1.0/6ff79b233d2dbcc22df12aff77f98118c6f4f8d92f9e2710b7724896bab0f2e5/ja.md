```
eigenfrequencies(m::LinearLayeredModel, freq=1.0; kwargs...) -> freqs
```

モデル `m` の通常モードの固有周波数を計算します。`m` は `SeisModels` の `LinearLayeredModel` でなければなりません。

固有周波数は、キーが放射状の順序 $n$、モードタイプ（`：S` は球状、`：T` はトロイダル、`：C` は内核トロイダル）、および角度の順序 ℓ のタプルである `OrderedDict` として返されます。

例えば、₀S₂を取得するには `freqs[(0,:S,2)]` を実行し、₃T₈の場合は `freqs[(3,:T,8)]` と書きます。構文 `freqs[3,:T,8]` も許可されています。

### キーワード引数

#### モード選択

  * `radial=true`: 放射状モードを計算する（またはしない）。この場合、`nmin` と `nmax` は無視されます。
  * `spheroidal=true`: 球状モードを計算する（またはしない）。
  * `toroidal=true`: トロイダルモードを計算する（またはしない）。
  * `ic_toroidal=true`: 内核トロイダルモードを計算する（またはしない）。

#### 計算パラメータ

  * `eps=1e-10`: 積分スキームの精度。固有周波数は約 `2eps` から `3eps` に対して正確です。`eps` は、100 mHz（10秒以上の周期）未満の周波数の場合は `1e-7` に設定できます。100 mHz から 200 mHz（5秒から10秒の周期）の周波数の場合、`eps` は `1e-12` と `1e-10` の間であるべきです。
  * `wgrav=10`: 重力項が無視される mHz の周波数。
  * `lmin=1`, `lmax=20`: モードの最小および最大角度の順序。
  * `wmin=0.0`, `wmax=166.0`: 考慮するモード固有周波数の最小および最大値。
  * `nmin=0`, `nmax=10`: 計算する最小および最大分散ブランチ番号。
