```
TextField(name; weight=1.0, no_stem=false, phonetic=nothing, kwargs...)
```

Creates a TextField.

# Arguments

  * `name`: Name of the field.

# KeyWords

  * `weight`: Weight of the field.
  * `no_stem`: Stem bool.
  * `phonetic`: Phonetic reference.
  * `kwargs...`: Additional fields defined in the `Field` object.

# Returns

  * `TextField::Field`
