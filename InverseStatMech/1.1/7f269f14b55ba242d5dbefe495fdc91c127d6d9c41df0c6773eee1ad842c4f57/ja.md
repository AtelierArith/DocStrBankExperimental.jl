```
iterative_boltzmann(pot::Function, dim::Int, ρ::Float64, targ_g2::Function, α = 1; n = 500, bin_size = 0.05, r_range = 10)::Function
```

ボルツマン反転法を使用して、ターゲットペア相関関数`targ_g2`に一致するようにペアポテンシャル`pot`を反復的に更新します。

# 引数

  * `pot::Function`: 最適化される初期ペアポテンシャル関数。
  * `dim::Int`: システムの次元。
  * `ρ::Float64`: システム内の粒子の数密度。
  * `targ_g2::Function`: ターゲットペア相関関数 $g_2(r)$。
  * `α::Float64`: ポテンシャルの更新パラメータ。 (オプション、デフォルト: 1)

# キーワード引数（すべてオプション）

  * `n`: シミュレーション内の各ボックスの粒子数。 (デフォルト: 500)
  * `bin_size`: ペア相関関数のヒストグラムのビンサイズ。 (デフォルト: 0.05)
  * `r_range`: ペア相関関数を計算する最大距離。 (デフォルト: 10)
  * `n_threads`: 並列計算に使用するスレッド数。 (デフォルト: 15)
  * `configs_per_thread`: 各スレッドのために生成する構成の数。 (デフォルト: 10)
  * `displace`: メトロポリスモンテカルロシミュレーションにおけるキックサイズ。 (デフォルト: 0.2)。
  * `Ψ_tol`: 停止基準のためのターゲットとシミュレーションされた $g_2$ の間の距離メトリックの許容誤差。 (デフォルト: 0.005)
  * `show_pb`: シミュレーション中に進捗バーを表示するかどうか。 (デフォルト: true)
  * `test`: テストモードで関数を実行するかどうか。 (デフォルト: false)

# 戻り値

  * `test=true`の場合、最適化が成功したかどうかを示すブール値を返します。
  * それ以外の場合、最適化されたペアポテンシャル関数を返します。

# 例

```
optimized_potential = iterative_boltzmann(r -> 0, 2, 1.0, r -> 1 - exp(-π*r^2))
```
