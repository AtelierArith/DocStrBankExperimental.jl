Identification entropy [Ahlswede2006](@cite).

## Description

The identification entropy is the functional

$$
H_I(p) = 2\left( 1 - \sum_{i=1}^N p_i^2 \right).
$$

Details about this entropy definition can be found in [Ahlswede2021](@citet).
