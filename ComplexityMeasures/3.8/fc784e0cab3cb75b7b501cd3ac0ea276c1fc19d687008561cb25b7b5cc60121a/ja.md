```
RegularDownsampling <: MultiScaleAlgorithm
RegularDownsampling(; f::Function = Statistics.mean, scales = 1:8)
```

マルチスケールエントロピー分析のための元のマルチスケールアルゴリズム [Costa2002](@cite) であり、スケール `s` ごとに単一のダウンサンプリングされた時系列を生成します。

## 説明

スカラー値の入力時系列 `x` が与えられたとき、`Regular` マルチスケールアルゴリズムは `x` を長さ `s` の非重複ウィンドウに分割することによってダウンサンプリングおよび粗粒化し、次に関数 `f` を各長さ `s` のウィンドウに適用することによって新しいダウンサンプリングされた時系列 $D_t(s, f)$ を構築します。

ダウンサンプリングされた時系列 `D_t(s)` は `t ∈ [1, 2, …, L]` であり、ここで `L = floor(N / s)` です：

$$
\{ D_t(s, f)  \}_{t = 1}^{L} = \left\{ f \left( \bf x_t \right) \right\}_{t = 1}^{L} =
\left\{
    {f\left( (x_i)_{i = (t - 1)s + 1}^{ts} \right)}
\right\}_{t = 1}^{L}
$$

ここで `f` は長さ `ts-((t - 1)s + 1)` のタプル `xₖ` に適用される要約統計量です。`f` の異なる選択は、文献に現れる異なるマルチスケール手法を生み出します。例えば：

  * `f == Statistics.mean` は元の第一モーメントマルチスケールサンプルエントロピー [Costa2002](@cite) を生成します。
  * `f == Statistics.var` は一般化されたマルチスケールサンプルエントロピー [Costa2015](@cite) を生成し、平均の代わりに第二モーメント（分散）を使用します。

## キーワード引数

  * **`scales`**. ダウンサンプリングレベル。`scales` が整数に設定されている場合、この整数は最大スケール数（すなわちダウンサンプリングのレベル）として扱われ、ダウンサンプリングはレベル `1:scales` に対して行われます。そうでない場合、ダウンサンプリングは提供された `scales` に対して行われます（範囲であるか、特定のスケール（例：`scales = [1, 5, 6]`）である場合があります）。最大スケールレベルは `length(x) ÷ 2` ですが、非常に短い時系列にメソッドを適用するのを避けるために、最大スケールを制限することを検討してください（例：`scales = length(x) ÷ 5`）。
