```
Ellipse(turtle; length = 1.0, width = 1.0, n = 20, move = false)
```

タートルの前に楕円を生成し、それを返します。

## 引数

  * `turtle`: 楕円を作成するために使用されるタートルの状態。
  * `length`: 楕円の長さ。
  * `width`: 楕円の幅。
  * `n`: 楕円を近似するメッシュの三角形の数（整数）。
  * `move`: タートルを前に移動させるかどうか（`true` または `false`）。

## 詳細

`n` 個の三角形を持つ三角形メッシュが生成され、楕円を近似します。楕円はタートルの前に、タートルの腕と頭の軸で定義された平面上に生成されます。引数 `length` はタートルの頭の軸に沿った楕円の軸を指し、`width` は直交する軸を指します。

`move = true` の場合、タートルは `length` に等しい距離だけ前に移動します。

## 返り値

三角形メッシュ（`Mesh` 型のオブジェクト）を返します。

## 例

```jldoctest
julia> turtle = Turtle();

julia> e = Ellipse(turtle; length = 1.0, width = 0.5, n = 40);
```
