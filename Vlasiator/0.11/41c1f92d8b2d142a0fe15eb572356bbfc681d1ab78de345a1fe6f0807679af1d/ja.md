```
find_reconnection_points(ψ::Array{T,2}; retol::Float64=1e-4,
   method::Int=1) -> indices_x, indices_o
```

フラックス関数 `ψ` から2D磁場トポロジーにおけるX点およびO点のインデックスを見つけます。現在の実装は境界近くの3層には対応していません。

# キーワード

  * `retol=1e-1`: 勾配を0として受け入れるための比率に対する相対許容誤差を決定します。
  * `method=1`: メソッド1はセル中心の1次および2次導関数を計算し、ヘッセ行列をチェックします。メソッド2は各点のフラックス関数をその8つの隣接点と比較し、より決定的です。
