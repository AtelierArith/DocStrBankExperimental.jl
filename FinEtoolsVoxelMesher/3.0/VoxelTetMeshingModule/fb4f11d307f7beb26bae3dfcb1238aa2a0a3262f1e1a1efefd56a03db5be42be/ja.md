```
remesh!(self::ImageMesher, edge_length_ratio = 1.0)
```

再メッシュステップを実行します。

メッシュが存在しない場合、初期メッシュが作成されます。それ以外の場合は、粗化シーケンス（表面を粗化 -> スムーズ -> ボリュームを粗化 -> スムーズ）が実行されます。現在の要素サイズ（`self.currentelementsize`）が使用されます。次の再メッシュアルゴリズムの反復のために、現在の要素サイズを更新するのを忘れないでください。

  * `edge_length_ratio` = 粗化によって生成される四面体の最長辺と最短辺の長さの比率の最大許容値（デフォルトは1.0）。これが1.0より大きい値（例えば2.0）として指定された場合、余分に長い尖った四面体が防止されます。

頂点、四面体、および材料識別子をメッシュ化した後、`self.v`、`self.t`、および`self.tmid`として取得できます。
