```
autocor(x, [lags]; demean=true)
```

ベクトルまたは行列 `x` の自己相関関数 (ACF) を計算し、オプションで `lags` を指定します。`demean` は、ACF を計算する前に `x` の平均を `x` から引くべきかどうかを示します。

`x` がベクトルの場合、`lags` と同じ長さのベクトルを返します。`x` が行列の場合、サイズ `(length(lags), size(x,2))` の行列を返し、結果の各列は `x` の列に対応します。

指定されていない場合、使用されるラグは 0 から `min(size(x,1)-1, 10*log10(size(x,1)))` までの整数です。

出力は `x` の分散で正規化されており、すなわちラグ 0 の自己相関は 1 になります。未正規化の形式については [`autocov`](@ref) を参照してください。
