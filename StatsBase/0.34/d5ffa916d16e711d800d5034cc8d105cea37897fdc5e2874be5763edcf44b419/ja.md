```
crosscov!(r, x, y, lags; demean=true)
```

実数ベクトルまたは行列 `x` と `y` の間のクロス共分散関数 (CCF) を `lags` で計算し、その結果を `r` に格納します。`demean` は、CCF を計算する前に `x` と `y` のそれぞれの平均を引くべきかどうかを指定します。

`x` と `y` が両方ともベクトルである場合、`r` は `lags` と同じ長さのベクトルでなければなりません。もし `x` が行列で `y` がベクトルである場合、`r` はサイズ `(length(lags), size(x, 2))` の行列でなければなりません；もし `x` がベクトルで `y` が行列である場合、`r` はサイズ `(length(lags), size(y, 2))` の行列でなければなりません。もし `x` と `y` が両方とも行列である場合、`r` はサイズ `(length(lags), size(x, 2), size(y, 2))` の三次元配列でなければなりません。

出力は正規化されていません。正規化された関数については [`crosscor!`](@ref) を参照してください。
