```
crosscor(x, y, [lags]; demean=true)
```

実数値ベクトルまたは行列 `x` と `y` の間のクロス相関を計算し、オプションで `lags` を指定します。`demean` は、クロス相関を計算する前に `x` と `y` のそれぞれの平均を引くべきかどうかを指定します。

`x` と `y` が両方ともベクトルである場合、`lags` と同じ長さのベクトルを返します。それ以外の場合、`x` と `y` の各列のペア間のクロス共分散を計算します。

指定されていない場合、使用されるラグは `-min(size(x,1)-1, 10*log10(size(x,1)))` から `min(size(x,1), 10*log10(size(x,1)))` までの整数です。

出力は `sqrt(var(x)*var(y))` で正規化されます。非正規化形式については [`crosscov`](@ref) を参照してください。
