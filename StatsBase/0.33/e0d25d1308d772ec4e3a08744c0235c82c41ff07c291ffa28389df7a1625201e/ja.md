```
autocov(x, [lags]; demean=true)
```

ベクトルまたは行列 `x` の自己共分散を計算し、オプションで自己共分散を計算するラグ `lags` を指定します。`demean` は、自己共分散を計算する前に `x` から `x` の平均を引くべきかどうかを示します。

`x` がベクトルの場合、`lags` と同じ長さのベクトルを返します。`x` が行列の場合、サイズが `(length(lags), size(x,2))` の行列を返し、結果の各列は `x` の列に対応します。

指定されていない場合、使用されるラグは 0 から `min(size(x,1)-1, 10*log10(size(x,1)))` までの整数です。

出力は正規化されていません。正規化された関数については [`autocor`](@ref) を参照してください。
