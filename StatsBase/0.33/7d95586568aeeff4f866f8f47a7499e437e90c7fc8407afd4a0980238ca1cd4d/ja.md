```
crosscor!(r, x, y, lags; demean=true)
```

実数値ベクトルまたは行列 `x` と `y` の間のクロス相関を `lags` で計算し、その結果を `r` に格納します。`demean` は、クロス相関を計算する前に `x` と `y` のそれぞれの平均を引くべきかどうかを指定します。

`x` と `y` がベクトルの場合、`r` は `lags` と同じ長さのベクトルでなければなりません。もし `x` が行列で `y` がベクトルの場合、`r` はサイズ `(length(lags), size(x, 2))` の行列でなければなりません；もし `x` がベクトルで `y` が行列の場合、`r` はサイズ `(length(lags), size(y, 2))` の行列でなければなりません。もし `x` と `y` の両方が行列の場合、`r` はサイズ `(length(lags), size(x, 2), size(y, 2))` の三次元配列でなければなりません。

出力は `sqrt(var(x)*var(y))` で正規化されます。未正規化の形式については [`crosscov!`](@ref) を参照してください。
