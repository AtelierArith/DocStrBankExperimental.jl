```
pmra(orbit, t)
```

時刻 `t` [日] における *二次* の赤経の瞬時の固有運動異常 [mas/ジュリア年] を取得します。

```
pmra(o)
```

`AbstractOrbitSolution` のインスタンスから *二次* の赤経の瞬時の固有運動異常 [mas/ジュリア年] を取得します。

```
pmra(elem, t, M_planet)
```

時刻 `t` [日] における *一次* の赤経の瞬時の固有運動異常 [mas/ジュリア年] を取得します。`M_planet` と `elem.M` の単位は一致している必要があります。

```
pmra(o, M_planet)
```

上記と同じですが、軌道解から取得します。
