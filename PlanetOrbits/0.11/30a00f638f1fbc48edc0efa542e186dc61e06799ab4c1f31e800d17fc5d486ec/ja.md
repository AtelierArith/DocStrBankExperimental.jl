```
Visual{OrbitType}(..., plx=...)
```

これは別の軌道をラップして、視差距離フィールド `plx` を追加し、投影された量を計算できるようにします。それ以外のすべては親軌道に転送されます。

例えば、KepOrbit型はAUでのxおよびy位置の計算をサポートしています。Visual{KepOrbit}は、追加で投影された赤経および赤緯のオフセットを計算することもサポートしています。

!!! note
    `ThieleInnesOrbit`型は、Thiele-Innes定数がすでにミリ秒で表現されているため、`Visual`でラップする必要はなく、常に`plx`値を必要とします。
