```
posangle(orbit, t)
```

二次体の位置角 [rad] を、時刻 `t` [日] における一次体に対する私たちの視点から計算します。

```
posangle(o)
```

二次体の位置角 [rad] を、`AbstractOrbitSolution` のインスタンスからの私たちの視点で計算します。

```
posangle(elem, t, M_planet)
```

二次体の位置角 [rad] を、時刻 `t` [日] における一次体に対する私たちの視点から計算します。この場合のみ、M_planet の値は任意です。

```
posangle(o, M_planet)
```

**一次体** の位置角 [rad] を、`AbstractOrbitSolution` のインスタンスからの私たちの視点で計算します。この場合のみ、M_planet の値は任意です。
