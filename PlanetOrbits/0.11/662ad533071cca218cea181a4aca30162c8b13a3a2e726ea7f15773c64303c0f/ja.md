```
radvel(orbit, t)
```

*二次*と*一次*の相対的な半径速度 [m/s] を視線に沿って取得します（正の値は離れていることを意味します）時刻 `t` [日] において。

```
radvel(o)
```

*二次*と*一次*の相対的な半径速度 [m/s] を視線に沿って取得します（正の値は離れていることを意味します）`AbstractOrbitSolution` のインスタンスから。

```
radvel(elem, t, M_planet)
```

*一次*の絶対的な半径速度 [m/s] を視線に沿って取得します（正の値は離れていることを意味します）時刻 `t` [日] において。`M_planet` と `elem.M` の単位は一致している必要があります。

```
radvel(o, M_planet)
```

*一次*の絶対的な半径速度 [m/s] を視線に沿って取得します（正の値は離れていることを意味します）`AbstractOrbitSolution` から。`M_planet` と `elem.M` の単位は一致している必要があります。
