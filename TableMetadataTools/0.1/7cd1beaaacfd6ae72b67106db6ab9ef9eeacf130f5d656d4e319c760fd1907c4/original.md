```
dict2metadata!(table, dict; style::Bool=false, defaultstyle::Symbol=:default)
```

Store table-level metadata stored in `dict` in `table` (discarding all previously present table-level metadata).

`style` keyword argument indicates if `dict` has style information (`false` by default). If `style` is `false` then `defaultstyle` (`:default` by default) is used when setting metadata style. If style is passed it can be any value, but it will be converted to `Symbol` internally.

The funcion assumes that `dict` stores key-value metadata; it is typically previously generated by the `metadata` function.
