```
conv(u, v[, dims])
```

`u`を`v`と`dims`次元で畳み込みます。これはFFTベースの方法です。この方法は、適切なパディング/ウィンドウなしではラップアラウンドアーティファクトを導入することに注意してください。

# 引数

  * `u`は実空間の配列です。
  * `v`は実空間で畳み込む配列です。
  * デフォルトでは`ntuple(+, min(N, M)))`は、次元数が少ない配列のすべての次元で畳み込みを行うことを意味します。`dims`が整数の配列である場合、これらの次元でのみ畳み込みを行います。例えば、`dims=[1,3]`は最初と第三の次元で畳み込みを行います。第二の次元は畳み込まれません。

`u`と`v`の両方が実数値の配列である場合、`rfft`を使用し、したがって出力も実数になります。`u`または`v`のいずれかが複素数である場合、`fft`を使用し、出力はしたがって複素数になります。

# 例

すべての次元でFFTを使用した1D。`v`をデルタピークに設定します。したがって、畳み込みは恒等作用を持つべきです。

```jldoctest
julia> u = [1 2 3 4 5]
1×5 Array{Int64,2}:
 1  2  3  4  5
julia> v = [0 0 1 0 0]
1×5 Array{Int64,2}:
 0  0  1  0  0
julia> conv(u, v)
1×5 Matrix{Float64}:
 4.0  5.0  1.0  2.0  3.0
```

異なる`dims`引数を持つ2DでのFFT。

```jldoctest
julia> u = 1im .* [1 2 3; 4 5 6]
2×3 Matrix{Complex{Int64}}:
 0+1im  0+2im  0+3im
 0+4im  0+5im  0+6im
julia> v = [1im 0 0; 1im 0 0]
2×3 Matrix{Complex{Int64}}:
 0+1im  0+0im  0+0im
 0+1im  0+0im  0+0im
julia> conv(u, v)
2×3 Matrix{ComplexF64}:
 -5.0+0.0im  -7.0+0.0im  -9.0+0.0im
 -5.0+0.0im  -7.0+0.0im  -9.0+0.0im
```
