```
iTpb
```

超タイプ `iT` の合成再帰型で、絶滅のない二項系統樹を表し、`λ` がジオメトリックブラウン運動として進化する `insane` 用で、以下のフィールドを持ちます：

d1: 娘樹 1   d2: 娘樹 2   e: エッジ   fx: 固定樹の場合   dt: 時間遅延の選択   fdt: 最終 `δt`   lλ: `log(λ)` のブラウン運動進化の配列

```
iTpb()
```

空の `iTpb` オブジェクトを構築します。

```
iTpb(e::Float64)
```

ペンダントエッジ `pe` を持つ空の `iTpb` オブジェクトを構築します。

```
iTpb(d1::iTpb, d2::iTpb, e::Float64)
```

2つの `iTpb` 娘とペンダントエッジ `pe` を持つ `iTpb` オブジェクトを構築します。
