```
sTfbd
```

スーパタイプ `sTf` の最も単純な合成再帰型で、`insane` 用の二項系統樹を表し、以下のフィールドを持ちます：

d1: 娘の木 1   d2: 娘の木 2   e: エッジ   iμ: 絶滅ノードであるか   iψ: 化石ノードであるか   fx: 固定されているか

```
sTfbd(e::Float64)
```

エッジ `e` を持つ空の `sTfbd` オブジェクトを構築します。

```
sTfbd(d1::sTfbd, d2::sTfbd, e::Float64)
```

二つの `sTfbd` 娘とエッジ `e` を持つ `sTfbd` オブジェクトを構築します。

```
sTfbd(d1::sTfbd, e::Float64)
```

一つのサンプル先祖、一つの `sTfbd` 娘、そしてエッジ `e` を持つ `sTfbd` オブジェクトを構築します。
