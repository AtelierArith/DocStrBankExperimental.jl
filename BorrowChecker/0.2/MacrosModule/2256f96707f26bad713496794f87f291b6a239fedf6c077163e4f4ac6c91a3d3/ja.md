```
@bc func(args...; kwargs...)
```

与えられた引数とキーワード引数を使って `func` を呼び出し、所有されている変数のように見える引数のために一時的な借用を自動的に作成します。

# 例

配列を参照で安全に変更したいとします。所有されている変数 `ar1` と `ar2` があり、`ar2` の最初の要素を `ar1` に追加したいとします。

```
@own :mut ar1 = [1, 2]
@own ar2 = [3, 4]

add_first!(x, y) = (x[1] += y[1]; nothing)
```

手動でライフタイムスコープを設定する場合、次のように書くことができます。

```julia
@lifetime lt begin
    @ref ~lt :mut ref1 = ar1
    @ref ~lt ref2 = ar2
    add_first!(ref1, ref2)
end
```

しかし、ほとんどの場合、単一の関数呼び出しのためにライフタイムスコープを作成するだけで済むので、`@bc` を使うことでこれを自動的に行うことができます。

```julia
@bc add_first!(@mut(ar1), ar2)
```

これは、手動のライフタイムスコープに非常に似たものとして評価されます。

`@bc` は所有されていない変数もサポートしており、それらはそのまま渡されます。
