```
@cc closure_expr
```

「クロージャチェック」は、クロージャが借用チェッカーと互換性があるかどうかを確認しようとするマクロです。

キャプチャされるのは不変参照（`@ref`および`@bc`で作成されたもの）のみが許可されており、キャプチャされるすべての他の所有および借用された変数はエラーを引き起こします。

# 例

```julia
@own x = 1
@own :mut y = 2

@lifetime lt begin
    @ref ~lt z = x
    @ref ~lt :mut w = y
    
    # これらはキャプチャが借用ルールを破るためエラーになります
    bad = @cc () -> x + 1
    bad2 = @cc () -> w + 1
    
    # しかし、不変参照をキャプチャすることは許可されています
    good = @cc () -> z + 1
    # これはエラーになりません。
end
```
