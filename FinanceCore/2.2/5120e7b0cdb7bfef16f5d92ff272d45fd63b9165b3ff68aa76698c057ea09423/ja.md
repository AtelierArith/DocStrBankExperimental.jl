概要 ≡≡≡≡≡≡≡≡≡

```
struct Composite{A, B}
```

`Composite{A,B}` は、タイプ `A` とタイプ `B` の2つの他の契約で構成される契約です。コンポジットの満期は、2つのコンポーネントの満期の最大値です。

これは、より単純な契約から任意に複雑な契約を組み立てるために使用されます。

フィールド ≡≡≡≡≡≡≡≡

```
a :: A
b :: B
```
