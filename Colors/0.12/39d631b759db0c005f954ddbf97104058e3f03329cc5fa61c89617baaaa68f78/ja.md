```
colordiff(a, b; metric=DE_2000())
```

色 `a` と `b` の間の知覚的差異の近似値を計算します。オプションで、[`DE_2000`](@ref)（デフォルト）、[`DE_94`](@ref)、[`DE_JPC79`](@ref)、[`DE_CMC`](@ref)、[`DE_BFD`](@ref)、[`DE_AB`](@ref)、[`DE_DIN99`](@ref)、[`DE_DIN99d`](@ref)、および [`DE_DIN99o`](@ref) の中から選択した `metric` を指定できます。

返り値は、色とメトリックに依存する非負の数です。

!!! note
    サポートされているメトリックは、`Lab` またはその変種の色空間内の差異を測定します。入力色がメトリックによって内部的に使用される色空間にない場合、色（例：`RGB`）はデフォルトのホワイトポイント CIE D65 (`Colors.WP_D65`) を使用して変換されます。別のホワイトポイントを使用したい場合は、メトリックで使用される色空間（例：[`DE_2000`](@ref) の場合は `Lab`）に色を事前に変換してください。
