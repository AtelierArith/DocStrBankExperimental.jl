```
diverging_palette(h1, h2, N::Int=100; <keyword arguments>)
```

2つの順次パレットを組み合わせて発散パレットを作成します

# 引数

  * N        - 色の数
  * h1       - 左側の主要な色相 [0,360]
  * h2       - 右側の主要な色相 [0,360]
  * c        - 全体の明度コントラスト [0,1]
  * s        - 彩度 [0,1]
  * b        - 明るさ [0,1]
  * w        - 冷暖パラメータ、すなわち開始色の強さ [0,1]
  * d1       - 左側の終了色の深さ [0,1]
  * d2       - 右側の終了色の深さ [0,1]
  * wcolor   - 開始色 (暖かさ)
  * dcolor1  - 左側の終了色 (深さ)
  * dcolor2  - 右側の終了色 (深さ)
  * logscale - ログ間隔を切り替えるための真/偽
