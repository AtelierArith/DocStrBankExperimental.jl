```
DE_94(kl=1, kc=1, kh=1, k1=0.045, k2=0.015)
```

CIE Delta E 94 推奨（1994）のメトリックを構築します。重み付けパラメータ `kl`、`kc`、`kh`、`k1`、および `k2` は、推奨に従って提供されています。`kl`、`k1`、および `k2` はアプリケーションに依存します：

|      | グラフィックアート | テキスタイル  |
|:----:|:--------- |:------- |
| `kl` | `1`       | `2`     |
| `k1` | `0.045`   | `0.048` |
| `k2` | `0.015`   | `0.014` |

デフォルト値はグラフィックアート用です。`kc` と `kh` はデフォルトで `1` です。

`DE_94` は [`DE_AB`](@ref) よりも知覚的に均一ですが、[`DE_2000`](@ref) によって解決されるいくつかの非均一性があります。

!!! note
    `DE_94` は準メトリックであり、すなわち対称性を破ります。したがって、`colordiff(a, b, metric=DE_94())` は `colordiff(b, a, metric=DE_94())` と等しくない場合があります。`colordiff` の最初の引数は参照（標準）色として取られます。
