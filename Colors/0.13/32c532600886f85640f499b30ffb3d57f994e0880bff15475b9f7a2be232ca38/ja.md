```
MSC(h)
MSC(h, l; linear=false)
```

与えられた色相 `h` に対して、LCHuv 空間の対応するコーナーを見つけることによって、sRGB ガマット内の最も飽和した色を計算します。オプションで、明度 `l` も指定できます。

# 引数

  * `h`: LCHuv 空間での色相 [0,360]
  * `l`: LCHuv 空間での明度 [0,100]

# キーワード引数

  * `linear` : true の場合、飽和度は黒/白と `MSC(h)` の間で線形補間されます。ガマットは L-C セクションでおおよそ三角形だからです。

!!! note
    `MSC(h)` は `LCHuv` 色を返しますが、`MSC(h, l)` は飽和度の値を返します。この動作は将来のリリースで変更される可能性があります。
