```
Near <: IntSelector

Near(x)
Near(a, b)
```

`x`に最も近いインデックスを選択するセレクタです。

[`Points`](@ref)を使用する場合、これは単に`x`に最も近い値を検索しますが、[`Intervals`](@ref)を使用する場合、`x`に最も近い区間の*中心*が選択されます。これは`Start`および[`End`](@ref)のローカスのインデックス値からオフセットされます。

`x`は単一のインデックスを選択するための任意の値、またはインデックスのベクトルを選択するための値の`Vector`であることができます。2つの値`a`と`b`が使用される場合、それぞれの最も近い値の間の範囲が選択されます。

## 例

```jldoctest
using DimensionalData

A = DimArray([1 2 3; 4 5 6], (X(10:10:20), Y(5:7)))
A[X(Near(23)), Y(Near(5.1))]

# 出力
4
```
