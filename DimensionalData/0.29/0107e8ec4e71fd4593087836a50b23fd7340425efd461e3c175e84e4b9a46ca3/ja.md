`DimTree`は緩く型付けされており、`NamedTuple`の`DimStack`ではなく`OrderedDict`に基づいているため、インデックス付けは遅くなりますが、コンパイルは非常に速く、非常に柔軟です。

ツリーは無限にネストでき、枝はツリーから次元を継承します。

# レイヤーとブランチの取得と設定

ローカルレイヤーは`getindex`と`Symbol`を使用してアクセスされ、`DimArray`を返します。例えば、`dt[:layer]`や`dt[(:layer1, :layer2)]`で`DimStack`を取得できます。

ブランチは`getproperty`を使用してアクセスされます。例えば、`dt.branch`です。

レイヤーとブランチはそれぞれ`setindex!`と`setproperty!`を使用して設定できます。

## 次元とブランチ

ツリーの基部と共有される次元は同一でなければなりません。それらは使用されるツリーの基部レベルに保存され、ブランチに伝播します。

ブランチ内では、すべてのレイヤーがブランチで利用可能な次元のサブセットを使用します。

ブランチ間では、異なるルックアップ値を持つ同じ次元のバージョンが存在する場合があります。これらは異なる範囲、解像度、またはルックアップに必要な特性をカバーすることがあります。

この特性は、異なるX/Y範囲を持つタイルや、異なる解像度のピラミッドレイヤーに使用できます。

## 例

`julia xdim, ydim = X(1:10), Y(1:15),  z1, z2 = Z([:a, :b, :c]), Z([:d, :e, :f]) a = rand(xdim, ydim) b = rand(Float32, xdim, ydim) c = rand(Int, xdim, ydim, z1) d = rand(Int, xdim, z2) DimTree(a, b)``
