```
Bins(f, bins; labels, pad)
Bins(bins; labels, pad)
```

関数 `f` を適用した後にグループを減らすためにビンを指定します。

  * `f`: ルックアップ値のグループ化関数で、デフォルトは `identity` です。
  * `bins`:

      * `Integer` はグループ値を等間隔のセクションに分割します。
      * 値の `AbstractArray` は `f` の戻り値の正確な一致として扱われます。例えば、`1:3` は 3 つのビン - 1, 2, 3 を作成します。
      * `IntervalSets.Interval` の `AbstractArray` を使用して、明示的に区間を定義できます。重複する区間は未定義の動作を持ちます。

## キーワード

  * `pad`: `Bins` が `Integer` を含むときに、各端にパッドするための全体の区間の割合です。これにより、端の値を失うことを避けます。この解決策は煩雑であることに注意してください - `Integer` と `pad` を渡すことに頼るよりも、選択した `Interval` の `Vector` を手動で指定する方が好ましいことが多いです。
  * `labels`: ビンのための説明的なラベルのリストです。ラベルは `bins` と同じ長さである必要があります。

`f` の戻り値がタプルである場合、ビン分けはタプルの *最後* の値に適用されます。
