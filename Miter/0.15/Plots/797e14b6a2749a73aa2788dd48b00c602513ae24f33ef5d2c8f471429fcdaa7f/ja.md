```julia
sync_bounds(tag, collection)

```

`tag`によって決定された軸の範囲がすべての軸で同じであることを確認してください（下記参照）。

タグは`Val(tag)`の形式でも指定できます。これは便利なラッパーです。

可能なタグ：

  * `:X`, `:Y`, `:XY`: コレクションのすべてのアイテムで指定された軸を同じにします
  * `:x`, `:y`, `:xy`: 同じ列/行にあるプロットでx/y軸を同じにします。これは

`:x`, `:y`, および `:xy`は行列のような引数にのみ適用されます。

すべてのメソッドは（修正された）最初の引数を返します。

# 説明

小文字のタグを示すために、次の配置を考えます

```ascii
y/vertical axis
│
│ C  D
│ A  B
└────────x/horizontal axis
```

これは例えば次のように入力できます

```julia
t = Tableau([A C; B D])
```

その後、`sync_bounds(:x, t)`はAとCがx軸の同じ範囲を持つことを保証し、同様にBとDも保証します。
