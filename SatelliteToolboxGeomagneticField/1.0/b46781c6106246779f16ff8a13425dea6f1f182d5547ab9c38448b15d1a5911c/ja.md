```
geomagnetic_dipole_field(r_e::AbstractVector{T}, year::Number = 2020) where T -> SVector{3, T}
```

位置 `r_e` (ECEF参照フレーム) [m] で簡略化された双極子モデルを使用して、地磁気場 [nT] を計算します。この関数は、南地磁気極の位置（北半球に位置する）と双極子モーメントを取得するために、年 `year` を使用します。`year` が省略された場合、デフォルトは2020年です。

# 備考

1. 出力ベクトルはECEF参照フレームで表されます。
2. 戻り値のベクトル型は、`T` を浮動小数点数に変換することによって得られます。
3. 南地磁気極の位置と双極子モーメントは、**[1]** に提供された値を補間することによって取得されます。
