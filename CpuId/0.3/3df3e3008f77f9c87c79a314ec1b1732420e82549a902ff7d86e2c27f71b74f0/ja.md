```
cachesize()
cachesize(lvl::Integer)
```

CPUの*データ*キャッシュサイズに関する情報を取得します。

CPUによって報告された各キャッシュレベルのデータキャッシュサイズを、`cpuid`命令への一連の呼び出しを使用して決定します。タプルのインデックスはキャッシュレベルに対応しており、サイズはバイト単位で示されます。

整数が与えられた場合、対応するキャッシュレベルのデータキャッシュサイズが返されます。これは上記のタプルバージョンよりも大幅に高速です。

これは総キャッシュサイズであり、いくつかのキャッシュレベルは通常複数のCPUコアによって共有され、高いキャッシュレベルには低いレベルが含まれる場合があります。キャッシュレベルをキロバイト単位で表示するには、例えば`CpuId.cachesize() .÷ 1024`を使用します。

この関数は、キャッシュサイズの検出がサポートされていない場合にエラーをスローします。
