```
cacheinclusive()
cacheinclusive(lvl::Integer)
```

CPUの*データ*キャッシュの包含性に関する情報を取得します。下位キャッシュレベルを含むキャッシュの場合は`true`を返し、それ以外の場合は`false`を返します。

CPUによって報告された各キャッシュレベルのデータキャッシュサイズを取得するために、`cpuid`命令への一連の呼び出しを使用します。キャッシュレベルに一致するタプルインデックスを持つタプルを返します。

整数が与えられた場合、対応するキャッシュレベルのデータキャッシュの包含性が返されます。これは上記のタプルバージョンよりも大幅に高速です。
