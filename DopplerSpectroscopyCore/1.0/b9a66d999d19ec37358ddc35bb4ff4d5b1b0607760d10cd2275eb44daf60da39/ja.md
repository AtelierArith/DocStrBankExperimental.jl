```
LightSource
```

量子二準位系と相互作用するために使用される光源を説明するクラスです。

# フィールド

  * `wavelength` :   主な光源の波長の値を表します (m)。
  * `rabi_freq_1` :   直接光ビームと二準位系との相互作用から生じるラビ周波数の値を表します (自発放出周波数 `γ₀` の単位)。
  * `rabi_freq_2` :   逆/反射光ビームと二準位系との相互作用から生じるラビ周波数の値を表します (自発放出周波数 `γ₀` の単位)。

# 引数

フィールドの説明を参照してください。

# 戻り値

  * `Laser` :   複合型インスタンス。
