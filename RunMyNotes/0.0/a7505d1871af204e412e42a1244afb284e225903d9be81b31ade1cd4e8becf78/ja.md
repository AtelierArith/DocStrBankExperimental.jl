```
folder(in, [out])
```

フォルダー `in` に見つかったすべての `*.jl` ファイルを実行し、それぞれの Jupyter `.ipynb` ノートブックを生成し、同じフォルダーまたは指定された場合は `out` に保存します。

キーワード:

  * `html=true` の場合、このノートブックを HTML にも変換し、`jupyter nbconvert` を使用します。
  * `throw=true` の場合、エラーが発生した場合はエラーで終了します。
  * `all=true` の場合、すべてを実行し、`all=false` の場合は `.ipynb` バージョンよりも古い `.jl` ファイルをスキップします。
  * `verbose=true` の場合、最後に時間とコンピュータに関する詳細を追加します。
  * `credit=true` の場合、リンクを追加します。
