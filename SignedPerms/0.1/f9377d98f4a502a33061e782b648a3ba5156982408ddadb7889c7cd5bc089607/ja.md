`sstab_onmats([G,]M[,l])`

引数 `G` が与えられた場合（これは `SPermGroup` であるべきです）、これは `centralizer(G,M,onmats)` の高速な実装です。`G` が省略された場合、`hyperoctaedral_group(size(M,1))` として扱われます。このプログラムは高度なアルゴリズムを使用しており、最大80×80の行列を処理できます。`l` が与えられた場合、返される群は `l` も中心化する必要があります（作用 ^ のため）。

```julia-repl
julia> n=[-1 -1 -1 -2 2 -2 -3 -3 -3; -1 -1 -1 -3 3 -3 -2 -2 -2; -1 -1 -1 -1 1 -1 -1 -1 -1; -2 -3 -1 -3 1 -2 -1 -3 -2; 2 3 1 1 -2 3 3 2 1; -2 -3 -1 -2 3 -1 -2 -1 -3; -3 -2 -1 -1 3 -2 -2 -3 -1; -3 -2 -1 -3 2 -1 -3 -1 -2; -3 -2 -1 -2 1 -3 -1 -2 -3];

julia> length(sstab_onmats(n))
8
```
