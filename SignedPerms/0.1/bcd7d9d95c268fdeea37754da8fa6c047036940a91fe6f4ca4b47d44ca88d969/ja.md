符号付き置換 `1:n` の符号付き置換は、ペア `(-i,i)` を保持する `-n,…,-1,1,…,n` の集合の置換です。内部的には `1:n` の画像として表現されます。符号付きサイクルの積として印刷されます。

# 例

```julia-repl
julia> SPerm([-2,-1,-3])
SPerm{Int64}: (1,-2)(3,-3)
```

`1:n` の符号付き置換の群は、ハイパーオクタヘドラル群と呼ばれます。

```
julia> W=hyperoctaedral_group(2)
Group([(1,-1),(1,2)])

julia> elements(W)
8-element Vector{SPerm{Int8}}:
 ()
 (1,-1)
 (1,2)
 (1,-2,-1,2)
 (1,2,-1,-2)
 (1,-2)
 (2,-2)
 (1,-1)(2,-2)
```

私が符号付き置換を使用する動機は、2つの行列が行と列の同時符号付き置換によってのみ異なるかどうかを調べることです。以下の例では `SPerm(m,n;dims=(1,2))` を使用しています。

符号付き置換の型は `SPerm{T}` であり、`T<:Integer` で、1つのフィールドを持つ `struct` で、`1:n` の画像を保持する `Vector{T}` です。`Int` とは異なる `T` を使用することで、スペースや時間を節約できる可能性があります。`T` が指定されていない場合、速度とコンパクトさの良い妥協点である `Int16` を使用します。

`SPerm` には `copy, hash, ==, isless` （全順序）メソッドがあり、ハッシュのキーや集合の要素として使用できます。2つの `SPerm` は、同じ点を同じ画像に移動させる場合に等しいと見なされます。例えば、

```julia-repl
julia> SPerm([-2,-1,-3])==SPerm([-2,-1,-3,4])
true
```

`SPerm` はブロードキャストのスカラーとして扱われます。
