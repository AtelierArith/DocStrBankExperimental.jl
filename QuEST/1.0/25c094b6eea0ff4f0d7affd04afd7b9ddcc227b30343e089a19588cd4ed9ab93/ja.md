```
unsafe_load_state_vec(state_vec_pointer, num_elements)
```

メモリから状態ベクトルをロードします。

この関数は、状態ベクトルへのポインタとベクトル内の要素数を受け取ります。次に、メモリから状態ベクトルの実部と虚部をロードし、状態ベクトルを表す複素数配列を返します。

# 引数

  * `state_vec_pointer`: 状態ベクトルへのポインタ。これは、`real`および`imag`フィールドを持つ構造体であり、それぞれが`Float64`へのポインタである必要があります。
  * `num_elements`: 状態ベクトル内の要素数。

# 戻り値

  * 状態ベクトルを表す`Complex{Float64}`配列。

# 警告

この関数は、`unsafe_load`関数を使用してメモリからデータを直接ロードします。ポインタが有効なメモリを指していない場合、安全でない可能性があります。使用には注意してください。
