```
@pyexec [inputs =>] code [=> outputs]
```

与えられた `code` を新しいローカルスコープで実行します。

グローバルスコープは持続的で、現在のモジュールに固有です。

`code` はリテラル文字列またはコマンドでなければなりません。

`inputs` はローカルスコープに含める `v=expr` の形式の入力のタプルです。`v` のみが必須で、`expr` は `v` にデフォルトします。

`outputs` は `x::T=v` の形式の出力のタプルであり、これは `v` がローカルから抽出され、`T` に変換されて `x` に割り当てられることを意味します。`x` のみが必須で、`T` は `Py` にデフォルトし、`v` は `x` にデフォルトします。

# 例

次のコードは `1.1+2.2` を計算し、その値を `Float64` として `ans` に割り当てます：

```
@pyexec (x=1.1, y=2.2) => `ans=x+y` => ans::Float64  # returns 3.3
```

変数を `global` としてマークすると、それらはモジュールスコープに保存され、以降の呼び出しで利用可能になります：

```
@pyexec `global x; x=12`
@pyeval `x` => Int  # returns 12
```
