```
@pyeval [inputs =>] code [=> T]
```

与えられた `code` を新しいローカルスコープで評価し、答えを `T` として返します。

グローバルスコープは持続的で、現在のモジュールに固有です。

`code` はリテラル文字列またはコマンドでなければなりません。

`inputs` はローカルスコープに含める `v=expr` 形式の入力のタプルです。`v` のみが必要で、`expr` は `v` にデフォルト設定されます。

# 例

次の計算は `1.1+2.2` を行い、`Float64` を返します：

```
@pyeval (x=1.1, y=2.2) => `x+y` => Float64  # returns 3.3
```
