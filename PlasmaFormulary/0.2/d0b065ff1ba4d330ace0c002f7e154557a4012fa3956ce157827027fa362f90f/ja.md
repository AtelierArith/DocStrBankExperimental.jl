```
ion_sound_speed(T_e, T_i, m_i, Z; γ_e=1, γ_i=3, n_e=nothing, k=nothing)
```

電子-イオンプラズマのためのイオン音速を計算します：

$$
V_S = \sqrt{\frac{γ_e Z k_B T_e + γ_i k_B T_i}{m_i (1 + k^2 λ_{D}^2)}}
$$

`n_e` と `k` の両方が与えられている場合、分散補正を含めます。

# 引数

  * `T_e`: 電子温度（`K` または粒子あたりのエネルギー）
  * `T_i`: イオン温度（`K` または粒子あたりのエネルギー）
  * `m_i`: イオン質量
  * `Z`: イオン電荷状態（デフォルト: 1）
  * `γ_e`: 電子断熱指数（デフォルト: 1）
  * `γ_i`: イオン断熱指数（デフォルト: 3）
  * `n_e`: 電子数密度（オプション）
  * `k`: 波数（オプション）
