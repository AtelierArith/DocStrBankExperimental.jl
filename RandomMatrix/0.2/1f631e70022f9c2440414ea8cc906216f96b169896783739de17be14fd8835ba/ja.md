```julia
Resolvent(A::Matrix)
```

  * 行列 `A` の **関数** である解決子を返します。$R(z)= (A-zI)^{-1}$

# 例

正規化された GOE(500) の解決子 **関数** を取得します。

そして `0` で解決子を評価します。

R = Resolvent(rand(GOE(500),norm = true)) # これは関数です
