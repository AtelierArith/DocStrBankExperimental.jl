```
meshscatter(positions)
meshscatter(x, y)
meshscatter(x, y, z)
```

Plots a mesh for each element in `(x, y, z)`, `(x, y)`, or `positions` (similar to `scatter`). `markersize` is a scaling applied to the primitive passed as `marker`.
