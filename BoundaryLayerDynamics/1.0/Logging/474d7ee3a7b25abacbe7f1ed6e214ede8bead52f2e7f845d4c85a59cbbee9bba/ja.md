```
スナップショット(; kwargs...)
```

瞬時の状態 $\mathbf{\hat{s}}$ をシミュレーション中の定期的な間隔で [Cartesian Binary Data](https://gitlab.com/efpl-columbia/codes/cbd-format) ファイルのコレクションに保存します。

# キーワード

  * `frequency`: スナップショットが保存されるシミュレーション時間の単位での頻度。これは時間ステップ $Δt$ の倍数である必要があります。
  * `path = "output/snapshots"`: スナップショットが保存されるフォルダのパス。
  * `centered = true`: ドメインを $N₁×N₂$ セグメントに分割して得られた間隔の中心での $x₁$ および $x₂$ の位置で値を保存します。`false` に設定すると、位置は 0 から始まり、ドメインの終わりの前のグリッド間隔で終了します。
  * `precision::DataType`: 出力に使用される浮動小数点データ型で、`Float32` または `Float64` のいずれかです。デフォルトでは、これは `Model` によってシミュレーションに使用されるデータ型に設定されています。
