```
MeanProfiles(; kwargs...)
```

時間と水平空間にわたって平均化された項のプロファイルを収集します。時間平均は、各時間ステップの前後に収集されたサンプルを使用して台形法で計算されます。平均値は、シミュレーション中に定期的に書き込むことができる[HDF5](https://en.wikipedia.org/wiki/Hierarchical_Data_Format)ファイルに保存されます。

# キーワード

  * `profiles = (:vel1, :vel2, :vel3)`: プロファイルが収集される項。現在サポートされているのは、`:vel1`、`:vel2`、`:vel3`、`:adv1`、`:adv2`、`:adv3`、`:sgs11`、`:sgs12`、`:sgs13`、`:sgs22`、`:sgs23`、`:sgs33`です。
  * `path = "output/profiles"`: プロファイルが保存されるHDF5ファイルのパスのプレフィックス。ファイルは番号付けされているため、デフォルトのパスの場合、最初のファイルは`output/profiles-01.h5`に書き込まれます。
  * `output_frequency`: 出力がディスクに書き込まれる間隔。この値は時間ステップの倍数である必要があるため、そうでない場合は一部の出力がスキップされる可能性があります。
