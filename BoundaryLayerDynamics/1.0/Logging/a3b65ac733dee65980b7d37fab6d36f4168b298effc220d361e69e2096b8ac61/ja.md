```
ProgressMonitor(; kwargs...)
```

シミュレーションの進行状況に関する情報を、壁時間の定期的な間隔で標準出力に印刷します。

# キーワード

  * `frequency = 10`: 各進行報告の間の壁時間の間隔（秒単位）。前回の報告からの壁時間が指定された間隔を超えた場合、報告は常に時間ステップの終了時に表示されます。
  * `always_update = true`: 定期的な報告の間に各時間ステップの後にドットを印刷して、シミュレーションの進行状況を示します。
  * `tstep = nothing`: 時間ステップ $Δt$ を `ProgressMonitor` に渡すことで、輸送コーラン数 $C = Δt / (∂₃^{max} |u|^{max})$ を計算できます。設定されていない場合、輸送時間スケール $1 / (∂₃^{max} |u|^{max})$ が代わりに報告されます。
