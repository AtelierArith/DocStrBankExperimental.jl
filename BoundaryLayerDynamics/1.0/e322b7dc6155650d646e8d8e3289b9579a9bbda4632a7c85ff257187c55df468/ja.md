```
Model(resolution, domain, processes)
```

`Model`は、指定された`processes`の動力学の離散化された表現を、指定された`domain`内で提供します。

それは、初めはゼロに設定された速度場などの離散化された状態変数を含み、これらの状態変数の変化率を効率的に計算するために必要なすべての情報を含んでいます。必要な状態変数は、指定されたプロセスから自動的に決定されます。

# 引数

  * `resolution::NTuple{3, Integer}`: $x_1$および$x_2$方向のモード数、ならびに$x_3$方向のグリッドポイント数。最初の2つの値は奇数である必要があり、偶数の場合は、解決された波数がゼロの周りで対称になるように1つ減らされます。
  * `domain::Domain`: [`Domain`](@ref)型で表現されたシミュレーションされた宇宙。
  * `processes`: 状態変数の動的挙動を支配する物理プロセスのリスト。便利なことに、このリストは任意の反復可能なコレクションであり、ネストすることもできます。

# キーワード

  * `comm::MPI.Comm`: モデルが使用するMPIコミュニケーター。指定されていない場合は、`MPI.COMM_WORLD`が使用されます。異なるコミュニケーターを指定することは現在あまりテストされておらず、予期しない動作を引き起こす可能性があることに注意してください。
