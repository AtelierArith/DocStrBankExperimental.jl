```
MomentumAdvection(; dealiasing = :quadratic)
```

回転形式での運動量 ($u_i$、すなわち密度で正規化された) の輸送。速度–渦度の積のみが計算され、運動エネルギーの勾配は修正された圧力項の一部として処理されると仮定されます。

# キーワード

  * `dealiasing`: 演算が行われる物理領域の解像度を決定するデアリアシングのレベル。デフォルトの `:quadratic` 設定は、運動量輸送項の一部として計算される積に対するエイリアシングエラーを回避するために「3/2ルール」に依存しています。代わりに、`dealiasing` を `nothing` に設定すると、周波数領域の解像度（偶数値に切り上げ）に一致する物理領域の解像度が設定されるか、解像度を手動で設定するために2つの整数のタプルに設定できます。
