```
StaticSmagorinskyModel(; kwargs...)
```

解決された運動量のサブグリッドスケールの対流輸送を、静的スマゴリンシー模型を用いて近似します。

# キーワード

  * `Cs = 0.1`: スマゴリンシー係数 $C_\mathrm{S}$。
  * `dealiasing`: 非線形操作が行われる物理領域の解像度を決定するディーアリアシングのレベル。デフォルトの `nothing` は、周波数領域の解像度に一致する物理領域の解像度を提供します（偶数値に切り上げ）。代わりに、`dealiasing` を `:quadratic` に設定すると「3/2ルール」に基づく解像度が得られ、2つの整数のタプルを設定することで解像度を手動で設定することもできます。エイリアシング誤差は、より大きな解像度で減少するはずですが、どの解像度でも予想される可能性があります。
  * `wall_damping = true`: 壁に向かって期待される値 $κ x₃$ に向けてスマゴリンシーの長さスケールを調整します。
  * `wall_damping_exponent = 2`: 近壁調整の指数。
