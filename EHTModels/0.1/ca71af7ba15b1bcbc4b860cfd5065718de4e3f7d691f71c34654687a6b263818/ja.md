```
GaussianFilter(θmaj, [θmin, ϕ]; [θunit, ϕunit])
```

原点を中心とした単位の総フラックス密度を持つ楕円形ガウシアンフィルターを作成します。

引数:

  * `θmaj::Real`:   ガウシアンの主軸FWHMサイズ。
  * `θmin::Real`:   ガウシアンの副軸FWHMサイズ。`θmin < 0` の場合、`θmin = θmax`（すなわち円形ガウシアン）となります。デフォルトは -1。
  * `ϕ::Real`:   ガウシアンの位置角。デフォルトは 0。
  * `θunit, ϕunit::Unitful`:   それぞれ `θmaj` と `θmin` および `ϕ` の単位。デフォルト: `θunit=rad` および `ϕ=deg`。
