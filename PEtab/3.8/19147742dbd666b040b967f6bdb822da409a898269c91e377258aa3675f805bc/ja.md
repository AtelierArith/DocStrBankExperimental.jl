```
remake(prob::PEtabODEProblem, xchange::Dict)::PEtabODEProblem
```

`prob`を再構築することなく、推定されるパラメータのセットを修正し、結果として削除します。

`xchange`は、パラメータ名とその新しい値を持つ`Dict`として提供する必要があります。例えば、`k1`を`1.0`に固定するには、`xchange = Dict(:k1 => 1.0)`を提供します。固定されるパラメータは、元々推定されるように設定されていたものだけです。

`remake`は、問題を再コンパイルしないため、パラメータを固定するための最も効率的なアプローチです。
