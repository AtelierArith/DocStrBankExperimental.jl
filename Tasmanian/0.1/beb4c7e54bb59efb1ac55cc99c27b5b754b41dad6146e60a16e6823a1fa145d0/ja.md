```
setDomainTransform!(tsg::TasmanianSG, Transform::VecOrMat)
```

各次元の下限と上限を設定します。

注意: ガウス・ラグランジュおよびガウス・エルミートルールは無限の領域で定義されており、その場合、これがシフトとスケールのパラメータを設定します。マニュアルを参照してください。

Transform: サイズ dimension X 2 の行列で、変換は各方向の領域の下限と上限を指定します。

```
       ガウス・ラグランジュおよびガウス・エルミートグリッドの場合、
       変換は重みの a および b パラメータを与えます。
        exp(-b (x - a))
        exp(-b (x - a)^2)
```
