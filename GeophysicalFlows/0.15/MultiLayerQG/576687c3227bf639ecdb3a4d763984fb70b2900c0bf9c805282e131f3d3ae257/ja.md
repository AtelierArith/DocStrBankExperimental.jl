```
streamfunctionfrompv!(ψh, qh, params::TwoLayerParams, grid)
```

PVを反転させて、二層流体構成の特別な場合における流れ関数 `ψh` のフーリエ変換を取得します。この場合、次のようになります。

$$
ψ̂₁ = - [k² q̂₁ + (f₀² / g′) (q̂₁ / H₂ + q̂₂ / H₁)] / Δ ,
$$

$$
ψ̂₂ = - [k² q̂₂ + (f₀² / g′) (q̂₁ / H₂ + q̂₂ / H₁)] / Δ ,
$$

ここで、$Δ = k² [k² + f₀² (H₁ + H₂) / (g′ H₁ H₂)]$です。

（ここでは、PV-流れ関数の関係がハードコーディングされており、GPU上でのスカラー操作を回避しています。）
