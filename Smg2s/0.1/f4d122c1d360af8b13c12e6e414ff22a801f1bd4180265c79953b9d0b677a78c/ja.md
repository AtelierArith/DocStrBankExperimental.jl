```
initMat!(matrix::SparseMatrixCSC{Tv, Ti}, diag_l::Ti, diag_u::Ti, size::Ti; scale::Real = 1.0, shift::Real = 0.0, sparsity::Real = 0.9) where {Tv<:Complex, Ti<:Integer}
```

生成される行列の初期化。この関数は非エルミートの場合に使用され、行列のエントリは複素スカラーです。

この関数では、パラメータ `diag_l` と `diag_u` が、埋められる下三角部分の範囲（`[diag_l, diag_u]`）を決定します。これらの2つのパラメータは負である必要があり、下三角部分の対角線のオフセットを指します。パラメータ `size` は生成される行列のサイズを定義します。

初期化された行列の非ゼロ要素は、区間 (0,1) でランダムに生成されます。オプションの2つのパラメータ `shift` と `scale` は、この区間のシフトとスケーリングを可能にします。

もう1つのオプションのパラメータ `sparsity` はより重要です。これは、`diag_l` と `diag_u` によって決定される対角線のバンド内の要素が非ゼロである可能性を決定します。言い換えれば、初期化された行列は `sparsity` パラメータの数が小さいほどよりスパースになります。
