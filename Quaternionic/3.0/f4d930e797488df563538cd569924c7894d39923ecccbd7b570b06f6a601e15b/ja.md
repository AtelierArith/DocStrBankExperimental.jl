```
slerp(q₁, q₂, τ; [unflip=false])
```

クォータニオンのペアの「球面線形補間（Spherical Linear intERPolation）」。

「slerp」の結果は次のように与えられます。

```
    (q₂ / q₁)^τ * q₁
```

`τ` が 0 のとき、これは `q₁` に評価されます。`τ` が 1 のとき、これは `q₂` に評価されます。他の値の場合、結果は二つの間で変化します。

`τₐ` に対して `slerp(q₁, q₂, τₐ)`、および `τᵦ` に対して `slerp(q₂, q₃, τᵦ)` のように、連続したクォータニオンのペアに適用すると、連続的になりますが、最初のペアから二番目のペアに移動する際に導関数は不連続になります。より連続的な曲線については [`squad`](@ref) を参照してください。

キーワードとして `unflip=true` が渡され、入力クォータニオンが平行よりも反平行である場合、結果が計算される前に `q₂` の符号が反転されます。

また、同時にこの関数と `τ` に関する導関数を評価するために [`slerp∂slerp∂τ`](@ref) を、入力の各パラメータに関するこの関数とその導関数を評価するために [`slerp∂slerp`](@ref) を参照してください。
