```
from_rotation_matrix(ℛ)
```

3x3 回転行列をクォータニオンに変換します。

3x3 行列 `ℛ` がベクトル `v` を次のように回転させると仮定します。

```
v' = ℛ * v,
```

この回転をクォータニオン `R` を用いて次のように表現することもできます。

```
v' = R * v * R⁻¹.
```

この関数は、そのクォータニオンを返します。非直交行列を許可するために Bar-Itzhack のアルゴリズム（バージョン 3）を使用しています。 [J. Guidance, Vol. 23, No. 6, p. 1085](http://dx.doi.org/10.2514/2.4654)

!!! note
    `Float64` または `Float32` 以外の型の要素を持つ行列に対してこの関数を使用したい場合は、最初に `GenericLinearAlgebra` を（インストールして）インポートする必要があります。この関数は `ℛ` の固有分解を計算するため、そのパッケージを介してのみ、より一般的な浮動小数点型に対して利用可能です。少なくともバージョン 0.3.11 の `GenericLinearAlgebra` が必要です。以前のバージョンにはバグがありました。
