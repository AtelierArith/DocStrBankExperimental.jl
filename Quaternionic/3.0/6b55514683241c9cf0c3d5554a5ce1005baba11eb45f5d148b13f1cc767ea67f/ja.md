```
to_euler_phases(q)
```

入力クォータニオンをオイラー角の複素位相に変換します。

入力クォータニオンを回転として解釈すると（その正規化はスケールアウトしますが）、オイラー角（α, β, γ）から複素オイラー位相を次のように定義できます。

```
zₐ ≔ exp(i*α)
zᵦ ≔ exp(i*β)
zᵧ ≔ exp(i*γ)
```

これらは角度そのものよりも有用な幾何学的量であり、球面調和関数やウィグナーの𝔇行列の計算に関与しており、対応するクォータニオンの成分から代数的に（超越関数を使用せずに）計算できます。

`to_euler_phases!(z, q)`は後方互換性のためにサポートされていますが、この関数が`SVector`を返すため、インプレースアプローチにはおそらく利点はありません。

# 返り値

  * `z::SVector{Complex{T}}`: 複素位相（zₐ, zᵦ, zᵧ）をその順序で。
