```
unflip(q, [dim=1])
unflip!(q, [dim=1])
```

次の次元 `dim` に沿って連続性ができるだけ保たれるように、連続するクォータニオンの符号を反転します。

もし `q` が一連の回転を表す場合、各要素の符号は任意です。しかし、補間や微分などの特定の目的のために、クォータニオンの連続性が重要であり、表す *回転* を変更することなく *クォータニオン* をできるだけ連続的にしたいと考えています。

# 例

```jldoctest
julia> q = [imx, -imx, imx, -imx];

julia> unflip(q)
4-element Vector{QuatVec{Int64}}:
  + 1𝐢 + 0𝐣 + 0𝐦
  + 1𝐢 + 0𝐣 + 0𝐦
  + 1𝐢 + 0𝐣 + 0𝐦
  + 1𝐢 + 0𝐣 + 0𝐦
```
