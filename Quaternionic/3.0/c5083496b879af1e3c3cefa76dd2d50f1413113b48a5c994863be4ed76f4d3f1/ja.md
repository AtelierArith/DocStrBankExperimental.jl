```
squad(Rin, tin, tout; [kwargs...])
```

入力 `Rotor` の `Rin` と対応する時間 `tin` に対する出力時間 `tout` の "球面四辺形補間"。

これは、異なるサイズの時間ステップを許可するために、[Shoemake の "球面ベジェ曲線"](https://doi.org/10.1145/325165.325242) の少し一般化されたバージョンです。

入力 `Rin` と `tin` は同じ長さのベクトルでなければなりません。出力 `tout` は単一の実数または実数のベクトルのいずれかである可能性があります。`tin` と `tout` は両方ともソートされていると仮定され、`tout` は完全に `tin` の範囲内に含まれていると仮定されます。外挿は行われません。

この関数のインプレースバージョンについては、[`squad!`](@ref) を参照してください。

# キーワード引数

`unflip=true` がキーワードとして渡されると、[`unflip`](@ref) 関数が `Rin` に適用されます。

`validate=true` がキーワードとして渡されると、入力 `tin` と `tout` の時間順序がテストされ、外挿が行われないことが確認されます。

`compute_angular_velocity=true` がキーワードとして渡されると、戻り値はタプルになります。タプルの最初の要素は以前と同様に `Rotor` のベクトルですが、2 番目の要素は角速度を表す `QuatVec` のベクトルになります。

`compute_derivative=true` がキーワードとして渡されると、戻り値はタプルになります。タプルの最初の要素は以前と同様に `Rotor` のベクトルですが、最後の要素はローターの時間微分を表す `Quaternion` のベクトルになります。`compute_angular_velocity=true` の場合、このタプルは 3 つの要素を持つことに注意してください。
