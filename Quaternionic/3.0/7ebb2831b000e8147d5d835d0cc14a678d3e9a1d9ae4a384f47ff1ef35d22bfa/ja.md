```
from_euler_angles(α, β, γ)
```

暗黒面から来てください。

オイラー角がクォータニオン `R` に対応することを前提としています。

```
R = exp(α𝐫/2) * exp(β𝐣/2) * exp(γ𝐤/2)
```

ここで、𝐣 と 𝐤 はそれぞれ固定された $y$ 軸と $z$ 軸の周りに回転するため、これは最初に $z$ 軸の周りに角度 $γ$ だけ正の方向に回転し、その後 $y$ 軸の周りに $β$ だけ回転し、最後に $z$ 軸の周りに $α$ だけ回転することを表します。これは、最初に $z$ 軸の周りに $α$ だけ回転し、その後 *回転された* $y'$ 軸の周りに $β$ だけ回転し、最後に *二重回転された* $z''$ 軸の周りに $γ$ だけ回転することと同等です。角度は自然にラジアンであると仮定されます。

注意: この関数に「間違っている」と報告する問題を開く前に、[このページ](https://github.com/moble/quaternion/wiki/Euler-angles-are-horrible)をすべて読むことを確認してください。特に、問題やプルリクエストを開くことに関する最後のセクションを読んでください。
