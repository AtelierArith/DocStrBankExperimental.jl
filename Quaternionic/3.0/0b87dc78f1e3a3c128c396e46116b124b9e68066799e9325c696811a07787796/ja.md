```
from_euler_phases(zₐ, zᵦ, zᵧ)
from_euler_phases(z)
```

これらのオイラー位相に対応するクォータニオンを返します。

入力クォータニオンを回転として解釈すると（その正規化はスケールアウトしますが）、オイラー角（α, β, γ）から複素オイラー位相を次のように定義できます。

```
zₐ ≔ exp(i*α)
zᵦ ≔ exp(i*β)
zᵧ ≔ exp(i*γ)
```

これらは角度自体よりも有用な幾何学的量であり、球面調和関数やウィグナーの𝔇行列の計算に関与しており、対応するクォータニオンの成分から代数的に計算できます（超越関数を使用せずに）。

# パラメータ

  * `z::Vector{Complex{T}}`: 複素位相（zₐ, zᵦ, zᵧ）をその順序で表す長さ3の複素ベクトル。

# 返り値

  * `R::Quaternion{T}`
