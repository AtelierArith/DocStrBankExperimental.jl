```
precessing_nutating_example([Ωₒ], [Ωₚ], [α], [α̇], [ν], [R₀])
```

時間の関数としての正確な四元数関数を返し、これは nutating precessional orbital motion を表します。これは [この論文](https://arxiv.org/abs/1604.08139) の Sec. 6.2 で説明されています。この例は、物理的に現実的でありながら *非常に* 複雑な動きを提供するため、コーディングが簡単であり、解析的に微分することができます。

出力は、軌道を描く天体を結ぶ線の回転とその角速度を表します。以下は、入力パラメータとそのデフォルト値、物理的解釈です：

  * `Ωₒ=2π/1_000`: 軌道周波数
  * `Ωₚ=2π/10_000`: 前進周波数
  * `α=π/8`: $t=0$ における前進円錐の開口角
  * `α̇=2α/100_000`: 前進円錐の開口速度
  * `ν=π/80`: 振動の角度
  * `R₀=exp(-3α*imx/10)`: 系全体の回転

デフォルト値は、合併直前の（潜在的に）実際の前進バイナリブラックホールシステムに典型的なものとして選ばれています。

返されるオブジェクトは、時間の関数である `R`、`ω⃗`、および `Ṙ` の3つで、`Rotor` としての向き、続いて `QuatVec` としての角速度、そして `Quaternion` としての `R` の時間微分を返します。

# 例

