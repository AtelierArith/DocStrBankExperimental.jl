```
to_euler_angles(R)
```

パンドラの箱を開けてしまった。

もし誰かがあなたにオイラー角を使うように言ってきたら、ノーと言って立ち去り、母親に話しに行きなさい。

オイラー角を使いたくはありません。彼らはひどいです。離れてください。オイラー角からクォータニオンに変換するのは一つのことですが、少なくとも正しい方向に進んでいます。しかし、逆に行くのは？！それは正しくありません。

オイラー角はクォータニオン `R` に次のように対応していると仮定します。

```
R = exp(α𝐤/2) * exp(β𝐣/2) * exp(γ𝐤/2)
```

ここで、𝐣 と 𝐤 はそれぞれ固定された $y$ 軸と $z$ 軸の周りを回転するため、これは最初に $z$ 軸の周りを角度 $γ$ だけ正の方向に回転し、その後 $y$ 軸の周りを $β$ だけ回転し、最後に $z$ 軸の周りを $α$ だけ回転することを表します。これは、最初に $z$ 軸の周りを $α$ だけ回転し、その後 *回転された* $y'$ 軸の周りを $β$ だけ回転し、最後に *二度回転された* $z''$ 軸の周りを $γ$ だけ回転することと同等です。角度は自然にラジアンであると仮定されます。

注意：この関数に「間違っている」と報告する問題を開く前に、必ず[このページ](https://github.com/moble/quaternion/wiki/Euler-angles-are-horrible)をすべて読んでください。特に、問題やプルリクエストを開くことに関する最後のセクションを読んでください。

# 戻り値

  * `αβγ::Vector{T}`
