```
BatchReactor(states; initial_states = nothing, name)
```

バッチリアクター。 このシステムは、バッチリアクターの質量移動を定義します（実際には空です）。 反応は外部システムから供給される必要があります。 タンクの体積は必要なく、濃度のみが考慮されるため、方程式は体積に依存しません。 バッチリアクターにおける成分 $C$ の方程式は次のように定義されます： $\frac{dC}{dt} = r_C$ ここで、$r_C$ は成分に対応する外部供給反応速度です。

# パラメータ：

  * `states`: バッチリアクターが持つべき成分と、その質量移動が定義される成分
  * `initial_states`: バッチリアクター内の初期濃度

# ステート：

  * `states` パラメータで定義された各成分に対して1つのステート

# コネクタ：

  * `state`: バッチリアクター内のすべてのステートの現在の値
  * `rates`: 各ステートの反応速度の入力
