```
Sensor(states, sensed; name)
```

特定の状態を実際の出力として提供するコンポーネントです。制御のために特定の状態を抽出する必要がある場合に便利です。

!!! note
    このシステムには測定モデルやノイズは含まれていません。単に情報を取得するだけです。


# パラメータ:

  * `states`: フロー内のコンポーネント
  * `sensed`: 値を定義するソース。ベクトルまたは単一の値のいずれか
  * `flowrate`: フロー率を指定するための名前付きパラメータ。状態に含まれている場合は`nothing`に設定します。

# コネクタ:

  * `inflow`: センサーへの流入
  * `outflow`: センサーの流出（流入と等しい）
  * 各感知された状態に対して1つの外因性出力。状態の名前を付けます。
