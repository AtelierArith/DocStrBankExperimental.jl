```
FlowPump(state; flowrate=nothing, name)
```

フローポンプをモデル化するコンポーネントです。提供された流れがポンプを通過する理想的なフローポンプをモデル化します。

`FlowPump`は1つの流入と1つの流出を持ち、単に通過する流れを指定します。

!!! warning
    `FlowPump`によってポンプされた流れが流入と一致しない場合、流れは別の方法を見つける必要があります。これは、流れが別のルートを取る（[`FlowSeparator`](@ref)を通じて）か、前のシステムで蓄積/消費される（すなわち、可変容量を持つ）ことができます。これらのいずれも不可能な場合、作成されたシステムは正常にシミュレーションできず、方程式が矛盾することになります。


!!! note
    `FlowPump`は流れを分割することはありません。メインの流れからサイドストリームを抽出するつもりであれば、流れを分割するために[`FlowSeparator`](@ref)の組み合わせと、1つの出口での`FlowPump`（そこで流量を指定するため）を使用してください。


# パラメータ:

  * `states`: 流れの中のコンポーネント
  * `flowrate`: ポンプする流量。`nothing`（デフォルト）の場合、動的に設定するための入力が追加されます。

# コネクタ:

  * `inflow`: ポンプの流入
  * `outflow`: ポンプの流出
  * `q_pumped`: **オプション**: ポンプされた流量を提供する[`ModelingToolkitStandardLibrary.Blocks.RealInput`](@extref)。パラメータ`flowrate`が`nothing`の場合のみ追加され、それ以外の場合は流量は指定された値で固定されます。
