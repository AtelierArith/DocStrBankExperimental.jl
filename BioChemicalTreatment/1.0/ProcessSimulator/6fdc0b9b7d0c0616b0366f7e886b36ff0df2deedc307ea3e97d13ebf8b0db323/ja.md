```
rates(sys)
```

プロセス要素 `sys` によって表されるシステムのレートコネクタを取得します。

この関数はシステムのレートを返します。すべてのプロセス要素がレートを持っているわけではありませんが、これは主に [`Reactor`](@ref)s と [`Process`](@ref)es に使用されます。[`Reactor`](@ref) は、[`Reactor`](@ref) 内で発生するプロセスの反応速度としてレートを使用し、[`Process`](@ref) はこれらのレートをリアクターに提供します。

# 拡張ヘルプ

# 実装

この関数は、レートを必要とするか提供する [`AbstractProcessElement`](@ref) から継承する各タイプのためにオーバーライドされるべきです。レートを必要とするか提供するかによって、異なるコネクタが必要であることに注意してください（[`ReactionInputPort`](@ref) 対 [`ReactionOutputPort`](@ref)）。
