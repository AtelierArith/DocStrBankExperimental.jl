```
abstract type CombinatorialSolution
```

ソルバーが `solve` 関数を呼び出したときに返される解。 この型は、統一された解の型を持つために、各組合せインスタンスに対して指定する必要があります。

2つのフィールドを実装する必要があります： `instance` は解決されたインスタンスでなければならず、 `variables` は実際の解を示さなければなりません。 `variables` には、解の一部として選択された離散項目のみを含めることができます（つまり、このオブジェクトの要素は `CombinatorialInstance` の報酬のインデックスとして使用できます）。 解オブジェクトは、必要に応じてさらに多くのプロパティを定義できます。
