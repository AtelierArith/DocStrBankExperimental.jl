```
make_solution(::CombinatorialInstance, ::Dict{K, Float64}) where {K}
```

与えられた組合せインスタンスに対して適切なタイプの解オブジェクトを作成します。辞書は返されるオブジェクトに含めるべき解を示します。

バイナリ決定変数のみがサポートされているため、この関数の呼び出し元は、*いずれか* 解に含まれる変数のみ（すなわち `variable => 1.0` エントリ）またはすべての変数を、0（解に含まれない）または1（解に含まれる）の関連値で提供する辞書を指定できます。たとえば、解が2つの可能なエッジのうちのエッジ `1`-`2` のみを含む場合（もう一方は `1`-`3`）、渡すことができる2つの同等の辞書があります：`Dict(Edge(1, 2) => 1.0)` または `Dict(Edge(1, 2) => 1.0, Edge(1, 3) => 0.0)` のいずれかです。
