```
DiamondSquare <: NeutralLandscapeMaker

DiamondSquare(; H = 0.5) 
DiamondSquare(H)
```

このタイプは、ダイヤモンド-スクエアアルゴリズムを使用して中立的な風景を生成します。このアルゴリズムは、変動する空間自己相関を持つフラクタルを生成します。

https://en.wikipedia.org/wiki/Diamond-square_algorithm

このアルゴリズムは、反復手順である「ダイヤモンド」と「スクエア」ステップにちなんで名付けられました。

空間自己相関の度合いは、パラメータ `H` によって制御され、0.0（低自己相関）から1.0（高自己相関）まで変化します - これは非包括的であり、H = 0およびH = 1は期待通りに動作しません。ダイヤモンド-スクエアアルゴリズムの結果は、次元D = 2 + Hのフラクタルです。

類似のアルゴリズムであるミッドポイント-ディスプレイスメントは、ほぼ同様に機能しますが、DiamondSquareでは、スクエアステップが最も近い2つのコーナーとスクエアの中心からエッジの中点を補間するのに対し、ミッドポイント-ディスプレイスメントは最も近いコーナーからのみ補間します（`MidpointDisplacement`を参照）。
