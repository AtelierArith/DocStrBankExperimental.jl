```
perm,invp = find_ped_order(pedlist [,start])
```

系譜を走査し、祖先がその子孫に先行するような順列を見つけます。結果として得られるベクトル `perm` は、現在のIDを新しいIDに関連付けます。すなわち、`perm[newid]=oldid` です。ベクトル `invp` は逆順列を持ちます。すなわち、`invp[oldid]=newid` です。デフォルトでは、関数は各動物を開始動物として扱います。オプションのベクトル `start` に開始IDを指定すると、追跡はこれらの子孫のみで開始されます。この場合、`perm` と `invp` は追跡されていない動物に対して `0` を含みます。
