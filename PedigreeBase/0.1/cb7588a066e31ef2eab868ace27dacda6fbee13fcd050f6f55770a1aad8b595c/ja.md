```
A = get_nrm(pedlist [, check=false, sorthere=true])
A = get_nrm(T, pedlist [, check=false, sorthere=true])
```

系譜リスト `pedlist` から再帰的（「表形式」）メソッドを使用して、分子（または加算）関係行列、いわゆる *A-matrix* を計算します。系譜は、先祖がその子孫に先行するようにソートされている必要がありますが、`sorthere=true` を指定することでソートを無視できます。結果として得られる行列は密です。`A` の型を `T`（デフォルトは Float64）として指定できます。デフォルトでは、この関数は系譜が時間的にソートされているかどうかをチェックします。これをオフにするには `check=false` を使用します。オプション `sorthere=true` を使用すると、関数は内部的に系譜を再配置し、元の順序で正しい A-matrix を返します。ただし、再配置されたインデックスを持つ一時的な A-matrix を作成するため、2倍のメモリが必要です。
