```
@binarize(expr::Expr)
```

与えられた式内のすべてのn-ary関数呼び出しを、同じ関数へのネストされたバイナリ呼び出しに変換します。つまり、`f(a, b, c)`の形のすべての呼び出しを`f(f(a, b), c)`に変換し、必要に応じてネストの層を追加します。`+`や`*`のような演算子も他の関数呼び出しと同様に扱われるため、

```
@binarize a + b + c + d
```

は次のようになります:

```
((a + b) + c) + d
```

これは、https://github.com/rdeits/CommonSubexpressions.jl/issues/14 のような長い連鎖関数呼び出しの部分式排除を容易にすることを目的としています。
