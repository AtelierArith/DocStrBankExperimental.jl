副作用がない純粋な関数がマクロの本体内で呼び出されるという仮定の下で、単純な共通部分式の排除を行います。詳細については[Readme.md](https://github.com/rdeits/CommonSubexpressions.jl/blob/master/Readme.md)を参照してください。

このマクロは、部分式の排除を行う前に、式内のマクロ呼び出しを再帰的に展開します。これと組み合わせると便利なのが`@binarize`マクロで、n-ary関数呼び出しをネストされたバイナリ呼び出しに変換し、したがって部分式の排除の機会を増やすことができます。使用法：

```
@cse(@binarize(<your code here>))
```
