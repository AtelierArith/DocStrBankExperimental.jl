```
grid(items; ...)
grid(items...; ...)
```

与えられたアイテムから [`GridBox`](@ref) を作成します。

`items` 引数は次のいずれかです：

  * 他のアイテムのベクター。最上位のベクターはそのアイテムを列に配置します。ベクターのその後のネストは行と列を切り替え、複雑なレイアウトを簡単に作成できます。
  * 2つのネストレベルを指定する行列。
  * グリッドの端末エントリである `LayoutDOM` オブジェクト。
  * グリッドの行または列を明示的に指定する [`Row`](@ref) または [`Column`](@ref)。
  * 空のセルを表す `nothing`。

## キーワード引数

  * `sizing_mode`: 結果のグリッドのサイズの設定方法。
  * `nrows` または `ncols`: `items` がベクターで、これらの引数のいずれかが指定されている場合、指定された行数または列数で2Dレイアウトに分割されます。
  * `item_width`, `item_height`: 各アイテムの幅と高さ。
  * `merge_tools=true`: true の場合、構成プロットのツールバーが1つに統合されます。
  * `toolbar_location="above"`: 統合されたツールバーを配置する場所。`nothing` である場合もあります。
  * `toolbar_options`: 統合されたツールバーのオプションの名前付きタプル。
  * 残りの引数は [`GridBox`](@ref) に渡されます。
