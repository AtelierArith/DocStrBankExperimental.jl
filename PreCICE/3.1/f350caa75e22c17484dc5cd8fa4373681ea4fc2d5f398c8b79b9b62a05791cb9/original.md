```
getDataDimensions(meshName::String, dataName::String)::Integer
```

Returns the spatial dimensionality of the given data on the given mesh. Note that vectorial data dimensionality directly depends on the spacial dimensionality of the mesh.

# Arguments

  * `meshName::String`: Name of the associated mesh.
  * `dataName::String`: Name of the data to get the dimensionality of.
