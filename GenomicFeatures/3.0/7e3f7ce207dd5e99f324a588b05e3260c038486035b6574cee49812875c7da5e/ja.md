# 外部コンストラクタ

  * [`Strand(strand::Char)`](@ref)
  * [`Strand(strand::UInt8)`](@ref)

[`Strand`](@ref) は、次の表に示す4種類の値を取ることができます。

| シンボル  | 定数                    | 意味                |
|:----- |:--------------------- |:----------------- |
| `'?'` | [`STRAND_NA`](@ref)   | ストランドが不明または適用できない |
| `'+'` | [`STRAND_POS`](@ref)  | 正のストランド           |
| `'-'` | [`STRAND_NEG`](@ref)  | 負のストランド           |
| `'.'` | [`STRAND_BOTH`](@ref) | ストランドに特有でない特徴     |
