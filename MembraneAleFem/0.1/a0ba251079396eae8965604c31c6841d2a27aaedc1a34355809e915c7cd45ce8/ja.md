```
GpBasisFnsζ(gpwζ::Float64, ζ::Float64, kv::KnotVector)
```

提供された数値点 `ζ` での 1-D 基底関数 $N(ζ)$ と重み `gpwζ`。

B-スプラインの値と導関数は [`get_bspline_ders`](@ref) を使用して決定され、ここに格納されます。この構造体には4つのフィールドがあります：

  * `w`   –> ガウス点の重み、`gpw` として渡されます
  * `N`   –> 非ゼロ基底関数 $N(ζ)$ の (`poly`+1)×1 ベクトル
  * `dN`  –> 非ゼロの一次導関数 $N'(ζ)$ の (`poly`+1)×1 ベクトル
  * `ddN` –> 非ゼロの二次導関数 $N''(ζ)$ の (`poly`+1)×1 ベクトル
