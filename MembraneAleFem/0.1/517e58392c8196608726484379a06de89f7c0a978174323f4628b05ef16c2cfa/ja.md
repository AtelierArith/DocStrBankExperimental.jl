```
GpBasisFnsζα(fns1::GpBasisFnsζ, fns2::GpBasisFnsζ)
```

2-D 基底関数 $N(ζ^α)$ は、1-D [`GpBasisFnsζ`](@ref) `fns1` と `fns2` によって設定された単一のガウス点 $ζ^α$ で定義されます。

ここで $ζ^1$ と $ζ^2$ はそれぞれ `fns1` と `fns2` に渡された `ζ` 値に対応しますが、この情報は保存されていません。2-D 基底関数は 1-D 関数のテンソル積として計算されます。この構造体には4つのフィールドがあります：

  * `w` –> ガウス点の重み、`fns1.w × fns2.w` によって与えられます
  * `N` –> テンソル積構造によって順序付けられた非ゼロ基底関数 $N(ζ^α)$ の `NEN`×1 ベクトル
  * `∂Nα` –> 一階導関数の `NEN`×2 行列：行はそれぞれ $N_{, 1}$ と $N_{, 2}$ を含みます
  * `∂∂Nαβ` –> 二階導関数の `NEN`×3 行列：行はそれぞれ $N_{, 1 1}$、$N_{, 2 2}$、および $N_{, 1 2}$ を含みます
