```
step!(model; multi_threading = false)
```

この関数は、経済モデルの単一エポックをシミュレートし、異なる経済エージェント間の相互作用に基づいてモデルのさまざまなコンポーネントを更新します。シミュレーションの状態をカプセル化する`model`オブジェクトと、マルチスレッドを有効または無効にするためのオプションのブールパラメータ`multi_threading`を受け取ります。

実行される主な操作には以下が含まれます：

  * 企業や銀行の財務調整、破産チェックや利益計算を含む。
  * 成長、インフレ、中央銀行の金利などの経済的期待と調整。
  * 賃金の更新やローン処理を含む労働市場と信用市場の操作。
  * 消費や投資予算を含む家計の経済活動。
  * 予算編成や貿易収支を含む政府および国際貿易の財務活動。
  * 経済指標やポジションの変化を反映するための一般的な市場マッチングと会計更新。

この関数はモデルをインプレースで更新し、値を返しません。
