# リーマン・リウヴィル意味の積分を三角ストリップ行列を用いて離散化する。

```
fracint(f, α, end_point, h, RLIntMatrix())
```

三角ストリップ行列を使用して分数積分を近似します。

### 例

```julia-repl
julia> fracint(x->x^5, 0.5, 2.5, 0.0001, RLIntMatrix())
```

!!! info
    三角ストリップ行列法は、区間 $[0, T]$ における導関数を `Vector` で返します。


!!! tip
    進化した三角ストリップ行列法を使用すると、分数積分だけでなく、整数次数や高次の積分も計算できます!!


αを整数として設定してみてください。もちろん任意の整数で構いません！楽しんでいただけることを約束します😏

[三角ストリップ行列](https://en.wikipedia.org/wiki/Triangle_strip)を使用して積分を離散化します。
