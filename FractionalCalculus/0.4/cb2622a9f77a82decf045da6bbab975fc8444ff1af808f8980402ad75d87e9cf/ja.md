# リーマン・リウビル意味の分数積分を立方スプライン補間を用いて近似する。

### 例

```julia-repl
julia> fracint(x->x, 0.5, 1, 0.0001, RLIntCubicSplineInterp())
0.7529119186452693
```

この方法の誤差推定は $\mathcal{O(h^4)}$ であり、立方スプライン補間の誤差によって決まります。

!!! warning "hを0.001以上に設定する"
    何らかの理由で、**RLIntCubicSplineInterp** メソッドでは、**h** を0.001に設定するとより良い結果が得られます。
