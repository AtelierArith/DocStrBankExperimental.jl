# リーマン・リウヴィル意味での直接計算

```
fracint(f::Function, α, start_point, end_point, h, RLDirect())
```

リーマン・リウヴィル分数積分は、複素ステップ微分を使用しています。

$$
(J^αf)(t)=\frac{1}{\Gamma(α)} \int_0^t(t-τ)^{α-1}f(τ)dτ
$$

[QuadGK](https://github.com/JuliaMath/QuadGK.jl)を使用して積分を計算し、値を取得します。

### 例:

```julia-repl
julia> fracint(x->x^5, 0.5, 0, 2.5, 1e-8, RLDirect())
(64.36234189727955, 7.742994054849976e-7)
```

タプル (1.1639316474512205, 1.0183453796725215e-8) を返し、この導関数の値は 1.1639316474512205 であり、誤差推定は 1.0183453796725215e-8 です。
