# リーマン・リウヴィル意味の分数積分を区分的補間を用いて。

```
fracint(f, α, end_point, h, RLPiecewise())
```

### 例

```julia-repl
julia> fracint(x->x^5, 0.5, 2.5, 0.0001, RLPiecewise())
64.36234206345209
```

小さなステップサイズで元の関数を近似するために区分的補間を展開することで、この方法は高速であり、メモリの割り当ても少なくて済みます。

区分線形補間を使用すると：

$$
    y_n(t)=\frac{t-t_{i+1}}{t_i-t_{i+1}}y(t_i)+\frac{t-t_i}{t_{i+1}-t_i}y(t_{i+1})
$$

補間を用いて元の関数を構成し、総和を実装します。
