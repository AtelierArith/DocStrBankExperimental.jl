# リーマン・リウビル意味のG1スキーム

```
fracdiff(f, α, start_point, end_point, h, RLG1())
```

グルンワルド・レトニコフ分数導関数の定義から限界記号を取り除き、切り捨てた級数の形の離散化スキームを導出します。

!!! tip
    ```
    **RLG1** は分数導関数のために ``+\alpha``、分数積分のために ``-\alpha`` を計算するためにも使用できます。
    ```
