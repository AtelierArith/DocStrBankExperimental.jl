1つのパズルインスタンスを保持します。

# `Puzzle{T}`のフィールド

  * `palette`: パズルの許可された色の反復可能なコレクションで、各色は型`T`です。色がないことは`zero(T)`で表されるため、`zero(T)`は定義されている必要があり、`palette`には含まれていない必要があります。
  * `sR::Vector{Vector{Int}}`: 上から下への各行の数値ヒント。空の行は`Int[]`として入力されます（例えば`[0]`ではなく）。
  * `cR::Vector{Vector{T}}`: `sR`のヒントに対応する`palette`内の色。
  * `sC::Vector{Vector{Int}}`: 左から右への各列の数値ヒント。空の列は`Int[]`として入力されます。
  * `cC::Vector{Vector{T}}`: `sC`のヒントに対応する`palette`内の色。

# 非デフォルトコンストラクタ

以下の`Puzzle`のコンストラクタは、デフォルトコンストラクタよりも推奨されます。

## モノクロパズル用

```
Puzzle(sR::Vector{Vector{Int}}, sC::Vector{Vector{Int}})
```

行のヒントを含む`sR`と列のヒントを含む`sC`を持つモノクロパズルを保持します。

このコンストラクタは、パズルの単一の色（伝統的に「黒」）を`1`に、色がないこと（伝統的に「白」）を`0`に、`Puzzle.palette`を`1:1`に設定します。

## 多色パズル用

```
Puzzle(sR, cR, sC, cC)
```

`cR`と`cC`から`Puzzle.palette`を構築します。それ以外はデフォルトコンストラクタと同じです。

[`read_puzzle_from_cwc`](@ref)も参照してください。
