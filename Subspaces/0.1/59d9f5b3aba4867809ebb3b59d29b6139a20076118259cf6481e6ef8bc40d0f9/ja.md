```
Subspace(basis::AbstractArray{AbstractArray{N}, 1}; tol::Real=1.0e-6)
```

与えられた基底から、基底ベクトルのリストとして表現された部分空間を作成します。

```
Subspace(basis::AbstractArray{N+1}; tol::Real=1.0e-6)
```

多次元配列として与えられた基底から部分空間を作成します。最後の配列インデックスは基底要素を列挙します。例えば、行列が与えられた場合、このコンストラクタはその行列の列のスパンを表す部分空間を作成します。

`tol` パラメータは、ベクトルが線形依存であるかどうかを判断するための許容誤差を設定します。

  * `basis::Array`: この部分空間のための直交正規基底。 この配列の最終インデックスは基底ベクトルをインデックスします。
  * `perp::Array`: 垂直部分空間のための直交正規基底。
  * `tol::AbstractFloat`: ベクトルが線形依存であるかどうかを判断するための許容誤差。

```jldoctest
julia> Subspace([[1, 2, 3], [4, 5, 6]]) == Subspace([ 1 4; 2 5; 3 6])
true
```
