```
model, Av, selected = nnmf_model(m = 100, n = 50, k = 10, T = Float64)
```

非負行列因子分解目的を表す `NLPModel` のインスタンスを返します

```
f(W, H) = ½ ‖A - WH‖₂²,
```

ここで A ∈ Rᵐˣⁿ は非負のエントリを持ち、k クラスタに分けることができます。`Av = A[:]`。インデックスのベクトル `selected = k*m+1: k*(m+n)` は、正則化子を適用する W ∈ Rᵐˣᵏ および H ∈ Rᵏˣⁿ の成分を示すために使用されます（正則化子は H のエントリのみに適用されます）。

## 引数

  * `m :: Int`: A の行数
  * `n :: Int`: A の列数（`n` ≥ `m`）
  * `k :: Int`: クラスタの数
