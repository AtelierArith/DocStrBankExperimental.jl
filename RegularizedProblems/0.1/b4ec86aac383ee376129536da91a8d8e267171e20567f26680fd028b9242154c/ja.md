```
model, nls_model, sol = group_lasso_model(; kwargs...)
```

グループラッソ問題を表す`NLPModel`と`NLSModel`のインスタンスを返します。すなわち、過小決定線形最小二乗目的

½ ‖Ax - b‖₂²,

ここで、Aは直交正規行であり、b = A * x̄ + ϵ、x̄はスパースであり、ϵは平均ゼロおよび標準偏差σの正規分布に従うノイズベクトルです。この形式では、すべてのグループが同じ数の要素を持ち、グループの数は全要素の数に均等に割り切れます。

## キーワード引数

  * `m :: Int`: Aの行数（デフォルト: 200）
  * `n :: Int`: Aの列数、`n` ≥ `m`（デフォルト: 512）
  * `g :: Int`: グループの数（デフォルト: 16）
  * `ag :: Int`: アクティブグループの数（デフォルト: 5）
  * `noise :: Float64`: ノイズ量（デフォルト: 0.01）
  * `compound :: Int`: `m`、`n`、`g`、および`ag`の乗数（デフォルト: 1）。

## 戻り値

グループラッソ問題を表す`FirstOrderModel`のインスタンス。グループラッソ問題を表す`FirstOrderNLSModel`のインスタンスも返します。また、真のx、グループの数g、アクティブなグループを示すグループインデックス、およびxのグループインデックスが行である行列を返します。
