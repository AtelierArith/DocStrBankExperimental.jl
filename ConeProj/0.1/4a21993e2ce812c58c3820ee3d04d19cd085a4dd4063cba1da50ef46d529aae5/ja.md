QR因子分解に列を追加する（Qを使用せずに）。

`R = qraddcol(A,R,v)` は、m-by-n行列 `A` のQR因子分解にmベクトル `v` を追加します。入力時、`R` は密なn-by-n上三角行列であり、`R'*R = A'*A` です。

`R = qraddcol(A,R,v,β)` は上記と似ていますが、このルーチンは次のQR因子分解を更新します。

[A; β I],   そして   R'*R = (A'*A + β^2*I) = R'*R。

`A` は列数が行数より少ない必要があります。`A` と `v` は疎または密である可能性があります。出力時、`R` は次に対応する(n+1)-by-(n+1)因子です。

Anew = [A        V ],    Rnew = [R   u  ],   Rnew'Rnew = Anew'Anew.          [beta*I     ]            [  gamma]          [       beta]

新しい列 `v` は非ゼロであると仮定されます。`A` にまだ列がない場合は、`A = []`、`R = []` を入力します。
