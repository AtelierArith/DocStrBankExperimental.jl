```
Codata
```

CODATAからの自然定数を保持するオブジェクトです。関数[`castCodata`](@ref)を使用して作成するのが最適です。

フィールドは次のとおりです：

  * `.∆νCs`: Cs超微細遷移周波数（`::Value`）
  * `.c`: 真空中の光速（`::Value`）
  * `.h`: プランク定数（`::Value`）
  * `.ħ`: 簡約プランク定数（`::Value`）
  * `.e`: 基本電荷（`::Value`）
  * `.kB`: ボルツマン定数（`::Value`）
  * `.NA`: アボガドロ定数（`::Value`）
  * `.Kcd`: 輝度効率（`::Value`）
  * `.me`: 電子の静止質量（`::Value`）
  * `.R∞`: リュードベリ定数（`::Value`）
  * `.Ry`: リュードベリ周波数（`::Value`）
  * `.Eh`: ハートリー a.u.（`::Value`）
  * `.α`: 微細構造定数（`::Value`）
  * `.μ0`: 真空の磁気透過率（`::Value`）
  * `.ε0`: 真空の電気透過率（`::Value`）
  * `.KJ`: ジョセフソン定数（`::Value`）
  * `.RK`: フォン・クリッツィング定数（`::Value`）
  * `.R`: モルガス定数（`::Value`）
  * `.u`: 統一原子質量単位（`::Value`）
  * `.matE`: 単位変換行列（Matrix{Float64}）

#### 例：

```
julia> codata = castCodata(2022);

julia> codata.μ0
Value(1.2566370612696005e-6, "N A⁻²")

julia> codata.μ0.val
1.2566370612696005e-6
```
