```
castTerm(n::Int; ℓ=0, S=1//2, L=0, J=1//2, msg=false)
```

*Term記法*でのTermを指定します：

  * `.n`: 主量子数
  * `.n′`: 放射量子数（ノードの数 - 自動生成）
  * `.ℓ`: 軌道角運動量の価電子
  * `.S`: 全電子スピン
  * `.L`: 全軌道角運動量
  * `.J`: 全電子角運動量

#### 例：

```
julia> castTerm(1; ℓ=0, S=1//2, L=0, J=1//2, msg=true)
Term created: 1s ²S₁⸝₂; n = 1,  n′ = 0, ℓ = 0, S = 1//2, L = 0, J = 1//2
Term("1s ²S₁⸝₂", 1, 0, 0, 1//2, 0, 1//2)
```
