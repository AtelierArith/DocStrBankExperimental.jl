```
mutable struct Pos{T} where T<:Real
```

フィールドを持つ型:

  * `.Na::Int`: 最後の先行点のグリッドインデックス
  * `.Nlctp::Int`: 下の古典的転回点のグリッドインデックス
  * `.Nmin::Int`: (スクリーンされた)ポテンシャル最小値のグリッドインデックス
  * `.Nuctp::Int`: 上の古典的転回点のグリッドインデックス
  * `.Nb::Int`: 最初の後行点のグリッドインデックス
  * `.N::Int`: 最後の点のグリッドインデックス
  * `.nodes::Int`: 縮小波動関数におけるノードの数 (r ≠ 0)
  * `.ΔNlctp::Float64`: Nlctp に対する lctp オフセット (1.0 ≤ ΔN ≤ 1.0)
  * `.ΔNuctp::Float64`: Nuctp に対する uctp オフセット (-1.0 ≤ ΔN ≤ 0.0)

特別なグリッドインデックスとノードの数、及び Nuctp に対する正確な uctp の (負の) オフセットを保持するための可変構造体。`Pos` は [`Def`](@ref) オブジェクトのフィールドの一つです。

#### 例:

```
julia> pos = Pos(1, 2, 3, 4, 5, 6, 7, 8.0, 9.0, 10.0);
julia> pos.Nuctp
4

julia> pos.Nuctp = 8;
julia> pos
Pos(1, 2, 3, 8, 5, 6, 7, 8.0, 9.0, 10.0)
```
