```
castSpinorbit(;n=1, ℓ=0, mℓ=0, up=true, msg=false)
```

`Spinorbit`をフィールドで作成します：

  * `.name`: スピン軌道名（文字列）
  * `.n`: 主量子数
  * `.n′`: 放射量子数（放射波動関数のノード数）
  * `.ℓ`: 価電子の軌道角運動量
  * `.mℓ`: 価電子の軌道角運動量の投影
  * `.ms`: スピン磁気量子数（Rational{Int}）

#### 例：

```
julia> castSpinorbit(n=1, ℓ=0, msg=true)
スピン軌道: 1s↑
    主量子数: n = 1
    放射量子数: n′ = 0（放射波動関数のノード数）
    価電子の軌道角運動量: ℓ = 0
    価電子の軌道角運動量の投影: mℓ = 0
    スピン磁気量子数: ms = 1/2
Spinorbit("1s↑", 1, 0, 0, 0, 1//2)
```

```
castSpinorbit(config::String; msg=false)
```

#### 例：

```
julia> castSpinorbit1("2p"; msg=true)
スピン軌道: 2p↓
    主量子数: n = 2
    放射量子数: n′ = 0（放射波動関数のノード数）
    価電子の軌道角運動量: ℓ = 1
    価電子の軌道角運動量の投影: mℓ = 1
    スピン磁気量子数: ms = -1/2
Spinorbit("2p↓", 2, 0, 1, 1, -1//2)

julia> castSpinorbit1("2p↑-1"; msg=true)
スピン軌道: 2p↑
    主量子数: n = 2
    放射量子数: n′ = 0（放射波動関数のノード数）
    価電子の軌道角運動量: ℓ = 1
    価電子の軌道角運動量の投影: mℓ = -1
    スピン磁気量子数: ms = 1/2
Spinorbit("2p↑", 2, 0, 1, -1, 1//2)
```
