```
castOrbit(;n=1, ℓ=0, mℓ=0, msg=true)
```

`Orbit`を作成します。フィールドは次のとおりです：

  * `.name`: 名前
  * `.n`: 主量子数
  * `.n′`: 放射量子数（放射波動関数のノード数）
  * `.ℓ`: 価電子の軌道角運動量
  * `.mℓ`: 価電子の軌道角運動量の投影

#### 例：

```
julia> castOrbit(n=1, ℓ=0; msg=true)
Orbital: 1s
    主量子数: n = 1
    放射量子数: n′ = 0 (放射波動関数のノード数)
    価電子の軌道角運動量: ℓ = 0
    価電子の軌道角運動量の投影: mℓ = 0
Orbit("1s", 1, 0, 0, 0)
```

```
castOrbit(strOrbit::String; mℓ=0, msg=false)
```

#### 例：

```
julia> castOrbit("2p"; mℓ=-1, msg=true)
Orbital: 2p
    主量子数: n = 2
    放射量子数: n′ = 0 (放射波動関数のノード数)
    価電子の軌道角運動量: ℓ = 1
    価電子の軌道角運動量の投影: mℓ = -1
Orbit("2p", 2, 0, 1, -1)
```
