```
output::Array{Complex{T}} = finufft_exec(plan::finufft_plan{T},
                  input::FINUFFT.InputArray{Complex{T}}) where T <: finufftReal
```

計画内で単一または多ベクトルのFINUFFT変換を実行します。

output = finufft_exec(plan, input)

`plan`は、必要な非一様点座標をすべて含む以前に作成された`finufft_plan`オブジェクトであり、単一（または`ntrans>1`の場合は複数）のNUFFT変換を実行し、`input`からの強度またはフーリエ係数入力ベクトルを使用します。変換の結果は（多次元の可能性がある）配列として返されます。

# 入力

```
- `plan`    `finufft_plan`オブジェクト、すでに計画されており非一様点を含む。
- `input`   強度（タイプ1または3）またはフーリエ係数（タイプ2）ベクトル、行列、
            または適切なサイズの配列。タイプ1および3の場合、これは長さMの
            ベクトル（Mは`xj`の長さ）であるか、`ntrans>1`の場合は
            `(M,ntrans)`行列です。タイプ2の場合、1Dではサイズが`(ms,)`、
            2Dではサイズが`(ms,mt)`、3Dではサイズが`(ms,mt,mu)`、
            またはこれらのそれぞれに`ntrans`の追加の最後の次元がある場合です。
            連続している必要があります。
```

# 出力

```
 `output`   ターゲットでの出力強度のベクトル（タイプ2または3）、またはフーリエ
            係数の配列（タイプ1）、または`ntrans>1`の場合は、そのようなベクトルまたは
            配列のスタックで、適切なサイズです。具体的には、`ntrans=1`の場合、
            タイプ1では1Dでサイズが`(ms,)`、2Dでサイズが`(ms,mt)`、3Dでサイズが
            `(ms,mt,mu)`です。タイプ2および3の場合は、長さ`M`（タイプ2の`xj`の長さ）
            の列ベクトル、または長さ`nk`（タイプ3の`s`の長さ）です。`ntrans>1`の場合は、
            そのようなオブジェクトのスタックであり、つまり、追加の最後の次元`ntrans`があります。
```
