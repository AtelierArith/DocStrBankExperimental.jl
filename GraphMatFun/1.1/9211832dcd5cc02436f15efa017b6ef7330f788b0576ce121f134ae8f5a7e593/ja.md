```
 (graph,crefs)=graph_horner_degopt(a; scaling=1.0, input=:A)
```

ホーナーの評価スキームを使用して多項式を生成します。多項式

```
 p(s) = a[1] + a[2]*(αs) + ... + a[n-1]*(αs)^(n-2) + a[n]*(αs)^(n-1),
```

は次のように評価されます。

```
 p(s) = a[1] + (αs)*(a[2] + (αs)*(... + (αs)*(a[n-1] + a[n]*(αs))...)),
```

ここで、α=`scaling`です。ただし、この関数は[`graph_degopt`](@ref)への呼び出しを使用しており、`crefs`における自由度が増加します。さらに[`graph_horner`](@ref)も参照してください。
