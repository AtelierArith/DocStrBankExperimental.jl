```
e_bwd=compute_bnd_rel_bwd_err(
    f,
    graph::Compgraph{T};
    coefftype = T,
    numterms = 100,
    numdigits = 100,
)
```

関数 `f` の相対後方誤差の上限を計算します。

返される関数 $e_{bwd}$ は、計算グラフ `graph` の基になる多項式の相対後方誤差の上限であり、`f` の近似として見なされます。基になる多項式 $p$ に対して、この関数は `|δ|` の上限を返します。ここで `δ` は $f(z+δ) = p(z)$ を満たすものです。

最初の引数 `f` はシンボルです。現在サポートされている値は以下の通りです：

  * `:exp` 上限は、恒等式 δ = log((exp(-z) p(z)-1)+1) によって計算されます。コードは方程式の右辺の級数展開を計算し、最初の `numterms` の係数に切り捨て、続く多項式の各係数をその絶対値で上限を取ります。この計算は `numdigits` 桁の精度を使用します。
