```
(graph,cref)=graph_sastre_yks_degopt(k,s,c)
```

式 (62)-(65) で与えられた多項式評価形式を `degop` 形式に変換します。`graph` は `y_{k,s}` の表現です。入力 `c` は、表現 (62)-(65) によって与えられた係数のグループ化です。`c` は長さ `k+1` の `Vector{Vector{Vector}}` であり、次のように表されます。

```
[
[c_i^{(0,1)}, c_i^{(0,2)}]
[c_i^{(1,1)}, c_i^{(1,2)}, c_i^{(1,3)}, c_i^{(1,4)}, c_i^{(1,5)}, c_i^{(1,6)}]
...
[c_i^{(k,1)}, c_i^{(k,2)}, c_i^{(k,3)}, c_i^{(k,4)}, c_i^{(k,5)}, c_i^{(k,6)}]
]
```

したがって、`c[1]` には2つのベクトルが含まれ、最初のベクトルの長さは `s`-1 で、2番目のベクトルの長さは `s` です。（注：最初のベクトルでは I の定数がゼロに設定されています）`c[2]` から `c[k+1]` までには6つのベクトルが含まれています：偶数番号のベクトルは長さ `s`+1、奇数番号のベクトルは長さ j-1 で、ここで j は `c` のインデックスです。例えば、`c[2][1]` には1つの要素があり、`c[3][5]` には2つの要素があります。例えば、式 (57)-(59) は次のように実装されています。

```
c = [
[[c15, c16], [0.0, 0, 0]], # (57)
[[1.0], [0, c13, c14], [1.0], [c11, 0, c12], [c10], [0.0, 0, 0]], # (58)
[[0.0, 1], [0, c8, c9], [c7, 1], [0, c6, 0], [c4, c5], [c1, c2, c3]], # (59)
]
```
