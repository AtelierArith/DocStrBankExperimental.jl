```
 (graph,crefs)=graph_horner(a; input=:A, B=:B, C=:C, scaling=1.0)
```

ホーナー法を使用して多項式のグラフを生成します。より正確には、これは多項式の評価に対応します。

```
p(s) = a[1] + a[2]*(αs) + ... + a[n-1]*(αs)^(n-2) + a[n]*(αs)^(n-1),
```

再帰として

```
Cj=s*B{j+1}
Bj=a[j]*I+α*Cj,
```

ここで `α=scaling` です。

キーワード引数 `B` と `C` は、これらの中間変数のベース名を指定します。
