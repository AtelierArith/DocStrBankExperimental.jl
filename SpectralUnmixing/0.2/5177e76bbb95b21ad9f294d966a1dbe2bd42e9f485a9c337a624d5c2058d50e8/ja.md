```
scale_data(refl::Matrix{Float64}, wavelengths::Vector{Float64}, criteria::String,
    bad_regions_wl=[[1300, 1500], [1800, 2000]])
```

指定された基準に基づいて反射率データをスケーリングします。

# 引数

  * `refl::Matrix{Float64}`: 反射率データはサイズ (n,) または (m,n) で、ここで n はバンドの数、m はピクセルの数です。
  * `wavelengths::Vector{Float64}`: サイズ (n,) の波長のベクトル。
  * `criteria::String`: 正規化オプション:

      * `"none"`: スケーリングは適用されず、元の反射率データが返されます。
      * `"brightness"`: 指定された悪い波長領域 [1300, 1500] および [1800, 2000] を除外して、データが明るさに基づいて正規化されます。
      * 特定の波長（文字列として）: この波長での反射率値を使用してデータが正規化されます。
  * `bad_regions_wl`=[[1300, 1500], [1800, 2000]]: 無視する波長領域のリスト（nm単位）。

# 戻り値

  * `refl` と同じ次元のスケーリングされた反射率データの行列。
