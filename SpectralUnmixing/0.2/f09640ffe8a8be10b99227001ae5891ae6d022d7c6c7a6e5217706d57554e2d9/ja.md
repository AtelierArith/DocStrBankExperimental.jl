```
write_results(output_files::Vector{String}, output_bands::Vector, x_len::Int64,
              y_len::Int64, results, args)
```

指定された出力ENVIラスターファイルに処理された結果を書き込みます。これには、主な出力、推定不確実性、および提供されたパラメータに基づく完全な割合が含まれます。

# 引数

  * `output_files::Vector{String}`: 作成または更新される出力データセットのファイルパス。各エントリは別々の出力ラスタに対応します。
  * `output_bands::Vector`: 各出力データセットのバンド数。このベクトルの長さは出力ファイルの数と一致する必要があります。
  * `x_len::Int64`: 出力データセットの幅。
  * `y_len::Int64`: 出力データセットの高さ。
  * `results`: 各結果がタプルまたは類似の構造であることが期待される結果のコレクション。特定の要素には、出力データセットに書き込むデータが含まれています。
  * `args`: 書き込み動作を指示する追加のパラメータを含むオブジェクトまたは構造体。利用可能なオプションは次のとおりです：     - `args.n_mc`: 1より大きい場合、2番目の出力ファイルにも不確実性出力が書き込まれます。     - `args.write_complete_fractions`: 1に設定されている場合、次の出力ファイルに完全な割合を書き込みます。

# 注意事項

  * 主な出力は、`-9999`の無効値で初期化され、`output_files`で指定された最初の出力ファイルに書き込まれます。
  * 各出力配列は、書き込む前に希望の次元に合わせて順列されます。
