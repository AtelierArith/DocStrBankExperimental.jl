```
Halo{X} <: Padding
```

配列の周りにメモリ内のハローを使用するパディング。これにより、配列の端を超えるステンシルの一部が、境界チェックや条件なしに直接インデックスできるようになります。これにより、ウィンドウブロードキャスト中のパフォーマンスが向上する可能性がありますが、いくつかの欠点もあります。

`:out` モードでは、元の配列よりも大きい新しい配列が全体的に割り当てられます。これを複数回使用しない限り、行う価値はないかもしれません。`:in` モードでは、配列の外側の端がパディングとして使用されます。パディング値を使用することによる境界効果がないため、こちらの方がより正確かもしれません。

# 例

```julia
halo_in = Halo(:in)
halo_out = Halo(:out)
```
