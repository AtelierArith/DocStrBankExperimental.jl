Repository of International Planning Competition (IPC) domains and problems, hosted at https://github.com/potassco/pddl-instances.

  * Collections: Formatted as "ipc-YYYY", where YYYY is a IPC year.
  * Domains: Specified as a string (e.g. "blocks-strips-typed"). When the   collection is not provided as an argument, it should be prepended to the   domain name with a dash (e.g. "ipc-2000-blocks-strips-typed").
  * Problems: Formatted as "instance-i", generally organized by increasing difficulty.
