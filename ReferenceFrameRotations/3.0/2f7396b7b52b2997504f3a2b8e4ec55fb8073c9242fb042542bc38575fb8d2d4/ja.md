```
Quaternion{T}
```

クォータニオンの定義。

# フィールド

  * `q0::T`: クォータニオンの実部。
  * `q1::T`: クォータニオンの虚部のX成分。
  * `q2::T`: クォータニオンの虚部のY成分。
  * `q3::T`: クォータニオンの虚部のZ成分。

!!! note
    この構造体におけるクォータニオン `q` は次のように表されます：

    ```
    q = q0 + q1.i + q2.j + q3.k
    ```


# 例

```julia-repl
julia> Quaternion(cosd(45), sind(45), 0, 0)
Quaternion{Float64}:
  + 0.7071067811865476 + 0.7071067811865476.i + 0.0.j + 0.0.k
```
