```
mcm(spec::Symbol, alm₁::Alm{T}, alm₂::Alm{T}; lmax=nothing)
```

モード結合行列を計算します。例については、ドキュメントの[スペクトル分析](@ref)セクションを参照してください。これらは、`SpectralArray{T,1}`に対して線形解法演算子`\`を適用することで使用されます。

`spec`の選択肢：

  * `:TT`、`M⁰⁰`と同じ
  * `:TE`、`:ET`、`:TB`、`:BT`、`:M⁰²`、`:M²⁰`と同じ
  * `:EE_BB`、スタックされたEEおよびBBベクトルの結合行列を返します
  * `:EB_BE`、スタックされたEBおよびBEベクトルの結合行列を返します
  * `:M⁺⁺`、スピン2モード結合行列のサブブロック
  * `:M⁻⁻`、スピン2モード結合行列のサブブロック

# 引数：

  * `spec::Symbol`：モード結合行列のクロススペクトル
  * `alm₁::Alm{T}`：最初のマスクの球面調和係数
  * `alm₂::Alm{T}`：2番目のマスクの球面調和係数

# キーワード

  * `lmin=0`：モード結合行列の最小倍数
  * `lmax=nothing`：モード結合行列の最大多極

# 戻り値：

  * モード結合行列。単一のシンボルの場合、`SpectralArray{T,2}`を返します。`spec`が`:EE_BB`または`:EB_BE`の場合、2×2ブロックを持つ`BlockSpectralMatrix{T}`を返します。
