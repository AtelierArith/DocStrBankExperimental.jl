```
statproducts_league(pd, league_cp; level_limit=50.0, iv_floor=0) → statprods
```

各可能なIVの組み合わせに関連するステータスプロダクトを計算します。`statprods[a, d, h]`は、IVが`(a, d, h)`の`pd`に対するステータスプロダクトを返します。`statprods`のインデックスは0から始まります。

`level_limit`はパワーアップの制限を設定し、`iv_floor`は3つのカテゴリすべてにおける最小IVを設定します。以下のフロアが適用されます：

  * レイド、卵の孵化、リサーチ：10
  * 天候ブースト：4
  * トレード：

      * 良い友達：1
      * 素晴らしい友達：2
      * ウルトラ友達：3
      * 親友：5
      * ラッキー：12

`statprods[a, d, h]`は、これらの要件を満たさないIVの組み合わせに対してはゼロです。
