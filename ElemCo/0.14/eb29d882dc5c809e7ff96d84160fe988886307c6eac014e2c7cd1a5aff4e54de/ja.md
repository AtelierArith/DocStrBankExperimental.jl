```
@cc(method, kwargs...)
```

カップルクラスター計算を実行します。

メソッドのタイプは最初の引数（ccsd/ccsd(t)/dcsdなど）によって決まります。メソッドは文字列または変数として指定できます。例えば、`@cc CCSD`または`@cc "CCSD"`または`ccmethod="CCSD"; @cc ccmethod`のように指定します。

# キーワード引数

  * `fcidump::String`: fcidumpファイル（デフォルト: ""、すなわち`EC`からの積分を使用）。
  * `occa::String`: 占有α軌道（デフォルト: "-"）。
  * `occb::String`: 占有β軌道（デフォルト: "-"）。

占有文字列は`+`で区切られたリストとして指定できます。例えば、`occa = 1+2+3`または同等の`1-3`のように指定します。さらに、軌道の空間対称性は`orb.sym`の構文で指定できます。例えば、`occa = "-5.1+-2.2+-4.3"`のように指定します。

# 例

```julia
geometry="bohr
O      0.000000000    0.000000000   -0.130186067
H1     0.000000000    1.489124508    1.033245507
H2     0.000000000   -1.489124508    1.033245507"
basis = Dict("ao"=>"cc-pVDZ", "jkfit"=>"cc-pvtz-jkfit", "mpfit"=>"cc-pvdz-mpfit")
@dfhf
@dfints
@cc ccsd
```
