```
opsmethods()
```

パッケージ内の利用可能なアルゴリズムを表示します。

# 例

```julia
julia> using OnlinePortfolioSelection

julia> opsmethods()

      ===== OnlinePortfolioSelection.jl =====
            現在利用可能なメソッド
       =====================================

        up: ユニバーサルポートフォリオ - `up`を呼び出す
        eg: 指数勾配 - `eg`を呼び出す
     cornu: CORN-U - `cornu`を呼び出す
          ⋮
```
