```
qcla_out_adder_circuit(N)
```

`N`量子ビットで表される2つの整数のためのアウトオブプレイス加算器を構築します。`CircuitGateChain{3N+1}`オブジェクトを返します。Draperらによる量子キャリー先読み加算器回路に基づいています（Quant. Inf. Comp. 6, 351-369 (2006), arXiv:quant-ph/0406142）。N+1の補助ワイヤがあるため、`CircuitGateChain{3N+1}`を返します。加算される2つの整数が次のように表されるとします：$A = a_{0}\times 2^{0} + a_{1} \times 2^{1} + a_{2} \times 2^{2} + .. + a_{N} \times 2^{N} \\ B = b_{0}\times 2^{0} + b_{1} \times 2^{1} + b_{2} \times 2^{2} + .. + b_{N} \times 2^{N}$ 入力インデックスは、`1`からインデックスが始まるJuliaにおいて、$a_{0}a_{1}a_{2}..a_{N}b_{0}b_{1}b_{2}..b_{N} + 1$とする必要があります。出力インデックスは、$C = A+B = c_{0}\times 2^{0} + c_{1} \times 2^{1} + c_{2} \times 2^{2} + ... + c_{N+1} \times 2^{N+1}$の形で、$a_{0}a_{1}a_{2}..a_{N}b_{0}b_{1}b_{2}..b_{N}c_{0}c_{1}c_{2}..c_{N+1} + 1$となります。
