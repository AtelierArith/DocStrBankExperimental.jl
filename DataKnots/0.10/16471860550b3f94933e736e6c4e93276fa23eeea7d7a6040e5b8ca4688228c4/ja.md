```
 @query expr
```

特化されたパスのような表記法からクエリオブジェクトを作成します：

  * ベア識別子は `Get` を使ったナビゲーションに変換されます；
  * `Count(X)` のようなクエリコンビネータは小文字の名前を使用します；
  * ピリオド（`.`）はクエリの合成（`>>`）に使用されます；
  * `Count` のような集約クエリは括弧を必要とします；
  * レコードは波括弧 `{}` を使用して構築できます；および
  * 関数と演算子は自動的にリフトされます。

```jldoctest
julia> @query 2x+1
Lift(+, (Lift(*, (Lift(2), Get(:x))), Lift(1)))
```
