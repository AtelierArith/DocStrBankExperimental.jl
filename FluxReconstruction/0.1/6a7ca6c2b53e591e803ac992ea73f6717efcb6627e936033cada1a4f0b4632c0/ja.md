X = λ¹V¹ + λ²V² + λ³V³ + λ⁴V⁴

λsは双線形矩形形状関数です:   (http://people.ucalgary.ca/~aknigh/fea/fea/rectangles/r1.html)   λ¹ = (r-1)(s-1)/4   λ² = (r+1)(1-s)/4   λ³ = (r+1)(s+1)/4   λ⁴ = (1-r)(s+1)/4

ヤコビ行列:   Xᵣ = (s-1)V¹/4 + (1-s)V²/4 + (s+1)V³/4 - (s+1)V⁴/4   Xₛ = (r-1)V¹/4 - (r+1)V²/4 + (r+1)V³/4 + (1-r)V⁴/4

線形単体要素とは異なり、Jは一般的な線形四角形内の点ごとに異なります。特別な場合として、ヤコビ行列は矩形メッシュの各要素に対して定数です。
