```
writefile!(edfh, newpath; acquire=dummyacquire, sigformat=same)
```

edfh構造体のデータをnewpathで示されたファイルに書き込みます。書き込まれたファイルのファイルハンドルを返し、読み取り用に開かれます。

注意: ヘッダーは、すべてのデータレコードが書き込まれた後に更新される最終的なレコード数を除いて、関数の開始時に完全に指定する必要があります。データが書き込まれると同時に記録されるシステムの場合、acquire(edfh)関数はヘッダーのパラメータに従ってデータを書き込む必要があります。注: 関数がBDFからEDFまたはEDFからBDFに変換する場合、edfh構造体は変更されます。
