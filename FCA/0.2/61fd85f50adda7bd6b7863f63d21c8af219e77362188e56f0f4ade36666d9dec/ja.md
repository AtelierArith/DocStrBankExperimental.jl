```
mat_embed(X, dim; zero_pos = -1, tpin = "her", tpout = "her")
```

この関数は、Xを新しい次元dimに埋め込み、エントリの直交順序を保持します。必要に応じてzero_posにゼロを埋めます。  

# 引数

  * `X`: エルミート行列または長方形行列
  * `dim`: 目標次元、配列のような形 [Nnew,Mnew]
  * `zero_pos`: ゼロのためにユーザーが指定した直交位置の配列。-1の場合は、ランダムに生成された位置を使用します。
  * `tpin`: 入力行列のタイプ、有効なオプションは "her" と "rec" です。
  * `tpout`: 出力行列のタイプ、有効なオプションは "her" と "rec" です。

# 出力

  * `Xnew`: 次元dimおよびタイプtpinの行列。エントリXnewはXからのもので、zero_posに追加のゼロがある可能性があります。
