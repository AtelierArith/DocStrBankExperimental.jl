```
icf(Z; opt = "orth")
```

Zに独立成分分析を適用し、推定された混合行列と独立成分を返します。

# 引数

  * `Z`: 同じ長さのベクトルの配列で、混合された独立信号の実現を表します
  * `obj`: 損失関数のタイプ、有効なオプションは "kurt" と "ent" です
  * `opt`: 文字列、最適化のタイプ、有効なオプション: "orth", "sphe"。 "sphe" は "kurt" 損失関数専用に設計されています

# 出力

  * `Aest`: サイズ `s`-by-`s` の推定混合行列、ここで `s` = size(Z,1)
  * `Xest`: 推定された独立成分 `Xest = pinv(Aest)*Z`
