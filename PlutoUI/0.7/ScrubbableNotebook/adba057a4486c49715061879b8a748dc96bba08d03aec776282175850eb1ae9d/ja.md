インライン番号は、マウスをクリックしてドラッグすることで変更できます。

# 例

```julia
md"""
_アリスが $(@bind a Scrubbable(20)) 個のリンゴを持っていて、 
彼女がボブに $(@bind b Scrubbable(3)) 個のリンゴを渡すと..._
"""
```

```julia
md"""
_...するとアリスは **$(a - b)** 個のリンゴを残します。_
"""
```

上記の例では、**初期値**をパラメータとして与え、読者はそれを低くしたり高くしたりできます。

## カスタム範囲

初期値に加えて、スクラバブルな数値には到達可能な値の配列もあります。`Scrubbable`に**単一の数値**を渡すと、この配列は自動的に作成されます。

配列を**手動で指定することもできます**：

```julia
@bind apples Scrubbable(200 : 300; default=220)
```

## フォーマット

ライブラリ [`d3-format`](https://github.com/d3/d3-format) は、浮動小数点数をフォーマットするために使用されます。スクラバブルな値をフォーマットするために使用される**フォーマット文字列**を `".2f"` のように指定できます。詳細な例については、彼らの[ドキュメント](https://github.com/d3/d3-format)を参照してください。

`@bind money Scrubbable(30e6, format=".0s", prefix="€ ")`

`@bind coolness Scrubbable(0.80 : 0.01 : 1.00, format=".0%", prefix="あなたは🌝 ", suffix=" クール")`
