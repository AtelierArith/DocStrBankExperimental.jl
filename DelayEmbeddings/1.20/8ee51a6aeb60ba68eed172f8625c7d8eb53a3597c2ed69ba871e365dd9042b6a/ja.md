```
GeneralizedEmbedding(τs, js = ones(length(τs)), ws = nothing) -> `embedding`
```

遅延座標埋め込み構造を関数として使用するために返します。時系列 *または* 軌道（すなわち `Dataset`） `s` が与えられ、次のように呼び出すと

```julia
embedding(s, n)
```

一般化埋め込みを使用して、埋め込まれた空間の `s` の `n` 番目の点の遅延ベクトルが作成されます（詳細は [`genembed`](@ref) を参照）。

`js` は時系列入力 `s` に対して無視されます（この場合、`js` のすべてのエントリは `1` でなければならないため）し、さらに `js` はすべての `τ` に対してデフォルトで `(1, ..., 1)` になります。

**`n` を選択する際は非常に注意してください。内部で `@inbounds` が使用されています。** [`τrange`](@ref) を使用してください！
