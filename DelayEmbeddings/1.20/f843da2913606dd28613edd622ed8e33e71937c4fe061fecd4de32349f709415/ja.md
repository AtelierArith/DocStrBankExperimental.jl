```
delay_ifnn(s::Vector, τ::Int, ds = 1:10; kwargs...) → `FNNs`
```

時系列 `s` に対して、均一な時間遅延 `τ` と埋め込み次元 `ds` のための `FNNs`-統計量を計算して返します [^Hegger1999]。この表記法では `γ ∈ γs = d-1` であり、`d` は埋め込み次元です。この割合は、適切な遅延を伴う最適な埋め込み次元に達すると 0 に近づきます。

## キーワード

*`r = 2`: 再構成空間における軌道の最大許容拡散を決定する必須の閾値。 *`metric = Euclidean`: 距離計算に使用されるノルム。 *`w = 1` = [ザイラーウィンドウ](@ref)。

参照: [`optimal_traditional_de`](@ref)。
