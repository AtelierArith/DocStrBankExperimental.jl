```
delay_f1nn(s::AbstractVector, τ::Int, ds = 2:6, metric = Euclidean())
```

`s`から作成されたデータセットに対して、`embed(s, d, τ) for d ∈ ds`の「偽の最初の最近接隣人」（FFNN）の比率を計算します。

## 説明

`embed(s, d, τ)`によって作成されたデータセットにおいて、「偽の最初の最近接隣人」（FFNN）は、次元`d`で最も近い点のペアであり、次元`d+1`では最も近い隣人でなくなる点のペアです。

返される値は、各`d ∈ ds`に対するFFNNの数とデータセット内の点の数の比率を持つベクトルです。この比率がゼロに近づく点で最適な`d`の値が見つかります。
