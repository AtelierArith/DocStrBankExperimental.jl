```
pecora(s, τs, js; kwargs...) → ⟨ε★⟩, ⟨Γ⟩
```

与えられた入力 `s`（時系列または `Dataset`）と `(τs, js)` で定義された入力一般化埋め込みに従って、Pecora et al.[^Pecoral2007]（アトラクタ再構成への統一アプローチ）に従って、（平均）連続性統計量 `⟨ε★⟩` とアンダーサンプリング統計量 `⟨Γ⟩` を計算します。連続性統計量は、既存の埋め込みのコンポーネントと追加の時系列との間の機能的独立性を表します。返される結果は、サイズ `T`x`J` の *行列* です。

## キーワード引数

  * `delays = 0:50`: 可能な時間遅延値 `delays`（サンプリング時間単位）。`delays` の各 `τ` に対して、連続性統計量 `⟨ε★⟩` が計算されます。`undersampling = true`（後述）であれば、考慮されたすべての遅延値に対してアンダーサンプリング統計量 `⟨Γ⟩` も返されます。
  * `J = 1:dimension(s)`: `J` のすべての時系列インデックスについて計算します。入力 `s` が時系列である場合、これは常に 1 です。
  * `samplesize::Real = 0.1`: 平面空間のすべての点（=`length(s)`) のうち、考慮される割合（基準点 v）を決定し、ε★ を平均化して `⟨ε★⟩, ⟨Γ⟩` を生成します。
  * `K::Int = 13`: δ-ボール内の最近傍の数（アルゴリズムの説明を参照）。有効な統計を保証するために、少なくとも 8 でなければなりません。`⟨ε★⟩` は、すべての `k ∈ K` に対して最小の結果を取ることで計算されます。
  * `metric = Chebyshev()`: 入力埋め込み内の最近傍を見つけるためのメトリック（ℝᵈ 空間、`d = length(τs)`）。
  * `w = 1`: タイムウィンドウ（インデックス `w` の近くの点の時間的隣接者で、真の隣接者から除外される）。`w=0` は、点自体のみを除外し、時間的隣接者は除外しないことを意味します。
  * `undersampling = false` : アンダーサンプリング統計を計算するかどうか（計算しない場合、`⟨Γ⟩` にはゼロが返されます）。`⟨Γ⟩` の計算は `⟨ε★⟩` よりも何千倍も遅くなります。
  * `db::Int = 100`: アンダーサンプリング統計のために各時系列のヒストグラムを計算する際に使用されるビンの数。
  * `α::Real = 0.05`: 連続性統計を取得するための有意水準
  * `p::Real = 0.5`: 連続性統計の計算に使用される二項分布の p パラメータ。
