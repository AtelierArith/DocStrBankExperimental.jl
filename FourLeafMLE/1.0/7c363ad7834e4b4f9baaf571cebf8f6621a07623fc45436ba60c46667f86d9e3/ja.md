```
compute_probability_vector(θ,τ)
```

# 引数

  * `θ' : ベクトル θ = [θ₁,...,θ₅] ハダマール分岐パラメータ。
  * `τ' : {1,2,3,4} の数で、木のトポロジーに対応します。ここで、1,2,3 はそれぞれ四重奏トポロジー 13|23, 12|34, 14|23 に対応し、4 は星型木のトポロジー（内部分岐長がゼロの四葉の木）に対応します。

# 説明

トポロジー τ と分岐長 θ に対応する 4 葉の木の CFN 確率モデルを計算します。センプルとスティールの系統学に関する本の公式を使用します。p201 の Cor 8.6.6 を参照してください。

# 例

`compute*probability*vector([.2,.3,.02,.4,.7],1)'

数式を象徴的に表示するには、次のように実行します。

`@var θ[1:5]; compute*probability*vector(θ,1)'
