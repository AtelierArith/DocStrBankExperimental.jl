```
li2(z::Complex)
```

複素数 $z$ の複素二重対数 $\operatorname{Li}_2(z)$ を返します。$z$ の型は `Complex` です。

実数引数 $z\in\mathbb{R}$ のみを考慮し、二重対数の実部 $\Re[\operatorname{Li}_2(z)]$ のみを興味がある場合は、より高速な代替手段となる可能性のある関数 [`reli2`](@ref) を参照してください。

他にも [`reli2`](@ref) を参照してください。

著者: アレクサンダー・フォイヒト

ライセンス: MIT

# 例

```jldoctest; setup = :(using PolyLog)
julia> li2(1.0 + 1.0im)
0.6168502750680851 + 1.4603621167531196im

julia> li2(BigFloat(1) + 1im)
0.6168502750680849136771556874922594459571062129525494141508343360137528014012052 + 1.460362116753119547679775739491787597608795299373993707847946932920340157070418im
```
