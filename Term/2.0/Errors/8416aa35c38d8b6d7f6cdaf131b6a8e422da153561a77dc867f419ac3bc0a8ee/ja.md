```
install_term_stacktrace(; reverse_backtrace::Bool = true, max_n_frames::Int = 30)
```

デフォルトのJuliaスタックトレースエラーのスタックトレース印刷をTermのものに置き換えます。

Termは`StackTrace`を解析し、印刷する前に追加情報とスタイルを加えます。印刷された出力は2つの部分から構成されています：     - "フレーム"のリスト：エラーが発生した場所を示すネストされたコードポイント、"エラースタック"     - メッセージ：一般的にはJuliaによって提供される標準情報メッセージですが、追加のフォーマットオプションがあります。

フレームが表示される順序を逆にするオプション（Juliaのデフォルトの順序と比較して）、トレース内に多くのフレームがある場合に余分なフレームを隠すオプション（例：スタックオーバーフローエラー）、およびBaseおよび標準ライブラリのエラー情報を隠すオプション（つまり、フレームがそれらに属するモジュールにある場合）をいくつか提供しています。
