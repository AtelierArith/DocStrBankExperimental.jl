```
MvNormalMeanScalePrecision{T <: Real, M <: AbstractVector{T}} <: AbstractMvNormal
```

平均 `μ` とスケールパラメータ `γ` を持つ多変量正規分布で、単位精度行列をスケーリングします。

# 型パラメータ

  * `T`: 平均ベクトルとスケールパラメータの要素型
  * `M`: 平均ベクトルの型で、`AbstractVector{T}` のサブタイプでなければなりません

# フィールド

  * `μ::M`: 多変量正規分布の平均ベクトル
  * `γ::T`: 単位精度行列をスケーリングするスケールパラメータ

# 注意事項

この分布の精度行列は `γ * I` であり、ここで `I` は単位行列です。共分散行列は精度行列の逆であり、すなわち `(1/γ) * I` です。
