```
MvNormalWishart{T, M <: AbstractArray{T}, V <: AbstractMatrix{T}, K <: Real, N <: Real} <: ContinuousMatrixDistribution
```

多変量正規-ウィシャルト分布であり、ここで `T` は配列 `M` と行列 `V` の要素型であり、`K` と `N` は実数です。この分布は、平均 `μ` を持つ多変量正規乱数変数と、スケール行列 `Ψ`、自由度 `ν`、およびスケーリングパラメータとしてのスカラー `κ` を持つウィシャート分布のランダム行列の同時分布です。

# フィールド

  * `μ::M`: 多変量正規分布の平均ベクトル。
  * `Ψ::V`: ウィシャート分布のスケール行列。
  * `κ::K`: ウィシャート分布のスケーリングパラメータ。
  * `ν::N`: ウィシャート分布の自由度。
