```
@cg [1.0] A \ b
```

共役勾配法を使用して最小二乗問題 $\min\|Ax - b\|^2$ を解きます。

`@cg` と `rdiv` の間で `iterpart` を指定できます。

## 例

```julia-repl
julia> @cg [1 2; 3 4] \ [1, 2]
2-element Array{Float64,1}:
5.551115123125783e-17
0.5000000000000012

julia> @cg 0.5 [1 2; 3 4] \ [1, 2]
2-element Array{Float64,1}:
0.2343820224719101
0.33483146067415726
```
