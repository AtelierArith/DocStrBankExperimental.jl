```
logqforeach(fn, itrs...)
```

`Base.foreach`を模倣しますが、関数の評価をすべての利用可能なスレッドに対して開始し、非均一なタスクの所要時間に適したキュー方式のスケジューリング戦略を使用します。

# 例

```julia
julia> pool = logqforeach(x -> println((x,Threads.threadid())), 1:8);
(2, 4)
(3, 3)
(4, 2)
(5, 4)
(6, 3)
(7, 2)
(8, 4)
(1, 1)

julia> plot(pool)
```

実行順序は保証されておらず、主スレッドが使用されることに注意してください。 ログが記録されたプールが返され、ログ関数と`plot`で分析できます。
