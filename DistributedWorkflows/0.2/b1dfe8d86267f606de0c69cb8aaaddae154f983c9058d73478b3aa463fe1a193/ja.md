```
implementation(port_name::String, path::String)
```

関数シグネチャの明確さと可読性のための便利なキー-バリュー ペアラッパー。

# 引数

  * `port_name::String`: パス文字列を含むポートの名前。
  * `path::String`: Julia ソースファイルへのパス。

関連情報としては [`Workflow_PetriNet`](@ref)、[`generate_workflow`](@ref)、[`compile_workflow`](@ref)、[`port_info`](@ref) を参照してください。
