```
as(Real, left, right)
```

与えられた（開）区間に単一の実数を変換する変換を返します。

`left < right` が必要ですが、`-∞` または `∞` である場合もあり、その場合は適切な変換が選択されます。 [`∞`](@ref)を参照してください。

いくつかの一般的な変換は定数として事前定義されており、[`asℝ`](@ref)、[`asℝ₋`](@ref)、[`asℝ₊`](@ref)、[`as𝕀`](@ref)を参照してください。

!!! note
    有限の引数は共通の型に昇格され、昇格に影響を与えます。例えば、`transform(as(0, ∞), 0f0) isa Float32` ですが、`transform(as(0.0, ∞), 0f0) isa Float64` です。
