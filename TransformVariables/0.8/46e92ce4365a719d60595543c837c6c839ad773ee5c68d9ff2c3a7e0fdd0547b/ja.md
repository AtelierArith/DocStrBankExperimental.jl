```
as(T, args...)
```

`T`の画像を持つ変換を構築するための省略形です。`args`は動作を決定または変更し、詳細は`T`に依存します。

すべての変換に`as`メソッドがあるわけではなく、直接コンストラクタを持つものもあります。リストについては`methods(as)`を参照してください。

# 例

```julia
as(Real, -∞, 1)          # 実数を(-∞, 1)に変換
as(Array, 10, 2)         # 20の実数を10x2の行列に再形成
as(Array, as𝕀, 10)       # 10の実数を(0, 1)に変換
as((a = asℝ₊, b = as𝕀)) # 2つの実数をNamedTupleに変換、a > 0, 0 < b < 1
as(SArray{1,2,3}, as𝕀)  # 正の数の静的配列に変換
```
