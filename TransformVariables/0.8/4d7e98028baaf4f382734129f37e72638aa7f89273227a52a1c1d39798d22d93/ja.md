```
CorrCholeskyFactor(n)
```

!!! note
    [`corr_cholesky_factor`](@ref) を使用する方がスタイルとしては良いです。これは非推奨になります。


サイズ `n` の相関行列のコレスキー因子。

$$
n×(n-1)/2
$$

の実数を $n×n$ の上三角行列 `U` に変換します。ここで `U'*U` は相関行列（正定値で、単位対角）です。

# 注意事項

もし

  * `z` が `n` の IID 標準正規変数のベクトルであり、
  * `σ` が標準偏差の `n` 要素のベクトルであり、
  * `U` が `CorrCholeskyFactor(n)` から得られる場合、

ならば `Diagonal(σ) * U' * z` は与えられた分散と相関行列 `U' * U` を持つ多変量正規分布になります。
