**時間モデル:**

```
  プロジェクト時間 τ:               0
  実時間 t:     t_start      t_0
  製品時間 s:  0            s_0
                   |------------|-------------------|-----------
                                |-------------------|
                                         dur
                   |--------------------------------|
                              product.dur
```
