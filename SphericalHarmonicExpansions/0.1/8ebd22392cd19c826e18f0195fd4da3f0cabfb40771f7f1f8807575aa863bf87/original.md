fastfunc(polynomial::Polynomial)

Generates a function for fast numerical evaluation, with a single input argument.

If you only plan a few evaluations of polynomial use the standard method (see e.g. TypedPolynomials for details). Only for a large number of evaluations the one can break even with the function generated by this method due to the just  in time compilation overhead.

# Examples

```
julia> using SphericalHarmonics

julia> @polyvar x y z
(x, y, z)

julia> p = 15.0*x*y^2+7.5*x*z^13
7.5xz¹³ + 15.0xy²

julia> ps = fastfunc(p)

julia> r = [1.0,2.0,3.0]

julia> ps(r)
1.19574825e7
```
