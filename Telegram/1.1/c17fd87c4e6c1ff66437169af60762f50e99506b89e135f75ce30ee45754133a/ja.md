```
run_bot(f::Function, [tg::TelegramClient]; timeout = 10, brute_force_alive = false, offset = -1)
```

テレグラムボットを実行し、関数 `f` を繰り返し実行します。

# 引数

  * `f`: ボットと一緒に実行する関数。入力として [Message](https://core.telegram.org/bots/api#message) を JSON3 オブジェクトの形式で受け取ります。
  * `tg`: テレグラムクライアント。設定されていない場合は、グローバルなテレグラムクライアントが使用されます。
  * `timeout`: メッセージのタイムアウト制限。ロングポールメッセージを中断すべき時期を定義します。
  * `brute_force_alive`: すべての対策にもかかわらず、メッセージが「ハング」することがあります。この引数を使用して、定期的にテレグラムサーバーを起こすことができます。このオプションが `true` に設定されている場合は機能しますが、追加の呼び出しを行うため推奨されません。この機能は実験的なものと考え、自己責任で使用してください。
  * `offset`: テレグラムメッセージのオフセット。メッセージを処理したがボットが中断され、メッセージの状態が失われた場合に便利です。
