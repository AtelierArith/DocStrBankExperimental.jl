```
×(M, N)
cross(M, N)
cross(M1, M2, M3,...)
```

[`ProductManifold`](@ref) を返します。二つの `AbstractManifold` である `M` と `N` の場合、片方が [`ProductManifold`](@ref) である場合、もう片方は (もし `N` がプロダクトの場合) 先頭に追加されるか、(もし `M` がプロダクトの場合) 末尾に追加されます。両方がプロダクト多様体である場合、それらは順序を保持しながら一つのプロダクト多様体に結合されます。

これらのうち複数がプロダクト多様体である場合、上記と同じアプローチで構築されます。
