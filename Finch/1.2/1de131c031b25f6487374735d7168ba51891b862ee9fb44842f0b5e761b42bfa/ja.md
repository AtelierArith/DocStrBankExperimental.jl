```
fsprand([rng],[type], M..., p, [rfn])
```

COO形式のサイズ`m`のランダムスパーステンソルを作成します。2つのケースがあります：     - `p`が浮動小数点数の場合、任意の要素が非ゼロである確率は独立して`p`によって与えられます（したがって、非ゼロの期待密度も`p`です）。     - `p`が整数の場合、正確に`p`の非ゼロがテンソル全体に均等にランダムに分布します（したがって、非ゼロの密度は正確に`p / prod(M)`です）。非ゼロの値は`rfn`で指定された分布からサンプリングされ、型は`type`です。`rfn`が指定されていない場合は一様分布が使用されます。オプションの`rng`引数は乱数生成器を指定します。

参照： (`sprand`)(https://docs.julialang.org/en/v1/stdlib/SparseArrays/#SparseArrays.sprand)

# 例

```julia
julia> fsprand(Bool, 3, 3, 0.5)
SparseCOO (false) [1:3,1:3]
├─├─[1, 1]: true
├─├─[3, 1]: true
├─├─[2, 2]: true
├─├─[3, 2]: true
├─├─[3, 3]: true

julia> fsprand(Float64, 2, 2, 2, 0.5)
SparseCOO (0.0) [1:2,1:2,1:2]
├─├─├─[2, 2, 1]: 0.6478553157718558
├─├─├─[1, 1, 2]: 0.996665291437684
├─├─├─[2, 1, 2]: 0.7491940599574348
```
