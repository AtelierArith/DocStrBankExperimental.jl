```
SparseCOOLevel{[N], [TI=Tuple{Int...}], [Ptr, Tbl]}(lvl, [dims])
```

スパースレベルのサブファイバーは、完全に [`fill_value`](@ref) であるスライスを表す必要はありません。代わりに、潜在的に非フィルスライスのみが `lvl` のサブファイバーとして保存されます。スパース COO レベルはサブファイバー内の `N` インデックスに対応するため、サブレベル内のファイバーはスライス `A[:, ..., :, i_1, ..., i_n]` です。`N` 個のリストのセット（各インデックスごとに1つ）が、どのスライスが保存されているかを記録するために使用されます。座標（`N` インデックスのセット）は、列優先順にソートされます。オプションで、`dims` は最後の次元のサイズです。

`TI` は最後の `N` テンソルインデックスの型であり、`Tp` はレベル内の位置に使用される型です。

型 `Tbl` は、各エントリ k がサブタイプ `AbstractVector{TI[k]}` である NTuple 型です。

型 `Ptr` はポインタ配列の型です。

```jldoctest
julia> tensor_tree(Tensor(Dense(SparseCOO{1}(Element(0.0))), [10 0 20; 30 0 0; 0 0 40]))
3×3-Tensor
└─ Dense [:,1:3]
   ├─ [:, 1]: SparseCOO{1} (0.0) [1:3]
   │  ├─ [1]: 10.0
   │  └─ [2]: 30.0
   ├─ [:, 2]: SparseCOO{1} (0.0) [1:3]
   └─ [:, 3]: SparseCOO{1} (0.0) [1:3]
      ├─ [1]: 20.0
      └─ [3]: 40.0

julia> tensor_tree(Tensor(SparseCOO{2}(Element(0.0)), [10 0 20; 30 0 0; 0 0 40]))
3×3-Tensor
└─ SparseCOO{2} (0.0) [:,1:3]
   ├─ [1, 1]: 10.0
   ├─ [2, 1]: 30.0
   ├─ [1, 3]: 20.0
   └─ [3, 3]: 40.0
```
