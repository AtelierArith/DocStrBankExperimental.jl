```
ASM(u, λ, Δx, Δy, z; expand=true)
```

角度スペクトル法（ASM）によって回折場を返します。エバネッセント波は排除されず、$\exp(-2{\pi}wz)$として減衰します。減衰がない場合、総エネルギー$\iint|u|\mathrm{d}x\mathrm{d}y$は保存されます。

# 引数

  * `u`: 入力場。
  * `λ`: 波長。
  * `Δx`: x軸のサンプリング間隔。
  * `Δy`: y軸のサンプリング間隔。
  * `z`: 回折距離。
  * `expand=true`: true（デフォルト）の場合、エイリアシング抑制のために4倍の拡張とゼロパディングを行います。

!!! note
    x軸は水平方向で、y軸は垂直方向です。
