```
convert_eltype(T, A) -> B
```

は、オブジェクト/型 `A` の要素型を型 `T` に変換します。返されるオブジェクト/型は、要素型を除いて `A` に似ています。たとえば、`A` が範囲である場合、`B` も範囲です。もし `T` が `A` の要素型である場合、`A` が返されることがあります。

オブジェクトが遅延的に変換を行うように [`as_eltype(T, A)`](@ref as_eltype) を使用することを検討してください。

指定された型のオブジェクト `A` に対して `convert_eltype` を拡張するのを簡素化するために、デフォルトの動作は次のようになります。

```
convert_eltype(T, A) = as(convert_eltype(T, typeof(A)), A)
```

したがって、オブジェクトの型に対して `convert_eltype` を拡張するだけで十分な場合があります。
