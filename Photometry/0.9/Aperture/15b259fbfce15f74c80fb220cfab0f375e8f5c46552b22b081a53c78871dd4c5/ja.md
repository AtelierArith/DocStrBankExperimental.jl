```
EllipticalAperture(x, y, a, b, θ=0)
EllipticalAperture(position, a, b, θ=0)
```

半長軸 `a`、半短軸 `b`、および位置角 `θ` を持つ楕円形アパーチャ。`a` と `b` は ≥ 0 でなければならず、`θ` は標準の x 軸に対して反時計回りに度で測定されます。
