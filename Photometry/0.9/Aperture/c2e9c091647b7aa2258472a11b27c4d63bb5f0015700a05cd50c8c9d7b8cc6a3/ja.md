```
EllipticalAnnulus(x, y, a_in, a_out, b_out, θ=0)
EllipticalAnnulus(position, a_in, a_out, b_out, θ=0)
```

内半主軸 `a_in`、外半主軸 `a_out`、外半副軸 `b_out`、および位置角 `θ` を持つ楕円環。`a_out` ≥ `a_in` ≥ 0 であり、`b_out` は ≥ 0 でなければならず、`θ` は標準の x 軸に対して反時計回りに度で測定されます。

`b_in` は自動的に `(a_in / a_out) * b_out` から計算されます。これは型の不安定性を引き起こす可能性があります。
