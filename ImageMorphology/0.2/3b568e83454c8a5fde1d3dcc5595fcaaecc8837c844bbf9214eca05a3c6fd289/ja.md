```
label = label_components(tf, [connectivity])
label = label_components(tf, [region])
```

バイナリ配列 `tf` の接続成分を見つけます。`connectivity` は次の2つの形式を取ることができます：

  * `tf` と同じ次元のブール配列で、各次元に沿ってサイズが1または3のものです。配列の各エントリは、接続性分析に使用される隣接点を決定します。たとえば、`connectivity = trues(3,3)` は8接続性を使用し、現在のピクセルに接触するすべてのピクセル（コーナーも含む）をテストします。
  * 接続性を決定するために使用される次元を示すリストを提供できます。たとえば、`region = [1,3]` は接続性のために次元2に沿った隣接点をテストしません。これは、最近接の隣接点、すなわち2次元での4接続性および3次元での6接続性に対応します。

デフォルトは `region = 1:ndims(A)` です。

出力 `label` は整数配列で、0は背景ピクセルに使用され、各接続領域には異なる整数インデックスが割り当てられます。
