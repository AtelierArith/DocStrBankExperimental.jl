```
isboundary(img::AbstractArray; background = 0, dims = coords_spatial(A), kwargs...)
```

各オブジェクトのちょうど内側にある境界を見つけます。`background`は、境界としてマークされない背景ピクセルのスカラー値です。キーワード引数は、境界を発見する次元を示す`dims`を含む`extremefilt!`に渡されます。

そのインプレースバージョン[`isboundary!`](@ref)や、厚い境界を見つける代替バージョン[`isboundary_thick`](@ref ImageMorphology.isboundary_thick)も参照してください。
