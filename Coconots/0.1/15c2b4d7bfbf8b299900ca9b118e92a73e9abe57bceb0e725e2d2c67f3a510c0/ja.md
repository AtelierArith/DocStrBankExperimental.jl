```
cocoSim(type, order, parameter, n, covariates=nothing, link="log", n_burn_in=200, x=zeros(Int(n + n_burn_in)))
```

整数自己回帰モデルに基づいてカウント時系列をシミュレートします。この関数は、指定されたリンク関数を介してオプションの共変量情報を持つ一次および二次モデルの両方に対応しています。

# 引数

  * `type`: モデルタイプを示す文字列（例: `"GP"` は一般化ポアソン）。このパラメータは、分散パラメータ `eta` の設定方法に影響します。
  * `order`: 自己回帰モデルの次数を指定する整数（1または2）。
  * `parameter`: モデルパラメータのベクトル。次数1のモデルでは、最初の要素が `alpha` であり（`type` が `"GP"` の場合）、2番目の要素が `eta` です。次数2のモデルでは、最初の3つの要素が `alpha1`、`alpha2`、`alpha3` であり（`type` が `"GP"` の場合）、4番目の要素が `eta` です。
  * `n`: バーンイン期間後にシミュレートする観測値の数。
  * `covariates`（オプション）: 共変量の値の行列。提供された場合、これらは指定されたリンク関数と共に使用されて、レートパラメータ `lambda` を計算します。
  * `link`（オプション）: 使用するリンク関数を指定する文字列（デフォルトは `"log"`）。
  * `n_burn_in`（オプション）: 廃棄する初期バーンイン観測値の数（デフォルトは200）。
  * `x`（オプション）: 長さ `n + n_burn_in` の時系列の初期ベクトル。デフォルトはゼロで初期化された整数配列です。

# 戻り値

バーンイン期間後の時系列に対応する長さ `n` のシミュレートされたカウントデータのベクトル。

# 詳細

  * レートパラメータ `lambda` は、提供された場合は共変量に適用されるリンク関数（`get_link_function(link)` を介して取得）を使用して計算され、そうでない場合は `parameter` の最後の要素に設定されます。
  * モデルの次数に応じて、適切な自己回帰パラメータが `parameter` から抽出されます。
  * 各時間ステップ（t = 3 から開始）において、シミュレートされたカウントは、カスタム自己回帰成分（`draw_random_g` を介して）からの引き出しと一般化ポアソン成分（`draw_random_generalized_poisson_variable` を介して）からの引き出しの合計です。
