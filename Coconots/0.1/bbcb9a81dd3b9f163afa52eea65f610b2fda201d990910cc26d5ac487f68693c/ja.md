```
cocoPit(cocoReg_fit, n_bins=21)
```

適合したカウント回帰モデルの確率積分変換（PIT）ヒストグラムを計算します。

# 引数

  * `cocoReg_fit`: 適合したモデルの結果を含む辞書。期待されるキーには `"data"`、`"parameter"`、`"order"`、`"type"`、およびオプションで `"covariates"` と `"max_loop"` が含まれます。
  * `n_bins`（オプション）: PIT ヒストグラムに使用するビンの数を指定する整数（デフォルトは 21）。

# 戻り値

以下を含む辞書:

  * `"Pit_values"`: 連続する PIT 値の差のベクトルで、推定されたヒストグラム確率を表します。
  * `"bins"`: ビンの端値のベクトル（初期のゼロを除く）。
