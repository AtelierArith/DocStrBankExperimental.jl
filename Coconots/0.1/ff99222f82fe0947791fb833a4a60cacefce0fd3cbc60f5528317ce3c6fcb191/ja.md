```
cocoForwardSim(n, x_prev, type, order, parameter, covariates=nothing, link="log", add_help=order * -1 + 2, x=zeros(Int(n + length(x_prev) + add_help)))
```

前のカウント、モデルパラメータ、およびオプションの共変量に基づいて、`n` ステップの前方カウントデータをシミュレートします。

# 引数

  * `n`: シミュレートする将来のステップ数。
  * `x_prev`: シミュレーションの初期化に使用される以前のカウント値のベクトル。
  * `type`: モデルタイプを示す文字列（例: `"GP"` は一般化ポアソン）。
  * `order`: 自己回帰モデルの次数 (1 または 2)。
  * `parameter`: モデルパラメータのベクトル。
  * `covariates` (オプション): シミュレーションに使用される共変量データの行列。
  * `link` (オプション): 使用するリンク関数を指定する文字列（デフォルトは `"log"`）。
  * `add_help` (オプション): シミュレーションのための補助パラメータ（デフォルトは `order * -1 + 2` として計算される）。
  * `x` (オプション): 長さ `n + length(x_prev) + add_help` の整数カウントの事前割り当てベクトル。

# 戻り値

`n` ステップの前方シミュレーションカウントのベクトル。
