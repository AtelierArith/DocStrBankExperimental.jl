```
cocoBoot(cocoReg_fit, lags=[1:1:21;], n_bootstrap=400, alpha=0.05, n_burn_in=200, store_matrix = Array{Float64}(undef, length(lags), 2))
```

適合したカウント回帰モデルの部分自己相関関数（PACF）をブートストラップします。PACF値のブートストラップサンプルを生成し、信頼区間を形成するために分位点を計算し、観測データから計算されたPACFとこれらの区間を比較します。

# 引数

  * `cocoReg_fit`: 適合したモデルの結果を含む辞書。
  * `lags`（オプション）: PACFが計算されるラグ値のベクトル（デフォルトは`[1:1:21;]`）。
  * `n_bootstrap`（オプション）: ブートストラップの反復回数（デフォルトは400）。
  * `alpha`（オプション）: 信頼区間の有意水準（デフォルトは0.05）。
  * `n_burn_in`（オプション）: データをシミュレーションする際に破棄するバンインサンプルの数（デフォルトは200）。
  * `store_matrix`（オプション）: 各ラグの下位および上位分位点を格納するための事前に割り当てられた行列。

# 戻り値

以下のキーを持つ辞書:

  * `"upper"`: 各ラグのブートストラップされたPACFの下位分位点値（最初の列）。
  * `"lower"`: 各ラグのブートストラップされたPACFの上位分位点値（2列目）。
  * `"in_interval"`: 観測されたPACF値がブートストラップ信頼区間内にあるかどうかを示すブールベクトル。
  * `"pacf_data"`: 観測データから計算されたPACF。
  * `"pacfs"`: ブートストラップされたPACF値の行列。
  * `"lags"`: ラグ値のベクトル。
