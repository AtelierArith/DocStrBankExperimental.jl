```
cocoPredictOneStep(cocoReg_fit, x=0:10, covariates=nothing, safe_array = Array{Float64}(undef, length(x)))
```

フィッティングされたカウント回帰モデルから1ステップ先のカウント確率を予測します。

# 引数

  * `cocoReg_fit`: フィッティングされたモデル情報を含む辞書（例：データ、パラメータ、順序、タイプ、リンクなど）。
  * `x`（オプション）: 予測が計算される可能性のある将来のカウント値の範囲またはベクトル（デフォルトは `0:10`）。
  * `covariates`（オプション）: 予測に使用されるオプションの共変量情報。
  * `safe_array`（オプション）: 計算された確率を格納するための事前に割り当てられた配列。

# 戻り値

以下を含む辞書:

  * `"probabilities"`: `x` の各カウントに対する予測確率のベクトル。
  * `"x"`: 入力されたカウントの範囲。
  * `"prediction_mode"`: 最大確率に基づくモード（最も可能性の高いカウント）。
  * `"prediction_median"`: 累積確率から計算された中央値の予測。
