```
comoving_radial_dist([u::Unitlike,] c::AbstractCosmology, [z₁,] z₂)
```

コモビング半径距離 ($D_C$) を Mpc 単位で、赤方偏移 `z₂` において、赤方偏移 `z₁` の観測者から見たものです。赤方偏移 `z₁` は省略された場合、デフォルトで 0 になります。提供された場合、互換性のある単位 `u` に変換されます。

これは $D_C = D_{H0} Z$ として計算され、ここで $D_{H0}$ は現在の時代におけるハッブル距離であり、$Z = \int_{z_1}^{z_2} \frac{dz}{E(z)}$ です。
