```
function esdd(;kwargs...)
```

AWSからデータキューブをデータセットとして開きます。これはどのシステムでも動作しますが、接続速度に応じて遅延が発生する可能性があります。`bucket`と`store`を指定するか、解像度、チャンク化、キューブ領域を選択することができます。

### キーワード引数

  * `bucket=nothing` OBSバケットを指定します。例えば "obs-esdc-v2.0.0"
  * `store=""` キューブのルートパスを指定します。例えば "esdc-8d-0.25deg-184x90x90-2.0.0.zarr"
  * `res="low"` データキューブの解像度を選択します（v2の場合は`"low"`または`"high"`、v3の場合は`"low"`または`"tiny"`）
  * `chunks="ts"` チャンク化を選択します（時間系列アクセスの場合は`"ts"`、空間分析の場合は`"map"`）
  * `region="global"` データキューブを選択します（`"global"`または`"Colombia"`）、esdc v2のみに対応
  * `version=3`
