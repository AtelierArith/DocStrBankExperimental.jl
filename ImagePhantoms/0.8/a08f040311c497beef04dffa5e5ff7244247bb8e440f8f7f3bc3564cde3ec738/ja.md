```
params = disk_phantom_params( ; ...)
```

頭の大きさのディスクとその中にある多くのディスクのための `ndisk` エリプスファントムパラメータの6タプルを生成します。ディスクが重ならないように、またパッチベースのモデルフィッティングを簡素化するために、ディスク間に最小の間隔 `minsep` を持つように設計されています。

# オプション

  * `fov::Real = 240` 画像の視野 [mm]
  * `rhead::Function = () -> 100` "頭" の背景半径 [mm]
  * `muhead::Function = () -> 1000` 背景頭ディスクの "μ" (強度) 値
  * `rdisk::Function = () -> 10 + 10 * rand()` ランダムディスク半径 [10,20]
  * `mudisk::Function = () -> 100 + 200 * rand()` ディスクの "μ" 値 [100,300]
  * `ndisk::Function = () -> 10` ランダムディスクの数
  * `minsep::Real = 8` 最小ディスク間隔 [mm]
  * `maxtry::Int = 500` この値に達した場合、ディスクの追加を諦める
  * `warn::Bool = false` maxtry に達した場合に警告を出すか？
  * `seed::Int = 0` ゼロ以外の場合、このシードを使用する

関数のオプションは、他の分布のために rand() で置き換えることができます。
