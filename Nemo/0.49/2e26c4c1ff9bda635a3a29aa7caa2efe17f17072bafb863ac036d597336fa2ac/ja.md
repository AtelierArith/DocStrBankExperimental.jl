```
next_signed_minimal(a::QQFieldElem)
```

符号付き有理数 $a$ が標準形であると仮定して、`next_minimal` によって生成される最小高さのシーケンスの次の要素を返しますが、負の数が交互に挿入されます。このシーケンスは $0, 1, -1, 1/2, -1/2, 2, -2, 1/3, -1/3, \ldots$ で始まります。ゼロから始めることで、すべての有理数を一度だけ、最小高さの順序で生成します。

# 例

```jldoctest
julia> next_signed_minimal(-ZZ(21)//31)
31//21
```
