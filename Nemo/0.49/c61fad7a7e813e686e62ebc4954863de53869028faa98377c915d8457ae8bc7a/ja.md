```
lll_with_transform(x::ZZMatrix, ctx::LLLContext = LLLContext(0.99, 0.51))
```

タプル $(L, T)$ を返します。ここで、$L$ の行は $x$ の行によって生成される $\mathbb{Z}$-格子の LLL-縮小基底を形成し、$T$ は $L = Tx$ となる変換行列です。$L$ には追加のゼロ行が含まれる場合があります。使用されるデフォルトパラメータについては、オプションのコンテキストオブジェクトを提供することで上書きできる [`lll`](@ref) を参照してください。

入力としてグラム行列を受け取る関数については、[`lll_gram_with_transform`](@ref) を参照してください。
