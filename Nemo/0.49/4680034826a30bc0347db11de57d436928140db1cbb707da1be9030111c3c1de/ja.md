```
lll_gram_with_transform(x::ZZMatrix, ctx::LLLContext = LLLContext(0.99, 0.51, :gram))
```

タプル $(L, T)$ を返します。ここで $L$ は、グラム行列 $x$ によって与えられる格子の LLL 簡約基のグラム行列であり、$T$ は $L = T^\top x T$ を満たす変換行列です。行列 $x$ は対称で非特異でなければなりません。

使用されるデフォルトパラメータについては、オプションのコンテキストオブジェクトを提供することで上書きできる [`lll_gram`](@ref) を参照してください。
