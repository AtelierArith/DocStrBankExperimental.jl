```
cyclotomic_field(n::Int, s::VarName = "z_$n", t = "_\$"; cached::Bool = true)
```

タプル $R, x$ を返します。ここで、$R$ は $n$-次サイクロトミック体の親オブジェクト、$\mathbb{Q}(\zeta_n)$ の生成元 $x$ です。提供された文字列 `s` は、数体の生成元がどのように印刷されるべきかを指定します。もし提供されていれば、文字列 `t` は、数体が構築される多項式環の生成元がどのように印刷されるべきかを指定します。提供されていない場合、変数を表すためにデフォルトのドル記号が使用されます。
