```
next_calkin_wilf(a::QQFieldElem)
```

$ a $ の次の数を、Calkin-Wilf 木の幅優先探索で返します。ゼロから始めると、これはすべての非負有理数を一度だけ生成し、最初のいくつかのエントリは $ 0, 1, 1/2, 2, 1/3, 3/2, 2/3, 3, 1/4, 4/3, 3/5, 5/2, 2/5, \ldots $ です。初期のエントリの見た目にもかかわらず、Calkin-Wilf 列挙は高さの順に有理数を生成するわけではありません：いくつかの小さな分数は列の後半に現れます。この順序は、最小高さの順序よりも生成が速いという利点があります。

# 例

```jldoctest
julia> next_calkin_wilf(ZZ(321)//113)
113//244
```
