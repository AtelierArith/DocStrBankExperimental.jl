```
is_unimodular(A::ZZMatrix)
is_unimodular(A::ZZMatrix; algorithm=:auto)
```

正方行列 `A` がユニモジュラーであるかどうかを判断します。すなわち、`abs(det(A)) == 1` であるかどうかです。使用される方法は、Pauderis–Storjohann の方法または CRT を使用するものであり、選択は2つのアプローチのコスト見積もりに基づいて行われます。

キーワード引数 `algorithm` は `:CRT` または `:pauderis_storjohann` に指定することができ、対応する基礎アルゴリズムが使用されることを保証します（迅速な初期ユニモジュラーチェックの後）。行列がユニモジュラーであり、大きなエントリを含む逆行列を持つ場合（またはユニモジュラーでない場合）、`:CRT` の方が良いです。行列がユニモジュラーであり、その逆行列が大きなエントリを含まない場合、`:pauderis_storjohann` の方が漸近的に速いですが、中間表現のためにより多くのスペースを必要とします。 ```
