```
next_minimal(a::QQFieldElem)
```

与えられた $a$ に対して、すべての正の分母 $q$ を列挙し、各 $q$ に対して $1 \le p < q$ の順に分子を列挙し、$p/q$ と $q/p$ の両方を生成しますが、gcd$(p,q) \neq 1$ のものはスキップします。ゼロから始めると、この方法で非負の有理数を一度だけ生成します。最初のいくつかのエントリは $0, 1, 1/2, 2, 1/3, 3, 2/3, 3/2, 1/4, 4, 3/4, 4/3, \ldots$ です。この列挙は最小の高さの順に有理数を生成します。Calkin-Wilf 列挙よりも計算がやや遅くなるという欠点があります。もし $a < 0$ であれば、`DomainError()` を投げます。

# 例

```jldoctest
julia> next_minimal(ZZ(2)//3)
3//2
```
