```
lll_with_transform(x::ZZMatrix, ctx::LLLContext = LLLContext(0.99, 0.51))
```

タプル $(L, T)$ を返します。ここで、$L$ の行は $x$ の行によって生成される $\mathbb{Z}$-格子の LLL-縮小基底を形成し、$T$ は $L = Tx$ となる変換行列です。$L$ には追加のゼロ行が含まれる場合があります。オプションのコンテキストオブジェクトを提供することで上書きできるデフォルトパラメータについては [`lll`](@ref) を参照してください。

グラム行列を入力として受け取る関数については [`lll_gram_with_transform`](@ref) を参照してください。
