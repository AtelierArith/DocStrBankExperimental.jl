```
MPRK43II(γ; [linsolve = ..., small_constant = ...])
```

生産-消滅システムのための三次改良パタンカール-ルンゲ-クッタ法のファミリーであり、非負のルンゲ-クッタ係数を持つ三次明示的ルンゲ-クッタ法の一パラメータファミリーに基づいています。このファミリーの各メンバーは、適応型の一段階法であり、四つのステージを持ち、三次精度を持ち、無条件に正の保持、保存的であり、線形的に暗黙的です。この実装では、ステージ値も保存的です。パラメータ `γ` は `3/8 ≤ γ ≤ 3/4` を満たさなければなりません。さらなる詳細はKopeczとMeister（2018）に記載されています。

この方法は、エラーを推定するための二次近似としてパタンカール重み分母 $σ_i$ を使用して、適応的な時間ステッピングをサポートします。詳細はKopeczとMeister（2018）を参照してください。

このスキームは、保存的な生産-消滅システムのためにKopeczとMeisterによって導入されました。非保存的な生産-消滅システムに対しては、[`MPE`](@ref) に類似した直接的な拡張を使用します。

これらの改良パタンカール-ルンゲ-クッタ法は、[`PDSProblem`](@ref) または [`ConservativePDSProblem`](@ref) の特別な構造を必要とします。

オプションで、キーワード引数 `linsolve` として [LinearSolve.jl](https://github.com/SciML/LinearSolve.jl) からアルゴリズムを渡すことで使用する線形ソルバーを選択できます。また、ゼロ除算を避けるために、すべてのパタンカール重み分母に追加されるパラメータ `small_constant` を選択することもできます。データ型 `type` のデフォルト値を表示するには、`MPRK43II(gamma).small_constant_function(type)` を評価します。ここで、`type` は例えば `Float64` です。
