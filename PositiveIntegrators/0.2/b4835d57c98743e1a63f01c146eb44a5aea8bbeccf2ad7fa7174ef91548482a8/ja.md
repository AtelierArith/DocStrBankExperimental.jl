```
MPRK43I(α, β; [linsolve = ..., small_constant = ...])
```

生産-消滅システムのための三次改良パタンカール-ルンゲ-クッタ法のファミリーであり、二つのパラメータファミリーの三次明示的ルンゲ–クッタ法に基づいています。このファミリーの各メンバーは、適応型の一段階法であり、四段階で三次精度を持ち、無条件に正の保持、保存的であり、線形暗黙的です。この実装では、ステージ値も保存的です。パラメータ `α` と `β` は、ルンゲ–クッタ係数が非負となるように選択する必要があります。詳細については、Kopecz と Meister (2018) を参照してください。

これらの方法は、パタンカール重み分母 $σ_i$ を使用して適応的な時間ステッピングをサポートし、誤差を推定するための二次近似として機能します。

このスキームは、保存的な生産-消滅システムのために Kopecz と Meister によって導入されました。非保存的な生産-消滅システムの場合は、[`MPE`](@ref) に類似した直接的な拡張を使用します。

これらの改良パタンカール-ルンゲ-クッタ法は、[`PDSProblem`](@ref) または [`ConservativePDSProblem`](@ref) の特別な構造を必要とします。

オプションで、キーワード引数 `linsolve` として [LinearSolve.jl](https://github.com/SciML/LinearSolve.jl) からアルゴリズムを渡すことで使用する線形ソルバーを選択できます。また、ゼロ除算を避けるためにすべてのパタンカール重み分母に追加されるパラメータ `small_constant` を選択することもできます。値を明示的に渡すことができますが、そうでない場合は `small_constant` は使用される浮動小数点型の `floatmin` に設定されます。
