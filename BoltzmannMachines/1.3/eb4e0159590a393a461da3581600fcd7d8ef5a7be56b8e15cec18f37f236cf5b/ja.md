```
oneornone_encode(x, categories)
```

値 0.0、1.0、2.0 ... カテゴリをエンコードするデータセット `x` を期待します。`x` の変数/列を、すべての値が 0.0 と 1.0 の複数の列でエンコードするデータセットを返します。これは、ゼロがすべてゼロとしてエンコードされるという点で、ワンホットエンコーディングに似ています。

`categories` は次のように指定できます。

  * すべての変数が同じ数のカテゴリを持つ場合は整数
  * 各変数のエンコードされたカテゴリの数を含む整数ベクトル

逆変換については `oneornone_decode` も参照してください。
