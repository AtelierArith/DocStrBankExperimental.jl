```
samples(bm, nsamples; ...)
```

ボルツマンマシンモデル `bm` から `nsamples` サンプルを生成するためにギブスサンプラーを実行します。これは *条件付き分布* からサンプリングするためにも使用できます（下記の引数 `conditions` を参照してください。）

# オプションのキーワード引数:

  * `burnin`: ギブスサンプリングステップの数、デフォルトは50。
  * `conditions`: 条件付けされる変数とその値のペアを含む `Vector{Pair{Int,Float64}}`。例: `[1 => 1.0, 3 => 0.0]`
  * `samplelast`: 最後のステップでサンプリングするかどうかを示すブール値（true、デフォルト）または活性化ポテンシャルを使用するかどうか。
