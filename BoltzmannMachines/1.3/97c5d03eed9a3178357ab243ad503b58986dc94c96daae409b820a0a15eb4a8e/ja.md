```
aislogimpweights(rbm; ...)
```

与えられた `rbm` の分配関数の比率を推定するための対数化された重要度重みを計算します。これは、`rbm` と同じ可視バイアスおよび隠れバイアスを持つ重みがゼロのRBMとの比率を推定します。この関数は、[Salakhutdinov, 2008] のセクション 4.1.3 に記載されているように、アニーリング重要度サンプリングアルゴリズム (AIS) を実装しています。

# オプションのキーワード引数（すべてのタイプのボルツマンマシン用）:

  * `ntemperatures`: 開始モデルからターゲットモデルへのアニーリングのための温度の数、デフォルトは100
  * `temperatures`: 温度のベクトル。デフォルトでは `ntemperatures` の昇順の数値で、0.0 から 1.0 まで等間隔です
  * `nparticles`: 並列チェーンの数と計算された重み、デフォルトは100
  * `burnin`: モデル間のギブス遷移のためにサンプリングするステップ数
