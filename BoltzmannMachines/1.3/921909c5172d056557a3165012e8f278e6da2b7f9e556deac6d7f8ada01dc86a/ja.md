```
traindbm!(dbm, x; ...)
```

`dbm`（`BasicDBM`またはより一般的な`MultimodalDBM`）を、トレーニングデータセット`x`を使用して一般的なボルツマンマシンの学習手順でトレーニングします。学習ステップは、平均場推論（正のフェーズ）、ギブスサンプリングによる確率的近似（負のフェーズ）、およびパラメータの更新で構成されます。

# オプションのキーワード引数（重要度順）：

  * `epoch`: トレーニングエポックの数
  * `learningrate`/`learningrates`: 各エポックの重みとバイアスを更新するための学習率のベクトル。学習率はエポックとともに減少する必要があります。例えば、因子`a / (b + epoch)`で。`learningrate`として1つの値のみが与えられた場合、`a`と`b`はそれぞれ11.0と10.0です。
  * `batchsize`: ミニバッチ内のサンプル数。デフォルトではミニバッチは使用されず、各エポックで勾配を計算するために完全なデータセットが使用されます。
  * `nparticles`: サンプリングに使用される粒子の数、デフォルトは100
  * `monitoring`: 各トレーニングエポックの後に実行される関数。トレーニングされたDBMと現在のエポックを引数として受け取る必要があります。
