```
initparticles(bm, nparticles; biased = false)
```

ボルツマンマシン `bm` におけるギブスサンプリングのための粒子を作成します。（参照: `Particles`）

ベルヌーイ分布のノードの場合、粒子はベルヌーイ(p)分布の値で初期化されます。`biased == false` の場合、p は 0.5 ですが、そうでない場合はバイアス値にシグモイド関数を適用した結果がノードの個々の p の値として使用されます。

ガウスノードは、`biased == false` の場合、正規分布からサンプリングされます。`biased == true` の場合、ガウス分布の平均はバイアスベクトルによってシフトされ、ノードの標準偏差がサンプリングに使用されます。
