function get*existing*chiller*default*cop(; existing*chiller*max*thermal*factor*on*peak*load=nothing,                                              max*load*kw=nothing,                                              max*load*kw*thermal=nothing) This function returns the default value for ExistingChiller.cop based on:     1. No information about load, returns average of lower and higher cop default values (`cop_unknown_thermal`)     2. If the cooling electric `max_load_kw` is known, we first guess the thermal load profile using `cop_unknown_thermal`,         and then we use the default logic to determine the `existing_chiller_cop` based on the peak thermal load with a thermal factor multiplier.     3. If the cooling thermal `max_load_kw_thermal` is known, same as 2. but we don't have to guess the cop to convert electric to thermal load first.
