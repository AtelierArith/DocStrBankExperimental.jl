```
@remote ex
```

式 `ex` を現在の RemoteREPL 接続の反対側で実行し、その値を返します。

これは両方向で使用できます：

1. 通常の `julia>` プロンプトから、リモートサーバーで `ex` を実行し、その値をクライアントに返します。
2. リモートプロンプトから、*クライアント* で `ex` を実行し、結果の値をリモートサーバーにプッシュします。

# 例

クライアントからサーバーに値をプッシュします：

```
julia> client_val = 1:100;

julia@localhost> server_val = @remote client_val
1:100
```

サーバーから変数のペア `(x,y)` を取得し、クライアントで単一の行でプロットします：

```
# 2行で
julia> x,y = @remote (x, y)
       plot(x, y)

# または単一の式として
julia> plot(@remote((x, y))...)
```
