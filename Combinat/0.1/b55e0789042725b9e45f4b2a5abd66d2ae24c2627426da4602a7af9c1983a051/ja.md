`combinations(mset[,k];dict=false)`,  `ncombinations(mset[,k];dict=false)`

`combinations` は、重複の可能性があるコレクションまたはイテラブルである多重集合 `mset` のすべての組み合わせを返します。第二の整数引数 `k` が与えられた場合、`k` 要素の組み合わせを返します。`k` は整数のベクトルでもあり、その場合は要素数がこれらの整数のいずれかである組み合わせを返します。

`ncombinations` は（より速く）組み合わせの数を返します。

*組み合わせ* は順序のない部分列です。

デフォルトでは、`mset` の要素はソート可能であると仮定され、組み合わせはソートされた `Vector` で表されます。固定された数 `k` の要素を持つ組み合わせは、辞書順でリストされます。`mset` の要素がソート可能ではないがハッシュ可能な場合、キーワード `dict=true` を指定することができ、（やや遅い）計算は `Dict` を使用して行われます。

`mset` に重複がない場合、すべての組み合わせのリストは `mset` の *冪集合* です。

```julia-repl
julia> ncombinations([1,2,2,3])
12

julia> combinations([1,2,2,3])
12-element Vector{Vector{Int64}}:
 []
 [1]
 [2]
 [3]
 [1, 2]
 [1, 3]
 [2, 2]
 [2, 3]
 [1, 2, 2]
 [1, 2, 3]
 [2, 2, 3]
 [1, 2, 2, 3]
```

組み合わせは、大きな多重集合の組み合わせを列挙できるイテレータ [`Combinations`](@ref) によって実装されています。
