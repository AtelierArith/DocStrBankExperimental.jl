`multisets(set,k)`, `nmultisets(set,k)`

`multisets` は、集合 `set` の要素からなる長さ `k` のすべての多重集合の集合を返します（重複のないコレクション）。`nmultisets` は多重集合の数を返します。

長さ `k` の *多重集合* は、集合 `set` からの長さ `k` の重複を許した無順序の選択であり、集合 `set` からの要素で構成された長さ `k` のソートされたベクターで表されます（これは「置換を伴う組み合わせ」と呼ばれることもあります）。

```julia-repl
julia> multisets(1:4,3)
20-element Vector{Vector{Int64}}:
 [1, 1, 1]
 [1, 1, 2]
 [1, 1, 3]
 [1, 1, 4]
 [1, 2, 2]
 [1, 2, 3]
 [1, 2, 4]
 [1, 3, 3]
 [1, 3, 4]
 [1, 4, 4]
 [2, 2, 2]
 [2, 2, 3]
 [2, 2, 4]
 [2, 3, 3]
 [2, 3, 4]
 [2, 4, 4]
 [3, 3, 3]
 [3, 3, 4]
 [3, 4, 4]
 [4, 4, 4]
```
