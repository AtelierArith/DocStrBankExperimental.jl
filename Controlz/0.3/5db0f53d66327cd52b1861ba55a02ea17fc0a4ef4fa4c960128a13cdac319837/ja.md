```
axs = bode_plot(tf, log10_ω_min=-4.0, log10_ω_max=4.0, nb_pts=300)
```

伝達関数 `tf` のボードプロットを描画して、その周波数応答を視覚化します。プロットの2つの軸を返し、`matplotlib` コマンドを使用してさらに調整できます。

ボードプロットが表示する周波数の範囲を `log10_ω_min` と `log10_ω_max` で調整します。

`nb_pts` を使用してボードプロットの解像度を上げます。

さらなる修正のために `CairoMakie.jl` の `Figure` オブジェクトを返します。
