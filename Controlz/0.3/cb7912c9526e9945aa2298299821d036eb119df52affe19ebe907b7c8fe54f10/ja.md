```
tf = zpk_form(tf)
```

伝達関数 `tf` をゼロ、極、k-因子形式で表します：

$$
g(s)=k\dfrac{\Pi_j (s-z_j)}{\Pi_j (s-p_j)}
$$

ここで、$z_j$ はゼロ $j$、$p_j$ は極 $j$、$k$ は伝達関数を一意に指定する定数因子（ゼロ周波数ゲインとは異なる）です。

これは、分母の最高次数の $s$ に係数 $1$ が関連付けられるように、1.0 を巧妙に掛けることで達成されます。

# 例

```jldoctest
g = 8.0 / (2 * s^2 + 3 * s + 4)
g_zpk = zpk_form(g)
# 出力
         4.0
---------------------
1.0*s^2 + 1.5*s + 2.0
```
