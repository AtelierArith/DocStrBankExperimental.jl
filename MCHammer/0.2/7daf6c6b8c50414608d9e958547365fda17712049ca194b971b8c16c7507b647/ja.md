```
sensitivity_chrt(ArrayName::DataFrame, TargetCol, Chrt_Type=1)
```

感度指標を要約する横棒グラフを生成します。これは、モンテカルロ試行や探索的データ分析を分析するための特別なユーティリティを備えたDataFrame内の変数に対して行われます。

# 説明

この関数は、1つ以上のターゲット列と入力DataFrame内のすべての列との間のペアワイズ相関を、スピアマン（順位）およびピアソン相関測定を使用して計算します。さらに、スピアマン相関の二乗に基づく分散への寄与度を計算します。関数は次のような横棒グラフを構築します。

  * `Chrt_Type == 1` の場合、x軸はスピアマン（順位）相関を示します。
  * `Chrt_Type == 2` の場合、x軸はピアソン相関を表示します。
  * `Chrt_Type` の他の値の場合、x軸は分散への寄与の割合を示します。

各変数は色分けされており（負の相関は赤、正の相関は青）、影響の方向について即座に視覚的な手がかりを提供します。結果として得られるグラフは、シミュレーション結果に最も重要な影響を与える入力を評価するのに特に役立ち、ユーザーがさらなる分析やモデルの洗練のために変数を優先順位付けすることを可能にします。

# 引数

  * `ArrayName::DataFrame`: 各列が異なる変数を表すシミュレーションまたは実験データを含むDataFrame。
  * `TargetCol`: 感度を測定するための `ArrayName` 内の1つ以上の列識別子（シンボルまたは文字列）。関数はこれらのターゲット列と他のすべての列との相関を計算します。
  * `Chrt_Type`: x軸の指標を選択するためのオプションの整数フラグ（デフォルトは `1`）：

      * `1` — スピアマン相関（順位相関）
      * `2` — ピアソン相関
      * その他の値 — 分散への寄与の割合（符号付き）
