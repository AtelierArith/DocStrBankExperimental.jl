単一またはペアパラメータの結合ルール。非対角成分が次のように等しいペアパラメータを返します：

```
ϵᵢⱼ = √(ϵᵢϵⱼ)*(σᵢᵢ^3 * σⱼⱼ^3)/σᵢⱼ^6 
```

`σᵢⱼ`が定義されていない場合、定義は単純な幾何平均に簡略化されます：

```
ϵᵢⱼ = √(ϵᵢϵⱼ)
```

すでに設定されている非対角成分は無視されます。

単一パラメータが入力として渡された場合、それは`ϵᵢᵢ = ϵᵢ`を用いてペアパラメータに変換されます。
