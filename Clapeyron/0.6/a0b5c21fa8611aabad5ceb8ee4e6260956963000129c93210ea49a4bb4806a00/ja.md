```
Tw, vw = widom_temperature(model, p; T0 = nothing, v0 = nothing)
```

与圧力に対して熱力学モデルのウィドム温度と対応する体積を計算します。

ウィドム点は、超臨界領域の等圧線に沿った等圧比熱容量の最大値が発生する温度を表します。特に、`widom_temperature`は指定された温度でウィドム点を計算します。

# 引数

  * `model`: 計算に使用される熱力学モデル
  * `p`: ウィドム温度を計算する圧力
  * `T0`: オプションの初期温度推定値。提供されない場合、デフォルト値が使用されます
  * `v0`: オプションの初期体積推定値。提供されない場合、デフォルト値が使用されます

# 戻り値

  * `Tw`: 指定された圧力でのウィドム温度
  * `vw`: ウィドム温度と指定された圧力での体積

# 例

```julia
model = PCSAFT(["methane"])
p = 150e5  #150 bar, 臨界圧を超える
Tw, vw = widom_temperature(model, p)
```
