```
sigma_LorentzBerthelot(σ::SingleOrPair)::PairParam
```

単一またはペアパラメータの組み合わせルール。非対角成分が次のように等しいペアパラメータを返します：

```
σᵢⱼ = (1 - ζᵢⱼ)*(σᵢ + σⱼ)/2
```

`ζᵢⱼ`が定義されていない場合、定義は単純な算術平均に簡略化されます：

```
σᵢⱼ = (σᵢ + σⱼ)/2
```

すでに設定されている非対角成分は無視されます。

単一パラメータが入力として渡された場合、それはペアパラメータに変換され、`σᵢᵢ = σᵢ`となります。
