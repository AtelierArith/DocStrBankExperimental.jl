```
pw, vw = widom_pressure(model, T; p0 = nothing, v0 = nothing)
```

与えられた温度で熱力学モデルのウィドム圧力と対応する体積を計算します。

ウィドム点は、超臨界領域の等圧線に沿って等圧熱容量の最大値が発生する温度を表します。特に、`widom_pressure`は指定された温度でウィドム点を計算します。

# 引数

  * `model`: 計算に使用される熱力学モデル
  * `T`: ウィドム圧力を計算する温度
  * `p0`: オプションの初期圧力推定値。提供されない場合はデフォルト値が使用されます
  * `v0`: オプションの初期体積推定値。提供されない場合はデフォルト値が使用されます

# 戻り値

  * `pw`: 指定された温度でのウィドム圧力
  * `vw`: ウィドム圧力および指定された温度での体積

# 例

```julia
model = PCSAFT(["methane"])
T = 200.0  # 200 K
pw, vw = widom_pressure(model, T)
```
