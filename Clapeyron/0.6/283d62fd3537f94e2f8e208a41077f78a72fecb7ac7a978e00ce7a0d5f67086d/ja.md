```
FugBubbleTemperature(kwargs...)
```

フェーズ組成を更新するために逐次代入法を使用し、温度を更新するために外部ニュートンループを使用して、[`bubble_temperature`](@ref)をフガシティ係数を介して計算するメソッドです。`itmax_newton`回の反復後に収束が達成されない場合、システムは多次元非線形方程式系を使用して解決されます。

入力:

  * `y = nothing`: オプション、蒸気相組成の初期推測。
  * `T0 = nothing`: オプション、バブル温度[`K`]の初期推測。
  * `vol0 = nothing`: オプション、液体および蒸気相の体積の初期推測
  * `itmax_newton = 10`: オプション、ニュートン法を使用して温度を更新するための反復回数
  * `itmax_ss = 5`: オプション、逐次代入法を使用して液体相組成を更新するための反復回数
  * `tol_x = 1e-8`: オプション、逐次代入サイクルを停止するための許容誤差
  * `tol_T = 1e-8`: オプション、ニュートンサイクルを停止するための許容誤差
  * `tol_of = 1e-8`: オプション、目的関数がゼロであるかどうかを確認するための許容誤差。
  * `nonvolatiles`: オプション、揮発性のない化合物を含む文字列のベクター。これらは蒸気相でゼロに設定されます。
