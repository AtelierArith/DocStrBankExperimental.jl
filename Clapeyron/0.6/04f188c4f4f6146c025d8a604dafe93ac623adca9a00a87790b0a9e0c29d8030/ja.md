```
@f(func,a,b,c,...)
```

このマクロは以下のエイリアスです。

```
func(model, V, T, z, a, b, c, ...)
```

ここで `func` は関数の名前、`model` はモデル構造体、`V` は体積、`T` は絶対温度、`z` は各成分のモル数の配列、`a`、`b`、`c`、... は `func` に渡される任意のパラメータです。

モデルに関与する関数は `model`、`V`、`T`、`z` パラメータを含むことが非常に一般的であるため、このマクロは関数の最初の4つのパラメータが上記のように正確に書かれている限り、コードの繰り返しを減らすのに役立ちます。
