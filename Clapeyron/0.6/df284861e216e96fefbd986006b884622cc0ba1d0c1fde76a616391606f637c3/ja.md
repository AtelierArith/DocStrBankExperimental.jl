```
ChemPotBubblePressure(kwargs...)
```

化学ポテンシャルを介して[`bubble_pressure`](@ref)を計算する関数です。化学ポテンシャルの等式系の方程式を直接解きます。

入力:

  * `y0 = nothing`: オプション、蒸気相組成の初期推定
  * `p0 = nothing`: オプション、バブル圧力[`Pa`]の初期推定
  * `vol0 = nothing`: オプション、液相と蒸気相の体積の初期推定
  * `atol = 1e-8`: オプション、非線形方程式系の絶対許容誤差
  * `rtol = 1e-12`: オプション、非線形方程式系の相対許容誤差
  * `max_iters = 1000`: オプション、最大反復回数
  * `nonvolatiles = nothing`: オプション、揮発性でない化合物を含む文字列のベクター。これらは蒸気相でゼロに設定されます。
