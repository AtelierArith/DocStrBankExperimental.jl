```
ActivityDewPressure(kwargs...)
```

活動係数を使用して [`dew_pressure`](@ref) を計算する関数。活動係数モデルでは、連続置換法を用いて問題を解決します。ヘルムホルツベースのモデルでは、活動係数のためにチャップマン近似を使用します。

入力:

  * `gas_fug = true`: ソルバーが気体のフガシティ係数を使用するかどうか。`ActivityModel` ではデフォルトで `false` に設定されています。
  * `poynting = true`: ソルバーが液体のフガシティ係数にポインティング補正を使用するかどうか。`ActivityModel` ではデフォルトで `false` に設定されています。
  * `x0 = nothing`: オプション、液相組成の初期推定
  * `p0 = nothing`: オプション、露点圧の初期推定 [`Pa`]
  * `vol0 = nothing`: オプション、液相と蒸気相の体積の初期推定
  * `atol = 1e-8`: オプション、非線形方程式系の絶対許容誤差
  * `rtol = 1e-12`: オプション、非線形方程式系の相対許容誤差
  * `itmax_ss = 40`: オプション、連続置換の最大反復回数
