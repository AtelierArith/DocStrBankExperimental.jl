```
tpd(model,p,T,z;break_first = false,lle = false,tol_trivial = 1e-5, di = nothing)
```

接線平面距離関数（`tpd`）を計算します。返されるのは：

  * `tpd < 0` の試行相組成を持つベクトル
  * `tpd` 値を持つベクトル
  * 入力組成の相を示す記号を持つベクトル
  * 試行組成の相を示す記号を持つベクトル

純粋な試行組成から始めて、各二相の組み合わせを反復し、次に逐次置換を行い、その後ギブス最適化を行います。

ベクトルが空である場合、手順は負の `tpd` を見つけることができませんでした。これは、相が（ほぼ）確実に安定していることを示す指標です。
