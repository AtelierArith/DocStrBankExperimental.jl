```
kij_mix(f,p::ClapeyronParam,k::PairParam)::PairParam
kij_mix(f,p::ClapeyronParam)::PairParam
```

`kᵢⱼ`相互作用パラメータを持つペアパラメータの一般的な結合ルール。対角以外のエントリが次のように等しいペアパラメータを返します：

```
pᵢⱼ = f(pᵢ,pⱼ,kᵢⱼ)
```

ここで、`f`は次のルールに従う「結合」関数です：

```
f(pᵢ,pⱼ,0) = f(pⱼ,pᵢ,0)
f(pᵢ,pᵢ,0) = pᵢ
```

そして、`k`は次のように従わなければなりません：

```
kᵢᵢ = 0 
```

すでに設定されている対角以外のエントリは無視されます。

単一のパラメータが入力として渡された場合、それは`pᵢᵢ = pᵢ`を持つペアパラメータに変換されます。
