```
MC(vol::Array{F,3}, I::Type = Int; x = [], y = [], z = [])
```

# 説明

マーチングキューブアルゴリズムの一時的なデータと出力を保持する構造体です。頂点、法線、および三角形は `m.vertices`、`m.normals`、`m.triangles` を使用してアクセスできます。1ベースのインデックスが想定されています。オプションで、`x`、`y`、`z` を渡すことで、頂点座標を `0:nx-1`、`0:ny-1`、`0:nz-1` の代わりに正規化できます。

# 引数

```
- `vol`: マーチングキューブアルゴリズムが適用される浮動小数点数のランク3配列（ボリューム）。
- `I`: Int32、Int64、...: 頂点 / 法線 / 三角形インデックス `Integer` 型（デフォルトは `Int`）。
- `x` 頂点.x を `x` の座標に正規化します。
- `y` 頂点.y を `y` の座標に正規化します。
- `z` 頂点.z を `z` の座標に正規化します。
```
