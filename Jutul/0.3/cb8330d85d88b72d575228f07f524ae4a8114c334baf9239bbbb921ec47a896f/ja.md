```
setup_state(model::JutulModel, name1 = value1, name2 = value2)
```

与えられたモデルのために、モデルで定義された主要変数の値を使って状態を設定します。通常、すべての主要変数はこの方法で初期化する必要があります。

# 引数

  * `name=value`: 主要変数の名前と、主要変数を初期化するために使用される値。

スカラー（または[`VectorVariables`](@ref)に対して適切なサイズの短いベクトル）は、全ドメインにわたって繰り返されます。一方、長さ（[`VectorVariables`](@ref)に対する列数）がエンティティ数（例えば、セル変数のセル数）に等しいベクトル（または[`VectorVariables`](@ref)に対する行列）は、直接使用されます。

注意: カスタムモデルのために[`setup_state!`]@refをオーバーロードしたいと思うかもしれませんが、`setup_state`の代わりに。
