```
NLSMeta
```

非線形最小二乗モデルに関連するメタデータのベースタイプ。

---

```
NLSMeta(nequ, nvar; kwargs...)
```

`nequ` 方程式と `nvar` 変数を持つ `NLSMeta` を作成します。次のキーワード引数が受け入れられます：

  * `x0`: 初期推定
  * `nnzj`: 残差のヤコビ行列の非ゼロ要素を格納するために必要な要素数
  * `nnzh`: 残差のヘッセ行列の合計の非ゼロ要素を格納するために必要な要素数
  * `lin`: 線形残差のインデックス

`NLSMeta` には、上記の変数から計算される次の属性も含まれています：

  * `nequ`: 残差のサイズ
  * `nvar`: 変数の数
  * `nln`: 非線形残差のインデックス
  * `nnln`: 非線形一般残差の数
  * `nlin`: 線形残差の数
