```
lambdamatrix(λ, inds::Vector{Int}, sings) -> Λk
```

新しい動的システムを作成するために使用される行列 $\mathbf{\Lambda}_k$ を返します。このシステムでは、いくつかの不安定な固定点が関数 [`periodicorbits`](@ref) で安定に変わります。

## 引数

1. `λ<:Real` : $C_k$ 行列の乗数で、`0<λ<1` です。
2. `inds::Vector{Int}` : このベクトルの `i` 番目のエントリは、$C_k$ の `i` 番目の列の非ゼロ要素の *行* を示します。
3. `sings::Vector{<:Real}` : $C_k$ の `i` 番目の列の要素は、`signs[i] > 0` の場合は +1 で、そうでない場合は -1 です（`sings` は `Bool` ベクトルでも構いません）。

`lambdamatrix(λ, D::Int)` を呼び出すと、すべての可能な組み合わせからランダムに `inds` と `signs` を生成することによってランダムな $\mathbf{\Lambda}_k$ が作成されます。これらの組み合わせの *コレクション* は、関数 [`lambdaperms`](@ref) から取得できます。

## 説明

`inds` の各要素は *一意でなければなりません*。そうでないと、結果の行列は直交せず、特別な反射と置換の群を表しません。

`λ, inds, sings` の適切な値を決定することは簡単ではありません。しかし、文献 [^Pingel2000] には、その決定に役立つ多くの情報があります。また、さまざまな値を適切に選ぶことで、例えば最も不安定なものから最も不安定なものまで周期的軌道を並べ替えることができます。詳細については [^Diakonos1998] を参照してください。

[^Pingel2000]: D. Pingel *et al.*, Phys. Rev. E **62**, pp 2119 (2000)

[^Diakonos1998]: F. K. Diakonos *et al.*, Phys. Rev. Lett. **81**, pp 4349 (1998)
