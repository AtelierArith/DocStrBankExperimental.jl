```
testchaos01(x::Vector [, cs, N0]) -> chaotic?
```

GottwaldとMelbourneによって導入された「0-1」カオステスト[^Gottwald2016]を時系列`x`に対して実行します。`x`がカオスであれば`true`を、そうでなければ`false`を返します。

## 説明

このメソッドは、与えられた時系列がカオスであるかどうかを、次のように二次元拡散過程に変換することでテストします：

$$
p_n = \sum_{j=1}^{n}\phi_j \cos(j c),\quad q_n = \sum_{j=1}^{n}\phi_j \sin(j c)
$$

時系列がカオスであれば、過程の平均二乗変位は`sqrt(length(x))`のように成長しますが、時系列が規則的であれば一定のままです。

ここでの実装は、平均二乗変位の成長を測定する係数`K`を計算し、単純に`K > 0.5`かどうかをチェックします。`K`は与えられた`c`に対する$K_c$の中央値です。参照を見てください。

さまざまな`Kc`にアクセスしたい場合は、メソッド`testchaos01(x, c::Real, N0)`を呼び出す必要があります。これにより`Kc`が返されます。実際、高レベルのメソッドは単に`median(testchaos01(x, c, N0) for c in cs) > 0.5`です。

`cs`のデフォルトは`3π/5*rand(100) + π/4`で、二次元過程の長さ`N0`は`N0 = length(x)/10`です。

連続的な動的システムからサンプリングされたデータに関しては、`cs`の値に注意が必要です。また、このメソッドはわずかなノイズでもかなり悪い結果を出すため、わずかなノイズのある時系列に対しても`true`を返すことがあります。これを緩和するためのいくつかの可能性がありますが、アプリケーションに特有の文脈に依存します。詳細については[^Gottwald2016]を参照してください。

[^Gottwald2016]: Gottwald & Melbourne, “The 0-1 test for chaos: A review” [Lect. Notes Phys., vol. 915, pp. 221–247, 2016.]( www.doi.org/10.1007/978-3-662-48410-4_7)
