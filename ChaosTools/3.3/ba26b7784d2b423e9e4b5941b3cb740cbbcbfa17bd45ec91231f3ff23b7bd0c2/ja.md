```
local_growth_rates(ds::DynamicalSystem, points::StateSpaceSet; kwargs...) → λlocal
```

動的システム `ds` の摂動の局所指数成長率を、`points` に与えられた初期条件に対して計算します。各初期条件 `u ∈ points` に対して、合計 `S` の摂動が生成され、正確に時間 `Δt` だけ進化します。指数的な局所成長率は、単純に `log(g/g0)/Δt` によって定義され、ここで `g0` は初期の摂動サイズ、`g` は `Δt` 後のサイズです。したがって、`λlocal` はサイズ `(length(points), S)` の行列です。

この関数は [`lyapunov`](@ref) の修正です。完全な非線形ダイナミクスと [`ParallelDynamicalSystem`](@ref) を使用して摂動を進化させますが、再スケーリングは行わず、摂動の成長の状態依存性と時間依存性を探ることを可能にします。実際の成長は `exp(λlocal * Δt)` によって与えられます。

この関数の出力は、時々「非線形局所リャプノフ指数」と呼ばれます。

## キーワード引数

  * `S = 100`
  * `Δt = 5`
  * `perturbation`: 与えられた場合、これは関数 `perturbation(ds, u, j)` であり、システム、現在の初期条件 `u`、およびカウンター `j ∈ 1:S` に基づいて摂動ベクトル（できれば `SVector`）を出力します。与えられない場合、キーワード `e = 1e-6` によって与えられるノルムを持つランダムな摂動が生成されます。
