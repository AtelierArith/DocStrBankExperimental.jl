```
AbstractBSplineBasis{k,T}
```

Bスプライン基底を定義する抽象型、またはより一般的にはBスプラインから定義された関数基底。

基底はBスプラインの次数 `k` とノット要素型 `T` によって表されます。

---

```
(B::AbstractBSplineBasis)(
    x::Real, [op = Derivative(0)], [T = float(typeof(x))];
    [ileft = nothing],
) -> (i, bs)
```

`x` で非ゼロのすべての基底関数を評価します。

これは `evaluate_all` の便利なエイリアスです。オプション引数や返される値の詳細については [`evaluate_all`](@ref) を参照してください。
