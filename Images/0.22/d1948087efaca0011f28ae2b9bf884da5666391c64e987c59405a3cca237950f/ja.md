```
ColorizedArray(intensity, label::IndirectArray) -> A
```

配列を作成します。`label` 配列（各ピクセルが離散的な色のリストの1つに割り当てられる）と `intensity` 配列（各ピクセルがスカラー値を持つ）を組み合わせます。`A` は次の条件を満たします。

```
A[i,j,...] = intensity[i,j,...] * label[i,j,...]
```

ラベル配列は、グレースケールの強度にその点のラベルに関連付けられた色を「着色」します。

この計算は遅延的に実行され、大きな配列にも適しています。
