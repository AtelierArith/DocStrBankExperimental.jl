```
rankplot(test, ranks; kwargs...)
```

`simulate_ranks`を使用してシミュレートされたランクをプロットします。テスト対象が正しい場合、ランクは視覚的に一様分布からのサンプルに似ているはずです。黒い水平線は一様分布の密度を示し、色付きのバンドは1σ、2σ、3σの信頼区間です。

!!! info
    信頼区間は二項観測の正規近似から導出されており、均等なサイズのビンを仮定しています。


!!! info
    このプロットレシピを使用するには`Plots`をインポートする必要があります。


# 引数

  * `test::ExactRankTest`: ランクをシミュレートするために使用される正確なランクテストオブジェクト。
  * `ranks`: `simulate_rank`の出力。

# キーワード引数

  * `stats_names`: ランクシミュレーションに使用される統計の名前。デフォルト引数は自動的にデフォルト名を割り当てます。（デフォルト: :auto）。
  * `bins`、`layout`、`size`など、`Plots`属性に対応するキーワード引数が適用される場合があります。
