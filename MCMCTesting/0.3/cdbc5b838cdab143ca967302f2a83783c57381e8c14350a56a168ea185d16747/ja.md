```
ExactRankTest(n_samples, n_mcmc_steps; n_mcmc_thin)
```

可逆MCMCカーネルのための正確な順位仮説検定戦略。Gandy & Scott (2021)のアルゴリズム2。

# 引数

  * `n_samples::Int`: シミュレーションする順位の数。
  * `n_mcmc_steps::Int`: 単一の順位をシミュレーションするためにシミュレーションされるMCMC状態の数。
  * `n_mcmc_thin::Int`: MCMCチェーンに適用される間引きの数。

# 戻り値

  * `pvalues`: `statistics`から返される統計量の各次元に対して計算されたp値。

# 要件

このテストには、`model`および`kernel`のために以下の関数が実装されている必要があります：

  * `markovchain_transition`
  * `sample_joint`

さらに、このテストは明示的に以下を仮定します：

  * `kernel`は可逆である。

参考までに、以下のカーネルは可逆です：

  * 任意のメトロポリス・ヘイスティングスカーネル
  * スライスサンプラー
  * ハミルトニアンモンテカルロ/ノーUターンサンプラー（軌道上での再サンプリングを伴う）
  * ランダムスキャン/順列スキャンギブスサンプラー
  * 調整されていないランジュバン

しかし、以下のカーネルは可逆ではありません：

  * 永続的な運動量を持つハミルトニアンモンテカルロ（いわゆるホロウィッツの方法、一般化ハミルトニアンモンテカルロ）
  * 系統的スキャンギブスサンプラー
  * 部分的決定論的マルコフ過程

!!! info
    可逆でない`kernel`にこのテストを適用すると、定常分布が正しくても偽陰性が発生します。


# テストのためのキーワード引数

`mcmctest`または`seqmcmctest`を呼び出す際、このテストには追加のキーワード引数があります：

  * `uniformity_test_pvalue`: p値計算戦略。

デフォルトの戦略は$\chi^2$テストです。均一性仮説検定から単一のp値を返す任意の関数が機能します。フォーマットは次のとおりです：

```julia
uniformity_test_pvalue(x::AbstractVector)::Real
```
