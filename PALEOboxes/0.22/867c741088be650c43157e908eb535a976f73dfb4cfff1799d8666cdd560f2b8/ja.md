```
smoothstepcubic(x, xedge, xwidth) -> y
```

幅 `xwidth` で位置 `xedge` におけるスムーズなステップ関数。

不連続なステップ関数をスムージングする方法を提供し、導関数が存在するようにして、ヤコビ行列を必要とする数値解法（例えば、硬いODEソルバー）を助けます。

範囲 `xedge +/- xwidth/2` で三次関数を使用するため、一次導関数は連続ですが、高次導関数はそうではありません。

[`zero_ad`](@ref) を使用して、ヤコビ行列のスパースパターンをトレースするためのAD依存情報を保持します。

返す値:

  * x < (xedge - xwidth/2) の場合は 0.0
  * x > (xedge + xwidth/2) の場合は 1.0
  * xedge-xwidth/2 < x < xedge+xwidth/2 の場合はスムーズなステップ
