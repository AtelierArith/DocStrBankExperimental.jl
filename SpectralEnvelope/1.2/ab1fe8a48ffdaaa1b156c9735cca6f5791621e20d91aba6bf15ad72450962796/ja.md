```
 get_mappings(data, freq; m = 3)
```

周波数 'freq' に対応する最適なマッピングを返します。制御目的で 'f' でのピークの位置と強度を印刷します。入力 :     - data : 入力カテゴリカル時系列     - freq : 最適なマッピングを求める周波数     - m : スムージングパラメータ（'spectral_envelope' を参照） 返すもの :     - mappings : 希望する周波数に関連するピークでのマッピング。

例:     # データがあり、スペクトルエンベロープをプロットすると、0.33 でピークが見えます     m = get_mappings(data, 0.33)
