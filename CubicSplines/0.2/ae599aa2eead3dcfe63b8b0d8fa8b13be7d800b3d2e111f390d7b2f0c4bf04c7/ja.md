Akimaスプラインを[1]に従って構築します。元のメソッドに加えて、この実装では、スプラインの両側でカスタマイズされた外挿を許可します。外挿の動作は、コンストラクタに追加のキーワード引数extraplおよびextraprとして与えられる3次多項式の係数によって定義されます。extraplおよび/またはextraprは、「nothing」または係数のセットである必要があります。

# ケース1: 外挿が「nothing」に設定されている

この場合、[1]のセクション2.3に従ったデフォルトの二次外挿が使用され、スプラインは境界の外で評価されるとエラーをスローします。

# ケース2: 外挿がベクトルに設定されている

コンテナはn要素の値のコレクションとして与えられます:     extrapl = [p1, p2, p3, ..., pn] これらは次の関数を評価するために使用されます:     y = p0 + p1*(x-x3) + p2*(x-x3)^2 + p3*(x-x3)^3 + ... + pn*(x-x3)^n 定数p0は定義によりy3に等しいです（連続スプライン）。x3およびy3はそれぞれ最初/最後のデータポイントの値です。
