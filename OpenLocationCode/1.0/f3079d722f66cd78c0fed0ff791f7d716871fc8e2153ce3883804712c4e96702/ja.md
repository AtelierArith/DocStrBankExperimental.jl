```
encode(latitude, longitude[, codelength])
```

位置をオープンロケーションコードにエンコードします。指定された長さのコードを生成し、長さが提供されていない場合はデフォルトの長さを使用します。

長さはコードの精度を決定します。デフォルトの長さは10文字で、約13.9x13.9メートルのコードを返します。長いコードは小さなエリアを表しますが、長さが15を超えるとセンチメートル未満になり、11または12が有用なコードの限界である可能性があります。

# 引数

  * `latitude`: 符号付き度数の緯度。範囲-90から90にクリップされます。
  * `longitude`: 符号付き度数の経度。範囲-180から180に正規化されます。
  * `codelength`: 出力コードの有効桁数で、区切り文字は含まれません。

# 例:

```julia
julia> encode(50.173168, 8.338086, 11)
"9F2C58FQ+768"
```
