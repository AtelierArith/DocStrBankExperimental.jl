```
stationary_statistics(tpt)
```

Compute and return the following statistics in a `NamedTuple`:

  * `stationary_distribution`
  * `forward_committor`
  * `backward_committor`
  * `reactive_density`
  * `reactive_current`
  * `forward_current`
  * `reactive_rate`
  * `reactive_time`
  * `remaining_time`
  * `hitting_location_distribution`
