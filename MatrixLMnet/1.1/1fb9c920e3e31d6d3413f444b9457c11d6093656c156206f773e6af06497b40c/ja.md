```
make_folds(n::Int64, k::Int64=10, k2::Int64=k)
```

`k` 個の重複しないフォールドを生成します。

# 引数

  * n = フォールドに分割する観測値の総数。
  * k = 作成するフォールドの数。デフォルトは 10。`k=1` の場合、この次元のすべてのデータが各フォールドに含まれます。
  * k2 = `k=1` の場合、この次元のすべてのデータが各フォールドに含まれます。`k2` はフォールドの数を指定します。デフォルトは `k` で、少しばかげていますが、プレースホルダーが必要です。

# 値

インデックスの配列の長さ `k` の 1d 配列
