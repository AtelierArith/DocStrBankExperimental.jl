EQDSKファイルの内容。

属性：

```
case    識別文字列
nw      水平方向のグリッドポイントの数
nh      垂直方向のグリッドポイントの数
rdim    計算ボックスの水平方向の寸法（メートル）
zdim    計算ボックスの垂直方向の寸法（メートル）
rcentr  真空トロイダル磁場BCENTRのメートル単位のR
rleft   長方形計算ボックスの最小R（メートル）
zmid    計算ボックスの中心のZ（メートル）
rmaxis  磁気軸のR（メートル）
zmaxis  磁気軸のZ（メートル）
simag   磁気軸でのポロイダルフラックス（ウェーバー/ラジアン）
sibry   プラズマ境界でのポロイダルフラックス（ウェーバー/ラジアン）
bcentr  RCENTRでの真空トロイダル磁場（テスラ）
current プラズマ電流（アンペア）
fpol    フラックスグリッド上のm-T単位のポロイダル電流関数、F = RB_T
pres    均一フラックスグリッド上のプラズマ圧力（ニュートン/ m^2）
ffprim  均一フラックスグリッド上のFF’(ψ)（(mT)^2 / (ウェーバー/ラジアン)）
pprime  均一フラックスグリッド上のP’(ψ)（(ニュートン/m^2) / (ウェーバー/ラジアン)）
psirz   長方形グリッドポイント上のポロイダルフラックス（ウェーバー/ラジアン）
qpsi    軸から境界までの均一フラックスグリッド上のq値
nbbbs   境界ポイントの数
limitr  リミターポイントの数
rbbbs   境界ポイントのR（メートル）
zbbbs   境界ポイントのZ（メートル）
rlim    周囲のリミター輪郭のR（メートル）
zlim    周囲のリミター輪郭のZ（メートル）
```
