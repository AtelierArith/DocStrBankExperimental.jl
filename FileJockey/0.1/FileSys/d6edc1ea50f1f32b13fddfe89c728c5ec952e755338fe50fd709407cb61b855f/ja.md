```
find(path)
find(path, path2,.. ; skip_paths=[])

eachentry(path,..)
findfiles(path,..)  [TODO]
finddupl(path,..)   [TODO]
```

ディレクトリツリーのファイルシステムエントリを取得します。

  * `find` は、ディレクトリやシンボリックリンクを含むすべてのエントリを返します。
  * `eachentry` は同様のことを行いますが、イテレータを返します。
  * `findfiles` は通常のファイルとファイルシンボリックリンクのターゲットのみを返します。また、パスが一意であるかどうかも確認します（同一のパスは、既知のファイルへのシンボリックリンクによって引き起こされることがあり、ほとんどの場合、持っているのは良くありません）。
  * `finddupl` は重複を返します（<original => [dupes]> の辞書）；これらは `rm` で削除できます；削除されるのは重複のみです。

複数のパスを入力できます。これは、新しいインポートディレクトリとライブラリを重複チェックする際に便利です。
