```
Jackson2017()
```

`Fuzzy logic` classifiers defined in [Moore et al 2009](https://doi.org/10.1016/j.rse.2009.07.016) and [Jackson et al 2017](http://dx.doi.org/10.1016/j.rse.2017.03.036) can be used to assign optical class memberships from an `Rrs` vector. This function provides vector `M` and inverse matrix `Sinv` for the Jackson et al 2017 classifier.

Credits: T. Jackson kindly provided the `J17.nc` classifier file
