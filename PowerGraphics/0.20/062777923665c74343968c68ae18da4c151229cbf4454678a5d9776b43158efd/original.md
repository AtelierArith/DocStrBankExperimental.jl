```
load_palette()
load_palette(file)
```

Loads color palette yaml from environment, the `DEFAULT_PALETTE_FILE`, or a given file. See [`color-palette.yaml`](https://github.com/NREL-Sienna/PowerGraphics.jl/blob/main/report_templates/color-palette.yaml) for an example.

# Arguments

  * `file` : path to YAML color palette
