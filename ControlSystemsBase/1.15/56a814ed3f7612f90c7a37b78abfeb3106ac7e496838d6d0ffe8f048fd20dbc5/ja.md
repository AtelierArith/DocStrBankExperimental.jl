離散時間における多項式合成。

「コンピュータ制御システム」第10章に従った多項式合成により、コントローラを設計します $R(q) u(k) = T(q) r(k) - S(q) y(k)$

入力:

  * `BPLUS`  : オープンループの分子の一部
  * `BMINUS` : オープンループの分子の一部
  * `A`      : オープンループの分母
  * `BM1`    : 追加のゼロ
  * `AM`     : クローズドループの分母
  * `AO`     : オブザーバ多項式
  * `AR`     : Rの事前指定係数、

例えば積分部分 [1, -1]^k

  * `AS`     : Sの事前指定係数、

例えばノッチフィルタ [1, 0, w^2]

出力: `R,S,T`  : コントローラの多項式

Diophantine-Aryabhatta-Bezout恒等式の解がどのように選ばれるかは、関数 `dab` を参照してください。
