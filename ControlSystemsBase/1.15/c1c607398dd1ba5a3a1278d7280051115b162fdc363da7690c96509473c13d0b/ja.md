```
y, t, x = impulse(sys[, tfinal])
y, t, x = impulse(sys[, t])
```

システム `sys` に対するインパルスの応答を時間 `t = 0` で計算します。連続時間システムの場合、インパルスは単位ディラックインパルスです。離散時間システムの場合、インパルスは1サンプルの間持続し、振幅は `1/Ts` です。最終時間 `tfinal` または時間ベクトル `t` が提供されていない場合、システムの極の位置に基づいて1つが計算されます。

戻り値は `SimResult` 型の構造体です。`SimResult` は `plot(result)` でプロットすることができ、または `y, t, x = result` として分解することができます。

`y` のサイズは `(ny, length(t), nu)` であり、`x` のサイズは `(nx, length(t), nu)` です。

詳細は [`lsim`](@ref) を参照してください。
