```
sminreal(sys)
```

状態空間システム `sys` の構造的に最小な実現を計算します。構造的に最小な実現とは、`sys` の 0 の位置に基づいて制御不能および観測不能であると判断できる状態のみが削除されるものです。

係数に数値ノイズがあるシステム、例えば `eps` のオーダーのノイズは、構造的簡略化の影響を受けるためにゼロに切り捨てる必要があります。例えば、

```julia
trunc_zero!(A) = A[abs.(A) .< 10eps(maximum(abs, A))] .= 0
trunc_zero!(sys.A); trunc_zero!(sys.B); trunc_zero!(sys.C)
sminreal(sys)
```

線形代数演算を使用して極-零キャンセルを行う [`minreal`](@ref) に対して、𝑂(nₓ^3) の複雑さを持ち、数値的許容範囲の影響を受けるのに対し、`sminreal` は計算的に非常に安価で数値的に正確です（整数で操作します）。ただし、`sminreal` がモデルの次数を減少させる能力ははるかに強力ではありません。

他にも [`minreal`](@ref) を参照してください。
