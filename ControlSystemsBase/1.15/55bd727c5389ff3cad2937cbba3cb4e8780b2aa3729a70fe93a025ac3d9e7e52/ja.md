```
feedback2dof(P, C, F)
```

伝達関数 `P(F+C)/(1+PC)` を返します。これは、プロセス `P`、コントローラー `C`、およびフィードフォワードフィルター `F` を持つ閉ループシステムで、参照から制御信号への経路（`C` をバイパス）です。

```
         +-------+
         |       |
   +----->   F   +----+
   |     |       |    |
   |     +-------+    |
   |     +-------+    |    +-------+
r  |  -  |       |    |    |       |    y
+--+----->   C   +----+---->   P   +---+-->
      |  |       |         |       |   |
      |  +-------+         +-------+   |
      |                                |
      +--------------------------------+
```
