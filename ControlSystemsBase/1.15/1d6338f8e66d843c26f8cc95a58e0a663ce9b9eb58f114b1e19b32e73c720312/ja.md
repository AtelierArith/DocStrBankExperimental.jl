See [`output_sensitivity`](@ref)

出力[sensitivity function](https://en.wikipedia.org/wiki/Sensitivity_(control_systems)) $S_o = (I + PC)^{-1}$は、参照入力から制御誤差への伝達関数であり、入力感度関数$S_i = (I + CP)^{-1}$は、プラント入力の外乱から全プラント入力への伝達関数です。SISOシステムでは、入力と出力の感度関数は等しいです。一般的に、ロバスト性と性能を向上させるために感度関数を最小化したいですが、実際の制約により感度関数は高周波数で常に1に近づきます。ロバスト設計は感度関数のピーク$M_S$を最小化します。$S$のピークマグニチュードは、オープンループナイキスト曲線と臨界点-1との距離の逆数です。感度ピーク$M_S$を上限することで、位相およびゲインマージンの下限が得られます。

$$
ϕ_m ≥ 2\text{sin}^{-1}(\frac{1}{2M_S}), g_m ≥ \frac{M_S}{M_S-1}
$$

一般的に、ゲインと位相マージンを見るよりも$M_S$を制約する方が良い目的です。なぜなら、ゲインと位相の変動が組み合わさる可能性があり、大きなゲインと位相マージンにもかかわらずロバスト性が低下する可能性があるからです。

```
         ▲
         │e₁
         │  ┌─────┐
d₁────+──┴──►  P  ├─────┬──►e₄
      │     └─────┘     │
      │                 │
      │     ┌─────┐    -│
 e₂◄──┴─────┤  C  ◄──┬──+───d₂
            └─────┘  │
                     │e₃
                     ▼
```

  * [`input_sensitivity`](@ref) は d₁ から e₁ への伝達関数です、       (I + CP)⁻¹
  * [`output_sensitivity`](@ref) は d₂ から e₃ への伝達関数です、      (I + PC)⁻¹
  * [`input_comp_sensitivity`](@ref) は d₁ から e₂ への伝達関数です、  (I + CP)⁻¹CP
  * [`output_comp_sensitivity`](@ref) は d₂ から e₄ への伝達関数です、 (I + PC)⁻¹PC
  * [`G_PS`](@ref) は d₁ から e₄ への伝達関数です、                    (1 + PC)⁻¹P
  * [`G_CS`](@ref) は d₂ から e₂ への伝達関数です、                    (1 + CP)⁻¹C
