```
nonlinearity(f)
nonlinearity(T, f)
```

純粋な非線形性を作成します。`f` は静的（メモリなし）非線形関数であると仮定されます $f : R -> R$。

型 `T` のデフォルトは `Float64` です。

注意：ControlSystemsBase.jl の非線形機能は現在実験的であり、セマンティックバージョニングを尊重しない破壊的変更の対象となる可能性があります。自己責任で使用してください。

# 例：

入力を [-1,1] に飽和させる静的入力非線形性を持つ LTI システムを作成します。

```julia
tf(1, [1, 1])*nonlinearity(x->clamp(x, -1, 1))
```

事前定義された非線形性 [`saturation`](@ref)、[`offset`](@ref) も参照してください。

注意：非線形性を持つ線形システムを構成する際には、動作点を正しく処理することが重要です。動作点の処理については [`ControlSystemsBase.offset`](@ref) を参照してください。 ```
