```
RMSProp(η = 0.001, ρ = 0.9)
```

[RMSProp](https://www.cs.toronto.edu/~tijmen/csc321/slides/lecture_slides_lec6.pdf)アルゴリズムを使用するオプティマイザ。再帰ネットワークにとってはしばしば良い選択です。学習率以外のパラメータは一般的に調整する必要はありません。

# パラメータ

  * 学習率（`η`）：重みを更新する前に勾配がどの程度割引かれるか。
  * モーメンタム（`ρ`）：勾配降下法の主要な方向への加速を制御し、実質的に振動を抑える。

# 例

```julia
opt = RMSProp()
opt = RMSProp(0.002, 0.95)
```
