```
AdaBelief(η = 0.001, β::Tuple = (0.9, 0.999))
```

[AdaBelief](https://arxiv.org/abs/2010.07468) オプティマイザは、よく知られた ADAM オプティマイザの変種です。

# パラメータ

  * 学習率 (`η`): 勾配が重みを更新する前にどれだけ割引かれるかの量。
  * モーメンタムの減衰 (`β::Tuple`): 最初の (β1) および 2 番目の (β2) モーメンタム推定のための指数的減衰。

# 例

```julia
opt = AdaBelief()
opt = AdaBelief(0.001, (0.9, 0.8))
```
