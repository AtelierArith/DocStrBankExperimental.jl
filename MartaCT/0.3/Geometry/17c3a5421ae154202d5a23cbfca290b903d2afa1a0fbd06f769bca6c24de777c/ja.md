```
FanBeamGeometry([T=Float32]; <keyword arguments>) where {T}
```

指定されたパラメータを使用して新しいFanBeamGeometryオブジェクトを作成します。

検出器は平面ではなく、円弧上に配置されていると仮定されます。

# 引数

  * `nϕ`: スキャン角度の数。
  * `nd=nothing`: 検出器の数。
  * `rows=nothing`: 再構成された画像の行数。
  * `cols=nothing`: 再構成された画像の列数。
  * `width=nothing`: `cols`のエイリアス。
  * `height=nothing`: `rows`のエイリアス。
  * `D`: 焦点からISOまでの距離。
  * `D′=nothing`: 焦点から検出器までの距離。指定されていない場合、かつ`γ`も指定されていない場合、全体のファン角が1ラジアンになるように定義されます。
  * `γ=nothing`: 度単位の全体のファン角。
  * `δ=1`: 検出器の間隔（セルサイズ）。
  * `α=360`: 度単位のスキャン角度。
  * `α₀=0`: 度単位のスキャン開始角度。
  * `center=nothing`: 仮想中心チャネル、デフォルトは`(nd-1)/2`です。
