構造コスパン。

最初の型パラメータ `L` は、基底カテゴリ `A` から「追加の構造」を持つカテゴリ `X` への関手 L: A → X をエンコードします。L-構造コスパンは、X のコスパンであり、その足は A のオブジェクトの L による像です。カテゴリ X はプッシュアウトを持つと仮定されています。

構造コスパンは、関手 L に対するさらなる仮定なしに二重カテゴリを形成します。対称モノイダル二重カテゴリを得るためには、L が有限コプロダクトを保つ必要があります。実際には、L は通常、右随伴 R: X → A を持ち、これにより L がすべての有限コリミットを保つことが示唆されます。また、これにより、X のオブジェクト x と頂点 R(x) を持つ A のコスパンから構造コスパンをより便利に構築することができます。
