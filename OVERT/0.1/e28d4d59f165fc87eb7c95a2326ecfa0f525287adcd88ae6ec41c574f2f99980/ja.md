```
overapprox(expr,
              range_dict::Dict{Symbol, Array{T, 1}} where {T <: Real};
              N::Integer=2,
              ϵ::Real=1e-2)
-> OverApproximation
```

n次元関数を関係抽象を使用して過大評価します。

Nは、定数凸性の領域ごとに使用される線形セグメントの数です。この数を自動的に選択させるには、Nを-1に設定します。関数と境界の間の最大偏差が相対誤差許容範囲内になるまでセグメントを追加します。

ϵは、境界が関数の最も近い接触点で接触しないように境界に追加される相対的な「エアギャップ」です。関数の境界が範囲[a,b]にある場合、サイズϵ*(b-a)のギャップが上限のすべての点のy値に追加され、下限のすべての点からは減算されます。デフォルト値は1%またはϵ = 0.01です。
