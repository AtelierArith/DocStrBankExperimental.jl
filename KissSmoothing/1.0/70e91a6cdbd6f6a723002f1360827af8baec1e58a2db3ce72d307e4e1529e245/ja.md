```
denoise(V::Array; factor=1.0, rtol=1e-12, dims=ndims(V), verbose = false)
```

`V` に従ってデータを平滑化します:

```
`V` : 平滑化するデータ

`factor` : 平滑化の強度

`rtol` : 平滑化の強度がどれだけ正確に決定されるかの相対許容誤差

`dims` : 平滑化される配列の次元

`verbose` : 内部情報の一部を印刷することを有効にします
```

タプル (S,N) を返します:

```
`S` : 平滑化されたデータ

`N` : 抽出されたノイズ
```

特に `S + N` は元のデータ `V` を再構成します。
