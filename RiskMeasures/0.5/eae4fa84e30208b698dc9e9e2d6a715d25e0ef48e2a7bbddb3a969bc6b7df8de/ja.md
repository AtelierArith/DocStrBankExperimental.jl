```
VaR(values, pmf, α; ...)
```

離散確率変数のVaRを`values`と確率質量関数`pmf`を用いて計算します。また、リスクを達成するインデックスも計算します。

もし`α = 0`の場合、VaRは本質的下限を返し、`α = 1`の場合、VaRは最大可能値を返します。なぜなら、VaR_1は無限大だからです。

これは$n \log(n)$の時間で実行されます。ここで、`n = length(x̃)`です。

# 戻り値

VaRの`value`とそれを達成する`index`を持つ名前付きタプルを返します。そのようなインデックスが存在しない場合は、-1を返します。

  * `check_inputs=true`: 入力が有効であることを確認します。
  * `fast=false`: 線形時間の実験的実装を使用します。
