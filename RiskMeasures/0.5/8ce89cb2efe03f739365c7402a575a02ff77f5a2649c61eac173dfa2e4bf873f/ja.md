```
expectile(x̃, α)
```

確率変数 `x̃` の期待値リスク測定をリスクレベル `α ∈ (0,1)` で計算します。`α = 1/2` のとき、関数は期待値を計算します。

```
expectile(values, pmf, α; ...)
```

確率質量関数 `pmf` と `values` を持つ離散確率変数の期待値を計算します。

期待値は `α ∈ (0,1/2]` のときのみ一貫性があります。

`α` の範囲には `0` または `1` は含まれません。

この関数は次を解きます。

$$
\argmin_{x ∈ Real} α \mathbb{E}((x̃ - x)^2_+) - (1-α) \mathbb{E}((x̃ - x)^2_-)
$$
