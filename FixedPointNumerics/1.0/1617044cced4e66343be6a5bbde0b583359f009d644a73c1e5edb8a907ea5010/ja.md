```
FixedPoint(v,p)
```

整数値(v)と整数小数点(p)を含む可変構造体を作成します。

値は常に整数です。p（精度）値は、表現される小数点以下の桁数を決定します。

vが浮動小数点型の場合、数はp桁に丸められ、その後10^pにスケーリングされます。

# 例

```
x = FixedPoint(4125,2)
println(x)
41.25
println(x * 2)
82.50 
```

すべての数学演算子がサポートされており、ほとんどの数学関数もサポートされています。数学関数を呼び出すと、数はFloatに変換され、その後FixedPoint型に戻されます。桁数は保持されます。異なる精度の型が混在する場合、すべてのオペランドと結果は、すべてのオペランドの最大精度に拡張されます。方程式や数学関数の戻り値の型は、ほぼ常にFixedPoint型の値になります。
