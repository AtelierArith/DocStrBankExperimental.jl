```
ShiftInvariantWaveletTransformNode{N, T₁<:Integer, T₂<:AbstractFloat}
```

SIWTノードのインデックス、係数、およびコスト値を保持するデータ構造。

# パラメータ

  * `Depth::T₁`: 現在のノードの深さ。ルートノードの深さは0。
  * `IndexAtDepth::T₁`: 現在の深さでのノードインデックス。インデックスは各深さで0から始まる。
  * `TransformShift::T₁`: 変換を計算する前に親ノードに対して行われるシフトのタイプ。受け入れられるシフト値のタイプは次のとおりです：

      * `0`: 親ノードの係数は変換前にシフトされません。
      * `1`: 1Dおよび2D信号の両方に対して、親ノードの係数は1インデックス左に円形シフトされます。
      * `2`: 2D信号に対して、親ノードの係数は1インデックス下から上に円形シフトされます。1D信号には利用できません。
      * `3`: 2D信号に対して、親ノードの係数は1インデックス下から上および右から左に円形シフトされます。1D信号には利用できません。
  * `Cost::T₂`: 現在のノードの[`ShannonEntropyCost`](@ref)。
  * `Value::Array{T₂,N}`: 現在のノードの係数。
