```
CoalCCSRetrofit(;kwargs...) <: Retrofit
```

`CoalCCSRetrofit` は、石炭燃焼プラント（gentype="coal"）を炭素捕集技術を持つように変更し、（gentype="coal*ccsus*retrofit"）に変更するための [`Retrofit`](@ref) を表します。

キーワード引数:

  * `crf = 0.115642438` - 資本回収係数（デフォルト値は12年の経済寿命を想定）
  * `capt_co2_percent = 0.9` - （0と1の間）レトロフィットによって捕集されるCO₂の割合
  * `reduce_nox_percent =  0.25` - （0と1の間）NOₓ排出量の削減割合（デフォルトは0%と50%削減の中間値）
  * `reduce_so2_percent = 0.985` - （0と1の間）SO₂排出量の削減割合（デフォルトは97%と100%削減の中間値）
  * `reduce_pm25_percent = 0.33` - （0と1の間）PM2.5排出量の削減割合（デフォルトは-4%と70%削減の中間値）
  * `econ_life = 12.0` - レトロフィットの想定経済寿命。年*シャットダウンが経済寿命の終了よりも早い場合、計画された年*シャットダウンをレトロフィットの経済寿命の終了時に移動します。

その他の要件:

  * `gen` テーブルには `heat_rate` 列が必要です
  * `gen` テーブルには `pcap_plant_avg` 列が必要であるか、各発電機が単一のプラントを表すと仮定されます。この値はコスト曲線で使用されます。

コスト調整値は、EPAスケジュール6データの回帰から得られます。
