```
create_causal_mask(seq_len::Int)
```

未来の位置に注意を向けることを防ぐ因果（自己回帰）注意マスクを作成します。これは、予測が前のトークンのみに依存することを保証するために、言語モデルで一般的に使用されます。

マスクは、位置 i が位置 j ≤ i のみに注意を向けることを保証し、対角線を含む上三角形がマスクされ（True）、下三角形がマスクされない（False）三角形パターンを作成します。

# 引数

  * `seq_len::Int`: マスクを作成するシーケンスの長さ

# 戻り値

  * 形状 (seq*len, seq*len, 1) の 3D ブール配列で、True はマスクする位置を示します

# 例

```
julia> mask = create_causal_mask(3)[:,:,1]
3×3 Matrix{Bool}:
 1  1  1  # 最初の位置は何にも注意を向けられない
 0  1  1  # 2 番目の位置は最初の位置のみに注意を向けられる
 0  0  1  # 3 番目の位置は最初と2 番目の位置に注意を向けられる
```
