```
render!(source::Source{G, A, nw}; n = 20, alpha = 0.2, point = false,
        scale = 0.2)
```

3Dシーンに光源を表すメッシュを追加します（`point = false`の場合）または光源の中心を表す一連の点を追加します（`point = true`の場合）。`point = false`の場合、各タイプの光源に対して三角形メッシュが作成され、`n`は三角形の数です（詳細は幾何学的プリミティブのドキュメントを参照してください）および`alpha`は各三角形に使用される透明度です。`point = true`の場合、光源の中心のみがレンダリングされ、その点での法線ベクトル（光線が生成される方向を表す）も表示されます。現在のバージョンでは、`point = true`は方向性光源にのみ可能です。
