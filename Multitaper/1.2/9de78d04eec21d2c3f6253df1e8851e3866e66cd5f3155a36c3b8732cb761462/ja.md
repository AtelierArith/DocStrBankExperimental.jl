```
gpss(w, k, t, f; <キーワード引数>)
```

不均一グリッド上の一般化されたプロレート球面系列

...

# 引数

## 位置引数

  * `w::Float64`: 帯域幅
  * `k::Int64`: スレピアンテーパーの数、2*bw*length(x)以下でなければならない
  * `t::Vector{Int64}`: 時間インデックスを含むベクトル
  * `f::Float64`: テーパーを計算する周波数

## キーワード引数

  * `beta::Float64 = 0.5`: 分析用ハーフ帯域幅（ナイキストレートに似ている）

...

...

# 出力

  * `lambda::Vector{Float64}` 一般化されたプロレート球面系列の濃度
  * `u::Matrix{Float64}` 系列自体を含む行列
  * `R` 一般化された固有値問題のためのコレスキー因子
