```
StatLossCov(mode::Symbol)
```

最適な推定共分散のための損失関数を指定します。`mode` は、Donoho et al. (2018) の表2（p. 1757）に指定されている `:st`、`:ent`、`:div`、`:aff`、または `:fre` のいずれかです。以下の表では、`A` と `B` はそれぞれターゲット共分散とサンプル共分散です：

| `mode` |                                            loss |                                                          解釈 |
| ------:| -----------------------------------------------:| -----------------------------------------------------------:|
|  `:st` | `st(A, B) = tr(A⁻¹ B - I) - log(det(B)/det(A))` | `N(0, A)` と `N(0, B)` の間のKLダイバージェンスを最小化します。ここで `N` は正規分布です。 |
| `:ent` |                                      `st(B, A)` |                                         マハラノビス距離の誤差を最小化します。 |
| `:div` |                           `st(A, B) + st(B, A)` |                                                             |
| `:aff` |  `0.5 * log(det(A + B) / (2 * sqrt(det(A*B))))` |                     `N(0, A)` と `N(0, B)` の間のヘリング距離を最小化します。 |
| `:fre` |                        `tr(A + B - 2sqrt(A*B))` |                                                             |
