```
NormLossCov(norm::Symbol, pivotidx::Int)
```

最適な推定共分散のための損失関数を指定します。`norm` は `:L1`、 `:L2`、または `:Linf` のいずれかであり、`pivotidx` はDonoho et al. (2018)の表1（p. 1755）で指定された1から7の整数です。以下の表では、`A` と `B` はそれぞれターゲット共分散とサンプル共分散であり、損失関数は `pivot` 列の量に対する指定されたノルムです：

| `pivotidx` |                      `pivot` |           Notes |
| ----------:| ----------------------------:| ---------------:|
|          1 |                      `A - B` |                 |
|          2 |                  `A⁻¹ - B⁻¹` |                 |
|          3 |                  `A⁻¹ B - I` | `:L1` では利用できません |
|          4 |                  `B⁻¹ A - I` | `:L1` では利用できません |
|          5 |         `A⁻¹ B + B⁻¹ A - 2I` |     サポートされていません |
|          6 |  `sqrt(A) \ B / sqrt(A) - I` |                 |
|          7 | `log(sqrt(A) \ B / sqrt(A))` |     サポートされていません |

[`StatLossCov`](@ref) も参照してください。

参考文献:     Donoho, D.L., Gavish, M. and Johnstone, I.M., 2018.     スパイク共分散モデルにおける固有値の最適なシュリンク. 統計学の年報, 46(4), p.1742.
