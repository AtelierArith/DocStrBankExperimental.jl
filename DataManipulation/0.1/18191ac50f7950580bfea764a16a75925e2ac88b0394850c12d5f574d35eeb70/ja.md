```
sr"regex"
```

型ドメインにおける「静的」正規表現。最も直接的な使用法は、`NamedTuple`のインデックス付けや、`StructArray`のような型安定なテーブルからの列の選択です。基になる値は`unstatic()`関数を使って抽出できます。

参照: `ss"substitution"`, `nest`.

# 例

```julia
nt = (a_1=1, a_2=10, b_1=100)

# 正規表現によってntのサブセットを選択:
nt[sr"a_\d"] === (a_1 = 1, a_2 = 10)

# 正規表現と置換文字列によって選択し、名前を変更:
nt[sr"a_(\d)" => ss"xxx_\1_xxx"] === (xxx_1_xxx = 1, xxx_2_xxx = 10)
```
