与えられた有理数を `1/a_1 + 1/a_2 + ...` の形の分数の和に貪欲（フィボナッチ–シルベスター）展開しますが、奇数の分母のみを使用します。

任意の有理数 `x/y` で `y` が奇数である場合、すべての分母が奇数である有限の分数の和としてこれを表すことができることが示されています。この方法は、従来の貪欲アルゴリズムよりも要素数が少ない展開を生成することがあります。

この関数は、供給された有理数の分母が偶数である場合、例外をスローします。

この関数は、展開の分母のみを返します（すなわち、`[a_1, a_2, ...]` のみ）。与えられた有理数 `r` が `r ≥ 1` を満たす場合、展開の最初の `n` 要素は 1 であり、ここで `n = floor(r)` です。与えられた有理数 `r` が `r < 0` を満たす場合、すべての返される分母は 0 より小さくなり、関係式

```
r == sum(1 .// efoddgreedy(r))
```

が常に成り立ちます。
