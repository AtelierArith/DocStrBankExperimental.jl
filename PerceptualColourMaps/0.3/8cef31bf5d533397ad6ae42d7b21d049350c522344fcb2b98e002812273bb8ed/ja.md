relief -  凹凸の影付き画像を生成します

```
Usage:  shadeimg = relief(img, azimuth, elevation, gradscale, rgbimg)

Arguments: img - 凹凸の影を付ける画像/高さマップ。 ::Array{T<:Real,2}
       azimuth - 光の方向の方位角（度）。ゼロの方位角は
                 上向きで、時計回りに増加します。デフォルトは45です。
     elevation - 光の方向の高度（度）。デフォルトは45です。
     gradscale - 表面の勾配に適用するスケーリング。影が
                 過剰な場合はスケーリングを減少させます。良い値を見つけるために
                 連続的に倍増または半減を試みてください。
        rgbimg - オプションのRGB画像で、'img'から導出された
                 影のパターンが適用されます。代わりに、rgbimgは
                 cmap()から得られた型::Array{ColorTypes.RGBA{Float64},1}
                 のカラーマップであることもできます。このカラーマップは
                 入力画像/高さマップに適用され、影のパターンが
                 適用されるRGB画像を取得します。
```

ラバルティアンシェーディングが凹凸画像を形成するために使用されます。これは、表面法線と光の方向との間の角度のコサインから得られます。影は無視されることに注意してください。したがって、近くの大きな構造物の影にある可能性のある小さな特徴は、大きな特徴が存在しないかのようにレンダリングされます。

関連情報: cmap, applycolourmap
