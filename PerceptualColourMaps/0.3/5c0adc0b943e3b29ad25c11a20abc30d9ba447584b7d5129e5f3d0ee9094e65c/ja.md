viewlabspace: L*a*b*空間の視覚化

```
Usage:    viewlabspace(L = 50, figNo = 1)

Arguments:     L - L*a*b*空間のスライスを表示する際の明度レベル
           figNo - 使用するPyPlot図
```

この関数は、L*a*b*空間における異なる明度のスライスに対応する一連の画像をインタラクティブに表示することを可能にします。明度は0から100まで変化します。最初は明度50のスライスが表示されます。

'l'または矢印の上/右を押すと、明度がdLだけ増加します。'd'または矢印の下/左を押すと、dLだけ暗くなります。'x'を押すと終了します。
