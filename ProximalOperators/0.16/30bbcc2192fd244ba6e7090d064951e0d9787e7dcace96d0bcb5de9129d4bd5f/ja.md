```
IndGraph(A)
```

行列 `A`（密または疎）に対して、そのグラフの指示関数を返します：

$$
G_A = \{(x, y) : Ax = y\}.
$$

`prox!` の評価は、LDLt（密の場合はLL）行列因子分解とバックソルブに基づく直接的な方法を使用します。

`prox!` メソッドは、入力/出力としてペア `(x, y)` に対して動作します。したがって、もし `f = IndGraph(A)` がグラフ $G_A$ の指示関数であり、`(x, y)` と `(c, d)` が同じサイズのベクトルのペアであるならば、

```
prox!((c, d), f, (x, y))
```

は `(x, y)` の $G_A$ への射影を `(c, d)` に書き込みます。
