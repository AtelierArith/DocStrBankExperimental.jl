```
Epicompose(L, f, [mu])
```

関数 $f$ と線形演算子 $L$ のエピ合成、または最小後合成または画像関数として知られるものを返します。関数 f と線形演算子 L が与えられたとき、彼らのエピ合成は次のようになります：

$$
g(y) = (Lf)(y) = inf_x { f(x) : Lx = y }
$$

これは前合成に対する双対操作です。Rockafellar と Wets の「Variational Analysis」、定理 11.23 を参照してください。

もし $mu > 0$ が指定されている場合、$L$ は $L'*L == mu*I$ であると仮定され、任意の凸 $f$ に対して近接演算子が計算可能です。$mu$ が指定されていない場合、$f$ は $Quadratic$ 型でなければなりません。
