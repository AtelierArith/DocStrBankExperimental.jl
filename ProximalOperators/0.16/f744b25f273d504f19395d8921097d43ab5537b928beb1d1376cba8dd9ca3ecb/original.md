Given `ind_S` the indicator function of a set $S$, and an optional positive parameter `位`, return the (weighted) squared Euclidean distance from $S$, that is function

$$
g(x) = \tfrac{位}{2}\mathrm{dist}_S^2(x) = \min \left\{ \tfrac{位}{2}\|y - x\|^2 : y \in S \right\}.
$$
