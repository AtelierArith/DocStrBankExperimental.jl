```
gsklf(A, E, B, C, D; disc = false, jobopt = "none", offset = sqrt(ϵ), fast = true, atol1 = 0, atol2 = 0, rtol, 
     withQ = true, withZ = true) -> (F, G, Q, Z, dimsc, nmszer, nizer)
```

構造行列ペンシル `M - λN` を削減します。

```
           | A-λE | B | 
  M - λN = |------|---|
           |  C   | D |
```

`n x n` ペンシル `A-λE` が正則であり、`[E B]` がフル行ランク `n` を持つ場合、等価な形 `F - λG = diag(Q',I)*(M - λN)*Z` に削減します。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用し、変換された行列 `F` と `G` は以下の特別な行の分割を保持するクロンカー様式の形になります。

```
            | Ar-λEr    *     *   *   |
            |   0     Al-λEl  Bl  *   |
   F - λG = |   0       0     0   Bn  | ,    
            |-------------------------|
            |   0       Cl    Dl  *   |
```

ここで: (1) `Ar-λEr` は `mr x nr` フル行ランクペンシルで、`M - λN` の右クロンカー構造と、キーワード引数 `disc` と `jobopt` によって定義される複素平面の関心領域 `Cb` の外にあるゼロの部分を含みます; (2) `Al-λEl` は `nl x nl` 正則正方形ペンシルで、`El` は可逆で上三角です; (3) `Bl` は `nl x ml` 行列で、`ml` は伝達関数行列 `C*inv(λE-A)*B+D` の通常のランクです; (4) `Bn` は順序 `nsinf` の可逆行列です。

列の次元 `(nr, nl, ml, nsinf)` は `dimsc` に返されます。

フル列ランクのサブペンシル 

```
            | Al-λEl | Bl | 
Ml - λNl := |--------|----|
            |   Cl   | Dl |
```

は `M - λN` の左クロンカー構造と、関心領域 `Cb` にあるゼロを含み、`rank [Al-λEl Bl] = nl` の性質を持ちます。これは、すべての有限 `λ ∈ Cb` に対して `rank [A-λE B] = n` が成り立つ場合です（`Cb` に関する有限安定性）。もし利用可能であれば、`Cb` の境界にある有限ゼロの数は `nmszer` に返されます。そうでなければ、`nmszer` は `missing` に設定されます。`disc = false` の場合、もし利用可能であれば無限ゼロの数は `nizer` に返されます。そうでなければ、`nizer` は `missing` に設定されます。`disc = true` の場合、`nizer` は 0 に設定されます。境界オフセット `β` は、`Cb` の境界にあるゼロを評価するために使用され、キーワードパラメータ `offset` によって指定されます。したがって、`disc = false` の場合、`Cb` の境界には実部が区間 `[-β,β]` にある複素数が含まれ、`disc = true` の場合、`Cb` の境界には絶対値が区間 `[1-β,1+β]` にある複素数が含まれます。`β` に使用されるデフォルト値は `sqrt(ϵ)` です（`A` の要素型の機械精度）。

ペンシル `Ml - λNl` のゼロの関心領域 `Cb` は以下のように定義されます：

`jobopt = none` の場合、`Cb` は空集合であり、`M - λN` のゼロは `Ml - λNl` に含まれません;

`jobopt = all` の場合、`Cb` は拡張複素平面（無限を含む）であり、`M - λN` のすべてのゼロが `Ml - λNl` に含まれます;

`jobopt = finite` の場合、`Cb` は複素平面（無限を含まない）であり、`M - λN` のすべての有限ゼロが `Ml - λNl` に含まれます;

`jobopt = infinite` の場合、`Cb` は無限の点であり、`M - λN` のすべての無限ゼロが `Ml - λNl` に含まれます;

`jobopt = stable` の場合、`disc = false` のとき、`Cb` は実部が `-β` より小さい複素数の集合であり、`disc = true` のとき、`Cb` は絶対値が `1-β` より小さい複素数の集合であり、`Cb` のすべての有限ゼロが `Ml - λNl` に含まれます;

`jobopt = unstable` の場合、`disc = false` のとき、`Cb` は実部が `β` より大きいまたは無限の複素数の集合であり、`disc = true` のとき、`Cb` は絶対値が `1+β` より大きいまたは無限の複素数の集合であり、`Cb` のすべての有限および無限ゼロが `Ml - λNl` に含まれます;

`jobopt = s-unstable` の場合、`disc = false` のとき、`Cb` は実部が `-β` より大きいまたは無限の複素数の集合であり、`disc = true` のとき、`Cb` は絶対値が `1-β` より大きいまたは無限の複素数の集合であり、`Cb` のすべての有限および無限ゼロが `Ml - λNl` に含まれます。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。削減は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左の直交またはユニタリ変換は、`withQ = true` の場合、`n x n` 行列 `Q` に蓄積されます。そうでなければ、`Q` は `nothing` に設定されます。実行された右の直交またはユニタリ変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでなければ、`Z` は `nothing` に設定されます。

方法: [1] の削減アルゴリズムは、いくつかのゼロ選択オプションに対処するように適応されています。
