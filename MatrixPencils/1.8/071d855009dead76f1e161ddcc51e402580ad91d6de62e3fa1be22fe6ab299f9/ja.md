```
pmzeros(P; CF1, fast = false, atol::Real = 0, rtol::Real = atol>0 ? 0 : n*ϵ) -> (val, iz, KRInfo)
```

多項式行列 `P(λ)` の有限および無限のゼロを `val` に、無限のゼロの重複度を `iz` に、`P(λ)` のクロンネッカー構造に関する情報を `KRInfo` オブジェクトに返します。ゼロとクロンネッカー構造の計算は、`CF1 = true` の場合は最初の伴随形式で、`CF1 = false` の場合は第二の伴随形式で、`P(λ)` の強い線形化を用いて行われます。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形の次数 `k` の多項式行列として指定でき、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納され、`P[:,:,i]` には `i` 番目の係数行列 `P_i`（`λ**(i-1)` を掛けたもの）が含まれます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

クロンネッカー構造に関する情報は、右クロンネッカーインデックス `rki`、左クロンネッカーインデックス `lki`、無限の基本除数 `id`、有限の固有値の数 `nf`、通常のランク `nrank` で構成され、`KRInfo` から `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` として取得できます。詳細については、[`KRInfo`](@ref) を参照してください。

ゼロの計算は、多項式行列 `P(λ)` の伴随形式に基づく線形化 `M-λN` を構築し、その後、クロンネッカー構造に関する情報を示す適切なクロンネッカー様形式（KLF）にペンシル `M-λN` を縮約することによって行われます。`P(λ)` の左および右クロンネッカーインデックスはそれぞれ `KRInfo.rki` と `KRInfo.rki` に返され、その値は [1] の結果を使用して `M-λN` の左および右クロンネッカーインデックスから回収されます。`P(λ)` の無限のゼロの重複度は `iz` に返され、これは `M-λN` の無限の固有値の重複度と `P(λ)` の次数の間の正の差です [2]。`val` の有限ゼロの数は `M-λN` の有限固有値の数（`KRInfo.nf` に返される）に等しく、`val` の無限ゼロの数は `iz` の重複度の合計です。

`M-λN` を KLF に縮約するには、直交類似変換を使用し、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を含み、`fast = false` の場合はより信頼性の高いSVD分解を使用します。効率のために、縮約は部分的にのみ行われ、実行された直交変換は蓄積されません。

キーワード引数 `atol` と `rtol` は、それぞれ `P(λ)` の非ゼロ係数の絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` で、ここで `n` は `P(λ)` の最小次元のサイズであり、`ϵ` は `P(λ)` の係数の要素型の機械イプシロンです。

[1] F. De Terán, F. M. Dopico, D. S. Mackey, 多項式行列のスペクトル同値性とインデックス和定理, 線形代数とその応用, vol. 459, pp. 264-333, 2014.

[2] A. Varga, Juliaを使用した多項式行列のクロンネッカー構造の計算について, 2020年6月, [arXiv:2006.06825](https://arxiv.org/pdf/2006.06825).
