```
sklf_right!(A, B, C; fast = true, atol1 = 0, atol2 = 0, rtol, withQ = true) -> (Q, νr, nc, nuc)
```

部分行ランク行列ペンシルを減少させます 

```
  M - λN = | B | A-λI | n
             m    n
```

等価な形 `F - λG = Q'*(M - λN)*diag(I,Q)` に変換するために、直交またはユニタリ変換行列 `Q` を使用します。ここで `M - λN` は、右側および有限のクロンカー構造（制御可能性階段形式とも呼ばれる）を示すクロンカーのような形 `| Bt | At-λI |` に変換されます：

```
                    |  Bc | Ac-λI     *    | nc
  | Bt | At-λI | =  |-----|----------------|
                    |  0  |  0     Auc-λI  | nuc
                       m     nc      nuc
```

`Bt = Q'*B` および `At = Q'*A*Q` はそれぞれ `B` と `A` に返され、`C*Q` は `C` に返されます（`C = missing` でない限り）。                 

部分ペンシル `| Bc | Ac-λI |` は完全行ランク `nc` を持ち、階段形式になっており、`M - λN` の右側のクロンカーインデックスを含んでいます。 `nr` 次元ベクトル `νr` は、階段形式 `| Bc | Ac-λI |` のブロックの行と列の次元を含み、`i` 番目のブロックは次元 `νr[i] x νr[i-1]` を持ち（`νr[0] = m`）、完全行ランクを持ちます。 差 `νr[i-1]-νr[i]` は、`i = 1, 2, ..., nr` の場合のサイズ `(i-1) x i` の基本クロンカー ブロックの数です。

`nuc x nuc` 行列 `Auc` は、`M - λN` の（有限の）固有値を含みます（`A` の制御不能な固有値とも呼ばれます）。  

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`B` の非ゼロ要素の絶対許容誤差、および `A` と `B` の非ゼロ要素の相対許容誤差を指定します。 減少は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ変換は、`withQ = true` の場合に行列 `Q` に蓄積されます。 そうでない場合、`Q` は `nothing` に設定されます。
