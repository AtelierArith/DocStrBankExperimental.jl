```
pm2lpCF2(P; grade = l) -> (M, N)
```

多項式行列 `P(λ)` の第二伴随フロベニウス形式の強い線形化 `M - λN` を構築します。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形を持つ、次数 `k` の多項式行列と仮定されます。係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納されており、`P[:,:,i]` には `λ**(i-1)` を掛けた `i` 番目の係数行列 `P_i` が含まれています。線形化に使用する有効次数 `l` は、キーワード引数 `grade` を介して `grade = l` として指定できます。ここで、`l` は `P(λ)` の次数以上である必要があります。デフォルト値として使用される `l` は `l = deg(P(λ))` です。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

`P(λ)` が有効次数 `l` および次数 `d` の `m x n` 多項式行列である場合、得られる行列ペンシル `M - λN` は以下の条件を満たします [1]:

(1) `M - λN` の次元は `l*m x (n+(l-1)*m)` であり、`P(λ)` が正則であれば `M - λN` も正則です;

(2) `M - λN` と `P(λ)` は同じ有限固有値を持ちます;

(3) `M - λN` の無限固有値の部分重複度は `l-d` だけ `P(λ)` の無限固有値の部分重複度を超えます;

(4) `M - λN` と `P(λ)` は同じ右クロンカー構造（すなわち、同じ右クロンカー指数）を持ちます;

(5) `M - λN` と `P(λ)` は同じ数の左クロンカー指数を持ち、`M - λN` の左クロンカー指数は `P(λ)` の左クロンカー指数を `l-1` だけ超えます。

[1] F. De Terán, F. M. Dopico, D. S. Mackey, Spectral equivalence of polynomial matrices and the Index Sum Theorem, Linear Algebra and Its Applications, vol. 459, pp. 264-333, 2014.
