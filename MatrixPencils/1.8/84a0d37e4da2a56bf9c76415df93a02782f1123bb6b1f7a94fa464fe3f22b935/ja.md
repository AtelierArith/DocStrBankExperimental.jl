```
klf_right!(M, N; fast = true, roff = 0, coff = 0, rtrail = 0, ctrail = 0, atol = 0, rtol, withQ = true, withZ = true) -> (Q, Z, νr, μr, nf, ν, μ, tol)
```

分割された行列ペンシル `M - λN` を次のように減少させます（`*` は関連しないサブペンシルを表します）

```
          [   *         *        *  ] roff
 M - λN = [   0      M22-λN22    *  ] m
          [   0         0        *  ] rtrail
            coff        n     ctrail
```

等価な形 `F - λG = Q1'*(M - λN)*Z1` に変換します。ここで、直交またはユニタリ変換行列 `Q1` と `Z1` を使用し、サブペンシル `M22 - λN22` が次のようなクロンカー様式に変換され、右および有限のクロンカー構造を示します：

```
               |  Mr-λNr    |     *      |    *     |
               |------------|------------|----------|
 F22 - λG22 =  |    O       |   Mf-λNf   |    *     |
               |------------|------------|----------|
               |    O       |     0      | Mil-λNil |
```

`F` と `G` はそれぞれ `M` と `N` に返されます。                

完全行ランクペンシル `Mr-λNr` は階段形式で、サブペンシル `M22 - λN22` の右クロンカーインデックスを含み、次のような形を持ちます。

```
     Mr-λNr  = | Br | Ar-λEr |,
```

ここで `Er` は上三角で非特異です。 `nr` 次元のベクトル `νr` と `μr` は、階段形式 `Mr-λNr` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `νr[i] x μr[i]` を持ち、完全行ランクです。 差 `μr[i]-νr[i]` は `i = 1, 2, ..., nr` の場合、サイズ `(i-1) x i` の基本クロンカーブロックの数です。

`nf x nf` ペンシル `Mf-λNf` は正則で、サブペンシル `M22 - λN22` の有限基本因子を含みます。 `Nf` は上三角で非特異です。

完全列ランクペンシル `Mil-λNil` は階段形式で、サブペンシル `M22 - λN22` の左クロンカーインデックスと無限基本因子を含みます。 `nb` 次元のベクトル `ν` と `μ` は、階段形式 `Mil-λNil` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `ν[nb-i+1] x μ[nb-i+1]` を持ち、完全列ランクです。 差 `ν[nb-i+1]-μ[nb-i+1]` は `i = 1, 2, ..., nb` の場合、サイズ `i x (i-1)` の基本クロンカーのブロックの数です。 差 `μ[nb-i+1]-ν[nb-i]` は `i = 1, 2, ..., nb` の場合、次数 `i` の無限基本因子の数です（`ν[0] = 0`）。

キーワード引数 `atol` と `rtol` は、`M` の非ゼロ要素に対する絶対および相対許容誤差を指定します。 `M` の非ゼロ要素に対する内部的に使用される絶対許容誤差は `tol` に返されます。 減少は、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左の直交またはユニタリ変換は、行列 `Q` に蓄積されます（すなわち、`Q <- Q*Q1`） `withQ = true` の場合。 そうでない場合、`Q` は変更されません。 実行された右の直交またはユニタリ変換は、行列 `Z` に蓄積されます（すなわち、`Z <- Z*Z1`） `withZ = true` の場合。 そうでない場合、`Z` は変更されません。   

`注意:` サブペンシル `M22 - λN22` が完全行ランクである場合、正則ペンシル `Mil-λNil` は、正方形の上三角対角ブロックを持つ階段形式になります（すなわち、`μ[i] = ν[i]`）、および差 `ν[nb-i+1]-ν[nb-i]` は `i = 1, 2, ..., nb` の場合、次数 `i` の無限基本因子の数です（`ν[0] = 0`）。 ```
