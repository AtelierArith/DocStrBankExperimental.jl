```
pmdivrem(N,D) -> (Q, R)
```

`N(λ)`と`D(λ)`の要素ごとの多項式除算`N(λ)./D(λ)`の商を`Q(λ)`に、余りを`R(λ)`に計算します。

`N(λ)`は、`N(λ) = N_1 + λ N_2 + ... + λ**k N_(k+1)`の形をした多項式行列であり、係数行列`N_i`（`i = 1, ..., k+1`）は3次元行列`N`に格納されており、`N[:,:,i]`には`λ**(i-1)`を掛けた`i`-th係数行列`N_i`が含まれています。

`D(λ)`は、`D(λ) = D_1 + λ D_2 + ... + λ**l D_(l+1)`の形をした多項式行列であり、係数行列`D_i`（`i = 1, ..., l+1`）は3次元行列`D`に格納されており、`D[:,:,i]`には`λ**(i-1)`を掛けた`i`-th係数行列`D_i`が含まれています。

また、`N(λ)`と`D(λ)`は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl)パッケージによって提供される`Polynomial`型の要素の行列として指定することもできます。

商の多項式行列`Q(λ)`と余りの多項式行列`R(λ)`は、それぞれ3次元行列`Q`と`R`に格納されており、`Q[:,:,i]`と`R[:,:,i]`には`λ**(i-1)`を掛けた`i`-th係数行列が含まれています。
