```
klf_left_refine!(ν, μ, M, N, tol; fast = true, ut = false, roff = 0, coff = 0, rtrail = 0, ctrail = 0, withQ = true, withZ = true) -> (νi, νl, μl)
```

分割された行列ペンシル `M - λN` を次のように縮小します（`*` は関連しないサブペンシルを示します）

```
         [   *         *        *  ] roff
M - λN = [   0      Mil-λNil    *  ] mli
         [   0         0        *  ] rtrail
           coff      nli     ctrail
```

直交またはユニタリ変換行列 `Q1` と `Z1` を使用して、同等の形 `F - λG = Q1'*(M - λN)*Z1` に変換します。このとき、フルコラムランクサブペンシル `Mil-λNil` は、無限および左クロンカー構造を示す次のクロンカーのような形に変換されます：

```
              |  Mi-λNi    |     *      |
 Fil - λGil = |------------|------------|
              |    O       |   Ml-λNl   |
```

フルコラムランクペンシル `Mil-λNil` は階段形式であり、`nb` 次元のベクトル `ν` と `μ` は、階段形式 `Mil-λNil` のブロックの行および列の次元を含みます。ここで、`i` 番目のブロックは次元 `ν[i] x μ[i]` を持ち、フルコラムランクです。行列 Mil はフルコラムランクを持ち、`Nil` の先頭 `μ[1]+...+μ[nb-1]` 列はフル行ランクサブマトリックスを形成します。

通常のペンシル `Mi-λNi` は階段形式であり、`Mil-λNil` の無限の基本因子を含み、`Mi` は `ut = true` の場合は上三角行列であり、非特異で、`Ni` は上三角行列であり、ニルポテントです。`nbi` 次元のベクトル `νi` は、階段形式 `Mi-λNi` の正方形対角ブロックの次元を含み、`i` 番目のブロックは次元 `νi[i] x νi[i]` を持ちます。差 `νi[i]-νi[i-1] = ν[nbi-i+1]-μ[nbi-i+1]` は、`i = 1, 2, ..., nbi` の場合の無限の基本因子の数です（`νi[0] = 0` および `μ[nbi+1] = 0`）。

フルコラムランクペンシル `Mil-λNil` は階段形式であり、サブペンシル `M22 - λN22` の左クロンカーインデックスと無限の基本因子を含みます。`nb` 次元のベクトル `ν` と `μ` は、階段形式 `Mil-λNil` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `ν[nb-i+1] x μ[nb-i+1]` を持ち、フルコラムランクです。差 `ν[nb-i+1]-μ[nb-i+1]` は、`i = 1, 2, ..., nb` の場合のサイズ `i x (i-1)` の基本クロンカーブロックの数です。差 `μ[nb-i+1]-ν[nb-i]` は、`i = 1, 2, ..., nb` の場合の無限の基本因子の数です（`ν[0] = 0`）。

フルコラムランクペンシル `Ml-λNl` は階段形式であり、`M-λN` の左クロンカーインデックスを含み、次の形を持ちます。

```
           | Al-λEl |
 Ml-λNl  = |--------|,
           |   Cl   |
```

ここで `El` は上三角行列であり、非特異です。`nl` 次元のベクトル `νl` と `μl` は、階段形式 `Ml-λNl` のブロックの行および列の次元を含み、`j` 番目のブロックは次元 `νl[j] x μl[j]` を持ち、フルコラムランクです。差 `νl[nl-j+1]-μl[nl-j+1] = ν[nl-j+1]-μ[nl-j+1]` は、`j = 1, 2, ..., nl` の場合のサイズ `j x (j-1)` の基本クロンカーブロックの数です。`ut = true` の場合、`Ml` のフルコラムランク対角ブロックは、上三角で非特異な形 `[Y; 0]` に縮小され、`Nl` のフル行ランク超対角ブロックは、上三角で非特異な形 `[0 Y]` に縮小されます。

`F` と `G` はそれぞれ `M` と `N` に返されます。

実行された左の直交またはユニタリ変換は、`withQ = true` の場合、行列 `Q` に蓄積されます（すなわち、`Q <- Q*Q1`）。そうでない場合、`Q` は変更されません。実行された右の直交またはユニタリ変換は、`withZ = true` の場合、行列 `Z` に蓄積されます（すなわち、`Z <- Z*Z1`）。そうでない場合、`Z` は変更されません。
