```
 lpmfd2ls(DEN, NUM; fast = true, contr = false, obs = false, minimal = false, atol = 0, rtol) -> (A, E, B, C, D)
```

構造化線形化をシステム行列 `S(λ)` の形で構築します。

```
        | A-λE | B | 
 S(λ) = |------|---|
        |  C   | D |
```

左の多項式行列分数表現 `R(λ) = inv(DEN(λ))*NUM(λ)` のために、`R(λ) = C*inv(λE-A)*B+D` となるようにします。結果として得られる線形化 `S(λ)` は、`minimal = false` の場合は一部を保持し、`minimal = true` の場合は `R(λ)` の完全なクロンカー構造を保持します。この場合、`A-λE` の次数 `n` は可能な限り最小のものになります。

`DEN(λ)` と `NUM(λ)` は、`X(λ) = X_1 + λ X_2 + ... + λ**k X_(k+1)` の形の多項式行列として指定できます。ここで、`X = DEN` または `X = NUM` の場合、係数行列 `X_i`（`i = 1, ..., k+1`）は3次元行列 `X` に格納され、`X[:,:,i]` には `λ**(i-1)` を掛けた `i` 番目の係数行列 `X_i` が含まれます。

`DEN(λ)` と `NUM(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。この場合、`N(λ)` と `D(λ)` が同じ不確定要素を持つかどうかのチェックは行われません。

計算された構造化線形化は以下を満たします：

(1) `A-λE` は正則です；

(2) `rank[B A-λE] = n` （制御可能性）もし `minimal = true` または `contr = true` の場合、この場合右のクロンカー構造が保持されます；

(3) `rank[A-λE; C] = n` （可観測性）もし `minimal = true` または `obs = true` の場合、この場合左のクロンカー構造が保持されます；

(4) `S(λ)` は `minimal = true` の場合、単純な無限固有値を持ちません。

条件 (1)-(4) が満たされる場合、線形化は `minimal` と呼ばれ、結果として得られる次数 `n` は達成可能な最小の次数です。条件 (1)-(3) が満たされる場合、線形化は `irreducible` と呼ばれ、結果として得られる次数 `n` は直交類似変換を使用して達成可能な最小の次数です。不可約線形化 `S(λ)` は、`R(λ)` の極-ゼロ構造（有限および無限）と左および右のクロンカー構造を保持します。

キーワード引数 `atol` と `rtol` は、それぞれ行列 `DEN(λ)` と `NUM(λ)` の非ゼロ係数に対する絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `nt*ϵ` であり、ここで `nt` は正方行列 `DEN(λ)` のサイズで、`ϵ` はその係数の要素型のマシンイプシロンです。

構造化線形化は、[1] に記載された方法を使用して構築されます。

[1] A. Varga, On computing the Kronecker structure of polynomial and rational matrices using Julia, 2020,  [arXiv:2006.06825](https://arxiv.org/pdf/2006.06825). ```
