```
 pm2lps(P; contr = false, obs = false) -> (A, E, B, F, C, G, D, H)
```

構造化された線形化をシステム行列 `S(λ)` として次の形式で構築します。

```
        | A-λE | B-λF | 
 S(λ) = |------|------|
        | C-λG | D-λH |
```

多項式行列 `P(λ)` のクロンカー構造の一部を保持します。

`P(λ)` は、次の形式の次数 `k` の多項式行列として指定できます。 `P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` であり、係数行列 `P_i` は `i = 1, ..., k+1` の範囲で、3次元行列 `P` に格納され、`P[:,:,i]` には `i` 番目の係数行列 `P_i` (λ**(i-1) を掛けたもの) が含まれます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

もし `d` が `p x m` 多項式行列 `P(λ)` の次数であるなら、計算された線形化は次の条件を満たします：

(1) `A-λE` は `n x n` の正則ペンシルであり、ここで `n = p(d-1)` もし `contr = false` かつ `p <= m` の場合、そうでなければ `n = m(d-1)` です； 

(2) `P(λ) = (C-λG)*inv(λE-A)*(B-λF)+D-λH`；

(3) `rank[B-λF A-λE] = n` すべての有限および無限の `λ` に対して (強い制御可能性) もし `contr = true` の場合、このとき右のクロンカー構造が保持されます；

(4) `rank[A-λE; C-λG] = n` すべての有限および無限の `λ` に対して (強い可観測性) もし `obs = true` の場合、このとき左のクロンカー構造が保持されます。

条件 (1)-(4) が満たされる場合、線形化は `強い最小` と呼ばれ、得られる次数 `n` は達成可能な最小の次数であり、`S(λ)` は `P(λ)` の極-零構造 (有限および無限) と左および右のクロンカー構造を保持します。

ペンシルに基づく線形化は、[1] に記載された方法を使用して構築されます。

[1] A. Varga, On computing the Kronecker structure of polynomial and rational matrices using Julia, 2020,  [arXiv:2006.06825](https://arxiv.org/pdf/2006.06825).
