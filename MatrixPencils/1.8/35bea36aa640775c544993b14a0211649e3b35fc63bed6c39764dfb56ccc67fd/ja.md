```
pzeros(M, N; fast = false, atol1::Real = 0, atol2::Real = 0, rtol::Real=min(atol1,atol2)>0 ? 0 : n*ϵ) -> (val, iz, KRInfo)
```

行列ペンシル `M-λN` の有限および無限のゼロを `val` に返し、無限のゼロの重複度を `iz` に、`KRInfo` オブジェクト内のクロンカー構造に関する情報を返します。

無限のゼロの重複度に関する情報はベクトル `iz` に提供されており、各 `i` 番目の要素 `iz[i]` は `k-1` に等しく、ここで `k` は無限の基本因子の次数で `k > 0` です。 `val` に含まれる無限のゼロの数は `iz` の成分の合計です。

クロンカー構造に関する情報は、右クロンカーインデックス `rki`、左クロンカーインデックス `lki`、無限の基本因子 `id`、有限の固有値の数 `nf`、および通常のランク `nrank` で構成され、`KRInfo` からそれぞれ `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` として取得できます。詳細については、[`pkstruct`](@ref) を参照してください。

ゼロの計算は、ペンシル `M-λN` の無限および有限の固有値構造の分割を示す適切なクロンカー様式 (KLF) にペンシルを縮約することによって行われます。縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を含み、`fast = false` の場合はより信頼性の高いSVD分解を使用します。効率のために、縮約は部分的にのみ行われ、実行された直交変換は蓄積されません。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、`M` および `N` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `M` の最小次元のサイズで、`ϵ` は `M` の要素型の機械イプシロンです。
