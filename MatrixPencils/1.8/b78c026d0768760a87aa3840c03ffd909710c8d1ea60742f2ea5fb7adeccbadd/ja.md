```
klf_leftinf(M, N; fast = true, ut = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) 
            -> (F, G, Q, Z, n, m, νi, νl, μl)
```

行列ペンシル `M - λN` を、直交またはユニタリ変換行列 `Q` と `Z` を用いて、同等の形 `F - λG = Q'(M - λN)Z` に還元します。このとき、変換された行列 `F` と `G` は、無限および左クロンカー構造を示す以下のクロンカー様式にあります：

```
               |  Mrf-λNrf  |     *      |    *    |
               |------------|------------|---------|
     F - λG =  |    O       |   Mi-λNi   |    *    |
               |------------|------------|---------|
               |    O       |     0      |  Ml-λNl |
```

完全行ランクペンシル `Mrf-λNrf` は、ペンシル `M-λN` の右クロンカーおよび有限クロンカー構造を含み、標準形 

```
      Mrf-λNrf = | B  | A-λE | ,
```

で表されます。ここで `E` は `nxn` の非特異上三角行列であり、`A` と `B` はそれぞれ `nxn` および `nxm` 次元の行列です。

通常のペンシル `Mi-λNi` は階段形式であり、`M-λN` の無限の基本因子を含み、`Mi` は `ut = true` の場合に上三角で非特異であり、`Ni` は上三角でニルポテントです。次元 `ni` のベクトル `νi` は、階段形式 `Mi-λNi` の正方対角ブロックの次元を含み、`i` 番目のブロックは次元 `νi[i] x νi[i]` を持ちます。差 `νi[i]-νi[i-1] = ν[ni-i+1]-μ[ni-i+1]` は、次数 `i` の無限基本因子の数です（`νi[0] = 0` および `μ[nb+1] = 0`）。

完全列ランクペンシル `Ml-λNl` は階段形式であり、`M-λN` の左クロンカー指標を含み、次の形を持ちます：

```
               | Al-λEl |
     Ml-λNl  = |--------|,
               |   Cl   |
```

ここで `El` は上三角で非特異です。

次元 `nl` のベクトル `νl` と `μl` は、階段形式 `Ml-λNl` のブロックの行および列の次元を含み、`j` 番目のブロックは次元 `νl[j] x μl[j]` を持ち、完全列ランクです。差 `νl[nl-j+1]-μl[nl-j+1]` は、サイズ `j x (j-1)` の基本クロンカーブロックの数です。`ut = true` の場合、`Ml` の完全列ランク対角ブロックは、上三角で非特異な `X` を持つ形 `[X; 0]` に還元され、`Nl` の完全行ランク超対角ブロックは、上三角で非特異な `Y` を持つ形 `[0 Y]` に還元されます。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。還元は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

実行された左の直交またはユニタリ変換は、`withQ = true` の場合に行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された右の直交またはユニタリ変換は、`withZ = true` の場合に行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。
