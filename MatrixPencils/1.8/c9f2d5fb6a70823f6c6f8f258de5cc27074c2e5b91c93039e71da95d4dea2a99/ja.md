```
klf_rightinf(M, N; fast = true, ut = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) 
            -> (F, G, Q, Z, νr, μr, νi, n, p)
```

行列ペンシル `M - λN` を、直交またはユニタリ変換行列 `Q` と `Z` を使用して、同等の形 `F - λG = Q'(M - λN)Z` に還元します。変換された行列 `F` と `G` は、無限および左クロンカー構造を示す以下のクロンカー様式にあります：

```
               |  Mr-λNr  |     *      |    *     |
               |----------|------------|----------|
     F - λG =  |    O     |   Mi-λNi   |    *     |
               |----------|------------|----------|
               |    O     |     0      | Mfl-λNfl |
```

完全行ランクペンシル `Mr-λNr` は階段形式であり、ペンシル `M-λN` の右クロンカーインデックスを含み、次の形を持ちます：

```
     Mr-λNr  = | Br | Ar-λEr |,
```

ここで `Er` は上三角で非特異です。次元 `nr` のベクトル `νr` と `μr` は、階段形式 `Mr-λNr` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `νr[i] x μr[i]` を持ち、完全行ランクです。`μr[i]-νr[i]` の差は、サイズ `(i-1) x i` の基本クロンカー ブロックの数です。`ut = true` の場合、`Mr` の完全行ランク対角ブロックは、`X` が上三角で非特異な形 `[0 X]` に還元され、`Nr` の完全列ランク超対角ブロックは、`Y` が上三角で非特異な形 `[Y; 0]` に還元されます。

通常のペンシル `Mi-λNi` は階段形式であり、`M-λN` の無限基本因子を含み、`Mi` は `ut = true` の場合上三角で非特異であり、`Ni` は上三角でニルポテントです。次元 `ni` のベクトル `νi` は、階段形式 `Mi-λNi` の正方形対角ブロックの次元を含み、`i` 番目のブロックは次元 `νi[i] x νi[i]` を持ちます。`νi[ni-i+1]-νi[ni-i]` の差は、次数 `i` の無限基本因子の数です（`νi[0] = 0`）。

完全列ランクペンシル `Mfl-λNfl` は、ペンシル `M-λN` の左クロンカーおよび有限クロンカー構造を含み、次の形を持ちます：

```
                 | A-λE | 
      Mfl-λNfl = |------| ,        
                 |  C   |
```

ここで `E` は nxn の非特異上三角行列であり、`A` と `C` はそれぞれ `nxn` および `pxn` 次元の行列です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。還元は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、また `fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左の直交またはユニタリ変換は、`withQ = true` の場合に行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された右の直交またはユニタリ変換は、`withZ = true` の場合に行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。
