```
pmpoles2(P; fast = false, atol::Real = 0, rtol::Real = atol>0 ? 0 : n*ϵ) -> (val, ip, id)
```

多項式行列 `P(λ)` の無限極を `val` に、無限極の重複度を `ip` に、`P(λ)` の基礎となる既約記述子系に基づく線形化の極ペンシルの無限基本除数を `id` に返します。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形の次数 `k` の多項式行列として指定でき、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納され、`P[:,:,i]` には `i` 番目の係数行列 `P_i`（`λ**(i-1)` を掛けたもの）が含まれます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

多項式行列 `P(λ)` の無限極に関する情報は、[1] および [2] の結果を使用して基礎となる線形化の極構造情報から取得されます。極構造情報の決定は、`A-λE` が単位行列である既約記述子系の実現 `(A-λE,B,C,D)` を構築することによって行われ、`R(λ) = C*inv(λE-A)*B+D` を満たし、極ペンシル `A-λE` を無限固有値の重複度を示す適切なクロンカー様形式 (KLF) に削減します（`P(λ)` の無限極の重複度に対して1つ余分に）。`A-λE` の無限ゼロの重複度と `P(λ)` の無限極の重複度は同じであり、`ip` に返されます。`val` の無限極の数は `ip` の重複度の合計に等しいです。`A-λE` の無限固有値の重複度は `id` に返されます。

`A-λE` を KLF に削減するためには、直交類似変換を使用し、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を含み、`fast = false` の場合は、より信頼性の高いSVD分解を使用します。効率のために、削減は部分的にのみ行われ、実行された直交変換を蓄積しません。

キーワード引数 `atol` と `rtol` は、それぞれ `P(λ)` の非ゼロ係数の絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `P(λ)` の最小次元のサイズで、`ϵ` は `P(λ)` の係数の要素型の機械イプシロンです。

[1] G. Verghese, B. Levy, and T. Kailath, Generalized state-space systems, IEEE Trans. Automat. Control, 26:811-831 (1981).

[2] G. Verghese, Comments on ‘Properties of the system matrix of a generalized state-space system’, Int. J. Control, Vol.31(5) (1980) 1007–1009.
