```
pmkstruct(P; CF1, grade=l, fast = false, atol::Real = 0, rtol::Real = atol>0 ? 0 : n*ϵ) -> KRInfo, iz, ip
```

多項式行列 `P(λ)` のクロンカー構造と無限極限ゼロ構造情報を決定し、`KRInfo` オブジェクトと無限ゼロおよび極限の重複度を返します。クロンカー構造の計算は、`CF1 = true` の場合は第一伴随形式で、`CF1 = false` の場合は第二伴随形式で `P(λ)` の強線形化を用いて行われます。線形化に使用される効果的な次数 `l` は、キーワード引数 `grade` を介して `grade = l` として指定でき、`l` は `P(λ)` の次数以上である必要があります。`l` のデフォルト値は `l = deg(P(λ))` です。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形の次数 `k` の多項式行列として指定でき、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納され、`P[:,:,i]` には `i` 番目の係数行列 `P_i`（`λ**(i-1)` を掛けたもの）が含まれます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

クロンカー構造に関する情報は、右クロンカーインデックス `rki`、左クロンカーインデックス `lki`、無限基本除数 `id`、有限固有値の数 `nf`、および通常のランク `nrank` で構成され、`KRInfo` からそれぞれ `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` として取得できます。詳細については、[`KRInfo`](@ref) を参照してください。

クロンカー構造情報の決定は、`P(λ)` の伴随形式線形化 `M-λN` を構築し、適切なクロンカー様形式 (KLF) にペンシル `M-λN` を縮約することによって行われ、有限固有値の数、無限固有値の重複度、左および右クロンカーインデックス、通常のランクが示されます。`P(λ)` に関するクロンカー構造情報と極限ゼロ構造情報は、[1] および [2] の結果を使用して `M-λN` のクロンカー構造情報から回収されます。

右クロンカーインデックスは整数ベクトル `rki` に提供されます。`rki` の要素数は多項式行列 `P(λ)` の右ヌル空間の次元であり、その合計は右多項式ヌル空間基底の最小次数です。

左クロンカーインデックスは整数ベクトル `lki` に提供されます。`lki` の要素数は多項式行列 `P(λ)` の左ヌル空間の次元であり、その合計は左多項式ヌル空間基底の最小次数です。

無限固有値の重複度は整数ベクトル `id` に提供され、各 `i` 番目の要素 `id[i]` は無限基本除数の次数（すなわち、無限固有値の重複度）です。

`P(λ)` の無限ゼロの重複度は `iz` に返され、`P(λ)` の無限固有値の重複度と `P(λ)` の効果的な次数 `l` との間の正の差を表します [2]。

`P(λ)` の無限極限の重複度は `ip` に返され、`P(λ)` の無限固有値の重複度と `P(λ)` の効果的な次数 `l` との間の負の差の絶対値を表します [2]。

KLF への縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボットを伴うランクを明らかにする QR 分解に基づくランク決定が含まれ、`fast = false` の場合は、より信頼性の高い SVD 分解が使用されます。

キーワード引数 `atol` と `rtol` は、`P(λ)` の非ゼロ係数の絶対および相対許容誤差をそれぞれ指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `P(λ)` の最小次元のサイズで、`ϵ` は `P(λ)` の係数の要素型の機械イプシロンです。

[1] F. De Terán, F. M. Dopico, D. S. Mackey, Spectral equivalence of polynomial matrices and the Index Sum Theorem, Linear Algebra and Its Applications, vol. 459, pp. 264-333, 2014.

[2] A. Varga, On computing the Kronecker structure of polynomial matrices using Julia, June 2020,  [arXiv:2006.06825](https://arxiv.org/pdf/2006.06825). ```
