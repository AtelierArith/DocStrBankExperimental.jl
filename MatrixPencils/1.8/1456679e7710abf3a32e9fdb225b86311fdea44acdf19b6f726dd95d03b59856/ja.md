```
sklf_left!(A, E, C, B; fast = true, atol1 = 0, atol2 = 0, atol3 = 0, rtol, withQ = true, withZ = true) -> (Q, Z, μl, no, nfuo, niuo)
```

分割されたフルコラムランク行列ペンシルを減少させます 

```
  M - λN = | A-λE | n
           |  C   | p
              n
```

を同等の形 `F - λG = diag(Q',I)*(M - λN)*Z` に変換します。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用し、`M - λN` が無限、有限、左クロンカー構造（一般化された可観測性階段形式とも呼ばれる）を示すクロンカーのような形に変換されます：

```
                | Aiuo-λEiuo     *       |   *    | niuo
                |    0       Afuo-λEfuo  |   *    | nfuo
  | At-λEt |    |    0           0       | Ao-λEo | no
  |--------| =  |------------------------|--------|
  |  Ct    |    |    0           0       |   Co   | p
                    niuo        nfuo         no
```

`Ct = C*Z`、`At = Q'*A*Z` および `Et = Q'*E*Z` はそれぞれ `C`、`A` および `E` に返され、`Q'*B` は `B` に返されます（`B = missing` でない限り）。                 

サブペンシル `| Ao-λEo |` はフルコラムランク `no` を持ち、階段形式にあり、`M - λN` の左クロンカーインデックスを含みます。                `|   Co   |` 次元ベクトル `μl` は、階段形式のブロックの行と列の次元を含み、`i` 番目のブロックは次元 `μl[nl-i] x μl[nl-i+1]` を持ち（μl[0] = p）、フルコラムランクを持ちます。差 `μl[nl-i]-μl[nl-i+1]` は、`i = 1, 2, ..., nl` の場合、サイズ `i x (i-1)` の基本クロンカーブロックの数です。

`niuo x niuo` サブペンシル `Aiuo-λEiuo` は `M - λN` の無限固有値を含みます（`A-λE` の観測不可能な無限固有値とも呼ばれます）。  

`nfuo x nfuo` サブペンシル `Afuo-λEfuo` は `M - λN` の有限固有値を含みます（`A-λE` の観測不可能な有限固有値とも呼ばれます）。  

キーワード引数 `atol1`、`atol2`、`atol3` および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、および `A`、`E`、`C` の非ゼロ要素の相対許容誤差を指定します。減少は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

実行された直交またはユニタリ左変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された直交またはユニタリ右変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。
