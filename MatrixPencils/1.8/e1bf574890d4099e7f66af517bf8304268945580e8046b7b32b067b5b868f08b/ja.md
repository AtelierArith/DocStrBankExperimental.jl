```
pmeigvals(P; CF1, grade = l, fast = false, atol::Real = 0, rtol::Real = atol>0 ? 0 : n*ϵ) -> (val, KRInfo)
```

多項式行列 `P(λ)` の有限および無限の固有値を `val` に、`P(λ)` のクロンカー構造に関する情報を `KRInfo` オブジェクトに返します。固有値とクロンカー構造の計算は、`CF1 = true` の場合は第一伴随形式で、`CF1 = false` の場合は第二伴随形式で、`P(λ)` の強い線形化を用いて行われます。線形化に使用される効果的な次数 `l` は、キーワード引数 `grade` を介して `grade = l` として指定でき、`l` は `P(λ)` の次数以上である必要があります。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形の次数 `k` の多項式行列として指定でき、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納され、`P[:,:,i]` には `i` 番目の係数行列 `P_i`（`λ**(i-1)` を掛けたもの）が含まれます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

クロンカー構造に関する情報は、右クロンカーインデックス `rki`、左クロンカーインデックス `lki`、無限の基本因子 `id`、有限固有値の数 `nf`、および通常のランク `nrank` で構成され、`KRInfo` から `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` として取得できます。詳細については、[`KRInfo`](@ref) を参照してください。

固有値の計算は、多項式行列 `P(λ)` の伴随形式に基づく線形化 `M-λN` を構築し、その後、有限固有値の数、無限固有値の重複度、左および右のクロンカーインデックス、通常のランクを示す適切なクロンカー様形式（KLF）にペンシル `M-λN` を縮約することによって行われます。`P(λ)` の左および右のクロンカーインデックスはそれぞれ `KRInfo.rki` と `KRInfo.rki` に返され、その値は [1] の結果を使用して `M-λN` の左および右のクロンカーインデックスから回収されます。効果的な次数 `l` の `P(λ)` の無限固有値の重複度は `KRInfo.id` に返されます。`val` の有限固有値の数は `M-λN` の有限固有値の数（`KRInfo.nf` に返される）と等しく、`val` の無限固有値の数は `KRInfo.id` に返される重複度の合計です。

`M-λN` を KLF に縮約するには、直交類似変換を使用し、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を行い、`fast = false` の場合はより信頼性の高いSVD分解を行います。効率のために、縮約は部分的にのみ行われ、実行された直交変換は累積されません。

キーワード引数 `atol` と `rtol` は、それぞれ `P(λ)` の非ゼロ係数の絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` で、ここで `n` は `P(λ)` の最小次元のサイズであり、`ϵ` は `P(λ)` の係数の要素型のマシンイプシロンです。

[1] F. De Terán, F. M. Dopico, D. S. Mackey, 多項式行列のスペクトル同値性とインデックス和定理, 線形代数とその応用, vol. 459, pp. 264-333, 2014. ```
