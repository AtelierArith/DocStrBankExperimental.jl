```
sklf_right!(A, E, B, F, C, G, D, H; fast = true, atol1 = 0, atol2 = 0, atol3 = 0, rtol, withQ = true, withZ = true) -> (Q, Z, nc)
```

分割されたフル行ランク行列ペンシルを削減します 

```
  M - λN = | B-λF | A-λE | n
              m      n
```

`A-λE` が正則である場合、同等の形 `Mt - λNt = Q'*(M - λN)*Z = | Bt-λFt | At-λEt |` に変換します。これは、直交またはユニタリ変換行列 `Q` と `Z` を使用して、`M - λN` がその右、有限および無限のクロンカー構造（強制御可能形式とも呼ばれる）を示すクロンカーのような形 `| Bt-λFt | At-λEt |` に変換されるようにします：

```
                         | Bc-λFc | Ac-λEc     *     | nc
  | Bt-λFt | At-λEt | =  |--------|------------------|
                         |   0    |   0     Auc-λEuc | n-nc
                             m        nc      n-nc
```

行列 `Bt`, `Ft`, `At`, `Et` は `[Bt At] = Q'*[B A]*Z`, `[Ft Et] = Q'*[F E]*Z` から決定され、`B`, `F`, `A` および `E` にそれぞれ返されます。さらに、互換的に分割された行列 `[Dt Ct] = [D C]*Z` および `[Ht Gt] = [D C]*Z` から決定される行列 `Ct`, `Dt`, `Gt`, `Ht` は、`C`, `D`, `G` および `H` にそれぞれ返されます（`C`, `D`, `G` および `H` が `missing` に設定されていない限り）。

サブペンシル `| Bc-λFc | Ac-λEc |` はフル行ランク `nc` を持ち、`(n-nc) x (n-nc)` のサブペンシル `Auc-λEuc` は `M - λN` の有限および無限の固有値を含みます（これを `M - λN` の制御不能な固有値とも呼びます）。

`(m+n) x (m+n)` の直交行列 `Z` は分割された形を持ちます

```
       | Z11 |  0  |  *  | m 
   Z = |-----|-----|-----|
       |  *  |  *  |  *  | n
          m    nc    n-nc
```

先頭の `m x m` サブ行列 `Z11` は可逆で上三角です。

`注意:` もし `Ct-λGt` が `[ Cc-λGc | Cuc-λGuc ]` と分割されると、次の関係が満たされます

```
  (Cc-λGc)*inv(λEc-Ac)(Bc-λFc) + Dt-λHt = ((C-λG)*inv(λE-A)(B-λF) + D-λH)*Z11
```

そして構造化されたペンシル線形化 (Ac-λEc,Bc-λFc,Cc-λGc,Dt-λHt) は `強制御可能` です。

キーワード引数 `atol1`, `atol2`, `atol3`, および `rtol` は、それぞれ `A` と `B` の非ゼロ要素の絶対許容誤差、`E` と `F` の非ゼロ要素の絶対許容誤差、`A`, `B`, `E` および `F` の非ゼロ要素の相対許容誤差を指定します。削減は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ左変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された直交またはユニタリ右変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。`withZ = false` の場合、`Z` には上三角行列 `Z11` が含まれます。
