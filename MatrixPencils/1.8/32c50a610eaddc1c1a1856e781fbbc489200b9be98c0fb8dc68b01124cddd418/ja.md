```
spkstruct(A, E, B, C, D; fast = false, atol1::Real = 0, atol2::Real = 0, rtol::Real=min(atol1,atol2)>0 ? 0 : n*ϵ) -> KRInfo
```

構造行列ペンシル `M-λN` のクロンネッカー構造情報を決定します。

```
          | A-λE | B | 
 M - λN = |------|---|
          |  C   | D |
```

そして `KRInfo` オブジェクトを返します。

クロンネッカー構造に関する情報は、右クロンネッカーインデックス `rki`、左クロンネッカーインデックス `lki`、無限の基本因子 `id`、有限固有値の数 `nf`、通常のランク `nrank` で構成され、`KRInfo` からそれぞれ `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` として取得できます。詳細については、[`pkstruct`](@ref) を参照してください。

右クロンネッカーインデックスは整数ベクトル `rki` に提供されており、各 `i` 番目の要素 `rki[i]` はサイズ `k x (k+1)` の基本クロンネッカーブロックの行次元 `k` です。`rki` の要素数はペンシル `M-λN` の右ヌル空間の次元であり、その合計は右多項式ヌル空間基底の最小次数です。

左クロンネッカーインデックスは整数ベクトル `lki` に提供されており、各 `i` 番目の要素 `lki[i]` はサイズ `(k+1) x k` の基本クロンネッカーブロックの列次元 `k` です。`lki` の要素数はペンシル `M-λN` の左ヌル空間の次元であり、その合計は左多項式ヌル空間基底の最小次数です。

無限固有値の重複度は整数ベクトル `id` に提供されており、各 `i` 番目の要素 `id[i]` は無限基本因子の次数（すなわち、無限固有値の重複度）です。

クロンネッカー構造情報の決定は、有限固有値の数、無限固有値の重複度、左および右クロンネッカーインデックス、通常のランクを示す適切なクロンネッカー様式（KLF）にペンシル `M-λN` を縮約することによって行われます。縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボッティングを伴うランク明示的QR分解に基づくランク決定に基づいて行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。効率のために、縮約は部分的にのみ行われ、実行された直交変換は蓄積されません。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、`M` および `N` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `M` の最小次元のサイズで、`ϵ` は `M` の要素型の機械イプシロンです。 ```
