```
salocd(A, C; evals, sdeg, disc = false, atol1 = 0, atol2 = 0, rtol) -> (K, Scl, blkdims)
```

ペア `(A,C)` に対して、行列 `A+K*C` のすべての固有値が安定度パラメータ `sdeg` と安定性タイプパラメータ `disc` によって指定された安定性領域 `Cs` に存在するような行列 `K` を計算します。`disc = false` の場合、`Cs` は実部が最大 `sdeg` の複素数の集合であり、`disc = true` の場合、`Cs` は原点を中心とした半径 `sdeg` の円の内部にある複素数の集合です。`evals` は、`Cs` 内にある行列 `A+K*C` の望ましい固有値を含む実数または複素数のベクトルです。実データ `A` と `C` に対して、`evals` は結果として得られる `K` も実行列であることを保証するために自己共役複素集合でなければなりません。

`A` の次数が `n` のペア `(A,C)` に対して、割り当て可能な固有値の数は `nc := n-nu` であり、ここで `nu` は `A` の固定固有値の数です。割り当て可能な固有値は *観測可能固有値* と呼ばれ、固定固有値は *非観測可能固有値* と呼ばれます（これらはペンシル `[A-λI; C]` のゼロです）。スペクトルの割り当ては、安定性領域 `Cs` の外にある `A` の観測可能固有値を、`evals` で提供された固有値に順次置き換えることによって達成されます。`Cs` にある `A` のすべての固有値は変更されずに保持されます。`evals` に指定された固有値の数が `Cs` の外にある `A` の観測可能固有値の数よりも少ない場合（例：`evals = missing` の場合）、`A` の観測可能固有値の一部は `Cs` の境界上の最も近い値に割り当てられます。`sdeg = missing` かつ `evals = missing` の場合、`disc = false` のときは `sdeg` のデフォルト値は -0.05 であり、`disc = true` のときは 0.95 です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、および `A` と `C` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は `A` の要素型のマシンイプシロンです。

シュール形式の行列 `Acl := Z'*(A+K*C)*Z` は、ここで `Z` は行列 `A+K*C` をシュール形式にするために使用される直交/ユニタリ行列であり、次の形を持ちます。

```
       ( Au  *   *  )     
 Acl = ( 0   Aa  *  ) 
       ( 0   0   Ag )
```

ここで、`Au` は `Cs` の外にある `A` の `nu` 個の非観測可能固有値を含み、`Aa` は `Cs` にある `na` 個の割り当てられた固有値を含み、`Ag` は `Cs` にある `A` の `ng` 個の固有値を含みます。行列 `Acl` と `Z`、および `Acl`（および `A+K*C`）の固有値のベクトル `α` は、`Schur` オブジェクト `Scl` に返されます。`nu`、`na`、および `ng` の値は、3次元ベクトル `blkdims = [nu, na, ng]` に返されます。

メソッド： [1] のシュール法が双対ペア `(A',C')` に適用されます（非観測可能ペアに拡張される可能性があります）。

参考文献：

[1] A. Varga.      A Schur method for pole assignment.     IEEE Trans. on Automatic Control, vol. 26, pp. 517-519, 1981. ```
