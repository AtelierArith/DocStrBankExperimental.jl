```
 lps2ls(A, E, B, F, C, G, D, H; compacted = false, atol1 = 0, atol2 = 0, atol3 = 0, rtol = min(atol1,atol2,atol3)>0 ? 0 : n*ϵ) -> (Ad,Ed,Bd,Cd,Dd)
```

入力-出力同等の記述子システムの線形化 `(Ad-λdE,Bd,Cd,Dd)` を、ペンシルベースの線形化 `(A-λE,B-λF,C-λG,D-λH)` に対して構築し、次の条件を満たします。

```
            -1                        -1
 Cd*(λEd-Ad)  *Bd + Dd = (C-λG)*(λE-A)  *(B-λF) + D-λH .
```

`compacted = true` の場合、行列 `F`、`G`、および `H` の可能なランク欠損を利用して圧縮線形化が決定されます。

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `F` の非ゼロ要素に対する絶対許容誤差、`G` の非ゼロ要素に対する絶対許容誤差、`H` の非ゼロ要素に対する絶対許容誤差、および `F`、`G`、`H` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズ、`ϵ` は `A` の要素型のマシンイプシロンです。
