```
 pm2lpCF1(P; grade = l) -> (M, N)
```

多項式行列 `P(λ)` の最初の伴随フロベニウス形式の強い線形化 `M - λN` を構築します。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形を持つグレード `k` の多項式行列と仮定され、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納され、`P[:,:,i]` には `λ**(i-1)` を掛けた `i` 番目の係数行列 `P_i` が含まれます。線形化に使用する効果的なグレード `l` は、キーワード引数 `grade` を介して `grade = l` として指定でき、`l` は `P(λ)` の次数以上である必要があります。デフォルト値として `l = deg(P(λ))` が使用されます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

`P(λ)` が効果的なグレード `l` と次数 `d` の `m x n` の多項式行列である場合、結果として得られる行列ペンシル `M - λN` は以下の条件を満たします [1]:

(1) `M - λN` の次元は `(m+n*(l-1)) x n*l` であり、`P(λ)` が正則であれば `M - λN` も正則です;

(2) `M - λN` と `P(λ)` は同じ有限固有値を持ちます;

(3) `M - λN` の無限固有値の部分重複度は `l-d` だけ `P(λ)` の無限固有値の部分重複度を超えます;

(4) `M - λN` と `P(λ)` は同じ数の右クロンネッカー指標を持ち、`M - λN` の右クロンネッカー指標は `P(λ)` の右クロンネッカー指標を `l-1` だけ超えます;

(5) `M - λN` と `P(λ)` は同じ左クロンネッカー構造（すなわち、同じ左クロンネッカー指標）を持ちます。

[1] F. De Terán, F. M. Dopico, D. S. Mackey, Spectral equivalence of polynomial matrices and the Index Sum Theorem, Linear Algebra and Its Applications, vol. 459, pp. 264-333, 2014. ```
