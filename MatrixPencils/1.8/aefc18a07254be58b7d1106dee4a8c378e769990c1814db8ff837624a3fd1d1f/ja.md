```
isunimodular(A, E; atol1::Real = 0, atol2::Real = 0, rtol::Real=min(atol1,atol2)>0 ? 0 : n*ϵ) -> Bool
```

行列ペンシル `A-λE` がユニモジュラーであるかどうかをテストします（すなわち、`A-λE` が正方行列であり、正則であり、`det(A-λE) == 定数` であること）。基盤となる計算手順は、ペンシル `A-λE` を適切なクロンカー様の形に還元し、`A-λE` のフルランクをチェックし、有限の固有値が存在しないことを確認するための情報を提供します。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A` および `E` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズ、`ϵ` は `A` の要素型のマシンイプシロンです。
