```
rmeval(NUM,DEN,val) -> Rval
```

有理行列 `R(λ) := NUM(λ)./DEN(λ)` を `λ = val` のときに評価します。

`NUM(λ)` は次の形の多項式行列です：`NUM(λ) = N_1 + λ N_2 + ... + λ**k N_(k+1)`。ここで、係数行列 `N_i`（`i = 1, ..., k+1`）は3次元行列 `N` に格納されており、`N[:,:,i]` には `i` 番目の係数行列 `N_i`（`λ**(i-1)` を掛けたもの）が含まれています。

`DEN(λ)` は次の形の多項式行列です：`DEN(λ) = D_1 + λ D_2 + ... + λ**l D_(l+1)`。ここで、係数行列 `D_i`（`i = 1, ..., l+1`）は3次元行列 `D` に格納されており、`D[:,:,i]` には `i` 番目の係数行列 `D_i`（`λ**(i-1)` を掛けたもの）が含まれています。

また、`NUM(λ)` と `DEN(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列として指定することもできます。この場合、`N(λ)` と `D(λ)` が同じ不定元を持つかどうかのチェックは行われません。
