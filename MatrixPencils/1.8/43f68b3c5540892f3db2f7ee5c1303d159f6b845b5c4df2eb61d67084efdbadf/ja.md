```
isregular(A, E, γ; atol::Real = 0,  rtol::Real = atol > 0 ? 0 : n*ϵ) -> Bool
```

行列ペンシル `A-λE` が `λ = γ` で正則であるかどうかをテストします（すなわち、`A-λE` が正方行列であり、${\small\det(A-λE) \neq 0}$ であること）。 基本的な計算手順は、`γ` が有限の場合は `A-γE` の最大ランクを、`γ` が無限の場合は `E` の最大ランクをチェックします。

キーワード引数 `atol` と `rtol` は、それぞれ `A-γE` の非ゼロ要素に対する絶対および相対許容誤差を指定します。 デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `A` のサイズ、`ϵ` は `A` の要素型のマシンイプシロンです。
