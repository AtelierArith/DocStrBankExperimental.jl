```
sklf_left!(A, C, B; fast = true, atol1 = 0, atol2 = 0, rtol, withQ = true) -> (Q, μl, no, nuo)
```

分割されたフルコラムランク行列ペンシルを減少させます 

```
  M - λN = | A-λI | n
           |  C   | p
              n
```

を同等の形 `F - λL = diag(Q',I)*(M - λN)*Q` に変換します。ここで、直交またはユニタリ変換行列 `Q` を使用して、`M - λN` が有限および左クロンカー構造（可観測階段形式とも呼ばれる）を示すクロンカーのような形に変換されます：

```
               | Auo-λI  |  *    | nuo
  | At-λI |    |    0    | Ao-λI | no
  |-------| =  |---------|-------|
  |  Ct   |    |    0    |   Co  | p
                   nuo       no
```

`Ct = C*Q` および `At = Q'*A*Q` はそれぞれ `C` と `A` に返され、`Q'*B` は `B` に返されます（`B = missing` でない限り）。                 

サブペンシル `| Ao-λI; Co |` はフルコラムランク `no` を持ち、階段形式になっており、`M - λN` の左クロンカーインデックスを含んでいます。 

`nl` 次元ベクトル `μl` は、階段形式のブロックの行と列の次元を含み、`i` 番目のブロックは次元 `μl[nl-i] x μl[nl-i+1]` を持ち（μl[0] = p）、フルコラムランクを持ちます。 差 `μl[nl-i]-μl[nl-i+1]` は、`i = 1, 2, ..., nl` の場合、サイズ `i x (i-1)` の基本クロンカー ブロックの数です。

`nuo x nuo` 行列 `Auo` は、`M - λN` の（有限の）固有値を含みます（`A` の非可観測固有値とも呼ばれます）。  

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、および `A` と `C` の非ゼロ要素の相対許容誤差を指定します。 減少は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。 そうでない場合、`Q` は `nothing` に設定されます。
