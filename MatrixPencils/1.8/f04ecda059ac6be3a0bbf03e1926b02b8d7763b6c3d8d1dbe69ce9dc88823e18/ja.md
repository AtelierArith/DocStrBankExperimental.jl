```
pmroots(P; fast = false, atol::Real = 0, rtol::Real = atol>0 ? 0 : n*ϵ) -> val
```

行列式 `P(λ)` の根を `val` に計算します。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形の次数 `k` の多項式行列として指定できます。このとき、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納され、`P[:,:,i]` には `λ**(i-1)` を掛けた `i` 番目の係数行列 `P_i` が含まれます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

`det(P(λ))` の根は、`P(λ)` の伴随形式線形化 `M-λN` の有限固有値として計算されます。有限固有値は、無限および有限固有値構造の分割を示す適切なクロンカー様形式 (KLF) にペンシル `M-λN` を縮約することによって計算されます。縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解に基づくランク決定が含まれ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。効率のために、縮約は部分的にのみ行われ、実行された直交変換は蓄積されません。

キーワード引数 `atol` と `rtol` は、それぞれ `P(λ)` の非ゼロ係数の絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `P(λ)` の最小次元のサイズで、`ϵ` は `P(λ)` の係数の要素型のマシンイプシロンです。
