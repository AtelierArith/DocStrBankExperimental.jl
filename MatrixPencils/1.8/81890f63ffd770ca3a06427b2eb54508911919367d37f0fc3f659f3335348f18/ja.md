```
sfischursep(A, E; smarg, disc = false, fast = true, finite_infinite = false, stable_unstable = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (At, Et, Q, Z, ν, blkdims)
```

正則行列ペンシル `A - λE` を、安定性マージンパラメータ `smarg` と安定性タイプパラメータ `disc` に関して、無限、安定、不安定な固有値が分離されたブロック上三角形形式 `At - λEt = Q'*(A - λE)*Z` に等価に還元します。`disc = false` の場合、`Cs` は実部が `smarg` より小さい複素数の集合であり、`disc = true` の場合、`Cs` は `smarg` の半径の円の内部にある複素数の集合です（すなわち、原点を中心とした半径 `smarg` の円の内部）。`smarg = missing` の場合、デフォルト値は `smarg = 0`（`disc = false` の場合）または `smarg = 1`（`disc = true` の場合）です。

ペンシル `At - λEt` は、次のいずれかのブロック上三角形形式になります：

(1) `finite_infinite = true` の場合、

```
               | Ai1   *      *       *     |
    At - λEt = |  0  A1-λE1   *       *     |
               |  0    0    A2-λE2    *     |
               |  0    0      0    Ai2-λEi2 |
```

ここで、`ni1 x ni1` 行列 `Ai1` と `ni2 x ni2` サブペンシル `Ai2-λEi2` は無限の基本因子を含み、`n1 x n1` サブペンシル `A1-λE1` はペア `(A1,E1)` が一般化シュール形式にあり、`n2 x n2` サブペンシル `A2-λE2` はペア `(A2,E2)` が一般化シュール形式にあります。ペンシル `A1-λE1` は不安定な有限固有値を持ち、`A2-λE2` は安定な有限固有値を持つ場合（`stable_unstable = false`）、`A1-λE1` は安定な有限固有値を持ち、`A2-λE2` は不安定な有限固有値を持つ場合（`stable_unstable = true`）です。

サブペンシル `Ai2-λEi2` は階段形式であり、`Ai2` は非特異で上三角形、`Ei2` は nilpotent で上三角形です。`nb` 次元のベクトル `ν` は、階段形式 `Ai2-λEi2` の対角ブロックの次元を含み、`i` 番目のブロックは次元 `ν[i] x ν[i]` を持ちます。差 `ν[nb-j+2]-ν[nb-j+1]` は、`j = 1, 2, ..., nb+1` の場合の無限の基本因子の次数 `j` の数です（`ν[0] := 0` および `ν[nb+1] := ni1`）。

対角ブロックの次元は `blkdims = (ni1, n1, n2, ni2)` に返されます。

(2) `finite_infinite = false` の場合、

```
               | Ai1-λEi1   *      *     *  |
               |    0     A1-λE1   *     *  |
    At - λEt = |    0       0    A2-λE2  *  |
               |    0       0      0    Ai2 |
```

ここで、`ni1 x ni1` サブペンシル `Ai1-λEi1` と `ni2 x ni2` 上三角形かつ非特異な行列 `Ai2` は `A-λE` の無限の基本因子を含み、`n1 x n1` サブペンシル `A1-λE1` はペア `(A1,E1)` が一般化シュール形式にあり、`n2 x n2` サブペンシル `A2-λE2` はペア `(A2,E2)` が一般化シュール形式にあります。ペンシル `A1-λE1` は不安定な有限固有値を持ち、`A2-λE2` は安定な有限固有値を持つ場合（`stable_unstable = false`）、`A1-λE1` は安定な有限固有値を持ち、`A2-λE2` は不安定な有限固有値を持つ場合（`stable_unstable = true`）です。

サブペンシル `Ai1-λEi1` は階段形式であり、`Ai1` は非特異で上三角形、`Ei1` は nilpotent で上三角形です。`nb` 次元のベクトル `ν` は、階段形式 `Ai1-λEi1` の対角ブロックの次元を含み、`i` 番目のブロックは次元 `ν[i] x ν[i]` を持ちます。差 `ν[i]-ν[i+1]` は、`i = 0, 1, 2, ..., nb` の場合の無限の基本因子の次数 `i` の数です（`ν[nb+1] = 0` および `ν[0] = ni2`）。

対角ブロックの次元は `blkdims = (ni1, n1, n2, ni2)` に返されます。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、および `A` と `E` の非ゼロ要素の相対許容誤差を指定します。

還元は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左の直交またはユニタリ変換は、`withQ = true` の場合に行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された右の直交またはユニタリ変換は、`withZ = true` の場合に行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。
