```
sklf(A, E, B, C, D; fast = true, finite_infinite = false, ut = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (F, G, Q, Z, νr, μr, νi, nf, νl, μl)
```

構造行列ペンシル `M - λN` を次の同等の形 `F - λG = Q'*(M - λN)*Z` に減少させます。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用し、変換された行列 `F` と `G` は以下のクロンカーのような形を持ち、完全なクロンカー構造を示します：

```
            |  Mr-λNr  |     *      |    *    |
            |----------|------------|---------|
  F - λG =  |    O     | Mreg-λNreg |    *    |
            |----------|------------|---------|
            |    O     |     0      |  Ml-λNl |
```

完全行ランクペンシル `Mr-λNr` は階段形式で、ペンシル `M-λN` の右クロンカーインデックスを含み、次の形を持ちます：

```
     Mr-λNr  = | Br | Ar-λEr |,
```

ここで `Er` は上三角で非特異です。次元 `nr` のベクトル `νr` と `μr` は、階段形式 `Mr-λNr` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `νr[i] x μr[i]` を持ち、完全行ランクです。差 `μr[i]-νr[i]` は、`i = 1, 2, ..., nr` の場合、サイズ `(i-1) x i` の基本クロンカー ブロックの数です。`ut = true` の場合、`Mr` の完全行ランク対角ブロックは形 `[0 X]` に減少し、`X` は上三角で非特異であり、`Nr` の完全列ランク超対角ブロックは形 `[Y; 0]` に減少し、`Y` は上三角で非特異です。

ペンシル `Mreg-λNreg` は正則で、`M-λN` の無限および有限の基本除数を含み、次の形を持ちます：

```
                   | Mi-λNi |    *    |
     Mreg-λNreg  = |--------|---------|, もし `finite_infinite = false` の場合、または次の形
                   |   0    |  Mf-λNf |


                   | Mf-λNf |    *    |
     Mreg-λNreg  = |--------|---------|, もし `finite_infinite = true` の場合、 
                   |   0    |  Mi-λNi |
```

ここで：(1) `Mi-λNi` は階段形式で、`M-λN` の無限基本除数を含み、`Mi` は上三角で `ut = true` の場合は非特異であり、`Ni` は上三角でニルポテントです；(2) `Mf-λNf` は `M-λN` の無限基本除数を含み、`Nf` は上三角で非特異です。次元 `ni` のベクトル `νi` は、階段形式 `Mi-λNi` の正方形ブロックの次元を含み、`i` 番目のブロックは次元 `νi[i] x νi[i]` を持ちます。`finite_infinite = true` の場合、`νi[i]-νi[i+1]` の差は、次数 `i` の無限基本除数の数です（`νi[ni] = 0`）。`finite_infinite = false` の場合、`νi[ni-i+1]-νi[ni-i]` の差は、次数 `i` の無限基本除数の数です（`νi[0] = 0`）。

完全列ランクペンシル `Ml-λNl` は階段形式で、`M-λN` の左クロンカーインデックスを含み、次の形を持ちます：

```
               | Al-λEl |
     Ml-λNl  = |--------|,
               |   Cl   |
```

ここで `El` は上三角で非特異です。次元 `nl` のベクトル `νl` と `μl` は、階段形式 `Ml-λNl` のブロックの行および列の次元を含み、`j` 番目のブロックは次元 `νl[j] x μl[j]` を持ち、完全列ランクです。差 `νl[nl-j+1]-μl[nl-j+1]` は、`j = 1, 2, ..., nl` の場合、サイズ `j x (j-1)` の基本クロンカー ブロックの数です。`ut = true` の場合、`Ml` の完全列ランク対角ブロックは形 `[X; 0]` に減少し、`X` は上三角で非特異であり、`Nl` の完全行ランク超対角ブロックは形 `[0 Y]` に減少し、`Y` は上三角で非特異です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。減少は、`fast = true` の場合は列ピボッティングを伴うランク決定に基づくランクを明らかにするQR分解を使用して、または `fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左直交またはユニタリ変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された右直交またはユニタリ変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。
