```
klf_left_refineut!(ν, μ, M, N, Q, Z; roff = 0, coff = 0, withQ = true, withZ = true)
```

分割された行列ペンシル `M - λN` を次のように縮小します（`*` は関連しないサブペンシルを表します）

```
         [  *     *     ] roff
M - λN = [  0  Mil-λNil ] 
          coff
```

等価な形 `F - λG = Q1'*(M - λN)*Z1` に変換します。ここで、直交またはユニタリ変換行列 `Q1` と `Z1` を使用し、フルコラムランクサブペンシル `Mil-λNil` を階段形式に変換します。`Mil` のフルコラムランク対角ブロックは `[Y ; 0]` の形に縮小され、`Y` は上三角で非特異です。また、`Nil` のフルローランク超対角ブロックは `[0 Y]` の形に縮小され、`Y` は上三角で非特異です。`Mil-λNil` には `nb` の対角ブロックがあり、対角ブロックの次元は `ν` と `μ` の `nb` 次元ベクトルによって指定され、`i` 番目のブロックは `ν[i] x μ[i]` の次元を持つと仮定します。

実行された直交またはユニタリ変換は `Q` に蓄積されます（すなわち、`Q <- Q*Q1`）、`withQ = true` の場合、`Z` にも蓄積されます（すなわち、`Z <- Z*Z1`）、`withZ = true` の場合。
