```
salocinfd(A, E, C; atol1 = 0, atol2 = 0, atol3 = 0, rtol, sepinf = true, fast = true) -> (K, L, Scl, blkdims)
```

ペア `(A-λE,C)` に対して、`A-λE` が通常のペンシルであるとき、すべての固有値が無限大であるように行列 `K` と `L` を計算します。固定された（観測不可能な）有限固有値を持つペア `(A-λE,C)` に対しては、割り当て可能な（観測可能な）有限固有値のみが無限大に移動されます。

`A` の次数が `n` であるペア `(A-λE,C)` に対して、割り当て可能な無限固有値の数は `nia := n-ninf-nfu` であり、ここで `ninf` は `A-λE` の無限固有値の数、`nfu` は `A-λE` の固定された有限固有値の数です。割り当て可能な有限固有値は *観測可能な有限固有値* と呼ばれ、固定された有限固有値は *観測不可能な有限固有値* と呼ばれます（これらはペンシル `[A-λE; C]` の有限ゼロです）。スペクトルの割り当ては、`A-λE` の観測可能な有限固有値を無限固有値に置き換えることによって達成されます。`A-λE` のすべての無限固有値は変更されずに保持されます。

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、および `A`、`E`、`C` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は `A` の要素型のマシンイプシロンです。

有限および無限固有値の予備的な分離は、`fast = true` の場合は列ピボットを用いたランクを明らかにするQR分解に基づくランク決定を使用し、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

得られたペンシル `Acl-λEcl := Q'*(A+K*C-λ(E+L*C))*Z` は、ここで `Q` と `Z` は一般化シュール形式（GSF）でペア `(Acl,Ecl)` を得るために使用される直交/ユニタリ行列であり、次の形を持ちます。

```
            ( Afu-λEfu    *        *     )     
 Acl-λEcl = (   0      Aia-λEia    *     )  
            (   0         0     Aig-λEig )
```

ここで：`Afu-λEfu` は、ペア `(Afu,Efu)` が一般化シュール形式にあり、`A-λE` の `nfu` 固定（観測不可能）有限固有値を含み、`Aia-λEia` は上三角で可逆、`Eia` は上三角でニルポテントであり、`nia` 割り当てられた無限一般化固有値を含みます。`Aig-λEig` は上三角で可逆の `Aig` と上三角でニルポテントの `Eig` を持ち、`A-λE` の `ninf` 無限固有値を含みます。行列 `Acl`、`Ecl`、`Q`、`Z` およびベクトル `α` と `β` は、`α./β` がペア `(Acl,Ecl)` の一般化固有値であるように、`GeneralizeSchur` オブジェクト `Scl` に返されます。`nfu`、`nia` および `ninf` の値は、3次元ベクトル `blkdims = [nfu, nia, ninf]` に返されます。

メソッド：一般的なペア `(A-λE,C)` に対して、[1] の修正された一般化シュール法の双対が使用され、ペア `(A+K*C,E+L*G)` が無限固有値のみを持つように `K` と `L` を決定します。
