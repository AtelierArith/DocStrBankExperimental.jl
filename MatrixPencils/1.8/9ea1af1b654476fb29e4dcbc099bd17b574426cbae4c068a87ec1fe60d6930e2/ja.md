```
sklf_right!(A, E, B, C; fast = true, atol1 = 0, atol2 = 0, atol3 = 0, rtol, withQ = true, withZ = true) -> (Q, Z, νr, nc, nfuc, niuc)
```

分割されたフル行ランク行列ペンシルを減少させます 

```
  M - λN = | B | A-λE | n
             m    n
```

`A-λE`が正則であるとき、同等の形 `F - λG = Q'*(M - λN)*diag(I,Z)` に変換します。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用して、`M - λN` を右、有限および無限のクロンネッカー構造（一般化された制御可能性階段形式とも呼ばれる）を示すクロンネッカーのような形 `| Bt | At-λEt |` に変換します：

```
                     |  Bc | Ac-λEc     *          *      | nc
  | Bt | At-λEt | =  |-----|------------------------------|
                     |  0  |  0     Afuc-λEfuc     *      | nfuc
                     |  0  |  0         0      Aiuc-λEiuc | niuc
                        m     nc       nfuc        niuc
```

`Bt = Q'*B`、`At = Q'*A*Z` および `Et = Q'*E*Z` はそれぞれ `B`、`A` および `E` に返され、`C*Z` は `C` に返されます（`C = missing` でない限り）。                 

部分ペンシル `| Bc | Ac-λEc |` はフル行ランク `nc` を持ち、階段形式になっており、`M - λN` の右クロンネッカーインデックスを含みます。 `nr` 次元ベクトル `νr` は、階段形式 `| Bc | Ac-λI |` のブロックの行と列の次元を含み、`i` 番目のブロックは次元 `νr[i] x νr[i-1]` を持ち（`νr[0] = m`）、フル行ランクを持ちます。 差 `νr[i-1]-νr[i]` は、`i = 1, 2, ..., nr` の場合のサイズ `(i-1) x i` の基本クロンネッカーブロックの数です。

`nfuc x nfuc` 部分ペンシル `Afuc-λEfuc` は、`M - λN` の有限固有値（`A - λE` の制御不能な有限固有値とも呼ばれる）を含みます。  

`niuc x niuc` 部分ペンシル `Aiuc-λEiuc` は、`M - λN` の無限固有値（`A - λE` の制御不能な無限固有値とも呼ばれる）を含みます。  

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`B` の非ゼロ要素の絶対許容誤差、および `A`、`E`、`B` の非ゼロ要素の相対許容誤差を指定します。 減少は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、また `fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ左変換は、`withQ = true` の場合に行列 `Q` に蓄積されます。 そうでない場合、`Q` は `nothing` に設定されます。 実行された直交またはユニタリ右変換は、`withZ = true` の場合に行列 `Z` に蓄積されます。 そうでない場合、`Z` は `nothing` に設定されます。
