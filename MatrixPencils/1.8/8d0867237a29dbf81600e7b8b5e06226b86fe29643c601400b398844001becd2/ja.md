```
rm2lspm(NUM, DEN; contr = false, obs = false, atol = 0, rtol) -> (A, B, C, D, blkdims)
```

有理行列 `R(λ) := NUM(λ)./DEN(λ)` の表現を、その厳密に適切な部分として構築し、構造化線形化として実現します。

```
          | A-λI | B | 
          |------|---|
          |  C   | 0 |
```

およびその多項式部分 `D(λ)`。結果として得られる表現は次の式を満たします。

```
 R(λ) = C*inv(λI-A)*B + D(λ).
```

厳密に適切な部分の線形化 `(A-λI,B,C,0)` は一般に非最小です。`contr = true` の場合は制御可能な実現が構築され、`obs = true` の場合は可観測な実現が得られます。行列 `A` はブロック対角行列となり、ベクトル `blkdims` には `A` の対角ブロックのサイズが含まれ、`blkdims[i]` には `i` 番目の対角ブロックのサイズが提供されます。詳細については [1] を参照してください。

`NUM(λ)` は次の形式の多項式行列です。`NUM(λ) = N_1 + λ N_2 + ... + λ**k N_(k+1)` であり、係数行列 `N_i`（`i = 1, ..., k+1`）は 3 次元行列 `NUM` に格納され、`NUM[:,:,i]` には `i` 番目の係数行列 `N_i`（`λ**(i-1)` を掛ける）が含まれます。

`DEN(λ)` は次の形式の多項式行列です。`DEN(λ) = D_1 + λ D_2 + ... + λ**l D_(l+1)` であり、係数行列 `D_i`（`i = 1, ..., l+1`）は 3 次元行列 `DEN` に格納され、`DEN[:,:,i]` には `i` 番目の係数行列 `D_i`（`λ**(i-1)` を掛ける）が含まれます。

あるいは、`NUM(λ)` と `DEN(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列として指定することもできます。この場合、`N(λ)` と `D(λ)` が同じ不確定要素を持つかどうかのチェックは行われません。

多項式部分 `D(λ)` は `D` に含まれ、`D(λ)` の次数が 0 の場合は 2 次元配列、次数が 0 より大きい場合は 3 次元配列となります。後者の場合、`D[:,:,i]` には `i` 番目の係数行列が含まれ、`λ**(i-1)` を掛けます。

キーワード引数 `atol` と `rtol` は、それぞれ `DEN(λ)` の非ゼロ係数に対する絶対および相対許容誤差を指定します。

[1] A. Varga, On computing the Kronecker structure of polynomial and rational matrices using Julia, 2020,  [arXiv:2006.06825](https://arxiv.org/pdf/2006.06825).
