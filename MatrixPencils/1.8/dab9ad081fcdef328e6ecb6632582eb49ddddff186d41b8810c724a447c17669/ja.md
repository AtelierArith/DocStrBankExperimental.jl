```
speigvals(A, E, B, C, D; fast = false, atol1::Real = 0, atol2::Real = 0, rtol::Real=min(atol1,atol2)>0 ? 0 : n*ϵ) -> (val, iz, KRInfo)
```

構造行列ペンシル `M-λN` の（有限および無限の）固有値を返します。

```
          | A-λE | B | 
 M - λN = |------|---|
          |  C   | D |
```

`val` に固有値を、`KRInfo` オブジェクトにクロンカー構造に関する情報を格納します。

クロンカー構造に関する情報は、右クロンカーインデックス `rki`、左クロンカーインデックス `lki`、無限の基本因子 `id`、有限固有値の数 `nf`、通常のランク `nrank` で構成され、`KRInfo` からそれぞれ `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` として取得できます。詳細については、[`pkstruct`](@ref) を参照してください。

固有値の計算は、無限および有限の固有値の分割、無限固有値の重複度、左および右のクロンカーインデックス、通常のランクを示す適切なクロンカー様形式（KLF）にペンシル `M-λN` を縮約することによって行われます。縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解に基づくランク決定が含まれ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。効率のために、縮約は部分的にのみ実行され、実行された直交変換は蓄積されません。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `M` の非ゼロ要素に対する絶対許容誤差、`N` の非ゼロ要素に対する絶対許容誤差、`M` および `N` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `M` の最小次元のサイズ、`ϵ` は `M` の要素型の機械イプシロンです。
