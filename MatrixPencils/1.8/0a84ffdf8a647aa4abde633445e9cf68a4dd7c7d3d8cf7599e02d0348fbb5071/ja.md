```
klf_right(M, N; fast = true, ut = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (F, G, Q, Z, νr, μr, nf, ν, μ)
```

行列ペンシル `M - λN` を同等の形 `F - λG = Q'(M - λN)Z` に縮約します。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用し、変換された行列 `F` と `G` は以下のクロンネッカー様式を示し、右および有限のクロンネッカー構造を持ちます：

```
               |  Mr-λNr    |     *      |    *     |
               |------------|------------|----------|
     F - λG =  |    O       |   Mf-λNf   |    *     |
               |------------|------------|----------|
               |    O       |     0      | Mil-λNil |
```

完全行ランクペンシル `Mr-λNr` は階段形式で、`M-λN` の右クロンネッカー指標を含み、次の形を持ちます：

```
     Mr-λNr  = | Br | Ar-λEr |,
```

ここで `Er` は上三角で非特異です。次元 `nr` のベクトル `νr` と `μr` は、階段形式 `Mr-λNr` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `νr[i] x μr[i]` を持ち、完全行ランクです。差 `μr[i]-νr[i]` は、`i = 1, 2, ..., nr` に対して、サイズ `(i-1) x i` の基本クロンネッカーブロックの数です。`ut = true` の場合、`Mr` の完全行ランク対角ブロックは形 `[0 X]` に縮約され、`X` は上三角で非特異であり、`Nr` の完全列ランク超対角ブロックは形 `[Y; 0]` に縮約され、`Y` は上三角で非特異です。

次元 `nf x nf` のペンシル `Mf-λNf` は正則で、`M-λN` の有限基本因子を含みます。`Nf` は上三角で非特異です。

完全列ランクペンシル `Mil-λNil` は階段形式であり、ペンシル `M-λN` の左クロンネッカー指標と無限基本因子を含みます。次元 `nb` のベクトル `ν` と `μ` は、階段形式 `Mil-λNil` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `ν[i] x μ[i]` を持ち、完全列ランクです。差 `ν[nb-j+1]-μ[nb-j+1]` は、`j = 1, 2, ..., nb` に対して、サイズ `j x (j-1)` の基本クロンネッカーブロックの数です。差 `μ[nb-j+1]-ν[nb-j]` は、`j = 1, 2, ..., nb` に対して、次数 `j` の無限基本因子の数です（`ν[0] = 0`）。`ut = true` の場合、`Mil` の完全列ランク対角ブロックは形 `[X; 0]` に縮約され、`X` は上三角で非特異であり、`Nil` の完全行ランク超対角ブロックは形 `[0 Y]` に縮約され、`Y` は上三角で非特異です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。縮約は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

実行された左直交またはユニタリ変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された右直交またはユニタリ変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。

`注意:` ペンシル `M - λN` が完全行ランクである場合、正則ペンシル `Mil-λNil` は、正方形の上三角対角ブロックを持つ階段形式になります（すなわち、`μ[j] = ν[j]`）、そして差 `ν[nb-j+1]-ν[nb-j]` は、`j = 1, 2, ..., nb` に対して、次数 `j` の無限基本因子の数です（`ν[0] = 0`）。 ```
