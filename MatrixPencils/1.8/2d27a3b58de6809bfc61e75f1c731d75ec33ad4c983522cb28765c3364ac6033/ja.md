```
sklf_left(A, E, B, C, D; fast = true, ut = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (F, G, Q, Z, ν, μ, nf, νl, μl)
```

構造行列ペンシル `M - λN` を次のように変換します。

```
           | A-λE | B | 
  M - λN = |------|---|
           |  C   | D |
```

同等の形 `F - λG = Q'*(M - λN)*Z` に変換するために、直交またはユニタリ変換行列 `Q` と `Z` を使用します。変換された行列 `F` と `G` は、有限および左クロンカー構造を示す次のようなクロンカー様式になります。

```
            |  Mri-λNri  |     *      |    *    |
            |------------|------------|---------|
  F - λG =  |    O       |   Mf-λNf   |    *    |
            |------------|------------|---------|
            |    O       |     0      |  Ml-λNl |
```

完全行ランクペンシル `Mri-λNri` は階段形式であり、ペンシル `M-λN` の右クロンカー指標と無限の基本除数を含みます。

`nf x nf` ペンシル `Mf-λNf` は正則であり、`M-λN` の有限基本除数を含みます。Nf は上三角行列であり、非特異です。

完全列ランクペンシル `Ml-λNl` は階段形式であり、`M-λN` の左クロンカー指標を含み、次の形を持ちます。

```
               | Al-λEl |
     Ml-λNl  = |--------|,
               |   Cl   |
```

ここで `El` は上三角行列であり、非特異です。

`nb` 次元ベクトル `ν` と `μ` は、階段形式 `Mri-λNri` のブロックの行および列の次元を含み、`i` 番目のブロックは `ν[i] x μ[i]` の次元を持ち、完全行ランクです。`μ[i]-ν[i]` の差は、`i = 1, 2, ..., nb` に対して、サイズ `(i-1) x i` の基本クロンカーブロックの数です。`ν[i]-μ[i+1]` の差は、`i = 1, 2, ..., nb` に対して、次数 `i` の無限基本除数の数です（`μ[nb+1] = 0`）。`ut = true` の場合、完全行ランクの対角ブロック `Mri` は `[0 X]` の形に変換され、`X` は上三角行列であり、非特異です。完全列ランクの超対角ブロック `Nri` は `[Y; 0]` の形に変換され、`Y` は上三角行列であり、非特異です。

`nl` 次元ベクトル `νl` と `μl` は、階段形式 `Ml-λNl` のブロックの行および列の次元を含み、`j` 番目のブロックは `νl[j] x μl[j]` の次元を持ち、完全列ランクです。`νl[nl-j+1]-μl[nl-j+1]` の差は、`j = 1, 2, ..., nl` に対して、サイズ `j x (j-1)` の基本クロンカーブロックの数です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、`M` および `N` の非ゼロ要素の相対許容誤差を指定します。変換は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

実行された左直交またはユニタリ変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された右直交またはユニタリ変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。

`注意:` ペンシル `M - λN` が完全列ランクの場合、正則ペンシル `Mri-λNri` は、正方形の上三角対角ブロックを持つ階段形式になります（すなわち、`μ[i] = ν[i]`）、および `ν[i+1]-ν[i]` の差は、`i = 1, 2, ..., nb` に対して、次数 `i` の無限基本除数の数です（`ν[nb+1] = 0`）。
