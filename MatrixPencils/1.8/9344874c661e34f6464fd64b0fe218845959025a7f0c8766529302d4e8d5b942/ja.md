```
sklf_right(A, E, B, C, D; fast = true, ut = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (F, G, Q, Z, νr, μr, nf, ν, μ)
```

構造行列ペンシル `M - λN` を次のように変形します。

```
           | A-λE | B | 
  M - λN = |------|---|
           |  C   | D |
```

同等の形 `F - λG = Q'*(M - λN)*Z` に変形するために、直交またはユニタリ変換行列 `Q` と `Z` を使用します。変換された行列 `F` と `G` は、右側および有限のクロンカー構造を示す次のようなクロンカー様式である必要があります。

```
            |  Mr-λNr    |     *      |    *     |
            |------------|------------|----------|
  F - λG =  |    O       |   Mf-λNf   |    *     |
            |------------|------------|----------|
            |    O       |     0      | Mil-λNil |
```

完全行ランクペンシル `Mr-λNr` は階段形式で、`M-λN` の右クロンカー指標を含み、次の形を持ちます。

```
     Mr-λNr  = | Br | Ar-λEr |,
```

ここで `Er` は上三角で非特異です。次元 `nr` のベクトル `νr` と `μr` は、階段形式 `Mr-λNr` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `νr[i] x μr[i]` を持ち、完全行ランクです。差 `μr[i]-νr[i]` は、`i = 1, 2, ..., nr` に対して、サイズ `(i-1) x i` の基本クロンカーブロックの数です。`ut = true` の場合、`Mr` の完全行ランク対角ブロックは、`[0 X]` の形に変形され、`X` は上三角で非特異です。また、`Nr` の完全列ランク超対角ブロックは、`[Y; 0]` の形に変形され、`Y` は上三角で非特異です。

`nf x nf` ペンシル `Mf-λNf` は正則で、`M-λN` の有限基本因子を含みます。`Nf` は上三角で非特異です。

完全列ランクペンシル `Mil-λNil` は階段形式で、ペンシル `M-λN` の左クロンカー指標と無限基本因子を含みます。次元 `nb` のベクトル `ν` と `μ` は、階段形式 `Mil-λNil` のブロックの行および列の次元を含み、`i` 番目のブロックは次元 `ν[i] x μ[i]` を持ち、完全列ランクです。差 `ν[nb-j+1]-μ[nb-j+1]` は、`j = 1, 2, ..., nb` に対して、サイズ `j x (j-1)` の基本クロンカーの数です。差 `μ[nb-j+1]-ν[nb-j]` は、`j = 1, 2, ..., nb` に対して、次数 `j` の無限基本因子の数です（`ν[0] = 0`）。`ut = true` の場合、`Mil` の完全列ランク対角ブロックは、`[X; 0]` の形に変形され、`X` は上三角で非特異です。また、`Nil` の完全行ランク超対角ブロックは、`[0 Y]` の形に変形され、`Y` は上三角で非特異です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。変形は、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

左側の直交またはユニタリ変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。右側の直交またはユニタリ変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。

`注意:` ペンシル `M - λN` が完全行ランクである場合、正則ペンシル `Mil-λNil` は、正方形の上三角対角ブロックを持つ階段形式になります（すなわち、`μ[j] = ν[j]`）、および差 `ν[nb-j+1]-ν[nb-j]` は、`j = 1, 2, ..., nb` に対して、次数 `j` の無限基本因子の数です（`ν[0] = 0`）。 ```
