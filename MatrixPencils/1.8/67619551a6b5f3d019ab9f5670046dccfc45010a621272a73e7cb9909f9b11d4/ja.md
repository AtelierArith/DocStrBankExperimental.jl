```
klf_left!(M, N; fast = true, roff = 0, coff = 0, rtrail = 0, ctrail = 0, atol = 0, rtol, withQ = true, withZ = true) -> (Q, Z, ν, μ, nf, νl, μl, tol)
```

分割された行列ペンシル `M - λN` を縮小します（`*` は関連しないサブペンシルを表します）

```
          [   *         *        *  ] roff
 M - λN = [   0      M22-λN22    *  ] m
          [   0         0        *  ] rtrail
            coff        n     ctrail
```

等価な形 `F - λG = Q1'*(M - λN)*Z1` に変換します。ここで、直交またはユニタリ変換行列 `Q1` と `Z1` を使用し、サブペンシル `M22 - λN22` が有限および左クロンカー構造を示す以下のクロンカー様式に変換されます。

```
               |  Mri-λNri  |     *      |    *    |
               |------------|------------|---------|
 F22 - λG22 =  |    O       |   Mf-λNf   |    *    |
               |------------|------------|---------|
               |    O       |     0      |  Ml-λNl |
```

`F` と `G` はそれぞれ `M` と `N` に返されます。

完全行ランクペンシル `Mri-λNri` は階段形式であり、サブペンシル `M22 - λN22` の右クロンカーインデックスと無限の基本因子を含みます。 `nb` 次元のベクトル `ν` と `μ` は、階段形式 `Mri-λNri` のブロックの行および列の次元を含み、`i` 番目のブロックは `ν[i] x μ[i]` の次元を持ち、完全行ランクを持ちます。 `μ[i]-ν[i]` の差は、`i = 1, 2, ..., nb` の場合、サイズ `(i-1) x i` の基本クロンカーブロックの数です。 `ν[i]-μ[i+1]` の差は、`i = 1, 2, ..., nb` の場合、次数 `i` の無限の基本因子の数です（`μ[nb+1] = 0`）。

`nf x nf` ペンシル `Mf-λNf` は正則であり、`M-λN` の有限の基本因子を含みます。 `Nf` は上三角であり、非特異です。

完全列ランクペンシル `Ml-λNl` は階段形式であり、`M-λN` の左クロンカーインデックスを含み、次の形式を持ちます。

```
               | Al-λEl |
     Ml-λNl  = |--------|,
               |   Cl   |
```

ここで `El` は上三角であり、非特異です。 `nl` 次元のベクトル `νl` と `μl` は、階段形式 `Ml-λNl` のブロックの行および列の次元を含み、`j` 番目のブロックは `νl[nl-j+1] x μl[nl-j+1]` の次元を持ち、完全列ランクを持ちます。 `νl[nl-j+1]-μl[nl-j+1]` の差は、`j = 1, 2, ..., nl` の場合、サイズ `j x (j-1)` の基本クロンカーブロックの数です。

キーワード引数 `atol` と `rtol` は、`M` の非ゼロ要素に対する絶対および相対許容誤差を指定します。 `M` の非ゼロ要素に対する内部で使用される絶対許容誤差は `tol` に返されます。 縮小は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、また `fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左の直交またはユニタリ変換は、行列 `Q` に蓄積されます（すなわち、`Q <- Q*Q1`） `withQ = true` の場合。そうでない場合、`Q` は変更されません。 実行された右の直交またはユニタリ変換は、行列 `Z` に蓄積されます（すなわち、`Z <- Z*Z1`） `withZ = true` の場合。そうでない場合、`Z` は変更されません。

`注意:` ペンシル `M22 - λN22` が完全列ランクである場合、正則ペンシル `Mri-λNri` は平方対角ブロックを持つ階段形式になります（すなわち、`μ[i] = ν[i]`）、および `ν[i]-ν[i+1]` の差は、`i = 1, 2, ..., nb` の場合、次数 `i` の無限の基本因子の数です（`ν[nb+1] = 0`）。 ```
