```
salocd(A, E, C; evals, sdeg, disc = false, atol1 = 0, atol2 = 0, atol3 = 0, rtol, sepinf = true, fast = true) -> (K, Scl, blkdims)
```

ペア `(A-λE,C)` に対して、`A-λE` が正則ペンシルであるとき、安定度パラメータ `sdeg` と安定性タイプパラメータ `disc` によって指定された安定性領域 `Cs` にすべての有限固有値が存在するような行列 `K` を計算します。 `disc = false` の場合、`Cs` は実部が最大 `sdeg` である複素数の集合ですが、`disc = true` の場合、`Cs` は絶対値が最大 `sdeg` である複素数の集合です（すなわち、原点を中心とした半径 `sdeg` の円の内部）。 `evals` は、ペンシル `A+K*C-λE` のための有限の望ましい固有値のセットを含む実数または複素数のベクトルです。実データ `A`、`E`、および `B` に対して、`evals` は結果として得られる `F` も実行列であることを保証するために自己共役複素セットでなければなりません。

`A` の次数が `n` であるペア `(A-λE,C)` に対して、割り当て可能な有限固有値の数は `nfc := n-ninf-nfu` であり、ここで `ninf` は `A-λE` の無限固有値の数、`nfu` は `A-λE` の固定有限固有値の数です。割り当て可能な有限固有値は *観測可能な有限固有値* と呼ばれ、固定有限固有値は *非観測可能な有限固有値* と呼ばれます（これらはペンシル `[A-λE; C]` の有限ゼロです）。スペクトルの割り当ては、安定性領域 `Cs` の外にある `A-λE` の観測可能な有限固有値を `evals` で提供された固有値に順次置き換えることによって達成されます。`Cs` にある `A-λE` のすべての有限固有値は変更されずに保持されます。`evals` に指定された固有値の数が `Cs` の外にある `A-λE` の観測可能な有限固有値の数よりも少ない場合（例えば、`evals = missing` の場合）、`A-λE` の観測可能な有限固有値のいくつかは `Cs` の境界上の最も近い値に割り当てられます。`sdeg = missing` かつ `evals = missing` の場合、`disc = false` のときは `sdeg` のデフォルト値は -0.05 であり、`disc = true` のときは 0.95 です。

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、および `A`、`E`、`C` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は `A` の要素型のマシンイプシロンです。

キーワード引数 `sepinf` は、ペンシル `A-λE` の無限固有値の事前分離のオプションを指定します。`sepinf = false` の場合、無限固有値の分離は行われず、`sepinf = true`（デフォルトオプション）の場合、有限固有値から無限固有値の事前分離が行われます。`E` が非特異である場合、`sepinf = false` を使用することが推奨されます。`E` が数値的に特異である場合、オプション `sepinf = false` が使用されます。有限固有値と無限固有値の分離は、`fast = true` の場合は列ピボットを用いたランクを明らかにするQR分解に基づくランク決定を使用し、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

得られたペンシル `Acl-λEcl := Q'*(A+K*C-λE)*Z` は、ここで `Q` と `Z` は一般化シュール形式（GSF）でペア `(Acl,Ecl)` を得るために使用される直交/ユニタリ行列であり、次の形を持ちます。

```
            ( Afu-λEfu    *        *       *    )     
 Acl-λEcl = (   0      Afa-λEfa    *       *    )  
            (   0         0     Afg-λEfg   *    )    
            (   0         0        0     Ai-λEi )
```

ここで、`Afu-λEfu` は `Cs` の外にある `A-λE` の `nfu` の非観測可能な有限固有値を含み、`Afa-λEfa` は `Cs` にある `nfa` の割り当てられた有限一般化固有値を含み、`Afg-λEfg` は `Cs` にある `nfg` の有限固有値を含み、`Ai-λEi` は上三角で可逆な `Ai` と上三角でニルポテントな `Ei` を持ち、`A-λE` の `ninf` の無限固有値を含みます。行列 `Acl`、`Ecl`、`Q`、`Z` および `α` と `β` のベクトルは、`α./β` がペア `(Acl,Ecl)` の一般化固有値であるように、`GeneralizeSchur` オブジェクト `Scl` に返されます。`nfu`、`nfa`、`nfg` および `ninf` の値は、4次元ベクトル `blkdims = [nfu, nfa, nfg, ninf]` に返されます。

メソッド:  `E = I` のペア `(A-λE,C)` に対して、[1] の双対シュール法が使用され、一般的なペア `(A-λE,C)` に対しては、[2] の双対一般化シュール法が使用され、双対ペア `(A'-λE',C')` のR安定化問題を解決します。
