```
pmdeg(P) -> deg
```

多項式行列 `P(λ)` の次数 `deg` を求めます。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形を持つ次数 `k` の多項式行列であり、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納されています。ここで、`P[:,:,i]` は `i` 番目の係数行列 `P_i`（`λ**(i-1)` を掛ける）を含みます。`P(λ)` の次数は `deg = j-1` であり、ここで `j` は `P[:,:,j]` が非ゼロである最大のインデックスです。ゼロ多項式行列の次数は `deg = -1` と定義されます。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。`P(λ)` の次数は `P(λ)` の要素の中で最大の次数です。ゼロ多項式行列の次数は `-1` と定義されます。
