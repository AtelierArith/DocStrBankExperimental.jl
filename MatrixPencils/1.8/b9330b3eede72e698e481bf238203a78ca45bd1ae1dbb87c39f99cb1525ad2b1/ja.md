```
 spm2lps(T, U, V, W; fast = true, contr = false, obs = false, minimal = false, atol = 0, rtol) -> (A, E, B, F, C, G, D, H)
```

構造化線形化を構築します。

```
          | A-λE | B-λF | 
 M - λN = |------|------|
          | C-λG | D-λH |
```

構造化多項式行列の

```
          | -T(λ) | U(λ) |
   P(λ) = |-------|------|
          |  V(λ) | W(λ) |
```

を次のようにします。`V(λ)*inv(T(λ))*U(λ)+W(λ) = (C-λG))*inv(λE-A)*(B-λF)+D-λH`。結果として得られる線形化 `M - λN` は、`minimal = false` の場合は一部を保持し、`minimal = true` の場合は `P(λ)` の完全なクロンケル構造を保持します。この場合、`A-λE` の次数 `n` は可能な限り小さく、`M - λN` は `P(λ)` の強い線形化です。

`T(λ)`, `U(λ)`, `V(λ)`, および `W(λ)` は、`X(λ) = X_1 + λ X_2 + ... + λ**k X_(k+1)` の形の多項式行列として指定できます。ここで、`X = T`, `U`, `V`, `W` のために、係数行列 `X_i`, `i = 1, ..., k+1` は 3 次元行列 `X` に格納され、`X[:,:,i]` には `i` 番目の係数行列 `X_i` (λ**(i-1) を掛けたもの) が含まれます。

`T(λ)`, `U(λ)`, `V(λ)`, および `W(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

計算された構造化線形化は次の条件を満たします：

(1) `A-λE` は正則です；

(2) `rank[B-λF A-λE] = n` (強い制御可能性) は `minimal = true` または `contr = true` の場合に成り立ち、この場合有限かつ右のクロンケル構造が保持されます；

(3) `rank[A-λE; C-λG] = n` (強い可観測性) は `minimal = true` または `obs = true` の場合に成り立ち、この場合有限かつ左のクロンケル構造が保持されます。

キーワード引数 `atol` と `rtol` は、それぞれ行列 `T(λ)`, `U(λ)`, `V(λ)` および `W(λ)` の非ゼロ係数に対する絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `nt*ϵ` であり、ここで `nt` は正方行列 `T(λ)` のサイズで、`ϵ` はその係数の要素型のマシンイプシロンです。
