```
saloc(A, B; evals, sdeg, disc = false, atol1 = 0, atol2 = 0, rtol) -> (F, Scl, blkdims)
```

ペア `(A,B)` に対して、行列 `A+B*F` のすべての固有値が安定度パラメータ `sdeg` と安定性タイプパラメータ `disc` によって指定された安定性領域 `Cs` にあるような行列 `F` を計算します。 `disc = false` の場合、`Cs` は実部が最大 `sdeg` の複素数の集合ですが、`disc = true` の場合、`Cs` は原点を中心とした半径 `sdeg` の円の内部にある複素数の集合です。 `evals` は、`Cs` 内の行列 `A+B*F` の望ましい固有値を含む実数または複素数のベクトルです。実データ `A` と `B` に対して、`evals` は結果として得られる `F` も実行列であることを保証するために自己共役複素集合でなければなりません。

`A` の次数が `n` のペア `(A,B)` に対して、割り当て可能な固有値の数は `nc := n-nu` であり、ここで `nu` は `A` の固定固有値の数です。割り当て可能な固有値は制御可能固有値と呼ばれ、固定固有値は *制御不能固有値* と呼ばれます（これらはペンシル `[A-λI B]` のゼロです）。スペクトルの割り当ては、安定性領域 `Cs` の外にある `A` の *制御可能固有値* を `evals` で提供された固有値に順次置き換えることによって達成されます。`Cs` にある `A` のすべての固有値は変更されずに保持されます。`evals` に指定された固有値の数が `Cs` の外にある `A` の制御可能固有値の数よりも少ない場合（例えば、`evals = missing` の場合）、`A` の制御可能固有値の一部は `Cs` の境界上の最も近い値に割り当てられます。`sdeg = missing` かつ `evals = missing` の場合、`disc = false` の場合は `sdeg` のデフォルト値は -0.05 であり、`disc = true` の場合は 0.95 です。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素の絶対許容誤差、`B` の非ゼロ要素の絶対許容誤差、および `A` と `B` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は `A` の要素型のマシンイプシロンです。

結果として得られる行列 `Acl := Z'*(A+B*F)*Z` はシュル形式であり、ここで `Z` は行列 `A+B*F` をシュル形式にするために使用される直交/ユニタリ行列です。形式は次のようになります。

```
       ( Ag  *   *  )     
 Acl = ( 0   Aa  *  ) 
       ( 0   0   Au )
```

ここで、`Ag` は `Cs` にある `A` の `ng` 個の固有値を含み、`Aa` は `Cs` に割り当てられた `na` 個の固有値を含み、`Au` は `Cs` の外にある `A` の `nu` 個の制御不能固有値を含みます。行列 `Acl` と `Z`、および `Acl` の固有値のベクトル `α`（`A+B*F` の固有値でもある）は、`Schur` オブジェクト `Scl` に返されます。`ng`、`na`、および `nu` の値は 3 次元ベクトル `blkdims = [ng, na, nu]` に返されます。

メソッド:  [1] のシュル法が、制御不能なペアにも拡張されて使用されます。

参考文献:

[1] A. Varga.      A Schur method for pole assignment.     IEEE Trans. on Automatic Control, vol. 26, pp. 517-519, 1981. ```
