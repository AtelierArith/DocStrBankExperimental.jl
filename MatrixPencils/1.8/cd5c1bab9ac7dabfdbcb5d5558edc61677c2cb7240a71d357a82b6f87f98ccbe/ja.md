```
lps2pm(A, E, B, F, C, G, D, H; fast = true, atol1 = 0, atol2 = 0, gaintol = 0, rtol = min(atol1,atol2) > 0 ? 0 : n*ϵ, val) -> P
```

多項式行列 `P(λ) = (C-λG)*inv(λE-A)*(B-λF)+D-λH` を、その構造化線形化に対応させて構築します。

```
 | A-λE | B-λF | 
 |------|------|
 | C-λG | D-λH |
```

各多項式エントリの係数をその根と対応するゲインから明示的に決定します。

キーワード引数 `atol1` と `atol2` は、それぞれ `A`、`B`、`C`、`D` および `E`、`F`、`G`、`H` の要素に対する絶対許容誤差を指定し、`rtol` は `A`、`B`、`C`、`D`、`E`、`F`、`G`、`H` の非ゼロ要素に対する相対許容誤差を指定します。デフォルトの相対許容誤差は `(n+2)*ϵ` であり、ここで `n` は `A` のサイズ次元のサイズで、`ϵ` は `A` の係数の要素型の機械イプシロンです。

キーワード引数 `gaintol` は、ゲイン行列 `C*inv(γE-A)*B+D` の非ゼロ要素の大きさの閾値を指定します。ここで `γ = val` もし `val` が数値である場合、または `γ` は `val = missing` の場合に単位大きさのランダムに選ばれた複素数値です。一般に、`val` は `P` のいずれかのエントリの根であってはなりません。
