```
sklf_leftfin!(A, E, C, B; fast = true, atol1 = 0, atol2 = 0,  
              rtol, withQ = true, withZ = true) -> (Q, Z, μl, no, nuo)
```

分割されたフルコラムランク行列ペンシルを削減します 

```
  M - λN = | A-λE | n
           |  C   | p
              n
```

`A-λE` が正則であるとき、同等の形 `F - λG = Q'*(M - λN)*diag(I,Z)` に変換します。ここで、`Q` と `Z` は直交またはユニタリ変換行列であり、`M - λN` は有限の固有値の分離を示す階段形式に変換されます：

```
                | Auo-λEuo  |   *    | nuo
  | At-λEt |    |   0       | Ao-λEo | no
  |--------| =  |-----------|--------|
  |  Ct    |    |   0       |   Co   | p
                    nuo         no
```

`Ct = C*Z`、`At = Q'*A*Z` および `Et = Q'*E*Z` はそれぞれ `C`、`A` および `E` に返され、`Q'*B` は `B` に返されます（`B = missing` でない限り）。結果として得られる `Et` は上三角行列です。もし `E` がすでに上三角行列であれば、`E` を上三角形式にするための予備的な削減は行われません。

部分ペンシル `| Ao-λEo |` は、すべての有限値の `λ` に対してフルコラムランク `no` を持ち、階段形式にあり、`|   Co   |` を含み、`E` が非特異であれば、`M - λN` の左クロンカー指標を含みます。次元 `nl` のベクトル `μl` は、階段形式のブロックの行と列の次元を含み、`i` 番目のブロックは次元 `μl[nl-i] x μl[nl-i+1]` を持ち（μl[0] = p）、フルコラムランクを持ちます。もし `E` が非特異であれば、`μl[nl-i]-μl[nl-i+1]` の差は、`i = 1, 2, ..., nl` に対してサイズ `i x (i-1)` の基本クロンカーブロックの数です。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`C` の非ゼロ要素の絶対許容誤差、および `A` と `C` の非ゼロ要素の相対許容誤差を指定します。削減は、`fast = true` の場合は列ピボッティングを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ左変換は、`withQ = true` の場合に行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された直交またはユニタリ右変換は、`withZ = true` の場合に行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。

`注意:` この関数は、逆の入力パラメータ `E` と `A`（すなわち、`A` と `E` の代わりに）で呼び出されると、ペンシル `M - λN` のすべての無限および非ゼロ有限固有値の分離を行います。 ```
