```
salocinf(A, E, B; atol1 = 0, atol2 = 0, atol3 = 0, rtol, fast = true) -> (F, G, Scl, blkdims)
```

制御可能なペア `(A-λE,B)` の計算を行います。ここで、`A-λE` は通常のペンシルであり、すべての固有値が無限大となるような行列 `F` と `G` を求めます。固定された（制御不能な）有限固有値を持つペア `(A-λE,B)` に対しては、割り当て可能な（制御可能な）有限固有値のみが無限大に移動されます。

`A` の次数が `n` であるペア `(A-λE,B)` に対して、割り当て可能な無限固有値の数は `nia := n-ninf-nfu` であり、ここで `ninf` は `A-λE` の無限固有値の数、`nfu` は `A-λE` の固定された有限固有値の数です。割り当て可能な有限固有値は *制御可能な有限固有値* と呼ばれ、固定された有限固有値は *制御不能な有限固有値* と呼ばれます（これらはペンシル `[A-λE B]` の有限ゼロです）。スペクトルの割り当ては、`A-λE` の制御可能な有限固有値を無限固有値に順次置き換えることによって達成されます。`A-λE` のすべての無限固有値は変更されずに保持されます。

キーワード引数 `atol1`、`atol2`、`atol3`、および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`E` の非ゼロ要素の絶対許容誤差、`B` の非ゼロ要素の絶対許容誤差、そして `A`、`E`、`B` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `ϵ` は `A` の要素型のマシンイプシロンです。

有限および無限固有値の予備的な分離は、`fast = true` の場合は列ピボットを用いたランクを明らかにするQR分解に基づくランク決定を使用し、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

得られたペンシル `Acl-λEcl := Q'*(A+B*F-λ(E+B*G))*Z` は、ここで `Q` と `Z` は一般化シュール形式（GSF）でペア `(Acl,Ecl)` を得るために使用される直交/ユニタリ行列であり、次の形を持ちます。

```
            ( Aig-λEig    *        *     )     
 Acl-λEcl = (   0      Aia-λEia    *     )  
            (   0         0     Afu-λEfu )
```

ここで：`Aig-λEig` は上三角で可逆な `Aig` と上三角でニルポテントな `Eig` を持ち、`A-λE` の `ninf` 個の無限固有値を含みます； `Aia-λEia` は上三角で可逆な `Aia` と上三角でニルポテントな `Eia` を持ち、`nia` 個の割り当てられた無限一般化固有値を含みます；および `Afu-λEfu` は一般化シュール形式のペア `(Afu,Efu)` を持ち、`A-λE` の `nfu` 個の固定（制御不能）有限固有値を含みます。行列 `Acl`、`Ecl`、`Q`、`Z` と、`α` と `β` のベクトル（`α./β` がペア `(Acl,Ecl)` の一般化固有値である）を含む `GeneralizeSchur` オブジェクト `Scl` が返されます。`ninf`、`nia`、`nfu` の値は3次元ベクトル `blkdims = [ninf, nia, nfu]` に返されます。

メソッド：一般的なペア `(A-λE,B)` に対して、[1] の修正された一般化シュール法を使用して、ペア `(A+B*F,E+B*G)` が無限固有値のみを持つように `F` と `G` を決定します。
