```
lpseval(A, E, B, F, C, G, D, H, val; atol1, atol2, rtol, fast = true) -> Gval
```

`Gval`を評価します。これは、`G(λ) = (C-λG)*inv(λE-A)*(B-λF)+D-λH`の値で、`λ = val`のときのものです。計算された`Gval`は、`val`が`G(λ)`の極（有限または無限）である場合、無限のエントリを持ちます。`val`が有限であり、`val*E-A`が特異であるか、`val`が無限であり、`E`が特異である場合、`Gval`のエントリは、各入力-出力チャネルに対して別々に評価され、記述子システムに基づく最小実現が使用されます。

キーワード引数`atol1`、`atol2`、および`rtol`は、それぞれ、行列`A`、`B`、`C`、`D`の非ゼロ要素に対する絶対許容誤差、行列`E`、`F`、`G`、`H`の非ゼロ要素に対する絶対許容誤差、および行列`A`、`B`、`C`、`D`、`E`、`F`、`G`、`H`の非ゼロ要素に対する相対許容誤差を指定します。

個々の入力-出力チャネルの最小実現の計算は、ペンシル操作アルゴリズムに依存しており、これは、`fast = true`の場合は列ピボットを用いたランクを明らかにするQR分解を使用するか、またはSVD分解に基づくランク決定を行います。SVD分解に基づくランク決定は一般的により信頼性がありますが、関与する計算コストは高くなります。
