```
ispmregular(P; fastrank = true, atol::Real = 0, rtol::Real = atol>0 ? 0 : n*ϵ) -> Bool
```

多項式行列 `P(λ)` が正則であるかどうかをテストします（すなわち、`P(λ)` が正方行列であり、${\small\det(P(λ)) \not\equiv 0}$ であること）。基礎となる計算手順は、正方行列 `P(λ)` の通常のランクがその次数に等しいことを確認します。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形を持つ次数 `k` の多項式行列であり、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納されており、`P[:,:,i]` には `λ**(i-1)` を掛けた `i` 番目の係数行列 `P_i` が含まれています。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。`fastrank = true` の場合、ランクは `P(γ)` の特異値のうち、絶対値が `max(atol, rtol*σ₁)` より大きいものの数を数えることによって評価されます。ここで、`σ₁` は `P(γ)` の最大特異値であり、`γ` は絶対値が1のランダムに生成された複素数です。`fastrank = false` の場合、まず次数 `n` の伴随形式 `M-λN` で `P(λ)` の線形化が構築され、その後、通常のランク `nr` が決定されます。`P(λ)` は `nr = n` の場合に正則です。

キーワード引数 `atol` と `rtol` は、それぞれ `P(λ)` の非ゼロ係数に対する絶対および相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `P(λ)` の最小次元のサイズで、`ϵ` はその係数の要素型のマシンイプシロンです。
