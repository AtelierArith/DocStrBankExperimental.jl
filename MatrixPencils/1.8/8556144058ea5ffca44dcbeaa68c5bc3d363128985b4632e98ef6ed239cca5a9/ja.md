```
spzeros(A, E, B, C, D; fast = false, atol1::Real = 0, atol2::Real = 0, rtol::Real=min(atol1,atol2)>0 ? 0 : n*ϵ) -> (val, iz, KRInfo)
```

構造行列ペンシル `M-λN` の（有限および無限の）スミスゼロを返します。

```
          | A-λE | B | 
 M - λN = |------|---|
          |  C   | D |
```

`val` に無限ゼロの重複度に関する情報を、`iz` に無限ゼロの重複度に関する情報を、`KRInfo` オブジェクトにクロンカー構造に関する情報を提供します。

無限ゼロの重複度に関する情報はベクトル `iz` に提供され、各 `i` 番目の要素 `iz[i]` は `k-1` に等しく、ここで `k` は無限の基本因子の次数で `k > 0` です。 `val` に含まれる無限ゼロの数は `iz` の成分の合計です。

クロンカー構造に関する情報は、右クロンカーインデックス `rki`、左クロンカーインデックス `lki`、無限基本因子 `id`、有限固有値の数 `nf`、通常のランク `nrank` から `KRInfo` で取得できます。それぞれ `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` です。詳細については [`pkstruct`](@ref) を参照してください。

ゼロの計算は、無限および有限固有値の分割、無限固有値の重複度、左および右クロンカーインデックス、通常のランクを示す適切なクロンカー様形式（KLF）にペンシル `M-λN` を縮約することによって行われます。縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を含み、`fast = false` の場合はより信頼性の高いSVD分解を含みます。効率のために、縮約は部分的にのみ行われ、実行された直交変換は蓄積されません。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、`M` および `N` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` で、ここで `n` は `M` の最小次元のサイズ、`ϵ` は `M` の要素型の機械イプシロンです。
