```
sklf_rightfin!(A, E, B, C; fast = true, atol1 = 0, atol2 = 0,  
               rtol, withQ = true, withZ = true) -> (Q, Z, νr, nc, nuc)
```

分割されたフル行ランク行列ペンシルを削減します 

```
  M - λN = | B | A-λE | n
             m    n
```

`A-λE`が正則であるとき、同等の形 `F - λG = Q'*(M - λN)*diag(I,Z)` に変換します。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用して、`M - λN` を階段形式 `| Bt | At-λEt |` に変換し、その有限固有値の分離を示します：

```
                     |  Bc | Ac-λEc     *    | nc
  | Bt | At-λEt | =  |-----|-----------------|
                     |  0  |  0     Auc-λEuc | nuc
                        m     nc      nuc
```

`Bt = Q'*B`、`At = Q'*A*Z` および `Et = Q'*E*Z` はそれぞれ `B`、`A`、`E` に返され、`C*Z` は `C` に返されます（`C = missing` でない限り）。結果として得られる `Et` は上三角行列です。もし `E` がすでに上三角行列であれば、`E` を上三角形にするための予備的な削減は行われません。

部分ペンシル `| Bc | Ac-λEc |` は、すべての有限値の `λ` に対してフル行ランク `nc` を持ち、階段形式になっており、`E` が可逆であれば、`M - λN` の右クロンカー指数を含みます。`nr` 次元のベクトル `νr` は、階段形式 `| Bc | Ac-λI |` のブロックの行と列の次元を含み、`i` 番目のブロックは次元 `νr[i] x νr[i-1]` を持ち（`νr[0] = m`）、フル行ランクを持ちます。もし `E` が可逆であれば、`i = 1, 2, ..., nr` に対して `νr[i-1]-νr[i]` の差は、サイズ `(i-1) x i` の基本クロンカーブロックの数です。

`nuc x nuc` の部分ペンシル `Auc-λEuc` は、`M - λN` の有限固有値（`A - λE` の制御不能な有限固有値とも呼ばれる）を含みます。もし `E` が特異であれば、`Auc-λEuc` は `M - λN` の無限固有値の一部（`A - λE` の制御不能な無限固有値とも呼ばれる）を含む場合があります。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `A` の非ゼロ要素の絶対許容誤差、`B` の非ゼロ要素の絶対許容誤差、`A` および `B` の非ゼロ要素の相対許容誤差を指定します。削減は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ左変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。実行された直交またはユニタリ右変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。

`注意:` この関数は、入力パラメータ `E` と `A` を逆にして呼び出すと（すなわち、`A` と `E` の代わりに）、ペンシル `M - λN` のすべての無限および非ゼロ有限固有値の分離を行います。
