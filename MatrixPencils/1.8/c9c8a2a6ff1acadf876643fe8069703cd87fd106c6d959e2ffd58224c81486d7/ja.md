```
klf_rlsplit(M, N; fast = true, finite_infinite = false, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (F, G, Q, Z, ν, μ, n, m, p)
```

行列ペンシル `M - λN` を、直交またはユニタリ変換行列 `Q` と `Z` を使用して、同等の形 `F - λG = Q'*(M - λN)*Z` に還元します。このとき、変換された行列 `F` と `G` は以下のいずれかのクロンカー様式の形になります。

(1) `finite_infinite = false` の場合、

```
               | Mri-λNri |     *      |
      F - λG = |----------|------------|, 
               |    O     | Mfl-λNfl   |
```

ここで、サブペンシル `Mri-λNri` は右クロンカー構造と無限の基本因子を含み、サブペンシル `Mfl-λNfl` はペンシル `M-λN` の有限および左クロンカー構造を含みます。

完全行ランクのサブペンシル `Mri-λNri` は階段形式になっています。 `nb` 次元のベクトル `ν` と `μ` は、階段形式のブロックの行および列の次元をそれぞれ含み、`i` 番目のブロックは次元 `ν[i] x μ[i]` を持ち、完全行ランクです。 `μ[i]-ν[i]` の差は、`i = 1, 2, ..., nb` に対して、サイズ `(i-1) x i` の基本クロンカーブロックの数です。 `ν[i]-μ[i+1]` の差は、`i = 1, 2, ..., nb` に対して、次数 `i` の無限の基本因子の数です（`μ[nb+1] = 0`）。

完全列ランクのサブペンシル `Mfl-λNfl` は次の形です。

```
                 | A-λE | 
      Mfl-λNfl = |------| ,        
                 |  C   |
```

ここで、`E` は非特異な上三角行列の `nxn` 行列であり、`A` と `C` はそれぞれ `nxn` および `pxn` 次元の行列であり、`m = 0` です。

(2) `finite_infinite = true` の場合、

```
               | Mrf-λNrf |     *      |
      F - λG = |----------|------------|, 
               |    O     | Mil-λNil   |
```

ここで、サブペンシル `Mrf-λNrf` は右クロンカーおよび有限クロンカー構造を含み、サブペンシル `Mil-λNil` は左クロンカー構造と無限の基本因子を含みます。

完全行ランクのサブペンシル `Mrf-λNrf` は次の形です。

```
      Mrf-λNrf = | B  | A-λE | ,
```

ここで、`E` は `nxn` の非特異な上三角行列であり、`A` と `B` はそれぞれ `nxn` および `nxm` 次元の行列であり、`p = 0` です。

完全列ランクのサブペンシル `Mil-λNil` は階段形式になっています。 `nb` 次元のベクトル `ν` と `μ` は、階段形式のブロックの行および列の次元をそれぞれ含み、`i` 番目のブロックは次元 `ν[i] x μ[i]` を持ち、完全列ランクです。 `ν[nb-j+1]-μ[nb-j+1]` の差は、`j = 1, 2, ..., nb` に対して、サイズ `j x (j-1)` の基本クロンカーのブロックの数です。 `μ[nb-j+1]-ν[nb-j]` の差は、`j = 1, 2, ..., nb` に対して、次数 `j` の無限の基本因子の数です（`ν[0] = 0`）。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、および `M` と `N` の非ゼロ要素の相対許容誤差を指定します。

還元は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、また `fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された左の直交またはユニタリ変換は、`withQ = true` の場合は行列 `Q` に蓄積されます。そうでない場合、`Q` は `nothing` に設定されます。 実行された右の直交またはユニタリ変換は、`withZ = true` の場合は行列 `Z` に蓄積されます。そうでない場合、`Z` は `nothing` に設定されます。
