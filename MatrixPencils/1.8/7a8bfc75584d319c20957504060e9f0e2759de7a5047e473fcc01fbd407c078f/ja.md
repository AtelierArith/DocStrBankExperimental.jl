```
_svdlikeAE!(A, E, Q, Z, B, C; svdA = true, fast = true, atol1 = 0, atol2 = 0, rtol, withQ = true, withZ = true) -> (rE, rA22)
```

通常の行列ペンシル `A - λE` を、直交またはユニタリ変換行列 `Q1` と `Z1` を用いて、同等の形 `A1 - λE1 = Q1'*(A - λE)*Z1` に還元します。このとき、変換された行列 `A1` と `E1` は、`svdA = true` の場合、以下のSVDのような座標形式になります。

```
               | A11-λE11 |  A12  |  A13  |
               |----------|-------|-------|
    A1 - λE1 = |    A21   |  A22  |   0   | ,
               |----------|-------|-------|
               |    A31   |   0   |   0   |
```

ここで、`rE x rE` 行列 `E11` と `rA22 x rA22` 行列 `A22` は非特異であり、`E11` と `A22` は、`fast = true` の場合は上三角行列、`fast = false` の場合は対角行列です。

`svdA = false` の場合、`E` のみがSVDのような形式に還元され、`A1 - λE1` は次の形になります。

```
               | A11-λE11 |  A12  |
    A1 - λE1 = |----------|-------| , 
               |    A21   |  A22  |
```

ここで、`rE x rE` 行列 `E11` は非特異の上三角行列（`fast = true` の場合）または対角行列（`fast = false` の場合）であり、`A22` は未還元でランク `rA22` を持ちます。

キーワード引数 `atol1`、`atol2`、および `rtol` は、それぞれ、`A` の非ゼロ要素に対する絶対許容誤差、`E` の非ゼロ要素に対する絶対許容誤差、`A` と `E` の非ゼロ要素に対する相対許容誤差を指定します。

還元は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して行われ、`fast = false` の場合はより信頼性の高いSVD分解が使用されます。

実行された左側の直交またはユニタリ変換 `Q1` は、`withQ = true` の場合に行列 `Q <- Q*Q1` に蓄積されます。そうでない場合、`Q` は変更されません。実行された右側の直交またはユニタリ変換 `Z1` は、`withZ = true` の場合に行列 `Z <- Z*Z1` に蓄積されます。そうでない場合、`Z` は変更されません。

`Q1'*B` は `B` に返され、`B = missing` の場合を除き、`C*Z1` は `C` に返され、`C = missing` の場合を除きます。
