```
sklf_left!(A, E, C, G, B, F, D, H; fast = true, atol1 = 0, atol2 = 0, atol3 = 0, rtol, withQ = true, withZ = true) -> (Q, Z, no)
```

分割されたフル行ランク行列ペンシルを減少させます 

```
M - λN = | A-λE | n
         | C-λG | p
            n
```

`A-λE` が正則である場合、同等の形 `F - λG = Q'*(M - λN)*Z` に変換します。ここで、直交またはユニタリ変換行列 `Q` と `Z` を使用し、`M - λN` が左側、有限および無限のクロンケル構造（強可観測性形式とも呼ばれる）を示す階段形式に変換されます：

```
                | Auo-λEuo  |   *    | n-no
  | At-λEt |    |   0       | Ao-λEo | no
  |--------| =  |-----------|--------|
  | Ct-λGt |    |   0       | Co-λGo | p
                   n-no         no
```

行列 `Ct`, `Gt`, `At`, `Et` は `[At; Ct] = Q'*[A; C]*Z`, `[Et; Gt] = Q'*[E; G]*Z` から決定され、`C`, `G`, `A` および `E` にそれぞれ返されます。さらに、互換的に分割された `[Bt; Dt] = Q'*[B; D]` および `[Ft; Ht] = Q'*[F; H]` から決定される行列 `Bt`, `Dt`, `Ft`, `Ht` は、`B`, `D`, `F`, `H` にそれぞれ返されます（ただし、`B`, `D`, `F` および `H` が `missing` に設定されていない限り）。

サブペンシル 

```
   | Ao-λEo |   
   |--------|
   | Co-λGo |
```

はフル列ランク `no` を持ち、`(n-no) x (n-no)` サブペンシル `Auo-λEuo` には `M - λN` の有限および無限の固有値（`M - λN` の非可観測固有値とも呼ばれる）が含まれます。  

`(n+p) x (n+p)` 直交行列 `Q` は次のような分割形式を持ちます

```
       |  *  |  *  |  *   | n 
   Q = |-----|-----|------|
       |  *  |  0  | Q22  | p
         n-no  no     p
```

`p x p` の後ろのサブ行列 `Q22` は可逆で上三角です。 

`注意:` `Bt-λFt` が `[ Buo-λFuo | Bo-λFo ]` と分割される場合、次の関係が成り立ちます

```
  (Co-λGo)*inv(λEo-Ao)(Bo-λFo) + Dt-λHt = Q22'*((C-λG)*inv(λE-A)(B-λF) + D-λH)
```

そして構造化ペンシル線形化 (Ao-λEo,Bo-λFo,Co-λGo,Dt-λHt) は `強可観測` です。 

キーワード引数 `atol1`, `atol2`, `atol3`, および `rtol` は、それぞれ `A` と `C` の非ゼロ要素の絶対許容誤差、`E` と `G` の非ゼロ要素の絶対許容誤差、`A`, `C`, `E` および `G` の非ゼロ要素の相対許容誤差を指定します。減少は、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を使用して、`fast = false` の場合はより信頼性の高いSVD分解を使用して行われます。

実行された直交またはユニタリ左変換は、`withQ = true` の場合、行列 `Q` に蓄積されます。`withQ = false` の場合、`Q` には上三角行列 `Q22` が含まれます。実行された直交またはユニタリ右変換は、`withZ = true` の場合、行列 `Z` に蓄積されます。それ以外の場合、`Z` は `nothing` に設定されます。
