```
   pmrank(P; fastrank = true, atol = 0, rtol = atol > 0 ? 0 : n*ϵ)
```

多項式行列 `P(λ)` の通常のランクを決定します。

`P(λ)` は、`P(λ) = P_1 + λ P_2 + ... + λ**k P_(k+1)` の形を持つ次数 `k` の多項式行列であり、係数行列 `P_i`（`i = 1, ..., k+1`）は3次元行列 `P` に格納されており、`P[:,:,i]` には `λ**(i-1)` を掛けた `i` 番目の係数行列 `P_i` が含まれています。`P(λ)` の通常のランクは、線形独立な行または列の数です。

`P(λ)` は、[Polynomials](https://github.com/JuliaMath/Polynomials.jl) パッケージによって提供される `Polynomial` 型の要素の行列、ベクトル、またはスカラーとしても指定できます。

`fastrank = true` の場合、ランクは `P(γ)` の特異値のうち、`max(atol, rtol*σ₁)` より大きい大きさを持つものの数を数えることによって評価されます。ここで、`σ₁` は `P(γ)` の最大特異値であり、`γ` は大きさが1のランダムに生成された複素数です。`fastrank = false` の場合、まず `P(λ)` の構造化線形化が `[A-λE B; C D]` の形で構築され、`A-λE` は次数 `n` の正則サブペンシルであり、その後、通常のランク `nr` が決定されます。`P(λ)` の通常のランクは `nr - n` です。

キーワード引数 `atol` と `rtol` は、それぞれ `P(λ)` の非ゼロ係数に対する絶対および相対の許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `P(λ)` の最小次元のサイズで、`ϵ` はその係数の要素型のマシンイプシロンです。
