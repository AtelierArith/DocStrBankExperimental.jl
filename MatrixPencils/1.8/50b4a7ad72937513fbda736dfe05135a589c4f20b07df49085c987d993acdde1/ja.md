```
saloc2(A, B, evc, tola, tolb) -> (F, U)
```

実数のペア `(A,B)` に対して、`A` が次数2の行列であるとき、固有値が行列 `A+B*F` の固有値が `evc` に含まれる複素共役ペアと等しくなるような行列 `F` を計算します。非ゼロ要素に対する絶対閾値 `tola` と `tolb` は、制御可能性のチェックに使用されます。

ペア `(A,B)` が制御不可能な場合、`F = nothing` となり、`U` には変換行列が含まれ、変換されたペア `(U'*A*U, U'*B)` は次の形になります。

```
                 [ A1  X    B1 ]
[ U'*A*U U'*B] = [             ] ,
                 [  0  A2   0  ]
```

ここで、ペア `(A1,B1)` は制御可能です。もし `norm(B) < tolb` であれば、`U` は `2x2` の単位行列となります。
