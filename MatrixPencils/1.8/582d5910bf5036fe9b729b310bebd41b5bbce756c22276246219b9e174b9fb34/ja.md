```
peigvals(M, N; fast = false, atol1::Real = 0, atol2::Real = 0, rtol::Real=min(atol1,atol2)>0 ? 0 : n*ϵ) -> (val, KRInfo)
```

行列ペンシル `M-λN` の（有限および無限の）固有値を `val` に返し、Kronecker構造に関する情報を `KRInfo` オブジェクトに返します。

Kronecker構造に関する情報は、右Kroneckerインデックス `rki`、左Kroneckerインデックス `lki`、無限の基本因子 `id`、有限固有値の数 `nf`、および通常のランク `nrank` で構成され、これらはそれぞれ `KRInfo.rki`、`KRInfo.lki`、`KRInfo.id`、`KRInfo.nf`、`KRInfo.nrank` から取得できます。詳細については、[`pkstruct`](@ref) を参照してください。

固有値の計算は、ペンシル `M-λN` の無限および有限固有値構造の分割を示す適切なKronecker様式（KLF）にペンシルを縮約することによって行われます。縮約は直交類似変換を使用して行われ、`fast = true` の場合は列ピボットを伴うランクを明らかにするQR分解に基づくランク決定を含み、`fast = false` の場合はより信頼性の高いSVD分解を使用します。効率のために、縮約は部分的にのみ行われ、実行された直交変換は蓄積されません。

キーワード引数 `atol1`、`atol2` および `rtol` は、それぞれ `M` の非ゼロ要素の絶対許容誤差、`N` の非ゼロ要素の絶対許容誤差、`M` および `N` の非ゼロ要素の相対許容誤差を指定します。デフォルトの相対許容誤差は `n*ϵ` であり、ここで `n` は `M` の最小次元のサイズで、`ϵ` は `M` の要素型の機械イプシロンです。
