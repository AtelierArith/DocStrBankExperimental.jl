phi(g, H, δ, γ*2, r, print*level)                     (3)

この関数は、δに関して単調減少する単変数関数です。H + δ I >= 0 かつ ||d*k|| < γ*2 r のとき、-1に等しくなります。H + δ I >= 0 かつ γ*2 r <= ||d*k|| <= r のとき、0に等しくなります。H + δ I < 0 または ||d*k|| > r のとき、1に等しくなります。d*k = (H + δ * I) ^ {-1} (-g) です。

# 入力:

  * `g::Vector{Float64}`. (1)を参照してください。現在の反復点xでの勾配。
  * `H::Union{Matrix{Float64}, SparseMatrixCSC{Float64, Int64}, Symmetric{Float64, SparseMatrixCSC{Float64, Int64}}}`.

(1)を参照してください。現在の反復点xでのヘッセ行列。

  * `δ::Float64`. (1)を参照してください。関数内の変数。
  * `γ_2::Float64`. (2)を参照してください。δ > 0 のとき、ステップd_kが信頼領域の境界からどれだけ近くなるべきかを指定します。
  * `r::Float64`. (1)を参照してください。信頼領域の半径。
  * `print_level::Float64`. ログの冗長性レベル。

# 出力:

-1、0、または1の値を取る整数値。
