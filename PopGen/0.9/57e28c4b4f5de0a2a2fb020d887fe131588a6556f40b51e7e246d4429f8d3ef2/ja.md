```
summary(data::PopData; by::String = "global")
```

`PopData`オブジェクトの要約統計を提供します。ローカスごとの要約情報には`by = "locus"`を使用します。グローバル値は、ローカスごとのパラメータの加重されていない平均として示されます。

### Het_obs

観察されたヘテロ接合性は次のように示されます：

1 - ∑ₖ ∑ᵢ Pₖᵢᵢ/np 

ここで、Pkiiはサンプル`k`におけるホモ接合体`i`の割合を表し、`np`はその集団のサンプル数です。

### HT

全体の遺伝子多様性は次のように示されます：

1 - ∑ᵢ(p̄ᵢ² + (HS / (ñ × np)) - Het_obs / (2 × ñ × np)) 

ここで、p̄ᵢ = ∑ₖpₖᵢ / npです。

### HS

集団内の遺伝子多様性は次のように示されます：

1 - ∑ᵢ(pᵢ² + HS / (ñ × np) - Het_obs / (2 × ñ × np)) 

ここで、ñ = np / ∑ₖ(1/nₖ) 

ここで、p̄ᵢ² = ∑ₖ(pᵢₖ² / np)です。

### DST

サンプル間の遺伝子多様性の量は次のように示されます：

HT - HS

### DST′

サンプルサイズに調整されたサンプル間の遺伝子多様性の量は次のように示されます：

(np / (np-1)) × Dst

### HT′

サンプルサイズに調整された全体の遺伝子多様性は次のように示されます：

HS + DST′

### FST

サブ集団における全遺伝的分散の割合は、全遺伝的分散に対して次のように示されます：

DST / HT

### FST′

ヘテロ接合性に対して調整されたサブ集団における全遺伝的分散の割合は次のように示されます：

DST′ / HT′

### FIS

ローカスにおける遺伝的分散の割合は、サブ集団内の遺伝的分散に対して次のように示されます：

1 - (Het_obs / HS)

### DEST

集団の分化（Jost 2008）は次のように示されます：

(np/(np-1)) × (Ht'-Hs) / (1-Hs) ```
