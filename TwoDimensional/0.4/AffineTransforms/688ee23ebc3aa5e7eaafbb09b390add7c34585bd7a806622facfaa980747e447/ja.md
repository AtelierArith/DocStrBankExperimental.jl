### アフィン変換の回転

角度 `θ`（ラジアンで反時計回り）による回転をアフィン変換 `A` と組み合わせる方法は2つあります。左回転の場合は次のようになります：

```julia
B = rotate(θ, A)
```

これは変換の出力を回転させる結果になります。一方、右回転の場合は次のようになります：

```julia
C = rotate(A, θ)
```

これは変換の入力を回転させる結果になります。上記の例は次のように似ています：

```julia
B = R∘A
C = A∘R
```

ここで `R` は点 `(0,0)` の周りで角度 `θ` による回転を実装します。
