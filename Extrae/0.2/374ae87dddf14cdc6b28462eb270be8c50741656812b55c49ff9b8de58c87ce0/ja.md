```
user_function(enter)
```

トレースファイルにイベントを発生させ、ソースコードを参照します（データには：ソース行番号、ファイル名、関数名が含まれます）。

enterが0の場合は終了を示し（つまり、関数を離れること）、それ以外の場合はルーチンの開始を示します。ユーザーは、このルーチンの呼び出しを常に実行されるコードの場所に配置するよう注意し、if文やreturn文の内部に配置しないように注意する必要があります。この関数は参照のアドレスを返します。

```julia
function routine1()
    user_function(1);
    [routine1 code]
    user_function(0);
end

function routine2()
    user_function(1);
    [routine2 code]
    user_function(0);
end
```

これらの呼び出しの実行中にパフォーマンスカウンタを収集するために、ユーザー関数のタグとそのカウンタの両方をセクション内で有効にする必要があります。

# 警告

キャプチャされたアドレスを関数名、ファイル名、行番号などの有用な情報に変換するには、アプリケーションバイナリをデバッグ情報付きでコンパイルする必要があります（通常は`-g`コンパイラフラグ）。
