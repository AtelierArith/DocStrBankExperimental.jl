```
simulate_component(rng, c::MultichannelComponent, design::AbstractDesign)
```

"source" から "sensor" 空間への `MultichannelComponent c` の投影を返します。

# 戻り値

  * `Array{Float64,3}`: イベントデータフレーム内の各イベントに対して投影されたシミュレーションコンポーネント。出力の次元は `length(c.projection) x length(c.basis) x length(design)` です。
