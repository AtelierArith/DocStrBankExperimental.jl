```
n400(; sfreq = 100)
```

N400 EEGコンポーネントを模倣するハニングウィンドウを生成します。400msで負のピーク(!)があり、幅は400msです。

# キーワード引数

  * `sfreq = 100`: サンプリング周波数（Hz）。

# 戻り値

  * `Vector`: シフトされた（すなわちゼロパディングされた）ハニングウィンドウを含みます。

# 例

```julia-repl
julia> n400(; sfreq = 250)

```
