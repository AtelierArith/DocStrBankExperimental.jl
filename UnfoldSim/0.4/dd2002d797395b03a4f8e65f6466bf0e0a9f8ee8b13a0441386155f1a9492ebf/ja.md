```
pad_array(arr::Vector, len::Int, val)    
pad_array(arr::Vector, len::Tuple, val)
```

入力配列 `arr` を指定された値 `val` で、`len` の符号に基づいて既存の要素の前または後にパディングします。

# 引数

  * `arr::Vector`: パディングされる入力配列。
  * `len::Union{Int, Tuple}`: `val` を追加する回数。`len` が負の場合、値は既存の配列の前に追加されます。そうでない場合は、既存の配列の後に追加されます。`len` がタプルの場合、`pad_array` は2回呼び出され、1回の関数呼び出しで配列の前後にパディングを行うことができます。
  * `val`: パディングに使用される値。

# 戻り値

  * `Vector`: 新しい長さ `length(arr) + sum(abs.(len))` のパディングされたベクター。

# 例

```julia-repl
# パディングされる配列を作成
julia> my_array = rand(5)
5-element Vector{Float64}:
 0.0420017254437951
 0.19179144973603235
 0.5388760239550549
 0.6973699906283798
 0.9966598131018376

 # 元の配列の前にゼロでパディング
julia> pad_array(my_array, -2, 0)
7-element Vector{Float64}:
 0.0
 0.0
 0.0420017254437951
 0.19179144973603235
 0.5388760239550549
 0.6973699906283798
 0.9966598131018376

# 元の配列の前後に値5でパディング
julia> pad_array(my_array, (-2, 1), 5)
8-element Vector{Float64}:
 5.0
 5.0
 0.0420017254437951
 0.19179144973603235
 0.5388760239550549
 0.6973699906283798
 0.9966598131018376
 5.0
```
