```
PuRF(; n = 10.1, tmax = 0.93, sfreq = 100)
```

PuRF瞳孔反応関数のデフォルトジェネレーター。標準的なPRFはガンマ関数であり、Denison 2020の方程式(2)に従って実装されており、Hoeks & Levelt, 1993に遡ります。

瞳孔反応はt = 0:1/sfreq:3*tmaxで評価されます。反応はtmaxでのピーク最大値で正規化されるため、通常は1の瞳孔反応が返されます（数値的な問題は無視します）。

# キーワード引数:

  * `n = 10.1`: 形状パラメータ
  * `tmax = 0.93`: ピーク最大値
  * `sfreq = 100`: サンプリング周波数

# 戻り値

  * `Vector`: 長さ(0:1/sfreq:3*tmax)のエントリを持つ標準的な瞳孔反応。
