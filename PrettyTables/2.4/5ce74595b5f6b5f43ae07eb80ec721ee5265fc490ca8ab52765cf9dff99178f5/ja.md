```
@pt(expr...)
```

`expr`内のテーブルを、マクロ`@ptconf`で選択されたグローバル設定を使用して`stdout`にきれいに表示します。

複数のテーブルは、次のように複数の式を渡すことで印刷できます。

```
@pt table1 table2 table3
```

ユーザーは、次の式を渡すことでテーブルのヘッダーを選択できます。

```
:header = [<ヘッダーのベクター>]
```

ヘッダーは次に印刷されるテーブルにのみ有効であることに注意してください。したがって：

```
@pt :header = header1 table1 :header = header2 table2 table3
```

は、`header1`を使用して`table1`を印刷し、`header2`を使用して`table2`を印刷し、デフォルトのヘッダーを使用して`table3`を印刷します。

!!! info
    このマクロに複数のテーブルが渡されると、`pretty_table`への複数の呼び出しが発生します。したがって、クロッピングアルゴリズムは、テーブルを個別に印刷するのとまったく同じように動作します。
