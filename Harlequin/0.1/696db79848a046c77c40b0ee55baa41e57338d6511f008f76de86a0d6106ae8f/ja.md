```
mutable struct Observation{T <: Real}
```

*観測*は、ある検出器によって取得された時間順序のデータ（TOD）のシーケンスです。以下のフィールドを実装しています：

  * `time`: サンプルの時間を表す `N` 要素の配列。`AbstractArray` として定義されているため、範囲（例：`0.0:0.1:3600.0`）を使用してメモリを節約できます。
  * `pixidx`: マップ内の各ピクセルインデックスを持つ `N` 要素の配列。Healpix ピクセル化スキームは強制されず、実際にはマップは球状である必要すらありません。
  * `psi_angle`: 固定された参照系（例：黄道面）に対する偏光角の向きを含む `N` 要素の配列。角度は **ラジアン** でなければなりません。
  * `tod`: 測定器によって測定された実際のデータを含む `N` 要素の配列で、すでにいくつかの物理単位（例：K_CMB）にキャリブレーションされています。
  * `sigma_squared`: 各サンプルの二乗RMSを含む `N` 要素の配列。メモリを節約するために、通常はここで `RunLengthArray` を使用するべきです。
  * `flagged`: サンプルがマップ作成プロセス中に破棄されるべきかどうかを示す `N` 要素のブール配列（`true` または `false`）。
  * `name`: 検出器を表す文字列。これはデバッグ目的でのみ使用されます。

観測を作成するために、コンストラクタはパラメータの代わりに *キーワード引数* を取ります。このようにして、コードはより読みやすくなります。また、欠落しているフィールドには適切なデフォルトが提供されます：

```julia
# "sigma_squared" や "flagged" を指定しないので、それらは
# 1 のベクトルと `false` のベクトルで初期化されます
obs = Observation{Float64}(
    time = 0.0:0.1:3600.0,
    pixidx = 1:3601,
    psi_angle = zeros(3601),
    tod = randn(3601),
)
```
