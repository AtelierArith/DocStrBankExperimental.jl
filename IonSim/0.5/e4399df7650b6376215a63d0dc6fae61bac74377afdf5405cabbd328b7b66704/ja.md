```
hamiltonian(
        chamber::Chamber; timescale::Real=1, lamb_dicke_order::Union{Vector{Int},Int}=1,
        rwa_cutoff::Real=Inf, displacement="truncated", time_dependent_eta=false
    )
```

`chamber`の時間の関数としてのハミルトニアンを構築します。戻り値の型は関数 `h(t::Real, ψ)` であり、それ自体が `QuantumOptics.SparseOperator` を返します。

**args**

  * `timescale`: 例えば、1e-6の値はμsの時間を取ります。
  * `lamb_dicke_order`: フォノン数をこの値まで変化させる項のみを考慮します。これが `Int` の場合、カットオフはすべてのモードに適用されます。これが `Vector{Int}` の場合、`lamb_dicke_order[i]` は `basis(chamber)` の順序に従ってiᵗʰモードに適用されます。注意: これはラム・ディッケ近似と完全に同じではなく、`lamb_dicke_order=1` を設定すると、例えば $a^\dagger a$ に比例する項が保持されます。
  * `rwa_cutoff`: このカットオフよりも速く振動するハミルトニアンの項を削除します。
  * `displacement`: これは `"truncated"`（デフォルト）または `"analytic"` のいずれかです。

    原子が照射されると、原子のエネルギーと運動量は一般的に影響を受けます。調和ポテンシャル内の原子に対して、運動量の交換は $D(α=iηe^{-iνt}) = exp[αa^† - α^*a]$ という変位操作としてモデル化できます。ここで、$η$ はラム・ディッケパラメータであり、これは原子の運動の基底状態エネルギーに対する反跳周波数の比の平方根に比例するか、基底状態波動関数の広がりとレーザーの波長の比として同等に説明できます。

    `"truncated"` が選択されると、$D(α)$ の行列要素は、モデルで指定された次元に従って切り捨てられた基底で $α^* a, αa^†$ を構築することによって計算され、その差を指数関数化します。これは、他の多くの利点の中でも、ユニタリティを保証するという利点があります。

    `"analytic"` が選択されると、行列要素は無限次元のヒルベルト空間を仮定して計算されます。

    小さな変位の場合（$η ≪ N$、ここで $N$ は運動のヒルベルト空間の次元）、これらの方法はどちらも良い近似となります。
  * `time_dependent_eta::Bool`: 振動サブスペースに直接影響を与えるだけでなく、トラップ周波数の変化 $δν$ はラム・ディッケパラメータも変化させます。通常、$δν≪ν$ であるため、この効果は小さく $η ≈ η₀(1 + δν/2ν)$ となり、時間に応じて計算して更新するために必要な追加の計算リソースを必要としません。この場合、`time_dependent_eta=false`（デフォルト）を設定することができ、$η(t) = η₀$ となります。
