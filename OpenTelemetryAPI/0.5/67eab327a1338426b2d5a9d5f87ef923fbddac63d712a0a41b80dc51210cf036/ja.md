```
Histogram{T}(name, meter; unit = "", description = "") where {T}
```

`Histogram`は、統計的に意味のある値を報告するために使用できる`AbstractSyncInstrument`です。ヒストグラム、サマリー、パーセンタイルなどの統計に意図されています。

詳細については、[the specification](https://github.com/open-telemetry/opentelemetry-specification/blob/main/specification/metrics/api.md#histogram)を参照してください。
