```
spin_weighted_spherical_harmonic(s::Int, l::Int, m::Int; method="auto")
```

スピン重み付き球面調和関数をスピン重み `s`、調和指数 `l`、および方位指数 `m` で構築します。

任意の点で評価できる `SpinWeightedSphericalHarmonicFunction` オブジェクトを返します。

デフォルトでは、値を計算するためのメソッドは調和指数 `l` の値に基づいて自動的に選択されます。`l < 30` の場合、直接評価メソッド（`method="direct"`）が使用され、式 (A8) に示される正確な解析解を評価します。ただし、和の各項の前因子は非常に大きくなる可能性があり、オーバーフローを引き起こすことがありますが、和自体は有限です（実際にはオーダー1です）。したがって、`l >= 30` の場合は、チェビシェフ擬似スペクトル法（`method="chebyshev"`）が代わりに使用されます。
