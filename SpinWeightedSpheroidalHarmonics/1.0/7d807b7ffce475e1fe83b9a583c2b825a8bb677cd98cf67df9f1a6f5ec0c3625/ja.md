```
spin_weighted_spheroidal_harmonic(s::Int, l::Int, m::Int, c; N::Int=-1, method="auto")
```

このスピン重み付き球面調和関数のスペクトル分解を構築します。スピン重み `s`、調和数 `l`、方位数 `m`、および楕円率 `c` ($c = a\omega$) を使用して、`N` 個のスピン重み付き *球面* 調和関数を使用します。

デフォルト値 `N=-1` は、適切な `N` の値が自動的に決定されることを示します。

任意の点で評価できる `SpinWeightedSpheroidalHarmonicFunction` オブジェクトを返します。

デフォルトでは、基礎となるスピン重み付き球面調和関数の値を計算する方法は、調和数 `l` の値に基づいて自動的に選択されます。`l < 30` の場合、直接評価法（`method="direct"`）が使用され、式 (A8) に示される正確な解析解を評価します。ただし、和の各項の前因子は非常に大きくなる可能性があり、オーバーフローを引き起こす可能性がありますが、和自体は有限（実際には1次のオーダー）です。したがって、`l >= 30` の場合は、チェビシェフ擬似スペクトル法（`method="chebyshev"`）が代わりに使用されます。
