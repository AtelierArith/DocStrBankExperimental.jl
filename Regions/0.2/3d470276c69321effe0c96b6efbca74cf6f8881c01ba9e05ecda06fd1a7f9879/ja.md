```
binarize(image, predicate)
```

画像を二値化し、領域を返します。

述語は、ピクセル値を受け取り、ピクセル値に基づいてブール値の結果を返す関数でなければなりません。

以下はいくつかの便利な例です：

# 返される領域にはすべてのピクセル > 0.3 が含まれます

reg = binarize(img, x -> x > 0.3)

# 返される領域にはすべてのピクセル <= 0.5 が含まれます

reg = binarize(img, x -> x <= 0.5)

# 返される領域にはすべてのピクセルが 0.3 と 0.8 の間に含まれます

reg = binarize(img, x -> x > 0.3 && x < 0.8)
