`BufferedOutputStream{T}` は、型 `T` のシンクへのバッファ付き書き込みを提供します。

`BufferedOutputStream` にラップされた任意の型 `T` は、次のメソッドを実装する必要があります：

```
BufferedStreams.writebytes(sink::T, buffer::Vector{UInt8}, n::Int, eof::Bool)
```

この関数は次のことを行うべきです：

  * `buffer` から最初の位置から `n` バイトを書き込む
  * `eof` が `true` の場合、シンクへの書き込み操作がこれ以上行われないことを示し、シンクのフラッシュやクローズなどを容易にします。
  * `writebytes` は書き込まれたバイト数を返すべきです。これは `n` または 0 でなければなりません。返り値が 0 の場合、データはバッファから排出されるべきではありません。

この関数に渡されるバッファは、`BufferedOutputStream` によって再割り当てされることはないため、例えば、いくつかのバイトを書き込まれたと報告するために、それを保持することは安全ですが、遅延または非同期で行うことができます。
