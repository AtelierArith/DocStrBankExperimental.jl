`FiniteDifferenceMethod(Δr=0.1, rₘₐₓ=50.0, R=Δr:Δr:rₘₐₓ, l=0, direction=:c, solver=:LinearAlgebra)`

| 引数                  | デフォルト            | 説明                                                               |
|:------------------- |:---------------- |:---------------------------------------------------------------- |
| `Δr::Real`          | `0.1`            | 放射状グリッドの間隔。均一なグリッド間隔が使用され、$r_{i+1} = r_{i} + \Delta r$ となります。    |
| `rₘₐₓ::Real`        | `50.0`           | 放射状グリッドの最大値。この値は計算には直接使用されませんが、`R` を決定するために使用されます。               |
| `R::StepRangeLen`   | `Δr:Δr:rₘₐₓ`     | 放射状グリッド。原点は、コロンブのポテンシャルと原点での遠心ポテンシャルの発散を避けるためにグリッドから除外する必要があります。 |
| `l::Int`            | `0`              | 角運動量量子数。これは正の整数で、$0 \leq l$ です。                                  |
| `direction::Symbol` | `:c`             | 有限差分の方向、`：c` は中央、`:f` は前方、`:b` は後方を示します。                         |
| `solver::Symbol`    | `:LinearAlgebra` | 固有値問題のソルバー、`:LinearAlgebra` または `:ArnoldiMethod`。                |
