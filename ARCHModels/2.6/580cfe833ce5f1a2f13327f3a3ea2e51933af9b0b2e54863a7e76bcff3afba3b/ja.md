```
TGARCH{o, p, q}(coefs) -> UnivariateVolatilitySpec
```

与えられたパラメータでTGARCH仕様を構築します。

# 例:

```jldoctest
julia> TGARCH{1, 1, 1}([1., .04, .9, .01])
TGARCH{1, 1, 1} 仕様。

─────────────────────────────────
               ω    γ₁   β₁    α₁
─────────────────────────────────
パラメータ:  1.0  0.04  0.9  0.01
─────────────────────────────────
```
