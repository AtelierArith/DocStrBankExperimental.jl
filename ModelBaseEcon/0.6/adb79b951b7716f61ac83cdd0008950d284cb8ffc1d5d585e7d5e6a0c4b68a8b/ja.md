```
@link expr
```

パラメータリンクを作成します。モデル定義の[`@parameters`](@ref)セクションで`@link`を使用します。

パラメータが他のパラメータに依存している場合、`@link`を使用してそれを宣言します。式は有効なJuliaコードであれば何でも構いません。

```
@parameters model begin
    a = 5
    b = @link a + 1
end
```

リンクが依存しているパラメータに新しい値が割り当てられると、それに依存するリンクが自動的に更新されます。

!!! note "重要な注意"
    リンクの値が自動的に更新されない2つのケースがあります。依存しているパラメータが可変である場合、例えば`Vector`の場合、インプレースで更新される可能性があります。もう1つのケースは、リンクがグローバル変数またはカスタム関数を含む場合です。

    その場合、[`update_links!`](@ref)を呼び出す必要があります。
