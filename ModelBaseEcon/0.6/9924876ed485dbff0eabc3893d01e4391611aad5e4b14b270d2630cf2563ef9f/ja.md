```
@equations model begin
    :eqnkey => lhs = rhs
    lhs = rhs
    ...
end
```

与えられたキーで方程式を置き換え、提供された方程式を追加します。キーなしまたは存在しないキーで提供された方程式はモデルに追加されます。キーは、`:`を含む完全なシンボル参照で提供する必要があります。

方程式のキーを見つけるには、[`summarize`](@ref)を参照してください。方程式の詳細については、[`Equation`](@ref)を参照してください。

このような変更の後は、モデルに対して[`@reinitialize`](@ref)を呼び出す必要があります。
