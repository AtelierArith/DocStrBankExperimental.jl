```
@steadystate model [type] lhs = rhs
@steadystate model begin
    lhs = rhs
    @delete _SSEQ1 _SSEQ2
    @level lhs = rhs
    @slope lhs = rhs
end
@steadystate model @delete _SSEQ1 _SSEQ2
```

モデルに定常状態方程式を追加します。

モデルの定常状態システムは動的システムから自動的に導出されます。このマクロを使用して定常状態の追加方程式を定義します。これは、複数の定常状態を持つ可能性のある非線形モデルや、定常状態を解くのが難しい場合に特に便利で、定常状態ソルバーが使用したい定常状態を見つけるのを助けます。

  * `model` は更新したいモデルインスタンスです
  * `type`（オプション）は追加したい制約のタイプです。これは `level` または `slope` です。省略した場合、デフォルトは `level` です
  * `lhs = rhs` は定常状態制約を定義する式です。この方程式では、モデルからの変数とショックを使用しますが、t参照は含めません。

これには、いくつかの定常状態方程式を許可するブロック形式もあります。これらは、タイプを指定するために @level または @slope で前置きすることができます。@level がデフォルトです。定常状態方程式は、制約キーのリストに続く @delete 行で削除することもできます。
