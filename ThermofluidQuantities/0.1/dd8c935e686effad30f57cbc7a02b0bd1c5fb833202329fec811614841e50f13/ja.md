```
@gas name temp visc gamma mmass
```

指定された名前の気体を作成し、粘度 `visc` と比熱比 `gamma` を基準温度 `temp` で、モル質量 `mmass` で指定します。これらはデフォルトでない場合、単位を付けて提供する必要があります。

# 例

```jldoctest mygas
julia> @gas MyO2 20u"°C" 2.04e-5 1.395 31.999u"g/mol"

julia> MyO2
完璧な気体で
   密度 = 1.330236729981785 kg m⁻³
   粘度 = 2.04e-5 kg m⁻¹ s⁻¹
   比熱比 = 1.395
   気体定数 = 259.83507666343445 J kg⁻¹ K⁻¹
   cp = 917.645397330357 J kg⁻¹ K⁻¹
   cv = 657.8103206669226 J kg⁻¹ K⁻¹
   基準温度 293.15 K において
```
