```
isprime(n::Integer) -> Bool
```

INT64変数の範囲内の値に対して、`n`が素数であれば`true`を返し、そうでなければ`false`を返します。より大きな値に対しては、`n`が素数である可能性が高ければ`true`を返し、そうでなければ`false`を返します（偽陽性率 = 0.25^reps、ここでreps=25 –> 安全と見なされます）。

```
より詳細には：
偶数に対して：決定的かつ正確な結果を返します。
INT64変数の範囲内の値に対して：決定的かつ正確な結果を返します（ルックアップテーブル、試行割り算、ミラー・ラビン、ルーカステストによって）。
より大きな値に対して：GNU多倍長精度算術ライブラリから確率的な結果を返します。
```

```julia
julia> isprime(3)
true
```
