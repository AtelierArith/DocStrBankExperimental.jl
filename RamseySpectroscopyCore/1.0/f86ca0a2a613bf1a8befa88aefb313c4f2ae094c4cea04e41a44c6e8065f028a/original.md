```
probe(state_vector::StateVector) -> Real
```

Method that computes the value of probability `nₑ` for a quantum system to be in excited state and can be interpret as population of excited state.

# Arguments

  * `self` :   The current state vector of a quantum system.

# Returns

  * `nₑ` :   Real number.
