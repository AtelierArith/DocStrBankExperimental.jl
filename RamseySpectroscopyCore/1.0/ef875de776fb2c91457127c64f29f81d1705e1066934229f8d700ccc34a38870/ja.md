```
StateVector
```

量子二状態システムの状態をディラック記法のケートベクトルで表す構造体です。

# フィールド

  * `state` :   (|e>, |g>) 基底におけるシステムの状態を表すベクトル。

# 引数

  * `e` :   量子システムが励起状態にある確率または基底状態にある粒子の量を示す数値。 |e> ケートベクトルの値。
  * `g` :   量子システムが基底状態にある確率または基底状態にある粒子の量を示す数値。 |g> ケートベクトルの値。

# 戻り値

  * `StateVector` :   複合型インスタンス。
