```
FreeEvol
```

量子物理演算子の構造体で、摂動のない量子二準位系の進化を記述します。この進化は、`|e>` と `|g>` 状態の間の位相差の変化として記述できます。

# フィールド

  * `operator`:   行列の形で演算子自体を表します。

# 引数

  * `x` :   自由進化の結果としての `|e>` と `|g>` 状態の間の `位相差` 。

# 戻り値

  * `FreeEvol` :   複合型インスタンス。
