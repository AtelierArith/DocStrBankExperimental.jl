```
createRegressionSetting(formula, data)

与えられた式とデータの回帰設定を作成します
```

# 引数

  * `formula::FormulaTerm`: 式オブジェクトは線形回帰モデルを説明します。
  * `data::DataFrame`: DataFrameオブジェクトはデータを保持します。

# 注意事項

```
このパッケージで実装されたメソッドは、回帰設定オブジェクトとして線形モデルを受け入れます。
このオブジェクトは、モデルの式と回帰推定に使用されるデータを保持します。
createRegressionSettingは、回帰設定オブジェクトを作成するためのヘルパー関数です。
```
