```
NoPad()
NoPad(border)
```

入力配列にパディングを適用しないこと、または入力画像がすでに事前にパディングされていることを示します。`border`オブジェクトを渡すことで、ボーダーの選択の「メモリ」を保持することができ、`[]`でインデックス指定することで取得できます。

# 例

コマンド

```
np = NoPad(Pad(:replicate))
imfilter!(out, img, kernel, np)
```

はフィルタリングを直接実行し、パディングステップをスキップします。`out`のすべてのエントリは、`img`と`kernel`に対する境界内操作を使用して計算可能でなければなりません。
