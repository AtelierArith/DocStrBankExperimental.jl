```
kernelfactors(factors::Tuple)
```

フィルタリングのための因子化されたカーネルを準備します。長さ `m` と `n` のベクトルの2タプルが渡された場合、これは実質的にサイズ `m×1` と `1×n` の `ReshapedVector` の2タプルを返します。一般に、各連続する `factor` は、対応する次元に沿って拡張されるように再形成されます。

一般的な配列のタプルが渡された場合、各配列はその「先頭」次元に沿って適切に形状が整えられていると仮定されます。各配列の次元は `N = length(factors)` に「拡張」され、必要に応じてサイズに1が追加されます。
