```julia
struct Fill{T,N} <: AbstractBorder
    value::T
    lo::Dims{N}
    hi::Dims{N}
end
```

`Fill`は、画像の境界を超えてピクセルを外挿するために使用される特定の値を指定する型です。

# 出力

境界をパディングするために使用される値を指定する型`Fill`。

# 詳細

二次元の空間画像フィルタリング操作を画像と$D \times D$フィルタとの間の離散畳み込みとして表現する場合、画像の境界から$D$ピクセル以内のピクセルに対する結果は未定義です。境界付近および境界での操作を定義するには、エッジを超えてピクセルを外挿するためのスキームが必要です。`Fill`型を使用すると、外挿に使用される特定の値を指定できます。より複雑な外挿スキームについては、[`Pad`](@ref)を参照してください。

この型は、1次元、2次元、または多次元の画像のパディングを容易にします。

画像の各次元の下部および上部の境界に異なる量のパディングを指定できます（2次元の場合、上、左、下、右）。

# 例

これを示すために、アルファベット順に指定された6つのピクセルの行からなる画像を考えます：

```plain
    ┏━━━━━━┓ 
    ┃abcdef┃ 
    ┗━━━━━━┛ 
```

左と右の境界にのみ定数値`m`でパディングすると、次のようになります：

```plain
╭────┏━━━━━━┓────╮
│mmmm┃abcdef┃mmmm│
╰────┗━━━━━━┛────╯
```

（上部および下部の境界にも類似の結果が適用されます。）
