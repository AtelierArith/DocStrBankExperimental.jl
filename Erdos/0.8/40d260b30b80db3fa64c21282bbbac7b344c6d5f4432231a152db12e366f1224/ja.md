```
adjacency_list(g::AGraph)
adjacency_list(g::ADiGraph, dir=:out)
```

グラフの隣接リスト `a` を返します（整数のベクトルのベクトル）。隣接リストの `i` 番目の要素は、グラフ `g` における `i` の隣接ノードを含むベクトルです。

有向グラフの場合、2 番目のオプション引数を指定できます（`:out` または `:in`）。返される隣接リストの隣接ノードは、出力エッジまたは入力エッジを通じて関連するものとして考慮されます。

`a[i]` の要素は、イテレータ `(out_/in_)neighbors(g,i)` の順序と同じです。

*注意*: 一部のグラフタイプでは、コピーではなく参照を返すため、返されたオブジェクトは変更しないでください。
