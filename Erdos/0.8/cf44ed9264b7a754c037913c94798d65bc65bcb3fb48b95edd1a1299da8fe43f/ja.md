```
incidence_matrix(g::AGraphOrDiGraph, T::DataType=Int; oriented=false)
```

グラフのスパースノード-アークインシデンス行列を返します。インデックスは `[v, i]` で、ここで `i` は `1:ne(g)` の範囲にあり、エッジ `e` をインデックス付けします。有向グラフの場合、値 `-1` は `src(e) == v` を示し、値 `1` は `dst(e) == v` を示します。それ以外の場合、値は `0` です。

無向グラフの場合、`oriented=false` の場合は両方のエントリが `1` になります。それ以外の場合は、`[v, i] -> -1` および `[u, i] -> 1` となります（`v < u` の場合）。
