```
erdos_renyi(n::Int, p::Real, G=Graph; seed=-1)
erdos_renyi(n::Int, m::Int, G=Graph; seed=-1)
```

指定されたタイプ `G` の `n` 頂点を持つ [エルデシュ–レーニモデル](http://en.wikipedia.org/wiki/Erdős–Rényi_model) のランダムグラフを作成します。最初のメソッドでは、頂点のペア間に確率 `p` でエッジが追加されます。2 番目のメソッドでは、`m` エッジがランダムに選択されます。

デフォルトでは無向グラフが作成されます。最後の引数として有向グラフタイプ（例：`DiGraph`）を渡すことで、有向グラフを作成できます。

また、`erdos_renyi(n, ne)` または `Graph(nv, ne)` コンストラクタを使用することで、エルデシュ–レーニグラフを迅速に生成できることにも注意してください。これにより、すべての潜在的なエッジの中からランダムに `ne` エッジが選択されます。
