```
CardinalCubicSplinePrime{T}(a)
```

は、浮動小数点型 `T` とパラメータ `a` に対する Keys カーディナルキュービックスプラインの1次導関数であるカーネルインスタンスを生成します（詳細は [`CardinalCubicSpline`](@ref) を参照）。この導関数は次のように与えられます：

```
ker′(x) = (3(a + 2)*|x| - 2(a + 3))*x           もし |x| ≤ 1
          (3a)*(|x| - 2)*(|x| - 4/3)*sign(x)    もし 1 ≤ |x| ≤ 2
          0                                     もし |x| ≥ 2
```
