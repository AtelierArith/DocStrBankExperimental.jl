# 補間カーネル

補間カーネル `Kernel{T,S}` は、その係数の浮動小数点型 `T` とサポートの整数サイズ `S` によってパラメータ化されます。効率の理由から、`InterpolationKernels` では（小さい）有限サイズのサポートを持つカーネルのみが実装されています。

カーネル `ker` は、実数引数を持つ関数として使用できる呼び出し可能なオブジェクトです：

```
ker(x::Real)
```

はオフセット `x` におけるカーネル値を返します。`x` の型に関わらず、`ker(x)` は常に `T = eltype(ker)` の型であり、これは `ker` に関連付けられた浮動小数点型です。すべてのカーネルサポートは対称です。つまり、`ker(x)` は `abs(x) > S/2` の場合にゼロになります。ここで、`S = length(ker)` はカーネルサポートのサイズです。

## カーネルの浮動小数点型変換

実数引数を持つ関数として呼び出されると、指定されたカーネルはその関連付けられた浮動小数点型の値を返します。これは高速な補間メソッドを持つために選ばれています。カーネル `ker` を浮動小数点型 `T` を使用するように変換するのは、次のいずれかによって簡単に行えます：

```
T(ker)
Kernel{T}(ker)
convert(Kernel{T}, ker)
```

浮動小数点型を変更すると、カーネルに数値パラメータがある場合、精度の損失が生じる可能性があることに注意してください。

## 一般的なメソッド

いくつかの一般的なメソッドは、任意の補間カーネル `ker` に特化されています：

```
eltype(ker) -> T
```

は計算のための浮動小数点型を返し、

```
length(ker) -> S
```

はこのカーネルによる補間に関与する隣接点の数でもある `ker` のサポートのサイズを返します。

```
values(ker)
```

は、次のようにして同一のインスタンスを構築できる `ker` のパラメータのタプルを返します：

```
typeof(ker)(values(ker)...)
```

最後に：

```
compute_offset_and_weights(ker, x) -> off, (w1, w2, ..., wS)
```

はオフセット `off` と、座標 `x` で配列を補間するための補間重みの `S` タプルを返します（分数インデックス単位で）。
