```
LanczosKernel{S,T}()
```

は、サポートサイズ `S`（偶数でなければならない）および浮動小数点型 `T` のランツォス再サンプリングカーネルのインスタンスを生成します。

ランツォス再サンプリングカーネルは、サポートサイズが大きい場合に正規化される傾向のある偶数のカーディナル関数です。これらは次のように定義されます：

```
ker(x) = S/(2*(π*x)^2)*sin(π*x)*sin(2*π*x/S)     もし |x| ≤ S/2
         0                                       もし |x| ≥ S/2
```

式 `ker'` は、ランツォス再サンプリングカーネル `ker` の1階導関数を返します（コンストラクタ [`LanczosKernelPrime`](@ref) も参照してください）。
