```
CatmullRomSpline{T}()
```

は、浮動小数点型 `T` の Catmull-Rom 補間カーネルのインスタンスを生成します。省略した場合は `Float64` と仮定されます。

Catmull-Rom 補間カーネルは、次のように定義された基本的な区分的三次スプラインです：

```
ker(x) = ((3/2)*|x| - (5/2))*x^2 + 1             if |x| ≤ 1
         (((5/2) - (1/2)*|x|)*|x| - 4)*|x| + 2   if 1 ≤ |x| ≤ 2
         0                                       if |x| ≥ 2
```

式 `ker'` は、Catmull-Rom 補間カーネル `ker` の 1 次導関数であるカーネルインスタンスを生成します（コンストラクタ [`CatmullRomSplinePrime`](@ref) も参照してください）。
