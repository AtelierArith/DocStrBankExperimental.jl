```
CardinalCubicSpline([T=Float64,] c, B=Flat)
```

は、浮動小数点型 `T`、テンションパラメータ `c` および境界条件 `B` のためのカーディナル三次スプライン補間カーネルのインスタンスを生成します。

`x = ±1` における傾きは `±(c - 1)/2` です。通常 `c ≤ 1` であり、`c = 0` を選択すると Catmull-Rom スプラインが得られます（[`CatmullRomSpline`](@ref) を参照）、`c = 1` はすべての接線がゼロになります、`c = -1` はカーディナルサインの切り捨て近似を生成し、`c = -1/2` は連続した二次導関数を持つ補間三次スプラインを生成します（そのサポート内で）。

式 `ker'` は、カーディナル三次スプライン `ker` の1次導関数であるカーネルインスタンスを生成します（コンストラクタ [`CardinalCubicSplinePrime`](@ref) も参照）。
