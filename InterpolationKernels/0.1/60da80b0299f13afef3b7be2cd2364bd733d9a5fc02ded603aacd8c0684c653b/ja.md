```
LanczosKernel([T=Float64,] S, B=Flat)
```

は、浮動小数点型 `T` と境界条件 `B` に対して、サポートサイズ `S`（偶数でなければならない）のランツォス再サンプリングカーネルのインスタンスを生成します。

ランツォス再サンプリングカーネルは、サポートサイズが大きい場合に正規化される傾向のある偶数のカーディナル関数です。 それらは次のように定義されます（詳細は https://en.wikipedia.org/wiki/Lanczos_resampling を参照）：

```
ker(x) = S/(2*(π*x)^2)*sin(π*x)*sin(2*π*x/S)     もし |x| ≤ S/2
         0                                       もし |x| ≥ S/2
```

式 `ker'` は、ランツォス再サンプリングカーネル `ker` の最初の導関数を生成します（コンストラクタ [`LanczosKernelPrime`](@ref) も参照）。
