```
CubicSpline([T=Float64,] B=Flat)
```

は、浮動小数点型 `T` と境界条件 `B` のための三次スプラインのインスタンスを生成します。

4次（立方）Bスプラインカーネルは、*パルゼンウィンドウ*または*ド・ラ・ヴァレ・プシンウィンドウ*としても知られています。これは次のように与えられます：

```
ker(x) = 2/3 + (|x|/2 - 1)*x^2       if |x| ≤ 1
         (1/6)*(2 - |x|)^2           if |x| ≤ 2
         0                           if |x| ≥ 2
```

式 `ker'` は、三次スプライン `ker` の1次導関数であるカーネルインスタンスを生成します（コンストラクタ [`CubicSplinePrime`](@ref) も参照してください）。
