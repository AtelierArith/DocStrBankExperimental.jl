```
LinearSpline([T=Float64,] B=Flat)
```

は、浮動小数点型 `T` と境界条件 `B` の線形スプラインのインスタンスを生成します。

線形スプライン（*三角カーネル*、*バートレットウィンドウ*、または *フェジェウィンドウ* とも呼ばれます）は、次のように定義される2次（線形）Bスプラインです：

```
ker(x) = 1 - |x|       もし |x| ≤ 1
         0             もし |x| ≥ 1
```

式 `ker'` は、線形スプライン `ker` の1次導関数であるカーネルインスタンスを生成します（コンストラクタ [`LinearSplinePrime`](@ref) も参照してください）。
