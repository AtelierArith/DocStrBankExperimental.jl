区間 `[0,1]` のフーリエ基底。基底関数は `exp(2 π i k)` で与えられ、`k` は奇数長（`2N+1`）のフーリエ級数の場合は `-N` から `N` まで、偶数長の場合は `-N+1` から `N` までの範囲です。後者の場合、周波数 `N` に対応する最高周波数の基底関数はコサインです。

基底関数は、典型的なFFT実装で期待される順序で並べられています。周波数 k は次の順序で配置されています：

```
0 1 2 3 ... N -N -N+1 ... -2 -1,
```

奇数長の場合と

```
0 1 2 3 ... N -N+1 ... -2 -1,
```

偶数長の場合です。

フーリエ基底は、連続測度（奇数長フーリエ基底のみ）および離散測度に関して直交正規です。
