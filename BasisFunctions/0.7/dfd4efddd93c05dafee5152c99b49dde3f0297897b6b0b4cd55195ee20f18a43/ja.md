微分関数は、辞書内の関数を微分するために使用できる演算子を返し、結果は別の辞書での展開として得られます。

微分演算子は、ソース辞書と宛先辞書のいくつかの組み合わせに対して効率的です。辞書が少なくとも1つの微分演算子をサポートしている場合、`hasderivative(dict)`はtrueとなり、`derivative_dict(dict)`は宛先辞書を返します。

微分の順序は、オプションの第2引数として渡すことができます。

例は次の通りです：

```
differentiation(Φ::Dictionary)
```

これは、デフォルトの宛先辞書、順序1、および次元1を使用します。別の例は次の通りです：

```
differentiation(Φ, 1)
```

または

```
differentiation(Φ, (0,1))
```
