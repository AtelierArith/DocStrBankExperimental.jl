```
psifluxes=streamfunctionvolumeflux(mask::BitArray,velocities,pmnin,xiin;dim::Integer=ndims(mask))
```

# 入力:

  * `mask` : 水中は真、陸上は偽のブール配列。
  * `velocities` : マスクと同じグリッド上の配列のタプル。各タプル要素は、空間の対応する方向に対して法線の速度場。
  * `pmnin`: divandと同様のメトリックのタプル。ただし、m/sの速度を得るためには、メトリックもメートルあたりである必要があります。
  * `xiin`: グリッドポイントの位置のタプル。
  * `dim` : 配列のどのインデックスが垂直方向に対応するかを示すオプションのパラメータ。デフォルトでは最後の次元が使用されます。

# 出力:

  * `psifluxes` 各深さと方向での体積フラックスのタプル、各座標線の法線および左側に対して。

表面から計算された体積フラックスストリーム関数を計算します。このフィールドの値は、見ているzレベルの深さの上の断面を横切る総流量（Sverdrup単位）を提供します。
