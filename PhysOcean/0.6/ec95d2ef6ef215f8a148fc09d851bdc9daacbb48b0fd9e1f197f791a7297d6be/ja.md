```
rhoi = integraterhoprime(rhop,z,dim=ndims(rhop))
```

深さにわたる密度異常を統合します。重力と一緒に使用する場合、重力がzに依存しないと仮定すると、定数までの動的圧力を計算するために使用できます。この関数は1D、2D、...で使用できます。

# 入力:

  * `rhop`: 密度異常配列
  * `z`: 垂直位置配列。表面でゼロ、下向きに正、rhopと同じ次元
  * `dim`: 深さが見つかり、積分が行われる次元。指定されていない場合は、最後の次元が取られます

# 出力:

  * `rhoi` : rhopが与えられたのと同じレベルに対する統合値。基本的に現在の深さの上にある総密度異常

# 注意:

密度異常の垂直積分を計算します
