```
GAST = gast(JD_UT1)
```

UT1ユリウス日をその時点でのGASTに変換します。

入力値は、通常のSOFA方式で返される2つの部分からなるユリウス日であり、時間の解像度を保持するように設計されています。完全なユリウス日は、ベクトルの2つの成分を加えることで単一の数値として利用可能です。

グリニッジ見かけの恒星時は、春分点からの本初子午線の真の回転を表す角度です。いくつかの実装があり、`model`入力によって選択できます。1982/1994（`94`、デフォルト）、2000（`00`）または2006（`06`）です。

SOFAの`gst94`、`gmst00`、および`gmst06`から派生しています。
