```
r_TEME = tod2teme(r_tod, JD)
```

与えられたTTユリウス日を使用して、TODベクトルをTEMEに変換します。IAU-76モデルを使用しています。

状態ベクトルは長さ3でなければなりません。

入力ユリウス日は、通常のSOFA方式で2つの部分に分けて与えられ、時間解像度を保持するように設計されています。完全なユリウス日は、ベクトルの2つの成分を加えることで単一の数値として利用できます。

TEMEフレームのValladoの説明から派生しています。TEMEは完全には公開されておらず、その定義の曖昧さからいくつかの誤差を含む可能性があることに注意してください。
