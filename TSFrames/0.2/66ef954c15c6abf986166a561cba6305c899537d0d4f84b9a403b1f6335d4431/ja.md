# 行のマージ (vcat/rbind)

```julia
vcat(ts1::TSFrame, ts2::TSFrame; colmerge::Symbol=:union)
```

2つのTSFrameオブジェクトの行を連結し、`ts2`を`ts1`に追加します。

`colmerge`キーワード引数は、列のマージ戦略を指定します。`colmerge`の値は、`DataFrames.vcat`の`cols`引数に直接渡されます。

現在、`DataFrames.vcat`は4種類の列マージ戦略をサポートしています：

1. `:setequal`: 両方のオブジェクトが同じ列名を持つ場合のみマージし、`ts1`の列の順序を使用します。
2. `:orderequal`: 両方のオブジェクトが同じ列名を持ち、列が同じ順序である場合のみマージします。
3. `:intersect`: 両方のオブジェクトに共通する列のみをマージし、残りは無視します。
4. `:union`: 列が異なっていてもマージし、必要に応じてすべての列が`missing`で埋められた結果のオブジェクトを生成します。
