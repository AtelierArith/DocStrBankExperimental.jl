```
orbit(g::AbstractVector{<:SymOperation}, kv::KVec, cntr::Char)  -->  Vector{KVec{D}}
orbit(lg::LittleGroup)
orbit(lgir::LGIrrep)
```

逆格子ベクトル `kv` の群 `g` による作用の下での軌道を返します。これは **k** の星とも呼ばれます。

群 `g` における `kv` の軌道は、`g` のすべての対称操作と `kv` の合成によって得られる非同等の **k**-点の集合です。二つの逆格子ベクトル $\mathbf{k}$ と $\mathbf{k}'$ は、原始逆格子ベクトルによって異なる場合に同等と見なされます。

`kv` と `g` が従来の基底で指定されているが、非原始格子を参照している場合、中心タイプ `cntr` を提供する必要があります。これにより、原始（従来ではない）逆格子ベクトルによる同等性のみが考慮されます。群 `g` の中心タイプが `g` から推測できる場合（例えば、`g` が `SpaceGroup` の場合）、`orbit` は従来の設定を仮定し、推測された中心タイプを使用します。そうでない場合、`cntr` が明示的に設定されていないか推測できない場合は、原始設定が仮定されます。
