```
littlegroups(sgnum::Integer, D::Union{Val{Int}, Integer}=Val(3)) 
                                                    -> Dict{String, LittleGroup{D}}
```

与えられた空間群番号 `sgnum` と次元 `D` に対して、高対称k点、線、平面における関連するリトルグループ（`LittleGroups{D}`）を返します（詳細は [`lgirreps`](@ref) を参照）。

リトルグループ **k**-点ラベルをキーとし、`LittleGroup{D}` のベクトルを値とする `Dict` を返します。

## 注意事項

従来の結晶学的設定が仮定されています（[`spacegroup`](@ref) を参照）。

`spacegroup` とは異なり、対称操作の「センタリング」コピーは返される `LittleGroup` に含まれていません。例えば、空間群110（体心、センタリング記号 'I'）は、従来の設定でセンタリング変換 `[1/2,1/2,1/2]` を持ちます：したがって、`spacegroup` によって返される対称操作には `{1|0}` と `{1|½,½,½}` の両方が含まれますが、`littlegroups` によって返される対称操作には `{1|0}` のみが含まれます（このように続きます）。

現在、`D = 3` のみがサポートされています。
