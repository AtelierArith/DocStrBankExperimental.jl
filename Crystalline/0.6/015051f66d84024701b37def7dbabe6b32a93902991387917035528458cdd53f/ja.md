```
subduction_count(Dᴳᵢ, Dᴴⱼ[, αβγᴴⱼ]) --> Int
```

2つの群 $G$ と $H$ を考えます。ここで、$H$ は $G$ の部分群であり、すなわち $H<G$ です。関連する不可約表現は `Dᴳᵢ` = $D^G_i(g)$ および `Dᴴⱼ` = $D^H_j(g)$ であり、操作 $g∈G$ および $h∈H<G$ に対して定義されます。サブダクション還元公式（または「魔法」公式/シュール直交性関係）から、2つの不可約表現の間の互換性関係を計算し、サブダクションされた表現 $D^G_i↓H$ が不可約表現 $D^H_j$ を何回含むか $n^{GH}_{ij}$ を返します。言い換えれば、これは2つの不可約表現の間の互換性を示します。

オプションとして、ベクトル `αβγᴴⱼ` を提供することができ、具体的な $(α,β,γ)$ の値で `Dᴳᵢ` のキャラクター/不可約表現を評価します。これは、非特別な **k**-ベクトルにおける `LGIrrep` に対して意味があります。デフォルトは `nothing` です。

結果は、還元公式を使用して計算されます [参照：例えば [arXiv:1706.09272v2](https://arxiv.org/abs/1706.09272) の式 (15)]:

$$
n^{GH}_{ij} = |H|^{-1} \sum_h \chi^G_i(h)\chi^H_j(h)^*
$$

## 例

空間群 207 における2つの互換性のある **k**-ベクトル Γ（点）と Σ（直線）を考えます：

```jl
lgirsd  = lgirreps(207, Val(3));
Γ_lgirs = lgirsd["Γ"]; # Γ ≡ [0.0, 0.0, 0.0] のとき
Σ_lgirs = lgirsd["Σ"]; # Σ ≡ [α, α, 0.0] のとき
```

次のようにして、互換性をテストできます：

```jl
[[subduction_count(Γi, Σj) for Γi in Γ_lgirs] for Σj in Σ_lgirs]
> # Γ₁ Γ₂ Γ₃ Γ₄ Γ₅
>  [ 1, 0, 1, 1, 2] # Σ₁
>  [ 0, 1, 1, 2, 1] # Σ₂
```

Γ と Σ における不可約表現間の互換性関係の解釈は次の通りです：

|         互換性関係 |       重複度 |
| -------------:| ---------:|
|       Γ₁ → Σ₁ |     1 → 1 |
|       Γ₂ → Σ₂ |     1 → 1 |
|  Γ₃ → Σ₁ + Σ₂ | 2 → 1 + 1 |
| Γ₄ → Σ₁ + 2Σ₂ | 3 → 1 + 2 |
| Γ₅ → 2Σ₁ + Σ₂ | 3 → 2 + 1 |

この場合、すべての小さな不可約表現は1次元です。
