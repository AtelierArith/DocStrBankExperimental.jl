```
cartesianize(op::SymOperation{D}, Rs::DirectBasis{D}) --> SymOperation{D}
```

`opˡ`を格子基底から直交基底に変換します。変換された演算子 `opᶜ = 𝐑*opˡ*𝐑⁻¹` を直交基底行列 𝐑（その列は `DirectBasis` ベクトル `Rs[i]` です）を用いて計算します。

# 注 1

行列 𝐑 は、格子基底 𝐯ˡ のベクトル係数を直交基底 𝐯ᶜ の係数にマッピングします。すなわち、𝐯ˡ = 𝐑⁻¹𝐯ᶜ であり、逆に 𝐯ᶜ = 𝐑𝐯ˡ です。一般的な変換 P は、係数 𝐯 を持つ「元の」ベクトルを新しい係数 𝐯′ に変換します：𝐯′ = P⁻¹𝐯。ここで、格子基底を「元の」基底と考えるため、P = 𝐑⁻¹ となります。このように、演算子 `op` の変換は `opᶜ = P⁻¹*opˡ*P` となり、すなわち `opᶜ = transform(opˡ,P) = transform(opˡ,𝐑⁻¹)` です。

# 注 2

`SymOperation` の表示（例えば、Seitz および xyzt 表記）は、REPL でその点群行列の整数係数を暗黙的に仮定します。その結果、直交基底での `SymOperation` の表示は未定義の動作を引き起こす可能性があります。ただし、行列表現は有効なままです。
