```
ENANTIOMORPHIC_PAIRS :: NTuple{11, Pair{Int,Int}}
```

3Dの11のエナンチオモルフィック空間群ペアの空間群番号を返します。

各ペア `(sgnum, sgnum')` に関連付けられた空間群タイプは、鏡変換によって関連しています。すなわち、2つの群 $G = \{g\}$ と $G' = \{g'\}$ の間に、$\mathbb{P} = \{\mathbf{P}|\mathbf{p}\}$ という変換が存在し、$G' = \mathbb{P}^{-1}G\mathbb{P}$ となるようなものであり、ここで $\mathbf{P}$ は不適切です（すなわち、$\mathrm{det}\mathbf{P} < 0$）。

我々は、適切な変換（すなわち、$\mathrm{det}\mathbf{P} > 0$）によって関連付けることができない空間群の*タイプ*を区別します。その観点から、230の空間群タイプがあります。不適切な回転を許可する条件に緩和すると、$230-11 = 219$ の異なる*アフィン*空間群タイプがあります。例えば、ITA5のセクション8.2.2を参照してください。

エナンチオモルフィック空間群タイプは、3Dのキラル空間群タイプでもあります。低次元にはエナンチオモルフィックペアは存在せず、3Dではすべてのエナンチオモルフィックペアがスクリュー対称性を含み、その方向はペア間で反転します（すなわち、逆の手の性質を持ちます）。
