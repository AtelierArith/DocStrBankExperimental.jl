```
α(tables::AbstractVector{String} [, profile=:hartmann_tran; kwargs...])
```

データベースのテーブルに保存された行ごとのデータを使用して吸収係数を計算します。行形状は、`profile` 引数と `:hartmann_tran`, `:voigt`, `:sdvoigt`, `:lorentz`, `:gauss` のいずれかのシンボルキーを使用してオプションで指定できます。キーワード引数が指定されていない場合、それらは提供されたテーブルから自動的に選択されます。

# キーワード引数

  * `components`: 計算のためのガス混合物の成分。`(molecule_id, local_iso_id)` のタプルのベクターまたは、`(molecule_id, local_iso_id)` タプルをキーとし、豊富さを値とする `Dict` である必要があります。タプルのベクターが提供されると、自然豊富さが使用されるため、同じ分子の同位体以外のガス混合物には意味がありません。
  * `intensity_threshold`: 最小線強度 $cm^{-1}/(\text{molecule} \cdot cm^{-2})$
  * `pressure`: 環境圧力（大気圧、デフォルト: 1.0 atm）
  * `temperature`: 環境温度（ケルビン、デフォルト: 296.0 K）
  * `ν`: 吸収を計算するための波数のベクターまたは範囲。出力の ν ベクターを複数の計算で再利用するのに便利です。ν が提供されると ν_range は無視されます。[]
  * `ν_range`: 吸収スペクトルの最小/最大波数をそれぞれ $cm^{-1}$ で表す (ν*min, ν*max) の形式のタプル
  * `ν_step`: 波数ステップ $cm^{-1}$（デフォルト: 0.01 $cm^{-1}$）
  * `ν_wing`: 波数の絶対計算幅 $cm^{-1}$（デフォルト: 0 $cm^{-1}$）
  * `ν_wing_hw`: 半幅の倍数での線の相対計算幅（デフォルト: 50）
  * `diluent`: 希釈物質の `Dict`、`Symbol` として指定され、例えば `:air` や `:H2O` をキーとし、相対濃度を値とします（デフォルト: `Dict(:self => 1.0)`）。詳細については例を参照してください。
