```
CFL(Δt [, timescale = Oceananigans.Advection.cell_advection_timescale])
```

時間ステップ `Δt` または `TimeStepWizard` と `timescale` に関連するクーラン-フリードリッヒス-ルーウィ (CFL) 数を計算するためのオブジェクトを返します。

[`AdvectiveCFL`](@ref Oceananigans.Diagnostics.AdvectiveCFL) および [`DiffusiveCFL`](Oceananigans.Diagnostics.DiffusiveCFL) も参照してください。
