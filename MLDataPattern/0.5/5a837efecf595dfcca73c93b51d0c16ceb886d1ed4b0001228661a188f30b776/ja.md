```
leaveout(data, [size = 1], [obsdim]) -> FoldsView
```

`data` コンテナを k-fold 戦略を使用して再分割します。ここで `k` は、結果のフォールドの各検証サブセットが大体 `size` の観測値を含むように選ばれます。デフォルトは `size = 1` で、これは「leave-one-out」分割とも呼ばれます。

結果のフォールドのシーケンスは遅延 [`FoldsView`](@ref) として返され、インデックスを付けたり反復処理したりできます。いずれにせよ、実際のデータサブセットのみが作成されます。つまり、[`getobs`](@ref) が呼び出されるまで、実際のデータはコピーされません。

```julia
for (train, val) in leaveout(X, size = 2)
    # もし nobs(X) が 2 で割り切れる場合、
    # それなら nobs(val) は各反復で 2 になります、
    # そうでなければ最初の数回の反復で 3 になるかもしれません。
end
```

詳細については [`FoldsView`](@ref) を参照するか、関連する関数については [`kfolds`](@ref) を参照してください。
