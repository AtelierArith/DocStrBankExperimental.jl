```
Pkn_NGG(n, β, σ)
```

k=0からk=nまでのkクラスタを得るための事前確率質量を計算します。直接計算またはCnkの再帰計算を使用して、任意の精度で計算します。kの大きな値に対して安定しています。kの小さな値で不安定になった場合、関数は警告を表示し、Gibbs型プロセスの予測分布に基づく近似スキームを使用することに戻ります（Bystrova, 2020）。

参考文献：

  * Bystrova, D, Kon Kam King, G, Deslandes, F, Arbel, J. ベイズ非パラメトリックモデルにおけるクラスタの事前分布の近似, 2020（準備中）。

# 例

```julia-repl
julia> GibbsTypePriors.Pkn_NGG(10, 100, 1.2, 0.6)
```
