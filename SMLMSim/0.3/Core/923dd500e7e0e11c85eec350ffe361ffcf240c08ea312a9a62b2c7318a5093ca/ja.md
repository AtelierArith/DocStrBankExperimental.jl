```
Nmer2D <: Pattern2D
```

直径 d の円の周りに対称的に配置された N 分子。

# フィールド

  * `n::Int`: パターン内の分子の数
  * `d::Float64`: マイクロメートル単位の円の直径
  * `x::Vector{Float64}`: マイクロメートル単位の分子の X 位置
  * `y::Vector{Float64}`: マイクロメートル単位の分子の Y 位置

# 例

```julia
# 直径 100nm の 8 分子パターンを作成
nmer = Nmer2D()

# 6 分子と 200nm の直径を持つカスタムパターンを作成
nmer = Nmer2D(; n=6, d=0.2)
```
