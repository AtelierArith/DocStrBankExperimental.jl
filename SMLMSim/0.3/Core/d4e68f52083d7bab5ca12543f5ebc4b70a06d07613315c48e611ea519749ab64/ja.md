```
Nmer3D <: Pattern3D
```

z=0の円の直径dの周りに対称的に配置されたN分子。

# フィールド

  * `n::Int`: パターン内の分子の数
  * `d::Float64`: 円の直径（マイクロメートル単位）
  * `x::Vector{Float64}`: 分子のX位置（マイクロメートル単位）
  * `y::Vector{Float64}`: 分子のY位置（マイクロメートル単位）
  * `z::Vector{Float64}`: 分子のZ位置（マイクロメートル単位）

# 例

```julia
# 直径100nmの8分子パターンを作成
nmer = Nmer3D()

# 6分子と直径200nmのカスタムパターンを作成
nmer = Nmer3D(; n=6, d=0.2)
```
