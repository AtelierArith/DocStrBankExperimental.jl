```
TabulatedInterpolator([T,] [d = nothing,] ker, pos, nrows, ncols)
```

は、カーネル `ker` を使用して、長さ `ncols` の次元に沿って補間する線形マップを生成し、長さ `nrows` の次元を生成します。関数 `pos(i)` は `i ∈ 1:nrows` に対して、長さ `ncols` の次元に沿った分数インデックス単位で補間する位置を提供します。引数 `d` は、多次元配列に対して演算子が適用される際に補間する次元のランクです。演算子が作成されるときに指定されていない場合、演算子が適用されるたびに指定する必要があります（下記参照）。

補間する位置は、次のようにベクトル `x` で指定することもできます：

```
TabulatedInterpolator([T,] [d = nothing,] ker, x, ncols)
```

これにより、出力次元が `nrows = length(x)` の補間器が生成されます。ここで `x` は抽象ベクトルであることができます。

オプションの引数 `T` は、線形マップの係数の浮動小数点型です。デフォルトでは、引数 `ker` の要素型の昇格によって与えられ、指定されている場合は `x` の要素型によっても与えられます。

タブulated演算子、例えば `A` は、引数 `x` に適用できます：

```
apply!(α, P::Operations, A, [d,] x, scratch, β, y) -> y
```

これにより、`y` が `α*P(A)⋅x + β*y` で上書きされます。`x` と `y` が多次元の場合、補間する次元 `d` を指定する必要があります。
