```
A = SparseInterpolator{T=eltype(ker)}(ker, pos, grd)
```

は、カーネル `ker` を用いて位置 `pos` で補間するのに適した疎線形補間器を生成します。オプションのパラメータ `T` は、演算子 `A` の係数の浮動小数点型です。疎補間器 `A` の係数の型を照会するには `eltype(A)` を呼び出します。

次に `y = apply(A, x)` または `y = A(x)` または `y = A*x` は、補間配列 `x` の結果を返します。`y` の形状は `pos` と同じです。形式的には、これは次の計算に相当します：

```
y[i] = sum_j ker((pos[i] - grd[j])/step(grd))*x[j]
```

ここで `step(grd)` はグリッド `grd` のノード間の（定数の）ステップサイズであり、`grd[j]` はグリッドの `j` 番目の位置です。
