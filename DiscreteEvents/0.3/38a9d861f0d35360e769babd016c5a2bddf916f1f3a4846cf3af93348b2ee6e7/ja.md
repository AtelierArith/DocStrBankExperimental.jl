```
Prc(id, f, arg...; kw...)
Prc(f, arg...; kw...)
```

プロセスとして実行され、時計に登録される関数を準備します。

# 引数、フィールド

  * `id`: 登録のための一意の識別子、
  * `f::Function`: 関数 `f(clk, arg...; kw...)`、最初の引数として `clk`（[`Clock`](@ref)）を取る必要があります、
  * `arg...`: `f`へのさらなる引数
  * `kw...`: `f`へのキーワード引数

# 登録中に特定されたフィールド

  * `id`: 提供されていない場合、登録中にいくつかの整数が計算されます、
  * `task::Union{Task,Nothing}`: 診断に使用されるタスク構造、
  * `clk::Union{AbstractClock,Nothing}`: プロセスが登録される時計、

!!! note "時計は動的に特定されます！"
    プロセスが実行され、登録される時計 `clk` はプロセスの起動中に特定され、その後 `f` への最初の引数として渡されます。
