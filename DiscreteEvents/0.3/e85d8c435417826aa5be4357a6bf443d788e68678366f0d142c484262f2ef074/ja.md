```
process!([clk], prc, cycles; <キーワード引数>)
```

[`Prc`](@ref)をクロックに登録し、プロセス関数をループで実行する非同期タスクを開始し、登録された`id`を返します。これは`clk.processes[id]`の下で見つけることができます。

# 引数

  * `c<:AbstractClock`: 提供されない場合、プロセスは`𝐶`の下で実行されます、
  * `prc::Prc`: 関数とその引数を含みます、
  * `cycles<:Number=Inf`: プロセスが実行すべきループサイクルの数、

# キーワード引数

  * `cid::Int=clk.id`: cid ≠ clk.idの場合、イベントをid == cidの並列クロックに割り当てます。これは`spawn`を上書きします、
  * `spawn::Bool=false`: trueの場合、プロセスは別のスレッドで並行してスケジュールされ、スレッド特有のクロックに登録される可能性があります。
