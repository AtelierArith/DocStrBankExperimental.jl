```
DiscreteEvents
```

離散イベントを生成およびシミュレーションするためのJuliaパッケージです。これはJulia `VERSION ≥ v"1.0"`で動作します。マルチスレッドは`VERSION ≥ v"1.3"`を必要とします。

そのGithubリポジトリは[https://github.com/pbayer/DiscreteEvents.jl]です。

現在の安定した登録バージョンは次のコマンドでインストールできます。

```julia
pkg> add DiscreteEvents
```

開発バージョンは次のコマンドでインストールできます。

```julia
pkg> add("https://github.com/pbayer/DiscreteEvents.jl")
```
