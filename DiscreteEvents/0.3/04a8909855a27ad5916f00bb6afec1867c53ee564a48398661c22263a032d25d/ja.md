```
fun(f::Function, args...; kwargs...)
```

関数 `f` とその引数をクロージャに保存して、後で実行します。

# 引数

`fun` に渡される引数 `args...` とキーワード引数 `kwargs...` は、実行時に `f` に渡されますが、`fun` でキャプチャされた時と `f` の後の実行時で値が変わる可能性があります。`f` が実行時に現在の値を必要とする場合、2つの可能性があります：

1. `fun` は、値や可変引数の代わりに `fun`、関数クロージャ、シンボル、または式を取ることができます。これらは、`f` に渡される直前のイベント時に評価されます。1つの例外があります：`f` が `event!` の場合、その引数は未評価のまま渡されます。
2. 可変型引数（Array、構造体など）は常に現在のものです。関数内からその内容を変更することもできます。

`fun` で `Symbol` や `Expr` を使用すると、一度だけ警告が表示されます。これらは `Main` モジュールのグローバルスコープでのみ評価されるため、他のモジュールでは使用できません。
