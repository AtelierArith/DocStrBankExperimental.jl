```
@event f(farg...) c(carg...)
@event f(farg...) ca
```

関数 `f(farg...)` を条件付きイベントとして時計にスケジュールします。

# 引数

  * `f`: イベント時に実行される関数、
  * `farg...`: その引数、最初の引数は時計でなければなりません、
  * `c`: 時計のサンプルレートで評価される関数、trueを返すとイベントがトリガーされます、
  * `carg...`: c への引数、
  * `ca`: 時計のサンプルレートで評価される形式 `()->...` の匿名関数、trueを返すとイベントがトリガーされます。
