```
RTClock{E <: ClockEvent} <: AbstractClock
```

リアルタイムクロックは、スケジュールされたイベントを実行するために、指定された期間ごとにチェックを行います。これは、開始または最後のリセットからの秒数での時間を持ち、更新にはシステム時間を使用します。

リアルタイムクロックはチャネルを介して制御されます。複数のリアルタイムクロックを任意の期間（≥ 1 ms）で設定できます。リアルタイムクロックは直接作成するのではなく、`CreateRTClock`を使用して作成するべきです。

# フィールド

  * `Timer::Timer`:     クロックの周期（秒単位）、最小は0.001（1 ms）
  * `clock::Clock`:     クロックの動作
  * `cmd::Channel{T}`:  非同期クロックへのコマンドチャネル
  * `back::Channel{T}`: 非同期クロックからのバックチャネル
  * `id::Int`:          任意のID番号
  * `thread::Int`:      非同期クロックが存在するスレッド
  * `time::Float64`:    開始からのクロック時間（秒単位）
  * `t0::Float64`:      クロック開始時のシステム時間（秒単位）
  * `T::Float64`:       クロックの周期（秒単位）
