```
createRTClock(T::Float64, id::Int, thrd::Int=nthreads(); ch_size::Int=256)
```

リアルタイムクロックを作成し、開始して返します。

クロックは現在のシステム時間を取得し、指定された周期 `T` で秒単位でカウントを開始します。イベントやサンプリング関数はそれにスケジュールできます。

# 引数

  * `T::Float64`:           秒単位の周期（クロック解像度）、T ≥ 0.001
  * `id::Int`:              0:nthreads() 以外のクロック識別番号
  * `thrd::Int=nthreads()`: クロックタスクが実行されるスレッド
  * `ch_size::Int=256`:     クロック通信チャネルのサイズ
