```
mie_cdf(m, x, num; norm = :albedo)
```

非偏光散乱のCDFをcos(θ)で均等に間隔を空けて作成します。CDFは散乱（出口）角が180度から0度までの範囲をカバーします。（コサインは-1から1まで均等に分布しています。）角度がcos(theta)で均等に分布しているため、散乱関数は均等にサンプリングされず、したがって非常に異方性の強い位相関数を適切にサンプリングするためには巨大な配列サイズが必要です。この関数は累積分布関数であるため、最大値は1であるべきです。

# パラメータ

  * `m`: 球体の複素屈折率
  * `x`: 球体のサイズパラメータ
  * `num`: 希望するCDF配列の長さ

# 出力

  * `µ`: 角度のコサインの配列
  * `cdf`: 累積分布関数の値の配列
