```
ez_intensities(m, d, λ0, μ, n_env = 1.0, norm = :albedo)
```

球体からの散乱強度を返します。これらは、入射平面波の場に平行（ipar）および垂直（iper）な平面における散乱強度です。散乱強度は、偏光されていない強度の4𝜋ステラジアンにわたる積分が単一散乱アルベドに等しくなるように正規化されています。散乱強度の単位は逆ステラジアン [1/sr] です。偏光されていない散乱は、2つの散乱強度の平均です。

# パラメータ

  * `m`: 球体の複素屈折率 [-]
  * `d`: 球体の直径 [lambda0と同じ単位]
  * `λ0`: 真空中の波長 [dと同じ単位]
  * `µ`: 各方向のcos(θ) [-]
  * `n_env`: 球体の周りの媒質の実数屈折率、オプション。

# 出力

`ipar`, `iper`: 平行および垂直平面における散乱強度 [1/sr]
