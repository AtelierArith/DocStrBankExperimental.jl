```
ez_mie(m, d, λ0, n_env = 1.0)
```

球の効率を計算します。

# パラメータ

  * `m`: 球の複素屈折率                             [-]
  * `d`: 球の直径                                   [lambda0と同じ単位]
  * `λ0`: 真空中の波長                             [dと同じ単位]
  * `n_env`: 球の周りの媒質の実数屈折率、オプション。
  * `use_threads`（オプション）: スレッドを使用するかどうかのフラグ（デフォルト: true）

# 出力

  * `qext`: 総消失効率                               [-]
  * `qsca`: 散乱効率                                 [-]
  * `qback`: 後方散乱効率                           [-]
  * `g`: 散乱位相関数の平均コサイン                 [-]
