```
mie_cdf(m, x, num; norm = :albedo)
```

非偏光散乱のためのCDFを作成します。CDFは、散乱（出口）角度が180度から0度までの範囲をカバーします。（コサインは-1から1まで均等に分布しています。）これらの角度μは、非偏光Mie散乱の累積分布関数の均等間隔に対応し、cdf[i] = i/(num-1)となります。これは、CDFを多くの点で計算し、CDFの均等間隔に対応する特定の角度を見つけるためにスキャンすることで問題を解決する単純な実装です。これは累積分布関数であるため、最大値は1であるべきです。

# パラメータ

  * `m`: 球体の複素屈折率
  * `x`: 球体のサイズパラメータ
  * `num`: 希望するCDF配列の長さ

# 出力

  * `µ`: 角度のコサインの配列（不規則に間隔を空けて）
  * `cdf`: 累積分布関数の値の配列
