```
direct(f, a::Vector{Float64}, b::Vector{Float64}; max_iterations::Int = 100, min_radius::Float64 = 1e-5)
```

グローバル最適化を行うために、探索空間を反復的に細分化するDIRECT（DIvided RECTangles）アルゴリズムを実装します。このアルゴリズムは、元の探索空間（境界 `a` と `b` によって与えられる）から単位ハイパーキューブ [0, 1]^n へのマッピングをオンザフライで行います。

# 引数

  * `f`: 最小化される目的関数。実数のベクトルを受け取り、スカラーを返す必要があります。
  * `a::Vector{Float64}`: 元の探索空間の各次元の下限を示すベクトル。
  * `b::Vector{Float64}`: 元の探索空間の各次元の上限を示すベクトル。
  * `max_iterations::Int`: 実行する最大反復回数（デフォルトは100）。
  * `min_radius::Float64`: さらなる細分化のために許可される最小ハイパーレクタングル半径（デフォルトは1e-5）。

# 戻り値

  * 指定された反復回数後の最終的なハイパーレクタングル区間のセットを表す `DirectRectangle` インスタンスのベクトル。各 `DirectRectangle` には以下が含まれます：

      * `c`: 中心点（単位ハイパーキューブ内）、
      * `y`: 中心での目的関数の値、
      * `d`: 次元ごとの分割数、
      * `r`: 矩形の計算された半径。
