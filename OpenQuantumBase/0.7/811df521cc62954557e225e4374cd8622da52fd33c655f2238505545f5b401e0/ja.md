```julia
haar_unitary(dim)

```

次元 `(dim, dim)` のHaarランダムユニタリ行列をQR分解法を使用して生成します。

# 引数

  * `dim::Int`: 行列の次元。

# 戻り値

  * サイズ `(dim, dim)` のHaarランダムユニタリ行列。

# 詳細

Haarランダムユニタリ行列は、複素数のモジュラスが1のエントリを持つ行列で、ユニタリ群U(dim)上のHaar測度に従ってランダムに選ばれます。この実装ではQR分解法を使用し、サイズ `(dim, dim)` の複素ガウス行列を生成し、それを直交行列と対角成分が正の上三角行列の積にQR分解します。対角成分はその絶対値で正規化され、ユニタリ行列が得られます。

# 例

```julia
julia> haar_unitary(3)
3×3 Matrix{ComplexF64}:
  0.407398+0.544041im   0.799456-0.213793im  -0.439273+0.396871im
 -0.690274-0.288499im   0.347444+0.446328im   0.426262+0.505472im
  0.437266-0.497024im  -0.067918-0.582238im   0.695026+0.186022im
```
