```julia
partial_trace(ρ, qubit_2_keep)

```

密度行列 `ρ` の部分トレースを計算します。すべてのサブシステムがキュービットであると仮定します。`qubit_2_keep` は、対応するキュービットがトレースアウトされないインデックスを示します。

# 例

```julia-repl
julia> ρ1 = [0.4 0.2; 0.2 0.6]; ρ2 = [0.5 0; 0 0.5];
julia> partial_trace(ρ1⊗ρ2, [1])
2×2 Array{Float64,2}:
 0.4  0.2
 0.2  0.6
```
