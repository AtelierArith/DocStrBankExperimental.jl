```julia
partial_trace(ρ, sys_dim, dim_2_keep)

```

密度行列 `ρ` の部分トレースを計算します。`ρ` は $ℋ₁⊗ℋ₂⊗…$ のテンソル空間にあり、`sys_dim` は対応するサブシステムの次元の配列です。`dim_2_keep` はトレースアウトされない対応するサブシステムのインデックスの配列です。

# 例

```julia-repl
julia> ρ1 = [0.4 0.2; 0.2 0.6]; ρ2 = [0.5 0; 0 0.5];
julia> partial_trace(ρ1⊗ρ2, [2, 2], [1])
2×2 Array{Float64,2}:
 0.4  0.2
 0.2  0.6
```
