# pcor_test

```julia
pcor_test(d, u, q, n)

```

2つの変数間の部分相関を、他の変数のセットが与えられた場合に計算します。

### メソッド

```julia
pcor_test(;
* `u::Vector{Symbol}`                  : 相関を計算するために使用される変数
* `q::Int`                             : 条件付きセット内の変数の数
* `n::Int`                             : 観測の数
* `S::Matrix`                          : サンプル共分散行列
)
```

ここで：

u[1], u[2]: 相関を計算するために使用される変数、残りの変数は条件付きセットです

### 戻り値

```julia
* `res::Float64`                       : u[1]とu[2]の間の相関
```
