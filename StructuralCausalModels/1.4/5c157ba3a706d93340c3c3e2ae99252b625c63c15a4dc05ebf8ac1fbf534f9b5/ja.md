# ancestral_graph

```julia
ancestral_graph(amat; m, c)

```

周辺化と条件付けの後の祖先グラフ。

### 必要な引数

```julia
* `amat::NamedArray{Int, 2}`           : DAGの隣接行列
```

### オプションの引数

```julia
* `m::Vector{Symbol}`                  : 周辺化されるDAGのノード
* `c::Vector{Symbol})`                 : 条件付けされるDAGのノード
```

### 戻り値

```julia
* `ag::NamedArray`                     : 残る祖先グラフ
```
