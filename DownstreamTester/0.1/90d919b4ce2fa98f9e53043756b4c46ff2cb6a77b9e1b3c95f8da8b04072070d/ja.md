```
nightly(configfile::String = "DownstreamTester.json"; do_clone::Bool = true, nightlylabels::Vector{String} = ["nightly"])
```

指定された設定ファイルに基づいてJuliaパッケージの夜間テストを実行します。この関数は、夜間のJuliaバージョンで新しいテストの失敗を見つけて報告するために、スケジュールされたGitHubアクションで毎日実行されることを目的としています。呼び出されると、設定ファイルに提供されたパッケージリポジトリのHEADリビジョンをクローンし、それに対して完全なテストスイートを実行します。新しい失敗したテストが見つかった場合、パッケージリポジトリに問題をオープンします。テストが成功した場合は、関連する問題を報告し、そこに報告されたすべてのテストが成功した場合はそれをクローズします。
