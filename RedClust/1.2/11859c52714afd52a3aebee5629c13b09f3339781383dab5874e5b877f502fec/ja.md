```
example_dataset(n::Int)
```

n$^{\mathrm{th}}$シミュレーション例からのデータセットを含む名前付きタプルを返します。このデータセットは、Julia v1.8.1で以下のコードを使用して生成されました：

```julia
	using RedClust
	using Random: seed!
	seed!(44)
	K = 10 # クラスタの数
	N = 100 # 点の数
	data_σ = [0.25, 0.2, 0.18][n] # 正規カーネルの分散
	data_dim = [10, 50, 10][n] # データ次元
	data = generatemixture(N, K; α = 10, σ = data_σ, dim = data_dim);
```

ただし、上記のコードは、Juliaの乱数生成器が異なるコンピュータ、異なるバージョンのJulia、または異なるバージョンのRandom.jlパッケージ間で再現可能な結果を意図していないため、あなたのコンピュータで異なる結果を生成する可能性があります。したがって、データセットはこのパッケージに含まれており、この関数を介してアクセスすることをお勧めします。

[`generatemixture`](@ref)も参照してください。
