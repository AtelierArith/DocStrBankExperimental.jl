```
getpointestimate(samples::MCMCResult; method = "MAP", loss = "VI")
```

クラスタ割り当てのサンプルベクトルから点推定を計算し、いくつかの損失関数の事後期待の最小化を探します。

# 引数

  * `method::String`: 次のいずれかでなければなりません -

```
- `"MAP"`: 事後確率の最大化。
- `"MLE"`: 最尤推定。
- `"MPEL"`: 最小事後期待損失。
`"MAP"` と `"MLE"` は、それぞれ最大の対数事後確率と対数尤度を持つクラスタリングをMCMCサンプルの中から探します。 `"MPEL"` は、`loss` 引数で指定された損失関数の事後期待損失を最小化するクラスタリングを探します。探索空間は `samples` のサンプルの集合です。
```

  * `loss`: `"MPEL"` メソッドで使用される損失関数を決定します。文字列または関数のいずれかでなければなりません。文字列として指定された場合、`"binder"` (バインダー損失)、`"omARI"` (調整済みランダムインデックスの1引き)、`"VI"` (情報の変動距離)、または`"ID"` (情報距離) のいずれかでなければなりません。関数として指定された場合、`(x::Vector{Int}, y::Vector{Int}) -> Real` のメソッドが定義されている必要があります。

# 戻り値

`clust` が `samples` のクラスタリング割り当てで、`i` がそのサンプルインデックスであるタプル `(clust, i)` を返します。
