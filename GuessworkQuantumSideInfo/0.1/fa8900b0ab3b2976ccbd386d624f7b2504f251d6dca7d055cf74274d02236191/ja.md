```
pmfN(data; tol = 1e-5) -> Vector
```

戦略に基づいて、推測の数 `N` に対する確率質量関数を計算します。出力ベクトルの `n` 番目のエントリは、`n` 回目の試行で正しい答えを推測する確率を示します。`n = 1 : K` の範囲です。許可される推測の数 `K` が `length(p)` より小さい場合、正しい答えを決して推測しない確率を示す追加の最後のエントリがあります。

  * `data` は、`p`、`ρBs`、`Es`、`K`、および `povm_outcomes` のエントリを持つ `NamedTuple` である必要があります。
  * `tol` は、虚数または負の確率について警告するための許容範囲を提供します。
