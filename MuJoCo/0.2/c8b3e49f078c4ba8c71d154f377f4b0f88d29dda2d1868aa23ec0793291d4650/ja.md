```
mj_addM(m, d, dst, rownnz, rowadr, colind)
```

慣性行列を宛先行列に追加します。宛先は、すべての int* が NULL の場合は疎圧縮されていないか、密であることができます。

# 引数

  * **`m::Model`** -> 定数。
  * **`d::Data`**
  * **`dst::Vector{Float64}`** -> 可変サイズのベクター。`dst` はベクターである必要があり、行列ではありません。`dst` のサイズは nM である必要があります。
  * **`rownnz::Vector{Int32}`** -> 可変サイズのベクター。`rownnz` はベクターである必要があり、行列ではありません。`rownnz` のサイズは nv である必要があります。
  * **`rowadr::Vector{Int32}`** -> 可変サイズのベクター。`rowadr` はベクターである必要があり、行列ではありません。`rowadr` のサイズは nv である必要があります。
  * **`colind::Vector{Int32}`** -> 可変サイズのベクター。`colind` はベクターである必要があり、行列ではありません。`colind` のサイズは nM である必要があります。
