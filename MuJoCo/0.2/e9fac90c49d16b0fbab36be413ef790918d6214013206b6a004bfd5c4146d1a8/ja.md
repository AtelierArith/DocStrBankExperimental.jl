```
mj_mulJacVec(m, d, res, vec)
```

密なまたは疎な制約ヤコビ行列をベクトルで乗算します。

# 引数

  * **`m::Model`** -> 定数。
  * **`d::Data`**
  * **`res::Vector{Float64}`** -> 可変サイズのベクトル。`res`はベクトルである必要があり、行列ではありません。`res`の長さはnefcである必要があります。
  * **`vec::Vector{Float64}`** -> 可変サイズのベクトル。`vec`はベクトルである必要があり、行列ではありません。`vec`の長さはnvである必要があります。定数。
