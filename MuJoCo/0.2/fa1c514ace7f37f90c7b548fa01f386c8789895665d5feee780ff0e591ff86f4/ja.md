```
mju_decodePyramid(force, pyramid, mu)
```

ピラミッド表現を接触力に変換します。

# 引数

  * **`force::Vector{Float64}`** -> 可変サイズのベクトル。`force` はベクトルである必要があり、行列ではありません。
  * **`pyramid::Vector{Float64}`** -> 可変サイズのベクトル。`pyramid` はベクトルである必要があり、行列ではありません。定数。
  * **`mu::Vector{Float64}`** -> 可変サイズのベクトル。`mu` はベクトルである必要があり、行列ではありません。定数。
