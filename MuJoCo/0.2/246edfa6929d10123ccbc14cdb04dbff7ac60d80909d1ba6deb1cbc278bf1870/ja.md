```
mj_mulM(m, d, res, vec)
```

ベクトルを慣性行列で乗算します。

# 引数

  * **`m::Model`** -> 定数。
  * **`d::Data`** -> 定数。
  * **`res::Vector{Float64}`** -> 可変サイズのベクトル。`res`はベクトルである必要があり、行列ではありません。`res`のサイズはnvである必要があります。
  * **`vec::Vector{Float64}`** -> 可変サイズのベクトル。`vec`はベクトルである必要があり、行列ではありません。`vec`のサイズはnvである必要があります。定数。
