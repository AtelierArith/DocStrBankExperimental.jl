```
mj_mulM2(m, d, res, vec)
```

ベクトルを（慣性行列）^(1/2)で乗算します。

# 引数

  * **`m::Model`** -> 定数。
  * **`d::Data`** -> 定数。
  * **`res::Vector{Float64}`** -> 可変サイズのベクトル。`res`はベクトルである必要があり、行列ではありません。`res`はサイズnvである必要があります。
  * **`vec::Vector{Float64}`** -> 可変サイズのベクトル。`vec`はベクトルである必要があり、行列ではありません。`vec`はサイズnvである必要があります。
