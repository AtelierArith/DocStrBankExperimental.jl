```
mju_addScl(res, vec1, vec2, scl)
```

res = vec1 + vec2*scl を設定します。

# 引数

  * **`res::Vector{Float64}`** -> 可変サイズのベクトル。`res` はベクトルである必要があり、行列ではありません。`res` と vec1 は同じサイズである必要があります。`res` と vec2 は同じサイズである必要があります。
  * **`vec1::Vector{Float64}`** -> 可変サイズのベクトル。`vec1` はベクトルである必要があり、行列ではありません。res と `vec1` は同じサイズである必要があります。定数です。
  * **`vec2::Vector{Float64}`** -> 可変サイズのベクトル。`vec2` はベクトルである必要があり、行列ではありません。res と `vec2` は同じサイズである必要があります。定数です。
  * **`scl::Float64`**
