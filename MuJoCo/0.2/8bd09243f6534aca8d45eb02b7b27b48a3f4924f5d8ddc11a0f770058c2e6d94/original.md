```
init_visualiser()
```

Loads the packages necessary for the running the visualiser.

Add the following packages to your project to be able to use the visualisation features, or run "install_visualiser()".

# Packages:

  * BangBang
  * FFMPEG
  * GLFW
  * Observables
  * PrettyTables
  * Printf
  * StaticArrays
