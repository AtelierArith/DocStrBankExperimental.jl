```
mju_mulVecMatVec(vec1, mat, vec2)
```

平方行列を両側のベクトルと掛け算します：vec1' * mat * vec2を返します。

# 引数

  * **`vec1::Vector{Float64}`** -> 可変サイズのベクトル。`vec1`はベクトルである必要があり、行列ではありません。定数です。
  * **`mat::Matrix{Float64}`** -> 可変サイズの行列。定数です。
  * **`vec2::Vector{Float64}`** -> 可変サイズのベクトル。`vec2`はベクトルである必要があり、行列ではありません。定数です。
