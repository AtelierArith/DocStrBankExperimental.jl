```
mj_differentiatePos(m, qvel, dt, qpos1, qpos2)
```

2つの位置を有限差分法で微分して速度を計算します。

# 引数

  * **`m::Model`** -> 定数。
  * **`qvel::Vector{Float64}`** -> 可変サイズのベクトル。`qvel`はベクトルである必要があり、行列ではありません。`qvel`のサイズはnvである必要があります。
  * **`dt::Float64`**
  * **`qpos1::Vector{Float64}`** -> 可変サイズのベクトル。`qpos1`はベクトルである必要があり、行列ではありません。`qpos1`のサイズはnqである必要があります。定数。
  * **`qpos2::Vector{Float64}`** -> 可変サイズのベクトル。`qpos2`はベクトルである必要があり、行列ではありません。`qpos2`のサイズはnqである必要があります。定数。
