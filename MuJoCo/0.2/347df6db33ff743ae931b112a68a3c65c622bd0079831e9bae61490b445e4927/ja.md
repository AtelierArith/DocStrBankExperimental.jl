```
mjd_subQuat(qa, qb, Da, Db)
```

mju_subQuatの導関数。

# 引数

  * **`qa::Vector{Float64}`** -> 可変サイズのベクトル。`qa`はベクトルである必要があり、行列ではありません。`qa`はサイズ4でなければなりません。定数。
  * **`qb::Vector{Float64}`** -> 可変サイズのベクトル。`qb`はベクトルである必要があり、行列ではありません。`qb`はサイズ4でなければなりません。定数。
  * **`Da::Matrix{Float64}`** -> **オプション**の可変サイズの行列。`Da`はサイズ9でなければなりません。必要ない場合は`nothing`に設定できます。
  * **`Db::Matrix{Float64}`** -> **オプション**の可変サイズの行列。`Db`はサイズ9でなければなりません。必要ない場合は`nothing`に設定できます。
