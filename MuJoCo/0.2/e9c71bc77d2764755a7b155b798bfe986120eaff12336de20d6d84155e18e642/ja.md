```
mj_zeros([element_type=mjtNum], dims...)
```

ゼロで満たされた配列を割り当て、基盤となるMuJoCo C APIと互換性があります。

C APIは配列を行優先として扱いますが、デフォルトではJuliaの配列は列優先です。この関数は行優先の方法でアクセスされる配列を作成しますが、あなたのJuliaコード内では通常の配列として扱うことができます。

# 引数

  * **element_type**: 配列の要素型。デフォルトはmjtNum（通常はFloat64）です。
  * **dims**: 出力配列の次元。整数のタプルまたは一連の整数である必要があります。
