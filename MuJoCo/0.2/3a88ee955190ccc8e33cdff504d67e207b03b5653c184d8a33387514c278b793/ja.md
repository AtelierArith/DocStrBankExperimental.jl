```
mju_encodePyramid(pyramid, force, mu)
```

接触力をピラミッド表現に変換します。

# 引数

  * **`pyramid::Vector{Float64}`** -> 可変サイズのベクトル。`pyramid` はベクトルである必要があり、行列ではありません。
  * **`force::Vector{Float64}`** -> 可変サイズのベクトル。`force` はベクトルである必要があり、行列ではありません。定数です。
  * **`mu::Vector{Float64}`** -> 可変サイズのベクトル。`mu` はベクトルである必要があり、行列ではありません。定数です。
