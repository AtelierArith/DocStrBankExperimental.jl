```
JUA_Guid
```

グローバルに一意の識別子を定義する可変構造体です。これは `UA_Guid` の同等物ですが、CではなくJuliaによってメモリが管理されます。

以下のコンストラクタメソッドが定義されています：

```
JUA_Guid()
```

空の `JUA_Guid` を作成します。これは `UA_Guid_new()` を呼び出すのと同等です。

```
JUA_Guid(guidstring::AbstractString)
```

文字列 `guidstring` を解析して `JUA_Guid` を作成します。文字列は、Part 6, 5.1.3 で定義されたOPC標準に従ってフォーマットされている必要があります。

```
JUA_Guid(ptr::Ptr{UA_Guid})
```

ポインタ `ptr` に基づいて `JUA_Guid` を作成します。これは、低レベルインターフェースを介して生成された `UA_Guid` を高レベル関数に渡すために使用できるフォールバックメソッドです。このメソッドを使用する際は、メモリ管理がC側に残ることに注意してください。つまり、オブジェクトがもはや必要ない場合は `UA_Guid_delete(ptr)` で `ptr` を手動でクリーンアップする必要があります。これを保証するのはユーザーの責任です。
