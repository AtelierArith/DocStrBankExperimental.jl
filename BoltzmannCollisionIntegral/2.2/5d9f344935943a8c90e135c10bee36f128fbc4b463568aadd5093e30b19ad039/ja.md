```
fload_All(fileLocation,fileName)
```

`fileLocation`に保存されている`fileName`に格納されたすべてのデータを読み込みます。

# 例

```julia-repl
    (Parameters,SAtot3,SAtot4,TAtot,SAtal3,SAtal4,TAtal,SMatrix3,SMatrix4,TMatrix1,TMatrix2,p3Max,p4Max,u3MinMax,u4MinMax,SConv3,SConv4,TConv) = fload_All(fileLocation,fileName);
```

ファイルに保存されたデータのタプルを返します。フィールドは次のとおりです：

  * `Parameters` : 評価に使用されたパラメータのタプル。
  * `Stot3` : 12->34相互作用のためにサンプリングされたすべての放出スペクトル値を合計した6D行列。
  * `Stot4` : 12->43相互作用のためにサンプリングされたすべての放出スペクトル値を合計した6D行列。
  * `Ttot` : サンプリングされたすべての吸収スペクトル値を合計した4D行列。
  * `Stal3` : 12->34相互作用のためにサンプリングされた放出スペクトル値の数を集計した5D行列。
  * `Stal4` : 12->43相互作用のためにサンプリングされた放出スペクトル値の数を集計した5D行列。
  * `Ttal` : サンプリングされた吸収スペクトル値の数を集計した4D行列。
  * `SMatrix3` : 12->34相互作用のための放出スペクトルの6D行列。
  * `SMatrix4` : 12->43相互作用のための放出スペクトルの6D行列。
  * `TMatrix1` : 12->34相互作用のための吸収スペクトルの4D行列。
  * `TMatrix2` : 21->34相互作用のための吸収スペクトルの4D行列、すなわちTMatrix1の置換と、種1が種2でない場合の位相空間因子の正しい適用によるもの。
  * `p3Max` : 各ビンのためにサンプリングされた運動量空間変数p3の最大値。（数値拡散を修正するのに便利）
  * `u3MinMax` : 各ビンのためにサンプリングされた運動量空間変数t3の最小値と最大値。（数値拡散を修正するのに便利）
  * `p4Max` : 各ビンのためにサンプリングされた運動量空間変数p4の最大値。（数値拡散を修正するのに便利）
  * `u4MinMax` : 各ビンのためにサンプリングされた運動量空間変数t4の最小値と最大値。（数値拡散を修正するのに便利）
  * `SConv3` : 12->34相互作用のために与えられた`Parameters`での前回の実行と比較した放出スペクトルの収束の6D行列。
  * `SConv4` : 12->43相互作用のために与えられた`Parameters`での前回の実行と比較した放出スペクトルの収束の6D行列。
  * `TConv` : 与えられた`Parameters`での前回の実行と比較した吸収スペクトルの収束の4D行列。
