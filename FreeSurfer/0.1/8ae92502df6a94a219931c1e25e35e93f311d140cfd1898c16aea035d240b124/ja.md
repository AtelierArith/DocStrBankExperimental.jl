```
get_tmp_path(tmpdir::String="")
```

一時ファイルを保存できるディレクトリへのパスを返します。

候補ディレクトリを以下の順序で検索します：

1. `$``TMPDIR`: 環境変数が定義されていて、ディレクトリが存在するか確認
2. `$``TEMPDIR`: 環境変数が定義されていて、ディレクトリが存在するか確認
3. `/scratch`: ディレクトリが存在するか確認
4. `/tmp`: ディレクトリが存在するか確認
5. `tmpdir`: `tmpdir` 引数が渡されていて、ディレクトリが存在するか確認

上記のいずれも存在しない場合は、現在のディレクトリ（`./`）を使用し、警告を表示します。
