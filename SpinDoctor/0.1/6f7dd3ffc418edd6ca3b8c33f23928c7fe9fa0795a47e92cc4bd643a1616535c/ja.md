```
create_geometry(setup; recreate = true)
```

セル、表面、および有限要素メッシュを作成します。`recreate = false`の場合、以前のジオメトリが再利用されます。

この関数は以下を行います：

  * ジオメトリ設定の整合性をチェック
  * セル構成を作成またはロード
  * 表面三角形分割を作成またはロード
  * TetGenを呼び出す
  * ドメインを変形
  * メッシュを区画に分割

複数の区画を持つカスタムジオメトリの場合は、代わりに`split_mesh`を直接呼び出してください。これにはファセットおよび要素ラベルが必要です。
