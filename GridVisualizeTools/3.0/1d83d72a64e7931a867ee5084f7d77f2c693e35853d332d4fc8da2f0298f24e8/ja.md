```julia
marching_triangles(
    coord,
    cellnodes,
    func,
    lines,
    levels;
    Tc,
    Tp,
    Tv
)

```

与えられたグリッドを通過し、指定された等高線レベルおよび/または指定された交差線のためのポイントと値を抽出します。返されたポイントリストと値リストからラインプロットを作成できます。

入力:     coord: グリッドの座標を格納する行列     cellnodes: 接続行列     func: 評価されるグリッドノード上の関数     lines: 線の定義のベクトル [a,b,c]、すなわち ax + by + c = 0 が線を定義します     levels: 等面のためのレベルのベクトル     Tc: 座標のスカラー型     Tp: 座標のベクトル型     Tv: 関数値のスカラー型

出力:     points: 等面または線との交差点の2Dポイントのベクトル     adjacencies: グリッド内の接続されたポイントを格納する2Dベクトルのベクトル     value: 交差点での `func` の補間値

非空の `lines` と `levels` の両方を渡すと、両方のタイプのポイントが混在した結果が生成されることに注意してください。
