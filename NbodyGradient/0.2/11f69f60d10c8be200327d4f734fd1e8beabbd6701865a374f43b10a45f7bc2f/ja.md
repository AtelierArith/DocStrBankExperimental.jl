Elements{T<:AbstractFloat} <: AbstractInitialConditions

バイナリの軌道要素と「外部」天体の質量。単位と慣習については[チュートリアル](@ref)を参照してください。

# フィールド

  * `m::T` : 外部天体の質量。
  * `P::T` : 周期 [日]。
  * `t0::T` : 初期通過時間 [日]。
  * `ecosω::T` : 離心率ベクトルのx成分（離心率と近日点引数のコサインの積）
  * `esinω::T` : 離心率ベクトルのy成分（離心率と近日点引数のサインの積）
  * `I::T` : 空の平面から測定された傾斜 [ラジアン]。
  * `Ω::T` : +x軸から測定された昇交点の経度 [ラジアン]。
  * `a::T` : 軌道の半長軸 [AU]。
  * `e::T` : 離心率。
  * `ω::T` : 近日点引数 [ラジアン]。
  * `tp::T` : 近日点通過の時間 [日]。
