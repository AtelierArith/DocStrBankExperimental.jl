フルスクリーンの紙吹雪アニメーションを表示します！紙吹雪は自動的に消え、セルが再実行されるたびに発射されます。

# 例

正しい答えを祝うためにこれをトリガーできます：

```julia
# xを非常に大きな数にする
x = 5
```

次のセルは非表示になります：

```julia
if x > 1000000
	confetti()
end
```

`confetti()`は「紙吹雪発射装置を返す」ことに注意してください。これはセルの出力値である必要があります。したがって、これは機能します：

```julia
let
	x = 1
	y = x + 2
	if y == 3
		confetti()
	end
end
```

しかし、これは機能しません：

```julia
let
	x = 1
	y = x + 2
	if y == 3
		confetti()
	end
	y
end
```
