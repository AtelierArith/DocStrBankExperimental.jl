# コンストラクタ

```julia
Label(;content=missing, captionstyle=CaptionStyle())
```

# キーワード引数

  * `content::Union{String, Missing}` はラベルの内容です。
  * `captionstyle::Union{CaptionStyle, Missing}` はラベルが印刷されるフォントのスタイルです。
