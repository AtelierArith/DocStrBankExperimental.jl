```
GaussLegendre{C, N, W}
```

積分を（複合）ガウス・レジェンドルの数値積分法で評価するための構造体です。フィールド `C` は `subintervals > 1` の場合は `true` になり、それ以外の場合は `false` になります。

フィールド `nodes::N` と `weights::W` は、与えられたノード数 `n` に対して `nodes, weights = gausslegendre(n)` によって定義されます。

フィールド `subintervals::Int64 = 1`（デフォルト値は `1`）は、元の積分区間 `[a, b]` を分割する区間の数を定義し、`[xⱼ, xⱼ₊₁]` に分割します。ここで、`j = 1,…,subintervals` であり、`xⱼ = a + (j-1)h` で、`h = (b-a)/subintervals` です。ガウス・レジェンドルの数値積分法は各サブ区間に適用されます。例えば、もし `[a, b] = [-1, 1]` で `subintervals = 2` の場合、ガウス・レジェンドルの数値積分法は `[-1, 0]` と `[0, 1]` にそれぞれ適用され、2つの結果が合計されます。
