```
VEGASMC(; kws...)
```

Markov-chain based Vegas algorithm from MCIntegration.jl

Refer to [`MCIntegration.integrate`](https://numericaleft.github.io/MCIntegration.jl/dev/lib/montecarlo/#MCIntegration.integrate-Tuple%7BFunction%7D) for documentation on the keywords, which are passed directly to the solver with a set of defaults that works for conforming integrands.
