```
generate_N_equally_spaced_points(num_points::Int, e::Ellipse; start_point_shift::Float64=rand())
```

`num_points` 個の等間隔の点を、`e` に含まれるパラメータで定義された楕円上に生成します。点は、2 行 `num_points` 列の配列として返され、各点は列に格納されます。

# 引数

  * `num_points`: 楕円上に等間隔で生成する点の正の整数の数。
  * `e`: 楕円を定義する有効な [`EllipseSampling.Ellipse`](@ref) 構造体。

# キーワード引数

  * `start_point_shift`: 数値 ∈ [0.0,1.0]。デフォルトは `rand()`（[0.0,1.0] で定義）であり、デフォルトではこの関数が呼ばれるたびに異なる点のセットが生成されます。

# 詳細

点は、回転前の主軸上の最も正の頂点から始まり、楕円の周りを反時計回りにサンプリングされます。回転が適用された後、この頂点は主軸上の最も正の頂点でなくなる可能性があります。ここでの頂点とは、主軸上にあり、楕円と交差する2つの端点を指します。`start_point_shift=0.0` の場合、等間隔の点の最初の点は、回転前の主軸上の最も正の頂点に配置されます。

楕円上の点の等間隔は、弧の長さに関して行われます。楕円上に配置される最初の点の位置は、[0.0,1.0] で定義された `start_point_shift` によってシフトされ、与えられた `num_points` に対して生成される可能性のある点のセットが全周をカバーできるようになります。このシフトは正規化されており、`start_point_shift=1.0` の場合、楕円上に配置される最初の点の位置は、`start_point_shift=0.0` の場合に楕円上に配置される2番目の点の位置と等しくなります。
