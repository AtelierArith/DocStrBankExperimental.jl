```
RandomCropNormalize([; output_size, open_size, mean, std])
```

前処理パイプラインは、入力画像をランダムな位置で `output_size` にクロップし、`mean` と `std` に従って正規化します。WHCフォーマット `(幅、高さ、チャンネル)` の配列を返します。

[`transform`](@ref) と [`inverse_transform`](@ref) を使用して適用されます。

## キーワード引数:

  * `output_size`: 中心クロップの出力サイズ `(幅、高さ)`。デフォルトは `(224, 224)` です。
  * `open_size`: JpegTurboを使用して画像を開くための推奨サイズ `(幅、高さ)`。デフォルトは `(256, 256)` です。
  * `mean`: カラーチャンネルに対する正規化の平均。デフォルトは `(0.485f0, 0.456f0, 0.406f0)` です。
  * `std`: カラーチャンネルに対する正規化の標準偏差。デフォルトは `(0.229f0, 0.224f0, 0.225f0)` です。
