```
named_ss(sys::AbstractStateSpace{T}; x, u, y)
```

`NamedStateSpace` システムを作成します。この種のシステムは、状態、入力、および出力を参照するために整数インデックスではなく名前を使用します。

  * 単一の名前が提供されるが、名前のベクトルが期待される場合、この名前は接頭辞として使用され、その後に数値インデックスが続きます。
  * 名前が提供されない場合、デフォルトの名前（`x,y,u`）が使用されます。

# 引数:

  * `sys`: 名前を追加するシステム。
  * `x`: 状態の名前を持つシンボルのリスト。
  * `u`: 入力の名前を持つシンボルのリスト。
  * `y`: 出力の名前を持つシンボルのリスト。

# 例

```julia
G1 = ss(1,1,1,0)
G2 = ss(1,1,1,0)
s1 = named_ss(G1, x = :x, u = :u1, y=:y1)
s2 = named_ss(G2, x = :z, u = :u2, y=:y2)

s1[:y1, :u1] # シンボルを使用してインデックス。正確な一致が見つからない場合は接頭辞マッチングを使用します。

fb = feedback(s1, s2, r = :r) # 
```
