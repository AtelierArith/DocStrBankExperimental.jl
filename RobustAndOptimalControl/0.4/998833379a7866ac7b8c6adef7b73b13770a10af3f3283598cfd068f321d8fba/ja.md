```
K, Cl = h2synthesize(P::ExtendedStateSpace, γ = nothing)
```

H₂最適コントローラKを合成し、`w`から`z`への閉ループ伝達関数を計算します。参考文献: Robust and Optimal Controlの第14.5章。

`γ = nothing`の場合、14.5章のH₂の公式を使用します。γが大きな値の場合、H∞の公式が使用されます。γ → ∞のとき、これら二つは同等です。H∞の公式は、やや一般的なシステムを扱う座標変換を行うため、エラーが発生した場合は、γを大きな値（例: 1000）に設定してみる価値があります。
