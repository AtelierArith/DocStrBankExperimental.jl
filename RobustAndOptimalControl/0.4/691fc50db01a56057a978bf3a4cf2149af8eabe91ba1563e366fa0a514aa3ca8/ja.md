```
sysm, T, E = modal_form(sys; C1 = false)
```

`sys`をモーダル形式に変換します。

モーダル形式は、主対角線に`A`の固有値の実数値があり、最初の下対角線と上対角線に複素部分がある三重対角行列であることが特徴です。`T`は、次のようにシステムに適用される類似変換です。

```julia
sysm ≈ similarity_transform(sys, T)
```

`C1`が`true`の場合、SISOシステムに対して追加の規約が使用され、実数の固有値の`C`行列係数は1になります。`C1 = false`の場合、`B`および`C`係数はバランスの取れた方法で選択されます。

`E`は`A`の固有因子分解です。

モーダル形式は、どのモードがどの入力から制御可能で、どの出力から観測可能であるかを明示にします。非最小実現は特異性例外を引き起こす可能性があります。

[`hess_form`](@ref)および[`schur_form`](@ref)も参照してください。
