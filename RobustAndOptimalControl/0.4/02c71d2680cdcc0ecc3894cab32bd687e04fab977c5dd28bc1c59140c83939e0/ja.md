```
feedback(s1::NamedStateSpace, s2::NamedStateSpace;
        w1 = [],  u1 = (:), z1 = (:), y1 = (:),
        w2 = (:), u2 = (:), z2 = (:), y2 = [], kwargs...)
```

2つの名前付きシステム間のフィードバック。接続する信号のリストは、信号名のシンボルのリストであることが期待され、`(:)`はすべての信号を示し、`[]`は信号がないことを示します。

すべての信号セット `w1,u1,z1,y1,u2,y2,w2,z2` は、通常のシステムの高度なフィードバックメソッドと同じ意味を持ちます。追加された信号セット `r` は、フィードバックループの入力に新しい名前をオプションで提供するために使用されます。

複雑なフィードバック相互接続を簡素化するには、`connect`を参照してください。

すべての入力と出力が接続されていない場合、返されるシステムは最小実現ではない可能性があります。入力-出力マップのみが関心のある場合は、システムを簡素化するために `sminreal`（場合によっては `minreal` も）を使用してください。
