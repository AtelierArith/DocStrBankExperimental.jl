```
nu_reduction(G, g=0.1; gap = nugap(G))
```

不確実なシステム `G` の粒子数を、名目システム `Gₙ` の νgap `g` 内にあるすべての粒子を削除することによって削減します。

注意: `G` に確率的解釈がある場合、すなわち係数が何らかの分布から来ている場合、この解釈は削減後に失われ、平均値と標準偏差は保持されません。削減されたシステムは、最悪のケースの不確実性を保持するものとして解釈されるべきです。

もし `gap = nugap(G)` がすでに事前に計算されている場合、再計算のコストを避けるために引数として供給することができます。
