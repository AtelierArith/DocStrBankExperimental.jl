# フィールド:

`α`: ディスクマージン `ω0`: 最悪ケース周波数 `f0`: 不安定化する摂動 `f0` は、同時にゲインと位相の変動を持つ複素数です。この重要な摂動は、臨界周波数 ω0 で虚軸上に閉ループ極を持つ不安定性を引き起こします。 `δ0`: f0 を生成する不確実な要素。 `γmin`: ディスクの下側実軸切片（古典的な下側ゲインマージンに類似）。 `γmax`: ディスクの上側実軸切片（古典的な上側ゲインマージンに類似）。 `ϕm`: 古典的な位相マージンに類似。 `σ`: マージンを計算するために使用されたスキューパラメータ。

注意、`γmax` と `ϕm` は、古典的なゲインと位相マージンよりも小さいです。なぜなら、古典的なマージンはゲインと位相の同時摂動を考慮しないからです。

「ディスク」マージンは、`α = 2` の場合は半平面になり、`α > 2` の場合は逆円になります。この場合、上側ゲインマージンは無限大です。詳細については論文を参照してください。特に図6を見てください。
