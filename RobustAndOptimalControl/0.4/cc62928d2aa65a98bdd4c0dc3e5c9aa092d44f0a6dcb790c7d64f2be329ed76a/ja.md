```
nu_reduction_recursive(G, g = 0.1; gap = nugap(G), keepinds = Set{Int}(1), verbose = false)
```

半径 `g` のボールの νgap カバーを見つけます（νgap メトリックにおいて）すべての実現を `G` に含むものです。

もし `gap = nugap(G)` がすでに事前に計算されている場合、再計算のコストを避けるために引数として供給できます。手動で計算された `gap` を供給する場合、`keepinds=Set{Int}(index)` も供給する必要があります。ここで `index` は `gap` を計算するために使用された `G` 内の名目システムのインデックスです。

返されるカバー `Gr` は `G` と同じタイプですが、粒子の数が少なくなっています。`Gr` のために設計されたコントローラーは、`Gr` 内のすべての実現に対して少なくとも `g` の [`ncfmargin`](@ref) を達成することで、元の `G` 内のすべての実現を安定化させます。`Gr = Gnominal` という極端なケースのカバーは、`g = nugap(G, i)` を呼び出すことで計算できます。ここで `i` は `G` 内の名目システムのインデックスです。

# 引数:

  * `G`: `StateSpace{TE, Particles}` の形をした不確実なモデル（マルチモデル）。
  * `g`: νgap カバー内のボールの半径。
  * `gap`: オプションの事前計算されたギャップ
  * `verbose`: 進行状況を表示
