```
loop_diskmargin(P, C, args...; kwargs...)
```

Calculate the loop-at-a-time diskmargin for each output and input of `P`. See also [`diskmargin`](@ref), [`sim_diskmargin`](@ref). Ref: ["An Introduction to Disk Margins", Peter Seiler, Andrew Packard, and Pascal Gahinet](https://arxiv.org/abs/2003.04771)
