```
nugap(G; map = map)
```

名目システム `Gₙ` と `G` の最初の粒子インデックスで表されるすべての他のシステムとの間の νgap を計算します。`G` の各システムに対する νgap を持つ `Particles` オブジェクトを返します。

不確実な値に名目値を付与するには `with_nominal` を、名目値を抽出するには `nominal` を参照してください。

この関数によって返される値 `νᵧ` は、名目システム `Gₙ` のためのコントローラーを設計することでロバスト合成に役立ちます。これは、[`ncfmargin`](@ref) が少なくとも `νᵧ` であることを達成することが保証されており、`G` 内のすべての実現を安定化させます。

大規模システムの計算を高速化するために、スレッド化または分散された `map` 関数を提供することができます。例えば、`ThreadTools.tmap` または `Distributed.pmap` です。
