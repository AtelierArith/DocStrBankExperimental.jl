```
l, res = find_lft(sys::StateSpace{<:Any, <:StaticParticles{<:Any, N}}, δ) where N
```

注意: この関数は実験的です。

不確実な係数を持つシステム `sys` が `StaticParticles` の形で与えられた場合、`lft(M, δ) ≈ sys` となるような下線形分数変換 `M` を検索します。

`δ` は、`sys` の不確実性の源、すなわち `sys` を作成するために使用されたユニークな不確実なパラメータのベクトルである可能性があります。これらは、最もロバスト制御理論が適用可能であるために、均一にランダムに分布した粒子として構築されるべきです。`δ` は整数でもあり、その場合は `δ` の不確実性の源が自動的に作成されます。これは、`sys` の不確実性の源の数が多い場合に次数削減に使用できます。

注意: `sys` の不確実性は `A` と `B` のみサポートされており、`C` と `D` は決定論的でなければなりません。

`l::LFT` を返し、内部に `M` の4つのブロックと `δ` を含みます。`ss(l,sys)` を呼び出して `lft(M, δ) ≈ sys` を取得します。

`Matrix(l)` を呼び出して `M = [M11 M12; M21 M22]` を取得します。
