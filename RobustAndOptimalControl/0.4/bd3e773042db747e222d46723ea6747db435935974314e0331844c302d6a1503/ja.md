```
μ = structured_singular_value(M; tol=1e-4, scalings=false, dynamic=false)
```

複素摂動の対角Δに対する構造的特異値μ（他の構造のΔはまだサポートされていません）を（上限を）計算します。`M`は（n × n × N_freq）配列または行列であると仮定します。

現在、下限を計算する方法はありませんが、すべての摂動が複素数である場合、スペクトル半径`ρ(M)`は常に下限となります（通常は良いものではありません）。

`scalings = true`の場合、対角スケーリング`D`を持つ`n × nf`行列`Dm`も返されます。これは次のようにして最小化されます。

```
D = Diagonal(Dm[:, i])
σ̄(D\M[:,:,i]*D)
```

`dynamic = true`の場合、摂動は時間変化する`Δ(t)`であると仮定されます。この場合、すべての周波数に対して同じスケーリングが使用され、`scalings=true`の場合に返される`D`は、`D = Diagonal(d)`となるベクトル`d`です。 ```
