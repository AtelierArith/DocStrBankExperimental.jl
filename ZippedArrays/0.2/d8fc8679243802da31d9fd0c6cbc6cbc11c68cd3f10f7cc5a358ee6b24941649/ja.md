```
Z = ZippedArray(A,B,C,...)
```

は、配列 `A`、`B`、`C` などに基づいて、ジップされた配列 `Z` を構築します。ここで、構文 `Z[i]` は値のタプル `(A[i],B[i],C[i],...)` を返し、構文 `Z[i] = (a,b,c,...)` は `(A[i],B[i],C[i],...) = (a,b,c,...)` と同等です。

任意の数の配列をジップすることができますが、`axes` メソッドを呼び出すことによって与えられる同じインデックスを持っている必要があります。

構文 `Z.args` を使用して、配列 `A`、`B`、`C` などを取得します。
