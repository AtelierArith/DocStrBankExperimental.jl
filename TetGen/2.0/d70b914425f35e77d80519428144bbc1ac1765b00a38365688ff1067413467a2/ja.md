```julia
tetrahedralize(
    input::TetGen.RawTetGenIO{Float64},
    flags::String
) -> TetGen.RawTetGenIO{Float64}

```

入力を四面体化します。

```
  flags: -pYrq_Aa_miO_S_T_XMwcdzfenvgkJBNEFICQVh 
    -p  ピースワイズリニア複合体（PLC）を四面体化します。
    -Y  入力表面メッシュを保持します（変更しません）。
    -r  以前に生成されたメッシュを再構築します。
    -q  メッシュを洗練します（メッシュの品質を向上させるため）。
    -R  メッシュの粗さを調整します（メッシュ要素を減らすため）。
    -A  異なる領域の四面体に属性を割り当てます。
    -a  最大四面体体積制約を適用します。
    -m  メッシュサイズ関数を適用します。
    -i  追加のポイントのリストを挿入します。
    -O  メッシュ最適化のレベルを指定します。
    -S  追加ポイントの最大数を指定します。
    -T  共面テストの許容誤差を設定します（デフォルト1e-8）。
    -X  正確な算術の使用を抑制します。
    -M  共面ファセットまたは非常に近い頂点のマージを行いません。
    -w  重み付きデローニ三角形分割を生成します。
    -c  PLCの凸包を保持します。
    -d  PLCのファセットの自己交差を検出します。
    -z  出力項目をゼロから番号付けします。
    -f  すべての面を .face ファイルに出力します。
    -e  すべてのエッジを .edge ファイルに出力します。
    -n  四面体の隣接情報を .neigh ファイルに出力します。
    -v  ボロノイ図をファイルに出力します。
    -g  Meditで表示するためにメッシュを .mesh ファイルに出力します。
    -k  Paraviewで表示するためにメッシュを .vtk ファイルに出力します。
    -J  出力 .node ファイルから未使用の頂点を廃棄しません。
    -B  境界情報の出力を抑制します。
    -N  .node ファイルの出力を抑制します。
    -E  .ele ファイルの出力を抑制します。
    -F  .face および .edge ファイルの出力を抑制します。
    -I  メッシュの反復番号の出力を抑制します。
    -C  最終メッシュの整合性をチェックします。
    -Q  静かに: エラーを除くターミナル出力なし。
    -V  詳細: 詳細情報、より多くのターミナル出力。
    -h  ヘルプ: TetGenの使用に関する簡単な説明。
```
