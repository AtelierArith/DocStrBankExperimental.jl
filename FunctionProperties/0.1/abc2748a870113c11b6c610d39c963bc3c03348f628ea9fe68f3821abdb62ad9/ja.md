```
hasbranching(f, x...)
```

関数 `f` が、ADトレースを行うパッケージ（ReverseDiff.jl など）によるトレースシステムで取得される値 x に依存する分岐（if文）を持っているかどうかをチェックします。

## 引数:

```
* `f`: 検査する関数
* `x`: 検査のためのテスト引数。これらの値は、関数への実際の呼び出しで使用される値である必要はなく、代わりに実際の関数呼び出しで使用される型に一致するプロトタイプ値である必要があります。これは、正しい内部ディスパッチをトレースするために使用されます。
```

## 出力:

```
関数が分岐を持っているかどうかのブール値。
```

## チェックからのディスパッチのカスタマイズと削除

パッケージの一部の内部関数は、分岐がコンパイル時に解決されることが知られているため、誤検出を引き起こす可能性があります。これが知られている場合、その関数を分析から除外するためにディスパッチを追加できます：

```julia
function FunctionProperties.Cassette.overdub(::FunctionProperties.HasBranchingCtx, ::typeof(f), x...) 
    f(x...)
end
```
