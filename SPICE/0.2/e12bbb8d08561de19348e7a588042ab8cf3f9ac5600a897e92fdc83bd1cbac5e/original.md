```
spkssb(targ, et, ref)
```

Return the state (position and velocity) of a target body relative to the solar system barycenter.

### Arguments

  * `targ`: Target body
  * `et`: Target epoch
  * `ref`: Target reference frame

### Output

Returns the state of target.
