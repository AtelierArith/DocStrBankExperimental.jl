```
ekaclc(handle, segno, column, cvals, nlflgs, rcptrs)
```

EKセグメントに全体の文字列列を追加します。

### 引数

  * `handle`: EKファイルハンドル。
  * `segno`: 列を追加するセグメントの番号。
  * `column`: 列名。
  * `cvals`: 列に追加する文字列値。
  * `nlflgs`: 列エントリのヌルフラグの配列。
  * `rcptrs`: セグメントのレコードポインタ。
