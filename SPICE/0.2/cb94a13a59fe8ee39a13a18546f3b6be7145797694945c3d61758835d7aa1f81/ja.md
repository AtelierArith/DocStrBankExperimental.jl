```
matchw(string, templ, wstr, wchar)
```

ワイルドカードを含むテンプレートに文字列が一致するかどうかを判断します。

### 引数

  * `string`: テストする文字列
  * `templ`: 文字列に対してテストするテンプレート（ワイルドカード付き）
  * `wstr`: ワイルド文字列トークン
  * `wchr`: ワイルドキャラクタートークン

### 出力

文字列が一致する場合は `true` を返します。
