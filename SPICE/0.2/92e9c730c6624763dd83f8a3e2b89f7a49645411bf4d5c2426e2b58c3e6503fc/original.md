```
spkgps(targ, et, ref, obs)
```

Compute the geometric position of a target body relative to an observing body.

### Arguments

  * `targ`: Target body
  * `et`: Target epoch
  * `ref`: Target reference frame
  * `obs`: Observing body

### Output

  * `pos`: Position of target
  * `lt`: Light time
