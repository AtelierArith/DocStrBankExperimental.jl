```
pl2nvc(plane)
```

Return a unit normal vector and constant that define a specified plane.

### Arguments

  * `plane`: A plane

### Output

Returns a tuple consisting of

  * `normal`: A normal vector and...
  * `constant`: ... constant defining the geometric plane represented by `plane`
