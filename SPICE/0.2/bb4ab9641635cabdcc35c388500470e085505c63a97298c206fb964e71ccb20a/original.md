```
cgv2el(center, vec1, vec2)
```

Form an ellipse from a center vector and two generating vectors.

### Arguments

  * `center`: Center vector
  * `vec1`: Generating vector
  * `vec2`: Generating vector

### Output

Returns the ellipse defined by the input vectors.
