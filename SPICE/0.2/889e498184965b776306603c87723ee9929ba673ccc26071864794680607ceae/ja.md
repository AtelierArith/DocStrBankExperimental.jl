```
lxqstr(string, qchar, first)
```

引用された文字列をスキャンします。

### 引数

  * `string`: スキャンする文字列
  * `qchar`: 引用区切り文字
  * `first`: スキャンを開始する文字位置

### 出力

  * `last`: トークンの終わりの文字位置
  * `nchar`: トークン内の文字数
