```
pltnrm(v1, v2, v3)
```

Compute an outward normal vector of a triangular plate.  The vector does not necessarily have unit length.

### Arguments

  * `v1`, `v2`, `v3`: Vertices of a plate

### Output

Returns the plate's outward normal vector.
