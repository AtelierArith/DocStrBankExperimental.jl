```
dskn02(handle, dladsc, plid)
```

Compute the unit normal vector for a specified plate from a type 2 DSK segment.

### Arguments

  * `handle`: DSK file handle
  * `dladsc`: DLA descriptor
  * `plid`: Plate ID

### Output

Return the plate's unit normal vector.
