```
inelpl(ellips, plane)
```

Find the intersection of an ellipse and a plane.

### Arguments

  * `ellips`: An ellipse
  * `plane`: A plane

### Output

  * `nxpts`: Number of intersection points of ellipse and plane
  * `xpt1`, `xpt2`: Intersection points
