```
set(a::T, b::T) where T <: SpiceCell
```

Given a relational operator, compare two sets of any data type.

### Arguments

  * `a`: First set
  * `op`: Comparison operator
  * `b`: Second set

### Output

Returns the result of the comparison: `a (op) b`.
