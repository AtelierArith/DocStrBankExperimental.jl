```
deltet(epoch, eptype)
```

Return the value of ΔET (ET-UTC) for an input epoch.

### Arguments

  * `epoch`: Input epoch (seconds past J2000)
  * `eptype`: Type of input epoch ("UTC" or "ET")

### Output

Returns ΔET (ET-UTC) at input epoch.
