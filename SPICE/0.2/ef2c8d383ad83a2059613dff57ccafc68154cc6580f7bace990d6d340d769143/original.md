```
spkobj!(ids, spk)
```

Find the set of ID codes of all objects in a specified SPK file.

### Arguments

  * `ids`: A preallocated set of ID codes of objects in SPK file
  * `spk`: Name of the SPK file

### Output

Returns the set of id codes.
