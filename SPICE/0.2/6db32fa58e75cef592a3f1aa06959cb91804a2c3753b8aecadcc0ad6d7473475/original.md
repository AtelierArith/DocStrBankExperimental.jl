```
pjelpl(elin, plane)
```

Project an ellipse onto a plane, orthogonally.

### Arguments

  * `elin`: An ellipse to be projected
  * `plane`: A plane onto which `elin` is to be projected

### Output

Returns the ellipse resulting from the projection.
