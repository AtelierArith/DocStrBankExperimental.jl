```
cpos(str, chars, start)
```

文字列内の指定された位置から前方に検索し、文字のコレクションに属する文字の最初の出現を見つけます。

### 引数

  * `str`: 任意の文字列
  * `chars`: 文字のコレクション
  * `start`: chars のいずれかを探し始める位置

### 出力

`chars` の文字のいずれかである `str` の最初の文字のインデックスを返します。文字が見つからなかった場合は -1 を返します。
