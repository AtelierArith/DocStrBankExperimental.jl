```
dskp02(handle, dladsc, start, room)
```

Fetch triangular plates from a type 2 DSK segment.

### Arguments

  * `handle`: DSK file handle
  * `dladsc`: DLA descriptor
  * `start`: Start index
  * `room`: Amount of room in output array

### Output

Returns an array of plates.
