```
ckopn(fname, ifname="CK_file", ncomch=0)
```

新しいCKファイルを開き、開かれたファイルのハンドルを返します。

### 引数

  * `fname`: 開くCKファイルの名前
  * `ifname`: CKの内部ファイル名（デフォルト: "`CK_file`"）
  * `ncomch`: コメントのために予約する文字数（デフォルト: 0）

### 出力

  * `handle`: 開かれたCKファイルのハンドル
