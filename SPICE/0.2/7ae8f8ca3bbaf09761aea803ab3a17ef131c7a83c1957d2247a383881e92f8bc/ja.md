```
eul2xf(eulang, axisa, axisb, axisc)
```

オイラー角の因子分解とそれらのオイラー角の導関数から状態変換を計算します。

### 引数

  * `eulang`: オイラー角とその導関数の配列
  * `axisa`: オイラー角の因子分解の軸 A
  * `axisb`: オイラー角の因子分解の軸 B
  * `axisc`: オイラー角の因子分解の軸 C

### 出力

状態変換行列を返します。
