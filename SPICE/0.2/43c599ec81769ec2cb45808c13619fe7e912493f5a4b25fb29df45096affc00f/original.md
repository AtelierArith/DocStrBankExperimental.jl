```
diags2(symmat)
```

Diagonalize a symmetric 2×2 matrix.

### Arguments

  * `symmat`: A symmetric 2×2 matrix

### Output

  * `diag`: A diagonal matrix similar to `symmat`
  * `rotate`: A rotation used as the similarity transformation
