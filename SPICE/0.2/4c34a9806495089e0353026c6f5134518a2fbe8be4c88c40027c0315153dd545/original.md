```
ekbseg(handle, tabnam, cnames, decls)
```

Start a new segment in an E-kernel.

### Arguments

  * `handle`: File handle
  * `tabnam`: Table name
  * `cnames`: Names of columns
  * `decls`: Declarations of columns

### Output

Returns the segment number.
