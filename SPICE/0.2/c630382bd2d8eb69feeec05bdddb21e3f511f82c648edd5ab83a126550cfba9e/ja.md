```
wnincd(window, left, right)
```

ダブル精度ウィンドウに区間が含まれているかどうかを判断します。

### 引数

  * `window`: 入力ウィンドウ
  * `left`: 入力区間の左端点
  * `right`: 入力区間の右端点

### 出力

`(left, right)` が `window` に含まれている場合は `true` を返します。
