```
dafus(sum, nd, ni)
```

Unpack an array summary into its double precision and integer components.

### Arguments

  * `sum`: Array summary
  * `nd`: Number of double precision components
  * `ni`: Number of integer components

### Output

  * `dc`: Double precision components
  * `ic`: Integer components
