```
spkgeo(targ, et, ref, obs)
```

Compute the geometric state (position and velocity) of a target body relative to an observing body.

### Arguments

  * `targ`: Target body.
  * `et`: Target epoch.
  * `ref`: Target reference frame.
  * `obs`: Observing body.

### Output

  * `state`: State of target.
  * `lt`: Light time.
