```
utc2et(utcstr)
```

Convert an input time from Calendar or Julian Date format, UTC, to ephemeris seconds past J2000.

### Arguments

  * `utcstr`: Input time string, UTC

### Output

Returns the equivalent of utcstr, expressed in ephemeris seconds past J2000.
