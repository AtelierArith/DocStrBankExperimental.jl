```
dskz02(handle, dladsc)
```

Return plate model size parameters - plate count and vertex count - for a type 2 DSK segment.

### Arguments

  * `handle`: DSK file handle
  * `dladsc`: DLA descriptor

### Output

  * `nv`: Number of vertices
  * `np`: Number of plates
