```
wnreld(a, op, b)
```

Compare two double precision windows.

!!! note
    Consider using overloaded operators instead, i.e. `a == b`, `a ⊆ b`, and `a ⊊ b`.


### Arguments

  * `a`: First window
  * `op`: Comparison operator
  * `b`: Second window

### Output

Returns the result of comparison `a (op) b`.
