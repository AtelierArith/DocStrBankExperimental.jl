```
wnreld(a, op, b)
```

2つの倍精度ウィンドウを比較します。

!!! note
    代わりにオーバーロードされた演算子を使用することを検討してください。すなわち、`a == b`、`a ⊆ b`、および `a ⊊ b`。


### 引数

  * `a`: 最初のウィンドウ
  * `op`: 比較演算子
  * `b`: 2番目のウィンドウ

### 出力

比較 `a (op) b` の結果を返します。
