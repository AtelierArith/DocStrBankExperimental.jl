```
vproj(a, b)
```

Finds the projection of one vector onto another vector. All vectors are 3-dimensional.

### Arguments

  * `a`: The vector to be projected
  * `b`: The vector onto which `a` is to be projected

### Output

Returns the projection of `a` onto `b`.
