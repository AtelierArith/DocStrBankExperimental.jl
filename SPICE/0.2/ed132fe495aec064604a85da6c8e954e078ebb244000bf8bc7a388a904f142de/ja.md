```
pckfrm!(ids, pck)
```

指定されたバイナリPCKファイル内のすべてのフレームの参照フレームクラスIDコードのセットを見つけます。

### 引数

  * `ids`: 初期化された `SpiceIntCell`
  * `pck`: PCKファイルのパス

### 出力

PCKファイル内のフレームのフレームクラスIDコードのセットを含む `ids` を返します。
