```
eul2m(angle3, angle2, angle1, axis3, axis2, axis1)
```

オイラー角のセットから回転行列を構築します。

### 引数

  * `angle3`, `angle2`, `angle1`: 第三、第ニ、第一回転軸に関する回転角（ラジアン）
  * `axis3`, `axis2`, `axis1`: 第三、第ニ、第一回転軸の軸番号

### 出力

3つの回転の積に対応する回転行列。
