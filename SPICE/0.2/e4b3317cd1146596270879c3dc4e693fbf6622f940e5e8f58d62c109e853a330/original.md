```
scpart(sc)
```

Get spacecraft clock partition information from a spacecraft clock kernel file.

### Arguments

  * `sc`: NAIF spacecraft identification code

### Output

  * `pstart`: Array of partition start times
  * `pstop`: Array of partition stop times
