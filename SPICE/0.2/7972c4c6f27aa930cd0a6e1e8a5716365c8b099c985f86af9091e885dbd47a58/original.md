```
rotate(angle, iaxis)
```

Calculate the 3Ã—3 rotation matrix generated by a rotation of a specified angle about a specified axis. This rotation is thought of as rotating the coordinate system.

### Arguments

  * `angle`: Angle of rotation (radians)
  * `iaxis`: Axis of rotation (X=1, Y=2, Z=3)

### Output

Returns rotation matrix associated with `angle` and `iaxis`.
