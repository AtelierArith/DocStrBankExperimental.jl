```
pl2nvp(plane)
```

Return a unit normal vector and point that define a specified plane.

### Arguments

  * `plane`: A plane

### Output

Returns a tuple consisting of

  * `normal`: A normal vector and...
  * `point`: ... point defining the geometric plane represented by `plane`
