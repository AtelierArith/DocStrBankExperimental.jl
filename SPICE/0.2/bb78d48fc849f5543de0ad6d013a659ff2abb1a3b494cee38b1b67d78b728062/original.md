```
dskgd(handle, dladsc)
```

Return the DSK descriptor from a DSK segment identified by a DAS handle and DLA descriptor.

### Arguments

  * `handle`: Handle of a DSK file
  * `dladsc`: DLA segment descriptor

### Output

Returns the DSK segment descriptor.
