```
spk14e(handle)
```

End the type 14 SPK segment currently being written to the SPK file associated with `handle`. See also [`spk14a`](@ref) and [`spk14b`](@ref).

### Arguments

  * `handle`: The handle of an SPK file open for writing

### Output

Returns the handle of the SPK file.
