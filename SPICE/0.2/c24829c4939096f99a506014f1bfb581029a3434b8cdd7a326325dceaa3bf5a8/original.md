```
scs2e(sc, sclkch)
```

Convert a spacecraft clock string to ephemeris seconds past J2000 (ET).

### Arguments

  * `sc`: NAIF integer code for a spacecraft
  * `sclkch`: An SCLK string

### Output

Returns ephemeris time seconds past J2000.
