```
wnfetd(window, n)
```

Fetch a particular interval from a double precision window.

### Arguments

  * `window`: Input window
  * `n`: Index of interval to be fetched

### Output

Returns a tuple consisting of the left and right endpoints of the n-th interval in the input window.
