```
axisar(axis, angle)
```

Construct a rotation matrix that rotates vectors by a specified `angle` about a specified `axis`.

### Arguments

  * `axis`: Rotation axis
  * `angle`: Rotation angle in radians

### Output

Rotation matrix corresponding to `axis` and `angle`
