```
ckobj!(ck, ids)
```

Find the set of ID codes of all objects in a specified CK file.

### Arguments

  * `ck`: Name of CK file
  * `ids`: Set of ID codes of objects in CK file. Data already present in   `ids` will be combined with ID code set found for the file `ck`.
