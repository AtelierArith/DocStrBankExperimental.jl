TumorGrowth.jlは、腫瘍成長のための以下のモデルを提供します：

| モデル                             | 説明                             | パラメータ, `p`            | 解析的? |
|:------------------------------- |:------------------------------ |:--------------------- |:---- |
| [`bertalanffy`](@ref)           | 一般的なバートランフィ（GB）                | `(; v0, v∞, ω, λ)`    | はい   |
| [`bertalanffy_numerical`](@ref) | 一般的なバートランフィ（テスト専用）             | `(; v0, v∞, ω, λ)`    | いいえ  |
| [`bertalanffy2`](@ref)          | 一般的なバートランフィの2D拡張               | `(; v0, v∞, ω, λ, γ)` | いいえ  |
| [`gompertz`](@ref)              | 古典的なゴンペルツ（GB, `λ=0`）           | `(; v0, v∞, ω)`       | はい   |
| [`logistic`](@ref)              | 古典的なロジスティック/ヴェルフスト（GB, `λ=-1`） | `(; v0, v∞, ω)`       | はい   |
| [`classical_bertalanffy`](@ref) | 古典的なバートランフィ（GB, `λ=1/3`）       | `(; v0, v∞, ω)`       | はい   |
| [`exponential`](@ref)           | 指数的減衰または成長                     | `(; v0, ω)`           | はい   |
| [`neural(rng, network)`](@ref)  | Lux.jl `network`を用いた1DニューラルODE | `(; v0, v∞, θ)`       | いいえ  |
| [`neural2(rng, network)`](@ref) | Lux.jl `network`を用いた2DニューラルODE | `(; v0, v∞, θ)`       | いいえ  |

ここでの*モデル*は、呼び出し可能なオブジェクトであり、関連する常微分方程式を解くことによって、与えられた時間に対する病変体積のシーケンスを出力します（以下のパラメータ `p` による）：

```julia
using TumorGrowth

times = [0.1, 6.0, 16.0, 24.0, 32.0, 39.0]
p = (v0=0.0002261, v∞=2.792e-5,  ω=0.05731) # `v0`は初期体積
volumes = gompertz(times, p)
6-element Vector{Float64}:
 0.0002261
 0.0001240760197801191
 6.473115210101774e-5
 4.751268597529182e-5
 3.9074807723757934e-5
 3.496675045077041e-5
```

すべてのモデルにおいて、`v0`は初期体積であり、したがって`volumes[1] == v0`です。

基礎となるODEの解析解が知られていない場合、[DifferentialEquations.jl](https://docs.sciml.ai/DiffEqDocs/stable/)ソルバーのオプションキーワード引数をモデル呼び出しに渡すことができます。

TumorGrowth.jlはまた、モデルパラメータをキャリブレーションするための[`CalibrationProblem`](@ref)ツールと、ホールドアウトセット上でモデルを比較するための[`compare`](@ref)ツールを提供します。
