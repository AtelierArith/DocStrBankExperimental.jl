```
Bb!(mod::Model)
```

行列 `B = A' * A` とベクトル `b = A' * y` を返します。

## デフォルト

  * メソッド [A](@ref A) と [y](@ref y) に基づく直接計算。

## 備考

  * `χ²`（およびその導関数）の計算のための必須ルーチン
  * パフォーマンスを向上させるために、モデル固有の実装に置き換えることができます。
  * `(B, b)::Tuple` を返すことが期待されます。
  * 一般的なモデルの場合、返り値の型は `B::SMatrix{Nc,Nc,T}` と `b::SVector{Nc,T}` である可能性があります。
  * ほとんどの場合、モデル固有の実装の方が効率的です。
