```
SummarizedExperiment()
```

Create an empty `SummarizedExperiment` with no assays and empty row/column annotations.

# Examples

```jldoctest
julia> using SummarizedExperiments

julia> SummarizedExperiment()
0x0 SummarizedExperiment
  assays(0):
  rownames:
  rowdata(1): name
  colnames:
  coldata(1): name
  metadata(0):
```
