```
rfldiv(A,b,f=lu; kwargs...) -> x,bnorm,bcomp,flags
```

線形システム $A x = b$ に対して、誤差境界を持つ超高精度反復精緻化を使用して正確な解を計算します。

解 `x`、ノルムに基づく相対前方誤差推定 `bnorm`、および最大成分ごとの相対誤差推定 `bcomp` を返します。具体的には、`bnorm` は $‖xtrue - x‖ / ‖x‖$ (最大ノルム) の推定値です。問題が非常に条件が悪く、良い解が実現不可能な場合、`bnorm` と `bcomp` は単位に設定されます（`expert` でない限り）。`flags` には、専門家にとって興味深い収束診断が含まれています。

# 引数

  * `A`: 行列、
  * `b`: 同じ `eltype` を持つベクトル、
  * `f`: `lu` のような因数分解関数。

## キーワード

  * `DT`: 精緻化のための高精度型; デフォルトは `widen(eltype(A))`
  * `verbosity`: 0(デフォルト): 静か、1: 反復の報告、2: 詳細。
  * `equilibrate::Bool`: 関数が `A` を均衡化すべきかどうか (デフォルト `true`)。
  * `maxiter`: デフォルト 20。
  * `tol`: 収束のための相対許容誤差、`eps(T)` の単位で。
  * `expert::Bool`: 極端な場合に疑わしい境界を返すかどうか。
  * `κ`: `A` の (最大ノルム) 条件 (以下を参照)。
  * `F`: `A` の因数分解 (以下を参照)。

`A` がすでに均衡化されており、因数分解オブジェクト `F` と条件推定 `κ` がすでに計算されている場合、それらをキーワード引数として提供できます; 一貫性のチェックはここでは行われません。

Demmel et al. ACM TOMS, 32, 325 (2006) のアルゴリズムを使用します。
