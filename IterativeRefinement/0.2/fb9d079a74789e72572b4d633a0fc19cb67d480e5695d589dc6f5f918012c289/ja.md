```
rfeigen(A, S::Schur, idxλ, DT, maxiter=5) -> vals, vecs, status
```

行列 `A` の固有値のクラスターの精度を、より高精度の実数型 `DT` を使用して多倍精度反復精度向上により改善します。固有値と、それに対応する不変部分空間を生成するベクトルの改善された推定値を返します。

このメソッドは、`S.values` にインデックス付けされた固有値のセットで動作します。これは（ほぼ）欠陥のあるケースを処理するように設計されていますが、行列が非常に非正規であるか、初期推定が不十分な場合は失敗します。`S` は真のシュール分解でなければならず、「実シュール」ではありません。
