```
rem_prop!(g, prop)
rem_prop!(g, v, prop)
rem_prop!(g, e, prop)
rem_prop!(g, s, d, prop)
```

Remove property `prop` from graph `g`, vertex `v`, or edge `e` (optionally referenced by source vertex `s` and destination vertex `d`). If property, vertex, or edge does not exist, will not do anything.
