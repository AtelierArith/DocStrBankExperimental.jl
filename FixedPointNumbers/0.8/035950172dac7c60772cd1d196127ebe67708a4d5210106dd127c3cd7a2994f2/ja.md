```
Fixed{T <: Signed, f} <: FixedPoint{T, f}
```

`Fixed{T,f}` は `Signed` 整数を `-2^f` から `2^f` の範囲を [-1.0, 1.0] にマッピングします。例えば、`Fixed{Int8,7}` は `-128` を `-1.0` に、`127` を `127/128 ≈ 0.992` にマッピングします。

`QXfY` 表記法における `Fixed` の型エイリアスがあり、ここで `Y` は小数ビットの数（すなわち `f`）で、`X+Y+1` は使用される基礎ビットの数を表します（`+1` は符号ビットを意味します）。例えば、`Q0f7` は `Fixed{Int8,7}` にエイリアスされ、`Q3f12` は `Fixed{Int16,12}` にエイリアスされます。
