```
pairwise_hamming(X, Y; step=1, step_left, step_right, as_vec=true, kwargs...)
pairwise_hamming(X; as_vec, step, kwargs...)
```

`X` と `Y` のシーケンス間のすべてのハミング距離を返します。2 番目の形式では、`X` のシーケンスのペアを考慮します。

`step` ごとにシーケンスのみを考慮します。`step_left` と `step_right` は、`X` または `Y` のいずれかのシーケンスをスキップするために使用できます。これは、大きなアライメントの場合に便利で、計算の数はアライメントのサイズの積に比例して増加します。

デフォルトでは、返される値は `[H(1,2), H(1,3), ..., H(M-1, M)]` のように整理されたベクトルで、`H` はハミング距離を、`M` はシーケンスの数を表します。行列が好ましい場合は、`as_vec=false` を使用してください。

追加のキーワード引数は `hamming` に渡されます。
