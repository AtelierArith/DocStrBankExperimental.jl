```
Foil(; kws...)
```

トレーニングプロセスの加速を担当するタイプです。

# コンストラクタ引数

  * **accelerator**: 異なるアクセラレータタイプを渡すことをサポートします：

      * `:auto`（デフォルト）：利用可能な場合は自動的にGPUを選択し、そうでない場合はCPUにフォールバックします。
      * `:gpu`：`:auto`と同様ですが、GPUが利用できない場合はエラーをスローします。GPUが利用可能であるためには、対応するパッケージがロードされている必要があります（例：`using CUDA`）。トリガーパッケージは、Nvidia GPU用の`CUDA.jl`、AMD GPU用の`AMDGPU.jl`、Apple Silicon用の`Metal.jl`です。
      * `:cpu`：CPUの使用を強制します。

    `devices`オプションも参照してください。
  * **devices**：整数`n`を渡して`n`デバイスでトレーニングを行う（現時点では`1`のみサポート）、または特定のデバイスでトレーニングを行うためのデバイスIDのリストを渡します（例：`[2]`はインデックス2のGPUでトレーニングを行います）。IDのインデックスは`1`から始まります。`nothing`の場合、デフォルトデバイスを使用します（`MLDataDevices.gpu_device`のドキュメントを参照）。デフォルト：`nothing`。
  * **precision**：異なる精度タイプ`(:bf16, :f16, :f32, :f64)`を渡すことをサポートします。ここで、`:bf16`はBFloat16、`:f16`はFloat16、`:f32`はFloat32、`:f64`はFloat64です。デフォルト：`:f32`。
