```
struct TEModel
```

ツリーアンサンブルモデルに関する情報を格納するためのユニバーサルデータ型です。これは、ツリーアンサンブルから整数最適化問題を作成する際に使用されるデータ型です。

異なるツリーモデル（EvoTrees、XGBoost、RandomForest）は、このデータ型への個別の変換関数を必要とします。

# フィールド

  * `n_trees`: アンサンブル内のツリーの数
  * `n_feats`: モデル内の特徴（入力変数）の数
  * `n_leaves`: 各ツリーの葉の数
  * `leaves`: 各ツリーの葉のインデックス
  * `splits`: [特徴、分割点インデックス] ペア、[ツリー、ノード] でアクセス可能
  * `splits_ordered`: 各特徴の分割値で順序付けられた分割点
  * `n_splits`: 各特徴の分割点の数
  * `predictions`: 各ノードの予測（葉でないノードはゼロ）
  * `split_nodes`: ノードが分割ノードであるかどうかの情報を含むブール配列

分割点は、アンサンブルからのユニークな条件値のセットです。各ノードは条件値に関連付けられています。
