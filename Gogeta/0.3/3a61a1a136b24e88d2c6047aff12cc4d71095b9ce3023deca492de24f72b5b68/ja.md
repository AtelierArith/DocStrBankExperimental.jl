```
struct CNNStructure
```

畳み込みニューラルネットワークの層構造のコンテナ。

この構造体は、CNNパラメータをある関数から別の関数に渡すために使用されます。この構造体は、`get_structure`関数を使用して作成できます。

# フィールド

  * `channels`: 畳み込み層またはプーリング層のための (層, チャンネル数) ペアの辞書
  * `dims`: 畳み込み層またはプーリング層のための (層, (# 行, # 列)) ペアの辞書
  * `dense_lengths`: 密な層およびフラット層のための (層, ニューロン数) ペアの辞書
  * `conv_inds`: 畳み込み層のインデックスのベクター
  * `maxpool_inds_inds`: マックスプール層のインデックスのベクター
  * `meanpool_inds`: ミーンプール層のインデックスのベクター
  * `flatten_ind`: `Flux.flatten`層のインデックス
  * `dense_inds`: 密な層のインデックスのベクター
