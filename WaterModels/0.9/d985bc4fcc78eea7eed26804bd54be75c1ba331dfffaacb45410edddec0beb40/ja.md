```
variable_flow(
    wm::AbstractPWLRDModel;
    nw::Int=nw_id_default,
    bounded::Bool=true,
    report::Bool=true
)
```

流に関連する変数を、区分線形、緩和および方向（PWLRD）に基づく最適化モデルにリンクして作成します。まず、ネットワーク内のすべてのノード接続コンポーネント（例：パイプ）に対して、サブネットワーク（または時間）インデックス `nw` で方向に基づく流量変数を作成します。例えば、`pipe` の `a` に対して `qp_pipe[a]` と `qn_pipe[a]` を作成します。また、方向値の流量に対するJuMP式を作成します。例えば、`pipe` の `a` に対して `q_pipe[a] = qp_pipe[a] - qn_pipe[a]` を作成します。さらに、バイナリ流方向変数を作成します。例えば、`pipe` の `a` に対して `y_pipe[a]` を作成し、これは流れが `node_fr`（アークの尾）から `node_to`（アークの頭）に向かうことを示します。また、ネットワーク内のパイプおよび設計パイプに対して、方向に基づくヘッド差変数を作成します。例えば、`pipe` の `a` に対して `dhp_pipe[a]` と `dhn_pipe[a]` を作成します。次に、必要な区分線形緩和を構築するために使用される連続的な凸結合変数を作成します。例えば、`pipe` の `a` に対して `lambda_p_pipe[a]` と `lambda_n_pipe[a]` を作成し、各々はゼロと一の間に制約されます。最後に、緩和制約の区分線形モデリングに使用されるバイナリ凸結合変数を作成します。例えば、`pipe` の `a` に対して `x_p_pipe[a]` と `x_n_pipe[a]` を作成します。
