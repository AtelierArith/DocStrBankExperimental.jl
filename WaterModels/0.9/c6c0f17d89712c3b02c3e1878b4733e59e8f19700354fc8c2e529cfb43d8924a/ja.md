```
variable_flow(
    wm::AbstractLAModel;
    nw::Int=nw_id_default,
    bounded::Bool=true,
    report::Bool=true
)
```

線形近似（LA）ベースの最適化モデルに関連するフロー関連変数を作成します。まず、ネットワーク内のすべてのノード接続コンポーネント（例：パイプ）に対して、サブネットワーク（または時間）インデックス `nw` でフロー変数を作成します。例えば、`pipe` の `a` に対して `q_pipe[a]` を作成します。次に、必要な線形近似を構築するために使用される連続的な凸結合変数を作成します。例えば、`pipe` の `a` に対して `lambda_pipe[a]` を作成し、各変数はゼロと一の間に制約されます。最後に、近似制約の区分線形モデリングに使用されるバイナリ凸結合変数を作成します。例えば、`pipe` の `a` に対して `x_pipe[a]` を作成します。
