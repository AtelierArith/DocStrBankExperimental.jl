```
variable_flow(
    wm::AbstractNCDModel;
    nw::Int=nw_id_default,
    bounded::Bool=true,
    report::Bool=true
)
```

ネットワーク内のすべてのノード接続コンポーネント（例：パイプ、ポンプ）に対して、サブネットワーク（または時間）インデックス `nw` で、デフォルトで制約のある（bounded）または制約のない（unbounded）方向ベースの流量変数を作成します。例えば、`pipe` の `a` に対して `qp_pipe[a]` と `qn_pipe[a]`（すなわち、パイプを通る正方向および逆方向の流れ）を作成します。また、方向値の流れに対する JuMP 式も作成します。例えば、`pipe` の `a` に対して `q_pipe[a] = qp_pipe[a] - qn_pipe[a]` を作成します。さらに、パイプの `a` に対して、流れが `node_fr`（アークの尾）から `node_to`（アークの頭）に向かうことを示すバイナリ流れ方向変数 `y_pipe[a]` も作成します。また、ネットワーク内のパイプおよび設計パイプに対する方向ベースのヘッド差変数も作成します。例えば、`pipe` の `a` に対して `dhp_pipe[a]` と `dhn_pipe[a]` を作成します。これは、流れの方向に基づくネットワークモデルの定式化にのみ使用されます。
