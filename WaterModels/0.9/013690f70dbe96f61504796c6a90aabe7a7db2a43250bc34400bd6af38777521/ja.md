```
function constraint_pump_switch_off(
    wm::AbstractWaterModel,
    a::Int,
    n_1::Int,
    n_2::Int;
    kwargs...
)
```

[`constraint_pump_switch_off`](@ref) 制約を追加するための制約テンプレートで、ポンプのスイッチを *オン* から *オフ* に切り替え、オフに切り替えた場合は少なくとも一定の最小時間オフのままであることを制約します。ここで、`wm` は WaterModels オブジェクトであり、`a` はポンプのインデックス、`n_1` は制約によってモデル化される最初の時間インデックス、`n_2` は制約によってモデル化される隣接する（次の）時間インデックスであり、ポンプの状態の変化（すなわち、オンからオフへの変化）を制限することを許可します。
