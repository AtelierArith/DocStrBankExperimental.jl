```
variable_pump_power(
    wm::AbstractWaterModel;
    nw::Int=nw_id_default,
    bounded::Bool=true,
    report::Bool=true
)
```

ネットワーク内のポンプに対して、サブネットワーク（または時間）インデックス `nw` での制約付き（デフォルト）または制約なしの電力消費変数を作成します。すなわち、`P[a]` は `a` が `pump` の場合です。これらの変数は常に非負であることに注意してください。なぜなら、各ポンプは電力を*消費する*だけだからです。さらに、これらの電力変数からも2つのセットのJuMP式が導出されます：エネルギー消費、すなわち `E[a]` は `a` が `pump` の場合と、時間ステップにおけるポンプ運転のコスト、すなわち `c[a]` は `a` が `pump` の場合です。
