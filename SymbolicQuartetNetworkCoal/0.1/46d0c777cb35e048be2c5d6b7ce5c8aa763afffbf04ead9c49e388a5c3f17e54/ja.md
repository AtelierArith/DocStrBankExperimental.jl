```
readTopologyrand(net; scaleparameter=1.0, dpoints=7)
```

入力ネットワークのすべてのエッジ長と継承確率のランダム化された値を生成します。

## 説明

入力ネットワークは、Newick/Extended Newick 文字列または PhyloNetworks の `HybridNetwork` オブジェクトとして提供できます。パラメータが指定されている場合とされていない場合の両方に対応しています。この関数は **バイナリネットワーク** を前提としており、**非ウルトラメトリックネットワーク** を生成します。

  * エッジの長さは次のように決定されます:   `scaleparameter * (edge.number + [0,1]の範囲のランダム値)`   `scaleparameter` のデフォルトは `1.0` です。
  * 継承確率（ガンマ値）は、範囲 (0,1) のランダム値が割り当てられます。

**注意:**   この関数は、空のネットワークトポロジーにおいて現実的なパラメータを**シミュレートしません**。この関数は、生物学的背景を持つパラメータをシミュレートすることを目的としていません。代わりに、トポロジーが下流の関数によって処理されることを可能にするために、**ダミー** 値を割り当てます。

## 引数

  * `net`: 入力ネットワーク（Newick 文字列、Newick を含むファイル、または `HybridNetwork` オブジェクト）。
  * `scaleparameter`: 生成されたエッジの長さに適用される乗数。デフォルトは `1.0` です。
  * `dpoints`: 四捨五入の小数点以下の桁数。デフォルトは `7` です。

## 戻り値

  * ランダムに割り当てられたエッジの長さと継承確率を持つ PhyloNetworks の `HybridNetwork` オブジェクト。
