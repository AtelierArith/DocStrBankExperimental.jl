```
fgt(v, w, α, z)
```

ベクトル `v` のフォスター–グリア–ソーベック指数を指定された `α` と与えられた貧困閾値 `z` で計算し、重みベクトル `w` によって与えられた重みを使用します。

重みは負であってはならず、欠損またはNaNであってはなりません。重みとデータベクトルは同じ長さでなければなりません。

# 例

```julia
julia> using Inequality
julia> fgt([8, 5, 1, 3, 5, 6, 7, 6, 3], collect(0.1:0.1:0.9), 2, 4)
0.05555555555555555
```
