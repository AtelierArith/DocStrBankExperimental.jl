```
poverty_gap(v, w, z)
```

ベクトル `v` の貧困ギャップを指定された貧困閾値 `z` で計算し、重みベクトル `w` によって与えられた重みを使用します。

重みは負であってはならず、欠損またはNaNであってはなりません。重みとデータベクトルは同じ長さでなければなりません。

# 例

```julia
julia> using Inequality
julia> poverty_gap([8, 5, 1, 3, 5, 6, 7, 6, 3], [0.1,0.5,0.3,0.8,0.1,0.5,0.3,0.8,0.2], 4)
0.13194444444444445
```
