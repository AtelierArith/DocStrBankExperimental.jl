```
plot_hourly_solar(city::String = "",
                  i_row::Int64 = 1;
                  lat::Float64 = 0.0,
                  long::Float64 = 0.0,
                  days::Int64 = 6)
```

前の時間の平均として短波太陽放射を示します。これは、総合的な水平照射量に等しいです。

# 引数

  * `city::String` : 有効な都市名、例："Oslo"、"Paris"、"Amsterdam"など。
  * `i_row::Int64` : 特定の場所に対して複数の一致がある場合、

印刷されたDataFrameから行インデックスを提供することで、希望のタイムゾーンを選択します。デフォルトは1に設定されています。

# オプションのキーワード

  * `lat::Float64` : 場所の地理的WGS84座標 (°S < 0, °N > 0)
  * `long::Float64` : 場所の地理的WGS84座標 (°W < 0, °E > 0)
  * `days::Int64` : データが返される日数。デフォルトは6日で、これが最大です。
