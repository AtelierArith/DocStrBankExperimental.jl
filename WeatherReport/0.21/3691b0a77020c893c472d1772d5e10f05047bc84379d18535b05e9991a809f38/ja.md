```
show_daily(city::String, i_row::Int64 = 1)
```

指定された都市の毎日の天気条件を表示します。

# 引数

  * `city::String` : 有効な都市名、例："Oslo"、"Paris"、"Amsterdam" など。
  * `i_row::Int64` : 指定された場所に対して複数の一致がある場合、                  印刷された DataFrame から行インデックスを提供することで                  希望のタイムゾーンを選択します。デフォルトは 1 に設定されています。
