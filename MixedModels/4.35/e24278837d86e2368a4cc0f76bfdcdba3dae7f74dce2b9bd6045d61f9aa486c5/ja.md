```
confint(pr::MixedModelBootstrap; level::Real=0.95, method=:shortest)
```

係数と分散成分のブートストラップ信頼区間を計算します。信頼レベルは level で、デフォルトは 95% です。

キーワード引数 `method` は、`:shortest`、すなわち最高密度の区間を使用するか、`:equaltail`、すなわち分位数ベースの区間を使用するかを決定します。歴史的な理由から、デフォルトは `:shortest` ですが、`:equaltail` はプロファイルおよびワルド信頼区間と最も比較可能な区間を提供します。

!!! note
    API の保証は、Tables.jl 互換のテーブルに対するものです。正確な戻り値の型は実装の詳細であり、将来のマイナーリリースで変更される可能性がありますが、破壊的な変更とは見なされません。
