```
StatsAPI.predict(m::LinearMixedModel, newdata;
                new_re_levels=:missing)
StatsAPI.predict(m::GeneralizedLinearMixedModel, newdata;
                new_re_levels=:missing, type=:response)
```

新しいデータに対する応答を予測します。

!!! note
    現在、これらのメソッドは内部で新しいモデルを構築するため、応答ベクトルだけでなく、他の多くの行列も割り当てるため、インプレースメソッドは提供されていません。


!!! warning
    `newdata` には、応答（従属変数）の列が含まれている必要があり、数値（`missing` ではない）で初期化されている必要があります。これは、予測を計算するために使用される新しいモデルを構築するために使用されます。`missing` は無効です。なぜなら、`missing` データはモデル行列を構築する前に削除されるからです。


!!! warning
    これらのメソッドは、背後で全体の MixedModel を構築するため、`newdata` が大きい場合には大量のメモリを使用する可能性があります。


!!! warning
    ランク欠損は驚くべき一貫した動作を引き起こす可能性があります。たとえば、2つの完全に共線的な予測因子 `A` と `B`（例えば、お互いの定数倍）がある場合、適合したモデルで `A` がピボットアウトされ、その関連係数がゼロに設定される可能性があります。新しいデータで `B` がゼロに設定され、`A` が設定されていない場合、結果の予測には `A` も `B` も寄与しません。


キーワード引数 `new_re_levels` は、以前に観測されなかったグループ変数の値の扱いを指定します。可能な値は次のとおりです：

  * `:population`: 関連するグループ変数の母集団値を返します。言い換えれば、関連するランダム効果を 0 として扱います。すべてのグループ変数に新しいレベルがある場合、これは固定効果だけと同等です。
  * `:missing`: `missing` を返します。
  * `:error`: この条件でエラーを返します。エラーの種類は実装の詳細です：特定のエラータイプがスローされることに依存しないでください。

グループ変数の未観測レベルに対するシミュレーション値が必要な場合は、[`simulate!`](@ref) および `simulate` メソッドを検討してください。

固定効果のみに基づく予測は、ランダム効果の以前に観測されなかったレベルを指定し、`new_re_levels=:population` を設定することで得られます。同様に、他のグループ変数の以前に観測されたレベルを含めながら、以前に観測されなかったレベルを指定することで、任意のグループ変数の寄与を除外できます。将来的には、使用するグループ変数のサブセットや全体のランダム効果構造を指定できるようになるかもしれませんが、現時点ではできません。

!!! note
    `new_re_levels` は、以前に観測されなかったランダム効果レベル、すなわち新しい RE レベルの動作にのみ影響します。以前に観測されたランダム効果レベルについては、予測は固定効果とランダム効果の両方を考慮します。


`GeneralizedLinearMixedModel` の場合、`type` パラメータは、予測が線形予測子のスケール（`:linpred`）で返されるべきか、応答スケール（`:response`）で返されるべきかを指定します。これらの用語の違いがわからない場合は、おそらく `type=:response` を選択することになります。

回帰重みは、予測ではまだサポートされていません。同様に、オフセットも `GeneralizedLinearMixedModel` ではサポートされていません。
