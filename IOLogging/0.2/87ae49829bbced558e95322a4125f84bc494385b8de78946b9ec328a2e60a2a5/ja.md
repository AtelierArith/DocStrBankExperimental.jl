汎用ファイルロガーは、ファイルへのログ記録を行います。デフォルトでは、各書き込み（すなわち各ログイベント）の後に必要な出力ストリームをフラッシュし、最終化時にファイルを閉じます。オープンされたファイルはデフォルトで追記されます。`append = false`を指定すると、上書きされます。

```
FileLogger(logPaths = Dict(Info => "default.log"; flush = true, append = true)
```

`logPaths`が指定されていない場合、ログレベルが`Info`以上のログイベントは「default.log」に記録されます。`Info`と`Error`のように2つのログレベルが存在する場合、`Info`から（ただし`Error`を除く）までのすべてのログイベントが`Info`によって指定されたファイルに記録されます。`Error`以上のログは`Error`によって指定されたファイルに記録されます。ログイベントを「クランプ」することも可能で、Unix/Macでは`/dev/null`、Windowsでは`NUL`にログを記録する上限を提供します。ただし、メッセージは実際のファイルに書き込む前に構成されるため、注意が必要です（ホットワイヤリングはありません）。

デフォルトでは、ログ中に発生する例外は捕捉されません。これは、将来的にログ中の例外をどのように処理するかが決定されるまでの予定です。
