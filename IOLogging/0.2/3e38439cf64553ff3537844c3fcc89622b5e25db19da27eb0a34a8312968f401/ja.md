任意の `IO` にログを記録するための汎用ロガー。フラッシュは行わず、フラッシュは一般的な IO 操作ではないためです。また、指定された IO を閉じることもありません。

```
IOLogger(logIOs = Dict(Info => stdout))
```

`logIOs` が指定されていない場合、`Info` 以上の LogLevel を持つログイベントは stdout に記録されます。`Info` と `Error` のように二つの LogLevel が存在する場合、`Info` から (ただし `Error` は除く) までのすべてのログイベントは `Info` によって指定されたストリームに記録されます。`Error` 以上のものは `Error` によって指定されたストリームに記録されます。上限を設定してログイベントを「クランプ」することも可能で、その場合は `devnull` に記録されます。ただし、メッセージは実際のストリームに書き込む前に構成されるため、注意が必要です（ホットワイヤリングはありません）。

デフォルトでは、ログ中に発生する例外は捕捉されません。これは、ログ中の例外をどのように処理するかが決定されるまで、将来的に変更されることが期待されています。
