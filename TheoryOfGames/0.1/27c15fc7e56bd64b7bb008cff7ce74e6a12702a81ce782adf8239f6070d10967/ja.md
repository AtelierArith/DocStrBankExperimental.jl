```
utility_combs(player_set, utility)
```

プレイヤーのすべての組み合わせに対するユーティリティを計算する関数。この表は、シャープレイ値の計算など、ゲーム理論で使用される可能性があります。この関数は、player_set内のすべてのプレイヤーの組み合わせを反復し、ユーティリティ関数を実行して共有される利益を特定します。

## 入力

player_set : ベクター     プレイヤーのベクター utility : 関数     任意の連合を与えると、その連合の利益を返すユーティリティ関数     utility(::Vector)::T<:Number である必要があります verbose : ブール     true の場合、現在の実行状況を示すプログレスバーが表示されます parallel : ブール     true の場合、並列処理が使用されて操作が計算されます

## 出力

utilities : 辞書     player_set内の各連合のユーティリティを指定する辞書
