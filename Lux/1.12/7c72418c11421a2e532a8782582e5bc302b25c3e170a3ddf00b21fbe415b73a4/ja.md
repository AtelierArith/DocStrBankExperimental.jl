```
LPPool(window; stride=window, pad=0, dilation=1, p=2)
```

LPプーリング層で、サイズ`window`のブロック内のすべてのピクセルを削減操作lpで置き換えます。

## 引数

  * `window`: ウィンドウのサイズを指定する整数のタプル。例えば、2Dプーリングの場合は`length(window) == 2`

## キーワード引数

  * `stride`: 単一の整数または`N`個の整数を持つタプルのいずれかである必要があります
  * `dilation`: 単一の整数または`N`個の整数を持つタプルのいずれかである必要があります
  * `pad`: データ配列の境界に追加される要素の数を指定します。次のいずれかです。

      * 周囲に均等にパディングを施すための単一の整数、
      * 各空間次元の開始/終了に同じパディングを適用するための`N`個の整数のタプル、
      * 非対称パディングのための`2*N`個の整数のタプル、または
      * `SamePad()`の単一要素で、`size(output,d) == size(x,d) / stride`（おそらく丸められた）となるようにパディングを計算します。

!!! danger "GPUサポート"
    この層は現在CPUでのみサポートされています。


# 拡張ヘルプ

## 入力

  * `x`: `ndims(x) == N + 2`を満たすデータ、すなわち`size(x) = (I_N, ..., I_1, C, N)`

## 戻り値

  * プーリングの出力`y`のサイズは`(O_N, ..., O_1, C, N)`であり、次のようになります。

$$
    O_i = \left\lfloor\frac{I_i + p_i + p_{(i + N) \% |p|} - d_i \times (k_i - 1)}{s_i} + 1\right\rfloor
$$
