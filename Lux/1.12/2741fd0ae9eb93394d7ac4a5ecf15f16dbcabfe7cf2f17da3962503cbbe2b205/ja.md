```
MeanPool(window; stride=window, pad=0, dilation=1)
```

平均プーリング層で、サイズ `window` のブロック内のすべてのピクセルを平均という縮小操作で置き換えます。

## 引数

  * `window`: ウィンドウのサイズを指定する整数のタプル。例えば、2Dプーリングの場合 `length(window) == 2`

## キーワード引数

  * `stride`: 各々が単一の整数、または `N` 個の整数を持つタプルであるべきです
  * `dilation`: 各々が単一の整数、または `N` 個の整数を持つタプルであるべきです
  * `pad`: データ配列の境界に追加される要素の数を指定します。次のいずれかです。

      * 周囲全体に均等にパディングするための単一の整数、
      * 各空間次元の始まり/終わりに同じパディングを適用するための `N` 個の整数のタプル、
      * 非対称パディングのための `2*N` 個の整数のタプル、または
      * `SamePad()` の単一要素で、各空間次元に対して `size(output,d) == size(x,d) / stride` (おそらく丸められた) となるようにパディングを計算します。

# 拡張ヘルプ

## 入力

  * `x`: `ndims(x) == N + 2` を満たすデータ、すなわち `size(x) = (I_N, ..., I_1, C, N)`

## 戻り値

  * プーリングの出力 `y` のサイズは `(O_N, ..., O_1, C, N)` であり、ここで

$$
    O_i = \left\lfloor\frac{I_i + p_i + p_{(i + N) \% |p|} - d_i \times (k_i - 1)}{s_i} + 1\right\rfloor
$$

  * 空の `NamedTuple()`
