```
PixelShuffle(r::Int)
```

アップスケールファクター `r` を持つピクセルシャッフル層。通常、画像をアップスケールしながら高解像度の画像を生成するために使用されます。

詳細については `NNlib.pixel_shuffle` を参照してください。

## 引数

  * `r`: アップスケールファクター

## 入力

  * `x`: N枚の画像を表す4D配列の場合、操作は入力 `size(x) == (W, H, r² x C, N)` を出力サイズ `(r x W, r x H, C, N)` に変換します。D次元データの場合、チャネルとバッチ次元を持つ `ndims(x) == D + 2` を期待し、チャネル数を `rᴰ` で割ります。

## 戻り値

  * 4D配列の場合は出力サイズ `(r x W, r x H, C, N)`、D次元データの場合は `(r x W, r x H, ..., C, N)` で、ここで `D = ndims(x) - 2` です。
