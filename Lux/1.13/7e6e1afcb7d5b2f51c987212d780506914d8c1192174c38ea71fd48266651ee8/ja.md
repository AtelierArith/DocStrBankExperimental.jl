```
batched_jacobian(f, backend::AbstractADType, x::AbstractArray)
```

関数 `f` のジャコビアンを入力のバッチ `x` に関して計算します。これは `y = f(x)` に対して以下の特性を期待します：

1. `ndims(y) ≥ 2`
2. `size(y, ndims(y)) == size(x, ndims(x))`

## バックエンドとADパッケージ

| サポートされているバックエンド   | 必要なパッケージ    |
|:----------------- |:----------- |
| `AutoForwardDiff` |             |
| `AutoZygote`      | `Zygote.jl` |

## 引数

  * `f`: ジャコビアンを計算する関数。
  * `backend`: ジャコビアンを計算するために使用するバックエンド。
  * `x`: 関数への入力。`ndims(x) ≥ 2` でなければなりません。

## 戻り値

  * `J`: `x` に関する `f` のジャコビアン。これは3D配列になります。`x` の次元が `(N₁, N₂, ..., Nₙ, B)` で、`y` の次元が `(M₁, M₂, ..., Mₘ, B)` の場合、`J` は `((M₁ × M₂ × ... × Mₘ), (N₁ × N₂ × ... × Nₙ), B)` の配列になります。

!!! danger
    `f(x)` はバッチ次元を混在させてはいけません。そうしないと結果が不正確になります。例えば、`f` にバッチ正規化のような操作が含まれている場合、結果は不正確になります。
