```
expand!(ghost, forest, nodes)
```

ゴーストレイヤーを拡張して、隣接するランクと共有されているノードを持つすべての要素を含めます。

次のクワッドツリーの森を考えてみましょう。

```
+-------+---+---+
|       | 5 | 6 |
|   2   +---+---+
|       | 3 | 4 |
+-------+---+---+
|       |       |
|   0   |   1   |
|       |       |
+-------+-------+
```

これは、ランク0が象限{0,1}を所有し、ランク1が象限{2, 3, 4, 5, 6}を所有するように分割されています。ランク0の完全に接続されたゴーストレイヤーには、象限{2, 3, 4}が含まれます。QuadrantWrapper 5はランク0とグローバルノードを共有していますが、完全に接続されたゴーストレイヤーには含まれていません。この関数は、このような象限を含めるためにゴーストレイヤーを拡張します。

隣接するランクと共有されているグローバルノードのリストを取得するには、[`sharers`](@ref)を参照してください。
