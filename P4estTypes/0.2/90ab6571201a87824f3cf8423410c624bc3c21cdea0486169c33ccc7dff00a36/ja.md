```
pxest(connectivity::Connectivity{X}; kw...) where {X}
```

接続性に基づいて、クアッドツリー（`X=4`の場合）またはオクトリー（`X=8`の場合）の分散フォレストを生成します。接続性の各要素がツリーのルートになります。

接続性はすべてのランクで複製されますが、フォレストの葉は（空間充填曲線の順序に基づいて）ランク間で分割されます。

フォレストの構築を制御するキーワード引数（`kw...`）は次のとおりです：

  * `comm = MPI.COMM_WORLD`: フォレストを共有するランクのMPIコミュニケーターオブジェクト。
  * `min_quadrants = 0`: ランクごとの最小クアドラント数。（これにより、初期の細分化パターンが`MPI.Comm_size`に特有になります。）
  * `min_level = 0`: フォレストのクアドラント細分化の最小レベル。
  * `fill_uniform = true`: `true`の場合、フォレストは均一なメッシュで満たされます。そうでない場合は、可能な限り粗いメッシュになります。
  * `data_type = Nothing`: 各クアドラントに保存されるユーザーデータの`isbitstype`。
  * `init_function = nothing`: 各クアドラントのユーザーデータを初期化するために呼び出されるプロトタイプ`init_function(forest, treeid, quadrant)`を持つコールバック関数。
