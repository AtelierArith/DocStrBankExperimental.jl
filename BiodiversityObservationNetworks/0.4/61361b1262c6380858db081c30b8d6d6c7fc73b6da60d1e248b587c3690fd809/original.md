```
BalancedAcceptance
```

`BalancedAcceptance` is a type of [`BONSampler`](@ref) for generating [`BiodiversityObservationNetwork`](@ref)s with spatial spreading.

*Arguments*:

  * `number_of_nodes`: the number of sites to select
  * `grid_size`: if being used on a polygon, the dimensions of the grid used to cover the extent. Balanced Acceptance sampling uses discrete Cartesian indices
