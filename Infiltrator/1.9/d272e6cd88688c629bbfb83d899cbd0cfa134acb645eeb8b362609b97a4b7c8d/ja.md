```
@infiltrate
@infiltrate condition::Bool
```

`@infiltrate` は浸透ポイント（またはブレークポイント）を設定します。

浸透ポイントに到達すると、ローカル変数やコールスタックを調査し、現在の関数のモジュールのコンテキストで任意のステートメントを実行できるインタラクティブなREPLセッションに入ります。

このマクロは、ブール値に評価される必要があるオプションの引数 `cond` も受け入れます。このマクロは「条件付きブレークポイント」として機能し、その条件が `true` のときのみ検査を開始します。例えば：

```
@infiltrate false # 浸透しない
```
