```
BlochMcConnellMatrix(A)
BlochMcConnellMatrix{T}(N)
BlochMcConnellMatrix(N)
```

`N` 個のコンパートメントを持ち、固定サイズの 3N×3N 行列を表す `BlochMcConnellMatrix` オブジェクトを作成します。

# プロパティ

  * `A::NTuple{N,NTuple{N,BlochMatrix{<:Real}}}`: `BlochMcConnellMatrix` のブロックを構成する 3×3 行列のリスト; `A[i][j]` は (i,j) 番目のブロックです
