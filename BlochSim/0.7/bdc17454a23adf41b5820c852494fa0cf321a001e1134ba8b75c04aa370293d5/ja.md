```
RF(waveform, Δt, [Δθ], [grad]) <: AbstractRF
```

与えられた（おそらく複素値の）波形（G）と時間ステップ `Δt`（ms）を持つRFパルスを表します。`Δθ` は波形に追加される追加の位相（デフォルトは `0`）であり、`grad` はRFパルス中にオンになるB0勾配（デフォルトは `Gradient(0, 0, 0)`、すなわちオフ）です。

# プロパティ

  * `α::Vector{<:Real}`: 各時間点での瞬時フリップ角（ラジアン）；`waveform` の大きさから計算されます
  * `θ::Vector{<:Real}`: 各時間点での瞬時位相（ラジアン）；`waveform` の位相から計算されます
  * `Δt::Real`: 時間ステップ（ms）
  * `Δθ_initial::Real`: 位相サイクリングの増分が適用される前に `θ` に追加される位相
  * `Δθ::Ref{<:Real}`: `θ` に追加される位相；位相サイクリング/RFスプーリングをシミュレートするために更新可能
  * `grad`: RFパルス中に適用される勾配

      * `::Gradient`: 定常勾配
      * `::Vector{<:Gradient}`: 時間変化する勾配
