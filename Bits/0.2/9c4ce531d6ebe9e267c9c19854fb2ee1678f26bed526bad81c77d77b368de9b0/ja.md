```
mask(T::Type{<:Integer}:=UInt, i::Integer=bitsize(T)) -> T
```

整数型 `T` の `i` ビットの右端が `1` で、他のビットが `0` である整数を返します（すなわち、正確に `i` 個の `1` を持つ形式 `0b0...01...1`）。`i` が指定されていない場合、すべてのビットが `1` に設定されます。`i < 0` の場合、結果は未定義です。`T` のデフォルトは `UInt` です。

# 例

```jldoctest
julia> mask(3)
0x0000000000000007

julia> mask(UInt8)
0xff

julia> bits(mask(Int32, 24))
<00000000 11111111 11111111 11111111>
```
