```
mask(T::Type{<:Integer} := UInt, j::Integer, i::Integer) -> T
```

整数型 `T` の値を返します。この値は `j` ビットの右端が `0` で、次の `i-j` ビットが `1`、残りのビットが `0` である（すなわち、正確に `i-j` 個の `1` が `j` 個の `0` に続く形 `0b0...01...10...0`）。`j < 0` の場合、結果は未定義です。`i < 0` の場合、結果は `~mask(T, j)` に等しく、すなわち正確に `j` 個のゼロを持つ形 `1...10...0` になります。注意：不安定なAPIであり、`mask(j, i-j)` に変更される可能性があります。

# 例

```jldoctest
julia> bits(mask(UInt8, 2, 5))
<00011100>

julia> bits(mask(BigInt, 3, -1))
<...1 11111111 11111111 11111111 11111111 11111111 11111111 11111111 11111000>
```
