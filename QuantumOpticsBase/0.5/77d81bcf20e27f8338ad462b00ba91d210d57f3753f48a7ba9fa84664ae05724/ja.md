```
manybodyoperator(mb::ManyBodyBasis, op)
```

与えられた一体演算子 `op` から多体演算子を作成します。

与えられた演算子は、一体演算子または二体相互作用のいずれかである可能性があります。高次の相互作用は現在実装されていません。

一体の場合の数学的形式は次のように表されます。

$$
X = \sum_{ij} a_i^† a_j ⟨u_i| x | u_j⟩
$$

相互作用の場合は次のように表されます。

$$
X = \sum_{ijkl} a_i^† a_j^† a_k a_l ⟨u_i|⟨u_j| x |u_k⟩|u_l⟩
$$

ここで、$X$ は N 粒子演算子、$x$ は一体演算子、$|u⟩$ は N 粒子基底の異なるモードに関連付けられた一体状態です。
