```
apply_column!(v, op, addr, num, boost=1) -> stats::Tuple
```

オペレーター `op` の列 `addr` の積とスカラー `num` をベクトル `v` に [`StochasticStyle`](@ref) に従って適用します。期待値としてこれは次のように等価です。

```
v .+= op[:, add] .* num
```

これはFCIQMCにおけるスポーンステップを実行し、オペレーター-ベクトルの乗算を実装するために使用されます。`v` を変更し、スポーン統計を報告します。

`boost` 引数は、手続きの期待値に影響を与えることなく、実行されるスポーンの数を乗法的に増加させます。
