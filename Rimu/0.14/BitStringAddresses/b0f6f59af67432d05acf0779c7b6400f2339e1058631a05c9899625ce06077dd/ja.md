```
BitString{B,N,T<:Unsigned}
```

静的サイズのビット列を格納するための型。`N` チャンクに `B` ビットを保持し、各チャンクは型 `T` です。

`N` は、`B` ビットをできるだけ効率的に収容するように自動的に選択されます。

# コンストラクタ

  * `BitString{B,N,T}(::SVector{N,T})`: 安全でないコンストラクタ。ゴーストビットのチェックは行いません。
  * `BitString{B,N,T}(i::T)`: 上記と同様ですが、`i` を最右のチャンクとして設定します。
  * `BitString{B}(::Integer)`: 整数を `BitString` に変換します。整数は正しいビット数に切り捨てられます。
