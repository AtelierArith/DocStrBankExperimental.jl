```
annotation_3D!(sz=(128,128, 1); numbers_or_alphabets="alphabets", font_size=Float64.(minimum(sz[1:2]))-10.0, bkg=0.9)

アルファベットまたは数字の3D配列を、異なるフォントサイズと背景レベルで作成します。
```

# 引数

  * `sz::Tuple`: ボリュームのサイズを表す3つの整数のタプル。デフォルトは(128, 128, 1)です。
  * `numbers_or_alphabets::String`: アルファベットまたは数字を使用するかを表す文字列。デフォルトは"alphabets"です。
  * `font_size::Float64`: フォントサイズを表す浮動小数点数。デフォルトは`sz`の最初の2つの要素の最小値から10.0を引いたものです。
  * `bkg::Float64`: 背景レベルを表す浮動小数点数。デフォルトは0.9です。

# 戻り値

```
異なるフォントサイズと背景レベルのアルファベットまたは数字の3D配列。
```
