```
filaments3D!(obj; radius = 0.8, rand_offset=0.05, rel_theta=1.0, num_filaments=50, apply_seed=true, thickness=0.8)
```

フィラメントの3D表現を作成します。

# 引数

  * `obj`: フィラメントが追加されるボリュームを表す3D配列。
  * `radius`: フィラメントが作成されるボリュームの相対半径を表す実数のタプル（または単一の実数）。デフォルトは0.8です。タプルが使用される場合、フィラメントは対応する楕円形のボリューム内に作成されます。半径は、配列を作成する`filaments3D`バージョンでのみ強制されることに注意してください。
  * `rand_offset`: サイズに対するフィラメントのランダムオフセットを表す実数のタプル。デフォルトは0.05です。
  * `rel_theta`: フィラメントの相対的なシータ範囲を表す実数。デフォルトは1.0です。
  * `num_filaments`: 作成されるフィラメントの数を表す整数。デフォルトは50です。
  * `apply_seed`: ランダム数生成器にシードを適用するかどうかを表すブール値。デフォルトはtrueです。
  * `thickness`: ピクセル単位でのフィラメントの厚さを表す実数。デフォルトは0.8です。

結果はobj入力配列に追加されます。

# 例

```julia

# 中央スライスにのみランダムなフィラメントの配置がある10本のフィラメントを持つ100x100x100のボリュームを作成
julia> obj = rand(100,100,100); # ランダムな数の配列を作成
julia> filaments3D!(obj; num_filaments=10, rel_theta=0, rand_offset=(0.1,0.1,0), intensity=2.0);

```
