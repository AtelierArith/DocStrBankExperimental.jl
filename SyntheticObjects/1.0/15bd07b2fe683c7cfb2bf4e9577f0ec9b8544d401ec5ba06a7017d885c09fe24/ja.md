```
Pollen3D(sv = (128, 128, 128), dphi::Float64=0.0, dtheta::Float64=0.0)
```

花粉粒子の3D表現を作成します。

# 引数

  * `sv::Tuple`: 花粉粒子が作成される体積のサイズを表す3つの整数のタプル。デフォルトは (128, 128, 128) です。
  * `dphi::Float64`: ラジアンで表される phi 角のオフセットを表す浮動小数点数。デフォルトは 0.0 です。
  * `dtheta::Float64`: ラジアンで表される theta 角のオフセットを表す浮動小数点数。デフォルトは 0.0 です。

# 戻り値

  * `ret::Array{Float64}`: 花粉粒子を表す3D配列。

# 例

```jldoctest
Pollen3D((256, 256, 256), 0.0, 0.0)
```
