```
filaments3D([DataType], sz= (128, 128, 128), rand_offset=0.05, rel_theta=1.0, num_filaments=50, apply_seed=true, thickness=0.8)
```

フィラメントの3D表現を作成します。

# 引数

  * `DataType`: 出力配列のデータ型。デフォルトは Float32 です。
  * `sz`: フィラメントが作成されるボリュームのサイズを表す整数のタプル。デフォルトは (128, 128, 128) です。
  * `radius`: フィラメントが作成されるボリュームの相対半径を表す実数のタプル（または単一の実数）。デフォルトは 0.8 です。タプルが使用される場合、フィラメントは対応する楕円形のボリューム内に作成されます。半径は、配列を作成する `filaments3D` バージョンでのみ強制されることに注意してください。
  * `rand_offset`: サイズに対するフィラメントのランダムオフセットを表す実数のタプル。デフォルトは 0.05 です。
  * `rel_theta`: フィラメントの相対的なシータ範囲を表す実数。デフォルトは 1.0 です。
  * `num_filaments`: 作成されるフィラメントの数を表す整数。デフォルトは 50 です。
  * `apply_seed`: ランダム数生成器にシードを適用するかどうかを表すブール値。デフォルトは true です。
  * `thickness`: ピクセル単位でのフィラメントの厚さを表す実数。デフォルトは 0.8 です。

結果は obj 入力配列に追加されます。

# 例

```jldoctest

# 中央スライスにのみランダムなフィラメント配置がある100x100x100ボリュームに100本のフィラメントを作成
julia> obj = filaments3D((100,100,100); rel_theta=0, rand_offset=(0.2, 0.2, 0));

# 3Dに配置された100本のフィラメントを持つ100x100x100ボリュームを作成
julia> obj = filaments3D((100,100,100));

```
