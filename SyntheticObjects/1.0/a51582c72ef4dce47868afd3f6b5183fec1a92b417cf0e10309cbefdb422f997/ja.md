```
object_3D([DataType], sz, radius=0.8, center=sz.÷2 .+1; thickness=0.8)
```

3Dオブジェクトを作成します。中空の球体（細胞膜）、別の中空の球体（核）、中空の小球（小胞）、充填された小球（）、および線を含みます。

# 引数

  * `DataType`: 出力配列のオプションのデータ型。デフォルトはFloat32です。
  * `sz`: オブジェクトのサイズを表す整数のベクトル。デフォルトは(128, 128, 128)です。
  * `radius`: 細胞球の相対半径を表す浮動小数点数（またはタプル）。
  * `center`: オブジェクトの中心を表すタプル。
  * `thickness`: ピクセル単位で膜の厚さを表す浮動小数点数。デフォルトは0.8です。
