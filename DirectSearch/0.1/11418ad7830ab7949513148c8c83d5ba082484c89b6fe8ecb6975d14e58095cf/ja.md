```
AddExtremeConstraint(p::AbstractProblem, c::Function
                     index::CollectionIndex=CollectionIndex(1)
                    )::ConstraintIndex where T
```

単一の関数を登録して、極端なバリア制約を定義します。制約インデックスを返します。

提供された関数はベクトル入力を受け取り、制約が満たされているかどうかを示すブール値または数値を返す必要があります。`true` または 0 以下は制約が満たされていることを示します。`false` または 0 より大きいは制約が違反していることを示します。

`index` 引数は、制約を追加するコレクションを指定するために使用できます。指定されたコレクションは存在し、極端なバリア制約を受け入れることができなければなりません。`index` が指定されていない場合は、デフォルトの極端な制約コレクションであるコレクション 1 に追加されます。
