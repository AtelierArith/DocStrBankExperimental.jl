```
この関数は、確率の範囲内で一連のグラフにわたるRFプロセスをシミュレートします。
パラメータ:
    - samples::Integer => サンプルするグラフの数,
    - first_infected => 最初に感染したノードのセット,
    - model_type::DataType => サンプリングするモデル（SirModelのサブタイプである必要があります）,
    - constructor_params::Dict => モデルを構築するためのパラメータ（指定されたモデルの要件に準拠する必要があります）
    - ε::Float64 => 考慮される確率の粒度（デフォルト = 0.01）,
    - probs_limits::Tuple => シミュレートする確率の制限（デフォルト = (1.0e-6, 1 - 1.0e-6)）,
    - plot_result => 結果グラフをプロットするためのフラグ（デフォルト = false）
    - log => シミュレーションの状態に関する情報を取得するためのフラグ（デフォルト = true）
出力: 
    infected => 確率範囲内の各感染確率に対する総感染者の割合
    standard_deviation => 各確率に対する感染者の標準偏差
    time => 各確率に対するリード-フロストプロセスを完了するための平均時間
```
