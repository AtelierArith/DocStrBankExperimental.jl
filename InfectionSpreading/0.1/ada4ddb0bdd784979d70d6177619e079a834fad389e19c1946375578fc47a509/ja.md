与えられたグラフ、感染者のセット、および感染確率に基づいて、この関数は
リード-フロストプロセスを計算し、以下の情報を含む辞書を返します：

 - "感染者系列" => 時間に対する感染者の数、
 - "総感染者数" => 総感染者の数、
 - "% 感染者" => 総感染者の割合、
 - "生存者" => 総生存者の数、
 - "% 生存者" => 総生存者の割合、
 - "年齢" => プロセスを終了するのに必要な時間、
 - "感染確率" => 与えられた感染確率、
 - "ノードの数" => 与えられたグラフのノードの数、
 - "エッジの数" => 与えられたグラフのエッジの数
