ある種のスイッチ文：

```
@switch x begin
  1; "x は1です！"
  2; "x は2です！"
  "x は他の何かです！"
end
```

ただし、通常のスイッチよりも一般的で、単なる等価性だけでなく、他のテストも行うことができます：

```
@switch isa(x, _) begin
  Integer; "x は整数です！"
  FloatingPoint; "x は浮動小数点数です！"
  "x は他の何かです！"
end

@switch _ begin
  a > b;  "より大きい！"
  a < b;  "より小さい！"
  a == b; "等しい！"       # このレベルの熱意は必須ではありません。
end
```

ここで、`_` は各ケースでテストする値に置き換えられます。最終的な式がある場合、それはデフォルト値として使用されます。デフォルトがなく、何も一致しない場合はエラーがスローされます。
