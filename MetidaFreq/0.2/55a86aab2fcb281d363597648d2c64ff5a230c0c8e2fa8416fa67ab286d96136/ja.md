```
colorder(ct::ConTab, v::AbstractVector{AbstrctString})
```

`v` ベクターの列順でコンティンジェンシーテーブルを作成します。`v` にアイテムが存在しない場合、そのアイテムのためにゼロの列が作成されます。

例 

```
コンティンジェンシーテーブル:
--------- -------- ----- -------
              B      D    合計 
--------- -------- ----- -------
   G1         1     123     124
   G2         0     124     124
--------- -------- ----- -------
 ID: ColName => id;
```

`colorder(ct, ["A", "B", "C", "D"])` の後:

```
コンティンジェンシーテーブル:
--------- -------- --------- --------- ----- -------
              A         B         C      D    合計 
--------- -------- --------- --------- ----- -------
    G1        0         1         0     123     124
    G2        0         0         0     124     124
--------- -------- --------- --------- ----- -------
ID: ColName => id;
```
