```
StrictlyIncreasing(algorithm::OrderedSamplingAlgorithm; n::Int = 50000)
```

要素ごとのサンプリングを示すサンプリングスキームで、結果の値は大きさが厳密に減少します。

減少する逐次サンプリングは、分布が有限のサポートを持つ場合にのみ保証されます。したがって、分布は要素ごとに下位および上位の分位数 `lq` と `uq` に切り捨てられます。各分布について、これは `n` 個の値をそこから引き出し、そのサンプルの分位数を見つけ、最後に分布を経験的な分位数範囲に切り捨てることによって行われます。

`algorithm` はいくつかの `OrderedSamplingAlgorithm` のインスタンスです（例：`StartToEnd`）。`n` は分位数を計算する際に引き出すサンプルの数です。

通常、測定に対して既知の物理的制約がある場合に使用されます。たとえば、堆積物コアの異なる深さでの地球化学的測定は、コア内の物理的に別々の深さで行われます。したがって、インデックスの順序を反転させることはできず、厳密に減少/増加しなければなりません。
