```
StrictlyIncreasing(algorithm::OrderedSamplingAlgorithm; 
    n::Int = 50000, lq = 0.05, uq = 0.95)
```

要素ごとのサンプリングを示すサンプリングスキームであり、結果として得られる値は大きさにおいて厳密に増加します。

増加する逐次サンプリングは、分布が有限のサポートを持つ場合にのみ保証されます。したがって、分布は下位および上位の分位数 `lq` と `uq` に対して要素ごとに切り捨てられます。各分布について、まず `n` 個の値をそこから引き出し、そのサンプルの分位数を見つけ、最後に分布を経験的な分位数範囲に切り捨てます。

`algorithm` は、いくつかの `OrderedSamplingAlgorithm` のインスタンスです（例： `StartToEnd`）。 `n` は分位数を計算する際に引き出すサンプルの数です。

通常、測定に対して既知の物理的制約がある場合に使用されます。たとえば、堆積物コアの異なる深さでの地球化学的測定は、コア内の物理的に別々の深さで行われます。したがって、インデックスの順序を反転させることはできず、厳密に減少/増加しなければなりません。
