```
bin_mean(left_bin_edges::AbstractRange, xs, ys)
```

`ys`の要素を、`left_bin_edges`で定義された`N`のグリッドポイントに基づいて、`N - 1`の異なるビンベクトルに分配します。次に、各ビンのビン平均を計算します。

もし`xs[i]`が`n`-番目のビン区間に入る場合、`ys[i]`は`n`-番目のビンベクトルに割り当てられます。値がグリッドの外にある場合は無視されます（`xs[i] < minimum(left_bin_edges)`の場合、`ys[i]`を無視します）。`ys`の値がビンベクトルに割り当てられた後、要約関数`f`を各ビンベクトルに要素ごとに適用し、`args`と`kwargs`を引数およびキーワード引数として使用します。

`N - 1`の平均値を返します。各ビンごとに1つです。

## 例

```julia
xs = [1.2, 1.7, 2.2, 3.3, 4.5, 4.6, 7.1]
ys = [4.2, 5.1, 6.5, 4.2, 3.2, 3.2, 2.5]
left_bin_edges = 0.0:1.0:6.0
bin_mean(left_bin_edges, xs, ys)

# 出力
6-element Array{Float64,1}:
 NaN   
   4.65
   6.5 
   4.2 
   3.2 
 NaN
```
