```
cov(d1::AbstractUncertainValueDataset, d2::AbstractUncertainValueDataset, n::Int; kwargs...)
```

2つの不確実なデータセット `d1` と `d2` の間の共分散の分布を取得します。これは、両方のデータセットを複数回再サンプリングし、それらの引き出しの間の共分散を計算することによって行われます。これにより、共分散推定値の分布が得られます。

手順は次のとおりです。

1. まず、`d1` の不確実な値を提供する分布に従って `d1` の実現を引き出します。
2. 次に、`d2` の提供する分布に従って `d2` の実現を引き出します。
3. これら2つの引き出し/実現の間の共分散を計算します。どちらも長さ `L = length(d1) = length(d2)` のベクトルです。
4. 手順を `n` 回繰り返し、`d1` と `d2` の `n` 組の実現を引き出します。これにより、`d1` と `d2` の間の共分散の `n` の推定値が得られ、ベクトルとして返されます。
