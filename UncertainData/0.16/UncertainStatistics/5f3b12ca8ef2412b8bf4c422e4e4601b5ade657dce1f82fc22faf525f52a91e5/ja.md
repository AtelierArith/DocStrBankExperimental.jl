```
OneSampleTTestPooled(d1::UncertainDataset,
    d2::UncertainDataset,
    n::Int = 1000; μ0::Real = 0) -> OneSampleTTest
```

まず、各データセットの各不確実な値から `n` 回のサンプリングを行い、`d1` の要素からのサンプリングと `d2` の要素からのサンプリングをそれぞれプールします。次に、`d1` と `d2` の不確実な値のペア間の差が平均 `μ0` を持つ分布から来ているという帰無仮説に対して、分布が平均 `μ0` を持たないという対立仮説のもとで対応のあるサンプル t 検定を実施します。
