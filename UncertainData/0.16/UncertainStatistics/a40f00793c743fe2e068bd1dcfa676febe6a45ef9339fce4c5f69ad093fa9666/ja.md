```
quantile(x::UVAL_COLLECTION_TYPES, q, n::Int)
```

不確実な値のコレクションの分位数 `q` を取得します。これは、まず `x` の長さ `L` の実現を `n` 回描くことによって行われます。ここで、 `L = length(x)` です。次に、これらの長さ `L` の実現ごとに分位数が計算され、分位数推定の分布が得られます。

詳細な手順：

1. まず、データセットを提供する各不確実な値から1つのランダムな数を描くことによって、 `x` の長さ `L` の実現を描きます。描画は独立しているため、データにすでに存在しない要素間の依存関係（例：逐次的相関）が実現に導入されることはありません。
2. 実現の分位数を計算します。これは長さ `L` のベクトルです。
3. 手順を `n` 回繰り返し、 `x` の `n` 個の独立した実現を描きます。これにより、 `x` の分位数の `n` 個の推定値が得られ、ベクトルとして返されます。
