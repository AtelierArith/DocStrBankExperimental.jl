```
cov(x::UVAL_COLLECTION_TYPES, y::UVAL_COLLECTION_TYPES, n::Int; corrected::Bool = true)
```

2つの不確実な値のコレクション間の共分散の分布を取得します。

これは、次の手順を `n` 回繰り返すことによって行われます：

1. 最初に、データセットを提供する各不確実な値から1つのランダムな数を引くことによって、`x` の長さ `L` の実現を引きます。引きは独立しているため、データにすでに存在しない要素間の依存関係（例えば、逐次的な相関）は実現に導入されません。
2. 次に、同様の方法で `y` の長さ `L` の実現を引きます。
3. 2つの長さ `L` の引きの間の共分散を計算します。

これにより、`x` と `y` の `n` 組の独立した実現の間の共分散の `n` の推定値が得られます。共分散推定値の `n` メンバーの分布がベクトルとして返されます。

`corrected` が `true`（デフォルト）である場合、引きの各ペアに対して合計は `n - 1` でスケーリングされますが、`corrected` が `false` の場合は合計は `n` でスケーリングされます（ここで `n = length(x)`）。
