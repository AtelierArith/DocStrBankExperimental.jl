```
var(x::UVAL_COLLECTION_TYPES, n::Int)
```

不確実な値のコレクションの分散の分布を取得します。これは、まず `n` 個の長さ `L` の実現を `x` から引き出すことによって行われます。ここで、 `L = length(x)` です。その後、これらの長さ `L` の実現ごとに分散が計算され、分散推定値の分布が得られます。

詳細な手順：

1. まず、データセットを提供する各不確実な値から1つのランダムな数を引き出すことによって、長さ `L` の `x` の実現を引き出します。引き出しは独立しているため、データに既に存在しない要素間の依存関係（例えば、連続的な相関）が実現に導入されることはありません。
2. 実現の分散を計算します。これは長さ `L` のベクトルです。
3. 手順を `n` 回繰り返し、`x` の `n` 個の独立した実現を引き出します。これにより、`x` の分散の `n` 個の推定値が得られ、ベクトルとして返されます。
