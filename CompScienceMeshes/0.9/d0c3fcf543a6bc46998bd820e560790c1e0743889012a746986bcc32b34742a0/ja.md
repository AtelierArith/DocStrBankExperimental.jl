```
meshicosphere(;nu::Int = 1, r::F = 1.0, nr_verts) where F
```

は Mesh(vertices, faces) を返します。

この関数は、分割周波数 nu を持つ測地線アイコサhedronを生成します。周波数 nu = 1 は正規の単位アイコサhedronを返し、nu>1 は分割を行います。結果は構造化された球体です。

分割周波数を使用すると、可能な頂点の数は nu に応じて増加します。     [12+10*(nu+1)*(nu-1) for nu in range(1,33)]     これは      [12, 42, 92, 162, 252, 362, 492, 642, 812, 1002, 1212, 1442, 1692, 1962,       2252, 2562, 2892, 3242, 3612, 4002, 4412, 4842, 5292, 5762, 6252, 6762,       7292, 7842, 8412, 9002, 9612, 10242] を与えます。

例えば、nu = 3 の場合、アイコサhedronの多角形の各三角形を次のように分割します。

```
                .
               / \
              .---.
             /\ / \
            .---.---.
           /\ /\ / \
          .---.---.---.
```

kwargs: 半径 - 与えられた nu に基づいて単位アイコサフィアに対してスケーリングされます。  nr*verts - nr*verts が指定されている場合、nu はアイコサフィアが少なくとも nr_verts 頂点を含むように調整されます。
