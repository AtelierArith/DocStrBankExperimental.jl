```
meshcuboid(length::F, breadth::F, width::F, edge length::F) where F
```

は Mesh(vertices, faces) を返します。

関数は直方体の単純な面メッシュを返します。キーワード引数 - generator :compsciencemeshes - がデフォルトで、直方体の構造化メッシュを提供します。直方体の寸法はエッジの長さの倍数で近似されます -

```
面の数 = 2*(2*m*n) + 2*(2*n*p) + 2*(2*m*p) -> 前面、背面; 上面、底面;
左面、右面
ここで m は y に沿った要素の数、n は x に沿った要素の数、p は z に沿った要素の数です。

奇数の面は底で直角の三角形です、
  /|
 /_|      または左右反転

 偶数は上面です
  _      
 | /    
 |/        または左右反転

頂点の数は 2*(n + 1)*(m + 1) + 2*(m + 1)*(p - 1) + 2*(n - 1)*(p - 1) です。
これは、直方体のエッジに沿ったノードの数を繰り返さないためです;
前面と背面に沿ったすべてのノードはスタックされ、
次に左面と右面に沿ったノードが配置され、左前、左後、右前、右後のノードはスキップされます。
次に、上面と底面に沿ったノードが配置され、上面前、後、左、右、および底面前、後、左、右のノードはスキップされます。

これは中空の直方体メッシュであるため、
ノードは最初に前面と背面に沿って番号付けされ、その後、z方向のx-yエッジに沿って番号付けされます。

                             20----- 23----- 26
                             |       |       |
                             19      22      25
背面 =>                      |       |       |
                             18----- 21----- 24


                             12----- 14----- 17
中間                       |               |
層 =>                      11      .       16
                             |               |
                             10----- 13----- 15


                             3 ----- 6 ----- 9
                             |       |       |
                             2       5       8
前面 =>                     |       |       |
                             1 ----- 4 ----- 7
```

:gmsh - gmsh と Closed Box のみを使用して直方体の面メッシュを返します。

立方体の表面の他の物理的構成については、gmshcuboid 関数を参照してください。
