```
readmesh(filename)
```

`filename`から*in*形式のメッシュを読み込みます。形式は次の通りです：

```
1
V C
x1_1    x1_2    ... x1_U
x2_1    x2_2    ... x2_U
...
xV_1    xV_2    ... xV_U
i1_1    i1_2    ... i1_D1
i2_1    i2_2    ... i2_D1
...
iC_1    iC_2    ... iC_D1
```

ここで、`U`はメッシュのユニバース次元、`D1`はメッシュの次元に1を加えたもの、`V`は頂点の数、`C`はメッシュ内のセルの数です。
