```
meshrectangle(len::F, breadth::F, edge_length::F, udim = 3) where F
```

は Mesh(vertices, faces) を返します。

関数呼び出しの引数 - udim またはユニバーサル次元 -     2 - 2D空間における長方形を返します     3 - デフォルトで、z = 0 の3D座標系での長方形を返します。

関数はまた、kwarg - generator -  :compsciencemeshes - デフォルトで、ノードと面のベクトルを事前に割り当てた構造化メッシュです。

```
ノード:
(b に沿って (m+1) ノード
(a に沿って (n+1) ノード
= (m+1)*(n+1) 合計ノード

面:
b に沿って m 要素 
a に沿って n 要素 
= 2*m*n 合計面 -> 各長方形の面に2つの三角形

y軸に沿った面は (2*要素番号) と 
(2*要素番号 - 1) で取得でき、これは x軸の各要素に対して繰り返されます。

奇数の面は底で直角の三角形です、
  /|
 /_|      または左右反転

 偶数は上部にあります
  _      
 | /    
 |/        または左右反転
```

:gmsh - Delaunay三角形分割を使用して、非構造化メッシュを返します。

kwarg（まだ実装されていません）：boundary_only - true の場合、長方形の境界のメッシュを返します。

また、gmsh関数 - gmshrectangle も参照してください。
