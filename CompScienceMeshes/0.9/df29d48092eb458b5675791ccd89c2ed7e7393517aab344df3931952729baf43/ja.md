```
restriction(submesh, supermesh)
```

サブメッシュ `submesh` に対する制限行列を計算します。これはスーパーメッシュ `supermesh` のものです。

制限行列のサイズは `(m,n)` であり、ここで

```
m == numcells(submesh)
n == numcells(supermesh)
```

行列の位置 `[i,j]` に `1` があるのは、サブメッシュのセル `i` がスーパーメッシュのセル `j` と等しい場合のみです。残りのエントリはゼロです。

この行列は制限行列と呼ばれます。なぜなら、スーパーメッシュのセルで取得されたサンプルの配列に作用すると、サブメッシュに保持されているセルのサンプルを選択し、再番号付けを考慮するからです。同様に、その転置は時々ゼロによる拡張演算子と呼ばれます。なぜなら、サブメッシュのセルで取得されたサンプルの配列を、サブメッシュで保持されなかったセルにゼロを挿入することによってスーパーメッシュのセルで取得されたサンプルの配列にマッピングするからです。
