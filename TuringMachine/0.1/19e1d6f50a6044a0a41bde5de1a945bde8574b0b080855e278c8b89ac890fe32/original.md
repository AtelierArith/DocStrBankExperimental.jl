```
load_input(file_name)
```

Takes a text file containing a continuous string of `n` integers (no spaces) and returns an `n`-element array, where each element is an integer. The array is ordered as the input string.

Input

  * `file_name`::string : path of text file containing an input string of `n` integers (no spaces)

Output

  * ::`n`-element Array{Char, 1} : output array of integers, ordered as the input string
