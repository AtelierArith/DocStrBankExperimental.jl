動的な「ブランチ」を表すために使用される型。ブランチは、定義域 `X=(a,b)` を持つマップ `f` によって表されます。X[1] と X[2] は a,b の区間包絡です。

このマップは [a,b] 上で単調でなければなりません。これは、hull(X[1], X[2]) 上で単調であることとは異なることに注意してください。例えば、[√2, 1] 上のマップ x → (x-√2)^2 を考えてみてください：左端点 X[1] は prevfloat(√2)..nextfloat(√2) になりますが、その場合、マップは hull(X[1], X[2]) 全体では単調ではありません。なぜなら、それは √2 の左側にある点も含むからです。これは対処しなければならない厄介なケースです。

f(a), f(b) のための包絡 `Y[1], Y[2]` と `increasing` が提供される場合があります（例えば、`Y=(0,1)` であることがわかっている場合）。そうでない場合は、自動的に計算されます。
