```
breakpoint(f, [sig], [line], [condition])
```

`f`に指定された引数の型`sig`でブレークポイントを追加します。`sig`が指定されていない場合、ブレークポイントは`f`のすべてのメソッドに適用されます。`f`がメソッドである場合、ブレークポイントはそのメソッドのみに適用されます。オプションで、ソースファイル内の絶対行番号`line`を指定できます。デフォルトでは、ボディの最初の行に入るときにブレークします。`condition`がない場合、ブレークポイントは遭遇するたびにトリガーされます。`condition`が`true`に評価される場合のみ、2回目にトリガーされます。`condition`は、`f`の引数とローカル変数に基づいて記述する必要があります。

# 例

```julia
function radius2(x, y)
    return x^2 + y^2
end

breakpoint(radius2, Tuple{Int,Int}, :(y > x))
```
