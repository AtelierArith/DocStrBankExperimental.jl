南アフリカ マクロ経済データベース API

南アフリカの公的マクロ経済データにアクセスするためのJulia APIで、南アフリカ準備銀行（SARB）および南アフリカ統計局（STATSSA）から取得され、EconData (https://www.econdata.co.za/) プラットフォームを介して毎週更新され、SARBおよびSTATSSAのウェブサイトからの自動スクレイピングによって維持されています。このデータベースは、ステレンボッシュ大学の経済学部で管理されています。このパッケージは、DataFrames.jlライブラリを中心に構築されています。その内容は以下のように要約されます。

**利用可能なデータに関する情報を提供するデータセット**

`datasources()`   - データソース

`datasets()`      - データセット

`series()`        - シリーズ（データセットによってクエリ可能）

**データベースからデータを取得する**

`data()`          - デフォルトですべてのデータを取得

**データを再形成し、時間的識別子を追加するための関数**

`pivot_wider()`   - DataFrames.unstack()のラッパー

`pivot_longer()`  - DataFrames.stack()のラッパー

`expand_date()`   - 日付から年、四半期、月、日列を作成

**入力を日付文字列に変換するためのヘルパー関数**

`as_date()`       - 例: "2011M01" -> "2011-01-01"

**識別子を持つ文字列ベクター**

`SAMADB_ID`       - 横断的識別子

`SAMADB_T`        - 時間的識別子
