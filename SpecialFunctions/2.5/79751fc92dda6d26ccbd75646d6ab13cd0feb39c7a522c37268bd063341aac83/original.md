```
besselkx(nu, x)
```

Scaled modified Bessel function of the second kind of order `nu`, $K_\nu(x) e^x$.

External links: [DLMF 10.25.3](https://dlmf.nist.gov/10.25.3), [Wikipedia](https://en.wikipedia.org/wiki/Bessel_function#Modified_Bessel_functions:_I%CE%B1,_K%CE%B1)
