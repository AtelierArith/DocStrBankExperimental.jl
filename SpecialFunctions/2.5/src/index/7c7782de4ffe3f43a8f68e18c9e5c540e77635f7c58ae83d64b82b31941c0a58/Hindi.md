# SpecialFunctions.jl

[`SpecialFunctions.jl`](https://github.com/JuliaMath/SpecialFunctions.jl/) विशेष कार्यों का एक व्यापक संग्रह प्रदान करता है जो [OpenSpecFun](https://github.com/JuliaLang/openspecfun) और [OpenLibm](https://github.com/JuliaLang/openlibm) पुस्तकालयों पर आधारित है।

जूलिया में विशेष गणितीय कार्यों में बेस्सेल, हैंकेल, एयरि, त्रुटि, डॉसन, घातांक (या साइन और कोसाइन) समाकल, एटा, ज़ेटा, डिगामा, उल्टा डिगामा, ट्रिगामा, और पॉलीगामा कार्य शामिल हैं।

## Installation

पैकेज का नवीनतम संस्करण जूलिया संस्करण 1.3 और उससे ऊपर के लिए उपलब्ध है। इसे स्थापित करने के लिए, जूलिया REPL में निम्नलिखित चलाएँ:

```julia
import Pkg; Pkg.add("SpecialFunctions")
```
