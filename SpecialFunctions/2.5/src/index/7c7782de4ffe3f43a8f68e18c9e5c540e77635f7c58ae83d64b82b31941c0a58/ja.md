# SpecialFunctions.jl

[`SpecialFunctions.jl`](https://github.com/JuliaMath/SpecialFunctions.jl/) は、[OpenSpecFun](https://github.com/JuliaLang/openspecfun) および [OpenLibm](https://github.com/JuliaLang/openlibm) ライブラリに基づいた特別な関数の包括的なコレクションを提供します。

Juliaにおける特別な数学関数には、ベッセル関数、ハンケル関数、エアリー関数、誤差関数、ドーソン関数、指数関数（または正弦および余弦）積分、エータ関数、ゼータ関数、ディガンマ関数、逆ディガンマ関数、トリガンマ関数、およびポリガンマ関数が含まれます。

## Installation

パッケージの最新バージョンは、Julia バージョン 1.3 以降で利用可能です。インストールするには、Julia REPL で以下を実行してください：

```julia
import Pkg; Pkg.add("SpecialFunctions")
```
