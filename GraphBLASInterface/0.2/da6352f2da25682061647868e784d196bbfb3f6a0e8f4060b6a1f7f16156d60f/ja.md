```
GrB_eWiseMult(C, mask, accum, op, A, B, desc)
```

要素ごとの行列およびベクトル操作のための一般的なメソッド：集合の交差を使用。

`GrB_eWiseMult` は `C<Mask> = accum (C, A .* B)` を計算します。ここで、2つの行列（またはベクトル）の要素のペアが対ごとに「乗算」され、C(i, j) = mult (A(i, j), B(i, j)) となります。「乗算」演算子は任意の二項演算子にすることができます。結果 T = A .* B のパターンは、A と B の集合の交差（和ではなく）です。交差の外にあるエントリは計算されません。これは `GrB_eWiseAdd` との主な違いです。入力行列 A および/または B は、記述子を介して最初に転置される場合があります。半環の場合、乗算演算子は半環の乗算演算子です。これは、半環の加算演算子を使用する eWiseAdd メソッドとは異なります。
