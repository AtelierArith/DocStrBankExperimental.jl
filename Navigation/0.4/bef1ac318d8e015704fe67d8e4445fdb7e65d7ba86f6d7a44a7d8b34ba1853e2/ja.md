```
intermediate_point(pos₁::Point, pos₂::Point[, fraction::Float64 = 0.5])
```

`intermediate_point` [deg] を、位置 `pos₁` と `pos₂` [deg] の間の大円経路に沿った任意の `fraction` で返します。大円経路に沿った割合は、`fraction` = 0.0 が `pos₁` [deg] にあり、`fraction` 1.0 が `pos₂` [deg] にあるように設定されています。

出典: www.movable-type.co.uk/scripts/latlong.html
