```
Dudt
```

[`ConditionType`](@ref) のインスタンスで、エッジまたはポイントが `Dudt` タイプの境界条件を持つことを宣言するために使用されます。`Dudt` タイプの条件は次の形を取ります。

$$
\dv{u(x, y, t)}{t} = a(x, y, t, u).
$$

`Dudt` 条件を提供する際、提供する関数は次の形を取ります。

```
a(x, y, t, u, p)
```

ここで、`(x, y)` はポイント、`t` は現在の時間、`u` は上記のように時間 `t` におけるポイント `(x, y)` での解であり、追加のパラメータのための引数 `p` が含まれています。
