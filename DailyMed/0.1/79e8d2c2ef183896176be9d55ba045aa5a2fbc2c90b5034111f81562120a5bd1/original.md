```
`drugclasses(; extra = [])`
```

Returns a list of all drug classes associated with at least one SPL in the Pharmacologic Class Indexing Files.

extra is optional. If provided it should be a `Dict` or list of string `Pair`s, and can be "drug*class*code", "drugclass*coding*system", "code*class*type", "class*name", "unii*code", "pagesize", "page"
