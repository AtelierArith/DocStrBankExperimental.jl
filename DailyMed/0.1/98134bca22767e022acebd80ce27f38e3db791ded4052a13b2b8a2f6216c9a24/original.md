```
function spls(; extra = [])
```

Returns a list of all SPLs.

extra is optional. If provided it should be a `Dict` or list of string `Pair`s, and can be "application*number", "boxed*warning", "dea*schedule*code", "doctype", "drug*class*code", "drugclass*coding*system", "drug*name", "name*type", "labeler", "manufacturer", "marketing*category*code", "ndc", "published*date", "published*date*comparison", "rxcui", "setid", "unii*code", "pagesize", "page"
