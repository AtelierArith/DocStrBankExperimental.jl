```
function spls(; extra = [])
```

すべてのSPLのリストを返します。

extraはオプションです。提供される場合、それは`Dict`または文字列の`Pair`のリストである必要があり、"application*number"、"boxed*warning"、"dea*schedule*code"、"doctype"、"drug*class*code"、"drugclass*coding*system"、"drug*name"、"name*type"、"labeler"、"manufacturer"、"marketing*category*code"、"ndc"、"published*date"、"published*date*comparison"、"rxcui"、"setid"、"unii*code"、"pagesize"、"page"のいずれかであることができます。
