```
rescale_magnitude(localaniso; r1, r2, r3, clip=[0.05,0.95])
rescale_magnitude!(localaniso; r1, r2, r3, clip=[0.05,0.95])
```

`localaniso`の大きさの値を最小-最大スケーリングを使用して希望の制限に再スケーリングします。`r1`、`r2`、`r3`はそれぞれの主軸に沿った範囲を指します。`clip`は最小および最大の制限が定義される分位数を定義します。デフォルトは[0.05,0.95]に設定されており、外れ値の影響を避けます。

## 例

以下の例では、3-D局所異方性オブジェクトが再スケーリングされます。`r1`はすべての値が1.0に設定されています。`r2`と`r3`はそれぞれ(0.2, 1.0)と(0.1, 0.5)の間に再スケーリングされます。つまり、範囲=10のSphericalVariogramに関連付けると、range1はすべての場所で10に等しくなります。range2は2と10の間で変動します。そして、range3は1と5の間で変動します。

```julia
example = rescale_magnitude(localaniso3d, r1=1, r2=(0.2,1.0), r3=(0.1,0.5))
```
