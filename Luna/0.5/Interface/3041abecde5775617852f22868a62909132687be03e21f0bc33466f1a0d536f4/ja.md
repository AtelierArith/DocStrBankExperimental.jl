```
prop_gnlse(γ, flength, βs; λ0, λlims, trange, kwargs...)
```

GNLSEを使用してパルス伝播をシミュレートします。

# 必須引数

  * `γ::Number`: 非線形係数。
  * `flength::Number`: 光ファイバの長さ。
  * `βs`: `λ0`の周りの伝播定数のテイラー展開。
  * `λ0`: (キーワード引数) シミュレーションの基準波長。単純な単一パルス入力の場合、これは入力パルスの中心波長でもあります。
  * `λlims::Tuple{<:Number, <:Number}`: シミュレーショングリッドの波長制限。
  * `trange::Number`: 時間グリッドの全幅。サンプル数を2の累乗にするために、実際に使用されるグリッドは通常大きくなります。

# グリッドオプション

  * `δt::Number`: 非線形相互作用に使用される細かいグリッドの時間ステップ。デフォルトでは、これは波長グリッドによって決定されます。`δt`が指定され、**必要な値より小さい**場合は、代わりに使用されます。

# 入力パルスオプション

単一のパルスは、以下のキーワード引数によって指定できます。より複雑な入力は、単一の`AbstractPulse`または`Vector{AbstractPulse}`によって定義できます。この場合、`λ0`以外のすべてのキーワード引数は無視されます。現在のGNLSEモデルは単一モードのみであることに注意してください。

  * `λ0`: 中心波長
  * `τfwhm`: 半値全幅で定義されるパルスの持続時間。
  * `τw`: 「自然な」パルス持続時間。パルス形状が`sech`の場合のみ利用可能。
  * `ϕ`: 変換制限されたパルスに適用されるスペクトル位相。要素は通常の多項式位相ϕ₀ (CEP)、ϕ₁ (群遅延)、ϕ₂ (GDD)、ϕ₃ (TOD)などです。
  * `energy`: パルスエネルギー。
  * `power`: **スペクトル位相が追加された後の**ピークパワー。
  * `pulseshape`: 変換制限されたパルスの形状。ガウスパルスの場合は`:gauss`、sech²パルスの場合は`:sech`を指定できます。
  * `polarisation`: 入力パルスの偏光。`:linear`（デフォルト）、`:circular`、または楕円率-1 ≤ ε ≤ 1の数値を指定できます。ここで、ε=-1は左円偏光、ε=1は右円偏光、ε=0は線形偏光に対応します。楕円偏光の主軸は常にy軸です。
  * `propagator`: シミュレーション開始前にパルスに任意の伝播を適用するために、最初の引数を**変異**させる関数`propagator!(Eω, grid)`。
  * `shotnoise`: `true`（デフォルト）の場合、モードごとの1光子量子ノイズが含まれます。

# GNLSEオプション

  * `shock::Bool`: ショック導関数項を含めるかどうか。デフォルトは`true`。
  * `raman::Bool`: ラマン効果を含めるかどうか。デフォルトは`true`。
  * `ramanmodel`; 使用するラマンモデル。デフォルトは`:sdo`で、これはシリカに一般的に使用される値にデフォルト設定された単純な減衰振動子モデルを使用します。`ramanmodel`は、HollenbeckとCantrellのより高度なモデルを使用する`:SiO2`に設定することもできます。
  * `loss`: パワーロス[dB/m]。デフォルトは0。
  * `fr`: `γ`へのラマン寄与の割合。デフォルトは`fr = 0.18`。
  * `τ1`: ラマン振動子の周期。
  * `τ2`: ラマン減衰時間。

# 出力オプション

  * `saveN::Integer`: フィールドを保存するz方向のポイント数。
  * `filepath`: `nothing`（デフォルト）の場合、シミュレーション結果を作業メモリ内の`MemoryOutput`にのみ保存します。`nothing`でない場合は、`String`としてのファイルパスであり、結果はこの場所のファイルに保存されます。`scan`が渡された場合、`filepath`はスキャンの出力**ディレクトリ**を決定します。
  * `scan`: パラメータスキャンを定義する`Scan`インスタンス。`scan`が指定された場合、`Output.ScanHDF5Output`が使用されてスキャンの出力ファイルが自動的に命名され、生成されます。`scanidx`も指定する必要があります。
  * `scanidx`: 実行中のスキャン内の現在のスキャンインデックス。`scan`が渡された場合のみ使用されます。
  * `filename`: パラメータスキャンを実行する際にスキャン名を上書きするために使用できます。実行中の`scanidx`がこのファイル名に追加されます。`scan`が指定されていない場合は無視されます。
  * `status_period::Number`: 印刷されたステータス更新の間隔（秒単位）。
