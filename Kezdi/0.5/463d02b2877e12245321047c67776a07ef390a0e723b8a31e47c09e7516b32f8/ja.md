```
@reshape long y1 y2 ... i(varlist) j(var) 
@reshape wide y1 y2 ... i(varlist) j(var)
```

データフレームをワイド形式からロング形式、またはロング形式からワイド形式に変換します。変換する変数は `y1`, `y2` などです。`i(var)` と `j(var)` は、変換されたデータフレームにおける行および列のインデックスを定義する変数です。

オプション `i()` には複数の変数を含めることができます（例：`i(var1, var2, var3)`）。オプション `j()` には1つの変数のみを含める必要があります。
