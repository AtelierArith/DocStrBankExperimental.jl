```
 ppeval(f, D...; dim::NTuple)
```

呼び出し可能な引数 `f` を `D` タプルの要素のスライスに対して評価します。

#### 引数

`f` は、`D` のスライスまたはブロードキャストされた要素を受け入れる任意の呼び出し可能なオブジェクトです。`f` から返される結果は、配列またはスカラーのいずれかでなければなりません。

`D` は任意の数の要素を持ち、要素は任意の型を持つことができます。`D` の要素が `dim` で指定された次元に沿った分散配列である場合、`D` の要素が分散されていない場合、その要素はデフォルトでブロードキャストされ、`f` のすべての評価に適用されます。

`dim` は、`D` の要素がスライスされる次元を指定する整数のタプルです。したがって、タプルの長さは `D` の引数の数と同じでなければなりません。デフォルトでは、分散配列は最後の次元に沿ってスライドします。値がゼロ以下の場合、要素は `f` のすべての評価にブロードキャストされます。

#### 結果

`ppeval` は、最初の `p` サイズが `f` の返り値のサイズに対応する次元 `p+1` の分散配列を返します。`ppeval` からの返り値の配列の最後の次元は、入力配列がスライスされる次元と同じ長さです。

#### 例

```jl
addprocs(Sys.CPU_THREADS)

using DistributedArrays

A = drandn((10, 10, Sys.CPU_THREADS), workers(), [1, 1, Sys.CPU_THREADS])

ppeval(eigvals, A)

ppeval(eigvals, A, randn(10,10)) # 2番目の引数をブロードキャスト

B = drandn((10, Sys.CPU_THREADS), workers(), [1, Sys.CPU_THREADS])

ppeval(*, A, B)
```
