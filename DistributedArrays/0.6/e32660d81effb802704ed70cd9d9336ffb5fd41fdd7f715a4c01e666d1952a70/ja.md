```
DArray(init, dims, [procs, dist])
```

分散配列を構築します。

パラメータ `init` は、インデックス範囲のタプルを受け取る関数です。この関数は、分散配列のローカルチャンクを割り当て、指定されたインデックスの初期化を行う必要があります。

`dims` は、分散配列の全体のサイズです。

`procs` は、使用するプロセスIDのベクトルをオプションで指定します。指定しない場合、配列はすべてのワーカープロセスに分散されます。通常、分散モードで実行している場合、すなわち nprocs() > 1 の場合、これはインタラクティブなJuliaプロンプトをホストしているプロセスには分散配列のチャンクが存在しないことを意味します。

`dist` は、分散配列が各次元でいくつのチャンクに分割されるべきかを指定する整数ベクトルです。

例えば、分散配列を作成し、値 `v` で埋める `dfill` 関数は次のように実装されています：

### 例

```jl
dfill(v, args...) = DArray(I->fill(v, map(length,I)), args...)
```
