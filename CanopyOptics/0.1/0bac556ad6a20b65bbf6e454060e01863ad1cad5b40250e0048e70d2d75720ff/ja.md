```
struct PigmentOpticalProperties{FT}
```

色素、水などの重要な吸収断面積を格納する構造体

# フィールド

  * `λ`: 波長 `[length]`
  * `nᵣ`: 葉の材料の屈折率
  * `Kcab`: クロロフィル (a+b) の特定吸収係数 `[cm² μg⁻¹]`
  * `Kcar`: カロテノイドの特定吸収係数 `[cm² μg⁻¹]`
  * `Kant`: アントシアニンの特定吸収係数 `[cm² nmol⁻¹]`
  * `Kb`: 茶色の色素の特定吸収係数 (任意単位)
  * `Kw`: 水の特定吸収係数 `[cm⁻¹]`
  * `Km`: 乾燥物質の特定吸収係数 `[cm² g⁻¹]`
  * `Kp`: タンパク質の特定吸収係数 `[cm² g⁻¹]`
  * `Kcbc`: 炭素ベースの成分の特定吸収係数 `[cm² g⁻¹]`
