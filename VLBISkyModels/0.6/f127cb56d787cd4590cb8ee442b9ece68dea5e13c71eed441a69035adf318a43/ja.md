```julia
CosineRingwFloor(r0, σ0, σ, ξσ, s, ξs, floor, x0, y0)

```

リングの中心に[`GaussDisk`](@ref)があり、CosineRingの`r0`と`σ0`に一致するコサインリングです。

これは、より基本的なVLBISkyModel画像コンストラクタの便利なコンストラクタです。これは次のように等価です。

```julia-repl
julia> CosineRing(r0, σ0, σ, ξσ, s, ξs, x0, y0) + floor*modify(GaussDisk(σ0/r0), Stretch(r0), Shift(x0, y0))
```

### 詳細

リングに楕円率を追加します。リングの半径`r0`は、長軸(a)と短軸(b)の長さの幾何平均として定義されます。すなわち、

r0 = √(a*b)。

楕円率`τ`はτ = 1-b/aで与えられます。

## 引数

  * `r0`: リングの幾何平均半径 (√ab)
  * `σ0`: ガウスリングの標準偏差
  * `σ` : リングの厚さの`N`次コサイン展開の振幅
  * `ξσ`: リングの厚さの`N`次コサイン展開の位相
  * `s` : 方位明るさの`M`次コサイン展開の振幅
  * `ξs`: 方位明るさの`M`次コサイン展開の位相
  * `floor`: 中心ガウスのフラックス。これはCosineRingに対して相対的です。
  * `x0`: リング中心の水平方向の位置
  * `y0`: リング中心の垂直方向の位置
