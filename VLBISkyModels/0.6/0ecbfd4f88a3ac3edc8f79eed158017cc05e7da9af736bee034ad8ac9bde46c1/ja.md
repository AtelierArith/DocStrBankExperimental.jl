```
@unpack_params a,b,c,... = m(p)
```

モデル `m` をドメイン `p` で評価したときに、パラメータ `a,b,c,...` を抽出します。これは基本的に次のように展開されるマクロです。

```julia
a = getparam(m, :a, p)
b = getparam(m, :b, p)
...
```

`DomainParams` 型に依存する可能性のあるモデルに対しては、このマクロを使用してパラメータを抽出する必要があります。

!!! warn
    この機能は実験的であり、公開された安定したAPIの一部とは見なされていません。
