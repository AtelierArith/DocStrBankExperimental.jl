```julia
EllipticalCosineRing(r0, σ0, σ, ξσ, τ, ξτ, s, ξs, x0, y0)

```

楕円コサインリング。これは、より基本的なVLBISkyModel画像コンストラクタの便利なコンストラクタです。これは次のように等価です。

```julia-repl
julia> modify(CosineRing(σ0/r0, σ/r0, ξσ, s, ξs .- ξτ),
            Stretch(r0*sqrt(1-τ), r0/sqrt(1-τ)),
            Rotate(ξτ),
            Shift(x0, y0)
        )
```

### 詳細

リングに楕円形を追加します。リングの半径 `r0` は、長軸 (a) と短軸 (b) の長さの幾何平均として定義されます。すなわち、

r0 = √(a*b)。

楕円率 `τ` は τ = 1-b/a で与えられます。

## 引数

  * `r0`: リングの幾何平均半径 (√ab)
  * `σ0`: ガウスリングの標準偏差
  * `σ` : リングの厚さの `N` 次コサイン展開の振幅
  * `ξσ`: リングの厚さの `N` 次コサイン展開の位相
  * `τ` : リングの非対称性 τ = 1-b/a
  * `ξτ`: 北の東側で測定された長軸
  * `s` : 方位の明るさの `M` 次コサイン展開の振幅
  * `ξs`: 方位の明るさの `M` 次コサイン展開の位相
  * `x0`: リング中心の水平方向の位置
  * `y0`: リング中心の垂直方向の位置
