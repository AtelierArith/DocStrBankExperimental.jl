```julia
struct ExtendedRing{T} <: VLBISkyModels.GeometricModel{T}
```

逆ガンマ分布に従う放射プロファイルを持つ対称拡張リング。

画像領域での式は次のように与えられます。

```
I(r,θ) = βᵅrᵅ⁻²exp(-β/r)/2πΓ(α)
```

ここで `α = shape` および `β = shape+1` です。

# 注意

これは主に非解析フーリエ変換の例として使用されます（複雑な表現を持っていますが）。

# フィールド

  * `shape`: 放射分布の形状

`T` が構築時に指定されていない場合、デフォルトで `Float64` になります。
