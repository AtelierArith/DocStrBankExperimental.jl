```
Cron(special::Symbol)
```

`Cron`の6つの引数の代わりに、次のいずれかの特別なシンボルが代わりに現れることがあります：

|   `special` |                                  意味 |
| -----------:| -----------------------------------:|
|   `:yearly` |          年に1回実行、`Cron(0,0,0,1,1,0)` |
| `:annually` |                      （`:yearly`と同じ） |
|  `:monthly` |      月に1回実行、`Cron(0,0,0,1,'*','*')` |
|   `:weekly` |      週に1回実行、`Cron(0,0,0,'*','*',1)` |
|    `:daily` |    日に1回実行、`Cron(0,0,0,'*','*','*')` |
| `:midnight` |                       （`:daily`と同じ） |
|   `:hourly` | 時間に1回実行、`Cron(0,0,'*','*','*','*')` |
|     `:none` |       決して繰り返さない、`Cron(0,0,0,0,0,0)` |

注意：Linux crontabの特別な`:reboot`はここではサポートされていません。

毎分実行するには、単に`Cron()`を使用してください。
