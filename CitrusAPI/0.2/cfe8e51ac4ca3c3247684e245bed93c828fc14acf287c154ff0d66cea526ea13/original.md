```
set_participant_properties!(client, survey_id, query, data)
```

Set properties of a participant of a remote survey with `survey_id`. `query` can either be a participant token id, or an array of participant properties.

See also: [https://api.limesurvey.org/classes/remotecontrol_handle.html#method_set_participant_properties](https://api.limesurvey.org/classes/remotecontrol_handle.html#method_set_participant_properties)
