```
mail_registered_participants(client, survey_id)
```

Send register emails for a remote survey with `survey_id` to all not invited, not reminded, not completed participants. Overwrite this behaviour by specifying `override_all_conditions`.

See also: [https://api.limesurvey.org/classes/remotecontrol_handle.html#method_mail_registered_participants](https://api.limesurvey.org/classes/remotecontrol_handle.html#method_mail_registered_participants)
