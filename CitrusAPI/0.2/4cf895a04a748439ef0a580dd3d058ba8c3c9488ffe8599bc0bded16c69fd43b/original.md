```
invite_participants(client, survey_id, token_ids; email=true)
```

Invite participants by `token_ids` in a remote survey with `survey_id`. If `email == true` only pending invites are sent. Otherwise invites are resent.

See also: [https://api.limesurvey.org/classes/remotecontrol_handle.html#method_invite_participants](https://api.limesurvey.org/classes/remotecontrol_handle.html#method_invite_participants)
