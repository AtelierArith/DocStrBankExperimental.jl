```
add_survey!(client, survey_id, title, language; format)
```

Adds an empty survey with `survey_id` , `title` and default language `language`. Optional parameter `format` sets the question appearance format of the survey.

See also: [https://api.limesurvey.org/classes/remotecontrol_handle.html#method_add_survey](https://api.limesurvey.org/classes/remotecontrol_handle.html#method_add_survey)
