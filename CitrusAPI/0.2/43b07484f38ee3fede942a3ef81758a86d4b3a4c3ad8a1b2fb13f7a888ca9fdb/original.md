```
add_participants!(client, survey_id, data; create_token=true)
```

Add participants to the remote survey with `survey_id`. `data` contains participant data to be added. If `create_token=true` (the default), access tokens will be automatically created.

See also: [https://api.limesurvey.org/classes/remotecontrol_handle.html#method_add_participants](https://api.limesurvey.org/classes/remotecontrol_handle.html#method_add_participants)
