```
add_group!(client, survey_id, title; description="")
```

Add a question group to remote survey with `survey_id`. Requires the specification of a question group `title`. Optionally a question group `description` can be added.

See also: [https://api.limesurvey.org/classes/remotecontrol_handle.html#method_add_group](https://api.limesurvey.org/classes/remotecontrol_handle.html#method_add_group)
