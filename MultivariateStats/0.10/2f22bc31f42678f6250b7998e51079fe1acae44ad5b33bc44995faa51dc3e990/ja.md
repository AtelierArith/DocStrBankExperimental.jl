```
ldacov(Cp, Cn, μp, μn)
```

与えられた共分散と平均ベクトルに基づいてLDAを実行します。タイプ[`LinearDiscriminant`](@ref)の線形判別関数を返します。

*パラメータ*

  * `Cp`: 正のクラスの共分散行列。
  * `Cn`: 負のクラスの共分散行列。
  * `μp`: 正のクラスの平均ベクトル。
  * `μn`: 負のクラスの平均ベクトル。

**注意:** コサインベクトルは、$\mathbf{w}'\boldsymbol{μ}_p + b = 1$ および $\mathbf{w}'\boldsymbol{μ}_n + b = -1$ となるようにスケーリングされています。
