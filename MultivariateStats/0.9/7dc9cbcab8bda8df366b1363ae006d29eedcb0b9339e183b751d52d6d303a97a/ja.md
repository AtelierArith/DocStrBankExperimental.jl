```
fit(PCA, X; ...)
```

行列 `X` に与えられたデータに対して PCA を実行します。`X` の各列は **観測** です。

**キーワード引数**

  * `method`: メソッドの選択:

      * `:auto`: `d < n` の場合は `:cov` を使用し、それ以外の場合は `:svd` を使用します (*デフォルト*)。
      * `:cov`: 共分散行列の分解に基づく。
      * `:svd`: 入力データの SVD に基づく。
  * `maxoutdim`: 出力次元、すなわち変換された空間の次元 (*min(d, nc-1)*)
  * `pratio`: 主成分空間に保存される分散の比率 (*0.99*)
  * `mean`: 平均ベクトルで、次のいずれかです。

      * `0`: 入力データはすでに中心化されています
      * `nothing`: この関数は平均を計算します (*デフォルト*)
      * 事前に計算された平均ベクトル

**注意:**

  * 出力次元 `p` は `maxoutdim` と `pratio` の両方に依存します。最初の `k` 主成分が全体の分散の少なくとも `pratio` を保存し、最初の `k-1` が `pratio` 未満を保存する場合、実際の出力次元は $\min(k, maxoutdim)$ になります。
  * この関数は、メソッドの選択に応じて内部的に [`pcacov`](@ref) または [`pcasvd`](@ref) を呼び出します。
