```
EndEffectorForces(thetalist, Ftip, Mlist, Glist, Slist)
```

エンドエフェクタの力 `Ftip` を生成するために、オープンチェーンロボットが必要とする関節の力/トルクを計算します。

# 引数

  * `thetalist`: 関節変数の $n$-ベクトル。
  * `Ftip`: フレーム `{n+1}` で表現されたエンドエフェクタによって適用される空間力。
  * `Mlist`: ホームポジションでの `i` に対するリンクフレームのリスト。
  * `Glist`: リンクの空間慣性行列 `Gi`。
  * `Slist`: 空間フレームにおける関節のねじ軸 `Si`、列として軸を持つ行列の形式。

エンドエフェクタの力 `Ftip` を生成するために必要な関節の力とトルクを返します。この関数は、`g = 0`、`dthetalist = 0`、および `ddthetalist = 0` で InverseDynamics を呼び出します。
