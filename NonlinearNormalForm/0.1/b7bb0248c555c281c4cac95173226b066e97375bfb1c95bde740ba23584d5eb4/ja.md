```
mat_eigen(mat; sort=true, phase_modes=true)
```

行/列の数が偶数の行列 `mat` が与えられたとき、固有ベクトルと固有値を計算します。安定した固有モード（複素共役の固有ベクトル/固有値ペア）については、固有ベクトルは `vⱼ'*S*vⱼ = +im` となるように正規化され、奇数の `j` に対しては +im、偶数の `j` に対しては -im になります。

`sort` が true の場合、各固有ベクトル/固有値ペアは、それが最も特定するモードに従ってソートされます。モードのソートが失敗した場合は警告が表示されます。1つ以上のモードが不安定な場合、モードのソートは自動的に失敗します。デフォルトは true です。

`phase_modes` が true の場合、各安定したモードは位相因子で乗算され、そのモードの最初の成分（例えば x、y、または z 成分、px、py、または pz 成分ではない）が完全に実数になるようにします。これにより、固有ベクトルが「美しく」なり、弱い結合の場合には、`vⱼ'*vⱼ₊₁` が j=(1, 3, 5) のときに正の虚部を持つことが保証され、奇数の `j` の固有ベクトル/固有値が正のチューンに関連付けられ、偶数の `j` が負のチューンに関連付けられます。この位相因子は、高度に結合された行列に含めることは無害で無意味です。

複素行列の場合、`mat_eigen` によって呼び出される Julia の `eigen` は型が不安定です。
