```
ArrowheadMatrix{O, T, Z, P} <: AbstractMatrix{O}
```

矢頭行列を表す構造体で、特別なタイプの疎行列です。

# フィールド

  * `α::T`: 行列の右下隅にあるスカラー値。
  * `z::Z`: 最後の行/列を表すベクトル（コーナー要素を除く）。
  * `D::P`: 対角要素を表すベクトル（コーナー要素を除く）。

# コンストラクタ

```
ArrowheadMatrix(a::T, z::Z, d::D) where {T,Z,D}
```

与えられた α, z, および D 値を持つ `ArrowheadMatrix` を構築します。出力タイプ `O` はすべての入力要素の昇格されたタイプとして自動的に決定されます。

# 操作

  * 行列-ベクトルの乗算: `A * x` または `mul!(y, A, x)`
  * 線形システムの解法: `A \ b` または `ldiv!(x, A, b)`
  * 密行列への変換: `convert(Matrix, A)`
  * 逆行列: `inv(A)` （`InvArrowheadMatrix` を返します）

# 例

```julia
α = 2.0
z = [1.0, 2.0, 3.0]
D = [4.0, 5.0, 6.0]
A = ArrowheadMatrix(α, z, D)

# 行列-ベクトルの乗算
x = [1.0, 2.0, 3.0, 4.0]
y = A * x

# 線形システムの解法
b = [7.0, 8.0, 9.0, 10.0]
x = A \ b

# 密行列に変換
dense_A = convert(Matrix, A)
```

# 注意事項

  * 行列は、α - dot(z ./ D, z) = 0 の場合、または D の任意の要素がゼロの場合に特異です。
  * 最良のパフォーマンスのために、可能な場合は線形システムの解法に `ldiv!` を使用してください。
