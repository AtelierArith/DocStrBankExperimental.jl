```
Contingency(P, renormalize = Val(true))
```

コンティンジェンシー分布は、カテゴリカル分布の多変量一般化です。二変量分布として、コンティンジェンシー分布は二つのユニットベクトル `v1` と `v2` に対する結合確率を定義します。パラメータ `P` は共起の確率を指定するコンティンジェンシー行列をエンコードします。

```
v1 ∈ {0, 1}^d1 ただし Σ_j v1_j = 1
v2 ∈ {0, 1}^d2 ただし Σ_k v2_k = 1

P ∈ [0, 1]^{d1 × d2}, ただし Σ_jk P_jk = 1

f(v1, v2, P) = Contingency(out1, out2 | P) = Π_jk P_jk^{v1_j * v2_k}
```

二つ以上の変数に対する `Contingency` 分布は、パラメータとして高次テンソルを必要とします。これらはReactiveMPでは実装されていません。

# 引数:

  * `P`、必須、コンティンジェンシー行列
  * `renormalize`、オプション、`Val(true)` または `Val(false)` のいずれかをサポートし、行列 `P` を自動的に再正規化する必要があるかどうかを指定します。元の `P` を変更せず、再正規化されたバージョンのために新しいものを割り当てます。`false` に設定された場合、コンティンジェンシー行列 `P` **は** 手動で正規化されなければならず、そうでないと関連する計算の結果が間違っている可能性があります。
