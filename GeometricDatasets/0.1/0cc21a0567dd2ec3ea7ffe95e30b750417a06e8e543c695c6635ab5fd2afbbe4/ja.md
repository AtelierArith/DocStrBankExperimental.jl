```
farthest_points_sample(
    X::PointCloud, n::Integer; 
    metric = Euclidean()
    )
```

与えられた `X` と整数 `n` に対して、互いにできるだけ遠く離れた点のサブセットを `X` から返します。

## 詳細

`X` を `k` 点を持つ距離空間とします。ランダムな点 `x_1` ∈ `X` を選択します。次に、与えられた距離に関して `x_1` から最も遠い点を `x_2` として選択します。その後、`x_1` と `x_2` の両方から最も遠い点を `x_3` として選びます。このようにして、`n` 点を得るまで点を選び続けます。
