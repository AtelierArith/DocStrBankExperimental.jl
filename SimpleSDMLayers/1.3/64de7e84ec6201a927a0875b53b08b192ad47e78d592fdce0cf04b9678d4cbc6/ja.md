```
SDMLayer{T}
```

地理空間情報のレイヤーを定義します。

この型には二つのデータフィールドがあります：

  * **grid**: 型 `T` の `Matrix`
  * **indices**: どの位置が値を持つかを示す `BitMatrix`

`grid` フィールドの各 *行* は、等しい *北方向* のラスタのスライスを表します。すなわち、情報は表示されたときに地図上で表現されるようにマトリックスに配置されています。同様に、列は同じ *東方向* を持ちます。

地理空間情報は三つの位置フィールドによって表されます：

  * **x** と **y**: *コーナー* の座標を示す二つのタプルで、*x* および *y* 次元（例：東方向/北方向）に沿っています - デフォルト値は `(-180., 180.)` と `(-90., 90.)` で、これは WGS84 における地球全体の表面を表します
  * **crs**: `Proj.jl` によって処理可能な CRS の任意の `String` 表現 - デフォルトは `"+proj=longlat +datum=WGS84 +no_defs"` で、これは緯度/経度座標系を表します
