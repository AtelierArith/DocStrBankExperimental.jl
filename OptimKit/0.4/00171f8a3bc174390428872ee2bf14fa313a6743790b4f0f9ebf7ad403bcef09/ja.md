```
struct HagerZhangLineSearch{T<:Real} <: AbstractLineSearch
HagerZhangLineSearch(; c₁::Real = 1//10, c₂::Real = 9//10, ϵ::Real = 1//10^6,
                    θ::Real = 1//2, γ::Real = 2//3, ρ::Real = 5//1)
```

指定されたパラメータを持つHager-Zhang線形探索オブジェクトを構築します。

## 引数:

  * `c₁::Real`: （近似的な）最初のWolfe条件（Armijoルール）のパラメータで、関数値の十分な減少を制御します: c₁ < 1/2 < c₂。デフォルトは`1//10`です。
  * `c₂::Real`: 第二のWolfe条件（曲率条件）のパラメータで、傾きの十分な減少を制御します: c₁ < 1/2 < c₂。デフォルトは`9//10`です。
  * `ϵ::Real`: 目的関数の期待される精度のパラメータで、関数値の最大許容増加を制御します。デフォルトは`1//10^6`です。
  * `θ::Real`: 二分探索ステップを調整するパラメータです。デフォルトは`1//2`（変更しない方が良いでしょう）。
  * `γ::Real`: 二分探索ステップをトリガーするパラメータで、すなわちブランケット縮小率が`γ`より遅い場合です。デフォルトは`2//3`です。
  * `ρ::Real`: 初期ブランケット拡張率を制御するパラメータです。デフォルトは`5//1`です。

## 戻り値:

このメソッドは`HagerZhangLineSearch`オブジェクト`ls`を返します。これはその後、`ls(fg, x₀, η₀; kwargs...)`として適用され、与えられた点で目的関数とその勾配を計算する関数`fg`の線形探索を実行します。`x₀`から方向`η₀`に沿って開始します。
