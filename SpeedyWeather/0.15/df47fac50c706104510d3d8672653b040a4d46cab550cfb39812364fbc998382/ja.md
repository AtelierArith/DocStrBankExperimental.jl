重力波の増幅を防ぐための半暗黙的補正のために、さまざまな事前計算された配列を保持する構造体。i-1とi+1の間の暗黙的時間ステップは、範囲[0, 1]のパラメータ`α`によって制御されます。

```
α = 0   は重力波項がi-1で評価されることを意味します（前方）
α = 0.5 はi+1とi-1で評価されます（中心暗黙的）
α = 1   はi+1で評価されます（後方暗黙的）
α ∈ [0.5, 1] も可能で、重力波の減衰の強さを制御します。
α = 0.5 は重力波を遅くし、増幅を防ぎます。
α > 0.5 は重力波を数日から数タイムステップ（α=1）で減衰させます。
```

フィールドは次のとおりです。

  * `α::Any`: 重力波をフィルタリングするための半暗黙的計算の係数、0.5 <= α <= 1
  * `gravity::Base.RefValue`: 重力 [m/s²]、initialize!時にmodel.planetから取得
  * `layer_thickness::Base.RefValue`: 層の厚さ [m]、initialize!時にmodel.atmosphereから取得
  * `time_step::Base.RefValue`: 時間ステップ [s]、= αdt = 2αΔt（リープフロッグ用）
