温度フラックス長波放射は、JeevanjeeとRomps（2018）によるもので、SeeleyとWordsworth（2023）の式（1）に従います。

```
dF/dT = α*(T_t - T)
```

ここで、`F`は温度差`dT`を持つ二つの層間の上向き温度フラックス、`α = 0.025 W/m²/K²`、`T_t = 200K`は指定された成層圏温度です。最下層へのフラックスは0です。最上層からのフラックスも0ですが、`dT/dt = (T_t - T)/τ`が追加され、最上層を時間スケール`τ = 24h`で成層圏温度`T_t`に向かって緩和させます（SeeleyとWordsworth（2023）は6hを使用していますが、ここでは低解像度で不安定です）。フィールドは以下の通りです。

  * `α::Any`: 放射強制定数 (W/m²/K²)
  * `temp_tropopause::Any`: 成層圏温度 [K]
  * `time_scale::Dates.Second`: 成層圏温度`temp_tropopause`への緩和時間スケール
