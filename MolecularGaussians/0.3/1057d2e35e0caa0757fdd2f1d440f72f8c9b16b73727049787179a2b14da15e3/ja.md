# MolecularGaussians.jl

MolecularGaussians.jlは、分子とそのファーマコフォア機能をガウス混合モデルとしてモデル化することによって整列させるために使用されるパッケージです。これは、分子間の整列、重なり、および距離を計算するためにGaussianMixtureAlignment.jlを利用します。

# REPLヘルプ

?の後にアルゴリズムまたはコンストラクタ名を続けると、ヘルプがターミナルに表示されます。次を参照してください：

```
	?MolGMM 

	?PharmacophoreGMM 

	?gogma_align 

	?tiv_gogma_align 

	?rocs_align
```
