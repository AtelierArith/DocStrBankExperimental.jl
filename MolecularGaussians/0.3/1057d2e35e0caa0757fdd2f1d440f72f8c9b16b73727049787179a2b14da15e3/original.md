# MolecularGaussians.jl

MolecularGaussians.jl is a package used to align molecules and their pharmacophore features by modeling them as Gaussian mixture models. It makes use of GaussianMixtureAlignment.jl to compute alignments, overlap, and distances between molecules.

# REPL help

? followed by an algorith or constructor name will print help to the terminal. See: 

```
	?MolGMM 

	?PharmacophoreGMM 

	?gogma_align 

	?tiv_gogma_align 

	?rocs_align
```
