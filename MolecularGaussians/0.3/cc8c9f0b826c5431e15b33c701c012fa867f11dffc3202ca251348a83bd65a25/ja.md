```
model = PharmacophoreGMM(mol, σfun=vdwvolume_sigma, ϕfun=ones, nodes=nodeset(mol), features=pubchem_features)
```

分子またはサブグラフ `mol` から一連のガウス混合モデルを作成します。各モデルは特定のタイプの分子特徴（例：環構造）に対応します。

オプションで、`σfun` と `ϕfun` の関数を提供することができ、これらは `mol` を入力として受け取り、ノードインデックスを分散 `σ` とスケーリング係数 `ϕ` にマッピングする辞書を返します。

`nodes` が提供される場合、ガウス混合モデルは分子のグラフのノードインデックスに対応する原子からのみ構築されます。

`features` は許可される分子特徴のキーを指定します。他のキーを持つ特徴は無視されます。

`directional` は、環構造および水素結合供与体の幾何学的制約が含まれるかどうかを指定します。
