```
FENodeToFEMap
```

有限要素ノードからそれらを接続する有限要素へのマップ。

入力の有限要素セットの接続性で参照される各ノードについて、そのノードを参照する個々の有限要素の番号が配列 `map` に格納されます。

```
    例:
```

```
fes.conn= [7,6,5;
            4,1,3;
            3,7,5];
マップは次のように読み取ります
    map[1] = [2];
    map[2] = [];#  ノード番号2は接続性によって参照されていないことに注意
    map[3] = [2,3];
    map[4] = [2];
    map[5] = [1,3];
    map[6] = [1];
    map[7] = [1,3];
```

ノード番号5を参照する接続性の個々の要素は1と3であるため、`fes.conn(map[5],:)`はノード5に接続されているすべてのノード（ノード5自体を含む）を含みます。
