```
filtered_probs(model::MSM; kwargs...)
```

各時間帯における各状態のフィルタリングされた確率を返します。モデルのみが提供された場合、インサンプルデータが使用されます。

フィルタリングされた確率は、スムーズ化された確率とは異なり、時刻Tまでに利用可能なデータを使用して計算されます。

# 引数

  * `model::MSM`: 推定されたモデル。
  * `y`: 従属変数のためのオプションデータ。
  * `exog_vars`: モデルの非スイッチング部分のためのオプションの外生変数。
  * `exog_switching_vars`: モデルのスイッチング部分のためのオプションの外生変数。
  * `exog_tvtp`: モデルのtvtp部分のためのオプションの外生変数。

[`smoothed_probs`](@ref)および[`expected_duration`](@ref)も参照してください。
