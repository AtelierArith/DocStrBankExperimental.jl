```
Yindex(ℓ, m, ℓₘᵢₙ=0)
```

モード重みの配列へのインデックスを計算します

## パラメータ

ℓ : int     ℓₘᵢₙ <= ℓ <= ℓₘₐₓ を満たす整数 m : int     -ℓ <= m <= ℓ を満たす整数 ℓₘᵢₙ : int, オプション     0 <= ℓₘᵢₙ を満たす整数。デフォルトは 0。

## 戻り値

i : int     モード重み配列の特定の要素のインデックス

## 関連項目

Ysize : モード重みの配列の総サイズ Yrange : この配列に対応する (ℓ, m) インデックスの配列

## 注意事項

これは、モードが (固定された s 値で) 次のように配置されていると仮定します

```
[
    Y(s, ℓ, m)
    for ℓ ∈ ℓₘᵢₙ:ℓₘₐₓ
    for m ∈ -ℓ:ℓ
]
```
