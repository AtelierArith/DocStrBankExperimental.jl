```
WignerHrange(ℓₘₐₓ, m′ₘₐₓ=ℓₘₐₓ)
```

H配列のように(ℓ, m', m)インデックスの配列を作成します

参照してください [`WignerHsize`](@ref) と [`WignerHindex`](@ref)

# 注意事項

ここでは、m≥|m'|のデータのみが保存され、対応する値のみが渡されると仮定しています。また、|m|≤ℓ および |m'|≤ℓ も仮定しています。これらはチェックされません。この関数がインデックスを付けるウェッジ配列は次のように順序付けられています。

```
[
    H(ℓ, m′, m) for ℓ ∈ 0:ℓₘₐₓ
    for m′ ∈ -min(ℓ, m′ₘₐₓ):min(ℓ, m′ₘₐₓ)
    for m ∈ abs(m′):ℓ
]
```
