```
d_matrices!(d_storage, β)
d_matrices!(d_storage, expiβ)
d_matrices!(d, β, ℓₘₐₓ)
d_matrices!(d, expiβ, ℓₘₐₓ)
```

ウィグナーの $d^{(\ell)}$ 行列を、すべての $\ell \leq \ell_\mathrm{max}$ に対して要素 $d^{(\ell)}_{m',m}(\beta)$ を計算します。 $d$ 行列は、完全な $\mathfrak{D}$ 行列と対比して「縮小」ウィグナー行列と呼ばれることがあります。

すべての場合において、結果は次のように順序付けられた1次元配列で返されます。

```
[
    dˡₘₚ,ₘ(β)
    for ℓ ∈ 0:ℓₘₐₓ
    for mp ∈ -ℓ:ℓ
    for m ∈ -ℓ:ℓ
]
```

最初の引数が `d` の場合、それは変更されるため、その配列と同じかそれ以上の大きさでなければなりません。最初の引数が `d_storage` の場合、それは [`d_prep`](@ref) によって返される量であり、結果はそのタプルの `d` フィールドに書き込まれます。これらのオプションの両方、特に後者は、対応する関数への各呼び出しで必要な割り当ての数を減らし、速度を大幅に向上させるはずです。

!!! warn
    `d_storage` 引数を使用する場合（推奨）、返される量 `d` は `d_storage[1]` のエイリアスになります。次の [`d_matrices!`](@ref) の呼び出し後にそのデータを保持したい場合は、`copy(d)` でコピーする必要があります。


単一の値の `β` または `expiβ` に対して行列を評価するだけの場合は、より簡単な関数呼び出しのために [`d_matrices`](@ref) も参照してください。

# 例

```julia
using SphericalFunctions
ℓₘₐₓ = 8
T = Float64
β = T(1)/8
d_storage = d_prep(ℓₘₐₓ, T)
d = d_matrices!(d_storage, β)
```
