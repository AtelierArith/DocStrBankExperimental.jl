```
ncon(tensorlist, indexlist, [conjlist, sym]; order = ..., output = ..., backend = ..., allocator = ...)
```

`tensorlist`（`Vector`または`Tuple`型）のテンソルを`indexlist`で指定されたネットワークに従って収束させます。ここで、`indexlist`は`tensorlist`と同じ長さのリスト（すなわち`Vector`または`Tuple`）であり、そのエントリはそれぞれリスト（できれば`Vector{Int}`）で、各整数エントリは`tensorlist`内の対応するテンソルの対応するインデックス/次元のラベルを提供します。正の整数は収束させる必要があるインデックスをラベル付けするために使用され、したがって`indexlist`内の2つの異なるエントリに現れます。一方、負の整数は出力テンソルのインデックスをラベル付けするために使用され、1回だけ現れるべきです。

同じ長さの別のリストにあるオプションの引数`conjlist`は、`Bool`型であり、対応するテンソルオブジェクトを共役させるべきか（`true`）そうでないか（`false`）を示します。デフォルトはすべてのエントリに対して`false`です。

デフォルトでは、収束は収束させるインデックスが増加する整数でラベル付けされる順序で行われます。すなわち、最初にラベル`1`に対応する収束が行われます。出力テンソルは、減少する（負のため絶対値で増加する）インデックスラベルに対応するインデックス順序を持ちます。キーワード引数`order`と`output`はこれらのデフォルトを変更することを可能にします。
