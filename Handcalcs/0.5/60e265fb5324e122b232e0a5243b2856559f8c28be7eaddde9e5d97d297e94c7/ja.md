```
set_default(; kwargs...)
```

handcalcsのデフォルトkwarg値を設定します。

これはすべてのキーワード引数に対して機能します。複数回呼び出すと、デフォルトが追加または置き換えられますが、リセットはされません。

## Kwargs

  * `cols`: 出力の列数を設定します
  * `spa`: 行間を設定します
  * `len`: `:long`に設定すると、方程式を複数行に分割します
  * `color`: 出力の色を変更します（`:blue`, `:red`など）
  * `precision`: 数字を最大精度でフォーマットします。`precision = 2`の場合、`2.567`は`2.57`として表示され、`2.5`は`2.5`として表示されます
  * `h_env`: "aligned"（デフォルト）、"align"および他のLaTeXオプションの中から選択します
  * `not_funcs`: "unroll"したくない関数の名前を指定します
  * `parse_pipe`: 方程式からパイプを削除するためのブール値（デフォルト=true）。これは単位付き方程式を対象としています。
  * `parse_ifs`: if文を展開するためのブール値（デフォルト=true）。関数の展開が機能し、満たされたif文の部分のみが表示されます。
  * `disable`: シミュレーションを実行するためにhandcalcsのレンダリングを無効にし、必要に応じて再度有効にします。

例:

```julia
set_handcalcs(cols = 2, spa = 5)
```

設定したデフォルトをリセットするには、`reset_handcalcs`を使用します。指定したデフォルトを確認するには、`get_handcalcs`を使用します。
