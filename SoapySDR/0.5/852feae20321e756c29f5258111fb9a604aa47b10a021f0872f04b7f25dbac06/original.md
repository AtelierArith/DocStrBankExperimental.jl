```
Device
```

A device is a collection of SDR channels, obtained from the `Devices()` list.

Fields:

  * `info`
  * `driver`
  * `hardware`
  * `tx`
  * `rx`
  * `sensors`
  * `time_source`
  * `time_sources`
  * `clock_source`
  * `clock_sources`
  * `frontendmapping_rx`
  * `frontendmapping_tx`
