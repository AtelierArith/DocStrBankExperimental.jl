```
init_telemetry_source([::Type{T}, vargs...; kwargs...]) where T<:TelemetrySource -> Union{T, Nothing}
```

テレメトリソース `T` を初期化します。すべての引数とキーワードが省略された場合、関数は対話的にソースを選択します。

ソースを初期化するための引数 `vargs...` とキーワード `kwargs...` は、ソースタイプによって異なります。

この関数は、初期化が成功した場合は型 `T` のオブジェクトを返し、そうでない場合は `nothing` を返します。

!!! note
    ソースが正しく初期化されると、それがデフォルトソースとして選択されます。
