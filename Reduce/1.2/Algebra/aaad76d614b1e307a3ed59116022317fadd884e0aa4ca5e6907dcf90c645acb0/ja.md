```
factor(r...)
```

この宣言は、識別子またはカーネルのリストを引数として受け取ります。`factor`は因数分解コマンドではなく（この目的には`factorize`または`factor`スイッチを使用してください）、むしろ分離コマンドです。宣言された式の固定された冪に関わるすべての項は、固定された冪の積と残りの項の和として印刷されます。

例えば、次の宣言の後に

```Julia
julia> Algebra.factor(:x)
```

多項式 $(x + y + 1)^2$ は次のように印刷されます。

```
         2                  2  
        x  + 2*x*(y + 1) + y  + 2*y + 1
```

特定の接頭辞演算子に関わるすべての式も、演算子名を因数分解された識別子のリストに入れることで因数分解できます。例えば：

```Julia
julia> Algebra.factor(:x,:cos,:(sin(x))
```

は、すべての$x$の冪と$sin(x)$および$cos$のすべての関数を因数分解します。
