```
@switchlang!(lang)
```

ドキュメント文字列の翻訳のためのターゲット言語を切り替え、ドキュメントシステムを修正してドキュメント文字列を自動的に翻訳します。

# 引数

  * `lang`: ドキュメント文字列を翻訳するためのターゲット言語コード（例："ja", "en"）。

# 詳細

このマクロは以下の操作を実行します：

1. 翻訳のためのターゲット言語を設定します
2. 個々のドキュメント文字列のために翻訳エンジンを挿入するために `Docs.parsedoc` をオーバーライドします
3. ドキュメント全体のページの翻訳を可能にするために `Documenter.Page` コンストラクタをオーバーライドします

# 例

```julia
@switchlang! "ja"  # 日本語翻訳に切り替え
```
