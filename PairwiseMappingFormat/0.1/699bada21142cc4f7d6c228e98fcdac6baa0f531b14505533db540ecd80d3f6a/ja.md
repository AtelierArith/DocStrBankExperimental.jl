```
is_mapped(x::PAFRecord) -> Bool
```

`PAFRecord`がマッピングされているかどうかを計算します。マッピングされていないレコードは、`is_rc`および`tname`のプロパティが利用できません。マッピングされていないレコードの`qname`および`qlen`、および補助データは信頼できますが、残りのプロパティには任意のデータが含まれています。いくつかのPAFパーサーはマッピングされていないレコードを正しく処理しないため、マッピングされていないレコードを書く際には注意してください。

# 例

```jldoctes
julia> is_mapped(record)
true
```
