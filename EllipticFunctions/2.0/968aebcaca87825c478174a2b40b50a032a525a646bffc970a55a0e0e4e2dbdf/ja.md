```
jellip(kind, u; tau, m)
```

ヤコビ楕円関数。パラメータ `tau` または `m` のいずれか一方のみを指定する必要があります。

# 引数

  * `kind`: 'c', 'd', 'n' または 's' の中から2文字の文字列; この文字列は関数を指定します: 2つの文字はそれぞれ基本関数 `sn`, `cn`, `dn` と `1` を示し、文字列はそのような2つの関数の比を指定します。例えば、 `ns=1/sn` と `cd=cn/dn`
  * `u`: 実数または複素数、または数の配列
  * `tau`: 非負の虚部を持つ複素数
  * `m`: 実数または複素数、楕円モジュラスの平方
