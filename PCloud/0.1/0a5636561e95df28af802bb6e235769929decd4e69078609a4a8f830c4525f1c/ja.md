```
file_open(client::PCloudClient; kwargs...)
```

ファイルディスクリプタを開きます。

ソース: https://docs.pcloud.com/methods/fileops/file_open.html

# 引数

  * `flags::Int`: file_open フラグの組み合わせである可能性があります。

# オプション引数

  * `path::String`: ファイルディスクリプタが作成されるファイルへのパス。
  * `fileid::Int`: ファイルディスクリプタが作成されるフォルダの ID。
  * `folderid::Int`: 新しいファイルが作成され、ファイルディスクリプタが返されるフォルダの ID。
  * `name::String`: 新しいファイルが作成され、ファイルディスクリプタが返されるファイルの名前。

これらのパラメータの使用は、与えられたフラグに依存します。詳細は以下を参照してください。

# 出力

成功した場合、連続操作で使用できる `fd` ファイルディスクリプタを返します。また、ファイルの `fileid` も返します（ファイルを作成する際に便利です）。

# 出力例

```
{
    result: 0,
    fd: 1,
    fileid: 3489
}
```
