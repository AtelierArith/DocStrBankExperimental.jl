```
P4estMesh{NDIMS, NDIMS_AMBIENT} <: AbstractMesh{NDIMS}
```

木に基づく非構造化曲面メッシュで、Cライブラリ `p4est` を使用して木とメッシュの細分化を管理します。

パラメータ `NDIMS` は、メッシュ自体が表す空間領域または多様体の次元を示し、`NDIMS_AMBIENT` はメッシュが埋め込まれている周囲空間の次元を示します。たとえば、型 `P4estMesh{3, 3}` は三次元ボリュームの標準メッシュに対応し、`P4estMesh{2, 3}` は三次元空間における二次元表面またはシェルのメッシュに対応します。

!!! warning "実験的な実装"
    `NDIMS != NDIMS_AMBIENT` の使用は実験的な機能であり、将来のリリースで変更される可能性があります。
