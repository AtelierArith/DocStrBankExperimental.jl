```
VolumeIntegralSubcellLimiting(limiter;
                              volume_flux_dg, volume_flux_fv)
```

DG法のためのサブセル制限ボリューム積分タイプで、低次のFV法を用いたサブセルブレンディングアプローチに基づいています。リミッタ [`SubcellLimiterIDP`](@ref) と共に使用されます。

!!! note
    サブセル制限法は、非適合メッシュでは完全には機能しません。これは主に、実装が低次および高次スキームが同じ表面項を持つと仮定しているためであり、非適合メッシュでは保証されません。高次モルタルを持つ低次スキームは、不変領域保存性を持ちません。
