```
Trixi
```

**Trixi.jl** は、双曲線保存則のための数値シミュレーションフレームワークです。このフレームワークの主要な目的は、科学者と学生の両方にとって有用であることです。したがって、迅速な実装を持つ拡張可能な設計に加えて、Trixi.jl は新しいユーザーや経験の浅いユーザーにとって使いやすいことに重点を置いており、インストールや後処理手順も含まれています。

始めるには、次のコマンドを実行して Trixi.jl で最初のシミュレーションを行います。

```
trixi_include(default_example())
```

参照: [trixi-framework/Trixi.jl](https://github.com/trixi-framework/Trixi.jl)
