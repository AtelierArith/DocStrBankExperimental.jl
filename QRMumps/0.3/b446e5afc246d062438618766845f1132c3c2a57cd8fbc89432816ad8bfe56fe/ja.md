```
qrm_shifted_spmat = qrm_shift_spmat(spmat, α = 0)
```

与えられた `spmat` 構造体は、いくつかのスパース行列 `A` を表します。ブロック行列 `(A  √α)` を `qrm_shifted_spmat` 型として返します。詳細については `qrm_shifted_spmat` を参照してください。これは、`A` がランク欠損である場合に特に便利であり、`α > 0` を選択することで正則化として機能します。

### 入力引数

  * `spmat`: 入力行列
  * `α ≥ 0`: 正則化パラメータ（ブロック行列表現における平方根に注意）。
