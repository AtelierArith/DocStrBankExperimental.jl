```
qrm_vecnrm!(x, nrm; ntype='2')
```

このルーチンは、ベクトルの1ノルム $\|x\|_1$、無限ノルム $\|x\|_{\infty}$ または2ノルム $\|x\|_2$ を計算します。

#### 入力引数 :

  * `x`: x ベクトル。
  * `nrm`: 計算されたノルム。x が行列の場合、この引数はベクトルでなければならず、その各要素には x の対応する列のノルムが含まれます。
  * `ntype`: 計算されるノルムのタイプ。無限ノルム、1ノルム、2ノルムのそれぞれに対して `'i'`、`'1'` または `'2'` のいずれかを指定できます。
