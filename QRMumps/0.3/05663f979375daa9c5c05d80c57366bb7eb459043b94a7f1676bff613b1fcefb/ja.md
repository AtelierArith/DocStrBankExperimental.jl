```
qrm_spmat_mv!(spmat, alpha, x, beta, y; transp='n')
```

このサブルーチンは、`y = αAx + βy`、`y = αAᵀx + βy`、または`y = αAᴴx + βy`のタイプの行列-ベクトル積を実行します。

#### 入力引数 :

  * `spmat`: 入力行列。
  * `alpha`, `beta` : αおよびβスカラー
  * `x`: xベクトル(s)。
  * `y`: yベクトル(s)。
  * `transp`: A、Aᵀ、またはAᴴで乗算するかどうか。`'t'`、`'c'`、または`'n'`のいずれかです。
