```
qrm_golub_riley!(shifted_spmat, spfct, x, b, Δx, y; α = ϵm, max_iter = 50, tol = ϵm, transp = 'n')
```

このメソッドは、Golub-Riley反復を実装しています。システム `Ax = b` が与えられた場合（おそらく条件が悪いかランクが不足している）、`A` は任意の形 `m×n` を持つことができ、`x = A†b = Aᵀ(AAᵀ)†b` を計算します。ここで、`A†` は `A` のムーア-ペンローズ擬似逆行列です。

### 入力引数：

  * `shifted_spmat`: Golub-Riley反復の正則化パラメータである `α` を持つ行列 `(A  √α)` を表す `qrm_shifted_spmat` 型。
  * `spfct`: `qrm_spfct` 型のスパース因子分解オブジェクト。
  * `x`: 近似解ベクトル x := A†b = Aᵀy、このベクトルのサイズは n+m であり、その値はこの関数によって上書きされ、関数が呼び出されるときに初期化されていない可能性があります。
  * `b`: 線形システムの右辺ベクトル、このベクトルのサイズは m です。
  * `Δx`: 解を計算するために使用される補助ベクトル、このベクトルのサイズは n+m であり、関数が呼び出されるときに初期化されていない可能性があります。
  * `y`: y := (AAᵀ)†b を計算するために使用されるベクトル、このベクトルのサイズは n であり、関数が呼び出されるときに初期化されていない可能性があります。
  * `Δy`: 解を計算するために使用される補助ベクトル、このベクトルのサイズは n であり、関数が呼び出されるときに初期化されていない可能性があります。

`shifted_spmat` の定義は最初は奇妙に見えるかもしれません。`qrm_golub_riley!` の引数としてこのようなブロック行列を使用するのは、この行列のQR因子分解が `RᵀR = AᵀA + αI` という性質を持っているためであり、これはGolub-Riley反復のために繰り返し解く必要があるシステムです。

Golub-Rileyメソッドは次のように機能します：

  * A と b が与えられた場合、固定の正則化パラメータ `α > 0` を選択し、x := 0 とします。
  * 各反復で、Δx := Aᵀ(AAᵀ + αI)⁻¹ (b - Ax) を計算し、x := x + Δx を更新します。

このメソッドは、`qrm_shifted_spmat` 行列のQR因子分解を1回だけ行うコストの利点があります。詳細については、A. Dax と L. Eldén の「ランク不足の最小二乗問題の最小ノルム解の近似」、Numerical Linear Algebra with Applications, 5, pp. 79-99, 1998, DOI 10.1002/(SICI)1099-1506(199803/04)5:2<79::AID-NLA126>3.0.CO;2-4 を参照してください。
