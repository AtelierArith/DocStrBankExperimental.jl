```
qrm_refine!(spmat, spfct, x, z, Δx, y)
```

線形システム RᵀRx ≈ z の近似解 x が与えられたとき、R はサイズ (m, n) の QR 分解の R 因子であり、洗練された解を計算します。

### 入力引数 :

  * `spmat`: 入力行列。
  * `spfct`: `qrm_spfct` 型のスパース因子化オブジェクト。
  * `x`: 近似解ベクトルまたは行列。このベクトル（または行列）のサイズは n（または n×k）です。
  * `z`: 線形システムの右辺ベクトルまたは行列。このベクトル（または行列）のサイズは n（または n×k）です。
  * `Δx`: 洗練を計算するために使用される補助ベクトル（または x と z が行列の場合は行列）。このベクトル（または行列）のサイズは n（または n×k）であり、関数が呼び出されるときに初期化されていなくてもかまいません。
  * `y`: 洗練を計算するために使用される補助ベクトル（または x と z が行列の場合は行列）。このベクトル（または行列）のサイズは m（または m×k）であり、関数が呼び出されるときに初期化されていなくてもかまいません。
