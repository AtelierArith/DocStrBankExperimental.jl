```
lim(f, c; n=6, m=1, dir="+-")
lim(f, c, dir; n-5)
```

は、`h` が `c` に近づくにつれて `f` の値の数値テーブルを生成することを意味します。

  * `n`, `m`: `c` に加える（引く）`10` の累乗。
  * `dir`: `"+-"`（左と右を表示）、`"+"`（右極限）、または `"-"`（左極限）のいずれか。関数 `+`, `-`, `±` も使用できます。

例:

```
julia> f(x) = sin(x) / x
f (generic function with 1 method)

julia> lim(f, 0)
 0.1        0.9983341664682815
 0.01       0.9999833334166665
 0.001      0.9999998333333416
 0.0001     0.9999999983333334
 1.0e-5     0.9999999999833332
 1.0e-6     0.9999999999998334
   ⋮          ⋮
   c          L?
   ⋮          ⋮
-1.0e-6     0.9999999999998334
-1.0e-5     0.9999999999833332
-0.0001     0.9999999983333334
-0.001      0.9999998333333416
-0.01       0.9999833334166665
-0.1        0.9983341664682815
```
