```
atomicmass(element::Element)
atomicmass(i::Union{AbstractString,Integer,Symbol})
```

Return the atomic mass of an element.

!!! warning
    This function is by default not implemented. You need to load either package [`Mendeleev.jl`](https://github.com/Eben60/Mendeleev.jl) or [`PeriodicTable.jl`](https://github.com/JuliaPhysics/PeriodicTable.jl) to use it.
