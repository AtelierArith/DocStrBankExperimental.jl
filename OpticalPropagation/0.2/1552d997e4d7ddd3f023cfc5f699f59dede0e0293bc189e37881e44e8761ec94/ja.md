```
fresnel1(Uin, d, λ, lx, ly)
```

単一のフーリエ変換を用いたフレネル回折に基づく光場の伝播を計算します。

## 引数

  * `Uin::AbstractArray{<:Number,2}`: 入力複素振幅の複素配列。
  * `d::Real`: メートル単位の伝播距離。
  * `λ::Real`: 伝播する光の波長。
  * `lx::Real`: x軸に沿った入力データの物理サイズ。
  * `ly::Real`: y軸に沿った入力データの物理サイズ。

## 戻り値

```
(Uout, (lxo, lyo))
```

  * `Uout::Array{<:Number,2}`: 伝播後の複素振幅データ。
  * `lxo::Real`: x軸に沿った回折データの物理サイズ。
  * `lyo::Real`: y軸に沿った回折データの物理サイズ。
