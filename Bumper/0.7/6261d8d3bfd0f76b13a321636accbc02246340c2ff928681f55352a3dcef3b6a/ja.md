```
@alloc_ptr(n::Integer) -> Ptr{Nothing}
```

これは、`@no_escape` ブロック内で使用され、`n` バイトを保持できるポインタを割り当てるために使用されます。このポインタを割り当てるために使用されるメモリは、囲む `@no_escape` ブロックに関連付けられたバッファから来ます。

このポインタへの参照が囲む `@no_escape` ブロックから逃げることを許可せず、これらのポインタを同時実行タスクに渡さないでください。ただし、そのタスクが `@no_escape` ブロックが終了する前に終了することが保証されている場合を除きます。この方法で割り当てられたポインタがその親 `@no_escape` ブロックの外に見つかった場合、その内容は未定義であり、このポインタに書き込むことは未定義の動作を引き起こします。
