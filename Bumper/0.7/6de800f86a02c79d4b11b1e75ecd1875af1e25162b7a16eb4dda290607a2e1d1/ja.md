```
@alloc(T, n::Int...) -> UnsafeArray{T, length(n)}
```

これは、`@no_escape` ブロック内で使用され、`n` によって決定される次元を持つ `UnsafeArray` を割り当てるために使用できます。この配列を割り当てるために使用されるメモリは、囲む `@no_escape` ブロックに関連付けられたバッファから取得されます。

この配列への参照が囲む `@no_escape` ブロックから逃げることを許可せず、これらの配列を同時タスクに渡さないでください。タスクが `@no_escape` ブロックが終了する前に終了することが保証されている場合を除きます。この方法で割り当てられた配列がその親 `@no_escape` ブロックの外に見つかった場合、その内容は未定義であり、このポインタに書き込むことは未定義の動作を引き起こします。
