ProjMPOSumは、MPSによって定義された基底に対してMPOの暗示された和の射影を計算し保存しますが、各MPOの特定の数のサイトインデックスは射影されません。どのサイトが射影されないかは、`position!`メソッドを呼び出すことで変更できます。ProjMPOSumに入力として使用されるMPOは、事前に加算されることはなく、代わりに`product`メソッドが呼び出されると、MPOのセット内の各射影MPOが入力テンソルと効率的に1つずつ乗算されます。

ProjMPOSum `P([H1,H2,...])`によって表されるネットワークの図で、`nsite(P)==2`およびMPS `psi`に対して`position!(P,psi,4)`を示しています（左側の和Σⱼに注意）：

```
     o--o--o-      -o--o--o--o--o--o <psi|
     |  |  |  |  |  |  |  |  |  |  |
 Σⱼ  o--o--o--o--o--o--o--o--o--o--o Hⱼ
     |  |  |  |  |  |  |  |  |  |  |
     o--o--o-      -o--o--o--o--o--o |psi>
```
