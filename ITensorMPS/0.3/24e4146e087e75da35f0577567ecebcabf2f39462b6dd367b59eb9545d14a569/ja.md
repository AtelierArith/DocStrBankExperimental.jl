ProjMPOは、MPSによって定義された基底へのMPOの射影を計算して保存し、MPOの特定のサイトインデックスを未射影のままにします。未射影のサイトは、`position!`メソッドを呼び出すことで移動できます。

ProjMPO `P(H)`によって表されるネットワークの図で、`nsite(P)==2`およびMPS `psi`に対して`position!(P,psi,4)`の場合を示します：

```
o--o--o-      -o--o--o--o--o--o <psi|
|  |  |  |  |  |  |  |  |  |  |
o--o--o--o--o--o--o--o--o--o--o H
|  |  |  |  |  |  |  |  |  |  |
o--o--o-      -o--o--o--o--o--o |psi>
```
