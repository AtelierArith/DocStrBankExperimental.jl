```
Planet{f,a,t,Gm}
```

天体 `Planet` 楕円体基準体で、`flattening` は `f`、半長軸は `a`、恒星の `period` は `t`、標準の `gravitation` パラメータは `Gm` です。さらに導出されたパラメータには、`semiminor`、`frequency`、`angularfrequency`、`eccentricity`、`eccentricity2`、`lineareccentricity`、`aspectratio`、`oblateness`、`dynamicformfactor`、`secondzonalharmonic`、`latitudegeodetic`、`latitudegeocentric`、`latitudeparametric`、`radius`、`radiusgeodetic`、`gravity`、`gravitygeodetic`、および `gravitycomponents` が含まれます。
