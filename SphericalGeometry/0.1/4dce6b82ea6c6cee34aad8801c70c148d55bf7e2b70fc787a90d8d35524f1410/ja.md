```
intermediate_point(point₁::Point, point₂::Point, fraction::Float64)
```

`intermediate_point` [deg] を、2つの点 `point₁` と `point₂` [deg] の間の大円線分に沿った任意の `fraction` で返します。大円線分に沿った割合は、`fraction` = 0.0 が `point₁` [deg] にあり、`fraction` 1.0 が `point₂` [deg] にあるように設定されています。

出典: www.movable-type.co.uk/scripts/latlong.html
