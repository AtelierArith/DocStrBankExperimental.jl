```
transpiration(;gsw = 0.1, gbw = 1.0, Tleaf = 300.0, Tair = 298.0, P = 101e3,
               RH = 0.75)
```

水蒸気に対する導電率と環境変数から蒸発散率 (mol/m^2/s) を計算します。

# 引数

  * `gsw`: 水蒸気に対する気孔導電率 (mol/m^2/s)
  * `gbw`: 水蒸気に対する境界層導電率 (mol/m^2/s)
  * `Tleaf`: 葉の温度 (K)
  * `Tair`: 空気の温度 (K)
  * `P`: 空気圧 (Pa)
  * `RH`: 相対湿度
