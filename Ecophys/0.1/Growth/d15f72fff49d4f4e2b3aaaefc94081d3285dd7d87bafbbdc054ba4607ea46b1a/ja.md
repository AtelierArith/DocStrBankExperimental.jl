```
Organ(f = [0.53, 0.25, 0.05, 0.05, 0.06, 0.06],
      te = 100.0, tm = 50.0, tb = 10.0, wmax = 0.1)
```

植物器官の成長モデルのすべてのパラメータを格納するデータ構造。

# 引数

  * `f`: 新しく形成された材料の異なる炭素プールに割り当てられる乾燥物質の割合、[炭水化物、タンパク質、脂質、リグニン、有機酸、鉱物]
  * `te`: 成長率が消失する時点、[DD]
  * `tm`: 成長率が最大になる時点、[DD]
  * `tb`: 基準温度、[°C]
  * `wmax`: 最大重量、[g]
