```
AutoRegressive(n, method = LPCLevinson())
```

次数`n`の自己回帰サロゲート。自己回帰係数`φ`は`DSP.lpc(x, n, method)`を使用して推定されるため、可能な`method`についてはDSP.jlのドキュメントを参照してください。

これらのサロゲートは、データが自己回帰プロセスから来ているかどうかの帰無仮説をテストするのに明らかに適していますが、フーリエ変換に基づくサロゲートの方がおそらくより良い選択です。現在の方法は、モデルをフィッティングすることによって同じ仮説のためのサロゲートを生成する明示的な方法に近いです。これは、自己回帰係数を推定し、それに基づいて自動的にサロゲートを生成する便利な方法として使用できます。

自己回帰フィットの係数φは、次のようにして見つけることができます。

```julia
sg = surrogenerator(x, AutoRegressive(n))
φ = sg.init.φ
```
