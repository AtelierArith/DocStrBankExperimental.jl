```
simulate(model; kwargs...)
```

指定された `model` のシミュレーションを実行します。

シミュレーションに関するすべての時系列情報を含む `NamedTuple` を返します。

# 必須キーワード引数

  * `T`: シミュレーションの総時間。必須

# 任意のキーワード引数

  * `uc`: 時間 `t` における連続時間モデルへの入力を定義する関数 `(t) -> u` を期待します。デフォルトは `(t) -> nothing` です。
  * `ud`: 時間 `t` における離散時間モデルへの入力を定義する関数 `(t) -> u` を期待します。デフォルトは `(t) -> nothing` です。
  * `Δt_max`: 連続時間モデルの積分に使用される最大ステップサイズ。デフォルトは `SimpleSim.jl` に設定された `ΔT_DEFAULT` です。
  * `t0`: 初期時間。デフォルトは `0 // 1` です。
  * `xc0`: 連続時間モデルの初期状態。モデル自体で定義された初期状態を上書きします。デフォルトは `nothing` です。
  * `xd0`: 離散時間モデルの初期状態。モデル自体で定義された初期状態を上書きします。デフォルトは `nothing` です。
  * `integrator`: 連続時間モデルに使用される積分法。サポートされている積分法については以下を参照してください。デフォルトは `RK4.` です。
  * `options`: 設定可能な追加オプションについては以下を参照してください。

# サポートされている数値積分法

これらのオプションは、`integrator` キーワード引数として `simulate` 関数に渡すことができます：

```julia
@enum SimpleSimIntegrator RK4 = 1 Euler = 2 Heun = 3 RKF45 = 4
```

# オプション

`SimpleSim.jl` には、一般的に変更する必要のないシミュレーションを実行するためのいくつかのデフォルトパラメータがあります。ただし、必要に応じて、以下のオプションを `NamedTuple` で `options` キーワード引数に渡すことができます。

  * `Δt_default`: 連続時間積分に使用されるデフォルト（最大）ステップサイズを置き換えます。有理数である必要があります。デフォルトはグローバルパラメータ `ΔT_DEFAULT` です。
  * `Δt_min`: 連続時間積分に使用される最小ステップサイズを置き換えます。特に適応ステップサイズ積分法に関連します。デフォルトは `1 // 1_000_000` です。
  * `zero_crossing_tol`: ゼロ交差の時間を計算する際に使用される絶対許容誤差。デフォルトは `1e-5` です。
  * `RKF45_rel_tol`: 切断誤差と5次のルンゲ・クッタ推定値との間の相対許容誤差で、`RKF45` 積分法の終了につながります。デフォルトは `1e-6` です。
  * `RKF45_abs_tol`: `RKF45` 積分法の終了につながる切断誤差の絶対許容誤差。デフォルトは `1e-7` です。
  * `silent`: `true` に設定すると、警告やエラーを含むすべての出力が無効になります。エラーと警告のみを表示し、他のすべての出力を無効にするには、`display_progress` と `debug` を `false` に設定します。デフォルトは `false` です。
  * `debug`: `true` に設定すると、モデルのデバッグに役立つ追加情報がターミナルに表示されます。デフォルトは `false` です。
  * `display_progress`: シミュレーションの進行状況についてターミナルで更新を受け取りたくない場合は `false` に設定します。デフォルトは `true` です。
  * `progress_spacing`: ターミナルでの進行状況更新の間隔時間。デフォルトは `1 // 1` です。
  * `base_rng`: ランダム抽出関数に使用される乱数生成器。デフォルトは `MersenneTwister` です。
  * `out_stream`: コンソール出力に使用されるIOストリーム。デフォルトは `stdout` です。

# オプションを使用した例

```julia
out = simulate(my_model,
    T = 20 // 1,
    options = (
        silent = true,
        base_rng = Xoshiro,
    )
)
```
