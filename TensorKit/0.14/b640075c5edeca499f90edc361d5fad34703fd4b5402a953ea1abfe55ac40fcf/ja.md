```
abstract type ElementarySpace <: VectorSpace end
```

テンソルのインデックスに対応するインデックス空間として使用できる体上の基本的な有限次元ベクトル空間。ElementarySpaceは、テンソルの個々のインデックスに関連付けることができるすべてのベクトル空間（オブジェクト）のスーパタイプであり、そのエイリアスであるIndexSpaceによって示唆されています。

すべての基本的なベクトル空間は、[`conj`](@ref)および[`dual`](@ref)メソッドに応答し、それぞれ複素共役空間と双対空間を返す必要があります。双対空間の複素共役は、`dual(conj(V)) === conj(dual(V))`として得られます。これらの異なる空間は同じ型である必要があるため、テンソルはこれらの空間の同次テンソル積の要素として定義できます。
