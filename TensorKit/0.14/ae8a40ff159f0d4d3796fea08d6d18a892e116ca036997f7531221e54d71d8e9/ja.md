```
eig(t::AbstractTensor, (leftind, rightind)::Index2Tuple; kwargs...) -> D, V
```

テンソル `t` の固有値因子分解を、`rightind` から `leftind` への線形写像として計算します。関数 `eig` は線形写像がエルミートでないことを前提としており、実数および複素数値の `t` に対して型安定な複素数値の `D` および `V` テンソルを返します。エルミート線形写像については `eigh` を参照してください。

`leftind` と `rightind` が指定されていない場合、`t` の現在の左インデックスと右インデックスの分割が使用されます。その場合、`eig!(t)` を使用して `t` のデータが破壊/上書きされることを許可すると、より少ないメモリが割り当てられます。`eig!` が呼び出される順列テンソルは、ドメインとコドメインが等しい必要があります。そうでない場合、固有値分解は無意味であり、次の条件を満たすことができません。

```
permute(t, (leftind, rightind)) * V = V * D
```

密行列の `eigen` と同じキーワード引数 `scale` と `permute` を受け入れます。詳細については、対応するドキュメントを参照してください。

また、`eigen` および `eigh` も参照してください。
