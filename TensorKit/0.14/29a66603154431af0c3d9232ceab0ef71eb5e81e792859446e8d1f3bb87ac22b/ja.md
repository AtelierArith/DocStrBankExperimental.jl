```
isposdef(t::AbstractTensor, (leftind, rightind)::Index2Tuple) -> ::Bool
```

テンソル `t` が `rightind` から `leftind` への線形写像として正定値であるかどうかをテストします。

`leftind` と `rightind` が指定されていない場合、`t` の現在の左インデックスと右インデックスの分割が使用されます。その場合、`isposdef!(t)` を使用することで、`t` 内のデータが破壊/上書きされることを許可すれば、より少ないメモリが割り当てられます。`isposdef!` が呼び出される順列テンソルは、ドメインとコドメインが等しい必要があることに注意してください。そうでない場合は意味がありません。
