```
braid(f::FusionTree{<:Sector, N}, levels::NTuple{N, Int}, p::NTuple{N, Int})
-> <:AbstractDict{typeof(t), <:Number}
```

融合木 `f` の結合されていないインデックスのブレーディングを行い、出力木と対応する係数の `<:AbstractDict` として結果を返します。ブレーディングは、新しいセクターが位置 `k` において、元々位置 `i = p[k]` にあったセクターに対応することを指定し、元の融合木の各インデックス `i` に異なるレベルまたは深さ `levels[i]` を割り当てることによって決定されます。この置換は、隣接するインデックス間の基本的なスワップに分解され、スワップはブレードとして適用されます。すなわち、もし `i` と `j` が交差する場合、$τ_{i,j}$ が `levels[i] < levels[j]` のときに適用され、$τ_{j,i}^{-1}$ が `levels[i] > levels[j]` のときに適用されます。これにより、最も一般的なブレードをエンコードすることはできませんが、そのような操作を組み合わせることで一般的なブレードを得ることができます。
