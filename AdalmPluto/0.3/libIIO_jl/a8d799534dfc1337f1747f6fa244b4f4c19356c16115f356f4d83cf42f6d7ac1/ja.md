```
C_iio_context_get_devices_count(context)
```

与えられたコンテキスト内で見つかったデバイスを列挙します。

# パラメータ

  * `context::Ptr{iio_context}` : iio_context 構造体へのポインタ

# 戻り値

  * 見つかったデバイスの数
