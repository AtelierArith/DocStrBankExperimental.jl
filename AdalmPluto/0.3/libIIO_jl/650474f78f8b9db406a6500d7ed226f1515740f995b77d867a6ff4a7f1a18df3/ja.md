```
C_iio_channel_get_data(channel)
```

iio_channel構造体の以前に関連付けられたポインタを取得します。

# パラメータ

  * `channel::Ptr{iio_channel}` : iio_channel構造体へのポインタ

# 戻り値

  * 以前に関連付けられたポインタが存在する場合はそれ、またはNULL
