```
C_iio_device_get_buffer_attrs_count(device)
```

指定されたデバイスのバッファ固有の属性を列挙します。

# パラメータ

  * `device::Ptr{iio_device}` : iio_device 構造体へのポインタ

# 戻り値

  * 見つかったバッファ固有の属性の数
