```
C_iio_channel_get_device(channel)
```

iio_device構造体へのポインタを取得します。

# パラメータ

  * `channel::Ptr{iio_channel}` : iio_channel構造体へのポインタ

# 戻り値

  * iio_device構造体へのポインタ
