```
C_iio_channel_is_output(channel)
```

与えられたチャネルが出力チャネルである場合はTrueを返します。

# パラメータ

  * `channel::Ptr{iio_channel}` : iio_channel構造体へのポインタ

# 戻り値

  * チャネルが出力チャネルである場合はTrue、そうでない場合はFalse
