```
C_iio_channel_is_scan_element(channel)
```

与えられたチャネルがスキャン要素である場合はTrueを返します。

# パラメータ

  * `channel::Ptr{iio_channel}` : iio_channel構造体へのポインタ

# 戻り値

  * チャネルがスキャン要素である場合はTrue、そうでない場合はFalse
