```
C_iio_buffer_get_device(buffer)
```

Retrieve a pointer to the iio_device structure.

# Parameters

  * `buffer::Ptr{iio_buffer}` : A pointer to an iio_buffer structure

# Returns

  * A pointer to an iio_device structure
