```
C_iio_device_get_attrs_count(device)
```

指定されたデバイスのデバイス固有の属性を列挙します。

# パラメータ

  * `device::Ptr{iio_device}` : iio_device 構造体へのポインタ

# 戻り値

  * 見つかったデバイス固有の属性の数
