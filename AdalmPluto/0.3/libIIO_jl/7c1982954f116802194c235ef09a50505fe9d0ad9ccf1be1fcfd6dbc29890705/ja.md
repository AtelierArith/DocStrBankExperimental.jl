```
C_iio_device_is_trigger(device)
```

与えられたデバイスがトリガーである場合はTrueを返します。

# パラメータ

  * `device::Ptr{iio_device}` : iio_device構造体へのポインタ

# 戻り値

  * デバイスがトリガーである場合はTrue、そうでない場合はFalse
