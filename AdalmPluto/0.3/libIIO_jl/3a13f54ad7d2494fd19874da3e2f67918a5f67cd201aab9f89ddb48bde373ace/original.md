```
C_iio_context_info_get_uri(context)
```

Get the URI of a discovered context.

# Parameters

  * `context::Ptr{iio_context_info}`: A pointer to an `iio_context_info` structure.

# Returns

  * A pointer to a static NULL-terminated string
