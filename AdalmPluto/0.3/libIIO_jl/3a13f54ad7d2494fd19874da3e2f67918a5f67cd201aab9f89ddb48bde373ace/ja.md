```
C_iio_context_info_get_uri(context)
```

発見されたコンテキストのURIを取得します。

# パラメータ

  * `context::Ptr{iio_context_info}`: `iio_context_info`構造体へのポインタ。

# 戻り値

  * 静的なNULL終端文字列へのポインタ
