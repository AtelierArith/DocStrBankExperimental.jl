```
C_iio_device_get_data(device)
```

iio_device構造体の以前に関連付けられたポインタを取得します。

# パラメータ

  * `device::Ptr{iio_device}` : iio_device構造体へのポインタ

# 戻り値

  * 以前に関連付けられたポインタが存在する場合はそれ、またはNULL
