```
C_iio_device_get_channels_count(device)
```

iio_context構造体へのポインタを取得します。

# パラメータ

  * `device::Ptr{iio_device}` : iio_device構造体へのポインタ

# 戻り値

  * iio_context構造体へのポインタ
