```
C_iio_buffer_end(buffer)
```

バッファ内の最後のサンプルに続くアドレスを取得します。

# パラメータ

  * `buffer::Ptr{iio_buffer}` : iio_buffer 構造体へのポインタ

# 戻り値

  * バッファ内に存在する最後のサンプルに続くアドレスに対応するポインタ
