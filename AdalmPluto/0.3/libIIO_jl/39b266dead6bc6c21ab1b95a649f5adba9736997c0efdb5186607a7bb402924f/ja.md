```
C_iio_channel_get_attrs_count(channel)
```

指定されたチャネルのチャネル固有の属性を列挙します。

# パラメータ

  * `channel::Ptr{iio_channel}` : iio_channel 構造体へのポインタ

# 戻り値

  * 見つかったチャネル固有の属性の数
