```
C_iio_buffer_destroy(buffer)
```

Destroy the given buffer.

# Parameters

  * `buffer::Ptr{iio_buffer}` : A pointer to an iio_buffer structure

# NOTE

After that function, the iio_buffer pointer shall be invalid.
