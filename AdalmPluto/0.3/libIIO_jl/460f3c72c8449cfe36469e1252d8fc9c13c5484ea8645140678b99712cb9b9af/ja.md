```
C_iio_context_info_get_description(context)
```

発見されたコンテキストの説明を取得します。

# パラメータ

  * `context::Ptr{iio_context_info}`: `iio_context_info` 構造体へのポインタ。

# 戻り値

  * 静的なNULL終端文字列へのポインタ
