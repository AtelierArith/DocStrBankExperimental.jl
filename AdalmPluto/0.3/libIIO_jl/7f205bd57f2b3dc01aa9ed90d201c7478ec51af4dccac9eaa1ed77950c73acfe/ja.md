```
C_iio_context_get_attrs_count(context)
```

コンテキスト固有の属性の数を取得します。

# パラメータ

  * `context::Ptr{iio_context}` : iio_context 構造体へのポインタ

# 戻り値

  * コンテキスト固有の属性の数

バージョン 0.9 で導入されました。
