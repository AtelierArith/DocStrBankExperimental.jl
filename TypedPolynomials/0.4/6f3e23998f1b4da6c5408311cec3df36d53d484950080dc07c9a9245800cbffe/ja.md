多項式変数を構築し、それらを同じ名前のローカル変数にバインドします。使用法：

単一の変数 x を作成します：

```
@polyvar(x)
```

複数の変数を同時に作成します：

```
@polyvar(x, y, z)
```

変数のタプル x = (x1, x2, x3, x4) を作成します：

```
@polyvar(x[1:4])
```

変数 x, y, a = (a1, a2, a3) を作成します：

```
@polyvar(x, y, a[1:3])
```

マクロの結果をタプルに割り当てることもできます：

```
vars = @polyvar(x[1:5])
@assert typeof(vars[1]) == Variable{:x1}
```
