```
open_stream(inlet::StreamInlet; timeout=LSL_FOREVER)
```

データストリームにサブスクライブします。

この瞬間以降、他の端からプッシュされたすべてのサンプルはキューに入れられ、最終的にはpull*sample()またはpull*chunk()呼び出しに応じて配信されます。前にopen_streamがない状態でサンプルをプルすることは許可されています（その場合、ストリームは暗黙的にオープンされます）。

関数はタイムアウトエラーまたはロストエラー（ストリームソースが失われた場合）をスローする可能性があります。

# キーワード引数

  * `timeout::Number`: 操作のタイムアウト。
