```
nominal_srate(info::StreamInfo)
```

ストリーム `info` のサンプリングレートを、ソースに従って返します（Hz単位）。

ストリームが不規則にサンプリングされている場合、これは IRREGULAR_RATE に設定する必要があります。

このサンプリングレートが不正確であったり、デバイスに一時的な問題があった場合でも、すべてのサンプルは記録されるため、データが失われることはありません（デバイス自体によってドロップされたものを除く）。ただし、録音がアプリケーションにインポートされるとき、良いインポータは、広告されたサンプリングレートがデバイスの仕様に近い場合、こうしたエラーをより正確に修正できるかもしれません。
