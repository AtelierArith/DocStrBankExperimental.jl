```
pull_chunk(inlet::StreamInfo; max_samples = 1024, timeout = LSL_FOREVER)
```

インレットからデータのチャンクを取得し、提供された行列に割り当てます。

関数は、長さNのタイムスタンプのベクターを返し、これは`max_samples`以下であり、各列がサンプルで構成されるデータの行列を返します。したがって、size(data) = (M,N) です。

関数は、ストリームが失われた場合にタイムアウトエラーやロストエラーをスローする可能性があります。タイムアウトが発生した場合、またはタイムアウトが0.0に指定され、新しいサンプルが利用できない場合、タイムスタンプは0.0になります。

# キーワード引数

`timeout::Number`: サンプルを取得するためのタイムアウト。
