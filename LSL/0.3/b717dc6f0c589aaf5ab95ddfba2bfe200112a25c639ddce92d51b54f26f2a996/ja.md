```
desc(info::StreamInfo)
```

ストリームの拡張説明をXMLElementとして取得します。

ここでは、少なくともチャネルラベルが記述されることを強く推奨します。LSLウィキのコード例を参照してください。アンプ設定、デフォルトから逸脱する測定単位、セットアップ情報、被験者情報など、他の情報もここで指定できます。メタデータの推奨事項はXDFファイルフォーマットプロジェクトに従います (github.com/sccn/xdf/wiki/Meta-Data またはウェブ検索で: XDFメタデータ)。

重要: メタデータの推奨事項が存在するストリームコンテンツタイプを使用する場合は、他のアプリケーションとの互換性のために、これらの推奨事項に従ってメタデータを配置するようにしてください。
