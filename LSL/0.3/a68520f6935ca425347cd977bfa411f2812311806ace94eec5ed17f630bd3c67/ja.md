```
resolve_streams(;timeout = 1.0)
```

現在利用可能なすべてのストリームをネットワーク上の任意のアウトレットから返します。

ネットワークは通常、ローカルルーターで指定されたサブネットですが、マルチキャストパケットを介して互いに可視な機械のグループ（ネットワークがそれをサポートしている場合）やホスト名のリストを含むこともあります。これらの詳細は、実験者が設定ファイルでカスタマイズすることができます（LSL wikiのネットワーク接続を参照）。  

空のdescフィールドを持つStreamInfoオブジェクトのベクターを返し、そのいずれかを使用してインレットを開くことができます。完全な説明はインレットから取得できます。

# キーワード引数:

-`timeout`: ストリームを検索するための操作の待機時間（秒単位）。警告: この時間が短すぎる（<0.5s）と、ネットワーク上に存在するアウトレットのサブセット（またはゼロ）が返される可能性があります。
