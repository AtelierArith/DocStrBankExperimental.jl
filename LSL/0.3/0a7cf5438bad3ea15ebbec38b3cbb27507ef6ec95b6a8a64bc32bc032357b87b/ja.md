```
type(info::StreamInfo)
```

ストリーム `info` のコンテンツタイプを返します。

コンテンツタイプは、チャネルが運ぶコンテンツを説明する短い文字列であり、"EEG" や "Gaze" などです（既知の場合）。ストリームが混合コンテンツを含む場合、この値は割り当てる必要はありませんが、代わりにチャネルタイプの説明に保存されることがあります。アプリケーションや自動処理システムにとって有用であるため、推奨されるコンテンツタイプを使用することが望まれます。コンテンツタイプは通常、https://github.com/sccn/xdf/wiki/Meta-Data で事前定義されたものに従います（または、ウェブ検索で: XDF メタデータ）。
