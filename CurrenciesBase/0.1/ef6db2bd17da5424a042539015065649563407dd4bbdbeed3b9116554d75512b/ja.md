通貨で表される金銭的価値の表現です。使用される通貨は型の一部であり、オブジェクトではありません。値は内部的にいくつかの整数型の量として表現されます。必要に応じて`Monetary`を直接構築する通常の方法は次のとおりです：

```
Monetary(:USD)      # 1.00 USD
Monetary(:USD, 325) # 3.25 USD
```

小数点に注意してください。`Monetary`コンストラクタは、通貨の最小単位の数を表す整数を取ります。通常、このコンストラクタは直接呼び出されることはありません。`@usingcurrencies`マクロと`100USD`形式を使用する方が簡単です。

この型は、`BigInt`のような任意の整数型として内部的に表現された値をサポートするのに十分柔軟ですが、より大きな型が必要でない限り、アーキテクチャ上の組み込みの`Int`型を使用することをお勧めします。異なる種類の内部型を混合しないでください。異なる内部表現を使用するには、`Monetary`の第二引数の型を変更します：

```
Monetary(:USD, BigInt(100))
```

一部のアプリケーションでは、通貨の小額単位が十分に正確ではありません。保存される小数点の数をオーバーライドすることが有用な場合があります。これらのアプリケーションでは、主要な単位の後に保持する小数点の数を示す第三の型パラメータを提供できます：

```
Monetary{:USD, BigInt, 4}(10000)            # 1.0000 USD
Monetary(:USD, BigInt(10000); precision=4)  # 1.0000 USD
```
