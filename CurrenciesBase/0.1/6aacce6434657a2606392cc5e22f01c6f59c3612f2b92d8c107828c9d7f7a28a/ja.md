新しい通貨を（グローバル）通貨リストに追加し、その通貨の単位を返します。可能な限り、より明確なコードを導く`@usingcustomcurrency`マクロを優先してください。この関数は3つの引数を取ります：通貨のシンボル、通貨の文字列説明（`currencyinfo`のドキュメントで概説された規則に従う）、および主要通貨単位に対するマイナー通貨単位を表す小数点の数を示す指数です。カスタム通貨を説明するために使用されるシンボルは、通常、小文字の文字のみで構成されるべきです。

```
btc = newcurrency!(:btc, "Bitcoin", 8)  # 1.00000000 BTC
```
