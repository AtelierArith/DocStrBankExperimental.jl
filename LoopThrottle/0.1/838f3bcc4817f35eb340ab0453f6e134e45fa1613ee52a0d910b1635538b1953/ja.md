```
@throttle
```

`for` ループまたは `while` ループをスロットルし、各ループの反復の最初に `sleep` を呼び出すことで、指定された変数が最大 `max_rate` の速度で増加するようにします（壁時間と比較して）。

`@throttle` が取る最初の引数は、レート制限される変数です。この変数はループ本体内でスコープ内にある必要があります。2番目の引数は `for` ループまたは `while` ループです。オプションで、ループ引数の後に次のキーワード引数を渡すことができます：

  * `max_rate`: 指定された変数が増加すべき最大レートを指定します。デフォルトは `1.` です。
  * `min_sleep_time`: 最小のスリープ時間を指定します（秒単位）。デフォルトは `0.01` で、`sleep` 関数の精度のために少なくとも 0.001 でなければなりません。

# 例

```julia
x = 0
@throttle t for t = 1 : 0.01 : 2
    x += 1
end max_rate = 2.
```

は約 0.5 秒で終了します。

```julia
x = 0.
@throttle x for i = 0 : 10
    x += 0.1
end
```

はデフォルトの `max_rate` 値 `1.` を使用し、したがって約 1 秒で終了します。

```julia
i = 0
@throttle i while i <= 10
    println(i)
    i += 1
end min_sleep_time=1.5 max_rate=1
```

は 0 から 10 までの数字を平均して 1 秒ごとに印刷し、1.5 秒未満でスリープすることはありません。
