```
gamma_flux_pions_PE04(rho_cgs::Real, m_cgs::Real, T_K::Real, α_p::Real, d::Real;
                      Xcr::Real=0.5,
                      Eγ_π0_min::Real=0.1, Eγ_π0_max::Real=200.0,
                      xH::Real=0.752)
```

プロトンスペクトルから生成されるγ線光子のフラックスは、`rho_cgs` [g/cm^3] と `T_K` [K] で定義されたエネルギー密度の割合 `Xcr` として与えられ、エネルギーに対するパワーロースロープ `α_p` を持ちます。質量 `m_cgs` [g] の粒子のためにSPH粒子体積で統合されます。距離 `d` [cm] からのフラックス。光子エネルギー `Eγ_π0_min` と `Eγ_π0_max` [GeV] の間で統合されます。エネルギーバンド内の光子の総数を [γ cm^-2 s^-1] の単位で返します。Pfrommer&Enßlin (2004), Eq. 25 を参照してください。

## 引数:

  * `rho_cgs`: SPH粒子の密度 [g/cm^3]
  * `m_cgs`: SPH粒子の質量 [g]
  * `T_K`: SPH粒子の温度 [K]
  * `α_p`: プロトンエネルギースペクトルのスロープ `S ~ 2.0 - 2.5`
  * `d`: SPH粒子またはハローまでの距離 [cm]。
  * `Xcr`: CRプロトンと熱圧の比。
  * `Eγ_π0_min`: γ線スペクトルの最小光子エネルギー [GeV]
  * `Eγ_π0_max`: γ線スペクトルの最大光子エネルギー [GeV]
  * `xH`: シミュレーションにおける水素質量分率
