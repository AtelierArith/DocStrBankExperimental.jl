```
stratifiedobs(d::AbstractDiskDataProvider, p::AbstractFloat, args...; kwargs...)
```

データを p の値に比例して複数の非重複のサブセットに分割します。観測値は、置換なしの層化サンプリングを使用してデータサブセットに割り当てられます。これらのサブセットは、最初の要素が p の最初の浮動小数点数によって指定されたデータの観測値の割合を含むタプルとして返されます。

例えば、p が Float64 自体である場合、返り値は二つの要素（すなわちサブセット）を持つタプルとなり、最初の要素には p によって指定された観測値の割合が含まれ、二番目の要素には残りが含まれます。以下のコードでは、最初のサブセット train には約 70% の観測値が含まれ、二番目のサブセット test には残りが含まれます。splitobs との主な違いは、y のクラス分布が train と test において積極的に保持されることです。

`train, test = stratifiedobs(diskdataprovider, 0.7)`
