```
(N, ωn) = cheb2ord(Wp::Real, Ws::Real, Rp::Real, Rs::Real; domain::Symbol=:z)
```

チェビシェフタイプII（逆）フィルタの整数および自然周波数の次数推定。`Wp` と `Ws` はバンドパス/バンドストップの場合の周波数エッジで、`Rp` と `Rs` はパスバンドにおけるリップルの最大損失とストップバンドにおける最小リップル減衰をdBで表します。パスバンドとストップバンドのエッジの順序に基づいて、ローパスまたはハイパスフィルタのタイプが推測されます。`N` は最低フィルタ次数を示す整数で、`ωn` は「-3 dB」カットオフ周波数を指定します。ドメインが `:s` と指定されている場合、パスバンドおよびストップバンドの周波数はラジアン/秒として解釈され、アナログフィルタの次数と自然周波数が得られます。デフォルトのドメインは `:z` で、入力周波数は0から1までの正規化された値として解釈され、1はπラジアン/サンプルに対応します。
