```
(N, ωn) = ellipord(Wp::Real, Ws::Real, Rp::Real, Rs::Real; domain::Symbol=:z)
```

エリプティック（カウアー）フィルタの整数および自然周波数次数の推定。`Wp` と `Ws` は通過帯域と阻止帯域の周波数エッジを示し、`Rp` と `Rs` は通過帯域での最大損失と阻止帯域での最小減衰（dB単位）を示します。通過帯域と阻止帯域のエッジの順序に基づいて、ローパスまたはハイパスフィルタのタイプが推測されます。`N` は所望の仕様を達成するための最小の整数フィルタ次数を示し、`ωn` にはフィルタの自然周波数（この場合、単に通過帯域エッジ）が含まれます。`:s` のドメインが指定されると、通過帯域と阻止帯域のエッジはラジアン/秒として解釈され、アナログフィルタの次数と自然周波数の結果が得られます。デフォルトのドメインは `:z` で、入力周波数は0から1までの正規化として解釈され、1はπラジアン/サンプルに対応します。
