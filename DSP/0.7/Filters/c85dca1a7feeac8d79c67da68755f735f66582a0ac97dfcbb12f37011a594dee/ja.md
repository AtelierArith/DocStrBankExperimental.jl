```
(N, ωn) = buttord(Wp::Tuple{Real,Real}, Ws::Tuple{Real,Real}, Rp::Real, Rs::Real; domain=:z)
```

バターワースの次数推定は、バンドパスおよびバンドストップフィルタタイプに対して行われます。 `Wp` と `Ws` は、2要素の通過帯域および阻止帯域の周波数エッジであり、`Rp` dB の通過帯域リップルを超えず、少なくとも `Rs` dB の阻止帯域減衰を持ちます。通過帯域とバンドストップエッジの順序に基づいて、バンドストップまたはバンドパスフィルタタイプが推測されます。 `N` は、推定されたフィルタ次数の最小整数を示し、`ωn` はカットオフまたは "-3 dB" 周波数を指定します。 `:s` のドメインが指定されている場合、通過帯域および阻止帯域の周波数はラジアン/秒として解釈され、アナログフィルタの次数と自然周波数が得られます。デフォルトのドメインは `:z` で、入力周波数は 0 から 1 までの正規化として解釈され、1 は π ラジアン/サンプルに対応します。
