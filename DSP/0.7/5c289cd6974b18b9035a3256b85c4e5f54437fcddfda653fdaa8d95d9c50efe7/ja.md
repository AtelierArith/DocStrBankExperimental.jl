```
kernel = diric(Ω::Real, n::Integer)
```

ディリクレカーネル、または周期的シンク関数として知られ、`n`は正の整数である必要があります。通常、`sin(Ω * n/2) / (n * sin(Ω / 2))`を返しますが、`Ω`が2πの倍数である場合は`±1`になります。

通常、`n`が奇数の場合、出力は$1/n \sum_{k=-(n-1)/2}^{(n-1)/2} e^{i k Ω}$に相当し、これは`n`点の移動平均フィルタの離散時間フーリエ変換（DTFT）です。

`n`が奇数または偶数の場合、関数はそれぞれ2πまたは4π周期です。一般の`n`に対する公式は`diric(Ω,n) =``e^{-i (n-1) Ω/2}/n \sum_{k=0}^{n-1} e^{i k Ω}``であり、これは`Ω`の実数かつ対称的な関数です。

2021年3月19日現在、ウィキペディアの定義には異なる係数があります。ここでの定義はscipyや他のソフトウェアフレームワークと一致しています。

# 例

```jldoctest
julia> round.(diric.((-2:0.5:2)*π, 5), digits=9)'
1×9 adjoint(::Vector{Float64}) with eltype Float64:
 1.0  -0.2  0.2  -0.2  1.0  -0.2  0.2  -0.2  1.0

julia> diric(0, 4)
1.0
```
