```
xcorr(u,v; padmode = :none)
```

2つのベクトルの相互相関を計算します。これは、さまざまなオフセットの `v` に対する `u` と `v` の類似性を計算することによって行われます。`u` を `v` に対して遅延させると、結果は右にシフトします。

出力のサイズは、padmode キーワード引数によって異なります。padmode = :none の場合、結果の長さは length(u) + length(v) - 1 になります。これは conv と同様です。padmode = :longest の場合、引数の短い方がパディングされて等しい長さになります。これにより、長さ 2*max(length(u), length(v))-1 の結果が得られ、ゼロ遅延条件が中心に配置されます。
