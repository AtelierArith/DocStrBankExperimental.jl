```
periodogram(s; onesided=eltype(s)<:Real, nfft=nextfastfft(n), fs=1, window=nothing)
```

信号の周期グラムをFFTによって計算し、Periodogramオブジェクトを返します。

実数信号の場合、二重側の周期グラムは対称であり、この関数はデフォルトで片側（実数のみ）の周期グラムを返します。二重側の周期グラムは、`onesided=false`を設定することで取得できます。

`nfft`はフーリエ変換に使用するポイントの数を指定します。もし`length(s)` < `nfft`の場合、入力はゼロでパディングされます。デフォルトでは、`nfft`はフーリエ変換が最大効率で計算できる最も近いサイズです。

`fs`は元の信号のサンプルレートであり、`window`はフーリエ変換を計算する前に元の信号に適用されるオプションのウィンドウ関数またはベクトルです。計算された周期グラムは、周期グラムの下の面積が元の信号の非中心化分散（または平均パワー）に等しくなるように正規化されています。
