カットプルーナーは、`size(A, 1)`（`== length(b)`）カットを表す行列 `A` とベクトル `b` を維持します。`lazy_minus` が真の場合、`a_i` は `-A[i,:]` であり、そうでない場合は `A[i,:]` です。また、`β_i` は `b[i]` であり、カットの意味はセンスによって異なります。`sense` が

  * `:Min` の場合、カットプルーナーは凹多面体関数 `min ⟨a_i, x⟩ + β_i` を表します；
  * `:Max` の場合、カットプルーナーは凸多面体関数 `max ⟨a_i, x⟩ + β_i` を表します；
  * `:≤` の場合、カットプルーナーは半空間 `⟨a_i, x⟩ ≤ β_i` の交差によって定義される多面体を表します；
  * `:≥` の場合、カットプルーナーは半空間 `⟨a_i, x⟩ ≥ β_i` の交差によって定義される多面体を表します。

内部的には、`sense` の代わりにブール値 `isfun` と `islb` が保存されます。`sense` とこれらの2つのブール値の間のマッピングは、次の表によって示されます。

| `sense` | `isfun` | `islb` |
| -------:| -------:| ------:|
|     Min |    true |  false |
|     Max |    true |   true |
|       ≤ |   false |  false |
|       ≥ |   false |   true |
