```
SeisSort(in, out;<keyword arguments>)
```

ヘッダーの単語を使用してseisファイルをソートします

# 引数

  * `in`: 入力ファイル名 >> データの範囲、データおよびヘッダーファイル名に関する情報を含むテキストファイル; データを含むバイナリファイルおよびヘッダーを含むバイナリファイル。
  * `out`: 出力ファイル名

# キーワード引数

  * `key=["imx","imy"]`
  * `rev=false` : ヘッダーを降順にソート
  * `ntrace=1000` : 一度に読み取るトレースの数

# 出力

ファイル `out` が作成され、データがソートされます。
