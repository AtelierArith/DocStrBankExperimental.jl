```julia
mutable struct Tikhonov <: Conditioner
    α
    threaded 
```

**ティホノフ正則化**コンディショナーのための可変構造体。

正定値行列のマニフォールドにおける点の集合$𝐏$が与えられたとき、次のように集合を変換します。

$$
P_j+αI, \ j=1,...,k
$$

、

ここで、$I$は単位行列であり、$α$は非負の数です。

このコンディショナー構造体には2つのフィールドがあります：

  * `.α`、これはデータにフィットしたときに構造体に書き込まれます。
  * `.threaded`、変換がマルチスレッドモードで行われるかどうかを決定します（デフォルトではtrue）。

インスタンスを構築するために、`α`は引数として、`threaded`はオプションのキーワード引数として指定します。

!!! warning "これはデータ駆動型のコンディショナーではありません"
    `α`パラメータは構築時に明示的に指定する必要があります（デフォルトでは0です）。


**例**：

```julia
using PosDefManifoldML, PosDefManifold

# コンディショナーを作成
T = Tikhonov(0.001)
T = Tikhonov(0.001; threaded=false)
```
