```julia
function rescale!(X::Matrix{T},	bounds::Tuple=(-1, 1);
		dims::Int=1) where T<:Real
```

実数行列 `X` の列または行を範囲 [a, b] に再スケーリングします。ここで、a と b はタプル `bounds` の最初と二番目の要素です。

デフォルトでは、列に適用されます。行を再スケーリングするには `dims=2` を使用します。

この関数は接線空間（特徴）ベクトルの正規化に使用されます。通常、これを必要とすることはありません。モデルを [`fit`](@ref) で適合させたり、[`crval`](@ref) で交差検証を行ったりする際には、これらの関数の引数 `rescale` に単に `bounds` を渡すことができます。
