```julia
function confusionMat(yTrue::IntVector, yPred::IntVector)
```

整数ベクトルの真のラベル `yTrue` と予測ラベル `yPred` を与えられたとき、*頻度*（カウント）を表す*混同行列*を返します。

`yTrue` と `yPred` の長さは等しくなければなりません。さらに、`yTrue` ベクトルは 1 から *z* までのすべての自然数を含む必要があります。ここで *z* はクラスの数です。

混同行列のサイズは *z* になります。これは、すべてのエントリがゼロで埋められた行列から始まり、各ラベルについて、真のラベルが `i` で予測ラベルが `j` のとき、行列のエントリ [`i`, `j`] に 1 を加えることで計算されます。したがって、最初の行はクラス 1 の真のラベルを報告し、2 行目はクラス 2 の真のラベルを報告します。

返される行列は整数の行列です。

**参照** [`predict`](@ref), [`predictAcc`](@ref), [`predictErr`](@ref)

**例**

```julia
using PosDefManifoldML
confusionMat([1, 1, 1, 2, 2], [1, 1, 1, 1, 2])
# return: [3 0; 1 1]
```
