```
@geometric_space name sig quote
  n = 1.0::e4 + 1.0::e5
  n̄ = (-0.5)::e4 + 0.5::e5
  ... # 他の定義
end [warn_override = true]
```

マクロ `@name` を生成し、署名 `sig` を持つ幾何空間を定義し、オプションのユーザー提供の定義を追加します。

結果として得られるマクロには二つのメソッドがあります：

  * `@name ex` は標準の `@ga sig ex` と同様に動作します（ただし、追加の定義がある可能性があります）。
  * `@name T ex` は結果を型 `T` にラップします。これは `@ga sig T ex` が行うのと同様です（[`@ga`](@ref) を参照）。

定義が `Expr` として提供される場合、それらは [`Bindings`](@ref) に解析され、デフォルトのバインディングに追加されます。定義がすでに [`Bindings`](@ref) の場合、それらはそのまま使用されます。`warn_override` は、デフォルトのバインディングの一部を意図的に上書きする場合は false に設定できます。
