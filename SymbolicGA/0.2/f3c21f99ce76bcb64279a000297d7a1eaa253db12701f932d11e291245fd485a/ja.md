```
Bindings(; refs = Dict{Symbol,Any}(), funcs = Dict{Symbol,Any}(), warn_override = true)
```

バインディングに関する情報を保持する構造体で、これは参照（単純な置換）または引数を持つ関数として機能します。これにより、代数式を構築する際に小さなドメイン特化型言語を使用することができます。

参照は `lhs::Symbol => rhs` のペアであり、左辺は単に解析中に右辺に展開されます。`lhs` を含む右辺もサポートされており、`x = x::Vector` の形式の参照が許可されていますが、これは一度だけ展開されます。関数は `name::Symbol => body` のペアであり、`rhs` は `Expr(:argument, <literal::Int>)` 表現を使用してその引数を参照しなければなりません。再帰はサポートされておらず、`StackOverflowError` を引き起こします。詳細は [`@arg`](@ref) を参照してください。

ほとんどのデフォルト関数とシンボルはこのメカニズムを使用して実装されています。`warn_override` が true に設定されている場合、そのようなデフォルト関数のオーバーライドは警告を引き起こします。
