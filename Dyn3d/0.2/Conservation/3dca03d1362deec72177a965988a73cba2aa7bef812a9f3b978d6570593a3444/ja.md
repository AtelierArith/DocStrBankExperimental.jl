```
force!(dict,bd,λ,k)
```

  * `dict` : エネルギー保存の項の構造。 "mlag", "mlag*temp", "mgra", "mgra*temp", "mkin" のキーを含む
  * `bd` : 現在の BodyDyn 構造
  * `λ` : ジョイントの制約された自由度に対する体による力
  * `k` : k-1 は rk スキームの現在のステージを表す

アクティブに指定された動きの下にあるボディ-ジョイントシステムの rk スキームのタイムステップの1つのステージで慣性フレームにおけるラグランジュ乗数による力を計算します。また、1つのステージで重力を計算します。この関数は、RK スキームのすべてのステージで呼び出す必要があります。
