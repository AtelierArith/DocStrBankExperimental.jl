オブジェクトからフィールドを抽出するためのマクロ。例えば、次のような文の代わりに

```
(obj.a + obj.b)^obj.c
```

最初に変数をローカルにバインドする方が読みやすいです。

```
a = obj.a
b = obj.b
c = obj.c

(a + b)^c
```

`@getfield`マクロを使用すると、これが次のようになります。

```
@getfield obj (a, b, c)

(a + b)^c
```

バインディングに異なる名前をローカルに割り当てることもできます。

```
@getfield obj (a, b, c) (α, β, γ)
(α + β)^γ
```
