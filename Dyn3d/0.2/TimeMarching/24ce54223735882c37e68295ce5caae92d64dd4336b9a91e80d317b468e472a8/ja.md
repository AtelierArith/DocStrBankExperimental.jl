HERKBodyは、V.BraseyとE.Hairerの論文に基づく制約付きボディモデルに基づいた半明示的ルンゲクッタソルバーです。
次のODEシステムが解かれています：
   | dq/dt = v                     |
   | M(q)*dv/dt = f(q,v) - GT(q)*λ |
   | 0 = G(q)*v + gti(q)           |
ここで、GTはGの転置に似ています。

これは、2つの変数qとuを持つインデックス2のシステムであることに注意してください。
ここでは、ボディチェーンに対して、vはボディの速度を表し、vJはジョイントの速度を表します。同様に、qとqJについても同様です。λはqに対して満たされるべき制約です。

特にダイナミクスの問題に対して、qJとvを解くことにします。
方程式のシステムは実際には次のようになります：
   | dqJ/dt = vJ                              |
   | M(qJ)*dv/dt = f(qJ,v,vJ) - GT(qJ)*lambda |
   | 0 = G(qJ)*v + gti(qJ)                    |
したがって、解決されたvJからvを計算するステップが必要です。運動制約（指定されたアクティブモーション）は、ボディではなくジョイントに従います。

ここで、方程式のシステムを一般的な形で書きます：
[A B₁ᵀ] * [v] = [r₁]
[B₂ 0 ]   [λ]   [r₂]
したがって、A = M(qJ)、B₁ᵀ = GT(qJ)、B₂ = G(qJ)、r₁ = f(qJ,v,vJ)、r₂ = -gti(qJ)
dqJ/dt = vJの方程式はHERKで更新されます。
