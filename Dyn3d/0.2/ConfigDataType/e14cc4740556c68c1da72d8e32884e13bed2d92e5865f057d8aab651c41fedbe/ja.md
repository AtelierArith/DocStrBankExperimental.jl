```
ConfigBody(nbody::Int,nverts::Int,verts::Matrix{Float64},ρ::Float64)
```

ボディシステム内の単一ボディの構成情報を設定します。ここでは、複数のボディが存在する場合、すべてのボディが同じ形状を持つと仮定します。単一ボディはy方向に無限に薄いボディです。多角形の形状を持ち、z-x空間で記述される必要があります。たとえば、z-x空間で長方形を記述する場合、3次元問題ではそれで問題ありません。2次元問題のx-y空間では、この長方形はx-y空間での線の投影を持ちます。頂点のローカル座標は慣例として時計回りの方向で記述されます。

## フィールド

  * `nbody`: ボディの総数
  * `nverts`: 1つのボディの頂点の数
  * `verts`: 左下の頂点から始まり、時計回りの方向に進む多角形の頂点座標。各行はこの頂点の(z,x)座標を記述します。通常、2次元問題ではz次元は単位長さ1です。
  * `ρ`: このボディの面積あたりの質量密度

## ボディの設定

```
     ^(y)
     |
     |----->(x)
    /
 (-z)
```
