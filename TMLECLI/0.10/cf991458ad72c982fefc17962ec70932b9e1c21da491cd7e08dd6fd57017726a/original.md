```
AdaptiveStratifiedCV(;shuffle=nothing, rng=nothing)
```

A StratifiedCV (see `MLJBase.StratifiedCV`) resampling strategy where the number of folds is determined  data adaptively based on the rule of thum described [here](https://academic.oup.com/ije/advance-article/doi/10.1093/ije/dyad023/7076266).
