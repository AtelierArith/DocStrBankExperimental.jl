```
AdaMax(η = 0.001, β = (0.9, 0.999), ϵ = 1e-8)
AdaMax(; [eta, beta, epsilon])
```

[AdaMax](https://arxiv.org/abs/1412.6980)は、∞-ノルムに基づくAdamの変種です。

# パラメータ

  * 学習率（`η == eta`）：重みを更新する前に勾配が割引かれる量。
  * モーメンタムの減衰（`β::Tuple == beta`）：最初の（β1）および2番目の（β2）モーメント推定のための指数的減衰。
  * マシンイプシロン（`ϵ == epsilon`）：ゼロ除算を防ぐための定数（デフォルトを変更する必要はありません）
