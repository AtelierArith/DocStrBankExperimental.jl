```
Rprop(η = 1e-3, ℓ = (0.5, 1.2), Γ = (1e-6, 50.0))
Rprop(; [eta, ell, gamma])
```

[Rprop](https://ieeexplore.ieee.org/document/298623)アルゴリズムを使用したオプティマイザ。勾配の符号のみに依存するフルバッチ学習アルゴリズムです。

# パラメータ

  * 学習率（`η == eta`）：重みを更新する前に勾配が割引かれる量。
  * スケーリングファクター（`ℓ::Tuple == ell`）：乗算的な増加および減少の因子。
  * ステップサイズ（`Γ::Tuple == gamma`）：許可される最小および最大のステップサイズ。
