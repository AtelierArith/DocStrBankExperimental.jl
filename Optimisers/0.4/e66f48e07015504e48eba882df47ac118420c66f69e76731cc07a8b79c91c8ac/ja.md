```
AdaBelief(η = 0.001, β = (0.9, 0.999), ϵ = 1e-16)
AdaBelief(; [eta, beta, epsilon])
```

[AdaBelief](https://arxiv.org/abs/2010.07468)オプティマイザは、よく知られたAdamオプティマイザのバリアントです。

# パラメータ

  * 学習率（`η == eta`）：勾配が重みを更新する前にどの程度割引かれるか。
  * モーメンタムの減衰（`β::Tuple == beta`）：最初の（β1）および2番目の（β2）モーメンタム推定のための指数的減衰。
  * マシンイプシロン（`ϵ == epsilon`）：ゼロ除算を防ぐための定数（デフォルトを変更する必要はありません）
