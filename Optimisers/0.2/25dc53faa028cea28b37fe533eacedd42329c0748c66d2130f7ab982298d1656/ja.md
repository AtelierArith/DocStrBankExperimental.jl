```
RMSProp(η = 1f-3, ρ = 9f-1, ϵ = eps(typeof(η)); centred = false)
```

[RMSProp](https://www.cs.toronto.edu/~tijmen/csc321/slides/lecture_slides_lec6.pdf)アルゴリズムを使用するオプティマイザ。再帰的ネットワークに対してしばしば良い選択肢です。学習率以外のパラメータは一般的に調整する必要はありません。

[Centred RMSProp](http://arxiv.org/abs/1308.08500)は、勾配の二次モーメントではなく、分散の推定値によって勾配を正規化する変種です。

# パラメータ

  * 学習率（`η`）：重みを更新する前に勾配が割引かれる量。
  * モーメンタム（`ρ`）：勾配降下の主要な方向への加速を制御し、実質的に振動を抑えます。
  * マシンイプシロン（`ϵ`）：ゼロ除算を防ぐための定数（デフォルトを変更する必要はありません）
  * キーワード `centred`（または `centered`）：アルゴリズムの中心化された変種を使用するかどうかを示します。
