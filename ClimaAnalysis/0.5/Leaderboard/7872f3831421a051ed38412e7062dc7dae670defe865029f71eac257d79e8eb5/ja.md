```
read_rmses(csv_file::String, short_name::String; units = nothing)
```

CSVファイルを読み込み、変数の`short_name`を持つRMSEVariableを作成します。

CSVファイルの形式は、"model_name"（または関数によって無視される他の任意のテキスト）というエントリからなるヘッダーを持ち、残りのエントリはカテゴリ名である必要があります。ヘッダーの後の各行は、モデル名とそのモデルの各カテゴリに対する二乗平均平方根誤差で始まる必要があります。CSVファイルのエントリはカンマで区切られている必要があります。

パラメータ`units`は、モデル名を単位にマッピングする辞書または文字列である可能性があります。`units`が文字列の場合、すべてのモデルで単位は同じになります。`units`が`nothing`の場合、単位は各モデルに対して欠落しており、空の文字列で示されます。 ```
