```
resampled_as(src_var::OutputVar, kwargs...)
```

`src_var`の`data`をキーワード引数で定義された次元に再サンプリングします。

ここで行われる再サンプリングは1次線形再サンプリングです。

例えば、`[0.0, 1.0, 2.0]`の経度次元で再サンプリングするには、`resampled_as(src_var, lon = [0.0, 1.0, 2.0])`を実行できます。

`dims`と`src_var.dims`の次元が一致する場合（順序は無視）、結果の`OutputVar`は`dims`の順序に合わせて次元が再配置されます。そうでない場合、結果の`OutputVar`の次元は変更されません。`dims`の次元が`src_var`の次元の厳密な部分集合である場合、部分的な再サンプリングが行われます。

次元の単位に対するチェックは行われないことに注意してください。
