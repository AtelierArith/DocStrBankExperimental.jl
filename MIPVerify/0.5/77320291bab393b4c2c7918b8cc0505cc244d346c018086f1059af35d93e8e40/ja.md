```julia
struct SkipBlock <: MIPVerify.Layer
```

スキップ接続パターンを実装するレイヤーで、残差ネットワーク（ResNet）で一般的に使用されます。

`SkipBlock`は複数の入力テンソルを受け取り、それぞれの入力に対応する変換レイヤーを適用します。これらの変換の出力は、要素ごとの加算を通じて結合されます。

[`SkipSequential`](@ref)内で使用されると、`SkipBlock`はネットワーク内の最新のレイヤーからの入力を処理します。ブロックに`n`レイヤーがある場合、最後の`n`レイヤーからの出力を順番に入力として受け取ります。たとえば、2層の`SkipBlock`の場合：

  * 2層目は直前のレイヤーからの出力を処理します
  * 1層目は2層前の出力を処理します

通常、スキップ接続を持つネットワークを作成するために[`SkipSequential`](@ref)と組み合わせて使用され、早いレイヤーからの出力が中間レイヤーをバイパスして後のレイヤーの出力と結合されることができます。

例：

```julia
skip_block = SkipBlock([
    Linear(input_size_1, output_size),  # 1つのパスからの変換
    Linear(input_size_2, output_size)   # 別のパスからの変換
])
```
