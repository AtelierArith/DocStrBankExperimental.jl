```
IPPSExcludedProviderCCN
IPPSExcludedProviderCCN(n::AbstractString)
```

入院前向き支払い制度（IPPS）から除外されたメディケアまたはメディケイドの提供者を表す型です。

IPPS除外提供者は、次の形式の6文字の識別子を使用します：

> `SSTAQQ`


ここで：

  * `SS` は2文字の英数字の州コードを表します；
  * `T` はアルファベットの施設タイプコードを表します；
  * `A` は、IPPS除外の親施設のIPPS除外ユニットの場合のアルファベットの親施設タイプコード、またはシーケンス番号の最上位桁を表します；
  * `QQ` はシーケンス番号の2つの最下位桁を表します。

!!! note
    IPPS除外提供者は常に親施設のサブユニットであり、そのため独自のCCNシーケンス番号は割り当てられません。CCNのシーケンス番号は親施設の最下位桁と一致します。


コンストラクタはエラーチェックを行いませんが、`n` が7文字を超える場合は例外をスローします。
