```
conformal_cubed_sphere_inverse_mapping(X, Y, Z; Z_map=Z_Rancic)
```

北極面の象限に対する準同型立方体球の逆マッピングで、`X` と `Y` の両方が正です。このマップの回転から他の立方体面座標へのすべての他のマッピングを復元できます。北極面には他に3つの象限と5つの他の面があり、合計で24の象限があります。対称性のため、単一の象限の逆だけが必要です。分岐切断と複素変換のため、逆マッピングは一般に多値ですが、単一の象限ケースを使用することで、適用可能な単純なルールのセットを持つことができます。

このマッピングは、$0 < x < 1$ および $0 < y < 1$ で定義される立方体面の象限に対して有効であり、完全な立方体面は $-1 < x < 1$ および $-1 < y < 1$ の範囲を持ちます。マッピングの象限は、中心に「北極」がある立方体面からのもので、$(x=0, y=0)$ に位置します。すなわち、中心に `X, Y, Z = (0, 0, 1)` を持ちます。このマッピングと慣習に対する `X` と `Y` の有効範囲は、半径 $R$ の球の殻上の点 A, B, C, D を結ぶ測地線によって定義される象限です。座標は次のようになります。

```
A = (0, 0)
B = (√2, 0)
C = (√3/3, √3/3)
D = (0, √2)
```
