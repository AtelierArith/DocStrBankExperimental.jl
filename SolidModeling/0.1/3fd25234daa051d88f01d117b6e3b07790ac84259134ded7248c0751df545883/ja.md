bintersect(first::Solid, second::Solid)

`A` と `B` の交差を計算した後、新しいソリッドを返します。

```
+-------+
|       |
|   A   |
|    +--+----+   =   +--+
+----+--+    |       +--+
     |   B   |
     |       |
     +-------+
```

# 例

```julia
c1 = cube(0.5, 0.5, 0.5, 1.0, 1.0, 1.0)
c2 = cube(0.5, 0.5, 0.5, 2.0, 2.0, 2.0)
r = bintersect(c1, c2) # c1 と c2 の交差

v = volume(r)
```
