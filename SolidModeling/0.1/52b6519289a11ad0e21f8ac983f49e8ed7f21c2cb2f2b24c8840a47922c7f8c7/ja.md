```
bsubtract(first::Solid, second::Solid)
```

`B`-`A`を実行した後の新しいソリッドを返します。

```
+-------+            +-------+
|       |            |       |
|   A   |            |       |
|    +--+----+   =   |    +--+
+----+--+    |       +----+
     |   B   |
     |       |
     +-------+
```

# 例

```julia
c1 = cube(0.5, 0.5, 0.5, 1.0, 1.0, 1.0)
c2 = cube(0.5, 0.5, 0.5, 2.0, 2.0, 2.0)
r = bsubtract(c1, c2) # c2をc1から引く

v = volume(r) # c1-c2の体積
```
