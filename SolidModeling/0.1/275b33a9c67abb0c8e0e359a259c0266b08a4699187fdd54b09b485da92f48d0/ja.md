```
bunion(first::Solid, second::Solid)
```

`A` と `B` の和集合の後に新しいソリッドを返します。

```
+-------+            +-------+
|       |            |       |
|   A   |            |       |
|    +--+----+   =   |       +----+
+----+--+    |       +----+       |
     |   B   |            |       |
     |       |            |       |
     +-------+            +-------+
```

# 例

```julia
c1 = cube(0.5, 0.5, 0.5, 1.0, 1.0, 1.0)
c2 = cube(0.5, 0.5, 0.5, 2.0, 2.0, 2.0)
r = bunion(c1, c2) # ソリッド c1 と c2 の和集合

v = volume(r) # 和集合 c1 c2 の体積
```
