```
BayesianNODE(data,X;kwargs ... )
```

データフレーム `X` が提供されると、モデルは共変量と共に実行されます。引数 `X` には、時間 `t` の列があり、残りの列には時間の値が含まれている必要があります。`X` の値は、データフレームに含まれていない時間の値について線形スプラインで補間されます。

# kwargs

  * `time_column_name`: `data` と `X` における時間に対応する列の名前。デフォルトは `"time"`。
  * `variable_column_name`: `X` における変数に対応する列の名前。デフォルトは `nothing`。
  * `value_column_name`: `X` における共変量に対応する列の名前。デフォルトは `nothing`。
  * `hidden_units`: 隠れ層のニューロンの数。デフォルトは `10`。
  * `seed`: 再現可能な結果のための固定ランダムシード。デフォルトは `1`。
  * `proc_weight`: プロセス誤差 $omega_{proc}$ の重み。デフォルトは `1.0`。
  * `obs_weight`: 観測誤差 $omega_{obs}$ の重み。デフォルトは `1.0`。
  * `reg_weight`: 正則化誤差 $omega_{reg}$ の重み。デフォルトは `10^-6`。
  * `reg_type`: 正則化のタイプ、`"L1"` または `"L2"` 正則化のいずれか。デフォルトは `"L2"`。
  * `l`: 予測のための外挿パラメータ。デフォルトは `0.25`。
  * `extrap_rho`: 予測のための外挿パラメータ。デフォルトは `0.0`。
