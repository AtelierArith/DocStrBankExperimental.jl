```
SimpleNeuralNetwork(inputs,outputs; kwargs ...)
```

ニューラルネットワークオブジェクトを構築し、ランダムに初期化されたパラメータ値を返します。ニューラルネットワークオブジェクトは、ベクトル `x` とニューラルネットワークの重みとバイアスを持つ名前付きタプル `parameters` の2つの引数を取る関数として評価できます。  

```
# kwargs
```

  * `hidden`: 隠れ層のニューロンの数。デフォルトは10です。
  * `nonlinearity`: ニューラルネットワークで使用される活性化関数。デフォルトは双曲線正接関数 `tanh` です。
  * `seed`: ニューラルネットワークの重みを初期化するための乱数生成器のシード。
