```
leave_future_out(model::UDE, training!, k; kwargs... )
```

UDeモデル`model`に対して、トレーニングルーチン`train!`を使用して`k`フォールドの将来除外交差検証を実行します。パスキーワードを使用してCSVファイルへのパスが提供されると、各フォールドの生のテストデータと予測が保存されます。

この関数は3つのデータフレームを返します。最初のデータフレームには、予測の平均絶対誤差の推定値と、予測のホライズン（未来の1からk時間ステップ）に対する関連する標準誤差が含まれています。2番目と3番目は、`horizon_by_var`と`raw`の2つの要素を持つ名前付きタプルで返されます。データフレーム`horizon_by_var`には、変数ごとに分けられた予測誤差が含まれ、データフレーム`raw`には生のテストおよび予測データが含まれています。モデルが複数の時系列でトレーニングされている場合、名前付きタプルには3番目のデータフレーム`horizon_by_var_by_series`が含まれます。

```
# kwargs
```

  * `path`: 出力データフレームを保存するディレクトリへのパス、デフォルトはNull
