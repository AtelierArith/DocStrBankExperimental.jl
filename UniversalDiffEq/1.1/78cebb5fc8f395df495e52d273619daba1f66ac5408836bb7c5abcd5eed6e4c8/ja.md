```
NODE(data;kwargs ... )
```

データセット `data` のための非パラメトリック連続時間モデルを構築し、システムのダイナミクスを表現するために単層ニューラルネットワークを使用します。

```
# kwargs
```

  * `time_column_name`: `data` の中で時間に対応する列の名前。デフォルトは `"time"`。
  * `hidden_units`: 隠れ層のニューロンの数。デフォルトは `10`。
  * `seed`: 再現可能な結果のための固定ランダムシード。デフォルトは `1`。
  * `proc_weight`: プロセス誤差 $omega_{proc}$ の重み。デフォルトは `1.0`。
  * `obs_weight`: 観測誤差 $omega_{obs}$ の重み。デフォルトは `1.0`。
  * `reg_weight`: 正則化誤差 $omega_{reg}$ の重み。デフォルトは `10^-6`。
  * `reg_type`: 正則化のタイプ、`"L1"` または `"L2"` 正則化。デフォルトは `"L2"`。
  * `l`: 予測のための外挿パラメータ。デフォルトは `0.25`。
  * `extrap_rho`: 予測のための外挿パラメータ。デフォルトは `0.0`。
  * `ode_solver`: 微分方程式の解を近似するための方法。デフォルトは `Tsit5()`。
  * `ad_method`: ODEソルバーの導関数を評価するための方法。デフォルトは `ForwardDiffSensitivity()`。
