```
XPRSaddcbchgbranch(prob, cb, priority)
```

# 引数

  * `cbprob`: コールバック関数 `chgbranch` に渡される問題。
  * `entity`: ブランチを行う変数またはセットへのポインタ。整数変数はその列インデックス、すなわち `0`, `1`,...(COLS`- 1`) で識別され、セットはそのセットインデックス、すなわち `0`, `1`,...,(SETS`- 1`) で識別されます。
  * `up`: `p_entity` が変数の場合、上向きのブランチを最初に行う場合は `1`、そうでない場合は `0` です。`p_entity` がセットの場合、上向きのブランチを最初に行う場合は `3`、そうでない場合は `2` です。
  * `estdeg`: この値は廃止されました。ゼロに設定され、返される値は無視されます。

`cb` は次のシグネチャで呼び出されます：

```
cb(cbprob, estdeg)::entity, up, estdeg
```
