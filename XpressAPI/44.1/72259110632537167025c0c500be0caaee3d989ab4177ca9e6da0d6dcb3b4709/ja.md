```
XPRSaddcbpreintsol(prob, cb, priority)
```

# 引数

  * `cbprob`: コールバック関数 `preintsol` に渡される問題。
  * `soltype`: 見つかったMIPソリューションのタイプ: 0 現在のツリー探索のノードに対する連続緩和ソリューションで、整数的に実現可能であることが確認されたもの。 1 ヒューリスティックによって見つかったMIPソリューション。 2 ユーザーによって提供されたMIPソリューション。
  * `reject`: ソリューションを拒否する場合は1に設定します。`soltype`がゼロの場合、これによりノード問題も削除されます。
  * `cutoff`: ソリューションが受け入れられた場合にオプティマイザーが使用する新しいカットオフ値。ユーザーが `p_cutoff` を変更した場合、新しい値が代わりに使用されます。ソリューションが拒否された場合、カットオフ値は更新されません。

`cb` は次のシグネチャで呼び出されます:

```
cb(cbprob, soltype, cutoff)::reject, cutoff
```
