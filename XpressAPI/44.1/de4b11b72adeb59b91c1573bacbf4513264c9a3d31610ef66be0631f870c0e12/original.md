```
XPRS_bo_getid(bo)::id
```

Returns the unique identifier assigned to a branching object.

# Arguments

  * `bo::XPRSbranchobject`: A branching object.

# Return value

  * `id::Int32`: Pointer to an integer where the identifier should be returned.

See also the documentation of the correponding function [XPRS*bo*getid](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRS_bo_getid.html) in the C API.
