```
XPRSgetversionnumbers()::major, minor, build
```

オプティマイザーのバージョン番号をメジャー、マイナー、ビルド番号に分割して返します。

# 戻り値

  * `major::Int32`: メジャーバージョン番号を受け取る整数へのポインタ。
  * `minor::Int32`: マイナーバージョン番号を受け取る整数へのポインタ。
  * `build::Int32`: ビルド番号を受け取る整数へのポインタ。

C APIの対応する関数のドキュメントも参照してください [XPRSgetversionnumbers](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRSgetversionnumbers.html)。
