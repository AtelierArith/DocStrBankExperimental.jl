```
XPRS_SOLVESTATUS
```

Status of the solve of the last problem solved with XPRSoptimize. (integer)

Values: 0 : `XPRS_SOLVESTATUS_UNSTARTED` The solve has not been started. 1 : `XPRS_SOLVESTATUS_STOPPED` Optimization has been interrupted. 2 : `XPRS_SOLVESTATUS_FAILED` Optimization has run into a nonrecoverable problem and failed. 3 : `XPRS_SOLVESTATUS_COMPLETED` Search completed.
