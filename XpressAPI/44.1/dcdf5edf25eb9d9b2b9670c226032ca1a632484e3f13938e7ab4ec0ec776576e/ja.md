```
XPRS_HEUREMPHASIS
```

分枝と境界: この制御は、プライマルヒューリスティクスおよびプライマル-デュアルギャップの収束速度に影響を与える他の手続きに関して、検索の強調を指定します。目標が小さなギャップを達成することであり、必ずしも最適解を求めることではない問題に対しては、`HEUREMPHASIS`を1に設定することをお勧めします。この設定は、検索の初期段階でギャップを減少させることを目指す多くの追加ヒューリスティック呼び出しをトリガーし、通常は最適性を証明するための時間が増加する代償を伴います。

デフォルト値: `-1`

値: -1 : オプティマイザのデフォルト戦略。 0 : すべてのヒューリスティックを無効にします。 1 : 検索の初期部分でプライマル-デュアルギャップを減少させることに焦点を当てます。 2 : 非常に攻撃的な検索ヒューリスティクス。
