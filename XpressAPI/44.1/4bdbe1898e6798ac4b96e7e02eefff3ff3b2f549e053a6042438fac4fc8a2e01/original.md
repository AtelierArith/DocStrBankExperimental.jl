```
XPRS_ge_getcomputeallowed()::allow
```

Query whether the current application is allowed to use the Insight Compute interface.

# Return value

  * `allow::XPRSAllowCompute`: Memory location in which the value will be returned.

See also the documentation of the correponding function [XPRS*ge*getcomputeallowed](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRS_ge_getcomputeallowed.html) in the C API.
