```
XPRS_CLAMPING
```

この制御は、返される解の値が常に範囲内に収まるように調整することを可能にします。（整数）

デフォルト値: `0`

値はビットセットです: ビット -1 : 自動的に決定されます。ビット 0 : プライマル解を常にプライマル境界内に収めるように調整します。提供されたスラックはそれに応じて調整されます。ビット 1 : プライマルスラック値を常に制約境界内に収めるように調整します。ビット 2 : デュアル解を常にスラックによって暗示されるデュアル境界内に収めるように調整します。提供された場合、削減コストはそれに応じて調整されます。ビット 3 : 削減コストを常にプライマル解によって暗示されるデュアル境界内に収めるように調整します。
