```
XPRS_bo_getbranches(bo)::nbranches
```

ブランチオブジェクトのブランチ数を返します。

# 引数

  * `bo::XPRSbranchobject`: 検査するユーザーブランチオブジェクト。

# 戻り値

  * `nbranches::Int32`: ブランチ数が返されるメモリ。

C APIの対応する関数のドキュメントも参照してください [XPRS*bo*getbranches](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRS_bo_getbranches.html)。
