```
XPRS_COMPUTELOG
```

Controls how the run log is fetched when a solve is performed on an Insight Compute Interface. (integer)

Default value: 1

Values: 0 : Run log will not be fetched 1 : Run log will be fetched in real-time 2 : Run log will be fetched at the end of the solve 3 : Run log will be fetched at the end of the solve if the solve fails with an error
