```
XPRSdumpcontrols(prob)::prob
```

現在の値がデフォルト値以外に設定されている制御のリストとその値を表示します。

# 引数

  * `prob::XPRSprob`: 制御がダンプされる問題。

# 戻り値

  * `prob::XPRSprob`: 制御がダンプされる問題。

C APIの対応する関数[XPRSdumpcontrols](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRSdumpcontrols.html)のドキュメントも参照してください。
