```
XPRSsetintcontrol64(prob, control, value)::prob
```

指定された整数制御パラメータの値を設定します。

# 引数

  * `prob::XPRSprob`: 現在の問題。
  * `control::Integer`: 値を設定する制御パラメータ。
  * `value::Integer`: 制御パラメータに設定する値。

# 戻り値

  * `prob::XPRSprob`: 現在の問題。

C APIの対応する関数[XPRSsetintcontrol64](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRSsetintcontrol64.html)のドキュメントも参照してください。
