```
XPRSaddcbmessage(prob, cb, priority)
```

# 引数

  * `cbprob`: コールバック関数に渡される問題。
  * `msg`: メッセージを含むヌル終端文字配列（文字列）で、単に改行である場合もあります。合計バイト数（`NUL` 終端を含む）は `XPRS_MAXMESSAGELENGTH` を超えません。この制限を満たすためにメッセージを切り詰める必要がある場合、`msg` の最後の4バイトは "...0" に設定されます。
  * `msglen`: ヌル終端を除くメッセージ文字列の長さ。
  * `msgtype`: 出力メッセージのタイプを示します：1情報メッセージ; 2（使用されていない）; 3警告メッセージ; 4エラーメッセージ。負の値は、オプティマイザが終了しようとしており、出力がファイルにリダイレクトされている場合はこの時点でバッファをフラッシュする必要があることを示します。

`cb` はこのシグネチャで呼び出されます：

```
cb(cbprob, msg, msglen, msgtype)::Nothing
```
