```
XPRSiisall(prob)::prob
```

不 feasibleな問題における独立した不可約不 feasible集合 (IIS) の自動検索を実行します。

# 引数

  * `prob::XPRSprob`: 現在の問題。

# 戻り値

  * `prob::XPRSprob`: 現在の問題。

C APIの対応する関数のドキュメント[XPRSiisall](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRSiisall.html)も参照してください。
