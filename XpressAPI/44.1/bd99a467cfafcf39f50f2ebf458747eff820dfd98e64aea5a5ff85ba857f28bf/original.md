```
XPRS_ge_setcomputeallowed(allow)::Nothing
```

Set whether the current application is allowed to use the Insight Compute interface.

# Arguments

  * `allow::XPRSAllowCompute`: Whether the Insight Compute interface may be used; must be one of the following constants: XPRS*ALLOW*COMPUTE_ALWAYSAlways allow solves to be sent to Compute.

See also the documentation of the correponding function [XPRS*ge*setcomputeallowed](https://www.fico.com/fico-xpress-optimization/docs/latest/solver/optimizer/HTML/XPRS_ge_setcomputeallowed.html) in the C API.
