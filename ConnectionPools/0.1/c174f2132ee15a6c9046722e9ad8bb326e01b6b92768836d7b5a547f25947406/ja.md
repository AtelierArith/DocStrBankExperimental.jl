free(pool::Pool{T}) where T -> Int

プール内で現在利用可能な無料リソースの数を返します。

この関数は、プール内で即座に取得可能なリソースがいくつあるかを確認するためのスレッドセーフな方法を提供します。現在使用中のリソース（取得済み）は含まれません。

# 引数

  * `pool::Pool{T}`: クエリするリソースプール。

# 戻り値

プール内の無料リソースの数。

# スレッドセーフ

この操作はスレッドセーフです。

# 例

```julia
using Pools

create(::Type{Int}) = rand(1:10)

pool = GenericPool{Int}(3)
free(pool)
```
