```
DiskCache(path)
```

`DiskCache{K,V}(path)` は、`serialize` と `deserialize` を使用してファイルシステムの `path` にマッピングされたキーの型 `K` と値の型 `V` を持つキャッシュを構築します。`DiskCache` はデフォルトでは `Dict` のように振る舞いますが、`DiskCache{K,V,D}` は `D`、すなわち `AbstractDict{K, V}` のように振る舞います。

`DiskCache` は、すでにチップ上にあるキーの高速読み取りと、新しいキーをディスクに書き込む遅い書き込みをサポートしますが、既存のキーの変更はサポートしていません。既存のキーへの書き込みはエラーを引き起こしますが、`get!` を呼び出すことは常に安全です。
