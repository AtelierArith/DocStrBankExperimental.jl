## `fiedler_vector`

隣接行列 A を持つグラフの正規化ラプラシアンに関連するフィードラベクトルを計算します。

この関数は、Float32 および Float64 値の疎行列とベクトルで動作します。

無向グラフの隣接行列を表す対称入力行列が必要です。入力が切断されたネットワークの場合、結果は必ずしもそうではありません。

返されるベクトルは符号付きで、正のエントリの数が負のエントリの数以上になるようにします。これは、重複のない第2固有値の場合に常に一意で決定的な出力を提供します。

## 関数

  * `(v,lam2) = fiedler_vector(A::SparseMatrixCSC{V,Int})`
  * `(v,lam2) = fiedler_vector(A::MatrixNetwork{V,Int})`

## 入力

  * `A`: 隣接行列
  * オプションのキーワード入力

      * `checksym`: 行列が対称であることを確認します
      * `tol`: 固有値、固有ベクトル推定の残差許容誤差。（これは絶対誤差です）
      * `maxiter`: ARPACK の最大反復回数
      * `nev`: ARPACK によって推定される固有ベクトルの数
      * `dense`: 疎（ARPACK）計算の代わりに密（LAPACK）計算のための閾値。行列のサイズが `dense` より小さい場合、maxiter と nev は無視され、LAPACK がすべての固有値を計算します。

## 返り値

  * `v`: 配列としてのフィードラベクトル
  * `lam2`: 固有値そのもの

## 例

```
# n ノードのパスグラフを構築
n = 25
A = sparse(1:n-1,2:n,1.,n,n)
A = A + A'
(v,lam2) = fiedler_vector(A) # コサインのようなフィードラベクトルを返します
# UnicodePlots を使用; lineplot(v)
```
