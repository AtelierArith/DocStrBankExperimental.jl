## `sweepcut`

スイープカットは、グラフの頂点に対する順序を取り、その順序の接頭辞によって誘導されるすべての分割のカットとボリュームを評価する操作です。つまり、頂点の順序が

```
v1, v2, v3, ...
```

であるとき、スイープカットは次のように評価します。

```
cut({v1}), vol({v1})
cut({v1,v2}), vol({v1,v2})
cut({v1,v2,v3}), vol({v1,v2,v3})
...
```

ここで、cutはS内の頂点とグラフを接続する全エッジ重みです。そして、volは接続する全エッジ重みです。

## 関数

  * `profile = sweepcut(A,x::Vector{T})` 頂点の順序を得るためにソートされたベクトルxに対してスイープカットを取得する標準的なスイープカット呼び出しです。これは、グラフ内の良いカットを示すべきベクトル（例えば、フィードラーベクトル）を持っている場合に便利です。
  * `profile = sweepcut(A,p,r,totalvol,maxvol)` 他のすべてが変換される詳細な呼び出しです。このコードを理解していない限り、自分で呼び出すことは強くお勧めしません。

## 入力

  * `A`: 対称グラフを表すスパース行列
  * `x`: 各頂点にスコアを付けるベクトル。このベクトルはソートされ、他の入力の1つに変換されます。
  * `p`: グラフの頂点に対する部分置換ベクトル。このベクトルは、各頂点を最大1回リストする必要があります。短くても構いませんし、すべての頂点をリストする必要はありません。
  * `r`: 置換ベクトル内のアイテムのランクを提供する一般的なデータ構造。pは降順にソートされている必要があり、i < jはr[p[i]] < r[p[j]]を意味します。
  * `totalvol`: グラフの総ボリューム
  * `maxvol`: 考慮される任意のセットの最大ボリューム

## 戻り値

スイープカットで計算されたすべての情報を持つ`SweepcutProfile`型。おそらく、以下に示すように結果`bestset`を取得したいでしょう。

## 例

```
A = load_matrix_network("minnesota")
v = fiedler_vector(A)[1] # 取得する
p = sweepcut(A,v)
S = bestset(p) # プロファイルからベストセットを取得
T = spectral_cut(A).set # 同じセットを返すはずです
# using UnicodePlots; lineplot(p.conductance) # 導電率を表示
```
