```
MatsubaraFreq(n)
```

マツバラ周波数 `ω = n*π/β` の前因子 `n`

虚時間 τ における伝播関数 G(τ) のフーリエ変換からマツバラ同等物 Ĝ(iω) への構造体を表す：

```
        β
Ĝ(iω) = ∫  dτ exp(iωτ) G(τ)      ただし    ω = n π/β,
        0
```

ここで β は逆温度であり、慣例として周波数引数に虚数単位を含めます。すなわち、Ĝ(iω) です。周波数は伝播関数の統計に依存し、次のようになります：

```
G(τ - β) = ± G(τ)
```

ここで + はボソン用、- はフェルミオン用です。周波数はそれに応じて制限されます。

  * ボソニック周波数 (`S == Fermionic`): `n` 偶数（β において周期的）
  * フェルミオン周波数 (`S == Bosonic`): `n` 奇数（β において反周期的）
