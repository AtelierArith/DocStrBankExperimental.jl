`qS2R(z::Vector{Float64}, q::Number, S::Number)`

`qS2R`は、製品とフィードの組成、フィードの品質、およびカラムの底部での再フラックス比を使用して、蒸留塔の上部での再フラックス比をMcCabe-Thiele法で計算します。

フィードが飽和液体の場合、フィード品質q = 1、フィード品質はq = 1 - 1e-10にリセットされます。

`qS2R`は、Juliaの`McCabeThiele`ツールボックスの主要な関数です。

関連情報: `stages`, `refmin`, `qR2S`, `RS2q`.

# 例

蒸留液の組成が88%、フィードの組成が46%、カラムの底部製品の組成が11%、フィードが飽和液体で、カラムの底部での再フラックス比が2.5であるとき、カラムの上部での再フラックス比を計算します:

```
x=[0.88;0.46;0.11];
q=1;
S=2.5;
R=qS2R(x,q,S)
```
