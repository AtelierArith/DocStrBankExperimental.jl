`RS2q(z::Vector{Float64}, R::Number, S::Number)`

`RS2q`は、製品とフィードの組成、カラムの底部での再フラックス比、およびカラムの上部での再フラックス比を考慮して、マッケイブ・ティーリー法を使用して蒸留塔のフィード品質を計算します。

フィードが飽和液体の場合、フィード品質 q = 1 となり、フィード品質は q = 1 - 1e-10 にリセットされます。

`RS2q`は、Juliaの`McCabeThiele`ツールボックスの主要な関数です。

関連情報: `stages`, `refmin`, `qR2S`, `qS2R`。

# 例

蒸留液の組成が88%、フィードの組成が46%、カラムの底部製品の組成が11%、カラムの上部での再フラックス比が2、カラムの底部での再フラックス比が2.5の場合のフィード品質を計算します：

```
x=[0.88;0.46;0.11];
R=2;
S=2.5;
q=RS2q(x,R,S)
```
