`qR2S(z::Vector{Float64}, q::Number, R::Number)`

`qR2S`は、製品とフィードの組成、フィードの品質、および塔の上部での再フラックス比を考慮して、蒸留塔の底部での再フラックス比をMcCabe-Thiele法を使用して計算します。

フィードが飽和液体の場合、フィード品質 q = 1、フィード品質は q = 1 - 1e-10 にリセットされます。

`qR2S`は、Juliaの`McCabeThiele`ツールボックスの主要な関数です。

関連情報: `stages`, `refmin`, `qS2R`, `RS2q`.

# 例

蒸留液の組成が88%、フィードの組成が46%、塔の底部製品の組成が11%、フィードが飽和液体で、塔の上部での再フラックス比が2であるとき、塔の底部での再フラックス比を計算します:

```
x=[0.88;0.46;0.11];
q=1;
R=2;
S=qR2S(x,q,R)
```
