`refmin(data::Union{Matrix{Float64},Function}, z::Vector{Float64}; q::Number=1)`

`refmin`は、液体分率xと蒸気分率yを関連付ける関数y = y(x)または液体と蒸気の分率のx-y行列を使用して、蒸留塔の上部と下部での最小再フラックス比をMcCabe-Thiele法により計算します。さらに、分留液とフィードの分率ベクトルおよびフィードの品質も考慮されます。

デフォルトでは、フィードはフィード段階で飽和液体であり、q = 1です。

フィードがフィード段階で飽和液体である場合、q = 1のとき、フィード品質はq = 1 - 1e-10にリセットされます。

`refmin`は、Juliaの`McCabeThiele`ツールボックスの主要な関数です。

関連情報: `stages`, `qR2S`, `qS2R`, `RS2q`.

# 例

液体分率と蒸気分率を関連付ける行列が与えられたときの蒸留塔の再フラックス比の最小値を計算します。分留液の組成は88%、フィードの組成は46%、塔の底製品の組成は11%、フィードはフィード段階で44%の蒸気を含む液-蒸気平衡です：

```
data=[0.  0.;
      0.1 0.212;
      0.2 0.384;
      0.3 0.529;
      0.4 0.651;
      0.5 0.752;
      0.6 0.833;
      0.7 0.895;
      0.8 0.942;
      0.9 0.974;
      1.  1.];
x=[0.88;0.46;0.11];
r,s=refmin(data,x,q=1-0.44)
```

液体分率から蒸気分率を計算する関数が与えられたときの蒸留塔の再フラックス比の最小値を計算します。分留液の組成は88%、フィードの組成は46%、塔の底製品の組成は11%、フィードはフィード段階で飽和液体です：

```
y(x)=x.^0.9 .* (1-x).^1.2 + x;
x=[0.88;0.46;0.11];
r,s=refmin(y,x,q=1)
```
