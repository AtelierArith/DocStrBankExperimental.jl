```
A = JopSlantStack(dom[; dz=10.0, dh=10.0, h0=-1000.0, ...])
```

ここで `A` は `z-h` から `tau-p` への 2D スラントスタック演算子のマッピングです。演算子のドメインは `nz` x `nh` で、精度 T、`dz` は深さの間隔、`dh` はハーフオフセットの間隔、`h0` はハーフオフセット軸の原点です。追加の名前付きオプション引数とそのデフォルト値は次のとおりです。

  * `theta=-60:1.0:60` - 開口角の範囲。レイパラメータは: p=sin(theta/2)/c
  * `padz=0.0,padh=0.0` - フーリエ変換を適用する前に深さとオフセットに適用する分数パディング
  * `taperz=(0,0)` - `z-h` から `kz-kh` に変換する前の z 方向の始まりと終わりのテーパー
  * `taperh=(0,0)` - `z-h` から `kz-kh` に変換する前の h 方向の始まりと終わりのテーパー
  * `taperkz=(0,0)` - `kz-kh` から `z-h` に変換する前の kz 方向の始まりと終わりのテーパー
  * `taperkh=(0,0)` - `kz-kh` から `z-h` に変換する前の kh 方向の始まりと終わりのテーパー

# 注意事項

  * この演算子を 3D に拡張することが可能であるべきです
  * ドメインが z-h ではなく t-h の場合でも、この演算子を使用することができます
