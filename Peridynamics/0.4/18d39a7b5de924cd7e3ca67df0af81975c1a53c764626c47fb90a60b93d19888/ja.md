```
round_sphere(diameter, ΔX0; kwargs...)
```

特定の `diameter` と点間隔 `ΔX0` を持つ滑らかな球体の表面にエッジのない点のグリッドを作成します。内部的には、`TrixiParticles.jl` の一部がコピーされ、この関数のために適応されました。

# 引数

  * `diameter::Real`: 球の直径。
  * `ΔX0::Real`: 点の間隔。

# キーワード

  * `center`: 球の中心の座標。デフォルト: `(0, 0, 0)`

# 戻り値

  * `position::Matrix{Float64}`: 点の位置を持つ `3×n_points` 行列。
  * `volume::Vector{Float64}`: 各点の体積を持つベクトル。
