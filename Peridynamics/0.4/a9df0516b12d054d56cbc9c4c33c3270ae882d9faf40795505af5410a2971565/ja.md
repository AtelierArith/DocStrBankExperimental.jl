```
round_cylinder(diameter::Real, height::Real, ΔX0::Real; kwargs...)
```

特定の `height` を z 次元に持ち、`diameter` と点間隔 `ΔX0` を持つ円筒形に分布した点のグリッドを作成します。x-y 平面での同心点分布のため、円筒の表面に鋭いエッジは現れません。

# 引数

  * `diameter::Real`: 円筒の直径。
  * `height::Real`: 円筒の高さ。
  * `ΔX0::Real`: 点の間隔。

# キーワード

  * `center`: 円筒の中心の座標。デフォルト: `(0, 0, 0)`

# 戻り値

  * `position::Matrix{Float64}`: 点の位置を持つ `3×n_points` 行列。
  * `volume::Vector{Float64}`: 各点の体積を持つベクトル。
