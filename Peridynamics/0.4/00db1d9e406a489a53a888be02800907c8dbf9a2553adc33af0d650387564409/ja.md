```
process_each_export(f, vtk_path; kwargs...)
process_each_export(f, job; kwargs...)
```

エクスポートされたすべてのファイルを後処理するための関数。この関数はマルチスレッドおよびMPIで動作し、バックエンドを[`submit`](@ref)関数と同様に正確に決定します。

# 引数

  * `f::Function`: シグネチャ `f(r0, r, id)` を持つ処理関数。

      * `r0`: 参照結果のエクスポートファイルに対する[`read_vtk`](@ref)の結果。
      * `r`: 時間ステップに対する[`read_vtk`](@ref)の結果。
      * `id::Ind`: エクスポートファイルの番号を示すID（1からカウントし、参照ファイルから始まる）。
  * `vtk_path::AbstractString`: シミュレーションのエクスポート結果を含むべきパス。
  * `job::Job`: ジョブオブジェクト。VTKファイルのパスは、ジョブオプションから処理されます。

# キーワード

  * `serial::Bool`: `true`の場合、すべての結果は時間ステップの正しい順序で、単一スレッドで処理されます。MPIルートランクを参照してください。
