```julia
predictor(hp, ds; verbose, ampfactor)

```

この関数は、ホップ分岐点から分岐する周期軌道の予測を提供します。ホップ標準形に `a,b` の係数が含まれていない場合、固有ベクトルと `ampfactor` を用いて推測が形成されます。含まれている場合は、二次予測子が計算されます。

# 引数

  * `bp::Hopf` 分岐点
  * `ds` 分岐点に対してどの距離で予測を行いたいか。負の値も可能です。基本的に新しいパラメータは `p = bp.p + ds` です。

# オプション引数

  * `verbose` 情報を表示
  * `ampfactor = 1` 周期軌道の振幅に掛けられる係数。

# 戻り値

  * `t -> orbit(t)` 分岐した軌道のための2π周期関数の推測。
  * `amp` 分岐した周期軌道の推測の振幅。
  * `ω` 周期軌道の周波数（標準形係数で修正済み）
  * `period` 周期軌道の周期（標準形係数で修正済み）
  * `p` 新しいパラメータ値
  * `dsfactor` 分岐点の正しい側を選択するために `abs(ds)` に掛けられた係数。
