TWProblem(prob, ∂::Tuple, u₀; DAE = 0, jacobian::Symbol = :AutoDiff)

この複合型は、対称性を凍結するための関数を実装しており、例えば、伝播波（TW）を計算するために使用されます。多くの対称性を凍結することができ、単一の対称性だけでなく、多くのLie生成子を渡すことができます。`pb(x, par)`を呼び出すと、次のように計算されます：

```
                ┌                   ┐
                │ f(x, par) - s⋅∂⋅x │
                │   <x - u₀, ∂⋅u₀>  │
                └                   ┘
```

## 引数

  * `prob` 連続対称性を持つ分岐問題
  * `∂::Tuple = (T1, T2, ⋯)` Lie生成子のタプル。実際には、これらの各々は（微分）演算子であり、（スパース）行列として指定することも、`LinearAlgebra.mul!`を実装する演算子として指定することもできます。
  * `u₀` 参照解

## 追加のコンストラクタ

```
pb = TWProblem(prob, ∂, u₀; kw...)
```

この簡略化された呼び出しは、単一の対称性を凍結する必要がある場合を処理します。

## 有用な関数

  * `updatesection!(pb::TWProblem, u0)` は、`u0`を使用して問題の参照解を更新します。
  * `nb_constraints(::TWProblem)` 制約の数（またはLie生成子）
