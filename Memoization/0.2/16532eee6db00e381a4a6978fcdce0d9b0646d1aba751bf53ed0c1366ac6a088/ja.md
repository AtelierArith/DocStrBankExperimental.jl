```
@memoize f(x) = ...                # 関数定義をメモ化する
@memoize Dict f(x) = ...           # カスタムキャッシュタイプを使用
@memoize LRU(maxsize=4) f(x) = ... # 別のカスタムキャッシュタイプを使用
@memoize (x::Foo)(y) = ...         # 呼び出し可能なものをメモ化する（xとyの両方に関して）
f(x) = @memoize g(y) = ...         # クロージャをメモ化する（yに関して、使用される場合はxに関しても）
@memoize h(2)                      # 任意のJulia関数への呼び出しをメモ化する
```

関数定義に適用されると、すべての引数とキーワード引数に関して関数、クロージャ、または呼び出し可能なものをメモ化します。関数呼び出しに適用されると、その関数呼び出しだけをメモ化します（任意のJulia関数で動作します）。

メモ化されたクロージャまたは呼び出し可能なものは、インスタンスごとにメモ化されるため、クロージャは閉じられた変数を自由に使用でき、呼び出し可能なものは呼び出し可能オブジェクトのフィールドを自由に使用できます。

デフォルトでは、IdDictがキャッシュとして使用されます。任意の辞書のようなオブジェクトは、関数定義の前にマクロへの最初の引数としてオブジェクトを構築するための型または式を渡すことで使用できます。たとえば、ベクトルの内容に基づいてメモ化したい場合は、`Dict`を使用できます。
