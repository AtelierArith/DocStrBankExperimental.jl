```
HampelS(kL::Vector{Real}, kU::Vector{Real}, ϵ::Real) <: HampelSetting
```

M推定における関数選択のためのデータ型で、Hampelの関数の平滑化バージョンを使用します。`kL`と`kU`はそれぞれ下限および上限の調整定数（ベクトル）を提供します。調整定数ベクトルが1つだけ提供された場合、対称関数が仮定されます。平滑化はLog-Exp-Smoothingによって行われます。詳細は[Xia (2019)](https://doi.org/10.1007/s10825-019-01356-w)を参照してください。

# 例

```julia
HampelS([1, 2, 5], [1, 4, 7], 0.1)
Hampel([1, 2, 5], [1, 4, 7], ϵ = 0.1)
Hampel([1, 2, 5], ϵ = 0.1)
```
