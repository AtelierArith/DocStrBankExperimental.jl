```
HuberS(kL::Real, kU::Real, ϵ::Real) <: HuberSetting
```

M推定における関数選択のためのデータ型で、Huberの関数の平滑化バージョンを使用します。`kL`と`kU`はそれぞれ下限および上限の調整定数を与えます。調整定数が1つだけ提供された場合、対称関数が仮定されます。平滑化はLog-Exp-Smoothingによって行われます。詳細は[Xia (2019)](https://doi.org/10.1007/s10825-019-01356-w)を参照してください。

# 例

```julia
HuberS(1.5, 2, 0.1)
Huber(1.5, 2, ϵ = 0.1)
Huber(1.5, ϵ = 0.1)
```
