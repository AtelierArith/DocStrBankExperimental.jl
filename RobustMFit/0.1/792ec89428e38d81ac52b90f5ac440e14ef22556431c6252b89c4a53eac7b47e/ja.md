```
MTPder(μ::Vector{Real}, d::UnivariateDistribution)
```

一変量分布のモーメントからパラメータへのマッピングの導関数。`μ` は分布の最初の p 個の生モーメントのベクトルでなければならず、p は推定されるパラメータの数です。分布 `d` のパラメータは無視されます。

25 の一般的に使用される分布のためにハードコーディングされた閉じた形式。その他の分布については数値勾配に依存します。

出力は行列で、i 番目の行と j 番目の列は i 番目のモーメントに対する j 番目のパラメータの導関数を示します。

# 例

```julia
d1 = Normal()
MTP([1, 5], d1)
d2 = Poisson(10)
MTP([5], d2)
```

有効なパラメータの数については [`nParEff`](@ref nParEff) を、モーメントからパラメータへのマッピングについては [`MTP`](@ref MTP) を参照してください。
