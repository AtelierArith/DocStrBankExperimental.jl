```
pix_write(
    stream::IO,
    pix::Pix,
    format::IFF = IFF_DEFAULT
)::Bool
```

指定された形式でIOストリームに画像を書き込みます。エラーが発生した場合は`false`が返されます。

**引数:**

| T   | 名前     | デフォルト         | 説明            |
|:--- |:------ |:------------- |:------------- |
| R   | stream |               | 画像を書き込むストリーム。 |
| R   | pix    |               | ストリームに書き込む画像。 |
| O   | format | `IFF_DEFAULT` | 画像を出力する形式。    |

**詳細:**

形式が指定されていない場合、Leptonicaは入力形式に基づいて画像を書き込むための正しい形式を選択します。
