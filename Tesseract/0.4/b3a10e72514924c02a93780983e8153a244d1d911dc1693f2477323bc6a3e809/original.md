```
pix_write_bmp(
    pix::Pix
)::Bool
```

Write an image to a byte array in the BMP image format.  If there is an error `nothing` is returned.

**Arguments:**

| T   | Name | Default | Description                           |
|:--- |:---- |:------- |:------------------------------------- |
| R   | pix  |         | The image to write to the byte array. |
