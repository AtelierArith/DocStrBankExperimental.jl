```
pix_write(
    filename::String,
    pix::Pix,
    format::IFF = IFF_DEFAULT
)::Bool
```

指定された形式で画像をディスクに書き込みます。エラーが発生した場合は `false` が返されます。

**引数:**

| T   | 名前       | デフォルト         | 説明            |
|:--- |:-------- |:------------- |:------------- |
| R   | filename |               | 画像を書き込むファイル名。 |
| R   | pix      |               | ディスクに書き込む画像。  |
| O   | format   | `IFF_DEFAULT` | 画像を出力する形式。    |

**詳細:**

ファイルが存在する場合は上書きされます。

形式が指定されていない場合、Leptonica は入力形式に基づいて画像を書き込むための正しい形式を選択します。
