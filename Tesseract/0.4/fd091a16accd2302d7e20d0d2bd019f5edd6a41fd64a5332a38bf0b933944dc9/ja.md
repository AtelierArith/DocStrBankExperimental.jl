```
pix_read_tiff(
    filename::AbstractString;
    page::Integer = Int32(1)
)::Union{Pix, Nothing}
```

指定されたファイルからTIFF画像を読み込みます。 エラーの場合は`nothing`を返します。

**引数:**

| T   | 名前       | デフォルト      | 説明               |
|:--- |:-------- |:---------- |:---------------- |
| R   | filename |            | 読み込むTIFFファイルの名前。 |
| O   | page     | `Int32(1)` | ファイルから読み込む画像。    |

**制限事項:**

  * `page` - 0より大きくなければなりません。

**詳細:**

TIFFファイルには複数の画像が含まれる場合があります。 `page`パラメータを使用して、読み込みたい画像を指定できます。
