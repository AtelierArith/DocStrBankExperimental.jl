```
pix_read_tiff(
    stream::IO;
    page::Integer = Int32(1)
)::Union{Pix, Nothing}
```

指定されたストリームからTIFF画像を読み込みます。エラーの場合は`nothing`を返します。

**引数:**

| T   | 名前     | デフォルト      | 説明                       |
|:--- |:------ |:---------- |:------------------------ |
| R   | stream |            | TIFFファイルを読み込むためのIOストリーム。 |
| O   | page   | `Int32(1)` | ファイルから読み込む画像。            |

**制限事項:**

  * `page` - 0より大きくなければなりません。

**詳細:**

この実装は、FILEポインタを渡すときにLeptonicaが提供するAPIを反映しています。ストリームの残りがTIFF画像を含むことを前提としています。

TIFFファイルは複数の画像を含むことができます。`page`パラメータを使用して、読み込みたい画像を指定できます。
