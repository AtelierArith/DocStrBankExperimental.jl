```
pix_delete!(
    pix::Pix
)::Nothing
```

Pixオブジェクトを解放して、メモリを解放できるようにします。

**引数:**

|   T | 名前  | デフォルト | 説明      |
| ---:|:--- |:----- |:------- |
|   R | pix |       | 解放する画像。 |

**詳細:**

このメソッドはガーベジコレクタによって自動的に呼び出されますが、オブジェクトを早期に解放するために手動で呼び出すこともできます。このメソッドは、悪影響なしに複数回呼び出すことができます。

このメソッドを呼び出すと、外部ライブラリによって参照が保持されていない限り、オブジェクトが解放されます。そのライブラリが参照を解放すると、Pixオブジェクトは完全に解放されるはずです。ただし、`pix_delete!()`がオブジェクトに対して呼び出されると、そのオブジェクトを他のライブラリ呼び出しに渡すとエラーが発生します。

注意: このメソッドはスレッドセーフではありません。
