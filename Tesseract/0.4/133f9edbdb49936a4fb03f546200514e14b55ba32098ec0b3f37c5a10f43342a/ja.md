```
pix_write_pdf(
    stream::IO,
    pix::Pix;
    ppi::Integer = Int32(300),
    title::AbstractString = ""
)::Bool
```

IOストリームにPDFとして画像を書き込みます。エラーが発生した場合は`false`が返されます。

**引数:**

| T   | 名前     | デフォルト        | 説明                       |
|:--- |:------ |:------------ |:------------------------ |
| R   | stream |              | PDFを書き込むストリーム。           |
| R   | pix    |              | PDFに書き込む画像。              |
| O   | ppi    | `Int32(300)` | インチあたりのピクセル数で使用する画像の解像度。 |
| O   | title  |              | PDFに使用するタイトル。            |

**制限:**

  * `ppi` - 0より大きくなければなりません。
  * `title` - 空の文字列はPDFにタイトルが使用されないことを意味します。

**詳細:**

ファイルが存在する場合は上書きされます。デフォルトではタイトルは追加されません。
