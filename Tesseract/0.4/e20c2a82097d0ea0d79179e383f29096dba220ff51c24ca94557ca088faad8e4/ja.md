```
pix_write_jpeg(
    pix::Pix;
    quality::Integer = Int32(75),
    progressive::Bool = false
)::Bool
```

JPEG画像形式でバイト配列に画像を書き込みます。エラーが発生した場合は`nothing`が返されます。

**引数:**

| T   | 名前          | デフォルト       | 説明                             |
|:--- |:----------- |:----------- |:------------------------------ |
| R   | pix         |             | バイト配列に書き込む画像。                  |
| O   | quality     | `Int32(75)` | 画像をエンコードする際の品質。                |
| O   | progressive | `false`     | プログレッシブエンコーディングアルゴリズムを使用するべきか？ |

**制限:**

  * `quality` - 1から100の範囲内でなければなりません。
