```
tess_delete!(
    inst::TessInst
)::Nothing
```

テッセラクトオブジェクトを破棄し、関連するメモリを解放します。

**引数:**

|   T | 名前   | デフォルト | 説明          |
| ---:|:---- |:----- |:----------- |
|   R | inst |       | 解放するインスタンス。 |

**詳細:**

このメソッドはガーベジコレクタによって自動的に呼び出されますが、オブジェクトを早期に解放するために手動で呼び出すこともできます。このメソッドは、悪影響なしに複数回呼び出すことができます。

`tess_delete!()`がオブジェクトに対して呼び出されると、そのオブジェクトを他のライブラリ呼び出しに渡すとエラーが発生します。

注意: このメソッドはスレッドセーフではありません。
