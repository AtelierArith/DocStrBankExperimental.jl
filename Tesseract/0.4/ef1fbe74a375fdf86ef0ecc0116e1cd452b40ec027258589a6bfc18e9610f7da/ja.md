```
pix_write(
    pix::Pix,
    format::IFF = IFF_DEFAULT
):::Union{Vector{UInt8}, Nothing}
```

指定された形式でバイト配列に画像を書き込みます。エラーが発生した場合は `nothing` が返されます。

**引数:**

| T   | 名前     | デフォルト         | 説明            |
|:--- |:------ |:------------- |:------------- |
| R   | pix    |               | ストリームに書き込む画像。 |
| O   | format | `IFF_DEFAULT` | 画像を出力する形式。    |

**詳細:**

形式が指定されていない場合、Leptonica は入力形式に基づいて画像を書き込むための正しい形式を選択します。
