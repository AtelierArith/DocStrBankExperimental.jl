```
pix_read_webp(
    stream::IO
)::Union{Pix, Nothing}
```

指定されたストリームからWEBP画像を読み込みます。エラーの場合は`nothing`を返します。

**引数:**

| T   | 名前     | デフォルト | 説明                       |
|:--- |:------ |:----- |:------------------------ |
| R   | stream |       | WEBPファイルを読み込むためのIOストリーム。 |

**詳細:**

この実装は、FILEポインタを渡すときにLeptonicaが提供するAPIを反映しています。ストリームの残りがWEBP画像を含むことを前提としています。
