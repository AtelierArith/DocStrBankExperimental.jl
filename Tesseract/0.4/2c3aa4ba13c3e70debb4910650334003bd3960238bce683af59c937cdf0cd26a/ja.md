```
tess_set_input_name(
    inst::TessInst,
    input_name::AbstractString
)::Nothing
```

入力ファイルの名前を設定します。

**引数:**

| T   | 名前         | デフォルト | 説明                          |
|:--- |:---------- |:----- |:--------------------------- |
| R   | inst       |       | 入力名を設定するためのTesseractインスタンス。 |
| R   | input_name |       | 入力ファイルの名前。                  |
