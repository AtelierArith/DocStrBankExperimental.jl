```
tess_set_input_name(
    inst::TessInst,
    input_name::AbstractString
)::Nothing
```

Set the name of input file.

**Arguments:**

| T   | Name       | Default | Description                                   |
|:--- |:---------- |:------- |:--------------------------------------------- |
| R   | inst       |         | The Tesseract instance to set the input name. |
| R   | input_name |         | The name of the input file.                   |
