```
pix_read(
    stream::IO
)::Union{Pix, Nothing}
```

ストリームから画像を読み込みます。ファイルを読み込めなかった場合は `nothing` を返します。

**引数:**

| T   | 名前     | デフォルト | 説明               |
|:--- |:------ |:----- |:---------------- |
| R   | stream |       | 画像を読み込むためのストリーム。 |

**詳細:**

この実装は、FILEポインタを渡すときにLeptonicaが提供するAPIを反映しています。これは、ストリームの残りの部分に画像が含まれていると仮定しており、ストリームの残りの部分が画像として読み込まれ、デコードされます。
