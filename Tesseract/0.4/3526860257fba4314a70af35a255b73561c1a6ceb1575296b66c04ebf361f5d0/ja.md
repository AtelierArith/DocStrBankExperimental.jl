```
pix_read_png(
    stream::IO
)::Union{Pix, Nothing}
```

指定されたストリームからPNG画像を読み込みます。エラーが発生した場合は`nothing`を返します。

**引数:**

| T   | 名前     | デフォルト | 説明                      |
|:--- |:------ |:----- |:----------------------- |
| R   | stream |       | PNGファイルを読み込むためのIOストリーム。 |

**詳細:**

この実装は、FILEポインタを渡すときにLeptonicaが提供するAPIを反映しています。ストリームの残りがPNG画像であることを前提としています。
