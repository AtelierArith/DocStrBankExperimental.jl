```
pix_read_webp(
    filename::AbstractString
)::Union{Pix, Nothing}
```

Read a WEBP image from the specified file.  Returns `nothing` on error.

**Arguments:**

| T   | Name     | Default | Description                        |
|:--- |:-------- |:------- |:---------------------------------- |
| R   | filename |         | The name of the WEBP file to load. |
