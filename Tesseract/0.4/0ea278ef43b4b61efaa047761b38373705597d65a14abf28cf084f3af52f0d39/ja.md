```
pix_read_gif(
    stream::IO
)::Union{Pix, Nothing}
```

指定されたストリームからGIF画像を読み込みます。エラーが発生した場合は`nothing`を返します。

**引数:**

| T   | 名前     | デフォルト | 説明                      |
|:--- |:------ |:----- |:----------------------- |
| R   | stream |       | GIFファイルを読み込むためのIOストリーム。 |

**詳細:**

この実装は、FILEポインタを渡すときにLeptonicaが提供するAPIを反映しています。ストリームの残りがGIF画像を含むことを前提としています。
