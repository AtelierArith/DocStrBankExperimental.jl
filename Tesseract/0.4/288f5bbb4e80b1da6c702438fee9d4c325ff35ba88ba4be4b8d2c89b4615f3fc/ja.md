```
pix_write_pnm(
    stream::IO,
    pix::Pix
)::Bool
```

IOストリームにPNM画像形式で画像を書き込みます。エラーが発生した場合は`false`が返されます。

**引数:**

| T   | 名前     | デフォルト | 説明                 |
|:--- |:------ |:----- |:------------------ |
| R   | stream |       | 画像を書き込むためのIOストリーム。 |
| R   | pix    |       | IOストリームに書き込む画像。    |
