```
pix_write_gif(
    filename::AbstractString,
    pix::Pix
)::Bool
```

GIF画像形式でファイルに画像を書き込みます。エラーが発生した場合は`false`が返されます。

**引数:**

| T   | 名前       | デフォルト | 説明           |
|:--- |:-------- |:----- |:------------ |
| R   | filename |       | 書き込むファイルの名前。 |
| R   | pix      |       | ファイルに書き込む画像。 |

**詳細:**

ファイルが存在する場合は上書きされます。
