```
pix_write_ps(
    stream::IO,
    pix::Pix;
    box::Union{PixBox, Nothing} = nothing,
    ppi::Integer = Int32(300),
    scale::AbstractFloat = Float32(1.0)
)::Bool
```

PostScriptファイルに画像を書き込みます。エラーが発生した場合は'false'を返します。

**引数:**

| T   | 名前     | デフォルト          | 説明              |
|:--- |:------ |:-------------- |:--------------- |
| R   | stream |                | 画像を書き込むストリーム。   |
| R   | pix    |                | ストリームに書き込む画像。   |
| O   | box    | `nothing`      | ページ上に画像を表示する位置。 |
| O   | ppi    | `Int32(300)`   | 画像に使用する解像度。     |
| O   | scale  | `Float32(1.0)` | ページ上の画像のスケール。   |

**制限:**

  * `ppi` - 0より大きくなければなりません。
  * `scale` - `0.0`以上でなければなりません。

**詳細:**

画像をスケーリングしたくない場合は、`scale`を`1.0`に設定します。ページ上に画像を配置したい場合は`box`を使用できますが、`w`と`h`を`0`に設定すると、画像は希望の位置に配置され、スケーリングされません。

画像を特定のサイズにしたい場合は、スケールを`0.0`に設定し、`w`と`h`を希望の幅と高さに千分の1インチで設定します。したがって、幅が`5000`、高さが`2000`の場合、5インチの幅と2インチの高さの画像が得られます。

画像をスケーリングしたい場合は、`scale`を希望の係数に設定します。`0.5`の値は画像を半分にし、`2.0`の値は画像のサイズを2倍にします。ページ上に画像を配置したい場合は`box`を使用しますが、`w`と`h`の値を`0`に設定します。
