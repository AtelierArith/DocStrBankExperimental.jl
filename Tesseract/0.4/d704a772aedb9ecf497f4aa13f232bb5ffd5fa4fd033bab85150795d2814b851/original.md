```
pix_get_depth(
    pix::Pix
)::Int32
```

Retrieves the color bit depth of the image.

**Arguments:**

|   T | Name | Default | Description                          |
| ---:|:---- |:------- |:------------------------------------ |
|   R | pix  |         | The image to get the bit depth from. |
