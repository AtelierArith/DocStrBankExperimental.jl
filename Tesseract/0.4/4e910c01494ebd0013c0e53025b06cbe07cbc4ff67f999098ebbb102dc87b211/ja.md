```
pix_write_png(
    filename::AbstractString,
    pix::Pix
    gamma::AbstractFloat = Float32(0.0)
)::Bool
```

PNG画像形式でファイルに画像を書き込みます。エラーが発生した場合は`false`が返されます。

**引数:**

| T   | 名前       | デフォルト          | 説明             |
|:--- |:-------- |:-------------- |:-------------- |
| R   | filename |                | 書き込むファイルの名前。   |
| R   | pix      |                | ファイルに書き込む画像。   |
| O   | gamma    | `Float32(0.0)` | ヘッダーに書き込むガンマ値。 |

**制限:**

  * `gamma` - 0.0から1.0の範囲内でなければなりません。

**詳細:**

ファイルが存在する場合は上書きされます。ガンマが`0.0`に設定されている場合、ファイルにガンマ値は書き込まれません。
