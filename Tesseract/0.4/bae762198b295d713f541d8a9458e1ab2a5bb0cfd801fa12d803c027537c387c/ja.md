```
struct PixBox
    x::Int32
    y::Int32
    w::Int32
    h::Int32
end
```

PixBox構造体は、画像内の領域を特定するために使用されます。

**値:**

| 名前  | 説明                    |
|:--- |:--------------------- |
| x   | このボックスによって特定された領域の左側。 |
| y   | このボックスによって特定された領域の上端。 |
| w   | ボックスの幅。               |
| h   | ボックスの高さ。              |
