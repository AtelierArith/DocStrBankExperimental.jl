```
pix_write_png(
    stream::IO,
    pix::Pix
    gamma::AbstractFloat = Float32(0.0)
)::Bool
```

PNG画像形式でストリームに画像を書き込みます。エラーが発生した場合は`false`が返されます。

**引数:**

| T   | 名前     | デフォルト          | 説明             |
|:--- |:------ |:-------------- |:-------------- |
| R   | stream |                | データを書き込むストリーム。 |
| R   | pix    |                | ストリームに書き込む画像。  |
| O   | gamma  | `Float32(0.0)` | ヘッダーに書き込むガンマ値。 |

**制限:**

  * `gamma` - 0.0から1.0の範囲内でなければなりません。

**詳細:**

ガンマが`0.0`に設定されている場合、画像ヘッダーにはガンマ値は書き込まれません。
