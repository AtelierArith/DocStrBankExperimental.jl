```
struct Tsv
    level::Int
    page::Int
    block::Int
    paragraph::Int
    line::Int
    word::Int
    left::Int
    top::Int
    width::Int
    height::Int
    conf::Float32
    text::String
end
```

この構造体は、tesseractライブラリによって提供されたTSV形式のテキストの行の詳細を保持します。

**値:**

| 名前        | 説明                                                        |
|:--------- |:--------------------------------------------------------- |
| level     | 行が何を説明しているかを識別します。                                        |
| page      | これは`tess_tsv()`メソッドに渡されたページ番号です。                          |
| block     | ページ上のブロックを識別します。                                          |
| paragraph | ブロック内の段落番号です。                                             |
| line      | 段落内の行です。                                                  |
| word      | 行内の単語です。                                                  |
| left      | アイテムの左端のピクセル数です。                                          |
| top       | アイテムの上端のピクセル数です。                                          |
| width     | アイテムの幅のピクセル数です。                                           |
| height    | アイテムの高さのピクセル数です。                                          |
| conf      | OCRエンジンが単語に対してどれだけ自信を持っているか（`0` - `100`）。レベルが5でない場合は`-1`。 |
| text      | 画像からデコードされた単語です。                                          |

**詳細:**

レベルは、行が提供している情報を識別します：

  * `1` - ページ情報、ページの開始時に追加されます。
  * `2` - ブロック情報、ブロックの開始時に追加されます。
  * `3` - 段落情報、段落の開始時に追加されます。
  * `4` - 行情報、行の開始時に追加されます。
  * `5` - 単語情報、ページから読み取られた単語を識別します。

left、top、width、およびheightの値は、アイテムを囲むピクセル単位のボックスを定義します。したがって、レベルが1の場合、ボックスは全体の画像を説明します。レベルが`1`の場合、ボックスは抽出されたブロックを囲み、以下同様に抽出された単語まで続きます。
