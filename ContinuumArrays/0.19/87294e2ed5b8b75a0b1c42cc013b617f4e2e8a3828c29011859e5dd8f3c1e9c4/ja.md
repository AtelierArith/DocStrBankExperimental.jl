```
grid(P, n...)
```

ポイントのグリッドを作成します。`n`が指定されていない場合、`size(P,2)`係数を決定するのに十分な数のポイントが生成されます。`n`が整数または`Block`の場合、`n`係数を決定するのに十分なポイントが生成されます。`n`がタプルの場合、テンソル積に対応するグリッドのタプルが返されます。つまり、5⨱6の2D変換は次のようになります。

```julia
(x,y) = grid(P, (5,6))
plan_transform(P, (5,6)) * f.(x, y')
```

そして、5×6×7の3D変換は次のようになります。

```julia
(x,y,z) = grid(P, (5,6,7))
plan_transform(P, (5,6,7)) * f.(x, y', reshape(z,1,1,:))
```
