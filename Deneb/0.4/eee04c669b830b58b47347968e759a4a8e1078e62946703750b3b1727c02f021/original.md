```
Mark(type; kw...)
Mark(; spec...)
```

Creates a `MarkSpec` containing the `mark` property of a single view specification. The `type` property of the mark can be specified as a positional argument. See more in [Vega-Lite's](https://vega.github.io/vega-lite/docs/mark.html) and [Deneb's](https://brucala.github.io/Deneb.jl/dev/data_mark_encoding/#Mark) documentation.
