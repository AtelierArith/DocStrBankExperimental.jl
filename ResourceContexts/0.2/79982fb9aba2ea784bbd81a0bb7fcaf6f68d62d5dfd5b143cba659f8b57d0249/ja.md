```
@! func(args)
```

`@!` マクロは `func(ctx, args)` を呼び出します。ここで `ctx` は `@context` マクロによって作成された「現在のコンテキスト」です。`@!` が `@context` ブロックの外で使用されると、警告が発生し、グローバルコンテキストが使用されます。

---

```
@! function my_func(...)
    ...
end
```

この形式では、`my_func` の最初の引数として `context::AbstractContext` 引数が追加されます。これにより、リソースのクリーンアップの責任を、`my_func` 内で `@defer` を使用するか、リソース処理をさらに連鎖させるために `@!` を使用することで呼び出し元に戻すことができます。
