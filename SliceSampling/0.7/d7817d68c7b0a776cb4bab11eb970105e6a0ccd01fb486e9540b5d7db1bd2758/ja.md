```
GibbsPolarSlice(w; max_proposals)
```

P. Schär、M. Habeck、および D. Rudolf によるギブス極スライスサンプリングアルゴリズム [^SHR2023]。

# 引数

  * `w::Real`: 半径縮小手続きの初期ウィンドウサイズ。

# キーワード引数

  * `w::Real`: 半径縮小手続きの初期ウィンドウサイズ
  * `max_proposals::Int`: エラーを投げるまでに許可される最大提案数（デフォルト: `10000`）。

!!! info
    極座標の性質上、GPSSは次元が少なくとも $d \geq 2$ のターゲットに対してのみ信頼性を持って動作します。


!!! info
    初期ウィンドウサイズ `w` は、他のスライスサンプラーにとって妥当なサイズよりも少なくとも1桁大きく設定する必要があります。そうしないと、大量の拒否が発生する可能性があります。


!!! warning
    チェーンを初期化する際（*e.g.* `AbstractMCMC.sample` の `initial_params` キーワード引数）、妥当なノルム $\lVert x_0 \rVert > 0$ を持つ点 $x_0$ から初期化する必要があります。そうしないと、チェーンは極座標における病的な点から開始されます。これにより、サンプラーが無限ループに陥る可能性があります。（これは `max_proposals` を設定することで防ぐことができます。）もし $\lVert x_0 \rVert \leq 10^{-5}$ であれば、現在の実装は警告を表示します。


!!! info
    Turing ユーザー向け: `Turing` は `initial_params` を変更して事後のサポートに合わせる可能性があります。これにより、渡したベクトルのノルムが十分に大きいにもかかわらず、$\lVert x_0 \rVert$ が小さくなることがあります。これが疑われる場合は、単に別の初期化値を試してください。
