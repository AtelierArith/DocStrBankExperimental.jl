```
quantile_space(A, B; n = 50) → Aq, Bq, q
```

配列 `B` を `A` の分位空間に表現します。`B` と `A` は同じインデックスを持つ必要があります。

配列 `B` は `A` の要素の分位値に従ってビン分けされます。合計で `n` 個のビンが作成され、ビンの境界 `p` が返されます。`i` 番目のビンには、`A` の分位値が ∈ [`q[i]`, `q[i+1]`) のデータが含まれます。これらの `A` 値のインデックスが `B` をグループ化するために使用されます。

各ビンでは、ビン分けされた `A` と `B` の値が平均化され、`Aq` と `Bq` が得られます。

分位ごとのデータポイントの数は定義上 `length(A)/n` です。
