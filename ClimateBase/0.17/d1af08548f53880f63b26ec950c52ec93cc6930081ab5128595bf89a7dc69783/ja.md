```
seasonality(t, x; y0 = year(t[1])) → dates, vals
```

日時ベクトル `t` に関して定義されたベクトル `x` の「季節性」を計算し、`dates, vals` を返します。`dates` は `t` に存在するすべてのユニークな日付で、*年を無視して*（したがって、月と日だけが比較されます）。`dates` の年のエントリは `y0` です。`vals` はベクトルのベクトルであり、`vals[i]` は `dates[i]` と同じ日と月を持つ `x` のすべての値です。`vals` の要素は `x` で出会った順にソートされています。

通常、`mean.(vals)` に関心があり、これは実際には季節性であり、`std.(vals)` は各日付での年間変動性です。

```
seasonality(A::ClimArray) → dates, vals
```

`ClimArray` が与えられた場合、配列は1次元（時間）のみでなければなりません。
