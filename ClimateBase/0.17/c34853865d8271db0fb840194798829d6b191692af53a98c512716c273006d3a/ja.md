```
dropagg(f, A::ClimArray, d [, W])
```

統計/集約関数 `f`（例：`sum` または `mean`）を配列 `A` に対して次元 `d` に沿って適用し、結果から対応する次元を削除します（Julia は本質的にシングルトン次元を保持します）。

`A` が一次元の場合、`dropagg` は `f(A)` を適用した単一の数値を返します。

オプションで、`W::ClimArray` の形で統計的重みを提供することができます。`W` は `A` と同じサイズであるか、または一つの次元のみを持ち、`length(W) == length(dims(A, d))` を満たさなければなりません（すなわち、次元 `d` の各値に対する重み）。後者のケースは、`d` が単一の次元である場合にのみ機能します。`missing` 値を持つデータを（適切に）処理するための詳細については、[`missing_weights`](@ref) を参照してください。 ```
