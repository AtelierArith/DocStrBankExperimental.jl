```
temporal_sampling(x) → symbol
```

`x`の時間的サンプリングタイプを返します。これは、`Date`の配列または次元配列（`Time`次元を持つ）です。

考えられる返り値は次のとおりです：

  * `:hourly`、連続するエントリ間の時間的差が正確に1時間である場合。
  * `:daily`、連続するエントリ間の時間的差が正確に1日である場合。
  * `:monthly`、すべての日付が同じ日であるが、異なる月である場合。
  * `:yearly`、すべての日付が同じ月と日であるが、異なる年である場合。
  * `:other`、これは`x`が上記のいずれのカテゴリにも該当しないことを意味します。
