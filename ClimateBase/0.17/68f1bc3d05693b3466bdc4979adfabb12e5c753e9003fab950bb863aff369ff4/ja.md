```
otheridxs(A::ClimArray, Dim)
```

次のインデックスのイテレータを返します。これを使用すると、`A`のすべてのインデックスにアクセスできますが、次元`Dim`に属するインデックスは除外されます。

例えば、`A`の次元が`(Lon, Lat, Time)`の場合、`A`のすべての時系列を取得できます：

```julia
for i in otheridxs(A, Time())
    x = A[i...] # これは各lon × latの組み合わせに対する時系列（ベクトル）です
end
```

または、すべての時間+緯度スライスを取得するには、次のようにします。

```julia
for i in otheridxs(A, (Time(), Lat()))
    x = A[i...] # 各経度に対する時間 × 緯度スライスの行列です
end
```

（`i...`のスプラッティングが必要であることに注意してください。なぜなら、`otheridxs`はタプルを生成するからです）
