```
timeagg(f, A::ClimArray, W = nothing)
```

関数 `f`（例：`mean, std`）の適切な時間集約を `A` に対して実行します。ここで：

  * `A` の完全な年の範囲のみが含まれます。詳細は [`maxyearspan`](@ref) を参照してください（ほとんどのプロセスは年周期の影響を受けるため、不均一なサイクル数での平均化は通常アーティファクトを引き起こします）
  * `A` の各月は、その日数に応じて重み付けされます（毎月サンプリングされたデータの場合）

これらの機能が不要な場合は、単に [`dropagg`](@ref)`(f, A, Time, W)` を実行してください。これは、時間サンプリングが不明な場合にも行われます。

`W` は、時間的重み付けと組み合わせて使用される可能性のある統計的重みであり、各時点を異なる重みで重み付けします。ベクトルでない場合（各時点の重み）、`A` と同じ次元レイアウトの次元配列である必要があります（各データポイントの重み）。

他に [`monthlyagg`](@ref)、[`yearlyagg`](@ref)、[`seasonalyagg`](@ref) も参照してください。

```
timeagg(f, t::Vector, x::Vector, w = nothing)
```

上記と同様ですが、時間ベクトル `t` に伴う任意のベクトル `x` に対してです。
