```
@enum ResidualCategory
```

残差のタイプ（状態のタイプについては[`StateCategory`](@ref)を参照）。

| 値             | 説明                          |
|:------------- |:--------------------------- |
| `FD`          | 微分方程式（XD、XLAMBDA、XMUE変数に依存） |
| `FC_ALG`      | 純粋に代数的な制約方程式（XALG変数に依存）     |
| `FC_LOW_HIGH` | 最低および最高の導関数レベルに関する制約方程式     |
| `FC_MUE`      | スタビライザーXMUEに関連する制約方程式       |
