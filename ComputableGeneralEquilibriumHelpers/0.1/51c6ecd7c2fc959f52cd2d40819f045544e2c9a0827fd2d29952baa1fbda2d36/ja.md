```
       cde(α, β, e, u, p, c)

   CDE需要システムの需要ベクトルと消費シェアの合計を計算します。
   パラメータ`α`、`β`、`e`、効用レベル`u`、価格`p`、および所得`c`を使用します。

   # 例
   ```jldoctest
   julia> cde([0.1, 0.2, 0.3], [1, 2, 3], [1.1, 0.9, 1.0], 1, [2, 1, 1], [4])
    4-element Vector{Float64}:
    0.5341500255889858
    1.1690947909189762
    1.762605157903052
    2.3324281115373924
    ```
