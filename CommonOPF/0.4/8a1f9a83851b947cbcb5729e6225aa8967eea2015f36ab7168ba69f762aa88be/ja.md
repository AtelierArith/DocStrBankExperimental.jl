```
splitting_busses(net::Network, source::String; threshold::Int64=10)
```

木グラフで分割するバスを決定するために、最も深い葉から上向きに検索し、各サブグラフのしきい値が満たされるまで最も近いバスを集めます。

分割するバス名の `Vector{String}` と、各サブグラフ内の対応するバスの `Vector{Vector{String}}` を返します。

!!! note
    `max_busses` 制限を遵守するためには、分割バスだけでは不十分であり、各分割バスからどのサブブランチを取るかも知っておく必要があります。言い換えれば、各サブグラフ内のすべてのバスも適切に分割するために必要です。たとえば、分割バスに2つのサブブランチがある場合、`max_busses` 制限を遵守するには、分割バスのサブブランチのうち1つだけを含める必要があるかもしれません。どのブランチを取るかを知るために、サブグラフ内の他のバスを使用できます（これが、このメソッドが各サブグラフ内のバスも返す理由です）。
