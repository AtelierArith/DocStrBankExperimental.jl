```
ゲート
```

安定器回路におけるゲート。

# フィールド

  * `type::String`: ゲートタイプを説明する文字列。
  * `index::Int32`: 回路内のゲートのユニークな層の出現をラベル付けするインデックス。
  * `targets::Vector{Int16}`: ゲートの量子ビットターゲットまたはターゲット群。

# サポートされているゲート

  * `H`: ハダマードゲート。
  * `S`: フェーズゲート。
  * `S_DAG`: 共役フェーズゲート。
  * `CX` または `CNOT`: 制御-Xゲート; 最初の量子ビットが制御で、2番目の量子ビットがターゲット。
  * `CZ`: 制御-Zゲート。
  * `I`: 恒等ゲート。
  * `Z`: パウリZゲート。
  * `X`: パウリXゲート。
  * `Y`: パウリYゲート。
  * `II`: 2量子ビット恒等ゲート。
  * `AB`: 2量子ビットパウリゲート、ここで `A` と `B` はパウリ `Z`、`X`、または `Y`。
  * `SQRT_AB`: 2量子ビットパウリ回転、ここで `A` と `B` はパウリ `Z`、`X`、または `Y`。
  * `SQRT_AB_DAG` : 2量子ビットパウリ回転、ここで `A` と `B` はパウリ `Z`、`X`、または `Y`。
  * `PZ`: パウリZ固有状態を準備する。
  * `PX`: パウリX固有状態を準備する。
  * `PY`: パウリY固有状態を準備する。
  * `M` または `MZ`: 計算パウリZ基底で測定する。
  * `MX`: パウリX基底で測定する。
  * `MY`: パウリY基底で測定する。
  * `R`: 計算Z基底でリセットする。
