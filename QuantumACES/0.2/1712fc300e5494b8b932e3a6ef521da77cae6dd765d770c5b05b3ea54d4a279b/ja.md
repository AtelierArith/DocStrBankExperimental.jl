```
DepolarisingParameters
```

デポラリゼーションパウリノイズモデルをパラメータ化します。

# フィールド

  * `params::Dict{Symbol, Any}`: 以下に説明するノイズパラメータの辞書。
  * `noise_name::String`: データを保存するためのノイズパラメータ名。

# パラメータ

  * `r_1::Real`: 単一量子ビットゲートのエンタングルメント不完全性、すべての3つの非単位パウリエラー確率の合計。
  * `r_2::Real`: 二量子ビットゲートのエンタングルメント不完全性、すべての15の非単位パウリエラー確率の合計。
  * `r_m::Real`: 測定エラー確率。
  * `r_im::Real`: 回路中間測定アイドルエンタングルメント不完全性。
  * `r_r::Real`: 回路中間リセットエラー確率。
  * `combined::Bool`: パウリX、Y、Z基底のSPAMノイズを同じものとして扱うかどうか。
