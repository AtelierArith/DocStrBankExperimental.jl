```
getobs!(buffer, data, idx)
```

`getobs(data, idx)`のインプレースバージョン。このメソッドが`data`の型に対して定義されている場合、`buffer`は結果を格納するために使用され、専用のオブジェクトを割り当てることはありません。

この関数の実装はオプションです。`data`の型に対してそのようなメソッドが提供されていない場合、`buffer`は*無視され*、[`getobs`](@ref)の結果が返されます。これは、`data`の型が`copy!`の概念に適さない可能性があるためです。したがって、カスタム`getobs!`をサポートすることはオプションであり、必須ではありません。

カスタム実装の`getobs!`は、出力形式に関して[`getobs`](@ref)と一貫性があるべきです。つまり、`getobs!(buffer, data, idx) == getobs(data, idx)`です。

[`getobs`](@ref)および[`numobs`](@ref)も参照してください。
