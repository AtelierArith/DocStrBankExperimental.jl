```
BC2Copula{P}
```

フィールド:

```
- θ1::Real - パラメータ
- θ1::Real - パラメータ
```

コンストラクタ

```
BC2Copula(θ1, θ2)
```

二変量BC₂コピュラは、二つのパラメータ $\theta_{i} \in [0,1], i=1,2$ によってパラメータ化されます。これは、ピカンズ依存関数を持つ極値コピュラです:

$$
A(t) = \max\{\theta_1 t, \theta_2(1-t) \} + \max\{(1-\theta_1)t, (1-\theta_2)(1-t)\}
$$
