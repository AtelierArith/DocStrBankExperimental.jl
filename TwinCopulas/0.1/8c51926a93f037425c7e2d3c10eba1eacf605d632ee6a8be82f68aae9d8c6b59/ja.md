```
IndependentCopula
```

コンストラクタ

```
IndependentCopula()
```

二変量 [独立コピュラ](https://en.wikipedia.org/wiki/Copula_(probability_theory)#Most_important_Archimedean_copulas) は、次の形を持つ最も単純なコピュラです：

$$
C(u_1,u_2) = u_1u_2
$$

  * [nelsen2006](@cite) Nelsen, Roger B. An introduction to copulas. Springer, 2006.
