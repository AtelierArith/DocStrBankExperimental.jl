```
LogCopula{P}
```

フィールド:

```
- θ::Real - パラメータ
```

コンストラクタ

```
LogCopula(θ)
```

二変量ロジスティックコピュラ（またはガンベルコピュラ）は、$\theta \in [1,\infty)$ でパラメータ化されています。これは、ピカンズ依存関数を持つ極値コピュラです：

$$
A(t) = (t^{\theta}+(1-t)^{\theta})^{\frac{1}{\theta}}
$$

いくつかの特別なケースがあります：

  * θ = 1 のとき、これは独立コピュラです
  * θ = ∞ のとき、これはMCopula（上フレシェ-ホエフディング境界）です
