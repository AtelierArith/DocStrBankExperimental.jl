```
GalambosCopula{P}
```

フィールド:

```
- θ::Real - パラメータ
```

コンストラクタ

```
GalambosCopula(θ)
```

二変量ガランボスコピュラは、$\alpha \in [0,\infty)$ でパラメータ化されています。これは極値コピュラで、ピカンズ依存関数を持ちます:

$$
A(t) = 1 - (t^{-\theta}+(1-t)^{-\theta})^{-\frac{1}{\theta}}
$$

いくつかの特別なケースがあります:

  * θ = 0 のとき、これは独立コピュラです
  * θ = ∞ のとき、これは M コピュラ (上フレシェ-ホエフディング境界) です
