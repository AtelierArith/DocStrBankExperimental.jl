```
ExpandingHist(nbins)
```

すべての観測値を含むために、ビンのエッジがサイズを倍増し続ける適応ヒストグラムです。`nbins`は偶数でなければなりません。ビンは左閉で、最も右のビンは閉じています。例えば、

  * `[a, b)`, `[b, c)`, `[c, d]`

# 例

```
o = fit!(ExpandingHist(200), randn(10^6))

using Plots
plot(o)
```

# 詳細

`ExpandingHist`の動作は、例を通じて最もよく理解できます。次のようにエッジ/カウントのヒストグラムから始めるとしましょう：

```
|1|2|5|3|2|
```

  * ここで、ビンのエッジに含まれていないデータポイントを観測します：

```
|1|2|5|3|2|       *
```

  * このポイントを含むために、エッジの範囲が新しいデータポイントの方向に倍増し、隣接するビンがカウントをマージします：

```
|1|2|5|3|2|       *
 \ / \ / \ /      ↓
  ↓   ↓   ↓       ↓
| 3 | 8 | 2 | 0 | 1 |
```

  * 新しいポイントがビンのエッジに含まれるまで、ビンの倍増が複数回発生する可能性があることに注意してください。
