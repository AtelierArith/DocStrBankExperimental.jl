```
Quantile(q::Vector{Float64} = [0, .25, .5, .75, 1]; b=500)
```

データストリームから（近似的な）分位数を計算します。内部的には、分布を推定するために[`ExpandingHist`](@ref)を使用し、`b`はヒストグラムビンの数です。`b`を大きな数に設定すると、速度の代わりに精度が向上します。

# 例

```
q = [.25, .5, .75]
x = randn(10^6)

o = fit!(Quantile(q, b=1000), randn(10^6))
value(o)

quantile(x, q)
```
