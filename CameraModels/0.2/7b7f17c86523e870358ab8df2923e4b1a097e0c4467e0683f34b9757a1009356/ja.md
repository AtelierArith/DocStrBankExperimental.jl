```julia
    c_H_a,

```

ピクセル座標から重力に沿ったローカルレベル参照の床へのレイトレース。ローカルレベルフレーム（座標）での交差点を返します。

ノート

  * ローカルレベルに対して実装されており、ローカルまたはワールド参照の使用を容易にします。

      * ワールドがローカルレベルであると仮定します。すなわち、`l_nFL = w_nFL` および `l_FL = w_FL`。
      * ユーザーは（仮定された動的な）ローカルレベル変換を `l_T_ex` を介して提供する必要があります – 下記の例を参照してください！
  * 使用される座標チェーンは、（ピクセル配列（a） –> カメラ（c） –> 外部（ex） –> レベル（l））です。

      * ピクセル配列からカメラへの `c_H_a`（ホモグラフィ行列として）
      * 配列ピクセルフレーム内の特徴 `a_F`
      * ボディ内の外部（またはローカルレベルへの外部）での `l_T_ex`、ArrayPartitionを使用したSE3多様体要素
      * 外部でのカメラ（またはカメラから外部への） `ex_T_c`
  * ローカルレベルフレームでの線と平面の交差計算が行われます。

例

```
# ボディ座標 x,y,z == fwd,prt,upw

# カメラ設定
f = 800.0        # ピクセル
ci,cj = 360,640  # 720x1280画像を仮定
# イメージング配列からカメラフレームへの変換
c_H_a = [0 1 -cj; 1 0 -ci; 0 0 f] # カメラ行列

# カメラの外部へのボディからの変換 -- 例えば、カメラが0.2下向き、0.2左向き
# ローカルレベルからボディへの外部変換 
l_T_b = ArrayPartition([0;0;0.], R0)
b_T_ex = ArrayPartition([0;0;0.], exp_lie(Mr, hat(Mr, R0, [0;0.2;0.2])))
l_T_ex = compose(M, l_T_b, b_T_ex) # ここでボディ参照が折りたたまれます。

# ローカルレベル座標での床、法線と点
l_nFL = [0; -0.05; 1.]
l_FL = [0; 0; -2.]

# 平面へのレイトレース
l_Forb = intersectRayToPlane(
  c_H_a,
  a_Forb,
  l_nFL,
  l_FL;
  l_T_ex
)
```
