# フィールド

  * data: 分布を比較するために使用される配列
  * log*likelihood*ratio: データの対数尤度比
  * sig_level: シグマレベル
  * xmin: 分布を比較するために使用された最小要素
  * V*test*stat: ヴォンテスト統計量
  * V*p*val: ヴォンテストからのp値
  * V*preff*distr: ヴォンテストに基づく好ましい分布
  * C_b: クラークテストb値 - 対数尤度比の正の値の合計
  * C*p*val: クラークテストからのp値
  * C*preff*distr: クラークテストに基づく好ましい分布
