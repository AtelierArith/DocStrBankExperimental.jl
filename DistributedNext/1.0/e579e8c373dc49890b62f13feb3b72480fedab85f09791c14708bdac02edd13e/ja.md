```
start_worker([out::IO=stdout], cookie::AbstractString=readline(stdin); close_stdin::Bool=true, stderr_to_stdout::Bool=true)
```

`start_worker` は、TCP/IP 経由で接続するワーカープロセスのデフォルトエントリポイントである内部関数です。プロセスを Julia クラスターのワーカーとして設定します。

ホスト:ポート情報はストリーム `out` に書き込まれます（デフォルトは stdout です）。

この関数は、必要に応じて stdin からクッキーを読み取り、空いているポート（または指定された場合は `--bind-to` コマンドラインオプションのポート）でリッスンし、受信した TCP 接続とリクエストを処理するタスクをスケジュールします。また、（オプションで）stdin を閉じ、stderr を stdout にリダイレクトします。

`--bind-to` を通じて特定のインターフェースが指定されていない場合、リッスンするために最も速い利用可能なネットワークインターフェースを選択するために最善を尽くします。これに使用されるヒューリスティックはシステム構成に依存し、常に最も速いインターフェースを選択することを保証するものではありません。

戻り値はありません。
