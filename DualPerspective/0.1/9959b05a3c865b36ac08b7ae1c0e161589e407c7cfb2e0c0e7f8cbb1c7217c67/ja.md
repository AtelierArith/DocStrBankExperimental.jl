```
solve!(kl::DPModel, ::SequentialSolve; kwargs...) -> ExecutionStats
```

KL正則化最小二乗問題を解くために、双対目的を最大化する最適なスケーリングパラメータ`t`を見つけます。最適な`t`は、`t`に関する双対目的の導関数のルートファインディングを適用することによって見つけられます。

# 引数

  * `kl`: 解くべきKL正則化最小二乗モデル
  * `::SequentialSolve`: シーケンシャルソルブアルゴリズムのタイプ

# キーワード引数

  * `t::Real=1.0`: スケーリングパラメータの初期推定値
  * `rtol::Real=1e-6`: ルートファインディング最適化の相対許容誤差
  * `atol::Real=1e-6`: ルートファインディング最適化の絶対許容誤差
  * `xatol::Real=1e-6`: `t`の収束に対する絶対許容誤差
  * `xrtol::Real=1e-6`: `t`の収束に対する相対許容誤差
  * `δ::Real=1e-2`: 内部最適化のために`atol`および`rtol`に適用される許容誤差ファクター
  * `zverbose::Bool=false`: ルートファインディングからの詳細な出力を印刷するかどうか
  * `logging::Int=0`: ロギングレベル (0=なし, 1=基本, 2=詳細)

# 戻り値

`ExecutionStats`構造体を返します:

  * 解のステータス
  * 実行統計
  * 最適なプライマルおよび双対解
  * 残差および最適性測定
