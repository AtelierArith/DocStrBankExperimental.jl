```julia
struct Gas
```

気体状態の化学元素。流体計算に使用される元素の特性のコンテナ。

# フィールド

  * `name::String`: ガスのフルネーム（例：キセノン）
  * `short_name::String`: 短縮名/記号（例：キセノンのXe）
  * `γ::Float64`: 比熱比 / 断熱指数
  * `M::Float64`: モル質量（グラム/モル）または原子質量単位
  * `m::Float64`: 原子の質量（kg）
  * `cp::Float64`: 定圧での比熱
  * `cv::Float64`: 定積での比熱
  * `R::Float64`: ガス定数
