```
mosaic(A1, A2...; center=true, kwargs...)
mosaic([A1, A2, ...]; center=true, kwargs...)
```

入力配列 `A1`, `A2`, ... からモザイクを作成します。`mosaic` は本質的に `cat` または `hvcat` のより柔軟なバージョンであり、それらと同様に入力のコピーを作成し、「ビュー」を返すことはありません。

`center` が `true` に設定されている場合、パディングされた配列は中央にシフトされます。`false` に設定されている場合、左上隅にシフトされます。このパラメータは、配列のサイズが異なる場合にのみ有用です。

[`mosaicview`](@ref) のすべてのキーワード引数もサポートされています。
