```
bng2lonlat(e::T1, n::T2) where {T1<:Real, T2<:Real} -> 経度, 緯度
```

イギリス国立グリッドのメートル単位の `easting` と `northing` から WGS84 の `経度` と `緯度`（度）に変換します。

!!! 注     この関数は、`e` と `n` が国立グリッドの範囲外であってもエラーをスローしません。したがって、この関数は注意して使用する必要があります。

# 例

```jldoctest
julia> bng2lonlat(175154, 225430)
(-5.268407238735794, 51.88199105278528)

```
