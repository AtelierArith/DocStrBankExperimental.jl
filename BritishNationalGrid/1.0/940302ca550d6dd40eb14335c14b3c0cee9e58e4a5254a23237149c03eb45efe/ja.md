#### タイプ

```
BNGPoint{T<:Real}
```

イギリス国立グリッド内の点の東西座標と北南座標を保持する構造体です。

#### コンストラクタ

```
BNGPoint(e, n)
```

グリッドの原点からのメートル単位での東西座標 `e` と北南座標 `n` を提供します。

```
BNGPoint(e, n, square)
```

100 kmの正方形内の点とその名前を2文字の文字列として提供し、完全な参照を持つ点を作成します。

```jldoctest
julia> BNGPoint(101, 12345, "OV")
BritishNationalGrid.BNGPoint{Int64}(500101, 512345)

```

!!! 注     提供された座標が国立グリッドの有効範囲内にない場合、エラーが発生します。

---

```
BNGPoint(; lon=0, lat=0)
```

度単位のWGS84経度 `lon` と緯度 `lat` をグリッドポイントに変換します。

```jldoctest
julia> BNGPoint(lon=-1.54, lat=55.5)
BNGPoint{Float64}(429158.6966862687, 623009.0927592682)

```

#### フィールドへのアクセス

`BNGPoint` には以下のフィールドが含まれています：

  * `e`: メートル単位の東西座標
  * `n`: メートル単位の北南座標

これらのフィールドはAPIの一部であり、したがってユーザーが `BNGPoint` から東西座標と北南座標を次のように抽出することは安全で期待されています：

```jldoctest
julia> p = BNGPoint(lon=-1.54, lat=55.5)
BNGPoint{Float64}(429158.6966862687, 623009.0927592682)

julia> p.e, p.n
(429158.6966862687, 623009.0927592682)
```
