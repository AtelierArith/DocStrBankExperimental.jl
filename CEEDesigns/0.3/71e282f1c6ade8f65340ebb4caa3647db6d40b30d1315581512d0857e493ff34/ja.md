```
front(v)
front(f, v; atol1=0, atol2=0)
```

`v`のパレートフロントを構築します。`f`によって`v`の要素が計算でマスクされます。

最初と2番目の（目的）座標は、それぞれフロント上の最新の点に対して少なくとも`atol1`、`atol2`異なる必要があります。

# 例

```jldocstest
v = [(1,2), (2,3), (2,1)]
front(v)

# 出力
[(1, 2), (2, 1)]
```

```jldoctest
v = [(1, (1, 2)), (2, (2, 3)), (3, (2, 1))]
front(x -> x[2], v)

# 出力

[(1, (1, 2)), (3, (2, 1))]
```

```jldoctest
v = [(1, 2), (2, 1.99), (3, 1)]
front(v; atol2 = 0.2)

# 出力

[(1, 2), (3, 1)]
```
