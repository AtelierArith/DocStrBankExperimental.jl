```
ftest!(data::DataMatrix, h1; h0, kwargs...)
```

与えられた `h1`（対立仮説）と `h0`（帰無仮説）を用いてF検定を実行します。F検定の例にはANOVAや二次回帰が含まれますが、任意の線形モデルを使用できます。

`ftest!` は `data.var` にF統計量とp値の列を追加します。

使用例や計算およびパラメータの詳細については、[`ftest_table`](@ref)を参照してください。

さらに、`ftest!` は次の `kwarg` をサポートしています：

  * `prefix` - F統計量とp値の出力列名はこの文字列で接頭辞が付けられます。指定がない場合は、`h1` と `h0` から構築されます。

他にも: [`ftest_table`](@ref), [`ftest`](@ref), [`ttest!`](@ref)
