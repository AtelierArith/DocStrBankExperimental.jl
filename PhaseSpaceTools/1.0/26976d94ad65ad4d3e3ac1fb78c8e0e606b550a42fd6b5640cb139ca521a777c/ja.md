```
a = randnc(N...)
```

次のような次元 `length(N)` の配列を返し、平均がゼロで分散が1の複素ランダム変数のサンプルを含みます：

$$
⟨|a|^2⟩ = 1; ⟨a^2⟩ = ⟨(a^*) ^2⟩ = 0,
$$

位相空間シミュレーションに現れるさまざまなノイズを生成するのに便利です。

使用法：

`a = randnc(10)` は `10-element Array{Complex{Float64},1}` を返します。

`a = randnc(50,100)` は `50x100-element Array{Complex{Float64},2}` を返します。
