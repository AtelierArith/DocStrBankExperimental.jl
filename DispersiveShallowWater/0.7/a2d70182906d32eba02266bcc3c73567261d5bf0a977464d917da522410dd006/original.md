```
RelaxationCallback(invariant)
```

Use a relaxation method in time in order to exactly preserve the (nonlinear) `invariant` for a conservative semidiscretization. A possible choice for `invariant` is `invariant = entropy`.

Reference

  * Hendrik Ranocha, Mohammed Sayyari, Lisandro Dalcin, Matteo Parsani, David I. Ketcheson (2020) Relaxation Runge–Kutta Methods: Fully-Discrete Explicit Entropy-Stable Schemes for the Compressible Euler and Navier–Stokes Equations [DOI: 10.1137/19M1263480](https://doi.org/10.1137/19M1263480)
