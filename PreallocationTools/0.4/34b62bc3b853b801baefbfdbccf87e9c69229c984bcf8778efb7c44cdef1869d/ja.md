```
b = LazyBufferCache(f = identity; initializer! = identity)
```

遅延割り当てされたバッファオブジェクトです。配列 `u` が与えられると、`b[u]` は同じ型とサイズ `f(size(u))`（デフォルトでは同じサイズ）を持つ配列を返し、必要に応じて割り当てられ、その後 `b` 内にキャッシュされて再利用されます。

デフォルトでは、作成されたバッファは初期化されていませんが、`initializer!` という関数を提供することができ、これはバッファが作成されるときに適用されます。例えば `buf -> fill!(buf, 0.0)` のように使用します。

オプションとして、呼び出し時にサイズを明示的に指定することができ、`b[u,s]` を使用するとサイズ `s` のキャッシュが返されます。
