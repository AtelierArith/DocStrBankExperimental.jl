```
simplexgrid(X; bregions=[1,2],cellregion=1)
```

1Dグリッドのコンストラクタ。

ノード座標の配列から1Dグリッドを構築します。デフォルトでは、左端にインデックス1、右端にインデックス2の2つの境界領域を作成します。

キーワード引数を使用すると、デフォルトの領域番号を上書きできます。

未知数を保持する原始グリッド：`o`でマーク、制御体積をマークする双対グリッド：`|`でマーク。

```@raw html
 o-----o-----o-----o-----o-----o-----o-----o-----o
 |--|-----|-----|-----|-----|-----|-----|-----|--|
```
