```
N,D,E,X = ratfn_minimax(f, interval, n, d, w)
```

最上位関数は、ミニマックス有理関数近似を見つけるためのものです。

引数:

  * f         近似される関数。BigFloat -> BigFloat をマップします。
  * interval  f を近似する区間の端点を与えるタプル           （どちらの順序でも）です。
  * n, d      希望する近似の分子と分母の次数です。
  * w         誤差重み付け関数。2つの BigFloat 引数 x,y を取り、その位置での誤差のスケーリング係数を返します。           返される近似 R は、abs((f(x)-R(x)) * w(x,f(x))) の最大値が可能な限り最小になるべきです。オプションのパラメータで、常に1を返す関数がデフォルトです。

返り値: タプル (N,D,E,X) で、ここで

  * N,D       返された有理関数の係数を与える BigFloat の配列のペアです。N のサイズは n+1; D のサイズは d+1 です。両方とも定数項から始まり、すなわち N[i] は x^(i-1) の係数です（Julia の配列は 1 ベースです）。D[1] は 1 になります。
  * E         最大重み付き誤差 (BigFloat) です。
  * X         n+2 点の位置とそれぞれの点での重み付き誤差を与える BigFloat のペアの配列です。重み付き誤差の値は交互の符号を持ち、これはチェビシェフの交代定理が、同じ次数の他の関数がこれらの点の少なくとも1つでこの関数の誤差を超えなければならないことを保証します。
