```
GadgetPhysical(l_unit::T=3.085678e21, m_unit::T=1.989e43, v_unit::T=1.e5;
               a_scale::T=1.0, hpar::T=1.0,
               γ_th::T=5.0/3.0, xH::T=0.752) where T
```

構造体 `GadgetPhysical` を作成し、コモービングコード単位と物理単位の間の変換係数を保持しますが、単位情報は含まれません。

キーワード引数は以下を指定します：

# 引数

  * `a_scale::T = 1.0`:  シミュレーションの宇宙論的スケールファクター。ヘッダー `h` として `h.time` で渡すことができます。
  * `hpar::T = 1.0`:     '小さな h' としてのハッブル定数。ヘッダー `h` として `h.h0` で渡すことができます。
  * `γ_th::T = 5.0/3.0`: ガスの断熱指数。
  * `xH::T = 0.752`:      化学モデルなしで実行された場合のシミュレーションの水素比。

# フィールド

| 名前                | 意味                            |
|:----------------- |:----------------------------- |
| `x_cgs::T`        | cm 単位での位置                     |
| `x_physical::T`   | kpc 単位での位置                    |
| `v_cgs::T`        | cm/s 単位での速度                   |
| `v_physical::T`   | km/s 単位での速度                   |
| `m_cgs::T`        | g 単位での質量                      |
| `m_msun::T`       | Msun 単位での質量                   |
| `m_physical::T`   | 10^10 Msun 単位での質量             |
| `t_s::T`          | 秒単位での時間                       |
| `t_Myr::T`        | Myr 単位での時間                    |
| `E_cgs::T`        | erg 単位でのエネルギー                 |
| `E_eV::T`         | eV 単位でのエネルギー                  |
| `B_cgs::T`        | ガウス単位での磁場                     |
| `rho_physical::T` | 10^10 Msun/kpc^3 単位での密度       |
| `rho_cgs::T`      | $g/cm^3$ 単位での密度               |
| `rho_ncm3::T`     | $n_p/cm^3$ 単位での密度             |
| `T_K::T`          | K 単位での温度                      |
| `T_eV::T`         | eV 単位での温度                     |
| `P_th_cgs::T`     | Ba 単位での熱圧力                    |
| `P_CR_cgs::T`     | Ba 単位での宇宙線圧力                  |
| `CR_norm::T`      | cgs 単位での LMB*SPECTRAL*CRs ノルム |
