`DataSet`型はデータを格納するための多目的コンテナです。通常、`x`、`y`、`sigma`の3つのベクトルを渡すことで構築され、`sigma`の各成分は各y値に関連する標準偏差を定量化します。あるいは、標準偏差のベクトルの代わりに`ydata`に完全な共分散行列を供給することもできます。`DataSet` `DS`の内容には、後で`xdata(DS)`、`ydata(DS)`、`ysigma(DS)`を介してアクセスできます。

例:

すべてのデータポイントが相互に独立で、各々が単一の$x$成分と単一の$y$成分を持つ最も単純なケースでは、4つのポイントからなる`DataSet`を次のように構築できます。

```julia
DataSet([1,2,3,4], [4,5,6.5,7.8], [0.5,0.45,0.6,0.8])
```

または、次のようにしても構いません。

```julia
using LinearAlgebra
DataSet([1,2,3,4], [4,5,6.5,7.8], Diagonal([0.5,0.45,0.6,0.8].^2))
```

ここで、2行目の対角共分散行列は、1行目で供給された標準偏差のベクトルと同等です。

複数の成分を持つ測定の場合、成分に対応する列を持つ`Matrix`として入力することも可能です。

```julia
DataSet([0, 0.5, 1], [1 100; 2 103; 3 108], [0.5 8; 0.4 5; 0.6 10])
```

不確実性行列が正方形の場合、標準偏差の列指定としてではなく、共分散行列として誤って解釈される可能性があることに注意してください。

より一般的には、データセットが$N$ポイントから構成され、各$x$値が$n$個の成分を持ち、各$y$値が$m$個の成分を持つ場合、これはベクトル`x`、`y`および共分散行列に加えて、タプル$(N,n,m)$を介して`DataSet`コンストラクタに指定できます。例えば：

```julia
X = [0.9, 1.0, 1.1, 1.9, 2.0, 2.1, 2.9, 3.0, 3.1, 3.9, 4.0, 4.1]
Y = [1.0, 5.0, 4.0, 8.0, 9.0, 13.0, 16.0, 20.0]
Cov = Diagonal([2.0, 4.0, 2.0, 4.0, 2.0, 4.0, 2.0, 4.0])
dims = (4, 3, 2)
DS = DataSet(X, Y, Cov, dims)
```

この場合、`X`は4つの異なるデータポイントのための3成分を持つ連結されたx値からなるベクトルです。`Y`の値は、これら4つのデータポイントの対応する連結されたy値（それぞれ2成分）です。したがって、共分散行列は正定値の$(m \cdot N) \times (m \cdot N)$行列でなければなりません。
