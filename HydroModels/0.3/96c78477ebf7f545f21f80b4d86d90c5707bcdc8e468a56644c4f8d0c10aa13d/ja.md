```
ManualSolver{mutable} <: AbstractHydroSolver
```

水文学的モデリングに最適化された軽量で型安定なODEソルバーです。

# 型パラメータ

  * `mutable::Bool`: 配列の可変性とパフォーマンス特性を制御します

      * `true`: インプレース更新のために可変配列を使用します（30%速い）
      * `false`: 関数型プログラミングスタイルのために不変配列を使用します

# 説明

ManualSolverは、常微分方程式（ODE）を解くための固定ステップ明示オイラー法を実装しています。これは、安定性と計算効率が高次の精度よりも優先される水文学モデルのために特別に設計されています。

## パフォーマンス特性

  * 予測可能なパフォーマンスのための型安定な実装
  * `mutable`パラメータを介したオプションのインプレース操作
  * シミュレーションの長さに対して線形の時間計算量
  * `mutable=true`の場合の定数の空間計算量

## メモリ管理

  * `mutable=true`: 

      * 配列をインプレースで変更します
  * `mutable=false`:

      * 各ステップで新しい配列を作成します
