```
LWR(weightfun=h -> exp(-3 * h^2), distance=Euclidean())
```

クリーブランドによって1979年に導入された局所加重回帰（LOESSとも呼ばれる）モデルです。これは、距離に基づく重みの代わりにカスタム重み関数を使用することが許可される[`IDW`](@ref)の最も自然な一般化です。
