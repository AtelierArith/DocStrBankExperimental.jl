```
TiltRegion <: GeoRegion
```

**傾斜した**矩形領域を直交グリッド上に定義します。定義は以下の通りです：

  * 中心の (lon,lat) 座標。
  * 経度および緯度方向の幅（回転前）。
  * 傾斜角度（時計回り）を度で。

`GeoRegion` 抽象型に共通するすべてのフィールドに加えて、`TiltRegion` には以下のフィールドも含まれます：

  * `tilt` : `Float` 型のベクトルで、[X,Y,ΔX,ΔY,θ] を含みます。ここで：

      * `X`  : `Float` 型、領域中心の経度座標。
      * `Y`  : `Float` 型、領域中心の緯度座標。
      * `θ`  : `Float` 型、矩形領域の傾斜角度（度）で、時計回りの方向。
      * `ΔX` : `Float` 型、経度座標における半幅（傾斜前）。
      * `ΔY` : `Float` 型、緯度座標における半幅（傾斜前）。
