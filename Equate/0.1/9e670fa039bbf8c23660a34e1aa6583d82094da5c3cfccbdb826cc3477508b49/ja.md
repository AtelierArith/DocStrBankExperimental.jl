```
Equipercentile(X::EG, Y::EG; case = :middle)
```

Equipercentile 同等の（単一）グループデザインにおける等価。平滑化された頻度を使用できます。関数によって返されるテーブルには2つの列が含まれています。最初の列はテスト X の基準スケールスコアを表します。2番目の列はテスト Y からの等価スコアで、基準スケール上の等百分位スコアです。

`case` はどの等価ケースを使用するかを表します。以下の記号を渡してください。

  * `:upper` 任意のパーセンタイル P に対応するスコアを計算し、P より大きい累積パーセントを持つ最小の整数スコアを使用します。
  * `:lower` （デフォルト）上記のケースとは逆に、P より小さい累積パーセントを持つ最大の整数スコアを使用します。
  * `:both` （実践用ではない）上記の上と下の両方のケースを表示します。
  * `:middle` 下の間の上の中点ケースを使用します。

私の例の場合、R の equate パッケージは等価を求めるために `lower` ケースを使用しているようです。
