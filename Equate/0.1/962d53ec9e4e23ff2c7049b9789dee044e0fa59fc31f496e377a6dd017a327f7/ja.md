```
BraunHolland(X::NEAT, Y::NEAT; w₁ = length(X.rawX) / (length(X.rawX) + length(Y.rawX)), w₂ = 1.0 - w₁)
```

ブラウン-ホランド線形等化を実施します。これは、合成母集団分布によって推定された平均と標準偏差を使用します。

# 引数

  * `X` は `freqtab(X, V)` によって生成された頻度表で、`V` は共通項目部分のテストスコアです。
  * `Y` は上記と同様です。
  * `w₁`, `w₂` は合成母集団の重みで、w₁ + w₂ = 1 に制約されています。
