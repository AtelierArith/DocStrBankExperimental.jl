```
asset_return(price::TSFrame, period::Int=1; method::String="simple")
```

資産価格から`returns`を計算します。

# 引数:

  * `price::TSFrame`: 資産価格のTSFrameオブジェクトの列
  * `period::Int=1`: リターン期間
  * `method::String="simple"`: リターン方法、利用可能な方法; `"simple"` と `"log"`
