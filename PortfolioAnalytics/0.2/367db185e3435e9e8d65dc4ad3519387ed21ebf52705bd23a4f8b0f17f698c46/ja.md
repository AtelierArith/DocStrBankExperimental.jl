```
drawdowns(R::TSFrame; geometric::Bool=true, max_drawdown::Bool=false)
```

資産またはポートフォリオのリターンのドローダウンを計算します。出力はドローダウンのTSFrameオブジェクトです。

# 引数:

  * `R::TSFrame`: 資産リターンのTSFrameオブジェクトの列
  * `geometric::Bool=true`: trueの場合、幾何学的リターンからドローダウンを計算します
  * `max_drawdown::Bool=false`: trueの場合、資産またはポートフォリオの最大ドローダウンを返します
