```
graphsummary([io], graph, extracolumns...; [inputhl], [outputhl], kwargs...)
```

`graph`の要約テーブルを`io`に`PrettyTables.pretty_table`を使用して印刷します。

テーブルには、次のいずれかの`extracolumns`を提供することで追加の列を追加できます。

  * 頂点を入力として受け取り、列の内容を返す関数（または呼び出し可能なオブジェクト）
  * 最初の要素が列名で、他の要素が前の箇条書きで説明されている内容である`Pair`

キーワード引数`inputhl`（デフォルトは`crayon"fg:black bg:249"`）と`outputhl`（デフォルトは`inputhl`）を使用して、`graph`への入力と出力のハイライトを設定できます。`nothing`に設定すると、特別なハイライトは使用されません。

他のすべてのキーワード引数は`PrettyTables.pretty_table`に転送されます。これにより、デフォルトのフォーマット、整列、ハイライトを上書きすることができます。

!!! warning "APIの安定性"
    この関数は自然な理由から公開APIの一部ですが、その出力の正確な形状は安定しているとは見なされません。

    `CompGraph`の`Base.show`はこの関数を呼び出すだけです。将来的に変更される可能性があります。


### 例

```jldoctest
julia> using NaiveNASlib

julia> g = let 
            v1 = "v1" >> inputvertex("in1", 1) + inputvertex("in2", 1)
            v2 = invariantvertex("v2", sin, v1)
            v3 = conc("v3", v1, v2; dims=1) 
            CompGraph(inputs(v1), v3)
        end;

julia> graphsummary(g)
┌────────────────┬───────────┬────────────────┬───────────────────┐
│ グラフの位置   │ 頂点番号   │ 入力頂点       │ 演算              │
├────────────────┼───────────┼────────────────┼───────────────────┤
│ 入力          │ 1         │                │                   │
│ 入力          │ 2         │                │                   │
│ 隠れ          │ 3         │ 1,2            │ + (要素ごと)      │
│ 隠れ          │ 4         │ 3              │ sin               │
│ 出力          │ 5         │ 3,4            │ cat(x..., dims=1) │
└────────────────┴───────────┴────────────────┴───────────────────┘

julia> graphsummary(g, name, "input sizes" => nin, "output sizes" => nout)
┌────────────────┬───────────┬────────────────┬───────────────────┬──────┬─────────────┬──────────────┐
│ グラフの位置   │ 頂点番号   │ 入力頂点       │ 演算              │ 名前  │ 入力サイズ   │ 出力サイズ   │
├────────────────┼───────────┼────────────────┼───────────────────┼──────┼─────────────┼──────────────┤
│ 入力          │ 1         │                │                   │ in1  │             │ 1            │
│ 入力          │ 2         │                │                   │ in2  │             │ 1            │
│ 隠れ          │ 3         │ 1,2            │ + (要素ごと)      │ v1   │ 1,1         │ 1            │
│ 隠れ          │ 4         │ 3              │ sin               │ v2   │ 1           │ 1            │
│ 出力          │ 5         │ 3,4            │ cat(x..., dims=1) │ v3   │ 1,1         │ 2            │
└────────────────┴───────────┴────────────────┴───────────────────┴──────┴─────────────┴──────────────┘
```
