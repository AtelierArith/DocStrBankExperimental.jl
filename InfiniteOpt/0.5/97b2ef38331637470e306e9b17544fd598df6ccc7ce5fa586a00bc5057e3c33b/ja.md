```
expand(mref::MeasureRef)::JuMP.AbstractJuMPScalar
```

測定 `mref` の明示的な展開を含む JuMP スカラー関数を返します。この展開は測定データに従って行われます。変数は、展開を収容するために必要に応じてモデルに追加されることに注意してください（すなわち、ポイント変数や半無限変数が必要に応じて作成されます）。測定データおよび/または測定式に対して展開が未定義の場合はエラーが発生します。必要に応じて、これを [`measure`](@ref) と組み合わせて、測定を即座に展開するために使用できます。

これは、カスタムオプティマイザーモデルを使用する拡張にとって便利です。なぜなら、式が新しいモデルに変換される前に測定を評価するために使用できるからです。このメソッドは、[`expand_measure`](@ref) を拡張することでカスタム測定データ型を処理するように拡張することもできます。オプションとして、特定のケースで解析的展開が可能な場合に [`is_analytic`](@ref) によってトリガーされる [`analytic_expansion`](@ref) も拡張できます。

**例**

```julia-repl
julia> tdata = DiscreteMeasureData(t, [0.5, 0.5], [0, 1])

julia> expr = expand(measure(g + z + T - h - 2, tdata))
0.5 g(0) + 0.5 g(1) + z + 0.5 T(0, x) + 0.5 T(1, x) - h(x) - 2
```
