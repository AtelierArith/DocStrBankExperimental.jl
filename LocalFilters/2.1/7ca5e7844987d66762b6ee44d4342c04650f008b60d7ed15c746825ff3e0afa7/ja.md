```
localreduce!(op, [dst = A,] A, dims, rngs; kwds...)
```

は、`A` の値を、`dims` の次元に沿った区間 `rngs` によって定義された連続したハイパー長方形の近傍における、結合的な二項演算子 `op` によるローカルリダクションで `dst` の内容を上書きします。`dims` に関心のある単一の次元が指定されていない限り、宛先 `dst` はソース `A` と同じインデックスを持つ必要があります（つまり、`axes(dst) == axes(A)`）。操作はインプレースで行うことができ、`dst` と `A` は同じであっても構いません。`dst` が指定されていない場合は、これがデフォルトの動作です。リダクションを計算するために van Herk-Gil-Werman のアルゴリズムが使用されます。

メソッドの完全な説明と受け入れられるキーワードについては、[`localreduce`](@ref) を参照してください。

配列 `A` の中心に幅 7 の構造要素を用いたインプレースの *形態学的侵食*（ローカル最小値）は、次のようにして取得できます：

```
localreduce!(min, A, :, -3:3)
```
