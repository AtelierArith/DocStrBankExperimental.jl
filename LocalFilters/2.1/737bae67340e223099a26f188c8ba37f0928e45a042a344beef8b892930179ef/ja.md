```
bottom_hat(A, B=3[, C]; order=FORWARD_FILTER, slow=false) -> dst
```

は、特徴検出のために `B` で定義された構造要素を使用して、配列 `A` にボトムハットフィルタを適用することによって *谷の検出* を行います。ボトムハットフィルタリングは次のように等価です：

```
dst = closing(A, B) .- A
```

オプションの引数 `C` は、ボトムハットフィルタリングの前に `A` を平滑化するための構造要素を指定します。`B` と `C` が構造要素の半径として指定される場合、`C` は `B` よりも小さい必要があります。

キーワード `order` はフィルタの方向を指定し、デフォルトでは `FORWARD_FILTER` です。`C` が指定されている場合、`order` は `B` のための第一の順序と `C` のための第二の順序を指定するための2タプルであることができます。

関連する操作については [`top_hat`](@ref) を、インプレースバージョンについては [`LocalFilters.bottom_hat!`](@ref) を参照してください。
