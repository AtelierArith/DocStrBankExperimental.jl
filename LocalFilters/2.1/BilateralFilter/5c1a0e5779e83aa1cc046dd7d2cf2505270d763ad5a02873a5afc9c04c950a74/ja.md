```
bilateralfilter([T = float(eltype(A)),] A, F, G...=3; order = FORWARD_FILTER)
```

は、配列 `A` にバイラテラルフィルタを適用した結果を返します。

引数 `F` は、値の違いを平滑化する方法を指定します。これは次のようにできます：

  * 関数 `f` で、`f(A[i],A[j])` として呼び出され、`i` が中央インデックス、`j` が近くの位置のインデックスに対して非負の重みを生成します。
  * 正の実数 `σ` で、ガウス分布の標準偏差と見なされます。

引数 `G, ...` は、座標の違いを平滑化するための距離フィルタの設定を指定します。いくつかの可能性があります：

  * `G... = wgt` 非負の重みまたはブール値の配列。`wgt` の軸はオフセットを持っている必要があり、ゼロインデックスが `wgt` のインデックスの一部である必要があります。
  * `G... = f, w` で、`f` は関数、`w` は近傍の拡張を指定するウィンドウ `win` を構築するために使用できる任意の引数です。距離フィルタの値は、`win[i]` が真であるすべてのデカルトインデックス `i` に対して `max(f(i),0)` になります。ウィンドウを指定するさまざまな方法については、[`kernel`](@ref) を参照してください。
  * `G... = σ` または `G... = σ, w` で、`σ` はガウス関数の標準偏差と見なされる正の実数で、`w` は近傍の拡張を指定するウィンドウ `win` を構築するために使用できる任意の引数です。`w` が指定されていない場合、デフォルトのウィンドウサイズ `±3σ` が仮定されます。

オプションの引数 `T` は、結果の要素タイプを指定するためのものです。デフォルトが不適切な場合に必要です。

この関数のインプレースバージョンについては [`bilateralfilter!`](@ref) を参照し、バイラテラルフィルタの説明については [Wikipedia](https://en.wikipedia.org/wiki/Bilateral_filter) を参照してください。
