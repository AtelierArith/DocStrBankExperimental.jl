```
localmap(f, [T=eltype(A),] A, B=3; null=zero(T), order=FORWARD_FILTER)
```

`A`の各位置に対して、`B`で定義された近傍から抽出された`A`の値に関数`f`を適用します。

オプション引数`T`は結果の要素型を指定するためのもので、デフォルトでは`T`は`A`の要素型です。

キーワード`order`はフィルタの方向を指定し、デフォルトでは`FORWARD_FILTER`です。

## 備考

  * 関数`f`は空の値ベクトルで呼び出されることはありません。キーワード`null`は近傍が空のときの結果の値を指定するために使用できます。デフォルトでは、`null = zero(T)`で、`T`は結果の要素型です。
  * `f`に渡される値のベクトルは、必要に応じて`f`によって変更される可能性があります（例えば、値の高速なソートのために）。

## 例

引数`f`を`minimum`または`maximum`に設定すると、`localmap`はそれぞれ入力配列の*侵食*と*膨張*を生成します。ただし、[`erode`](@ref)および[`dilate`](@ref)メソッドは、`localmap`がこれらの関数に特化されるまで、より高速です。

2次元画像`img`に対してスライディング`5×5`ウィンドウの*メディアンフィルタ*を適用するには、次のようにします：

```julia
using Statistics
med = localmap(median!, img, 5; eltype=float(eltype(img)), null=NaN)
```
