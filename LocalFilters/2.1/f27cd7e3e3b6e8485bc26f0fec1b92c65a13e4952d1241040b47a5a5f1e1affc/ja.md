```
top_hat(A, B=3[, C]; order=FORWARD_FILTER, slow=false) -> dst
```

は、特徴検出のために `B` で定義された構造要素を使用して配列 `A` にトップハットフィルタを適用することによって *サミット検出* を実行します。トップハットフィルタリングは次のように等価です：

```
dst = A .- opening(A, B)
```

オプションの引数 `C` は、トップハットフィルタリングの前に `A` を平滑化するための構造要素を指定します。`B` と `C` が構造要素の半径として指定されている場合、`C` は `B` よりも小さい必要があります。例えば：

```
top_hat(bitmap, 3, 1)
```

は、ビットマップ画像内のテキストや線を検出するために使用できます。

キーワード `order` はフィルタの方向を指定し、デフォルトでは `FORWARD_FILTER` です。`C` が指定されている場合、`order` は `B` のための第一の順序と `C` のための第二の順序を指定するための2タプルであることができます。

関連する操作については [`bottom_hat`](@ref) を、インプレースバージョンについては [`LocalFilters.top_hat!`](@ref) を参照してください。
