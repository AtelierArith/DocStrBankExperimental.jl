```
backdoor_criterion(g::AbstractGraph, u::Integer, v::Integer, Z; verbose = false)
```

与えられた有向グラフ `g` に対して、`Z` のノードのいずれも `u` の子孫でなく、`Z` が `u` を `v` から d-分離することをテストします。これは、`u` のバックドアのみを残したサブグラフ（`u` の出力エッジが削除されたもの）において行われます。

もしそうであれば、`u` が `v` に与える因果効果は識別可能であり、次の式で表されます：

```
∑{z∈Z} p(v | u, z)p(z)
```

線形ガウスモデルにおいて、`E[Y | X = x, Z = z] = α + βx + γ'z` を求め、`E[Y | do(x)] = α + βx + γ'E[Z]` を得ます。
