```
stress_tri3_hs(X::T, Y::T, Z::T, P1::V, P2::V, P3::V, Ss::T, Ds::T, Ts::T, λ::T, μ::T) where {T, V}
```

弾性*半空間*における三角形のひずみから生じる応力を計算します。詳細については、特にここで使用される**座標系**については[元のバージョン（補足情報にて）](https://academic.oup.com/gji/article/201/2/1119/572006#86405752)を参照してください。

## 引数

  * `X`, `Y`, `Z`: 観測座標
  * `P1`, `P2`, `P3`: 三角形の頂点座標
  * `Ss`, `Ds`, `Ts`: 三角形のひずみベクトル、ストライクスリップ、ディップスリップ、テンシルスリップ
  * `λ`: ラメの第一パラメータ
  * `μ`: せん断弾性率

## 出力

順番は次の通り: $σ_{xx}$, $σ_{yy}$, $σ_{zz}$,     $σ_{xy}$, $σ_{xz}$, $σ_{yz}$. ```
