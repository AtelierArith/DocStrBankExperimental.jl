```
frechet_mono_via_refinement( P, Q, approx )

PとQの間の「真の」単調フレシェ距離を計算します。
ve_rアルゴリズムを使用します。単調性を得るために
必要に応じて頂点を追加するためのリファインメントを行います。epsパラメータがあることに注意してください -
実数入力の場合はかなり小さく設定できます。いずれにせよ、最悪の場合はフレシェ（単調）
距離を近似するだけです。モーフィングと、結果が真のフレシェ距離である場合はtrueとなるブール値を返します。

この関数は、2つの曲線が巨大な場合、距離を計算する前にそれらを簡略化しないため、非常に遅くなる可能性があることに注意してください。
```

# Returns

4タプル ( m, f, P, Q ) を返します: `m`: 結果を実現するモーフィング。 `f`: 返される距離が正確な連続フレシェ距離であるかどうか。 `P`: リファインされた最初の入力ポリゴン `Q`: リファインされた2番目の入力ポリゴン

指定された場合は、解を計算するために使用された中間モーフィングを返します。
