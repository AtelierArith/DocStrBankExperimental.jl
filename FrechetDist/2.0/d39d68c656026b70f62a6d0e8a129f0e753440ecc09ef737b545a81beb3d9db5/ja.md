```
frechet_c_compute
```

2つのポリゴン間の正確な連続（単調）フレシェ距離を計算します。これは比較的速いはずです。

この関数は、近似バージョンよりもやや遅くなります。正確な答えが本当に必要な場合にのみ使用してください。代わりに frechet*continous*approx を使用することを検討してください。

# 詳細

これは、まず非常に粗い近似を計算し、その後曲線の距離感度の高い簡略化を行うことによって機能します。次に、簡略化された曲線間の単調フレシェ距離を計算し、それを組み合わせて元の2つの曲線間の距離を得ます。答えが同じであることを確認し、そうでない場合は、より細かい簡略化/近似で繰り返します。

最後に、アルゴリズムは、2つの簡略化された曲線間のオフセットを持つフレシェ距離を使用して下限を計算し、これが計算されたフレシェ距離と等しいことを確認します。もし等しければ、2つの曲線のフレシェ距離の上限/下限が同じであることを意味し、計算された距離が実際に望ましいフレシェ距離であることを示唆します。

# さらなる詳細

収束を本当に保証するために、簡略化間で計算された単調距離は、精緻化を使用して計算されるため、ve_r距離が得られます。
