```
frechet_d_compute_sample
```

2つの曲線間の離散フレシェ距離をサンプリングによって計算します。

最初に、2つの曲線をほぼ均一にサンプリングします。nは頂点の数を示すはずですが、コードが頂点が選ばれることを確実にしようとするため、出力はわずかに大きいサンプルになる可能性があります。

# 引数

  * `n`: 2つの曲線を「洗練」する際に追加する頂点の数。計算される頂点の数は大きくなる可能性があります（しかし、あまり大きくはならないことを願っています）。
  * f_lopt    true: 再構成可能なフレシェ距離バージョンを使用    false: 標準の離散フレシェバージョンを使用。
