```
review::Verbosity
```

A flag of [`Verbosity`](@ref) which specifies that [`speculate`](@ref) will show a summary of the number of methods generated, the number of generic methods found, and the duration. If `dry = false`, this also shows the number of generated methods that were compiled, skipped, and warned.

# Examples

```jldoctest
julia> debug
debug::Verbosity
```
