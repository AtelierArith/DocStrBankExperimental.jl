```
waterfall(x, y; kwargs...)
```

個々の正の要素と負の要素を視覚化し、合計結果を隣接するスタックバーの棒グラフとして表示するための[ウォーターフォールチャート](https://en.wikipedia.org/wiki/Waterfall_chart)をプロットします。

## プロットタイプ

`waterfall`関数のプロットタイプエイリアスは`Waterfall`です。

## 属性

**`color`** =  `@inherit patchcolor`  — *ドキュメントは利用できません。*

**`cycle`** =  `[:color => :patchcolor]`  — *ドキュメントは利用できません。*

**`direction_color`** =  `@inherit backgroundcolor`  — *ドキュメントは利用できません。*

**`dodge`** =  `automatic`  — *ドキュメントは利用できません。*

**`dodge_gap`** =  `0.03`  — *ドキュメントは利用できません。*

**`final_color`** =  `plot_color(:grey90, 0.5)`  — *ドキュメントは利用できません。*

**`final_dodge_gap`** =  `0`  — *ドキュメントは利用できません。*

**`final_gap`** =  `automatic`  — *ドキュメントは利用できません。*

**`gap`** =  `0.2`  — *ドキュメントは利用できません。*

**`marker_neg`** =  `:dtriangle`  — *ドキュメントは利用できません。*

**`marker_pos`** =  `:utriangle`  — *ドキュメントは利用できません。*

**`n_dodge`** =  `automatic`  — *ドキュメントは利用できません。*

**`show_direction`** =  `false`  — *ドキュメントは利用できません。*

**`show_final`** =  `false`  — *ドキュメントは利用できません。*

**`stack`** =  `automatic`  — *ドキュメントは利用できません。*

**`width`** =  `automatic`  — *ドキュメントは利用できません。*
