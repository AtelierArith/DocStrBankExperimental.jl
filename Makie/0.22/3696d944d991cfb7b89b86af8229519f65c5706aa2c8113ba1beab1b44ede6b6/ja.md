```
Linestyle(value::Vector{<:Real})
```

プロット関数の `linestyle` キーワード引数の値として使用できる型で、線のスタイルを任意にカスタマイズします。

`value` は、線のダッシュの境界を指定するベクトルです。値 1 と 2 は最初のダッシュを区切り、値 2 と 3 は最初のギャップを区切ります。これは通常、パターンが奇数の値を持つべきであることを意味し、ダッシュの後には常にギャップがあるようになります。

以下はASCIIコードの例です。`[0, 3, 6, 11, 16]` を指定すると、次のパターンが得られます：

```
#  0  3   6   11   16  3  6   11
#   ---   -----     ---   -----
```
