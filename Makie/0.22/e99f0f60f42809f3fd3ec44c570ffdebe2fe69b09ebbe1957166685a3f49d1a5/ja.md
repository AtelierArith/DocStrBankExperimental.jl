```
hidedecorations!(la::Axis; label = true, ticklabels = true, ticks = true,
                 grid = true, minorgrid = true, minorticks = true)
```

x軸とy軸の両方の装飾を隠します：ラベル、目盛りラベル、目盛り、グリッド。キーワード引数を使用して、特定の種類の装飾の隠蔽を無効にすることができます。

関連項目として[`hidexdecorations!`], [`hideydecorations!`], [`hidezdecorations!`]を参照してください。
