```
cam2d!(scene::SceneLike, kwargs...)
```

指定された `scene` のための2Dカメラを作成します。このカメラは、スクロールによるズームとマウスドラッグによる移動を実装しています。また、矩形選択も実装しています。

## キーワード引数

  * `zoomspeed = 0.1f0` ズーム速度を設定します。
  * `zoombutton = true` ズームを有効にするために押す必要があるボタン（組み合わせ）を設定します。デフォルトでは、ボタンを押す必要はありません。
  * `panbutton = Mouse.right` カメラを移動するために使用されるボタンを設定します。これにはマウスボタンが含まれている必要があります。
  * `selectionbutton = (Keyboard.space, Mouse.left)` 矩形選択に使用されるボタンを設定します。これにはマウスボタンが含まれている必要があります。
