```
scatterlines(xs, ys, [zs]; kwargs...)
```

Plots `scatter` markers and `lines` between them.

## Attributes

Available attributes and their defaults for `MakieCore.Plot{Makie.scatterlines}` are: 

```
  color             :black
  colormap          :viridis
  colorrange        MakieCore.Automatic()
  colorscale        identity
  cycle             [:color]
  inspectable       true
  linestyle         "nothing"
  linewidth         1.5
  marker            :circle
  markercolor       MakieCore.Automatic()
  markercolormap    MakieCore.Automatic()
  markercolorrange  MakieCore.Automatic()
  markersize        9
  strokecolor       :black
  strokewidth       0
```
