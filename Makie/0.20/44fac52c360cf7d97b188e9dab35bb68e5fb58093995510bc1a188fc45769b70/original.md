```
pie(fractions; kwargs...)
```

Creates a pie chart with the given `fractions`.

## Attributes

Available attributes and their defaults for `MakieCore.Plot{Makie.pie}` are: 

```
  color           :gray
  inner_radius    0
  inspectable     true
  normalize       true
  offset          0
  radius          1
  strokecolor     :black
  strokewidth     1
  transparency    false
  vertex_per_deg  1
  visible         true
```
