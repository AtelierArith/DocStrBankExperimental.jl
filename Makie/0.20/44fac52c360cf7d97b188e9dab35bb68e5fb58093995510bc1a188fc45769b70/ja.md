```
pie(fractions; kwargs...)
```

指定された `fractions` を使用して円グラフを作成します。

## 属性

`MakieCore.Plot{Makie.pie}` の利用可能な属性とそのデフォルト値は次のとおりです：

```
  color           :gray
  inner_radius    0
  inspectable     true
  normalize       true
  offset          0
  radius          1
  strokecolor     :black
  strokewidth     1
  transparency    false
  vertex_per_deg  1
  visible         true
```
