```
vupdate!(y, [sel,] α, x) -> y
```

は `y` を `α*x + y` で上書きし、`y` を返します。このコードは、乗数 `α` の特定の値に対して最適化されています。たとえば、`α` がゼロの場合、`y` は `x` を使用せずに変更されません。計算は `promote_eltype(x,y)` の数値精度で行われます。

オプションの引数 `sel` は、操作を適用するインデックスの選択です。インデックスが繰り返される場合、その位置で操作が複数回実行されることに注意してください。
