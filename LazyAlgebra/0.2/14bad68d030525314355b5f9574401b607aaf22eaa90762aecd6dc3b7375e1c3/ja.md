```
vdot([T,] [w,] x, y)
```

は `x` と `y` の内積を返します。つまり、`conj(x[i])*y[i]` の合計、または `w` が指定されている場合は `w[i]*conj(x[i])*y[i]` の合計（`w` は実数値の要素を持つ必要があります）を、すべてのインデックス `i` に対して計算します。オプションの引数 `T` は結果の浮動小数点型です。

もう一つの可能性は次の通りです：

```
vdot([T,] sel, x, y)
```

ここで `sel` は内積の計算を特定の要素に制限するためのインデックスの選択です。これは、すべての `i ∈ sel` に対して `x[i]*y[i]` の合計を返します。

引数が複素数値の要素を持ち、`T` が浮動小数点型として指定されている場合、複素数は実数のペアのベクトルと見なされ、結果は次のようになります：

```
vdot(T::Type{AbstractFloat}, x, y)
-> ((x[1].re*y[1].re + x[1].im*y[1].im) +
    (x[2].re*y[2].re + x[2].im*y[2].im) + ...)
```
