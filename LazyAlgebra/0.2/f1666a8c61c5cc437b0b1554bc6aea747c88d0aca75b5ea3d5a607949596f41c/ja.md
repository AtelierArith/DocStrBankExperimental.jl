```
GeneralMatrix(A)
```

は、係数が多次元配列 `A` によって与えられる線形マッピングを作成し、次元を変更するために `reshape` を呼び出すことなく行列-ベクトル積の定義を一般化します。

例えば、`G = GeneralMatrix(A)` で `A` が通常の配列であると仮定すると、`y = G*x` は `x` の次元が `A` の末尾の次元と一致する必要があり、結果 `y` の次元は `A` の残りの先頭の次元となります。したがって、`axes(A) = (axes(y)..., axes(x)...)` となります。`y = G'*x` のように `G` の随伴を適用するには、`x` の次元が `A` の先頭の次元と一致する必要があり、結果 `y` の次元は `A` の残りの末尾の次元となります。したがって、`axes(A) = (axes(x)..., axes(y)...)` となります。
