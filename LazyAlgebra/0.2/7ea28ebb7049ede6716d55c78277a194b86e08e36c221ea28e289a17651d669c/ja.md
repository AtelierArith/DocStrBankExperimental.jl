```
vcombine(α, x, β, y) -> dst
```

は線形結合 `dst = α*x + β*y` を生成します。

---

結果の割り当てを避けるために、宛先配列 `dst` をメソッドのインプレースバージョンで指定できます：

```
vcombine!(dst, α, x, β, y) -> dst
```

このコードは、乗数 `α` と `β` の特定の値に対して最適化されています。たとえば、`α`（または `β`）がゼロの場合、`x`（または `y`）の以前の内容は使用されません。

ソースと宛先は同じであってもかまいません。たとえば、次の2行のコードは同じ結果を生成します：

```
vcombine!(dst, 1, dst, α, x)
vupdate!(dst, α, x)
```

関連情報： [`vscale!`](@ref), [`vupdate!](@ref).
