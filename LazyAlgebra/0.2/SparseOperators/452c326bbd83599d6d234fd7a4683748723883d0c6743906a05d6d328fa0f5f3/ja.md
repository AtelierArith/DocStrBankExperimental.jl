```
SparseOperator{T,M,N}
```

は、スパースオペレータータイプによって継承される抽象型です。パラメータ `T` は要素の型です。パラメータ `M` と `N` はそれぞれ*行*と*列*の次元数です。スパースオペレーターは、`N` 次元の引数に適用して `M` 次元の結果を生成する線形マッピングを実装するという意味で、スパース行列の一般化です（以下で説明します）。*密*行列に対する類似の一般化については、[`GeneralMatrix`](@ref)を参照してください。

圧縮ストレージフォーマットを実装するスパースオペレーターの使用については、[`CompressedSparseOperator`](@ref)を参照してください。
