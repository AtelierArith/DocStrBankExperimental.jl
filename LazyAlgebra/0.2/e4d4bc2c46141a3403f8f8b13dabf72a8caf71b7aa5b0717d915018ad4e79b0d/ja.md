```
Identity()
```

は恒等線形写像を生成します。この写像の目的は、できるだけ効率的であることです。そのため、この写像を適用した結果は入力引数と同じである場合があります。

恒等写像はシングルトンであり、次のようにも利用できます：

```
Id
```

標準ライブラリの `LinearAlgebra` モジュールは、恒等行列に対応する定数 `I` をエクスポートします。`I` が任意の LazyAlgebra の写像と組み合わされると、`Id` のエイリアスとして認識されます。したがって、例えば `I/A`、`A\I`、`Id/A` および `A\Id` はすべて、任意の LazyAlgebra の写像 `A` に対して `inv(A)` を生成します。
