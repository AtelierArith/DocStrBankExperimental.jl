```
vscale!(dst, α, src) -> dst
```

は `dst` を `α*src` で上書きし、`dst` を返します。計算は `promote_eltype(src,dst)` の数値精度で行われます。ソース引数は省略可能で、*インプレース* スケーリングを行うことができます：

```
vscale!(x, α) -> x
```

は `x` を `α*x` で上書きし、`x` を返します。慣例として、結果は `α=0` の場合はゼロで埋められます（ソースの値に関係なく）。

メソッドはデフォルトで提供されており、因子 `α` とソースベクトルの順序を逆にすることができます：

```
vscale!(dst, src, α) -> dst
vscale!(α, x) -> x
```

また、[`vscale`](@ref) や [`LinearAlgebra.rmul!](@ref) も参照してください。
