`alpha, c = forward(Nt,a,A,y)`

隠れマルコフモデル（HMM）のスケーリングされた前向き確率（$\bar{\alpha}$）を計算します。`Ns` 状態を持つモデルで、遷移行列 `A`（`Ns` × `Ns` 行列である必要があります）、初期確率 `a`（`Ns` 長のベクトルである必要があります）、および観測尤度 `y`（サイズが `Nt2` × `Ns` で `Nt2` ≥ `Nt` である必要があります）。

返されるもの：

  * `alpha` 前向き確率を含むサイズ `(Nt,Ns)` の行列
  * `c` 正規化係数を含む `Nt` 長のベクトルで、これを使用して後向き確率（`backward` を参照）や対数最大尤度（`sum(log.(c))`）を計算できます。

`alpha, c = forward(Nt,t2tr,A,y)`

ここで `t2tr` は `Pair{Vector{D},Vector{D}}` の `Nt-1` ベクトルであることができます。例えば、`Nt=3` の場合：

```julia
t2tr = [[1,1]=>[1,2],[1,2]=>[3,3],[3]=>[4]]
```

これは、時刻 `t=1` で状態 `1` から `1` への遷移と `1` から `2` への遷移が許可されていることを示します。時刻 `t=2` では、状態 `1` から `3` への遷移と `2` から `3` への遷移が許可されています。時刻 `t=3` では、状態 `3` から `4` への遷移のみが許可されています。実際には、これらのインデックスを使用して、[理論セクション](@ref theory) で説明されている（おそらくスパースな）時間依存の遷移行列を構築できます。
