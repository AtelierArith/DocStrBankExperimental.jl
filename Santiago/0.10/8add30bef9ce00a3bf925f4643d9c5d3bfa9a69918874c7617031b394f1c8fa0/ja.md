```julia
massflow_summary!(
    s::Santiago.System,
    args...;
    kwargs...
) -> Dict{String, Any}

```

モンテカルロ質量流量計算を実行し、システムプロパティに要約統計を追加します。

## 引数:

  * `M_in`: 各ソースの各物質の流入を含む辞書。
  * `MC`: false の場合、ディリクレ分布の期待値が移転係数として使用されます。そうでない場合、移転係数はディリクレ分布からサンプリングされます。

## キーワード引数

  * `n`: モンテカルロシミュレーションの数。`MC=false` の場合は無視されます。
  * `techflows`: true の場合、個々の技術の流れが要約されます。
  * `scale_reliability`: すべての `Techs` の `transC_reliability` をスケールするための係数。
  * `rng`: オプション、乱数生成器。これはスレッドセーフを得るためにマルチスレッド処理にのみ必要です。
