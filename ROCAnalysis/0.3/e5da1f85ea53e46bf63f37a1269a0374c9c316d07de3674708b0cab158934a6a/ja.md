```
dcf()
```

古典的な決定コスト関数を計算します：

```
dcf = ptar Cmiss pmiss + (1-ptar) Cfa pfa.
```

コスト関数の*相対的*な値は、比率によってのみ決定されます。

```
oeff = (ptar / (1 - ptar)) * (Cmiss / Cfa).
```

したがって、この関数は `ber()` を使用して計算され、適切にスケーリングされます。 引数：

  * `tar`, `non`: ターゲットおよび非ターゲットスコア
  * `tnt::TNT`: ターゲットおよび非ターゲットスコアを含む `TNT` オブジェクト。
  * `r::Roc`: `roc()` の結果である `Roc` オブジェクト
  * `; d::DCF`: 決定コスト関数、デフォルトは `getdcf()`。 これはベクトルまたはDCFである可能性があります。
  * `; thres`: 決定を下すために使用される閾値、デフォルトは `-plo(d)`
  * `; norm`: コストが事前のみを使用して決定する自明なシステムに正規化されているかどうか、デフォルト：`false`
