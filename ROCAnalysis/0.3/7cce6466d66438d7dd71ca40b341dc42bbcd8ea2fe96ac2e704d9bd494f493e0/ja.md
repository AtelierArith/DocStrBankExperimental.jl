```
roc(tar, non; laplace; collapse=true, laplace=false)`
```

二クラス分類器の性能評価のための基本的な統計を計算します。 スコアの真のクラスは、それらが出現する配列にエンコードされており、すなわち、`tar`ゲットまたは`non`-ターゲットスコアです。

結果は、テストのための受信者動作特性（ROC）データを含む型`Roc`で提供されます。

オプションの引数：

  * `collapse=true`は、偽陽性（アラーム）および偽陰性（ミス）の確率に対する結果の配列を統合するかどうかを示します。つまり、同じ偽陽性または偽陰性率を持つROC上の連続した点が削除され、ROCの「コーナー」点のみが保持されます。
  * `laplace=false`は、スコアスケールの両端にターゲットおよび非ターゲットスコアに追加のデータポイントを追加するかどうかを示します。これはラプラス事前分布に対応し、ROCの凸包の線分に関連する最適な対数尤度比の大きさを制限する結果になります。
