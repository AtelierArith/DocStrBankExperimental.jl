```
setdcf()
```

グローバルな決定コスト関数を設定します。この DCF は `dcf()` と `mindcf()` で使用できます。決定コスト関数は次のように定義されます。

```
dcf = ptar Cmiss pmiss + (1-ptar) Cfa pfa.
```

これは、パラメータ `ptar`（ターゲットの事前確率）、`Cfa`（偽陽性のコスト）、および `Cmiss`（偽陰性のコスト）によって定義されます。引数：

  * `ptar`。ターゲットの事前確率、デフォルトは `0.5`。
  * `cfa`。偽陽性（誤警報）のコスト、デフォルトは `1`
  * `cmiss`。偽陰性（ミス）のコスト、デフォルトは `1`。

これらのパラメータのいずれか、または組み合わせをベクトルとして指定することで、複数の同時コスト関数を設定できます。

DCF のパラメータの現在の値は `getdcf()` を使用して確認できます。
