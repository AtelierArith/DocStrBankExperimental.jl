```
eerch(r::ROC)
```

Convex Hullの解釈を使用して、Equal Error Rateを計算します。これは、ROC曲線の凸包がROCの`y=x`対角線を横切るときのエラー率を計算します。

この値は、試行の数が少ない場合の試行サンプリングに対してあまり敏感ではなく、`最小決定コストの最大値が変動する`という解釈を持ちます。これは、二クラス分類システムの決定コスト関数分析において有用な量です。

引数:

  * `r::Roc`: `roc()`の結果である`Roc`型のオブジェクト
  * `tar, non`: ターゲットおよび非ターゲットスコアのベクトル
  * `pfa, pmiss, ch`: 偽陽性率および偽陰性率のベクトル、および凸包上の`(pfa, pmiss)`点のメンバーシップを示す配列。これらの点は`roc()`によって見つけられます。
