```
req = @request expr
```

関数から求められる内部結果のリクエストを作成します。[`@espy`](@ref)の例として提示された関数を考慮すると、可能な構文の例には以下が含まれます。

```
req       = @request gp[].(s,z,material.(a,b))
req       = @request gp[].(s)
req       = @request gp[].(material.(a))
```

最初の式は次のように解釈できます。「関数内には変数`igp`に対する`for`ループがあり、結果はベクトルとして求められます（ループの各サイクルに対して1つの要素）。ベクトルの各要素は`material`というフィールドを持つ`type`（構造体）である必要があります。なぜなら、その名前の関数がforループ内で呼び出されるからです。その関数内で、変数`a`が取得されることになります。

単一要素のリストに対しても括弧を使用する必要があることに注意してください、例えば`(s)`のように。

参照してください: [`@espy`](@ref), [`@espydbg`](@ref), [`makekey`](@ref)
