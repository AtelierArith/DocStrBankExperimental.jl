```
ladmm(x0, prox_f!, prox_g!, A; λ=1., μ=λ*inv(norm(A)), α=1., ϵ_abs=1e-7, ϵ_rel=1e-4, max_iter=1000)
```

目的関数 $f(x) + g(Ax)$ を最小化します。ここで、$f(x)$ と $g(Ax)$ はどちらも非滑らかであり、線形化された交互方向法を使用します。交互方向法には、3つのハイパーパラメータ `μ`、`λ`、`α` があります。`μ` と `λ` は更新ステップのスケーリングを制御し、擬似ステップサイズとなります。`λ` は拡張ラグランジアンパラメータの逆数であり、`μ` は近似項に対応する拡張ラグランジアンパラメータの逆数であり、$μ ∈ (0,λ/||A||²₂]$ です。$α ∈ [0,2]$ は緩和パラメータであり、$α < 1$ は過小緩和を示し、$α > 1$ は過大緩和を示します。

#### 引数

  * `x0::AbstractVector`: 初期パラメータ値 (n x 1)
  * `prox_f!::Function` : $f(x)$ の近接演算子
  * `prox_g!::Function` : $g(x)$ の近接演算子
  * `A::AbstractMatrix` : $g(x)$ の行列変換 (p x n)
  * `λ::Real`           : $g(x)$ の近接スケーリングパラメータ
  * `μ::Real`           : $f(x)$ の近接スケーリングパラメータ
  * `α::Real`           : 緩和パラメータ
  * `ϵ_abs::Real`       : 絶対許容誤差
  * `ϵ_rel::Real`       : 相対許容誤差
  * `max_iter::Integer` : 最大反復回数

#### 戻り値

  * `x::AbstractVector` : 最小化された値 $∈ dom f$ (n x 1)
  * `z::AbstractVector` : 最小化された値 $∈ dom g$ (p x 1)
