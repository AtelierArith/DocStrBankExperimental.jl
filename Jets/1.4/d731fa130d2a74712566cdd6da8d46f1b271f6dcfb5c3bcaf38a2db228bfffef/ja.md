```
lhs,rhs = dot_product_test(A, m, d; mmask, dmask)
```

*ドット積テスト*の左辺と右辺を計算して返します：

`<d,Am> ≈ <Aᴴd,m>`

ここで `Aᴴ` は `A` の共役転置または随伴であり、`<x, y>` はベクトル `x` と `y` の内積を示します。ドット積テストの左辺と右辺は、演算子 `A` に対して機械精度に近い等価であることが期待されます。等式が成り立たない場合、これは演算子 `A` の実装に問題があることを示す可能性があります。

この関数は、演算子の適用前にベクトル `m` と `d` をマスクするために要素ごとの乗算を介して適用される、`A` の定義域と値域のベクトルであるオプションの名前付き引数 `mmask` と `dmask` を提供します。以下に示すように、ここでは `∘` を使用して2つのベクトルのアダマール積（要素ごとの乗算）を表します。

`<dmask ∘ d, A (mmask ∘ m)> ≈ <Aᵀ (dmask ∘ d), mmask ∘ m>`

この関係を使用して、この関数によって返される左辺 `lhs` と右辺 `rhs` の演算子の相対精度をテストできます：

`|lhs - rhs| / |lhs + rhs| < ϵ`
