`@computation`を使用する場合：

```
'''
    something(daf::DafWriter, x::Int = $(DEFAULT.x); y::Bool = $(DEFAULT.y))

...
`x`（デフォルト: $(DEFAULT.y)）が偶数の場合、...
...
`y`（デフォルト: $(DEFAULT.y)）が設定されている場合、...
...
'''
@computation Contract(...) function something(daf::DafWriter, x::Int = 0; y::Bool = false)
    return ...
end
```

この場合、`$(DEFAULT.x)`はパラメータ`x`のデフォルト値で展開されます。各パラメータの効果の説明をドキュメントのどこかに含めることは良い習慣であり、そのデフォルト値を提供することも礼儀です。これは、シグネチャ行またはテキスト、またはその両方で行うことができます。`DEFAULT`を使用することで、ドキュメントに正しい値が使用されることが保証されます。
