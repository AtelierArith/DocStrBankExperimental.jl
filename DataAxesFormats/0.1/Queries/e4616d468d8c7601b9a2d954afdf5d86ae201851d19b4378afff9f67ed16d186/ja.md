```
Names(kind::Maybe{AbstractString} = nothing) <: Query
```

名前のセットを検索するためのクエリ操作です。文字列 [`Query`](@ref) では、`?` 演算子を使用して指定され、オプションで名前を付けるオブジェクトの種類が続きます。

  * クエリ状態が空の場合、`kind` を指定する必要があり、`scalars` または `axes` のいずれかでなければならず、結果はそれらの名前のセットです（`? scalars`、`? axes`）。
  * クエリ状態に単一の軸が含まれている場合（マスクなし）、`kind` は指定する必要がなく、結果はその軸のベクトルプロパティの名前のセットです（例：`/ cell ?`）。
  * クエリ状態に二つの軸が含まれている場合（マスクなし）、`kind` は指定する必要がなく、結果はその軸の行列プロパティの名前のセットです（例：`/ cell / gene ?`）。

!!! note
    これ、[`Lookup`](@ref) および [`Axis`](@ref) は、完全な [`Query`](@ref) としても機能する唯一の [`QueryOperation`](@ref) です。
