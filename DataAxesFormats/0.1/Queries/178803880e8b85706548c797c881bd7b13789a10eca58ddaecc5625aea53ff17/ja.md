```
IfNot(value::Maybe{StorageScalar} = nothing) <: QueryOperation
```

ベクトル内の「偽っぽい」値（空の文字列、ゼロの数値、または偽のブール値）に使用する値を提供するクエリ操作です。文字列 [`Query`](@ref) では、これは `??` 演算子を使用して示され、オプションで使用する値を続けて指定できます。

値が `nothing`（デフォルト）の場合、これらのエントリは結果から削除されます（マスクされます）（例：`/ cell : type ?` は `/ cell & type : type` と同じ動作をし、非空のタイプを持つセルのタイプを返します）。そうでない場合、この値は「偽っぽい」値の代わりに使用されます（例：`/ cell : type ? Outlier` は、空のタイプを持つセルに対して `Outlier` の値を持つ各セルのタイプのベクトルを返します）。プロパティを取得する際には、これは最終的な値です（例：`/ cell : type ? red => color` は、空のタイプを持つセルに対して `red` の色を持つ各セルのタイプの色のベクトルを返します）。

`value` が `AbstractString` の場合、自動的に結果ベクトルの要素のデータ型に変換されます。
