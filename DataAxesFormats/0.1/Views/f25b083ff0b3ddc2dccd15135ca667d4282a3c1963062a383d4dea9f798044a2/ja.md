単一のデータをビューから公開するように指定します。これは、ペアのベクターとして指定されます（`Dict`を初期化するのに似ています）。ペアの順序は重要であり（最後のものが勝ちます）、ペアの代わりにタプルを指定することも許可されており、Pythonなどの他の言語からAPIを呼び出すのが簡単になります。

**スカラー**は、[`ViewAxes`](@ref)と同様に指定されますが、クエリはベクターではなくスカラーを返す必要があります。つまり、`"*"`（または[`ALL_SCALARS`](@ref)）と言うことで、ビューからすべての元の`daf`データスカラーを公開します。これに続けて`"version" => nothing`と言うことで、ビューから`version`を隠します。`"total_umis" => q"/ cell / gene : UMIs %> Sum %> Sum"`を追加すると、すべての細胞のすべての遺伝子のUMIの合計を含む`total_umis`スカラーが公開されます。

**ベクター**はスカラーと同様に指定されますが、軸とプロパティ名の両方を指定するキーが必要です。軸はビューによって公開されている必要があります（`axes`パラメータに基づいて）。軸が`"*"`の場合、`axes`パラメータで指定されたすべての公開された軸名に置き換えられます。同様に、プロパティ名が`"*"`（例：`("gene", "*")`）の場合、基本データの公開された軸のすべてのベクタープロパティに置き換えられます。したがって、`("*", "*")`（または[`ALL_VECTORS`](@ref)）を指定すると、すべての（公開された）軸のすべてのベクタープロパティも公開されます。

ベクターの値は、適切な軸に基づくベクタークエリのサフィックスでなければなりません；値が`"="`の場合、プロパティをそのまま公開するために再度使用されます。

たとえば、`axes = ["cell" => q"/ cell & type = TCell"]`を指定し、次に`data = [("cell", "total_noisy_UMIs") => q"/ gene & noisy : UMIs %> Sum`を指定すると、`total_noisy_UMIs`が`cell`ごとのベクタープロパティとして公開され、クエリ`/ gene & noisy / cell & type = TCell : UMIs %> Sum`を使用して、各細胞（`type`が`TCell`である）のすべてのノイジー遺伝子の`UMIs`の合計を計算します。

**行列**は、両方の軸とプロパティ名を指定するキーが必要です。軸は両方ともビューによって公開されている必要があります（`axes`パラメータに基づいて）。再度、いずれかまたは両方の軸が`"*"`の場合、それらはすべての公開された軸（`axes`パラメータに基づいて）に置き換えられ、名前が`"*"`の場合、軸のすべての行列プロパティに置き換えられます。行列の値は再度`"="`を使用してプロパティをそのまま公開するか、行列クエリのサフィックスにすることができます。したがって、`("*", "*", "*")`（または`ALL_MATRICES`）を指定すると、すべての（公開された）軸のすべての行列プロパティも公開されます。

つまり、`gene`と`cell`の軸が`axes`パラメータによって公開されていると仮定すると、`("cell", "gene", "log_UMIs") => q": UMIs % Log base 2 eps"`を指定すると、各細胞と遺伝子の行列`log_UMIs`が公開されます。

軸の順序は重要ではないため、`data = [("gene", "cell", "UMIs") => "="]`は`data = [("cell", "gene", "UMIs") => "="]`と同じ効果があります。

**3Dテンソル**は、主軸、2つの軸、およびプロパティ名を指定するキーが必要です。すべての軸はビューによって公開されている必要があります（`axes`パラメータに基づいて）。この場合、いずれの軸も`"*"`であってはなりません。値は、テンソルのすべての行列プロパティをそのまま公開するための`"="`またはすべてを隠すための`nothing`のみです。つまり、ビューは既存の（マスクされた可能性のある）3Dテンソルを公開または隠すことができますが、新しいものを作成するためには使用できません。

つまり、`gene`、`cell`、および`batch`の軸が`axes`パラメータによって公開されていると仮定すると、`("batch", "cell", "gene", "is_measured") => "="`を指定すると、各細胞ごとに遺伝子ごとの行列`batch1_is_measured`、`batch2_is_measured`などのセットが公開されます。
