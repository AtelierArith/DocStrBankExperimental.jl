```
chain_reader(dafs::AbstractVector{<:DafReader}; name::Maybe{AbstractString} = nothing)::DafReader
```

[`DafReader`](@ref)の読み取り専用チェーンラッパーを作成し、それらを単一の[`DafReader`](@ref)として提示します。コンテンツにアクセスする際、公開される値はデータを含む最後のデータセットによって提供されるものであり、つまり、後のデータセットは前のデータセットを上書きすることができます。ただし、チェーン内の複数のデータセットに軸が存在する場合、そのエントリは同一でなければなりません。これは通常手動で作成されるものではなく、代わりに[`chain_reader`](@ref)を呼び出します。

!!! note
    これはチェーンを作成する際に軸が一貫していることを確認しますが、その後にチェーンされたデータを変更して不整合な軸を作成することに対する防御にはなりません。*それはしないでください*。
