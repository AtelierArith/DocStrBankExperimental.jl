```
DurationHorizons{T} <: AbstractHorizons{T}
```

リセーディングホライズンモデルの最適化および実装ホライズンを、ホライズンの期間として指定するために使用される型です。これは、異なる反復における運用期間の数が変動する可能性があることを意味します。

`DurationHorizons`を反復すると、単一の実行に必要なすべての情報を含む`SingleHorizon`が得られます。

# フィールド

  * **`len::Int64`** は、調査された時間枠の総長で、1つの運用期間の期間の倍数です。
  * **`dur::Vector{T}`** は、各個別の運用期間の期間のベクトルです。
  * **`optim::Int64`** は、***最適化***ホライズンで使用される運用期間の***期間***の合計です。
  * **`impl::Int64`** は、***実装***ホライズンで使用される運用期間の***期間***の合計です。

!!! note "最適化および実装ホライズン"
    最適化ホライズンは、各個別の最適化実行で使用されるホライズンに対応し、実装ホライズンは最適化ホライズンよりも短くなければなりません。これは内部コンストラクタを通じて強制されます。これは、モデル実行から抽出された運用期間に対応します。
