```
firstinds(ic::Vector{Int})
firstinds(ib::Vector{Vector{Int}})
```

入力整数ベクトル `ic` の各ユニークな値の最初のインデックス位置を含む整数のベクトル、または整数ベクトルの入力ベクトル `ib` の各エントリの最初のインデックス位置を返します。

`unique(A, dim)` から返された出力に対して操作する場合、返される整数のベクトルは、元の入力多次元配列 `A` の次元 `dim` に沿った各ユニークスライスの最初の位置に対応します。

整数のベクトルを受け入れる `firstinds` の実装は、`groupslices(A, dim)` から返された出力に対して操作します。

整数のベクトルのベクトルを受け入れる `firstinds` の実装は、`groupinds(ic::Vector{Int})` から返された出力に対して操作します。
