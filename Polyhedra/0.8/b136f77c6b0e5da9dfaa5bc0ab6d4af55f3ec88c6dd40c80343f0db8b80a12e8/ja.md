```
hchebyshevcenter(p::HRep[, solver]; linearity_detected=false, proper=true)
```

ポリヘドロン `p` に含まれる最大のユークリッド球の中心と半径をタプルで返します。ポリヘドロンが空であるか、半径が無限大の場合はエラーをスローします。`linearity_detected` が false でない限り、最初に線形性が検出されます。

ポリトープには複数のチェビシェフ中心が存在する場合があります。一般に、ポリトープ `p` のチェビシェフ中心の集合は、`p` が正の次元を持つ場合、`p` よりも次元が低いポリトープです。たとえば、`p` が長方形 `[-2, 2] x [-1, 1]` の場合、`p` のチェビシェフ半径は 1 で、チェビシェフ中心の集合は `[-1, 1] x {0}` です。*proper* チェビシェフ中心は `(0, 0)` であり、`[-1, 1] x {0}` のチェビシェフ中心です。`!proper` の場合は、任意のチェビシェフ中心が返されます（返されるものはソルバーによって異なります）。そうでない場合、適切なチェビシェフ中心が計算されます。適切なチェビシェフ中心は、`p` の次元に関して帰納的に定義されます。`p` の次元が 0 の場合、それは単一要素であり、その適切なチェビシェフ中心は `p` の唯一の要素です。そうでない場合、`p` のチェビシェフ中心の集合 `q` の次元は `p` の次元よりも小さく、`p` の適切なチェビシェフ中心は `q` の適切なチェビシェフ中心です。
