```
aai"user_prompt"[model_alias] -> AIMessage
```

結果が準備でき次第ログに記録される`@ai_str`マクロの非同期バージョンです。

提供されたメッセージに対する非同期の返信が必要な場合は、`aai!""`も参照してください。会話を続けることができます。    

# 例

GPT-4に非同期リクエストを送信し、応答を待つ必要がないようにします：遅いモデルで非常に実用的で、その間に作業を続けることができます。

```julia m = aai"Say Hi!"gpt4;

# ...しばらく待って...

# [ Info: Tokens: 29 @ Cost: 0.0011 in 2.7 seconds

# [ Info: AIMessage> こんにちは！今日はどのようにお手伝いできますか？
```
