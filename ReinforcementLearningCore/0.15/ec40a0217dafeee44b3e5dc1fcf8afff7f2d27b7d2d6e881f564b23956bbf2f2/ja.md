```
TargetNetwork(network; sync_freq = 1, ρ = 0f0, use_gpu = false)
```

強化学習のためのターゲットネットワークを構築します。

# 引数

  * `network`: トレーニングに使用されるメインネットワーク。
  * `sync_freq`: ターゲットネットワークがメインネットワークと同期される頻度（`optimise!`への呼び出し回数）。デフォルトは1。
  * `ρ`: ターゲットネットワークを更新するために使用される補間係数。範囲は[0, 1]でなければなりません。デフォルトは0（古い重みが新しい重みに完全に置き換えられます）。
  * `use_gpu`: ターゲットネットワークにGPUを使用するかどうかを指定します。デフォルトは`false`。

# 戻り値

`TargetNetwork`オブジェクト。
