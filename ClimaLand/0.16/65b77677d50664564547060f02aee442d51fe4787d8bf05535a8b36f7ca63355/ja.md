```
make_update_boundary_fluxes(
    land::SoilSnowModel{FT, SnM, SoM},
) where {
    FT,
    SnM <: Snow.SnowModel{FT},
    SoM <: Soil.EnergyHydrology{FT},
    }
```

関数を作成するメソッド; 返される関数は、統合モデルの追加の補助変数を更新し、すべてのコンポーネントモデルの境界補助変数も更新します。

この関数は、傾向関数の評価の前に、各ODE関数評価ごとに呼び出されます。

このメソッドでは、

1. 土壌と雪の間の地面熱フラックスを計算します。これは、雪と土壌の境界フラックスを更新するために必要です。
2. 雪の境界フラックスを更新し、雪が完全に溶ける際に発生するエネルギーまたは水の過剰フラックスを計算します。この場合、その過剰は保存のために土壌に行かなければなりません。
3. 土壌の境界フラックスを更新し、事前に計算された地面熱フラックスと雪からの過剰フラックスを使用します。
4. 大気のネットフラックスを計算し、保存の評価に役立てます。
