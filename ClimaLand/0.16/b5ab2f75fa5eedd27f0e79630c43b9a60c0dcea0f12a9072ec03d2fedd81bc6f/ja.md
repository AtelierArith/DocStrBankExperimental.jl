```
call_count_nans_state(state, mask::ClimaCore.Fields.Field = nothing, verbose = false)
```

これは、状態内のNaNの数をカウントする関数を呼び出します。例えば、`solve`を呼び出した後の`sol.u[end]`によって与えられるFieldVectorです。この関数は、シミュレーションが安定しているかどうかを定量的に判断するために、シミュレーションのデバッグに役立ちます。これは、非カラムシミュレーションでのみ使用されます。

この関数がFieldVectorに対して呼び出されると、FieldVector内の各Fieldに対して再帰的に自分自身を呼び出します。Fieldに対して呼び出されると、そのField内のNaNの数をカウントし、見つかった場合は警告を生成します。この動作は2Dフィールドと3Dフィールドで異なるため、`space`に基づいてディスパッチします。

`mask`としてClimaCore Fieldが提供されると、関数はマスクが1である状態変数内のNaNのみをカウントします。これは、海上のNaNをカウントしないように、陸/海マスクと一緒に使用することを意図しています。このマスクは、陸上で1、海上で0であると仮定され、マスクは2Dフィールドである必要があります。

`verbose`引数は、NaNが見つからなかった場合に関数が出力を生成するかどうかを切り替えます。
