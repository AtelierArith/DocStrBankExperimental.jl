```
PrescribedRadiativeFluxes{FT, SW, LW, DT, T, TP} <: AbstractRadiativeDrivers{FT}
```

スタンドアロンモードで陸上モデルを駆動するために必要な指定された放射関数のコンテナです。

一部のモデルでは、天頂角と拡散比率の両方が必要です。拡散比率は直接提供される場合（TimeVaryingInput型）もあれば、経験的に計算される場合もあります。後者の場合、計算するために熱力学的パラメータも必要です。

したがって、許可される組み合わせは次のとおりです：

1. 天頂角と拡散比率が不要：天頂角=nothing、熱力学的パラメータ=nothing、拡散比率=nothing
2. 天頂角が提供され、拡散比率が経験的に計算される：熱力学的パラメータが使用され、拡散比率=nothing
3. 天頂角が提供され、拡散比率が提供される：熱力学的パラメータは使用されず、拡散比率はTimeVaryingInput

  * `SW_d`: 時間の短波放射関数（W/m^2）：正の値は表面に向かうことを示す
  * `frac_diff`: 短波放射の拡散比率（無次元、[0,1]）
  * `LW_d`: 時間の長波放射関数（W/m^2）：正の値は表面に向かうことを示す
  * `start_date`: 開始日 - シミュレーションのt=0に対応する日時
  * `θs`: 太陽の天頂角（ラジアン）
  * `thermo_params`: 熱力学的パラメータ
