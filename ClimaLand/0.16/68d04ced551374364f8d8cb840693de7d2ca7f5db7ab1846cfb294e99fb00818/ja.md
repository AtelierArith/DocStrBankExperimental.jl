```
make_set_initial_state_from_file(ic_path, model::EnergyHydrology{FT}) where {FT}
```

`ic_path`から初期条件を取得し、引数(Y,p,t0,model)を受け取って状態Yをその場で更新する関数を返します。キャッシュ`p`のフィールドは事前に割り当てられたメモリとして使用され、更新されますが、これはキャッシュの状態がYおよびtと完全に一致することを意味するものではありません。

現在、グローバルシミュレーションのみに対してテストされ、使用されていますが、返される関数はカラムシミュレーションにも適用できるはずです。

返される関数は`ic_path`のクロージャです。多くの他のClimaLand関数と同様に`model`のためのものでもありますが、ユーザーが独自の初期条件関数を定義したい場合に必要なパラメータなどが`model`に格納される可能性があるため、`set_ic!`の引数`model`を保持したいと考えています。
