```
make_set_initial_state_from_file(ic_path, land::LandModel{FT}) where {FT}
```

`ic_path`から初期条件を取得し、引数(Y,p,t0,land)を使って状態Yをその場で更新する関数を返します。キャッシュ内のフィールド`p`は事前に割り当てられたメモリとして使用され、更新されますが、これはキャッシュの状態がYおよびtと完全に一致することを意味するものではありません。

現在、グローバルシミュレーションのためにのみテストされ、使用されていますが、返される関数はカラムシミュレーションでも機能するはずです。

返される関数は`ic_path`のクロージャです。多くの他のClimaLand関数と同様に`land`のためのものでもありますが、ユーザーが独自の初期条件関数を定義したい場合、パラメータなどが`land`に格納される可能性があるため、`set_ic!`の引数`land`を保持したいと考えています。
