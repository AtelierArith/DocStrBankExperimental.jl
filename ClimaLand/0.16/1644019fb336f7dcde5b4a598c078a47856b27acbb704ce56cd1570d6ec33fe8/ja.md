```
make_set_initial_state_from_file(ic_path, land::SoilCanopyModel{FT}) where {FT}
```

`ic_path`からの初期条件で状態Yをインプレースで更新する関数を返します。この関数は(Y,p,t0,land)を引数として受け取ります。キャッシュ`p`のフィールドは事前に割り当てられたメモリとして使用され、更新されますが、これはキャッシュの状態がYおよびtと完全に一致することを意味するものではありません。

現在、グローバルシミュレーションのためにのみテストされ、使用されていますが、返される関数はカラムシミュレーションでも機能するはずです。

返される関数は`ic_path`のクロージャです。また、他の多くのClimaLand関数のように`land`のためのものでもありますが、ユーザーが独自の初期条件関数を定義したい場合に必要なパラメータなどが`land`に格納される可能性があるため、`set_ic!`の引数`land`を保持したいと考えています。
