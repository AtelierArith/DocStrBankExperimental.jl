```
cutree(hclu::Hclust; [k], [h]) -> Vector{Int}
```

`hclu` のデンドログラムを切り取って、指定された粒度レベルでクラスタを生成します。

クラスタ割り当てベクトル $z$ を返します ($z_i$ は $i$-番目のデータポイントのクラスタのインデックスです)。

# 引数

  * `k::Integer` (オプション) 希望するクラスタの数。
  * `h::Real` (オプション) 木が切られる高さ。

`k` と `h` の両方が指定されている場合、クラスタの数は `k` 以上であり、その高さは `h` を超えないことが保証されます。
