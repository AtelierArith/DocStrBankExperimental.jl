```
get_Δat(JD::Number) -> Float64
```

指定された `JD` における UTC と国際原子時 (TAI) の間の累積うるう秒 (ΔAT) [s] を取得します。この関数は配列 `ΔAT_Data` で ΔAT を検索します。

# 備考

`JD` が `_ΔAT[1, 1]` より前の場合、10 が返されます。**これによりエラーが発生する可能性があることに注意してください。**

`JD` が `_ΔAT[end, 1]` より後の場合、`_ΔAT[end, 2]` が返されます。これは、うるう秒がいつ追加されるかを予測することがまだ不可能であるためです。
