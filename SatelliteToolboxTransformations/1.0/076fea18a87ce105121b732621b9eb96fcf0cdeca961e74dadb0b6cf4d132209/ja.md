```
mean_obliquity_iau2006(jd_tt::Number) -> Float64
```

ユリウス日 `jd_tt` [地球時間] を使用して、春分点に基づくIAU-2006理論を用いて黄道の平均傾斜角 [rad] を計算します。

このアルゴリズムは**[1]**から得られました。
