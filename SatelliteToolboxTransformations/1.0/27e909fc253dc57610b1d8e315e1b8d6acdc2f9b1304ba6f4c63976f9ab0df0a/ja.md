```
r_mj2000_to_mod_iau2006([T, ]jd_tt::Number) -> T
```

J2000平均赤道フレームをジュリアン日`jd_tt` [地球時]の平均日（MOD）参照フレームに整列させる回転を計算します。このアルゴリズムはIAU-2006理論を使用しています。

回転のタイプはオプションの変数`T`によって説明されます。もし`DCM`であれば、DCMが返されます。そうでなければ、`Quaternion`が返されます。このパラメータが省略された場合、`DCM`にフォールバックします。

# 戻り値

  * `T`: MJ2000フレームをMODフレームに整列させる回転。

# 備考

ここでのJ2000参照フレームは、FK5理論での以前の定義とは等しくありません。そのため、内部的には`MJ2000`と呼ばれています。**[1]**によれば：

> 考慮すべきJ2000.0の平均春分点は、過去に使用された「回転動力学的平均春分点J2000.0」ではなく、最近の数値または解析的解に関連する「慣性動力学的平均春分点J2000.0」です。後者は、地球-月重心の軌道運動の角運動量ベクトルに対して、慣性系に対する重心の速度から計算された面である黄道に関連しています。回転春分点は、地球-月重心の回転軌道面に対して参照される速度から計算された角運動量ベクトルに対して垂直な回転の意味での黄道に関連しています。（2つの角運動量の違いは、軌道面の回転に関連する角運動量です。）
