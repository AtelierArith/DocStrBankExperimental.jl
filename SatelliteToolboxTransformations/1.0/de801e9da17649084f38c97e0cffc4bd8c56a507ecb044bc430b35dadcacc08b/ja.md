```
r_tirs_to_ers_iau2006([T, ]jd_ut1::Number, jd_tt::Number, δΔΨ_2000::Number = 0) -> T
```

地球中間基準系（TIRS）をユリウス日 `jd_ut1` [UT1] および `jd_tt` [地球時間] における地球基準系（ERS）に整列させる回転を計算します。このアルゴリズムはIAU-2006理論を使用しています。

通常、IERS EOPデータから取得される経度の摂動に対する補正（`δΔψ_2000`）[rad]を提供することができます（[`fetch_iers_eop`](@ref) および [`compute_δΔϵ_δΔψ`](@ref]を参照）。この補正は、液体の地球コアの影響をモデル化する自由コア摂動（FCN）に関連しています。

回転のタイプはオプションの変数 `T` によって説明されます。もしそれが `DCM` であれば、DCMが返されます。そうでなければ、`Quaternion` であれば、Quaternionが返されます。このパラメータが省略された場合、`DCM` にフォールバックします。

# 戻り値

  * `T`: TIRSフレームをERSフレームに整列させる回転。

# 備考

基準フレームTIRSとERSは、グリニッジの見かけの恒星角（GAST）に関するZ軸周りの回転によって分離されています。この角度はIAU-2006理論を使用して計算され、地球回転角（ERA）を取得し、起源の方程式（EO）の結果を引き算することから成ります。
