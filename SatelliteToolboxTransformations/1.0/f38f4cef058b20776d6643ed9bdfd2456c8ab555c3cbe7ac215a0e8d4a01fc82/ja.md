```
r_mod_to_tirs_iau2006([T, ]jd_ut1::Number, jd_tt::Number, δΔϵ_2000::Number = 0, δΔΨ_2000::Number = 0) -> T
```

平均日付 (MOD) 参照フレームをユリウス日 `jd_ut1` [UT1] と `jd_tt` [地球時間] における地球中間参照系 (TIRS) に整列させる回転を計算します。このアルゴリズムはIAU-2006理論を使用しています。

傾斜の摂動に対する補正 (`δΔϵ_2000`) と経度に対する補正 (`δΔψ_2000`) [rad] を提供することができ、これらは通常IERS EOPデータから取得されます (参照: [`fetch_iers_eop`](@ref) および [`compute_δΔϵ_δΔψ`](@ref))。これらの補正は、液体の地球コアの影響をモデル化する自由コア摂動 (FCN) に関連しています。

回転のタイプはオプションの変数 `T` によって説明されます。もし `DCM` であれば、DCMが返されます。そうでなければ、`Quaternion` であれば、Quaternionが返されます。このパラメータが省略された場合、`DCM` にフォールバックします。

!!! info
    この合成回転 TIRS <=> ERS <=> MOD は新しい関数として実装されています。なぜなら、単一の回転 TIRS <=> ERS と ERS <=> MOD は、計算負荷が高い `nutation_eo` 関数を呼び出すからです。この場合、合成アルゴリズムはそれらの関数を個別に呼び出すよりも約2倍速くなります。


# 戻り値

  * `T`: TIRSフレームをERSフレームに整列させる回転。
