```
r_pef_to_itrf_fk5([T, ]x_p::Number, y_p::Number) -> T
```

擬似地球固定（PEF）を国際地球基準系（ITRF）に整列させる回転を計算します。これは、IERS EOPデータから取得された角度 `x_p` [rad] および `y_p` [rad] で表される極運動を考慮します（[`fetch_iers_eop`](@ref)を参照）。

`x_p` はX軸周りの極運動の変位であり、これはIERS基準子午線の方向（0˚経度の子午線に沿って南方向に正）です。`y_p` はY軸周りの極運動の変位（90˚Wまたは270˚Eの子午線）です。

回転のタイプはオプションの変数 `T` で説明されます。もしそれが `DCM` であれば、DCMが返されます。そうでなければ、`Quaternion` であればQuaternionが返されます。このパラメータが省略された場合、デフォルトで `DCM` に戻ります。

# 戻り値

  * `T`: PEFフレームをITRFに整列させる回転。

# 備考

ITRFは国際的な委員会によって合意された地球の極の位置である国際基準極（IRP）に基づいて定義されています **[1]**。一方、擬似地球固定（PEF）は地球の回転軸、または天体中間極（CIP）に基づいて定義されています。したがって、PEFのXY平面には真の赤道が含まれます。さらに、回復された緯度と経度はCIPに敏感であるため、PEFフレームを考慮して計算する必要があります。

# 参考文献

  * **[1]**: Vallado, D. A (2013). Fundamentals of Astrodynamics and Applications. Microcosm   Press, Hawthorn, CA, USA.
