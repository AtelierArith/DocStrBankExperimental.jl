```
r_tirs_to_mod_iau2006([T, ]jd_ut1::Number, jd_tt::Number, δΔϵ_2000::Number = 0, δΔΨ_2000::Number = 0) -> T
```

地球中間基準系 (TIRS) をユリウス日 `jd_ut1` [UT1] および `jd_tt` [地球時間] における平均日 (MOD) 基準系に整列させる回転を計算します。このアルゴリズムは IAU-2006 理論を使用しています。

通常、IERS EOP データから取得される傾斜の摂動 (`δΔϵ_2000`) および経度の摂動 (`δΔψ_2000`) [rad] の補正を提供することができます（[`fetch_iers_eop`](@ref) および [`compute_δΔϵ_δΔψ`](@ref) を参照）。これらの補正は、液体の地球コアの影響をモデル化する自由コア摂動 (FCN) に関連しています。

回転のタイプはオプションの変数 `T` によって説明されます。もしそれが `DCM` であれば、DCM が返されます。そうでなければ、`Quaternion` であれば、Quaternion が返されます。このパラメータが省略された場合、`DCM` にフォールバックします。

!!! info
    この合成回転 TIRS <=> ERS <=> MOD は、新しい関数として実装されています。なぜなら、単一の回転 TIRS <=> ERS および ERS <=> MOD は、計算負荷が高い `nutation_eo` 関数を呼び出すからです。この場合、合成アルゴリズムはそれらの関数を個別に呼び出すよりも約 2 倍速くなります。
