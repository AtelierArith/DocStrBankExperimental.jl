```
hamilton_smoother(y, T, R, C, Q, Z, D, E, s_0, P_0; Nt0 = 0)

hamilton_smoother(regime_indices, y, Ts, Rs, Cs, Qs,
    Zs, Ds, Es, s_0, P_0; Nt0 = 0)
```

これは、James Hamiltonの「時系列分析」に基づくカルマン平滑化プログラムです。クープマン平滑化器とは異なり、これはモア・ペンローズ擬似逆行列を使用して潜在的に特異な行列を反転することに依存しています。

状態空間はショックで拡張され（`?augment_states_with_shocks`を参照）、拡張された状態空間はカルマンフィルタリングおよび平滑化されます。最後に、平滑化された状態とショックは拡張された状態ベクトルからインデックス化されます。

ショックで拡張する前の元の状態空間は次のように与えられます：

```
s_{t+1} = C + T*s_t + R*ϵ_t    (遷移方程式)
y_t     = D + Z*s_t + u_t      (測定方程式)

ϵ_t ∼ N(0, Q)
u_t ∼ N(0, E)
Cov(ϵ_t, u_t) = 0
```

### 入力

  * `y`: データ `y_1, ... , y_T` を含む `Ny` x `Nt` 行列
  * `s_0`: `Ns` x 1 初期状態ベクトル
  * `P_0`: `Ns` x `Ns` 初期状態共分散行列

**メソッド1のみ:** 状態空間システム行列 `T`, `R`, `C`, `Q`, `Z`, `D`, `E`。`?kalman_filter`を参照

**メソッド2のみ:** 各レジームの `regime_indices` とシステム行列 `Ts`, `Rs`, `Cs`, `Qs`, `Zs`, `Ds`, `Es`。`?kalman_filter`を参照

ここで：

  * `Nt`: データがある期間の数
  * `Ns`: 状態の数
  * `Ne`: ショックの数
  * `Ny`: 観測可能な数

### キーワード引数

  * `Nt0`: 0より大きい場合、返される平滑化された状態とショックの行列は、その数の列（先頭から取られる）だけ短くなります。

### 出力

  * `s_smth`: 平滑化された状態 `s_{t|T}` の `Ns` x `Nt` 行列
  * `ϵ_smth`: 平滑化されたショック `ϵ_{t|T}` の `Ne` x `Nt` 行列
