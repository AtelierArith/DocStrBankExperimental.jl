```
durbin_koopman_smoother(y, T, R, C, Q, Z, D, E, s_0, P_0;
    Nt0 = 0, draw_states = true)

durbin_koopman_smoother(regime_indices, y, Ts, Rs, Cs, Qs,
    Zs, Ds, Es, s_0, P_0; Nt0 = 0, draw_states = true)
```

このプログラムは、DurbinとKoopmanの「A Simple and Efficient Simulation Smoother for State Space Time Series Analysis」（Biometrika, 2002）に基づくシミュレーションスムーザーです。

他のシミュレーションスムーザー（例えば、CarterとKohnの1994年のもの）とは異なり、この方法は各期間ごとに別々のサンプルを必要とせず、状態ベクトルのサンプルや条件付き分布からのサンプルも必要としません。代わりに、ショックのベクトルはショックの無条件分布から引き出され、その後（カルマン平滑化ステップを介して）データに条件付けられたショックのサンプルを得るために修正されます。これを使用して、データに条件付けられた状態のサンプルを生成します。このように状態を描画することは、状態ベクトルの複数のサンプルを必要とせず（特異値分解を必要とし）、状態共分散行列を逆転させる必要もないため、他の方法よりもはるかに効率的です（計算集約的で比較的不安定なムーア・ペンローズ擬似逆行列を使用する必要があります）。

状態空間は次のように与えられます：

```
s_{t+1} = C + T*s_t + R*ϵ_t    (遷移方程式)
y_t     = D + Z*s_t + u_t      (測定方程式)

ϵ_t ∼ N(0, Q)
u_t ∼ N(0, E)
Cov(ϵ_t, u_t) = 0
```

### 入力

  * `y`: データ `y_1, ... , y_T` を含む `Ny` x `Nt` 行列
  * `s_0`: `Ns` x 1 初期状態ベクトル
  * `P_0`: `Ns` x `Ns` 初期状態共分散行列
  * `pred`: カルマンフィルターからの1ステップ予測状態ベクトル `s_{t|t-1}` の `Ns` x `Nt` 行列
  * `vpred`: 予測状態ベクトルの平均二乗誤差 `P_{t|t-1}` の `Ns` x `Ns` x `Nt` 配列

**メソッド1のみ:** 状態空間システム行列 `T`, `R`, `C`, `Q`, `Z`, `D`, `E`。`?kalman_filter` を参照してください。

**メソッド2のみ:** 各レジームの `regime_indices` とシステム行列 `Ts`, `Rs`, `Cs`, `Qs`, `Zs`, `Ds`, `Es`。`?kalman_filter` を参照してください。

ここで：

  * `Nt`: データがある期間の数
  * `Ns`: 状態の数
  * `Ne`: ショックの数
  * `Ny`: 観測可能な数

### キーワード引数

  * `Nt0`: 0より大きい場合、返される平滑化された状態とショックの行列は、その数の列（先頭から取られる）だけ短くなります。
  * `draw_states`: 平滑化された状態を分布 `N(s_{t|T}, P_{t|T})` から引き出すか、平均 `s_{t|T}` を使用するかを示します（クープマンのスムーザーに減少します）。

### 出力

  * `s_smth`: 平滑化された状態 `s_{t|T}` の `Ns` x `Nt` 行列
  * `ϵ_smth`: 平滑化されたショック `ϵ_{t|T}` の `Ne` x `Nt` 行列
