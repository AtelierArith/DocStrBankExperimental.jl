```
init_stationary_states(T, R, C, Q)
```

定常条件の下で初期状態 `s_0` と状態共分散行列 `P_0` を計算します：

```
s_0  =  (I - T) \ C
P_0 = reshape(I - kron(T, T)) \ vec(R*Q*R'), Ns, Ns)
```

ここで：

  * `kron(T, T)` は次元 `Ns^2` x `Ns^2` の行列で、`T` のクロネッカー積です
  * `vec(R*Q*R')` は `R*Q*R'` の `Ns` 列をスタックして構成された `Ns^2` x 1 の列ベクトルです

状態空間モデルが定常であるとき、`T` のすべての固有値は単位円の内側にあります。前述の式が適用できない場合、初期状態ベクトルの推定値は `C` に設定され、その共分散行列は `1e6 * I` で与えられます。
