```
pslyapd2(A, C, E; stability_check = false) -> (X, Y)
```

周期的な離散時間リャプノフ方程式のペアを解きます。

平方の `n` 次の周期行列 `A(i)`, `i = 1, ..., pa`,  `C(i)`, `i = 1, ..., pc`, および `E(i)`, `i = 1, ..., pe` の周期 `pa`, `pc` および `pe` に対して、周期 `p = lcm(pa,pc,pe)` の周期的リャプノフ方程式の周期的解 `X(i)`, `i = 1, ..., p` および `Y(i)`, `i = 1, ..., p` が計算されます：

```
A(i)*X(i)*A(i)' + C(i) = X(i+1), i = 1, ..., p ,  
 
A(i)'*Y(i+1)*A(i) + E(i) = Y(i), i = 1, ..., p .
```

周期行列 `A`, `C` および `E` はそれぞれ `n×n×pa`, `n×n×pc` および `n×n×pe` の3次元配列 `A`, `C` および `E` に格納され、`X` および `Y` は `n×n×p` の3次元配列として結果が得られます。

また、周期行列 `A`, `C` および `E` はそれぞれ `pa`-, `pc`- および `pe`-次元の行列ベクトルに格納され、`X` および `Y` は `p`-次元の行列ベクトルとして結果が得られます。

`stability_check = true` の場合、`A` の特性乗数の安定性がチェックされ、特性乗数のいずれかが絶対値1以上である場合はエラーが発生します。

周期行列 `A` の周期的シュール形式に基づくバーテルス-スチュワート法の周期的離散アナログが使用されます [1]。

*参考文献:*

[1] A. Varga. Periodic Lyapunov equations: some applications and new algorithms.                Int. J. Control, vol, 67, pp, 69-87, 1997.
