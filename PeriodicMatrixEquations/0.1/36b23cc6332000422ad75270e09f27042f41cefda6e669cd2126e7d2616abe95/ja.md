```
prcplyap(A, C; K = 10, solver, reltol, abstol) -> U
```

解の上三角周期因子 `U(t)` を計算します。`X(t) = U(t)'U(t)` 

```
 .
-X(t) = A(t)'X(t) + X(t)A(t) + C(t)'C(t).
```

周期行列 `A` と `C` は同じ型で、同じ列の次元を持ち、同等の周期を持たなければなりません。結果として得られる周期因子 `U` の周期は `A` と `C` の最小公倍数の同等の周期に設定され、サブ周期の数はそれに応じて調整されます。

この関数は単に [`pcplyap`](@ref) へのインターフェースです（キーワードパラメータの説明についてはこの関数を参照してください）。
