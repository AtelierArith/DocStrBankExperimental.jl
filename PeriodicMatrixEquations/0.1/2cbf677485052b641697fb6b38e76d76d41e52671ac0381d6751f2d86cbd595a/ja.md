```
 prdric(A, B, R, Q[, S]; itmax = 0, nodeflate = false, fast, rtol) -> (X, EVALS, F)
```

周期リカッティ差分方程式を解きます。

```
  X(i) = Q(i) + A(i)'X(i+1)A(i) - (A(i)'X(i+1)B(i) + S(i))*
                                 -1
         (B(i)'X(i+1)B(i) + R(i))  (A(i)'X(i+1)B(i) + S(i))'
```

安定化周期状態フィードバックを計算します。

```
                                  -1
  F(i) = -(B(i)'X(i+1)B(i) + R(i))  (B(i)'X(i+1)A(i) + S(i)')
```

および、`A(i)-B(i)F(i)`の対応する安定閉ループ*コア*特性乗数を`EVALS`に格納します。

`n(i+1)×n(i)`および`n(i+1)×m`の周期行列`A(i)`と`B(i)`は、`PeriodicMatrix`オブジェクト`A`と`B`に含まれ、同じサンプリング時間を持つ必要があります。`R(i)`, `Q(i)`および`S(i)`は、`A`および`B`と同じサンプリング時間を持つ`m×m`、`n(i)×n(i)`および`n(i)×m`の周期行列であり、`R(i)`および`Q(i)`は対称です。`R(i)`, `Q(i)`および`S(i)`は、`PeriodicMatrix`オブジェクト`R`, `Q`および`S`に含まれています。`R`, `Q`および`S`は、定数実行列として代わりに提供することもできます。結果として得られる`n(i)×n(i)`の対称周期解`X(i)`と`m×n(i)`の周期状態フィードバックゲイン`F(i)`は、`A`, `B`, `R`および`Q`の最小公倍数の周期に設定され、サブ周期の数はそれに応じて調整されます。

`fast = true`の場合、[1]のペンシル削減に基づく高速構造を利用して、`X(j)`の周期生成器を決定し、全周期にわたって解を反復的に生成できるようにします。`j`の値は、`n(i)`の最小次元`nc`に対応します（これはコア特性乗数の数でもあります）。`fast = false`（デフォルト）の場合、周期シュル分解に基づくアプローチ[1]が適用され、周期行列のシンプレクティックペアに適用されます。`nodeflate = false`（デフォルト）の場合、基礎となる周期ペンシルは、問題構造から生じる無限特性乗数を排除（縮小）するために前処理されます。`nodeflate = true`の場合、事前の縮小は行われません。

計算された解の精度を反復的に改善するには、`itmax = k`を使用し、`k > 0`（デフォルト：`k = 0`）とします。

関与する行列の特異性を検出するために、キーワードパラメータ`rtol = tol`を使用して、1ノルムの逆条件数の下限を指定できます。`tol`のデフォルト値は`n*ϵ`であり、ここで`ϵ`は作業*マシンエプシロン*、`n`は`n(i)`の最大値です。
