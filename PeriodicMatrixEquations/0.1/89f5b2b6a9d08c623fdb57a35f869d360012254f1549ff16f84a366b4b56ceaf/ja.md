```
 pslyapdkr(A, C; adj = true) -> X
```

周期的離散時間リャプノフ行列方程式を解きます。

```
  A'σXA + C = X, もし adj = true の場合、
```

または 

```
  A*X*A' + C =  σX, もし adj = false の場合、
```

ここで `σ` は前方シフト演算子 `σX(i) = X(i+1)` です。周期行列 `A` は単位円上に特性乗数を持ってはいけません。周期行列 `A` と `C` は、3次元配列として保存されるか、行列のベクトルとして保存されます。

方程式のクロンカー積展開が使用されるため、この関数は大きな次数の行列や大きな周期には推奨されません。
