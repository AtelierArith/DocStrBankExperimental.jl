```
pfclyap(A, C; K = 10, N = 1, solver, reltol, abstol, intpol) -> X
```

周期的前進時間リャプノフ微分方程式を解きます。

```
.
X(t) = A(t)X(t) + X(t)A(t)' + C(t) .
```

周期行列 `A` と `C` は同じ型、同じ次元、かつ整合性のある周期を持つ必要があり、さらに `C` は対称でなければなりません。得られる対称周期解 `X` の周期は `A` と `C` の最小公倍数の整合性のある周期に設定され、サブ周期の数はそれに応じて調整されます。

この関数は単に [`pclyap`](@ref) へのインターフェースです（キーワードパラメータの説明についてはこの関数を参照してください）。
