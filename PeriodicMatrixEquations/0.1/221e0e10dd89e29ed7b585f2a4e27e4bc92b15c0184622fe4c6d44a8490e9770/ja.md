```
pfdplyap(A, B) -> U
```

前進時間周期離散時間リューボビッチ方程式の解 `X = U*U'` の上三角因子 `U` を計算します。

```
AXA' + BB' = σX
```

ここで `σ` は前進シフト演算子 `σX(i) = X(i+1)` です。周期行列 `A` は安定でなければならず、すなわちすべての特性乗数の絶対値が1未満である必要があります。

周期行列 `A` と `B` は同じタイプ、同じ次元、および整合性のある周期を持っている必要があります。結果として得られる上三角周期行列 `U` の周期は `A` と `B` の最小公倍数の整合性のある周期に設定され、サブ周期の数はそれに応じて調整されます。

注: `X` は周期ペア `(A,B)` の*到達可能性グラミアン*です。
