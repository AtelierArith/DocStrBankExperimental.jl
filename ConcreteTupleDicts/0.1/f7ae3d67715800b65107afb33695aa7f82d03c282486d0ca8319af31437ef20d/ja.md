```
TupleDict(td)
TupleDict(d1, d2...)
```

イテラブルな `td` の `Dict` から `TupleDict` を構築するか、`Dict` のシーケンス `d1, d2, d3...` から構築します。

同じ `valtype` を共有する `Dict` は、キーが同じ `Dict` に格納されるようにマージされます。異なる `valtype` の `Dict` は、`getkey` の型推論を可能にするために、別々の `Dict` として残されます。

すべての入力 `Dict` は、以下の制約を満たす必要があります：

```
1. すべての `Dict` は具体的な `keytype` を持つ必要があるか、具体的な型の `Union{...}` である `keytype` を持つ必要があります
2. `keytype` が重複する `Dict` を渡すことはできません
```

最初の要件は、抽象型の `Dict` が正しいキーを特定する際にあいまいさを避けるのに役立ちます（例：`Number` のキーを持つ `Dict` と `Float64` のキーを持つ `Dict` の2つ）。

2番目の要件は、特定の型のすべてのキーが1つの `valtype` のみにマッピングされることを保証/要求します。
