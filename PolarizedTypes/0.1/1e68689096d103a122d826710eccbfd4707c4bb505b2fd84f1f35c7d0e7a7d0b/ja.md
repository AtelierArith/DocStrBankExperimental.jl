```julia
polellipse(s)

```

ストークスパラメータ `s` の偏光楕円を返します。結果は、以下の要素を持つ名前付きタプルです。

  * `a`   : 偏光楕円の長半径
  * `b`   : 偏光楕円の短半径
  * `evpa`: `s` の電気ベクトル位置角または楕円のPA。
  * `sn`  : ストークス `V` の符号。

## 注意事項

長半径と短半径は次のように定義されます。

```
    a = 1/2(Iₚ + |L|)
    b = 1/2(Iₚ - |L|)
```

ここで `Iₚ = √(Q² + U² + V²)` および `|L| = √(Q² + U²)` です。

一般に、楕円の面積は次のように与えられます。

```
    πab = π/4|V|²
```

線形偏光がゼロのソースでは `a = b` となり、半径 `|V|` の円になります。純粋な線形偏光の場合は `b = 0` となり、長さ `|L|` の直線になります。
