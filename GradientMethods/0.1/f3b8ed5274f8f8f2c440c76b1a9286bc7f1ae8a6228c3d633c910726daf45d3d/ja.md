```
cg(A, b, [maxiter=size(A,1)])
```

最小二乗問題 `min |Ax - b|^2` を共役勾配法を使用して解決します。

初期値を使用する必要がある場合は、`cg!` を参照してください。

`A` は対称ではないことに注意してください。

## 例

```julia-repl
julia> cg([1 2; 3 4], [1, 2])
2-element Array{Float64,1}:
5.551115123125783e-17
0.5000000000000012
```

他にも [`cg!`](@ref) や [`@cg`](@ref) を参照してください。
