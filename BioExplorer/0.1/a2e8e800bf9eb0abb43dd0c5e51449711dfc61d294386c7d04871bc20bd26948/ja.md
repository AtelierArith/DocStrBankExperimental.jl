```
generate_incidence_communities(nb_species::Int64, nb_sites::Int64, probability::Number = Float64(50))
```

バイノミアル分布を使用して発生データを持つ偽のコミュニティ行列を生成します。

# 引数

  * `nb_species::Int64`: 各コミュニティ内の種の数。
  * `nb_sites::Int64`: サイト（コミュニティ）の数。
  * `probability::Number`: 発生データを生成するために使用されるバイノミアル分布の確率パラメータ。デフォルトは50です。

# 戻り値

  * ランダムに生成された発生データを含むコミュニティ行列。

# 詳細

この関数は、バイノミアル分布を使用して種の存在/不在をランダムにサンプリングすることによって、発生データを持つ偽のコミュニティ行列を生成します。行列の各行はサイト（コミュニティ）を表し、各列は種を表します。発生データは最も近い整数（0または1）に丸められます。バイノミアル分布の確率パラメータのデフォルト値はp = 0.5に設定されています。

[`generate_abundance_communities`](@ref)も参照してください。
