```
generate_abundance_communities(nb_species::Int64, nb_sites::Int64, poisson_lambda::Number = Float64(50))
```

ポアソン分布を使用して、豊度データを持つ偽のコミュニティ行列を生成します。

# 引数

  * `nb_species::Int64`: 各コミュニティ内の種の数。
  * `nb_sites::Int64`: サイト（コミュニティ）の数。
  * `poisson_lambda::Number`: 豊度データを生成するために使用されるポアソン分布のラムダパラメータ。デフォルトは50です。

# 戻り値

  * ランダムに生成された豊度データを含むコミュニティ行列。

# 詳細

この関数は、ポアソン分布から種の豊度をランダムにサンプリングすることによって、豊度データを持つ偽のコミュニティ行列を生成します。行列の各行はサイト（コミュニティ）を表し、各列は種を表します。豊度データは最も近い整数に丸められます。ポアソン分布のラムダパラメータのデフォルト値は50に設定されています。

関連情報は[`generate_incidence_communities`](@ref)を参照してください。
