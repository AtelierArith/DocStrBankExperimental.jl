```
SAC(community_matrix::Community_Matrix, npermut::Int64 = 0)
```

コミュニティマトリックスの種の蓄積曲線を計算します。

# 引数

  * `community_matrix::Community_Matrix`: 入力コミュニティマトリックス。
  * `npermut::Int64`: 平均蓄積曲線の周りの信頼区間を計算するための置換の数。デフォルトは0で、置換を行わないことを示します。

# 戻り値

  * `npermut`が0の場合、種の蓄積曲線を表すベクトルを返します。
  * `npermut`が0より大きい場合、平均蓄積曲線と信頼区間を含むDataFrameを返します。
  * 種の蓄積曲線を表すMakie Figure。

# 詳細

この関数は、入力コミュニティマトリックスの種の蓄積曲線を計算します。 `npermut`が0より大きい場合、コミュニティマトリックスの置換を行い、平均蓄積曲線の周りの信頼区間を推定するために複数の蓄積曲線を計算します。この関数は、蓄積曲線のプロットを生成し、それを表示します。
