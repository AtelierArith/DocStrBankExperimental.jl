```
mat_com_convert(community_matrix::Community_Matrix)
```

豊富度データを持つコミュニティマトリックスを発生データを持つコミュニティマトリックスに変換します。

# 引数

  * `community_matrix::Community_Matrix`: 入力の豊富度データを持つコミュニティマトリックス。

# 戻り値

  * 発生データを持つ対応するコミュニティマトリックス。

# 詳細

この関数は、豊富度データを持つコミュニティマトリックスを発生データを持つコミュニティマトリックスに変換します。豊富度が0より大きい値を1（存在を示す）に置き換え、豊富度が0の値はそのまま（不在を示す）にします。結果として得られるマトリックスは、入力マトリックスと同じ構造を持ちますが、発生データを持っています。
