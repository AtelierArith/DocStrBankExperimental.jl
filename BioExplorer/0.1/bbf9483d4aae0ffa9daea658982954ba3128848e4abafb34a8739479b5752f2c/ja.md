```
biodiversity_surface(community_matrix::Community_Matrix)
```

コミュニティマトリックスのための生物多様性サーフェスプロットを作成します。

# 引数

  * `community_matrix::Community_Matrix`: 複数のコミュニティの種の豊富さデータを含むコミュニティマトリックス。

# 戻り値

  * 生物多様性サーフェスプロットを表すMakie Figure。

# 詳細

この関数は、Makieを使用してコミュニティ間の生物多様性を視覚化する3Dサーフェスプロットを生成します。 `BioExplorer.hill`関数を使用して各コミュニティのヒル系列を計算し、ヒル数の順序に対する同等の種数をプロットします。 x軸は異なるコミュニティを、y軸はヒル数の順序を、z軸は同等の種数を表します。 サーフェス上の異なる色は、コミュニティ間の生物多様性の変動を示します。
