```
FD_rich(trait_matrix::Trait_Matrix, weight, display_graph::Bool)
```

Gower距離とPCAを使用して、特性マトリックスの機能的豊かさを計算します。

# 引数

  * `trait_matrix::Trait_Matrix`: 異なる種の特性値を含む特性マトリックス。
  * `weight`: 特性マトリックス内の各特性に対する重みパラメータのベクトル。欠損する場合があり、その場合は各特性が等しいと見なされます。
  * `display_graph::Bool`: グラフを表示するかどうかを示すブールフラグ。

# 戻り値

  * `rich`: 機能的豊かさの値。

# 詳細

この関数は、特性マトリックスからGower距離行列を計算し、それに対して主成分分析（PCA）を実行します。次に、データを最初の2つの主成分に投影し、投影された点の凸包を計算します。`display_graph`がtrueに設定されている場合、凸包が強調表示された投影点の散布図を生成します。最後に、凸包の面積に対するガウスの靴ひも定理を使用して機能的豊かさを計算します。
