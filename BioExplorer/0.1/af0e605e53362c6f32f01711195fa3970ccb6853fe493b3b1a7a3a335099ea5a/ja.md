```
mutable struct Community_Matrix
```

`Community_Matrix`は、BioExplorer.jlパッケージ内の分類学的多様性（TD）分析に関連するデータを表すために設計された可変コンポジットオブジェクトです。これは、4つの重要なコンポーネントで構成されています。

# フィールド

  * `sites::Vector{String}`: 種に関するデータが収集される生態学的サイトまたは場所のユニークな識別子または名前を表す文字列のベクター。
  * `species::Vector{String}`: 生態学的サイト内で観察または記録された種のユニークな識別子または名前を示す文字列のベクター。
  * `species_data::Array{Float32, 2}`: 種の出現または豊富さに関する実際のデータを含む単精度浮動小数点数（`Float32`）の配列。この配列は、行が異なる生態学的サイトに対応し、列が異なる種に対応するように構成されています。
  * `type::String`: オブジェクト内に格納されているデータのタイプ（'occurrences'または'abundances'のいずれか）を示す文字列。
