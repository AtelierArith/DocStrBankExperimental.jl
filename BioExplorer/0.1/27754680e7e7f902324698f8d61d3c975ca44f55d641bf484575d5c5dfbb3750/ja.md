```
matrix_Gowdis(trait_matrix::Trait_Matrix, weight)
```

トレイトマトリックス内のすべての種のペア間のGower距離を計算します。

# 引数

  * `trait_matrix::Trait_Matrix`: 種のトレイトデータを含むトレイトマトリックス。
  * `weight`: トレイトマトリックス内の各トレイトに対する重みパラメータのベクトル。欠損している場合は、各トレイトが等しいと見なされます。

# 戻り値

  * すべての種のペア間のGower距離を表す行列。

# 詳細

この関数は、与えられたトレイトマトリックス内のトレイトデータに基づいて、すべての種のペア間のGower距離を計算します。すべての種の組み合わせを反復処理し、`pairwise_Gowdis`関数を使用してペアワイズGower距離を計算します。結果として得られる行列は、すべての種のペア間のGower距離を表します。
