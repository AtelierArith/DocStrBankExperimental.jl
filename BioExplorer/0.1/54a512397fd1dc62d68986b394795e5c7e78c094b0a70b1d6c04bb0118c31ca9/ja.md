```
FD_obs_metrics(trait_matrix::Trait_Matrix, weight)
```

機能的多様性の観察に基づく指標を計算します：独自性、ユニーク性、貢献度。

# 引数

  * `trait_matrix::Trait_Matrix`: 異なる種の特性値を含む特性行列。
  * `weight`: 特性行列内の各特性に対する重みパラメータのベクトル。欠損する場合もあり、その場合は各特性が等しいと見なされます。

# 戻り値

  * `Obs_metrics`: 各種の観察に基づく指標を含むDataFrame。

# 詳細

この関数は、機能的多様性の観察に基づく3つの指標を計算します：

1. 種の貢献度：全体の機能的豊かさに対する各種の貢献度。
2. 種の独自性：特性空間内の他のすべての種との平均距離。
3. 種のユニーク性：特性空間内の他の任意の種との最小距離。

各種の貢献度は、1つの種を逐次的に除去し、機能的豊かさを再計算し、総豊かさとの差を計算することによって算出されます。独自性は各種と他のすべての種との平均距離として計算され、ユニーク性は各種と他の任意の種との最小距離として計算されます。
