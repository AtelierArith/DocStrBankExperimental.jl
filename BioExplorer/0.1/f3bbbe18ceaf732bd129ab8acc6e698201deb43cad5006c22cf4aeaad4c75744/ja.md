```
jaccard_dissim(community_matrix::Community_Matrix)
```

2つのコミュニティ間のジャッカード非類似度を計算します。

# 引数

  * `community_matrix::Community_Matrix`: 2つのコミュニティの種データを含むコミュニティマトリックス。コミュニティが2つ以上存在する場合、最初の2つのみが考慮されます。

# 戻り値

  * `jaccard_dissim_value::Float64`: 2つのコミュニティ間のジャッカード非類似度値。

# 詳細

ジャッカード非類似度は、2つの集合間の非類似度を、交差点と和集合を比較することによって測定します。バイナリの種の存在-不在データで表される2つのコミュニティの場合、これは、いずれかのコミュニティに見られる種の総数で割った、1つのコミュニティにのみ見られる種の数として計算されます。
