```
FD_dispersion(trait_matrix::Trait_Matrix, weight)
```

Gower距離とPCAを使用して、特性行列の機能的分散を計算します。

# 引数

  * `trait_matrix::Trait_Matrix`: 異なる種の特性値を含む特性行列。
  * `weight`: 特性行列内の各特性に対する重みパラメータのベクトル。欠損している場合は、各特性が等しいと見なされます。

# 戻り値

  * `dispersion`: 機能的分散値。

# 詳細

この関数は、特性行列からGower距離行列を計算し、それに対して主成分分析（PCA）を実行します。次に、データを最初の2つの主成分に投影し、投影された点の凸包を計算します。次に、凸包の重心を計算し、重心から凸包上の各点までの距離を計算します。機能的分散は、重心から凸包上のすべての点の平均距離として計算されます。
