保守変数係数ポアソン方程式のための複合型:

```
∮ds β ∂x/∂n = σ
```

得られる線形システムは

```
Ax = [L+D+L']x = z
```

ここで、Aは対称でブロックトリディアゴナルかつ非常にスパースです。さらに、 `D[I]=-∑ᵢ(L[I,i]+L'[I,i])` です。これは、行列のストレージ、乗算などが外部ライブラリなしで簡単に実装および最適化できることを意味します。

上記のシステムを反復的に解決するのを助けるために、ポアソン構造は `inv(D)`、誤差 `ϵ`、および残差 `r=z-Ax` のためのヘルパー配列を保持します。反復解法は次に誤差 `ϵ=̃A⁻¹r` を推定し、`x+=ϵ`、`r-=Aϵ` を増加させます。
