# 2Dパルスシーケンスの反転:

```
invert(seq, x_direct, x_indirect, Data; lims1, lims2, alpha, solver, normalize)
```

この関数はカーネルを構築し、選択したアルゴリズムを使用して反転を実行します。出力は `inv_out_2D` 構造体です。

必要な（位置的）引数:

  * `seq` は2Dパルスシーケンス（例: IRCPMG）です。
  * `x_direct` は直接次元の取得パラメータ（例: CPMGエコーを取得する時刻）です。
  * `x_indirect` は間接次元の取得パラメータ（例: IRシーケンスのすべての遅延時間 τ）です。
  * `Data` は複素データの2Dデータ行列です。

オプションの（キーワード）引数:

  * `lims1` は直接次元の反転の出力「範囲」を決定します（例: IRCPMGのT₂時間）。
  * `lims2` は間接次元の反転の出力「範囲」を決定します（例: IRCPMGのT₁時間）。

上記の両方の場合で、範囲の制限を指定するタプルまたは値のベクトルを使用できます。これは1D反転の `lims` 引数と同様です。

  * `alpha` は平滑化項を決定します。固定のアルファには実数を使用します。選択しない場合は、デフォルトの方法である `gcv` を通じて自動的にアルファが決定されます。
  * `solver` は反転計算に使用されるアルゴリズムです。デフォルトは `brd` です。
  * `normalize` は `Data` を `Data` の最大値で1に正規化します。デフォルトは `true` です。
