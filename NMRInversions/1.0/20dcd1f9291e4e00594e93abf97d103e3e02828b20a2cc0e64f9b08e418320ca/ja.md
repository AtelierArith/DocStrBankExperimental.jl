expfit関数の出力。多重指数フィットに関する情報を含む構造体。

フィールドは次のとおりです：

  * `seq`: パルスシーケンス
  * `x` : x取得値（例：緩和の時間または拡散のb因子）。
  * `y` : y取得値。
  * `xfit` : フィッティングされたx値の配列（プロット目的）。
  * `yfit` : フィッティングされたy値の配列（プロット目的）。
  * `u` : `mexp`関数のフィッティングパラメータ。
  * `u0` : `mexp`関数の初期パラメータ。
  * `r` : 残差。
  * `eq` : フィッティングされた関数の方程式。
  * `eqn` : 初期関数の方程式。
  * `title` : データを説明するタイトル。
