# 1Dパルスシーケンスの反転:

```
invert(seq, x, y ; lims, alpha, solver, normalize)
```

この関数はカーネルを構築し、それを使用して選択したアルゴリズムを用いて反転を実行します。出力は`inv_out_1D`構造体です。

必要な（位置的）引数:

  * `seq`は1Dパルスシーケンス（例: IR, CPMG, PFG）です。
  * `x`は実験のx軸（時間またはb因子など）です。
  * `y`は実験のy軸（NMR信号の強度）です。

オプションの（キーワード）引数:

  * `lims=(a,b,c)`は出力Xの「制限」を設定します。

これにより、10^aから始まり、10^bで終わり、cの対数間隔の値を持つようになります（デフォルトはリラクゼーションの場合は(-5, 1, 128)、拡散の場合は(-10, -7, 128)です）。代わりに、より自由度が必要な場合は、値のベクトルを直接使用できます（例: `lims=exp10.(range(-4, 1, 128))`）。

  * `alpha`はスムージング項を決定します。固定されたalphaのために実数を使用してください。

選択がない場合、デフォルトの方法である`gcv()`を通じてalphaが自動的に決定されます。

  * `solver`は反転計算に使用されるアルゴリズムです。デフォルトは`brd`です。
  * `normalize`は`y`を`y`の最大値で1に正規化します。デフォルトは`true`です。
