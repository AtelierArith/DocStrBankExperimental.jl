```
pdhgm(σ, τ)
```

L1正則化のためのプライマルデュアルハイブリッド勾配法、[この論文](https://doi.org/10.1016/j.jmr.2017.05.010) [Reci2017](@cite) に従っています。これは、逆関数の「ソルバー」として使用できます。

σとτの特定の選択はヒューリスティックです。σを小さくすると、アルゴリズムの収束速度が低下する一方で安定性が向上します。σ = 0.1およびτ = 10のときに、両者の良い妥協点が見つかりました。σとτの最適な値は、信号のスケーリングにわずかに依存します。したがって、NMR信号を最大1に正規化することが最善であり、これは引用された研究で採用された技術です。

この方法では、αの役割が逆転していることに注意してください。大きなα値は、より少ないスムージングをもたらします。そのため、`alpha=gcv()`（ミッチェル法）は機能しません。代わりに、`alpha=gcv(starting_value)`または`alpha=gcv(lower*limit, upper*limit)`、または他のlcurveメソッドを使用してください。
