```
lcurve(lower, upper ; kwargs...)
```

lcurve曲率単変数最適化を通じて最適なアルファ値を見つけるためのコンストラクタで、いくつかの下限と上限が与えられます。

必要な（位置的）引数：

  * `lower` は下限であり、アルゴリズムが考慮する最低のアルファ値です。
  * `upper` は上限であり、アルゴリズムが考慮する最高のアルファ値です。

オプションの（キーワード）引数：

  * `algorithm` は、Optim.jlが問題を解決するために使用するメソッドを決定します。

デフォルトは `Brent()` で、唯一の代替は通常遅い `GoldenSection()` です。

  * `abs_tol` は、アルゴリズムが気にするべきアルファの最小の変化を決定します。

デフォルトは 1e-3 です。
