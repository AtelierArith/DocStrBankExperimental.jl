```
gcv(lower, upper ; kwargs...)
```

最適なアルファ値をgcvスコアの単変量最適化を通じて見つけるためのコンストラクタで、いくつかの下限と上限を考慮します。

必要な（位置的）引数：

  * `lower` は下限であり、アルゴリズムが考慮する最も低いアルファ値です。
  * `upper` は上限であり、アルゴリズムが考慮する最も高いアルファ値です。

オプションの（キーワード）引数：

  * `algorithm` は、Optim.jlが問題を解決するために使用するメソッドを決定します。

デフォルトは `Brent()` で、唯一の代替は通常遅い `GoldenSection()` です。

  * `abs_tol` は、アルゴリズムが気にするべきアルファの最小の変化を決定します。

デフォルトは 1e-3 です。
