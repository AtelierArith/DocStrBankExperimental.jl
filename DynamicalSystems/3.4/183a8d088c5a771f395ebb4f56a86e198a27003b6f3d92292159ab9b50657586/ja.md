```
interactive_cobweb(ds::DiscreteDynamicalSystem, prange, O::Int = 3; kwargs...)
```

1D離散動的システムのコブウェブ図を探索するためのインタラクティブなアプリケーションを起動します。2つのスライダーがプロットされた軌道の長さと現在のパラメータ値を制御します。パラメータ値は、指定された`prange`から取得されます。

コブウェブプロットでは、動的ルール`f`の高次反復もプロットされ、順序1から指定された順序`O`まで表示されます。コブウェブ内の軌道や任意の反復`f`は、いくつかのボタンを使用してオフにすることができます。

## キーワード

  * `fkwargs = [(linewidth = 4.0, color = randomcolor()) for i in 1:O]`: プロットされた各反復`f`のためのプロットキーワード
  * `trajcolor = :black`: 軌道の色
  * `pname = "p"`: パラメータスライダーの名前
  * `pindex = 1`: パラメータインデックス
  * `xmin = 0, xmax = 1`: 動的システムが取ることができる状態の制限
  * `Tmax = 1000`: 最大軌道長
  * `x0s = range(xmin, xmax; length = 101)`: x0スライダーの可能な値。
