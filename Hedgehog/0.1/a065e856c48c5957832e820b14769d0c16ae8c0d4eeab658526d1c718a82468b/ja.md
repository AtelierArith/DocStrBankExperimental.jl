コックス・ロス・ルービンスタインの二項木価格設定方法。

この構造体は、オプション価格設定のためのコックス・ロス・ルービンスタイン（CRR）二項価格モデルを表します。

# フィールド

  * `steps`: 二項木の時間ステップの数。

# 注意事項

  * この実装は、**フォワード価格**または**スポット価格**のいずれかに基づくオプションをサポートしています。
  * フォワード価格に基づくオプションを価格設定する場合、ディスカウントはすでにフォワードに組み込まれています。
  * スポット価格に基づくオプションを価格設定する場合、各ステップで明示的にディスカウントを適用する必要があります。
  * 上昇確率は次のように定義されます：

```
p = 1 / (1 + u)
```

ここで：

  * `u = exp(σ√ΔT)` は上昇要因、
  * `ΔT` は年単位の時間ステップの期間です。
