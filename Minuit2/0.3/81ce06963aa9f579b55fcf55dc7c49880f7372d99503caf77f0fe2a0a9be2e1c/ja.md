```
scan!(m::Minuit, maxfcn = 0, strategy=1)
```

最小値を見つけるためにスキャンアルゴリズムを実行します。これは、初期値の周りのハイパーキューブ内で関数をスキャンするブルートフォースアルゴリズムです。値は制限内でなければなりません。

## 引数

  * `m::Minuit` : 最小化するMinuitオブジェクト。
  * `maxfcn::Int=0` : 最大関数呼び出し回数。0に設定すると、Minuitはデフォルト値を使用します。
  * `strategy::Int=1` : 最小化戦略。デフォルト値は1で、ほとんどのケースで推奨される値です。値0は速いですが、信頼性が低くなります。値2は遅いですが、より信頼性があります。値3以上はさらに遅くなりますが、さらに信頼性が高くなります。
