```
Minuit(fcn, x0...; grad=nothing, error=(), errordef=1.0, names=(), limits=(), fixed=(), method=:migrad, maxfcn=0, 
            tolerance=0.1, precision=nothing, strategy=1, kwargs...)
```

Minuitオブジェクトを初期化します。

これは最小化を開始したり、他の作業を行ったりするものではありません。アルゴリズムは、対応するメソッドを呼び出すことで開始されます。

## 引数

  * `fcn::Union{Function,CostFunction}` : 最小化する関数。受け入れられる関数の種類についての詳細はノートを参照してください。
  * `x0::AbstractArray` : 最小化のための開始値。開始値の設定方法についての詳細はノートを参照してください。
  * `grad::Union{Function,Nothing, Bool}` : `grad`が関数である場合、それは勾配を計算し、各パラメータに対する`fcn`の導関数を持つイテラブルオブジェクトを返す関数でなければなりません。`nothing`（デフォルト）の場合、Minuitは提供されたコスト関数の勾配を使用します。存在しない場合、Minuitは数値的に勾配を計算します。`grad`が`false`の場合、Minuitは勾配を使用しません。
  * `error::AbstractArray` : パラメータの誤差のための開始値。提供されない場合、Minuitはすべてのパラメータに対して0.1を使用します。
  * `errordef::Real` : 関数の誤差定義。Minuitは、関数の値を`errordef`だけ変化させるために必要なパラメータ値の変化としてパラメータの誤差を定義します。通常、カイ二乗フィットの場合は1であり、負の対数尤度の場合はその値は0.5です。ユーザーがカイ二乗フィットの2シグマ誤差を望む場合、それは4になります。`Chi2(x+n*sigma) = Chi2(x) + n*n`です。
  * `names::Vector{String}` : パラメータの名前。提供されない場合、Minuitは関数シグネチャから名前を抽出しようとします。
  * `limits::AbstractArray` : パラメータの制限。提供されない場合、Minuitはすべてのパラメータに制限を使用しません。
  * `fixed::AbstractArray` : パラメータの固定状態。提供されない場合、Minuitはすべてのパラメータに対して`false`を使用します。制限は2つの値のタプルであり、下限と上限です。パラメータが固定されている場合、制限は`(-Inf, Inf)`に設定されます。
  * `method::Symbol` : 使用する最小化アルゴリズム。可能な値は`:migrad`、`:simplex`です。
  * `maxfcn::Int` : 最大関数呼び出し回数。0に設定すると、Minuitはデフォルト値を使用します。
  * `tolerance::Real` : 最小化のための許容誤差。0に設定すると、Minuitはデフォルト値を使用します。
  * `kwargs` : 追加のキーワード引数。最小化のための開始値をキーワード引数として指定します。開始値の設定方法についての詳細はノートを参照してください。

## ノート

### 最小化する関数

デフォルトでは、Minuitは呼び出し可能な`fcn`がカイ二乗関数のように振る舞うと仮定します。つまり、同一のランダム実験を繰り返した場合の関数の最小値は、任意の加算定数までカイ二乗分布に従います。これは正しい誤差計算にとって重要です。`fcn`が対数尤度を返す場合、結果に-2を掛けて適応させる必要があります。関数が否定された対数尤度を返す場合、Minuitが誤差を正しく計算できるように`errordef`属性を設定することもできます。

Minuitは、`fcn`の関数シグネチャを読み取って、関数パラメータの数と名前を検出します。理解される関数シグネチャには2種類があります。

a. 位置引数を持つ関数。

関数は位置引数を持ち、各フィットパラメータに1つずつ対応します。例：

```
fcn(a, b, c) =  ...
```

パラメータa、b、cは実数を受け入れる必要があります。この場合、Minuitは自動的にパラメータ名を検出します。

b. 単一のAbstractArrayとして渡される引数を持つ関数。

関数は単一の引数を持ち、それはAbstractArrayです。例：

```
function fcn_v(x) =  ...
```

この形式を使用するには、開始値（`x0`）を`Tuple`または`Vector`の形式でMinuitに渡す必要があります。場合によっては、検出が失敗することがあり、その場合は`names`キーワードを使用してパラメータ名を設定する必要があります。

### パラメータの初期化

最小化のための初期値は、位置引数またはキーワードを介して設定できます。これは例を通じて最もよく説明されます：

```
fcn(x, y) =  (x - 2)^2 + (y - 3)^2
```

開始値を渡す以下の方法は同等です：

```
Minuit(fcn, x=1, y=2)
Minuit(fcn, y=2, x=1) # キーワードを使用する場合、順序は重要ではありません...
Minuit(fcn, [1,2])    # ... しかしここでは順序が重要です
```

関数にシグネチャがない場合、位置引数も使用できます：

```
fcn_no_sig(args...) =  ...
Minuit(fcn_no_sig, [1,2])
```

引数が`names`キーワードで明示的に名前付けされている場合、初期化にキーワードを使用できます：

```
Minuit(fcn_no_sig, x=1, y=2, names=("x", "y"))  # これも機能します
```

関数が単一のAbstractVectorを受け入れる場合、初期値は単一の配列のようなオブジェクトとして渡す必要があります：

```
fcn_v(x) = return (x[1] - 2) ** 2 + (x[2] - 3) ** 2
Minuit(fcn_v, (1, 2))
```
