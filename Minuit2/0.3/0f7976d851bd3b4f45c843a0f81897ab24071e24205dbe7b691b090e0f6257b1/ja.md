```
migrad!(m::Minuit, strategy=1; ncall=0)
```

Migrad最小化を実行します。

Minuit2ライブラリのMigradは、高エネルギー物理学での40年以上のほぼ独占的な使用により、その評判を得た堅牢な最小化アルゴリズムです。Migradの動作は[Minuit]()論文で説明されています。最小値付近で二次収束を達成するために、一次および近似二次導関数を使用します。

## パラメータ

  * `m::Minuit` : 最小化するMinuitオブジェクト。
  * `strategy::Int` : 最小化戦略。デフォルト値は1で、ほとんどのケースで推奨される値です。値0は速いですが、信頼性が低いです。値2は遅いですが、より信頼性があります。値3以上はさらに遅いですが、さらに信頼性があります。

## キーワードパラメータ

  * `ncall::Int=0` : 呼び出し回数の上限の近似値。0に設定すると、Minuit2ライブラリの適応ヒューリスティックを使用します。

次の2つのパラメータは、最初の反復後に収束に失敗した場合にMigrad手続きを再起動しようとする[`robust_low_level_fit`](@ref)の動作を制御します：

  * `iterate::Int=5` : 最小化を実行する反復回数。デフォルトは5です。
  * `use_simplex::Bool=true` : `true`の場合、最小値を見つけるために再試行で単体アルゴリズムを使用します。`false`の場合、migradアルゴリズムを使用します。
