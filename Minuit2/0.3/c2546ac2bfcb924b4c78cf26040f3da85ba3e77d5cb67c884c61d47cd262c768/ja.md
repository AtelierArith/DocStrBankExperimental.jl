```
nmcontour(x, y; cl=0.68, size=50, interpolated=0, ncall=0, iterate=5, use_simplex=true)
```

2D Minos信頼領域を取得します。

これは2つのパラメータをスキャンし、各スキャンポイントに対して他のすべての自由パラメータを最小化します。このスキャンは、信頼レベル`cl`に従って[プロファイル尤度法](https://en.wikipedia.org/wiki/Likelihood_function)に基づく統計的信頼領域を生成します。これは、[ウィルクスの定理](https://en.wikipedia.org/wiki/Wilks%27_theorem)に従って信頼領域のカバレッジ確率に漸近的に等しくなります。2D信頼領域の1D投影は、同じ信頼レベルに対して計算された1D Minos区間よりも大きいことに注意してください。これはエラーではなく、ウィルクスの定理の結果です。

計算は高コストであり、さまざまなポイントで数値最小化を実行する必要があります。

## 引数

  * `x` : 最初のパラメータの変数名。
  * `y` : 2番目のパラメータの変数名。
  * `cl::Real=0.68` : 等高線の信頼レベル。設定されていない場合、標準の68%等高線が計算されます（デフォルト）。0 < cl < 1の場合、その値は信頼レベル（確率）として解釈されます。便宜上、cl >= 1の値は、正規分布の標準偏差をカバーする中央対称区間の確率内容として解釈されます。
  * `size::Int=50` : 等高線上で見つけるポイントの数。これを増やすと等高線が滑らかになりますが、計算時間が長くなります。
  * `interpolated::Int=0` : 等高線上の補間ポイントの数。これをsizeより大きい値に設定すると、滑らかな曲線を生成するために三次スプライン補間が使用され、補間された座標が返されます。sizeより小さい値は無視されます。size=20、interpolated=200で良い結果が得られます。

## 戻り値

  * `contour::Vector(Tuple{Float64,Float64})` : [(x1, y1)...(xn, yn)]の形の等高線ポイント。
