```
hesse!(m::Minuit, ncall::Int=0)
```

Hesseアルゴリズムを実行して漸近誤差を計算します。

Hesse法は、[最小値での二次導関数の行列（Hesse行列）](https://en.wikipedia.org/wiki/Hessian_matrix)を逆行列にすることによって共分散行列を推定します。パラメータの相関を得るには、これを使用する必要があります。Minosアルゴリズムは、パラメータの不確実性を推定する別の方法です。関数`minos`を参照してください。

## 引数

  * `ncall::Int=0` : Hesseアルゴリズムによって行われる呼び出しの数の上限の近似値。0に設定すると、Minuit2ライブラリからの適応的ヒューリスティックを使用します。

## 注意事項

共分散行列は漸近的に（大きなサンプルで）有効です。有効であるとは、誤差から構築された信頼区間が、よく知られたカバレッジ確率（各区間の68％）で真の値を含むことを意味します。有限サンプルでは、コスト関数が最小値の周りで双曲放物線のように見える場合、これは真である可能性が高いです。

実際には、Minos法とHesse法の結果が一致する場合、誤差は正しいカバレッジを持つ可能性が非常に高いです。このルールが破られる人工関数を構築することは可能ですが、実際には常に機能するはずです。
