```
contour(m::Minuit, x, y; size=50, bound=2, grid=nothing, subtract_min=false)
```

最小値周辺の関数の2D等高線を取得します。

これは、他のすべてのパラメータを固定しながら、2つのパラメータに対して関数スキャンを行うことで等高線を計算します。関連する:meth:`mncontour`は異なる動作をします：スキャン内の各パラメータ値のペアに対して、他のすべてのパラメータに関して関数を最小化します。

このメソッドは、最小値付近の関数を検査して問題を検出するのに役立ちます（等高線は滑らかに見えるべきです）。関数が2つのパラメータのみを持つ場合を除き、これは信頼領域ではありません。信頼領域を計算するには:meth:`mncontour`を使用してください。

## 引数

  * `x` : スキャンのための最初のパラメータ（名前またはインデックス）。
  * `y` : スキャンのための2番目のパラメータ（名前またはインデックス）。
  * `size=50` : パラメータごとのスキャンポイントの数（デフォルト：50）。タプル`(nx,ny)`として、パラメータごとのスキャンポイントの数を指定できます。`grid`が設定されている場合は無視されます。
  * `bound=2` : ((v1min,v1max),(v2min,v2max))または最小値から対称的にスキャンする`sigma`の数。
  * `grid::Tuple{AbstractVector,AbstractVector}` : スキャンするグリッドポイント。`grid`が設定されている場合、`size`と`bound`は無視されます。
  * `subtract_min::Bool=false` : 戻り値から最小値を引きます。

## 戻り値

  * タプル(`xv`, `yv`, `zv`) : xおよびyの値を持つ1D配列と関数値を持つ2D配列のタプル。
