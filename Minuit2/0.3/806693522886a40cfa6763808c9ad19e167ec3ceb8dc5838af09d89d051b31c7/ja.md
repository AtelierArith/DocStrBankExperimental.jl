```
UnbinnedNLL(data::AbstractArray, pdf::Function; log=false, verbose=0, mask=nothing, grad=nothing, names=())
```

未ビン化負の対数尤度コスト関数。

## 引数

  * `data::AbstractArray` : 観測のサンプル。観測が多次元の場合、データは形状 (D, N) を持つ必要があり、ここで D は次元の数、N はデータポイントの数です。
  * `pdf::Function` : f(data, par0, [par1, ...]) の形の確率密度関数で、data はデータサンプル、par0, ... はモデルパラメータです。データが多変量の場合、f に渡されるデータは形状 (D,) を持ち、ここで D は次元の数、N はデータポイントの数です。
  * `verbose::Int` : 冗長性レベル。0: 出力なし（デフォルト）。1: 現在の引数と負の対数尤度値を印刷します。
  * `log::Bool=false` : 指数族の分布（正規分布、指数分布、ポアソン分布など）は、pdf の対数を直接計算することを可能にし、数値的に $log(pdf)$ を計算するよりも正確で効率的です。モデルが pdf の代わりに logpdf を返す場合は、これを `true` に設定します。
  * `mask::Union{Vector{Bool}, BitVector, Nothing}` : データのサブセットを選択するためのオプションのマスク。データと同じ長さである必要があります。
  * `grad::Union{Function, Nothing}` : pdf の勾配をオプションで渡します。pdf と同じ呼び出しシグネチャを持ちますが、形状 (K, N) の配列を返す必要があります。ここで N はデータポイントの数、K はパラメータの数です。`log` が True の場合、関数は pdf の代わりに logpdf の勾配を返す必要があります。勾配は Minuit によって収束を改善または加速し、パラメータ推定の分散のサンドイッチ推定量を計算するために使用できます。
  * `names` : モデルの各パラメータのオプションの名前（順番に）。モデルパラメータの数と同じ長さである必要があります。
