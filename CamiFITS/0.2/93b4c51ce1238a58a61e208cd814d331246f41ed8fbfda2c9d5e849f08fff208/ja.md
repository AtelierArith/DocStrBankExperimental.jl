```
fits_remove_zero_offset(data)
```

`BZERO` キーワードで指定された適切な整数オフセット値を `data` に *加算* することによって、`Int` 値の範囲を `UInt` 値の範囲にシフトします。

注意: FITS 形式は *ネイティブの符号なし整数データ型* をサポートしていないため（`UInt8` を除く）、`UInt16`、`UInt32`、および `UInt64` の符号なし値は、それぞれ `Int16`、`Int32`、および `Int64` の保存されたネイティブ符号付き整数から、(正の) `BZERO` キーワード値で指定された適切な整数オフセットを *加算* することによって復元されます。バイトデータ型 (`UInt8`) に関しては、保存されたネイティブ符号なし値 (`UInt`) から符号付きバイト値 (`Int8`) を復元するために、(負の) `BZERO` オフセット値を *加算* する逆の手法が使用できます。

このメソッドは、`Int8`、`UInt16`、`UInt32`、および `UInt64` のネイティブ値をサポートしていないソフトウェアとの後方互換性を確保するために、保存されたデータの *読み取り* に含まれ、使用されます。

#### 例:

```
julia> fits_remove_zero_offset(Int32[-2147483648])
1-element Vector{UInt32}:
 0x00000000

julia> Int(0x00000000)
0

julia> fits_remove_zero_offset(UInt8[128])
1-element Vector{Int8}:
 0
```
