```
coe2rv(coe, μ=1)
```

Convert classical orbital elements 'coe' to Cartesian position and velocity 'rv'.

Input classical orbital elements in order: [a, e, i, ω, Ω, ν].  See Algorithm 10 from Vallado, D. "Fundamentals of Astrodynamics and Applications."  4th Edition. 2013.
