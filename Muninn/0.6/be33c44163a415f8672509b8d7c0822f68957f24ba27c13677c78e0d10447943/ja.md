```
MB_timestep(model::Model, glacier::G, step::F, t::F) where {F <: AbstractFloat, G <: AbstractGlacier}
```

氷河の与えられた時間ステップに対する質量収支（MB）を計算します。

# キーワード引数

  * `model::Model`: 質量収支パラメータを含むモデル。
  * `glacier::G`: 気候およびDEMデータを含む氷河オブジェクト。
  * `step::F`: 時間ステップの期間。
  * `t::F`: 現在の時間。

# 戻り値

  * `MB::Matrix{F}`: 与えられた時間ステップに対して計算された質量収支行列。

# 詳細

1. 現在の時間`t`と前のステップ`t - step`の間の期間を計算します。
2. 指定された期間の累積気候データを取得します。
3. 氷河のDEMに基づいて気候データを2Dグリッドにダウンスケールします。
4. ダウンスケールされた気候データを使用して質量収支を計算します。
