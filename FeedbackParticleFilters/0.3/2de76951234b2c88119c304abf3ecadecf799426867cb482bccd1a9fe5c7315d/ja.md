```
LinearDiffusionStateModel(A, B; init)
```

線形拡散過程の隠れ状態モデル $dX_t = A X_t dt + B dW_t$ を返します。ここで、行列 $A$ と $B$ は適切なサイズです。

オプション引数 `init` はプロセスの初期条件を表し、次のいずれかです。

  * 固定（決定論的）初期条件のための長さ `n` のベクトル
  * ランダム初期条件のための `Distributions.Sampleable` 型

引数 `init` が省略された場合、次のいずれかに設定されます。

  * 定常分散に設定された共分散行列を持つ多変量正規分布（存在する場合）
  * ゼロベクトル
