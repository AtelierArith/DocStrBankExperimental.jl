```
ObservationModel{S1, S2, T} <: AbstractModel{S2}
```

観測の型 `S2` と隠れ状態の型 `S1` の任意のモデルのための抽象型です。

モデルを呼び出すと、新しい観測が生成されます。離散時間または静的モデルの場合、

```
model(x)
```

は隠れ状態 `x` に基づいて新しい観測を返します。

連続時間モデルの場合、

```
model(x, dt)
```

は隠れ状態 `x` と時間ステップ `dt` に基づいて新しい観測を返します。

もし `x` が `state_type` 要素のベクトルまたはバッチ行列である場合、バッチの各要素に対して観測が生成されます。
