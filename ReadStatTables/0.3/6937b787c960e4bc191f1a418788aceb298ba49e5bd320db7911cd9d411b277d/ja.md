```
getvaluelabels(tb::ReadStatTable)
getvaluelabels(tb::ReadStatTable, name::Symbol)
```

`tb`からデータファイルから取得したすべての値ラベル辞書の辞書を返します。`name`が指定されている場合は、特定の値ラベルの辞書を返します。

各値ラベルの辞書は、データ列に添付された値ラベルの辞書を識別するために、`vallabel`キーの下にある変数レベルのメタデータに表示される可能性のある名前に関連付けられています。同じ辞書は複数のデータ列に関連付けられる場合があります。データ列の`vallabel`のメタデータ値を変更すると、そのデータ列に関連付けられた値ラベルが切り替わります。メタデータ値が`Symbol("")`に設定されている場合、そのデータ列は値ラベルに関連付けられていません。
