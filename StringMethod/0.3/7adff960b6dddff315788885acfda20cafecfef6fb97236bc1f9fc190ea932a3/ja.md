`SaddleOptions(;nmax = 10^3, tol = 1e-6, verbose = false, save_trajectory = true, print_iters = 10^3)` - クライミングイメージ法の鞍点オプションを設定します

### フィールド

  * `nmax = 10^3` - ソルバーの最大反復回数
  * `tol = 1e-6` - 停止許容誤差
  * `verbose = false` - 診断および収束情報を表示
  * `save_trajectory = true` - 各アルゴリズムステップで全体の軌跡を保存
  * `print_iters = 10^3` - この頻度で診断情報を表示
