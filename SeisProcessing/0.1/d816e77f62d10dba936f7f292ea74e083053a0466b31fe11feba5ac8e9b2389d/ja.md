```
SeisMute(d; <キーワード引数>) -> Array{Real,2}
```

最初の到着前にトレースノイズをミュートします。ミューティング関数は、ソースとレシーバーの間のオフセットと最初の層の速度に依存します。t*mute = t*0 + offset/v1。

# 引数

  * `d`: 入力トレースの配列。

# キーワード引数

  * `offset=[0.]`: ソースとレシーバーの間の距離のベクトル。
  * `tmute=0.`: 初期ミューティング時間
  * `vmute=1500.`: 最初の層の速度。
  * `taper=0.1`: ミューティング関数のテーパー
  * `dt::Real=0.004`: サンプリング間隔（秒単位）。
