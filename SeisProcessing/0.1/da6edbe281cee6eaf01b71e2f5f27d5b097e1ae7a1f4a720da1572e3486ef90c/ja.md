```
SeisLinearEvents(; <キーワード引数>)
```

線形イベントからなる最大5次元のデータを生成します。

# 引数

  * `ot=0.0`: 時間軸の最初のサンプル（秒単位）。
  * `dt=0.004`: 時間サンプリング間隔（秒単位）。
  * `nt=500`: 時間サンプルの数。
  * `ox1=0.0`: 最初の空間次元の最初のサンプル（メートル単位）。
  * `dx1=10.0`: 最初の空間次元のサンプル間隔（メートル単位）。
  * `nx1=100`: 最初の空間次元のサンプル数。
  * `ox2=0.0`: 第二の空間次元の最初のサンプル（メートル単位）。
  * `dx2=10.0`: 第二の空間次元のサンプル間隔（メートル単位）。
  * `nx2=1`: 第二の空間次元のサンプル数。
  * `ox3=0.0`: 第三の空間次元の最初のサンプル（メートル単位）。
  * `dx3=10.0`: 第三の空間次元のサンプル間隔（メートル単位）。
  * `nx3=1`: 第三の空間次元のサンプル数。
  * `ox4=0.0`: 第四の空間次元の最初のサンプル（メートル単位）。
  * `dx4=10.0`: 第四の空間次元のサンプル間隔（メートル単位）。
  * `nx4=1`: 第四の空間次元のサンプル数。
  * `tau=[1.0, 1.5]`: 各イベントの切片到達時間。
  * `p1=[0.0001,-0.0003]`: 最初の次元におけるイベントの傾斜。
  * `p2,p3,p4=[0, 0]`: 次の次元におけるイベントの傾斜。
  * `amp=[1.0,-1.0]`: 各線形イベントの振幅。
  * `f0=20.0`: 各線形イベントのウェーブレットの中心周波数。

# 例

```julia
julia> using SeisPlot
julia> d = SeisLinearEvents(); SeisPlotTX(d);
```
