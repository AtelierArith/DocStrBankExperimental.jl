```
SeisRadonFreqInv(d; <キーワード引数>)
```

時間オフセットの集約からτ-p集約に変換するために、最小二乗逆変換を介して周波数領域の逆放物線または線形ラドン演算子を使用します。

# 引数

  * `d::Array{Float64,2}`: 2Dデータ、`d[1:nt,1:nh]`、ここで`nt`は時間サンプルの数、`nh`はレシーバーの数です。

# キーワード引数

  * `order::"parab"`: 放物線変換の場合は`"parab"`、線形変換の場合は`"linear"`。

  * `dt=0.004`: サンプリング間隔（秒）。
  * `h=collect(0.0:20.0:1000.0)`: オフセットベクトル`h[1:nh]`。
  * `href=0.0`: 放物線ラドン変換の基準オフセット。デフォルト値`href=0.0`が指定された場合、`href`は`max(abs(h))`に設定されます。
  * `p=collect(-0.05:0.01:2.2)`: `p[1:np]`。`order="parab"`の場合、`p`は基準オフセット`href`における残差移動時間（"曲率"）のベクトル（秒）です；`order=linear`の場合、`p`はレイパラメータのベクトル（s/m）です。
  * `flow=0.0`: データの最小周波数（Hz）。
  * `fhigh=125.0`: データの最大周波数（Hz）。
  * `mu=0.001`: L.S.逆変換のトレードオフパラメータまたはダンピング。

# 出力

  * `m`: 逆ラドンパネル`m[1:ntau, 1:np]`。

# 参考文献

  * Hampson, D., 1986, 逆速度スタッキングによる複数排除: Canadian Journal of Exploration Geophysics, 22, 44-55。
  * Sacchi, M.D. and Ulrych, T.J., 1995, 高解像度速度集約とオフセット空間再構成: Geophysics, 60, 1169-1177。
