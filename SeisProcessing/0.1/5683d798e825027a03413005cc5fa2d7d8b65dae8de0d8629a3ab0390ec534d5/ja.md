```
SeisFKFilter(in; <キーワード引数>)
```

周波数-波数領域での集約にフィルターを適用することによって、地震データからエネルギーを除去します。

# 引数

  * `in`: TXドメインの2D入力データ。最初の次元は時間です。

# キーワード引数

  * `dt=0.004`: 秒単位の時間サンプリング間隔。
  * `dx=10`: メートル単位の空間サンプリング間隔。
  * `va=-2000,vb=-3000,vc=3000,vd=2000`: フィルタリングする見かけの速度のコーナー

# 出力

  * `out`: TXドメインのフィルタリングされたデータ

# 例

```julia
julia> d = SeisLinearEvents(); df = SeisFKFilter(d,va=-4000.,vb=-6000.,vc=6000.,vd=4000.);
```
