```
SeisDecimate(in; <キーワード引数>)
```

多次元データ配列にランダムまたは規則的なデシメーションを適用します。入力と出力は同じ次元を持ちます。

# 引数

  * `in`: 2D、または3D、4D、5Dテンソルとしての入力データ。最初の次元は時間です。

# キーワード引数

  * `mode="random"`: デシメーションモード。デフォルトはランダムで、そうでなければ均等にデシメートします。
  * `perc=50`: ゼロに等しいトレースの割合（デシメートされたもの）。ランダムモードのみ
  * `incx1=1`: 最初の次元でゼロにされる連続トレース。規則的デシメーションのみ
  * `incx2=1`: 第二次元でゼロにされる連続トレース。
  * `incx3=1`: 第三次元でゼロにされる連続トレース。
  * `incx4=1`: 第四次元でゼロにされる連続トレース。

# 例

```julia
julia> d = SeisLinearEvents(); deci = SeisDecimate(d);
```
