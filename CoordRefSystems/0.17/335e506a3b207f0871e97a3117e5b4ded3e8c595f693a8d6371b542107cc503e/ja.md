```
メルカトル(x, y)
メルカトル{Datum}(x, y)
```

メルカトル座標は長さ単位（デフォルトはメートル）で、指定された`Datum`（デフォルトは`WGS84`）を使用します。

## 例

```julia
メルカトル(1, 1) # デフォルト単位を追加
メルカトル(1m, 1m) # 整数は浮動小数点数に変換されます
メルカトル(1.0km, 1.0km) # 長さの量はメートルに変換されます
メルカトル(1.0m, 1.0m)
メルカトル{WGS84Latest}(1.0m, 1.0m)
```

[EPSG:3395](https://epsg.io/3395)を参照してください。
