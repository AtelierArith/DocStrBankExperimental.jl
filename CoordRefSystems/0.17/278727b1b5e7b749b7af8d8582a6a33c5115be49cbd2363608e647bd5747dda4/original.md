```
ETRF{Year}
```

ETRF (European Terrestrial Reference Frame) datum with a given `Year`. Each ETRF is a realization of the European Terrestrial Reference System 89 (ETRS89) ensemble. Currently, ETRF has realizations in the following years: 1989, 1990, 1991, 1992, 1993, 1994, 1996, 1997, 2000, 2005, 2014 and 2020.

See [http://etrs89.ensg.ign.fr/](http://etrs89.ensg.ign.fr/)
