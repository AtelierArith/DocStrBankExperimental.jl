```
vall([p=identity,] A::AbstractArray, dims=:)
```

与えられた `dims` に対して、述語 `p` がすべての要素に対して真を返すかどうかを判断します。`p` が省略された場合、与えられた `dims` に沿ったすべての値が `true` であるかどうかをテストします（この場合、`A` は `AbstractArray{Bool}` である必要があります）。

# 使用推奨

`A` が比較的小さい場合、`vall` は `all` よりも速いかもしれません。しかし、`A` のサイズが大きくなるにつれて、すべての要素が `false` を返す確率は必然的に増加します（これは繰り返しのベルヌーイサンプリングであり、非常に小さな成功確率であっても、大量の試行があれば `all` の `break` が勝つシナリオが生じる可能性があります）。したがって、個々の要素が `true` である確率が選択を決定すべきです。合理的な成功確率が疑われる場合、`A` のサイズに応じて `all` の方が好ましいかもしれません。潜在的なブレークポイントを特定するためには、さらなるテストが必要です。

# 追加の注意事項

この関数は `vfindmax` やその仲間と同じ問題を抱えています。最大マスクを含む最初の次元を持つ還元は、まだ LoopVectorization によってサポートされていません。特に、この関数は最初の次元を含まない任意の還元に対しては意図した通りに機能します。
