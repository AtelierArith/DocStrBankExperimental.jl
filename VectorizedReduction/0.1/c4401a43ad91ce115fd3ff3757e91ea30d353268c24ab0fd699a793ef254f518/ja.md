```
vfindmin(f, A::AbstractArray, dims=:) -> (f(x), index)
```

引数 `f` の値とインデックスを返し、`dims` にわたって `f` を最小化します。`dims` は `::Int`、`::NTuple{M, Int} where {M}` または `::Colon` である可能性があります。これは、Julia (v1.8) Base で提供される機能を拡張したものです。

# 追加の注意事項

現在の LoopVectorization の制限により、配列の最初の次元に対する検索は十分にサポートされていません。`A` を再形成することで回避策が可能ですが、結果として得られるパフォーマンスはしばしば `findmin` と同等です。一時的な便宜として、再形成戦略の明示的な使用のために `vfindmin1` が提供されていますが、ユーザーにはパフォーマンスの問題に注意が必要です。

# 警告

`NaN` 値は処理されません！
