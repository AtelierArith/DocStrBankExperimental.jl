plot_polar(     r;     trange = [0, 0],     rrange = [0, 0],     width = 0,     height = 0,     mode = "lines",     color = "", )

極座標プロットを描画します（θ軸は指定されていません）。

#### 引数

  * `r`: 放射座標データ（ベクトルのベクトルである可能性があります）
  * `trange`: 角度軸の範囲（デフォルト: `[0, 0]`）
  * `rrange`: 放射軸の範囲（デフォルト: `[0, 0]`）
  * `width`: プロットの幅（デフォルト: `0`）
  * `height`: プロットの高さ（デフォルト: `0`）
  * `mode`: プロットモード（デフォルト: `"lines"`、ベクトルである可能性があります）
  * `color`: プロット線の色（デフォルト: `""`、ベクトルである可能性があります）
  * `name`: プロット線の名前（デフォルト: `""`、ベクトルである可能性があります）
