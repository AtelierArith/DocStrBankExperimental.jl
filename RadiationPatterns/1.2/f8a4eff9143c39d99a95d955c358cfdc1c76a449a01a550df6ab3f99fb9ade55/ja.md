```markdown
plot_rect(     x,     y;     xlabel = "",     ylabel = "",     xrange = [0, 0],     yrange = [0, 0],     width = 0,     height = 0,     mode = "lines",     color = "",     name = "", ) 矩形（直交座標）プロットを描画します。

#### 引数

  * `x`: x座標データ（ベクトルのベクトル可）
  * `y`: y座標データ（ベクトルのベクトル可）
  * `xlabel`: x軸のラベル（デフォルト: `""`）
  * `ylabel`: y軸のラベル（デフォルト: `""`）
  * `xrange`: x軸の範囲（デフォルト: `[0, 0]`）
  * `yrange`: y軸の範囲（デフォルト: `[0, 0]`）
  * `width`: プロットの幅（デフォルト: `0`）
  * `height`: プロットの高さ（デフォルト: `0`）
  * `mode`: プロットモード（デフォルト: `"lines"`、ベクトル可）
  * `color`: プロット線の色（デフォルト: `""`、ベクトル可）
  * `name`: プロット線の名前（デフォルト: `""`、ベクトル可）
```
