```markdown
plot_rect(     y;     xlabel = "",     ylabel = "",     xrange = [0, 0],     yrange = [0, 0],     width = 0,     height = 0,     mode = "lines",     color = "",     name = "", ) 矩形（直交）プロットを描画します（x軸は指定されていません）。

#### 引数

  * `y`: y座標データ（ベクトルのベクトルである可能性があります）
  * `xlabel`: x軸のラベル（デフォルト: `""`）
  * `ylabel`: y軸のラベル（デフォルト: `""`）
  * `xrange`: x軸の範囲（デフォルト: `[0, 0]`）
  * `yrange`: y軸の範囲（デフォルト: `[0, 0]`）
  * `width`: プロットの幅（デフォルト: `0`）
  * `height`: プロットの高さ（デフォルト: `0`）
  * `mode`: プロットモード（デフォルト: `"lines"`、ベクトルである可能性があります）
  * `color`: プロット線の色（デフォルト: `""`、ベクトルである可能性があります）
  * `name`: プロット線の名前（デフォルト: `""`、ベクトルである可能性があります）
```
