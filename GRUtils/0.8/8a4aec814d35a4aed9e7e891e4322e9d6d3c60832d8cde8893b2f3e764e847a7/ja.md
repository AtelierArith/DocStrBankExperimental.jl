```
volume(v; kwargs...)
```

三次元配列 `v` をボリュームレンダリングを使用して描画します。

ボリュームデータは、放射または吸収モデル、または最大強度投影を使用して二次元画像に縮小されます。投影後、現在のカラーマップが結果に適用されます。

ボリュームデータを縮小する方法は、キーワード引数 `algorithm` で定義できます。以下の表からの数値または文字列です。

|  #  | String         | 説明           |
|:---:|:-------------- |:------------ |
|  0  | `"emission"`   | 放射モデル（デフォルト） |
|  1  | `"absorption"` | 吸収モデル        |
|  2  | `"mip"`        | 最大強度投影       |

# 例

```julia
# サンプルデータを作成
x = LinRange(-1, 1, 40)
y = LinRange(-1, 1, 40)
z = LinRange(-1, 1, 40)
v = 1 .- (x.^2 .+ y'.^2 .+ reshape(z,1,1,:).^2).^0.5 - 0.25 .* rand(40, 40, 40)
# 放射モデルを使用して3Dボリュームデータを描画
volume(v, algorithm="mip")

```
