```
scatter(x[, y, size, color; kwargs...])
```

散布図を描画します。

ポイントは、数値ベクトルとして与えられる `x` および `y` 座標によって定義されます。さらに、マーカーの `size` および `color` の値を提供することができます。サイズの値は、通常のサイズのパーセントでマーカーのサイズを決定し、色の値は現在のカラーマップと組み合わせて使用されます。

最後の変数は、前の変数の関数として定義する callable に置き換えることができます。

この関数は、単一の数値ベクトルまたは行列を受け取ることができ、その場合は Y 座標として解釈されます。その場合、X 座標は 1 から始まる整数のシーケンスになります。

# 例

```julia
# サンプルデータを作成
x = LinRange(0, 1, 11)
y = LinRange(0, 1, 11)
s = LinRange(50, 400, 11)
c = LinRange(0, 255, 11)
# サイズと色が増加するポイントをプロット
scatter(x, y, s, c)

```
