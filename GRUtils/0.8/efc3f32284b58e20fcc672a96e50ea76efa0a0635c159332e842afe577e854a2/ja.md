```
subplot(num_rows, num_columns, indices[, replace])
```

現在の図にサブプロットを設定します。

デフォルトでは、現在のプロットはウィンドウ全体をカバーします。複数のプロットを表示するには、ウィンドウを行と列の数に分割し、各プロットがその結果得られるグリッドの1つ以上のセルをカバーするようにします。

サブプロットのインデックスは1から始まり、左上隅から始まります。新しい行は、毎回`num_cols`のサブプロットの後に始まります。

引数`num_rows`と`num_cols`は、図が分割されるプロットのグリッドの行数と列数を示し、`indices`はそのグリッド内のセルのグループを識別する整数または整数のコレクションです。この関数は、すべてのセルを跨ぐ最小サイズのプロットを返し、それを図のプロットの配列に追加して、現在のプロットとなります。

新しいサブプロットのビューポートが既存のプロットのビューポートと一致する場合、デフォルトでは古いプロットが最前面に移動し、「現在のプロット」として扱われます。しかし、新しいサブプロットと他のプロットの間に部分的な重なりがある場合、重なっているプロットは削除されます。この動作をオーバーライドして、すべての以前のプロットを変更せずに保持するには、オプションの引数`replace`をfalseに設定します。

# 例

```julia
# サンプルデータを作成
x = randn(100_000)
y = randn(100_000)
# より大きな右下の領域に六角形プロットを描画
subplot(3, 3, (5, 9))
hexbin(x, y, colorbar = false)
# 周辺ヒストグラムを描画
subplot(3, 3, (2, 3))
histogram(x)
subplot(3, 3, (4, 7))
histogram(y, horizontal = true, xflip = true)
# より小さな左上の領域にシェードプロットを描画
subplot(3, 3, 1)
shade(x, y)

```
