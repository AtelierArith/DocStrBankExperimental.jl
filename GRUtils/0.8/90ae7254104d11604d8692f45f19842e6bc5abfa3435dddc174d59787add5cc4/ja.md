```
xlog(flag)
ylog(flag)
zlog(flag)
```

X軸、Y軸、またはZ軸を対数スケール（`flag == true`）または線形スケール（`flag == false`）で描画するように設定します。

プロット関数でキーワード引数 `xlog=<true/false>` などを使用して、プロット作成時に対数軸を設定します。

!!! note
    軸が対数スケールに設定されると、その下限は正の値のみを表すように調整されます。プロットのデータにゼロまたは負の値が含まれていてもです。負の値を含むように明示的に設定された対数軸の限界（[`xlim`](@ref) など）を持つ場合、その様子は未定義です。


# 例

```julia
# x軸の限界を対数スケールに設定
xlog(true)
# y軸が線形スケールであることを確認
ylog(false)
```
