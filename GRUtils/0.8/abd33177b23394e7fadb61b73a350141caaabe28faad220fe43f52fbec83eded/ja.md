```
contourf([x, y,] z; kwargs...)
```

塗りつぶし等高線プロットを描画します。

現在のカラーマップは、点の系列または二次元配列を塗りつぶし等高線プロットとして表示するために使用されます。次のいずれかを受け取ることができます：

  * `x` 値、`y` 値、および `z` 値。
  * *M* 個のソートされた `x` 軸の値、*N* 個のソートされた `y` 軸の値、および *N*×*M* グリッド上の `z` 値のセット。
  * *M* 個のソートされた `x` 軸の値、*N* 個のソートされた `y` 軸の値、および `z` 値を決定するための呼び出し可能な関数。
  * *N*×*M* グリッド上の `z` 値で、*x* 軸と *y* 軸はそれぞれグリッドの列と行のインデックスとして定義されます。

この関数に一連の点が渡されると、それらの値はグリッド上で補間されます。提供された点の凸包の外側にあるグリッドポイントには、0 の値が使用されます。

等高線の数とアスペクトを設定するために提供できる以下のキーワード引数があります：

  * `levels::Int`、データにフィットする等高線の数（デフォルトは 20）。
  * `majorlevels::Int`、ラベル付き等高線の間のレベルの数（デフォルトではラベルなし）。

# 例

```julia
# 1. サンプル点データを作成
x = 8 .* rand(100) .- 4
y = 8 .* rand(100) .- 4
z = sin.(x) + cos.(y)
# 等高線プロット
contourf(x, y, z)
# 2. 呼び出し可能な関数を持つサンプルグリッドデータを作成
x = LinRange(-2, 2, 40)
y = LinRange(0, pi, 20)
f(x, y) = sin(x) + cos(y)
# 25 本の線を持つ等高線プロット、4 分の 1 にラベル付け
contourf(x, y, f, levels = 25, majorlevels = 4)

```
