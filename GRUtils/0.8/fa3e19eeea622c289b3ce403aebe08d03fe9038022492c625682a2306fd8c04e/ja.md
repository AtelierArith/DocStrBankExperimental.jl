```
polarhistogram(data; kwargs...)
```

`data` に含まれる角度値の極座標ヒストグラムを描画します。

次のキーワード引数を指定できます：

  * `nbins`: ビンの数；デフォルトでは、または1未満の数が指定された場合、ビンの数は `3.3 * log10(n) + 1` として計算されます。ここで、`n` は `data` の要素数です。
  * `radians`: この引数を `false` に設定すると、角度を度数法で表示します。デフォルトでは、`data` はラジアンであると仮定され、グリッドの角度ラベルは π の因子として表示されます。
  * `fullcircle`: この引数を `true` に設定すると、ヒストグラムの角度座標がスケーリングされ、バーが全円にわたって広がります。
  * `color`: バーのための16進RGBカラーコード。

!!! note
    極座標プロットでは、対数スケールおよび逆スケールは無効です


# 例

```julia
# サンプルデータを作成
data = randn(100)
# 19ビンの極座標ヒストグラムを描画
polarhistogram(data, nbins = 19, fullcircle=true)

```
