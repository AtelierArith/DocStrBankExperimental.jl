```
tricont(x, y, z; kwargs...)
```

三角形の等高線プロットを描画します。

現在のカラーマップは、三角形の等高線プロットとして一連の点を表示するために使用されます。`z` 値は、[ドロネー三角形分割](http://mathworld.wolfram.com/DelaunayTriangulation.html)を通じて `x` と `y` 値の間で補間されます。

等高線の数はキーワード引数 `levels` で設定できます（デフォルトでは `levels = 20` です）。

!!! note
    点の系列が凹の場合、プロットの端に補間アーティファクトが発生する可能性があります。これは、補間が非常に鋭い三角形で行われる可能性があるためです。


# 例

```julia
# サンプル点データを作成
x = 8 .* rand(100) .- 4
y = 8 .* rand(100) .- 4
z = sin.(x) + cos.(y)
# Tricontour プロット
tricont(x, y, z)

```
