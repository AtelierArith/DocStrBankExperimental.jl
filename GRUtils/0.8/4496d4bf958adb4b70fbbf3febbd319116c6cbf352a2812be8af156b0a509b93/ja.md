```
viewpoint(rotation, tilt)
```

三次元プロットの視点を設定します。

`rotation` と `tilt` は、視線の「方位角」と「仰角」を示す整数値でなければなりません（度単位）。

両方の角度がゼロの場合、プロットはY軸の方向から見られます（すなわち、X-Z平面が見えます）。正の `rotation` 値は、視線の反時計回りの回転（またはシーンの時計回りの回転）を意味し、垂直（Z）軸の周りで回転します。正の `tilt` 値は、視点の上昇を意味します。

# 例

```julia
# 視点をX-Y平面にリセット
# (rotation=0, tilt=90)
viewpoint(0, 90)
```
