```
movefocus(target)
```

カメラの視点軸を回転させ、フォーカスを`target`ポイントに移動します。

これは、カメラ設定で作成された3-Dシーンにのみ影響します。たとえば、[`isosurface`](@ref)プロットです。フォーカスポイントを移動すると、カメラの位置は変わらずに回転します。シーンの中心の周りでカメラを回転させるには、[`rotate`](@ref)、[`tilt`](@ref)または[`viewpoint`](@ref)の関数を使用してください。

# 例

```julia
# フォーカスを点(1.0, 0.5, 0.0)に移動
movefocus([1.0, 0.5, 0.0])
```
