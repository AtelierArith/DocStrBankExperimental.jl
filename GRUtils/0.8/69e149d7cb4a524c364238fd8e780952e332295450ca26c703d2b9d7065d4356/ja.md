```
stairs(x[, y, spec; kwargs...])
stairs(x1, y1, x2, y2...; kwargs...)
stairs(x1, y1, spec1...; kwargs...)
```

1つ以上の階段またはステッププロットを描画します。

階段のアウトラインの座標と形式は、線プロットと同様に定義されています（cf. [`plot`](@ref)）。

さらに、キーワード引数 `where` を使用して、「階段」（Y値間の垂直不連続性）を配置する場所を定義できます：

  * `where = "pre"` は、各点 (`x[i]`, `y[i]`) でステップが停止し、最初の点を除いて前の `x` 座標から始まります。
  * `where = "post"` は、各点 (`x[i]`, `y[i]`) でステップが始まり、最後の点を除いて次の `x` 座標で停止します。
  * `where = "mid"`（デフォルト）は、各点 (`x[i]`, `y[i]`) を通過するステップを作成し、周囲の x-区間の中央で始まり、終了します。ただし、最初と最後の点を除きます。

# 例

```julia
# サンプルデータを作成
x = LinRange(-2, 2, 40)
y = x.^3 .+ x.^2 .+ x
# x と y をプロット
stairs(x, y)
# x 値のインデックスで y をプロット
stairs(y)
# x の各位置の直後にステップ
stairs(y, where="pre")
# 2つの x 位置の間でステップ
stairs(y, where="mid")
# x の各位置の直前にステップ
stairs(y, where="post")

```
