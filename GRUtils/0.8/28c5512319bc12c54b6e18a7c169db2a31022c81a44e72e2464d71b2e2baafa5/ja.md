```
colorscheme!(p, scheme)
```

与えられたプロット `p` に色の `scheme` を適用します。`p` は `PlotObject` または `Figure` であり（この場合、スキームはその中に含まれるすべてのプロットに適用されます）。

`scheme` の値は、利用可能な任意のカラースキームの番号または名前であることができます（詳細については [`colorscheme`](@ref) を参照してください）。

プロット関数でキーワード引数 `scheme` を使用して、プロットの作成中に特定のカラースキームを設定します（この場合、既存のスキームの番号のみが許可されます）。

# 例

```julia
# ダークスキーム (2) のプロットを作成
plot(x, y, scheme=2)
# 標準のライトスキームに変更
colorscheme!(currentplot(), "light")
```
