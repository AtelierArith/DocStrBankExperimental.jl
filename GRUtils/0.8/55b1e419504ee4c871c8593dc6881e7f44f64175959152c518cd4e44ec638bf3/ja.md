```
shade(x, y; kwargs...)
```

ポイントまたは線ベースのヒートマップを描画します。

現在のカラーマップは、`x`、`y` 値のペアによって残された足跡を表示するために使用されます。データに `NaN` または `missing` が含まれている場合、足跡はそれらの値によって分離された線に基づいています。それ以外の場合、足跡はポイントに基づいています。キーワード引数 `footprint = "lines"` または `footprint = "points"` によって、入力に関係なく足跡のタイプを強制することができます。

その足跡の値は、キーワード引数 `xform` によって調整可能な変換によって決定されます – 数字または以下の表からの文字列です：

|  #  | 文字列           | 説明               |
|:---:|:------------- |:---------------- |
|  0  | `"boolean"`   | ブール値             |
|  1  | `"linear"`    | 線形               |
|  2  | `"log"`       | 対数               |
|  3  | `"loglog"`    | 二重対数             |
|  4  | `"cubic"`     | 三次               |
|  5  | `"equalized"` | ヒストグラム均等化（デフォルト） |

# 例

```julia
# NaNをポリラインセパレーターとして持つ線データを作成
x = [randn(10000); NaN; randn(10000) .+ 5 ]
y = [randn(10000); NaN; randn(10000) .+ 5]
# shadeを描画
shade(x, y, xform="loglog")

```
