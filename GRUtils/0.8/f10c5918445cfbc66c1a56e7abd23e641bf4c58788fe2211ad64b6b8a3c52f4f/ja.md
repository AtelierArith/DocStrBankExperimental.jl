```
plot3(x, y, z[, spec; kwargs...])
plot3(x1, y1, z1, x2, y2, z2...; kwargs...)
plot3(x1, y1, z1, spec1...; kwargs...)
```

1つ以上の三次元線グラフを描画します。

線は、接続された点の `x`、`y`、`z` 座標によって定義され、数値ベクトルとして与えられ、オプションで `spec` というフォーマット文字列が指定され、[matplotlib](https://matplotlib.org/3.1.1/api/_as_gen/matplotlib.pyplot.plot.html) のように線とマーカーのスタイルおよび色を定義します。

`z` ベクトルは、X および Y 座標の関数として Z 座標を定義する呼び出し可能なもので置き換えることができます。

複数の線は、`x`、`y`、`z` 座標の三つ組（およびオプションでそのフォーマット文字列）によって定義され、`plot3` の引数として順次渡されます。

`specs` が指定されていない場合、系列は事前定義された色のシーケンスで実線としてプロットされます。

さらに、線とマーカーの仕様はキーワード引数によって定義できます：

  * `linewidth`: 線幅のスケールファクター。
  * `markersize`: マーカーサイズのスケールファクター。
  * `linecolor`: 線の16進RGBカラーコード。
  * `markercolor`: マーカーの16進RGBカラーコード。

# 例

```julia
# サンプルデータを作成
x = LinRange(0, 30, 1000)
y = cos.(x) .* x
z = sin.(x) .* x
# 実線と、10ポイントごとに星マーカーを持つ線を描画
plot3(x, y, z, x[1:10:end], y[1:10:end], z[1:10:end], "p")

```
