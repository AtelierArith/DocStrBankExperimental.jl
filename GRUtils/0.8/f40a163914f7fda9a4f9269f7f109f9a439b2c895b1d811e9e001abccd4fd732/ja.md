```
wireframe([x, y,] z; kwargs...)
```

三次元ワイヤーフレームプロットを描画します。

一連の点または二次元配列がワイヤーフレームプロットとして描画されます。以下のいずれかを受け取ることができます：

  * `x` 値、`y` 値、および `z` 値。
  * *M* のソートされた `x` 軸の値、*N* のソートされた `y` 軸の値、および *N*×*M* グリッド上の `z` 値のセット。
  * *M* のソートされた `x` 軸の値、*N* のソートされた `y` 軸の値、および `z` 値を決定するための呼び出し可能な関数。
  * *N*×*M* グリッド上の `z` 値で、*x* および *y* 軸はそれぞれグリッドの列と行のインデックスとして定義されます。

また、属性 `color` と `linecolor` を使用して、メッシュの表面と線の色をRGB16進数の色値として設定します。

この関数に一連の点が渡されると、それらの値はグリッド上で補間されます。提供された点の凸包の外側にあるグリッドポイントには、値0が使用されます。

# 例

```julia
# 例のグリッドデータを作成
x = LinRange(-2, 2, 40)
y = LinRange(0, pi, 20)
z = sin.(x') .+ cos.(y)
# ワイヤーフレームプロット
wireframe(x, y, z)

```
