```
annotations(x, y, s; kwargs...)
```

現在のプロットに1つ以上のテキスト注釈を追加します。

`x` と `y` はスカラーまたは座標のベクトルであり、`s` は文字列または文字列のベクトルです。注釈は2-Dプロットでのみ利用可能です。

デフォルトでは、座標はテキストボックスの左下隅を示します。これは次のキーワード引数によって変更できます：

  * `halign` は水平方向の整列（`"left"`、`"center"` または `"right"`）を指定します。
  * `valign` は垂直方向の整列（`"bottom"`、`"center"` または `"top"`）を指定します。

`x` と `y` がワールド座標ではなく正規化デバイス座標で表現されている場合は、キーワード引数 `wc = false` を使用します – これがデフォルトです。

# 例

```julia
# サンプルデータを作成
numbers = [10, 15, 35, 20]
# ラベル付きのバーをプロット
barplot(numbers)
annotations(1:4, numbers .+1 , string.(numbers), halign="center")

```
