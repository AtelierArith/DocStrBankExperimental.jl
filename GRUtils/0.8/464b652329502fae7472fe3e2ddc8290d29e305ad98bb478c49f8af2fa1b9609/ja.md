```
video(fun::Function, target="webm")
video(figs::AbstractArray{<:Fig}, target="webm")
```

関数または図の配列からビデオを作成します。

最初の引数は、ビデオのフレームとして表示される`Figures`の配列であるか、引数なしで図を描画する関数です（通常は、図が一つずつ作成されて描画されるループ）。その関数は、`do`構文を使って`video`の呼び出し内で匿名で定義できます（例を参照）。

2番目の（オプションの）引数`target`は、`"webm"`（デフォルト）、`"mp4"`、または`"mov"`のいずれかの形式の文字列でなければなりません。

出力は、指定された形式のビデオとして表示できるオブジェクトです（環境のサポートされているMIME出力に依存します）。[`videofile`](@ref)を使用して、そのようなビデオをファイルに保存します。

# 例

```julia
# サンプルデータでプロットを作成
x = LinRange(0, 800, 100)
y = sind.(x)
plot(x,y)
# X軸をスライドさせるビデオを作成
video("webm") do
  for d = 0:10:440
    xlim(d, d+360)
    draw(gcf())
  end
end
```
