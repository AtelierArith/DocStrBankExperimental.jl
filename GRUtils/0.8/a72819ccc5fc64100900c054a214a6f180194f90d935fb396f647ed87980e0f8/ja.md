```
radians(flag::Bool)
```

極座標プロットにおける角度のスケールを設定します。

`radians(true)`を使用して角度をラジアンで表し（デフォルト設定）、`radians(false)`を使用して角度を度で表します。

この操作は極座標プロットのグリッド線のガイドのみを変更します。既存のジオメトリは変更されません。

極座標プロットの作成中に角度のスケールを設定するために、プロット関数でキーワード引数`radians=<true/false>`などを使用します。

# 例

```julia
# 例のデータ
θ = LinRange(0, 2π, 40)
r = sin.(θ)
# 極座標プロットを描画（デフォルトではラジアン）
polar(θ, r)
# 角度スケールを変更
radians(false)
```
