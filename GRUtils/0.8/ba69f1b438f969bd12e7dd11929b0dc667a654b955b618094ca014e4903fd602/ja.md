```
tricont(x, y, z; kwargs...)
```

三角形の表面プロットを描画します。

一連の点または二次元配列が三角形の表面プロットとして描画されます。`z` 値は、[ドロネー三角形分割](http://mathworld.wolfram.com/DelaunayTriangulation.html)を通じて `x` と `y` 値の間で補間されます。

!!! note
    点の系列が凹の場合、プロットのエッジに補間アーティファクトが発生する可能性があります。これは、補間が非常に鋭角な三角形で行われる可能性があるためです。


# 例

```julia
# サンプル点データを作成
x = 8 .* rand(100) .- 4
y = 8 .* rand(100) .- 4
z = sin.(x) + cos.(y)
# トライコントールプロット
tricont(x, y, z)

```
