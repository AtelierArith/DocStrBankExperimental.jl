```
xlim(inf, sup [, adjust::Bool = false])
xlim((inf, sup), ...)
ylim(inf, sup, ...)
ylim((inf, sup), ...)
zlim(inf, sup, ...)
zlim((inf, sup), ...)
```

プロット軸の制限を設定します。

軸の制限は、個別の引数として渡すことも、`(inf, sup)` のタプルとして渡すこともできます。いずれかの制限を `nothing` に設定すると、データに基づいて自動的に決定されます。これはデフォルトの動作です。

制限に加えて、フラグ `adjust` を使用して、制限を調整する必要があるかどうかを指定できます。

プロット関数でキーワード引数 `xlim=(inf, sup)` などを使用して、プロットの作成時に軸の制限を設定します。

# 例

```julia
# x軸の制限を -1 と 1 に設定
xlim((-1, 1))
# x軸の制限を自動的に決定されるようにリセット
xlim()
# y軸の上限を設定し、下限を 0 に設定
ylim((0, nothing))
# y軸の下限をリセットし、上限を 1 に設定
ylim((nothing, 1))
```
