```
contour([x, y,] z; kwargs...)
```

等高線プロットを描画します。

現在のカラーマップは、ポイントの系列または二次元配列を等高線プロットとして表示するために使用されます。次のいずれかを受け取ることができます：

  * `x` 値、`y` 値、および `z` 値。
  * *M* 個のソートされた `x` 軸の値、*N* 個のソートされた `y` 軸の値、および *N*×*M* グリッド上の `z` 値のセット。
  * *M* 個のソートされた `x` 軸の値、*N* 個のソートされた `y` 軸の値、および `z` 値を決定するための呼び出し可能な関数。
  * *N*×*M* グリッド上の `z` 値で、*x* および *y* 軸はそれぞれグリッドの列と行のインデックスとして定義されます。

この関数にポイントの系列が渡されると、それらの値はグリッド上で補間されます。提供されたポイントの凸包の外側にあるグリッドポイントには、0 の値が使用されます。

等高線はデフォルトで色付けされており、現在のカラーマップをカラースケールとして使用します。色付きの線は、カラーバーを削除することで無効にできます（キーワード引数 `colorbar == false` を使用）。

等高線の数とアスペクトを設定するために提供できるキーワード引数は次のとおりです：

  * `levels::Int`、データにフィットする等高線の数（デフォルトは 20）。
  * `majorlevels::Int`、ラベル付き等高線の間のレベルの数（色付きの線にはデフォルトでラベルなし、色が削除された場合はすべての線にデフォルトでラベルあり）。

# 例

```julia
# 1. サンプルポイントデータを作成
x = 8 .* rand(100) .- 4
y = 8 .* rand(100) .- 4
z = sin.(x) + cos.(y)
# 左側の色なしの等高線プロット（デフォルトでラベル付きの線）
subplot(1, 2, 1)
contour(x, y, z, colorbar = false)
# 2. 呼び出し可能な関数を持つサンプルグリッドデータを作成
x = LinRange(-2, 2, 40)
y = LinRange(0, pi, 20)
f(x, y) = sin(x) + cos(y)
# 右側の色付きの等高線プロット
# 3 行ごとに明示的なラベル付き
subplot(1, 2, 2)
contour(x, y, f, majorlevels = 3)

```
