```
plot(x[, y, spec; kwargs...])
plot(x1, y1, x2, y2...; kwargs...)
plot(x1, y1, spec1...; kwargs...)
```

1つ以上の線グラフを描画します。

線は、接続された点の `x` および `y` 座標によって定義され、数値ベクトルとして与えられ、オプションで `spec` というフォーマット文字列が指定され、[matplotlib](https://matplotlib.org/3.1.1/api/_as_gen/matplotlib.pyplot.plot.html) のように線とマーカーのスタイルおよび色を定義します。

`y` ベクトルは、X 座標の関数として Y 座標を定義する呼び出し可能なオブジェクトに置き換えることができます。

複数の線は、`plot` の引数として順次渡される `x` および `y` 座標のペア（およびオプションでそのフォーマット文字列）によって定義できます。あるいは、さまざまな線が同じ X 座標を持つ場合、それらの Y 値を行列の列としてグループ化することができます。

`spec` が指定されていない場合、系列は事前定義された色のシーケンスを使用して実線としてプロットされます。

さらに、線とマーカーの仕様はキーワード引数によって定義できます：

  * `linewidth`: 線の幅のスケールファクター。
  * `markersize`: マーカーのサイズのスケールファクター。
  * `linecolor`: 線のための16進RGBカラーコード。
  * `markercolor`: マーカーのための16進RGBカラーコード。

この関数は、単一の数値ベクトルまたは行列を受け取ることができ、Y 座標として解釈されます。この場合、X 座標は1から始まる整数のシーケンスになります。

# 例

```julia
# サンプルデータを作成
x = LinRange(-2, 2, 40)
f(t) = t^3 + t^2 + t
y = f.(x)
# x と y をプロット
plot(x, y)
# 呼び出し可能なオブジェクトを使用して x をプロット
plot(x, f)
# y をプロットし、x 値にはそのインデックスを使用
plot(y)
# 2つの列をプロット
plot(x, [y 3x.^2 .- 3])
# カスタムライン仕様でさまざまな系列をプロット
y2 = 3x.^2 .- 3
plot(x, y, "-r", x, y2, ":*b")

```
