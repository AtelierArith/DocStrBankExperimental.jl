```
xticklabels(f)
yticklabels(f)
```

X軸とY軸の目盛ラベルの文字列をカスタマイズします。

目盛軸のラベルは、1つの引数（目盛位置の数値）を持つ関数によって定義され、その関数が文字列を返すか、X = 1, 2, などの位置に順次配置される文字列の配列によって定義できます。

プロット作成時に軸目盛ラベルを設定するには、プロット関数でキーワード引数 `xticklabels=s` などを使用します。

# 例

```julia
# Y軸の範囲 (0-1) をパーセント値としてラベル付け
yticklabels(p -> Base.Printf.@sprintf("%0.0f%%", 100p))
# X軸を文字列のシーケンスでラベル付け
xticklabels(["first", "second", "third"])
```
