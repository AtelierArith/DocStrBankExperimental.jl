```
scatter3(x, y, z[, color; kwargs...])
```

三次元散布図を描画します。

点は、数値ベクトルとして与えられる `x`、`y`、および `z` 座標によって定義されます。さらに、マーカーの `color` の値を提供することができ、これは現在のカラーマップと組み合わせて使用されます。

最後の変数は、前の変数の関数として定義する callable に置き換えることができます。

# 例

```julia
# サンプルデータを作成
x = 2 .* rand(100) .- 1
y = 2 .* rand(100) .- 1
z = 2 .* rand(100) .- 1
c = 999 .* rand(100) .+ 1
# 色付きの点をプロット
scatter3(x, y, z, c)

```
