```
panzoom(x, y[, s = 0])
```

現在のプロットの軸をパン/ズームします。

ズームの焦点は、現在の軸の中心から正規化デバイス座標 (NDC) で `(x, y)` ユニットのオフセットを持つ点に設定されます。軸の隅はその点に向かって線形に移動され、新しい軸のサイズは元のサイズの `s` 倍になります。

`s` が 0 (デフォルト値) に設定されている場合、軸の中心はリサイズせずに焦点に移動します。

# 例

```julia
# 中心を右に1ユニット、上に0.2ユニット移動 (NDC)
panzoom(1, 0.2)
# 軸の焦点を元のサイズの半分に縮小
# 前の点に焦点を合わせる
panzoom(1, 0.2, 0.5)
```
