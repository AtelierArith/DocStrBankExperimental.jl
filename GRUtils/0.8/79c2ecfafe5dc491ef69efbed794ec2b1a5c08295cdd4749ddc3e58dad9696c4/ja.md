```
savefig(filename[, fig])
```

ファイルに図を保存します。

現在の図を指定された名前のファイルに描画します。サポートされているファイルタイプはインストールされているワークステーションのタイプによって異なりますが、GRは通常、.png、.jpg、.pdf、.ps、.gifおよびその他のさまざまなファイル形式のサポートを備えています。

図が指定されていない場合（オプションの引数 `fig`）、現在の図が使用されます。

# 例

```julia
# 簡単なプロットを作成
x = 1:100
plot(x, 1 ./ (x .+ 1))
# 図をファイルに保存
savefig("example.png")
```
