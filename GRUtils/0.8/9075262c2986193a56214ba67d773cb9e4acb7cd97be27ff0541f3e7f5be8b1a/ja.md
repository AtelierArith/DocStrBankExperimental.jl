```
polarheatmap(data; kwargs...)
```

極座標ヒートマップを描画します

現在のカラーマップを使用して、二次元配列 `data` を極座標ヒートマップとして表示します。

`data` が *N*×*M* 配列である場合、行列のセルは *M* の角度セクターと *N* のリングに分割された円にプロットされ、最初の行の値は円の中心に集中し、次の行は増加する同心リングに描画されます。配列の列は、左に向かう水平軸から始まり、右に向かう水平軸で終わる反時計回りの順序で円の半径として描画されます。

# 例

```julia
# サンプルデータを作成
angle = LinRange(0, 2π, 40)
radius = LinRange(0, 10, 20)
z = sin.(angle') .* cos.(radius)
polarheatmap(z)

```
