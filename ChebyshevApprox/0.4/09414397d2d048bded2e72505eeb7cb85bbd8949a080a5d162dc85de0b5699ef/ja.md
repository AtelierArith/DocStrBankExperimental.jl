`N` のチェビシェフ多項式の極値を計算し、`domain` で指定された区間に極値をスケーリングします（デフォルトは [1.0,-1.0]）。指定されたドメインにある `N` のチェビシェフ極値を含むベクトルを返します。

# シグネチャ

points = chebyshev_extrema(N,domain)

# 例

```
julia> points = chebyshev_extrema(4)
[-1.0
 -0.5000000000000001
  0.5000000000000001
  1.0]

julia> points = chebyshev_extrema(4,[3.0,-1.0])
[-1.0
 -2.220446049250313e-16
  2.0
  3.0]
```
