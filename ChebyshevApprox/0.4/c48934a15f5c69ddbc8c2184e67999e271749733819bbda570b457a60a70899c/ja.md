`N` のチェビシェフ多項式の根を計算し、`domain` で指定された区間に根をスケーリングします（デフォルトは [1.0,-1.0]）。指定されたドメインにある `N` のチェビシェフ根を含むベクトルを返します。

# シグネチャ

points = chebyshev_nodes(N,domain)

# 例

```
julia> points = chebyshev_nodes(2)
[-0.7071067811865476
  0.7071067811865476]

julia> points = chebyshev_nodes(2,[3.0,-1.0])
[-0.41421356237309515
  2.414213562373095]
```
