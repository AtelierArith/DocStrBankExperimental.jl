`N` 個のチェビシェフ拡張点を計算し、指定された `domain`（デフォルトは [1.0,-1.0]）の範囲にスケーリングします。指定されたドメインにわたる `N` 個のチェビシェフ拡張点を含むベクターを返します。

# シグネチャ

points = chebyshev_extended(N,domain)

# 例

```
julia> points = chebyshev_extended(4)
[-1.0
 -0.41421356237309515
  0.41421356237309515
  1.0]

julia> points = chebyshev_extended(4,[3.0,-1.0])
[-1.0
  0.1715728752538097
  1.8284271247461903
  3.0]
```
