```
psisloo(log_lik, [wcpp, wtrunc])
```

PSIS leave-one-out ログ予測密度。

入力パラメータ `log*lik` における対数尤度項 p(y*i|	heta^s) の事後サンプルに基づいて、ログ予測密度を計算します。留出法のログ予測密度の合計 `loo`、個々の留出法のログ予測密度項 `loos`、およびパレート尾指数の推定値 `ks` を返します。尾指数 k>0.5 の場合、原始推定値の分散は存在せず、尾指数 k>1 の場合、原始推定値の平均は存在せず、PSIS 推定値は大きな変動といくつかのバイアスを持つ可能性があります。

# 引数

  * `log_lik::AbstractArray`: n x m のサイズを持つ配列で、n の対数尤度項 p(y_i|	heta^s) の事後サンプルを含みます。
  * `wcpp::Real`: GPD フィット推定に使用されるサンプルの割合 (デフォルトは 20)。
  * `wtrunc::Float64`: 非常に大きな重みを n^wtrunc に切り捨てるための正のパラメータ。False または 0 を指定すると切り捨てが無効になります。デフォルト値は 3/4 です。

# 戻り値

  * `loo::Real`: 留出法のログ予測密度の合計。
  * `loos::AbstractArray`: 個々の留出法のログ予測密度項。
  * `ks::AbstractArray`: 推定されたパレート尾指数。
