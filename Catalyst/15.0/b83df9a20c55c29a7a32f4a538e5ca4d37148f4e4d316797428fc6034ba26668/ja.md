```
lattice_kymograph(sol, sp, lrs::LatticeReactionSystem, kwargs...)
```

`LatticeReactionSystem`シミュレーションのキモグラフを、直交格子またはマスクされた格子に基づいて作成します。プロットはy軸に区画を、x軸にシステムの状態の時間発展を示します。種の量はヒートマップとして表示されます。

引数（すべての格子）:

  * `sol`: プロットしたいシミュレーション。
  * `sp`: プロットしたい種の値。シンボリック形式またはシンボルとして提供できます。
  * `lrs`: シミュレーションされた`LatticeReactionSystem`。
  * `colormap = :BuGn_7`: キモグラフで種の量を表示するためのカラーマップ。
  * `nframes = 200`: 時系列がサンプリングされる時間サンプルの数。
  * `plot_min = nothing`: カラースケールの最小値（この値未満の値は、カラースケールが計算されるときにこの値に設定されます）。`nothing`の場合、シミュレーションの最小値を使用します。
  * `plot_max = nothing`: カラースケールの最大値（この値を超える値は、カラースケールが計算されるときにこの値に設定されます）。`nothing`の場合、シミュレーションの最小値を使用します。

注意:

  * マスクされた格子の場合、区画に対応しないグリッドポイントの値は表示されません。
  * 現在のプロットインターフェースは進行中の作業であり、修正が期待されます。フィードバックがある場合は、パッケージの著者に連絡してください。
  * 追加の引数は`lattice_plot`に渡すことができ、次にMakieの`heatmap`プロットコマンドに渡されます。
