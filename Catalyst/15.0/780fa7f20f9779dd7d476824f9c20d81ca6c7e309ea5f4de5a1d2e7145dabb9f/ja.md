```
adjacencymat(rs::ReactionSystem, pmap = Dict(); sparse = false)

反応系が n 個の複合体を持つ場合、複合体 i と複合体 j の間の反応の速度定数 R_{ij} を持つ n-by-n 行列を出力します。辞書、ベクトル、または変数と値のマッピングのタプルを受け入れます。例: [k1 => 1.0, k2 => 2.0,...]。

これは、反応速度が重みである加重グラフの隣接行列に相当します。
デフォルトでは記号行列を返しますが、pmap を介してパラメータ値が指定されると数値行列を返します。

この行列と [`fluxmat`](@ref) の違いは非常に微妙です。両方の行列の非ゼロエントリは速度定数です。しかし：
    - `fluxmat` では、行は複合体を表し、列は反応を表し、エントリ (c, r) は反応 r のソース複合体が c である場合に非ゼロです。
    - `adjacencymat` では、行と列の両方が複合体を表し、エントリ (c1, c2) は反応 c1 --> c2 が存在する場合に非ゼロです。
```
