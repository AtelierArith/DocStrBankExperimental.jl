```
setdefaults!(rn, newdefs)
```

`ReactionSystem`である`rn`のパラメータと種のデフォルト（初期）値を設定します。

例えば、

```julia
sir = @reaction_network SIR begin
    β, S + I --> 2I
    ν, I --> R
end
setdefaults!(sir, [:S => 999.0, :I => 1.0, :R => 1.0, :β => 1e-4, :ν => .01])

# または
@parameter β ν
@variables t
@species S(t) I(t) R(t)
setdefaults!(sir, [S => 999.0, I => 1.0, R => 0.0, β => 1e-4, ν => .01])
```

は、`S`、`I`、および`β`のそれぞれに初期/デフォルト値を与えます。

注意：

  * サブシステムや制約システムの種、変数、またはパラメータのデフォルト値を設定するためには使用できません。それらのシステムに対して直接デフォルトを設定するか、[`flatten`](@ref)を使用してそれらを1つのシステムにまとめてからデフォルトを設定してください。
  * デフォルトは、シンボルと値のペアまたはシンボリックと値のペアの任意の反復可能なコンテナで指定できます。
