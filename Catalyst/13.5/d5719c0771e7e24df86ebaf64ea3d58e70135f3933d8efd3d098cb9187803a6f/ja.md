```
reactioncomplexmap(rn::ReactionSystem)
```

指定されたシステム内の各 [`ReactionComplex`](@ref) を見つけ、複合体から基質および生成物として出現する反応を示すベクトルへのマッピングを構築します。

ノート：

  * 各 [`ReactionComplex`](@ref) はペアのベクトルにマッピングされ、各ペアは `reactionidx => ± 1` の形式を持ち、`-1` は複合体が基質として出現することを示し、`+1` は整数ラベル `reactionidx` の反応で生成物として出現することを示します。
  * 定数種は複合体の一部として無視されます。すなわち、種 `A` が定数である場合、反応 `A + B --> C + D` は複合体 `B` と `C + D` から成ると見なされます。同様に、`A --> B` は `0 --> B` と同じように扱われます。
