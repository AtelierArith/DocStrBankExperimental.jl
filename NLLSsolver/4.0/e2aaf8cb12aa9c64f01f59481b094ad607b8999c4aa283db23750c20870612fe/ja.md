```
NLLSsolver.nvars(var)
```

変数ブロック `var` の固有次元数（自由度）を返します。この正の整数は、最適化の期間中固定される必要があります。

`var` のストレージサイズは固有次元数よりも大きくなる可能性があることに注意してください。たとえば、3D回転は固有次元が3つしかありませんが、有用な表現は9要素の3x3行列です。

このメソッドのデフォルト実装は、`Number`、`Vector`、および `StaticVector` の型に対して存在します。

# 例

3次元を持つ変数型 `Rotation3D` の場合、ユーザーは次の関数を定義する必要があります：

```julia
    NLLSsolver.nvars(::Rotation3D) = static(3)
```

!!! tip
    次元数がコンパイル時に既知である場合は、静的整数を返してください。


!!! note
    必要なユーザー特化API関数です。
